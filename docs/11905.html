Unity 和 ironSource 价值 44 亿美元的合并现已完成

Unity 与 ironSource 的拟议合并已经正式结束，两家公司将联手创建一个端到端平台，供开发者开发游戏并从中获利。

以其同名通用游戏引擎而闻名的【Unity 和为开发者提供整合广告、跨渠道营销等工具的广告技术公司 ironSource在 7 月份首次宣布计划联手达成价值 44 亿美元的全股票交易。

截至 7 月份，这两家上市公司的股票在 2022 年分别下跌了约 75%和 50%，他们合并的决定在一定程度上是由经济衰退驱动的，但也是——正如至少一位分析师指出的那样——由去年推出的苹果应用程序跟踪透明度(ATT)框架(T12)驱动的。Unity 和 ironSource 都依赖开发者购买广告来吸引新用户，而 ATT 在这方面制造了摩擦，因此通过集中他们的集体资源，这在一定程度上解决了他们各自的下滑。

“这一改变行业的合并背后的驱动力是在整个开发过程中为开发者创造更多的价值，”ironSource 首席执行官 Tomer Bar-Zeev 在新闻稿中说。“我们对未来的道路感到非常兴奋，因为我们开始更深入地整合我们的产品组合，并加强创作优秀游戏和将其发展成为成功企业之间的反馈回路。通过这样做，我们将能够创造一个有更多创造者比以往任何时候都更成功的世界。”

值得注意的是，在 Unity 和 ironSource 首次宣布他们的计划后的几周内，AppLovin 大张旗鼓地加入了对话，当时它提出了 200 亿美元的收购要约，条件是 Unity 终止与 AppLovin 的竞争对手 ironSource 合并的计划。经过考虑，Unity 最终拒绝了这个提议，其董事会指出 AppLovin 的提议不是一个“更好的提议”
