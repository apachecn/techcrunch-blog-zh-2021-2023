Americana Technologies 从 776 亿美元中筹集了 690 万美元的种子资金

原文：https://web.archive.org/web/https://techcrunch.com/2022/05/03/americana-chip-turns-objects-nfts-funding-ohanian/

NFT 不仅仅被用来证明你在互联网上拥有一张 JPEG 图片。设计师杰克·弗雷(Jake Frey)去年初创立的初创公司 Americana Technologies 允许品牌和创作者将街头服饰、收藏品、汽车和艺术品等实物转化为非功能性物品。

这家初创公司的核心产品是其 NFTA 通用芯片，它可以附着在一个物理对象上，并将该对象链接到区块链。弗雷说，传统艺术界提供了一个有用的例子。想象一下，走到一件艺术品前，用手机扫描它，看到它的整个网络存在和所有权历史都出现在你的屏幕上——这就是芯片如何把任何物体变成 NFT，他解释道。

NFT 创业公司 Americana Technologies 网站的图片。图片来源:美国科技

记录区块链上实物信息的价值在于，NFT 能够证明一件物品的真实性和出处，并提供一种方法，通过这种方法，该物品背后的艺术家或创作者可以在每次转手时赚取版税。

“当我们谈论非功能性文件时，有图像、歌曲、视频和所有这些东西，但(对美国人来说)这种灯泡是所有这些东西都是文件，”弗雷说。"当然，文件在互联网上很容易处理，但是其他东西呢？"

弗雷说，数字艺术家和创作者，如插画师和 3D 设计师，已经从推出 NFT 中赚了数百万，但他们只占创作者的一小部分。他补充说，Americana 背后的最初想法来自弗雷的想法，即“其他 99%的创造者”如何从 NFT 技术中获得价值，他们不是以数字为重点，而是创造实物。弗雷拒绝透露该公司目前的客户数量，也拒绝透露使用其平台的创作者数量。

Americana 刚刚筹集了 690 万美元的种子资金，由 Alexis Ohanian 的风险投资公司 Seven Seven Six 牵头，NFT 市场 OpenSea 和说唱歌手 Future 参与其中。

据弗雷称，社交网络论坛 Reddit 的联合创始人奥哈尼安是这家年轻公司的合作伙伴，因为他在建立、培育和盈利在线社区方面有着良好的记录。

“我想，(通过 Americana)进行交易有不同的用例，然后是收集或炫耀，比如吹牛。普通美国家庭实际上拥有大约几千美元的资产，而且，有史以来第一次，你可以把所有这些东西都放在链子上，你可以吹嘘你在现实生活中拥有的所有酷东西，这些东西已经具有巨大的价值，并且已经围绕它们建立了巨大的社区，”弗雷说。

弗雷计划利用这笔资金增加工程师和设计师，壮大 13 人的团队，并开发一种专注于创造数字资产物理表现的产品。据 Frey 称，该公司目前只支持以太坊区块链店的 NFTs，但随着业务的增长，正在探索与 Solana 和其他连锁店的整合。

“我们的目标是融合这些(物理和数字)世界，所以从两个角度来看，这肯定是我们想要做的事情，”弗雷说。
