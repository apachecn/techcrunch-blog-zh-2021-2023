英国政府正在扫描英国互联网空间寻找零日威胁

英国政府正在扫描英国互联网空间，寻找零日威胁

英国国家网络安全中心启动了一项新计划，该计划将持续扫描托管在英国的每一台联网设备的漏洞，以帮助政府应对零日威胁。

NCSC 是政府通信总部的一部分，作为英国面对公众的网络威胁的技术权威，它说,发起了倡议，以建立“英国的脆弱性和安全性”的数据驱动视图

这与挪威国家安全局的努力相似，去年该机构寻找针对挪威互联网用户的微软交换漏洞的证据。斯洛文尼亚的网络安全响应部门，即 SI-CERT，也在时间表示，它正在通知其互联网空间中的 Exchange 零日漏洞的潜在受害者。

NCSC 的扫描活动将覆盖托管在英国境内的任何可访问互联网的系统，该机构解释说，并将寻找常见的或由于广泛影响而特别重要的漏洞。

NCSC 表示，它将使用收集的数据来创建“漏洞披露后英国暴露于漏洞的概况，并随着时间的推移跟踪他们的补救措施。“该机构还希望这些数据将有助于向系统所有者提供日常安全状况方面的建议，并帮助英国更快地对事件做出响应，如正在被积极利用的零日漏洞。

该机构解释说，从这些扫描中收集的信息包括连接到服务和 web 服务器时发回的任何数据，如完整的 HTTP 响应，以及每个请求和响应的信息，包括请求的时间和日期以及源和目的地端点的 IP 地址。

它指出，请求旨在收集检查被扫描资产是否受漏洞影响所需的最少量信息。如果任何敏感或个人数据被无意中收集，NCSC 表示，它将“采取措施删除这些数据，并防止它们在未来再次被捕获。”

扫描是使用在 NCSC 的专用云托管环境中运行的工具来执行的，这使得网络管理部门可以在其日志中轻松识别该机构。总部位于英国的组织可以通过向 NCSC 发送电子邮件，列出他们希望排除的 IP 地址，选择不接受政府对其服务器的扫描。

“我们不是为了其他邪恶的目的而试图在英国寻找漏洞，”NCSC 即将离任的技术总监伊恩·利维在一篇博客文章中解释道。“我们从简单的扫描开始，慢慢增加扫描的复杂性，解释我们在做什么(以及为什么要这样做)。”
