当你信心十足的时候筹集资金

原文：https://web.archive.org/web/https://techcrunch.com/2022/05/07/fundraise-on-confidence-front-ceo-mathilde-collin-says/

Front 首席执行官玛蒂尔德·科林说，要靠信心筹集资金

初创公司需要资本，通常会从投资者那里筹集资金。这需要投球，数字，数据和故事。时间必须准确无误。这位首席执行官表示，把握时机的关键很简单:当你信心高涨时筹集资金。

每周在 TechCrunch Live 上，投资者和企业家分享从个人经历中学到的教训。Front 首席执行官兼联合创始人玛蒂尔德·科林了解筹款。在几轮融资中，她从风险资本那里筹集了 1.38 亿美元，其中包括弗雷德里克·克雷斯特(Frederic Kerrest)、俄克拉荷马州的首席运营官和风险资本家。他们谈论了几个话题，整个 TechCrunch 直播活动可以在 YouTube 或通过播客观看。

时机可以决定一次融资的成败，科林建议在你感觉良好的时候寻找外部投资——就像你这位创始人感觉良好一样。不幸的是，有时这与你公司的数字并不相关。

“可能是你雇了一个很棒的人，”她说。“你刚刚签下了一个非常大的客户——这让你对这家公司的未来超级有信心。”

为什么？根据科林的说法，投资者非常善于评估创始人的动机是否真实，这围绕着对公司的信心和兴奋。这意味着她总是从她为什么要做某事开始陈述，即使随着规模的扩大会变得更加复杂。

TechCrunch Live 即将举办的活动

弗雷德里克·凯雷斯特同意这一点，他指出，作为一名投资者，他希望把时间花在关心、积极和感兴趣的人身上。

科林说，每次融资时，她都会根据公司的需求对投资者进行评估。然后，到了 Front 的后期 C 轮融资时，她转向了几家运营商，他们可以提供资金以及业内人士对行业和企业的指导。
