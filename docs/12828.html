Discord 开放了付费订阅，因此服务器可以出售额外津贴

去年年底推出高级会员资格试点后，Discord 现在将允许更多服务器提供付费会员资格，以换取特定服务器的特殊津贴。

许多 Discord 社区已经在使用 Patreon 等第三方服务来提供优质内容，但扩展后的功能将允许以 Discord 为中心的社区从应用程序本身内部管理这一过程。随着 Discord 内置的货币化选项，任何运行服务器的人都可以创建定制的会员等级，提供特殊角色和其他好处，如早期访问 merch drops 或独家内容。

Discord 现在称这个功能为“服务器订阅”，它做标签上的事情。要获得资格，服务器所有者必须位于美国(至少目前如此)，并且在平台上信誉良好，最近没有违反其平台政策和社区准则。

订阅为服务器所有者提供了相当大的灵活性，让他们可以更好地赚钱。他们可以将订阅设置为从 2.99 美元到 199.99 美元之间的任何价格，并决定将哪种额外津贴与他们提出的层级相关联。Discord 对订阅收入的提成是 10%。

与 Twitter 不同，Twitter 在游戏后期强行推出付费选项，Discord 用户已经习惯于为应用程序的特殊功能付费。Nitro 是 Discord 的高级版本，为每月订阅者捆绑了高清流媒体、更大的文件上传和自定义表情符号。Discord 爱好者也可以付费“提升”他们最喜欢的服务器，释放所有人都可以使用的社区范围的特权。

除了为服务器所有者提供从内容中赚钱的方法，Discord 还为用户在服务器上增加了更多的事情，除了文本聊天、音频和直播。几个月前，该公司宣布了一套在 Discord 服务器上运行的新互动体验，包括象棋、扑克和社区成员一起观看 YouTube 视频的方式。

Discord 还宣布了一个应用程序目录，如 RPG 迷你游戏和新的运动鞋掉落警报，这些都在服务器内运行(大多数用户过去都称它们为“机器人”))一些精心挑选的开发者可以出售应用程序的高级订阅，让 Discord 更广泛的社区有更多的方法在应用程序本身内赚钱。
