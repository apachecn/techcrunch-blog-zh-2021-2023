Block 的现金应用程序采用闪电网络进行免费比特币支付

去年年底，Twitter 首席执行官杰克·多西辞去了的职位，以便将全部精力投入到他的另一家公司 Square ( 现称 Block )上，该公司已经越来越多地投资于区块链技术和加密货币——就像多尔西本人一样。现在，我们开始看到 Block 拥抱加密的结果，因为今天早上现金应用程序宣布它与闪电网络集成，允许其美国用户向全球任何人免费发送比特币。

在今天之前，该功能已经慢慢向现金应用客户推出，但现金应用尚未正式宣布。该公司表示，预计推出将在“未来几周”完成，覆盖所有美国现金应用客户。

一旦上线，Cash 应用程序用户将能够在国际上向任何外部兼容的钱包发送比特币，包括家人或朋友的钱包或自我管理的钱包，例如 Chivo 钱包、 、T5】 BlueWallet 或 Muun 钱包。用户还将能够向任何接受闪电网络支付的商家发送比特币，费用为零。虽然这还不是主流活动，但一些商家已经开始接受闪电支付，允许顾客通过闪电网络订购披萨或购买礼品卡。

闪电网络与现金应用的整合也有助于推动日益增长的创作者经济，因为如果粉丝接受闪电支付，他们可以发送比特币来表示对个人创作者或事业的支持。

图片来源:闪电网

Cash App 解释了这一系统的优势，指出与闪电网络(Lightning Network)相比，典型的比特币网络交易可能需要一些时间来处理，并且费用更高。闪电网络的名称旨在表达其更快的能力。它的交易也独立于区块链(链外)，进行，这有助于减少否则可能涉及的费用、时间和能源使用。但闪电网络仍将受益于区块链的技术和去中心化，因为网络上发生的交易后来被合并并记录到主要的比特币区块链。

多尔西本人对闪电网络表现出了兴趣，早在 2019 年就发推文称，这是# BitcoinTwitter 用户之间进行实验的一个“很酷的例子”。最近，Block 专注于比特币的公司 Spiral，展示了其闪电开发套件( LDK )，该套件提供了一种将比特币支付轻松集成到任何应用程序中的方法。新的 Lightning integration on Cash 应用程序也由 Spiral 的 LDK 提供支持，Cash 应用程序也是目前第一个也是最大的与 LDK 集成的支付应用程序，该公司指出。

LDK 的推出也是布洛克的战略愿景的一个实例，其中一个分支可以建立被其他布洛克拥有的企业采用的工具。
