谷歌未能推翻欧盟购物反垄断决定

原文：https://web.archive.org/web/https://techcrunch.com/2021/11/10/google-fails-to-overturn-eus-e2-42bn-shopping-antitrust-decision/amp/

图片来源: TechCrunch

谷歌对 2017 年欧盟针对其购物比较服务(Google Shopping)的反垄断调查结果的挑战基本上被欧盟普通法院驳回。

这是欧盟委员会反垄断部门的一次重要胜利，该部门近年来对大型科技公司采取了一系列强制措施，包括针对谷歌的多项裁决。但是今年夏天输掉了一场针对苹果补税的大官司。

今天，欧盟普通法院维持了四年前对谷歌及其母公司 Alphabet 处以的 24.2 亿€的罚款，原因是谷歌购物(一种产品比较搜索服务)相关的竞争滥用。

尚不清楚谷歌是否会寻求针对谷歌购物的进一步上诉。当被问及此事时，一位发言人拒绝置评。

早在 2017 年，委员会发现谷歌滥用其在搜索领域的主导地位，将同名购物比较服务放在显著位置，同时在有机搜索结果中降低竞争对手的排名。

谷歌及其母公司 Alphabet 对该决定提出上诉，但普通法院驳回了他们的大部分诉求——同意受制裁的行为是反竞争的，谷歌更喜欢自己的比较购物服务，而不是提供更好的结果。

在一份关于判决的新闻稿中，法院还指出了另一个有问题的策略，写道:“虽然谷歌后来确实通过出现在它的‘盒子’中以换取付款，使竞争的比较购物服务提高了搜索结果的显示质量，但普通法院指出，这种服务取决于比较购物服务改变其商业模式，不再是谷歌的直接竞争对手，而是成为它的客户。”

在额外的调查结果中，法院同意谷歌的反竞争行为对其竞争对手造成了有害影响。它拒绝了谷歌的一个论点——谷歌试图声称，由于该市场上存在商业平台，比较购物服务的竞争仍然很激烈——同意欧盟委员会的评估，即这些平台不在同一市场上。

谷歌的一个亮点是，法院发现委员会没有证明这家科技巨头的行为对普通搜索服务市场产生了(甚至是潜在的)反竞争影响。因此，法院取消了仅针对该市场的侵权认定。

但是，它再次支持委员会对比较购物专业搜索服务市场的分析(以及谷歌在其中的反竞争行为)。

法院还驳回了谷歌的主张，即其行为客观上是合理的，因为它“提高了其搜索服务的质量”。

此外，它还驳回了谷歌提出的另一项指控，即技术限制阻碍了谷歌提供平等待遇。

“谷歌没有证明这种做法带来的效率提升会抵消其对竞争的负面影响，”它在新闻稿中补充说。

在维持委员会施加的罚款水平时，法院指出，它所废除的决定部分对罚款数额没有影响(“因为委员会在确定基本罚款数额时没有考虑该市场的销售额”)；并强调了所谓的“侵权行为的特别严重性质”——同时考虑到该行为是故意的，而不是过失的。

欧盟委员会表示，该判决“传递了一个明确的信息，即谷歌的行为是非法的，它为市场提供了必要的法律澄清”。

“在电子商务对零售商和消费者变得越来越重要的时候，比较购物为消费者提供了一项重要的服务。该委员会在声明中说:“如今，数字服务在我们的社会中已经无处不在，消费者应该能够依靠它们来做出明智和公正的选择。”。

它补充说，它将继续使用“一切可用的工具来解决企业和用户分别依赖的大数字平台的作用，以访问最终用户和访问数字服务”——指的是它的数字市场法案监管提案，该提案目前正在由欧洲议会和理事会辩论，它表示旨在确保“公平和可竞争”。

谷歌发言人在自己的声明中回应了这一裁决，试图淡化任何意义，他写道:

购物广告总是帮助人们快速、轻松地找到他们想要的产品，并帮助商家接触到潜在客户。这一判决与一系列非常具体的事实有关，虽然我们将仔细审查它，但我们在 2017 年做出了修改以遵守欧盟委员会的决定。我们的方法已经在成功运作了三年多，为 700 多家比较购物服务产生了数十亿次点击。

然而，谷歌在本地搜索领域的一个竞争对手 Yelp 抓住了这一判决，称其建立了一个“迅速评估其他垂直领域此类行为的非法性”的框架，并呼吁欧盟委员会就本地搜索对谷歌采取行动。

“ Yelp 欢迎欧洲普通法院今天的裁决，该裁决认定谷歌滥用其在普通搜索领域的主导地位，在没有任何效率理由的情况下，消灭垂直搜索服务领域的竞争对手，”SVP 公共政策的 Luther Lowe 在一份声明中表示。“虽然该决定侧重于比较购物，但它建立了一个框架，用于快速评估其他垂直行业(即本地搜索)中此类行为的非法性。本地搜索市场尚未倾斜，欧洲消费者每周在谷歌上进行数十亿次本地搜索。

“与其接受一场得不偿失的胜利，欧盟委员会现在必须抓住这个有利的先例，起诉谷歌在本地搜索市场的类似侵权行为，并允许像 Yelp 这样的服务根据其价值进行竞争，”他继续说道。“现在可能很难记起，但谷歌及其大型科技公司并不总是这么不受欢迎。2015 年，副总裁韦斯塔格尔展现了令人难以置信的勇气，向世界表明谷歌的滥用行为是不可接受的。她这种勇气应该受到赞扬。

“但历史对这一时期的判断将基于这一奥德赛最终是否对欧洲消费者产生了切实的影响，这就是为什么在竞争仍可挽救的市场中使用这些工具至关重要。”

与此同时，谷歌有一系列针对其他欧盟反垄断调查结果的上诉( Android 和 AdSense )，以及对其广告技术的公开欧盟调查——更不用说在本土的一系列反垄断案件了。

因此，不管今天的损失会带来怎样的后果，它的律师们都会非常忙碌。
