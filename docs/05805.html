这就是为什么我们即将在网上看到超现实人造人的爆炸

原文：https://web.archive.org/web/https://techcrunch.com/2022/04/25/heres-why-were-about-to-see-an-explosion-of-hyper-real-artificial-humans-online/

合成生成的真人版本可以被编程说任何听起来像最新一集《黑镜》中的场景。

但事实上，基于真人的制作级视频角色——可以以超级链接的方式谈论任何产品或主题——可以说将成为电子商务和远程学习等领域下一波浪潮的一部分。此外，好莱坞名人可以简单地授权他们的虚拟形象来解释产品，其规模之大，使得实际拍摄变得不可能。但也许更现实的是，像这样的“数字双胞胎”制作的视频比虚拟角色更有说服力，因为它们具有类似人类的品质。

这项技术的市场正在扩大。该领域的主要参与者包括 SoulMachines(已融资 1.35 亿美元)和 Synthesia(融资 6660 万美元)。

早在 2020 年，我们报道了纽约和特拉维夫的初创公司 Hour One 如何基于真实人类创造人工智能驱动的合成角色，完成了一轮 500 万美元的种子资金。

它现在获得了由 Insight Partners 牵头的 2000 万美元的首轮融资。参与该轮投资的还有银河互动、Remagine Ventures、Kindred Ventures、Semble Ventures、Cerca Partners、Digital-Horizon 和 Eynat Guez。

这家初创公司计划扩展其 Reals 平台，这是一个自助服务平台，允许企业“在几分钟内从文本中自动创建人工视频”。

该公司表示，这将人们转化为商业和专业用例中的虚拟人物。人类首先被视频捕捉，然后 Hour One 的人工智能生成一个虚拟双胞胎。例如，这可能是一个虚拟的接待员、销售人员、人力资源代表或语言教师。

在某种程度上，Hour Pen 的观点是正确的，即向远程工作的转变意味着视频和更具沉浸感的媒体(如教育内容)变得更加重要。因此这种视频的人群将会扩大。

Hour One 首席执行官兼创始人柳文欢·阿哈龙在一份声明中说:“很快，任何人都将能够拥有一个虚拟的专业用途的双胞胎，可以代表他们发送内容，用任何语言说话，并以以前无法想象的方式提高他们的生产力。”

Insight Partners 董事总经理 Lonne Jaffe 补充说:“生殖人工智能的能力和准确性继续以极快的速度提高，Hour One 处于领先地位。”“你只需键入一些文本，在幕后，令人难以置信的可扩展的 Hour One 基础设施创建了一个流畅而逼真的视频，视频中的化身在说话，并配有相应的语音和图形。该团队的宏伟愿景是能够将这种非凡的能力嵌入到任何软件产品中，或者允许通过 API 实时调用它。”

语言和文化培训巨头 Berlitz 现在使用 Hour One 制作视频，在其数千个视频中提供虚拟讲师。Hour One 还与 NBC 环球、梦工厂和 Cameo 合作，后者允许名人为粉丝录制付费视频。

SoulMachines、Synthesia 和 Hour One 等产品的出现引发了关于这项技术可能会被滥用的问题。将视频标记为“人造的”可能是防止这种情况的一种方法，但我们仍然在未知的水域中游泳。Hour One 表示，对于如何使用其技术，它有一套道德政策准则。

我们肯定会看到围绕这项技术出现一些“有趣”的场景，这项技术的扩散速度比创业公司本身快得多。
