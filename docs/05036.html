Digits 以 5.65 亿美元的估值入账 6500 万美元，为传统会计工具带来更加动态、自动化的方法| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/03/24/digits-books-65m-on-a-565m-valuation-to-bring-a-more-dynamic-automated-approach-to-legacy-accounting-tools/

Digits 是一家初创公司，它正在通过一种被称为建立公司财务活动“活模型”的方法，对会计软件进行新的尝试。该公司已经获得了一些自有资金，以加倍发展其业务。该公司已经筹集了 6500 万美元，首席执行官兼联合创始人 Wayne Chang 向我证实，这家初创公司的估值为 5.65 亿美元。

在首次推出隐形模式 并积累了令人印象深刻的 1.6 万人的等候名单(以及 9750 万美元的资金)后，Digits 直到现在仍处于只有受邀者才能进入的阶段 。但是今天，为了配合融资新闻，它发布了它的第一个产品:一个叫做 Digits Reports 的报告工具。 (“悬停发现！”这是常在向我描述它的工作原理时差点唱出来的口号。)

Digits 也将向更多用户敞开大门。Chang 说，现在，它一直在有意地与“仅仅几百个”合作，以帮助教授它的算法，并了解客户中可能出现的不同用例。

软银的愿景基金(Vision Fund)领投，英国投资者哈里·斯特宾斯(Harry Stebbings)的 20VC Growth、GV 和 Benchmark 也参与其中。GV 和基准分别领先前几轮，目前总计 9750 万美元。

Digits 采取了一种非常合作的方式来积累支持者:在它的资本表上有 72 个独立的天使投资人。Digits 的吸引力(和牵引力)部分来自其创始人的过往记录:Chang 和 Jeff Seibert 还创建了 Crashlytics，这是一家车祸报告服务公司，首先被 Twitter 收购，再将其出售给谷歌，谷歌将其整合到 Android 中。它现在被用于数十亿台设备和数百万个应用程序。

你可能想知道构建一个非常专注于开发人员的工具和构建一个会计平台之间的界限是什么，但正如 Chang 所描述的那样，这种联系对他和 Seibert 来说是显而易见的。

“有直接的相似之处，”塞伯特告诉我。“在 Crashlytics，我们专注于产品和开发人员方面，这都是关于应用程序执行情况的分析。通过处理大量的数据，可以给这些团队以前没有的洞察力。但令我们震惊的是，这些公司对其(整体)业务的透明度如此之低。”他说，Digits“正在使用非常相似的方法，但应用于公司的另一半。”

这种精神在公司生产的产品中显而易见。

Digits 本身并不是一个数据摄取工具:Chang 指出，它本质上位于 Intuit 的 QuickBooks 之上(注意:这一选择是故意的，因为 QuickBooks 目前占美国小型企业会计软件市场的 80%，尽管随着时间的推移，Digits 将根据需求与其他来源合作)。然后，它使用这些数据，再加上基于 API 的与公司用来管理其业务收入和支出的任何其他东西的集成，从本质上创建了一个巨大的信息数据库。

然后，Digits 开始组织和读取这些数据，以围绕这些数据创造更多的智能。在新报告产品的情况下，它为会计师或其他财务专家可能围绕基本财务报告提出的各种“如何”或“为什么”问题提供了自动答案，这种答案以前只能通过人工查询以及能够阅读和理解书面记录和孤立数据源背后的故事来提供。

Digits Reports 将与 Digits 去年推出的一项搜索功能并列，并以该功能为基础，帮助用户找到与 Digits 类似的交易:答案不仅仅是通过关键词找到的，而是在结果中找到的，这些结果直观地显示了财务专家真正想要弄清楚的东西。

“以前你不得不搜查文件柜，甚至鞋盒，”常说，他坚称自己并没有过分夸张。"整个行业都非常陈旧。"

目前，该公司的产品主要专注于查询历史信息:查看金融交易和其他已经发生的事件，以理解它们。但其明显的进展是将 Digits 的活体模型应用于实时分析——对财务人员来说更贴切——未来报告。(另一家为小型企业开发下一代财务工具的初创公司 DataRails 恰当地将未来报告和建模称为这些财务专业人士的“圣杯”。)

常证实，这肯定是对该公司的卡，但拒绝透露更多。

然而，当与我分享他的屏幕来演示“悬停发现”工具时，我注意到 Chang 桌面上的另一个产品，他很快就不再说它仍在开发中。

它被称为“烧钱”,我有根据的猜测是，它可能与让用户——最有可能来自 Digits 服务所针对的小企业——看到一家公司的现金流在一段时间内的表现有关，以便确定它在这段时间内的跑道和烧钱率。这反过来可以帮助一家公司弄清楚，如果它需要这些数字以不同的方式发展，或者相反，如果它有能力投资资金来促进活动，它可能需要在哪些方面花费更少(或更多)。

活模型的另一个有趣之处是，在更广泛的层面上，它可以应用于不仅仅是会计查询，因为它提供了一个公司财务运作的完整画面。常告诉我，自 2019 年推出以来，Digits 已经抵挡了三次收购要约。

他没有具体说明是谁在接洽，但证实了不只是其他更大的会计平台感兴趣。

“我们知道我们正在建造别人没有的非常引人注目的技术，”他说，“而且这个活生生的模型也适用于其他东西。比如你是银行，想申请不同的放贷模式，我们的平台可以自动告诉你一个公司的财务状况。我们只是触及了我们能做的事情的表面。”

目前，它从会计角度找到了足够的吸引力，用户正在快速迁移到更多的数字工具，并在他们的体验中寻求更多的自动化和洞察力。这也有助于 Digits 成为其产品的签约用户。

“企业主很少考虑这类事情，只有当有人问‘为什么会超出预算？”这是他对似乎正在进化的传统思维方式的描述。

这就是投资者想要加入的原因。

“我们投资组合中的一家公司对其他公司有价值的情况很少见，”软银愿景基金的投资者 Eylul Kayin 说。 "Digits 对他们所有人都有用。”
