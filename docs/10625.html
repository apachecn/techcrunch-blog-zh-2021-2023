Solana 开发者平台 Coral 筹集 2000 万美元由 FTX 牵头，Jump Crypto 打造 web3 的 iPhone TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/09/28/solana-coral-ftx-jump-crypto-iphone-web3-apps-xnfts/

在竞争激烈的一线区块链，开发商是增长的巨大催化剂。无论开发者去哪里，他们都会构建应用程序和健壮的生态系统，用户的活动和兴趣也会随之而来。

Coral 在为索拉纳区块链生态系统构建开发工具方面发挥了重要作用。这家初创公司创建了 Anchor ，据称这是 Solana 最受欢迎的智能合约开发者框架。Anchor 类似于以太坊的 Hardhat 或 Web 2.0 世界中的 Ruby on Rails，创始人 Armani Ferrante 在接受 TechCrunch 采访时表示。

该公司今天宣布，它已经筹集了 2000 万美元的战略轮融资，由 FTX 风险投资公司和 Jump Crypto 领投，参与了 Multicoin Capital、Anagram、K5 全球等公司的投资。它计划利用这笔资金推出一款名为 Backpack 的产品，这是一款交互式加密钱包，据称通过该公司所谓的可执行 NFTs (xNFTs)提供加密体验。

费兰特通过一个类比来解释产品——苹果的 iPhone。Ferrante 说，背包钱包就像 iPhone 的 iOS 操作系统，用户可以管理自己的私钥并访问 xNFTs，这就像 iPhone 上的应用程序。

“在背包之上，我们有 xNFTs 这个概念。因此，我们不仅有这些分散的协议，而且还有分散的应用程序或用户界面，我们可以为 xNFTs 提供这种精心策划的体验，”Ferrante 说。

Coral 还提供了一个名为 ReactxNFT 的开发工具，Ferrante 称该工具可以与苹果的 Swift 和 UIKit 软件包相媲美。

“我们有能力在这个平台上构建，并利用这些 API 在 Backpack 上构建引人注目的用户体验，”Ferrante 说。

最终，Ferrante 希望将 Backpack 开发成一个“独立的网络”,通过 xNFTs 帮助 web3 用户建立多链身份的概念。

“我可以做一些事情，比如用 xNFTs 玩游戏，我可以做一些事情，比如看看他们是否在线和连接，以及他们在区块链上做什么。我认为将它作为一个原语来构建并向应用程序公开将会是一个非常强大的功能。费兰特说:“这将是一种独特的东西，它将所有这些结合在一起，创造一种真正的社交体验，解决我们今天在 web3 中看到的许多问题。”。

珊瑚的背包平台上的 xNFTs。图片来源:珊瑚

Backpack 周围的 Discord 社区目前约有 2000 名成员，Ferrante 表示，这些成员一直被有意锁定，直到今天宣布 Coral 将在私人测试版中推出 Backpack 并开源其代码，给予 Discord 成员优先访问权。Ferrante 说，Solana 上超过 10 个最大的项目已经在利用 Backpack 的协议。

当人们考虑到作为一个生态系统，索拉纳选择投资硬件时，费兰特的 iPhone 类比似乎特别恰当。Solana Labs，开发和改进 Solana 协议的公司，在今年夏天早些时候推出了一款专注于 web3 的智能手机，名为 Saga。Ferrante 表示，Coral 团队正在与 Saga 背后的团队保持密切联系，他最终希望看到 xNFTs 作为智能手机上的原生应用程序运行。

“就像你可以去 Saga 应用商店下载(像 Magic Eden 这样的应用)一样，这里也会有一个 xNFT 区。你可以下载 xNFTs 并在你的主屏幕上看到它们，”Ferrante 说。
