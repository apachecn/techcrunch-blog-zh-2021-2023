脸书在 F8 会议上宣布的一切

在脸书 F8 开发者大会的第一天，马克·扎克伯格展示了公司的 10 年路线图，并发布了大量新消息，如新的聊天机器人和一款疯狂的新 VR 相机。

在第二天，主题演讲人讨论了 React 原生平台的更新，演示了 VR 自拍杆，介绍了两个连接项目等等。

从直播 API 到聊天机器人，再到新的虚拟现实摄像机，这里有你需要了解的 2016 F8 的一切。
