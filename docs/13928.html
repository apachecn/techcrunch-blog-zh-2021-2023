微软现在在美国有了第一个官方工会

原文：https://web.archive.org/web/https://techcrunch.com/2023/01/03/microsoft-zenimax-union/

在微软旗下的游戏工作室 ZeniMax，300 名质量保证(QA)测试人员投票加入了工会。这标志着微软在美国的第一个官方联盟，也是该国最大的游戏联盟。

视频游戏行业的工会组织一直在增加，尤其是在质量保证人员中。2022 年，动视暴雪的 QA 测试人员已经通过通信工人联盟(CWA)在渡鸦软件和暴雪奥尔巴尼成功组建工会，这也将代表 ZeniMax 的工会。但是当动视暴雪试图处处阻挠工会组织的时候，微软在 6 月承诺不会阻碍员工组织工会。因此，通过赢得大多数支持工会化的选票，微软将正式承认 ZeniMax 工人联合会。

“根据最近工会投票的结果，我们承认美国通讯工人(CWA)为 ZeniMax 质量保证员工的谈判代表。微软和 ZeniMax 的一位发言人表示:“我们期待着在达成集体谈判协议的过程中进行真诚的谈判。”。

ZeniMax 包括 Bethesda Softworks 和 id Software 等子公司，这些子公司生产《上古卷轴》、《毁灭战士》和《辐射》等系列游戏。

“微软兑现了对员工的承诺，让他们自己决定是否要工会，”美国通讯工人协会主席克里斯·谢尔顿在一份声明中说。“当其他视频游戏和科技巨头联合起来成立工会时，他们有意识地选择攻击、削弱和挫伤自己员工的士气。微软正在制定一条不同的路线，这将加强其企业文化和服务客户的能力，应该成为行业的典范和监管机构的蓝图。”

id Software 的高级质量保证测试员 Zachary Armstrong 上个月告诉 TechCrunch 说，该部门正在组织起来争取更好的工资和工作条件。

“当涉及到游戏开发时，QA 测试人员总是被放在图腾柱的底部，以至于我们甚至不被认为是游戏开发者。”阿姆斯特朗说。QA 人员严格测试视频游戏的所有方面，以识别和解决影响用户体验的问题。“这反映在我们的薪酬中，也反映在我们的工作中，特别是在 crunch 方面。”

在截止日期之前，QA 测试人员可能会工作不可持续的时间来准备一款游戏供公众观看——这被称为“紧缩”在动视暴雪成立工会的 QA 测试人员也对关键时刻的预期表示了担忧。在宣布他们加入工会的意图之前，渡鸦软件质量保证测试人员举行罢工，抗议影响 12 名承包商的裁员——在那些合同被终止之前，质量保证测试人员已经连续五周加班。

ZeniMax 工会的下一步是与微软谈判他们的合同。

高级音频测试员维多利亚·巴诺斯在一份声明中说:“摆在我们面前的是一个做出重大改变并为视频游戏行业带来公平的机会。”。“我们希望结束公司内部突然出现的危机、不公平的薪酬和缺乏发展机会的状况。我们的工会将推动真正有竞争力的薪酬、管理层和工人之间更好的沟通、为那些想要发展职业生涯的人提供一条清晰的道路，等等。”

更新，1/3/23，美国东部时间晚上 8:00，来自微软和 ZeniMax 发言人的评论。
