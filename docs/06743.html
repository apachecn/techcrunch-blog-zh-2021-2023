纳米比亚 B2B 电子商务零售平台 JABU 融资 1500 万美元，由 Tiger Global TechCrunch 领投

原文：https://web.archive.org/web/https://techcrunch.com/2022/05/25/namibian-b2b-e-commerce-retail-platform-jabu-raises-15m-led-by-tiger-global/

更多的风险资本不断涌入非洲的 B2B 电子商务零售领域，在这个领域，初创公司正在将非正式贸易数字化，以使成千上万的商家更有效地运营。

这一次，它是今年 1 月新闻中的一家初创公司:总部位于纳米比亚的 JABU ，我们报道了其320 万美元的种子轮。现在，最后一公里配送电子商务公司获得了更多投资:由 Tiger Global 领投的 1500 万美元 A 轮投资。

这一轮于 3 月的某个时候结束，是 Tiger Global 在支持 Wasoko 的大型 B 轮之后在 B2B 电子商务领域的第二次投资。同样值得注意的是，这笔交易是在 Tiger Global 的报告在今年科技股抛售期间损失 170 亿美元之前完成的。

这轮增长轮的其他投资者包括 Box Group、Knollwood 和 D Global Ventures。一些种子轮的支持者:Afore Capital、Oldslip 和 FJ 实验室也双倍下注。

多年来，在非洲各地拥有中小型售货亭和商店的商人在向批发商和分销商订购产品时，都要处理物流问题。像 JABU 这样的初创公司和其他几家公司，如 Wasoko、 TradeDepot 、 Omnibiz 、 MarketForce 、 MaxAB 和 Chari 已经通过应用程序和更有效的分销渠道使这一过程变得更容易。

这家 YC 支持的公司表示，有了 JABU，商家可以通过 Jwallet 订购、库存和支付产品，并期待当天送达。今年 1 月，该公司在纳米比亚、南非和赞比亚有超过 6000 名商户使用其平台。首席执行官大卫·阿基宁说这个数字增加了 50%。

该公司还通过仪表板向 FMCGs 品牌和银行提供销售指标和代理绩效等数据驱动的服务。

最后，阿基宁表示，JABU 希望围绕其 Jwallet 进行开发，该钱包系统目前作为独立产品推出。Jwallet 允许南部非洲的商户使用他们的实物流量为他们的客户提供现金提取和存款服务。这种做法类似于代理银行，即尼日利亚和西非的无分支银行系统，由人工代理充当自动取款机，在偏远地区提供金融服务。Chari 在摩洛哥为提供了同样的服务。

阿基宁说:“你可以让你的终端消费者通过 JABU 向他们的钱包和银行账户存钱和取钱。“因此，我们将一个 API 连接到交换的银行，实际上允许通过钱包收款的人走到一个 JABU 商人面前，他可以使用他们的物理浮动来取款。”

Jwallet 还有其他可以工作的部分。据该公司称，为其 232 个物流合作伙伴处理配送并使用钱包进行支付的司机可以获得资产融资，对于商家来说，还可以获得股票融资。关于后者，阿基宁说，他打赌钱包系统可以为其他平台向商家提供的流行的 BNPL 模式提供更可持续的替代方案。

“我认为现在买，以后付是一种错觉。我认为有信贷，也有现金销售，没有任何介于两者之间的东西，”阿基宁说。“所以，我认为它(BNPL)将使小企业的情况恶化，它将创造更多的违约，产生一种低工资的文化，它将在非洲的中小企业层面产生真正的债务，这将很难证明是合理的。”

他的偏见来自于 JABU 过去尝试 BNPL 模式时的违约经历。阿基宁讲述了商家将如何使用平台的 BNPL 产品，产生收入，并利用这一利润支付下一张发票，或从完全不同的供应链中的另一家供应商那里购买库存。

Jwallet 通过与银行合作进行数字支付，并为商家创建社区，以便在平台上相互储蓄和提供信用额度，从而规避了这一问题。这一过程也有助于商家积累交易历史，同时他们从向最终客户提供金融服务中获得足够的收入来偿还。

“我们用邪武钱包做的很多事情是围绕社区和商店而不是围绕我们的资产负债表创建一个生态系统。作为一种产品，我们对此感到兴奋，因为我们试图证明，随着我们规模的扩大，有一种不同的方式与商店互动。”

首轮融资将见证 JABU 深化其在南部非洲的业务，并在今年晚些时候扩展到博茨瓦纳和埃斯瓦蒂尼等新市场。阿基宁说，他的创业公司与其他公司不同的地方在于，它如何为小企业创造一个更广泛的生态系统，而不仅仅是一个市场。

“许多像我们这样的企业正在将资金撤出市场。我们正试图建立一个业务，将产品带入市场，并在一个具有乘数效应的旅程中继续，将资金在市场中流动 20 倍。我认为这就是建立 JABU 钱包的目的。会有共享服务；其他产品将由此产生，为这些市场的服务和产品付费的能力也将如此。”
