微软将 JavaScript 引入 Excel

在 Ignite 大会上，微软今天宣布了对 Excel 的更新，为古老的电子表格应用程序带来了新的 JavaScript API。有了这个新的 API，开发人员将能够基于它们创建定制的数据类型和函数。

在过去几年中，Excel 开始添加许多新的数据类型，最初允许用户从云中提取股票和地理数据，随后添加了 Power BI 和 Power Query 数据类型，允许用户处理自己的数据。现在，开发人员可以更进一步，创建对他们的组织有意义的自定义数据类型。

“客户将能够创建他们自己的加载项，并扩展以前存在的加载项以利用数据类型，从而在 Excel 中获得更加集成的下一代体验，”微软在其新闻材料中解释道。“他们可以在整个组织内共享这些数据类型，并创建将数据类型连接到他们自己的服务或数据的插件或解决方案。该 API 为用户和开发人员提供了强大的功能，使他们能够更好地组织、访问和使用数据。”

微软已经与彭博合作，帮助该公司试用这些自定义数据类型。
