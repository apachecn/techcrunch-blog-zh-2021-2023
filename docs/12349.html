现代的氢燃料电池概念暗示了性能 N 品牌的未来

只是不要说它是受德罗宁的启发

现代汽车周四发布了一款名为 N Vision 74 的氢燃料电池混合概念车，该公司表示，这款车展示了高性能子品牌对电气化的愿景。

现代 N 子品牌是该汽车制造商专注于性能的部门，自 2015 年成立以来，已应用于一系列量产车，从现代 Veloster N 和伊兰特 N 到 Kona N。

N 品牌的灵感来自德国著名的纽伯格林赛道，现代汽车在这里测试这些车型，其 N 品牌瞄准了宝马 M、奔驰 AMG、奥迪 RS 和凯迪拉克 V 系列等豪华性能品牌。

N Vision 74 提高了赌注。只是不要说是受德罗宁的启发。

近距离观察 N Vision 74

图片来源:阿比盖尔·巴塞特

现代称 N Vision 74 为“滚动实验室”，是未来产品的试验台。虽然也有一些现代小马跑车的历史。

N Vision 74 向 1974 年的现代 Pony Coupe 概念车致敬，这款车是由传奇汽车设计师乔治亚罗·乔治亚罗开发的，他也是 DeLorean 的设计者。(DeLorean 于 1981 年在 Pony Coupe 之后首次亮相。)现代全球设计中心和现代汽车公司执行副总裁 Sang Yup Lee 很快指出了这个细节。

“不要说他们长得像，因为是我们先做的，”李在新闻发布会上说。

N Vision 74 拥有独特的氢混合动力和电池电动架构。支撑 N Vision 74 概念的是燃料电池组和电池组。前部的燃料电池堆输出 85 千瓦(最大 95 千瓦)，而 62.4 千瓦时的电池位于后部。

氢 W 燃料电池将氢转化为电能，为 62 千瓦时的电池充电。该车还在每个车轮上安装了独立的后置电机，以产生 500 千瓦的总功率输出，近 670 马力和 663 磅-英尺的扭矩。现代表示，这使得工程师能够调整左右车轮之间的动力分配，并优化 N Vision 74 的设置，以处理不同类型的轨道。

N Vision 74 概念车拥有双充电功能。由于底层的 800 伏架构 E-GMP 平台，它可以充满氢气或在 DC 快速充电器上充电。现代表示，它可以行驶 372 英里，最高时速 155 英里。

问题当然是这种氢燃料电池混合动力技术会来到量产车上吗？

现代不愿透露这种动力系统是否会投入生产。然而，李在结束他的演讲时说，“N Vision 74 概念具有不可否认的现代 DNA 和设计，可以作为指引我们未来的指南针。”
