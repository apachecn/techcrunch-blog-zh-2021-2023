如何计算你的初创公司的 TAM、SAM 和 SOM

对于许多首次创业的人来说，计算他们希望参与竞争的市场规模是他们面临的最大挑战之一。

计算 TAM、SAM 和 SOM 听起来像是一项存在主义练习，但“如果你有条不紊地估算市场规模”，资深投资者和企业家 Marjorie Radlo-Zandi 说。

在 TechCrunch+的一篇综合文章中，她分解了捕捉这些关键指标所需的步骤，这些指标“向潜在投资者展示他们如何从投资你的公司中获益，并将你自己置于实现目标的最佳位置。”

  • Twitter 让模式切换变得简单 : 我喜欢 Twitter 的一点是，我可以以时间序列格式获取我的推文。这是我想与服务互动的唯一方式。但有些人更喜欢让 Twitter 的霸王机器人(读作:算法)为他们排序。现在你可以在它的应用程序中更容易地在两者之间切换。

  • DoorDash 想帮你退包裹 : 送餐的市场不是无限的。几乎在你能想到的每个市场都有多个竞争对手，这是很有竞争力的。随着公开市场增长预期的加入，DoorDash 寻找 TAM 支持者也就不足为奇了。
