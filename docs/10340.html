特使于 2013 年推出，承诺将个人公司的访客管理带入现代。当疫情袭击时，它迅速转向根据当地法规、健康状况和疫苗接种情况等因素控制谁可以进入办公室。

今天，该公司宣布了该系统的下一步，一个面向高层建筑中所有租户的互联建筑管理系统。最终目标是开发一个完整的楼宇管理平台，在某个时候，客户可以在现有软件的基础上构建附加组件和定制。

但是公司首席执行官 Larry Gadea 表示，今天的公告为公司的未来愿景奠定了基础。

“我们现在推出的是(我们最终可以让客户在平台上构建应用)之前必须完成的第一步，即大楼中的所有租户都使用相同的软件，并且他们都连接在一起。然后我们用这位访客的产品证明了这一点，该产品现在能够自动将所有租户同步到一起，并同步到楼下，”Gadea 解释道。

在一篇宣布这些变化的博客文章中，他这样描述新系统:

我们正在解决我们的客户目前在高层建筑方面的大多数主要痛点，以使每个人的事情变得更容易:访客不再需要登录两次，员工为其办公室发出的邀请将立即传播到大楼的安全大厅，邀请的更改将实时同步，十字转门和电梯访问控制将自动提供，等等。

在今天的公告之前，Envoy 专注于帮助建筑物内的单个租户管理访客，但通过 Envoy Connect，租户、建筑物所有者和楼下的安全服务台可以自动同步，而不必明确告诉安全人员访客即将到来。

所有这些都将在一个集中的仪表板上显示给物业经理，使他们更容易看到和了解在任何给定时间谁在大楼里。

特使连接访问者仪表板。图片来源:特使

该公司在过去几年里增加了一些东西，如管理办公桌分配的办公桌和管理建筑交付的交付，在许多方面都是对不断变化的办公环境的反应。在疫情办公室关闭之初，然后是受限访问，最后是今天，即使我们恢复了一些正常的表象，混合办公室仍然是一个现实，了解谁在大楼里变得更加重要。

广泛的愿景是 Envoy 将不必继续自己构建这些组件，而是将提供一个软件开发工具包或 API 访问，让客户自己构建他们需要的部分，但 Gadea 表示这可能需要一两年的时间。此外，该公司希望更容易最终创建更流畅的办公环境，让大楼租户和业主可以通过该平台进行管理，以利用不断变化的工作性质。这可能包括预订桌子、会议室，甚至订购食物和咖啡，即使他们不一定在大楼里有永久的空间。

今年早些时候，该公司宣布了由 Brookfield 的投资部门 Brookfield Growth 牵头的大规模1.11 亿美元战略投资。这项投资对该公司的估值为 14 亿美元。值得注意的是，Brookfield 是一家房地产开发公司。

门罗风险投资公司(Menlo Ventures)的马特·墨菲(Matt Murphy)在 2018 年领导了该公司 4300 万美元的 B 轮投资，他表示，今天的公告是这笔钱被导向特使的一个重要原因。

“Connect 是 Envoy 的多租户产品，是 Brookfield 领先 1.11 亿美元的 c 轮融资的主要驱动力之一。最终，它将导致桌子/房间/停车场等的市场。而 Connect 是实现这一目标的第一步，”墨菲告诉 TechCrunch。
