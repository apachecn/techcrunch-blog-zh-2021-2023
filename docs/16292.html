Canva 推出了一系列新功能，包括几个人工智能工具| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/22/canva-new-features-including-suite-of-ai-powered-tools/

Canva 正在推出一系列新功能，包括一套人工智能工具和一个新的品牌中心。该公司在今天的 Canva Create virtual 活动中推出了新的功能和工具。

该公司正在推出助手，让用户搜索设计元素，并提供快速访问功能。该工具还可以向您推荐与您现有设计相匹配的图形和样式。Assistant 提供快速访问人工智能驱动的设计工具，如 Magic Write，这是该平台于 12 月推出的人工智能驱动的文案助手。

图片来源: Canva

从今天起，Magic Write 将在所有 Canva 项目类型中可用，从演示文稿到社交媒体图形、传单等等。在此扩展之前，Magic Write 只能在 Canva Docs 中访问。

您可以使用 Magic Write 来编写网站文案或演示摘要。例如，你可以要求 Magic Write 为新产品发布列出社交媒体战略想法。当建议复制时，该工具会考虑您正在处理的项目类型。除了扩展的可用性之外，Canva 还宣布 Magic Write 现在有 18 种语言版本。

Canva 还推出了一种自动生成演示文稿的新方法。您可以提示编辑在每张幻灯片上创建一系列带有大纲和内容的演示文稿。至于设计，该公司的新“魔术设计”工具让用户上传一张图片，然后选择一种风格，以获得个性化模板的精选，你可以定制。

我们还有一个新的“魔术编辑”功能，让您添加或替换图像中的任何内容。你可以确定你想在哪里添加东西，然后向 Magic Edit 描述它。一个新的“神奇橡皮擦”功能允许你通过刷过该区域来清理图像中不想要的细节。

图片来源: Canva

Canva 还推出了“Beat Sync”，可以自动将视频片段与您选择的配乐进行匹配，以及一个新的“翻译”功能，可以自动将设计中的文本翻译成 100 多种不同的语言。

除了新的人工智能工具，Canva 还推出了一系列用户要求的新功能。

首先，该公司正在推出“Draw”，让你画一个简单的形状，比如一个圆，然后让它变成一个抛光的圆。新的“布局”功能允许用户向页面添加内容，并获得设计布局想法的建议，而“样式”则允许用户浏览建议的调色板和字体。

一个新的“层”工具可以用来在一个地方看到所有的元素，包括文本，形状，图像和视频。

图片来源: Canva

Canva 现在支持 Alt 文本，这意味着您可以为您的图像和元素提供额外的上下文，以便为那些有视觉障碍或辅助功能需求的人描述它们。另一个受欢迎的更新是增加了 953 种新字体，包括 Arial、Courier、Helvetica 和 Times New Roman。

该公司还推出了 Brand Hub，这是一系列旨在帮助团队保持品牌的新工具和功能。该中心包括一个新的“神奇替换”工具，允许用户快速更新平台上所有设计的品牌资产，如徽标。

此外，该中心包括帮助您创建品牌模板、文件夹和指南的工具。管理员还可以直接在 Canva 中批准工作流，以确保设计在发布前经过审查和批准。

图片来源: Canva

“十年前，我们推出了 Canva，其愿景是让每个人都能设计任何东西，无论他们的技能或经验如何，”Canva 联合创始人兼首席执行官梅勒妮·帕金斯在一份声明中说。“今天，我们很高兴通过推出一系列全新的功能来推进这一愿景，专注于帮助品牌扩大其创意产出。随着技术的不断进步，我们正在重新构想设计流程，让人们更容易地将自己的想法写在纸上，并以前所未有的速度展现在世人面前。”

Canva 于 2013 年推出，现在每月用户超过 1.2 亿，比 2022 年 9 月推出可视工作套装之前增加了 3500 万。该公司表示，使用其平台已经创造了超过 150 亿个设计。
