Speechmatics 为语音转文本人工智能筹集了 6200 万美元

原文：https://web.archive.org/web/https://techcrunch.com/2022/06/28/speechmatics-raises-62m-for-its-inclusive-approach-to-speech-to-text-ai/

上周，我写了一篇关于一家人工智能初创公司的文章，该公司正在开发可以实时改变某人讲话的口音的技术。但是，如果人工智能的目标是让人们以任何方式说话都有可能被理解，并在这个过程中消除许多人工智能系统固有的偏见，那会怎么样？这也是一个很大的需求，现在一家名为 Speechmatics 的英国初创公司宣布投资 6200 万美元来扩展业务，该公司已经开发了人工智能，可以将语音翻译成文本，不管这个人的口音或说话方式如何。

美国的 Susquehanna Growth Equity 领投，英国投资者 AlbionVC 和 IQ Capital 也参与其中。这个系列 B 是演讲学的一大进步。该公司最初是在 2006 年由创始人托尼·罗宾逊博士从剑桥的人工智能研究中剥离出来的，在此之前只筹集了大约 1000 万美元(Albion 和 IQ 都是过去的支持者，还有中情局支持的 In-Q-Tel 和其他人)。

在此期间，它已经建立了大约 170 个客户群——它只销售 B2B，以推动面向消费者或面向企业的服务——虽然它没有披露完整的名单，但其中一些名字包括 what3words， 3Play Media，Veritone，Deloitte UK 和 Vonage，它们不仅在传统意义上使用这项技术进行转录；而是用于接收口语单词以帮助应用程序功能的其他方面，如自动字幕，或支持更广泛的辅助功能。

如今，它的引擎能够将语音翻译成 34 种语言的文本，除了利用资金继续提高准确性和业务发展外，它还将增加更多的语言，并研究不同的用例，例如将语音翻译成文本，以便在更复杂的机动车辆环境中使用(在这种环境中，马达噪音和振动会影响人工智能如何摄取声音)。

“我们所做的是收集数百万小时的数据，努力解决人工智能偏见。我们的目标是理解多种语言中的任何声音，”这家初创公司的首席执行官凯蒂·维格达尔(Katy Wigdahl)说。

这体现在该公司的产品重点和使命上，这也是它希望拓展的领域。

“我们看待语言的方式是全球化的，”维格达尔说。“谷歌将为每个版本的英语提供不同的包，但我们的包可以理解每一个版本。”它最初只是通过出售给客户的私有 API 来提供技术；现在，为了吸引更多用户和潜在的更多付费用户，它还向开发者提供更多开放的 API 工具来玩这项技术，并在其网站上提供拖放式采样器。

事实上，如果演讲家的挑战之一是训练人工智能在理解人们如何说话方面更像人类，那么另一个挑战就是在语音到文本技术的其他主要提供商面前为自己赢得名声。

Wigdahl 表示，该公司今天与“大技术”竞争，即像亚马逊、谷歌和微软(现在有 Nuance)这样的大公司已经建立了语音识别引擎，并将技术作为服务提供给第三方。

但它表示，在理解语言的多种表达方式的测试中，它的得分一直高于这些测试。(它向我引用的一项测试是斯坦福大学的“语音识别的种族差异”研究，，其中记录了“与谷歌(68.6%)和亚马逊(68.6%)相比，非裔美国人声音的总体准确率为 82.8%。”它说,“相当于语音识别错误减少了 45%——相当于一个普通句子中的三个单词。它还向 TC 提供了“竞争对手加权平均值”:

然而，当你考虑到在较小的开发者和像苹果、谷歌、微软和亚马逊这样的大规模、超大型技术巨头之间，有数百家大公司可能不完全处于为此目的建立内部人工智能的水平(或兴趣),但如果你以像 Spotify 这样的公司为例，肯定对它感兴趣，并且肯定不愿意依赖那些大公司，这些公司有时也是他们的竞争对手，有时是他们的彻底挫败(需要澄清的是，Wigdahl 并没有告诉我 Spotify 是一个客户，但他说这是一个典型的例子，说明在这种规模和情况下，有人可能会敲演讲家的门。)

这也是投资者如此热衷于资助这家公司的部分原因。萨斯奎汉纳有支持那些看起来可能会让权力玩家为他们的钱而战的公司的历史(它是 Tik Tok 早期的大支持者)。

Susquehanna Growth Equity 的董事总经理乔纳森·克拉尔(Jonathan Klahr)在一份声明中表示:“演讲团队无疑是不同血统的技术专家。“当我们的投资组合公司告诉我们，演讲术一次又一次地在准确性上胜过所有其他选择，包括来自‘大型科技’公司的选择时，我们开始跟踪演讲术。我们已经准备好与该团队合作，以确保更多的公司能够接触并采用这一卓越的技术。”克拉尔将加入董事会。

事实上，随着技术变得越来越自然，那些制造技术的人寻找更多的方法来减少任何和所有可能围绕该技术使用的摩擦，语音已经成为一个主要的机会点，也是一个棘手的问题。因此，拥有能够“阅读”和理解各种声音的技术可能会在各种方面得到应用。

“我们认为语音将成为越来越占主导地位的人机界面，语音学家是将深度学习应用于语音的类别领导者，类别定义了跨行业用例及需求的准确性和理解，”AlbionVC 的合伙人 Robert Whitby-Smith 补充道。“自 2019 年 A 轮投资以来，我们见证了团队和产品在过去几年中令人印象深刻的增长，作为负责任的投资者，我们很高兴支持公司的包容性使命，了解全球每一个声音。”
