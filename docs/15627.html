“信徒”,一种新的游戏方式，从光速、a16z 等网站筹集了 5500 万美元

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/07/believer-a-new-approach-to-gaming-raises-55m-from-lightspeed-a16z-and-more/

如今，游戏正处于十字路口:一家拥有数十年历史、围绕经典概念拥有非凡持久力的企业，如今正与混合现实、人工智能、区块链和网络等领域的创新展开激烈竞争，努力与日益分散的消费者群体建立联系。

今天，一家初创公司宣布从一些大牌投资者那里获得 5500 万美元的资金，以标志其在该领域的做法。创始人告诉我，由前 Riot 高管周英华和史蒂文·斯诺创立的“信徒 T1”仍处于秘密状态，在三到五年内不太可能推出任何东西。但是它有很大的野心去挑战大玩家，给游戏组合带来新的东西，特别是在大型多人“开放世界”游戏中。

“我们认为，开放世界类型已经被单一玩家、盒装产品所限制，”信徒的首席产品官斯诺在接受两位联合创始人的电子邮件采访时说。“我们热爱这些游戏；在古埃及或不久的将来的反乌托邦冒险是令人敬畏的！但现在是我们开始和朋友们一起冒险的时候了。”

它将在一些令人印象深刻的支持下做到这一点。光速创投(Lightspeed Venture Partners)领投首轮融资，安德森·霍洛维茨(Andreessen Horowitz)也参与其中，参与的还有 Bitkraft Ventures、Riot Games、1Up Ventures、唐·汤普森的 Cleveland Avenue 和迈克尔·D·艾斯纳的 Tornante Company。两位联合创始人告诉我，目前它不会披露估值。

《内心的信徒》将自己描述为一个工作室，这笔钱最初将用于招募更多的人才来填补这一空缺，补充最初聘用的人员，包括首席技术官兰登·麦克道尔(微软，Riot Games，林登实验室)，首席文化官杰里米·范霍泽(Bungie，电子艺界)，首席运营官·蒂姆·许(Twitter，Riot Games)，CMO·尚卡尔·古普塔-哈里森(Riot Games，电通 X)，运营总监格雷丝·朴(英雄联盟:荒野大裂谷)，以及设计副总裁杰夫·朱(英雄联盟，Runeterra 传奇)。

该公司表示，最初的重点将是原创 IP 和故事，“玩家的选择很重要”。

但除此之外，还会有一场技术竞赛，以及对如今任何新兴媒体帝国必不可少的一部分的追求:更大的娱乐焦点。

“我们肯定会建立新的技术，但要明智:只有在高度分化的情况下，没有技术支持我们的球员的梦想，以及信徒实现这些梦想的需要，”周说。然而，科技，至少在某些方面，根本不是纯粹的焦点。当我问信徒目前最感兴趣的平台是什么时——鉴于移动休闲游戏、个人电脑和联网电视上的多人游戏品牌以及虚拟现实世界的萌芽，答案可能是任何东西，真的——我得到了一个非常不具体的回答。

“最引人注目的平台是所有平台，”斯诺告诉我。“我们真的感觉到我们正处于一个最令人兴奋的时代，对平台对等性的渴望也满足了在所述平台上提供引人注目的游戏模式的能力。因此，我们希望建立在朋友群体之间广泛共享的游戏体验，而真正实现这一点的方法是尽可能减少进入壁垒，如平台排他性。”

与此同时，周星驰将《信徒》将做出的选择比作皮克斯在动画界做出的选择:它没有发明技术，但“必须创造全新的电影制作方式来与之配合。”

皮克斯的平行是有意的。“游戏公司正越来越多地转向娱乐生态系统的核心，”周补充道。“过去，我们看到媒体公司收购游戏公司，以将其知识产权延伸到互动媒体领域。现在，我们越来越多地看到这种动态逆转，伟大的 IP 诞生于视频游戏，然后被电视、电影、音乐和体育所包围。我们热爱这种模式，我们在 Riot 生活和呼吸着这种模式，在 Believer，我们努力做得更好。”

对于像迪斯尼前总裁艾斯纳这样的投资者来说，这是关键。艾斯纳在一份声明中说:“真正伟大的内容专营权不是建立在单一媒体类型上的——它们需要在游戏、电影、电视、收藏品和现实世界的互动体验中与观众产生共鸣。”“信徒公司明白这一点，我们 Tornante 很高兴为他们提供建议，帮助他们在各个平台上实现自己的愿景。”

由于公司迄今为止已经建立或计划建立的东西很少，而且在显示这些东西是否能吸引玩家之前，最终将是最大的考验，现在可能有趣的是听听联合创始人关于他们将会做什么而不是做什么。

首先，你可以忘掉 NFTs。

“我们他妈的不感谢 NFTs，”斯诺说。“这些技术在游戏中举步维艰，因为玩家没有要求它们，而且迄今为止还没有人展示它们如何让游戏变得更有趣。我相信游戏应该是有趣的。我们不是在这里迷上技术，这不会丰富任何人的行业，更不用说玩家了。”

这并不是说元宇宙和 web 3——不管你认为这意味着什么——被完全抛弃了。

“我认为最终会有人用这些概念提出解决方案，但这不是我们现在的目标。对我们来说，关键是这些技术如何让游戏更好，而现在我们还没有看到这些技术特别丰富了我们的对话，”斯诺说。“在他们为球员提供的乐趣变得清晰明了之前，这里需要更多的 R&D，根据我们到目前为止所看到的情况，他们可能不会去那里。”

另一方面，该公司希望用创成式人工智能做得更多。

斯诺说:“我们不仅要在建设和利用这项技术方面成为领导者，而且要在如何应用这项技术的道德规范方面成为领导者。”。他接着强调，这不会牺牲对人类创造者的关注，他们认为人类创造者对游戏“至关重要”。它更多的是关于人工智能能够实现的新功能，“最强大的工具，可以释放高质量的乐趣。人工智能和人工智能无疑在这方面有发言权，尤其是随着时间的推移，但如何利用它取决于我们的责任。”

鉴于任何初创公司都很难筹集资金，更不用说针对变幻莫测的消费游戏市场推出一款完全未经证实的产品，非常值得注意的是，信徒已经筹集了这么一大笔资金。斯诺知道这个圆很大，但它被抬高部分是因为他们能做到，也因为他们不知道拐角处有什么。

“通常情况下，这种规模的筹资是没有道理的，但我们也不确定今年晚些时候或明年的筹资会是什么样子，”他说。“迈克尔和我愿意放弃更多我们自己的所有权，以确保我们在未来三年或更长时间内能够继续专注于玩家和产品。这是获得适当支持以巩固我们未来的最佳方式。”

周小川补充说，投资者的“广泛背景”使得“思想、知识和观点的丰富多样性”。

“一旦我们找到了合适的投资者团队，从 Lightspeed 和 a16z 开始，就很容易找到更多当前心态正确的合作伙伴，”周补充道。“我们相信，我们拥有游戏公司有史以来最好的投资者和顾问。这种气候中有一种东西让人们从长远的角度来思考问题——那就是希望利用现在这个充满不确定性和可能性的时刻，大张旗鼓地制造破坏性影响。”

斯诺说，第一款产品的上市时间是 3 到 5 年，这是一个“合理”的时间表，所以希望那些支持它的投资者也能……耐心等待。

斯诺补充说:“我们将分阶段进行，并要求球员们尽早与我们合作。”。“我们当然有内部路线图，因为它们对问责制至关重要，但日期是对玩家的承诺，我们只是还没有准备好。”

就目前而言，关键员工招聘的早期信号是迄今为止投资者喜欢的。

“已经在信徒公司集合的团队随时准备通过对玩家的不屈不挠的奉献来改变这个行业，”加入信徒公司董事会的 Lightspeed Venture Partners 合伙人莫里茨·拜尔-伦茨说。“忘掉那些先入之见吧，那些只需要 70 美元入场费就能建立、销售和营销特许经营权的概念，还有那些严格创作的、固定不变的故事。作为一名终身球员和多年来一直关注和钦佩工作室创始人工作的“信徒”，我知道迈克尔、斯诺和他们不可思议的团队会在每一个转弯处做正确的事情，做出下一件大事。他们的远见卓识鼓舞了我，我很荣幸能够帮助他们实现这一目标。"

a16z 的普通合伙人陈楚翔在一份声明中补充道:“当迈克尔、斯诺和他们的创始人同事谈论电子游戏的未来时，游戏行业会停下来倾听。”。“一次又一次，通过新技术的创新使用和颠覆性的设计原则，他们挑战了常规，实现了其他开发者做梦也没想到的雄心壮志。我们很自豪能在这个历史时刻加入他们的行列，让他们能够在新兴技术中找到创造力。”
