谷歌将发布 Pixel 平板电脑...明年

原文：https://web.archive.org/web/https://techcrunch.com/2022/05/11/google-will-release-a-pixel-tablet-in-2023/

谷歌将于 2023 年发布 Pixel 平板电脑

提供 Pixel 7 的预览是不够的，所以谷歌真的在倾斜。今天在 I/O 大会上，该公司宣布将重返平板电脑业务，推出一款新设备，预计于 2023 年发布。

“通常我们不会在新产品准备好之前取笑它，”谷歌硬件主管里克·奥斯特洛(Rick Osterloh)说，“但在开发者社区中，平板电脑周围有如此惊人的能量，我们想让你们都参与进来。”

Pixel 平板电脑将成为 2018 年 Pixel Slate 的精神继承者，该公司去年悄悄停止了 Pixel Slate。像 Pixel Book 一样，硬件很好，但在一个充斥着超级便宜的 Chromebooks 的世界里，谷歌从未真正找到“为什么”

Pixel 平板电脑确实在一个关键方面与这些产品不同:它实际上是一款 Android 设备。在 ChromeOS 占主导地位的世界里，该公司试图重振 Android 平板电脑市场。

想必这里的部分想法是为谷歌的定制张量处理器找到另一个家，该处理器在去年的 Pixel 6 中首次亮相。至少，使用相同的芯片意味着手机和平板电脑应该可以很好地配合，因为该公司慢慢建立了一个硬件生态系统，其中也包括 Pixel Buds 和即将推出的 Pixel Watch。

除此之外，对于一款不会在一年后推出的设备来说，细节不足不足为奇。正如上面的照片所示，我们有一个后置摄像头，一些侧面扬声器狭缝，我猜还有一些按钮？我想，在 2023 年谷歌 I/O 大会上，我们会在这里见面。
