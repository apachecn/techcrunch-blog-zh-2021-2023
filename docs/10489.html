Cloudflare 推出 eSIM 来保护移动设备 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/09/26/cloudflare-launches-an-esim-to-secure-mobile-devices/

智能手机真的完全安全吗？这取决于一个人对“安全”的定义，尤其是在处理企业环境时。大多数实行自带设备政策的公司会在员工的智能手机上安装应用程序或代理程序来帮助保护它们，利用 Android 和 iOS 等操作系统内置的管理功能。但是这些可能还不够。

无论如何，这就是 Cloudflare 在本周推出的新服务中所主张的。今天，该公司宣布了面向移动运营商的零信任 SIM 和零信任，这两款产品面向智能手机用户、保护公司电话的公司和销售数据服务的运营商。

先说零信任 SIM。Zero Trust SIM 旨在保护离开智能手机的所有数据包，一旦在美国推出(开始)，将作为 eSIM 通过现有的移动设备管理平台部署到 iOS 和 Android 设备上。它将被锁定到一个特定的设备，降低了 SIM 交换攻击的风险，并且可以在独立配置中使用，也可以与 Cloudflare 的移动代理 WARP 配合使用。

在最近的一次电子邮件采访中，cloud flare CTO John Graham-Cumming 提出了零信任 SIM 可以完成 VPN 和其他安全层无法完成的事情:细胞级保护。他认为，SIM 卡可以作为另一个安全因素，结合硬件密钥，几乎不可能冒充员工。

“零信任 SIM 提供深度防御。Graham-Cumming 说:“VPN 层是其中的一个组成部分，但并没有消除今天仍然需要在所有移动设备上部署蜂窝连接的需求，传统的 AnyConnect 式’VPN 无法阻止攻击者在 VPN 内部横向移动。”“我们不断看到组织因为保护其应用程序和网络的挑战而遭到破坏，从 IT 安全的角度来看，曾经的房地产预算正迅速变成‘保护我的远程和分布式工作人员’的预算。”

具体来说，Graham-Cumming 表示，Zero Trust SIM 将使 Cloudflare 能够重写 DNS 请求，使设备转而使用 Cloudflare Gateway 进行 DNS 过滤。他补充说，它还将支持在每个主机和 IP 地址到达互联网之前对其进行验证，以及基于身份的服务和其他设备连接，它可以用作身份认证的第二个因素。

虽然定价尚未确定，但将在未来几个月推出的 Zero Trust SIM 将从计费角度被视为 cloud flare Zero Trust 平台的一部分，Graham-Cumming 表示，它将是目前 Zero Trust 客户按座位定价的延伸。他预计，大多数设备都将兼容，一旦 Cloudflare 开始为该服务提供物理 SIM 卡(计划在不久的将来实现)，甚至更多。

Graham-Cumming 说:“我们的目的是从美国开始，但很快就会使这成为一项全球服务——运营全球网络是我们工作的核心部分。”“虽然我们在这方面还处于早期发展阶段，但我们已经在工业物联网(IoT)领域(例如，车辆、支付终端、集装箱、自动售货机)开展了并行计划。零信任 SIM 卡本身就是一项基础技术，可以开启许多新的使用案例。”

关于物联网，Cloudflare 今天预览了一个物联网设备平台，被恰当地称为物联网平台，其目标是为一系列联网设备提供单一控制台视图。为了与微软 Azure、亚马逊 Web 服务和谷歌云的物联网管理服务竞争，Cloudflare 的产品处理物联网的订购、供应和管理蜂窝连接和安全。

根据 Cloudflare 的说法，离开每个物联网设备的每个数据包都可以在到达互联网、云或其他设备之前，由客户创建的策略进行检查、批准或拒绝。此外，设备可以锁定到特定的地理位置，以确保敏感流量不会到达公共频道。

Cloudflare 表示，随着物联网平台正式发布的临近，未来几个月将有更多信息可用。

在移动运营商零信任方面，Cloudflare 没有什么可分享的。Graham-Cumming 表示，作为一项运营商合作伙伴计划，零信任移动运营商将允许服务提供商从 Cloudflare 的零信任平台订阅移动安全工具。感兴趣的运营商可以从今天开始注册获取更多信息。

有人认为，对移动运营商的零信任——就此而言，新的零信任 SIM——是 Cloudflare beyond WARP 的一项利润丰厚的业务，该公司三年前推出了免费增值模式。根据 Allied Market Research 的数据，2020 年全球移动安全市场价值 33 亿美元，到 2030 年可能达到 221 亿美元。

鉴于物联网市场的稳健性，物联网平台对 Cloudflare 也有意义。据一位消息人士称，随着供应链挑战等顺风加强，企业物联网支出在 2021 年增长了 22.4%，达到 1580 亿美元。该领域充斥着现任者，但 Cloudflare 显然打赌它可以发挥足够的影响力，产生相当大的影响。
