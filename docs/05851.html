OwnID 用基于智能手机的生物识别技术取代了密码

原文：https://web.archive.org/web/https://techcrunch.com/2022/04/26/ownid-raises-6m-to-replace-passwords-with-smartphone-based-biometrics/?

OwnID 筹集 600 万美元，用基于智能手机的生物识别技术取代密码

OwnID 的创始人 Dor Shany 和 Rooly Eliezerov 之前创立了 Gigya，这是一家希望更容易帮助识别客户的初创公司。他们在 2017 年以 3.5 亿美元的价格将这家初创公司卖给了 SAP ，但让登录网站变得更容易而没有摩擦的想法一直在他们的脑海中，他们去年推出了 OwnID 来帮助解决这个特殊的问题。

Shany 解释说，创始团队从他们在 Gigya 的时候就了解身份，他们希望创建一家公司，从用户的角度尽可能简单地进行身份认证。任何忘记密码的人都知道这个过程有多让人抓狂。

他们的想法是将这个过程移到你的手机上，利用手机内置的认证系统。“手机本身实际上是一种双重认证，因为它是一种生物认证。本质是你去一个网站，你输入你的电子邮件地址，你点击‘跳过密码’，用你的生物识别(面部或手指)ID 登录你的手机。”就是这样。

如果你曾经试图在电视上登录网飞等流媒体服务，你就会知道这可能是一种令人沮丧的体验，在这种情况下，OwnID 会在屏幕上向你显示一个二维码，你用手机扫描代码，然后登录手机。

OwnID 处理的另一个麻烦是，当您在设备之间移动时，您需要记住您的密码(除非您有某种密码管理器)。有了 OwnID，你就可以一直用手机进行身份验证，所以这个问题应该会消失。

Shany 说，这很难被欺骗，因为当你第一次访问该网站时，你通过手机注册，密钥从那时起就存储在你的手机上。每当网站或应用程序看到你的手机有钥匙，它就知道是你。

从网站或应用程序所有者的角度来看，他们通过 API 连接到 OwnID，一旦开发人员添加了代码，用户就可以选择使用手机登录而无需密码。

该公司今天首次公开 OwnID，但它已经与雀巢和德隆基等大客户合作。该公司目前约有 20 名员工，计划在未来一年将人数增加一倍。他表示，疫情实际上让寻找多元化员工变得更加容易，因为他们可以从任何地方雇佣员工，这给了他们更多的灵活性。

OwnID 今天宣布了一轮 620 万美元的种子融资，由 Mayfield 牵头。根据 Crunchbase 的数据，在被 SAP 收购之前，这位创始人的前一家公司 Gigya 筹集了超过 1 亿美元。值得注意的是，梅菲尔德也是该公司的早期投资者。
