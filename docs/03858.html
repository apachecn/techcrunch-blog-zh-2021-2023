Clickatell 筹集 9100 万美元扩大其聊天商务平台和美国扩张 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/02/22/clickatell-raises-91m-to-scale-its-chat-commerce-platform-and-us-expansion/

全球聊天商务的潜在市场总价值约为 800 亿美元，被视为继电子商务和应用商务市场 之后的第三波数字商务浪潮。

作为世界上最大的数字平台，拥有比互联网 47 亿用户更多的用户，企业通过聊天渠道与消费者沟通和交易才有意义 。

成立于 2000 年的 Clickatell 是移动通信和聊天商务领域的先驱。该公司帮助企业通过移动通讯平台与客户沟通，今天宣布它在新一轮融资中筹集了 9100 万美元。

C 轮融资，这是红杉支持的公司上一轮融资十年后的产物， 由 葛根资领投，Kennedy Lewis Investment Management、EndeavorGlobal 和 Harvest 参投 。

Clickatell成立于南非，总部位于美国加州，在尼日利亚和加拿大设有办事处。公司创始人兼首席执行官 Pieter de Villiers 告诉 TechCrunch，他创办了公司，为互联网、增长最快的商务平台和手机沟通，快速增长的通讯平台 。

因此，Clickatell 成为允许公司向客户发送消息的首选 API 接口，从而通过 SMS 在客户参与方面获得可观的收益。

2011 年，Clickatell 从红杉资本(Sequoia Capital)和其他投资者那里获得了 1200 万美元的 B 轮注资，以超越客户参与，并向其众多产品之一 增加支付。

在一个聊天商务空间中，Twilio、Zendesk 和 Stripe power 分别在通信平台即服务(CPaaS)、联络中心即服务(CCaaS)和电子商务支付中开展活动，Clickatell 在这三者的交汇处发挥作用，并帮助企业在聊天渠道中部署支付 。

在其平台上，消费者可以通过短信和 USSD(在尼日利亚)和 WhatsApp(在南非)与品牌联系，找到商品和服务，进行购买，跟踪订单，并通过短信或聊天解决问题 。

该公司声称已经通过与南非最大的银行之一 ABSA 银行的聊天商务整合，在 WhatsApp 上开发出了世界上第一个聊天银行解决方案。

“在许多方面，我们不得不等待市场来找我们，因为如果你是第一个行动者，你也可能会远远领先于其他人，发现自己在沙漠中，”德·维利耶在 TechCrunch 上谈到该公司从通过聊天实现客户参与到支付 的进展时说。

“我们不得不等待 WhatsApp 频道的开放。 两年前，我们是全球首家推出基于 WhatsApp 和聊天银行的支付服务的公司 。”

去年 9 月，Clickatell 与 Visa 的欺诈管理平台 Cybersource 合作，推出了 Chat2Pay，为美国的企业提供聊天商务和无接触支付。 此次合作拓宽了这家支付巨头对实体销售点终端和信用卡之外的不同支付方式的接受范围 。

使用该平台的商家可以通过手机短信或 WhatsApp 向顾客发送支付链接，然后顾客会被引导到一个支付页面，在页面上输入他们的信用卡信息并进行支付。

我们相信聊天商务有潜力比电子商务更大，原因很简单，今天聊天的人已经比互联网上的人多，事实上几乎是两倍。 所以当 Visa 考虑到未来，以及他们如何服务他们的客户跨越多种支付工具，他们决定与我们合作，”他说 。

这种合作关系的市场机会是巨大的。2019 年价值 120 亿美元的全球非接触式支付市场 预计未来五年 将达到 520 亿美元 。 此次合作将看到 190 个国家的 Visa 商户和合作伙伴接入新推出的平台 。

Clickatell 声称，十年多来，它已经实现了两位数的利润增长。该公司表示，自 2020 年以来，它已经处理了超过 300 亿次互动和 20 亿次商业交易。 仅在尼日利亚，Clickatell 就拥有 1700 万最终客户，每天进行的支付交易高达 130 万笔 。

2019 年，这家总部位于加州的公司寻求资金在美国上市，当然，考虑到其新一轮私人资本 ，上市尚未实现。在谈到 Clickatell 为何仍保持私有时，首席执行官 de Villiers 表示，这家 22 岁的公司需要安全的资金来测试“这个刚刚兴起的行业中的产品”，然而，他很有信心 Clickatell 的下一轮融资将是公开的。

目前，Clickatell 打算利用 C 轮资本来扩大其在美国的业务，扩大其销售和营销努力，并 加速其在南非和尼日利亚的聊天商务服务的发展。

Clickatell 是一个鼓舞人心的例子，展示了一家南非公司如何推动未来发展。 Clickatell 是成功地将信息从互联网发送到手机的先驱公司，今天他们为通信领域的一些最大的名字供电，如 WhatsApp。
