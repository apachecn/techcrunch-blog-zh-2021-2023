以 API 为先的无密码创业公司 Stytch 以$1B 估值融资 9000 万美元

原文：https://web.archive.org/web/https://techcrunch.com/2021/11/18/stytch-api-passwordless-unicorn/

以 API 为先的无密码创业公司 Stytch 以$1B 估值融资 9000 万美元

API 第一的无密码认证初创公司 Stytch 获得了 9000 万美元的 B 轮融资，使该公司的估值超过了 10 亿美元。

这笔投资由 Coatue Management LLC 牵头，现有投资者 Benchmark Capital、Thrive Capital 和 Index Ventures参与其中。就在四个月前，Stytch 以约 2 亿美元的估值筹集了 3000 万美元的 A 轮。自那以后，这家初创公司使用其无密码认证平台的开发人员增加了近 1000%，从 7 月份的 350 名开发人员增加到 11 月份的约 4000 名。

该公司的首席执行官告诉我们，这是因为其 API 优先的方法。“如果你想想其他无密码创业公司，他们非常专注于小工具，”Stytch 首席执行官、前格子图案员工里德·麦金莱-斯坦普尔告诉 TechCrunch。“我们对非 API 优先的产品有足够的经验，我们知道你能做的有很多限制。”

“例如，我们从未预料到的一个常见用例是结账流程——增加了在结账时使用短信密码或电子邮件验证创建新账户的能力，而不是客人结账，”麦金莱-斯坦普尔说。

自 7 月份 A 轮融资以来，该公司还推出了许多新产品，包括支持登录苹果、谷歌和微软、可嵌入的魔法链接和通过电子邮件发送的一次性密码。本周它还增加了对 WebAuthn 的支持，允许 Stytch 支持基于硬件的认证密钥，如 Yubico ，以及基于生物特征的 Face ID 和指纹登录。

Stytch 表示，它计划在未来几个月进一步扩展其平台。作为 B 轮融资的一部分，它收购了 Y Combinator 支持的无代码无密码认证平台 Cotter ，允许用户一键登录网站和应用。Stytch 表示，这将使开发者更容易采用无密码技术。

该公司还将利用这笔资金扩大其 30 人的团队，并建设其基础设施。

Stytch 并不是唯一一家致力于消除密码的初创公司。6 月， Transmit Security 筹集了 5.43 亿美元，这被认为是网络安全历史上最大的 A 轮投资，用于其固有的无密码身份和风险管理解决方案。接下来的一个月，总部位于三藩市的初创公司 Magic，开发“即插即用”的无密码认证技术，宣布它已经在首轮融资中筹集了 2700 万美元。
