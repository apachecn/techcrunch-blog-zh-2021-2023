Astrix Security 从秘密行动中脱颖而出，帮助组织发现流氓第三方应用程序

以色列网络安全初创公司 Astrix Security ，为第三方应用集成提供访问管理，已经秘密获得 1500 万美元资金。

这家初创公司由首席执行官阿龙·杰克逊(Alon Jackson)和首席技术官伊丹·古尔(CTO Idan Gour)在 2021 年共同创立，他们都是以色列著名的情报部门 Unit 8200 的前成员，旨在帮助组织监控和控制连接到其关键系统的第三方应用程序的复杂网络。

由于广泛转向远程工作以及基于云的环境，组织使用的集成数量在过去两年中急剧增加。Astrix 声称，虽然企业在管理用户对关键系统的访问方面很大程度上处于领先地位，但大多数企业在管理 API 访问方面都做得不够，这使他们面临越来越大的攻击面，容易受到供应链攻击、数据泄漏和违规行为的影响。这就是为什么这家初创公司开发了 Astrix Security，这是一个为企业提供全面集成生命周期管理的平台。

“当前的解决方案提供了一个安全分数，可帮助您评估想要采用的应用程序的安全状况。其他人，如 NoName ，关注 API 安全性，专注于你开发并希望其他人使用的 API，”杰克逊在创立 Astrix 之前担任 Argus 的研发主管，他告诉 TechCrunch。“我们关注通过第三方完成的集成；这可能是你在 Salesforce 上的 CRM 或者你在 GitHub 上的知识产权。这些都不是您开发的系统，但是您可以对它们进行 API 访问。”

Astrix Security 为组织提供了所有第三方企业应用程序连接的即时清单。它会自动检测这些集成和低代码或无代码工作流配置中的更改和恶意异常，并提供实时补救。

杰克逊声称，这项技术可以防止组织成为去年 CodeCov 黑客攻击的受害者，攻击者突破了该公司的软件审计工具，获得了数百个客户的网络。

“发生的事情正是我们正在建设的；开发人员只是在他的 GitHub 代码库中添加了一个新的第三方连接。他删除了它，但没有撤销访问权限，这导致他们的整个知识产权在黑暗网络上出售，”杰克逊说。

Astrix Security 已经掌握在众多全球企业客户手中，涵盖科技、医疗科技和汽车行业。杰克逊表示，这家初创公司计划利用其 1500 万美元的种子投资，扩大其目前的 20 人团队，并加强其走向市场的努力。这笔投资由 Bessemer Venture Partners 和 F2 Capital 牵头，Venrock 和 20 多名网络安全天使投资人参与。
