Reddit 推出搜索改进

原文：https://web.archive.org/web/https://techcrunch.com/2022/04/14/reddit-search-improvements-search-comments/

Reddit 推出了新的搜索改进，包括查找评论的方式

Reddit 今天宣布将推出搜索评论的功能，以及其他一些与搜索相关的功能。有了新的评论搜索功能，用户在查找特定对话时，不再需要点击多个评论来查找线索。现在，您可以通过搜索栏中新的“评论”选项卡直接搜索评论。

用户还可以通过在特定社区中搜索评论来进一步细化他们的结果。例如，如果您想找到一个关于伦敦最佳位置的帖子，您以前必须解析 r/London 社区中的每个帖子并查看评论才能找到它。现在，你可以看到人们在评论中分享的所有不同的推荐。

Reddit 表示，去年它对用户进行了调查，询问他们希望看到什么样的搜索功能，发现最受欢迎的功能之一是评论搜索。当该公司最初测试该功能时，发现超过 26，000 人使用评论搜索浏览了超过 50 亿条评论。

Reddit 在一篇博客文章中说:“通过最新的更新，16 年来第一次，Reddit 上的所有内容都可以搜索——用户、帖子、社区，现在还有评论——这使 Reddit 成为首批具有这种功能的平台之一。”

图片来源: Reddit

此外，Reddit 正在推出一种更简单的搜索结果设计，这种设计基于桌面和移动设备上的用户反馈。该平台现在在其更新的搜索设计中，帖子优先于其他类型的内容。它还简化了结果页面，让用户更容易浏览结果并找到他们想要的东西。Reddit 还致力于根据搜索者的意图减少意外结果的数量，从而使搜索更加安全。

该公司还在更新其平台，通过减少限制性匹配来提高搜索的相关性。例如，100%的查询不一定要匹配文章的文本才能返回相关结果。Reddit 表示，通过这一改变，以前收不到结果的查询结果增加了 60%。它还表示，它正在使用机器学习来研究用户模式，以改善搜索结果。

“例如，如果有人在搜索一个很多人都在搜索的主题，我们会自动对他们的结果进行排序，以优先考虑最新的内容，以确保结果是新鲜的，”该公司在博客中说。

新功能将于今天推出。新的评论搜索功能，以及上面列出的其他新功能，可以通过主页提要的搜索栏访问。

今天的声明是在 Reddit 证实它正在探索将更多用户生成的视频内容引入其在线论坛的想法几周后发布的。关于这一潜在的新视频功能，还没有太多确定的消息——它还没有进入测试。但我们知道，这可能涉及使用类似抖音的视频编辑工具，其中最值得一提的是，人们可以通过将自己的视频添加到他人的视频中，对他人发布的视频做出“反应”。
