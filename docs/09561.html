Drop 的 DCX 键帽让你的旧键盘重获新生

原文：https://web.archive.org/web/https://techcrunch.com/2022/08/27/drops-dcx-keycaps-give-your-old-keyboard-a-new-lease-on-life/

大多数人不会花很多时间去考虑键盘上的键帽。很有可能，它们是白底黑字或白底黑字，如此而已。然而，就像所有的东西(机械)键盘一样，似乎有无穷无尽的选择。很长一段时间，最受欢迎的键帽是德国 GMK 公司生产的。但随着近年来机械键盘爱好的激增，GMK 根本跟不上需求，新的套装——在机械键盘领域往往采取团购的形式——往往需要一年或更长时间才能发货。

虽然许多爱好者喜欢 GMK 在其 ABS 键帽上印刷鲜艳颜色的能力，但该公司最近的制造问题和延迟使其在社区中失去了很多商誉。不出所料，GMK 的发行为其他高质量键帽制造商打开了大门，包括 EnjoyPBT 和 Drop 等公司。虽然 Drop 在其商店中也提供一些 GMK 制造的电视机(没有团购的麻烦)，但该公司最近也推出了其 DCX 键帽，目前有九种版本。

图片来源: TechCrunch

我承认我从来都不是 Drop 默认 MT3 轮廓键帽的忠实粉丝。有些人喜欢它们，但它们对我来说有点太高太圆了。不过，Drop 表示，花了两年时间开发的新 DCX 个人资料，更类似于你可能已经习惯的标准樱桃布局。事实上，除非你仔细看，否则你可能会把它们误认为樱桃形状的键帽。一整套厚厚的双镜头 ABS 键帽售价 99 美元，也有点便宜(至少按照机械键盘的标准)——尽管如果你是 Mac 用户，你可能会想多花 25 美元买 MacOS 专用键。

这些键帽很容易喜欢。Drop 给我发了一个黑白系列的评测样本。图例清晰——这是好的 ABS 键帽的标志之一——而且我没有发现任何制造缺陷。与更便宜的 ABS 套不同，Drop 的键帽感觉不太光滑，虽然在使用几年后它们可能会显示出一些光泽(所有 ABS 键帽都是如此)，但显然还没有这种迹象。它们没有我在个人键盘上常用的一些 PBT 键帽厚，但与 Drop 目前销售的 GMK 键帽相当。因此，在同一块电路板上，它们发出的声音只比更厚的 PBT 装置发出的声音高一点点，但这几乎不明显，就声音而言，开关、板和电路板上的其他任何东西都会产生更大的影响。

这套设备兼容标准 ISO 和 ANSI 布局，不过如果你使用爱丽丝风格的键盘，例如，你还需要15 美元的空格键套件。这在整个行业中是相当标准的，并使基本套件的价格保持较低，因为只有少数买家会需要这些。

Drop 还发给我一个它的预建的 CTRL 键盘，让我试试新的键帽。Paragon 系列全栈版本配有樱桃红静音开关和Drop+Oblotzky SA obliation 定制键帽套装(带白色字母键)。我不介意那套的审美，但是和新的 DCX 套相比，感觉有点廉价。我承认高键帽的 SA 配置文件真的不适合我，但也许同样重要的是，我不喜欢它们摸起来有多光滑。

尽管已经上市有一段时间了，但现在，具有铝制框架和有趣灯光效果的下拉 CTRL 仍然是一个可行的选择。准系统版本的价格为 150 美元，位于定制键盘市场的低端，同时提供高质量的构建(值得注意的是，我测试的预建 Paragon Full Stack 具有一些设计调整，如取消浮动键设计，价格为 500 美元)。因此，它正在一个竞争激烈的细分市场中发挥作用，一些较新的主板以类似的价格提供了更多的功能和选项(例如，包括垫圈安装)，而 Drop 自 2019 年以来一直没有发布新的内部设计。不过，该公司最近推出了一款新键盘，所以这种情况迟早会改变。
