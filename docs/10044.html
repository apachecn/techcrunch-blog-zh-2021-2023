Arpeggi Labs 为 web3 音乐软件融资 16z

原文：https://web.archive.org/web/https://techcrunch.com/2022/09/14/arpeggi-labs-banks-a16z-funding-to-build-web3-music-creation-software/

Arpeggi Labs 为构建 web3 音乐软件筹集了 16z 资金

寻找 web3 用例的确切定义并不总是容易的，但这让许多创始人在开发产品时发挥了创造力，这些产品旨在利用去中心化的精神和创造者经济的商业机会。

Arpeggi Labs 是一家新的加密初创公司，专注于利用区块链让音乐创作更具协作性，该公司从安德森·霍洛维茨的加密部门获得了 510 万美元的种子资金，还有许多艺术家，包括史蒂夫·青木、3LAU 和怀克里夫·金。

该团队的目标是利用区块链建立一个以开源精神为基础的音乐创作套件，允许制作人尝试各种各样的歌曲和声音，同时该平台确保信用总是适当地流回到原创者手中。

创始人埃文·迪隆(Evan Dhillon)、凯尔·迪隆(Kyle Dhillon)和詹姆斯·帕斯坦(James Pastan)认为，这种设置将鼓励一种新型的“混合文化”渗透到音乐行业，允许像抖音一样创造性地重新利用知识产权，这将提升所有为病毒式流行做出贡献的创作者。这是一项可能天生不需要区块链的功能，但 Arpeggi 的创始人表示，这项技术只是让它变得更容易。

“在音乐领域，我们已经看到人们立即抵制 web3 中的任何东西，因为他们认为这是一个骗局，”联合创始人帕斯坦告诉 TechCrunch。“我们已经尽可能多地抽象出了密码……而且我们总是远离投机因素。”

为了实现这个梦想，Arpeggi 的目标是超越开发协议，建立一个完整的浏览器内数字音频工作站，允许制作人混合区块链创作的节拍，同时与 Web 2.0 和 web3 音乐平台的更广泛领域相集成。

这个免费平台不仅是创作的中心，也是消费的中心，这一机会可能会激发区块链的积极性，但也可能给寻找有凝聚力的受众带来巨大挑战。

虽然该公司尚未决定他们是否打算在未来为该平台发行令牌，但创始人表示，他们主要关注的是吸引那些尚未与 web3 咒语结盟但对行业加倍开放源代码的想法感到兴奋的音乐专业人士。

“你听到的任何声音都是你应该创造的声音，”联合创始人凯尔·迪隆告诉 TechCrunch。
