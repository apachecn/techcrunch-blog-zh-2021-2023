安全合规和自动化平台 Drata 以 2B 估值获得 2 亿美元| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/12/07/security-compliance-and-automation-platform-drata-nabs-200m-at-2b-valuation/

Drata 是一个安全合规和自动化平台，帮助公司遵守 SOC 2 和 GDPR 等框架，已经在由 Iconiq Growth 和 GGV 资本共同牵头的 C 轮融资中筹集了 2 亿美元，对公司的估值为 20 亿美元。

就在一年前，这家总部位于圣地亚哥的公司在一轮 1 亿美元的 B 轮融资中成为独角兽，而在随后的几个月里，该领域的竞争对手，包括 Secureframe 和 Laika，都筹集了5000 多万美元，而 Vanta 今年通过两批获得了 1.5 亿美元，估值超过 16 亿美元。

因此，尽管在过去的一年里，裁员和低估值一直是风险投资领域的一个关键话题，但安全合规和自动化似乎在某种程度上与这一趋势背道而驰。

数据安全

原因可能有点显而易见，相对于数据几乎是所有现代业务的基石，为从人工智能应用和 CRM 系统到数字医疗服务的一切提供动力。与此同时，公司不得不升级他们的数据隐私游戏，特别是在涉及个人身份信息(PII)的情况下，以满足越来越多的旨在保护敏感用户数据的法规和框架。

Drata 联合创始人兼首席执行官亚当·马科维茨(Adam Markowitz)向 TechCrunch 解释说:“当你考虑到不断演变的数据威胁和不断增加的隐私法规时，安全性和合规性现在对一个组织的基础设施至关重要。“当你考虑到其他国家对独特的合规框架、标准或法规有不同的偏好时，这对快速增长的公司变得更加重要。合规性变得越正常，公司就不能再忽视或回避合规性，而那些采用合规性的公司需要一种可扩展的自动化方式。”

虽然 Drata 支持多种框架，包括今年 3 月的GDPR，但 SOC 2 仍然是它的面包和黄油。对于外行人来说，SOC 2 本质上是一个自愿的审计过程，公司通过它向潜在客户证明他们有一个强大的数据隐私制度。然而，实现 SOC 2 认证是一项资源密集型工作，不会在认证点停止，这是一个持续的过程，因为公司的数据利用实践通常会随着其技术堆栈的增长、向组合中添加更多 SaaS 集成、更换供应商等而发展。

简而言之，公司必须不断证明他们值得客户的信任。

这就是 Drata 及其同类进入竞争的地方，集成了数十种云平台、开发工具、SaaS 应用、网络安全系统等，帮助公司自动收集必要的“证据”，以验证他们的数据隐私和安全实践是井然有序的。

合规自动化有效地取代了涉及跨无数系统收集数据的多种手动工作，这一过程在完成时已经非常过时:通过接入保存数据的系统，从 AWS 和 GitHub 到 Okta 和 Gusto，一切都是实时维护的。

虽然许多公司不得不在经济逆风的情况下缩减业务，但事实是他们仍然需要寻求新的业务，证明数据隐私合规性现在是其中的一个重要部分，这不再是一个“美好的事情”

Markowitz 说:“一个强大的安全和合规计划是向大公司或企业销售的要求，所以公司必须在推出时就优先考虑这一点。”。

增长

Drata 成立于 2020 年年中，在去年年初获得 320 万美元融资之后悄然出现，这几年对它来说可谓是风风雨雨。最初，Drata 专注于帮助初创公司解决他们的 SOC 2 认证难题，但随着它扩展对其他框架的支持，该公司已经获得了更大的客户，包括价值数十亿美元的企业，如概念和柠檬水。

归根结底，问题并不局限于任何特定规模或类型的公司，合规性问题影响到每个人。

马科维茨说:“事实上，每家公司都在某种程度上在云端处理或存储客户数据。”。“法规遵从性是一种验证您的公司可以信赖这些数据的方式。无论你是一家两人创业的公司，还是一家向全球数百万公司销售产品的上市公司，合规都是必要的。”

在此之前，Drata 已经筹集了大约 1.28 亿美元，另外还有 2 亿美元的银行存款，该公司表示，它计划围绕其企业级产品在 R&D 进行大量投资，尽管它也打算投资于创业公司和审计机构的新功能。

Drata 的 C 轮融资包括机构投资者的支持，包括 Salesforce Ventures、Alkeon Capital、Cowboy Ventures、S Ventures (SentinelOne)和硅谷 CISO 投资公司(SVCI)。天使投资者包括前 LinkedIn 首席执行官杰夫·韦纳和雪花首席执行官弗兰克·斯洛特曼。作为一个有趣的旁注，德拉塔还透露，微软董事长兼首席执行官塞特亚·纳德拉去年投资了 1 亿美元的 B 轮融资，这是它以前没有披露过的。

Markowitz 继续说道:“我们一直将筹资视为一种策略，而不是目标或结果——筹资是我们战略增长的一部分，实际上也是对我们迄今为止执行情况的确认。”。"融资使我们能够加速下一阶段的增长。对我们来说，这意味着进一步扩大我们的产品能力，真正为全球大众带来持续的合规性。”
