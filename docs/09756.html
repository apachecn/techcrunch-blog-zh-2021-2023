Mode Designs 的第 65 首十四行诗让人觉得太奢侈了，无法打字

原文：https://web.archive.org/web/https://techcrunch.com/2022/09/05/mode-designs-sonnet-65-feels-almost-to-luxurious-to-type-on/

我还没有在键盘上打出像《风尚设计》的《十四行诗》一样好的感觉。在过去的几周里，确实有几天，公司给了我一个原型单元来测试在哪里打字是我开始写作的主要动力之一。

不带开关和键帽的起价为 299 美元，这不是最贵的定制键盘(尽管随着你增加几个选项，价格可能会很快上涨一倍)，但 Mode 几乎拥有一切，由于它的选项范围，这家总部位于波士顿和旧金山的公司允许你组装一个适合你的机械键盘。如果你想让它感觉良好和灵活，这是许多机械键盘爱好者目前努力的目标，你可以做到这一点——但如果你喜欢更坚固的打字体验，你可以选择堆叠安装，这也提供了更柔和的声音。

图片来源:模式设计

对大多数人来说，键盘就是键盘——台式机自带或笔记本电脑内置的基本设备。只要它能工作，就没有人会去想它。但对于机械键盘爱好者来说，键盘就是一切，很少有比键盘更深的兔子洞，因为由于开关、稳定器、键帽、开关板和其他大多数人从未想到的东西过多，即使是单板的变化也是永无止境的(使用 Krytox 205G 或 Krytox 105 开关的陆兵如何改变其声音和感觉？Tribosys 3204 怎么样？).

十四行诗的基础非常简单:它是一个 75%的板，这意味着没有数字小键盘，而是一整排的功能键，箭头键和右侧的几个额外的按钮，如 Home，PageUp，PageDown 和 End(或者其他任何你想要的东西，感谢十四行诗的 Via/QMK 兼容性)。顶部由加工精美的 铝制成，颜色由您选择，底部有多种选择，从铝到聚碳酸酯、黄铜和铜。然后你有选择——很多选择。你可以选择你想要使用哪种内部重量，你更喜欢哪种板(这对手感和声音有重大影响)以及你想要哪种重音颜色(对手感和声音没有影响但确实看起来相当不错)。

图片来源:模式设计

在我收到的变体中，有一半的板，没有板泡沫，声音和感觉接近我的机械键盘应该是什么样子的柏拉图式理想。在这个变体中，有一套 GMK 黑底白字的键帽，每次按键听起来就像两个台球相撞。对我来说，这正是我想从键盘上听到的声音。我也喜欢有一点点弹性，但不会走极端——这正是我得到的。

这是一个很重的板，在机械键盘世界里，这通常与质量有关，也经常让这个爱好的新手感到惊讶。毕竟，你的基本罗技主板不太可能有几磅重。

《十四行诗》在 Mode Designs 有售，不过如果你现在订购，12 月前你都拿不到键盘。这是一个漫长的等待时间，但需要注意的是，这不是一个有限的团购。Mode 希望在可预见的未来卖出这首十四行诗。
