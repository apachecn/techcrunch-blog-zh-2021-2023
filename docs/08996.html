ReturnLogic 的目标是让退货不再是一场噩梦

原文：https://web.archive.org/web/https://techcrunch.com/2022/08/09/returnlogic-bags-new-capital-to-make-returning-items-less-of-a-nightmare/

根据全国零售联合会的数据，去年网上销售额达到 1 万亿美元，其中 2180 亿美元的商品被退回。更重要的是，每售出 10 亿美元的商品，零售商就会获得超过 1.6 亿美元的回报。

我们已经看到公司变得有创意——例如，亚马逊在 2019 年与美国柯尔百货公司合作退货，以及快乐退货去年与史泰博等零售商合作。NewStore 总裁里克·伯杰(Rick Berger)告诉 TechCrunch，由于消费者“习惯于订购多种尺寸、颜色或变化的产品，在闲暇时试用，然后退回他们不想要的产品”，因此“在线购物热潮”导致了更多的退货。该平台使零售商能够在 iPhone 上运营他们的商店。此外，“电子商务巨头也让顾客习惯于期待快速、简单和免费的退货流程，不管卖家是谁。”

这些新习惯使得退货成为零售商的一大开支。虽然他们致力于让客户更容易退货，但像 ReturnLogic 这样的初创公司表示，根本问题实际上是退货流程中的操作缺陷。

这家总部位于费城的公司为电子商务品牌和零售商提供退货管理软件，将客户关系管理、第三方物流、库存管理和运输等技术整合在一起。用户可以插入 ReturnLogic 的 API 并访问他们可以用来进行产品、流程、制造和采购更改的数据，以提高他们的底线并满足客户。

公司创始人兼首席执行官彼得·索博塔(Peter Sobotta)在长期从事逆向物流工作后，于 2015 年创立了 ReturnLogic。他开始关注过去五年出现的新一代现代电子商务公司。他注意到的是，他们“无情地关注数据和终身价值，但他们就是做不到，”他说。

索博塔补充说:“我们看到了科维德的巨浪，每个人都看到了，海啸袭击了。”“我们现在看到的趋势是，零售商更加重视退货。因为当谈到投资时，如果你现在有一个地方可以存钱，并且你正在努力削减成本，但你像大多数服装公司一样有 30%到 40%的回报率，投资 1 美元的回报会直接达到底线。”

在过去的七年里，ReturnLogic 已经处理了超过 5 亿份退货，并发展到为数百个在线品牌和零售商提供服务，包括 Groove Life、EchelonFit、Oofos、Decathalon、filials 和 Sak。它还为大型零售商处理第三方保修退货，包括亚马逊、沃尔玛和百思买。

现在它准备扩张，并从一个投资者团体获得了 850 万美元的首轮融资，该投资者团体由 Mercury 牵头，参与方包括 Revolution's Rise of the Rest Fund、White Rose Ventures 和 Ben Franklin Technology Partners。新资本给 ReturnLogic 带来了总计 1150 万美元的股权。

Mercury 董事总经理布莱尔·加罗(Blair Garrou)告诉 TechCrunch，许多退货软件仍处于早期阶段，并专注于让客户更换产品，而不是其他退货方式，例如，如果有保修问题或其他原因。

“年轻的直接面向消费者的品牌直到收入达到 4000 万美元、5000 万美元或 1 亿美元时，才真正知道他们何时达到了这种运营效率低下的回报，”他说。“直到我和彼得谈过，了解了他的背景，我们才真正明白，这是有意义的。容量规划，以及了解你对客户的长期价值是什么，是市场正在走向成熟的东西。彼得的团队把握住了这一脉搏。”

尽管退货行业仍在蓬勃发展，但 Garrou 说，有很多公司都在努力做这件事。ReturnLogic 还加入了其他公司，包括 Seel 、 Loveseat 、 FloorFound 和 Loop Returns ，处理 Returns。像 Saara 这样的一些公司，目前已经筹集了 100 万美元，正试图以一种更加环保的方式来做这件事。

基于云的退货平台 ReverseLogix 的首席执行官 Gaurav Saran 告诉 TechCrunch，电子表格和手动流程不再适合退货管理。

“这就是为什么增加的技术升级投资正在流入供应链的这一领域，”Saran 说。“尽管退货在供应链问题上经常被忽视，但优化退货将产生巨大的影响，从高效的退货和维修到可持续发展目标、劳动力管理，甚至是新的收入机会。这就是为什么我相信退货管理系统将成为供应链中自己的解决方案类别。”

事实上，IBISWorld 估计今年美国产品退货管理市场的价值为 100 亿美元。

与此同时，ReturnLogic 在过去招聘了七名新员工，Sobotta 表示，计划在年底前使用新资本将员工人数从 35 人增加到 50 人，特别是在产品开发方面。此外，该公司将扩大其 API 产品，并融入其销售模式。

Sobotta 说，该公司正在与 Shopify 合作，但计划扩展到其他平台。

“我们的愿景是成为任何平台和工作流程上的退货操作系统，这让我们如此特别，因为这是一件非常困难的事情，”他补充道。“我们今年迈出的一大步是向我们服务的市场证明，我们开辟了更多的平台和更广泛的基础，这将带来我们的增长潜力。”
