欧盟利用关键的电信阶段推动重新考虑网络投资

原文：https://web.archive.org/web/https://techcrunch.com/2023/02/27/thierry-breton-mwc-2023-speech/

欧盟委员会(European Commission)发出了迄今最明确的信号，表示它打算在未来几年对欧盟的互联网连接融资方式进行重大干预。

今天上午，在移动行业协会 GSMA 的年度贸易展移动世界大会(MWC)上，欧盟内部市场专员 Thierry Breton 在舞台上发表了主题演讲，他警告说，当前的 gen 网络无法完成刚刚开始的“大规模转型”任务，这种转型是由虚拟世界等越来越沉浸式的技术和人工智能的发展推动的，这些技术为联网汽车和智能城市等新兴技术提供动力——这些技术依赖于高速率和带宽以及低延迟，但大规模的互联性来实现其颠覆性的承诺。

“我们需要为所需的巨额投资找到一种融资模式，尊重和保护我们欧洲法律的基本要素，”他在巴塞罗那年度电信大会第一天的早些时候告诉代表们，同时强调即将到来的支点必须确保最终用户的选择自由，尊重欧盟关于网络中立的现有规则，并通过确保公平、竞争的服务环境来实现竞争自由。

欧盟去年在“数字十年政策计划”下设立了自己的展望未来的摊位，该计划包括连通性的目标，以及培养数字技能和数字业务的计划。但是，由于网络基础设施是所有数字化事物的核心，因此委员会的政策目标与网络基础设施的发展息息相关，这使得电信公司在讨论发展该领域的监管格局时有了新的优势。

在他今天的演讲中，Breton 承认“连接革命正在对传统的垂直整合模式提出质疑”。

“数字行业将不得不走向去中心化和互操作性，专注于开发开放的、可互操作的系统，使不同的设备和应用程序能够协同工作，创造一个更加互联和高效的世界，”他预测道。(与此同时，GSMA 今天早上宣布了一项新的全行业倡议关于一个通用网络 API 的框架——称为GSMAOpenGateway——，它说这将“旨在为开发者提供对运营商网络的通用接入”——显然是唱着类似的赞歌。)

“真正的转变将是一切都无缝互联的 Web 4.0。通过虚拟双胞胎，万物的复制品，来管理和预测从建筑物到汽车，人体甚至地球的行为，”Breton 在他的演讲中进一步预测。“如果不使用超级计算，这种转变将是不可能的。我们将需要能够快速移动大量数据。这应该由遍布全球的可互操作系统来支撑。

“换句话说，明天，每个人的口袋里、车里、家里都会有一台超级计算机。”

他说，实现连通性方面的这一必要飞跃是欧盟周五发起的磋商的关键所在。欧盟磋商提出了许多未来如何资助固定和移动宽带网络基础设施的选项，包括所有数字服务提供商直接付款的可能性；或者来自所谓的“大流量生成器”的子集。(后者是电信公司一直公开呼吁的。)

与此同时，欧盟还通过了一项法规提案，即《千兆宽带法案》,据称该法案将提出新的规则，以便在整个欧盟范围内更快、更便宜、更有效地部署千兆网络。

“就在几天前，我们就互联互通领域及其基础设施的未来展开了广泛的磋商。许多人将此次磋商描述为大型电信公司和大型科技公司之间的公平份额之争。在今天提供网络的人和为他们提供流量的人之间的二元选择。但是让我说清楚:更多的事情处于危险之中。这不是一个既得利益是否应该压倒另一个既得利益的问题。这是关于实现我们面前连通性的巨大飞跃，”Breton 继续说道。

“我们没有充分利用拥有 4.4 亿公民的欧盟单一市场的潜力(乌克兰有 4.8 亿公民正在加入欧盟)。现在是我们认真讨论欧盟电子通信提供商跨境整合可能存在的障碍以及一体化无线电频谱市场的好处的时候了。与其他大陆相比，我认为这两个问题目前阻碍了我们的集体潜力。”

“我们正处于一场新革命的开端，”他补充道——随后，他表示愿意通过向创造性破坏的力量点头，来改变经济秩序。 “在未来几年，整个行业将需要进行彻底的转变，重新审视其商业模式。有些人可以把它描述为熊彼特式的时刻。工业将不得不适应生存。或者更积极地说，适应才能成功。”

在他的演讲之前，电信运营商 Telefonica 和 Orange 的首席执行官做了开场发言。

Telefonica 首席执行官兼董事长何塞·马里亚·阿尔瓦雷斯-帕莱特描绘了一幅根网络和分支网络转型的画面，这些网络转型将成为推动未来颠覆性技术(和经济增长)的引擎——但他表示，所有这些彻底的变化都取决于电信公司从那些在网络上产生最大流量的公司那里获得“公平的份额”。

阿尔瓦雷斯-帕莱特表示，运营商已经准备好并愿意重新装备他们的基础设施——成为他所说的“大规模分散的超级计算机”——并开发新的商业模式，以适应一个重新配置的管道充当“创新平台”(或各种破坏性应用的“实时大规模推动者”)的时代。但他认为，运营商应该从生态系统中获得比他们目前获得的更“平衡”的贡献——挑出大型技术，因为它们需要为其(受欢迎的)服务产生的流量做出贡献。

Orange 的首席执行官克里斯特尔·海德曼(Christel Heydemann)在她的信息中甚至更加直言不讳:认为欧洲电信公司处于一个完全“矛盾”的境地，他们的网络被认为是关键的社会基础设施，但她说他们需要进行的“大规模”投资“很难货币化”，消费者总是希望支付更少的钱，获得更多的东西。

“整个行业正处于十字路口，”她警告说，认为这种情况“不可持续”——并引用了一项对欧洲电信公司的调查，其中近一半接受调查的首席执行官表示，他们预计自己的业务不会再撑过十年。

“公平竞争规则始于承认今天的不平衡状况，”她补充道，敦促欧盟立法者“认识到电信行业一直是我们经济的最大贡献者之一”；接受运营商的抱怨是不公平的，他们应该背负数十亿美元的成本，而少数大型流量生成器可以骑在他们的网络上，一路笑到银行。

她敦促，未来网络的“适当融资框架”必须关注这些大型在线流量产生者——称欧盟的磋商是迈向“开放但公平”未来的“第一步”。

虽然 Breton 的讲话并没有明确说明对电信公司要求的屈服，但欧盟的情绪音乐肯定会取悦这些穿着行业套装的观众。

这位专员告诉 MWC 代表，他向他们传达的信息是，他们可以“依靠我共同实现 2030 年目标的决心”——这些目标要求欧盟抓住他的演讲中也表述为“连通性革命”的东西。

“当然，咨询只是我们在过去几年里一直在拼凑的拼图的一部分，”他补充说。“女士们，先生们，让我们停止用今天的眼光看世界，或者看后视镜。让我们着眼于未来，我们想要达到的目标以及达到目标需要采取的步骤。

“我鼓励你们都参加这次反思。每一次重大的技术革命，欧洲都参与其中，这一次，我希望欧洲也参与其中。我们不能错失引领互联领域技术革命的机会，也不能将它打造为一场欧洲革命。”

GSMA 的首席监管官 John Giusti 上周在接受 TechCrunch 采访时表示，运营商会倾向于一个具有法律约束力的谈判准则，这样他们就可以将最大的流量产生者带到谈判桌前讨论贡献。

在今天的新闻发布会上，当被问及委员会是否正在考虑通过磋商过程在各种选择中制定这样一部法典时，Breton 说现在说还为时过早。

“我们没有考虑任何事情——我们只是刚刚发起这次磋商。我们会看看会发生什么。”

“我们已经发起了一场非常大的磋商——接触所有的利益相关者，”他补充道。“以确保我们都有话要说。

“明天的网络当然将承载千万亿次数据，但也将使用群计算、边缘计算、极短延迟、网络验证和许多新功能，我们需要将这些功能放在一起，以建立我们为 web 4.0 做好充分准备的愿景，并确保我们正在建立其余的功能。

“我听说人们对此有巨大的兴趣。我见过的所有利益相关者。电信公司、平台、监管机构和消费者。我们现在还有 12 周的时间，然后我们当然会带着我们的回应回来。”
