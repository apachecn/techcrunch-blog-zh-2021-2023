谈到远程就业，雇员和雇主都面临着过多的好处和陷阱。虽然已经讨论了文化的利弊，但是从设置和维护的角度来看，还没有解决很多问题。当涉及到远程工作人员时，需要牢记重要的法律和税收影响。

虚拟团队早在新冠肺炎之前就存在了，但在过去的两年里，员工们通过搬出雇主所在的州，把不能进办公室变成了一项福利。对于创业公司来说，雇佣州外员工变得很普遍，因为远程优先的企业是从零开始创建的，人才比地点更重要。

如果你的创业启动或远程，请记住以下几点。

税收影响

远程工作人员对他们的公司有税务影响。具体来说，有一个国家的工资代扣税。这通常是雇员工作或提供服务的州/省所要求的，无论雇主在哪里。这意味着你的创业公司可能需要在几个州登记和扣缴所得税。

这些都是复杂的问题，通常，最好的方法是尽早聘请专家。

以下是我们向客户提出的问题:

  1. 你在各州的销售额和收入是多少？
  2. 你的员工在哪里？
  3. 你的办公室和其他财产在哪里？

美元数额和财产位置很重要，因为每个州在界定是否建立了联系时都有不同的门槛(稍后会详细介绍)。

这不是你可以忽视的事情。各州确实很关注。当您向政府机构注册时，州政府会收到您的纳税 ID 号和其他识别信息。这意味着你在该州有业务，你的业务将被监控，并追究任何由此产生的纳税义务。

例如，我们的一个客户在去年的一次收购中陷入停滞，因为他们被发现不符合其远程工作人员的要求。所以，在你有雇员的每个州注册是很重要的。
