Zoom 正在增加新功能，以与 Slack、Calendly、谷歌和微软竞争

在裁员1300 人(或 15%的员工)几周后，Zoom 正在推出新功能，以与包括 Slack、Calendly、谷歌和微软在内的众多公司竞争。这些功能包括人工智能驱动的会议摘要、基于提示的电子邮件回复、白板生成以及视频“Huddles”和会议日程安排。

公司希望你把更多的工作任务转移到它的工具上。为此，Zoom 向所有人开放了电子邮件和日历客户端。视频会议公司去年开始在会议之外的一个大的探索领域测试这些工具。也有托管的电子邮件和日历服务提供端到端的加密保护和付费用户的自定义域名。公司可以将这些服务作为微软 Exchange 和谷歌 Workspace 的替代品。

如今，很难花几个小时没有一家公司宣布生成性人工智能功能。Zoom 正在扩展其 Zoom IQ 助手，以提供人工智能支持的摘要，即使你中途参加会议，也可以“提出进一步的问题”。一旦会议结束，机器人会在 Zoom 的团队聊天功能上发布一份摘要。助理还可以总结团队聊天中的聊天线索。

到目前为止，Zoom IQ 能够记录精彩内容，将会议分成几个章节，并自动列出行动项目。去年，该公司还推出了 Zoom IQ for Sales ，旨在为销售团队提供视频通话的洞察力。

随着 Zoom IQ 帮助用户撰写聊天、电子邮件和白板会话，创建会议议程，Zoom 承诺了一个可生成的未来。该公司将在下个月邀请用户试用这些功能，并计划稍后进行更大范围的推广。该公司表示，它正在与 OpenAI 合作开发人工智能功能，但它没有说明这种合作关系是否只包括 API 的使用或更多。

该公司还推出了一些非人工智能产品。它在公开测试版中推出了 Zoom Scheduler 一个类似日历的工具，用于分享预约的可用性。Zoom 还引入了名为 Zoom Huddles 的虚拟共同工作空间，人们可以随时进入或退出。这项功能类似于的 Slack Huddles 功能，该功能于 2021 年推出，旨在进行快速的语音或基于视频的实时对话。

Zoom 似乎在这里打了很多仗。一方面，它正在引入生成性人工智能功能来创建电子邮件、会议议程和白板，以对抗微软和谷歌的冲击。这两家公司都宣布了新的工作场所生成人工智能功能。另一方面，它正在努力成为一个超越会议的相关工作场所工具，与 Slack、Calendly 和 Otter 竞争。

最近，Slack 宣布与 OpenAI 合作的 ChatGPT bot。与此同时，转录工具 Otter 推出了自动总结会议的 OtterPilot 助手。但不是这样。其他 会议相关 工具的大量 已经以不同的格式推出了 AI 驱动的摘要功能。

Zoom 的股票在过去 12 个月里下跌了 40%以上。在 2023 财年第四季度业绩中，该公司面临自 2018 年以来的首次季度亏损 1.08 亿美元。它预计本财年的增长将放缓至 1.1%，预期收入在 44.35 亿美元至 44.55 亿美元之间。
