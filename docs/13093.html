随着削减成本的压力越来越大，谷歌合并了地图和 Waze 团队

谷歌计划合并其地图产品和 Waze 的团队，Waze 是谷歌在 2013 年收购的地图服务。据华尔街日报报道，此次合并正值搜索引擎巨头感受到削减成本和整合业务的压力。

从周五开始，Waze 的 500 名员工团队将归属于谷歌的 Geo 组织，该组织负责监管地图、地球和街景。Waze 现任首席执行官 Neha Parikh 将离职。

谷歌告诉华尔街日报，它计划将 Waze 作为一项独立的服务——Waze 因其途中信息的众包而闻名，如测速摄像头、警车和道路杀人事件的位置。

谷歌还表示，预计重组中不会有任何裁员。然而，裁员在科技世界比比皆是，无论你是一家初创公司还是亚马逊的。在团队之间有冗余的地方，他们通常会受到最大的打击。事实上，谷歌表示，预计不同地图服务的重组将减少地图制作的重叠。

Alphabet 和谷歌首席执行官桑德尔·皮帅表示，他希望通过“使用更少的资源”让谷歌的生产力提高 20%这位高管在 9 月份的 Code Conference 上发表讲话时表示，由于过度招聘，该公司的速度变得更慢，他似乎暗示，合并从事重叠产品的团队将有助于公司保持领先地位。
