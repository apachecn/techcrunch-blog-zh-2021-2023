OpenAI 推出 ChatGPT Plus，起价每月 20 美元

原文：https://web.archive.org/web/https://techcrunch.com/2023/02/01/openai-launches-chatgpt-plus-starting-at-20-per-month/

OpenAI 推出 ChatGPT Plus，起价每月 20 美元

OpenAI today 旨在将这种已经成为病毒现象的现象货币化，为 ChatGPT 推出了一项新的试点订阅计划，ChatGPT 是一种生成文本的人工智能，可以撰写令人信服的类似人类的文章、诗歌、电子邮件、歌词等。OpenAI 表示，这项名为 ChatGPT Plus 的服务起价为每月 20 美元，与基本级别的 ChatGPT 相比有很多好处，包括即使在高峰时段也可以访问 ChatGPT，更快的响应时间以及优先访问新功能和改进。

免费的 ChatGPT 层将一直存在，不会消失。至于 ChatGPT Plus，目前只对美国的客户开放。OpenAI 表示，它将在未来几个月内开始邀请来自等待名单的人，并期待“很快”将 Plus 扩展到其他国家和地区

OpenAI 在一篇博客中写道:“我们推出 ChatGPT 作为研究预览，以便我们可以了解更多关于该系统的优势和弱点，并收集用户反馈，以帮助我们改善其局限性。”“从那时起，数百万人给了我们反馈，我们进行了几项重要的更新，我们看到用户在一系列专业用例中发现了价值，包括起草和编辑内容、集思广益、编程帮助和学习新主题。”

OpenAI 暗示，ChatGPT Plus 可能是未来几个计划中的第一个。在博文中，该公司表示，除了 API 之外，它正在“积极探索”低成本计划、商业计划和数据包的选项。

“我们热爱我们的免费用户，并将继续提供 ChatGPT 的免费访问。通过提供这种订阅价格，我们将能够帮助尽可能多的人支持免费访问，”该公司继续说道。“我们计划根据您的反馈和需求完善和扩展这项服务。”

OpenAI 在 1 月初预览了 ChatGPT Plus 的发布，宣布它“开始考虑如何将 ChatGPT 货币化”，并发布了一项调查，概述了“ChatGPT 专业版”计划的潜在定价和功能。然后，几周前，几个 ChatGPT 用户报告说被授予了每月 42 美元的 pro tier 权限，现在回想起来，这似乎是错误的。

尽管 争议 和 几个 禁令，ChatGPT 已被证明是 OpenAI 的宣传胜利，吸引了各大媒体的关注，并在社交媒体上催生了无数的模因。截至 12 月初，ChatGPT 拥有超过 100 万用户——无论以何种标准衡量，这都是一个令人羡慕的用户群。但这是一项昂贵的服务。根据 OpenAI 联合创始人兼首席执行官 Sam Altman 的说法，ChatGPT 的运营费用“令人眼馋”，在总计算成本中，每次聊天的运营费用只占几分钱。
