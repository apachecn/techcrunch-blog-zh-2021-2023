创业公司和风险投资

经济衰退对一些行业的打击比其他行业更大，但有一个行业似乎不受影响——至少目前如此——那就是应用程序开发。对应用程序的需求并没有消失，显然，对低代码平台和 API 的需求也没有消失，这些平台和 API 可以使应用程序的构建更快更容易。Appsmith 是一个开发商业应用的低代码平台，本周获得了 4100 万美元。与此同时, Courier 获得了 3500 万美元来建立一项应用通知服务。

硬件行业最近不那么宽容了。例子: Nothing ，一加联合创始人 Carl Pei 的新公司宣布，它不会把它的第一款手机——Phone(1)带到美国。障碍之一是在美国实现运营商的采用，因为美国运营商对不受欢迎的品牌是出了名的敌视—特别是在低迷的市场上。但还是很不幸。

寻求非稀释性资本以度过低迷期的生物技术创业公司的 3 条建议

对于生命科学初创公司来说，这是一个特别困难的时期。即使他们的技术改变了世界，它也需要几年时间才能上市。

在这种环境下寻求融资的大多数生物技术创始人都认为稀释性资本是他们唯一的选择，但这是短视的。

“在经济低迷时期，来自政府的非稀释性拨款或合同应该比以往任何时候都更有吸引力，因为它们提供了没有稀释的跑道，并成为头条新闻。”
