最后，苹果的 iPad 有一个默认的天气应用 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/06/06/finally-apples-ipad-has-a-default-weather-app/

显然，地狱已经结冰了。今天，苹果公司发布消息，解决了用户对 iPad 的长期抱怨，该设备最终将拥有自己的默认天气应用程序——如果你能相信的话，这是在平板电脑首次亮相大约 12 年后的。奇怪的是，苹果忽略了这个关键应用的添加，尽管在 2020 年收购了天气应用制造商黑暗天空和随着 iOS 15 的推出改进了 iOS 天气应用。该公司甚至为 iPadOS 主屏幕推出了天气小工具，但它没有推出原生应用程序，而是将用户链接到天气频道的网站。

这是一个不太理想的体验，因为该网站(weather.com)充斥着广告，不是苹果用户所期望的那种干净且易于导航的体验。苹果从未在这里解释过它的想法，但它为 iPad 推出了一个天气小工具，却没有配套的应用程序，这一事实让整个事情变得越来越奇怪。IBM 旗下的天气频道是否与苹果达成了未公开的流量交易？苹果真的认为用户更愿意访问网站而不是原生应用吗？这是反垄断吗？发生了什么事？！

虽然用户可以依靠大量高质量的第三方应用程序来访问 iPad 上的天气信息，但奇怪的是，像这样的苹果旗舰设备在这一点上却没有这样一个基本的实用程序。

但今天在苹果全球开发者大会的主题演讲上，该公司宣布，iPad 终于可以拥有自己的天气应用了。

“我们还将天气带入了 iPad，充分利用了令人惊叹的显示屏和美丽的动画，如轻轻移动的云，大雪和暴雨，”软件工程的苹果 SVP 克雷格·费德里吉(Craig Federighi)指出，他简要展示了新的原生应用程序的功能。这款应用并没有那么出色，很大程度上只是原生 iPhone 应用的 iPad 优化版。但这将是一个受欢迎的补充。

图片来源:苹果

此次更新与 iOS 16 的新功能不谋而合，该功能允许用户用实时天气壁纸定制设备的锁屏，显示一天中不断变化的当前状况。同样，iPad 应用程序也将包含这些相同种类的动画——云、雨、闪电、雪等。原生应用程序还将提供可点击的天气模块，允许用户深入到预报、温度、降水、紫外线指数、空气质量等领域。Federighi 解释说，当点击时，一个窗口将弹出并覆盖应用程序的主屏幕，显示详细信息。

此外，苹果紧随天气应用的消息，推出了期待已久的 WeatherKit。这款开发者工具包很好地实现了苹果的计划，允许第三方开发者在苹果自己的天气数据基础上开发应用程序——苹果在收购黑暗天空时曾表示对这一领域感兴趣。这是一个低调的宣布，对应用行业有更大的影响，因为它看到苹果推出自己的天气相关的创收服务。

随着该公司将开发者从黑暗天空天气服务转移到 WeatherKit，它表示，在测试期及以后，它将每月提供多达 50 万个 API 调用，作为其苹果开发者计划成员的一部分。此后，它将开始以下列速率充电:

  • 100 万次通话/月:49.99 美元
  • 每月 200 万次通话:99.99 美元
  • 500 万次通话/月:249.99 美元
  • 每月 1000 万次通话:499.99 美元
  • 每月 2000 万次通话:999.99 美元

相比之下，黑暗天空的 API 曾免费提供1000 次 API 调用，随后每次调用收费 0.0001 美元。

苹果表示，WeatherKit 需要 iOS 16、iPadOS 16、macOS 13、tvOS 16 或 watchOS 9，notes REST APIs 可以用于网站和其他平台。用于 WeatherKit 的 Swift APIs 将需要苹果操作系统和 Xcode 14 的测试版。

随着 iPadOS 16 的发布，原生 iPad 天气应用程序也随之推出。
