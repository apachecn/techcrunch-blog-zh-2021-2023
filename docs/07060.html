苹果没有告诉你的所有方便的 iOS 16 功能

苹果在它的 iOS 16 公告中有很多大热门，包括锁屏改造、照片更新和改进的邮件应用体验。除此之外，还有很多事情，该公司只是简单地浏览了一下，或者根本没有提及。

随着越来越多的人测试 iOS 16 开发者测试版，每天都有一些小而漂亮的功能被发现。所以这里总结了一些方便的功能。

  • 你可以根据时间表、地点或打开特定应用的时间来打开聚焦模式。
  • iOS 16 对焦点下的通知有单独的允许和静默列表。到目前为止，您只能将应用程序包含在允许列表中。
  • 苹果在今年的更新中真正关注焦点，敦促用户根据自己的需求建立可定制的配置文件。新的触发器非常有用，可以在你进入办公室或开始晨间散步时激活特定的设置。
  • Face ID 可以在横向模式下工作，但只适用于部分 iPhones。没有关于支持型号的官方信息。但是根据的一些 推文，它可能只适用于 iPhone 13 和以后的型号。
  • 有了新的 Live Text 更新，您将能够轻松翻译网页中嵌入的图像。
  • 该公司在 iOS 16 中为有听力障碍的用户引入了实时字幕——这也适用于 FaceTime。
  • 在辅助功能>触摸下有一个新的开关，可以让你通过按下锁定键来结束通话。当你的手脏兮兮的，不想触摸屏幕时，这非常有用。

这太酷了。图片来源:苹果

  • 苹果将在 iOS 16 的 Notes 中加入实时单元对话功能。您可以写下 1，000 g，点击完成，然后再次点击该数字以查看单位转换。在我们早期的测试中，它适用于重量、货币和液体体积的测量。默认情况下，此功能是打开的。

希望更多的应用程序将包括这一转换功能。图片来源:苹果

  • “快速笔记”选项将出现在“共享”菜单中，因此您可以快速记录任何内容，如文章或图片。

共享菜单中新的快速笔记选项。图片来源:苹果

  • 即将到来的 iOS 更新有漂亮的 iMessage 功能，如撤销发送和编辑。此外，您还可以恢复长达 30 天的已删除邮件。
  • “信息”应用程序将有一个专门的未读信息部分。

图片来源:苹果

  • 苹果将向 iOS 用户发布独立于版本更新的自动安全更新。一旦 iOS 16 可供所有人使用，该功能将自动启用，您将无需担心下载更新。
  • 您将最终能够看到并复制您存储的网络的 Wi-Fi 密码。

轻松查看您的 Wi-Fi 密码并复制它。图片来源:苹果

  • iOS 16 默认锁定隐藏照片和已删除照片文件夹。你必须使用 Face ID、触控 ID 或密码来解锁它们。这给了你一个内置的工具来隐藏你的敏感照片，而不需要使用第三方应用程序。
  • 在密码部分，苹果会让你选择一个默认的验证码应用程序，包括 iCloud Keychain 和 Google Authenticator。
  • 当你把东西从一个应用程序拷贝到另一个应用程序时，粘贴板会请求许可。这不仅仅是通知你一个应用程序可以访问你的粘贴板。

图片来源:苹果

以安全为重点的更新表明，苹果希望通过让消费者对应用权限和工具进行精细控制来保护敏感内容，从而让消费者的安全“更容易”。此外，它还消除了手动安装更新的负担。

  • iOS 16 中的新照片应用将允许您跨图片复制和粘贴样式。这意味着您可以将一些滤镜和效果应用到一个图像，并将其复制到其他图像中。
  • 你可以在 Apple Music 上找到喜欢的艺术家，并收到他们新作品的通知。这比不上 Spotify 的 What's New feed ，但总比没有好。

喜欢艺术家得到关于他们新作品的提示？帮我报名。图片来源:苹果

  • 在 iOS 16 中，Apple Music 将允许您按照标题、艺术家、专辑、发布日期以及最重要的播放列表顺序对播放列表进行排序。所以你可以从列表中添加的最后一首歌开始听。

如此多的排序选项。图片来源:苹果

这两个有用的功能都是针对那些使用音乐应用程序来发现音乐和建立播放列表的人的。

  • iOS 16 有一个新的通知设置，允许你在锁定屏幕上以不同的风格显示它们。

图片来源:苹果

这些是一些面向消费者的更新，预计将于秋季在 iOS 16 中发布。更重要的是，苹果发布了一些有趣的开发者 API，这些 API 可能会成为这个新版本应用程序的一些伟大功能。

  • 一些 新的 API表明，可能会支持向谷歌 Chromecast、亚马逊 Echo 和 Sonos 等第三方设备播放媒体。将媒体从你的 iPhone 投射到支持 Chromecast 的电视上会很棒。
  • 苹果公司的 RoomPlan API 将允许开发者创建一个基于 AR 的房间和其中物品的地图。
  • 苹果将允许网站在 iOS 16 上发送推送通知。这将是一个选择加入的功能，所以你不必担心通知垃圾邮件。
  • 该公司将使开发者能够使用其附近的交互框架——该框架支持 AirTags 和 Find My——来触发现实生活中的动作，例如在屏幕上用箭头指示方向以找到一个对象。
  • 苹果在 iOS 16 中引入了开发者模式，以防止用户“无意中在他们的设备上安装潜在有害的软件。”这意味着它让侧装应用变得更加麻烦。苹果在其所有的监管或法律斗争中强烈反对侧装。这种切换只代表了这种信念，即只为具有一定技术知识的人启用该功能。
  • 开发人员将能够使用新的 Live Activities API 在锁定屏幕上实现诸如出租车行程跟踪和体育比赛记分卡等体验。

实时活动将在锁定屏幕上实现实时更新体验。图片来源:苹果

这绝不是 iOS 16 的所有功能的详尽列表。当我们发现更多他们的时候，我们将确保更新帖子。
