TLDraw 提供无需任何登录的协作白板| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/12/16/tldraw-offers-a-collaborative-whiteboard-without-any-login/

有很多工具可以满足协作式白板的需求。Figma 在 9 月份完成了被 Adobe 以 200 亿美元收购的交易。有米罗公司，它是公司，价值 175 亿美元，在一月份进行了 C 轮融资；苹果公司在本周早些时候发布了面向用户的自由形式应用。在所有这些中， TLDraw 为用户提供了一个无需任何登录的协作画布。

TLDraw 消费者应用程序

TLDraw 从一开始就很容易使用。这是一个空白的无限画布，让你画线条或物体，写文本和插入媒体，如图像，视频或 gif。

更重要的是，您可以通过链接轻松地与您的同事共享此协作板。如果你不希望任何人改变董事会，你也可以分享一个只读链接。这就像 Google Docs 一样，它让你既可以与多个参与者共享文档，也可以只读方式共享。

TLDraw 的测试版。图片来源: TLDraw

网站的新版本处于测试阶段，提供了更多的功能。它有更多的形状、颜色和线条可供选择；您还可以插入新类型的对象，如框架或便笺条；还有复制和移动对象的附加选项；并有多种格式，如 SVG，PNG，JPG 和 JSON 输出板。

要使用所有这些功能，您不需要任何帐户或登录。因为这是在网络上，你和你的同事可以很容易地使用它，而不依赖于平台。

这个公司

TLDraw 是 Steve Ruiz 在 2021 年作为开源项目推出的。他在其他开源项目的基础上构建了 TLDraw，比如 perfect-freehand 和 globs.design 。

在完成这些项目后，我想创造一些新的更“形状不可知论”的东西，让我可以在画布上画出任何形状；这就是后来的 TLDraw。Ruiz 告诉 Techcrunch:“我在公共场合(主要是通过 Twitter 的 gif)构建了所有这些，TLDraw 的内容很快就变得非常受欢迎。

完美手绘项目演示。图片来源: TLDraw

TLDraw 已经筹集了 270 万美元的种子资金，由 Lux Capital 牵头，Amplify Partners、Sabrina Hahn、Vercel 的 Guillermo Rauch 和 NP-Hard Ventures 参与。这轮投资还包括其他投资者，如 Soleio、Badrul Farooqi、Michael Stoppelman、Tom Preston-Werner、Muse 的 Adam Wiggins、campus 的 Brian Lovin、estoral . ai 的 David Khourshid、Runway ML 的 Cristóbal Valenzuela 和 Prisma 的 Johannes Schickling。

该公司有五名全职员工，除了 GitHub 赞助商捐赠的资金外，目前没有任何收入。展望未来，这家初创公司希望在许可和支持方面努力创造收入。Ruiz 表示，该团队可能会在付费墙后提供一些功能，如团队成员之间的协作工具。

开发者工具和未来之路

虽然 TLDraw 可供所有人免费使用，但该公司还提供开发人员工具来集成其画布。可以把它想象成“白板即服务”

Ruiz 告诉 TechCrunch，当他开始从事这个项目时，他希望提供一个平台，让任何人都可以使用所有可用的基本工具来构建协作画布。

“当前的一组白板应用程序假设，如果您在一个无限的画布上工作，您只是在处理可视组件。但通过 TLDraw，我们希望重视各种格式，”他说。

这位初创公司创始人补充说，现有的工具提供了自己的协作设计，很难根据公司自己的需求进行定制。使用 TLDraw，团队可以在有效的交互设计和协作画布方面获得一个强大的起点。

图片来源: TLDraw

“有数百个小功能——选择、调整大小、旋转、在不同父对象之间拖动形状，甚至撤销和重做——解决方案在数学上非常复杂，同时也非常主观。Ruiz 补充道:“正确的实现是‘感觉正确的’，这可能是一个非常令人沮丧的目标。”。

已经有相当多的项目使用 TLDraw 作为基础。有整洁的基于网络的绘图应用，如 MacPaint 和OkSo；开发人员也使用 TLDraw 制作有用的故事构建工具，如lengendekeeper和world anvil；基于人工智能的视频生成平台 Vidext 正在将其作为基础画布；开源虚拟教室公司 BigBlueButton 将其用作课堂白板工具。

该公司现在正致力于开源 TLDraw 网站的新测试版，同时将所有协作版本和 Visual Studio 代码插件转移到该网站。更重要的是，在未来 12 个月内，这家初创公司计划根据社区反馈添加更多功能，如标尺、对象列表和动画。
