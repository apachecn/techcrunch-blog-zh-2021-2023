Slack 现在支持个人资料中的发音指南

不是每个人都赞同远程工作的想法。但对于那些人来说，Slack 今天发布的新个人资料功能应该会让一些日常任务变得更容易，特别是入职。

Slack 现在支持个人资料的发音指南，允许用户以语音拼写或录音的形式添加姓名发音。它们将于今天开始在 Slack 客户端的显示名称和发音部分下向一些用户显示，并在未来几周内全面推出。

名字发音建立在 Slack 的其他包容性努力的基础上，包括个人资料中的代词字段。

在一个相关的发展中，悬浮卡——本质上是弹出式的——已经在 Slack 中出现。将光标移动到一个人的名字上，将显示其简档的压缩悬停卡版本，以及开始“群聚”(即音频对话)、呼叫或直接消息的链接。

图片来源: Slack

最后，Slack 配置文件现在包括三个新字段:联系信息、人员和关于我。联系信息显示电子邮件和电话的详细信息，而人列出了该人工作的团队以及他们在组织层次结构中的位置。“关于我”本质上更具动态性，但可以包含开始日期、语言甚至宠物和生日等内容，这取决于用户选择分享的内容。

Slack 的其他升级涉及 Atlas，这是 Slack 为其 Business+和企业网格计划提供的附加服务。一种称为智能标记的新数据类型允许管理员创建自定义字段，这些字段可以转换为可搜索、可过滤的标记，如“专业知识”、“重点领域”和“语言”Flexible Text 是另一项升级，它使个人资料字段能够支持多达 5000 个丰富的文本字符，包括项目符号、链接、换行符、代码块、表情符号等。

图片来源: Slack

Slack 的产品总监 Maxwell Hayman 在一份声明中说:“通过帮助人们了解他们的同事是谁，我们重新设计的档案使队友更容易合作，感觉联系在一起，并一起做最好的工作。”。
