涂料安全从秘密行动中脱颖而出，撼动 SWG 市场

原文：https://web.archive.org/web/https://techcrunch.com/2022/09/14/dope-security-stealth/

总部位于旧金山的网络安全初创公司 Dope Security 已经从 stealth 启动了 400 万美元的资金，以实现安全网络网关市场的现代化。

安全网络网关(SWG)是一种网络安全设备，在用户和恶意网络流量、存在漏洞的网站、恶意软件和其他基于互联网的网络威胁之间充当屏障。虽然一点也不性感，但随着员工从严格控制的办公室环境转向不太安全的家庭网络，swg 在最近向远程和 T2 混合工作的转变中变得至关重要。

虽然 swg 对于员工不再处于内部企业网络的组织来说是一个重要工具，但 Dope Security 的创始人兼首席执行官 Kunal Agarwal 表示，大多数传统的 swg 在一个偏远的、云优先的世界中不再适用。

“已经出现了安全的网络访问，今天每个主要组织都保护或确保你可以从笔记本电脑上访问的内容，”Agarwal 告诉 TechCrunch。“他们这样做的方式是个问题。这相当于从伦敦飞往都柏林，中途在德国停留。”

他补充说，这些中途停留，加上难以部署的解决方案，会导致停机、设备外解密、页面加载明显变慢，并降低最终用户的工作效率。

Agarwal 是一名网络安全老手，从小就开始从事黑客活动，在 Forcepoint 和赛门铁克工作期间，他对传统的 SWG 解决方案感到失望，在那里他花了数年时间试图改造现有的 SWG 解决方案，以解决他说从未设计来解决的问题。“我开始看到所有这些客户抱怨停机、可靠性和性能问题，”Agarwal 说。

正是这一点导致了 Dope Security 的创立，这是一家以湾区俚语命名的初创公司。Dope Security 是一个直飞 SWG，它消除了传统提供商所需的数据中心中途停留架构，而是直接在端点上执行安全性。根据 Agarwal 的说法，这种架构将性能提高了四倍，并在保护企业免受基于网络的威胁时确保了隐私和可靠性。

Agarwal 表示，他的公司的技术可以在五分钟内部署完毕，并通过基于云的控制台为网络防御者提供见解，并与微软 365 和谷歌账户集成——并且已经在客户手中。

该公司的 400 万美元投资由 Boldstart Ventures 牵头。Agarwal 表示，该公司有 30 名员工，大多数是前 Forcepoint 和赛门铁克员工，已经在寻找下一个试图扰乱的市场。

“我们想打造的不是一个产品，而是一个完整的产品组合，”Agarwal 说。“我们希望客户看到这些产品后会说‘是啊，太棒了。’"
