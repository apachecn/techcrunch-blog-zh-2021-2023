ShareGPT 让您轻松分享您的 ChatGPT 对话

OpenAI 的 ChatGPT bot 是科技界炙手可热的新感觉，在发布仅几天后就获得了超过一百万的用户。聊天机器人正在为广泛的问题提供答案，以其能力和速度给许多人留下深刻印象，同时有时也让人们对其表现感到有趣和困惑。你可能已经在你的 Twitter 时间轴上看到了许多这样的截图，也可能已经捕捉并分享了一些答案，并获得了乐趣。现在，两位开发人员——Steven Tey 和 Dom Eccleston 已经开发了一个名为 ShareGPT 的 Chrome 扩展，以便更容易地捕捉和分享人工智能的答案。

ShareGPT 用 ChatGPT 捕捉完整的对话，并生成一个 URL 与他人分享。所以不用多张与 AI 聊天机器人的对话截图，直接分享网址(就像这张)。

一旦你安装了 Chrome 扩展，进入 ChatGPT 网站，开始与机器人的对话。收到答案后，您会在底部看到一个分享按钮。您可以继续聊天，或点击“共享”按钮为特定对话生成一个 URL。

ShareGPT 为 ChatGPT 对话启用共享按钮。图片致谢: ChatGPT/ShareGPT

有趣的是，当你在 Twitter 上分享网址时，你会看到与 ChatGPT 对话的预览，因此视觉元素不会丢失，人们不一定需要前往网站阅读简短的对话。

ChatGPT 目前处于研究预览阶段，可以免费使用。虽然它有语言和事实上的限制，但它仍然是一个有趣的工具。这是一种工具，让任何人都可以在没有太多技术知识的情况下尝试人工智能的能力(或弱点)。
