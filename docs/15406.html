Anthropic 开始为选择初创公司提供文本生成人工智能模型

原文：https://web.archive.org/web/https://techcrunch.com/2023/02/27/anthropic-begins-supplying-its-text-generating-ai-models-to-startups/

由前 OpenAI 员工共同创立的热门人工智能初创公司 Anthropic ，已经开始向合作伙伴提供其人工智能文本生成模型的访问权限。

第一家宣布整合人择模型的商业风险企业是罗宾·艾，这是一家法律科技初创公司，从包括 Plural、Episode 1 和谷歌黑人创始人基金在内的投资者那里筹集了超过 1300 万美元。Quora 为 iOS 和 Android 开发的实验性聊天机器人应用程序 Poe ，使用了人类模型，但目前还没有商业化。

Robin 首席执行官 Richard Robinson 透露了一些关于 Anthropic 关系的细节，但他告诉 TechCrunch，Robin 在法律文本数据集上对 Anthropic 模型进行了微调，以起草和谈判合同。

罗宾逊在一份声明中说:“我们非常幸运地成为 Anthropic 在法律领域的启动合作伙伴——该团队对人工智能安全的关注与我们的‘律师在回路中’软件即服务产品相吻合——旨在管理即使是最先进的模型‘幻觉’的风险。”。

Robin AI 是首批使用人类模型的商业企业之一。图片来源:罗宾·艾

Anthropic 对其在生成文本人工智能领域的产品化计划一直相对沉默，而是更愿意专注于学术研究。但去年年底，该公司推出了一个人工智能系统的封闭测试版，名为 Claude ，类似于 OpenAI 的 ChatGPT ，似乎在关键方面对原版进行了改进。克劳德是使用 Anthropic 开发的一种称为“宪法人工智能”的技术创建的，该技术旨在提供一种“基于原则的”方法来使人工智能系统与人类意图保持一致——让类似于 ChatGPT 的人工智能使用一套简单的原则(例如，避免给出有害的建议)作为指导来回答问题。

对克劳德的初步印象很好。但是和 ChatGPT 一样，这个系统也有局限性，比如对问题给出危险的答案(比如如何在家制造冰毒)以及做出不一致的、事实上错误的陈述。

目前还不清楚罗宾使用的模型是克劳德还是其他衍生模型——罗宾和 Anthropic 都不会说。即使在反复催促下，Anthropic 也不会透露它目前有多少合作伙伴(或者他们是如何与他们合作的)以及它计划开放多少模型用于商业用途。

但毫无疑问，Anthropic 正感受到来自投资者的某种压力，要求收回投入其人工智能技术的数亿美元。

最近，谷歌承诺向 Anthropic 投资 3 亿美元，收购这家初创公司 10%的股份。《金融时报》最先报道了该交易的条款，根据该条款，Anthropic 同意让谷歌云成为其“首选云提供商”，两家公司“共同开发人工智能计算系统”

Anthropic 的成立并不完全是为了追求利润。OpenAI 前研究副总裁达里奥·阿莫代伊(Dario Amodei)于 2021 年作为一家公益公司成立了该公司，并带走了一些 OpenAI 员工，包括 OpenAI 前公共政策主管杰克·克拉克(Jack Clark)。Amodei 从 OpenAI 中分离出来，原因是对公司的方向产生了分歧，即这家初创公司越来越注重商业。

但是人工智能系统的开发和维护成本很高。不断膨胀的成本导致 Anthropic 寻求外部支持，包括一批投资者的 5.8 亿美元，其中包括名誉扫地的 FTX 创始人萨姆·班克曼-弗里德、卡罗琳·埃里森、吉姆·麦克拉夫、尼萨德·辛格、贾恩·塔林和新兴风险研究中心。

创业伙伴关系——以及大型科技投资——是否意味着 Anthropic 的优先事项发生了转变尚不清楚。但是明确的是该公司认为其技术与 OpenAI、Cohere 和 AI21 Labs 等竞争对手有所区别，这些公司都通过 API 提供对其文本生成人工智能的付费访问。
