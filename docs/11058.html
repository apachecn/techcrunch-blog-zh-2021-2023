苹果将包括 iCloud 和 Apple Music 在内的更多服务引入微软平台

在今天早上的一场聚焦 Surface 的活动中，微软宣布将苹果的 iCloud 存储服务与 Windows 11 中的 Photos 应用整合。从微软商店安装 iCloud for Windows 应用程序并选择同步 iCloud 后，Windows 设备的 iPhone 用户将能够在照片中查看他们的 iPhone 照片和视频。

参与微软 Windows Insiders 计划的 Windows 用户可以从今天开始获得最新的 iCloud for Windows 应用程序，从而实现集成。

CNET 援引微软的话说:“在过去的几年里，拥有安卓手机的 Windows 用户已经体验到了将信息、通话和照片直接集成到他们的 Windows PC 上的承诺，让他们生活中最重要的两个设备更加紧密地联系在一起。”。“我们让顾客在他们的 Xbox 和 Windows 设备上访问他们的 iPhone 照片和他们喜欢的苹果娱乐变得比以往任何时候都更容易。”

Windows 11 中 iCloud 集成的初步观察。图片来源:微软

在一个相关的发展中，苹果宣布将为更多的微软平台带来包括 Apple Music 和 Apple TV 在内的第一方流媒体服务。苹果音乐将于今日开始登陆 Xbox 游戏机，而苹果音乐和苹果电视应用将于明年某个时候登陆 Windows。

近两年前，苹果电视在 Xbox 游戏机上推出后，Windows 和 Xbox 上新的苹果服务搭售。这是一个漫长的过程，但混合 Windows-iOS 家庭无疑会欣赏更紧密的集成。
