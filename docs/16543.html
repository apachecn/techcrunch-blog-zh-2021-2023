GitHub 大幅削减印度的工程团队

受欢迎的开发者平台 GitHub 几乎解雇了其在印度的整个工程团队，因为微软旗下的公司在全球市场疲软的情况下削减了开支。

一位知情人士说，这家开发商公司已经在南亚市场裁员 100 多人。GitHub 在周二通知了员工裁员的消息。经营时事通讯《务实工程师》的前软件工程师 gergely Orosz首先报道了裁员。

GitHub 发言人告诉 TechCrunch，裁员是该公司在 2 月份披露的精简工作的一部分。GitHub 曾表示，到 3 月结束的季度，它将在 T2 裁员约 10%。

“作为 2 月份共享的重组计划的一部分，今天进行的裁员是艰难但必要的决定和调整的一部分，以保护我们业务的短期健康，并赋予我们投资于我们长期发展战略的能力，”该发言人说。

印度是全球最大的开发商市场之一，也是许多大公司的关键工程中心。
