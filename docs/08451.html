可观察性创业公司 Better Stack 获得 1860 万美元新资金

原文：https://web.archive.org/web/https://techcrunch.com/2022/07/21/observability-startup-better-stack-lands-18-86m-in-new-cash/

在软件行业，可观察性工具为企业提供了关于其生产应用健康状况的见解。在最好的情况下，可观察性工具可以标记应用程序何时表现不佳，并帮助确定问题的根本原因。但很少会这么简单。根据 Splunk 最近进行的一项调查，86%的开发人员将与技术相关的困难视为对可观察性采用的挑战。在调查中，他们列举了遗留工具不足、缺乏平台选项和碎片化(即工具太多)等主要障碍，以及对开源工具的担忧。

近年来，随着像 Datadog 、 New Relic 、page duty和前面提到的 Splunk 这样的供应商的增长，可观察性市场已经成熟。但是还有改进的空间。至少，这是 Juraj Masar 和 Veronika Kolejak 的说法，他们是 Better Stack 的联合创始人，这是一种将监控、日志记录、事件管理和状态页面与协作工具相结合的可观察性产品。

Better Stack 今天宣布，它在由 Creandum 牵头的 A 轮融资中筹集了 1860 万美元，参与方包括 Susa Ventures、K5 全球、Credo Ventures、Kaya 和 Lachy Groom，以及 Codecademy 联合创始人扎克·西姆斯等知名天使投资人。担任 Better Stack 首席执行官的 Masar 表示，资金将用于招聘，目标是在年底前将团队从 14 人扩大到 30 人左右。

“我们本质上是软件建造者——我们正在制造我们一直想自己用的产品，”Masar 在一次采访中说。拥有数家创业公司的马萨尔之前是服装电子商务公司 Represent.com 的工程副总裁。在创建金融市场数据平台 Wallmine 和加入 Better Stack 之前，Kolejak 曾在 Shopify 和谷歌担任过各种软件工程工作。“首轮融资是一次机会主义融资——我们不需要融资就能生存下去，”Masar 继续说道。“我们希望利用目前的裁员来继续快速建立一个令人惊叹的团队。”

Better Stack 的产品建立在 ClickHouse 之上，click house 是 Yandex 开发的数据库管理系统，允许用户通过 SQL 查询生成实时分析报告。ClickHouse 于 2016 年开源，并于 2021 年成立了一个公司实体——click house，Inc .——来维护和改进该项目。

Better Stack 独立于主线 ClickHouse 项目，并针对可观察性用例进行了定制。例如，该平台将应用和服务的日志和指标聚合在一个技术堆栈中，以跟踪这些应用和服务的可用性、正常运行时间和性能。Masar 说，Better Stack 可以通过随叫随到计划和升级策略自动通知员工，并提供员工可以用来一起分析可观察性数据的协作工具。

除此之外，Better Stack 可以生成数据的可视化，并支持定制报告的 SQL 查询。这类似于其他软件可观察性解决方案，如观察、蝠鲼和克里尔。但是 Masar 声称 Better Stack 的产品更全面，而且便宜 10 倍。

“在更好的堆栈之前，用户通常需要组合至少三种不同的工具来获得相同的功能。“疫情对我们的业务产生了积极的影响，因为公司开始探索高成本效益的替代方案，以取代昂贵的传统可观测性提供商，”Masar 说。“工程团队可以直观地使用更好的堆栈，因为它利用 SQL 进行数据查询，而不是使用定制的查询语言。[此外，]它专注于类似 Figma 的协作，这使得开发人员可以实时地一起工作。最后，它简化了产品集成。”

无论如何，包括 Time.com、迪卡侬和埃森哲在内的 1400 多个公司团队正在为更好的堆栈付费。Masar 拒绝透露收入，但表示超过 70，000 名开发者正在积极使用该公司的平台。
