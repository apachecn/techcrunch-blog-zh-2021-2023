随着伊朗扼杀互联网，激进分子为上网而战

原文：https://web.archive.org/web/https://techcrunch.com/2022/10/05/iran-internet-protests-censorship/

22 岁的妇女马赫莎·阿米尼在被拘留期间死亡，根据伊朗严格的女性着装规定，她没有戴头巾而被逮捕。9 月，抗议者涌上街头，抗议的视频和图像在伊朗国内的网上传播。此前闻所未闻的行为，如销毁伊朗最高领袖或妇女摘下头巾的照片，通过智能手机视频传播开来。

但是政府打击了互联网接入；WhatsApp、Signal、Viber、Skype 和 Instagram 被屏蔽。

现在，一些团体正在动员起来攀登那些不断增长的高墙。最近，一群积极分子想出了一个新的方法，包括在伊朗境内安装服务器，并让国外的技术社区参与进来。

随着该组织寻求更多的参与，它的方法——使用国内的服务器作为一种“互联网接入的特洛伊木马”——正在获得自由言论社区的普遍认可。目前，大约有 200 人正在使用由发明这种方法的激进分子组织运行的服务器；这种方法现在已经发布在 GitHub 上，该组织还没有追踪有多少人可能也在使用它。

“如果伊朗已经将其住宅互联网接入与世界其他地区隔离，但其位于该国境内的服务器仍然可以访问伊朗住宅 IP 以及外部互联网，那么在该国境内建立服务器来中继流量应该是可行的，”电子前沿基金会的高级技术人员比尔·布丁顿告诉 TechCrunch。

诚然，伊朗互联网用户对互联网关闭并不陌生。超过 8500 万人生活在伊朗，大约 84%的居民可以访问互联网。为了压制不同意见，伊朗政权经常使用互联网封锁和审查来阻止抗议活动的视频和图像传到民众手中。

据大赦国际称，2019 年，当 100 多名抗议者在因燃料价格引发的大规模骚乱中丧生时，该国的互联网接入被切断了 12 天。

规避工具和技术建议的细节越来越多，但审查者的范围很广。据报道，甚至允许玩家聊天的在线视频游戏也被关闭。

据互联网监测人员称，在过去两周多的时间里，伊朗三大移动运营商从当地时间每天下午 4 点开始屏蔽服务长达数小时，最长可达 8 小时。这意味着有线网络成为一个重要的信息来源。

因此，寻找绕过这些街区的方法对伊朗人来说已经很熟悉了。

传统上，他们会求助于 VPN 来保持连接。

然而，随着许多 VPN 现在被屏蔽，Tor 网络——允许匿名浏览网页，可以绕过互联网屏蔽——对于传播有关最近反政府抗议活动的视频和信息变得尤为重要。维护 Tor 网络的美国非营利组织 Tor Project有一份波斯语和英语的详细指南，介绍如何在伊朗使用 Tor 访问互联网。

这就是为什么互联网自由活动人士每小时都在努力帮助伊朗人重新上网，并向国外寻求帮助，以保持信息渠道对抗议者开放。

伊朗以外的技术社区在帮助抗议者重新上网方面发挥了重要作用。

谷歌在一条推特中说，它的“团队正在努力使我们的工具广泛可用，遵循美国适用于通信服务的最新制裁。”

消息应用软件 Signal 和 WhatsApp 一直在通过代理让他们的服务在伊朗国内可用。

“我们正在努力让我们的伊朗朋友保持联系，并将在我们的技术能力范围内尽一切努力保持我们的服务正常运行，”Meta 拥有的消息应用程序在 9 月份发推文。

在其他方面，在美国政府允许私营公司向伊朗提供互联网接入后，埃隆·马斯克在伊朗启动了他的 Starlink(T1)低地球轨道卫星宽带服务。此前，在乌克兰的互联网因俄国入侵而中断后，Starlink 在乌克兰激活了 T2。

然而，需要一个特殊的终端，包括一个 20 英寸的卫星天线来接收信号，这使得几乎不可能将硬件运送到这个被封锁的国家。

现在，人权和其他观察人士担心，该政权正在切断互联网连接，并使 VPN 和代理服务器无法访问，这是 2019 年抗议活动的可怕重演，在大停电期间，数百人丧生。

所有这一切导致一群活动家采取了新的方法，这种方法也涉及 Tor，但也涉及伊朗以外的技术社区。

加拿大安大略省——2022 年 9 月 23 日:示威游行期间，在一个路标的背面看到一张贴纸，上面写着“伊朗:互联网瘫痪，他们正在杀害人民”。数百人聚集在加拿大多伦多纪念 Mahsa Amini，并抗议伊朗政府。图片来源: Katherine Cheng/SOPA 图片/LightRocket via Getty Images

TechCrunch 采访了一名代表该组织在中国的科技企业家，以了解该组织如何让互联网恢复工作，传播有关示威的信息。(我们不公布姓名是为了保护他们的安全。)

这位企业家说，获得准入已经成为与当局的“猫和老鼠的游戏”。Tor 项目已经成为解决这些问题的重要途径，该项目使用自由和开源软件来实现匿名通信。(事实上，Tor 还允许世界各地的用户使用反审查系统 Snowflake 来帮助伊朗用户访问被审查的网站和应用程序。)

随着 VPN 阻塞的增加，Tor 匿名网络的使用越来越多，但即使这样也遇到了障碍。

“VPN 服务为伊朗人提供免费服务。Tor 项目正在增加桥梁，但很少会起作用，”他通过一个加密的消息应用程序说。

他说，该政权现在也在“非常积极地断开 VPN 连接，在断开连接之前，你不会与常规 VPN(与伊朗以外的服务器)保持超过几分钟的连接”。

“政府已经屏蔽了大多数非伊朗 IP 地址的住宅连接(实质上是一个速度受到限制的白名单)和所有非伊朗 IP 地址的移动 3G/4G 数据(大多数人通过移动数据连接到互联网)。所有这些服务(VPN/Tor/etc)都在伊朗境外有服务器，这是没有用的，”他告诉我。“人们无法与它们建立联系。”

新方法使用伊朗数据中心内的服务器，这些服务器“可以全速连接到互联网。”他和其他几个人现在正在购买伊朗数据中心的服务器，在这些服务器上建立 VPN 服务器，并确保所有传入的流量都通过隧道传输到伊朗以外的另一台服务器。

“然后，伊朗的 VPN 服务器连接信息就与那些可以在一天中的任何时间从任何设备连接到他们的人共享，”他说。他补充说，在抗议最激烈的晚上，互联网几乎完全关闭，但与伊朗境内服务器的连接仍然工作。

但是这种方法不可扩展。伊朗科技公司不能购买伊朗数据中心的很多服务器，因为这给伊朗当局带来了太多危险信号。

“我们不能公开分享连接信息，因为连接信息包括服务器的 IP 地址，政府可以很容易地利用它来识别购买它的人，然后他们可以跟着我们，”他说。

相反，伊朗工程师一直在与 Tor 项目的成员接触，以帮助在伊朗境内架设桥梁。

为了实现这一目标，他和其他人一起编写了一份名为“ InternetForIran ”的文档，并发布到 GitHub 上。

这详细说明了位于伊朗数据中心内的机器如何被用来连接到载有伊朗境内抗议活动信息的网站和服务器，因为政府尚未阻止这些内部服务器的互联网接入，而且可能不会这样做，因为害怕切断自己对外部世界的接入。

该文件呼吁伊朗以外的技术行业，包括伊朗侨民，通过购买服务器来寻求帮助。

尚不清楚该组织的提议在实践中会有多成功或安全。

“我不知道这样做有多安全，也不知道如果他们被抓住会发生什么，”一位参与 Tor 项目的消息人士告诉我。如果被当局抓住，国内的活动人士可能面临报复。正在进行的互联网中断仍然是 Tor 论坛上的一个活跃话题。

Tor 项目的一名官方发言人没有回应记者的置评请求。

然而，EFF 的 Budington 认为，这种方法填补了一个空白，如果它能够避免当局取消访问的能力。他告诉 TechCrunch，活动人士似乎“想出了聪明的方法，先通过伊朗服务器，然后到伊朗以外的另一个服务器，再到 Tor 网络，从而将流量中继到 Tor 网络，而不会引起危险信号”。

这位科技企业家表示，绕过互联网关闭的方法正在积极使用。

“大多数联网的人都在使用这种方法或类似的方法(通过伊朗服务器转移流量)。我们大约有 200 人使用我们的服务器，但我们不能确定总共有多少人使用我们的方法，”他说。

与此同时，TechCrunch 采访的活动人士表示，由于缺乏信息，抗议活动“每晚都越来越小”，政府越来越有信心。德黑兰当局最近宣布，他们不会取消对 WhatsApp 和 Instagram 的限制，除非他们在伊朗注册公司，并且遵守伊朗法律。

活动人士表示，他们正在研究的重新获得访问权的方法可能对帮助反对独裁政权的抗议活动至关重要。“伊朗国内的人们看不到关于抗议的视频和信息。他们看到的都是政府宣传，”一个人告诉我。“我们可以让他们进入，但我们需要帮助。”
