Ravel 凭借隐私优先的数据工具脱颖而出

原文：https://web.archive.org/web/https://techcrunch.com/2022/08/23/ravel-emerges-from-stealth-with-privacy-first-data-tools-based-on-scalable-homomorphic-encryption/

世界对隐私和数据保护越来越重视，但在许多情况下，依赖这种或那种个性化的商业模式难以跟上。如今，巴黎一家名为 Ravel Technologies 的初创公司正在悄然崛起，它认为这种方法可能是这两者之间缺失的一环。它建立了一个基于同态加密的工具，以保持个人身份信息(PII)的端到端隐私，而无需接触数据本身。它首先推出了一个工具来实现“零知识”广告服务，另一个工具用于金融服务。

该公司已成立近四年，最初是靠创业起家的，雇佣了一个由学者和顾问组成的团队，其中包括菲尔兹奖获得者赛德里克·维拉尼。现在有消息称，空中客车风险投资公司已经开始了一轮金额不明的种子期投资。它没有透露任何客户名称，但该公司联合创始人之一的首席执行官 Mehdi Sabeg 表示，它正在与两家公司就这两种产品进行深入讨论。报告指出，法国巴黎银行(BNP Paribas)是正在进行概念验证的银行之一。

同态加密，正如其他人在之前描述的那样，是安全世界中的一个“圣杯”。这项技术最初是由学术界提出的，它涉及对一个组织的数据进行广泛的算法加密，即使该组织与第三方合作处理数据并基于数据提供自己的服务，也可以保持加密状态——例如，你可能会在广告网络中遇到这种情况。

圣杯方面的出现是因为尽管这个想法在理论上听起来很棒，但在实践中它需要大量的计算资源来运行，以至于到目前为止许多将同态加密付诸实践的努力都失败了。

这导致其他公司试图建立自己的方法，要么使用 he 的修改版本，要么将其应用于更小的、定义明确的数据集——我们已经介绍过的方法被像 Enveil 和 Duality 这样的公司使用，这两家其他基于 HE 的创业公司已经吸引了一些有趣的注意。

Ravel 的重大突破是一种新的方法，这种方法不仅允许它在所有其他方法中首次实现完全同态加密(FHE)，而且可以在任意大小的数据集上大规模实现。Sabeg 说，Ravel 处理数据的速度比其他人推出的其他基于 he 的解决方案快“四个数量级”。

Sabeg 补充说，Ravel 已经就其方法提交了专利申请。总的来说，它的效率足以驱动一个完全加密的 SQL 数据库——这是同类数据库中的第一个，他说——能够对大量加密数据进行加密查询。

当前的数据保护和隐私氛围创造了 Ravel 希望填补的真空。

如今，尤其是在某些司法管辖区，人们设置了数据来源和后续使用的网关，用户可以选择退出，并从根本上消除所有个性化，从而使许多围绕这一概念创造的广告技术和其他工具变得毫无用处。Sabeg 指出，对于采用其技术的公司——在零知识广告工具的情况下，它将使用 API 来运行服务，并在广告商和出版商端使用 SDK 来实现它；就金融服务工具而言，它将是一个金融平台，比如说，一个执行交易的第三方工具——虽然 GDPR·盖茨之类的东西仍将存在，但公司仍将能够运行其常规广告服务，因为它们使用的数据不再与 PII 相关。

同样，在金融交易所的应用程序中，Sabeg 表示，目的是确保保密性，并“消除明文数据中的市场偏见”，例如，这些数据可能会出现在投标中，这是区块链交易所的情况。

事实上，正是 GDPR 的出现，第一次让训练有素的物理学家萨贝格考虑如何将他的概念应用到在线广告的模型中，以及 DSP 是如何工作的。

“GDPR 即将实施，所有的广告客户都在抱怨它的限制，”他说。“我发现 GDPR 很有趣。本质上，我喜欢它所捍卫的价值观，但也能理解广告业所面临的问题。我想我们可以带来一个高效的技术解决方案。我们认为同态加密可以用作身份去认证技术。公司可以收集和处理数据，而不必使用 PII。”

我们已经报道了许多寻求应用同态加密来构建更多隐私优先数据服务的初创公司，但他们并不是唯一追求这一理念的公司，在某些情况下，这是因为中央广告和其他数据密集型服务对他们来说是多么重要。

脸书/梅塔去年进行了一场招聘狂欢，挑选了一些关键的同态加密研究专家，包括微软的长期员工克里斯汀·劳特，来领导其西海岸人工智能研究，并且它在 T2 发布了关于这个主题的研究。“这显示了他们对这项技术的重视，”萨贝格说。像谷歌这样的其他公司也在这个领域投入了一些研究，苹果也在它自己的一些隐私工具中应用它。

“鉴于拉威尔团队取得的令人印象深刻的重大算法突破，拉威尔全同态加密比最先进的 FHE 方案快几个数量级，”维拉尼在一份声明中指出。“随着全球范围内处理的个人和工业数据的不断增加，隐私和机密性保护至关重要。Ravel 的突破为关键的数据隐私和安全挑战带来了高效和可扩展的解决方案。”
