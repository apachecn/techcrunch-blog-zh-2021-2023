Cloudflare 非常重视基础架构服务

原文：https://web.archive.org/web/https://techcrunch.com/2022/05/11/with-new-serverless-database-cloudflare-gets-serious-about-infrastructure-services/

借助全新的无服务器数据库，Cloudflare 开始重视基础架构服务

当 AWS 在 2006 年推出存储产品时，它开始向云计算、存储和数据库服务进军。谷歌、微软、阿里巴巴和 IBM 等公司将紧随其后。在这个市场中，一个可能不是最受关注的玩家是 Cloudflare ，但是随着它今天宣布一项新的 SQL 数据库服务，它越来越认真地成为云基础设施玩家。

在发布新产品的博客文章中，该公司这样描述这个数据库:

D1 是建立在 SQLite 之上的。SQLite 不仅是世界上最普遍的数据库，每天有数十亿设备使用它，它还是第一个无服务器数据库。惊讶吗？SQLite 走在了时代的前面，在这个术语获得云服务的内涵之前，它自称为“无服务器”，最初的意思是“不涉及服务器”。

Cloudflare 帮助公司保持其网站以安全和快速的方式运行，作为其中的一部分，他们在世界各地都有数据中心。从 2017 年开始，当它推出 Cloudflare Workers 平台时，该公司开始在其庞大的网络上向开发人员提供计算。

去年，它发布了一款新的存储产品,他们称之为 R2，这是对亚马逊 S3 存储产品的一个小小的肯定。首席执行官兼联合创始人马修·普林斯在发布会上告诉我:

“我们认为，如果我们能够建立一个存储解决方案，提供其他存储解决方案所提供的所有功能，利用我们的全球网络，因此它具有极高的性能，然后我们还可以以对客户非常有吸引力的方式定价(我们应该这样做)，”他说。

当您将这三个部分放在一起时，您将获得一个一应俱全的计算、存储和数据库包，它为开发人员提供了一种在庞大的 Cloudflare 硬件网络上构建完整解决方案的方法。

根据 Synergy Research 的数据，云基础设施市场上个季度达到了 530 亿美元。亚马逊控制了其中的三分之一。微软以 22%的份额位居第二，谷歌云以大约 10%的份额位居第三。虽然三大巨头拥有 65%的市场，但仍有数十亿美元留给市场底层的公司。

Cloudflare 在世界各地拥有 200 多个数据中心，可能永远无法与三巨头抗衡，但它可以在其其他服务的基础上建立一个庞大的业务，这可能有利于该公司不断增长的收入。

Cloudflare 于 2010 年在 TechCrunch Disrupt 战地首次亮相。在于 2019 年上市之前，该公司筹集了超过 3.3 亿美元。在本月早些时候的最新收益报告中，该公司报告收入为 2.122 亿美元，比去年同期增长 54%。
