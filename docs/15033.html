微软新搜索的背后是 OpenAI 的“下一代”人工智能模型

微软正在用其改进的 Bing 搜索引擎和 Edge 网络浏览器进行人工智能的重大尝试，这两款浏览器似乎都是 OpenAI 流行的 ChatGPT 大型语言模型的继任者的独家访问。

新的人工智能目前还没有命名，只是在一篇博客中描述如下:

…新的下一代 OpenAI 大型语言模型，比 ChatGPT 更强大，专门为搜索定制。它吸取了 ChatGPT 和 GPT-3.5 的关键知识和进步，甚至更快、更准确、更强大。

“专门为搜索定制”在微软的活动中被多位高管反复提及，因此它必须是某种公认的语言，不会以某种方式过度使用该模型的功能。他们称之为“新的大型语言模型”，尽管很难说它比 OpenAI 以前的模型有多大进步。推测是 GPT-4，但是到目前为止这个术语还没有被使用。我已经向 OpenAI 询问了更多信息，如果有回音，我会及时更新。

像这样的大型模型的一个缺点是运行它们需要巨大的计算能力，这导致许多潜在的 ChatGPT 用户在开始会话之前等待几分钟。微软对扩展的关注是有意义的，特别是因为它肯定会为计算和服务器买单。

与 ChatGPT 和其他 GPT 模型不同，人工智能驱动的 Bing 可以通过正常的搜索界面直接访问，并包裹在微软创建的安全系统 Prometheus 中。毫无疑问，这是一个不祥的名字，因为尽管普罗米修斯是一个智慧和狡猾的神，但他也因无休止的折磨而闻名，他被锁在一块岩石上，一只大鸟永远在啄食他的内脏。希望必应不会落得同样的下场。

普罗米修斯是模型周围的安全和控制层的一部分，充当消毒剂和过滤器，监视明显不适当或不正确的结果。但是它也引入了相关数据，如位置、上下文和最新信息，以定制或改进核心模型的输入和输出。

下一代模型也被应用于必应的搜索排名指数，“这导致了 20 年来最大的相关性跃升。”承认这一点需要力量！
