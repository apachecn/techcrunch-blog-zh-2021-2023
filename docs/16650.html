Locus Robotics 首席执行官谈仓库自动化的未来

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/28/locus-robotics-ceo-on-the-future-of-warehouse-automation/

当你走在芝加哥麦考密克广场的地板上时，任何关于 ProMat 已经成为机器人赛事的怀疑都被搁置了。 Locus Robotics 在会议中心南大厅的正面和中央，以一种巨大而昂贵的方式插上了它的旗帜。在不到十年的时间里，Locus 已经成为该领域最大的公司之一，并徘徊在首次公开募股的边缘。

我们爬上该公司 ProMat 展位的二楼，与 2016 年担任首席执行官的 Rick Faulk 坐在一起。这位高管讨论了该公司作为第三方物流公司的诞生、发展以及自动化仓库的未来。

TechCrunch:你的解决方案很大程度上是一个棕色地带。

RF:棕色地带和绿色地带都有。我们现在大约 23%的业务实际上是在绿地建筑中。我们从布朗菲尔德开始，但是关于 Locus 的伟大之处在于我们可以两个都去。其他一些解决方案只能进入棕地——Fetch 可以做到，6 Rivers 可以做到，Locus 可以做到，真的就这么多了。进入从未被设计为自动化的建筑真的很难，通过对基础设施进行最小的改变来部署它也很难做到。

在一栋绿地建筑中，如果给你一辆车布兰奇，那会是什么样子？

我们现在有很多客户，我不能说出他们的名字，他们做绿地部署，这座建筑实际上是专门为 Locus 设计的。它的伟大之处在于它真的不需要任何伟大的设计工作——我们几乎可以在任何通道上操作。但是设计是为了最大效率。在空间的中间有一个理想的感应区，在外围有一个聚集区。在大多数这样的建筑中，我们也做回复。有一天，我在一个非常大的鞋类制造商的大楼里，他们实际上有卡车停在大楼里。机器人直接从卡车上拿箱子。

你做卡车卸货？

你会在[ProMat]上看到一些解决方案，可以自动进入并抓取盒子。一旦箱子从卡车上下来，它们就被放到机器人上，然后被送到建筑的正确区域。

给机器人添加有效载荷大多是手动完成的？

是的。如果你碰巧去[北馆]，你会看到发那科展台展示一只手臂。它将其中一个搬运箱放在机器人上，经过挑选后，手臂将它移走。一般来说，这是由人来完成的，但是我们正在试验将整个过程自动化。

如果很多人已经做得很好了，为什么还要尝试自己动手制作机器人手臂呢？

那是你最不想做的事。它本身就是一门学科。很多人都很擅长这个。我们最好的方法是与他们集成，而不是重新设计。

机器人是如何相互交流的？

我们有一个开放的 API 来交流。对于伯克希尔·格雷公司，我们的系统与他们对话，也与 WMS(仓库管理系统)对话。我们的系统首先与 WMS 对话，它会问，‘这个机器人填墙的理想数字和顺序是什么？’同样的事情也适用于我们所做的任何类型的整合。

所以你有可能拥有自动化整个过程的基础。

我们今天绝对会这么做。这不是梦。

熄灯了？

这不是熄灯。熄灯可能在 10 年后发生，但投资回报率却不是今天就能实现的。它可能在路的那头。我们的高级产品小组正在研究如何从等式中获得更多的劳动力。我们的策略是随着时间的推移尽量减少劳动力。我们正在与伯克希尔·格雷和其他公司进行整合，以最大限度地减少劳动力。要到达一座黑暗的建筑还需要几年的时间。

你是否探索过前台——零售店或餐厅？

我们接到很多关于餐馆的电话。我们的策略是专注。有 135，000 个仓库必须实现自动化。如今只有不到 5%是自动化的。最近在日本，饭是机器人填的。我环顾四周，说，'嘿，我们可以这样做。'但这是一个不同的市场。

什么是安全协议？如果一个机器人和我在地板上相向而行，它会先停下来吗？

它会停下来或者他们会绕过去。这是难以置信的聪明。如果你看到后端发生了什么，它会实时动态规划路径。每个机器人都在和其他机器人交谈。这个机器人会告诉这边的机器人，'你不能从这里通过，所以绕过去。'如果发生事故，我们就绕过去。

他们都在一起实时创建一个基于云的大型地图。

正是如此。

这家公司是什么时候成立的？

2014.我们实际上是从一家叫做安静物流的公司分离出来的。是 3PL。我们在 Kiva 上实现了全自动化。亚马逊在 2012 年收购了 Kiva，并表示，我们将把该产品撤出市场。我们又找了一个机器人，找不到，所以决定造一个。

外形相似。

它们的外形基本上是底部。它钻到一个架子下面，然后把架子带回工作站进行挑选。我们的解决方案最棒的一点是我们可以进入一个棕色地带的建筑。它们很棒，也能工作，但要做我们机器人做的同样的工作，还需要四倍数量的机器人。

在我与该领域的对话中，亚马逊一直是仓库采用技术保持竞争力的动力。但是这里有更深层次的联系。

亚马逊实际上是我们最好的营销组织。他们为 SLA(服务水平协议)设立了标杆。这里的每一家第三方物流公司都必须做到当天或次日送达，因为这是他们客户的要求。

系统的风格需要亲自部署吗？

COVID 期间有趣的事情是，我们实际上在 FaceTime 上部署了一个站点。

有人拿着电话在仓库里走来走去？

是啊。这不是我们的首选方法。他们可能比我们做得更好。太棒了。

就效率而言，这可能很有意义，继续前进。

是啊。它仍然需要人进去，进行安装和培训——诸如此类的事情。我认为我们要摆脱这种情况还需要一段时间。但这并不难做到。我们把人们从街上带走，训练他们，一个月后他们就知道如何部署。

它们是在哪里制造的？

信不信由你，我们在波士顿生产。我们有合同制造商生产一些部件，像底座和桅杆。然后我们在波士顿把它们整合在一起。我们进行最后的组装，然后进行所有的运输。

随着你们在全球范围内扩大销售，是否有计划开设更多的制造工厂？

我们最终会的。现在我们正在阿姆斯特丹进行一些组装。我们在阿姆斯特丹为欧洲做所有的翻新工作。[……]还有一个大的可持续发展故事。可持续发展对像 DHL 这样的大客户来说非常重要。我们的产品本质上是绿色的。我们有超过 12，000 个机器人在现场。你可以用两只手来数我们报废的机器人数量。所有的东西都被回收到地里。一个机器人会在三四年后回来，我们会重新包装它。我们可能需要更换相机、灯或其他东西。然后在 RaaS 模式下重新投入使用。

在不得不报废的情况下发生了什么。

他们被铲车撞了，无法修复。我是说粉碎。

有任何额外的融资计划吗？

我们已经筹集了大约 4.3 亿美元，通过了我们的 f 轮融资。我们融资的下一步将是首次公开募股。大概吧。我们现在有足够的人来做这件事。出于你所知道的所有原因，市场条件不适合这样做。

你有一个大致的时间表吗？

明年会是，但市场必须复苏。这不是我们能控制的。
