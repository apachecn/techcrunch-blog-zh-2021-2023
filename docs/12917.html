Stripe 宣布推出菲亚特转加密 on ramp widget TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/12/02/stripe-announces-fiat-to-crypto-onramp-widget/

支付巨头 Stripe 将推出一款新产品，该产品可以让用户更容易持有加密货币，而无需注册加密货币交易所。该公司新推出的菲亚特-to-crypto widget 可以嵌入到任何加密产品中，这样用户可以输入他们的卡信息，并获得可以在另一个 web3 产品中使用的加密。

在一些用例中，像这样的小部件是有意义的。最常见的大概就是非保管钱包了。一旦你的钱包里有了加密资产，你就可以使用 WalletConnect 与各种 web3 产品进行交互。可以买 NFT，用 DeFi 的产品等等。

但是，如何将加密资产上传到这些软件钱包中呢？最合乎逻辑的用例是，你首先注册一个像比特币基地或北海巨妖这样的中央加密交易所。一旦你验证了你的身份，你就可以购买加密货币，然后将这些资产转移到你的非托管钱包中。

那不是无缝的体验。这就是为什么像 Argent 或 ZenGo 这样的非托管钱包开发者一直在使用菲亚特到加密的小工具。Stripe 并不是第一家推出这款产品的公司。像 MoonPay 和 Ramp 这样的加密公司已经可以在许多加密钱包中使用。

在这种情况下，MoonPay 和 Ramp 负责 KYC 的要求、欺诈和合规。它们支持数十种法定货币和来自 150 多个国家的客户。本质上，当你想购买加密资产并为你的钱包充值时，你可以用 MoonPay 或 Ramp 创建一个账户——不需要从交易所到钱包的转账。

这是一个更流畅的体验，因为你不必在多个产品之间切换。当然，便利并不便宜。MoonPay 对信用卡支付收取 4.5%的费用，而 Ramp 收取 2.9%的费用。他们还提供银行转账，费用较低。

Stripe 的平面到加密 onramp 的工作原理或多或少类似。目前，它仅面向受邀测试该产品的美国客户。

当用户想要购买 crypto 时，Stripe 的小工具会询问你想花多少钱购买法定货币——该公司会在此基础上加收费用。然后，你可以选择你想要的加密货币，看看一旦交易被确认，你会收到多少。

图像信用:条纹

在 Stripe 的示例截图中，该公司以 4.99 美元的价格购买了价值 100 美元的 USDC。但费用可能因金额和支付方式而异。由于它是一个条纹产品，它应该从第一天起就与不同的支付方式配合使用，如卡支付、银行转账、Apple Pay、Google Pay 等。在幕后，Stripe 与零哈希合作获取加密货币。

Stripe 预计，它的小工具将吸引在 DEX、NFT 平台、钱包和 dApps 上工作的开发者。例如，该公司表示，它一直在与 Audius (一个位于区块链的音乐流媒体平台) Magic Eden (一个 NFT 市场)和 Argent (一个我已经介绍过的以太坊钱包)一起测试该产品。

支付公司处理 KYC、支付、欺诈和合规，因此加密公司可以专注于加密，而不是与 Stripe 竞争。Stripe 的菲亚特-to-crypto on lamp 产品也有一些附带好处。

例如，它与链接集成。如果您在过去进行了 Stripe-powered 购买并保存了您的支付信息，Stripe 可以帮助您检索这些信息并更快地支付。

在 FTX 的失败之后，像 Ramp、MoonPay 和 Stripe 的 fiat-to-crypto widget 这样的产品将变得必不可少。他们可以更容易地避免集中交易。很高兴看到有多家公司在研究这个问题。
