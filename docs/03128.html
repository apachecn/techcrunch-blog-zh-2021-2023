位置智能的新时代为企业带来了什么？

在成为 Foursquare 的首席执行官之前，Gary 是 Raine 的总经理，领导技术业务，重点是消费者互联网、企业软件和新兴技术方面的咨询任务和主要投资。

在当前的环境中，企业现在面临的任务是平衡复苏的推动力和发展所需的敏捷性，以保持在重新出现的新冠肺炎障碍之上。

位置数据对此类战略绝对至关重要，它不仅能帮助领先企业缓解挑战，还能开启前所未见的机遇。在整个新冠肺炎复苏时代，位置数据将成为推动商业智能和建立可持续消费者忠诚度的核心要素。

基于云的位置服务的进步通过帮助数据工程师、分析师和开发人员将位置数据集成到他们现有的基础设施中，构建数据管道，并更有效地获得洞察力，从而开创了位置智能的新时代。

可扩展且数据丰富的位置服务正帮助面向消费者的企业沿着三个战略前沿推动转型和增长:

创造更丰富的消费者体验

更好的应用内体验可以提高消费者参与度和忠诚度。许多世界上最大的科技公司已经通过 AWS 数据交换(ADX)平台访问兴趣点(POI)数据，以支持核心的搜索、发现和地图构建功能，使他们的应用程序更加有用和有趣。

例如，Nextdoor 帮助用户探索并与他们社区中的其他人互动，利用 POI 数据来提高业务数据的覆盖范围和质量，以及发现、验证和入职体验。

各行各业的品牌都在将云原生位置数据与其他下游云服务结合使用。例如，将独立的数据定位平台与知识图应用程序集成在一起，有助于品牌为其用户揭示流行的夜生活和休闲趋势，从而推动强大的应用内搜索和发现体验。
