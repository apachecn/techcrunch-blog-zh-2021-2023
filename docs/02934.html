肯尼亚保险科技初创公司米拉在收购 Bluewave 后进入马拉维、刚果民主共和国，关注非洲其他地区

原文：https://web.archive.org/web/https://techcrunch.com/2022/02/02/kenyan-insurtech-startup-lami-enters-malawi-drc-after-acquiring-bluewave-eyes-rest-of-africa/

肯尼亚保险科技初创公司米拉科技以未披露的金额收购了 Bluewave 保险机构，该公司寻求让更多非洲人获得保险。

同样来自肯尼亚的初创公司 Bluewave insurance 是由阿德莱德·奥德希安波创立的，他现在加入米拉，担任商业合作伙伴主管。

自 2016 年推出以来，Bluewave 已经建立了一个在线平台，允许客户通过多种渠道访问小额保险产品，包括 USSD、短信、WhatsApp 聊天机器人和网络应用程序。在新的安排中，米拉将把 Bluewave 的技术整合到其核心平台中，进一步增强其 B2B2C 能力。

这笔交易也为米拉在马拉维和刚果民主共和国打开了新的市场，Bluewave 已经在这两个国家开展业务。例如，通过进入马拉维，米拉将利用成千上万的小农户，包括已经在 Bluewave 数据库中的 6 万人。

米拉科技公司创始人兼首席执行官韩吉·阿巴斯在一次关于收购的采访中告诉 TechCrunch，“真正让我们对这个机会感到兴奋的事情之一是，我们正在努力实现一个非常共同的愿景，即实现保险的民主化。我们的愿景非常一致。”

“Bluewave 团队创建了一些模块(如 USSD 的小额保险产品)，我们认为这些模块可以特别增强我们的 B2B2C 产品。他们围绕小额保险产品的理赔模块做了大量工作。所有这些不同的东西都可以集成到核心的米拉平台中。我们还可以继续在它的基础上构建，并使用我们的部分技术来增强产品，如我们的高级定价模块，以使体验更好，”阿瓦斯说。

今年，米拉正着眼于北非和西非的更多市场。图片来源:米拉科技

此次收购正值米拉寻求进入尼日尼亚和一些北非国家。在肯尼亚之外，米拉通过合作伙伴关系进入了乌干达和坦桑尼亚等其他市场。

“我们今年正在寻找更多的市场。这是我们的核心关注点，也是这次收购令人激动的原因；这让我们在继续扩张时占据了上风，”她说。

像米拉这样的数字服务的创新和扩张，通过允许小额支付、灵活的签约、快速的索赔周转时间以及通过手机获得广泛的服务，推动了保险的发展。在非洲，保险渗透率不足 3%；如果你剔除南非，覆盖率会下降到 1.2%。正是这种差距促使 Abass 建造了米拉，并于 2018 年在肯尼亚推出。

米拉与其承保合作伙伴共同设计产品，向客户提供独特的保险产品，并通过其技术基础设施，以 B2B2C 方式促进保险产品的分销。

这家初创公司的 API 允许银行等实体向其客户提供数字保险产品——方便地触及大众。多年来，米拉已经将其 API 与 12 个不同领域的实体进行了整合，包括物流、电子商务、银行和金融科技，向用户提供不同的产品。

米拉现有的合作伙伴包括数字物流初创公司熊智娟和信用合作社新银行 Kwara。

与熊智娟的合作使东非(肯尼亚、乌干达和坦尚尼亚)的货运公司能够获得每次行程的运输保险，而与 Kwara 的合作使超过 60，000 名 SACCO 成员能够获得保险产品。

随着技术不断激发新产品和流程，Abbas 预测保险行业将面临面向更个性化产品的持续干扰。

“你将会看到更多考虑到相关风险的数据驱动定价，”她说。

“我认为这将真正有助于使产品更加实惠，因为将会有更多的定制产品。我认为我们还将看到人工智能的更多应用，特别是在索赔处理和定价方面。”
