在其他更新中，Windows 11 今天获得了一个人工智能支持的眼神交流功能，它将始终让你看起来像是在看屏幕。不要用。

没有什么比在视频会议期间盯着主持人没有灵魂的人工智能增强的眼睛更让你质疑你现实的本质了。演讲者的眼睛坚定不移地聚焦在屏幕上，只有偶尔的小故障表明另一边仍然有一个活生生的人，可能正在逐字阅读预先写好的演示文稿。

微软的 Windows 主管 Panos Panay 说，它使会议体验“更人性化”，但它看起来很假。你最好用一个卡通形象来代替。说清楚一点，没必要这样。你本来就很棒——不管有没有镜头。

(是的，苹果公司在 FaceTime 上做了这个，而且已经做了几年了，但是要微妙得多。)

在启用此功能之前，有一些硬件要求。我们昨天问微软公关这些是什么。他们仍在努力寻找答案，所以我们会在听到更多消息后更新这篇文章。
