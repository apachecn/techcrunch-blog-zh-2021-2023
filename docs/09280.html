Polygon 的投资主管仍然“高度看好 web3”

加密市场可能处于深度熊市和复苏之间，但这并没有阻止投资者向该领域投入资金。

Polygon 投资主管 Shreyansh Singh 对 TechCrunch 表示:“从某种意义上来说，熊市对该行业是有利的。”“它们允许具有明确长期愿景和可持续模式的平台继续构建新的解决方案，并帮助淘汰可能过于依赖炒作的项目。”

Polygon 是以太坊上更著名的第二层区块链建筑之一，大约有 37，000 个分散应用 (dApps)在其链上启动。它是一些最大的 web3 项目的所在地，如 Uniswap V3、OpenSea 和 Aave。

4 月，Polygon 启动了一个1 亿美元的生态系统基金，帮助 web3 开发者建立其网络。一个月后，Polygon 启动了一个“无上限”基金，帮助建立在现已不复存在的 Terra 生态系统上的项目迁移到其区块链。可以肯定地说，区块链及其背后的团队一直很忙。

超过 50 个项目已经开始从 Terra 迁移到 Polygon 的生态系统，包括 NFT 市场 OnePlanet 和 NFT 游戏 Derby Stars。
