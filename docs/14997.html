Elon Musk 称 Twitter 将向提供“好”内容的机器人提供免费的只写 API

原文：https://web.archive.org/web/https://techcrunch.com/2023/02/04/elon-musk-says-twitter-will-provide-a-free-write-only-api-to-bots-providing-good-content/

埃隆·马斯克表示，Twitter 将向提供“好”内容的机器人提供免费的只写 API

上周，Twitter 表示将从 2 月 9 日开始关闭对其 API 的免费访问。现在，在截止日期前几天，埃隆·马斯克表示，在获得开发者的反馈后，Twitter 将为“提供免费优质内容的机器人”提供只写 API。

这个决定和马斯克管理下的其他一些政策决定一样不透明。没有关于什么是“好内容”以及由谁来决定的信息。然而，如果 Twitter 最终实施了这一规则，一些机器人将在社交网络上获得新的生命线。

此前，Twitter 关闭了对第三方客户端的 API 访问，称他们打破了没有任何规范的“长期规则”。然后公司悄悄地更新了它的开发者条款，以反映应用程序不能“使用或访问许可材料来创建或试图创建 Twitter 应用程序的替代品或类似的服务或产品。”

声明发布后，许多取笑机器人的开发者批评了这一决定，称他们的自动化为人们提供了免费内容，从而增强了服务。上周，BuzzFeed 采访了几位对该决定不满的机器人开发者。其中包括 @_restaurant_bot ，它在推特上发布餐馆的随机照片，以及 @_weather_bot_ ，它在推特上发布不同地方的天气更新图片。

目前，还不清楚像 @BigTechAlert 这样的账户是否有资格获得这个免费层，因为他们可能需要扫描账户信息，这些账户在推特上发布关于大型科技高管和组织相互关注和不关注的信息。

开发人员大流士·卡泽米制作了 80 多个 T2 机器人，甚至在 2016 年组织了一次机器人开发者峰会，他在电话中告诉 TechCrunch，这些自动化账户多年来一直是 Twitter 不可或缺的一部分。他说，这些拥有成千上万追随者的机器人每天都给许多人带来欢乐。

他提到，维护这些向平台提供免费内容的机器人成本很高。

“我在推特上有超过 80 个机器人所以我每年要花几千美元来维护它们，我付不起那么多钱，”他说。

马斯克一直试图通过新的昂贵订阅计划和增加广告费等举措为 Twitter 创造更多收入。他还计划在回复时展示广告，与创作者分享收入。虽然这将如何运作的细节很少，但 Twitter 首席执行官表示，只有蓝色订户才能赚到这笔钱。因此，即使广告显示在内容机器人的账户上或推文下方的回复中，内容机器人也很可能赚不到钱。

Twitter 的免费 API 停产不仅仅影响 bot 开发者。有大量的学生开发人员和仇恨言论或错误信息研究人员可能没有预算支付月费。Twitter 的 v2 API 为学者提供了特殊的访问权限，但根据新的 API 规则，情况可能并非如此。

开发者还指出，许多传播垃圾邮件的机器人实际上并没有使用官方 API。因此，该公司关闭免费 API 以清除垃圾邮件的意图可能不会奏效。
