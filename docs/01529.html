7 投资者讨论 web3 的现在并展望其未来

原文：https://web.archive.org/web/https://techcrunch.com/2021/12/14/7-investors-discuss-web3s-present-and-peer-into-its-future/

大多数人只通过 Web 2.0 体验过互联网:在线应用、社交网络和软件即服务构成了我们的生活。

但是，即使其他技术在迅速发展，基本的网络基础设施在世纪之交以来基本上没有变化。最近，许多人支持 web3 作为互联网的下一个阶段，但这个术语包含了如此多的领域，对话可能相当分散。

我们距离 web3 获得主要市场份额还有几年的时间，有理由担心它的复杂性会让消费者和监管者感到气馁。然而，我们的研究表明，随着风险资本家受教育程度的提高和怀疑程度的降低，投资领域的竞争越来越激烈。

为了更清楚地了解市场现状，我们联系了几位活跃的投资者，以了解 web3 的现状和未来:

  • 兰迪·格林，创始人/合伙人萨姆·夏皮罗，负责人 DFJ 成长

为了让事情尽可能清楚，我们要求每位受访者分享他们的电梯推销:如果他们试图说服一位怀疑论者投资，他们会如何描述这项技术？

从卡通类人猿的潜在消费者吸引力开始，我们试图找出是什么吸引他们投资语义网，以及他们目前在哪里看到了需求。“我最初是通过企业市场上可验证的凭证和数据来源进入 web3 的，”马蹄资本的合伙人阿图尔·阿霍伊(Atul Ajoy)说。其他几个人说，他们在对加密产生兴趣后开始探索这个领域。

除了讨论广告、金融科技和企业应用的潜在应用，受访者还分享了他们对正在寻找资金的 web3 创始人的建议，以及他们对可能阻碍其发展的因素的担忧。最后，我们问每位受访者:持怀疑态度的 web3 投资者遗漏了什么？

“我从未见过一个持怀疑态度的投资者真正明白发生了什么。如果你得到了它，你可能已经系好安全带，准备出发了。

“在这一点上，web3 已经证明了自己不仅仅是一个现象，它是元宇宙的基础层。”

Lior Messika，Eden Block 创始人兼管理合伙人

请告诉我们您的 web3 电梯间推介:它是什么，它在当今的互联网框架中扮演什么角色？

用两句话来说: Web3 是真实元宇宙的关键。在不远的将来，Web3 将容纳我们的金融机构、社会互动、个人身份以及更多更多的东西。

扩张:当我看着 web3 时，我看到了一个能够驱动新世界的分散基础设施的混合体。web3 的两个最明显的特性是可组合性和去中心化。首先，加密网络正在吞噬价值，并创造一种环境，在这种环境中，每一种可能的价值链形式都相互联系、相互作用。Web3 拥有独特的能力，可以从几千年来无形的价值体系中创造出有形的价值体系。它开启了一个世界，在这个世界中，我们的社会资本、声誉和历史互动被赋予了不可磨灭的真实资产——这些资产通常会被定价到由去中心化协议驱动的市场中。

在 Web3 中，所有分散的系统将学会彼此交互，建立在更广泛的生态系统的安全性和价值(因此是网络效应)之上。它将是多层次、多方面的，由全新的基础设施支持。我们的应用程序将需要可靠地中继数据，完全秘密地传输数据，存储数据并保持高水平的可用性——所有这些都在分散的轨道上。在 web3 短暂的历史中，我们已经看到了集中式基础设施如何成为一种危险的权衡。

是什么让你决定进入太空？卡通猩猩和区块链游戏还没有广泛的消费者吸引力:你认为今天对 web3 产品的需求在哪里？

Web3 象征着巨大的社会转变，充满了创新和价值观。当我意识到我们正处于一场由技术推动的文化革命中时，我一头扎进了这个领域——而不是反过来。Crypto and web3 是我们对困扰我们数千年的一个社会问题的技术解决方案，所以我认为我们很自然地以尊重和热情对待这场运动。

关于产品:最近我们看到的一些最强大的 web3 产品非常关注所有权、创造和其他最终构成我们数字身份的上下文特征。有趣的是，我们已经开始看到加密产品是如何抓住我们的文化和我们共同的注意力的。在元宇宙，你的“个人隐私”不仅仅是某种形式的皮肤，而是通往另一个维度的钥匙。随着时间的推移，这一基本特征所带来的个人主义解释了疯狂的 NFT 市场，它无疑仍将是 web3 和元宇宙的一大部分。NFTs 的技术基础将开启跨越游戏、时尚、社交和创意经济的颠覆性价值体系。

另一个不太明显的问题是，大型分散团体(Dao)的协调需要精简的框架和透明的价值体系。随着社区的增长和规模扩大到数十亿美元的网络，分散治理变得非常困难。Colony 和 Coordinape 将通过专门的工具和自动化的治理和激励框架来实现下一代 Dao。

在不久的将来，我们可以期待哪种应用进入市场？你在哪里下注，你能说出几个值得关注的公司吗？

在 web3 的大背景下，我们看到堆栈中两个不同层的巨大增长— 基础设施和应用。与互联网出现之初的 TCP/IP(互联网基础设施的核心部分，不能货币化)不同，分散式协议和内置的激励系统使得运行分散式基础设施变得极其有价值。像袖珍网络这样的技术充分体现了这一点，每周有数十亿次转播。口袋网络为整个生态系统提供了分散的基础设施，并为其节点运营商的工作提供了丰厚的报酬。在目前的中继状态下，网络的预测年收益是数亿。

web3 的另一个核心基础设施是双经济。Biconomy 团队正在为分散式网络构建交易基础设施。它的平台为开发者和用户抽象出加密交易的复杂性。Biconomy 已经开发了一系列产品，以满足开发人员在 web3 中构建应用程序的需求——随着协议继续发展成为链上服务的大规模生产者，开发人员需要工具来简化他们的操作和集成，以实现大规模生产。

在应用层，我相信我们仅仅开始触及更广阔的 web3 空间内的可互操作价值系统的表面。通过分散融资，我们有一天会对每一种与我们有关联的资本形式进行定价、估值和货币化。我们最喜爱的艺术作品将偿还我们的贷款协议抵押，我们的“喜欢”将成为金融资产。简而言之，分散金融将启动一切事物的巨大货币化。像 Vega 协议这样的协议旨在通过公平有效的市场实现大规模的分散衍生品。

DeFi 的另一个万亿美元的机会与房间里的一只巨大的大象有关——由于市场需要稳定来保持竞争力和效率，DeFi 在很大程度上牺牲了分散化来实现稳定，选择美元作为大部分市场的主要基础抵押品。像浮动这样的协议提供了一种选择，通过创造一种低波动性的资产，可以保持完全分散。如果 DeFi 没有真正分散的抵押品，web3 将面临风险。

你能分享几个你最关心的问题吗？有哪些陷阱会阻碍 web3 实现其潜力？

首先，必须明白去中心化必须成为我们所说的 web3 中所有真正创新的基础。它是当今密码领域每一个真正颠覆性应用之间的共同主线: Uniswap 及其许多克隆，像 Compound 这样的货币市场，Cryptopunks 和 Bored Apes，以及像 Filecoin 和 IPFS 这样的存储解决方案——所有这些都由一个底层属性实现。加密网络的未来发展将继续利用去中心化作为核心驱动力，并围绕它建立额外的效用。

如果没有权力下放，破坏就会被扼杀。出于这个原因，我认为，当我们更深入地探索 web3 时，最大的风险是回到过去。进行简单的权衡会让我们偏离 web3 的核心前提——最大程度的去中心化，通常是以牺牲性能或可扩展性为代价。

鼓励主要网站转向 web3 的使用案例和盈利机会有哪些？互联网监管加强的前景会是一个因素吗？

互联网监管是一个极端，但更有可能的情况是，随着它的继续增长，web3 将吞噬足够的价值，以至于主要网站认为有必要转向 web3。Web3 类似于一个经济体，大多数玩家都想参与其中。

在未来十年，你预计会有哪些重大飞跃？web3 的哪些方面被夸大了？

作为一个社会，我们将实现的一个重大飞跃是接受真正的所有权。财务自由深深植根于财务责任中，而 web3 是一个尽可能直接的责任(和自由)的界面。

关于 web3 被夸大的方面:我认为 web3 的彻底开放和无许可访问为大量的实验和推测打开了大门。出于这个原因，web3 的大部分被夸大的领域实际上会在这个空间中最有价值的市场的底部被发现。例如，除了排名前 15-20 的 NFT 项目及其相关社区之外，看任何东西都是令人痛苦的。从长期来看，市场中被夸大的部分将继续作为一个信号，表明目前大部分创新和更广泛的关注点在哪里。

今天的 web3 投资市场竞争有多激烈？需要做些什么来培养兴趣？

T2 的网络投资市场从未像现在这样竞争激烈。投资者聚集在有想法和银行账户的开发者团队后面。我相信，尽管资本继续如此自由地流入这个领域，但基本面保持不变，而赌注(无论是风险还是收益)从未如此之高。强大的分散化用例中的潜在价值几乎是不可想象的，团队正在世界范围内寻找人才。为了培养更多的兴趣，资金最终必须与人才和人力资本相匹配，而这在目前看来极其难以找到。

你会给那些试图拿到第一张支票的 web3 创始人什么建议？

Web3 需要新的领导者，随着这个生态系统的发展，他们将被有机地指定。如今，web3 的创始人承担了传播者和社区建设者的角色。在一个开发是开源的，贡献在很大程度上是不允许的(在以太坊上构建的应用程序中更是如此)的世界里，团结整个生态系统支持你的事业成为了一个极其独特的挑战。

web3 的核心驱动力是基于价值观和财务驱动的——因此，对 web3 的迭代将是一项文化和资本主义使命。它将受到数字、算法和代码的激励，也将受到品牌、社区和人的推动。Web3 的创始人应该将这一点内在化，并利用开发者和社区之间的亲密关系来为自己谋利。从一开始就清楚地传达项目的价值观和核心使命的创始人，将使社区围绕共同的目标形成，并推动项目的存在和大规模分发。

没有市场情报，他们在投球时是否处于劣势？

没有一定的市场情报，创始人会处于严重的不利地位。Web3 的迭代如此之快，如此之快，以至于几个月的停机时间都会造成基本的知识缺口。在我们的投资实践中，我们看到了网络 3 出身的创始人和那些还没有冒险尝试的创始人之间的巨大差异。到目前为止，很明显，要在 web3 中构建一个成功的产品，团队需要一定程度的独创性，以正确识别市场中的差距、用户群的期望和需求等等。

你认识任何持怀疑态度的投资者吗？你觉得他们遗漏了什么？

我从未见过一个持怀疑态度的投资者真正明白发生了什么。如果你明白了，你可能已经系好安全带，准备出发了。在这一点上，web3 已经证明了它不仅仅是一个现象，它是元宇宙的基础层。持怀疑态度的投资者可能没有理解梅塔最近这部戏的真正含义，即扎克与加里·维(Gary Vee)关于非功能性数字技术和数字生活大方向的有趣对话。

Atul Ajoy，马蹄资本合伙人

请给我们你的 web3 电梯间推介:它是什么，它在今天的互联网框架中扮演什么角色？

Web3 是互联网的一种新的去中心化形式，用户成为所有者。今天的互联网，Web 2.0，在很大程度上被集中的玩家所控制，他们获取了互联网上创造的大部分货币价值。Web3 用分散的网络取代了这些集中的参与者，将价值分配给开发者、创造者和用户。这允许这些网络参与者为像 DAOs 这样的组织设计新的链上激励机制。
