Silverfort 利用“整体”方法保护跨 IT 孤岛和遗留系统的 ID 管理，赢得 6500 万美元| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/04/12/silverfort-nabs-65m-with-a-holistic-approach-to-protecting-id-management-across-it-silos-and-legacy-systems/

身份管理是许多组织旨在保持其系统安全的核心，但这留下了一个未回答的大问题:身份管理系统总体上有多安全，以及没有覆盖的所有缺口是什么？今天，一家名为 Silverfort 的初创公司宣布投资 6500 万美元进行扩张，该公司已经建立了一个它认为可以覆盖这些基地甚至更多的平台。

该公司的平台不仅可以与现有的身份管理软件提供商合作，还可以为他们尚未覆盖的系统和硬件提供一个安全层。 Silverfort 的联合创始人兼首席执行官 赫德·科维兹在接受采访时说，后一个群体比一些人可能意识到的更加棘手，也是一个更大的目标，他引用了遗留系统和机器对机器网络这两个当今最容易受到黑客攻击的领域。

“市场正在意识到身份是安全领域的下一件大事，但它是如此分散，”Kovetz 在接受采访时说。

总部位于以色列的 Silverfort 的 C 轮融资使融资总额超过 1 亿美元，估值未披露。PitchBook 给出的最新估计是 1.5 亿美元，但那是在 2020 年 8 月，自那以来公司一直在增长——年同比增长 2.5 倍，美国增长 3.5 倍。“数百人来找我们，”科维兹说。他说，今天的客户包括新加坡电信、新秀丽和 UPS 等“较小的”公司。更大的客户没有被披露，总的来说，在 200 多家客户中，约有一半位于美国。

由 Greenfield Partners 牵头，这轮融资还包括新的支持者 GM Ventures、Acrew Capital 和 Vintage Investment Partners 以及之前的投资者 StageOne Ventures、Singtel Innov8、花旗创投、Aspect Ventures 和 Maor Investments。这是一个金融和战略投资者的混合体，尽管 Kovetz 不愿透露哪些人在使用其服务以及如何使用。

然而，从这份名单中抽出来的一个有趣的人是通用汽车风险投资公司，它可能会与这家初创公司合作，以保护自己的系统，这些系统跨越内部和云系统，但也包括混合了下一代应用程序的遗留软件；以及它可能为自己的客户开发的服务，这些客户将越来越多地将更多基于身份的服务加载到联网汽车环境中。简而言之，这方面有很大的潜力，通用汽车在描述如何使用这项技术时似乎也同样含糊其辞:

“随着传统网络参数的快速变化和新威胁的出现，整个企业对强大威胁防护的需求比以往任何时候都更加普遍，”通用汽车风险投资公司(GM Ventures)总经理韦德·谢弗(Wade Sheffer)在一份声明中表示。“我们对 Silverfort 的投资强调了通用汽车风险投资公司致力于确定下一代技术，这些技术将增强企业的数字企业，同时支持通用汽车向技术领导者和平台创新者的转型。我们对 Silverfort 的增长持乐观态度，并相信他们的技术有潜力阻止通用汽车及其他公司未来基于身份的威胁。”

虽然您可能会认为身份管理系统本身就是为了帮助提高安全性而设置的系统，而且在任何情况下，它们中的许多都可能会与其他类型的服务一起销售或与其他类型的服务一起工作，以构建更大的安全堆栈，但 Kovetz 和 Silverfort 的观点是，围绕身份层将身份与安全性分离有很多好处。

Kovetz 说，部分原因是因为每个身份产品，就像许多应用程序本身一样，都处于孤岛中——这是在人们考虑到企业网络的不同部分可能没有被基于身份的身份认证所覆盖之前。综上所述，这为恶意黑客寻找小漏洞和入口创造了绝佳的机会。(在这方面，这就像有人在拥挤的地方找人扒窃一样——那个人不会考虑那些受到保护的人，而是寻找那些警惕性较低且可能不会受到保护的人。)

Silverfort 的方法是独特的(Kovetz 指出，它获得了专利)，尽管听起来有点古怪。正如他向我描述的那样，客户不一定被要求放弃他们当前的任何 ID 管理产品。Silverfort 和所有这些一起使用，不管有多少个。同时，它可以对公司的整个机器环境进行快照，并提供对所有这些 ID 服务和未被任何服务覆盖的机器的观察，从而提供一种一体化的方法来覆盖 it，同时提供威胁检测和响应(ITDR)以及身份威胁防御(ITP)功能。

正如 Kovetz 所描述的，Silverfort“坐在所有其他平台的后面”，而不是建立另一个 ID 平台。其他平台将它们的认证转发给 Silverfort，Silverfort 在认证用户时成为“第二意见”。他补充说，遗留点的整合更加复杂，也是一些专利技术发挥作用的部分原因。

Greenfield Partners 的合伙人埃弗里·施瓦茨(Avery Schwartz)在一份声明中说:“该公司花了数年时间建立了一个一流的平台，以全面解决日益增长的安全问题，现在已经获得了主流意识。”“公司技术带来的积极影响以及领导层的远见和热情都让我们感到振奋。我们很清楚，Silverfort 在市场上的强劲势头只是一个开始，我们很高兴能加入他们的行列。”
