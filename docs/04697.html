Tinder 推出应用内背景调查，由非营利组织 Garbo | TechCrunch 提供支持

原文：https://web.archive.org/web/https://techcrunch.com/2022/03/09/tinder-rolls-out-in-app-access-to-background-checks-powered-by-nonprofit-garbo/

继去年对背景调查非营利组织嘉宝的投资之后，Match Group 今天向其旗舰约会应用 Tinder 推出嘉宝背景调查。Tinder 用户将可以通过应用内安全中心获得这项服务，通过蓝盾图标可以在应用的任何地方访问该中心。

应用内体验将引导用户进入嘉宝网站，在那里他们可以填写关于他们比赛的基本信息。该公司表示，用户通常只需要匹配的名字和电话号码就可以开始。然而，如果嘉宝不能找到一个唯一的匹配，它不会返回任何结果，而是会提示更多的信息，如匹配的年龄。

当嘉宝公布其结果时，Tinder 用户可以选择他们下一步想做什么。例如，如果这个人的结果表明他们有暴力史，用户将被鼓励向 Tinder 报告匹配情况。该公司告诉 TechCrunch，根据其现有政策,它将删除所有 Match 约会应用品牌中被举报有暴力犯罪的账户。

图片来源:火绒/嘉宝

 

嘉宝网站还将引导用户使用各种心理健康资源，包括与热线直接聊天的选项(全国家庭暴力热线)。这条热线为幸存者提供了实时聊天体验和资源，可以引导人们获取更多信息，提供推荐并帮助制定安全计划。(这对于那些不立即进行背景调查，但已经参与了 Tinder match，现在有进一步问题和担忧的在线约会者来说可能是有用的。)

为在线约会者提供背景调查选项的想法源于 ProPublica 和哥伦比亚新闻调查公司 2019 年的一份调查报告，该报告强调了 Match 拥有的约会应用程序上的性侵犯者问题。

但此举也正值科技行业越来越关注消费者保护之际。美国一直在关注监管科技公司，这些公司多年来通过将参与度和增长指标提升到用户安全之上而获利。立法者和美国政府越来越担心技术会导致现实世界的伤害——这已经被证实了，在 Tinder 或任何其他在线约会服务这样的应用程序中，T2 对 T4 是正确的。

图片来源:嘉宝

在推出时，Tinder 将向每位用户提供两次免费背景调查搜索，总共多达 50 万次免费搜索。之后，搜索嘉宝的费用为 2.50 美元，外加每笔交易处理费。Tinder 指出，所有收益都将用于嘉宝的运营成本。

除了在 Tinder 上发布，嘉宝将于今天通过其网站 app.garbo.io 向美国公众发布。网站上的价格和提供给 Tinder 用户的价格一样。

总部位于纽约的嘉宝于 2018 年由凯瑟琳·科斯米德斯(Kathryn Kosmides)创立，她是基于性别的暴力的幸存者，希望让每个人都能够更容易地有能力查找有关某人背景的关键信息，这些信息可能表明暴力历史。与传统的背景调查提供商不同，嘉宝专注于它所谓的“公平背景”调查——指的是那些排除非暴力藏毒指控、游荡、流浪和更多轻微交通罚单的调查，除了酒后驾车和车辆过失杀人。

图片来源:嘉宝

“我们知道未来虐待或暴力的最大指标是这些类型行为的历史。无论是网上约会还是在今天的数字时代我们遇到陌生人的其他方式，我们都应该知道我们是否潜在地将自己的安全置于危险之中，”科斯米德斯说。“我们希望保护那些最容易遭受线上和线下伤害的人，这只是我们在数字时代主动预防伤害的使命的第一步。”

嘉宝认为，它从结果中排除的罪行是那些对边缘化群体影响更大的罪行，这就是为什么它的背景调查被认为是“公平的”。它也不会像竞争对手那样在搜索结果中返回个人信息，比如家庭住址和电话号码。

嘉宝表示，它与专家合作，包括通过其内部倡导委员会来帮助制定战略。该委员会包括来自国家犯罪受害者中心、法院创新中心和 ENDTAB 的个人。

去年，在公开推出之前，非营利组织从 Match Group 获得了七位数的投资。

Match Group 的安全和社会倡导负责人特雷西·布里登说:“长期以来，女性和传统上被边缘化的群体在资源和安全方面面临着许多障碍。“嘉宝深思熟虑和创新的消费者背景调查将推动该行业向前发展，同时赋予人们关键信息，帮助他们做出个人安全选择。”

Match Group 表示，Tinder 的美国用户目前可以使用功能，但它的目标是在未来几个月内将嘉宝推广到 Match Group 的其他美国品牌。
