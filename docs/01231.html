AWS 将从 2022 年开始在国际上推出 30 多个新的本地区域

在其年度 re:Invent 会议上，AWS 宣布将在全球主要城市推出 30 多个新的 AWS 本地区域。这些新的 AWS 本地区域将从 2022 年开始在超过 21 个国家提供，包括阿根廷、澳大利亚、奥地利、比利时、巴西、加拿大、智利、哥伦比亚、捷克共和国、丹麦、芬兰、德国、希腊、印度、肯尼亚、荷兰、挪威、菲律宾、波兰、葡萄牙和南非。

举例来说，AWS 本地区域是一种 AWS 基础架构部署，它将计算、存储和其他精选服务放在离客户更近的地方，使客户能够在 AWS 上运行对终端用户或内部安装要求个位数毫秒延迟的应用程序。

图片来源: AWS

“我非常 激动 向 宣布我们 将 成为 扩大局部 地区 国际 跨越 欧洲 世界"当然，我们也会在我的家乡阿姆斯特丹举办一次."

AWS 指出，这些新地点加入了美国 16 个本地区域，将帮助客户以更低的延迟为世界各地的最终用户提供服务。当前的本地分区位于波士顿、芝加哥、达拉斯、丹佛、休斯顿、堪萨斯城、拉斯维加斯、洛杉矶、迈阿密、明尼阿波利斯、纽约市、费城和波特兰。
