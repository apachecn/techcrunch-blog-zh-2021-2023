Bubbles 旨在通过异步视频信息增强聊天功能

原文：https://web.archive.org/web/https://techcrunch.com/2022/05/12/bubbles-aims-to-supercharge-chats-with-async-video-messaging/

Bubbles ，一家将其平台标榜为视频消息“反缩放”的初创公司，今天宣布完成了由科斯拉风险投资公司、Craft Ventures、Streamlined Ventures、468 Ventures 和贝恩资本牵头的一轮 850 万美元的融资。联合创始人兼首席执行官汤姆·梅德马(Tom Medema)表示，这笔资金将用于进一步开发 Bubbles 的平台，同时加大公司的客户获取力度。

Bubbles 在员工的桌面上添加视频、音频和文本评论——梅德玛说，这个想法是他在荷兰鲜花递送服务公司 Bloomon 担任首席技术官时产生的。在那里，梅德玛管理着一个分布式团队，该团队依赖于已经成为工作场所协作的 典型软件栈:Slack、Zoom 和 Google Docs。

随着团队的成长，协作变得越来越重要，我很快意识到这些工具的不足，我们的团队为此付出了代价。这是一个疯狂的日历俄罗斯方块游戏，不断的通知，不断的沟通失误，”梅德玛通过电子邮件告诉 TechCrunch。" 我意识到的是，我们所谓的生产力工具其实是在扼杀我们的生产力。我们“总是在工作”，却无法完成真正的工作。在寻找解决方案的过程中，我侵入了我们过时的系统和流程，产生了泡沫的愿景。”

梅德玛于 2019 年开始研究 Bubbles，但他认为该工具与在家工作设置的正常化变得更加相关。梅德玛，一个谈到远程工作的悲观主义者——或者更准确地说，典型的远程工作软件——指出了一项针对微软和 LinkedIn 员工的 2021 年研究 ，该研究发现员工筋疲力尽，“数字化疲惫”

图片来源:气泡

“我们的使命是让快乐回归工作:回归创造力和生产力，促进深度工作，并通过从超反应性和实时沟通转向更好的异步沟通来缓解倦怠，”梅德玛继续说道。“异步平台是他们的首选，因为它解放了员工的时间来做深入的工作，并减轻了因整天忙于会议和通知而产生的数字倦怠。”

使用 Bubbles，用户可以通过线程异步发送和接收消息——视频或其他 Medema 声称这允许更周到的反馈。从历史上看，异步视频工具面临的挑战是用户的认同。毕竟，快速发送一条松弛消息比录制和编辑视频更容易。但 Bubbles 试图通过简化许多方面来降低这一过程的吸引力，取消密码、入职步骤甚至用户名。

用户从网络浏览器开始泡泡对话，通过视频、音频或屏幕记录捕捉他们的观点，并邀请合作者。这就开始了一个线索化的对话，参与者可以对特定的时间代码进行评论和添加反应。

未来的计划包括改进发现，特别是语义搜索和主题建模，梅德玛说这将允许 Bubbles 用户“直观地搜索所有对话”除此之外，Medema 表示，Bubbles 将引入转录和摘要功能，自动突出显示视频中的重要部分，并建议行动项目。

梅德玛说:“Slack 上的快速提问、塞满电子邮件的收件箱以及源源不断的通知阻碍了我们实现深度工作和心流的珍贵状态。”。“Bubbles 是一个 异步优先 工具，设计用于在过时的会议架构之外工作，允许用户在每个人的最佳时间协作工作。”

Bubbles 并不是第一个尝试异步消息传递范式的人。其他包括螺纹、螺纹和夹头。Slack 最近试用了 Slack Clips，它可以让用户在频道和直接信息中嵌入音频、视频和屏幕记录。

Medema 声称 Bubbles 最近的月活跃用户超过了 20，000——当然，这是一个健康的数字——但是诀窍在于将其商业化。泡泡目前是免费的。货币化计划包括一个具有文件夹和粒度访问控制的团队层。

图片来源:气泡

过去的两年不仅让领导们认识到在家完成工作是绰绰有余的，也让他们认识到员工可以灵活地控制自己的时间表，从而更好地工作。然而，通过加速我们向远程工作的转换，它也暴露了我们现有远程工作工具中的深层缺陷，这些工具并不是真正设计用于远程世界的，”Medema 说。" 泡泡的异步特性会在对话发生时记录下来。这也有助于部分克服因员工流动而导致的知识流失，这是当今工作环境中的一个主要挑战。”

Bubbles 有八名员工。该公司计划在年底前将团队规模扩大一倍。
