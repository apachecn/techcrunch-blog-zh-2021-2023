微软和英伟达联手打造新的 Azure 托管人工智能超级计算机| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/11/16/microsoft-and-nvidia-team-up-to-build-new-azure-hosted-ai-supercomputer/amp/

图片来源:克里斯汀·博西/彭博/盖蒂图片社

大约两年前，微软宣布与 OpenAI 合作，open AI 是一个人工智能实验室，微软与 open AI 有着密切的商业关系，以建造这家科技巨头所谓的运行在 Azure 云中的“人工智能超级计算机”。包含超过 285，000 个处理器内核和 10，000 个显卡，微软当时声称它是世界上最大的超级计算机集群之一。

现在，大概是为了支持更雄心勃勃的人工智能工作负载，微软表示已经与英伟达签署了一项“多年”协议，以建立一台新的超级计算机，托管在 Azure 上，由英伟达的 GPU、网络和人工智能软件提供支持，用于训练人工智能系统。

微软云计算和人工智能集团执行副总裁斯科特·格思里在一份声明中说:“人工智能正在推动企业和工业计算的下一波自动化浪潮，使组织能够在应对经济不确定性时用更少的资源做更多的事情。”。“我们与英伟达的合作开启了世界上最可扩展的超级计算机平台，该平台为微软 Azure 上的每个企业提供了最先进的人工智能功能。”

截至发稿时，细节还很难获得。但在一篇博客文章中，微软和英伟达表示，即将推出的超级计算机将采用英伟达的 Quantum-2 400Gb/s InfiniBand 网络技术和最近详细介绍的 H100 GPUs 等硬件。当前的 Azure 实例提供了前代 Nvidia A100 GPUs 和 Quantum 200Gb/s InfiniBand 网络。

值得注意的是，H100——Nvidia 的 Hopper 架构的旗舰——配备了一个特殊的“变压器引擎”,以加速机器学习任务，并且——至少根据 Nvidia 的提供的—性能比 A100 好 1.5 到 6 倍。它的功耗也更低，提供了与 A100 相同的性能，能效提高了 3.5 倍。

纽约市熨斗研究所运行的联想制造的 Henri 系统是首批配备 H100 GPUs 的工业规模计算机之一，在今年最高效的超级计算机名单中名列榜首。但是，按照超级计算机的标准，Henri 是一个相对较小的系统，只有数千个 CPU，在全球 500 台最强大的超级计算机中排名第 405 位，这意味着 H100 绝不是圣杯。

作为微软合作的一部分，英伟达表示，它将使用 Azure 虚拟机实例来研究生成式人工智能的进展，或者可以创建文本、代码、图像、视频或音频的自学算法。(想想 OpenAI 的文本生成 GPT-3 和图像生成达尔-E 2 。)与此同时，微软将为新的英伟达硬件优化其 DeepSpeed 库，旨在降低人工智能训练工作负载期间的计算能力和内存使用，并与英伟达合作，向 Azure 企业客户提供该公司的人工智能工作流和软件开发工具包。

为什么英伟达会选择使用 Azure instances 而不是自己的超级计算机塞勒涅，这还不完全清楚；该公司已经利用塞勒涅来训练 GauGAN2 这样的生成性人工智能，gaugan 2 是一种文本到图像的生成模型，可以根据基本草图创作艺术。显然，英伟达预计，它正在工作的人工智能系统的范围最终将超过塞勒涅的能力，甚至可能超过英伟达正在建造的概念验证超级计算机 Eos ，它将配备 4608 个 H100 GPUs，最高可达到 18.4 万亿次人工智能计算性能。

“人工智能技术的进步以及行业的采用正在加速。Nvidia 企业计算副总裁 Manuvir Das 在一份声明中表示:“基础模型的突破引发了研究浪潮，培育了新的创业公司，实现了新的企业应用。“我们与微软的合作将为研究人员和公司提供最先进的人工智能基础设施和软件，以利用人工智能的变革力量。”

对强大的人工智能培训基础设施的贪得无厌的需求导致了云和硬件供应商之间的军备竞赛。就在本周， Cerabras (迄今已筹集超过 7.2 亿美元的风险投资，估值超过 40 亿美元)发布了一台价值 1350 万的核心人工智能超级计算机，名为 Andromeda，据称它可以实现超过 1 exaflop 的人工智能计算。谷歌和亚马逊继续投资于自己的专有解决方案，提供定制设计的芯片——例如 TPUs 和train ium——用于加速云中的人工智能训练。

在可预见的未来，对更强大硬件的推动将会继续。最近的一项研究发现，2016 年至 2022 年间，大规模人工智能模型的计算需求以平均 10.7 个月的速度翻了一番。OpenAI 曾经估计，如果在一个 Nvidia Tesla V100 GPU 上训练 GPT-3，需要大约 355 年。
