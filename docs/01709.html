Line 正在筹备明年的 NFT 服务

原文：https://web.archive.org/web/https://techcrunch.com/2021/12/20/line-to-launch-nft-service-linenext-next-year/

Line 正在为明年的 NFT 服务做准备

总部位于日本的消息应用程序 Line Corporation ，由 Z Holdings 全资拥有，将于明年通过其新的组织 LineNext 推出 NFT 服务，为公司和个人提供在全球(不包括日本)进行 NFT 交易的市场。Line 的发言人说，Line 正在通过适合日本市场的 Line Bitmax 钱包单独运营其 NFT 市场测试版。

Line 上周宣布，它已经在韩国和美国建立了 LineNext，专注于扩大全球 NFT 生态系统。

总部位于韩国的 LineNext 拥有约 100 名员工，正在制定全球 NFT 平台战略和规划，而总部位于美国的 LineNext 正在开发和运营其不可替代的令牌(NFT)平台业务。这位发言人指出，美国办公室有 55 名员工。

由于 LineNext 是平台提供商，“交易经纪费将是主要的收入来源，未来还会有额外的收入来源，”该发言人告诉 TechCrunch。

这位发言人说，LineNext 目前正在与大约 20 个潜在的全球合作伙伴进行讨论。

LineNext 的新全球 NFT 平台将支持全球各地的公司和创作者建立市场，并为普通用户创建社区和生态系统，以进行非功能性交易。

“NFT 是一种技术基础设施，将改变数字空间，并在内容、游戏、社交和商业等所有领域创新用户体验，”Line app products(和 Line fintech 公司)首席产品官 Youngsu Ko 表示，他已被任命为 LineNext 的首席执行官。“作为创新者，Line 拥有十多年的丰富经验，已成为亚洲最受欢迎的科技公司之一。现在，我们的目标是与我们的全球合作伙伴一起，在令人兴奋的新领域 NFTs 中做同样的事情，韩国办事处专注于全球 NFT 平台战略，我们在美国的办事处专注于 NFT 业务。”

LineNext 旨在利用 Line 在开发和发展 messenger 和区块链服务方面的丰富经验，为公司、创作者和用户转变 NFT 体验。

自从建立了 Line 区块链实验室以来，Line 已经发布了其加密资产链接，并在日本运营加密资产交易所 Line Bitmax ，在全球运营 Bitfront 。Line 还运营着区块链服务开发平台 Line 区块链开发者，以及日本 Line Bitmax 钱包上 NFT 市场的测试版。

该公司表示，Zepeto 和电通等区块链合作伙伴已经发行了超过 130 万张 NFT，引入了知识产权、内容和游戏。
