杰弗瑞·卡森伯格为 HUBUC 的嵌入式金融 API TechCrunch 提供了 1000 万美元的种子资金

原文：https://web.archive.org/web/https://techcrunch.com/2022/02/24/jeffrey-katzenberg-backs-the-10m-seed-round-for-hubucs-embedded-finance-api/

我最近写了一篇关于所谓的“嵌入式金融”趋势的文章，引用了 Intergiro 最近在该领域融资的例子。这种趋势现在又有了一个例子，那就是 HUBUC 。

这家初创公司自称为“金融服务的 AWS ”,它出现在西班牙加泰罗尼亚的巴塞罗那，现在已经筹集了 1000 万美元的种子资金，由 WndrCo 和 Runa Capital 共同牵头。

WndrCo 是该公司的一次重大胜利，因为这笔交易将 WndrCo 的管理合伙人和梦工厂动画公司的前联合创始人兼首席执行官杰弗瑞·卡森伯格带到了 HUBUC 的资本表。这是一个强大的盟友。

但有影响力的投资者不止这些。由杰夫·贝索斯、肯尼斯·陈纳德、阿比盖尔·约翰逊和比尔·盖茨支持的 Y Combinator 和 Village Global 也参加了会议，还有一些天使投资人(见下文)。

HUBUC 由两位金融科技企业家(首席执行官哈桑·纳瓦兹(Hasan Nawaz)和首席运营官的伊格纳西奥·贾维尔(Ignacio Javierre)于 2021 年创立，该公司表示，它通过一个无代码解决方案的 API，承担了将金融服务嵌入商业产品的重任。这意味着银行账户、支付、虚拟和实体卡以及实时汇率和数字钱包都可以提供给 HUBUC，因为它是与其合作的金融合作伙伴和该公司产品之间的一层。这意味着它管理合同、监管要求、反洗钱、其他集成和风险等。

在一份声明中，纳瓦兹说:“嵌入式金融不再是大型科技公司的专属特权。我们的愿景是成为一个平台，让任何公司都能够在其产品中嵌入金融服务。为此，我们简化了极其复杂的流程，将一切打包到一份合同和一个 API 中。”他表示，他们在 2017 年考察了各种金融科技垂直行业，但决定采用全面的 API 方法。

HUBUC 表示，它现在覆盖了 58 个地区，与旅游预订支付、营销和媒体支出以及员工福利等产品合作。客户包括万事达卡、Wagestream、PayFlow、OkTicket 和 Declarando。

当然，他们不是唯一的玩家。其中包括 FIS、特梅诺斯、阿德延和 Wex。一些分析师预测到 2025 年，嵌入式金融科技的支付部分收入可能达到 1410 亿美元。

华特·迪士尼电影公司前董事长兼 WndrCo 联合创始人卡森伯格表示:“嵌入式金融将颠覆每一个垂直市场的银行业，从零售、移动到物流和保险。但要实现这一目标，需要大量投资——因此也有风险——来开发和部署。我们可以看到 HUBUC 通过提供全套服务为客户带来的巨大价值。”

Runa Capital 普通合伙人 Andre Bliznyuk 补充说:“我们认为金融科技基础设施是未来 10 年最有前景的主题之一，因为推出金融产品的需求正在迅速加速。”

这轮融资的天使投资人包括 Sujay Jaswa (Dropbox)、John Lilly(Mozilla 前首席执行官)、Immad Akhund(Mercury 首席执行官)、Lars Fjeldsoe-Nielsen(优步前副总裁)、Bo Jiang(Lithic 首席执行官)、John Bautista(Orrick 合伙人)和 Anthony Saleh(比特币基地和 Robinhood 的天使投资人)。
