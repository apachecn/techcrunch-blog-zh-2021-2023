原文：https://web.archive.org/web/https://techcrunch.com/2022/10/05/apollo-launches-graphos-its-end-to-end-supergraph-platform/

阿波罗推出其端到端超级图形平台 GraphOS

GraphQL 公司 Apollo 今天宣布推出 GraphOS，这是一个新的端到端平台，旨在帮助企业构建、连接和扩展该公司所谓的“超级图形”对于阿波罗来说，这些超级图最近成为了的一大焦点。这里的基本思想是允许企业将来自各种来源的数据集成到单个超图中，然后开发人员可以在他们的应用程序中轻松使用该超图。

通过 GraphOS，该公司现在可以让企业更容易地开始使用这些超级图表，为他们提供一种服务，消除建立必要的基础设施来构建和管理这些图表的复杂性。

“GraphOS 实现了超图的基本承诺:让开发人员轻松获得构建优秀应用和体验所需的任何东西，而不必为他们需要的每样东西编写代码，”Apollo 首席执行官兼联合创始人 Geoff Schmidt 说。“当有人连接到互联网时，他们打开浏览器并开始输入。这应该是开发人员在访问他们需要构建的数据和服务时的相同体验。”

该公司去年筹集了 1.3 亿美元的 D 轮融资，认为新服务为用户提供了轻松连接所有数据的能力，以及作为托管服务一部分的快速、统一的查询执行。Apollo 还强调，开发人员仍然可以获得所有必要的 CI/CD、可观察性和协作工具来处理他们的图形。

值得注意的是，Apollo 将提供云托管和自托管路由选项，具有访问控制和 SOC 2 Type II 合规性，由 Doyensec 审核。

展望未来，Apollo 还计划扩展这一概念，很快允许用户在公司网络之外链接超级图，以包括第三方 API。Apollo 认为，GraphOS 将为其所谓的“全球超级图”提供基础，随着时间的推移，它可能成为数据访问的市场，允许开发人员“在一次操作中查询他们需要的任何东西。”这是一个崇高的目标，但你不能指责阿波罗想得太少。

施密特说:“supergraph 和 GraphOS 旨在解决应用程序开发人员的需求，为他们提供构建令人惊叹的体验所需的灵活性和资源，而没有不断妨碍他们的复杂性和摩擦。”。“我们已经看到业界一致认为超图满足了这些需求。GraphQL 中最初的开源规范，已经发展成为一个蓬勃发展的生态系统和围绕超级图的全球开发者运动。”
