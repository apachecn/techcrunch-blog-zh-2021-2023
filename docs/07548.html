Twitter 正式推出其长篇内容“笔记”功能

Twitter 今天宣布已经开始测试“笔记”，这是一种在社交媒体平台上撰写和发布长篇内容的方式。官方声明是在 TechCrunch 报道该功能将很快推出的一天后发布的。

美国、加拿大、加纳和英国的一小群作家现在可以在初步测试阶段使用 Notes。Twitter 表示，大多数国家的人都可以在 Twitter 上和 Twitter 外阅读笔记。测试阶段的用户将可以访问一个新的“写”标签，在这里他们可以写和访问他们所有的笔记。这些用户还将在他们的个人资料中有一个新的“注释”标签，用于保存他们已发布的作品，以便他们的追随者可以轻松找到他们的长篇内容。

有了这一新功能，用户将能够使用丰富的格式和上传的媒体来创建文章，然后在发布时可以在推特上发布并与关注者分享。用户可以选择在笔记中嵌入照片、视频、gif 和 tweets。像推文一样，笔记将有自己的链接，可以被推文，转发，在 DM's 中发送，喜欢和书签。

Twitter Notes 有可能改变一些人使用 Twitter 分享他们更深入的思想和想法的方式。这项新功能对于那些不经常发表文章长度的内容、不想为建立和维护自己的博客或网站而烦恼的用户来说尤其有用。同样值得注意的是，该功能标志着自将字符数从 140 增加到 280 后，Twitter 的一个重大变化。

新功能将通过允许用户将所有文本放入注释中来解决难以跟踪线索的情况。Notes 还将解决用户为了发布大部分文本而不得不发布 Notes 应用程序截图的情况。

Twitter 笔记的引入可能会与 WordPress 或 Medium 等传统博客平台形成竞争，后者碰巧是由 Twitter 联合创始人伊万·威廉姆斯开发的。如果 Twitter 决定将时事通讯整合到便签中，该功能可能会与 Substack 等流行平台形成竞争。

Twitter 今天还宣布，它将推出一个新的 Twitter 写作团队，专注于为读者构建工具。该公司正在将其时事通讯订阅合并到 Twitter Write 中，Twitter Write 现在是它用来涵盖其长篇写作成果的品牌，包括笔记和时事通讯。写作团队将专注于改善作家的 Twitter 体验。这家社交媒体巨头指出，它对作家的定义包括记者、博客作者、时事通讯出版商、喜剧演员、内容创作者、社交媒体社区经理、诗人、编剧等等。
