框架寻求将可修复性扩展到笔记本电脑之外

原文：https://web.archive.org/web/https://techcrunch.com/2022/02/01/framework-looks-to-expand-repairability-beyond-the-laptop/amp/

图片来源:框架

去年 11 月，苹果宣布推出自助维修服务——对于一家众所周知缺乏这种能力的公司来说，这是一个令人惊讶的新发现。当然，这些变化不是在真空中发生的。总统和国会都在推动开放所谓的修复权。这有很多原因，从消费者的选择到可持续发展的考虑。

但是，即使曾经抵制的公司开始接受这种变化，向用户开放可修复性和实际上使产品用户可修复之间还是有区别的。随着消费电子产品变得越来越薄，非专业人士修理它们也变得更加困难。

由前苹果公司和 Oculus/脸书工程师 Nirav Patel 于 2019 年底创立的 Framework 是那些专注于将可修复性作为产品设计关键功能的新兴硬件创业公司之一。今天，该公司宣布了一项 1800 万美元的首轮融资，称这是对这一使命的证明。

图片来源:框架

Patel 在宣布加薪的新闻稿中说:“你们所有人对我们的任务和 Framework 笔记本电脑的巨大和直接兴趣清楚地表明，我们正在正确的轨道上。”“这个行业早就应该有经久耐用的个性化产品了。这对你们所有人和我们来说都是显而易见的，我们在 Spark 的合作伙伴也完全相信这一点。”

这轮融资由 Spark Capital 牵头，正如 Patel 指出的那样，该公司也领导了 Oculus 的 a 轮融资。融资发现 Spark 普通合伙人 Kevin Thau 加入了 Framework 的董事会。种子投资者 Pathbreaker Ventures、Anorak Ventures 和蚁族 Ventures 也将重返该轮融资。帕特尔很快指出，这家总部位于湾区的公司并不“需要投资者的资金来维持运营”，而是利用资本来增加其产品供应，以“为更多的消费电子产品提供可升级性、定制和维修服务。”

没有关于这些类别可能是什么的消息，尽管该公司表示已经制定了未来两年的路线图。智能手机似乎是一个合理的选择，因为它无处不在，尽管目前这个市场比笔记本电脑更加饱和。这也是总部位于阿姆斯特丹的 Fairphone 相当积极的目标，去年发布了 Fairphone 4。

额外的资金将用于增加框架的人员。用户可修复性的吸引力是否足以让相对小众的受众产生兴趣，这仍然是个问题。虽然大公司在可修复性方面取得了一些进展，但似乎不太可能有许多大公司会像框架或 Fairphone 一样开放地对待设计。
