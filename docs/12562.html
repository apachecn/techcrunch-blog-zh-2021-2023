哈维利用人工智能回答法律问题，从 OpenAI | TechCrunch 获得现金

原文：https://web.archive.org/web/https://techcrunch.com/2022/11/23/harvey-which-uses-ai-to-answer-legal-questions-lands-cash-from-openai/

哈维，一家被其描述为“律师的副驾驶”的初创公司，今天在 OpenAI 创业基金的领导下，秘密获得了 500 万美元的资金，OpenAI 及其合作伙伴正通过这笔资金投资于解决重大问题的早期 AI 公司。谷歌人工智能的领导者杰夫·迪恩和 Mixer Labs 的联合创始人埃拉德·吉尔以及其他天使般的支持者也参加了这一轮。

哈维是由温斯顿·温伯格(Winston Weinberg)和加布里埃尔·佩雷拉创建的，前者是奥梅尔维尼&迈尔斯律师事务所的证券和反垄断诉讼律师，后者曾是 DeepMind、谷歌大脑(谷歌的另一个人工智能集团)和元人工智能的研究科学家。温伯格和佩雷拉是室友——佩雷拉向温伯格展示了 OpenAI 的 GPT-3 文本生成系统，温伯格意识到它可以用来改进法律工作流程。

“我们的产品为律师提供了现有法律工作流程的自然语言界面，”佩雷拉在一次电子邮件采访中告诉 TechCrunch。“哈维没有手工编辑法律文件或进行法律研究，而是让律师用简单的指令描述他们希望完成的任务，并接收产生的结果。为了实现这一点，哈维利用大型语言模型来理解用户的意图并生成正确的输出。"

更具体地说，哈维可以用自然语言回答这样的问题:“告诉我第四巡回法庭的雇员和独立承包商之间有什么不同？”和“告诉我租约中的这一条款是否违反了加利福尼亚法律，如果是，重写它，使它不再违反。”一读之下，哈维似乎可以取代律师，在接到通知后立即提出法律论点并提交草案。但佩雷拉坚称事实并非如此。

“我们希望哈维能成为科技和律师之间的中介，成为法律的自然语言界面，”他说。“哈维将提高律师的工作效率，让他们能够完成更高质量的工作，并将更多的时间花在他们工作的高价值部分。哈维为所有法律工作流程提供了一个统一、直观的界面，允许律师用简单的英语描述任务，而不是使用一套复杂而专业的工具来完成特殊任务。”

这在理论上是很强大的东西。但这也令人担忧。鉴于大多数法律纠纷的高度敏感性，律师和律师事务所可能不愿意让 Harvey 这样的工具访问任何案件文件。还有一个问题是，语言模型倾向于宣扬毒性和 T2 捏造的事实，这在法庭上是很难被接受的——如果不是伪证的话。

这就是为什么目前处于测试阶段的 Harvey 附带了一个免责声明:该工具不是为了向非律师提供法律建议，应该在有执照的律师的监督下使用。

在数据隐私问题上，Pereyra 表示，Harvey 尽力满足客户的合规需求，匿名化用户数据并在预定时间后删除数据。他说，用户可以根据要求随时删除数据，并且对 Harvey 不会在客户之间“交叉污染”数据这一事实感到欣慰。

现在还为时尚早。但 Pereyra 已经表示，从律师事务所到法律援助组织，“整个法律界的用户”都在使用哈维。

它面临着一些竞争。Casetext 使用人工智能，主要是 GPT-3，来查找法律案例，并协助一般的法律研究任务和简要起草。更多的手术工具，如 Klarity 使用人工智能从合同审查中剥离苦差事。有一段时间，初创公司 Augrented 甚至在探索如何利用 GPT-3 以简单的英语总结法律通知或其他来源，以帮助租户捍卫自己的权利。

首先，OpenAI 的首席合规官兼 OpenAI 创业基金经理布拉德·莱特卡普(Brad Lightcap)认为哈维已经足够与众不同。它也将受益于与 OpenAI 的关系；OpenAI 创业基金的参与者除了获得资本外，还可以从微软获得新的 OpenAI 系统和 Azure 资源。

“我们相信哈维将对我们的法律体系产生变革性的影响，使律师能够更有效地为更多的客户提供更高质量的法律服务，”Lightcap 通过电子邮件表示。“我们启动了 OpenAI Startup Fund，以支持公司使用强大的人工智能来推动社会层面的影响，哈维关于人工智能如何增加法律服务和改善结果的愿景正好符合我们的使命。”

Harvey 有一个五人团队，Pereyra 预计到今年年底，这个数字将增加到五到十名员工。当被问及收入数字时，他没有回答。
