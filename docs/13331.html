杰克·多西警告不要攻击 Twitter 员工，并每年投入 100 万美元用于 Signal | TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/12/13/jack-dorsey-warns-against-attacks-on-twitter-staff-and-dedicates-1m-a-year-to-signal/

Twitter 创始人杰克·多西对该社交网络的现状和前景发出警告，称其不符合他希望达到的标准，对其员工的骚扰是短视和危险的。正如他之前所说，是时候继续前进了，为此，他资助了“开放互联网发展”的新努力，从每年 100 万美元开始。

从一个 Twitter 帖子开始，但很快转变为一篇博客文章(他写道:“我不想把所有的东西都编辑成 280 个字符块——这可能是他从未预料到的)，多尔西说，随着一位不知名的维权投资者的加入，他根据自己的愿望建立 Twitter 的希望在 2020 年破灭了。

“我当时就计划好了离开，因为我知道自己不再适合这家公司了，”他写道。

他承认，希望建立在基础上的原则——对企业和政府控制的弹性、用户控制的内容无一例外以及算法审核——在今天的 Twitter 和他领导的 Twitter 中都不存在。

即便如此，他写道，与所谓的推特文件中的暗示相反，“没有恶意或隐藏的议程，每个人都是根据我们当时掌握的最佳信息采取行动的。”

各种线索在展示和编辑的内容上非常挑剔，同时将某些员工，特别是前信任和安全负责人 Yoel Roth，描绘成权力狂人和议程驱动者。据报道，罗斯亲身经历了严重的骚扰，以至于他不得不暂时离开自己的家。总的来说，除了为想象中的虐待找几个方便的替罪羊之外，已经出版的东西没有什么新意。

多尔西说:

至于这些文件，我希望它们能以维基解密的方式发布，需要考虑更多的视角和解释。与此同时，对当前和未来行动的透明度承诺。我希望这一切都会发生。没什么好隐瞒的…只有很多值得学习的地方。目前对我以前同事的攻击可能是危险的，并且不能解决任何问题。如果你想责备，就把矛头指向我和我的行为，或者我的缺失。

小心你的愿望，杰克。

正如我上周所写的，这些对话本身，事实上构成了一个非常有趣的视角，审视了在前所未有的情况下，适度的困难。关于如何解释一项规则或他们应该或不应该采取什么行动的坦率和公开的讨论正是人们希望在这样一个过程的幕后发生的事情。偏见的归罪背后很少或根本没有文献的分量，除了一个精心策划的公开旨在促进这种叙述的陈述。

至于实际的解决方案，Dorsey 当然在 Bluesky 努力工作(或者至少是现在),但是他指出 Mastodon 和 Matrix 是其他有价值的发展途径:

还会有更多。人们将有机会成为像 HTTP 或 SMTP 那样的标准。这不是一个“去中心化的推特”这是对基础核心技术标准的集中和紧急推动，以使社交媒体成为互联网的一部分。

他说到做到，宣布他将从每年 100 万美元资助 Signal(肯定对政府有弹性)开始。他说，更多的赠款即将到来，并征求建议。幸运的是，由于似乎是他的个人电子邮件被马特泰比无意中发表在第一个 Twitter 文件线程，每个人都应该能够取得联系。
