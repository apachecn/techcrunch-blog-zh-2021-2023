Snapchat for Web 现已面向全球所有用户开放

Snapchat 周四宣布，该公司正在向其整个全球社区提供其应用程序的桌面版本 Snapchat for Web 。这家社交网络最初于 7 月向美国、英国、加拿大、澳大利亚和新西兰的 Snapchat+ 用户推出了 Snapchat for Web。现在，所有 Snapchat 用户都可以访问桌面版。

使用 Snapchat for Web，您可以发送消息和快照，还可以通过视频和语音通话聊天。Snapchat for Web 还包括访问移动应用的消息功能，包括聊天反应。它现在还拥有 10 多个用于视频通话的镜头。

图片来源:抓拍

要访问 Snapchat 网络版，你需要前往web.snapchat.com，用你的 Snapchat 用户名和密码登录。然后，系统会提示您在手机上完成两步验证。打开 Snapchat for Web 后，您可以从手机上离开的地方继续您的对话。当您使用 Snapchat for Web 时，您的 Bitmoji 将出现在手持笔记本电脑的聊天中，以向其他人表明您正在通过桌面访问聊天。此外，该应用的网页版包括一个隐私屏幕，如果你离开去执行另一个任务，它会隐藏 Snapchat 窗口。

与移动 Snapchat 体验一样，通过 Snapchat for Web 发送的消息会在 24 小时后自动删除。该公司还表示，Snapchat for Web 将防止人们截图。在发布时，该公司表示，它意识到人们仍然可以从手机屏幕上拍照，并指出该产品并不完美。

该公司此前分享称，Snapchat 上的普通来电者每天花在通话上的时间超过 30 分钟。尽管 Snap 仍将自己视为移动优先的平台，但它表示，在听取了用户的反馈后，它决定是时候将 Snapchat 的核心功能引入网络了。

目前，Snap 专注于将该应用的核心功能带到网络上，并将在此基础上构建体验。该公司计划在未来将该应用程序的其他功能，包括查看故事和回忆引入 Snapchat for Web。

Snapchat for Web 的广泛推出正值社交媒体平台寻求更好地与抖音和 T2 的 Instagram 竞争，这两个平台都可以通过网络访问。

图片来源:抓拍

除了让所有用户都能使用 Snapchat for Web 之外，Snapchat 还在聊天选项卡的顶部推出了新的聊天快捷方式，以便轻松处理未读快照和朋友的聊天，查看未接来电和回复故事。该公司表示，这些快捷方式还会提醒你是否需要回复某人，并在你朋友的生日即将到来时向你显示。

Snapchat 还推出了一个新的“问题贴”功能，让用户以 AMA 风格在故事中提问。您可以使用该功能来做一些事情，如询问您的朋友您应该穿什么去学校，晚餐应该吃什么。值得注意的是，新功能类似于 Instagram 的问题贴。

该公司表示，这些新功能现在已经推出或即将推出。
