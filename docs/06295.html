Supabase 因其开源 Firebase 替代方案获得 8000 万美元

原文：https://web.archive.org/web/https://techcrunch.com/2022/05/10/supabase-raises-80m-series-b-for-its-open-source-firebase-alternative/amp/

图片来源:3 lexd/Getty Images

Supabase ，宣称自己是谷歌 Firebase 等服务的开源替代者，今天宣布它已经筹集了由 Felicis Ventures 牵头的 8000 万美元的 B 轮融资。Coatue 和 Lightspeed 也参与了这轮融资，使该公司迄今为止的总融资额达到 1.16 亿美元。

当然，该服务无法在逐个功能的基础上与 Firebase 相匹配，但它提供了开发人员入门所需的许多核心功能，包括数据库、存储和认证服务，以及最近推出的 Supabase Edge Functions ，这是一种无服务器的功能即服务产品。

图片来源: Supabase

正如 Supabase 首席执行官兼联合创始人保罗·科普斯通(Paul Copplestone)告诉我的那样，该公司去年增长迅速，社区现在已经发展到超过 80，000 名开发人员，他们在该服务上创建了超过 100，000 个数据库——在过去 12 个月中增长了 1，900%。

“我们正在向高端市场发展，发现越来越多的客户将我们作为 Postgres 即服务产品使用，基本上是作为[亚马逊] RDS 的替代产品。当然，Heroku 现在是一个有趣的地方，因为很多人想移民。因此，基本上，任何想寻求 Postgres 即服务产品的人，我们开始看到越来越多的人直接接触，”他说，并指出，用户感兴趣的不仅仅是数据库本身，还有公司围绕它的工具，包括自动生成的 API 和 GraphQL 扩展，以及根据需要扩展数据库的能力。“它真的专注于让开发者非常容易地开始和建立，”他说。

投资者也在关注 Supabase，因为它专注于自己的数据库。Felicis 创始人兼管理合伙人 Aydin Senkut 解释说:“即使在当前的市场环境下，如果你看看世界上最有价值的公司，它要么是数据库，要么是安全，这两者目前都是两个最重要的领域。“当我们看到数据库公司[……]时，首先，我需要赞扬[保罗·科普斯通]和[联合创始人安东尼·威尔逊]。作为创始人，他们令人印象深刻。他们真的走得很快。老实说，他们发送代码和打动客户的速度达到了我只在谷歌和 Shopify 等公司见过的水平。”

图片来源: Supabase

他还指出，Supabase 显然能够展示令人印象深刻的增长数字，并期望该团队现在能够在此基础上执行并有效地将该平台货币化。当我问及公司的盈利战略时，Copplestone 说:“我们正在利用基础设施盈利，这是一条非常明确的收入途径，我们希望在未来几年看到更多的增长。”。

有了这笔新资金，Supabase 计划将其团队扩大一倍，以构建其平台(侧重于企业功能)和走向市场战略。到目前为止，该公司的大部分增长都是有机的。
