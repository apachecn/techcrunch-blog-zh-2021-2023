Jarvis ML 筹集 1600 万美元帮助公司产品个性化

原文：https://web.archive.org/web/https://techcrunch.com/2022/04/21/jarvis-ml-snags-16m-to-help-companies-personalize-their-products/

Jarvis ML 是一个为销售产品、服务和体验的品牌提供人工智能个性化引擎的平台，今天宣布在由戴尔技术资本牵头的种子轮融资中筹集了 1600 万美元。在接受 TechCrunch 采访时，首席执行官 Rakesh Yadav 表示，新资金将用于壮大 Jarvis ML 的研发、销售和营销团队，以“加速产品开发和市场渗透”

随着疫情刺激品牌更新——或从零开始创造——网上形象，个性化的价值成为焦点。习惯了网飞和亚马逊式的定制产品推荐，客户开始要求各种规模的公司都提供同样的产品。根据麦肯锡的调查, 71%的购物者现在希望商家提供个性化的互动，而 76%的人对此感到失望。

一些研究——不出所料，特别是来自客户分析供应商的研究——表明个性化是一项值得的投资。对一项调查作出回应的 40%的消费者表示，由于个性化的体验，他们已经购买了比最初计划更贵的东西。但是从技术的角度来看，创建这种类型的个性化是具有挑战性的。

这也是 Yadav 在 2021 年创立 Jarvis ML 的原因。作为谷歌的前高级员工工程师，他领导了谷歌支付和谷歌广告 背后的机器学习平台的开发，亚达夫试图创造一种产品，使公司能够将数据转化为品牌参与，如营销活动或定制的网络体验。

疫情加速了消费者购买趋势向线上转移。这也意味着在线推荐策略对于企业适应这种不断变化的消费者模式至关重要，”亚达夫通过电子邮件告诉 TechCrunch。 " 亚马逊、Airbnb、谷歌和脸书等大型科技公司利用机器学习来取悦消费者，并限制成长期和中端市场公司的独立性，这些公司最终在科技巨头生态系统中被降级为供应商或履行角色。Jarvis ML 使这些公司能够利用他们已经拥有的数据来减少对科技巨头的依赖，同时实现可持续发展。”

Jarvis ML 的网站截图。

Yadav 将 Jarvis ML 描述为一个完全托管的“机器学习即服务”解决方案，旨在允许公司快速将个性化引擎部署到他们的产品中。该平台利用算法来学习公司提供给它的数据中的销售和库存模式，并建立 预测、定价和促销模型，使这些公司能够个性化他们的网站、应用程序和广告以及礼宾服务和客户服务。

人们发现，个性化引擎会产生各种各样的偏见。通常，它们是数据不平衡的结果-一组客户在用于开发引擎的数据中代表性不足。去年，LinkedIn 表示 修复了一个问题，该问题导致 LinkedIn 的连接建议对于那些不经常使用该服务的人来说变得不太准确。其他研究表明，在电子商务网站上，推荐系统可以在经济上不公平地对待处于不利地位的顾客，而不是那些大量购买的顾客。

Yadav 没有直接回答偏见问题，但强调 Jarvis ML 的客户“拥有他们的数据”，并且该平台针对不同购物者的“终生价值观、偏好和品味”优化了收入

Yadav 解释说:“arvis ML 只是一个平台，它有助于将这些数据转化为可操作的品牌活动，如营销活动或个性化网站体验。 “通过分析客户群，Jarvis ML 可以提供高度相关的推荐产品、服务和体验，从而最大限度地提高销售额。 我们的系统从这些基于品味的模型池中挑选最佳模型，并为我们的企业客户挑选表现最佳的模型。”

根据 Grand View Research 的数据，推荐引擎市场到 2028 年可能价值 173 亿美元，Jarvis ML 与专注于电子商务的初创公司竞争，如 T2 的 Constructor 和 T4 的 RichRelevance。其他对手包括 Flybits 和Monetate(2019 年被 Kibo 收购)。

但 Yadav 对 Jarvis ML 在竞争中的增长能力表示有信心，并指出 Twiddy & Company Vacation Rentals 等客户的早期采用。这家初创公司目前有 21 名员工，计划在今年年底前扩大到 40 多人。

"我们的产品易于装载，并提供深度机器学习功能，以帮助企业创造更多收入……例如，一家旅游公司可以在其网站上自动向一个来自贝弗利山的家庭展示海滨住宅，同时向一对来自盐湖城的退休夫妇展示更普通的公寓。根据生命周期价值的背景，两者都与每个客户相关，”Yadav 说。“技术上的 C-suite 级别的经理可以通过简单地利用 Jarvis ML JavaScript SDK 和一行代码更改来产生结果。”
