谷歌云的 AlloyDB Omni 让你可以在任何地方运行兼容 PostgreSQL 的数据库

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/29/google-cloud-launches-alloydb-omni-a-downloadable-version-of-its-postgresql-compatible-database/amp/

图片来源:anu cha sirivisansuwan/Getty Images

AlloyDB for PostgreSQL 是 Google Cloud 完全托管的基于云的数据库服务。通常情况下，这些诞生于云中的服务就是故事的结尾。但是谷歌云对 AlloyDB 采取了一种有点不寻常的方法。今天，该公司推出了 AlloyDB Omni ，这是 AlloyDB 的一个可下载版本，用户可以在公司内部、边缘或笔记本电脑上安装。将有一个免费的开发者版本，而商业用户将不得不支付他们的许可证。

谷歌认为，AlloyDB Omni(该公司将把它打包成一个可下载的容器)的速度是事务性工作负载的两倍，并且可以提供比 PostgreSQL 的标准开源版本快 100 倍的分析查询。谷歌还指出，AlloyDB Omni 可以扩展到更多的 CPU。

“当我们宣布 AlloyDB 时，我们谈论了很多事实，许多客户已经找到我们，谈论他们对摆脱遗留数据库并转向开放 API 的兴趣，无论这些客户是来自 Oracle、[微软] SQL Server、[IBM]Db2，谷歌的 Andi Gutmans 解释道。”AlloyDB 的部分使命实际上是帮助客户将其现有的传统数据库现代化到开放 API 上，而我们 AlloyDB 托管服务的重点是弥合客户在内部运行的高端传统数据库环境与他们可以从开源中获得的东西之间的一些差距。"

Gutmans 指出，Google 引起了那些希望迁移到 AlloyDB 的客户的极大兴趣，但也有不少人指出，将所有工作负载迁移到云需要花费相当多的时间，他们的策略通常围绕着多云方法。Gutmans 说，这些客户希望以比迁移到云更快的速度从传统数据库中迁移出来。不出所料，这一结果就是 AlloyDB Omni，它现在将允许谷歌的客户现代化他们的工作负载。

IDC 数据管理软件研究副总裁 Carl Olofson 表示:“虽然云通常是最终目的地，但许多组织的工作负载无法立即迁移，因此需要可下载的跨平台数据库来运行企业应用程序。“AlloyDB Omni 是谷歌云的一个伟大举措，因为它满足了客户的需求，而不是要求他们更换平台。具有加速性能的 PostgreSQL 兼容数据库、集成的分析引擎和 ML 驱动的管理满足了数据库市场的关键需求，AlloyDB Omni 对于寻求由主要供应商支持并提供云和非云部署选项的成熟数据库的组织来说是一个极具吸引力的选择。”

鉴于 AlloyDB 本质上是 PostgreSQL 的替代产品，企业可以使用他们目前可能已经使用的所有 PostgreSQL 兼容工具来备份和复制他们的数据库，例如，该服务显然与他们可能已经拥有的所有 PostgreSQL 支持的应用程序兼容。

谷歌表示，一旦 AlloyDB Omni 全面上市，它将提供全面的企业支持，包括 24/7 技术支持。

当被问及谷歌云是否会将一些工作贡献给开源 PostgreSQL 项目时，古特曼斯指出，该公司计划成为该项目的积极贡献者，但它也会将一些工作保密。“在某些领域，我们相信我们有一些秘密的优势和差异化，这对我们来说很重要，”他说。“我给你举个例子:我们帮助加速分析查询的矢量化引擎，它基于谷歌内部多年的创新和我们在 AlloyDB 之前已经做的工作。这可能是我们更有可能在 AlloyDB 产品中保留的东西。但我们认为，在某些领域，我们可以成为 Postgres 更重要的贡献者，并真正帮助确保我们回馈社会，真正让 Postgres 社区变得更好。”
