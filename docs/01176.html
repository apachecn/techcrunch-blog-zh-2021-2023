威盛融资 1.3 亿美元扩大按需穿梭软件业务

原文：https://web.archive.org/web/https://techcrunch.com/2021/11/30/via-raises-130m-to-expand-on-demand-shuttle-software-business/amp/

图片来源: Via

按需班车和软件公司 Via 又筹集了 1.3 亿美元，随着城市对更新其传统交通系统的需求增加，这笔资金使其估值达到约 33 亿美元。

这轮融资由骏利·亨德森(Janus Henderson)牵头，贝莱德(BlackRock)、ION Crossover Partners、科赫颠覆性技术(Koch Disruptive Technologies)和现有投资者 Exor 管理的基金和账户参与了融资。迄今为止，该公司已经筹集了 8 亿美元。

如今拥有约 950 名员工的威盛，其业务有两个方面。该公司在华盛顿特区和纽约运营面向消费者的班车。它的底层软件平台出售给城市、交通部门、学区和大学来部署他们自己的班车，这不仅是它的业务核心；它已经成为增长的主要驱动力。

联合创始人兼首席执行官丹尼尔·拉莫特此前告诉 TechCrunch，当该公司在 2012 年首次推出软件即服务平台时，城市对该平台兴趣不大。Via 在 2017 年底与奥斯汀达成了第一个城市合作伙伴关系，此前它向运输管理局免费提供了该平台。这足以让 Via 开展案例研究，并说服其他城市购买这项服务。Ramot 在去年的一次采访中说，2019 年，合伙业务“起飞了”。

今天，软件方面——内部称为 transit tech——已经超越了面向消费者的业务。Via 表示，TransitTech 的收入同比增长了一倍多，年运营成本超过了 1 亿美元。该软件平台被 500 多家合作伙伴使用，包括洛杉矶地铁公司。美国的泽西城和迈阿密，以及德国铁路公司 Arriva Bus UK，该公司使用它提供连接通勤者和英国肯特高速火车站的第一英里和最后一英里服务

Via 没有提供计划如何使用这笔资金的细节。该公司在过去的 18 个月里进行了两次收购，包括 2020 年的 Fleetonomy。

今年早些时候，威盛用1 亿美元现金和股权收购了 Remix，这是一家开发用于城市交通规划和街道设计的地图软件的初创公司。Remix 成为 Via 的子公司，这一安排将使这家初创公司保持其独立品牌。

此次收购已经产生了一个产品，该产品将 Remix 的协作制图和交通规划工具与 Via 的按需交通数据相结合。本月早些时候，Via 推出了一款软件工具，帮助城市了解和规划按需乘车和固定路线公交(如公交车)如何协调工作。
