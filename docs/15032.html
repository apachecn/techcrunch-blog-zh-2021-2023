微软的 Edge 网络浏览器获得类似 ChatGPT 的功能

原文：https://web.archive.org/web/https://techcrunch.com/2023/02/07/microsoft-updates-its-edge-web-browser-with-built-in-a-i-capabilities/

在今天的微软新闻发布会上，该公司展示了其新的人工智能搜索功能，该公司还宣布了其新版本的 Edge 网络浏览器，其中包括内置的人工智能技术。有了新的浏览器，网络用户能够在人工智能的帮助下总结搜索结果，并与人工智能聊天机器人交谈，以获得他们的问题的答案，等等。

微软公司副总裁优素福·迈赫迪在展示新版浏览器时说:“它更时尚、更轻巧，现在有了一些美学上的变化，比如圆角。

与 Bing 的更新类似，Edge 的侧边栏中有一个人工智能聊天机器人，启动时包括两个功能:聊天和撰写。

图片来源:来自 TechCrunch 的微软活动照片

在某些情况下，这些可以结合起来帮助用户做一些事情，比如总结冗长的网页内容，进行比较，甚至创建内容。

Mehdi 在活动期间展示了 Edge 的人工智能能力如何在少数演示中投入使用。其中一次，他访问了 Gap 网站，查看季度报告和收益。他点击了一个链接来查看第三季度的报告，然后弹出了一个 15 页的 PDF 文件。

“挺长的。我没时间读完这些。他说:“我喜欢的是要点总结。”

然后，点击一下，他从浏览器顶部打开一个聊天机器人，并要求它提供页面中最重要的内容。迈赫迪指出，人工智能会迅速返回一个摘要，代表着“大量的时间节省”。

然而，更令人印象深刻的是人工智能接下来要做的事情。他让 it 部门将 Gap 第三季度的财务状况与 Lululemon 第三季度的财务状况进行比较，并根据调查结果制作一个表格。

图片来源:来自 TechCrunch 的微软活动照片

在另一个演示中，他展示了人工智能如何与 Stack Overflow 网站上的开发人员资源一起使用。他展示了一个研究如何解析 JSON 文件的开发人员如何找到他们想要使用的代码片段。但是代码是 Python 的，开发者想要 Rust 的。

为了从人工智能那里获得帮助，Mehdi 突出显示了文本，然后将其显示在 Edge 的新侧边栏中，然后指示人工智能用 Rust 编写代码。确实如此。

在最后一个例子中，Mehdi 演示了使用侧边栏中的人工智能来编写 LinkedIn 帖子。(你觉得现在公司的帖子听起来像机器人！).

在这种情况下，他给人工智能一些提示来启动它。你也可以要求根据语气、格式或长度来细化帖子，Edge 会相应地理解并更新草稿。

在微软向 OpenAI 投资10 亿美元之后，关于 Edge 的消息与 OpenAI 技术在 Bing 搜索引擎中的预期到来一起宣布。在活动中，微软展示了如何将 OpenAI 的 GPT-4 模型集成到其搜索引擎中，为用户提供类似 ChatGPT 的体验。除了必应的情况，人工智能不仅限于覆盖到 2021 年的数据——它能够处理与更近期事件相关的查询。

除了必应的这些人工智能更新，更新后的 Edge 浏览器将于今天预览。

“人工智能将从根本上改变每一个软件类别，首先是最大的类别——搜索，”微软董事长兼首席执行官塞特亚·纳德拉在的一次声明中说。“今天，我们推出了由人工智能 copilot 和 chat 支持的 Bing 和 Edge，以帮助人们从搜索和网络中获得更多。”
