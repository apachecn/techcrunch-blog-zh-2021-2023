Point One Navigation 筹集 1000 万美元，将其精确定位技术应用于所有车辆 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/11/08/point-one-navigation-raises-10m-to-bring-its-precise-location-tech-to-all-vehicles/

驾驶新车或卡车上下班的司机很可能经历过乒乓效应——这是一种配备了先进驾驶辅助系统的车辆难以在车道上居中并来回跳动直到找到最佳位置的现象。

Point One Navigation 于 2016 年出现，该技术消除了当车辆不在它应该在的确切位置时出现的问题和其他问题。这家总部位于旧金山的初创公司开发了一种用于精确定位的 API，可以应用于任何移动的车辆，从城市空中出租车和无人机到滑板车，配备 ADAS 的车辆，自动驾驶汽车，甚至是农用设备。

“位置是机器人技术需要解决的支柱之一，”首席执行官兼联合创始人亚伦·内森在最近的一次采访中说。“每一家公司，不管他们是想造一个机器人在田里除草还是在高速公路上行驶，都有一个工程师团队试图弄清楚，这该死的东西在哪里？我们说:‘如果我们能把这个问题从桌面上拿下来，为所有这些用例通用地解决它，并真正让我们的客户专注于他们的应用，那会怎么样？’"

具体来说，Point One 的技术将增强全球导航卫星系统(GNSS)、计算机视觉和传感器融合结合到一个 API 中。这意味着，通过 API，无人机制造商、机器人初创公司或卡车运输公司可以知道其车辆的位置，精确到 10 厘米。第一点在 10 月份的自动赛车活动中展示了其技术，一辆自动驾驶汽车在雷小山赛道公园赛道上逆向行驶。

他们的产品将于 2020 年开始发货，已经吸引了许多投资者，他们认为这种产品在交通运输领域，甚至在包括消费电子产品在内的其他行业都有广阔的应用前景。 Point One 最近完成了由 UP 领投的 1000 万美元 A 轮融资。合作伙伴，现有投资者参与其中，包括 BOLT、IA Ventures 和 Ludlow Ventures。

如今的技术可以提供 Nathan 所说的部分定位服务。它可能告诉某人一个机器人在一个广阔的农田里的大致区域，或者它可能在一些区域工作而在其他区域不工作。Point One 能够为世界任何地方的客户提供 10 厘米的精度。

如今，大约一年前开始发货的 Point One 产品正在与两家未透露姓名的汽车客户进行批量生产。在这些汽车应用中，Point One 能够将这项技术部署为软件产品，因为如今的新车已经具备所需的硬件。对于其他客户，比如说一家摩托车公司，Point One 可能也需要包含芯片组。

该公司最初专注于汽车，包括应用于 2 级高级驾驶辅助系统。这家 20 人的公司现在正专注于拓展新的细分市场，包括微型汽车。Point One 还与农业客户——想想“智能”拖拉机——以及无人机送货部门达成了生产协议。

它也瞄准了 Nathan 所说的“新兴开发人员”，即那些从事可能还没有完全开发出来的产品的人。

“问题是我们如何非常迅速地扩张到这些市场，并确保每个人都意识到，第一点可以为你解决这个问题——你不必煮沸那部分海洋，”内森说。
