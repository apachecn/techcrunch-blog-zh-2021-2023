谷歌全力以赴将人工智能带入工作空间

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/14/google-goes-all-in-on-bringing-ai-to-workspace/

谷歌和微软陷入了一场针锋相对的竞争，试图将尽可能多的生成性人工智能引入他们的生产力服务。就在微软“工作的未来”活动的前几天，谷歌今天宣布了对 Workspace 的全面更新，将把它的生成式人工智能模型带到其生产力套件的几乎每个部分，此外，新的开发人员解决方案将使谷歌的基础模型，包括其 5400 亿参数的 PaLM 大型语言模型，通过 API 和新的低代码工具提供给开发人员。

首先值得一提的是:目前，这些新功能只对谷歌所谓的“可信测试者”开放。尚不清楚它们何时会面向更广泛的受众推出。目前还没有定价信息，尽管听起来至少这些功能的一部分将向消费者提供，包括 Google One 计划中的那些功能。基本上，这类似于谷歌几周前宣布的 LaMDA:它们听起来很棒，但要在实践中尝试这些还需要一段时间。

图片来源:谷歌

谷歌的计划是将其生成性人工智能模型带到 Workspace 的几乎每个部分，无论是在 Gmail 中为你编写电子邮件，帮助你在 Docs 中编写(或重写)文档，在工作表中生成公式，在 Meet 中捕捉笔记，还是在幻灯片中创建文本、图像、音频和视频。如果要我猜的话，微软很可能会在本周晚些时候宣布非常相似的更新。

“Google Workspace 一直是实现实时协作的先驱，人们可以在我们的产品中实时合作，”Google Cloud 首席执行官托马斯·库里安在今天发布之前的新闻发布会上说。“下一阶段，我们将让人类得到人工智能合作者的支持，人工智能合作者将与人类实时合作。”

图片来源:谷歌

库里安说，该公司计划“每隔几周”迅速推出新功能。今年，受信任的测试人员将进入第一阶段，这将包括在 Gmail 中编写电子邮件和 Docs 中的文档方面获得帮助。目前，这里的核心功能是在写作和调整文本的语气和风格方面获得帮助。但正如谷歌所强调的，这仅仅是个开始(至少对于可信的测试人员群体来说)。

图片来源:谷歌

这里的承诺是，谷歌最终将把人工智能注入 Workspace 的几乎每一个工作流程。这可能意味着总结一次电子邮件对话或集体讨论一份文档。当然，因为大部分都是基于聊天的，谷歌聊天也将获得对其中一些功能的支持，但迄今为止，谷歌只说它将“支持在聊天中完成事情的工作流。”

图片来源:谷歌

然而，谷歌展示的最令人印象深刻的演示可能是将大量基于文本的工作与生成图像和音乐的人工智能模型相结合——并使用它来创建一个完整的演示。“通过幻灯片，我们可以帮助您从幻灯片中的文本生成见解和图像，”Kurian 解释道。“因此，你可以使用丰富内容库中的图像、你的品牌图像(如果你有自己公司的图像)和私人存储库来生成这些图像，并赋予每个人用幻灯片构建的超能力，包括生成配乐。”

图片来源:谷歌

当被问及定价时，Kurian 指出，该公司计划“将这些解决方案广泛提供给我们的企业、小企业、消费者，甚至是订阅 Google One 的人。”这就是我们目前所知的全部，尽管提到 Google One 是一个有趣的事情，因为谷歌一直在推动其订阅服务——其主要优势是提供额外的云存储空间——随着时间的推移，慢慢包括更多的功能，包括其 VPN 服务。

众所周知，谷歌正在全力以赴地将生成性人工智能引入其产品的最大范围。有点遗憾的是，今天的许多公告感觉像是蒸汽软件，目前只有少数用户可以访问它。虽然像智能撰写和智能回复这样的功能显然已经广泛可用(并且非常有用)，但谷歌似乎仍然不愿将其最新的人工智能技术放到日常用户手中。

不过，这里的策略感觉是对的。谷歌工作空间为该公司向用户提供各种人工智能技术提供了最大的画布，Gmail 和 Docs 是这里最容易实现的成果，因此谷歌从那里开始是有意义的。这也是大型文本模型大放异彩的地方，同时谷歌可以避开 Bing 和 ChatGPT 面临的聊天机器人脱离脚本的问题(尽管微软似乎目前已经控制住了这一点)，并且不会给其搜索业务带来任何风险。
