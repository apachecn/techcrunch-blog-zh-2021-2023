无论好坏，特斯拉的机器人战略与其自动驾驶战略密不可分 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/09/30/teslas-robot-strategy-is-inextricably-tied-to-its-autopilot-strategy-for-better-or-for-worse/

特斯拉在周五发布了其第一个擎天柱人形机器人原型——这次是一个真正的机器人，按照最严格的定义，而不是一个穿着奇怪衣服的有血有肉的人类。根据特斯拉首席执行官埃隆·马斯克的说法，机器人执行了一些基本功能，包括行走一点点，然后举起双手——这是第一次没有支撑或起重机。

该公司可能正在向人形机器人迈出第一步，但它在业务上有很多依靠。马斯克曾表示，Optimus bot 最终将比汽车业务更有价值，价值超过 FSD(特斯拉的附加“完全自动驾驶”功能，这不是自动驾驶。)

周五晚上的活动中显而易见的是，特斯拉正在做出一个经济上明智但战略上有问题的决定，即将 Optimus 和它的 Autopilot(以及 FSD)野心的命运捆绑在一起。

特斯拉表示，它能够在机器人领域如此快速地前进的原因是，它已经在试图开发车辆自动驾驶的工作中打下了很多基础。

“考虑一下。该公司的一名工程师解释道:“我们只是从轮子转移到了腿。”。“所以有些组件非常相似[ … ]这是完全相同的占用网络。现在，我们将在稍后与自动驾驶团队讨论更多细节[……]唯一真正改变的是训练数据。”

这是整个演讲中反复出现的主题，来自特斯拉的各种演讲者(该公司展示了许多可能被认为是主要宣传为招聘活动的活动)提出了研发这两个领域实际上有多么紧密的联系。

事实上，特斯拉在活动舞台上展示的机器人只是一个非常简短的演示，几乎无法与多年来其他公司的大量人形机器人演示相提并论，当然也没有超过这些公司，包括最著名的波士顿动力公司。FSD 和 Optimus 之间的联系充其量也只是一种脆弱的联系。

领域专业知识，虽然被 Tesla 的介绍简化为简单的翻译，但实际上是相当复杂的一个。在步行路线上导航的两足机器人与自主车辆路线是非常不同的动物，过度简化这种联系不利于该主题的大量现有研发工作。

特斯拉的演示者一直在 Optimus 和其车辆的自主导航功能之间进行相对无缝的过渡。Optimus 的主要发言人之一是该公司的自动驾驶软件工程总监米兰·科瓦奇，他交给了自动驾驶总监阿肖克·埃卢斯瓦米(Ashok Elluswamy)，让他进一步深入研究特斯拉汽车的自动驾驶问题。

很明显，特斯拉认为这是一个相关的挑战，将导致市场在追求这两个问题时会欣赏的效率。现实是，要真正阐明这种联系不仅仅是表面上的，还有很多令人信服的工作要做。

更不用说，Autopilot(更具体地说，FSD)面临着来自公众和监管质疑和审查的挑战。一个每天和你生活在一起的机器人不需要这种潜在的风险。

特斯拉可能已经将它的西装男变成了一个真正的机器人，拥有真正的执行器和处理器，但它仍然有一段路要走，以实现它的承诺，这是一个价格低于 20，000 美元的可行产品，我们任何人都可以购买。
