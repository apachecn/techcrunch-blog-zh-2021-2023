拐点为其第二只专注于加密的基金 TechCrunch 筹集了 4000 万美元

原文：https://web.archive.org/web/https://techcrunch.com/2022/01/19/inflection-raises-40-million-for-its-second-crypto-focused-fund/

正如该公司在其投资论文中所言，早期风险公司拐点已经筹集了一只新基金投资于“开放经济”。用这个术语，公司的意思是它计划用它的 4070 万美元的新基金覆盖所有东西 crypto，web3 和区块链。

Mercury 基金的有限合伙人包括 Accolade Partners、Evanston Capital、异构体、Hutt Capital、Multiple Capital、Christian Angermeyer 的 Presight Partners、Galaxy Digital、DCG、洛克威、马克·安德森、克里斯·狄克逊、邵伯和埃里克·沃尔赫斯。

拐点计划为新的加密项目开出第一张风投支票——投资额可能在 50 万美元到 150 万美元之间。虽然该公司已经建立了一个以美元计价的传统基金，但它计划以不同的方式配置其资本。除了传统的股权融资，Inflection 还计划在私下销售中购买代币，但也通过购买公开上市的加密资产押注于一些加密货币。

拐点已经出现好几年了。该公司尚未披露其初始基金的规模，但已经全部投资到位。来自基金 1 的一些投资包括受欢迎的自动做市商平衡器，去中心化 Git 平台 Radicle 和专注于隐私的概念替代方案 Anytype 。

拐点基金也已经开始投资新的水星基金。组合公司包括不可阻挡的金融、紫罗兰、定义的和目录。有趣的是，拐点将开放经济定义为五个不同的垂直领域——开放资金、开放金融、开放数据、开放媒体和开放组织。下面是它在图表中的样子:

虽然拐点团队仍然相对较小，只有四名全职团队成员，但该公司不打算只签支票。“我们运行节点，管理加密网络，编写代码和扩展 Dao，”拐点团队在其网站上写道。有趣的是，在这种描述中，我们可以看到秘密创投公司与传统创投基金的不同之处。

亚历山大·兰格是该事务所的创始人和普通合伙人。Jonatan H. Luther-Bergquist 担任该公司的研究主管。他们都在德国工作。风险合伙人约翰·赞诺斯(John Zannos)以及公司的运营团队成员都在马萨诸塞州的波士顿。拐点基金主要投资于欧洲和美国，但也有全球授权。
