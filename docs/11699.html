Momento 秘密推出无服务器缓存| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/11/02/momento-launches-out-of-stealth-with-a-serverless-cache/

在美国宇航局担任火星车机器人专家后，Khawaja Shams 经历了职业生涯的转折，加入了 AWS，与工程师 Daniela Miao 合作开发 DynamoDB，这是一个完全托管的 NoSQL 数据库服务。Shams 和 Miao 并不满足于此，他们离开 AWS，共同创立了总部位于西雅图的初创公司 Momento ，这家公司如今正悄然崛起，拥有针对云计算优化的“无服务器缓存”。

你会问，什么是无服务器缓存？Shams 将其描述为一种弹性的“高可用性”缓存，可以更快地将常用数据传输到应用程序和数据库。他声称，Momento 的平台允许开发人员用大约五行代码向他们的云堆栈添加缓存，加速在公共云中运行的数据库，如亚马逊网络服务或谷歌云。

“传统缓存提供商提供了一种不同的模式，要求客户为其峰值容量进行配置和付费。这些缺乏弹性的服务非常低效、昂贵，需要大量的工作才能做好，而且根本无法扩展，”Shams 在一次电子邮件采访中告诉 TechCrunch。“Momento 允许客户通过一个 API 调用来调配一个每秒能够处理数百万次交易的安全缓存。”

Shams 认为传统的缓存服务复杂而低效，迫使工程团队浪费时间修补过多的配置。此外，他说，因为它们不会自动扩展，所以需要工程师为高峰使用量做准备，这导致了浪费性支出。

相比之下，Momento 在抽象配置的同时处理负载峰值。该平台可自动优化、扩展和管理缓存，并通过端到端加密和审计日志支持来保护缓存。

“云计算让客户比以往任何时候都更容易快速调配资源。遗憾的是，在近年来以增长为中心的阶段，客户最终大规模过度配置容量，并与云提供商的大额账单作斗争，”Shams 说。“Momento 正在帮助客户优化他们云账单上的一个主要项目。”

Shams 指出，在疫情期间，随着数字化转型工作的加速，一些企业的云成本飙升。分析平台 Anodot 的 2022 调查显示，近一半的企业(49%)发现很难控制他们的云成本。根据 Flexera 的另一项调查，超过 50%的公司现在每年在公共云上花费超过 240 万美元。

Momento 声称已经与包括 CBS、NTT Docomo 和智能家居公司 Wyze Labs 在内的客户达成了“多笔六位数的交易”。这家初创公司获得了包括贝恩资本风险投资公司(Bain Capital Ventures)在内的风投的早期信任票，贝恩资本风险投资公司领导了 Momento 的 1500 万美元种子融资轮，该轮融资于本周结束。普通合伙公司与 Flickr 首席执行官唐·麦卡斯基尔、前 Mozilla 首席执行官约翰·莉莉和其他天使一起参与。

Shams 表示，这笔资金将用于扩大 Momento 的 25 名员工的工程团队，建立一个“全周期”的上市团队，发展 Momento 平台，并增加对其他公共云的支持。
