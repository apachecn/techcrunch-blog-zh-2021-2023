Polymath Robotics 发布，将即插即用的自治软件应用于任何工业车辆 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/07/29/polymath-robotics-launches-to-bring-plug-and-play-autonomy-software-to-any-industrial-vehicle/

“机器人真烂。”

对于一家旨在与机器人合作的初创公司创始人来说，这是一个大胆的宣言——或者更准确地说，是一种帮助将拖拉机、舵柄或叉车变成自动化车辆的软件。但是 Stefan Seltz-Axmacher 试图证明这一点，他之前创建并领导了现已关闭的自动驾驶汽车初创公司 Starsky Robotics。

Seltz-Axmacher 在最近的一次采访中说:“它们真的很难，它们总是出故障，要获得一个稳定的产品真的很难。”。"几乎每个人都会为几乎所有的应用程序从头开始构建几乎所有的东西."

让事情变得更加复杂的是，用于仓库、采矿、农业和其他工业环境的机器人有着超特定的应用，这些应用是结构化的，并且经常重复数千次。换句话说，爱荷华州的农民、佛罗里达州的货场卡车运营商和在全国拥有 100 个仓库的电子商务巨头有着其他人没有的特定需求。

这就是塞尔兹-阿克斯马赫、联合创始人伊利亚·巴拉诺夫和他们的新创公司 Polymath Robotics 希望进入的领域。两人开发了一个即插即用的软件平台和配套的 SDK，使公司能够快速、经济高效地实现工业车辆的自动化。可以把它想象成工业机器人的 SaaS。

图片来源: Polymath 机器人公司

博学者机器人公司于周五亮相，是 Y Combinator 的 2022 年夏季团队，其目标是成为机器人世界的先知。这家初创公司正在建立基本的通用自动驾驶系统，旨在自动化今天在封闭环境中运行的大约 5000 万辆工业车辆。

这家总部位于旧金山的初创公司的软件与硬件和商业模式无关，专注于公司运行自动化机器人、拖拉机或叉车可能需要的所有功能，包括路径规划、危险检测、行为树、人类检测、控制调整和安全。

在巴拉诺夫(他是首席技术官，之前在 Clearpath Robotics 和亚马逊实验室 126 领导机器人团队)的领导下，Polymath 也创建并发布了一个名为 Caladan 的免费工具，允许用户在该公司软件的基础上进行模拟。据该公司称，与其他模拟人生不同，这可以在互联网浏览器上查看和创建， 不需要安装其他工具，如 ROS、Gazebo 甚至 Linux。

Polymath Robotics 软件平台让另一家初创公司、仓库所有者、农民或矿业公司跳过了构建 autonomy、安全层和前端应用程序的漫长过程。Seltz-Axmacher 表示，该软件让这些用户只需关注该应用程序，连接到其 REST API，并在 sim 中命令虚拟拖拉机、叉车或其他类型的机器人。

Polymath 已经在无人驾驶，并与潜在客户合作。但是有兴趣了解它如何工作的技术团队可以通过它的 API 开始免费构建模拟。

Polymath 的 API 告诉机器人要做什么，无论是在模拟工具中还是在现实世界中。例如，这位 TechCrunch 记者能够坐在亚利桑那州的办公桌前，通过互联网浏览器控制位于加利福尼亚州莫德斯托尘土飞扬的停车场的拖拉机。

当然，仅靠软件无法将拖拉机变成无人驾驶的自动化车辆。Polymath 已经与爱达荷州的初创公司 Sygnal Technologies 合作，通过提供线控驱动套件的改造来帮助硬件方面的事情。

Sygnal 的首席执行官兼联合创始人 Josh Hartung 对自动驾驶汽车和线控系统略知一二。他之前的初创公司 PolySync 已经关闭，它开发了一个自动车辆软件平台和一个线控套件，许多其他初创公司在自己的 AV 演示中使用了该套件。

这一次，Hartung 和他的联合创始人 Trey German 开发了一个线控套件，该套件设计有加速器、制动和转向的冗余控制以及专有的开关技术。它是为车队和商业部署而构建的，而不是为演示而构建的。在经历了多年围绕 AVs 的大肆宣传，并看到包括他自己在内的无数初创公司陷入演示和概念验证的循环中以获得资金后，Hartung 相信该行业最终正在向现实转变。

“我相信自治的下一阶段实际上是回归商业原则，”哈东说，并补充说，博学者似乎很好地适应了这种转变。

似乎许多天使投资者已经注意到了 Polymath 的 10 人团队。

虽然 Seltz-Axmacher 还没有准备好分享该公司筹集的资金，但他列出了该公司的一些天使投资者，他们都有自动驾驶汽车技术、软件和机器人方面的背景。据 Seltz-Axmacher 称，该集团包括 Catapult Ventures 董事总经理达伦·利卡多(Darren Liccardo)、Thursday Ventures 普通合伙人马特·斯威尼(Matt Sweeney)、Cruise 联合创始人兼首席执行官凯尔·沃格特(Kyle Vogt)以及 Voyage 前联合创始人兼首席执行官奥利弗·卡梅伦(Oliver Cameron)。

“我们希望让机器人在进出速度方面看起来更像 SaaS，”他补充道。

Sweeney 曾在 Neuralink 工作，在创办 Thursday Ventures 之前是优步的产品和工程主管，他认为这家初创公司在正确的时间拥有正确的产品。

Sweeney 说:“这种方法的吸引力在于，我可以看到一个类似甲骨文的机器人公司的未来。“各种各样的企业都来 Oracle 寻求适合其业务的硬件和软件解决方案，只需进行少量配置，就可以将这些解决方案融入到他们的业务中。如果你预测未来 10 到 15 年，这将是一个非常大胆的目标，但我认为它也可以一直持续提供价值。”
