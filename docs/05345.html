Stackblitz 筹集 790 万美元为你的浏览器带来更好的 IDE

原文：https://web.archive.org/web/https://techcrunch.com/2022/04/06/stackblitz-raises-7-9m-to-bring-a-better-ide-to-your-browser/amp/

图片来源:马斯科特/盖蒂图片社

StackBlitz 是一家专注于开发人员的初创公司，使用 WebAssembly 和 WebContainers 为您的浏览器提供完整的开发环境，今天宣布它已经筹集了 790 万美元的种子资金，由 Greylock 牵头，GV、GitHub 联合创始人 Tom Preston-Werner、atSpoke 联合创始人 Jay Srinivasan 和 Appurify 联合创始人 Pratyus Patnaik 参与。

在其核心，StackBlitz 在浏览器中给你一个完整的开发环境。该服务目前支持类似 Next.js、Nuxt、Node.js、React 和 Angular 的 JavaScript 邻近框架。代码编辑器由 Visual Studio 代码的精简版本提供支持，但它很快将在浏览器中运行微软开放源代码编辑器的完全可扩展版本，这将消除当今 StackBlitz 编辑器的限制。开发人员可以在几秒钟内开始一个空白项目，连接到他们在 GitHub 中的代码库来处理现有项目，甚至只是处理他们的本地文件。

图片来源: StackBlitz

Stackblitz 由埃里克·西蒙斯 (CEO)和艾伯特·帕伊 (CTO)共同创立。如果西蒙斯的名字听起来很熟悉，那么他就是那个在他 19 岁的时候，蹲在美国在线总部的家伙。Thinkster 提供在线编程课程和教程，重点是全栈开发。在这个过程中，他们意识到当你想教某人如何编程时，建立一个开发环境是多么困难。“这超级不靠谱，”西蒙斯说。“这完全是一种痛苦。简直是噩梦。就像是:这怎么还没解决？”

毫不奇怪，这正是 StackBlitz 试图解决的问题。“[在 Thinkster]我们正在教授浏览器能做什么的前沿技术，”他解释道。“我们意识到，理论上应该可以在浏览器选项卡中实际运行开发环境，就像在浏览器选项卡中运行操作系统一样，这样您就不必安装任何东西，也不必启动服务器。”

图片来源: StackBlitz

Simons 指出，该团队花了三年时间构建 WebContainer 技术，使您可以在浏览器标签中运行这个基于 WebAssembly 的系统，并在几毫秒内启动它。“这也意味着我们可以免费赠送，考虑到我们的出身，这对艾伯特和我来说在精神上是很重要的，”西蒙斯解释道。“而且，我们现在每月有 200 万开发人员使用 StackBlitz，我们的 AWS 账单是几百美元，因为所有的计算都被转移到了边缘。”

StackBlitz 团队还认为，使用其服务可以增强安全性，因为代码很难突破浏览器沙箱。Simons 将该服务在企业中的成功部分归功于这些固有的安全特性。

虽然 StackBlitz 应该为教授编程创造了一个良好的环境，但值得注意的是，该公司并没有直接追逐这个市场。到目前为止，这似乎是可行的。Simons 说，在测试阶段，StackBlitz 看到了来自 2000 多家公司的开发人员使用它的平台，他还看到了来自财富 500 强企业的强烈兴趣。该公司的企业收入在 2021 年增长了 10 倍，员工人数翻了两番，达到 20 人。

至于服务的货币化，StackBlitz 正在遵循 GitHub 的剧本，免费服务只允许公共项目，并与公共 GitHub 回购同步，每月 9 美元起的付费层也允许私人项目。
