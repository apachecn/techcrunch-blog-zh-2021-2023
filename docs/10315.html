“电话计划条纹”:Gigs 筹集 2000 万美元，帮助任何公司成为 MVNO TechCrunch

原文：https://web.archive.org/web/http://techcrunch.com/2022/09/21/a-stripe-for-phone-plans-gigs-raises-20m-to-help-any-company-be-an-mvno/

一家新创业公司希望让任何公司都能更容易地将电话和数据计划作为其自有品牌移动网络订阅的一部分进行销售——为了提供帮助，它已经获得了知名投资者的支持，包括谷歌的早期风险投资部门 Gradient Ventures 和优步首席执行官达拉·科斯罗萨西(Dara Khosrowshahi)。

Gigs 于 2020 年在德国成立，将自己定位为“手机计划的条纹”，迄今为止基本上没有引起人们的注意。然而，这家总部位于柏林的初创公司去年完成了 Y Combinator 的加速器项目，并在今天宣布的 2000 万美元首轮融资之前获得了约 400 万美元的资金(通过可转换贷款)。

简而言之，Gigs 允许任何公司——无论是银行、打车公司还是视频流媒体服务——向其客户销售手机订阅计划(包括数据、短信和语音)。这些计划完全可以针对特定的用例进行定制；例如，一家零售连锁店可能希望推出一个拥有自己品牌的全功能移动网络，或者一家支持 4G 的可穿戴设备制造商可能希望在他们销售的每台物理设备上实现数据订阅的货币化。或者，一家公司的人力资源部门决定将自己的手机计划与员工的设备一起发布。

当然，我们谈论的是移动虚拟网络运营商( MVNO )，全球已经有很多这样的运营商了。

例如，在美国，有基于 T-Mobile 和美国蜂窝网络的 Google Fi、和基于 T-Mobile 基础设施的瑞安·雷诺兹支持的 Mint Mobile 。在世界其他地方，有Aldi Talk——由德国超市巨头 Aldi 提供支持的 MVNO——在少数市场依赖 Telefónica 的网络，而英国有数十个 MVNO从该国四大主要运营商租赁频谱。

盒子里的 MVNO

但是，尽管任何公司都有可能成为 MVNO，但这通常是一个艰巨而耗时的过程——最终几乎没有灵活性。这就是演出进入竞争的地方。

今天，任何想要提供电话计划(即成为 MVNO)的实体都必须与美国的大型电信提供商谈判条款，其中包括美国电话电报公司、T-Mobile 和威瑞森，这不仅是一个昂贵的过程，而且需要大量的技术准备工作，包括网络集成和必须开发管理用户订阅的软件。

另一方面，Gigs 将所有电信运营商的 API(应用编程接口)整合到一个易于访问的层中，降低了准入门槛。

Gigs 首席执行官赫尔曼·弗兰克(Hermann Frank)向 TechCrunch 解释说:“通过 Gigs，公司将能够以低得多的成本在他们经营的任何市场提供电话计划，所有这些都是在 2 到 14 天内通过相同的集成实现的。”“最终，通过建立在 Gigs 的基础设施上，企业可以创建自己的移动服务，这种服务无缝地符合他们的品牌身份，速度快 20 倍，更简单，更便宜。”

Gigs 联合创始人丹尼斯·j·鲍尔和赫尔曼·弗兰克。图片来源:演出

Gigs 能够通过购买大量的数据、语音和短信容量来提供这种服务，然后根据客户在其经营的市场中的需求来分配这种容量——这比一家公司仅仅根据自己的需求来谈判价格要便宜得多。

“然后，我们可以根据客户的需求，以自己的价格制定自己的计划，并创建定制的套餐，”弗兰克说我们也有市场标准的计划，你会发现在其他运营商，这可以很容易地由我们的客户转售有吸引力的利润。"

模拟城市

虽然 Gigs 提供传统的“物理”SIM 卡作为白标产品，但现代嵌入式 SIM (eSIM)卡的出现使事情变得更简单，因为它允许公司在任意数量的设备上实时提供虚拟 SIM 卡。最新的 iPhones 手机在美国市场甚至没有实体 SIM 卡插槽，这就是为什么 Gigs 至少有一只眼睛关注未来，它支持没有任何实体足迹的数字 MVNOs。

事实上，该公司目前通过允许最终用户扫描简单的 QR 码来激活 eSIM。通过同时支持 SIM 和 eSIM，Gigs 可以有效地应对其进入的所有市场。

Frank 解释说:“与运营商整合、销售手机套餐的流程和障碍，以及我们提供手机套餐的方式，对于物理 SIM 和 eSIM 都是一样的。“[但是] eSIM 现在取消了将物理卡放入设备的最后一步，从而简化了激活过程。”

Gigs SIM 和 eSIM。图片来源:演出

除了主要的 API，该公司还提供了一个名为 Gigs Connect 的软件套件，根据 Frank 的说法，这基本上是一个“为高端用户转换而优化的”托管结账。可以使用粘贴到客户产品(例如，销售智能手表的在线商店)中的简单链接来嵌入该结帐。

这显然符合所有人的利益——它越容易让自己的客户销售智能手机、可穿戴设备或物联网设备的计划，Gigs 及其客户就能获得越多的收入。

音乐会:在电话和智能手表上展示计划。图片来源:演出

另外，Gigs 还为中小型企业提供了一个名为 Gigs Teams 的电话计划和设备订阅管理平台，以及一个在单一界面中为客户提供所有订阅、支付和分析的完整视图的仪表板。

Gigs 仪表板。图片来源:演出

这一切背后的精神实质与金融科技巨头 Stripe 如何通过一套简单的 API 服务支付基础设施来帮助商家在线销售，甚至亚马逊网络服务(AWS)如何成为数百万人的默认云计算基础设施非常相似。

这都是关于无差别的重担——允许公司增加其主要产品或服务的价值，而不失去对其核心竞争力的关注。根据 Frank 的说法，对于 Gigs，这意味着通过抽象出成为全球网络运营商通常涉及的所有复杂性，将其减少到仅仅五个 API 调用，来为嵌入式电话和数据计划提供动力。

“Gigs 正在创建电信即服务类别，”他说。“我们是第一个像 Stripe 和 AWS 那样做支付和托管的公司。”

市场机会

去年，MVNO 市场被认为是一个价值 620 亿美元的产业，这个数字预计在五年内将超过 910 亿美元。但是，这还没有考虑到可以在一夜之间开店的新型 MVNO，所以很难评估潜在市场到底有多大。

弗兰克说:“我们采访过的各行各业的许多公司都考虑过推出自己的 MVNO，或者收购一家 MVNO，通常是在他们尝试与运营商合作一年多之后。”“理论上，你可以通过 MVNO 轻松开启新的收入流，但事实证明，进入壁垒——与运营商的漫长谈判、非常高的设立成本和承诺——对大多数企业来说是不可逾越的。”

当一家公司希望在全球多个市场推出其 MVNO 时，这变得更加复杂。

“你将不得不在每一个市场中克服同样的进入壁垒，这将是一个漫长而成本高昂的过程，”弗兰克继续说道。“有了 Gigs，您可以通过单一集成管理所有市场的所有连接需求，并在几天内启动自己的 MVNO。”

也很难忽视 Gigs 的明星投资者。除了主要支持者 Gradient Ventures 和优步的掌门人 Dara Khosrowshahi，YC 还通过其后续的 YC 连续性基金进行了再投资，同时还有 DoorDash 首席执行官许宏涛、Instacart 首席执行官 Fidji Simo 以及来自整个技术领域的一系列天使投资人。

这也有助于突出 Gigs 的核心目标市场，它将主要围绕美国公司，尽管它对其他地方的业务非常开放。该公司的核心 API 今天正式退出测试，到目前为止，它仅限于“精选合作伙伴”，其中 70%位于美国，20%位于欧洲，10%位于亚洲。

同样值得注意的是，正如如今几乎所有的初创公司一样，Gigs 是一家远程优先的公司，30%的员工在德国，30%在美国，20%在英国，还有 20%分布在世界上的少数几个国家，包括意大利、格鲁吉亚、希腊、瑞士和南非。

“我们团队的大部分成员都是美国人——要么在美国工作，要么在欧洲工作，”弗兰克解释道。“对于大多数科技企业来说，美国仍然是最重要的单一市场，拥有最多样化的科技场景，这对于演唱会来说也不例外。”
