全球动荡将如何影响 2023 年的创新

大卫·马格曼撰稿人

过去一年的全球经济和政治动荡对科技行业及其他领域的企业创新产生了重大影响。

与 COVID 的全球斗争、乌克兰-俄罗斯冲突以及 COVID 封锁和供应链中断的经济影响共同造成了全球经济衰退、全球通胀和不可预测的不稳定的痛苦组合。

所有这些因素导致企业界勒紧裤腰带，裁员和冻结招聘，投资界也采取了更加保守的投资姿态。不可避免的是，这些变化将对未来几年的创新产生寒蝉效应。

然而，当谈到创新的前景时，也许还有一线希望。从某些方面来说，这些市场力量实际上可能成为创造力和技术进步的催化剂。

在这种环境下，购买和整合可能比从头开始构建更容易。

短期影响

在短期内，这些负面经济趋势和政治不稳定的影响将被企业和初创企业的创新中心所感受到。

企业可能会大幅削减内部和外部创新的支出。也就是说，他们将减少研发预算，并可能以牺牲长期愿景项目为代价，将 R&D 的重点放在能够对盈利能力产生直接影响的项目上。

企业也将减少与其他创新者的合作以及昂贵的先进技术收购。我们预计将会看到更多对早期公司的收购，因为它们变得越来越弱，企业希望通过打折购买而不是从头开始开发新技术，从而以更低的成本开发新技术。
