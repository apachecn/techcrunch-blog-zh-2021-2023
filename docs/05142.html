印度主导 Y Combinator 的最新创业批

原文：https://web.archive.org/web/https://techcrunch.com/2022/03/29/y-combinator-india-winter-2022/

再一次，印度是除美国之外，在最新的 Y Combinator 加速器批次中最具代表性的国家:2022 年冬季队列运动有 32 家创业公司，分别来自古鲁格拉姆、本加卢鲁、德里、新德里、海德拉巴、孟买和钦奈。值得一提的是，超过 191 家印度公司通过 Y Combinator 加速器获得了资助，其中近一半——一半！—在过去 12 个月中接受的公司中。

正如我们将在下面讨论的，这批公司计划应对科技领域的各种挑战，但似乎主要集中在金融服务领域。想想“现在购买，以后支付”的宣传，以储蓄为重点的新银行，当然还有比特币赌博。这与之前的展示形成了鲜明对比，在之前的展示中，大多数印度 YC 初创公司都属于 B2B 服务类别，但正如我们所见，金融科技公司在估值和投资者兴趣方面着火了，尽管市场出现了更广泛的调整。

其他值得注意的趋势包括一些来自 Accel Atoms 的公司，以及许多许多 IIT 大学的毕业生，Accel Atoms 是一个为印度早期创业者提供的种子前项目。

以下是所有 W22 初创公司的列表:

团队:

我们的想法:个人理财显然在国际上举足轻重，然而储蓄并不是什么新鲜事。费罗需要一个巨大的差异化优势，以便在竞争激烈的环境中赢得粘性牵引力，无论这是一种易于获得的用户体验，还是一种宣传其愿景的聪明方式。

他们正在构建:Telematica 自诩为“电动汽车的格子花呢”，它正在构建一个 API 工具，将云带入蓬勃发展的汽车订阅行业。使用 Telematica 的公司可以访问汽车的远程信息处理、电池和其他移动数据，并通过一个 API 控制电动汽车充电。

我们的想法:该公司声称印度的电动汽车市场正在增长，预计到 2030 年，超过 40%的新车销售将是电动的。如果是这样的话，早期押注于使这些汽车工作并能解决其维护和燃料需求的 API，肯定会是一个很大的机会。

他们正在打造的东西:一家专注于经济实惠的医疗保健公司。这家初创公司希望在医疗账单和其他大额支出方面，给更多消费者提供“先买后付”的选择。

我们的想法:在美国，这些年来还款服务出现了许多波动，尤其是在医疗保健方面。这是有道理的:基于一项简单服务的昂贵程度，账单是令人困惑的、过时的，而且通常是无法获得的。SaveIN 希望将这种模式引入印度，因为印度在计费和还款方面也有自己的挑战。SaveIN 的重点应该是获得供应商的信任，因为它寻求优化几十年来的低效率。

关键引言:“作为一名优步工程师，我想测试当驱动程序分配 API 失败时会发生什么，是自动重试还是我们要求用户重试，或者应用程序崩溃，”该公司在 Y Combinator 的博客上说。

我们的想法:在过去的几年里，API 初创公司已经爆炸式增长(并因此变得支离破碎)。Requestly 很聪明，它试图从一个更高的层面带来某种统一和方便，来管理这些不同的工具。
