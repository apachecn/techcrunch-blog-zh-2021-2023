一些投资者正在(谨慎地)将 ChatGPT 应用到他们的工作流程中

原文：https://web.archive.org/web/https://techcrunch.com/2023/01/10/some-investors-are-cautiously-implementing-chatgpt-in-their-workflows/

新的人工智能工具 ChatGPT 凭借其聪明的消息传递能力获得了病毒式传播，这无疑引起了共鸣。这个工具上个月刚刚向公众开放，它足够智能，可以回答关于深刻主题的严肃和愚蠢的问题，这使它陷入了由作家、教育家、艺术家等领导的辩论中。

但对于投资者来说，ChatGPT 的飙升不仅仅是激励他们运行和支持下一个大型人工智能工具。一些人正在想办法将技术整合到他们的工作流程中，以更好、更智能甚至更便宜的方式完成工作。

许多投资者对人工智能取代他们工作中更单调的部分的想法不屑一顾。毕竟，在一个由附加值和个性驱动的行业，谁会愿意承认人工智能可以做他们的工作？

“不过，写拒绝信是风险投资中最艰难、最耗费情感的部分之一，所以我可以理解为什么一些投资者可能会试图外包给 ChatGPT，但出于对创始人的尊重，我们不会这么做。” SignalFire 的乔希·康斯汀

但是抛开名声不谈， venture 多年来一直试图实现自动化，无论是在交易发现方面，还是在投资支持方面。但 ChatGPT 是一个专门基于文本的支持工具，自动化可能会成为拒绝信、市场地图甚至尽职调查的一部分——所有这些都是为了在不断变化的风险领域保持活力。

使用案例

Baukunst 的普通合伙人凯特·麦克安德鲁表示，鉴于人工智能是“片段的自然进化”，如果同事们不使用 ChatGPT，她会感到“震惊”至于她自己，她承认甚至没有使用谷歌的人工智能自动完成功能，从头开始写所有的电子邮件。她说这种方法很慢，“但对我来说很重要。”

一些投资者表示，ChatGPT 可用于围绕市场规模主张或增长潜力进行事实核查；与此同时，谷歌也可以。当然，人工智能的论点是，内容将是原创的，可能更针对某人的确切问题，而一般的谷歌搜索可能需要额外的挖掘和拼凑不同的文章。

Hustle Fund 的普通合伙人 Eric Bahn 是 ChatGPT 的支持者，上个月他用它为投资者写了一篇年终文章。他分享了他使用的提示和输出，并表示投资者对此反应良好。

Bahn 没有用它来做 pass 响应，尽管他觉得同事们可能会用它。
