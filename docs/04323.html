TechCrunch 前三名

  • 黑客泄露英伟达数据 :在一次网络攻击后，英伟达向 TechCrunch 证实黑客正在泄露其数据。什么被拿走了？我们写道:“来自其网络的敏感数据，包括员工凭证和公司专有信息。”。这种性质的攻击并不罕见，但它们具有潜在的破坏性。
  • Alphabet 的无人机送货公司达到里程碑 :有点不为人知，但 Alphabet 的机翼部门一直忙于以越来越快的速度用无人机向客户运送货物。Wing 刚刚宣布达到 20 万的商业交付量，大约是在达到 10 万之后的半年。由于大部分航班都在澳大利亚进行，所以 Wing 今天也说它正在连接澳大利亚的连锁店，这并不令人吃惊。
  • 科技限制了俄罗斯政府的影响力:乌克兰战争远未结束，但俄罗斯政府推行自己政策的能力正在迅速减弱。 YouTube 今日屏蔽俄罗斯，脸书和 Instagram 限制俄罗斯国有媒体的覆盖范围，以过滤图片闻名的 Meta property在乌克兰和俄罗斯发布加密的 DMs。

创业公司/风投

一段时间以来，TechCrunch 一直在跟踪市场上通过 API 销售产品的初创公司数量的增长，现在是。应用编程接口第一的初创公司的繁荣恰好符合软件定价从传统的 SaaS 方法向更按需的模型的演变。无论如何，我写了一点关于 GGV 正在整理的一个新的 API 引导的初创公司的索引。作为一个引子，它包含了大约 84 万亿个 API 初创公司的链接，如果你想了解这个细分市场的话。

  • 乔拉希望切断与俄罗斯的关系 :如果你不是一家大型科技平台公司，很难构建一个操作系统。见鬼，微软用 Windows Phone 教给我们，即使你是一家平台公司，也很难。因此，当乔拉，这是建立一个“基于移动 Linux 的替代谷歌的安卓系统”，决定切断与俄罗斯的联系，在那里它有用户，这是一个实际的物质选择。
  • up beat 筹集 615 万美元确保您的视频带有音乐 :每天有大量视频被制作并上传到互联网上。如果你想从中赚钱，你通常会想避开音乐，因为包含音乐会让你的鸭子们上当。因此，乌普垮掉的建立了一个服务，提供免费的音乐给人们在他们的剪辑中使用。它已经发展到 500，000 个用户，我认为这是值得注意的。
  • Commsor 想超越资本主义来扩展社区 :我通常会改写时事通讯的标题，使其篇幅更长，并开一些小玩笑，但是娜塔莎·马斯卡雷纳斯的标题非常完美，所以我没有改动。Commsor 已经建立了她所说的“帮助其他初创公司管理社区的操作系统”。那么，每家初创公司都需要一个社区吗？不，事实证明。
  • Zero Systems 想要自动化专业服务 :我喜欢 Zero Systems 背后的想法。 Ron Miller 将其目标描述为将自动化引入“像律师事务所这样的专业服务”，这很有道理。律师事务所的真正价值不在于它有能力处理单调的文书工作。这是准备好的法律思维。那么，为什么不把其他的东西自动化呢？这家初创公司刚刚完成了一笔 1200 万美元的首轮融资。
  • Figma 将白板引入 iPad : Figma 价值100 亿美元，回想一下。
  • Instacart 失去了对初创公司的投资 :在疫情期间筹集了大量资金后，价值1.19 亿美元的 Instacart 有望在今年或明年上市。因此，看到它在这个过程中裁员并不奇怪，尽管该公司可能不会乐意失去金融科技团队中的某个人。无论如何，为政府开发支付技术的 feed 很可能对最近的偷猎行为火上浇油。
  • OpsLevel 为微服务管理筹集 1500 万美元 :我有点知道什么是微服务了。它是一个小的独立的应用程序单元，你可以把它串成一个更大的系统，对吗？我也这么认为不管怎样，如果你有很多，我估计很难把它们都整理好？这是微服务管理背后的想法，也是 OpsLevel 所做的。它刚刚筹集了资金来维持下去。

此外还有更多的 : 卫星初创公司 Vu 即将把技术送入轨道， Veev 刚刚为预制房屋筹集了 4 亿美元， Nayya 筹集了 5500 万美元，为医疗保健和其他福利提供建议，亚空间筹集了资金，以减少区块链的碳密集度，Starship Technologies——我想补充一句，这是一个非常好的名字——为一支地面机器人舰队筹集了 4200 万美元。我收回我的点名表扬！

10 位投资者探讨 Q1 2022 无码和低码景观

当我们在 2020 年 8 月发布上一次低代码/无代码投资者调查时，前总统已经决定禁止抖音，Epic 正在对苹果和谷歌提起反垄断诉讼，美国各地的电影院正在关闭，以减缓当时新颖的冠状病毒的传播。

好像是很久以前了。

从那以后，我们提出的许多关键趋势和主题都成为了现实:在 Salesforce Ventures 和迈克尔戴尔的 MSD 资本的帮助下，Airtable 在 2021 年 12 月筹集了 7.35 亿美元的 F 轮融资后，获得了 110 亿美元的估值。

为了不被超越，微软的 Power Fx 低代码编程语言现在连接了数百个应用程序。
