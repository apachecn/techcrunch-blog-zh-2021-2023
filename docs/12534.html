跟踪和捕捉:注意力指标入门

曾几何时，像网站停留时间和每次访问页面数这样的指标对于增长型营销来说已经足够了，但现在不再是这样了。

为了跟踪参与度和表现，品牌需要准确地知道他们的广告出现在哪里，以及消费者是主动还是被动地观看它们。

MediaMath 的数据和分析主管 Sylvain Le Borgne 表示:“可见性已经不够了，而‘注意力指标’在业内正变得越来越受欢迎。”。

“随着注意力指标和测量的不断成熟，品牌可以了解更多信息，制定与关键业务成果相一致的计划，开始优化注意力，并启动其目标和测量策略来吸引消费者的注意力。”

让您的产品和客户成功团队步调一致，提高净保留率

你不需要移动指针很远就能优化客户成功。不断进步的 SaaS 初创公司会积累一些小的成功，这些成功有可能改变公司的发展轨迹。

Elastic Path 的产品和客户成功 SVP 布莱恩豪斯写道，创始人应该考虑将他们的产品和 CX 团队整合成一个单位，以促进凝聚力，而不是使用传统的孤岛式方法。

“不管你把这个角色叫做什么，产品管理、UX 和客户成功都应该向同一个人汇报，”他建议道。

“这位领导者应该创建一个共享激励和共同目标的结构，与客户的成功衡量标准保持一致。"

给风险投资的支柱和未来——微型基金的一封情书

图片来源: Getty Images

Rebecca Szkutak 报告说，风险资本基金可以达到数十亿美元，但最大值为 5000 万美元或更少的较小资金池“才是推动行业未来的真正资金”。

去年，PitchBook 记录了美国境内 1472 亿美元的一般风险融资，其中小额基金筹集了近 170 亿美元。

PitchBook 高级风险分析师凯尔斯坦福(Kyle Stanford)表示:“(它们)通常是对一家公司的第一笔投资，这在生命周期中创造了更多机会。”。

“特别是当我们谈到有多少资本进入风险市场时，如果没有大量的小额基金，这是不可能的。”

在经济衰退期间，希望通过留住客户来推动增长

Zoho 首席战略官 Vijay Sundaram 表示，找到减少客户流失的方法是任何通过保留实现增长的战略的核心，但集中第一方数据是第一步。

他写道，将社交媒体、网站行为和 CRM 数据等不同信号编织在一起，将有助于识别高价值客户并推动参与，但“第一方数据可能存储在多个部门和系统中，包括营销、客户宣传、销售和支持”。

“建立丰富的客户档案和建立信息共享文化需要成为面向客户团队的核心职能。”
