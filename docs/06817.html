Ayoken 筹集 140 万美元发展其 NFT 创意市场 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/05/30/ayoken-raises-1-4m-to-grow-its-nft-marketplace-for-creatives/

Ayoken 是 NFT 的一个创意市场，已经筹集了 140 万美元的种子前资金，使用户能够通过数字收藏品增加收入来源。

这家初创公司的市场名为 Ayokenlabs ，将展示来自世界各地的音乐家、运动品牌和有影响力的人的数字收藏品。

Ayoken 创始人兼首席执行官约书亚·金告诉 TechCrunch，市场是粉丝和艺术家之间的桥梁，让支持者对他们偶像的成功有一种主人翁感。

他说，通过 NFT 市场，粉丝们将可以获得幕后视频和专辑封面等纪念品。NFT 持有者还将获得其他额外待遇，如获得未发行的音乐和创意人员的独家现场活动。

“通过 VIP 通行证，粉丝们将能够在这些艺术家的音乐到达 Spotify、YouTube 或 Apple Music 之前实际直播这些音乐。粉丝在未来的活动中也会得到折扣，”金说，他在战略、增长和创新咨询以及企业家方面有 14 年的经验。他的职业生涯包括帮助 scale AZA (Bitpesa)，这是一个总部位于内罗毕的平台，利用比特币促进跨境汇款，在那里他第一次接触到加密和区块链技术。

King 说 Ayoken 将在接下来的几个月里发布一些主要的非洲艺术家和世界上其他艺术家的 NFT。

这家总部位于伦敦的初创公司已经与加纳非洲裔艺术家 KiDi (Dennis Nana Dwamena)合作，在 6 月的第一天首次在 NFT 降落。金表示，跨链市场(尽管目前建立在雪崩区块链的基础上)允许加密和信用卡支付，但计划增加移动货币，因为这家初创公司让非洲等新兴市场的人们更容易轻松交易。金说，他们正在与非洲大陆的一些电信公司谈判合作伙伴关系，以实现这一目标。

“我们正在减少用户的摩擦点，让人们使用他们的卡，而不是必须使用加密来购买；我们正在与电信公司合作，未来也将允许人们使用移动货币进行支付。没有什么能与我们正在做的相提并论，这就是为什么我们能够签下创意产业中一些最大的名字，”他说。

当用户购买 NFTs 或推荐人时，他们将获得 token (Ayo)奖励，稍后他们可以兑换 NFT。

金说，与其他 NFT 市场不同，他们有包括 YouTubers、影响者、时事通讯、加密交易所和电信在内的分销合作伙伴来推广 NFT 滴滴——让创意人员能够利用更广泛的受众，而不仅仅是他们的粉丝群。

“这意味着名人不必依赖他们的社交媒体关注来推动交易。只需按一下按钮，他们就能立即联系到全世界数百万人。我们的方法与地球上任何其他 NFC 市场都不一样。我们也有一个营销机构来帮助这些创意人员在他们的第一次 NFT 投放中取得成功，”金说。

“他们(分销合作伙伴)将根据其社交媒体促销活动产生的任何交易获得收入分成。”

利用从投资者那里筹集的资金，其中包括创始人 Factory Africa、总部位于德克萨斯州的 Kon Ventures、总部位于欧洲的风险资本 collective Crypto League、总部位于加纳的 R9C Ventures 和 Maximus Ventures，Ayoken 计划与艺术家签署一系列独家协议，并与电信公司建立合作伙伴关系，此外还扩大其团队和二级市场。

“大部分资金将用于购买独家许可和建设我们的技术团队；这是开发人员和工程师的四倍，”他说。
