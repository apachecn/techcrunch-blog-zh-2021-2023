Box 正在添加免费的白板工具

原文：https://web.archive.org/web/https://techcrunch.com/2022/04/12/box-is-adding-free-whiteboarding-tool-for-collaborating-on-visual-content/

Box 正在添加免费的白板工具，用于视觉内容的协作

当你与人们谈论自 2020 年我们从家里搬到办公室工作以来，他们最想念办公室的什么时，人们经常提到在会议室与同事一起写白板，他们说在数字环境下很难做到这一点。许多公司都试图填补这一空白，包括初创公司 fill 和 Miro，后者在最近一轮融资中获得了 T2 175 亿美元的巨额估值。

今天，Box 宣布推出 Box Canvas，这款工具不仅可以让您进行虚拟白板式的头脑风暴，还可以让您在各种类型的视觉内容上进行协作，如产品工作流程或实体销售计划。

Box 首席执行官兼联合创始人 Aaron Levie 表示，他的产品与微软和谷歌的 office 套件集成，用于结构化文档协作，并且它也有自己的 Box Notes 原生工具，但其想法是将这种协作能力从结构化文档领域带到更可视化的内容。

“通过 Canvas，我们带来了同样的体验，但更多的是视觉协作，即虚拟白板类型的体验。我认为尽管这个领域已经见证了许多公司的伟大创新，但我仍然相信它在这个市场上实际上还处于非常早期的阶段。我认为我们才刚刚开始看到这种混合工作方式的潜力。”

图片来源:框

虽然 Box 将免费把 Canvas 捆绑到其新的套件产品中，但它也打算让它作为一个独立产品免费提供给任何想无限制使用它的人。他说，我们的目的是永远不为这项功能收费。

“我们认为，这个星球上的每个用户都希望通过他们的内容开展一系列核心活动。你会想要储存它，分享它，围绕它合作，让它签名。因此，我们希望向尽可能多的人提供这些核心功能，然后我们将根据您管理数据或使其符合特定行业的能力提供高级功能。这些将是非常先进的能力，随着时间的推移，我们将继续拥有越来越多的能力(我们将为这些能力向公司收费)，”他说。

产品导向的增长从公司开始提供免费盒子的早期就开始了，Levie 认为免费提供 Canvas 是这种想法的延伸，公司打算完全接受这种想法。

至于其他生产类似软件的公司，Levie 说这不是要和他们竞争，因为这是一个免费的附加软件。“坦率地说，我真的不认为它必须与任何人竞争，因为它是为了我们的客户。因此，如果您是一名 Box 客户，并且使用 Box 来管理您最重要的内容，这是完成工作的另一种真正有价值的方式。”

据该公司称，Box Canvas 将于秋季上市。
