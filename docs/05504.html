斯特林路公司对风险投资的新举措

原文：https://web.archive.org/web/https://techcrunch.com/2022/04/12/sterling-road-fund-3/

在风险投资的世界里，没有什么新鲜的东西——看看 20 家风险投资网站，它们几乎都没有什么区别。是的，他们投资最好的团队。是的，它们是“增值”是的，他们…天啊，这太无聊了，我想用可转换票据合约的锋利边缘来测量我的眼睛。Ash Rust 的 Sterling Road 是对这一切的全新诠释，其商业模式颠覆了一切。

该公司刚刚筹集了第三只基金，价值 2000 万美元，前两只基金分别为 300 万美元和 900 万美元。该公司在 B2B 公司的旅程刚开始时就关注它们，通常会向其投资的公司开出第一张机构支票。该基金规模相对较小，但它开出了 25 万美元的支票，并保留了一些用于后续投资，因此有足够的干粉用于相当数量的投资。

问题在于，该基金是先帮助，后投资。音高？投资者可以深入了解创始人是如何运作、思考和行动的。创始人尝到了与投资者合作的滋味。如果“我喜欢你，你喜欢我”进行得很好，这种结合随着对公司的投资支票而完成。

该公司的观点与大多数融资过程中快速推进、FOMO 驱动的方式截然相反。这总是意味着该公司错过了一些好的交易，但 Rust 声称这是一个功能，而不是一个错误。

“我认为，很多时间都花在了追逐最新的热门创始人上。他们下周一就要结束一轮谈判了，你必须尽快见到他们，做好所有的参考资料，”拉斯特哀叹道。“这些交易对我来说都是‘不’。这是一个尊重的通行证——我真的很抱歉，我帮不了你。恭喜你，但你太远了。”

该公司特别致力于多样化。

“在多元化方面，我们还有很多工作要做，但我们会努力投资那些历史上获得风险投资份额较少的创始人。这可能是因为种族、性别甚至地理位置，”Rust 说。“在我们的第二只基金中，65%的资金流向了传统上在科技领域代表性不足的创始人。我们将努力在基金 3 中做得更好。我想找到那些没有走过传统道路的创始人。我自己也是混血儿，但我是在一个白人家庭长大的。我很容易找到白人。它们整天都在我的收件箱里。如果我想找到背景迥异、观点与我不同的伟大创始人，我发现最好的办法是公开营销，在一个非常小众的领域，专注于早期创始人及其相关内容，而不是像《人物》杂志最近的 IPO 这样的热门话题。

该基金的中心是技术型创始人，他们在某个领域拥有专业知识，但没有运营初创公司的经验；团队告诉我，这是他们增加价值最多的地方。这通常意味着创始人不太熟悉融资、早期销售或招募合适的团队。
