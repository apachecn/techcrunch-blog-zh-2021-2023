有消息称，Web Summit Ventures 将成为新的 4000 万美元后续基金

原文：https://web.archive.org/web/https://techcrunch.com/2022/09/20/sources-say-web-summit-ventures-will-be-a-new-40m-follow-on-fund/

TechCrunch 了解到，全球最大的科技创业活动之一网络峰会(Web Summit)将推出由两只新基金组成的全新风险投资工具。此前，Web Summit 的联合创始人之间发生了激烈的争吵，他们在 2018 年首次创立了现已解散的 Amaranthine VC 基金，部分原因是为了加入围绕 Web Summit 活动发展起来的不断膨胀的投资生态系统。

虽然之前曾报道过Web Summit 联合创始人帕迪·科斯格拉夫将很快推出他的新工具 Web Summit Ventures (WSV)，但该基金的性质和规模直到现在还没有透露。

TechCrunch 了解到，WSV 将获得 4000 万美元的资金，分成两个 2000 万美元的基金。他们将分别被称为“网络峰会风险投资种子”和“网络峰会风险投资增长”。种子基金将在早期阶段和 A 轮投资，而增长基金将在“B 轮及以后”阶段投资。消息人士称，这两个基金都将是“后续”基金，不打算牵头融资。这与之前的苋菜红战略如出一辙。

进一步证实这些基金存在的形式是一份新的招聘广告为该基金招聘一名助理。

据悉，在 Amaranthine Ventures 的创始人和合伙人卷入一系列错综复杂的法律纠纷后，WSV 打算取代科斯格拉夫此前进入投资领域的尝试。

正如爱尔兰媒体此前报道的那样，在爱尔兰都柏林(网络峰会最初发起地)的公司注册办公室提交的文件显示，网络峰会首席执行官科斯格拉夫被列为网络峰会风险管理有限公司的董事

据了解，只有科斯格拉夫和克里斯·墨菲会成为网络峰会的合伙人。墨菲是 Web Summit 的前雇员，后来在 Amaranthine 基金工作了近三年，担任董事总经理。

一位消息灵通人士告诉 TechCrunch，新 WSV 基金的一个主要区别是，许多科技创始人将作为有限合伙人加入，包括 Twitter、Tinder、N26、Checkout.com、Rappi、Algolia、Lightricks 和 Wise 的一些创始人，以及一些风投基金的一些普通合伙人，他们表示将以个人身份投资，尽管这尚未得到独立证实。

Web Summit 试图参与其积累的庞大创业公司生态系统的故事始于 2018 年。

苋菜红基金是科斯格拉夫设立的；网络峰会联合创始人大卫·凯利；以及 2018 年的基金经理帕特里克·墨菲。但是，尽管它设法支持了 Hopin(在线活动初创公司，在疫情的远程工作时代，其估值飙升至 56 亿美元)，但一系列激烈的分歧导致科斯格拉夫在美国法院起诉凯利和墨菲。

提起诉讼后，5000 万美元的 Amaranthine 基金已经在 Tapestry 更名。

但是这种激烈的争论并不仅限于美国。

科斯格拉夫也在爱尔兰高等法院起诉凯利。凯利和墨菲否认这些指控，而凯利则在高等法院单独起诉科斯格拉夫涉嫌压迫少数股东。科斯格拉夫否认了这些指控。

Web Summit 的发言人以监管限制为由拒绝对 WSV 的推出发表评论。
