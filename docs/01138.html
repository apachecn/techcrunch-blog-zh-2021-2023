开发者生产力工具初创公司 Raycast 从 Accel 和 coat UE TechCrunch 融资 1500 万美元

原文：https://web.archive.org/web/https://techcrunch.com/2021/11/30/developer-productivity-tools-startup-raycast-raises-15m-from-accel-and-coatue/

专注于开发人员的生产力工具 Raycast 已经在 A 轮融资中筹集了 1500 万美元，由 Accel 和 Coatue 牵头。参与该轮投资的还有天使投资人 Johnny Boufarhat，他是 Hopin 的首席执行官和创始人；杰夫·温斯坦，Stripe 的产品；Jason Warner，GitHub 前 CTO 和 Checkout.com 首席技术官 Ott Kaukver 等。

Raycast 由前脸书软件工程师托马斯·保罗·曼恩和切赫·尼古拉耶夫于 2020 年创立，他们试图创造一种方法来弥补开发人员每天因不同 SaaS 工具之间的连续上下文切换而损失的时间。

“我们创办了这家公司，因为我们想改善我们与计算机的工作方式，”首席执行官曼恩在接受采访时告诉 TechCrunch。“作为开发人员，我们发现自己花了很多时间在没有带来多少价值的无用功上。我们宁愿把时间花在真正重要的工作上。这是 Raycast 的前提——我们带给你一个工具，帮助你更快地完成工作。”

Raycast 的目标是让开发人员更容易通过其命令行启发的界面找到和更新信息。该平台支持日常流程的自动化，并允许开发人员专注于重要任务。桌面软件从 Superhuman 和 Command E 等同行那里获取笔记，允许用户使用键盘快捷键快速调出和修改数据。用户可以很容易地在吉拉创建和重新修改问题，在 GitHub 合并请求和查找文档。该软件本质上是苹果 Spotlight search 的一个面向开发人员的版本，旨在帮助软件工程师使用单一工具导航他们工作中非开发工作的所有部分。

就增长而言，Raycast 表示，在短短 12 个月内，其日活跃用户数量从 2020 年 10 月的 130 个增加到目前的 11，000 多个，在此期间该平台上执行了超过 2，000 万次操作。该团队目前由 12 名成员组成，预计今年年底会有更多人加入。

Raycast 还在公共测试版中发布了其扩展 API 和 Store，使开发人员能够构建自定义扩展并与他们的团队或社区共享。该公司表示，在一个月内，该社区在测试版上建立了 100 多个扩展，连接到 Figma，GitHub，Chrome，inconce，YouTube，Twitter 等服务。Raycast 现在公开发布扩展 API 和商店，向全世界的开发者开放。

至于新的资金，Mann 说 Raycast 希望成为开发者生产力的领导者。

“这是我们的资金目标。我们将利用这笔资金进一步壮大我们的团队。我们希望让每个人都能使用这个平台，这样他们就可以建立他们想要的工具，”他说。

Raycast 计划利用这笔资金建立其开发人员和工具社区，加速平台的发展，并将 Raycast 引入团队。尽管该公司仍将重点放在个人开发人员上，但 Raycast 看到了与他人分享生产力工具和工作流程的潜力。从今天开始，公司将能够注册一个项目，以获得新的团队功能。用户将能够创建他们自己的内部存储，在那里他们可以构建和分发特定于他们团队需求的自定义扩展和链接。

“团队现在可以构建扩展，并与他们的团队成员私下共享，”Mann 说。“如果他们有一个内部工具，可以让自己在使用自定义设置时更有效率，他们可以与团队成员分享，这样他们也可以从中受益。这有助于他们作为一个团队保持高效，并为他们节省平时忙碌的时间。”

Raycast 的首轮融资紧随该公司从 2020 年 10 月开始的 270 万美元种子轮融资之后。这轮融资由 Accel 牵头，参与方包括 YC、小杰夫·莫里斯的 Chapter One 基金，以及天使投资人查理·契弗(Charlie Cheever)、卡尔文·弗伦奇-欧文(Calvin French-Owen)和马尼克·古普塔(Manik Gupta)。

“自从领投了 Raycast 的种子投资以来，我们对 Raycast 在开发者社区中的增长和牵引力印象深刻，”Accel 的合伙人安德烈·布拉索维安努(Andrei Brasoveanu)在一份声明中说。“这支持了我们最初的信念，即 Raycast 有潜力成为开发人员不可或缺的工具，我们很高兴看到该团队通过推出 API 和 Store 并扩展到团队而走得更远。我们很高兴能继续支持托马斯和彼得的雄心之旅。”
