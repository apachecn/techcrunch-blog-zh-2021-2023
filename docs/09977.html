星巴克详细介绍了其位于区块链的忠诚度平台和 NFT 社区

原文：https://web.archive.org/web/https://techcrunch.com/2022/09/12/starbucks-unveils-its-blockchain-based-loyalty-platform-and-nft-community-starbucks-odyssey/?

星巴克今天正式推出 Starbucks Odyssey，将于今年晚些时候推出——这是这家咖啡连锁店首次尝试使用 web3 技术。新的体验将该公司成功的星巴克奖励忠诚计划与 NFT 平台相结合，允许其客户赚取和购买解锁独家体验和奖励的数字资产。

该公司早些时候向投资者调侃其 web3 计划，称其相信这种新的体验将建立在当前星巴克奖励模式的基础上，在该模式下，今天的顾客可以获得“星星”,这些星星可以用来换取额外津贴，如免费饮料。它设想星巴克奥德赛作为其最忠实的顾客赚取更广泛的奖励的一种方式，同时也建立社区。

为了开发这个项目，星巴克请来了移动订单支付系统和星巴克应用程序的设计师亚当·布罗特曼作为特别顾问。Brotman 的团队现在是 web3 忠诚度初创公司 Forum3 的联合创始人，与西雅图咖啡连锁店自己的营销、忠诚度和技术团队一起为星巴克奥德赛工作。

星巴克的 CMO·布雷迪·布鲁尔告诉 TechCrunch，虽然星巴克已经调查区块链科技好几年了，但它只参与了这个特定的项目大约六个月。他说，该公司希望在这一领域投资，但不是像许多公司正在做的那样，作为一个“噱头”项目。相反，它希望找到一种方法，利用这项技术来增强其业务，并扩大其现有的忠诚度计划。

它选择让 NFTs 成为进入这个数字社区的通行证，但它故意模糊支持体验的技术的本质，以便将更多的消费者——包括非技术人员——引入 web3 平台。

“它恰好建立在区块链和 web3 技术的基础上，但老实说，客户可能甚至不知道他们正在做的是与区块链技术的互动。布鲁尔解释道:“这只是一个促成因素。

为了参与星巴克奥德赛体验，星巴克奖励会员将使用他们现有的忠诚度计划凭证登录网络应用程序。

一到那里，他们就可以从事各种活动，星巴克称之为“旅行”——比如玩互动游戏或接受挑战，加深他们对星巴克品牌或咖啡的了解。当他们完成这些旅程时，成员们可以以 NFT(不可替代的代币)的形式收集早期的数字收藏品。然而，星巴克奥德赛(Starbucks Odyssey)摒弃了科技术语，将这些 NFT 收藏品称为“旅行邮票”。

此外，一套限量版的 NFT 将在星巴克的 Odyessy 网络应用程序中出售，该程序同样适用于移动设备。虽然在多边形区块链托管，这些 NFT 将使用信用卡或借记卡购买——不需要加密钱包。该公司认为，这将通过降低进入门槛，让消费者更容易参与 web3 体验。这也不会让消费者的交易变得复杂，比如“汽油费”，他们更愿意提供捆绑价格。

该公司还没有准备好分享它的 NFT 将花费多少或有多少将在发布时提供，称这些决定仍在敲定中。

然而，各种“邮票”(NFT)将包括基于其稀有性的点价值，并且可以在市场上的星巴克会员中购买或出售，所有权在区块链上有保障。NFTs 上的艺术品是由星巴克和外部艺术家共同创作的，出售限量版收藏品的部分收益将捐赠给支持星巴克员工和顾客选择的事业。

通过收集邮票，会员将获得积分，从而解锁专属福利。

这些额外福利超出了您通过传统的星巴克奖励计划账户及其“明星”所能获得的。如今，会员可以购买免费咖啡、免费食物或精选商品，而在星巴克获得的积分将转化为体验和其他福利。

在低端，这可能是一个虚拟的浓缩马提尼酒制作班，或获取独特的商品和艺术家的合作。随着您积分的增加，您可能会受邀参加在星巴克储备烘焙店(Starbucks Reserve Roasteries)举办的特别活动，甚至有机会参观位于哥斯达黎加的星巴克庄园咖啡农场。预计最大的额外福利将留给那些购买 NFT 的人，尽管那些通过购买 NFT 赚钱的人可能会得到更少的额外福利。

例如，付费的 NFT 可以提供完整的旅游套餐和农场旅游，而付费的 NFT 可以单独提供旅游，航班和酒店由用户决定。然而，星巴克尚未在这方面做出任何正式决定。

但该公司可以说的是，它希望将该计划与其现有的忠诚度奖励进行深度整合，而不仅仅是在两个计划中使用相同的用户账户凭证。

布鲁尔说，星巴克已经在想象如何将一些赚取 NFT 的活动与现实世界中的星巴克购物联系起来。

在 Odyssey 中，用户通过挑战获得 NFT，这可能还包括像“尝试浓缩咖啡菜单上的三样东西”这样的真实活动。这将要求用户在结账时出示他们的条形码——如果他们赢得了星级，他们的交易将计入星巴克奥德赛挑战赛。该公司仍在确定推出时将包括哪些游戏、挑战和任务。

布鲁尔强调说:“但我们将获得与顾客在我们店里的行为直接相关的体验。最重要的是，该公司希望让获得 NFTs 成为任何人都可以做的事情——而不仅仅是那些有钱购买数字收藏品的人，正如当前 NFT 社区的常见情况那样，这些社区的价格高于普通用户。

布鲁尔说:“人们将有很多方法不用花很多钱就能获得(奖励)。”“我们希望让这变得超级简单和容易实现。比如，客户可以获得大量的日常体验，比如虚拟课堂或限量版商品。”他补充道:“体验的范围将非常广泛，也非常容易获得。”。

星巴克表示，为了这个项目，它探索了所有不同的区块链，但最终选择了 Polygon 为这项工作建立的“证据”区块链技术，因为它比第一代“工作证据”区块链使用的能源更少，这更符合其对话目标。

对于一家以利用新兴技术、让消费者更容易接触到这些技术而闻名的公司来说，进入 web3 世界的想法是有意义的。在过去的几年里，星巴克在其门店引入了 Wi-Fi，以鼓励顾客在参观期间花更多的时间。早在 Apple Pay 普及之前，它就已经推动了移动钱包的概念。它使移动点餐远远领先于科维德·疫情，当其他连锁餐厅开始使用它时。

但是，当许多传统企业进入 web3 市场时，一个针对它们的批评是，它们把它当作一种营销噱头，而不是真正的努力。当然，星巴克认为情况并非如此——但只有时间才能证明它的兴趣有多认真。

布鲁尔声称:“我们对这些技术的未来充满信心，这些技术能够实现以前不可能实现的体验。”。他解释说，这样做的目的是为了保持灵活性，随着 web3 市场的变化而与客户同步。“我们着眼于长远，这真的很重要，”他继续说道。“但是，鉴于我们正在将其纳入我们行业领先的大规模奖励计划，我们承诺，”他说。

该公司表示，其 web3 平台将于 9 月 12 日开放其等候名单(waitlist.starbucks.com)，并将于今年晚些时候推出。它将在明年的某个时候删除等候名单，并更广泛地开放平台。
