Harmonic 帮助投资者查询他们最疯狂梦想的创业搜索 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/11/07/harmonic-helps-investors-query-the-startup-searches-of-their-wildest-dreams/

Siri，给我看看过去两年成立的金融科技公司，它们在过去一年没有融资，但同期的员工人数增长了 100%；它能由过去六个月 Twitter 访问量增长至少 50%的斯坦福校友创建吗？

这是谐波的愿景；嗯，除非你把 Siri 换成 Harmonic 的基于文本的启动搜索查询工具。这个数据平台由马克斯·鲁德曼领导，由布莱恩·凯西和
阿贾伊·索姆谢蒂共同创建，认为它可以帮助高管们发现下一个大的创业公司，而不需要数百小时的人工采购和研究。

Harmonic 是其最大竞争对手 Crunchbase 和 PitchBook 的一个更具体的版本，后者聚合并组织私人创业数据。鲁德曼告诉 TechCrunch:“我们走出去，查看网络上可能存在公司信息的每个角落，我们获取结构化和非结构化数据，并找出如何将这些数据融合到一家公司的一些规范表示中。”

根据鲁德曼的说法，Harmonic 的聚合差异是它用来帮助识别哪些公共数据对某些领域更准确的智能，然后合并这些来源以开发“任何时间点最准确、最新的表示”。

Harmonic 加入了其他创业公司的行列，试图让 venture 更加数据驱动、透明和公平。理论上，算法投资对冲了投资者的先入为主的观念，将情绪推到了一边。金融科技独角兽 Clearco 和风险公司 SignalFire 花了数年时间实施以数据为中心的投资流程，AngelList 和 Hum Capital 也加入了这一流程。

在投资者重新学习纪律的背景下，数据让人感到安全。但是，随着其他解决方案的成熟，所述数据的清洁度和可靠性受到质疑。(一位创始人甚至搞了一次恶作剧，称安德森·霍洛维茨(Andreessen Horowitz)是他在 Crunchbase 的初创公司的投资者；当其他投资者争相投资他的新贵时，他解释说，展示平台上糟糕的数据质量是一个笑话，彭博报道。

另一个需要考虑的问题是，谐波是否真的有助于投资者打破他们的模式匹配倾向，或者只是有助于加强已经存在的价值。

“核心问题是他们现在寻找(他们相信的想法)的方式是隐性的，而不是显性的，这意味着他们非常依赖他们的网络，对吗？，”鲁德曼说。“同事、关系网、合伙人和他们的校友网络……他们与已经在他们投资的公司中的创始人合作，这可以帮助他们与其他人联系。第一步只是将所有公司的所有信息纳入我们的索引，这样当他们明确进行搜索时，他们最终会找到许多符合标准的公司，而他们可能没有通过自己的网络找到这些公司。”

Ruderman 承认，数据可靠性和一致性是最难解决的问题之一，他们的策略是他们的一大优势。“我们能够在大规模范围内保持数据最新，并充满信心地将来自整个网络的结构化和非结构化数据的碎片合并在一起，”鲁德曼说，并补充说，其成功的主要衡量标准是他们使用的内部评分，用于捕捉新鲜度、库存和分类。

当被问及他们如何在新鲜度上获得优势的更多细节时，鲁德曼没有分享太多细节(鉴于这是一个竞争的护城河，我对此并不太惊讶)。他还表示，定价将随着产品的发展而发展，但目前这家初创公司收取许可费和 API 使用费。

鲁德曼的背景为他为什么对更好的搜索方式充满信心增添了一些色彩。这位联合创始人在谷歌工作了大约六年半——他的最后一份工作是在搜索团队担任高级软件工程师，该团队致力于开发工具，帮助谷歌进行大规模的 UX 研究和设计。在此之前，他在人力运营部门学习行为经济学，在商业智能团队学习技术基础设施，在财务团队学习机器学习，最终学习搜索。

到目前为止，他的方向和他背后的公司已经为 Harmonic 赢得了至少 150 个客户，包括 SaaS 的初创公司，如 Brex、bucket、inconce 和 Carta，以及风险投资公司，如 Floodgate、a16z 和 Accel。一些早期采用者甚至成为了这家初创公司的最大投资者。Harmonic 今天宣布完成了由 Sozo Ventures 领投的 2300 万美元 A 轮融资，Craft Ventures 也参与了该轮融资，后者去年领投了 700 万美元的种子轮融资。另一个客户 Floodgate 是 Harmonic 的第一个投资者。

“通过在 venture 中创建一个真正强大的发现工具，它让资本以更有效的方式流向更多的创新……如果我们将这一点带到销售团队，它让团队在正确的时间带来他们的服务并向前推进，”Ruderman 说。“最终，我们希望人才能够找到与他们的才华相匹配的创业公司，推动创业公司向前发展。”
