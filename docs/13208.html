三年多前，这位编辑在离开 Y Combinator 总裁的职位后不久，在三藩市的一个小型活动上与山姆·奥特曼(Sam Altman)坐在一起，成为他在 2015 年与埃隆·马斯克(Elon Musk)和其他人共同创立的人工智能公司 OpenAI 的首席执行官。

当时，奥特曼用听起来有些古怪的语言描述了 OpenAI 的潜力。例如，奥特曼说，人工通用智能——可以像人类一样解决问题的机器智能——的机会如此之大，如果 OpenAI 设法破解它，这套设备“可能会捕捉到宇宙中所有未来价值的光锥。”他说，该公司“将不得不停止发布研究”，因为它太强大了。当被问及 OpenAI 是否犯有散布恐惧罪时——马斯克一再呼吁所有开发人工智能的组织都受到监管——奥特曼谈到了当“你正在指数曲线上建造一些东西”时而不是考虑“社会后果”的危险

观众在谈话的不同点上笑了，不确定如何认真对待奥特曼。然而，现在没有人笑了。虽然机器还没有人类智能，但 OpenAI 已经发布的技术让许多人大吃一惊(包括马斯克)，一些批评者担心这可能会导致我们的毁灭，特别是据报道更复杂的技术即将到来。

事实上，尽管重度用户坚持认为它不太聪明，但 OpenAI 上周向公众提供的 T2 ChatGPT T3 模型能够像人一样回答 T4 的问题，因此各行各业的专业人士都在试图处理其中的含义。例如，教育工作者想知道他们如何将原创作品与他们一定会收到的算法生成的论文区分开来——这可以避开反抄袭软件。

保罗·凯德罗斯基本身并不是一名教育家。他是一名经济学家、风险投资家和麻省理工学院的研究员，自称“沮丧的正常人，喜欢思考复杂系统中的风险和意外后果。”但他是突然对我们的集体未来感到担忧的人之一，昨天他在推特上写道:“在 OpenAI 上的 hame，因为他向一个毫无准备的社会无限制地发射了这枚袖珍核弹。”Kedrosky 写道，“我明显觉得 ChatGPT(及其同类)应该立即撤销。而且，如果重新引入，也只有严格的限制。”

我们昨天和他讨论了他的一些担忧，以及为什么他认为 OpenAI 正在推动他认为是“美国经济 100 年来最具破坏性的变化”，而且不是以一种好的方式。

我们的聊天已经被编辑的长度和清晰度。

TC: ChatGPT 上周三出来了。是什么引发了你在推特上的反应？

PK:我过去玩过这些对话式用户界面和人工智能服务，这显然是一个巨大的飞跃。这里特别困扰我的是它的偶然的残忍，对许多不同的活动有着巨大的影响。不仅仅是那些显而易见的，比如高中论文写作，而是几乎在任何有语法的领域都是如此——意思是一种表达自己的有组织的方式。可能是软件工程，高中论文，法律文件。所有这些都很容易被这个贪婪的野兽吃掉，并再次吐出来，而没有补偿任何用于训练它的东西。

我从加州大学洛杉矶分校的一位同事那里听说，他们不知道在本学期结束时如何处理论文，他们每门课程得到数百份，每个系得到数千份，因为他们再也不知道什么是假的，什么不是。就像今天早些时候有人对我说的那样，如此随意地做这件事让人想起所谓的[道德]白帽黑客，他在一个广泛使用的产品中发现了一个错误，然后在更广泛的公众知道之前通知开发商，这样开发商就可以修补他们的产品，我们就不会有大规模破坏和电网瘫痪。这是相反的，一个病毒被释放到野外，没有考虑后果。

感觉它真的可以吞噬整个世界。

有些人可能会说，那么，当自动化进入汽车工厂，汽车工人失业时，你有同样的感觉吗？因为这是一种更广泛的现象。但这是非常不同的。这些特定的学习技术是自我催化的；他们从请求中学习。因此，制造工厂中的机器人，虽然具有破坏性，并为在那里工作的人创造了难以置信的经济后果，但并没有转过身来，开始吸收工厂内的一切，一个部门一个部门地移动，而这不仅是我们可以预期的，也是你应该预期的。

马斯克离开 OpenAI 的部分原因是对公司发展的分歧，他在 2019 年说，他一直在谈论人工智能是一个存在的威胁。但是人们嘲笑他不知道自己在说什么。现在我们正面临这种强大的技术，还不清楚谁会介入解决它。

我认为它将在一堆地方同时开始，其中大多数看起来真的很笨拙，人们会(然后)嘲笑，因为这是技术专家做的。但这太糟糕了，因为我们已经通过创造这种结果性的东西把自己带入其中。所以，就像联邦贸易委员会多年前要求经营博客的人[表明他们]有附属链接并从中赚钱一样，我认为在一个微不足道的层面上，人们将被迫披露‘这些都不是我们写的’。这都是机器生成的，”[编者按:OpenAI 表示，它正在研究一种对人工智能生成的内容进行“水印”的方法，以及其他“出处技术”。]

我还认为，在我们正在训练的机器学习算法的背景下，我们将看到正在进行的针对微软和 OpenAI 侵犯版权的诉讼中出现新的活力。我认为这项服务会引发更广泛的 DMCA 问题。

我认为，关于服务的后果，最终可能会有一场[大规模]诉讼和和解，你知道，这可能会花费太长时间，而且不能帮助足够多的人，但我不明白我们如何不能在这些技术方面结束[这个地方]。

麻省理工学院在想什么？

Andy McAfee 和他的团队更乐观，他们有一个更正统的观点，即任何时候我们看到颠覆，其他机会就会被创造出来，人们是流动的，他们从一个地方到另一个地方，从一个职业到另一个职业，我们不应该如此墨守成规，认为这种特定的技术进化是我们无法变异和迁移的。我认为这大体上是对的。

但特别是过去五年的教训是，这些变化可能需要很长时间。例如，自由贸易是一种令人难以置信的破坏性的经济经验，作为经济学家，我们都告诉自己，经济将会适应，人们将普遍受益于较低的价格。谁也没有预料到的是，有人会把所有愤怒的人组织起来，选唐纳德·特朗普。所以有这种想法，我们可以预见和预测后果会是什么，但[我们不能]。

你谈到了高中和大学的论文写作。我们的一个孩子已经问过了——理论上！—如果用 ChatGPT 写论文会不会算抄袭。

写论文的目的是为了证明你能思考，所以这缩短了过程，违背了目的。同样，就后果和外部性而言，如果我们不能让人们有家庭作业，因为我们不再知道他们是否作弊，这意味着一切都必须在教室里发生，并且必须受到监督。我们不能带任何东西回家。更多的东西必须口头完成，这意味着什么？这意味着学校变得越来越昂贵，越来越手工化，越来越小，而与此同时，我们正试图做相反的事情。就实际提供服务而言，高等教育的后果是毁灭性的。

你怎么看待普遍基本收入的想法，或者说让每个人都参与到 AI 的收益中来？

我不再像以前那样是一个坚定的支持者了。原因是，从某种意义上说，COVID 是一个普遍基本收入的实验。我们付钱让人们呆在家里，他们想出了 QAnon。所以我真的很担心，每当人们不必跳上一辆车，开车去某个地方，做一份他们讨厌的工作，然后再回家，会发生什么，因为魔鬼会为游手好闲的人找到工作，会有很多游手好闲的人和很多魔鬼。
