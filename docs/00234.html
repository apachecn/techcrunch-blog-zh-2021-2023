Tiger Global 因其发卡 API | TechCrunch 在赞比亚 Union54 中领先 300 万美元

原文：https://web.archive.org/web/https://techcrunch.com/2021/10/27/tiger-global-leads-3m-round-in-zambias-union54-for-its-card-issuing-api/

Y Combinator 支持的第一家赞比亚创业公司 Union54 的投资条款清单上又多了一家大型风险投资公司:Tiger Global 。

该公司向 TechCrunch 证实，这位风投巨头领导了其刚刚完成的 300 万美元种子轮，仅在两个月前从 Y Combinator 的夏季批次中毕业。

总部位于旧金山的风险投资公司 Runa Capital、Ace & Company、Todd & Rahul 天使基金和 Vibe VC 参与 。

赞比亚金融科技公司也收到了天使投资者的支票，如尼日利亚 neobank Kuda 的首席执行官 Babs OgundeyiRisana Zitha，复兴资本董事总经理；以及 Wise 前中小企业发展总监 Gbenga Ajayi。

Union54 是一个 API，允许非洲软件公司发行和管理他们的借记卡，而不需要银行或第三方处理器。

联合创始人珀尔修斯·姆兰博夫妇和 T21 于今年创办了 Union54，这是他们之前的公司沙祖于六年前创办的挑战者银行的一个分支。

发卡是一个越来越区域化的空间，因为每个地域的规定不同。在沙祖，创始人发现给顾客发卡既困难又耗时。 在解决问题的同时，他们发现了与发卡机构互动时的扭曲激励 。他们推出 Union54 不仅是为了解决他们自己的问题，也是为了解决其他金融科技公司 的问题。

首席执行官姆兰博告诉 TechCrunch ，自从加入 Y Combinator 以来，沙祖一直处于隐身状态，直到这个月才上线。

他补充道，目前有超过 50 家公司处于 Union54 的 API 沙盒环境中。从数字银行到后 A 轮融资的金融科技公司，以及“基于 Union54 可用性而成立的公司”。

四家公司目前已经投入生产，并开始向他们的客户发放虚拟卡。姆兰博说，今年年底前还会有 30 多名学生加入。首席执行官 说，对这些公司来说，开始生产和开始发行的平均时间需要三到九天。

其合作伙伴包括非洲独角兽 Flutterwave 和较新的公司，如 Payday 和 Plumter(跨境金融科技)以及加密交换平台 Bitmama。

那么，自该公司 8 月份与 TechCrunch 对话以来，还有什么变化吗？ “我们已经意识到，对于那些希望利用现有客户群赚钱的公司来说，存在着巨大的潜力 。但是我们不知道该如何去做。现在，我们有了，而且我们是非洲唯一一家会给你们(金融科技公司)提供交换的公司。”

本质上是 ，这是交换的工作方式——Union54 加入一家金融科技公司，它使用 union 54 的 API 向员工发放一张卡。当员工使用该卡在线支付物品时，可能是 30 美元的网飞订阅或 AWS 发票，金融科技公司将从这笔交易中赚取 1%，在这种情况下为 0.30 美元。

虽然这可能看起来很微小，但想象一下，一家金融科技公司使用 Union54 的 API 向超过 20，000 名经常使用它的客户发行卡。这就是这家赞比亚初创公司希望创造价值的地方:帮助金融科技公司从交换中获得可观的收入，而无需做大量与卡管理相关的繁重工作。

姆兰博说:“我们不仅允许金融科技公司以任何银行或信用卡发行商做梦都想不到的速度进入市场，而且我们也真正表明了我们的动机是一致的。”。“只有当他们赚钱时，我们才赚钱，这就是为什么我们乐意为那个交易所提供担保。”

谈到对 Runa Capital 的投资，普通合伙人 Andre Bliznyuk 表示，公司 很高兴 支持 Union54 的努力，即“通过让客户能够轻松推出新的基于卡的产品，并为消费者提供切实的价值，从而推动非洲金融科技生态系统。”

老虎环球拒绝对这笔投资置评。 然而 非常有趣的是，继 3 月份的 flutter wave、7 月份的 FairMoney和本月的 Mono之后，这家投资公司今年正在对一家非洲金融科技初创公司进行第四次投资。

但一个更令人印象深刻的事实是:Union54 似乎是它在尼日利亚或南非以外的第一个赌注，这两个国家是非洲四个最具主导地位的科技生态系统中的两个，包括肯尼亚和埃及。过去十年，Tiger Global 投资了尼日利亚初创公司 Jobberman、Cheki 和 Wakanow，以及南非电子商务公司 Takealot。

与尼日利亚和南非冒泡的科技生态系统截然不同，赞比亚拥有相对安静的 创业和风险投资场景；因此，老虎环球参与 Union54 的种子赛对这个南部非洲国家来说是一个巨大的胜利。

然而，回想起来，姆兰博和马丁尼作为沙祖创始人的经历，他们在那里筹集了超过 200 万美元，成为赞比亚资金最多的创业公司之一， 不能被视为 Union54 表面上成功的贡献者。

Mlambo 表示，当该公司筹集这一轮资金时，由于其主要位置在赞比亚，它优先考虑速度并寻找全球合作伙伴。老虎环球没有浪费时间证明它是投资者联盟 54 想要的。

“与泰格的讨论相当直截了当；他们很快就提交了，整个过程相当顺利，”这位首席执行官评论道。

与 Union54， 联盟的创始人正在处理一个泛非洲的问题，而不是赞比亚的问题。因此，前面的任务是多倍的，但回报也是多倍的。例如，姆兰博声称，在不到一个月的运作中，Union54 目前的交易量超过了沙祖头 10 个月的交易量。

姆兰博认为赞比亚是一个开展业务的好地方。然而，他说，说服人们搬到赞比亚工作很有挑战性，所以 Union54 在创业的大部分时间里不得不依赖远程工作(这是现在的常态)。目前，该公司在尼日利亚、马拉维、南非、赞比亚和欧洲拥有一支由 10 人组成的精干团队。

Union54 计划利用这笔投资来增加工程、产品、营销和销售团队的招聘。这家赞比亚公司还将利用这笔资金扩大其地区客户群。

“这笔资金的目的是帮助我们找到非洲最优秀的人才，那些想要解决大问题的人。这项投资有助于我们能够找到他们，并说，‘嘿，我们是一家资金雄厚的公司，我们有客户在使用我们，我们正在创收，这将是非洲最重要的公司之一。’

但最终，姆兰博说，目标是让公司处于更有利的地位，在接下来的几个月里筹集到有吸引力的 A 轮融资。
