娱乐网站 Fandom 增加了经常被要求的功能

原文：https://web.archive.org/web/https://techcrunch.com/2022/06/01/entertainment-site-fandom-adds-long-requested-creator-features-interactive-wiki-maps-and-fandom-trivia/

拥有超过 3 亿月活跃用户和 25 万维基社区的维基托管服务和粉丝平台 Fandom 刚刚宣布了其最新的功能和工具:交互式维基地图和 Fandom Trivia。

Fandom 上的创建者已经通过社区博客知道了互动维基地图和 Fandom 琐事一年多了，因为这些功能已经过测试和完善。今天是该平台首次向公众公布这些功能。

交互式 Wiki 地图引入了一个可定制的地图工具，这是创建者社区长期以来所要求的。与此同时，Fandom Trivia 是一个内置的维基功能，使创作者能够增强粉丝的体验，并测试他们对想象世界的每个细节的知识，他们对此还不够。现在，所有粉丝社区的创作者都可以设计自己的地图和测验，以进一步让粉丝沉浸在想象的世界中，为他们提供他们最喜欢的游戏、电视节目和电影的信息。

经过一段时间的测试、错误修复和接受创作者的反馈，这些功能终于在上个月推出。该团队仍在致力于改进地图工具和实现新功能，如自定义地图标记等。

一些已发布地图的例子包括"阿凡达:最后的气宗的世界和"星球大战:银河边缘"中的巴图星球。

图片来源:粉丝

交互式 Wiki 地图功能是使用开源传单 JavaScript 库创建的，并使用 JSON (JavaScript 对象符号)定义。它可以作为地图名称空间中的一个 wiki 页面来构建，并且可以嵌入到其他带有模板转换的页面中。跨包是使用 MediaWiki 的模板功能将相同的内容合并到许多文档中，而无需单独编辑这些文档。JSON 文件是源代码编辑器和底层代码。该团队还在此基础上构建了一个更加直观的编辑器。

然后，为了让每个人都可以创建交互式地图，Fandom 创建了一个可视化编辑器工具来简化构建体验。该工具类似于当前的主题设计器，允许用户轻松上传地图图像、在地图上放置自定义标记、添加描述等，而无需使用 JSON 源代码。熟悉 JSON 的用户仍然可以在源代码编辑器中编辑代码。

这不是 Fandom 第一次尝试实现地图。早在 2014 年，就有人尝试过更老的维基地图功能；然而，Fandom 的军火库中只有很少的资源来完成一些拥有合适工具并能获得足够多用户采用的东西。

Gamepedia 是 Fandom 在 2018 年收购的一个独立的维基平台，拥有针对交互式地图的 IP 特定解决方案，但从未集中到一个工具中。开发团队也比 Fandom 目前的团队要小。这个新的统一的、可定制的地图工具实现了对创作者的承诺，这一承诺花费了数年时间来完善。

Fandom Trivia 也经历了一系列测试，尤其是网站上测验的位置。

例如，Fandom 发现，当这个特性位于页面的顶部、侧面甚至底部时，没有多少粉丝会关注它。有了的数据和研究，团队发现当功能被放入内容本身时，参与度会增加。超过 1200 万人(4%的用户)参与了 Fandom 琐事，当时测验看起来像是内容的一部分，而不是在页面的广告一侧。

与 2019 年开发的有限问答功能不同，这种新开发的体验使用了工作人员、维基代表和维基专家可以访问的内部工具。工作人员和 Wiki 团队可以通过使用预先编写的问题目录或从头开始创建它们来有效地进行测验。

此外，经过大量研究，Fandom 确定游戏用户对参加测验最感兴趣(73%)。与此同时，44.4%的动漫迷想参加测验。此外，Fandom 发现人们强烈希望看到用户创建测验和 Fandom 创建的测验。事实上，64%的受访者认为两者的结合最有意义。

TechCrunch 采访了社区布兰登·雷亚的副总裁，他说 Fandom 正在就开放粉丝自己制作的测验进行早期对话。难度等级、社区排行榜、挑战甚至游戏化元素也在考虑之中。

Fandom 首席营销官 Stephanie Fried 也加入进来，并强调该平台的主要目标是提供粉丝可能需要的一切，并继续返回该平台获取更多。有了这两个功能和未来的其他增强功能，Fandom 的目标是成为粉丝的一站式商店。布兰登补充说，他希望用户说 Fandom 不仅仅是寻找答案的地方，也是他们可以获得完整粉丝体验的地方。

Rhea 说:“Fandom 很高兴有热情的创作者管理我们的 250，000 个维基，他们对我们平台的贡献使我们成为头号信息源和游戏、电视、电影和流行文化的领先网站。”“这些新功能将为他们提供更多身临其境的机会，让粉丝探索他们的维基，并通过提供更加无缝和身临其境的网站体验来欢迎新用户加入我们的社区。”
