虽然亚马逊经历了艰难的一年，但 AWS 仍然是一棵可靠的摇钱树

回到三月 2020 年，当世界关闭时，亚马逊成为世界上最受欢迎的在线商店。当人们不能离开自己的家时，商品送到他们手中就变得势在必行，于是亚马逊繁荣起来。资金涌入它的金库，它的股票价格暴涨，它疯狂地招聘和建造仓库来满足不断增长的需求。

根据Statista的数据，该公司在 2020 年第一季度开始疫情，拥有大约 84 万名员工。到 Q1 2022 年，它已经有超过 160 万工人。问题是，随着疫情放松对公共生活的控制，人们不再在网上购买任何东西，而是回到实体零售。

亚马逊首席执行官安迪·雅西似乎明白市场已经发生了变化，他已经让他的经理们寻找削减开支和降低运营成本的地方，这是许多大型组织在经济不确定时期正在做的事情。

如果报道准确的话，亚马逊的努力包括在短期内裁员 10，000 人，以抵消疫情高峰期的招聘。

从股票的角度来看，该公司已经放弃了在疫情获得的几乎所有收益，今年损失了近 50%的价值，据《美国消费者新闻与商业频道》报道。这意味着杰夫·贝索斯的财富比以前少了一点，他的前妻麦肯齐·斯科特的财富也少了一点。与此同时，Jassy 有更多令人头疼的事情要处理，还有削减运营成本的压力。

通过这一切，Jassy 在晋升到角落办公室之前运营的亚马逊云部门 AWS 继续保持着一贯的高水平。但即使是 AWS 也报告称第三季度增长放缓，因为公司试图削减云成本。

考虑到最近报告的 2022 年第三季度，AWS 收入达到 205 亿美元，低于分析师群体预期的 211 亿美元。这可能看起来不多，但云计算一直是少数几个超级增长领域之一，所以错过是一件大事。

也就是说，你不能忽视这样一个事实，即 AWS 现在有望成为 800 亿美元的企业，所以这不是一件值得挂在头上的事情，而且人们一致认为，尽管外部宏观经济条件不佳，云业务仍有很大的增长空间。

换句话说，无论货币问题、增长放缓还是客户期待在新的一年里仅适度增加 IT 支出，AWS 都可能很好。Jassy 可能不得不在全公司范围内削减成本，但 AWS 很可能基本上不会这么做。
