原文：https://web.archive.org/web/https://techcrunch.com/2023/03/17/pull-systems-launches-out-of-up-labs-porsche-partnership-to-tackle-ev-performance/

当保时捷与 venture studio UP 合作时。实验室，任务是在三年内创建六家初创公司，旨在解决德国汽车制造商的最大问题，并作为一个独立的商业 it 足够引人注目，可以吸引其他客户。

在保时捷的名单上:帮助管理和自动化电动汽车性能的软件。合作中诞生的第一家初创公司 Pull Systems 开发了一款软件产品，两家公司表示该产品可以解决这个问题。在 SXSW 2023 上亮相的 Pull Systems 也宣布在由 UP.Partners 牵头的种子轮融资中筹集了 500 万美元。

“汽车正在成为软件和电池的结合体——最终是电池的性能，”UP。实验室主席凯特琳·福利说。“为了保持竞争力，原始设备制造商需要真正了解这两个方面，因为他们真正擅长的东西实际上是汽车中更具商品性的部件。”

Pull Systems 是一个软件即服务平台，为电动汽车供应商、制造商和运营商提供性能管理软件。该产品不是电池管理软件(BMS)，BMS 在技术上负责收集电池数据并与电池管理系统通信。up 前产品风险主管亨利·弗曼解释说，初创公司的软件是一种补充。拉式系统公司的首席产品官。

它已经作为概念验证推广到一些保时捷 Taycan 车辆上。

这家初创公司开发了一个机器学习模型库，可以分析和预测车辆行为，例如在保时捷车队中驾驶和充电。这种信息，加上天气模式和道路状况等外部数据，可以用来预测并通知汽车制造商或电动汽车所有者车辆何时需要维护，何时部署无线软件更新，甚至提高售后收入。

弗曼补充说，从理论上讲，该软件可以跟踪和收集保时捷电动汽车车队中每辆车的数据，这也有助于确定可能通过新固件解决的性能问题，或在电池达到寿命终点时确定最佳的第二生命选项。

最终，该公司希望使用机器学习工具来自动化软件。

“在电气化的复杂性中，我们在这里的真正愿景是，汽车实际上能够自己承担自己推进系统的一些管理工作，”弗曼说。“我们看到了一个很好的机会，让我们可以针对这些不同的软件更新，自动做出许多本质上基于规则的结论。”

他解释说，例如，该软件可能会识别出进入某个地区的天气锋，并发布软件更新以帮助优化电池。

对于保时捷来说，这是一个令人信服的前景，该公司计划在未来几年内将其电动汽车阵容扩展到 Taycan 之外，包括 2024 年的 Macan，2025 年的 718，卡宴和尚未命名的全尺寸 SUV。

Pull Systems 计划在未来一年内增加几家汽车制造商。
