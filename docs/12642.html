AWS 推出 Graviton3E，这是一款面向高性能计算工作负载的新型 Arm 芯片

在 re:Invent 的传统晚间主题演讲中，AWS 今晚宣布了相当多的新硬件，包括新版本的 Nitro 虚拟机管理程序、新的实例类型和新版本的基于 Arm 的定制 Graviton 芯片，该芯片专为支持高性能计算工作负载而设计。这款新的 Graviton3E 芯片是现有 Graviton 系列的变体，有望显著提高性能，包括将严重依赖矢量指令的工作负载的性能提高 35%。

这些新芯片显然将支持新的 AWS EC2 实例类型，从逻辑上称为 Hpc7G 开始。这种新的实例类型将有各种大小，最多 64 个 vCPUs 和 128 GiB 内存。不过，这些实例要等到 2023 年初才能实现。对于更多的网络密集型工作负载，AWS 还推出了新的 Graviton3E 实例类型(c7gn)。

对于英特尔的粉丝来说，也有新的基于 Ice Lake 的至强处理器。

图片来源:弗雷德里克·拉迪诺瓦

所有这些新实例都将使用 AWS 的新 Nitro 5 硬件管理程序，该公司今天也宣布了这一消息。Nitro v5 承诺显著改善延迟，性能功耗比提高 40%，PPS 提高 60%。AWS 团队通过将定制 Nitro 芯片中的晶体管数量增加一倍来实现这一目标。

图片来源:弗雷德里克·拉迪诺瓦

AWS 效用计算高级副总裁 Peter DeSantis 在今晚的主题演讲中说:“如果你拒绝在安全性和成本等问题上让步，性能就很难实现。”在许多方面，这一直是 AWS 计算平台及其定制处理器工作的故事。
