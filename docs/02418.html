Opera】发布了其 Web3“加密浏览器”测试版，具有内置加密钱包、轻松访问加密货币/NFT 交易所、支持分散应用程序(d apps)等功能。Opera 执行副总裁桂鑫·阿内森在一份声明中说，目的是“简化主流用户经常感到困惑的 Web3 用户体验”。

一个关键的功能是内置的非托管钱包，从一开始就支持包括以太坊、比特币、Celo 和 Nervos 在内的区块链。它还宣布与 Polygon 和其他公司合作。这个想法是让你不需要任何扩展就能访问你的密码，也可以选择使用第三方钱包。您可以通过菲亚特到加密的入口购买加密货币，直接在钱包中交换加密，发送和接收加密货币，并检查您的钱包余额。它甚至有一个安全的剪贴板，确保其他应用程序不能在你复制/粘贴数据。

另一个主要功能是支持 Web3，也就是基于区块链的分散式互联网，也就是密码爱好者(和怀疑论者)中流行的新事物。除了通过区块链加密提供额外的安全性之外，它还允许用户访问 GameFi 之类的东西，“在那里你可以通过玩各种元诗句来赚钱，”Opera 指出。根据 Opera 的说法，它还提供了一个“加密角落”,里面有最新的区块链新闻，还“让你增长你的 Web3 技能”。

Opera 的竞争对手 Mozilla 最近宣布它将接受加密货币捐赠，但由于区块链的环境影响，包括联合创始人 Jamie Zawinski 在内的用户强烈反对，这给造成了打击。Opera 表示，或许是预计到了类似的反应，它正致力于“尽快”实现更节能的以太网第二层标准。

像育碧这样的公司跳上了区块链的火车，推出了 NFTs 和其他产品，也受到了类似的批评。然而，Opera 至少给了用户多种浏览器选择，因为它还提供了常规的 Opera 浏览器和游戏玩家专用的浏览器。该加密浏览器现已适用于 Android 、 Windows 和 Mac ，iOS 版本即将推出。
