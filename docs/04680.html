了解并展示你的目标市场规模对你的企业获得资金至关重要。天使投资人和风险投资人都希望看到一种突破性的产品或服务，这种产品或服务能够扩大规模，并能在一个相当大的市场中获得相当大的份额。

在投资的任何阶段，无论是种子期还是 B 轮，都有必要在你的推介材料中加入一张幻灯片，从三个角度解释你的市场规模——TAM、SAM 和 SOM。

当你向投资者展示你的市场规模数据时，他们会寻找 TAM、SAM 和 SOM 信息。这些数据点给那些看起来巨大而遥不可及的数字蒙上了一层神秘色彩，但如果你有条不紊地估算市场规模，你会发现其实并没有那么复杂。

我最近参加了一家生产创新、营养丰富的橄榄和辣椒的公司的演示。因为他们的数据点场景简单易懂，所以我在这里用它来解释他们的 TAM、SAM 和 SOM。

步骤 1:捕获 TAM

虽然 TAM(总可用市场)往往会引起最大的焦虑，但它是最容易处理的数据点。TAM 描述的是更大范围内的总收入。你可以用三种不同的方法计算 TAM。

首先是自上而下的方法，通常通过公开的市场研究报告或从公开的数据报告中推断得出。

接下来是自下而上的方法，它依赖于实际和预计定价的计算，以及您的产品或服务的当前和预计总使用量。

如果你做足了功课，你就能消除财务预测的猜测，向潜在投资者展示他们如何从投资你的公司中获益，并把自己置于实现目标的最佳位置。

最后是价值理论方法。这一理论应用了对你的产品或服务的预计价值和总预期使用的有根据的分析，然后计算有多少价值可以反映在它的定价中。价值理论通常适用于全新的产品、服务和类别。让我们将理论应用于 Airbnb。

Airbnb 通过想象一种能让房主与寻找住宿地点的旅行者匹配的技术，颠覆了酒店业。为了应用价值理论，评估旅行者愿意为这种新型住宿支付多少钱，房主希望每晚获得多少报酬，以及 Airbnb 可以从房主那里收取多少技术费用。

当一个产品或服务类别已经被很好地研究过，TAM 通常使用公开的市场研究报告来计算。如果产品或产品类别是新的，并且没有经过彻底的研究——这在技术领域经常发生，也适用于其他垂直行业——我建议聘请一位经验丰富的市场研究顾问，使用自下而上或价值理论的方法来保护 TAM 数据。
