Odilo 筹集了 6400 万美元，其白标电子学习图书馆拥有 8500 名客户和 1.7 亿用户 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/06/01/odilo-raises-64m-as-its-white-label-e-learning-library-passes-8500-customers-and-170m-users/

电子学习——无论是物理材料的补充还是现场课程；或者正在使用的主要甚至唯一的媒介——现在是教育经历中不可磨灭的一部分；现在，西班牙的一家初创公司建立了一个平台，帮助更多地方提供电子学习，该公司宣布将获得新一轮融资，以进一步扩大业务。

Odilo 是一家总部位于马德里的初创公司，它建立了一个白色标签平台，供企业或组织以 B2B2C 模式建立自己的定制电子学习服务，该公司已获得€6000 万英镑(6400 万美元)的投资。伦敦投资者 Bregal Milestone 领投了这一轮，之前的支持者斯万拉布和 CDTI 也参与了进来。Odilo 没有披露其估值，但它自 2012 年以来一直存在，此前筹集的资金不到 3000 万美元。

你可能不熟悉“奥迪罗”这个名字。创始人兼首席执行官罗德里戈·罗德里格斯(Rodrigo Rodriguez)向我描述 Odilo 是教育领域的“网飞”,因为它的目录很大——该公司迄今为止已经收集了 390 万项内容，其中包括约 300 万本书籍，还有播客、完整课程和来自 6300 家出版商的 43 种语言的其他材料——而且人们可以按需使用这些内容。

然而，与网飞不同的是，它是作为一个标签服务运营的，所以你可能永远不会真正看到奥迪罗品牌。然而，您可能已经使用了它所支持的一个教育门户网站。到目前为止，Odilo 已经在 52 个国家积累了 8500 名客户，总共覆盖了约 1.7 亿用户，其中包括政府机构、图书馆和麻省理工学院等教育机构，也包括谷歌和沃达丰等大型企业客户。

罗德里格斯说，新的资金将用于继续扩大该目录，以及向更多的市场，特别是北美和非洲大陆扩展。

该公司的成长源于科技领域的两大趋势。首先是电子学习的兴起，从最广泛的角度来看，它可以包括学校学习和专业培训，据估计，2021 年这两者之间的一切价值约为 3150 亿美元，并且根据研究人员的说法，未来六年将以 20%的 CAGR 增长。当然，整个趋势是由新冠肺炎加速的，当时学校和其他面对面的学习体验都被关闭了，但这种趋势甚至在疫情之前就已经很大了，鉴于虚拟体验的经济性，它可能会继续存在。

第二个是“无头”系统的兴起:构建为可扩展平台的服务，客户可以选择使用各种 API 和其他工具在其上构建自己的定制体验。这些在其他从头构建起来非常复杂的技术领域非常常见——电子商务后端(例如 Shopify 或 Commercetools)，内容管理系统(例如 Contentful)——现在 Odilo 正在有效地使用相同的模型并将其应用于电子学习。

(需要指出的是，还有其他人也在追求建立平台的想法，让其他人创建可定制的电子学习图书馆:伦敦的 Perlego 将自己描述为电子学习的“Spotify ”,追求类似的东西，并筹集资金来建立它；然而，在这方面，奥迪罗的规模令人印象深刻。)

值得注意的是，谷歌是奥迪罗的客户之一:该公司是世界上无可争议的科技巨头之一，通过其协作多媒体谷歌教室等产品大力推进教育；其谷歌学术学术资料搜索；当然是 YouTube；它的主要搜索引擎等等。Rodriguez 告诉我，它利用 Odilo 为它的人民建立培训和教育图书馆，但它有没有接触过 Odilo，以建立一个更面向消费者的产品？

不，这是他给我的简短回答，但他确实指出，该公司一直在与非常大的技术公司进行谈判，这些公司希望研究这种产品，但可能不想自己投资制造这种产品——至少最初不想——来探索这个想法。

这一点，加上现有的巨大市场，是投资者一直准备投资的两个原因，即使在创业公司融资变得有点棘手的时候。

“奥迪罗是高增长教育内容市场的创新参与者，拥有令人印象深刻的产品系列、庞大的内容目录和一流的数字基础设施，”Bregal Milestone 的负责人 Francesco Canessa 在一份声明中说。“国际化和扩展平台有很大的空间，这是 Bregal Milestone 拥有丰富专业知识的两个价值创造杠杆。作为该领域的领导者，Odilo 拥有强大而有粘性的客户群和独特的产品，这正是我们寻求的业务类型，随着我们的合作，我们对未来的增长前景感到兴奋。”
