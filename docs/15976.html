BlaBlaCar 将收购 Klaxit，这是一项针对日常通勤的拼车服务

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/15/blablacar-to-acquire-klaxit-a-ride-sharing-service-for-daily-commutes/amp/

图片来源:大卫·莱吉奥

BlaBlaCar 宣布计划收购一家较小的法国创业公司 Klaxit。Klaxit 是一种拼车服务，这样你就不用每天开着车去上班了。BlaBlaCar 有自己的通勤服务，名为 BlaBlaCar Daily 。Klaxit 将在收购完成后补充这项服务。

BlaBlaCar 更出名的是它的长途拼车服务。该公司将空的汽车座位与同方向的乘客相匹配。当你想去另一个城市度假或看望家人时，它是火车或公共汽车的替代选择。

许多人也依赖 BlaBlaCar 在小城市之间乘车，因为不是每个小城市都有火车站。随着时间的推移，该公司增加了其他服务，如通过收购 Ouibus 的巴士市场和通过收购 Busfor 的在线巴士售票平台。

2017 年，该公司推出了一款名为 BlaBlaLines 的新应用和服务。这一次，该公司希望将去上班的司机与寻找交通方式的其他员工匹配起来。该应用在推出后不久就更名为 BlaBlaCar Daily。

但是 BlaBlaCar 并不是唯一一家考虑通勤问题的公司。Klaxit 和 Karos 等其他初创公司也在致力于类似的服务。这是因为日常通勤占了汽车出行的很大一部分。然而，大多数人在去上班的时候都是一个人坐在车里。

优化这些乘坐方式是节省日常通勤费用的好方法。这对环境也有好处，地方政府也喜欢这个主意，因为它改善了交通状况。

像这样的服务有几种赚钱的方法。最终用户可以直接支付骑手。公司可以为这项服务做出贡献，因为它可以被视为一项福利。地方政府可以补贴拼车服务，以减少道路基础设施预算。

尤其是 Klaxit 自 2012 年成立以来，已经与当地政府签署了大量合作伙伴关系。超过 50 个地方政府和 350 家公司与该平台合作，促进拼车。2022 年，Klaxit 在其平台上促成了 260 万次乘车。

BlaBlaCar 表示，计划在 2024 年合并 BlaBlaCar Daily 和 Klaxit。虽然它将增加用户群，但仍有很长的路要走，因为在法国每天有超过 1000 万人独自开车上班。
