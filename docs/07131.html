Luminai(前 DigitalBrain)获得 1600 万美元投资，用于类似 RPA 的客户支持工具

DigitalBrain 的创始人有相当多的背景故事。一对移民住在旧金山的一个黑客之家，两人一度通过参加黑客马拉松比赛并获胜来支付账单。当他们想到一个名为 DigitalBrain 的业务想法时，他们能够在 2020 年夏天进入 Y Combinator，从那以后他们一直致力于建立一家公司。

今天，公司创始人的辛勤工作获得了 1600 万美元的 a 轮融资。在此期间，他们还宣布将更名为 Luminai 。

公司联合创始人凯萨瓦·基鲁帕·迪纳卡兰表示，自 TechCrunch 在 2020 年覆盖了的种子期以来，这家初创公司已经取得了很大的进步。他说，最初的产品有大约 10 个功能，但真正令人印象深刻的是类似 RPA 的功能，它将用户的一系列步骤自动化为一次单击。

“当我们第一次构建产品时，我们已经构建了大约 10 个功能。当我们与第一批设计合作伙伴和客户进行反复讨论时，我们发现基本上有一个特定的产品成为了今天的成熟产品，它采用了任何类型的多点击、多步骤流程，只需一次点击，”他告诉我。

Luminai 现在做的是记录一个用户在不同系统之间跳跃的视频。“使用 Luminai 记录器，我们可以跟踪点击和击键，我们的软件能够识别是否有可用的 API，然后在后台复制完全相同的工作流程。”

Dinakaran 说，他的客户将其描述为类固醇上的宏，但他说，它与纯 RPA 工具的真正区别在于当 API 网关可用时连接到 API 网关的能力。“不同的是，我们将 RPA 与传统的基于 API 的集成相结合，”他说。

当没有 API 时，或者当公司出于某种原因不想授予 Luminai 访问网关的权限时，例如如果它是一个金融网关，那么该工具可以使用标准的 RPA 风格的记录技术来遍历工作流。

该公司目前仍在精简运营，只有 11 名员工，尽管他们在 A 轮融资中两面下注，但他说他们有收入，公司 340 万美元种子资金的大部分仍在银行。他说，种子融资意味着他和他的联合创始人不再需要担心食物和租金等基本需求，可以完全专注于业务。

他特别引以为豪的一件事是，他在印度的表亲现在认为成为一名初创企业创始人是一件触手可及的事情。“故事的另一部分是，我的一些堂兄想成为创始人，现在他们看到了某种途径，这在许多方面给了我很多快乐，”他说。

General Catalyst 领投了今天的这轮融资，参与方包括 Moxxie Ventures、stress VC、Craft Ventures、YC Continuity 以及一系列知名的行业天使投资人。该公司现已筹集了近 2000 万美元。
