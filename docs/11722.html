在管理层交接期间，Twitter 取消了面向开发者的 Chirp 会议

在管理层交接期间，Twitter 取消了为开发者举办的 Chirp 会议

Twitter 公司周四晚些时候表示，在管理层过渡期间，Twitter 将取消其专注于开发者的 Chirp 会议。会议定于 11 月 16 日在旧金山举行。

埃隆·马斯克(Elon Musk)上个月接管公司后，公司的产品战略出现了几次高管离职和方向性变化。因此，社交网络在中断十多年后放弃会议回归的计划就不足为奇了。

在一条推文中，Twitter 的开发者相关公告官方账户表示，该公司“正在努力让 Twitter 对所有人都更好，包括开发者”，它可能很快就会分享一些关于这个主题的新闻。

该公司的开发产品负责人阿米尔·谢瓦特(Amir Shevat)用一条神秘的消息回应了这一消息，称:“变革之风。”

6 月，帕拉格·阿格拉瓦尔领导的 Twitter 宣布将在 11 月恢复 Chirp 会议。该公司还在发起了一场开发者竞赛，展示其新 v2 API 的创造性用例，不同类别的获胜者可以获得 10，000 美元的价格，并可以免费使用该 API 的企业级一年。它说，虽然会议将不再举行，但社交网络仍将宣布竞赛的获胜者。

Twitter 第一次举办 Chirp 是在 2010 年，但第二年就放弃了。虽然在过去的十年里，该平台与开发者的关系，但它近年来一直试图通过新程序和更新的 API 赢回社区。更重要的是，该公司去年向学术研究人员开放了 API 访问。

今年早些时候，它推出了一个名为 Twitter Toolbox 的程序，突出了一些第三方应用。当时，Shevat 说该公司对探索像 Twitter 自己的应用商店这样的模式持开放态度。

上周，Twitter 开放了新的端点来通过 v2 API 引导消息，这使得第三方应用能够为用户提供更好的 DM 体验。

目前还不清楚在马斯克时代，面向开发者的 Twitter 会是什么样子。特斯拉首席执行官已经表示，公司将优先考虑工程，因此开发人员可以希望他们将更好地使用公司的工具。

但在那之前，围绕一些产品的未来前景仍会有疑问，比如在社交网络的高级用户中很受欢迎的 TweetDeck。公司去年开始测试新版本的 TweetDeck，希望使其成为付费产品。今年早些时候，应用研究员简·满春·王的发现表明， Twitter 可能会将 TweetDeck 加入其订阅伞 Twitter Blue。但是，由于马斯克正在大规模改革订阅计划，因此无法确定 TweetDeck 是否仍有一席之地。
