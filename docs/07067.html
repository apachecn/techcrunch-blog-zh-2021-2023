Curio 让 NFT 的死忠们可以在一个浏览器标签中浏览、追踪和购买所有东西

原文：https://web.archive.org/web/https://techcrunch.com/2022/06/08/curio-lets-nft-die-hards-browse-track-and-buy-all-in-one-browser-tab/

致力于 NFT 世界的人们可以使用大量的分析和浏览工具，但是它们都不能真正协同工作。至少，新创公司 Curio 是这样看待事情的。

Curio 利用了几个不同的数据源，为浏览和跟踪 NFTs 的价值提供了一个强大的分析工具。

联合创始人 Chrys Bader 和 Sean Dadashi 大约两年前开始研究 Curio，但当时还不叫 Curio。他们开发了一款名为 Ikaria 的抗孤独应用，旨在以一种脆弱而真实的方式将人们聚集在一起。这款应用的采用速度很慢。大约在同一时间，Dadashi 和 Bader 开始进入 web3 领域。

巴德说:“就激励机制以及推动和激励人们的因素而言，Web2 已经相当饱和了。“一切都已经做了，或多或少。Web3 展示了一个全新的设计空间，在这个空间里，我们可以创造新的激励机制，在比现在更基础的层面上激励人们。”

他们试图将这种想法应用到他们的社交应用中，但进展并不顺利。巴德说:“如果你试图在人际关系中加入激励因素，事情就会变得很奇怪。”

从那时起，他们开始思考他们能给 web3 空间带来什么。他们对 NFT 特别感兴趣，但发现浏览它们、理解它们和追踪它们的价值的过程冗长而烦人。它需要多个浏览器窗口和信息交叉引用。

因此，他们着手建立一个一体化的场所，让人们不仅可以观看艺术品，还可以追踪这些非艺术作品的价值，做出更好的决定。

Curio 从 OpenSea API 和区块链本身获取信息，并计划很快整合来自 LooksRare 的数据。这家初创公司美化了这些数据，允许用户跟踪稀有度、楼层、每次收藏的平均价格、数量等。用户可以深入某个集合，获得完整的信息，甚至可以根据特征进行过滤。例如，用户可以在 Bored Ape collection 中搜索 3D 眼镜和棕色皮毛，以查看地板深度、销售散点图并跟踪实时销售。

那些只想看看今天有什么的人也可以通过最佳价值来浏览，或者看看今天有什么趋势。

Curio 目前已经有 1000 名用户处于测试阶段，今天将向公众推出。作为发布的一部分，也是收入战略的一部分，这家初创公司正在提供一张名为 Genesis 的 NFT 通行证，让人们可以终身使用 Curio 的高级工具。

该公司还通过其 Discord bot 创造收入，该公司允许服务器管理员向其 Discord 社区提供按需分析。

亚历克西斯·奥哈尼安的《七七六》为 Curio 带来了 370 万美元的种子资金，使该项目的总资金达到 510 万美元。
