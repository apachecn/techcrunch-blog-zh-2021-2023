欧盟无条件批准微软-Nuance TechCrunch

欧盟竞争监管机构已经批准微软以 197 亿美元收购转录技术公司 Nuance，该交易于今年早些时候在宣布。

欧盟今天表示，已经得出结论，如果收购继续进行，该地区不存在竞争问题，无条件批准。

该交易于 11 月 16 日通知了欧盟委员会的监管机构。

虽然微软-Nuance 已经获得了欧盟的批准，但英国竞争和市场管理局刚刚开始了自己的初步调查，因此一些地区审查仍在继续。

在欧盟方面，委员会的调查研究了 Nuance 和微软在转录软件市场上的横向重叠——发现两家公司提供了非常不同的产品(分别为最终用户提供现成的软件，而 API 则针对希望在其应用程序中添加语音识别技术的开发人员)。

这也决定了合并后的实体将继续面临来自其他参与者的“强大”竞争。

欧盟也研究了微软云计算服务和 Nuance 医疗保健下游转录软件之间的垂直联系，但发现该领域的竞争转录服务提供商并不依赖微软的云计算。

根据委员会的说法，这些类型的转录服务提供商也不是云计算的主要用户。

其调查考虑了 Nuance 的软件(仅适用于 Windows)和许多微软产品之间的联合关系，但认为合并后的实体没有能力和/或动机在(医疗保健)转录软件市场上排除竞争对手；企业通信服务；CRM 软件；生产力软件；和 PC 操作系统。

同样，欧盟决定合并后的实体仍将面临激烈的竞争。

也许最有趣的是，委员会查看了 Nuance 软件转录的数据的使用情况。

这很有趣，因为医疗保健数据是高度敏感的，而微软虽然不是广告技术领域的大玩家，但确实有增长这方面业务的雄心。就在今天宣布收购美国电话电报公司的广告技术公司 Xandr(前身为 AppNexus ，以加强其数字广告业务。

此外，最近几天，甲骨文——一个已经拥有庞大数字营销业务的科技巨头——已经发出了自己在健康方面的宏伟设计的信号——宣布收购美国电子健康记录(EHR)系统提供商 Cerner 。

嗯，广告技术公司收集健康数据的前景让很多人对隐私感到不安。

尽管如此，由于现有的“合同限制”和地区数据保护法规，委员会对微软-Nuance 数据方面的评估给了它一份健康证明。

虽然这一分析主要是从竞争的角度来看，但值得注意的是，数据保护在欧盟反垄断评估中(又)被大声疾呼。(早先的例子与欧盟对谷歌广告技术的(持续)审查有关；去年委员会批准了谷歌对 Fitbit 的收购，尽管——在那种情况下——附带的条件包括限制谷歌使用 Fitbit 健康数据做广告，这引起了隐私倡导者的大量批评。)

“委员会得出结论，Nuance 只能使用这些数据来提供其服务，”欧盟在一份关于微软-Nuance 许可的新闻稿中写道。“由于合同限制和数据保护法规，它不会被任何其他公司使用，也不能用于任何其他目的。”

欧盟反垄断部门还得出结论，鉴于“重要的转录信息通常存储在第三方应用程序中，如电子健康记录(EHR)系统，该系统结合了来自多个来源的数据，而不是 Nuance 的碎片化语音数据”，访问 Nuance 的数据不会给微软带来优势，使其能够将竞争对手的医疗保健软件提供商拒之门外。

这一趣闻引发了一个问题，即甲骨文收购 EHR 系统供应商 Cerner 的交易是否会面临欧盟竞争监管机构更多的尖锐问题——如果/当他们考虑这笔大型科技医疗保健行业交易时。

尽管 Cerner 相对缺乏地区客户——它去年出售了一些欧洲投资组合——可能会减少或限制任何欧盟担忧的范围。
