新工具“Movetodon”让你更容易在乳齿象| TechCrunch 上找到你的 Twitter 好友

原文：https://web.archive.org/web/https://techcrunch.com/2022/11/29/a-new-tool-movetodon-makes-it-easier-to-find-your-twitter-friends-on-mastodon/

在埃隆·马斯克收购 Twitter 后，开源 Twitter 替代品乳齿象出现了可观的增长，在交易完成后一周多一点的时间里突破了月活跃用户 100 万的新里程碑。但是对于第一次来到乳齿象的用户来说，T4 面临的众多挑战之一是重建他们多年来在 Twitter 上建立的朋友和粉丝网络。一款名为 Movetodon 的新工具旨在让用户通过最少的工作轻松找到并关注他们在开源社交网络上的 Twitter 朋友，从而使从 Twitter 到乳齿象的过渡更加容易。

在乳齿象上已经有几个寻找 Twitter 好友的工具，像 Debirdify 、 Twitodon 和 Fedifinder 。然而，目前的一些工具要求用户连接他们的 Twitter 帐户，扫描他们的网络，然后将数据导出到 CSV 文件中，以便导入到 Mastodon 中。如果你想简单地导入所有拥有乳齿象账户的 Twitter 好友，这是很有用的，但如果你想有一个新的开始，这不允许你轻松地管理你的网络。

Movetodon 的创建者，德国软件开发商 Tibor Martini 说，他自己也使用过这些工具，并承认它们符合他的目的。但他认为整体体验有点欠缺。

他在一封电子邮件中告诉 TechCrunch，“有几天，我使用了其他一些工具，即 Fedifinder 和 Debirdify，发现它们很棒。”。“他们做了需要做的事情——从 Twitter 导出你朋友的账户，然后导入乳齿象。然而，这样做的过程并不复杂。“你必须下载一个 CSV 文件，并将其导入乳齿象，”马丁尼解释道。此外，一些工具根本没有任何样式，所以看起来非常“原始”这些方面使得那些已经在努力寻找‘正确的’乳齿象标本的人更加困难，”他补充道。

Martini 表示，他希望开发一个工具，让不了解技术的人能够更容易地在 Mastodon 上找到他们的朋友，包括他们是否在移动设备上访问该服务。

他在 10 月底开始为 Movetodon 的概念工作，然后在 11 月中旬左右开始编码。

虽然仅在五天前推出，但 Movetodon 已经成为那些试图逃离 Twitter 的人的热门工具。Martini 表示，他的服务器日志显示，仅在过去一周，就有超过 50，000 名用户尝试过 Movetodon。他关于这项服务的帖子在乳齿象上被转发了近 2000 次，在推特上转发了数百次。

为了利用 Movetodon ，用户必须同时登录 Twitter 和 Mastodon 这两个应用，并授权 Movetodon 开始使用。然后，该工具会自动生成一个也在乳齿象上的朋友列表。在测试中，我们还发现 Movetodon 通过暂停列表生成并显示倒计时(以秒为单位)来帮助处理它遇到的 API 限制。

图片来源: Movetodon in action

列表生成完成后，您可以通过单击按钮来关注单个用户，也可以单击一次关注所有用户。您还可以根据用户加入 Mastodon 的时间对列表进行排序，以使新帐户弹出到列表的顶部。

开发者表示，他的工具通过查看 Twitter 上的相关字段来工作，用户经常在这些字段中分享他们的乳齿象账户信息，如他们的简历、用户名、位置字段、URL 字段或他们的推文中。然后使用 Regex 提取可能的乳齿象句柄和 URL。有了乳齿象 API，它就可以确定这个句柄实际上是乳齿象帐户，还是其他东西——比如电子邮件地址。该工具还会获取你当前在 Mastodon 上所有朋友的列表，以显示你是否已经关注了对方。

Movetodon 访问的数据从不存储在其服务器上，两个应用程序连接都要求有限的权限。例如，在 Twitter 上，Movetodon 只能阅读信息，不能发布或关注。在乳齿象上，Movetodon 要求有限的写权限，以允许用户关注平台上的其他人。

虽然 Martini 在 T4 的日常工作是德国出版商 stern 的社交媒体团队领导，但他表示，他可能会根据用户反馈进一步开发 Movetodon。例如，他添加了“关注全部”按钮来响应用户的请求。用户现在也在询问他们是否可以使用这个工具从他们的 Twitter 列表中查找账户。

即使将来其他 Twitter 替代品增加了 API，Martini 也不认为他会为它们开发工具，因为他更喜欢乳齿象。(Movetodon 显示他在 2245 天前加入了开源社交网络！)

“就个人而言，我看到了乳齿象的一些优势:它已经拥有庞大的用户群，对隐私友好，并且在管理和开发平台方面已经有多年的经验，”他说。
