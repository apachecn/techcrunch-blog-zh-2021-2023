ThreatX 筹集了新一轮资金来保护 API 和 web 应用

原文：https://web.archive.org/web/https://techcrunch.com/2022/08/17/threatx-raises-a-fresh-round-of-capital-to-protect-apis-and-web-apps/amp/

图片来源:安德烈·奥努里延科/盖蒂图片社

ThreatX 是一家主要向企业客户出售 API 保护服务的供应商，今天宣布它在由哈勃克成长伙伴公司牵头的 B 轮融资中筹集了 3000 万美元，Vistara Growth、. 406 Ventures、Grotech Ventures 和 Access Venture Partners 也参与了融资。随着新资金的增加，ThreatX 的总收入增加到 5200 万美元，首席执行官 Gene Fay 告诉 TechCrunch，ThreatX 将“加速”平台开发投资，同时扩大销售和营销计划。

尽管当前的宏观经济陷入困境，这次提价凸显了投资者对网络安全业务净回报的持续信心。根据 PitchBook 的数据，虽然有一些证据表明融资已经开始放缓，但网络安全初创公司在 1 月至 6 月间筹集了 24 亿美元。最近，保护 API 免受外部攻击的公司尤其富有成效，像 Ghost Security 和 Corsha 这样的初创公司筹集了数千万美元的资本。

ThreatX 于 2014 年由 Bret Settle 和 Andrius Useckas 共同创立。在开始 ThreatX 之前，Settle 是 BMC 的企业架构副总裁；Useckas 曾与 Bret 在 BMC 共事，当时他是一名企业安全架构师。两人也是 Corporate Express 的同事，该公司于 2008 年被史泰博收购，在那里，乌塞卡斯作为外部笔测试员加入进来。

2020 年被任命为 ThreatX 首席执行官的 Fay 表示:“经过几年的合作，Settle 和 Andrius 在保护 BMC 应用程序组合的解决方案方面看到了市场上的巨大差距。“可用的产品需要无休止的调整和规则编写，并返回大量的误报。通过这一切，在这个领域创新的概念——以及威胁——诞生了。”

ThreatX 为第一方和第三方 web 应用提供 API 保护、bot 和 DDoS 缓解以及传统 web 应用防火墙(WAF)。该平台构建威胁参与者的概况，利用检测和关联引擎来显示哪些参与者正在积极攻击，哪些可能构成最大的威胁。

图片来源: ThreatX

Fay 认为 ThreatX 主要与两类网络安全供应商竞争。首先是更新的 API 可观察性工具，比如 Salt Security 和 Noname。第二种是 bot 管理平台，如 Cequence 和 WAF 播放器，如 Akamai、F5 和 Imperva，它们通常依赖于对 web 应用程序和 API 应用基于规则的保护。

Fay 认为，前一组 bot 管理公司和 WAF 供应商——倾向于提供通过收购整合在一起的能力，因此它们的集成程度较低。至于后者 API observability 工具——Fay 声称，它们通常不提供 web 应用或 bot 保护，并且需要离线分析，这就排除了实时阻止攻击的能力。

“底线是，为了保护 API，你必须能够实时阻止攻击，”费伊说。“通过观察获取数据并在事后进行分析可能很有趣，但从直接的安全角度来看，这并没有什么作用。对于我们的客户来说，第一要务是保护—实时、全天候的保护。这是我们向客户提供的价值主张。”

不管有没有实时保护，API 攻击确实是一个日益增长的网络威胁。Gartner 预测到 2022 年，API 攻击将成为最常见的攻击媒介，导致企业网络软件的数据泄露。

“新冠肺炎·疫情加速了 API 的使用，因为公司正在考虑如何提供新的服务来从客户那里传递价值并获得收入，”Fay 补充道。“随着人们——无论是消费者还是专业人士——转向技术来完成更多工作，对 API 和 web 应用程序的依赖也在大幅增长。这反过来又增加了这种情况下的安全性需求，这为 ThreatX 提供了大量的机会。

当被问及财务状况时，费伊表示异议，他说 ThreatX 目前有“超过”100 名客户。他拒绝透露任何名字。

当被联系到发表评论时，哈勃克成长伙伴公司的普通合伙人汤姆·罗伯茨在一份声明中说:

API 是各种规模的企业的战略重点，并已成为威胁参与者的主要目标。如今，企业面临着持续不断的威胁，需要能够实时识别和响应攻击的 API 和 web 应用程序保护功能。这种对“实时攻击保护”的需求正在推动 API 安全市场向积极的方向发展。基于 ThreatX 强大的客户吸引力和独特的产品功能，我们相信该公司有能力迎接这一转变，成为寻求保护其攻击面的企业的重要合作伙伴。
