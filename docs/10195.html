新加坡的 KNN3 希望为分散应用程序实现社交发现 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/09/16/singapore-knn3-social-graph-database-web3/

有很多初创公司试图理解区块链应用程序产生的数据的爆炸式增长。南森得到 a16z 的支持，为密码投资者提供链上数据分析。该图为开发人员提供了查询区块链数据的 API。最近获得风投认可的是 KNN3，这是一家总部位于新加坡的初创公司，致力于帮助开发人员理解区块链各地的关系数据。

当我们进入社交网络时，首先出现的通常是关注建议。该信息基于对我们的数字足迹历史的分析。KNN3 想在 web3 中做同样的事情，通过建立图形数据库来分析用户的关系、状态、成员和其他链上行为。

区块链的数据空间已经相当拥挤，联合创始人 Thomas Yu 承认，但仍有空间提供更专业化的服务。南森和 web3 开发平台炼金术以集中 SaaS 产品的形式出现。俞认为，这个图形是“可编程的”，但它支持的数据结构相当“有限”。

这就是为什么俞和他在 BTC 中国的前同事刘(音译)着手开发 KNN3，一个无需许可(因此是去中心化的)的工具，供开发者从跨的用户数据中获取洞察力。

KNN3 开始瞄准亚洲面向消费者的 dApps。虽然 web3 的大部分基础设施建设都发生在西方，但亚洲通常被视为消费应用的创新中心，GameFi 平台的流行凸显了这一点，如 Axie Infinity 和 StepN 。KNN3 的一个更知名的客户是 Mask Network，，它使用户能够在 Web 2.0 服务上发送加密货币，并且现在正在使用 KNN3 的技术建立一个分散的身份系统。

相比之下，在美国，KNN3 计划追赶像 Chainlink 这样面向企业的组织，这些组织将被称为“甲骨文”的现实生活数据输入智能合同，也是 Yu 曾经工作过的地方。KNN3 正在权衡一款新产品，该产品将提供基于 chain link Oracle 的云服务，最终将允许开发人员构建和运行分散的应用程序和智能合同，而无需担心“基础数据层”。

“这意味着开发者可以使用像谷歌云这样的 web2 工具，但实际上是在构建 web3 工具，而不是编写智能合同并使其跨链工作。KNN3 使用 oracles 构建了不可信的基础设施，开发者只需在其中运行一个容器就可以了。

KNN3 表示，它已在一轮种子资金轮中筹集了 240 万美元，由专注于加密的风险投资公司 HashGlobal 和中国企业集团复星国际(Fosun International)前副董事长兼首席执行官梁信军牵头。这轮谈判于 4 月份结束，但直到本月才宣布。

种子投资也有一长串参与投资者——这似乎是区块链初创公司早期结盟的一种流行策略。它们包括 Mask Network、MetaWeb Venture、Eniac Venture、Tess Venture、分层资本、基本面实验室、Incuba Alpha、Zeuth Venture、Cogitent Venture、Atlas Capital 不可能金融、RSS3、ShowMe 和 ETHsign 的联合创始人和波特李。

KNN3 目前在新加坡、中国、欧洲和美国雇佣了一个 24 人的团队。有了这笔资金，它希望从硅谷吸引更多的科技人才。“现在是熊市招聘的好时机，因为许多竞争对手都在裁员，”俞说。
