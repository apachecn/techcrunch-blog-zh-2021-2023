Superplastic 筹集 2000 万美元扩大其卡通影响力 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/10/20/superplastic-guggimon-janky-virtual-celebrities-series-a/

Ello 和 Kidrobot 的创造者的一家公司宣布新一轮 2000 万美元的融资，以建立一个虚拟的虚拟世界，其中充满了疯狂但可爱的卡通影响者。

超塑的投资者包括谷歌风投、Index Ventures、Founders Fund、Craft Ventures、Day One Ventures、Galaxy Digital、Mantis VC/The Chainsmokers、Kakao Entertainment、LINE Friends 以及 Cyan Banister、杰瑞德·莱托、贾斯汀汀布莱克(Justin Timberlake)和 Scooter Braun 等个人投资者。2000 万美元的首轮融资使该公司的总资本增加到 3800 万美元。

超塑的提议很奇怪。该公司拥有一个由“合成巨星”组成的动画 collab house，这些角色生活在社交媒体上，通过与名人、标志性时尚品牌和游戏平台的合作，在其他任何地方出现。该公司已经在 Instagram 和抖音上积累了数百万粉丝，在 NFT 上销售了数百万，并建立了一个熙熙攘攘的 Discord 社区，围绕着一系列暴力、时尚的卡通人物。

“一切都是由角色和故事驱动的……在某种程度上，超塑公司的管理团队也是如此，”超塑公司创始人保罗·巴德尼茨告诉 TechCrunch。

Budnitz 的目标很高，渴望建立自己的“糟糕的迪士尼”超塑的核心人物之一古西蒙不时折磨他的对手，并制作虚拟的古驰齿轮模型——这只是超塑众多利润丰厚的合作伙伴之一。Guggimon 也出现在堡垒之夜最近的一季中，不仅仅是作为一个皮肤，而是作为一个成熟的特色角色，出现在超人旁边。与此同时，超塑与 LINE 和 Kakao 的关系已经帮助这些角色在亚洲市场获得了立足点，而 Kidrobot 从未成功过。

鉴于 Budnitz 与 Kidrobot 一起塑造 IRL 收藏品世界的经验，超塑性也深入到 NFT 现象中，这是有意义的。但他并不总是一个信徒。Budnitz 告诉 TechCrunch，他起初对这种现象不屑一顾。“在 NFT 的事情上，我就想，不，这太愚蠢了。突然我他妈的一下子全明白了。”(Budnitz 最初对比特币也有同样的感觉，后来在一次活动中遇到了一位留着“长长指甲”的神秘密码传播者，他说服他进行投资。)

今年早些时候，超塑与佳士得合作，在这座历史悠久的拍卖行举办了一场 NFT 拍卖会，其中有两个角色在这个过程中炸毁了大楼，这是一个恰到好处的混乱虚构故事。

图片来源:超塑

从 2 月份开始，Superplastic 开始涉足数字收藏品的疯狂世界，现在它正在运营一个名为 Jankyverse 的成熟的基于 Discord 的社区，成员们在这里展示他们获得的实物和数字小雕像。与大多数大量生产数字化身的 NFT 项目不同，Superplastic 预先开发了它的角色和虚拟世界，而不是在利用推动 OpenSea 天文价格的投机热潮后将它附加上去(或承诺将它附加在)。

超塑正在做 NFT 的事情，但这只是最终平台不可知愿景的一个渠道。该公司致力于扩展其跨越媒介的虚构混沌宇宙，它已经在玩具销售、服装、非功能性游戏和许可方面赚了钱，比如与 Epic Games、虚拟角色先驱街头霸王乐队和古驰的合作。

角色可能无处不在，但 Budnitz 非常清楚，他的公司不会出售部分知识产权——即使是为了换取一个主要的合作伙伴关系。该公司在去年不合时宜地倒闭之前拒绝了 Quibi，还拒绝了两家想永久拥有超塑角色的流媒体工作室。

有了这笔新投资，超塑将按照计划于 2022 年推出一部动画长片，由不安的另类迪士尼角色主演。该公司还计划明年初在纽约和迈阿密开设零售店，将其数字收藏品与店内专为 NFT 持有者提供的秘密区域等独家 IRL 体验联系起来。

“我对连接真实世界更感兴趣，而不仅仅是做那些虚拟的东西，”Budnitz 说。
