哈苏拉筹集了 1 亿美元

原文：https://web.archive.org/web/https://techcrunch.com/2022/02/22/graphql-developer-platform-hasura-raises-100m-series-c/amp/

图片来源: gonin / Getty Images

Hasura ，一家流行的开源 Hasura GraphQL 引擎背后的公司，该引擎可以将几乎任何数据库转换成 GraphQL API，今天宣布它已经筹集了 1 亿美元的 C 轮融资，由 Greenoaks 牵头。之前的投资者 Nexus Venture Partners、Lightspeed Venture Partners 和 Vertex Ventures 也参与了这轮投资，这使得该公司的总资金达到 1.365 亿美元，估值达到 10 亿美元。

正如 Hasura 首席执行官兼联合创始人 Tanmai Gopal 告诉我的那样，自从筹集了 2500 万美元的 B 轮融资后，该公司在过去的 18 个月中增长加速。随着开发人员越来越多地承担在更广泛的数据源上构建应用程序的任务，他们正在寻找像 Hasura 这样的服务。

“企业开发者完全被复杂性所困扰，”他说。“他们只是完全陷入僵局。他们说，‘伙计，我在这个遗留数据系统中有一千个模型。我有一堆新东西，在这个新系统中丰富了 ML/AI。我有这些传统的 API。我真正想做的是更新我的应用程序屏幕，以便有人可以看到他们的账单历史。[……]Hasura 所做的就是说:我们让这种数据访问自助化。我们将自动完成这项工作。我们将为您提供一个灵活、安全且您喜爱的 API——这是一个 graph QL API——并且我们将实现自动化。"

这是一个明显引起共鸣的信息。Gopal 指出，该公司的托管云产品去年增长了 10 倍。它的客户现在包括空客、Atlassian、BBVA、Netlify、Pipe、Swiggy 和沃尔玛等，Gopal 指出，该公司看到越来越多的传统公司正在寻求实现其开发堆栈的现代化。像许多以数据为中心的公司一样，Hasura 尤其关注金融服务领域，例如制造业和医疗保健行业。

Hasura 创始人(L-R)首席执行官 Tanmai Gopal 和首席运营官的 Rajoshi Ghosh。图片来源: Hasura

Hasura 的联合创始人首席运营官·拉乔希·戈什指出，虽然企业倾向于使用该产品来获取大量数据并加速他们的数字化转型项目，但对于创业公司来说，该服务往往提供了一种不同的价值主张。“在创业方面，我们听到很多创业公司都在说，超越竞争对手，”她告诉我。“事实上，我们只想充当为您提供优势的那一层。你只想处理业务逻辑，你只想建立你的用户体验。”因此，尽管这些初创公司可能不需要管理那么多数据，但他们仍然需要将许多不同的数据源和 API 集成到自己的产品中，即使这并不是他们最终与竞争对手的区别。

“托管 GraphQL 服务器完全改变了开发人员访问数据的方式，并且正在快速构建应用程序，”Greenoaks 合伙人 Neil Shah 说。“当我们与这些客户交谈时，他们谈到这实际上是一个根本的瓶颈。这不仅仅是一个瓶颈，也是他们在构建产品时面临的基本瓶颈之一。”

他告诉我，大约在公司宣布 B 轮融资时，Shah 开始与 Hasura 团队建立关系，这有助于他在投资前很好地了解团队和产品。沙阿指出:“很难找到既服务于全球 100 强企业又服务于初创企业的公司。”。

Hasura 计划使用新的资金来加速其 R&D 工作，同时也在全球范围内扩展其走向市场的活动，并投资于开源用户和贡献者社区。
