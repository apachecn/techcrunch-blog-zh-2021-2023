谷歌推出网站版儿童阅读教育应用

原文：https://web.archive.org/web/https://techcrunch.com/2022/08/09/google-launches-a-website-version-of-its-read-along-education-app-for-children/

谷歌推出了儿童阅读教育应用的网站版本

谷歌为其 Read Along 应用程序推出了一个网站，以鼓励儿童练习阅读。该网站是作为公测版推出的，可以在桌面和 Android 上与 Chrome、Firefox 和 Edge 浏览器兼容，并很快支持 iOS 和更多浏览器，如 Safari。

该网站的概念类似于该应用程序:通过使用谷歌的语音识别和文本到语音转换文本阅读故事，儿童可以学习说英语、葡萄牙语、西班牙语、印地语、马拉地语、孟加拉语、泰米尔语、泰卢固语、古吉拉特语和乌尔都语等语言。一个名为 Diya 的虚拟助手在检测到孩子们在挣扎时会帮助他们发音。孩子们也可以在说不认识的单词时向 Diya 寻求帮助。

该公司声称，就像这款应用一样，所有的语音识别过程都在本地浏览器上进行，不会向其服务器发送数据，以保护儿童的隐私。另外，整个体验是无广告的。

虽然该应用程序具有离线使用的优势，但该网站可以帮助手机存储容量低的人或拥有台式电脑的学校。网站和应用程序之间的另一个主要区别是，后者无需任何登录即可工作。新网站要求在同一台设备上登录谷歌账户，以跟踪不同孩子的进度。

谷歌于 2019 年在印度首次推出了这款名为 Bolo 的应用，支持印地语和英语，并于 2020 年为更广泛的受众重新命名为 Read 和其他语言支持。这家搜索巨头指出，自推出以来，该应用程序已经帮助超过 3000 万儿童阅读了超过 1.2 亿篇故事。

该公司表示，它正在与新的内容提供商合作，为该平台添加更多的故事。

“除了网站发布，我们还在增加一些全新的故事。它在一份声明中说:“我们已经与两家知名的 YouTube 内容创作者楚楚电视台和 USP 工作室合作，将他们的一些受欢迎的视频改编成故事书格式。”

该公司还与总部位于印度的 Kutuki learning app 合作，将他们的英语和印地语字母书和儿童语音书作为 Read Along app 中的故事，将于今年晚些时候推出。
