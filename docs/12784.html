杰克·多西的比特币项目 TBD 取消了将“Web5”注册为 TechCrunch 商标的计划

原文：https://web.archive.org/web/https://techcrunch.com/2022/11/30/jack-dorseys-bitcoin-project-tbd-kills-its-plan-to-trademark-web5/?utm_source=Telegram&utm_medium=iXNews&utm_campaign=PerspectiveIX.com

一项计划，将看到一个公司实体商标的术语“网络 5”已被搁置后，许多反弹。周二深夜， TBD ，杰克·多西支付公司 Block ( 前身为 Square )，专注于比特币的部门，宣布了一项考虑不周的计划来注册这个术语，指的是 TBD 最近几个月一直在推动的未来网络的去中心化、专注于隐私的迭代。在其声明中，TBD 解释说，将“Web5”注册为商标将保护该术语不被滥用，并确保其“按预期使用”

然而，这一举措并没有得到 TBD 社区的认可，因为许多人反对自由和开源技术需要这样一个看门人的想法。

“寻求开源协议的保护？一名推特用户在回应 TBD 的推特时问道。另一个嘲笑，“wow web5 是如此分散，一个集中的实体可以控制公关和 IP。”

“欢迎来到真正的去中心化网络。当转发该公告时，另一名用户评论道:“你需要得到我们的许可，才能使用‘去中心化网络’这个术语。”。“再也分不清真实和讽刺了。”

其他评论似乎也嘲笑这个想法，我们甚至必须命名和标记网络的每一次迭代— ,就像现在对 web3 所做的一样。

“Web4 怎么了？，"问一个。“好吧，听我说完:web SEVEN，”另一个人开玩笑说。

根据 TBD 的说法，将“Web5”注册为商标的计划产生于它看到该术语被应用于“与我们制定的 Web5 原则截然相反”的其他产品和服务的问题 TBD 认为，换句话说，骗子和诈骗犯不恰当地使用了这个术语。例如，人们一直试图出售类似“Web5”令牌的东西，或者将 NFT 作为“Web5”出售。

该公司表示，为“Web5”寻求保护将允许 TBD 防止对该术语的含义产生混淆，并捍卫其原则。它的计划是以后允许“Web5”的商业和非商业用途——只要参与者同意维护其主要属性，包括其去中心化原则、身份解决方案和开源协议。

“最终，我们希望建立一个公司、个人和其他利益相关者的联盟来维护这些标准，”TBD 最初的声明称。

在随后的推文中，TBD 负责人迈克·布洛克 回应了的抱怨，指出这样的计划在这个领域并不罕见。他说，Linux 是有商标的。自由软件基金会也持有商标。虽然有些人同意这一点，但其他人认为这一举措将与 Web5 的整体去中心化原则相违背。

“这么点集权，懂了，”嘲讽一用户。

几个月来，TBD 一直在推广术语 Web5，解释 Web5 技术如何将分散的身份和数据存储带到应用程序中，同时将数据和身份的所有权归还给个人。这一概念旨在解决当今网络中身份和个人数据成为第三方财产的问题。理论上，Web5 将允许用户使用他们分散的身份与应用程序连接，而不是不断地为他们想尝试的服务创建新的配置文件。如果他们想更换应用程序，用户可以使用他们的社交角色，包括他们的联系和关系。

但是在公布其注册“Web5”商标的计划仅六个小时后，TBD 宣布由于社区的强烈反对，其计划已被取消。

“……我们在机构群体中听到了强烈的声音，他们担心商标法可能会被滥用，从而破坏去中心化的使命。该公司写道:“我们听到了你的声音”，并承认此举可能会破坏人们对其使命的信任。

"因此，我们暂停和取消我们先前宣布的计划，另行通知."

除了已经发布到推特上的内容，TBD 拒绝发表更多评论。
