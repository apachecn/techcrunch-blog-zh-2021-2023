公开你的范围 3 排放，你这个懦夫

如果你想了解哪些公司认真对待碳排放问题，哪些公司不认真，看看提交给美国证券交易委员会的关于其提议的气候规则的公开评论。

你可以通过一家公司对所谓的范围 3 排放的立场来判断它是否是认真的。根据业务的不同，范围 3 的排放可能占公司碳足迹的很大一部分。这种排放可能是由公司不拥有或控制的活动和资产造成的，如租赁的办公场所、商务旅行或产品的报废处理。当客户使用他们的产品时，比如有人驾驶他们的汽油动力 SUV 时，也可能发生这种情况。

简而言之，如果你的公司真的想为气候变化做些什么，它可能应该估计一下它的范围 3 排放量。如果它在制造可持续发展的噪音，至少它不应该破坏使范围 3 披露成为标准的努力。

这就是为什么 SEC 网站上的评论会成为有趣的读物。从沃尔玛和贝莱德到富达、Gap、埃克森美孚和西南航空等公司都明确表示，他们宁愿不披露自己的范围 3 排放，即使 SEC 提供了安全港条款来限制责任。这些公司实际上是在说，他们没有认真对待气候变化，没有充分理解——并披露——他们自己对气候变化的影响。

还有很多很多公司采取了类似的立场，我在这里就不一一列举了。那么我为什么要挑出这些呢？沃尔玛，因为它是世界上最大的零售商。贝莱德和富达，因为他们是 T2 第一和第三大资产管理公司。埃克森美孚，因为它是最大的非国有石油公司。Gap，因为该公司声称它“感到有道德责任将我们的目标和战略与最佳科学和行业实践相结合”，根据其自己的气候价值观页面。西南航空，因为无论你用哪种标准衡量，它都是美国最大的航空公司之一。

异议和延迟

反对披露 Scope 3 数据的论点通常分为三类:公司抱怨数据太不可靠或不确定，太难获得，或者会让他们面临诉讼。

第一种感觉像是典型的 FUD 竞选——恐惧、不确定和怀疑。富达首席法律官 Cynthia Lo Bessette告诉 SECScope 3 的数据是“推测性的、新生的、不可靠的，目前没有标准来确保数据的一致性和可比性。”
