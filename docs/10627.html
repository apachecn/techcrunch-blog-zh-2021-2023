新的 Echo Dot 也是一个 Eero TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/09/28/the-new-echo-dot-is-also-an-eero/

你已经知道自己是不是亚马逊 Echo 类型的人。如果是这样的话，你可能正在看着一个，想着“我应该升级那个东西吗，还是它毫无意义？”可能是后者——除非你也有亚马逊的 Eero mesh Wi-Fi 产品，在这种情况下:它可能仍然毫无意义，除非它相当旧。在这种情况下，你可以得到一个新的或很少使用的点，它现在将充当 Wi-Fi 范围扩展器。

新的 Dot 和其他 Echo 产品在亚马逊今天的硬件活动上宣布；Dot 进行了轻微的重新设计，配备了更好的扬声器和几个怪物主题的印花，适合放在孩子们的房间里。

没有人真的期望这么小的设备除了有用的声音之外还能提供什么，所以这部分升级并不特别引人注目。但下一个部分很有趣:最新的 Dot 会自动连接到你的 Eero 网络，并扩展其范围。第四代 Dots 和球形回声也将在不久的软件更新后获得这一能力。(明确地说，有一个简单的设置过程，但它将作为网络的一部分工作。)

它不会像专用路由器那样扩大覆盖范围，但有一些技巧是固定的供电收发器可以做到的，而手机或笔记本电脑不能做到，这意味着即使信号很弱，它也可以充当中继器。然后它可以将信号传递给你的其他设备。

我们还没有机会测试这一点，尽管我们知道 Eero 可能是领先的消费者网状 Wi-Fi 品牌，即使是最便宜的 Echo，亚马逊也不太可能超过它。

你可能会期待更多的设备提供“内置 Eero”，因为它现在似乎是一件事，已经开始与铃声警报专业。不要都跑出去买 Echo Dots 作为廉价(50 美元)的无线扩展器。
