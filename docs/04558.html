Twitter 的 EU-唯一的地理区块 RT 了摇摇欲坠的开始

Twitter 声称它遵守了 EU-对今日俄罗斯(RT)和 Sputnik 的禁令，该禁令于昨天生效，是欧盟在俄罗斯入侵乌克兰后对俄罗斯实施的一揽子制裁的一部分，尽管是不情愿的。

在社交网络本周回应媒体要求而发布的一份声明中，Twitter 发言人表示:

欧盟(EU)制裁将依法要求我们在欧盟成员国保留某些内容，我们打算遵守。我们在欧盟以外的全球策略将继续专注于在我们的服务中消除这种国家附属媒体内容的影响，并通过我们的标签提供重要的背景信息。我们继续倡导自由开放的互联网，尤其是在危机时期。

然而，目前还不清楚推特在多大程度上真正遵守了禁止 RT 内容在欧盟传播的法令。

虽然自 RT 禁令生效以来，欧洲各地的一些用户报告说，如果他们试图访问克里姆林宫相关媒体的验证 Twitter 账户，就会收到“账户被扣”的通知(见下面的第一个截图)，但 TechCrunch 发现，在欧盟境内仍然可以查看 RT 的账户，而无需使用 VPN 来绕过地理区块(见第二个截图)。

但今天在西班牙测试完全相同的内容时，我们没有发现访问 RT 账户的障碍，也可以查看个人推文，包括国家附属媒体实体今天早上发布的以下示例，尽管 Twitter 自己的“账户被扣”通知在欧盟其他地方也可以看到，将西班牙列为实施该通知的国家之一…

西班牙的一名 Twitter 用户仍然可以访问 RT 的账户(截屏:TechCrunch)

在撰写本文时，我们在西班牙的测试人员仍然能够通过 Twitter 移动应用程序和移动网络访问 RT。

因此，在这种情况下，Twitter 似乎违反了欧盟的制裁。

目前还不清楚 Twitter 在西班牙的实施会有多严重(或多严重),或者说，在整个欧盟(有 27 个成员国)会有多严重。

不排除进一步“泄密”。也许尤其是在制裁生效后不久。对这里发生的事情的一种解释是，这可能是 Twitter 仍在解决其合规问题的一个案例。(更新:另一个，正如这位读者指出的，屏蔽不依赖于位置，而是用户声明的国家，所以可以简单地通过用户在 Twitter 设置中改变他们的国家来绕过。)

在英国，仍然可以访问 RT 在 Twitter 上的账户——但这是该公司为了遵守欧盟禁令而选择的最窄应用程序，而英国不再是欧盟成员。

英国也没有下令禁止 RT 的内容通过在线平台传播，至少现在还没有；其媒体监管机构 Ofcom 目前正在调查 RT 是否违反了该国的广播法规。(在在线安全法案成为法律之前，Ofcom 也在为一个大大扩展的角色做准备，监督传入的互联网内容规则，所以将来肯定会参与在线内容的审核决策。)

在欧盟禁令后，Twitter 表示将对 RT 进行地理封锁的国家名单也不包括冰岛、挪威、列支敦士登和瑞士——这些国家位于欧洲，属于欧洲自由贸易区，但不在欧盟内。

因此，推特再次选择了最狭隘的泛欧盟制裁措施。

这与苹果公司和谷歌公司形成了鲜明的对比——本周早些时候，苹果公司和谷歌公司都宣布在各自的移动商店中屏蔽 RT 的应用程序，苹果公司在所有国际市场都这么做了(除了俄罗斯)。

谷歌没有走得那么远，但它确实实施了比 Twitter 略广的地理屏蔽——也屏蔽了上述非欧盟欧洲国家(英国、冰岛、挪威、列支敦士登和瑞士)以及欧盟国家和乌克兰的应用访问。

周一，微软还宣布将从其 Windows 应用商店中移除 RT 新闻应用，并从其微软 Start 平台(包括 MSN.com)中清除 RT 和 Sputnik 内容，以“减少俄罗斯国家宣传的曝光”，正如其所说。

因此，Twitter 在克里姆林宫附属媒体上的狭窄的、EU-唯一的屏蔽开始看起来像一个局外人。

尽管如此，该公司的核心产品是实时信息服务——这解释了为什么它通常更喜欢标记和语境化，而不是审查(本周早些时候，它宣布为链接到俄罗斯媒体的推文添加额外的标签)。

此外，正如 Twitter 的声明所暗示的，其网络可以在危机时期发挥特别重要的作用——因此即使在战争时期，不愿降低其核心效用也是可以理解的。

与此同时，随着俄罗斯入侵乌克兰，RT 的经营环境无情地发生了变化。目前还不清楚 Twitter 是否因此重新评估了自己的政策。

我们现在正在目睹俄罗斯对欧洲一个主权国家的侵略战争——在这场战争中，以外国为目标的宣传发挥了关键的战略作用。

这就是为什么欧盟领导人很快决定入侵乌克兰标志着允许普京的宣传通过 RT 和 Sputnik 继续自由流动的红线，这两个主要的国家附属频道从事外国信息操纵，并明显附属于他的政权。

尽管如此，Twitter 似乎更愿意尽可能(合法地)保持中立。

当被问及我们的测试人员在西班牙(一个欧盟国家)遇到的问题时，Twitter 拒绝解释为何未能屏蔽 RT 的内容——只是指出了它早些时候的声明，当时它表示“打算遵守”欧盟制裁。

在进一步追问下，推特的女发言人证实:“我们已经实施了禁令。”

鉴于此，看起来我们发现的问题是禁令执行不完善的一个例子——想必不是 Twitter 的故意失败。这也许与禁令生效后的时间很短有关(尽管欧盟主席警告说禁令将在周末生效)。

考虑到该公司长期以来对尽可能多的推文流动的偏好，法律禁令的古怪实施也可能意味着 Twitter 草率执行的某种程度上的“功能而不是错误”。

我们确实在推特上澄清了这个 bug。但是，尽管我们多次试图得到关于我们标记的问题的直接答案，但没有一个是现成的。

相反，Twitter 的女发言人给我们发来了一个链接，指向其“国家保留内容”(CWC)页面——以及一个她称之为“ ”的指针，指向来自我们最新透明度报告的数据/解释，这些数据/解释是关于哪些国家要求我们保留内容并回复:哪些问题”，她补充道，“希望这有助于给你一个尺度感”。

她强调的那段文字(见文章底部)表明，现在有 20 个国家的 Twitter 限制访问内容。所以，试着从字里行间去理解，它可能是想说，有这么多不同的、特定于国家的要求要遵守，它很难完美地执行。

在欧盟对克里姆林宫相关媒体的禁令昨日生效之前，欧盟官员确实发出信号，鉴于切断内容可能以数字方式传播的最后一个渠道的挑战，他们预计在线实施制裁会有一定程度的渐进性。(不过，在特定地区屏蔽已验证的账户应该没那么难。)

无论我们对 Twitter 执行欧盟禁令的问题有什么具体解释，有一点是明确的:它仍然很容易摆脱另一次政策执行失败。
