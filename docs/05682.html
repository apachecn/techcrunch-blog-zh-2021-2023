比特币基地推出 NFT 市场

原文：https://web.archive.org/web/https://techcrunch.com/2022/04/20/coinbase-launches-nft-marketplace-in-hopes-of-appealing-to-crypto-and-mainstream-users/

对于成千上万的 NFT 社区成员来说，这可能是一个美好的早晨，也可能是“gm”，他们现在将获得另一个市场来交易他们的数字收藏品。

六个月前，比特币基地，美国交易量最大的加密交易所，宣布将推出一个 NFT 市场，但没有提供具体时间。快进到今天:“比特币基地 NFTs”在区块链以太坊推出了它的测试模式，收集了大量的非艺术作品。

比特币基地是一个公开交易的加密货币交易平台，据其网站称，该平台拥有约 8900 万用户、2780 亿美元的加密资产和 5470 亿美元的季度交易额。

在测试版发布期间，比特币基地正在策划和展示一些 NFT 的收藏，如《涂鸦》、《女人的世界》和《志那都红豆》。任何基于以太坊的系列都可以在比特币基地 NFT 上访问，“更多的(区块链)将很快推出，”比特币基地 NFT 的 NFT 产品负责人 Alex Plutzer 在新闻发布会上说。“随着时间的推移，我们将循环推出这些[收藏]，帮助人们了解平台上令人敬畏的非功能性测试，特别是对非功能性测试不太熟悉的人。”

“在有限的时间内”，其市场上的非金融交易也不会收取交易费。随着时间的推移，费用会增加，但将是一个“低个位数的费用，”比特币基地产品副总裁 Sanchan Saxena 在会议上说。用户可以使用比特币基地钱包或他们拥有的任何自保管钱包在平台上交易非金融交易。

“我们已经能够成功地以一种易于使用的方式将加密货币的复杂性带给大众，”Saxena 说。“我们相信，对于非功能性测试，我们也有类似的机会。”

但是与它的密码交易所不同，NFT 市场的目标是实现一个“web3 社交市场”，Saxena 说。作为比特币基地 NFTs 战略的一部分，该市场将允许用户购买、出售并与创作者、收藏家和他们所拥有的 NFT 社区进行互动。

那些在比特币基地 NFT 等待名单中名列前茅的人将能够成为其市场上的 beta 测试者，并创建个人资料和展示他们的 NFT，包括他们创建和交易的 NFT。Saxena 说，整个平台将在未来几周内对所有 18 岁以上的人开放。

这类似于 Instagram 个人资料，人们可以展示他们的照片——但在这种情况下，这是 NFTs。人们可以通过其市场进行关注、评论和互动。

Saxena 说，人们经常使用多个不同的社交媒体渠道，如 Discord 或 Twitter，来参与他们相信的 NFT 社区。“在我们的平台上，他们可以在一个地方这样做。”

Saxena 说，虽然像评论或关注其他 NFT 账户这样的功能不是链上或分散的，但公司的目标是逐步分散所有这些功能，并随着时间的推移使它们链上。

另外，该平台不打算对上传到市场的任何非功能性食物进行审核，除非在提供非功能性食物的国家是非法的。Saxena 说，例如，如果 NFT 包含在美国不合法的细节，比特币基地·NFT 会禁止它。

适度的想法经常引起整个加密社区的分歧，因为用户声称应该是一个开放的，无许可的市场带有一个受适度限制的星号。然而，Saxena 声称，只要在该国合法，比特币基地 not“没有判断什么是对的或错的业务”。

在未来，比特币基地 NFT 计划增加更多的功能，如 NFT 下降，铸造和令牌门控社区。Saxena 说，它还预计增加用比特币基地账户或信用卡购买 NFT 的选项，以便主流观众更容易进入生态系统。

Saxena 分享说，为了给其平台带来更多用户，它还将在全球范围内与 NFT 艺术家合作，这些艺术家将在不久的将来在比特币基地 NFT 进行收藏。“你可以期待比特币基地 NFT 成为下一个无聊的猿类游艇俱乐部”或下一个艺术家或组织在这个平台上做他们的发射的平台。

“这只是开始，”Saxena 说。
