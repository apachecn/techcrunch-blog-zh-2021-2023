用户称，第三方 Twitter 应用面临问题

原文：https://web.archive.org/web/https://techcrunch.com/2023/01/12/third-party-twitter-apps-tweetbot-twitterrific-echofon-down/amp/

影像学分:布莱斯·杜宾/ TechCrunch

使用第三方推特客户端的人面临许多问题，包括无法登录和访问推特订阅源。三个受欢迎的第三方推特应用 Tweetbot、Echofon 和 twitteel at 证实了这些问题，并指出他们不确定是什么引发了故障。

这些应用的制造商也在 Mastadon 上抱怨这些问题。推特的开发者肖恩·赫伯说:“推特刚刚杀死了第三方客户吗？”推特机器人的保罗·阿达德说，“我希望推特上发生的一切只是一些自动垃圾邮件保护机器人，错误地暂停了适当的应用程序。”

影像学分支:乳齿象

影像学分: Mastadon

在对 TechCrunch 的电子邮件回复中，哈达德表示，该问题始于太平洋时间今天下午 7:30 左右。他还提到，来自应用程序的所有 API 请求都失败了。

很可能是推特对第三方客户端的应用编程接口做了一些改变，导致这些应用崩溃。尚不清楚这是否是阻止访问该平台的一个步骤。

除了上述应用，用户还抱怨无法从菲尼克斯、推特面板、羽毛和泰龙等客户端访问推特。所以访问推特的唯一途径是通过官方客户或网站。

推特开发者论坛上的一篇帖子称，在开发者门户上，这些应用显示为“暂停”

自从埃隆·马斯克接管以来，Twitter 已经杀死了许多开发者程序，包括用于应用发现的 Twitter 工具箱。第三方开发者对他们围绕 Twitter 的开发计划持谨慎态度，因为该公司尚未公布其生态系统计划。上个月，该公司前开发者平台主管阿米尔·谢瓦特(Amir Shevat)为 TechCrunch 撰文称，新管理层辜负了开发者的信任。

虽然 Twitter 还没有就这个问题进行任何沟通，TwitterDev 账户上个月表示该公司“将继续投资我们的开发者平台，尤其是我们的 Twitter API。”

本周早些时候，Twitter 决定将算法时间线命名为“For You”作为 iOS 的默认提要。
