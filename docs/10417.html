Rad Power Bikes 和 Cycle pilot 消费者电动自行车订阅

Rad Power Bikes 是一个受欢迎的美国电动自行车品牌，在过去两年中已经筹集了超过 3 亿美元，将于下周开始提供自行车订阅服务。该公司正与总部位于柏林的电动自行车订阅服务公司 Cycle 合作，在柏林试点这项服务，该公司提供快递和最后一英里配送服务。

从 9 月 22 日开始，Cycle 将提供两辆货运自行车，即 RadRunner 和 RadWagon ，根据 Cycle 的说法，起价为 €每月 79.90 美元，可以选择滚动月租或年度合同。

根据 Rad 欧洲业务总监阿诺·萨拉丁(Arno Saladin)的说法，这种合作关系是 Rad 的第一次订阅活动——该公司一直高度专注于直接面向消费者的销售，但希望吸引一个新的客户群体，他们更喜欢为电动自行车的使用付费，而不是直接拥有电动自行车。

“通过这个试点项目，我们希望了解需求的规模以及这些客户正在寻找的确切价值，”萨拉丁告诉 TechCrunch。"这将有助于我们确定什么是最佳价值和需求潜力."

与 Rad 合作提供以消费者为中心的订阅服务对 Cycle 来说也是全新的。在其短暂的生命中，这家初创公司主要迎合了最后一英里送货工人和车队的需求。Cycle 拥有自己开发的货运自行车，许多车队和快递客户都使用这种自行车，但本着多样性和选择的精神，Cycle 在 2020 年开始将 Rad 自行车加入其产品线。

“Rad Power 自行车的独特外观吸引了街上许多人的注意，”Cycle 的联合创始人兼首席执行官 Luis Orsini-Rosenberg 告诉 TechCrunch。“自从我们将车辆加入车队以来，没有一个星期我们没有收到私人客户的电子邮件，询问从我们这里购买或租赁车辆。”

Cycle 在柏林有实地运营、自行车维护和订阅基础设施，因此 Rad 所要做的就是交付货物。客户将能够从 Cycle 位于柏林时尚街区 Kreuzberg 的车间收集他们完全组装好的电动自行车。服务显然包括在内，可以使用 Cycle 应用程序预订维修。偏执的客户可以添加一个“安心”包，在发生大规模损坏或盗窃的情况下，可以显著减少超额损失。
