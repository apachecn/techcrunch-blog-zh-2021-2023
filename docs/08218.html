Canvas Medical 筹集了 2400 万美元的 B 轮融资

原文：https://web.archive.org/web/https://techcrunch.com/2022/07/14/canvas-medical-the-tesla-of-emrs-raises-a-24-million-series-b-round/?utm_content=buffer57e4c&utm_medium=social&utm_source=twitter.com&utm_campaign=buffer

当 Andrew Hines 在 2015 年开始 Canvas Medical 的时候，他没有预料到远程医疗会从小众走向规范，也没有预料到疫情会激励企业家成立一个新的数字医疗创业公司舰队，这些公司需要新的基础设施。他只知道他想为像他妻子一样的医生开发软件，他妻子每天在数据文档上浪费数小时。

“我认为肯定有更好的解决方案，”海因斯告诉 TechCrunch。“我开始使用 Canvas 来设计具有正确数据结构和正确工作流程的软件，以便能够对患者健康结果产生积极影响。”

Canvas 是一家电子医疗记录(EMR)初创公司，它通过一个工作流系统帮助从医生到保险公司到社会工作者的 15 种利益相关者更好地跟踪他们的患者数据，该工作流系统可以在文档中自动填充信息，而不是通过一系列下拉菜单，这使得用户可以更快更方便地使用。

安德鲁·海因斯(Andrew Hines)在看到妻子在数据文档上浪费数小时后，创办了 Canvas Medical。

现在，这家总部位于旧金山的初创公司刚刚在 M13 牵头的一轮 B 轮融资中筹集了 2400 万美元，现有投资者包括 Inspired Capital、IA Ventures 和前期资本。

对于灵感资本首席运营官和合伙人马克·巴兹扬来说，对画布的需求是显而易见的，因为他的妻子在职业生涯中也不得不与糟糕的电子病历系统打交道。Inspired 在 2021 年 4 月投资了该公司的 A 轮融资，然后在今年加倍投资 B 轮融资。

Batsiyan 说:“我从未忘记的一句话是，他们的一位医生将 Canvas 称为 EMR 的‘特斯拉’”。“这是一种行业领先的产品体验，让我们有信心投资 A，并真正重新思考医疗保健的 EMR 和基础设施。”

尽管 Canvas 目前很成功，但 Hines 最初并没有打算成为一名企业家。

在他创建公司的一年前，他在 Practice Fusion 担任了一个软件工程师的角色，这是一家雄心勃勃的 EMR 公司，当时已经筹集了超过 1 亿美元的风险投资。海因斯说，他在那里学到了很多东西，但该公司并没有从 EMR 系统中赚钱，而是通过它开发的其他渠道赚钱，他认为这导致了增长方向的错位。

Canvas 软件的当前迭代花了五年时间开发，并于 2021 年秋季推出。它允许用户跟踪数据并链接到支付选项，这一切都是通过一个开放的 API 实现的，允许软件开发人员在其上进行构建。自去年推出最新产品以来，该公司的客户群增长了 300%，但它拒绝透露更多具体指标。

除了融资消息，这家初创公司还宣布，它已经获得了国家卫生信息技术协调员办公室(ONC)的联邦认证。由于其标准化的医疗 API，这是过去二十年来成立的第一家获得最新版本规则认可的电子病历公司，这意味着它可以与联邦计划如医疗保险合作，并使该公司领先于即将到来的 21 世纪治疗法 API 执行。

“令人鼓舞的是我们看到的客户类型的广度，”海因斯说。“许多远程医疗公司从事紧急护理、心理健康和行为健康。我们已经看到了大量新的特定类型的实践。”

虽然 Canvas 不是专门为数字健康初创公司社区服务的，但它已经能够利用自疫情开始以来兴起的一波公司，包括心理健康初创公司 upgrade 和虚拟记忆诊所 Isaac Health，并帮助推动传统公司的数字产品。

“软件和医学正在进入一个合作和创造价值的新时代，”海因斯说。“软件工程师如何与医生和护士合作？那是我们的北极星。”
