英伟达与谷歌云合作推出专注于人工智能的硬件实例

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/21/nvidia-partners-with-google-cloud-to-launch-ai-focused-hardware-instances/

通过与谷歌合作，Nvidia 今天推出了一款新的云硬件产品 L4 平台，该平台针对运行视频应用进行了优化。

Nvidia 表示，L4 平台旨在加速“人工智能驱动的”视频性能。作为通用 GPU，L4 提供视频解码以及代码转换和视频流功能。

除了通过谷歌云提供对 L4 平台的访问，谷歌还将 L4 集成到 Vertex AI 中，这是其为企业客户提供的托管机器学习服务。

对于那些不愿意与谷歌云签约的人来说，L4 将于今年晚些时候从英伟达的网络硬件合作伙伴那里推出，包括华硕、思科、戴尔、惠普企业和联想。

L4 与 Nvidia 今天宣布的其他专注于人工智能的硬件解决方案并列，包括 L40、H100 NVL 和格蕾丝·赫柏的推荐型号。L40 针对图形和人工智能支持的 2D、视频和 3D 图像生成进行了优化，而 H100 NVL 支持部署大型语言模型，如 ChatGPT。(顾名思义，推荐模型的格蕾丝·赫柏是以推荐模型为中心的。)

L40 本周可通过英伟达的上述硬件合作伙伴获得。Nvidia 预计格蕾丝·赫柏和 H100 NVL 将于今年下半年发货。

在相关新闻中，今天标志着英伟达的 DGX 云平台的推出，该平台使公司能够访问基础设施和软件，以训练生成式和其他形式的人工智能模型。今年早些时候宣布，DGX 云允许企业按月租用英伟达硬件集群——每月每个实例的起价为令人瞠目结舌的 36999 美元。

DGX 云的每个实例都配有八个 Nvidia H100 或 A100 80GB Tensor Core GPUs，每个节点总共有 640GB 的 GPU 内存，搭配存储。通过 DGX 云订阅，客户还可以访问 AI Enterprise，Nvidia 的软件层，包含人工智能框架、预训练模型和“加速”数据科学库。

英伟达表示，它正在与“领先”的云服务提供商合作，以托管 DGX 云基础设施，从甲骨文云基础设施开始。微软 Azure 预计将在下一个财政季度开始托管 DGX 云，该服务将很快扩展到谷歌云。

英伟达积极推进人工智能计算，正值该公司从游戏和专业虚拟化等其他领域的无利可图的投资中撤出。英伟达的最新财报显示，其数据中心业务，包括人工智能芯片，继续增长(至 36.2 亿美元)，表明英伟达可以继续受益于生成性人工智能热潮。
