谷歌联合实验室推出了一个现收现付的选项，高级 GPU 访问

Google Colaboratory(简称 Colab)是谷歌的一项服务，旨在允许任何人通过网络浏览器编写和执行任意 Python 代码，该服务正在推出一项现收现付计划。自从谷歌在 2020 年推出高级 Colab 计划以来，Colab 的第一次价格变化是，现在用户可以选择购买 Colab 中的额外计算时间，无论是否付费订阅。

谷歌表示，这一更新不会影响免费的 Colab 层，它仍保持目前的形式。唯一的实质性变化是，用户可以以“计算单位”的形式购买计算权限，100 个单位的起价为 9.99 美元，500 个单位的起价为 49.99 美元。

正如 Google Colab 产品负责人 Chris Perry 在博客中解释的那样:

付费用户现在可以灵活地以他们选择的任何速率用完计算配额，以计算单位衡量。当计算单元耗尽时，用户可以根据自己的判断选择购买更多的计算单元。一旦用户耗尽了他们的计算单元，他们的 Colab 使用配额将恢复到我们的免费层限制。

在推出按需付费的同时，谷歌宣布，付费的 Colab 用户现在可以在 Colab 中选择标准或“高级”GPU——后者通常是 Nvidia V100 或 A100 Tensor Core GPUs。(Colab 中的标准 GPU 通常是英伟达 T4 张量核心 GPU。)然而，该公司指出，不能保证获得特定的 GPU 芯片类型，这取决于许多因素，包括可用性和用户在 Colab 的付费余额。

不言而喻，高级 GPU 也会比标准 GPU 更快地耗尽 Colab 计算单元。

几周前，当通过电子邮件通知 Colab 用户，它将为订户采用上述计算单位系统时，谷歌开始在 Colab 中展示现收现付选项。它将这一转变框定为走向透明，允许用户“对如何和何时使用 Colab 有更多的控制。”

一些人认为此举是对用户的敌意——试图对 Colab 的使用收取更多费用或进行压制。但在给 TechCrunch 的一份声明中，谷歌发言人指出，限制一直适用于 Colab 使用付费计划的所有层级。

发言人通过电子邮件表示:“这些更新是为了让用户更清楚地了解……限制。”" Colab 将继续支持其免费层，包括基本的 GPU 访问."

围绕价格变化的敏感性反映了自 2017 年底从谷歌内部研究项目中剥离出来以来，Colab 的增长幅度。该平台已经成为人工智能研究社区内事实上的数字试验板——对于编写代码的研究人员来说，在托管代码的 GitHub 仓库上或旁边包含 Colab 页面的链接并不罕见。
