支付基础设施初创公司 Minka 获得 2400 万美元

原文：https://web.archive.org/web/https://techcrunch.com/2022/04/27/bogota-based-minka-lands-24m-from-tiger-kaszek-to-build-an-open-infrastructure-for-money/

总部位于波哥大的支付基础设施初创公司 Minka 在由 Tiger Global Management 和 Kaszek 共同牵头的一轮融资中获得了 2400 万美元。

在一次采访中，首席执行官兼联合创始人 Domagoj Rozic 将 Minka 描述为“一个从头开始构建的无缝支付网络。”该网络旨在通过区块链概念和 API 连接独立的遗留系统，“以简化复杂的资金流动”

换句话说， Minka 是一个开放的网络，旨在允许银行和票据交换所等组织通过将其“封闭、过时的核心系统”暴露在网络上来实时“发布”和转移资金。

“这反过来使他们能够实时收取、发送或交换资金，而不需要对账，几乎没有成本，”Rozic 告诉 TechCrunch。

在其祖国哥伦比亚，这家初创公司已经产生了影响。它与 ACH 哥伦比亚公司合作开展 Transfiya 项目，该项目包括对该国 80%的账户进行直接读写 API 访问。根据 Rozic 的说法，这使其成为该地区最大的开放式银行项目。

他说:“我们的目标是通过网络，使用共享的连接分类账，将世界各地的余额连接起来。”“哥伦比亚的项目证明了这是可能的，并已成为该地区发展最快的实时支付网络之一，使以前缓慢的资金能够以最低的成本立即流动。”

目前，近 200 万人使用该项目只用一个电话号码汇款。根据 Rozic 的说法，Minka 有 20 家金融机构和金融科技公司直接与其平台相连，另有 20 家正在整合过程中。

图片来源:敏卡

这家初创公司正在衡量我们平台上的交易数量、连接的机构和活跃用户。根据 Rozic 的数据，2022 年全年交易和用户逐月增长 20%，同比增长 250%。

“由于我们的模式是交易型 SaaS，收入也一直在增加，”他告诉 TechCrunch。

这家初创公司还计划利用新资本继续“现代化”票据交换所和央行的基础设施。它计划通过建立一个完整的自助服务工具来扩大网络，该工具用于向支持一键式购买、支付和收款的组织发布和转移资金。

“我们的客户主要是票据交换所和金融机构，”Rozic 说。“然而，随着新资本的筹集，我们也将很快向金融科技公司提供自助 SaaS。”

“我们已经改善了人们在哥伦比亚与金钱互动的方式，现在正在建立一个开放的网络，以扩大世界各地的利益，”他补充说。

Minka 这个名字来自一个古老的盖丘亚语单词，代表为社区的利益而合作。

Rozic 说，他五年前搬到哥伦比亚时，受到启发创办了 Minka。

“我意识到我可以免费给我女儿在克罗地亚的祖父母发送一张我女儿的照片，而且只需要一个电话号码，”他告诉 TechCrunch。“但要给隔壁的邻居汇 5000 比索，我得等两天，最后还会欠银行钱。”

Rozic 与 Paola Sanchez 和 Tomislav Biscan 合作，为基于现金的经济体建立了一个新的支付网络，以使资金像信息一样容易流动。

图片来源:明卡

这不是最初的计划。Minka 实际上是在 2018 年成立的，纯粹是一家专注于建立新支付协议的 R&D 公司。在这个过程中，它与中央银行和当地的 ACH 进行了接触，然后赢得了一个 RFP，以实现哥伦比亚支付基础设施的自动化。

它计划帮助拉丁美洲的西班牙语国家实现支付基础设施的现代化，并计划立即扩展到多米尼加共和国、玻利维亚和墨西哥。

这家初创公司是由万事达卡介绍给多米尼加共和国的一位客户的，Minka 和谷歌都是万事达卡的合作伙伴。值得注意的是，万事达卡也是该公司的投资者之一。

Rozic 告诉 TechCrunch:“我们正在通过与这些合作伙伴的推荐计划进入新市场，以帮助我们将连接转移到我们的网络中。”“这些公司充当了明卡的扩展销售团队，他们通过实时支付更快地进入市场。支付基础设施即服务对 LatAm 来说并不常见。我们的目标是由 Minka 提供动力，这是每个人都在使用但没有人看到的东西。”

Minka 有大约 30 名员工，其产品和技术团队位于克罗地亚，支持和销售人员位于哥伦比亚。

老虎环球合伙人亚历克斯·库克表示，他的公司渴望支持明卡公司“为金钱建立开放基础设施”的使命。

“Transfiya 在哥伦比亚的成功推出反映了 Minka 团队的实力，我们预计他们将继续加快实时支付在全球的采用，”他在一份声明中说。

之前的支持者包括金融科技集团(FinTech Collective)，该集团在 2021 年领导了种子轮融资，也参与了融资。迄今为止，该公司已经筹集了 2750 万美元的资金。
