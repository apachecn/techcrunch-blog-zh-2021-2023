你现在可以做的 3 件事来支持乌克兰的 IT 行业

原文：https://web.archive.org/web/https://techcrunch.com/2022/04/01/3-things-you-can-do-right-now-to-support-ukraines-it-sector/

艾美奖·根格勒是

Softjourn

是一家全球技术服务提供商，在加利福尼亚、乌克兰和波兰设有办事处。

“电话和短信很早就开始了，询问电源和修理引擎的工具，”在乌克兰西部小城伊万诺-弗兰基夫斯克生活和工作的卡蒂亚·里扎(Katia Ryzha)说。

自战争开始以来，前线的乌克兰士兵向他们在全国各地的朋友和同事索要工具和其他难以找到的物品。“我们的团队协调清理车库、地下室和工具棚，因为现在在乌克兰很难或不可能找到这些物资，”Katia 分享道。

卡蒂亚是美国科技公司 Softjourn 项目和交付管理部门的负责人，该公司在 R&D 设有办事处，在乌克兰西部有许多员工。虽然我们在战争开始时感到震惊，但我们的团队设法团结努力，组织支持，以保证我们同事及其家人的安全。

我们的团队与乌克兰更大的技术社区一起，团结一致，为战争努力提供帮助。这包括加入乌克兰的 IT 军队，计划前往前线补给士兵，以及向军队捐赠资金和军事装备。

卡蒂亚说，虽然志愿服务对战争努力很重要，但在经济战线上战斗更为重要。

Softjourn 的联合创始人兼董事总经理 Sergiy Fitsak 说:“通过工作，我们正在帮助推动战争努力。我们挣得的每一美元都有助于加强乌克兰的经济，因为我们急需这笔钱来资助军事行动、向受灾最严重的人提供人道主义救济、保护我们的社区、供养和照顾我们的家人和邻居。”

对于卡蒂亚和我们许多生活在乌克兰的同事来说，工作已经成为一种反抗行为。同样，当公司选择购买在该国生产的产品的许可证，或者与部分或全部团队都在该国的咨询公司合作时，它们培育了经济，从而保障了乌克兰的未来。

作为全球技术社区的成员，我们在这一关键时刻有着独一无二的优势。这里有三种方法可以帮助你:

继续关注乌克兰的 IT 和技术需求

根据 IT 乌克兰协会的数据，战前，乌克兰是价值 68 亿美元的科技产业的所在地。虽然一些公司不得不关闭或暂时停止运营，但许多公司仍在运营，甚至还在增长。

与乌克兰的许多其他科技公司一样，Softjourn 自 2014 年以来就制定了应急计划，以确保俄罗斯的入侵不会阻止我们的进步。我们不会停止在乌克兰的扩张，无论是招聘还是获取专业知识。

如何提供帮助:

  • 支持乌克兰科技公司的最佳方式是，其他地方的公司继续与该国的合作伙伴做生意。
  • 如果你的公司一直与在乌克兰有员工的公司合作，保持这种伙伴关系。用你的业务支持他们不仅有助于保持高昂的士气，而且在这个国家非常需要的时候，它还为这个国家提供了关键的资本注入。
  • 如果你是一个初创公司的创始人，还没有完全定义、架构或设计你的解决方案，那么与在乌克兰有建筑师的合作伙伴合作可能不是你现在想做的事情。但是为什么不呢？许多在乌克兰开展业务的咨询公司都是国际化的。他们准备帮助他们的客户降低风险，而你将支持乌克兰和那里的技术社区的持续扩张。

购买或许可乌克兰产品和服务

乌克兰的 IT 部门在多个行业表现出色。根据 2021 年乌克兰 IT 报告，金融科技、银行和电子商务引领该国的 IT 出口，但该国的科技公司也因其为运输、物流、医疗保健、教育、零售和娱乐行业提供的服务而闻名。

如何提供帮助:

  • 你可能已经在使用一家植根于乌克兰的公司提供的服务或产品，但你并不知道，比如 Grammarly、GitLab、Ajax 和 People.ai。如果你是，继续与他们合作并传播消息。
  • 许多咨询公司和服务提供商专注于特定领域，并许可产品来解决这些行业中的问题。考虑使用这些产品和服务。

增强对乌克兰重要科技行业的认识

乌克兰是欧洲最大的 IT 服务出口国之一。在这场战争开始之前，世界不太了解乌克兰作为欧洲领先科技中心的地位，而且越来越了解它作为世界领先科技中心的地位。我们可以提升乌克兰作为科技领导者的角色，在现在、未来的岁月里帮助这个国家。

如何提供帮助:

  • 如果你和一家在乌克兰有业务的公司做生意，要广泛分享。在你的网站或产品上添加一个徽章，上面写着“支持乌克兰技术”
  • 在社交媒体和谷歌上给乌克兰公司或有乌克兰团队的公司正面评价。向你认识的人推荐他们的服务。
  • 分享您的合作伙伴公司的技术人员如何开展业务的故事。

乌克兰继续为世界各地的企业和消费者开发软件和 IT 产品，这带来了支持军事努力和人道主义援助的资源，并让公司关心员工的家人、朋友和社区。

乌克兰和许多在该国工作的公司花了几十年时间建立工业和技术专长，这不会被与俄罗斯的战争摧毁。科技界有一个独特的机会来帮助乌克兰在经济战线上的斗争。
