原文：https://web.archive.org/web/https://techcrunch.com/2022/05/24/google-lens-comes-to-chrome-for-searching-using-images-from-webpages/

谷歌镜头来到 Chrome，使用网页上的图像进行搜索

谷歌一直在努力将谷歌眼镜(Google Lens)的视觉搜索工具更好地整合到其浏览器中，以实现能够识别你所看到的内容的新型搜索，而不仅仅是搜索你输入的内容。今天，谷歌正在推出一种在桌面上使用谷歌镜头的新方法。不用打开一个新标签来执行搜索，你可以在 Chrome 浏览器的同一页面上使用 Lens 来做一些事情，如翻译图像的文本，识别图像中的对象或从图像中获取原始来源。

此前，谷歌已经在网络上提供了图像搜索和谷歌照片的镜头功能，但其最全面的产品是在移动设备上。今年 4 月，谷歌还在手机上推出了镜头驱动的多搜索功能，允许用户结合文本和图像进行搜索——这暗示了该公司进一步投资镜头技术以使搜索感觉更自然的更广泛计划。

在此之前，公司已经宣布将在桌面上集成 Lens 和 Chrome，也将在“未来几个月”发布

今天的更新将允许桌面上的 Chrome 浏览器用户右键单击网页上的任何图像，然后进入新的菜单选项“用谷歌镜头搜索图像”这是同一个菜单，今天，你可以保存或复制图像或在一个新的标签中打开它。

图片来源:谷歌

这将在网页一侧的新面板中打开一组搜索结果，其中包含关于该照片的更多信息。然后，您可以选择单击一个按钮来查找图像源，这样您就可以看到包含该图像的其他网页。你也可以翻译图像中的文字，或者使用镜头来帮助你识别照片中的内容。

图片来源:谷歌

换句话说，它引入了一种方法，将图片作为你所在网页的新搜索的起点，而不是要求你从谷歌的图片搜索中开始一个新的关于图片的查询，或者通过在 Google.com 上键入标准的基于文本的搜索。

该功能类似于微软更新的网络浏览器 Microsoft Edge 中的反向图像功能，也允许用户在不离开当前标签的情况下在侧边栏中执行反向图像搜索。然而，搜索本身是由必应执行的。

谷歌表示，新功能将从今天开始向所有 Chrome 用户推出，这是其帮助人们以更直观的方式进行搜索的更大努力的一部分。
