Vianova 为共享移动公司和城市建立位置数据平台

认识一下 Vianova ，这是一家法国初创公司，正在为共享移动公司和地方政府建立一个数据平台。该公司既是数据存储库，也是数据可视化仪表板。

它已经筹集了 640 万美元(€600 万)的首轮融资，由 Baloise VC 牵头，XAnge 以及现有投资者 RATP 资本创新和 Ponooc 也参与其中。

随着自由浮动自行车、电动滑板车、轻便摩托车和可以用手机解锁的汽车的兴起，许多城市发生了巨大变化。随着时间的推移，城市也改变了它们的规则，并以不同的方式拥抱微移动性。

好消息是，这些移动车队都与 GPS 芯片和蜂窝调制解调器有某种形式的连接。换句话说，公司和地方政府可以利用数据让这些服务对使用它们的人和生活在这些城市的其他人来说更加高效和安全。

图片来源: Vianova

现在，Vianova 在其平台上跟踪 100 万辆联网车辆。一些公司使用该平台来管理他们的车队，并确保车辆在正确的时间出现在正确的地点。

政策制定者和数据科学家也可以利用这个平台来发现一些趋势。它可以用于改善基础设施，通过地理围栏等创建低速区或停车区。

所有这些都可以通过一个类似模拟城市的仪表盘来完成，仪表盘上有地图覆盖和实时数据。这个想法是，你可以使用 Vianova 作为一个实时仪表板和一个分析工具来生成历史报告，看看你做得如何——无论你是一家移动公司还是一个地方政府。

但是一个没有任何数据的分析工具会有点没用。这就是为什么您可以将自己的数据与不同的移动数据源相结合。对于那些希望从私营公司获取数据以更好地了解正在发生的事情的城市来说，这可能特别有趣。

例如，斯德哥尔摩市和 Voi 都使用 Vianova 来改善该市的共享移动服务。移动公司决定与谁共享数据。

Vianova 使用 API 从移动运营商获取数据，以便您可以在平台上看到实时数据。数据可以聚合和匿名。

Vianova 的一些客户包括苏黎世、阿姆斯特丹、斯德哥尔摩和马赛等城市，以及 Bolt 和 Voi 等移动运营商。一些大型基础设施公司也在使用该平台，如 RATP、巴黎机场和 KTH 大学。
