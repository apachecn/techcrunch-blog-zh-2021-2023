生成图像的人工智能可以从训练数据中复制和粘贴，引发知识产权问题

原文：https://web.archive.org/web/https://techcrunch.com/2022/12/13/image-generating-ai-can-copy-and-paste-from-training-data-raising-ip-concerns/amp/

图片来源:布莱斯·德宾/ TechCrunch

研究人员在一项新的研究中显示，像 DALL-E 2 和 Stable Diffusion 这样的图像生成人工智能模型可以——而且确实——从它们的训练数据中复制图像的某些方面，随着这些服务进入广泛的商业用途，这引起了人们的担忧。

由马里兰大学和纽约大学的科学家共同撰写的这项研究确定了图像生成模型(包括稳定扩散)从公共互联网数据(包括受版权保护的图像)中“复制”它们被训练的案例。

需要说明的是，这项研究还没有经过同行评议。一位不愿透露姓名的该领域研究人员通过电子邮件与 TechCrunch 分享了高层次的想法。

“尽管稳定扩散等扩散模型产生了美丽的图像，而且往往是高度原创和针对特定文本提示定制的图像，但我们表明，这些图像实际上可能是从它们的训练数据中复制的，要么是整体复制，要么是只复制训练图像的一部分，”研究人员说。“用扩散模型生成数据的公司可能需要重新考虑知识产权法。几乎不可能验证由稳定扩散生成的任何特定图像是新颖的并且不是从训练集中窃取的。”

来自噪声的图像

最先进的图像生成系统，如稳定扩散就是所谓的“扩散”模型。扩散模型在处理大量训练数据集时，学会根据文本提示创建图像(例如，“一只小鸟栖息在窗台上的草图”)。这些模型——被训练成“重新创建”图像，而不是从头开始绘制图像——从纯噪声开始，随着时间的推移完善图像，使其逐渐接近文本提示。

这不是很直观的技术。但它非常擅长生成几乎任何风格的艺术品，包括照片写实艺术。事实上，扩散已经实现了许多引人注目的应用，从 Lensa 的合成头像到 Canva 的艺术工具。DeviantArt 最近发布了一个稳定扩散驱动的应用程序，用于创建定制艺术作品，而微软正在开发 DALL-E 2，为微软 Edge 的生成艺术功能提供动力。

顶部是由模型训练集中随机字幕的稳定扩散生成的图像。底部是研究人员提示匹配原件的图像。图片鸣谢: Somepalli 等人。

需要澄清的是，扩散模型复制训练图像的元素并不神秘，训练图像通常是不加选择地从网上搜集来的。像 Hollie Mengert 和 Greg Rutkowski 这样的角色设计师，他们的古典绘画风格和幻想景观已经成为稳定扩散中最常用的提示之一，谴责他们认为是拙劣的人工智能模仿，但却与他们的名字联系在一起。

但很难根据经验来衡量复制发生的频率，因为扩散系统是针对来自一系列不同来源的数十亿张图像进行训练的。

为了研究稳定扩散，研究人员的方法是从一个名为 LAION-Aesthetics 的数据集(用于训练稳定扩散的图像集之一)和图像相应的标题中随机抽取 9000 张图像。LAION-Aesthetics 包含配有文字说明的图像，包括有版权的人物图像(如卢克·天行者和蝙蝠侠)，来自 iStock 等受知识产权保护的来源的图像，以及来自 Phil Koch 和 Steve Henderson 等在世艺术家的艺术作品。

研究人员将字幕输入稳定扩散，让系统创建新图像。然后他们为每张照片写下新的说明，试图让稳定的扩散复制合成图像。在使用自动相似性识别工具进行比较后，两组生成的图像——一组是根据 LAION-美学标题创建的，另一组是根据研究人员的提示创建的——研究人员表示，他们通过结果的稳定扩散发现了“大量复制”,包括从训练集中回收的背景和物体。

一个提示——“画布墙面艺术打印”——持续产生显示特定沙发的图像，这是扩散模型将语义概念与图像联系起来的一个相对普通的例子。其他包含单词“绘画”和“波浪”的图片生成的波浪类似于葛饰北斋的画作“神奈川外的巨浪”。

研究人员说，在他们的所有实验中，稳定扩散大约有 1.88%的时间是从训练数据集“复制”的。这听起来可能不多，但考虑到今天扩散系统的影响范围——根据一项粗略的估计，截至 10 月，稳定扩散已经创建了超过 1.7 亿张图像——很难忽视这一点。

“艺术家和内容创作者绝对应该警惕其他人可能在未经同意的情况下从他们的内容中获利，”这位研究人员说。

含义

在这项研究中，共同作者指出，没有一个稳定的扩散世代与他们各自的 LAION-Aesthetics 源图像相匹配，并且他们测试的所有模型都同样倾向于复制。模型复制的频率取决于几个因素，包括训练数据集的大小；较小的集合往往比较大的集合导致更多的复制。

研究人员探索的一个系统，即在开源 ImageNet 数据集上训练的扩散模型，显示“在任何一代中都没有明显的复制，”他们写道。

合著者还建议不要从研究结果中过度推断。受计算成本的限制，他们在实验中只能对稳定扩散的完整训练集的一小部分进行采样。

更多稳定扩散的例子从它的训练数据集中复制元素。图片鸣谢: Somepalli 等人。

尽管如此，他们表示，这些结果应该会促使公司重新考虑收集数据集和在数据集上训练模型的过程。Stable Diffusion 等系统背后的供应商长期以来一直声称，公平使用——美国法律中允许使用受版权保护的材料而无需首先获得版权所有者许可的原则——在他们的模型受到许可内容培训的情况下保护他们。但这是一个未经验证的理论。

“现在，数据是盲目筛选的，数据集如此之大，人工筛选是不可行的，”研究人员说。“扩散模型是惊人而强大的，已经展示了如此令人印象深刻的结果，我们不能抛弃它们，但我们应该考虑如何在不损害隐私的情况下保持它们的性能。”

对于使用扩散模式来推动应用和服务的企业来说，这项研究可能会让他们暂停。在此前接受 TechCrunch 采访时，MBHB 律师事务所的创始合伙人、知识产权法专家 Bradley J. Hulbert 表示，他认为法官不太可能将人工智能艺术作品中的版权作品视为合理使用——至少在 DALL-E 2 这样的商业系统中。出于同样的担忧，Getty Images 已经禁止人工智能生成的艺术作品进入其平台。

这个问题将很快在法庭上得到解决。11 月，一名软件开发商对微软、其子公司 GitHub 和商业伙伴 OpenAI 提起集体诉讼，指控其与 GitHub 的人工智能驱动的代码生成服务 Copilot 违反了版权法。这起诉讼的关键在于，Copilot——它接受了来自互联网的数百万个代码示例的训练——在不提供信用的情况下重复使用许可代码的部分。

除了法律后果，还有理由担心提示可能会直接或间接泄露图像训练数据集中嵌入的一些更敏感的数据。正如最近一份 Ars Technica 报告所揭示的，多达数千份的私人医疗记录就隐藏在 Stable Diffusion 的照片中。

合著者提出了一种解决方案(不是他们首创的)，其形式是一种叫做差分私人训练的技术，这种技术将使扩散模型对用于训练它们的数据“脱敏”——在这个过程中保护原始数据的隐私。研究人员说，不同的私人培训通常会损害表现，但如果其他方法失败，这可能是保护隐私和知识产权的代价。

“一旦模型记住了数据，就很难验证生成的图像是原始的，”研究人员说。“我认为内容创作者正在意识到这种风险。”
