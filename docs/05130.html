松果宣布投资 2800 万美元用于专门构建的数据库

原文：https://web.archive.org/web/https://techcrunch.com/2022/03/29/pinecone-announces-28m-series-a-for-purpose-built-database-aimed-at-data-scientists/

当松果去年推出时，该公司的信息是建立一个专为数据科学家需求设计的无服务器矢量数据库。虽然该数据库是该公司正在做的事情的核心，但它正朝着围绕人工智能驱动的搜索的更精细的数据库用例发展，帮助这些数据科学家在大海捞针。

去年，当我们采访松果创始人兼首席执行官 Edo Liberty 时，他正在进行1000 万美元的种子期，他的公司正在摸索着建立数据库。他来自亚马逊，在那里他帮助建立了 SageMaker 数据库服务。他说，从那时起，他们已经走了很长的路。

“自从我们宣布种子以来，发生了很多变化，所以首先也是最重要的是，我们在 10 月份推出了适当的生产付费服务，自那以来，它在采用率和收入方面都增长迅速，所以事情进展得非常好，”Liberty 说。

他这样描述了在获得种子基金时为数据科学家专门构建数据库的原因:

机器学习模型期望的数据不是 JSON 记录，而是一个高维向量，它是一系列特征或所谓的嵌入，是世界上物品或物体的数字表示。这种[格式]在语义上更加丰富，对于机器学习来说更具可操作性。

他说，如今这种语义丰富的方法正在促使客户使用松果。“矢量数据库的主要用途是搜索，广义上的搜索。它是在文档中搜索，但你可以将搜索视为一般的信息检索、发现、推荐、异常检测等等，”他说。

该系统被组织成多个单元，这些单元是为处理松果数据库中的数据而设计的一组资源。该公司免费提供单个 pod，以帮助客户熟悉产品并进行简单的概念验证。之后，他们开始根据豆荚的数量付费。

他相信该公司已经以这样一种方式构建了该系统，它可以扩展到数十亿个对象。“您可以扩展到您的软件实际能够承受的范围，并且您可以实际进行协调。我们设计了这个系统，所以对于你可以索引和使用多少数据，真的没有任何明确的限制，”他说。

作为一个无服务器的数据库，客户根本不用担心供应问题，但他们必须根据需要处理的数据量告诉 Pinecone 他们每月愿意花费多少。

“他们在信封的背面做了一些工作，认为 x pods 对于我们正在使用的东西来说已经足够了，就它能容纳的数据和它能给我的性能而言，就是这样。”之后，用户只需注册，在控制台上点击几下，调用一个 API 来创建索引，就可以开始运行并准备使用了。

Liberty 不想分享增长数字或员工人数，但他表示，他预计明年员工人数会翻倍(不管这意味着什么)。值得注意的是，这家初创公司在宣布种子期时有 10 名员工。

在多样性方面，他去年说，“我们已经指示我们的招聘人员积极主动地(寻找更多样化的申请人)，确保他们不会错过优秀的候选人，并为我们带来多样化的候选人。”他说，实际上，这意味着今年 50%的新技术人员(相对于员工总数)是女性。

该公司今天宣布了由 Menlo Ventures 牵头的 2800 万美元的 A 轮融资，新投资者 Tiger Global 以及之前的投资者包括 Wing Venture Capital，后者领导了该公司的种子资金。该公司现在已经筹集了 3800 万美元。
