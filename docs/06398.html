哦，太好了！Snap 抢购数据库开发人员 KeyDB

随着 Snapchat 的应用程序继续增长，该公司转向开源社区，以帮助进一步改善其技术基础设施。今天早上，Snap 宣布收购总部位于多伦多的 KeyDB ，这是一家开源、高性能数据库的开发商，具体金额未披露。

该公司表示，该初创公司的六人团队，包括其联合创始人约翰·萨利和本·斯切尔梅尔，将在交易完成后加入 Snap 的基础设施团队，并将努力改善 Snap 的缓存技术及其相当大的工程工作量。

作为一家由 T4 Y Combinator 支持的初创公司，KeyDB 宣称其解决方案是 Redis 的更快替代方案，提供了对多线程、内存效率和高吞吐量等方面的关注。对于 Snap 来说，它看到了将技术和团队的专业知识带入内部的优势，以帮助公司降低团队的运营成本，并释放工程带宽。该公司表示，这也使 Snap 能够更加专注于构建其他核心 Snapchat 体验。

Snap 发言人告诉 TechCrunch，KeyDB 的技术将在 Snap 的所有领域发挥重要作用，包括其货币化和 Spotlight 平台。后者是 Snapchat 的短片视频产品，是抖音的竞争对手，后者现在是 Snap 业务的关键部分。

该公司指出，KeyDB 的多线程数据库利用 Redis 数据结构、闪存和复制，并以快照规模运行。它也是一个开源项目，有一个很大的社区，有超过 300 万的公共下载。

对其他工程师来说幸运的是，Snap 表示将继续开源项目，并在收购后开源 KeyDB 的 Pro 和企业产品。我们被告知，这将使该公司继续受益于更广泛的社区贡献，但也将有助于 Snap 更好地在开源社区中建立自己的存在，并与更多开发者建立关系。

此外，随着 Snap 的工程师将他们自己的内部改进合并到 KeyDB 项目中，该公司相信它将能够帮助解决更广泛社区的挑战。

接下来， KeyDB GitHub 库的访问者将被重定向到 Snap GitHub 库。

与此同时，KeyDB 团队将继续在多伦多工作，Snap 在那里已经有了一个相当大的办公室，其中包括其 Bitmoji 团队。

Snap 在过去一年左右的时间里相当热衷于收购，最近的许多并购都集中在底层技术上。例如，2021 年，Snap 以 760 万美元收购了 3D 地图开发商 Pixel8earth，以 1.24 亿美元收购了试衣技术初创公司 Fit Analytics ，位置数据初创公司 StreetCred ，精神控制的头带制造商 NextMind ，以及一家为眼镜提供动力的 AR 初创公司wave optics，以 5.418 亿美元+13 万美元。

由于 FitAnalytics 和 WaveOptics 的规模，该公司在一份 SEC 文件中披露了它们的价格，但该公司表示，2021 年的所有收购总额仅为 2.661 亿美元。该公司不愿讨论 KeyDB 的价格，但 Crunchbase 报道称该公司获得了 130 万美元的种子资金。
