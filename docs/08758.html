红点创投设立第九个早期投资工具

原文：https://web.archive.org/web/https://techcrunch.com/2022/08/01/techs-riskiest-founders-are-getting-a-650-million-bet-from-redpoint-ventures/

对于风险投资者来说，噪音具有讽刺意味的重要。涉水通过不断寻求资本的创始人和创业项目可能是工作中最困难的部分，但它也是同一份工作成功的必要条件。

那么，如果围绕创业的能量放缓，会发生什么？随着经济衰退的逼近，愿意冒险的创始人越来越少了吗？据 Redpoint 董事总经理安妮·卡达维(Annie Kadavy)称，明年创办的公司总数将少于过去两年。此外，有点违反直觉的是，投资者认为即将到来的减速是“一件大好事”。

“在一个很容易筹集种子资金的环境中，只要你能在你试图解决的问题上投入更多资金，就很容易推出你的第一个产品……这是一种不同的风险，”她说，“相比之下，筹集资金真的很难，我必须开发这些产品，因为我非常关心这个问题。”

她补充道:“我认为，我们将看到的创始人总数将会减少，但质量标准正在提高。”

由 Kadavy、管理合伙人 Erica Brescia 以及普通合伙人 Alex Bard 和 Satish Dharmaraj 领导的 Redpoint Ventures 今天宣布，它已经关闭了一只 6.5 亿美元的基金，以支持初创公司。该投资工具是该公司迄今为止关闭的第九只早期专注基金，它将投资于从种子到 B 轮阶段的公司。根据公司的不同，支票的金额从 200 万美元到 1500 万美元不等。

该公司的目标是将该基金的大部分投资(约 70%)用于首轮融资领域，其余 30%用于种子和 B 轮创业公司。它的目标是获得 15%至 23%的首轮股权。

布雷西亚去年辞去 GitHub 首席运营官的职位后加入了红点，她表示，公司最近没有看到老虎环球(Tiger Global)或软银(SoftBank)等大型基金的太多活动。

“市场上的参与者越多，尤其是(去年)确实会推高价格……现在我们看到估值大幅回落，”她说。“我认为这对创始人和投资者来说都更健康，我相信这部分是因为我们看到越来越少的玩家积极收购同一家公司。”

由于市场情绪的转变，不仅仅是估值在变化；这位投资者表示，由于巨型基金的保守主义，初创企业领域的竞争也在发生变化。“建立一家早期公司更具挑战性、成本更高的一个因素是，你必须击败大量资金雄厚的早期竞争对手，”Kadavy 说。“但如果能有两家或三家公司，而不是 10 家、12 家或 15 家，那么成功的可能性、这些公司雇佣和留住优秀人才的能力，以及它们继续融资的能力都会提高。”

布雷西亚补充说，Redpoint 的产品和作为风险服务的 megafund 的产品看起来相当不同，Redpoint 最大的区别是其早期 GP 团队都由前创始人领导。该公司没有应要求分享其内部收益率目标。

该公司的新资本来自大量的招聘。去年，除了布雷西亚，红点还招募了 GitHub 首席技术官杰森·华纳(Jason Warner)。该团队还增加了米拉·克拉克和乔丹·西格尔作为投资者。
