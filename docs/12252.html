Akeyless 获得现金注入，帮助公司管理他们的密码、证书和密钥

原文：https://web.archive.org/web/https://techcrunch.com/2022/11/16/akeyless-secures-a-cash-infusion-to-help-companies-manage-their-passwords-certificates-and-keys/?utm_source=Telegram&utm_medium=iXNews&utm_campaign=PerspectiveIX.com

早在 2018 年，Intuit 的前安全软件工程师 Refael Angel 就有了一个在云上保护加密密钥的新方法的想法，加密密钥是为了加扰和解码数据而创建的随机比特串。他会见了 Shai Onn，然后会见了安吉尔五年前的同事 Oded Hareven，寻找产品市场契合的迹象。找到它后，三位联合创始人一起建立了一个管理密码、API 密钥和数字证书的服务，在接下来的几年里，它发展成了一个成熟的业务。

安吉尔告诉我，如今，Akeyless 正在蓬勃发展——尽管面临来自 Hashicorp Vault、AWS Secrets Manager 和谷歌云的 Secret Manager 等现有公司的激烈竞争。Akeyless 的客户遍布零售、金融科技、保险和游戏行业，其中包括 Wix 和 Outbrain。过去一年，该公司的收入增长了 350%。

“疫情和由此产生的劳动力趋势，如在家工作的倡议，只是增加了员工远程访问公司 IT 资源的需求，加速了云技术的采用，增加了所需的秘密数量，”沙伊在电子邮件采访中告诉 TechCrunch。在软件开发中，“秘密”指的是像密码和访问令牌这样的凭证。“同样，经济衰退和技术发展放缓只会进一步鼓励组织寻求基于软件即服务的解决方案，以提供更快的部署、低维护甚至零维护、全球自动可扩展性、更低的总拥有成本和更高的采用率。”

为了给未来的增长奠定基础，Akeyless 今天完成了 6500 万美元的 B 轮融资——4550 万美元的股权和 1950 万美元的债务——由 NGP 资本牵头，Team8 Capital 和 Jerusalem Venture Partners 参与。Hareven 通过电子邮件表示，迄今为止，Akeyless 的总资金达到 8000 万美元，新资本给了该公司至少两年半的时间，并将用于各种销售、营销、客户服务和产品开发计划。

“这将使我们能够应对当前的经济气候，并继续向市场提供我们急需的解决方案，”他补充说。

Akeyless 的联合创始人将这家初创公司的成功部分归因于其产品供应的全面性。Akeyless 对组织用来访问其系统、应用和数据的证书、凭证和密钥进行加密和签名。该平台使用位于不同地区和云提供商的加密密钥片段来执行加密操作。Hareven 声称，这些片段从不组合——即使在加密和解密过程中也不组合——其中一个片段是在客户端创建的，以确保 Akeyless 对密钥一无所知。

无钥匙秘密管理仪表板的抽象视图。图片来源:无背景

Akeyless 试图解决的核心问题是 Hareven 所说的“秘密蔓延”他指出，随着公司 IT 环境的扩展，公司用来实现流程、服务和数据库之间认证的密码、API 密钥和证书的数量也在增加。这些密码和密钥存在于代码、配置文件和自动化工具中，带来了可能导致数据泄露的风险。

根据代码安全平台 GitGuardian 的 2021 年调查，1000 个代码提交中有 3 个暴露了至少一个秘密。GitGuardian 估计，应用程序安全工程师平均要处理超过 3400 起秘密事件。在同一年发布的另一份来自 Forrester 的报告中，开发人员透露，在过去的两年中，他们的雇主中有 57%经历过与机密泄露相关的安全事件。

Hareven 继续说，Akeyless 的解决方案是通过现有 IT、开发和安全工具以及灾难恢复等功能的插件来集中秘密。平台存储的秘密可以在公司的所有环境中访问。

Hareven 说:“虽然现代机密管理解决方案解决了[开发]环境的安全挑战，但许多组织仍然被迫依赖孤立和互不关联的工具来保护传统环境中的机密。“我们的客户表示需要整合传统工具，以降低风险并提高所有环境和使用案例的合规性。”

Akeyless 无疑占据了一个巨大且有利可图的领域——大观研究公司预测到 2025 年，密码管理软件的市场价值将高达 20.5 亿美元。但是它将不得不抵挡像 Doppler 这样的竞争对手，后者最近为其平台筹集了 2000 万美元，以帮助公司管理他们的应用程序秘密。另一个挑战将是说服顽固分子接受秘密管理作为一门学科；根据一份报告，截至 2019 年，只有 10%的组织在使用秘密管理解决方案。

如果 Akeyless 的联合创始人有顾虑，他们没有表现出来。相反，Hareven 指出了该团队在网络安全方面的记录——Onn 之前的安全项目 Fireglass 被赛门铁克以 2.5 亿美元收购——并指出 Akeyless 正在扩张，计划到明年年底将其 80 人的员工人数增加一倍。

Hareven 在我们的谈话中没有提到，但 Akeyless 也可能受益于风险投资对网络安全的持续广泛兴趣。根据 PitchBook 的数据，今年对安全初创公司的风险投资超过了 130 亿美元，高于 2020 年的 114.7 亿美元。

“事实上，我们是一个软件即服务提供商，没有版本和支持的‘内部技术债务’，这使得我们的经济效率更高，使我们能够更快地响应市场需求和快速创新，”Hareven 说。
