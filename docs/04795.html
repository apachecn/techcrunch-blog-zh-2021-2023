随着 iOS 15.4 的到来，面具解锁终于在 iPhone 上上线了

就在苹果重大事件后不到一周，该公司推出了三个操作系统的最新版本:iOS/iPadOS 15.4 和 macOS 12.3 现已上线。对于那些没有在测试电路上生活的人来说，这意味着一系列关键功能，包括使用带有面具解锁的 Face ID 和通用控制，这提供了 iPad 和 Mac 之间更加无缝的集成。

操作系统在新的 iPhone SE 和 Mac Studio 之前到来——其中一个我们刚刚评论了。整个面具解锁的事情一直是一个相当大的 bugbear 在过去，哦，两年左右。苹果以 Apple Watch 解锁的形式提供了一个很好的权宜之计，但这个新版本让更多用户可以更容易地使用它(尽管，同样，你可以始终坚持使用 SE)。

我已经使用测试版几个星期了，对结果很满意。戴上面具和眼镜效果很好，这是我开始旅行以来第一次真正的旅行。当然，当我的眼镜起雾时，它会挣扎，但我不知道，这可能是霍尼韦尔和沃比·帕克的错。振作起来，伙计们。操作系统将引导你设置选择加入功能，同时警告你，当依赖全脸时，事情自然会更安全。

图片来源:苹果

手机方面的新功能还有点击支付功能，该功能取消了中间人读卡器，允许 iPhone 用户通过 NFC 芯片接受支付。与此同时，还有一大堆新的表情符号——总共 37 个。包括一张融化的脸和另一张偷看的脸，这可能没有给上周的“偷看表演”活动命名，但是，真的，谁能说得准呢？

这里的另一个大新闻是在 iPadOS/MacOS 端。确切地说，苹果没有把它定位为边斗杀手，但它无疑将为许多用户扮演这个角色。该功能可以在 Mac 和 iPad 之间同步键盘、触控板和鼠标。这意味着你可以在设备和操作系统之间拖放文件，有效地使用更加无缝的隔空投送版本。

这两种操作系统现在都可以在您的兼容设备上下载。
