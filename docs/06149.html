苹果、谷歌和微软联手开发无密码登录

原文：https://web.archive.org/web/https://techcrunch.com/2022/05/05/apple-google-microsoft-passwordless-logins/amp/

图片来源: Getty Images

在一场罕见的联盟秀中，苹果、谷歌和微软联手扩大对手机、桌面和浏览器无密码登录的支持。

众所周知，密码是不安全的，根据威瑞森的年度数据泄露报告，80%以上的数据泄露都是由脆弱且容易被猜到的凭证造成的。虽然密码管理器和多因素技术提供了渐进的改进，但苹果、谷歌和微软正在合作创造更方便、更安全的登录技术。

科技巨头周四宣布他们正在扩大对 FIDO 联盟和万维网联盟的无密码登录标准的支持，这意味着你很快就可以使用智能手机在附近的设备上登录应用程序或网站，无论你使用的是什么操作系统或浏览器。您将使用每天多次使用的相同操作来解锁您的智能手机，例如验证您的指纹、面部扫描或设备 PIN。

用户还将能够跨多个设备(包括新设备)自动访问其 FiDO 登录凭据或“密码”，而无需重新注册每个帐户。

虽然这三家公司长期以来一直支持由 FIDO Alliance 创建的无密码登录标准，但用户仍然被迫使用每台设备登录每个网站或应用程序，然后才能使用无密码功能。在接下来的一年里，三大科技巨头将在 macOS 和 Safari 上实现无密码 FIDO 登录标准；安卓和 Chrome 窗户和边缘。这意味着，举例来说，用户将能够使用苹果设备上的密钥登录运行微软 Windows 的谷歌 Chrome 浏览器。

这将使黑客更难远程泄露登录信息，因为登录需要访问物理设备。

“与行业合作，建立新的、更安全的登录方法，提供更好的保护，消除密码漏洞，这是我们致力于打造最大限度安全和透明用户体验的产品的核心——所有这些都是为了保护用户的个人信息安全，”苹果平台产品营销高级总监库尔特·奈特(Kurt Knight)在一份新闻稿中说。

这一新的集体承诺受到了美国网络安全和基础设施安全局(CISA)局长 Jen Easterly 的赞扬，他称之为“最终将使美国人民更加安全的前瞻性思维”

“在 CISA，我们正在努力提高所有美国人的网络安全底线，”伊斯特利补充说。“今天是安全之旅中的一个重要里程碑，它鼓励内置的安全最佳实践，并帮助我们超越密码。网络是一项团队运动，我们很高兴能继续合作。”

虽然这个密码迄今为止躲过了多次试图永远杀死他们的企图，但这可能是密码棺材中的最后一颗钉子。
