FTX 首席执行官领导可信媒体的种子轮，帮助建立社区拥有的网络 3 秀 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/07/27/ftx-ceo-leads-trustless-medias-seed-round-to-help-build-community-owned-web3-shows/

一个新的专注于网络 3 的制作中心，无信媒体，已经脱离秘密模式，它的目标是建立一个 NFT 社区拥有的媒体公司。

该公司在种子期筹集了 325 万美元，并计划推出“NFTV 节目”，这将允许拥有 NFT 的社区成员拥有节目内容。

这轮投资由 FTX 首席执行官山姆·班克曼-弗里德牵头，包括来自雪崩公司艾娃实验室和红岛创始成员梅根·卡斯帕等人的投资。

“NFTV 是 NFT 和电视的结合体。“当我们描述什么是新词时，这是一个不断出现的问题，”无信媒体的联合创始人扎克·古兹曼对 TechCrunch 说。“整个概念是试图找到一种方法，打破内容创作者和他们创作的节目观众之间的一堵墙。”

Guzman 说，这家初创公司的目标不是“通过销售 NFT 赚很多钱”，而是希望吸引真正优秀的合作伙伴和社区来打造 web3 节目。

制作公司将在秋季推出其第一个 NFTV 节目硬币，并计划以怀疑的眼光专注于探索 crypto 的最大故事。古兹曼之前是雅虎财经和美国消费者新闻与商业频道的记者，他将主持这场旗舰秀。

“归根结底是信任，尤其是在加密领域。古兹曼说:“我认为我们正在努力创造新词，这是我们需要这样一个节目的原因，我们所看到的崩溃已经证明了这一点。”。

班克曼-弗里德在一份声明中说:“由 web3 本地人开发的高质量教育和新闻资源，对该技术的基本理解，它的发展方向，以及它能带来的价值，将对该领域的持续增长非常重要。”“作为一个行业，我们必须回答一些大问题。”

“这就像是，人们能够更早地被警告一些项目中的弱点吗？或许，如果有更多诚实的对话，”古兹曼说。“人们会谈到集中式密码出借人的一些陷阱吗？可能，如果加密影响者没有得到报酬来推广他们。传统媒体是否也能更好地揭露他们呢？或许吧，如果他们有一个更专注于加密的编辑部。”

无信传媒最初推出的“网络 NFT”将围绕一个博弈论和信任实验来建立其管理社区。古兹曼从 1000 份可用的 NFT 中挑选了前四名受邀者:班克曼-弗里德；卡斯帕；艾娃实验室主席吴炯；以及 GV(前身为谷歌风投)合伙人特里·伯恩斯(Terri Burns)。然后，这四个人可以邀请其他人，这实质上就产生了邀请“网络 NFT”社区成员的连锁反应。

“我们从 crypto 的一些最聪明的人那里筹集了资金，我们将依靠他们，”Guzman 说。“他们显然支持这一愿景，我们认为他们知道向谁发出邀请，并邀请值得信任的人来帮助构建这一愿景。”

从历史上看，一些最成功的娱乐和新闻类主流电视节目拥有大量的追随者和社区意识。古兹曼说:“世界上任何一个节目的影响力都取决于它所吸引的观众。”。“为什么人们喜欢《老友记》或者《宋飞正传》？文笔不错，但也有狂热的粉丝群。”

无信传媒计划通过扩大网络，使其社区成员超过最初的 1000 名 NFT 社区成员。在第一季结束时，它将向公众推出“NFT 核心小组”，其中有 9000 个 NFT，这样一个社区就可以共同建设第二季。还有一种“订户 NFT”，没有最大供应量，可以自由铸造(除了交易费)。

“随着许多 NFT 下降，我觉得他们专注于铸造，所以价格上升，或者最近，有几个下降，创作者选择谁可以铸造或谁是第一批非功能性食品，”古兹曼说。“那里没有太多的透明度，也没有太多的分权倾向。因此，通过信任实验，我们试图建立一个社区如何相互信任。”

古兹曼说，NFTs 可以为媒体做很多尚未开发的事情。“这里的概念验证是你如何在任何利基市场创建一个社区运营的节目。”

未来，可信媒体计划将其 web3 节目扩展到加密领域之外，进入其他流行的数字媒体领域，如食品或旅游。

古兹曼说:“你可以让人们称体重，创建一个类似‘用餐、免下车和潜水’的网络。”。“不仅仅是盖伊·菲耶里选择他要去的地方，而是社区选择他们喜欢的餐馆让节目参观。”

但古兹曼表示，无信传媒的主要目标是在一个不信任的世界里，在社区中建立信任，这是对制作中心名称的认可。“我们试图从零开始创建一个信任社区，一个不仅会被该领域的其他人信任，也会被该领域之外的人信任的社区。”
