“虚拟友谊”人工智能聊天机器人 Replika 因儿童安全问题在意大利遭遇数据禁令

原文：https://web.archive.org/web/https://techcrunch.com/2023/02/03/replika-italy-data-processing-ban/

总部位于旧金山的人工智能聊天机器人制造商Replika——运营着一项基于可定制数字化身的免费增值“虚拟友谊”服务，其“个性化”回应由人工智能驱动(按照其宣传，旨在让人类用户感觉更好)——已被意大利隐私监管机构下令停止处理当地用户的数据。

Garante 表示，他们担心 Replika 的聊天机器人技术对未成年人构成风险，并且该公司缺乏根据欧盟数据保护规则处理儿童数据的适当法律基础。

此外，监管机构担心人工智能聊天机器人可能会给情感脆弱的人带来风险。它还指控 Replika 应用程序的开发者 Luka Inc .未能满足地区法律要求，明确传达其如何使用人们的数据。

停止处理意大利人数据的命令立即生效。

在宣布其干预的新闻稿中，监管机构表示:“人工智能聊天机器人使用文本和视频界面生成‘虚拟朋友’，暂时不能处理意大利用户的个人数据。意大利 Garante 对开发和运营该应用程序的美国公司实施了临时数据处理限制；该限制将立即生效。”

“最近的媒体报道以及 SA(监管机构)对‘Replika’进行的测试表明，该应用对儿童存在实际风险——首先也是最重要的是，他们收到的回复绝对不适合他们的年龄，”它补充道。

Replika 是 OpenAI 的文本生成大型语言模型技术 GPT-3 的早期 API 合作伙伴——尽管它的服务不是运行在 GPT-3 的副本上(也不是 OpenAI 的 buzzy ChatGPT 的相同技术)。相反，这家初创公司声称它“微调过的”GPT-3，使用一个经过对话训练的网络机器学习模型，来磨练其特定用例的生成技术:对话式(它声称“移情”)人工智能伴侣。

然而，在此之前，人们已经对该技术可能给儿童带来的风险提出了担忧——从担心儿童接触不适当的内容到更普遍的担忧，即他们可能会对互动上瘾，或者只是被鼓励花很多钱定制他们的头像或获得其他付费内容。但意大利监管机构似乎是第一个对儿童安全采取正式行动的监管机构。

Garante 的命令指出，该应用程序的几个用户评论报告了不适当的性内容。它还指出，虽然该应用程序在苹果的 iOS 和谷歌的 Android 应用商店中被列为 17 岁以上，但开发者的服务条款只禁止 13 岁以下的人使用。虽然 18 岁以下的人需要获得父母或监护人的授权，但监管机构指出，该应用程序并不寻求验证用户的年龄，也不阻止未成年人提供他们的年龄信息——因此它认为 Replika 未能保护儿童。

“实际上没有年龄验证机制:没有针对儿童的门控机制，如果用户声称自己未成年，也不会屏蔽该应用。在创建账户的过程中，该平台只会要求用户提供姓名、电子邮件账号和性别，”它观察道。“聊天机器人提供的‘回复’通常明显与儿童和弱势群体有权获得的强化保护相冲突。两个主要应用商店的一些评论包括用户对不适当性内容的标记。

“‘Replika’违反了欧盟数据保护条例:它不符合透明度要求，并且非法处理个人数据，因为合同的履行不能被援引为法律依据，即使是隐含的，因为根据意大利法律，儿童没有能力签订有效的合同，”Garante 补充说，并表示它已经命令其美国开发商停止处理与意大利用户有关的数据——给它 20 天时间来传达为遵守命令而采取的措施。

它进一步指出，不遵守该命令将面临最高 2，000 万€的罚款，相当于全球年营业额的 4%。

我们联系了 Replika，要求其对 Garante 的订单做出回应。

欧盟的《一般数据保护条例》( GDPR)非常强调保护儿童的信息和隐私——例如，建议可能将未成年人作为用户的服务应该考虑采用儿童友好型设计，并积极主动地进行风险评估，以确保他们发现潜在的安全和其他权利问题。

该地区的监管机构已经表示愿意关注这一领域的侵权行为。

例如，去年秋天，Instagram 因侵犯儿童隐私被罚款近 4.4 亿美元。欧洲的消费者保护机构也对抖音的儿童安全表示担忧——尽管对抖音处理儿童数据的调查仍在进行中。

意大利数据保护监管机构近年来显示出对儿童安全问题的特别敏感——两年前使用紧急干预措施，命令抖音屏蔽无法验证年龄的用户，作为回应，此前有报道称一名儿童在该平台上参与了危险挑战。这导致了 50 多万账户被清理。

然而，尽管围绕儿童安全问题执行了一些 GDPR(和消费者保护法),活动组织认为儿童仍然没有得到适当的保护——并继续推动更严厉的法律。因此，限制只会变得更加严格。

在英国，一项着眼于保护未成年人免受安全和隐私风险的适龄设计规范于 2021 年秋季生效。同时，法国数据保护监察机构也发布了一套建议以确保儿童的数字权利得到保护。

英国也在努力通过关注儿童安全的在线安全法案，以回应公众对儿童在网上接触到什么的担忧。

最近几个月，欧盟立法者也同意全面禁止为广告定位处理未成年人的数据，这是欧盟数字规则手册的两个旗舰更新:数字服务法案和数字市场法案，它们将于今年晚些时候开始适用。
