米拉保险技术公司筹集了 370 万美元种子基金，由哈莱姆资本公司牵头

原文：https://web.archive.org/web/https://techcrunch.com/2022/08/02/kenyan-insurtech-lami-raises-3-7m-seed-extension-led-by-harlem-capital/

从 2020 年推出第一款产品开始，肯尼亚保险技术公司米拉技术公司就着手增加肯尼亚和非洲其他地区的保险普及率。它通过构建和分发端到端的数字保险平台和 API 开始走上这条道路，允许不同行业的企业为他们的客户创建定制的保险解决方案。

随着其 API 平台在保险领域的发展，这家初创公司还计划提供技术解决方案，将代理人和经纪人也数字化，帮助他们简化运营——以覆盖广泛的客户群并在线销售。这些计划还包括随着创新不断塑造行业，实现传统保险公司的数字化。

在接受 Techcrunch 采访时，米拉的创始人兼首席执行官韩吉·阿巴斯进一步宣布了提供额外保险产品线的计划，同时还透露该初创公司已进入埃及和尼日利亚。所有这些都是在由哈林资本领导的一轮融资中筹集的 370 万美元种子期的背景下发生的——该公司投资少数族裔和女性创始人。

参与该轮投资的其他投资者包括早期风险投资公司 Newtown Partners、纽约研究驱动的风险投资公司 Social Impact Capital 的合伙人彼得·布鲁斯-克拉克(Peter Bruce-Clark)、insure tech Connect(insure tech 创新者的网络平台)的 Caribou Honig 和 Jay Weintraub，以及专注于新兴市场的企业融资和并购精品公司 Exotix Advisory 的高级成员。最新的资金增加了 insurtech 去年筹集的 180 万美元种子资金。

谈到其增长战略，阿巴斯肯定了该公司不断创新的计划，因为他们正在探索新的方法来提高整个非洲大陆的保险渗透率——目前低于 3%。

“在技术方面，我们希望迎合整个保险生态系统。因此，不仅是数字平台想要销售保险产品，而且要帮助现有的参与者更有效地分销产品，让他们在提高保险渗透水平方面发挥作用。这包括代理人和经纪人——我们正在研究如何让他们销售更多的保单。

Abass 补充说，米拉正在利用这笔资金招聘，快速跟踪其扩张计划，并推动承销商合作伙伴关系。该公司还任命其首席财务官罗伊·珀洛特(Roy Perlot)为联合创始人，他自 2020 年以来一直在 insurtech 工作。

Harlem Capital 的负责人 Gabby Cazeau 在解释他们在米拉的投资时说:“我们相信下一波金融科技将把保险等金融产品和服务嵌入到客户的购买体验中。米拉通过电子商务和金融领域的战略合作伙伴为人们服务的方式是与用户建立信任并以无缝、便捷的方式向非洲大陆的非洲人提供保险的最佳方式。”

米拉与其承保伙伴(目前有 25 个)共同设计产品，这些承保伙伴通过 B2B2C 方式帮助使用其 API 来促进保险产品的分销。

已经扩展到尼日尼亚和埃及的米拉技术公司计划提供额外的保险产品线。图片来源:米拉科技

这家初创公司的 API 允许银行等实体向其客户提供数字保险产品。米拉还将其 API 与超过 15 个不同领域的实体进行了整合，包括物流、电子商务、银行和金融科技。

米拉的合作伙伴之一是“现在购买，以后支付”的初创公司 Lipa Later，它通过为融资产品提供支付违约保险来支持该公司。insurtech 还与熊智娟合作，使东非(肯尼亚、乌干达和坦桑尼亚)的货运公司能够在每次旅行的基础上获得过境保险，并与 Kwara 合作，使超过 60，000 名 SACCO(信用社)成员能够获得保险产品。其他客户包括电子商务平台 Jumia、零售 B2B 和端到端分销平台 MarketForce 以及 Stanbic Bank Insurance，后者使用米拉的技术来支持银行保险产品。

米拉于 2020 年 1 月进入市场，推出了面向消费者的产品 Griffin insurance，这可以说是第一个数字汽车保险平台。然后它转向提供一个支持数字保险产品的产品无关的 API 平台。这帮助它在 2021 年底将其投资组合从 70，000 份保单增长到 85，000 份，并在去年将承保的保费翻了两番，达到 800，000 美元——该公司预计今年将增长到 200 万美元以上。

Stanbic 银行使用米拉的 SaaS 平台为其银行保险产品提供支持。图片来源:米拉科技

预期的增长将在很大程度上受到不断增加的合作伙伴关系和产品线的推动，以及今年在收购 Bluewave 后进入马拉维和刚果民主共和国(DRC)时开始的积极扩张计划。此次收购有助于米拉利用 Bluewave 的客户和技术，通过 USSD 和 WhatsApp 聊天机器人等不同渠道获得小额保险产品。

Abass 预计，随着更多科技企业的合作以及更多保险公司的数字化，将会有更多的增长机会。

“保险公司已经意识到，他们不能再依赖传统的分销渠道；为了自身的财务健康，他们还需要多元化。此外，当谈到数字平台时，我认为他们也意识到这是一种使收入来源多样化的简单方法。

“自从两年前我们推出第一款产品以来，情况已经发生了很大变化。人们对数字分销渠道更加开放。我们期望看到 B2B2C 方面的显著增长。随着我们向其他市场扩张，这是我们看到大量机会的地方。”
