谷歌引入印度的 DigiLocker 集成到文件应用程序来访问官方文档

谷歌宣布将印度的在线文档存储服务 DigiLocker 引入 Android 上的 Files 应用程序，让用户可以从该应用程序访问经过验证的政府发布的文档。

在周一的谷歌印度年会上，谷歌宣布与印度政府合作，在文件应用程序中推出 DigiLocker 集成。这家搜索巨头还宣布了一个基于机器学习的模型，该模型将帮助识别和组织重要文件，包括官方文件和政府身份证。

Abhishek Singh 在宣布与谷歌的合作时表示:“我们希望 DigiLocker 在 Android 上的集成和合作将推动我们所有用户以安全可靠的方式更顺畅、无处不在地访问数字化文档。”。辛格是 NeGD 的总裁兼首席执行官，数字印度公司(DIC)的董事总经理兼首席执行官，同时也是首席执行官 Karmayogi Bharat。

辛格表示，DigiLocker 拥有超过 1.37 亿注册用户。他说，该服务还拥有超过 2，300 家发行商，迄今已发行超过 56 亿份文件。

DigiLocker 在 3 月超过了 1 亿用户的用户基数。它使用 API 来检索、数字化和存储政府发布的文件。

谷歌没有透露用户何时可以使用该集成的确切时间表。它也没有透露这种体验是否会在未来的某个时候提供给 iOS。

在第八届年会上，谷歌还宣布了与搜索相关的人工智能驱动的更新，并展示了谷歌眼镜的人工智能产品，该产品将帮助用户解码医生的笔迹。此外，谷歌在 YouTube 上引入了课程作为特色，为开发教育内容的创作者提供更多的盈利空间。
