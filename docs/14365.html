微软终止 Windows 7 安全更新

为 Windows 7 倒一杯，这个十年前的操作系统今天到达了安全线的尽头。在微软停止对 Windows 7 的主流支持大约三年后，这家科技巨头将不再提供安全更新，剩下的用户可以选择升级到更新的操作系统，或者继续面对持续的安全威胁。

Windows 7 在服务了十多年后于 2020 年 1 月寿终正寝，但微软允许客户支付扩展安全支持，以帮助维护不容易升级的遗留或旧设备，如医院扫描设备和生产线系统。

那些扩展的安全更新今天结束了。目前还不清楚到底有多少台 Windows 7 电脑仍在运行(或连接到互联网，这增加了它们的风险)，但一些市场份额数据显示，这一数字高达约十分之一。

如果没有扩展的安全更新，Windows 7 将继续运行，但不会再收到针对新的和现有的安全漏洞的补丁。

继 Windows 7 之后的操作系统版本 Windows 8.1 也在周二达到了支持终止的里程碑，这距离它发布已经过去了将近 10 年。微软表示不会为 Windows 8.1 提供扩展的安全更新，这可能是因为它的历史低使用率，因为许多人跳过了操作系统，直接更新到 Windows 10。

与此同时，微软 Edge 的最新版本(版本 109)定于周四发布，是支持不再受支持的 Windows 7 和 Windows 8.1 的最新版本。
