从合作伙伴到竞争对手:Stripe 的最新举措对 Plaid TechCrunch 意味着什么

原文：https://web.archive.org/web/https://techcrunch.com/2022/05/04/from-partners-to-competitors-what-stripes-latest-move-means-for-plaid/

今天早些时候，支付巨头 Stripe 宣布了一款新产品，填补了其作为商户和其他业务的金融服务层的一些重大空白，这些业务的模式是基于交易的，据 TC 的 Ingrid Lunden 报道。

该新产品被称为金融连接，该公司表示，它将为 Stripe 的客户提供一种直接连接到其客户银行账户的方式，以访问金融数据来加速或运行某些类型的交易。

如果这听起来很熟悉，那是因为这几乎正是另一家金融科技巨头 Plaid 已经在做的事情。Stripe 的这一举动使该公司直接与以前的合作伙伴 Plaid 竞争。

给不熟悉的人一个快速的背景介绍:Stripe 一年前的估值为 950 亿美元，据报道现在的估值已经接近首次公开募股。Stripe 由约翰·科利森(总裁)和他的兄弟帕特里克·科利森(首席执行官)于 2010 年创立，在数字支付，特别是在线支付开始兴起的时候，Stripe 看到了为开发者建立一个简单的方法，通过几行代码将支付集成到任何应用或网站的价值。

如上所述，Plaid 帮助消费者将银行账户与金融应用联系起来，它在 2021 年 4 月筹集了 4.25 亿美元的 D 轮融资，估值为 134 亿美元。这家成立 9 年的公司去年成为头条新闻，当时它以 53 亿美元的价格被消费信贷巨头 Visa 收购的交易因监管担忧而告吹——许多人认为这一事件对这家初创公司来说是塞翁失马，焉知非福。

现在的紧张局势是，包括 Plaid 首席执行官和联合创始人扎卡里·佩雷特在内的一些人认为，Stripe 可能利用了与 Plaid 的关系来收集如何开发这种产品的信息。

格子和条纹都拒绝对这个话题发表评论。

但是在一条回应条纹首相杰伊·沙阿自己的关于他公司新产品的推特的推特中，佩雷特写道:“哇！Jay，在过去的几年里，你接受了 Plaid &的采访，问了很多试探性的问题，你的团队也重复发送了 RFP(在 NDA 的领导下！)向我们索要大量的详细数据。我希望你们在这些产品上一切顺利，但看到这些方法还是令人惊讶。”

有趣的是，Stripe 的新产品由格子的竞争对手 MX 和 Finicity 提供动力。虽然有些人可能会说，格子不拥有开放的银行市场，但其他人似乎对 Stripe 可能没有向格子公开其意图感到更不安。

虽然 Perret 没有对媒体说话，但他在黑客新闻上对关于 Plaid 的产品和刚刚发布的 Stripe 之间的差异直言不讳。在那次交流中，他与 Stripe 的“埃德温”就两家公司业务的各个方面进行了反复讨论。

这个消息可能不会让 Perret 或 Plaid 感到震惊，因为 3 月 29 日，他引用了帕特里克·科利森关于 Stripe 推出其首款 ACH 支持产品的推文，只说看到该公司的声明“很酷”，还有“很多东西要推出”Stripe 的 ACH 借记产品的这些更新是今天新闻的一个暗示，因为它推出了即时验证银行账户的能力，这一功能与 Plaid 的 Auth 产品具有竞争力。

这两家公司曾经是合作伙伴。在 Plaid 的网站上，有两家公司合作的细节，以“提供无摩擦的资金转移，而无需处理帐户或路由号码。”那里的语言说:

使用 Plaid Link 立即验证您客户的账户，并自动生成一个条带银行账户令牌，以便您可以通过其 ACH API 接受 ACH 付款。本指南专为那些已经在 Stripe 和 Plaid 上拥有 ACH 启用帐户的用户而设计。如果不是您，请前往条带 ACH 文档开始。你可以在那里注册一个格子账户。

没有消息表明这种合作关系是否仍然存在。但是从今天的新闻来看，即使从技术上来说仍然如此，它的未来似乎也岌岌可危。
