很长一段时间以来，第一次少有人谈论某种三个字母的密码交换，咳咳，FTX。但当然，这并没有持续多久，这位前 FTX 首席执行官(他也有三个字母——SBF)创建了一个子堆栈，并在周四分享了他的观点——在一份 2300 字的概述中——详细说明了 FTX 和阿拉米达的“死前”

在过去的七天里，加密领域也宣布了一些裁员，从第二大交易所比特币基地裁员 20%到 NFT 市场 SuperRare 裁员 30%。

这是比特币基地的第二轮大裁员，去年 6 月裁员 18%，约 1100 人，但“如果不考虑裁员，就没有办法大幅削减我们的开支，”比特币基地联合创始人兼首席执行官 Brian Armstrong 周二在一篇帖子中写道。

在这些裁员带来的厄运和悲观情绪中，许多市场参与者试图帮助那些受到裁员影响的人，包括索拉纳联合创始人阿纳托利·亚科文科，他在推特上对 T21 说，“如果你是一名即将离职的比特币基地员工，伸出援手！我很乐意在生态系统中给你找个家。”

看，不是所有人类的希望都要失去。

对于合作伙伴来说，这也是重要的一周，因为更多的加密公司加入了主流 Web 2.0 和金融公司如 AWS 和万事达卡的行列。但我不会在这里多说，我会让故事自己说话。请在下面阅读更多相关信息。

本周在 web3

AWS 与 Avalanche 合作，为企业和政府扩展区块链解决方案

两家公司独家告诉 TechCrunch，亚马逊网络服务(AWS)已经与艾娃实验室合作，该公司正在构建第一层区块链雪崩，以帮助企业、机构和政府扩大区块链的采用。“展望未来，web3 和区块链是不可避免的，”AWS 副总裁兼全球初创公司负责人 Howard Wright 对 TechCrunch 说。“没有人可以预测它将会发生的时间、日期或季度，它将成为主流，但我们以前已经看到过增长周期。这一次的速度似乎在加速，我们很高兴能成为其中的一员。”

6 位加密投资者谈论 DeFi 和 2023 年采用的未来之路 (TC+)

由于市场普遍低迷，以及一系列丑闻和市场混乱导致的信任动摇，秘密风险投资行业变得更加挑剔，但主要公司的投资者仍在该领域开出支票。随着市场展望未来，一些风险投资家正在修改他们的投资策略，而其他人则坚持他们目前的计划，也许有一两个小的调整。请继续阅读，了解积极的投资者对 DeFi 的看法，他们如何在缺乏资金的情况下为其投资组合公司提供建议，接近他们的最佳方式等。

万事达卡推出多边形网络 3 艺术家孵化器

世界上最大的金融支付提供商之一万事达卡(Mastercard)正在推出一个专注于 web3 的孵化器，以帮助艺术家通过一种新的媒体与粉丝联系，该公司分享道。万事达卡与 Polygon 合作，Polygon 是一个建立在以太坊基础上的缩放区块链，最近在 Web 2.0 生态系统中取得了巨大进展。万事达卡首席营销和沟通官拉贾·拉贾曼纳对 TechCrunch 表示，加入孵化器后，参与的艺术家应该知道如何制作 NFT，在虚拟世界中表现自己，并建立一个社区。

专注于网络 3 的 Beacon 与 13 家加密初创公司一起推出旗舰演示日

我们现在只是在 2023 年的第二周，但演示日已经开始，创始人们试图在不断变化的加密市场中保持活力。Beacon 是一个专注于 web3 的早期加速器项目，于去年推出，其旗舰团队刚刚毕业。第一组的团队被称为第 0 组，他们在周二的演示日展示了他们的想法，TechCrunch 独家报道。

当提到 web3 时，投资者说他们会长期参与其中

不管宣传与否，web3 公司看起来已经存在，投资者似乎更愿意继续支持它们。为了更好地了解填写支票的人对 web3 的看法，TechCrunch 调查了超过 35 名投资者，结果发现大多数人不仅积极投资该类别，还对他们认为具有潜在变革性的技术的光明未来抱有希望。
