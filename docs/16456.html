上周 ui path 报告其跟踪财务业绩时，这就是它所做的，其中包括与分析师预期相比的顶线和底线表现。其股价飙升。

事实证明，虽然裁员、缩减项目和更加尊重现金是当今科技公司及其许多客户的时尚，但这种趋势有一个小问题。降低员工成本的一个方法就是裁员。另一个是让它更有成效，让你的每一美元的花费更有效。

这就是 UiPath 和更大的自动化市场——机器人过程自动化，或 RPA——可能在其他软件类别上具有优势的地方。Appian 上个月的积极收益报告以及在收益电话会议上对其自动化工作的冗长讨论强调了科技公司对自动化帮助的强烈需求。

关于这一点，我们已经有了更多的数据。我们之前讨论过的最近一份关于电池企业软件支出的报告包含了更多乐观的数据。

简而言之，当然，每个人都想在软件支出上节省一美元。但是，如果你的初创公司正在开发自动完成任务和快速提高生产率的技术，你或许可以避开低迷。大家说说吧。
