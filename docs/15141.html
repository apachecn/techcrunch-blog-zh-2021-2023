与 Vanta 和 Sequoia 一起在 TechCrunch Live 上练习你的创业营销

在本周的 TechCrunch Live 上练习推销你的创业公司。我有两个很棒的客人，他们带来了很多创业经验。本周 TechCrunch Live 活动的三位嘉宾将有两分钟时间练习他们的电梯推销，他们将获得两位嘉宾四分钟的坦诚反馈。

本周我们邀请到了 Vanta 首席执行官克里斯蒂娜·卡乔波和红杉资本合伙人安德鲁·里德。安德鲁自 2014 年以来一直在红杉投资初创公司。克里斯蒂娜获得了斯坦福大学的本科和研究生学位，是联合广场风险投资公司的前投资者、教授和 Dropbox 产品经理。她知道一家企业如何获得启动资金。

被选中的参与者有两分钟时间推销他们的公司。然后，我们的嘉宾将提供关于推介的反馈:信息是否以符合逻辑的格式呈现，创始人说话是否权威，推介中缺少什么？

TCL 的使命仍然是帮助创始人建立更好的风险投资支持的企业。但进入 2023 年，这项任务背后有了新的紧迫性。TechCrunch Live 始于 2021 年令人兴奋的日子，现在是 2023 年初，创业世界正在经历新的挑战。融资更加困难，销售周期更长，投资者(和他们的有限合伙人)有不同的期望。让 TechCrunch Live 给你一点反馈。

以下是如何参与音高练习:

  • 在 Hopin 上注册并加入节目，节目将于美国东部时间下午 2:30/太平洋时间上午 11:30 开始。第一次面试从 3 点/12 点开始。
  • 使用此表格申请介绍您的公司。
  • 在与 Christina 和 Andrew 进行 30 分钟的交谈后，被选中的创始人将有两分钟的时间进行推介，并将获得四分钟的反馈。
  • 你不需要一个球场甲板来参与。

我们在寻找那些对早期创业公司有着精心准备的营销策略的创业者。

没有入选这个节目？没有烦恼；下周试试。这一部分是 TechCrunch Live 的常规功能。
