Shopify 收购 Remix 以支持其店面设计工具

Remix ，一家开发类似于 Next.js 的开源网络框架的初创公司，已经被 Shopify 收购，两家公司今天在一份联合声明中宣布。财务条款没有披露，但在博客帖子中，Remix 首席执行官迈克尔·杰克逊表示，Remix 将获得 Shopify 的“长期支持和支持”，这将使其“增长更快”，并“专注于性能和可扩展性。”

“你会在野外看到更多的(Remix 框架)，为一些最大的商业网站提供动力，”杰克逊说。“此外，Shopify 本身将在许多项目中使用 Remix，随着时间的推移，你可以期待看到更多 Shopify 的开发者平台包括对 Remix 的一流支持。”

Remix 是由前 Twitter 工程师杰克逊和瑞安·弗洛伦斯在 2020 年共同创立的。在决定推出同名的 Remix 框架之前，两人合作多年，围绕 React 创建了开源工具，React 是一个用于构建应用程序 ui 的 JavaScript 库。

杰克逊和佛罗伦萨最著名的项目之一是 React 路由器，这是 React 的一个库，已经被下载了近十亿次。并非巧合的是，Shopify 最初使用 React Router 来构建 Hydrogen，这是该公司用于构建定制 Shopify 店面的前端 web 开发框架。

至于 Remix，它是一个全栈 web 框架，旨在利用分布式系统和原生浏览器功能，同时抽象出后端服务器任务。与公共云环境兼容，包括亚马逊网络服务、谷歌云、Netlify、Vercel 和 Cloudflare Workers，Remix 的一个关键功能是预取——该框架可以在用户点击链接之前并行预取网页的元素，包括按钮和表单，以最小化页面加载。

在收购 Shopify 之前，Remix 已经从 OSS Capital 和天使投资人 Naval Ravikant、Ram Shriram 和 Sahil Lavingia 那里筹集了 300 万美元的种子资金。

在 Shopify 工程博客上的一篇帖子中，Shopify 的工程副总裁 Dion Almaer 说，购买 Remix 将通过改善氢气而使 Shopify 开发者和商家受益。

“Remix 将继续是一个独立的开源框架，”阿尔迈尔说。“Remix 将解决基于 Hydrogen 的开发人员在数据加载、路由和错误处理方面遇到的挑战……Shopify 将在许多有意义的项目中使用 Remix，随着时间的推移，你可以期待看到我们的开发人员平台更多地支持一流的 Remix。”

Remix 是 Shopify 自今年 5 月以 21 亿美元收购的履行技术提供商 Deliverr 以来的首次收购。今年早些时候，Shopify 收购了帮助品牌管理影响者营销活动的燕尾。在对 CMS developer Sanity 和 Klaviyo 营销自动化初创公司进行股权质押后，该公司最近还投资了 Shopify 上许多企业使用的音乐和视频应用软件 Single。

在经历了艰难的 Q2 之后，有迹象表明 Shopify 开始更好地度过经济衰退。该公司上周公布的第三季度亏损低于预期，导致股价飙升 17%。
