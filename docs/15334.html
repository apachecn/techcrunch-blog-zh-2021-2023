中国的 web3 社区正在聚集在腾讯进军去中心化领域的消息周围。周三，这家社交和游戏巨头在新加坡举行了首次全球 web3 峰会，在中国于 2021 年打击加密货币后，新加坡已成为中国区块链开发者的避难所。

很难忽略事件发生的时间。周一，香港宣布计划从 6 月开始允许散户投资者交易以太网和比特币，这增强了人们对中国可能软化其加密货币立场的信心。

正如我们推测的那样，中国可能不想错过下一波技术颠覆，也就是价值转移的数字化。虽然它对加密市场波动性的担忧是有根据的，正如从卢娜的垮台和 FTX 的崩溃的连锁反应中所看到的，但它至少想谨慎而又保持一定距离地进行实验。

事实上，彭博昨日报道称，香港成为加密中心的野心得到了中国官员的“暗中支持”。

高层的支持可能会鼓励腾讯举办一场高调的 web3 活动。该公司以前涉足过 web3，但它走得更小心。其国内项目在很大程度上仅限于政府批准的努力，包括与企业合作伙伴建立一个区块链财团，这与以太坊(Ethereum)等全球性的无许可项目形成了鲜明对比。

像它的同行一样，腾讯将其 NFT 平台标榜为“数字收藏品”市场，并在中国警告围绕不可替代代币的投机后禁止二级交易。其数字收藏平台何欢在推出仅一年后就关闭了。在中国以外，该公司试图通过金融投资在 web3 游戏中占有一席之地。例如，它支持澳大利亚的 web3 游戏公司 Immutable。

这一次，腾讯进军 web3 似乎更有信心，也更具体。在大会上，该公司的云计算部门腾讯云透露它已经与 web3 基础设施提供商 Ankr 签署了一份谅解备忘录，共同开发一套区块链 API 服务。这意味着腾讯正式加入了与 AWS 等云提供商的竞争，以吸引 web3 开发者。

腾讯云还与其他几家受欢迎的 web3 基础设施开发商合作，包括专注于速度和低交易费用的区块链Avalanche；Scroll，以太坊的第二层扩展解决方案；和隋，一个由前 Meta 员工创建的相对年轻的第 1 层区块链。对于香港和所有希望离家更近的中国 web3 开发者来说，这是一个激动人心的时刻。
