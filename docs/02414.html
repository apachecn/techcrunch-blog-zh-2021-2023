支付基础设施初创公司 M2P 金融科技公司获得 5600 万美元融资，估值超过 6 亿美元

原文：https://web.archive.org/web/https://techcrunch.com/2022/01/19/m2p-fintech-funding-insight-partners/

支付基础设施初创公司 M2P 金融科技(Alibaba fin tech)已经在印度建立了市场主导地位，在结束上一轮融资不到三个月的时间里，该公司在新一轮融资中筹集了 5600 万美元，该公司正在努力加深其在几个国际地理区域的足迹。

总部位于纽约的私募股权和风险投资公司 Insight Partners 领投了这家总部位于钦奈的初创公司的 C 轮融资。日本银行巨头 MUFG Innovation Partners 和现有投资者 Tiger Global 和 Better Capital 也参与了正在进行的一轮投资。

在新一轮融资中， M2P 的融资额达到 1.07 亿美元，在短短三个月内，这家初创公司的估值几乎翻了一番，达到约 6.05 亿美元。

M2P 金融科技公司的三位创始人来自威瑞森和 Visa，有两个核心产品。它帮助银行和其他金融机构获得监管许可，但需要技术帮助来推出银行服务。

“我们还帮助他们升级技术体系。M2P 金融科技公司的联合创始人兼首席执行官 Madhusudanan R 在接受 TechCrunch 采访时表示:“随着银行从资本支出为主的产品实施模式向 SaaS 的运营支出模式转变，一场运动正在进行中。”。

他说，数十家一级银行以及快速崛起的金融科技友好型银行是 M2P 金融科技的客户，这正帮助他们为新一代服务提供支持。

这家初创公司的第二个亮点是为数百家金融科技初创公司提供支付层，这些公司提供卡产品(如 Slice 和 Open)来管理他们的整个资金流。例如，如果一家金融科技初创公司想要推出新产品，他们可能希望与六家银行合作。这将需要建立自己的管道来利用银行的 API。他们可以转而与 M2P 金融科技公司合作，后者已经有了这些轨道。

Madhusudanan R(左)、Prabhu R 和 Muthukumar A 于 2014 年共同创立了 M2P 金融科技公司。图片来源: M2P 金融科技

Insight Partners 董事总经理 Nikhil Sachdev 在一份声明中表示:“M2P 是印度领先的金融科技基础设施即服务公司，他们整合了市场上最广泛的银行和金融科技关系，并构建了强大的产品，以 API 的形式提供，坚定地致力于满足客户的需求。”。

“M2P 出色的创始团队和快速的增长使该公司成为了一个明显的市场领导者，正在改变该地区的金融创新。随着 Madhu 及其团队不断扩大规模，我们很高兴能与他们合作。”

这家初创公司在近五年的时间里基本上实现了增长，如今在亚洲和北非的 20 多个市场开展业务。Madhusudanan 说，这家初创公司计划扩大在这些市场的存在，扩展到一些新的市场，并移植许多在印度受欢迎的产品功能。

“在非常微观的层面上，许多在印度发生的金融科技行动正在被复制到其他市场，”他说。

它还在关注一些收购机会，以推动相邻类别的无机增长。他说，这家初创公司去年收购了两家初创公司，目前正着眼于收购六家以上的公司。

M2P 金融科技目前处于一个独特的位置。它几乎没有面临竞争，并且是金融科技初创公司之间日益激烈的战斗的直接受益者，因为其中大多数都是它的客户。Marqeta 等公司的股票与 M2P 类似，该公司去年在美国上市，很快就获得了超过 160 亿美元的市值。但这些公司中的大多数——数量并不多——还没有扩展到亚洲国家。

Madhusudanan 说，他希望未来几年在印度和其他亚洲市场会出现更多的玩家。“我认为这个领域很容易容纳三到四家价值数十亿美元的公司，”他说。
