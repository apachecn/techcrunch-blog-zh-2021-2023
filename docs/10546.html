木瓜希望帮助欧洲最后一英里物流电气化

原文：https://web.archive.org/web/https://techcrunch.com/2022/09/27/papaya-wants-to-help-electrify-last-mile-logistics-in-europe/

内燃机仍然主宰着汽车的动力，但有迹象表明，至少在一些市场上，内燃机正逐渐被遗忘。像瑞典、T2、丹麦和英国都计划在 2020 年前禁止销售柴油车和汽油车，而澳大利亚、加州等市场也在朝着这个方向前进，尽管速度较慢。

这一过程的一部分必须包括让消费者和企业更容易过渡到电气化，例如通过扩大电动汽车(EV)充电站的接入，正如美国最近宣布的作为其 1 万亿美元基础设施法案的一部分。但是公司也需要帮助来获得和运营他们的电动汽车车队——这就是一家名为 Papaya 的新创公司开始发挥作用的地方。

早在 2 月份，Papaya 的软件就已试运行，旨在帮助车队运营商采购和管理电动或轻型电动汽车(lev)，解决联合创始人兼首席执行官 Santi Ureta 所说的通常“高度分散和不透明”的问题。为了让事情更上一层楼，这家总部位于伦敦的公司今天宣布，它已经从一系列机构和天使投资者那里筹集了 350 万美元。

就背景而言，现在已经不缺乏车辆管理系统，从 Automile 和 FleetCheck 到 Webfleet ，但 Papaya 希望通过其特定的行业专注于可能被最后一英里送货公司使用的小型电动汽车而脱颖而出。它是关于解决非常具体的痛点，减少碎片化，并作为每个人联系和交流的单一平台。

“没有人真正像我们这样连接市场的所有方面，并建立他们需要的工具来更好地管理关系，”Ureta 告诉 TechCrunch。

Ureta 和 Papaya 的首席技术官联合创始人 Renato Serra 都有在运输和物流对他们的底线至关重要的公司工作的经验，包括欧洲食品配送巨头 Deliveroo 和快速商务独角兽 Gopuff 。这个经历证明了木瓜的起源。

“我们直接意识到采购电动车队很难，有效管理更难，”Ureta 说。“用当前的软件工具管理一个混合动力车队是不可能在一个地方完成的。”

活动件

木瓜寻求解决的问题之一是多模式的复杂性——电动车队需要不同类型的车辆用于不同的用例。例如，电动货车可能更适合大规模的杂货交付，而货运自行车或电动自行车可能足以满足食品交付。对于每一种车辆，都有一大堆不同的供应商、维修公司和其他服务提供商来保持一切正常运转。

Papaya 本质上是车队运营商(例如 Gopuff 或 Deliveroo)和服务提供商之间的连接点，服务提供商可能包括车辆供应商(例如 Hop 或 Otto )、维护提供商(例如 Fettle 或 Cycledelik )、保险提供商(例如 Laka 或 Zego )，甚至是为电动汽车的存放和充电而设计的存储空间(例如

“每个供应商都有他们过时的系统——谷歌表单、电子表格、电子邮件或笨重的车队管理工具——车队需要与所有这些工具进行交互，以报告事故并保持其可用性，这使得它非常困难和低效，”Ureta 说。“Papaya 将所有这些不同的流程和工具集中到一个单一的操作系统中，让车队对其车辆的状态有全面的可见性、可问责性和透明度，并在同一个地方管理它们的所有关系。”

在最初的伪装下，Papaya 主要是为了实现现有电动汽车和电动汽车的管理，但其首要目标是帮助公司从传统的化石燃料燃烧汽车过渡到零排放的替代产品。这就是为什么该公司正准备推出其汽车市场，作为车队运营商采购电动汽车和电动车及所有相关服务的单一渠道。

“人们可以将它(市场)视为汽车供应商和服务提供商向车队展示其产品和服务的一种方式，在其运营的地理区域内，”Ureta 解释说，并补充说他预计该市场将于今年年底推出。“木瓜将使公司更容易采购和管理电动汽车——这将加速从内燃机车队向电动汽车车队的过渡。”

木瓜已经在五个市场上市，包括英国、西班牙、法国、德国和爱沙尼亚。到目前为止，在其短暂的生命周期中，该公司已经积累了令人印象深刻的客户名单，包括前面提到的 Gopuff(目前价值 150 亿美元)和包裹递送巨头 Evri 。

根据 Ureta 的说法，Gopuff 使用 Papaya 与车队中的所有车辆进行交互，跟踪可用性和成本，并在出现事故时进行管理。

“Gopuff 使用 Papaya 作为其主要的车辆管理系统——他们在平台上拥有所有的车辆，而他们的主要服务提供商则在另一边，”Ureta 说。“该平台由多个参与者使用，从乘客到枢纽运营商、车队经理和运营主管。”

除了采购和管理电动汽车之外，就像其他车辆管理系统一样，Papaya 也主要是生成数据和收集对车队在任何给定时间点发生的一切的见解。

降低排放

快速浏览一下数据就会发现，木瓜有所发现。欧盟委员会(EC)的目标是到 2050 年减少 90%的运输排放，而最后一英里物流目前占公司供应链排放的大约 5%，但随着电子商务的发展，这一数字可能会增加。事实上，世界经济论坛建议到 2030 年，前 100 个城市的运输车辆数量将增加 36%,交通排放也会随之增加。

简而言之，如果世界有希望实现崇高的气候目标，就需要解决排放问题。这就是木瓜增长计划的核心——该公司新的 350 万美元投资迎来了一系列支持者，包括 Giant Ventures、Seedcamp、20VC、FJ 实验室、Flexport、Cocoa、理查德·布兰森爵士的家族(具体来说:弗雷迪·安德鲁斯和霍利·布兰森，他们管理着家族基金)、Glovo 联合创始人奥斯卡·皮埃尔和前 TechCrunch 记者史蒂夫·奥黑尔。

该公司表示，计划利用其现金注入“建立欧洲最大的电动汽车生态系统，并使欧洲车队脱碳。”
