苹果用 widgets | TechCrunch 修改了 iOS 16 的锁屏

原文：https://web.archive.org/web/https://techcrunch.com/2022/06/06/apple-revamps-the-lock-screen-in-ios-16-with-interactive-widgets-and-other-personalization-features/

苹果在 iOS 16 中对 iPhone 的锁屏进行了大改造。在苹果的全球开发者大会上，该公司介绍了其更加个性化的锁屏计划，用户可以通过内置的编辑器进行定制，该编辑器提供了小工具、壁纸库和丰富多彩的主题。它还引入了一个名为 Live Activities 的新功能，允许开发者构建包含实时信息的实时更新锁屏小工具。

以前，小工具在锁定屏幕左侧的“今日视图”中可用。向右滑动，iPhone 用户可以在一个可定制的视图中跟踪新闻、天气、日历约会、股票等。现在，小工具将直接出现在锁屏界面上。

在新的锁定屏幕中，用户将能够通过一个新的编辑器更新这个关键的 iOS 不动产，当点击时，你可以进入一个模式，你可以滑动以尝试不同的主题和风格。这些样式可以自动更改颜色、滤镜、背景和字体，所有这些都是为了相互协调。使用这一功能，用户可以尝试不同的字体和选择不同的颜色。

苹果表示，在锁屏上使用的照片也将看到一种新的多层效果，照片的主题被设置在锁屏上的时间前面，创造出一种深度感。用户还可以用不同的风格和颜色选择来改变日期和时间本身的外观。

图片来源:苹果

此外，用户可以通过从可用的小部件库中选择新的小部件来使他们的锁定屏幕更有帮助。

这些功能允许用户快速查看信息，如即将到来的日历事件、天气、新闻、电池健康状况、警报等。一个名为“照片随机播放”的选项可以让您从一组照片中选择一天中要随机播放的照片。苹果公司还提供从用户个人图库中智能筛选的推荐照片。

从一个新的锁屏画廊，用户还可以浏览苹果的精选收藏，他们可以从预先配置的主题中挑选，如骄傲和团结主题。另一个选择是让用户预览天气状况，用实时天气壁纸显示闪电或暴雨等情况。天文学壁纸提供了地球、月球和太阳系的景观。苹果表示，用户还可以用自己喜欢的表情符号或颜色组合来创建锁屏。

用户也可以配置许多不同的锁屏，他们可以保存这些锁屏，然后在它们之间切换。

与此同时，开发人员可以通过 WidgetKit 构建锁定屏幕，这允许他们为 iOS 16 体验的这一部分构建可定制的简略信息。

随着这些变化，苹果还推出了一种叫做 Live Activities 的东西，这种东西可以帮助你从锁屏上实时了解正在发生的事情。开发人员将能够通过 Live Activities API 访问它，该 API 用于创建这些简略的体验。

苹果建议如何使用这些直播活动——例如，掌握体育比赛比分，或者跟踪优步骑行、锻炼等。其中一个将允许播放音乐的用户访问播放控件，这些控件可以扩展到全屏视图，使这个小工具成为一个更具交互性的体验。苹果解释说，这些元素将与其他锁屏元素共存。

此外，苹果新的对焦模式将更新为可通过锁屏进行定制。也就是说，用户可以为给定的聚焦模式定制锁定屏幕。这意味着，在工作时，你可以让小工具关注股票或工作日历等事情，而在私人时间，你可能会用不同的小工具和家庭照片更新屏幕。壁纸也可以随着不同的对焦模式而改变。

通知现在也将从锁定屏幕的底部滚动出来，而不是杂乱无章，以保持主要部件和功能清晰可见。

随着更新的 iOS 16 操作系统在秋季到来，新的锁屏将会发布。
