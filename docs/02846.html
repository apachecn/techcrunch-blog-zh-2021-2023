Docker 在重组两年后以超过 5000 万美元的收入卷土重来 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/02/01/docker-makes-comeback-reaching-over-50m-in-arr/amp/

图片来源:罗恩·米勒/TechCrunch

对于 2013 年成立的开源集装箱公司 Docker 来说，这无疑是动荡的几年，但它似乎已经再次找到了财务立足点。今天，该公司宣布，在过去的一个财年中，年度经常性收入(ARR)增长了 4 倍，超过 5000 万美元。

对于一家从 2019 年开始面临巨大动荡的公司来说，这是一次相当大的回归，当时的首席执行官史蒂夫·辛格下台，并由罗布·比尔登短暂接替。此后不久，它出售了其主要收入来源企业业务，最终将长期高管斯科特·约翰斯顿提升为首席执行官。

当时，它还接受了新的融资几乎回到了起点，实际上是作为 A 轮公司接受投资。它同时实施了一个新的以开发人员为中心的战略，同时将员工人数从 400 人减少到 60 人。几个月后，第一波疫情飓风来袭。约翰斯顿表示，这不是一个轻松的时期，他必须带领公司度过这段不稳定时期。

“2019 年 11 月是一个充满风险和不确定性的时期，但我们相信市场的顺风，我们相信开发者对我们产品的热爱，我们可以团结成一个团队，专注于开发者，提供伟大的产品，建立一个合法的业务，”约翰斯顿告诉我。

尽管存在这种不确定性，Docker 还是有一些优势的。它在开发人员中有着广泛的品牌认知度，是容器化的同义词，容器化是一种在云中将软件打包并交付为单独服务的方式，而不是一个单一的应用程序。

它还拥有一系列开源产品，可以作为最终销售活动的漏斗顶端，目标是将一些免费产品的用户转变为付费客户。从该公司同期的 ARR 增长来看，这种情况似乎在去年越来越频繁。

重组初期的目标是抓住围绕品牌的开发者势头，向他们提供免费的开源产品，然后随着时间的推移扩展到一定比例的付费产品。这与他们在销售 Docker Enterprise 时采用的方法非常不同，他们通常向 IT 部门销售，而不是开发人员或他们的经理。

当经理们开始购买相关的商业工具时，这种以产品为导向的增长方法从商业角度来看是有效的。“因此，当开发人员使用免费产品有很好的体验时，随着他们和他们的组织扩大对产品的使用，就会有经理们重视的、他们愿意为之付费的功能。”

他补充道，“我们在博客文章中描述的所显示的财务表现，完全是由那些认识到这些生产力优势的大型组织推动的……并且愿意为管理安全工具付费，以实现组织范围内的采用。”

Docker 成立于 2013 年，2019 年重组。在将企业业务出售给米兰蒂斯时，该公司接受了来自基准资本和洞察伙伴公司(Benchmark Capital and Insight Partners)的 3500 万美元首轮投资。去年三月，他们还获得了2300 万美元的 B 轮。

在重组的时候，我写道:“这种方法是否可行还不清楚，但约翰斯顿认为这是前进的方向。时间会证明这个策略是否成功。”

随着 ARR 超过 5000 万美元，陪审团可能仍然不在，但它肯定会朝着正确的方向发展，我估计大多数投资者都会满意。现在，他们必须继续保持这一势头。
