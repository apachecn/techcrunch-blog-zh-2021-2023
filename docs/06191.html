code 公证人扩展其软件供应链安全服务

原文：https://web.archive.org/web/https://techcrunch.com/2022/05/06/codenotary-launches-a-full-stack-monitoring-solution-for-kubernetes-and-vmware-environments/

code 公证人为 Kubernetes 和 VMware 环境推出全栈监控解决方案

不变的 immudb 开源数据库和基于该数据库的软件供应链安全服务背后的公司code 公证人今天宣布，它还将在此基础上为运营和 DevOps 团队添加全栈 Kubernetes 和 VMware vSphere 监控服务。基本上，虽然该公司以前的产品——code 公证人云——专注于保护软件供应链，但现在随着不可变度量&日志的推出，它正在将其服务扩展到监控生产系统。

鉴于 core 公证人的核心是一个新颖的数据库系统，该公司有必要扩展到其他领域，利用这一点带来额外的价值。

图片致谢:公证人

该公司辩称，其新服务允许用户监控从应用程序到虚拟机或 Kubernetes 容器以及网络和存储基础设施的整个堆栈。该公司认为，其解决方案将能够为企业提供单一工具来监控 Kubernetes 部署和虚拟化基础架构，这些工具由 VMware vSphere 等工具管理，他们通常位于这些工具之上，而不必求助于使用多种工具。

code 公证人联合创始人兼首席技术官 Dennis Zimmer 表示:“我们的技术提供了最快的部署价值比。“借助 code 公证人不变的指标和日志，客户可以从他们的部署中看到近乎即时的结果，无论是在云中还是在内部。此外，我们提供了唯一一种存储日志和配置更改的技术，这些更改完全可验证和可审计，让法规和许可证合规性变得无忧。”

显然，市场上还有其他监控解决方案，但 comes 公证人可能会认为，其不可变的数据库可以让它在竞争中占据优势，特别是在审计和许可证合规性方面。

今年 1 月，code 公证人宣布已经筹集了 1250 万美元的 B 轮融资，使其迄今为止的融资总额达到 1800 万美元。
