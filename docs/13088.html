自主电动卡车公司 Einride 通过股权和债务融资 5 亿美元

今天，Einride 宣布了一笔 5 亿美元的融资，以将其自动驾驶和电动卡车扩展到全球新市场。

现金注入构成了基于股权的 2 亿美元 C 轮部分，支持者包括 Northzone，EQT Ventures，淡马锡，瑞典养老基金 AMF，Polar Structure 和 Norrsken VC。此外，这家瑞典公司表示，它已经获得了巴克莱欧洲(Barclays Europe)牵头的 3 亿美元债务融资，这笔资金将用于支持其车队，因为它将进一步扩展到欧洲和北美的新市场和现有市场。

Einride 于 2016 年在斯德哥尔摩成立，最初开发了一种无驾驶室自动驾驶货车，如果需要，它也可以由人类操作员远程控制。

Einride in action 图片来源: Einride

这些吊舱已经进行了商业试验，但是监管障碍导致 Einride 采取了一种双管齐下的方法，其中还包括人类驾驶的电动卡车作为迈向自主的中间步骤。这些车辆已经在瑞典和美国的船运公司和运输公司上市，由 Saga 平台驱动，帮助客户运营和优化他们的车队。

膨胀驱动

在过去的几个月里，Einride 已经开始了某种扩张运动(可以这么说)，9 月在德国推出接着是上周在比利时、荷兰和卢森堡推出针对挪威的计划已经就绪。为伊莱克斯和通用电气电器等大公司提供服务，如果考虑到开发数百辆卡车和相关软件的相关成本，后者是一项巨大的资本密集型努力，Einride 已与后者合作部署完全自主的吊舱，作为美国试点的一部分。

这就是为什么 Einride 现在必须筹集这么大一笔资金，自成立以来已经筹集了大约 1.5 亿美元(包括去年 1.1 亿美元的 B 轮融资)。

虽然许多公司今年以明显较低的估值筹集了后续融资，但我们不可能知道经济衰退对 Einride 有何影响，因为它从未在之前的任何一轮融资中披露其估值——它也不会在最近一轮融资中改变策略。

然而，尽管受到全球疫情和其他宏观经济因素的影响，货运卡车行业几乎没有放缓的迹象，报告显示该行业预计将在未来五年内从 2020 年的 2.1 万亿美元增长到近 3 万亿美元。当然，今年有许多迹象表明，投资者仍然非常看好任何有助于货运和物流行业的技术。

Einride 创始人兼首席执行官罗伯特·法尔克在一份声明中说:“现在不仅要采取行动开发技术，还要加快技术的实施，创造更清洁、更安全、更有效的货物运输方式。”

Einride 表示，其 3 亿美元债务融资的前一半将于 1 月份开始陆续到位，而其股权部分包括今年早些时候筹集的 9，000 万美元可转换票据。
