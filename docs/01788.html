社交媒体和约会应用是时候面对音乐，一劳永逸地遏制他们平台上的欺诈、欺骗和虚假信息了。

起初，社交媒体和约会应用代表着互联网的一个小角落，只有少数用户。今天，脸书和推特是如此之大，以至于它们影响选举，决定疫苗运动的成败，并影响市场。

像 Tinder 和 Bumble 这样的约会应用紧随其后，数以百万计的人希望通过它们的服务来结识他们的“永远”伴侣。

但是乐趣和游戏现在结束了。你选择了利润而不是信任和安全。您为身份盗窃和在线欺诈创造了一个通道。

今天，我们都有一个朋友在 Bumble 或 Tinder 上被“猫钓”了；我们都有家庭成员成为网上推特和脸书骗局的受害者。每天，我们都会听到新的案例，其中恶意行为者窃取身份(或创建虚假的新身份)来实施欺诈，传播错误信息以获取政治和商业利益，或宣传仇恨言论。

在大多数行业中，使用假身份的用户只会影响业务。但是当约会和社交平台上的信任被打破时，它会伤害用户和整个社会。对一个人的经济、心理——有时是身体——影响是真实的。

那么，谁来负责阻止或打击这种欺诈行为的增长呢？显然，不是平台本身。尽管有些人声称正在采取行动。

2020 年第四季度，脸书拿下13 亿假账号。够了吗？差远了。事实是，如今的社交平台和约会应用在防止欺诈方面做得很少。虽然基本的人工智能和人类版主有所帮助，但他们的用户数量远远超过了他们。

脸书称有 35000 人检查内容；这是一个军团，但它计算出大约每 82，000 个帐户有一个版主。随着坏人变得越来越老练，他们使用深度假货和合成诈骗等不断发展的犯罪技术，他们的规模继续扩大。甚至精明的网络用户也成为这些骗局的牺牲品。

社交和约会平台因在解决这个问题上行动迟缓而受到抨击，但能做些什么呢？

钓猫是一件严肃的事情

不难想象这样一个场景:你在网上遇到一个人，开始了一段对话。这个人说正确的话，问正确的问题。这种关系开始变得“真实”，你开始感受到亲情。不知不觉，它就升级了；你所有的守卫都倒下了，你变得不受红旗的影响。你甚至称之为爱。

你和你的另一半计划最终见面。他们声称他们没有旅行的钱。你信任地、慈爱地把钱送去，只是为了这个人不久后死去。

虽然有些钓鱼事件会自行解决，造成的伤害极小，但其他事件(如上例)可能会导致金融勒索和犯罪活动。据美国联邦贸易委员会报道，2020 年，爱情诈骗造成的损失达到创纪录的 3.04 亿美元。

然而，在这个报道不足的领域，实际损失可能会成倍增加，如果算上“灰色地带”和网上乞讨，损失会更大。然而，大多数约会应用程序都没有提供验证身份的方法。一些流行的应用程序——像 Tinder 这样的——让身份验证变得自愿；其他人，嗯，什么也不提供。谁想给新用户添麻烦呢？

但是自愿核查只是触及了表面。这些企业必须采取更多措施来阻止匿名和伪造身份的进入。鉴于它们对社会和消费者造成的损害，我们作为一个社会必须要求它们站出来。

在社交网络上，身份验证可能是一把双刃剑

浪漫骗局并不仅限于约会应用；大约三分之一是从社交媒体开始的。但在社交网络上验证身份还有很多其他原因。消费者可能想知道他们接触的是真正的奥普拉·温弗瑞或爱莉安娜·格兰德，还是一些模仿账户；温弗瑞和格兰德可能也希望这种区别显而易见。

更严重的是，社交网络通过验证身份来控制在线滥用者的压力越来越大。在英国，主要由于英国真人秀明星凯蒂·普莱斯的努力，#TrackaTroll 运动越来越流行。近 700，000 人在她的请愿书上签名，为哈维法游说，哈维法是以她的残疾儿子命名的，他是匿名网络虐待者的主要目标。

然而，许多人强烈反对在社交网络上要求身份验证。通常，他们指出，在专制政权搜寻和伤害政治对手的国家，要求核实可能会危及家暴幸存者和持不同政见者。此外，身份验证不会阻止许多传播政治或疫苗虚假信息的人，因为他们希望被识别出来，以建立自己的受众和个人品牌。

如今，脸书和 Twitter 提供了一个“验证”审查程序，给真实的账户打上蓝色勾号。但这远非万无一失。Twitter 最近暂停了它的“验证”程序，因为它错误地验证了一些虚假账户。

脸书做得更多。该社交网络长期以来一直有条件地实施身份验证，例如，如果用户被锁定在他们的账户之外。他们还将身份要求基于发布的内容，其中某些行为、词语和图像会触发海报的某个区块，等待验证和人工审查。

身份军备竞赛

当坏演员在约会应用程序和社交媒体上创建假身份来欺诈和伤害他人时，这损害了公众的信任，无疑会影响这些平台的收入。社交媒体平台每天都在努力协调其最大化使用和保护用户隐私的业务目标，否则将面临越来越多的监管和消费者信任的丧失。

保护身份不被窃贼和黑客滥用是至关重要的。想象一下，一个假冒的 Twitter 或脸书账户声称是你，散布仇恨言论。没有办法证明你的参与，你可能会丢掉工作或者更糟。

这些平台会做出什么选择来保护他们的用户和他们自己的品牌？他们的决策集中在政策和收入保护上，而不是技术上。平衡建立信任措施与隐私问题以及他们对收入的需求，是他们必须解决的重大战略困境。不管怎样，为用户创造一个安全的空间是他们的责任。

社交媒体和约会平台必须在保护用户免受网络欺诈和不良行为者侵害方面承担更大的责任。
