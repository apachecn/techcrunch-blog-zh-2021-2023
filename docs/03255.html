Picsart 用新的 API 为开发者带来了创造性的工具

该公司周三宣布，数字创作平台 Picsart 为开发者推出了一个名为 Picsart 的新 API 程序。Picsart 提供了一系列面向消费者和专业人士的数字创作和编辑工具，让照片和视频编辑变得更加有趣和平易近人。该公司表示，随着世界各地的消费者向企业寻求定制的视觉产品和体验，Picsart 正在通过其新的 API 新产品，使其技术可用于各种规模的企业，以满足他们的需求。

有了这些新的 API，企业现在可以直接在自己的平台上实现 Picsart 的人工智能创意工具。基于人工智能的 API 包括一个移除背景功能，一个在不损失质量的情况下放大和增强内容的高档功能，以及一个转移任何源图像的外观以提升其风格的风格转移功能。图像处理 API 包括旨在使照片突出的滤镜和一个调整功能，让您可以调整亮度和对比度等设置。它还包括上传功能，使图像处理速度更快。

Picsart for Developers 将与八家合作伙伴一起推出，包括数字广告、网站建设和定制营销方面的创意公司。这八个合作伙伴是 Photobook、TPS Engage、Clos、Amaze、Make Your Move、The Flat Lay、Etch4U 和 Smiiley。Picsart 表示，Photobook 和 Smiiley 的整合目前正在进行，其余的将在未来几个月内推出。该公司计划在未来几个月增加更多受邀才能加入的合作伙伴。

“我们的愿景是为各地的创作者提供支持，Picsart 凭借我们创新的人工智能、照片和视频编辑能力引领市场。通过 API 向开发者提供我们的技术是一种令人兴奋的方式，我们可以支持蓬勃发展的创作者经济，”Picsart 的创始人兼首席执行官 Hovhannes Avoyan 在一份声明中说。“数字创作的速度，尤其是 Z 世代对个性化内容和脱颖而出的要求，意味着各地的公司都必须适应这些现实，并提供更深入的创作体验。我们的 API 让他们能够做到这一点。”

图片来源: Picsart

Picsart 于 2018 年通过收购 EFE kt进入视频市场，近年来使用量激增，主要是因为该平台已被使用视频的社交媒体创作者和电子商务商店采用。2021 年 12 月，Picsart 表示，其应用程序中编辑的视频超过 1.8 亿，同比增长 70%。

今天的发布正值 Picsart 最近收购了研发公司 DeepCraft。这笔交易包括现金和股票，价格在七位数范围内，但具体条款没有披露。Picsart 曾表示，DeepCraft 的人工智能技术人才及其在计算机视觉和机器学习方面的突破将增强 Picsart 自己的人工智能技术，并帮助该公司更好地支持其服务上最近的视频创作增长。该团队还将帮助补充 Picsart 的人工智能研究和开发部门 PAIR (Picsart 人工智能研究)的额外高级资源。

Picsart 在去年 8 月宣布获得由软银愿景基金 2 牵头的 1.3 亿美元融资后，成为独角兽。资本注入使该公司的估值突破了 10 亿美元大关。皮萨特·首席运营官·塔米·南告诉 TechCrunch，该公司拥有数百万用户，该平台还有很大的发展空间。
