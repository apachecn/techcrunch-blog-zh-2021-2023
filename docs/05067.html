在科技界，批评创业加速器和演示日的价值已经有几十年的历史了。这些项目承诺，餐巾阶段的创始人会提供一切帮助，从找到他们的联合创始人，到产品符合市场需求，到筹集关键的第一笔资金。在 Y Combinator、Techstars 和 500 Global 等全球项目的引领下，创业加速器诞生了比特币基地和 Stripe 等数十亿美元的公司，并成为活化能承诺的代名词。

然而，每隔几个月，企业家们都会问同样的问题:宝贵的股权值得进入一个网络吗？这个项目的真正价值仅仅是一种受人尊敬的认可吗？演示日过时了吗？加速器内的创始人最好的结果只是新一轮融资吗？YC 的批量是否太大而无法脱颖而出？

我们一直试图重新发明创业加速器，这本身就告诉我，这个机构仍然是相关的，即使不完美。毕竟，提出问题是改变做事方式的第一步。

一月份，我写了一篇关于创业加速器如何看待增值服务的文章。几天后，Y Combinator 宣布将支票金额从之前的 12.5 万美元增加到 50 万美元。随着 Y Combinator 年冬季演示日在下周举行，我们将看到第一批受到这些变化影响的人——YC 变得更加偏远，更加国际化，对其想要产生的影响更加雄心勃勃。

今年，正如每个人都会看到的那样，我们正在改变我们报道演示日的方式，以更好地反映我们认为加速器最重要的部分:一种看到一大批初创公司如何定向思考某个子行业最大问题的方式。感觉上，演示日已经完全脱离了传统的向投资者展示和推销的方式，更多的是提供一个创业公司的快照，以及创业初期的成长和个性。

下周会有更多内容，但在本新闻简报的剩余部分，我们将讨论金融科技的异常世界、Instacart 折扣和加密货币非营利重叠。你可以一如既往地支持我，把这篇时事通讯转发给朋友，在 Twitter 上关注我，或者订阅我的个人博客。

本周交易

Ramp 证实，它已经再一次筹集了的资金，但这一次的估值为 81 亿美元。鉴于不到一年前，该公司达到独角兽地位不到一年后，十玉米的估值即将到来。天啊。

这就是为什么它很重要: Ramp，以及更广泛的金融科技，感觉像是我们在过去一个季度报道的市场动荡中的一个局外人。金融服务行业是否受到保护，不会受到更大范围的风险撤出或估值调整的影响？本周，Alex 和 Mary Ann 谈到了一个关键问题:这是一个金融科技世界，我们只是生活在其中。

Instacart 迄今为止最大的折扣

Instacart 将其估值削减了近 40%，这为我们提供了另一个数据点，表明疫情时代的许多成功故事正在经历更大的市场调整。

这很重要:正如 Alex Wilhelm 指出的那样，另一家送餐公司 DoorDash 的价格销售比已经从去年的高点下降，而优步正希望扩大其送餐服务。Instacart 仍然是私有的，在股票市场首次亮相之前削减其账面价值可能会使它避免一个颠簸的反应。

为什么 web3 的富人捐赠密码而不是现金

加密记者安妮塔·拉马斯瓦米调查了 web3 的富人以加密方式而非现金捐赠的趋势。这篇报道特别探讨了本月支持乌克兰的秘密捐款热潮如何引发社区对通过硬币支持事业的更广泛兴趣。

以下是它的重要性:除了捐赠中的文化重叠和 crypto 对支持事业的更民主方式的看法，还有一个技术上的好处。Change 创始人 Sonia Nigam 正在与 Amar Shah 一起构建一个捐赠 API，她解释了传统慈善事业和创造者效用之间的区别:

智能合同技术允许影响存在于产品本身，然后永久地给予……我们将看到 NFT 系列上线，他们将设定一个目标；[例如]在所有二次销售中，2%用于终生应对气候变化。现在，在每一次转售中，创作者的最初意图从未丢失，这是让他们真正兴奋的地方。对于非营利组织来说，打开捐赠的循环渠道永远是首要目标。
