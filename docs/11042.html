OpenSea 在 Avalanche | TechCrunch 上推出 NFT 市场

原文：https://web.archive.org/web/https://techcrunch.com/2022/10/11/opensea-launches-nft-marketplace-on-avalanche/amp/

图片来源:Jakub Porzycki/Nur photo/Getty Images

随着 NFT 的活动继续放缓，该领域的一些主要参与者已经联手扩大其影响力。两家公司都独家告诉 TechCrunch，最近这样做的是 OpenSea，它已经在第一层区块链雪崩上本地启动。

“web3 的未来是多链的；OpenSea 的产品副总裁 Shiva Rajaraman 在一封电子邮件中对 TechCrunch 说:“我们的目标一直是提供最佳选择，将人们与他们喜欢的链上的项目和创作者联系起来。

世界上最大的 NFT 市场 OpenSea 将在区块链与现有平台一起运营，如 NFTrade 和其他专注于雪崩的 NFT 市场，包括 Joepegs 和 Kalao。该公司表示，有了 Avalanche，OpenSea 用户将能够以较低的交易费用在不到一秒的时间内完成 NFT 交易。

“我们倾听我们社区的声音，我们社区的一部分人真的希望有 OpenSea，”艾娃实验室的主席吴炯告诉 TechCrunch。“我们有一个崭露头角的、充满活力的创作者和 NFT 社区，他们希望 OpenSea 成为一个选择。”

所有基于雪崩的 NFT 收藏将在 OpenSea 上展示，随着时间的推移，还会推出更多。

此外，Avalanche 正在与一些 Web 2.0 企业合作，如商业和消费品牌以及运动队，吴说。吴说，这些公司希望 OpenSea 成为 Avalanche 的一个选择。

“他们想知道有一个可扩展的链，因为[Web 2.0]比 web3 大得多。他们想要一个能够处理来自用户的大量活动的连锁店，”吴说。“这些品牌与 OpenSea 合作要舒服得多。”

到目前为止，在 OpenSea 上已经有超过 200 亿美元的交易量通过 8000 万个 NFT 完成。今年 1 月，该平台达到了 133 亿美元的估值，当时这个细分市场正人气爆棚。cryptosslam数据显示，Avalanche 销售额约为 4.046 亿美元，是 NFT 销量排名第七的区块链。

根据 CryptoSlam 的数据，尽管 NFT 市场在最近几个月有所放缓，但在过去 30 天里，Avalanche 在 NFT 的销量增长了 38.52%，达到 210 万美元。

“整个市场都在下跌，而且跌幅很大，”吴说。“但如果你打破 NFT 市场………,许多绒毛就不见了。除了收藏领域之外的其他类别，一个新的实用类别和粉丝参与度开始上升。”
