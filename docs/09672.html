YouTube 正在为精选的创作者测试一种新的方式来推广他们的内容

YouTube 正在为创作者试验一种新的方式来推广他们的频道和内容，而不必通过谷歌广告管理器。谷歌旗下的公司正在 YouTube Studio 中测试一个新的“推广”标签，这将使创作者更容易购买广告。YouTube 在一个支持页面上指出，新标签正在有限数量的桌面创建者中进行测试。

该公司表示:“我们听说创作者希望有更多的工具来帮助扩大他们的渠道，触及更广泛的受众，所以我们正在 Studio 中测试一种更简单的端到端工作流程，而不是通过谷歌广告。”“如果你在这个实验中，你会在 Studio 的内容页面中看到一个新的‘推广’标签。”

新增加的内容让创作者更容易为他们的内容进行快速推广，也给他们一种方式来提高他们在平台上的存在。新功能也是 YouTube 带来更多广告收入的一种方式。

与任何其他测试功能一样，YouTube 何时或是否计划更广泛地推出推广标签还不得而知。

关于创作者的话题，YouTube 是也让创作者能够在主应用程序上查看关键指标，使其与 YouTube Studio 应用程序保持一致。在这一变化之前，创作者只能从 Studio 应用程序中查看有关视频、短片和直播的指标。现在，他们可以从主要的 iOS 和 Android YouTube 应用程序中查看这些指标。YouTube 表示，新增内容的目的是帮助创作者了解一种内容类型在他们的频道上如何优于其他类型。

该平台还推出了一种管理直播的新方法。创作者现在能够在实时控制室和 YouTube Studio 中安排、管理和编辑实时和即将到来的移动流。到目前为止，创作者只能在 YouTube 移动应用程序中主持实时聊天，而不能编辑或管理任何移动直播流。YouTube 表示，这一变化将在本周末对所有符合条件的创作者开放。
