亚马逊向开发者开放其低带宽、长距离人行道网络

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/28/amazon-opens-its-low-bandwidth-long-range-sidewalk-network-to-developers/amp/

图片来源:TechCrunch/Frederic Lardinois

早在 2019 年，亚马逊就宣布了 Sidewalk ，它的低带宽、远程无线网络使用 900 MHz 频谱来连接物联网(IoT)设备。它通过将这些设备连接到亚马逊自己的 Echo 和 Ring 设备来实现这一点，这些设备共享其所有者的一小部分带宽。理想情况下，这意味着 Sidewalk 将能够连接蓝牙或 Wi-Fi 信号覆盖不到的设备。然而，直到现在，只有少数开发人员能够为网络开发应用程序。但是今天这种情况正在改变。该公司现在正在推出软件和硬件开发套件，以及一个易于使用的测试套件，以测试您所在社区的可用人行道连接。

使用新的测试套件，开发人员将能够在地图上检查他们的本地信号强度，以便在开始开发产品之前更好地了解他们的设备是否能够连接到网络。

亚马逊给我发了一个戒指品牌的设备来试试。开始使用它只需要启动它(它有一个电池供移动使用)并登录亚马逊基于网络的人行道覆盖服务。从那里，您可以快速查看启用 GPS 的测试套件的所有原始数据，并访问亚马逊的美国范围覆盖地图。

为了确保用户隐私，覆盖地图仅显示 900 米乘 900 米平方区域内的覆盖范围。您将能够看到您自己设备的确切位置，但无法看到报告到覆盖地图中的其他设备。

亚马逊设备与服务高级副总裁戴夫·林普(Dave Limp)表示:“我们已经迅速建成了一个长距离、低带宽的网络，现在覆盖了超过 90%的美国人口，这是对开发者的公开邀请，让他们对其进行测试。”“许多类型的互联设备都受到 Wi-Fi 覆盖范围和蜂窝技术成本的限制，这阻碍了环境传感器、检漏仪和智能锁等设备的互联能力。 Sidewalk 旨在提供一种安全、低成本的方式来发明和连接一系列全新的设备，我们迫不及待地想看看开发人员都在做些什么。”

人行道覆盖图。图片来源:亚马逊

对于软件开发套件，亚马逊与北欧半导体、硅实验室、德州仪器和模块供应商 Quectel 合作。这些公司还将提供硬件开发套件，Quectel 推出了 Sidewalk 连接模块，允许开发人员快速将支持 Sidewalk 的设备投入生产。亚马逊也在为 iOS 和 Android 推出移动 SDK，以帮助开发者将 Sidewalk 构建到他们的移动应用中。该公司还推出了一款应用程序，帮助开发者在现场调试和排除设备故障。

值得注意的是，Sidewalk 还与 AWS 的物联网核心服务进行了原生集成，允许企业连接和管理他们的物联网设备车队。“AWS 物联网核心和亚马逊 Sidewalk 的整合标志着开发者、制造商和客户的一个重要里程碑，简化了基于亚马逊 Sidewalk 的物联网解决方案的设计、连接和部署，”亚马逊网络服务的物联网副总裁 Yasser Alsaied 说。“现在，借助亚马逊 Sidewalk 的 AWS 物联网核心，开发人员可以访问 200 多种 AWS 服务，在高度可靠、安全和免费连接的无线网络基础上构建可扩展的解决方案。”
