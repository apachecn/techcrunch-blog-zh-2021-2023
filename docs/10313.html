研究称，YouTube 上的“不喜欢”和“不感兴趣”选项对你的推荐没有太大帮助

原文：https://web.archive.org/web/https://techcrunch.com/2022/09/20/youtubes-dislike-and-not-interested-options-dont-do-much-for-your-recommendations-study-says/

众所周知，的观众和的创作者都对 YouTube 的推荐算法感到困惑。现在，Mozilla 的一项新研究表明，当用户使用“不喜欢”和“不感兴趣”等选项来阻止 YouTube 推荐类似视频时，他们的推荐不会有太大变化。

该组织的研究观察到，YouTube 为他们提供了与他们拒绝的视频相似的视频——尽管人们使用反馈工具或改变他们的设置。当谈到防止不良推荐的工具时，点击“不感兴趣”和“不喜欢”大多是无效的，因为它只能分别防止 11%和 12%的不良推荐。像“不推荐频道”和“从观看历史中删除”这样的方法在有效性方面排名更高，分别减少了 43%和 29%的不良推荐。总的来说，参与研究的用户对 YouTube 将不良推荐从他们的订阅源中剔除的能力感到不满。

图片来源: Mozilla

Mozilla 的研究从其自有的 RegretReporter 浏览器扩展的 22，722 名用户那里收集了数据，并分析了超过 5.67 亿个视频。该浏览器扩展允许用户报告“令人遗憾的”视频，并更好地控制他们的推荐。它还对 2，757 名 RegretReporter 用户进行了详细的调查，以更好地了解他们的反馈。

报告指出，78.3%的参与者使用 YouTube 自己的反馈按钮，改变设置或避免某些视频，以“教导”算法建议更好的东西。在采取任何措施来更好地控制 YouTube 推荐的人中，39.3%的人说这些措施不起作用。

“什么都没变。有时我会报告一些误导性的和垃圾邮件，第二天它又回来了。感觉就好像我对他们的建议提供的负面反馈越多，废话就越多。一位调查者说:“即使你屏蔽了某些信息来源，它们最终还是会回来。”。

事实上，23%努力改变 YouTube 建议的人给出了不同的回答。他们列举了一些影响，比如不想要的视频重新出现在 feed 中，或者花费大量持续的时间和精力来积极地改变推荐。

“是的，他们确实改变了，但以一种糟糕的方式。在某种程度上，我觉得主动尝试改变算法的行为是一种惩罚。另一位研究参与者说:“在某些方面，互动越少，提供的推荐数据就越少。”

Mozilla 得出结论，即使是 YouTube 最有效的屏蔽不良推荐的工具也不足以改变用户的订阅。它表示，该公司“对听取用户真正想要的东西并不感兴趣，更喜欢依赖不透明的方法来推动参与，而不顾用户的最佳利益。”

该组织建议 YouTube 设计易于理解的用户控件，并为研究人员提供粒度数据访问，以更好地了解视频共享网站的推荐引擎。

“我们让观众可以控制他们的推荐，包括将来阻止向他们推荐某个视频或频道的能力。重要的是，我们的控件不会过滤掉整个主题或观点，因为这可能会对观众产生负面影响，就像创建回音室一样。我们欢迎我们平台上的学术研究，这就是为什么我们最近通过我们的 YouTube 研究员计划扩展了数据 API 访问。YouTube 发言人 Elena 在回应这项研究时说:“Mozilla 的报告没有考虑到我们的系统实际上是如何工作的，因此我们很难收集到许多见解。”。

Mozilla 去年进行了另一项基于 YouTube 的研究，指出该服务的算法表明 71%的视频用户“后悔”观看，其中包括错误信息和垃圾信息的剪辑。这项研究公开几个月后，YouTube 给写了一篇博文，为其建立当前推荐系统并过滤掉“低质量”内容的决定辩护。

在多年依赖算法向用户推荐更多内容后，包括抖音、T2、推特和 T4 在内的社交网络正试图为用户提供更多选择来完善他们的订阅。

世界各地的立法者也在密切关注不同社交网络的不透明推荐引擎如何影响用户。欧盟在 4 月份通过了一项数字服务法案，以增加平台的算法问责，而美国正在考虑一项两党合作的过滤泡沫透明法案来解决类似的问题。
