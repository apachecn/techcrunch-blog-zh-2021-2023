微软为 GPT-3 推出 Azure OpenAI 服务

原文：https://web.archive.org/web/https://techcrunch.com/2021/11/02/microsofts-new-azure-openai-service-brings-gpt-3-to-a-few-more-developers/

微软新的 Azure OpenAI 服务为更多开发者带来了 GPT 3

微软今天宣布推出 Azure OpenAI 服务，顾名思义，这使得 OpenAI 的机器学习模型可以在 Azure 平台上使用。具体来说，这意味着 GPT-3 ，OpenAI 开创性的语言模型，在适当的情况下，只需几个提示就可以产生类似人类的文本。

不过，有一个问题。至少目前来看，并不是所有的 Azure 用户都会获得访问权限(即使他们愿意付费)。只有受到邀请的客户才能访问，并且“客户计划实施明确定义的用例，这些用例包含使用人工智能技术的负责任的原则和策略。”微软将提供安全监控和分析，以发现滥用或误用 GPT-3 的案例，并将提供过滤器，以确保你基于 GPT-3 的聊天机器人不会开始咒骂你的高管(即使 GPT-3 认为他们活该)。

图片来源:微软

值得注意的是，OpenAI 本身已经在去年推出了 GPT-3 API，尽管还有一个等待列表。微软也已经在使用 GPT-3 为 GitHub 的 Copilot 工具提供动力，帮助开发者为他们编写代码。但是，尽管已经有办法在 Azure 之外访问 GPT-3，微软认为它可以提供“额外的安全层，访问管理，私人网络，数据处理保护或扩展容量。”

微软早在 2019 年就向 OpenAI 投资了 10 亿美元，并获得了 GPT-3 的许可，所以该公司现在试图将其引入更广泛的产品也就不足为奇了。

“GPT-3 确实证明了自己是第一个强大的、通用的自然语言模型——这是一个可以用于所有这些事情的模型，开发者喜欢它，因为你可以非常容易地尝试，”OpenAI 首席执行官萨姆·奥尔特曼说。“一段时间以来，我们一直想找到一种方法来尽可能广泛地扩展它，这是与微软合作真正让我们兴奋的一部分。”
