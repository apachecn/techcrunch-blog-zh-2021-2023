Flipboard 通过乳齿象整合和社区加入了 Fediverse，计划为 ActivityPub | TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2023/02/28/flipboard-joins-the-fediverse-with-a-mastodon-integration-and-community-plans-for-activitypub/

杂志应用 Flipboard 正在加入Fe diversity——一组互联的服务器，为一系列开源、分散的应用提供动力，包括最近流行的 Twitter 替代品乳齿象。从今天开始，iOS 版 Flipboard 应用将包括一项测试功能，允许 Mastodon 用户直观地浏览他们的时间线，以查看他们关注的人的帖子，就像他们在 Twitter 上所做的那样。但该公司的 Fediverse 雄心不仅仅局限于产品集成，因为 flipboard 正在推出自己的乳齿象实例 flipboard.social ，为 Flipboard 用户和杂志策展人提供一种参与去中心化网络的简单方式。它还计划支持 ActivityPub，这是一种支持 Mastodon 和其他联合应用的底层协议。

目前，只有 Flipboard 管理员才能访问这个新实例。但 Flipboard 表示，随着时间的推移，它的目标是通过其实例让更多用户加入 Fediverse。

此前，博客平台 Medium 也发布了类似的公告，该平台上个月为其作者推出了自己的乳齿象社区。其他正在考虑 Fediverse 集成的公司包括 Tumblr 和 Flickr ，这两家公司也都讨论过增加对 ActivityPub 的支持。

对于 Flipboard 来说，新的乳齿象整合填补了其服务中的一个潜在漏洞。该公司明白，随着埃隆·马斯克(Elon Musk)领导下的 Twitter API 变化，其长期的 Twitter 集成可能变得不可靠，这已经影响到了一系列第三方应用程序———许多 Twitter 客户端。 Flipboard 最初是一份社交杂志，用户可以阅读他们的新闻以及 Twitter 等社交应用的更新，它知道自己与 Twitter 的时间可能有限。

“我们不知道这种情况还会持续多久，”Flipboard 首席执行官迈克·麦库埃在谈到该应用程序的 Twitter 集成时说。

他说，如果出现任何 API 问题，该公司不再与 Twitter 联系，尽管它尚未经历重大中断。不过，他说，接下来会发生什么“谁也说不准”

通过与 Mastodon 集成，Flipboard 用户将能够继续浏览他们在 Flipboard 应用程序中关注的人的简短更新。这包括带有文本、图像和链接预览的乳齿象帖子，就像 Twitter 提供的那样。他们也可以利用这种整合来撰写自己的乳齿象帖子，并上传自己的链接和照片。此外，他们还可以回复、点赞和转发他人的帖子，点击标签来关注更大范围的话题讨论，等等。

Flipboard 策展人也可以使用加号按钮“翻转”(添加)乳齿象的帖子到他们的杂志上，就像他们以前在 Twitter 上做的那样。而 Flipboard 新增加的笔记功能，允许策展人在他们的杂志中加入原创内容，将能够被转化为乳齿象文本帖子。

虽然它不是一个功能齐全的客户端——像 Ivory 、 Mammoth 或 Ice Cubes 这样的应用程序——Flipboard 提供了一种简单的方法来跟上你的乳齿象社区以及其他新闻和故事。

但是 Flipboard 的 Fediverse 雄心超出了 Masoton 整合的范围。麦库埃相信，费多元有潜力重塑网络，如今网络的权力集中在几个大型科技平台上。

“ActivityPub(为 Fediverse 提供动力的协议)真正强大的地方在于它是一个 W3 标准。它不依附于加密，也不依附于加密——它实际上是在试图解决一个问题，那就是一个普通的社交图和一个普通的命名空间，人们可以用它来有效地在一个开放的社交网络中生存，”他说。“这个机会是我从网络早期以来看到的最令人兴奋的事情。我真的认为这是一件大事，”麦库埃补充道。

到目前为止，像脸书(Meta)这样的公司已经拥有了社交图，但是 ActivityPub 的承诺是在这个新的社交层上构建各种服务。目前，麦库埃指出，我们正在看到大量现有应用的克隆出现——比如推特克隆版 Mastodon，或者联合 Instagram 克隆版pixel feed。但他认为，随着时间的推移，其他服务将会实现并增长。

Flipboard 是探索如何最好地将 ActivityPub 作为社交层的核心能力的公司之一。这将最终允许 Mastodon 和其他联合应用程序的用户关注 Flipboard 的策展人和杂志，同时也允许 Flipboard 的用户关注其他服务的用户和订阅源，比如 Mastodon、PixelFed 和 PeerTube。

该公司还通过在 flipboard.social 开设自己的 Mastodon 实例，直接投资于 Fe diversity。首席执行官将该实例描述为“高质量、可扩展的实例，它将是高度可调节、快速、安全和可靠的。”公司自己的审核团队现在有七名员工，除了现有的职责外，还将负责实例的审核。

今天，Flipboard 将开始邀请其顶级策展人加入，但其他人都可以在 flipboard.social 网站上登记其候补名单。最初，该实例将向几百个开放，但随着时间的推移，将以缓慢而稳定的速度扩展到一位数的几千个，然后扩展到数万个。这一实例运行在 Flipboard 自己的基础设施上，将由该公司现有的收入进行补贴，但麦库埃认为，将会出现一种让公司能够自我维持、适度参与和扩大规模的多元化商业模式。

“我认为，在某个时候，这将需要被弄清楚，但我认为，这是一种事情，以集体做的人谁是在多元化，”他说。

如果 Flipboard 的社区选择拥抱这一新领域，那么它对 Fediverse 的加入有可能增加大量用户。该公司此前表示，其应用程序上有“数百万”份杂志，但其中只有 25%-50%的杂志在某个月是活跃的。它还声称每月有 1 亿活跃用户，但这个数字包括了它的时事通讯订户，而不仅仅是它的应用程序用户，所以这不是一个有多少活跃贡献者的准确图片。尽管如此，即使一小部分人从 Flipboard 的社区加入了 Fediverse，这对于分散化的网络来说也是一个很好的推动，今天的每月活跃用户数约为 256 万。

iPhone 版 Flipboard 的最新版本现在将包括乳齿象功能，该实例的等候名单今天在 flipboard.social 上直播。
