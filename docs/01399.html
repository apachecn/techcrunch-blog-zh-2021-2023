，这是一个面向活跃在数字资产市场的资产管理公司和交易商的跨链协议。

点对点交易，面对面，眼对眼——这是几千年来交易的方式，在距离和缺乏信任迫使我们使用银行和经纪人等中间人进行交易之前。

现在，分散融资(DeFi)已经把我们带回到一个过度抵押的未来。我们不仅可以远程进行点对点交易，还可以通过智能合约进行无信任交易。这一创新为金融复兴奠定了基础，其意义远不止取代中介机构。

直到最近，监管机构在很大程度上忽视了这一新兴的平行金融体系。但随着前区块链大学教授加里·詹斯勒担任美国证券交易委员会主席，华盛顿已经觉醒了。问题是:当局如何执行不依赖中介机构的监管？该法规将如何保护用户和市场？

权力下放>非地域化

DeFi 协议可能会超出监管范围。区块链交易历史的副本存储在世界各地的节点上，如果其中一个受到威胁，随时可以像多头九头蛇一样重新出现。

然而，历史提供了监管者可能认为他们可以解决赤字问题的经验。

从历史上看，监管机构只对其管辖范围内的法律实体拥有权限。随着 2010 年《域外外国账户税收合规法案(FATCA)》的出台，这种情况发生了变化。根据该法案，美国当局不仅对本国货币和世界各地的美国人进行监管，还通过签署政府间执行协议(IgA)与其他司法管辖区进行协调。

欧盟在 2018 年的《通用数据保护条例》(GDPR)中采取了类似的方法，在他们的象牙塔中编写法规，以控制欧洲人在世界任何地方的数据——尽管目前尚不清楚当局如何对欧盟以外的组织实施执法。

展望未来，我们可以看到监管机构依靠类似的治外法权手段，试图进入网络空间，在不受限制的情况下实施监管。

阻塞点和入口匝道

然而，即使有了域外执法，监管者仍需要找出可以用来控制分散协议的瓶颈。

这些集中点似乎已经引起了监管者的注意。正如 Gensler 评论的那样:DeFi 可能是一个误称，平台通常“在某些方面是分散的，但在其他方面是高度集中的。”

拥有已知开发者的单个协议，或者那些由公司令牌持有者控制的协议，可能会被迫对协议进行修改。对于像他们声称的那样分散的协议——由分布式匿名社区运行——监管者可以将与协议的交互视为非法。或者，更有可能的是，通过瞄准入口或者将某些协议标记为有毒来阻碍资金的流动。

这些入口可以是法定到加密的交换或稳定的连接，可以强制纳入尽职调查和了解客户程序，以确保符合反洗钱和反恐怖主义(反洗钱/CFT)工作等。为了有效，这些未来的控制将需要建立与 DeFi 铭记。这可以看到制裁名单公布为一个链接查找或一个免费的 API 调用，从金融行动特别工作组(FATF)或经济合作与发展组织(经合组织)直接。

与此同时，寻求与实体经济整合的个别协议可能会做出有利于监管者的权衡。

例如， Aave 的“了解你的客户”版本的流动性池，通过向 KYC 参与者提供入口，为机构提供有限的 DeFi 准入。他们能够通过依赖 Chainalysis 这样的组织来分析区块链以了解你的交易(KYT)来降低风险，尽管这是以流动性深度为代价的，并且不会为所有参与者增加馅饼。

其他有前景的解决方案包括智能包装合同，允许经过验证的实体存放资金并自动铸造“完全合规的资产”，这些资产可用于任何 DeFi 协议，而不必每次都进行 KYC。

另一方面，协议可能会进一步分散；正如我们最近所看到的，MakerDAO 关闭了法律实体，并完全依赖于 DAO。但是，尽管这些完全去中心化的协议可能仍然不受监管机构的监管，但它们也可能在一定程度上脱离实体经济。

考虑到这些情况，问题不再是如何实施监管，而是监管应该旨在实现什么结果。

DeFi 应该如何监管？

至于应该在协议层面推动哪些变化，我们现在正站在一个十字路口。

适当水平的监管有机会让 DeFi 有足够的喘息空间来有所作为:提高透明度，增加金融包容性，为 80 亿人提供信贷，这将使世界朝着繁荣迈出巨大的一步。

然而，过度扩张也有可能扼杀创新和增长，并产生意想不到的后果。不幸的是，我们似乎已经在这条路上走得很远了。

我们需要的是认识到 DeFi 与金融监管机构有许多相同的目标:彻底改革不灵活的流程，提供更广泛的准入、更低廉的价格和更大的稳定性，同时确保市场所有参与者都能广泛分享这些好处。

例如，获得流动性一直是加密货币和区块链项目乃至整个金融市场的核心问题。根据英国央行(Bank of England)2019 年的 Run Lola Run 演讲，有证据表明，那些离流动性更远的人会得到越来越糟糕的交易。

DeFi 有潜力通过全新的机制创建更公平、更透明和更具流动性的市场，帮助每个人减少欺诈和抢注，解决碎片化问题，并创建高效、有弹性、公平和所有人都能平等进入的市场，而不仅仅是拥有正确联系的参与者。

定义正确的监管可以决定 DeFi 的成败，还有一些大问题需要回答:我们如何确定钱包得分？我们如何构建分散的标识符( W3C DIDs )？我们如何确保任何控制措施都不会对金融包容性产生不利影响？

考虑到这样一个从头开始重建金融的机会，我们需要大胆一点:设定明确的目标，建立监管，为进入新的金融世界铺平道路——而不是简单地满足于我们今天拥有的更快版本。
