Nerdio 斥资 1.17 亿美元在 Azure 虚拟桌面上构建管理工具| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/12/14/nerdio-lands-117m-to-build-management-tools-on-top-of-azure-virtual-desktop/

2005 年，三位企业家——Vadim Vladimirskiy、Stuart Gabel 和 Niall Keegan——共同创立了 Adar，这是一家总部位于芝加哥的公司，主要向中小型企业提供“流媒体 IT”和 IT 即服务产品。仅仅十多年后，Adar 创建了一个内部部门, Nerdio ,专注于帮助其他托管服务提供商将其客户迁移到云。

认识到进一步增长的机会，Vladimirskiy 和 Nerdio 的联合创始人、前微软高管 Joseph Landes 决定将 Nerdio 分拆为一家独立公司，并在 2020 年 1 月将 Adar 出售给一家私募股权公司。他们在那年 2 月为 Nerdio 的第一轮融资做出了重大贡献，这被证明是一次明智的押注。Nerdio 今天完成了由 Updata Partners 牵头的 1.17 亿美元的 B 轮融资，使该公司的融资总额达到 1.25 亿美元。

Vladimirskiy 通过电子邮件向 TechCrunch 表示，收益将用于 Nerdio 的客户获取、产品 R&D 和招聘工作。“这项投资的市场时机非常理想，”他补充道。“随着许多企业带着对混合和远程工作的新爱好走出疫情，对能够最大限度地实现数字工作空间的安全高效交付和管理的现代技术有着很高的需求。”

Nerdio 的平台让客户能够部署、管理和优化在微软 Azure 中运行的虚拟桌面，扩展了微软基于云的 Windows 虚拟化系统 Azure 虚拟桌面的功能。Vladimirskiy 声称，凭借自动扩展、“许可证优化”、安全性和合规性，以及微软本地提供的监控和报告功能，Nerdio 可以在现有 Azure 虚拟桌面部署的基础上“无中断”分层，从而大幅节省成本。(Nerdio 作为基于 Azure 的应用程序运行在客户自己的 Azure 订阅中。)

“有了 Nerdio，Azure 虚拟桌面中的每个部署或管理任务都可以由任何级别的 IT 人员通过几次点击来完成，”Vladimirskiy 说。“凭借强大的策略和访问管理功能以及 RBAC 角色，我们为采用新一代 IT 的组织提供了一个安全网，以充分掌握 Azure 的功能和价值，而不会迷失在它的复杂性中。”

Vladimirskiy 表示，在疫情期间，由于企业受到与流行病相关的停工以及随后转向混合和远程工作的影响，企业开始为其员工探索虚拟桌面解决方案，业务真正起飞了。调查数据说明了这一戏剧性的转变。商业技术评论网站 TrustRadius 在 2020 年进行的一项民意调查发现，超过 50%的企业预计明年将增加在远程桌面软件上的支出。

根据 Vladimirskiy 的数据，从 2020 年 2 月到本月，Nerdio 的年度经常性收入增长了 2000%，现在拥有超过 5000 家客户，包括托管服务提供商和系统集成商。与此同时，Nerdio 的渠道合作伙伴增加到 1000 多家。

“[在疫情期间，]每个人都必须立刻找到解决方案，让他们的员工能够继续工作，并且能够在家安全地工作。因此，管理服务提供商和企业采用都非常重要。“即使在经济不稳定时期，我们也能够保持资本效率，确保我们为成功做好准备。”

Nerdio 与微软的密切合作关系可能会有所帮助。这家初创公司的董事会中有许多微软的资深人士，包括微软全球渠道合作伙伴的前负责人 Gavriella Schuster，它定期在这家科技巨头的营销材料中出现在。

这也有助于 Azure 虚拟桌面的迅速普及，尽管存在来自 VMware 和 Citrix 等现有公司的竞争。eG Innovations 和 AVD TechFest 在 2022 年初对大约 500 家组织进行的调查发现，58%的组织打算在两年内安装 Azure 虚拟桌面。调查表明，Azure 虚拟桌面正在为小型企业找到一个利基市场，这些企业通常不会考虑 Citrix 或 VMware，因为成本或其他限制。

Vladimirskiy 说:“在托管服务提供商领域，我们真的看不到其他任何人能够为托管服务提供商提供我们在成本和时间管理以及可扩展性方面的定制功能。“我们不仅让 Azure 和它的虚拟桌面服务在时间和资源方面令人满意——通过 Nerdio，Azure 虚拟桌面中的每个部署或管理任务都可以由任何级别的 IT 人员通过几次点击来完成——而且我们解决了可以说是采用这些原生技术的最大障碍，即成本。”

总部位于伊利诺伊州斯科基的 Nerdio 目前有一个 100 人的团队，并计划在明年增加一倍。
