阅读最多

API 形式的 ChatGPT:open ai 推出了一种 API，允许任何企业将 ChatGPT 技术构建到他们的应用程序、网站、产品和服务中。(提醒一下， ChatGPT 是一个免费的文本生成人工智能，可以编写类似人类的代码、电子邮件、文章等等。)Snap、Quizlet、Instacart 和 Shopify 都是早期采用者。

变成人类: 本周，一家初创公司图从秘密行动中脱颖而出，有望制造出一款通用的两足人形机器人。(布莱恩 九月份爆料这家初创公司的存在，以防你错过。该公司于去年 12 月完成的人形机器人 alpha build 目前正在其桑尼维尔办公室进行测试。它目前专注于广泛的体力劳动任务。

无证监控: 扎克报道称，特勤局和 ice 的国土安全调查小组在进行侵入性手机监控时，多次未能获得正确的法律文书。调查结果于上周由国土安全部监察长公布，该监察长负责监督美国联邦部门及其许多执法部门，称这些机构经常在没有获得适当搜查令的情况下使用手机基站模拟器。

Salesforce 扭转乾坤: 本周，Salesforce 报告了其财年第四季度收益，包括超过预期的收入和超出华尔街估计的指导。这是该公司急需的胜利，该公司正面临来自包括艾略特管理公司在内的激进投资者越来越大的压力。

氢动力: 初创公司环球氢本周搭载有史以来最大的氢燃料电池升空。一架改装的 Dash-8 飞机的 15 分钟试飞很短，但是——正如马克所写的——它表明氢可以作为短程客机的燃料。(然而，许多技术和监管壁垒阻碍了这一进程。)

暂停你的连拍: 伊万报道称，Snapchat 将允许用户暂停他们的连拍——你可以每 24 小时向你的朋友发送一次照片——这样他们就不必担心如果他们决定暂时不访问该应用程序会破坏连拍。

新的人工智能非营利组织: 一个由社区驱动的人工智能研究组织 EleutherAI ，正在组建一个非营利基金会。由支持者的捐款和拨款资助，包括人工智能初创公司 Hugging Face 和 Stability AI，GitHub 前首席执行官 Nat Friedman，Lambda Labs 和 Canva，该非营利组织计划沿着 OpenAI 的 ChatGPT 路线研究大型语言模型的问题。

停止《继承者们》: 《继承者们》最后一季的官方预告片于本周首播，看起来该系列将以史诗般的麦克风掉落而告终。正如劳伦所写的，HBO 系列不仅获得了巨大的成功，获得了 13 项艾美奖奖和 5 项金球奖奖，而且也是对媒体行业的有趣评论。创作者兼导演杰西·阿姆斯特朗承认从很多地方获取灵感，包括鲁珀特·默多克的剧本。

科技博客+

TC+订阅者可以获得深入的评论、分析和调查，如果你已经是订阅者，你就会知道。如果你不是，考虑注册。以下是本周的一些亮点:

女性风投的“品牌化”问题: 做风投的目标是为有限合伙人创造回报，人们普遍认为，多样化的创业生态系统会给所有人带来更好的结果。但是 Natasha 和 Rebecca 写道，对于女性风险投资家来说，如何平衡这两者常常以不同的、令人沮丧的方式表现出来。

赶时髦:十一国际公司的公关顾问卡米拉·坦恩(Camilla Tenn)写道，科技初创公司是否应该将他们的信息转向与人工智能相关的话题。她认为，如果人工智能相关的报道能够让一个新的、未知的品牌进入其目标出版物，它可能有助于在明天将该品牌的宣传资料呈现在潜在投资者面前。

把开源变成生意: 尽管开源软件分发的前提是“免费”，但像 Red Hat、MongoDB、GitLab 和 Elastic 这样价值数十亿美元的公司已经开始以开源为核心建立有利可图的业务。但是，一个更小的开源项目有可能进入这片充满商业机会的土地吗？维多利亚·梅尔尼科娃调查。
