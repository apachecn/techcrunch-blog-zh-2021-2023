Kevin 筹集了 6500 万美元用于账户对账户支付

原文：https://web.archive.org/web/https://techcrunch.com/2022/05/03/kevin-raises-65m-as-it-charges-ahead-on-account-to-account-payments-over-point-of-sale-terminals/

在全球范围内，支付仍然是一个非常分散的行业:根据你在哪里买卖东西(以及你是在线还是离线销售)，你会有不同的“标准”支付方式、货币和结算方案等。今天，一家名为 Kevin 的初创公司正在解决这个难题的一部分——从一个账户到另一个账户的支付，这是一种绕过这些轨道的支付卡支付的替代方式——并通过开发一套全新的直接与银行集成的支付基础设施，使其更容易和更普遍地使用，该公司宣布了一笔 6500 万美元的重大 A 轮投资，在最初的强劲牵引下，对其业务进行了双倍投资。

它已经在欧洲 12 个市场吸引了 6，000 家商户，首先是电子销售点，最近是与物理 POS 终端的集成。它的计划是到今年年底，在欧洲约 35%的电子销售点终端上提供支付选项，然后在后年达到 85%，“与任何卡方案一样，”首席执行官 Tadas Tamosiunas 在接受采访时说。

英国将在今年晚些时候，但在今年年底，欧洲 EPOS 终端将达到 35%,明年将达到 85%,与卡计划相同。

这轮融资由 Accel 牵头，Eurazeo 和之前的支持者 OTB 风险投资公司、Speedinvest、OpenOcean 和 Global Paytech Ventures 也参与了进来。20VC 的哈利·斯特宾斯；Ilkka Paananen，Supercell 首席执行官兼联合创始人；和 Venmo 前首席执行官 Amitabh Jhawar 也参与了这轮投资。凯文现在已经筹集了 7700 万美元，它没有透露其估值。

总部位于立陶宛的 Kevin 由 Tamosiunas 和 Pavel Sokolovas(首席运营官)共同创立，他们在联合采访中表示，计划将利用这笔资金继续发展其技术，并雇佣更多的人进入更多的市场，首先覆盖整个欧洲。

凯文的学名是“凯文”—包括句号。Tamosiunas 表示，做出这一选择有几个原因:首先，“Kevin”是一个普通人的名字，其想法是这是一个对每个人都有用的技术支付解决方案；第二，句号暗示这是你在工作中需要知道的名字和姓氏；但是第三，作为谈话的开场白。“这给了我们一个讲述自己故事的机会，”他简单地说。

这个故事是商人和其他从事支付和商业的人所熟知的:每个国家在支付过程的前端和后端都有不同的支付系统。账户到账户支付，实质上是直接从买家那里扣款，然后存入卖家的账户，长期以来一直是这些选项之一，当有人没有使用现金时，它通常是比信用卡支付和产生的费用更便宜和直接的替代选择。

问题是，许多现有的账户到账户支付基础设施非常笨重，不是围绕 API 构建的，因此很难扩展和集成到任何新的服务中，无论是实体故事中的服务还是“电子销售点”中的服务，这些服务可能在商店中，但也可以很容易地在例如为停车场的时间付费的应用程序中。

“但是帐户对帐户是一个更便宜的过程，所以我们有一个巨大的机会来解决这个问题，特别是在 EPOS 中，”Sokolovas 说。Kevin 在大楼里呆了几年，最初有很多反对者，怀疑 API 可以被构建来与银行集成，银行传统上一直缓慢地接受它们并向其他人开放它们的服务。当然，也有例外，比如我们在英国看到的开放银行业的努力，但总的来说，这是一个分散的、仍然神秘的领域。“现在，我们是市场上唯一一家拥有技术解决方案的公司。”

Tamosiunas 说，现在有其他公司正在赶上——例如 POS 终端巨头 Worldline 正在开发一种接受帐户到帐户支付的解决方案，但他声称这需要几年的时间。

更大的主题是，电子商务仍然是一个巨大且快速增长的领域，但在新冠肺炎疫情的巅峰时期之后，随着向实体运动的回归，重心也在发生变化。“每个人都在寻找如何在销售点提高线下销售，”Tamosiunas 补充说。

Kevin 在这里寻求的颠覆不仅仅是它开启了一个已经存在多年但很难使用的流程并使之现代化；但它也给了商家、消费者和任何交易中的其他人一种更直接的方式来实现特定的支付。更直接意味着它也更便宜，这也是推销的一个重要部分:这意味着任何选择这一选项的人都可以在交易中获得更高的利润。相反，这也将支付生态系统中的许多传统参与者排除在外，这是另一种破坏。

这不仅引起了投资者的注意，也引起了潜在战略合作伙伴和潜在收购者的注意。创始人不会详细说明是谁在敲他们的门，但你可以想象支付技术领域的其他新老巨头(包括 Stripe、Adyen、PayPal，甚至可能是大型信用卡铁路公司)可能会对多元化投资中的这项技术感兴趣。目前，凯文甚至拒绝作为战略投资者与他们合作，以保持中立，不依赖于任何特定的平台。

“Tadas、Pavel 和 Kevin 团队正在用他们的下一代支付基础设施推动支付的未来，”Accel 的合伙人 Luca Bocchio 在一份声明中指出。“提供快速、无缝的支付体验，降低成本并提高认证率，现在是 A2A 支付的时候了，Kevin 已经凭借其产品取得了令人印象深刻的发展势头。随着其独特的 POS 支付产品的推出，未来的机会是巨大的，我们期待着在他们的旅程中与团队合作。”

一个有趣的转折是 Kevin 和类似的人是否以及如何与移动钱包整合。

如今，当商家将其技术集成到他们自己的销售点时，无论是物理的还是电子的，以及在应用程序中，Kevin 都在服务行业运营。但像 Apple Pay 或 Google Pay 这样的钱包现在只能用卡支付。鉴于如此多的信用卡交易正在被使用手机的 NFC 支付取代，如果 Kevin 不能为消费者提供这种支付方式的替代选择，它可能会限制 Kevin 的增长空间。

巧合的是，就在昨天，苹果因其如何向其他方开放(或不开放，视情况而定)基于 NFC 的钱包技术而被欧盟称为反竞争行为。这将是一场值得关注的比赛，也将对凯文未来的成长产生重大影响。
