集成平台 Cinchy 获得新资金来连接数据源| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/10/27/integration-platform-cinchy-lands-fresh-cash-to-connect-data-sources/

为企业客户提供数据管理服务的初创公司 Cinchy 今天宣布，它获得了由 Forgepoint Capital 领投的 1450 万美元 B 轮融资，IVP、SUV、Techstars 和 Mars 也参与了融资。首席执行官丹·德莫斯(Dan DeMers)在接受 TechCrunch 采访时表示，这笔资金将使公司的总融资额达到 2400 万美元，将用于扩大 Cinchy 的业务范围，并继续投资于这家初创公司的核心技术。

“数据管理仍然是一项昂贵的工作，应用程序的激增产生了不断增长的数据量，这只会增加挑战。因此，数据与其说是业务驱动力或竞争优势，不如说更经常是 IT 预算的消耗和合规团队的噩梦，”DeMers 说。“Cinchy 平台解决了许多与当今 IT 环境相关的挑战，特别是那些由数据孤岛、数据副本和复杂代码定义的挑战。”

DeMers 在 2017 年与 Karanjot Jaswal 共同创立了 Cinchy，雄心勃勃的目标是抽象出数据集成流程。德默斯之前是花旗集团(Citi)大宗金融和期货技术总监，在那里他为经纪公司建立并管理了一个技术交付和支持服务团队。贾斯瓦尔也在花旗，在风险和利润率数据仓库团队工作。

DeMers 和 Jasawal 都意识到公司正在努力克服数据整合的障碍。就他们的观点而言，在最近的 IBM 调查中，40%的 IT 领导者表示他们的数据集成变得过于昂贵，而 19%的人认为他们当前的数据集成解决方案无法处理所有数据源。

“现有的以应用程序和 API 为中心的架构要求各个应用程序管理自己的数据，这意味着每个新的应用程序或 API 都会增加另一个数据孤岛，”DeMers 说。“这就像是一种创新税，每推出一种新的解决方案，情况都会变得更糟。”

Cinchy 旨在通过将应用程序和其他孤岛连接到一个“基于网络的”平台，使组织能够将数据从应用程序和其他孤岛中分离出来，从而解决这一问题。项目团队首先将核心系统、软件即服务应用和电子表格中的数据连接到平台——CIN chy 处理数据备份、数据版本和数据参与跟踪等事情，而不实际托管数据。管理员可以访问该平台来查看、编辑或查询个人和团队的数据。其他拥有适当权限的用户可以使用这些数据来构建数据模型。

图片来源: Cinchy

Cinchy 使用平台本身来经营业务。DeMers 说，员工可以自助发现、查询、创建和更改数据。对数据的更改受版本控制和访问控制，并基于粒度控制提供给应用程序和用户。

“任何体验过 Google Drive 和 Docs 等协作工具的协作和效率的人都会理解将这些功能引入组织数据的重要性，”DeMers 说。“在速度、效率、控制和创造性解决问题方面的成果令人震惊。”

DeMers 认为 Cinchy 与任何承诺简化数据集成的供应商竞争。有相当多的公司，包括 Equalum 、 Airbyte 、 Hevo Data 和Jitsu——所有的都在追逐，这个市场到 2027 年可能价值 222.8 亿美元。对数据集成解决方案的需求显然很高，Dresner 咨询服务公司的 2020 年调查发现，67%的企业依靠数据集成来支持分析和商业智能平台，24%的企业计划在未来 12 个月内这样做。

但 DeMers 认为，大多数人都专注于更好地处理数据碎片的变通办法，特别是在分析的背景下。“大多数可能被视为与 Cinchy 竞争的产品实际上只会加剧与数据集成相关的敏捷性和合规性的挑战，”他说。

竞争对手无疑不同意。不过，Cinchy 确实拥有越来越多的客户群，尤其是在金融行业——这表明它正在赢得企业的支持。采用者跨越像道明银行、国家银行和 Natixis 这样的机构；Cinchy 最近推出了该平台的信用联盟版，以更好地服务于金融机构。

“各地的组织都在寻找节省资金的方法，同时继续利用新解决方案抓住市场机遇。这就是为什么我们相信 Cinchy 平台将越来越多地吸引首席体验官和负责沟通这些优先事项的团队领导，”DeMers 说。“Cinchy …使组织能够将其数据从应用程序、电子表格和其他孤岛中解放出来，并在需要时随时随地使其[可用于]实时协作。”

总部位于多伦多的 Cinchy 只有 50 多名员工，目前正在招聘。
