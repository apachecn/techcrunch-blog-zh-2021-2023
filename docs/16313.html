Brex 抨击 Concur 向旅游业扩张

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/22/fintech-brex-takes-a-swipe-at-concur-with-an-expansion-into-travel/

另一个表明消费管理领域竞争激烈的例子是，Brex 今天宣布它已经扩展到旅游业。

除了寻求从 Concur 等现有公司夺取市场份额，Brex 还在与 Navan(以前称为 TripActions)等初创公司竞争，Navan 实际上在 扩大其产品——和 Ramp 之前就开始了专注于差旅费管理的业务，Ramp 本身 去年通过一个让员工能够与第三方预订的工具扩展到了差旅 。

在接受 TechCrunch 采访时，联合首席执行官兼联合创始人 Henrique Dubugras 表示，Brex Empower 的用户现在可以在 Brex 移动应用程序中预订旅行并管理流程。新功能还可以与 Empower 的所有预算和策略功能配合使用。

Brex 已经花了一年多的时间来开发这款产品，并在本周发布之前进行了几个月的私下测试。尽管该领域有各种各样的参与者，但 Dubugras 表示，Concur 和 Navan 是 Brex Travel 的主要竞争对手。

Dubugras 说，新产品的部分目的是将 Empower 拥有的预算功能应用到差旅费上。

他说:“有时候，公司认为降低旅行成本的最佳方式是让人们选择更便宜的航班或更便宜的酒店，并在去两个地方的途中停留两次。“我们认为，实际管理它的最佳方式是实际帮助一些旅行根本不发生。”

Dubugras 说，例如，财务团队能够看到信息——比如一家公司有 10 万美元的预算——并实时在公司卡上消费，然后决定哪些旅行应该优先。

新的旅行功能还会自动提供收据，这样员工就不必说，退房后再上传收据。它还可以让员工知道某件事情是否违反了政策，是否需要额外的批准。如果公司要求，员工只能看到符合政策的航班和酒店选项。

图片来源: Brex

杜布格拉斯说，Brex 选择不接受航空公司或酒店的佣金，因为它们不会提供更便宜的航班或酒店作为替代。

他补充道:“一些公司得到报酬不展示这些信息，因为他们知道公司买家对价格不太敏感。”。“但我们认为这只会降低人们对这个系统的信任。因为旅游不是我们的核心业务，我们决定不要这笔钱，基本上有完全公正的库存，显示所有航班和所有可用的酒店。”

Brex 通过收取每次旅行的费用从新产品中获得收入，根据旅行量的不同，费用会因公司而异。

另一个旨在简化旅行预订的功能是让员工能够同时预订飞往另一个国家和在一个国家内的航班。通常，这两个预订需要分别进行。他补充说，该应用程序还旨在帮助在不同国家工作的员工更容易在国内旅行，并以不同货币支付旅行费用。

值得注意的是，在构建其新的旅游产品时，Brex 与一家名为spot nana的公司合作，本质上是将其“旅游即服务”软件贴上白色标签，该软件提供与全球航空公司和酒店集成的后端基础设施。

“我认为这有点像‘旅行条纹’，”杜布格拉斯说。

Brex 表示，它将 Spotnana 的旅行相关功能与其移动应用程序、卡和消费管理功能相集成，以支持旅行政策、旅行者档案以及预订和报告的工作流程。该公司表示，它确实在 Spotnana 的基础上建立了“大量定制”——如旅行者的旅行细节，以努力“使支出变得简单和无缝”此外，Spotnana 的库存搜索 API 有助于 Brex 向用户提供旅行预算估计。

从技术角度来看，该公司表示选择 Spotnana 是因为其基于微服务的架构、开放的 API、“全面的”旅游内容、可扩展的数据基础设施和扩展的白标 UI 组件库。

Brex 于 2022 年 4 月公布了其支出管理产品Empower。今天，Brex 表示，超过 120 个国家的公司，如 DoorDash，比特币基地，事实上，Superhuman，SeatGeek，Built Technologies 和 Scale AI，都是该产品的用户。Empower 上的旅游客户包括 GoGuardian、Miso Robotics、FMX、Satellite Healthcare 等。

该公司表示，它已经开始将所有信用卡客户迁移到 Empower。

初创公司正叫嚣着要帮助减轻管理旅行费用的痛苦。今年 2 月，Navan(以前的 TripActions)声称是第一家在其基础设施和产品系列中集成 OpenAI 和 ChatGPT APIs 的旅游公司。该公司表示，目前正在使用生成式人工智能技术来编写、测试和修复代码，目的是提高其运营效率并减少开销。当时，一位发言人表示，项目管理员将能够要求 Navan 的虚拟助理艾娃报告整个旅行和支出项目，无论是通过文本，图表，PDF 等。她补充说:“我们还使用人工智能做一切事情，从消除费用报告到自动列举项目——就酒店账户而言，我们在入住后立即从酒店获取它，对行项目进行分类，与公司政策进行比较，并为用户提交，所以他们没有必要为了平衡账户而四处转移资金——在我的经历中，这是一个非常痛苦的过程。”

去年，当 Ramp 推出自己的旅游产品时，首席执行官兼联合创始人 Eric Glyman 告诉 TechCrunch，它将免费提供给所有现有客户和任何新客户。Ramp 表示，它计划使用人工智能辅助软件来“完全”自动化公司的费用报告，比如自动收集收据和对所有与旅行相关的费用进行分类。该公司表示，其产品还会标记出违反政策的购买，并允许员工通过他们想要的任何服务预订旅行。

30 年前的 Concur 也有一个旅行专用产品，它具有 Concur Request 等功能，允许企业创建“可定制的预消费控制文档，以确定预期的费用。”
