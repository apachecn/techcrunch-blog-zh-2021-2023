宇迦实验室的妮可·穆尼斯将在 TC 会议上谈论 NFT 和无聊的猿类:加密

随着 NFT 生态系统继续摇摆不定，超级粉丝和蓝筹股持有者仍在强势持有。但是，数字资产行业如何才能重新点燃增长并吸引新的受众呢？根据不可买市场跟踪系统的数据，NFT 的销售量比去年同期下降了近 90%。

但是，炒作仍然在像 Reddit 这样的平台上增长，它见证了数百万个加密钱包被创建来铸造 NFT 或像 Cardano 这样的第一层区块链，这创下了NFT 卷的历史新高。

NFT 是加密领域最热门的话题之一，这就是为什么我们很高兴邀请到宇迦实验室的首席执行官妮可·穆尼斯于 11 月 17 日在迈阿密的 TC Sessions: Crypto 上登台。

今年早些时候，宇迦实验室在 Andreessen Horowitz 领导的一轮融资中筹集了 4.5 亿美元，估值达到 40 亿美元。总部位于迈阿密的 NFT 创业公司是 Bored Ape 游艇俱乐部和 variant Ape 游艇俱乐部的幕后公司，并且已经收购了其他受欢迎的 NFT 项目，如 CryptoPunks 和 Meebits。

宇迦也为 ApeCoin 的推出做出了贡献，这是一个受其 BAYC 系列启发的项目。这种代币在交易的第一天就聚集了数十亿美元的市值，到目前为止约为 14 亿美元。据 TechCrunch 此前报道，该公司希望建立一个“扩展宇宙”的无聊猿游艇俱乐部，但同时“邀请更大的 NFT 社区”加入。

综上所述，穆尼斯将会有很多话题可谈，从宇迦实验室的细微细节到她对更广泛的 NFT 生态系统的想法以及现状。我们想听听她对 NFT 市场的挑战和机遇的看法，以及她认为在扩大生态系统方面最有前途的领域。

在宇迦实验室之前，穆尼斯成立了一个名为“新事物”的机构，该机构曾与耐克、谷歌、艺电和 Square 等大型品牌合作。

TC Sessions: Crypto 于 11 月 17 日在迈阿密举行。今天就通过早鸟价格和购买您的通行证节省 150 美元，然后加入 web3、DeFi 和 NFT 社区，跟上不断发展和总是令人兴奋的加密世界。
