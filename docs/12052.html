Okta 首席执行官公开谈论 Auth0 收购、SaaS 滑坡和 Lapsus$攻击

Okta 早在 2009 年就推出了一款云身份产品，当时大多数人都被锁定在微软的 Active Directory 上，这是一款根深蒂固的本地产品，没有人相信任何人都可以接触到它。追赶这样一个巨人需要一点勇气，但是 Okta 采取了云优先的方法，这是一个与当时的 Active Directory 明显不同的策略。

在于 2017 年上市之前，该公司筹集了超过 2.3 亿美元的资金。早在 2015 年，随着以 12 亿美元的估值融资 7500 万美元，它成为了独角兽，当时这个称号的意义比现在多一点。

由于拥有劳动力市场的所有权，Okta 决定采取另一个大胆的举措，在 2020 年加速的股市泡沫期间以 65 亿美元收购 Auth0。这笔交易背后的想法不仅仅是拥有一个开发者青睐的身份工具——尽管这肯定是其中的一大部分——它实际上是拥有另一大块市场，可以让 Okta 成为一站式身份商店。

"在这个市场上，传统和现代之间有很深的鸿沟。" Okta 首席执行官托德·麦金农

Okta 既想拥有劳动力市场(这是其实现这一目标的核心)，也想拥有 Auth0 所在的客户身份市场。Okta 为这样规模的公司下了很大的赌注来实现这一目标。

Okta 在身份空间并不孤单；竞争对手包括大大小小的公司，如 ForgeRock、SAP、IBM、Ping Identity、Salesforce、Microsoft 和 Akamai 等。

像所有其他 SaaS 公司一样，Okta 在公开市场度过了艰难的一年，过去一年下跌了 80%以上(尽管周四午盘上涨了近 10%)。它还必须应对由 Lapsus$ 组织发起的一次攻击，该攻击发生在 1 月份，但在 3 月份被报道——以及它的回应所带来的后果。

尽管有这些不利因素，该公司仍有拥有云身份市场的长期目标，并相信它可以安然度过当前暂时的宏观经济条件和传统供应商，以实现这一目标。

最近，我们采访了首席执行官兼联合创始人托德·麦金农(Todd McKinnon ),询问他如何应对这些时代，以及在这一过程中他学到了什么。

增长授权 0

麦金农强调，他当时花了 14%的股票价值收购 Auth0，这是一个他知道的数字，因为他希望他的公司拥有云身份市场，他不认为没有 Auth0 他可以做到这一点。

“我们买它们是为了改变，我们买它们是因为我们需要改变来赢得这个客户身份市场，”他告诉 TechCrunch。“我们的战略是，我们必须赢得劳动力市场和客户身份市场。我们将身份转变为每个公司最重要的平台之一的唯一方法是，我们必须[拥有]这两种用例。”

他说，像这样整合两家公司并非没有挑战，他可能走得太快，没有将产品整合在一起。
