当 cookies 消失的时候，我们的数据会去哪里？

詹姆斯·埃弗里是的创始人兼首席执行官

Kevel

在广告技术行业有超过十年的经验。他还是软件工程和 API 技术方面的专家。

2020 年 1 月，谷歌宣布了消除第三方 cookies 的计划，预示着数字广告和互联网本身的巨大变革。这些饼干的终结预示着数字营销的黄金时代，在这个时代，互联网变得隐私优先。

乍一看，这一更新似乎是朝着正确方向迈出的一步，从许多方面来看，的确如此。不过，这并不是说谷歌的动机是纯粹的。禁止第三方 cookies 让谷歌领先一步，剥夺了竞争对手的权力，进一步巩固了它对数字广告的控制。该公司本质上是在用户至上、注重隐私的更新的幌子下限制对数据的访问，从而阻碍竞争对手。

禁止 cookies 将对许多出版商产生持久的影响，尽管并非所有出版商都会受到同样的影响。依赖于通过第三方广告服务器(如谷歌广告管理器)提供的程序化广告的出版商将受到巨大影响。广告交易商、需求方平台和广告商可以实时访问 cookie 数据，然后使用这些信息来决定对库存出价多少。如果没有第三方数据来增加他们的库存价值，传统出版商可以预计他们的程序化 eCPMs(每千次展示的预期收入)将会下降，从而导致广告收入大幅下降。

那么出版商如何赢回广告收入呢？无论接下来发生什么，数字广告都不会像现在这样简单，出版商将很快被迫重新思考他们的广告策略，并实施新的解决方案来实现广告货币化。

第一方数据将变得非常有价值，出版商必须开始确定如何利用这些数据并从中获利。

从长远来看，统一身份证是不可持续的

出版商需要意识到数字广告的这一巨大转变，并找到替代方案来维持他们的广告收入。一个潜在的途径是统一 ID 解决方案，即出版商以匿名的方式汇集第一方数据，创建一个可以识别整个供应链中用户的 ID。例如，这种“无 cookie”解决方案可以使用匿名电子邮件地址来取代第三方数据。

许多公司已经在建立统一 ID 程序，如 TradeDesk 的统一 ID 2.0 。开源表头竞价平台 Prebid 已经宣布将支持。Unified ID 2.0 只是市场上众多版本中的一个，每个版本在功能、实现和隐私方面都略有不同。

谷歌最近宣布了自己的统一 ID 形式，这有助于使用谷歌广告管理器的出版商。通过出版商提供的标识符(PPIDs) ，出版商可以以匿名的方式与外部投标人共享第一方数据。这似乎是谷歌在保护隐私的同时不疏远广告商和出版商的中间立场。当然，出版商将不得不交出他们的第一方数据，因此谷歌又一次拥有了大量它可能会用到的数据。
