Zealy 是 web3 社区的成就系统| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2023/04/03/zealy-is-an-achievement-system-for-web3-communities/

来认识一下 Zealy ，这是一家法国初创公司，你可能已经知道它的名字 Crew3。Zealy 帮助 web3(和 web2)公司参与他们的社区，给他们任务，他们可以完成这些任务来换取各种奖励。

该公司刚刚更名为 Zealy，这表明了一个超越 web3 公司的更大的关注点。去年，这家初创公司在 Redalpine 牵头的一轮融资中筹集了 350 万美元。其他投资者包括 Connect Ventures、Aglaé Ventures、基马风险投资公司、Purple、STATION F、Founders Future、Pareto Holdings 以及来自沙盒、POAP、DFNS、Starton 和 Pianity 的几位商业天使。

“Zealy 是每个应用程序之上的一个动作层，”联合创始人兼首席执行官 Mathis Grosjean 告诉我。公司将 Zealy 与 Discord 服务器、subreddit 或任何类型的社区中心结合使用，为最热情的社区成员创建游戏化的任务。任务包括创建用户生成的内容，在社交网络上推广某些东西或编写网页代码。

例如，他们将这些任务传递给他们的社区，以便他们可以在产品发布之前、期间或之后完成这些任务。公司可以用它来进行新的 NFT 投放、物理事件或主要产品发布。

Zealy 这样的产品之所以存在，是因为很多公司已经在为他们的社区实现游戏化的任务。但是他们使用像谷歌表单这样的产品，并在一个不和谐的渠道中传递这些任务。这是一个手动过程，Zealy 希望自动化那些用例。

“我们帮助公司上船，教育，娱乐和发展他们的社区，而不需要花费太多的钱，并以可扩展的方式，”Grosjean 说。Fredrika Lindh 和 Louis Demeslay 是该公司的另外两位联合创始人。

但是为什么用户想要完成这些任务呢？狂热的顾客可以为某些任务授予奖励。例如，用户可以在 Discord 服务器上获得特殊状态，获得数字资产、商品等等。

如果这些竞赛运行良好，社区成员可能希望完成尽可能多的任务来爬上排行榜。使用 Zealy 的公司甚至可以利用这些数据来确定其社区中最活跃的用户。

客户可以创建带有排行榜的社区冲刺，排行榜会在几周后重置。在许多方面，Zealy 的工作方式很像视频游戏成就和在线阶梯排名系统。

当谈到为平台带来新用户时，Zealy 客户大多会将他们自己的社区带到服务中。这就是为什么这家初创公司已经拥有相当多的用户:其 70 万月活跃用户迄今已完成 1 亿项任务。

虽然 Zealy 最初专注于 web3 项目，但这家初创公司意识到，更多传统公司可以使用 Zealy 这样的社区工具来改善他们的社区战略。例如，雷诺和 PMU 一直在使用这项服务。总体而言，有 2000 家公司正在使用该平台。

“在目前的市场中，世界上的每一家公司都有可能成为社区主导的公司，”格罗斯让说。Zealy 团队希望他们的初创公司有机会成为社区主导的公司的操作系统。
