意大利下令封锁 ChatGPT，理由是担心数据保护

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/31/chatgpt-blocked-italy/amp/

图片来源:斯特凡尼·雷诺兹/法新社/盖蒂图片社

两天后，一封公开信呼吁暂停开发更强大的生成性人工智能模型，以便监管机构可以赶上ChatGPT之类的人，意大利数据保护局刚刚发布了一个及时的提醒，一些国家已经有适用于前沿人工智能的法律:它已经命令 OpenAI 立即停止在本地处理人们的数据。

意大利 DPA 表示，它担心 ChatGPT 制造商违反了欧盟的一般数据保护条例(GDPR)，并开始调查。

具体来说， Garante 表示，由于担心 OpenAI 非法处理人们的数据，以及缺乏任何防止未成年人访问该技术的系统，它已经发布了阻止 ChatGPT 的命令。

这家总部位于旧金山的公司有 20 天的时间对该命令做出回应，如果不遵守，将受到一些巨额罚款的威胁。(提醒:违反欧盟数据保护制度的罚款最高可达年营业额的 4%，或 2000 万€，以金额较大者为准。)

值得注意的是，由于 OpenAI 没有在欧盟建立法律实体，根据 GDPR，任何数据保护机构都有权干预，如果它认为当地用户面临风险。(因此，意大利介入的地方，其他国家可能会效仿。)

更新: OpenAI 现已在意大利对 ChatGPT 进行地理封锁——详情请见我们的后续报道。

一套 GDPR 问题

每当处理欧盟用户的个人数据时，都会应用 GDPR。很明显，OpenAI 的大型语言模型一直在处理这种信息，因为它可以，例如，按需制作该地区命名个人的传记(我们知道；我们试过了)。尽管 OpenAI 拒绝提供用于该技术最新迭代 GPT-4 的训练数据的细节，但它透露，早期的模型是根据从互联网上收集的数据进行训练的，包括 Reddit 等论坛。所以，如果你经常上网，这个机器人很可能知道你的名字。

此外，ChatGPT 已经被证明产生了关于指定个人的完全错误的信息，显然是在弥补其训练数据所缺乏的细节。这可能会进一步引发 GDPR 方面的担忧，因为该规定为欧洲人提供了一系列对其数据的权利，包括纠正错误的权利。例如，还不清楚人们如何/是否可以要求 OpenAI 纠正机器人产生的关于他们的错误言论。

Garante 的声明还强调了该服务在本月早些时候遭受的数据泄露，当时 OpenAI 承认一个对话历史功能一直在泄露用户的聊天记录，并表示这可能暴露了一些用户的支付信息。

数据泄露是 GDPR 监管的另一个领域，重点是确保处理个人数据的实体充分保护信息。这项泛欧盟法律还要求公司在很短的时间内将重大违规行为通知相关监管机构。

最重要的是一个大问题，OpenAI 最初处理欧洲人数据的法律依据是什么。换句话说，这个过程的合法性。

GDPR 允许多种可能性——从同意到公共利益——但训练这些大型语言模型的处理规模使合法性问题变得复杂。正如 Garante 指出的(指向“个人数据的大规模收集和存储”)，数据最小化是该法规的另一大焦点，其中也包含要求透明和公平的原则。然而，至少，ChatGPT 背后的(现在的)盈利公司似乎没有告知人们它重新利用了谁的数据来训练其商业人工智能。这对 it 部门来说可能是一个相当棘手的问题。

如果 OpenAI 非法处理了欧洲人的数据，欧盟各国的 DPA 可以下令删除这些数据，尽管这是否会迫使该公司重新训练根据非法获得的数据训练的模型是一个悬而未决的问题，因为现有的法律正在努力应对尖端技术。

另一方面，意大利可能刚刚禁止了所有的机器学习，呃，意外…

“隐私担保人指出，OpenAI 收集的数据缺乏向用户和所有利益相关方提供的信息，但最重要的是，缺乏法律基础来证明大规模收集和存储个人数据是合理的，目的是‘训练’平台操作的算法，”DPA 在今天的声明中写道(我们使用 AI 从意大利语翻译过来)。

“正如所进行的检查所证明的那样，ChatGPT 提供的信息并不总是与真实数据相符，因此确定对个人数据的处理不准确，”它补充说。

该机构补充说，它担心 OpenAI 处理未成年人数据的风险，因为该公司并没有积极阻止 13 岁以下的人注册使用聊天机器人，例如通过应用年龄验证技术。

儿童数据的风险是监管机构非常活跃的一个领域，最近因儿童安全问题下令对虚拟友谊 AI 聊天机器人 Replika 实施类似禁令。近年来，它还就未成年人使用问题起诉抖音，迫使该公司清理了超过 50 万个无法确认是否不属于儿童的账户。

因此，如果 OpenAI 不能明确确认它在意大利注册的任何用户的年龄，它至少可以被迫删除他们的账户，并重新开始一个更强大的注册过程。

OpenAI 被联系，要求对 Garante 的订单做出回应。

纽卡斯尔大学(Newcastle University)数据保护和互联网法律专家莉莲·爱德华兹(Lilian Edwards)在研究记住的“算法的影响方面一直处于领先地位，她告诉 TechCrunch:“有趣的是，它或多或少地复制粘贴了 Replika 在强调儿童访问不适当内容方面的做法。但真正的定时炸弹是否认合法基础，这应该适用于所有或至少许多机器学习系统，而不仅仅是生成性人工智能。”

她指出了涉及谷歌搜索的关键的“被遗忘权”案例，在该案例中，一名西班牙人对谷歌未经同意处理个人数据的做法提出了质疑。虽然欧洲法院规定个人有权要求搜索引擎删除关于他们的不准确或过时的信息(与公共利益测试相平衡)，但谷歌在这种情况下对个人数据的处理(互联网搜索)并没有因为 processing point 的合法性而被欧盟监管机构否决，似乎是因为它提供了一项公共事业。但最终也是因为谷歌最终向欧盟数据主体提供了删除和更正的权利。

“大型语言模型不提供这些补救措施，也不完全清楚它们会不会、会不会或者会有什么后果，” Edwards 补充道，暗示对模型进行强制再训练可能是一个潜在的解决办法。

或者，像 ChatGPT 这样的技术可能只是违反了数据保护法…
