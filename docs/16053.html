Ledger 推出浏览器扩展以改善加密钱包连接| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/16/ledger-launches-browser-wallet-extension-to-improve-crypto-connectivity/

最大的冷存储加密钱包提供商之一 Ledger 向 TechCrunch 独家透露，该公司推出了一款浏览器扩展，以提高数字资产的在线安全性和连接性。

“你认为 Web 1.0 是用户名和密码，Web 2.0 是用脸书、谷歌、iCloud、Twitter 登录，web3 是(关于)连接到你的钱包，”Ledger 的首席体验官 Ian Rogers 说。“但这种连接体验仍然不是超级微不足道的，这就是我们在这里讨论的，并解决 web3 中连接的主要可用性问题。”

该公司分享说，Ledger 扩展与基于以太坊和多边形的 dApps 和平台兼容，并计划在未来支持更多的 EVM 兼容链和 Solana。它目前只能在 Safari、iOS 和 MacOS 上使用，但将在“发布后不久”推出对 Windows、Chrome 和基于 Chrome 的浏览器如 Brave、Opera 和 Edge 的额外支持

它目前可以通过一些 dApps 和平台使用，如 NFT 市场 OpenSea 和分散式交易所 PancakeSwap 和 Curve 等。

该扩展旨在通过 MetaMask 等浏览器扩展提供类似于加密钱包的体验，但它不是一个热钱包——它连接到互联网，更容易受到在线攻击——而是一个冷钱包，允许用户通过蓝牙直接与 dApps 交互。

“很多人说 web3 很难用，当你具体问它有多难用时，他们耸耸肩，但我认为是 onboarding 和连接性，”罗杰斯说。“这是为了解决这个问题。”

它有两个功能，旨在确保用户在与 crypto 交互时的安全:它分析智能合同，并警告用户交易是否具有潜在的恶意，消费者工程副总裁卡尔·安德森分享道。它还将模拟一次交易，以显示它将如何影响钱包，即使这是一次安全交易。“本质上，我们在这里做的是向用户展示如何保持他们的密钥安全，”安德森补充说。

Ledger 是一家成立九年的加密硬件钱包和冷钱包提供商，已经在 200 个国家和 100 多个金融机构和品牌中销售了 600 多万台设备。该公司表示，全球约有 20%的加密资产通过分类账进行保护。

“我们正在我们已经拥有的硬件安全的基础上进行建设，”安德森说。"分类帐扩展是为了易于使用和带来安全性."

去年 12 月，该公司与 iPod 的设计者托尼·法德尔(Tony Fadell)合作，为用户创造了一种更简单、更容易使用的方式，通过他们的 Ledger Stax 产品来保护他们的密码。2022 年早些时候，莱杰与价值 15 亿美元的风险投资公司国泰创新(Cathay Innovation)合作推出了一只价值 1.1 亿美元的 T2 基金(T3)，专门用于保护加密资产。2021 年，在 10T Holdings 牵头的一轮融资中，该公司为 T4 筹集了 3.8 亿美元，当时莱杰的估值为 15 亿美元。

罗杰斯说:“对我们来说，我们的目标是让加密数字资产和钱包连接应用的世界从托尼·法德尔所说的‘商业到极客’到‘商业到消费者’。“那里有足够的土地可以覆盖。但这一点也不困扰我。”

罗杰斯有一个数字音乐的背景，并比较了密码空间与早期的数字音乐。“一开始只有最专业的人才有 MP3，现在，很明显，音乐触手可及。”

罗杰斯指出，这些向新技术的转变总是从非常困难的可用性和有限的消费者选择开始。“但随着时间的推移，它们会不断发展，并接触到大量受众。数字音乐之路是类似的，因为它最初只是极客们感兴趣的东西，最终我们让 iPod、iPhone 和 Spotify 这样的服务成为主流。”

罗杰斯和安德森分享道，展望未来，莱杰计划专注于提高加密领域的连接性和安全性，让消费者和企业更容易接触该领域的 dApps 和平台。

罗杰斯说:“如果你看看这些拼图块，你会发现它们明天会更好，比昨天更好，在 dApps 中被广泛采用，并确保我们涵盖了各种平台。”。“总会有更多的地方需要覆盖，但碎片还是存在的。”

安德森认为，快进到 2030 年，这个行业将不会谈论 Web 2.0 和 web3。“它将只是网络。对我们来说，这是关于给它带来安全，因为区块链是不可改变的。我们希望缩小两者之间的差距，以便于使用。”
