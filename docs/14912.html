选择 Star 完成 1500 万美元的交易，为不同的数据添加背景| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2023/01/31/select-star-closes-15m-round-to-add-context-to-disparate-data/amp/

Select Star ，一家为大多数企业组织提供数据发现、血统和治理工具的初创公司，今天宣布在由光速风险投资公司领投的 A 轮融资中筹集了 1500 万美元，参与方有 Bowery Capital、Sozo Ventures 和 Pebblebed。这笔新资金使该公司的融资总额达到 2000 万美元，首席执行官 Shinji Kim 告诉我，这笔资金将用于产品工程、上市和客户成功方面的努力。

Kim 在接受 TechCrunch 的电子邮件采访时表示:“数据和应用的爆炸式增长使得确保良好的数据治理，同时允许数据团队以自助方式利用所有适当的数据变得非常困难。”。“传统工具用于企业数据目录和数据治理，但它们需要大量手动配置和培训才能在组织内充分利用，这正是 Select Star 的用武之地。”

Kim 之前是数据处理提供商 Concord Systems 的首席执行官，该公司于 2016 年底被 Akamai 收购。她加入了 Akamai，并开发了该公司的物联网数据平台，用于实时信息传递，在那里，她注意到了一种不幸的模式。根据 Kim 的说法，许多 Akamai 的企业客户在迁移到云数据库基础设施时遇到了理解和使用所有数据的问题，通常是因为数据的上下文丢失了。

对于“上下文”，Kim 指的是数据在服务、应用和工具内部和之间流动和使用的汇总视图，这正是 Select Star 提供的。该平台分析来自 Snowflake、Google BigQuery、Amazon Redshift、Databricks、Tableau、Looker 和其他流行平台的元数据和日志，然后尝试自动标记事件并将其提取为要点。

Select Star 还创建持续更新的关键数据资产、模型和业务指标清单。该平台为跨组织和团队的所有数据提供了一个记录系统，试图揭示最重要的数据库级关系。

图片来源:选择明星

“我们自动生成客户的数据模型、数据谱系、文档，检测个人身份信息等，因此在设置上只需最少的工作，您就可以立即获得洞察力，”金说。我们的列级沿袭模型深入了解端到端数据流，这有助于数据团队避免数据模型和仪表板损坏。"

Kim 断言，这种环境可以帮助企业变得更加数据驱动——特别是那些定期处理多个大型异构数据集的企业。例如，企业可能希望将销售数据、营销数据和产品使用数据放在一起，以获得客户的完整资料。收集和规范化所有这些信息不仅从技术角度来看是一个挑战，而且因为不同的部门和人员可能以不同的方式解释数据。

金认为，出于这样或那样的原因，尽管尽了最大努力，但很少有公司认为自己是真正的数据驱动型。在 2021 年 NewVantage 的一项调查中，只有 30% 报告称拥有清晰的数据策略。埃森哲最近的一份独立的报告发现，只有 33%的企业信任“他们的数据足以有效地使用它并从中获取价值”在 Arm Treasure Data 进行的 2019 年民意调查中，47%的受访者表示他们公司的数据“孤立且难以访问”

“拥有像 Select Star 这样的自动化数据发现平台可以作为每个人都可以参考的单一事实来源，”Kim 说。“这有助于消除任何混淆，培养对数据的信任，并鼓励更有效地完成更多分析工作。”

如果我没有提到其他初创公司也在解决同样的问题，那我就失职了，在某些情况下，他们从明显不同的角度出发。我想到了 Castor——它正在构建数据目录，或者元数据、数据管理和搜索工具的集合，旨在帮助用户在组织内找到他们需要的数据。另外还有 Alation ，数据。World 和 Collibra ，它们同样提供数据搜索和数据谱系追踪工具，以及数据观察公司，如 Manta 和解开数据。

Kim 表示相信 Select Star 可以竞争——并且正在竞争——非常好，尽管他指出客户群包括 Square 母公司 Block、Pitney Bowes、Fivetran、Opendoor 和 Handshake。她拒绝对收入指标发表评论，但表示计划在年底前将 Star 的 20 人员工队伍扩大一倍，这表明她对未来持乐观态度。
