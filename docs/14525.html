苹果电视和苹果音乐应用悄然出现在微软商店

原文：https://web.archive.org/web/https://techcrunch.com/2023/01/12/apple-tv-apple-music-apps-windows-microsoft-store/

苹果电视和苹果音乐应用悄然出现在微软商店

苹果终于将其 Apple Music 和 Apple TV 应用程序引入 Windows，这两款应用程序的预览版已经悄然出现在微软商店中。在微软宣布这些应用将于今年登陆 Windows 11 的几个月后，这款应用就推出了。这次发射首先被核查人员的发现。

这些应用程序看起来与 macOS 上的版本相似，但在 Windows 上略有改动。Apple TV 应用程序的功能与智能电视上的应用程序类似，用户可以访问 Apple TV+和 Apple TV 频道，以及 iTunes Store 中的电影和电视节目。苹果音乐应用程序与 macOS 版本几乎相同，但没有歌词功能。

还有一个名为“苹果设备”的预览应用，让用户从他们的 Window PC 上管理苹果设备。该应用程序取代了 iTunes 进行同步和备份，还允许用户在不需要 iTunes 的情况下恢复固件。

这三个新的苹果应用都需要 Windows 11 版本 22621.0 或更高版本。苹果指出，安装任何这些应用程序都会阻止 iTunes for Windows 打开。用户必须卸载这些应用程序才能继续使用 iTunes。

10 月，微软宣布将苹果的 iCloud 存储服务与 Windows 11 中的照片应用整合。从微软商店安装 iCloud for Windows 应用程序并选择同步 iCloud 后，Windows 设备的 iPhone 用户将能够在照片中查看他们的 iPhone 照片和视频。
