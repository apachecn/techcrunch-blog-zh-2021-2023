微软押注藻类减少其不断增长的碳足迹

像科技行业的所有同行一样，微软也面临碳污染问题。

尽管该公司承诺到 2030 年成为碳负的 T2，但这个软件巨头的排放量仍在上升。这个滴答作响的时钟解释了微软解决其环境代价的最新交易:它转向运行潮汐以抵消其通过海洋的一些排放。

Running Tide ，也与 Stripe 和 Shopify 合作，旨在用这笔钱锁住大量的二氧化碳。Running Tide 表示，它将通过在可生物降解的浮标上种植大量海藻等努力来实现这一目标，旨在让海藻最终沉入海底。创业公司有一份关于其工作的白皮书，但如果你想了解更多细节，以下是业务发展负责人 Jordan Breighner 今天告诉 TechCrunch 的内容:

“我们将木材和碱性矿物质结合起来，形成一个小型碳浮标，我们可以用藻类种子播种，并部署到公海深处，”Breighner 说。“浮标漂浮，碱性矿物质溶解，减少海洋酸化，并通过一种称为海洋碱度增强的过程去除碳。海藻迅速生长，吸收二氧化碳。不到三个月后，浮标、藻类和内含的快速碳沉入海底，如果它们沉入 1000 米以下，碳就会消失大约 1000 年。”

“然而，并不是所有的浮标都是播种的，”Breighner 补充道。"这是基于最适合藻类生长的海洋条件."

整体而言，除碳业务仍处于发展初期。它还没有证明它能以它最终渴望达到的规模持久地减少碳排放。《麻省理工科技评论》去年报道，一些科学家还担心，完全成熟的、风险投资支持的封存计划，如大型海藻农场，可能会无意中损害海洋生态系统。

到目前为止，Breighner 表示，Running Tide“在测试和研究部署中仅去除了不到 1000 吨的碳。”这家初创公司打算在两年内仅微软一家就清除 12，000 吨垃圾。

Running Tide 表示，这笔交易的价值为个位数百万美元。微软发言人拒绝对价格发表评论。

微软最近的可持续发展报告显示，从 2020 年到 2021 年，T2 的排放量增加了 21.5%。软件巨头将此归因于范围 3 的排放，称这与数据中心的发展和更多客户更频繁地使用其产品有关。换句话说，微软增长了其云和游戏业务，其净排放量也同步上升。公司的目标是在未来七年左右的时间里成为碳负，它实现这一目标的计划取决于碳的去除。
