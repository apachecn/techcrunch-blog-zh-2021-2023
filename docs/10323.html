Block Party，一个打击网上骚扰的工具，筹集了 480 万美元的种子资金

原文：https://web.archive.org/web/https://techcrunch.com/2022/09/21/block-party-a-tool-to-combat-online-harassment-raises-a-4-8m-seed/

如果你在过去几年里在 Twitter 上经历过任何形式的骚扰，那么你可能听说过街区派对。如果你没有，你真幸运！

由软件工程师和技术多样性倡导者 Tracy Chou 开发的 Block Party 帮助人们在 Twitter 上策划更安全的体验。通过建立在 Twitter 的 API 之上，Block Party 允许用户在 Twitter 上自动化阻止不良行为者和创建阻止列表的过程。例如，如果一条特定的推文滋生了骚扰，那么你可以自动屏蔽任何喜欢或转发该帖子的人。

Block Party 于 2021 年初推出了，但现在，该公司希望借助 480 万美元的种子资金扩展到其他社交平台。

“我们已经进行了一系列初步验证，这个产品对人们有用吗？有可能在平台之上构建吗？”周告诉 TechCrunch。“会有人愿意为此付费吗，比如这里的市场有多大？”

到目前为止，这个概念已经证明了自己。Chou 说，Block Party 吸引了各种各样的用户，包括公众人物和只是想要更多隐私控制的小账户。至于建立在 Twitter 的 API 去，周说，Twitter 主动接触，以阻止党，以确保他们可以协同工作。

“看到他们理解这种战略联盟真的很酷，让 Block Party 这样的公司生产安全工具是一种双赢的局面，”Chou 告诉 TechCrunch。“这对最终用户有好处，因为他们有更好的工具来更好地控制他们的体验。”这对平台也有好处，因为他们不必为此投入工程和产品设计资源。

图片来源:街区派对

街区派对有免费和付费两种服务。作为一名非付费用户，你可以使用“适度过滤”这样的工具，它可以屏蔽更有可能是骚扰的账户，比如关注者很少、没有个人资料图片的新创建账户。付费用户可以使用关键词过滤器等功能，允许用户在他们的用户名、显示名称或简历中使用特定的关键词或表情符号来自动阻止或屏蔽他们提及的账户。另一个付费功能“助手视图”允许用户指定一个朋友来帮助管理他们的帐户设置。这个工具可以让一个朋友帮助你管理过滤器、黑名单和观察名单，而不是你自己筛选有害的推文。

Chou 将 Block Party 描述为“中间件”，这个术语反映了用户如何通过个性化的第三方应用程序来更多地控制他们在社交媒体上的体验。

“对于平台来说，围绕着定义什么是可接受的或不可接受的，进入这些信息有时真的没有意义。他们将能够定义服务条款，”周解释道。“但中间会有一堆东西，个人用户可能不想看到这些内容，但 Twitter 没有理由禁止一个用户说我丑陋或愚蠢。”

图片来源:街区派对

用户可以转向 Block Party 这样的“中间件”,作为个人自己决定他们想看哪些内容，而不是被束缚在什么样的不良行为是应受惩罚的罪行的意识形态辩论中。正如周所说，称某人丑陋并不违反 Twitter 规则，但这并不意味着你应该在你的提及中看到这一点。

理论上，Twitter 可以自己构建这些过滤选项。但周认为，社交媒体平台根本没有动力将安全置于纯粹的使用之上。

“我认为观察结构性激励很能说明问题。平台被激励产生更多的参与——这是他们的北极星，”周说。“所以这就是他们将投入所有资源的目标。”

举例来说，现在，当 Twitter 面临与埃隆·马斯克的法律战时，440 亿美元危在旦夕，内部优先事项已经转移。The Verge 报道【Twitter 的健康团队被重组为一个负责识别垃圾账户的团队，因为马斯克已经将虚假账户作为他终止收购该平台的论点的一个关键方面。因此，如果 Twitter 不能像它所希望的那样为安全投入尽可能多的资源，至少 Block Party 可以。

随着 Block Party 计划扩展到其他社交平台，周和她的四名全职员工团队已经策划了一份战略投资者名单，可以帮助公司更上一层楼。一些投资者包括 Twitter 的机器学习伦理总监 Rumman Chowdhury 博士；前 Twitter 产品负责人杰夫·塞伯特；Pinterest 联合创始人埃文·夏普；前 Instagram 增长负责人 Bangaly Kaba 和其他经验丰富的社交媒体高管。本轮由 Stellation Capital 领投，参与方包括 imperive Ventures、Fuel Capital、Goodwater Capital 和 Hyphen Capital。
