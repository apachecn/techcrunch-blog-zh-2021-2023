Cubzh 希望打造下一代《我的世界》TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/12/06/cubzh-wants-to-build-the-next-generation-minecraft/

认识一下 Cubzh ，这是一款新的免费视频游戏，它完全是通过立方体系统由用户生成的内容。该公司从零开始设计了一个新的平台，并希望通过脚本环境赋予创作者创建新对象、构建世界和定义游戏规则的能力。

今年夏天，该公司筹集了 300 万美元的种子资金，由 New Wave 牵头，几位天使投资人也参与了进来，如 Docker 创始人所罗门·海克斯(Solomon Hykes)和 Sorare 首席执行官尼古拉斯·朱莉娅(Nicolas Julia)。

“这是一台自制的发动机。这就是为什么我们花了很长时间在隐形模式下，以确保它的工作，”联合创始人兼首席执行官 Adrien Duermaë告诉我。

这个 C/C++引擎可以在 PC、macOS、iOS、Android 和 web 浏览器上运行。换句话说，如果你正在阅读这篇文章，很可能你有一个可以运行 Cubzh 的设备。

你将在 Cubzh 玩什么将取决于其他用户。本质上，Cubzh 是一个游戏平台，可以提供不同的游戏体验——有点像 Roblox。

“不会写剧本的玩家将可以创造物品，绘制剑和载具。很快，我们将增加一个动画编辑器。那些会编码的人将能够在他们的游戏中加入这些对象，”杜尔曼说。

至于游戏开发，Cubzh 没有使用无代码方法。目前一切都基于 Lua 脚本。Cubzh 负责基本的东西，比如碰撞和多人游戏代码。剩下的就看玩家的想象力了。

现在，Cubzh 可以作为 alpha 测试使用。该公司正在考虑将 web3 机制作为一种盈利途径。例如，Cubzh 可以在物品的一级和二级销售中提成。内容创作者将根据使用情况从他们的创作中获得版税。

但该公司尚未推出市场。它也不知道是否会有实用程序令牌。现在基本特性已经在这里了，公司将把重点放在所有这些上层特性上。

最终，Cubzh 希望它能创造出一种数字版的乐高。也许一些《我的世界》的创作者会发现 Cubzh 更有趣，并转移到这个新的游戏平台。
