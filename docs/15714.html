通过 Project Clover，抖音吹捧新的欧盟数据隐私和安全措施

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/08/with-project-clover-tiktok-touts-new-eu-data-privacy-and-security-efforts/amp/

图片来源:丘克鲁特·布德鲁/SOPA 图片/光火箭/盖蒂图片

抖音今天加倍展开欧洲魅力攻势，以应对日益高涨的对广受欢迎的短片视频托管平台的政治不满。

抖音负责政府关系和公共政策的欧洲副总裁 Theo Bertram 在一篇博客文章中写道，一个名为 Project Clover 的新项目将为“欧洲抖音用户数据创造一个安全的飞地”。

抖音声称在全球拥有超过 10 亿用户，由于其与中国科技公司字节跳动的关系，多年来一直处于监管的聚光灯下。事实上，抖音最近的轨迹表明，一些限制可能正在进行中，一群美国参议员本周公布了两党立法，该立法可能允许政府限制或禁止基于外国的技术，如抖音，如果被视为国家安全威胁。

在其他地方，在美国众议院发布类似禁令后不久，欧盟委员会上个月命令员工将抖音从工作设备中移除。

正是在这种背景下，抖音现在正寻求通过一系列针对该地区的承诺来讨好欧洲监管机构，并根据欧洲即将到来的数字服务法案(DSA)解决对其数据利用做法的担忧。

数据主权

通过 Project Clover，抖音基本上将一些之前宣布的计划与一些新的隐私和安全措施捆绑在一起。我们已经知道，抖音计划在欧洲本地数据中心方面进行一些重大的基础设施投资。第一个数据中心本应于去年在爱尔兰开业，但一再推迟，而该公司最近宣布计划在该地区再建两个数据中心。我们现在知道他们将在哪里——一个也将在爱尔兰，而第三个将部署在挪威。据称，挪威数据中心将完全依靠可再生能源运行

抖音表示，将数据迁移到欧洲服务器的过程应该最终在今年开始，并持续到 2024 年，这对于满足欧盟监管机构至关重要，而且在此之前不久，有消息称中国的员工可以访问欧洲用户的数据。

考虑到这一点，Project Clover 显然还将引入新的数据访问和控制流程，包括决定哪些员工可以访问欧洲抖音用户数据的“安全网关”。但也许更重要的是，抖音表示，它将聘请欧洲的一家独立安全公司审计其数据控制和做法。

“我们正在与第三方进行讨论，并将在适当的时候宣布更多细节，”伯特伦说。

除此之外，Bertram 还指出，抖音打算与其他第三方合作，将“最新的先进技术”集成到其现有系统中。这包括所谓的个人数据“假名化”，本质上使得在数据泄露事件中识别个人用户更加困难。

Bertram 指出:“自去年以来，一个专门的内部团队一直在致力于 Project Clover，我们预计将在今年和 2024 年实施这些新颖的行业领先措施。

尽管抖音可能因为其母公司位于中国而面临更严格的审查，但它今天的声明与其他地方大型科技公司的努力大致一致。数据主权最终是游戏的名称，希望在欧洲做生意的公司应该将数据保存在本地，并采取措施确保消费者和企业确切知道他们的数据正在发生什么。

去年，微软推出了面向公共部门客户的微软主权云，同时最近也启动了一项为期多年的新欧盟数据本地化计划。在其他地方，谷歌和亚马逊 AWS 等公司也一直在兜售他们的数字主权凭证，欧洲通常是主要推动力。

“Project Clover 加强了我们对欧洲数据治理方法的承诺，这种方法将保护用户数据置于其核心位置，并符合数据主权原则，”Bertram 说。
