脸书向 2D 渐进式网络应用开放 Oculus 商店

脸书希望将其 Quest 2 耳机打造成新一代远程工作者的生产力设备，但为了实现这一目标，它需要应用程序。

今天，脸书宣布他们将首次把 2D 应用程序带入 Oculus 商店。新的应用程序基于渐进式网络应用程序(PWA)行业标准，将允许用户检查他们以前需要摘下耳机才能在手机上看到的应用程序。

截至今天，Oculus 商店中有少量 2D 应用程序可用，包括脸书、Instagram、Smartsheet 和 Spike。该公司表示，其他应用程序包括 Dropbox、Monday.com、壁画、My5(英国)、Pluto TV 和 Slack。

虽然 Oculus 商店已经看到少数应用程序制造商提供原生虚拟现实支持，但维护一个专用的虚拟现实应用程序在多个方面都很复杂，并导致网飞和 Hulu 等公司的应用程序很少更新。依靠 PWAs 将使开发人员的负担明显减轻，并有望比 Oculus 通常从应用程序制造商那里看到的更广泛的采用。

2D 应用程序的加入是将这种探索带入工作场所的更广泛努力的一部分。今年 8 月，该公司发布了一款名为 Horizon Workrooms 的应用，允许员工将桌面带入虚拟现实，并在共享的社交虚拟现实空间中进行交流。
