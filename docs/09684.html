Alloy 依靠欺诈防范获得 15.5 亿美元的新估值

原文：https://web.archive.org/web/https://techcrunch.com/2022/09/01/fintech-alloy-fight-fraud/

【Alloy 成立于 2015 年，其使命是利用其单一 API 服务和 SaaS 产品，帮助银行和金融科技公司做出更好的身份和风险决策。

从那时起，这家初创公司已经发展到不仅自动化入职身份决策，还自动化交易监控和信贷承保。

今天， Alloy 宣布，在以 13.5 亿美元的估值融资 1 亿美元11 个月后，它又以 15.5 亿美元的估值融资 5200 万美元。初创公司 在这样一个充满挑战的融资环境中成功筹集到这么多资金，这一事实令人印象深刻，但考虑到如今许多公司要么在艰难地筹集资金，要么在持平甚至下跌的情况下筹集资金，它还提高了估值，这一事实值得注意。

Alloy 联合创始人兼首席执行官汤米·尼古拉斯在接受 TechCrunch 采访时指出，对身份工具的需求不断增长，这些工具可以帮助金融机构获得更多“好”客户，淘汰“坏”客户，这使得 Alloy 的年度经常性收入(ARR)在过去一年翻了一番。

简而言之，Alloy 的使命是帮助银行和金融科技公司打击欺诈并保持合规，同时接纳美国和海外的新客户。它帮助客户通过单一集成点获取客户信息、传统征信机构数据和其他替代数据。

本月初，该公司宣布将其 全球扩张到 40 个国家 横跨北美、EMEA、拉丁美洲和 APAC。

这家总部位于纽约的初创公司拥有 300 多家客户，包括 Ally Bank、HMBradley、Gemini、Ramp and Evolve Bank & Trust、Brex 和 Petal，这些客户使用其基于 API 的产品连接到 160 多个数据源，在创建新账户时自动做出身份决定，并持续监控它们。Alloy 声称每天要处理超过一百万个决策。当然，最终目标是帮助其客户开发安全的金融科技产品，并帮助他们扩大客户群。

尼古拉斯说，随着时间的推移，欺诈威胁已经发展到了“专业欺诈品牌”试图使用偷来的和合成的身份开设账户，转移和窃取资金的地步。

他补充道，越来越多的组织和个人利用社交媒体欺骗人们进行欺诈。

“你可以想到火绒骗子类型的事情，在那里它是大规模组织的，”尼古拉斯说。“这真的成为一个越来越大的问题。”

在 C 轮融资中筹集 1 亿美元资金,“仍在银行”

对于公司来说，筹集到的资金只有上一次融资的近一半，这有点不寻常。但据 Nicholas 说，对 Alloy 来说，这个决定是有意的，也是战略性的。这是在它的 1 亿美元 C 轮资金“仍然存在银行”的情况下做出的。

“我们环顾四周，说好吧，世界已经以这些方式改变了。我们面前有一个巨大的机会。董事会正在以不同的方式做出投资决策，”他告诉 TechCrunch。“我们如何确保我们仍然能够执行我们需要执行的计划，并在需要时继续进攻？”

Nicholas 补充道:“此外，对于我们的客户来说，欺诈正在快速变化。我们已经走向全球，我们做的事情比以往任何时候都多。我们知道机会会出现在我们需要进行研发投资的地方。”

光速创投(Lightspeed Venture Partners)和 Avenir 成长(Bernstein Growth)共同主导了 Alloy 的最新融资，现有支持者 Canapi Ventures、Bessemer Venture Partners、Avid Ventures 和 Felicis Ventures 也参与了融资。

light speed 的合伙人贾斯汀·奥弗多夫(Justin Overdorff)在 Alloy 上押了双倍赌注(他的公司也领导了这家初创公司 2021 年 9 月的 C 轮融资)，因为他看到了“该公司的作用不仅在于帮助公司更快地将金融产品推向市场，而不会增加欺诈或合规风险，还在于帮助公司安全地扩大客户群。”

他在电子邮件中写道:“因此，作为投资者，我们看到了公司本身的巨大潜力，但也看到了它能做些什么来推动整个生态系统。”。

作为 Stripe 的前雇员和目前的金融科技投资者，奥多夫认为，许多人不理解的是这个领域的相关风险。

他补充说:“构建金融产品本身就有风险——因为有规则和法规来保护人们的资金安全(理应如此)，也因为有坏人试图利用任何漏洞。”。

尼古拉斯称，Alloy 计划利用其资本继续改善对现有市场的服务，“为全球公司解决全球问题”，并扩大其产品范围。它还想继续招聘。目前，这家初创公司有 290 名员工。

在 Alloy 的最后一次融资时，早期投资者布拉德·斯维尔卢加(Brad Svrluga)是 Primary Venture Partners 、的普通合伙人，他总结了该公司在充满挑战的环境中的崛起:“当汤米·尼古拉斯、劳拉·斯皮克曼和查尔斯·赫恩在 2015 年创办该公司时，他们正在逆流而上。作为一家初创公司，向保守的金融机构销售新技术绝非易事。但在过去几年里，Alloy 帮助引领了对颠覆性金融科技和合作伙伴关系的信任度的转变。”
