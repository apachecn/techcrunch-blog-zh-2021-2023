，参与了 50 多项 NFT 和加密货币交易，并且是分散式金融基础设施和技术公司 Liquid Meta 的董事会成员。

关于 NFT的讨论可能充斥着董事会会议室和新闻报道，但它们复杂而新颖的性质让投资者很难判断哪些项目有前途。事实上，根据 Chainalysis 的一项研究，只有一小部分投资者从非金融交易中获得了最多的利润。

我已经参与了 50 多项 NFT 和加密货币交易，并致力于扩大 DAO(去中心化自治组织)生态系统。然而，对 NFT 空间的不熟悉是许多投资者害怕涉足的原因。

NFT 不仅仅是著名的艺术品、歌曲和推特——它们还是更广泛的去中心化运动的一部分。从版权执法到购买不动产和身份验证，NFT 在远程数字世界中发挥着重要作用。2021 年上半年，NFT 市值增长了 2，100%，销售额达到 25 亿美元。与此同时，创造者经济的繁荣为非功能性交易打开了更多的大门，因为人们不必通过聚合者或中介来创造代币。

如果你与一位不深究商业模式、技术和竞争对手细节的创始人交谈，那就把它视为危险信号。

投资者需要了解 NFTs 的基础知识和潜力，但不需要很深的技术知识。这是因为任何 NFT 项目的真正价值在于建造它的人。当他们面临不可避免的动荡时刻时，他们将维持有前途的 NFT 项目。

以下是如何通过 NFT 项目的创建者对其进行最终检验的方法:

检查创始人和技术是否都开放

第一辆 NFT 于 2014 年制造，去年才售出。NFT 市场仍处于起步阶段，投资者不应指望 NFT 的项目会经历与其他科技项目相同的审查过程。没有足够的数据点可用，也没有工具来跟踪 NFT 的表现。相反，投资者应该寻求项目领导层和技术基础设施的透明度。与其说是评估目的地，不如说是相信有一扇观察旅程的窗户。
