原文：https://web.archive.org/web/https://techcrunch.com/2023/01/03/you-must-now-verify-your-drivers-license-to-watch-pornhub-in-louisiana/

今年年初，路易斯安那州一项新法律生效，该法律要求在线出版商进行年龄验证检查，如果他们网站的内容超过 33.3%的色情内容。

根据这项法律，Pornhub 现在要求游客通过路易斯安那州驾照的数字钱包 LA Wallet app 验证他们的年龄。根据主板的消息，其他流行的成人网站还没有设立年龄检查。但是根据新的法律，色情网站可以被起诉，要求对未成年人访问这些内容造成的损害进行赔偿。TechCrunch 联系 Pornhub 寻求置评。

“由于技术的进步，互联网的普及和有限的年龄验证要求，未成年人在更小的年龄就接触到色情制品，”立法说。“色情作品导致青少年和青春期前儿童的过度性化，并可能导致自卑、身体形象障碍、年轻时有问题的性活动增加，以及青少年参与危险性行为的欲望增加。”

这项立法是由众议员劳里·施莱格尔(R-LA)起草的，他也主张立法规定变性青少年根据他们的性别身份参加学校体育活动是非法的。在 2021 年就职之前，施莱格尔是一名性瘾顾问。

加州大学洛杉矶分校批判性网络调查中心的性工作者、教授和研究员奥利维亚·斯诺将这些年龄验证检查描述为“性恐慌”的一部分尽管这项立法规定进行年龄验证的网站不能保留身份信息，但 Snow 认为色情消费者可能仍然担心数据泄露，这是一个非常合理的担忧。

像 Pornhub 和 OnlyFans 这样的网站要求表演者证明他们的年龄和身份，作为打击非同意内容和儿童性虐待材料的一种方式(CSAM)。但是，如果消费者被要求交出法律文件才能观看色情内容，他们可能会寻找其他网站，那里的内容可能不会受到很好的审查。

斯诺告诉 TechCrunch:“这实际上只是进一步边缘化性工作者，我认为这将是主要的影响。”“我想这意味着未经同意传播的优质内容黑市将会增加。”

像路易斯安那州的 440 法案以及 SESTA/FOSTA 这样的立法都被定位为倡导儿童安全。但在实践中，这项立法通常只会让性工作者更难安全地工作。此外，在某些情况下，这些法律实际上增加了执法部门遏制性交易的难度。

2018 年，司法部的查封了 Backpage.com，性工作者将它用作帮助他们审查当面客户的工具。当网站因洗钱和性交易被关闭时，印第安纳州警方表示，他们抓捕从事性交易的人变得更加困难。

斯诺说，第 440 号法案“也可能产生类似于倒退的影响，你知道，这只是性工作者失去了另一种收入来源，不得不求助于保护较少的渠道。”

2020 年，参议员伊丽莎白·沃伦(D-MA)向参议院提出了一项关于 SESTA/FOSTA 对性工作者的次要影响的研究。

法案说“性工作者报告说，他们筛选潜在客户的能力降低了，他们就使用避孕套等界限进行谈判的能力降低了，这导致了身体暴力和性暴力的报告。”。“许多性工作者转向街头工作，这在历史上比其他形式的性交易涉及更高的暴力率。”

斯诺将要求人们上传 ID 观看色情片的行为描述为监视。她说，在最极端的情况下，这种监控会伤害 LGBTQ 人群。

“随着同性恋恐惧症和变性恐惧症——尤其是色情背景下的同性恋恐惧症——的上升，我完全可以看到国家将注意力集中在消费同性恋色情作品的人身上，要么进一步监视他们，要么将其定罪，”她说。在第 440 号法案的文本中，该法案警告说，色情可以激发“异常的性唤起”，但没有定义“异常”的含义。

斯诺说:“变性女性在性工作中所占的比例不成比例。”。“我不知道这是立法者有意识的决定，还是只是偶然的，但我认为变性女性性工作者，像往常一样，将受到最大的影响。”

多年来，英国政府一直致力于一项在线安全法案，但由于领导层不断变动，该法案一直处于搁置状态。按照现在的情况，该法案可以建立类似的年龄检查，要求用户在访问露骨的性内容之前核实他们的年龄。

过去，英国政府曾试图在色情网站上实施年龄检查，但在 2019 年放弃了该计划，原因是担心强制年龄验证的技术和监管挑战，以及隐私问题。现在，英国的年龄检查再次被考虑。
