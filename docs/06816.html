据 crypto VC Grace Isford | TechCrunch 报道，为什么 web3 公司经常被黑客攻击

原文：https://web.archive.org/web/https://techcrunch.com/2022/05/29/web3-hacks-crypto-vc-grace-isford-chain-reaction-podcast/

在本周的连锁反应播客中，Lux Capital 的最新投资者 Grace Isford 加入了我们，谈论不透明但至关重要的 web3 基础设施世界。在 Lux，Isford 投资于在幕后工作的公司，以确保加密交换足够安全可靠，以避免被黑客攻击。

在今年 2 月加入 Lux 之前，Isford 是 Canvas Ventures 的投资者，专注于企业软件和金融科技。她说，她在 Canvas 从事的一项数据基础设施投资向她揭示了 web3 领域的机会，公司可以“大规模不变地共享数据”，这促使她转向加密。

“这让我掉进了兔子洞，然后我最终投资了自己，”伊斯福德说。“我进入了高产农业，这与我搬到纽约的时间相吻合，在纽约，我的许多朋友也在加密和风险投资生态系统中。”

伊斯福德说，她在 web3 的投资方式根植于她所谓的“能力圈”，或者说，她可以与该领域的其他人相比具有竞争力的领域。

“NFT 投资与 DeFi 投资截然不同，DeFi 投资与加密数据基础设施投资也截然不同，我认为任何说自己投资 web 的人都不应该投资所有这些——他们可能应该选择自己核心竞争力中的最佳点，”伊斯福德说。

根据她之前的经验，Isford 自己的“能力圈”是在企业和金融科技基础设施领域，所以我们问她，她认为 web3 基础设施提供商面临的最大挑战是什么。

Isford 说，与 Web 2.0 相比，web3 缺乏企业级的安全解决方案。Alchemy 和 Infura 是业内仅有的两家主要节点服务提供商，这意味着大多数 crypto 都依赖于两家基础设施提供商来管理他们的数据。

“似乎每周(在 web3 中)都有新的安全黑客报告，”Isford 说，引用了最近源于 Infura 的 Metamask 和 Ethereum dApp 中断和二月份的虫洞桥黑客。

伊斯福德说，尽管许多初创公司正在开发安全解决方案，但在开发工具、数据基础设施监控和存储方面，这项技术“仍处于萌芽状态”。

伊斯福德补充说，另一个主要挑战是管理欺诈和下行风险。

“我认为[这个问题]现在真的让很多人置身于加密世界之外，[因为]他们害怕如果他们冒险深入加密就会失去所有的钱，”伊斯福德说。

Isford 乐观地认为，通过过去一年对 web3 初创公司的大量投资流入，公司将能够建立更可靠的解决方案。

“我认为 TRM Labs、Chainalysis 和该领域的其他几家公司在合规和监控方面有 10 倍的潜力，因为你还没有像我们在 web2 世界的金融基础设施方面创建这些复杂的反洗钱系统那样的规模，”Isford 说，他指的是传统金融机构的反洗钱技术。

Isford 说，更好的欺诈和风险管理系统是更多机构资金流入 crypto 的前兆。她补充说，随着富达、高盛和摩根大通等公司继续向加密领域大步迈进，这个市场将会成熟。

“我认为目前加密领域最大的机会之一仍然是安全性，如果你可以大规模地建立更可靠的智能合同……但是如果不安全，就不可能有可靠的系统，对吗？如果你不知道系统中有谁，你就无法安全地运行系统，所以我认为从优先级的角度来看，安全性可能是最重要的因素之一，”Isford 说。
