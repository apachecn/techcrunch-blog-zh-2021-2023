AWS re:Invent 2021 更多的是增量而非创新

AWS 本周在拉斯韦加斯举行了年度客户会议，重新发明。这通常是一场高能量的客户盛会——技术人员的马戏团——但今年的模型有一些不寻常的事情，使活动感觉比过去稍微低调一些。

首先，这是去年疫情迫使赛事进入虚拟模式后，第一次回到拉斯维加斯。现场的人比正常年份少(无论现在正常与否)，这也是新任首席执行官亚当·塞利普斯基(Adam Selipsky)在周二发表主旨演讲的第一次革新。

亚马逊首席技术官沃纳·威格尔指出，随着公司努力为客户简化运营，许多创新都在幕后进行。

有人认为他会想在事情上打上自己的印记，也许会为利润丰厚的部门的未来制定一条路线，但观众得到的更多是增量而不是创新，更多的是平淡而不是激动人心。这是一次很好的、组织良好的活动，没有大张旗鼓。

塞利普斯基的演讲流畅而专业，但缺乏真正吸引眼球的重大声明，就像安迪·杰西过去那样。虽然有一些有新闻价值的公告——我们当然报道了其中的一大堆——但 2021 年的《re:Invent》并没有什么让人觉得很酷的东西。感觉更像是亚马逊在检查盒子，填补产品路线图上的漏洞。

也许这就是我们今年都需要的。也许这是亚马逊在动荡的一年中所能做的最好的事情，当 Jassy 成为整个公司的 CEO 时，世界通过疫情奋力前进。但是不管是什么原因，看起来并没有发生什么大事。

AWS 首席执行官亚当·塞利普斯基在 AWS re:Invent 2021 的舞台上。图片来源:亚马逊

连续九年参加该会议的 Constellation Research 分析师霍尔格·穆勒(Holger Mueller)称之为他记忆中“最驯服的可再生能源发明之一”。

“总的来说，re:Invent 没有通常的重大宣布。例如，这是 AWS 很长时间以来第一次没有发布新的数据库。相反，它将一些服务转移到无服务器，并全面增加了(次要)功能，”他说。
