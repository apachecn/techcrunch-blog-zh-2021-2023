网飞将测试对密码共享收费的选项

原文：https://web.archive.org/web/https://techcrunch.com/2022/07/18/netflix-test-add-home-option-charge-password-sharing/

网飞将测试新的“添加家庭”选项，对密码共享收费

网飞今天宣布将开始测试一项新的“添加一个家”功能，向在特定国家与他人分享账户的用户收费。该测试标志着流媒体巨头打击密码共享的最新努力。如果用户使用家庭以外的账户，测试将提示他们支付额外的费用。下个月，网飞将开始在阿根廷、多明尼加共和国、萨尔瓦多、瓜地马拉和宏都拉斯测试这项功能。

在这些测试区域，每个网飞帐户将包括一个家庭，您可以在其中通过任何设备访问网飞。如果你想让别人在另一个家庭使用你的网飞账户，公司会要求你付费。你可以在阿根廷额外支付 219 比索，在其他测试地区支付 2.99 美元，为你的账户增加一套房子。基本网飞计划的会员可以增加一个额外的家，标准用户可以增加两个额外的家，高级用户可以增加三个额外的家。这些测试区域的用户将可以选择控制他们的帐户被使用的位置，并从他们的帐户设置页面中删除房屋。

“我们的会员如此喜爱网飞电影和电视剧，他们希望更广泛地分享它们，这太好了，”的产品创新总监成毅·龙在一份声明中说。“但如今家庭间广泛的账户共享削弱了我们投资和改善服务的长期能力。”

网飞表示，这项测试扩展了今年早些时候在智利、哥斯达黎加和秘鲁推出的类似功能，名为“添加额外成员”作为测试的一部分，如果用户与自己家庭以外的人共享服务，他们需要支付额外的费用。该功能允许家庭添加两个“子账户”,费用低于全价网飞服务的费用。

网飞曾表示，这种解决方案不依赖于基于位置的数据，如 GPS。取而代之的是，它正在利用它今天用来向其最终用户提供服务的相同信息，包括 IP 地址、设备 id 和家庭中登录到网飞帐户的设备的其他信息。通过这种方法，网飞可以识别出家庭以外的持续分享行为。

该公司在第一季度财报电话会议上宣布，它将扩大测试，如果会员与家庭以外的人分享账户，将向他们收取更高的费用。网飞澄清说，它需要在大约一年左右的时间里继续迭代这一功能，以确保在向与自己家庭以外的其他用户分享网飞账户的用户收取多少额外费用方面取得平衡。

今天，网飞估计全球大约有 1 亿个家庭共享他们的用户账户，其中仅在美国和加拿大就有超过 3000 万个。网飞表示，通过要求分享账户的会员支付更多费用，它希望在允许分享发生的同时，帮助每个观看并从其服务中获得价值的人带来收入，从而达到适当的平衡。

这些测试并不是网飞计划从其用户群中赚钱的唯一方式，因为该公司还表示将推出一项广告支持的计划。上周，网飞宣布与微软合作，成为其“全球广告技术和销售合作伙伴”，帮助流媒体服务通过广告获得收入。
