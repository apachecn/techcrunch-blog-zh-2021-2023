谷歌为谷歌文档增加了有限的降价支持

原文：https://web.archive.org/web/https://techcrunch.com/2022/03/30/google-adds-limited-markdown-support-to-google-docs/

谷歌与谷歌工作区的用户分享了的博客文章，宣布了一些给所有 Markdown 粉丝的好消息。谷歌文档现在可以自动将 Markdown 格式转换成富文本。

如果你不熟悉 Markdown，这种标记语言在过去几年里变得非常流行，如果你想写结构化文档的话——尤其是 web 内容。例如，流行的笔记工具如概念或支持降价。像 WordPress 这样的内容管理系统也可以让你启用 Markdown 作为写新帖的一种方式。

默认情况下，谷歌文档的这项新功能是关闭的。你必须先在设置中手动打开它。这就是我写这篇文章的原因。如果你想启用减价支持，在谷歌文档中打开一个文档，转到屏幕顶部，进入“工具”，然后“首选项”，启用“自动检测减价”之后，你就可以走了。如果你今天看不到这个功能，那是正常的，因为该公司将在未来几天逐步推出新功能。

同样，不要指望广泛的降价支持。Google Docs 仍然是一个富文本编辑器，一旦你启用了 Markdown 功能，这不会神奇地改变。

取而代之的是，谷歌文档将用富文本取代普通的降价格式。例如，你可以像[this](https://techcrunch.com)一样用圆括号和方括号创建链接。您可以用*字符将文本括起来，使其成为斜体。

谷歌文档支持不同级别的标题，粗体，斜体，删除线和链接。编号列表和非结构化列表仍然像以前一样自动格式化。

换句话说，这个新功能补充了一些键盘快捷键。它还可以更容易地将一个应用程序中的降价内容复制到谷歌文档中，如果你使用像 Ulysses 这样的降价编辑器，这可能会特别有用。

但是…反过来就不行了。Markdown 的伟大之处在于，它是一种纯文本格式，可以兼容多种应用和服务。

例如，Google Docs 应该允许您将当前文档导出为 Markdown，这样您就可以快速地将它复制并粘贴到任何支持 Markdown 的地方。我知道，有一个插件可以解决这个问题。但是并不是所有的公司都喜欢插件。

嘿，谷歌，这是一个很好的第一步，但你必须迭代，并提供更多的降价功能，请。
