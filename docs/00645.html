收藏卡交易平台 Alt 融资 7500 万美元，因为它关注其他资产类别，并准备推出移动产品| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/11/11/collectible-card-trading-platform-alt-raises-75m-as-it-eyes-other-asset-categories-and-preps-its-mobile-launch/

当我们上次在三月份与 Alt 联系时，它刚刚为其另类资产平台 筹集了 3100 万美元——到目前为止，这个平台主要被那些寻求研究、交易和安全储存高价值运动卡的人使用。

仅仅几个月后，该公司从顶级投资者和职业运动员名单中筹集了 7500 万美元的 B 轮融资，进行了大规模招聘，并准备推出其移动应用程序，同时它开始扩大其覆盖的替代资产的范围。

虽然 Alt 最初的重点是运动卡——创始人 Leore Avidar 非常了解这一类别——但它最近已经扩展到支持其他类型的交易卡。逛逛它的交易所，你就会发现第一版的 Charizards 和 Yu-Gi-Oh！混合了罕见的科比和亲笔签名的卡。随着时间的推移，他们希望扩展到信用卡之外的其他替代资产。

“我们的目标是涉足从运动鞋、手表到 NFT 的各种产品，”Avidar 告诉我，他把运动卡称为他们的“证据”。

要在 Alt 上出售一张卡片，它的状况必须首先由一个已经建立的分级小组(PSA、BGS 或 SGC)进行判断，然后被送到 Alt 的“保险库”——一个光控、温控、防火的设施，该公司随便称之为“卡片的诺克斯堡”一旦到了那里，所有权可以立即从买家转移到买家(Alt 从每笔销售中抽取 1.5%的佣金)。默认情况下，购买的卡留在保管库中；所有者可以自由地将他们的卡从保险库中取出并运送给他们，但 Avidar 告诉我，交易后“99%”的卡会留在保险库中。他们购买这些卡片主要是作为投资，而不是展示。我还被告知，到目前为止，Alt 用户已经在金库中储存了价值超过 7000 万美元的卡。

Alt 分享了卡上的大部分数据；它的市场趋势页面列出了最近几周最受欢迎的类别，并突出显示了在过去七天中卡片价值变化最大的玩家。单个列表显示给定卡在过去一年中的销售价格，并提供“Alt 值”——基于 Alt 可以访问的所有最近交易数据的 Zestimate 风格的价值估计。

Alt 的团队发展非常迅速，Avidar 指出，该公司目前约有 60 名员工。值得注意的是，它最近聘请了 Nicole Colombo，前易贝收藏品和交易卡总经理，作为它的第一任总裁。

与此同时，该公司正在悄悄探索一种新的收入来源:向用户贷款，贷款由用户在 Alt 上的储值卡担保。Avidar 指出，银行一般不会把运动卡收藏之类的东西视为“真正的抵押品”；与此同时，Alt 对任何给定的卡每天的价值都有相当深刻的理解——而且，由于这些卡在 Alt 的保险库中，他们可以假设这些卡不会凭空消失或突然损坏。该团队称这个程序为“ Alt Lending ”，并指出它目前处于测试阶段，只对精选用户开放。

该公司告诉我，这一轮融资的估值“超过 3.25 亿美元”。这轮融资由矛头基金(一种由杰夫·法格南(Jeff Fagnan)和纳瓦·拉维坎特(Naval Ravikant)共同创建的新型基金)牵头。我们在这里描绘矛头)，背后有七七六、Vibe Capital、Breyer Capital、耸肩资本、阿波罗项目(Max 和 Sam Altman 的公司，而不是我们新的同名企业霸主)、Hyperguap、A*和一些个人投资者——包括一些体育界的大腕，如汤姆·布拉迪、扬尼斯·阿德托昆博、艾利克斯·摩根、坎迪斯·帕克和马龙·汉弗莱。
