API 初创公司无法免受风险放缓的影响| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/21/api-startups-venture-slowdown/

即使是最热门的初创企业类别也不能免受风险投资放缓的影响

新贵科技公司通过应用编程接口提供产品或服务，在 2021 年初创热潮的最后一年筹集了巨额资金。在此期间，情况有所放缓。

新数据显示，尽管该集团在 2022 年筹集的资金比 2020 年多，但过去一个日历年筹资活动的下降趋势表明，没有一个初创企业群体能够免受风险资本放缓的影响。这也适用于最近经济衰退前最热门的新科技公司集团。

GGV 是一家跨行业、跨阶段、跨地域投资的风险投资公司，它编制了一份指数，列出了它认为是去年推出的 API 第一的公司。TechCrunch 报道了它的发布，因为我们有一个可以跟踪的初创公司的软肋，可以随时观察；来自贝塞默等公司的类似数据为创业市场提供了有用的测量工具，尤其是在快速发展的风险投资和退出环境中。

该风险投资集团最近汇编了 2022 年最后一份信息，并在发布前与 TechCrunch+分享了这些数据。蒂芙尼·勒克和切尔希·泰勒是 GGV 的投资者，他们和我们坐在一起讨论数据。应我们的要求，他们还提供了一个稍微扩展的数据集，包含了更多的 API 第一创业公司。

面对私募市场的逆风和一些市场对许多初创公司远离退出点的健康状况的悲观情绪，API 优先的初创公司表现如何？答案是喜忧参半。一方面，对创业商业模式的风险投资正在放缓，但也有好消息。随着科技巨头释放一波又一波有经验的人才，科技人才市场正在慢慢开放，这意味着资金雄厚的初创公司——以及 API first 初创公司在 2021 年期间确实筹集了大量资金——可能能够招募到以前遥不可及的新员工。
