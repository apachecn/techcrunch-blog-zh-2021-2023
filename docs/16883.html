陷入困境的航天公司表示无法获得额外资金来维持运营后，美国维珍轨道公司将裁员约 85%，以进一步削减开支。美国维珍轨道公司周四向美国证券交易委员会提交了这一消息，就在两周前，该公司让所有员工休假，并进入“运营暂停”以寻找更多现金。

报税时的日期: 有一款新的动漫约会模拟软件可以帮你报税——而且真的管用。Amanda 玩了税务天堂 3000 ，这是一款由 MSCHF 制作的游戏，MSCHF 是一家风险投资的创意工作室，支持像推党和 Lil Nas X 血鞋这样的项目。结论是什么？如果你不介意冒险与一个痴迷于纳税过程的动漫女孩分享你的个人信息，这不是最不愉快的报税方式。

替代者抖音: 随着美国议员推动实施抖音禁令或强制出售的计划，该应用的中国母公司字节跳动正推动其另一个社交平台进入美国应用商店的排行榜。字节跳动旗下的应用 Lemon8 是 Instagram 的竞争对手，它自称是一个“生活方式社区”，周一跃升为美国应用商店下载量最高的应用之一，成为应用和游戏综合排名第十的应用。

Groupon 有了新 CEO: 因推广在线团购模式而一举成名的 Groupon任命 Dusan Senkypl 为临时 CEO。正如 Ingrid 所写，Groupon 拥有 1400 万活跃用户，但在过去十年中，该公司的财务状况几乎一直在缓慢下滑——其核心业务模式停滞不前，多元化努力收效甚微，收入不断下降，持续亏损。

【获得你自己的 Lyft: Lyft 可能会再次放弃其共享乘车服务，这只是该公司新任命的首席执行官大卫·里舍(David Risher)可能做出的几项改变之一，以专注于 Lyft 的核心乘车业务并实现盈利。里舍在一次广泛的采访中告诉丽贝卡，其他功能也可能被取消，比如等待保存选项，允许某些地区的乘客在等待最佳位置的司机时支付更低的费用。

Twitter 的 API 开始付费: 经过数周的拖延，Twitter 终于在周三宣布了新的 API 定价结构。这三个等级包括主要针对内容发布机器人的基本免费等级、每月 100 美元的基本等级和昂贵的企业等级。任何级别的订阅都可以免费访问 Twitter 的 ads API。

艰难时期，大幅下调估值: Manish 报道称，一些最大的印度初创公司正在下调估值——至少在投资者眼中是如此，因为一些投资者在全球经济疲软之际调整了他们的估值。贝莱德将印度最有价值的初创公司 Byju's 的估值下调了近一半，至 115 亿美元，而印度最有价值的食品配送初创公司 Swiggy 的估值为 107 亿美元，被景顺(Invesco)下调至约 80 亿美元。

莱杰大获全胜: 法国初创公司莱杰向其 C 轮融资增加了更多资金——约€1 亿英镑(1.08 亿美元)，罗曼写道。该公司的主要产品是硬件加密钱包，提供高度的安全性，形状像 USB 钥匙，具有一个微型屏幕，可以在设备上确认交易。

供应链攻击: 卡莉写道，多家安全公司发出警报，称一项活跃的供应链攻击正在利用 3CX 广泛使用的语音和视频通话客户端的木马版本来瞄准下游客户。这种恶意软件特别危险，能够收集系统信息，从谷歌 Chrome、微软 Edge、Brave 和火狐用户档案中窃取数据和存储的凭证。

Canoo 与 SEC 达成和解: 根据一份监管文件显示，电动汽车初创公司 Canoo 已同意与美国证券交易委员会达成 150 万美元的和解。美国证券交易委员会于 2021 年 5 月开始调查 Canoo，重点关注这家初创公司的运营、商业模式、收入、收入战略、客户协议、收入以及某些公司高管的离职，包括联合创始人兼首席执行官 Ulrich Kranz。
