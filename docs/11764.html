现在，任何人都可以使用 DALL-E 2 开发应用程序来生成图像

原文：https://web.archive.org/web/https://techcrunch.com/2022/11/03/now-anyone-can-build-apps-that-use-dall-e-2-to-generate-images/amp/

图片来源: OpenAI

终于，OpenAI 的图像生成人工智能系统 DALL-E 2 以 API 的形式推出，这意味着开发者可以将该系统构建到他们的应用程序、网站和服务中。在今天的一篇博客文章中，OpenAI 宣布，任何开发者都可以开始利用 DALL-E 2 的力量——现在有超过 300 万人每天使用它来制作超过 400 万张图像——只要他们创建一个 OpenAI API 帐户作为公共测试版的一部分。

DALL-E 2 API 的定价因分辨率而异。对于 1024×1024 的图像，成本为每张图像 0.02 美元；512×512 图像每张 0.018 美元；256×256 的图片每张 0.016 美元。与 OpenAI 的企业团队合作的公司可以享受批量折扣。

与 DALL-E 2 beta 一样，该 API 将允许用户根据文本提示生成新图像(例如，“一只毛绒绒的兔子在花丛中跳跃”)或编辑现有图像。微软，OpenAI 的亲密合作伙伴，正在 Bing 和微软 Edge 中利用它的图像创建器工具，如果网页结果没有返回他们想要的，它可以让用户创建图像。时尚设计应用 CALA 正在使用 DALL-E 2 API 作为一种工具，允许客户从文本描述或图像中提炼设计思想，而摄影初创公司 Mixtiles 正在为其用户提供艺术创作流程。

API 的推出并没有改变多少政策，这可能会让那些担心 DALL-E 2 这样的生成性人工智能系统在发布时没有充分考虑它们所带来的道德和法律问题的人失望。和以前一样，用户受到 OpenAI 服务条款的约束，该条款禁止使用 DALL-E 2 生成公开的暴力、性或仇恨内容。OpenAI 还继续阻止用户上传未经他们同意的照片或他们无权上传的图像，并采用自动和人工监控系统来执行这一操作。

一个小小的调整是用 API 生成的图像不需要包含水印。OpenAI 在 DALL-E 2 beta 期间引入了水印，作为指示哪些图像来自系统的一种方式，但随着 API 的推出，它已选择使其成为可选的。

“我们鼓励开发者披露图像是人工智能生成的，但不要求它们包括 DALL-E 2 的签名，”OpenAI 负责监督 DALL-E 2 开发的产品经理卢克·米勒(Luke Miller)通过电子邮件告诉 TechCrunch。

微软的设计工具，由 DALL-E 2 API 支持。图片来源:微软

OpenAI 还在 DALL-E 2 上采用了即时和图像级别的过滤器，尽管一些客户抱怨过滤器过于热情和不准确。该公司已经将一部分研究工作集中在使 DALL-E 2 生成的图像类型多样化上，旨在打击文本到图像人工智能系统已知会成为受害者的偏见(例如，当被提示像“CEO 的例子”这样的文本时，主要生成白人男性的图像)。

但是这些措施并没有消除所有的批评。八月，Getty Images 禁止上传和出售使用 DALL-E 2 和其他类似工具生成的插图，包括 Newgrounds，PurplePort 和 FurAffinity 在内的网站也做出了类似的决定。Getty Images 首席执行官克雷格·彼得斯(Craig Peters)告诉 The Verge，禁令是出于对“未解决的权利问题”的担忧，因为 DALL-E 2 等系统的训练数据集包含从网络上搜集的受版权保护的图像。

Getty Images 的竞争对手 Shutterstock 最近试图找到一个中间立场宣布它将开始使用 DALL-E 2 来生成内容，但同时推出一个“贡献者基金”，在公司出售训练文本到图像人工智能系统的工作时补偿创作者。它还禁止第三方上传人工智能艺术，以尽量减少版权作品进入平台的可能性。

技术专家 Mat Dryhurst 和 Holly Herndon 正在领导一项名为 Source+ 的工作，允许人们不允许他们的工作或相似物用于人工智能训练目的。但这是自愿的。OpenAI 还没有表示它是否会参与其中——或者，事实上，它是否会推出一种自助服务工具，允许版权所有者将其作品从培训或内容生成中排除。

Mixtiles 是 DALL-E 2 API 的早期采用者之一。图片来源: Mixtiles

在一次采访中，米勒几乎没有透露有关新缓解措施的细节，只是说 OpenAI 一直在改进其技术，以防止系统产生偏见、有毒和其他令人不快的内容，客户可能会觉得这些内容令人反感。他将 open API beta 描述为一个“迭代”过程，在接下来的几个月里，随着 OpenAI 扩展支持 DALL-E 2 的基础设施，将涉及到与“用户和艺术家”的合作。

当然，如果 DALL-E 2 测试版有任何暗示的话，API 程序将会随着时间而进化。早期，OpenAI 禁用了 DALL-E 2 编辑人脸的功能，但后来在对其安全系统进行改进后启用了这一功能。

“我们在这方面做了大量工作，包括通过您上传的图像和发送的提示，使其符合我们的内容政策，并采用不同的缓解措施在提示级别和图像级别进行过滤，以确保符合我们的内容政策。因此，举例来说，如果有人上传一张包含仇恨符号或血块的图像——比如非常、非常、非常暴力的内容——将会被拒绝，”米勒说。“我们一直在考虑如何改进这个系统。”

但是，尽管 OpenAI 似乎急于避免围绕稳定扩散的争议，相当于 DALL-E 2 的开源软件已被用于创建色情、戈尔和名人 deepfakes ，但它让 API 用户来选择如何和在哪里部署其技术。有些公司，比如微软，无疑会采取谨慎的方式，慢慢推出 DALL-E 2 驱动的产品来收集反馈。其他人会一头扎进去，拥抱技术和随之而来的道德困境。

如果有一件事是肯定的，那就是对生殖式人工智能的需求被压抑了——让后果见鬼去吧。甚至在 API 正式发布之前，开发人员就已经发布了将 DALL-E 2 集成到应用程序、服务、网站甚至视频游戏中的解决方案。随着公开测试版的推出，在 OpenAI 强大的营销力量的推动下，合成图像有望真正进入主流。
