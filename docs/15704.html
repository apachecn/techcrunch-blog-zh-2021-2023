Twitter 的隐私保护 Tor 服务陷入黑暗

Twitter 的 Tor 服务是该网站的一个版本，即使在禁止社交网络的国家也可以访问，在该公司未能更新其证书后，该服务已被关闭。

Twitter 洋葱网站的证书于 3 月 6 日到期，就在该网站即将迎来一周年纪念日的前几天。匿名全球网络背后的非营利组织 Tor Project 的战略沟通主任 Pavel Zoneff 告诉 TechCrunch，该网站“似乎不再可用，也没有更新的计划。”

Twitter 的 Tor 网站现在向访问者显示一个警告，警告其证书已经过期，如果超过这个时间点，就会显示一个 Twitter 错误页面，这是不推荐的。

Zoneff 补充说:“Tor 项目已经联系了 Twitter，研究如何将洋葱版的社交媒体平台重新上线。”“那些依靠洋葱服务获得额外一层保护并保证他们正在访问他们正在寻找的内容的人现在少了一种安全的方式。让我们明确一点，对洋葱服务的需求、保护人们匿名的注重隐私的浏览器的使用以及其他形式的加密仍然存在。”

在社交网络因入侵乌克兰而被屏蔽几天后，Twitter 推出了现已不存在的 Tor 服务。Tor 也被称为“洋葱路由器”，因为它通过许多层加密互联网流量，并通过世界各地的数千台服务器进行路由，允许其用户匿名并能够绕过审查系统。

虽然 Twitter 不愿透露其 Tor 服务的推出是否与俄罗斯的屏蔽直接相关，但该公司的一名发言人当时告诉 TechCrunch，让其服务更容易获得是该公司“持续的优先事项”。

Twitter 已经解散了通信部门，因此无法联系到该公司就此事发表评论。

该公司让其 Tor 服务到期的决定引发了人们对 Twitter 在埃隆·马斯克领导下对用户隐私和安全的承诺的更多担忧。该公司的安全、隐私和合规负责人都在被马斯克收购不到两周后离开了公司。根据前安全主管 Peiter“mud ge”za tko 提交的举报者投诉，Twitter 已经在安全方面存在不足，还面临着来自 FCC 的加强审查，FCC 与 Twitter 有长期协议以确保隐私保护。

虽然 Twitter 已经让其 Tor 服务下线，但其他几个受欢迎的网站，包括 BBC、脸书和 TechCrunch，都有洋葱地址。Alec Muffett 是一名网络安全研究员，去年帮助 Twitter 推出了 Tor 网站，他的 GitHub 知识库上有一份的详细列表。
