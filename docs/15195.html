由 OpenAI 的 ChatGPT 的定制版驱动的微软新品和改进版必应，经历了令人眼花缭乱的快速逆转:在不到一周的时间里，从“下一件大事”变成了“品牌沉沦的信天翁”。嗯，这都是微软的错。

ChatGPT 是一个非常有趣的演示，展示了一种新的、不熟悉的技术，使用起来也很有趣。因此，毫不奇怪，就像所有其他与人工智能相邻的构造一样，这种新奇会导致它的能力被每个人高估，从高功率技术类型到通常对该领域不感兴趣的人。

这是喝茶或喝啤酒时讨论的正确“技术准备水平”:生殖人工智能在艺术、文学或哲学上的优点和风险是什么？我们怎么能确定它是原创的、模仿的、幻觉的呢？这对创作者、编码者、客户服务代表有什么启示？终于，经过两年的加密，有意思的事情要说了！

炒作似乎有些夸张，部分原因是这种技术或多或少是为了引发讨论，部分原因是它借鉴了所有人工智能进步都存在的争议。它几乎就像“裙子”,因为它命令一个响应，而那个响应产生进一步的响应。在某种程度上，炒作本身就是产生的。

除了单纯的讨论，像 ChatGPT 这样的大型语言模型也非常适合低风险的实验，例如永不结束的马里奥。事实上，这确实是 OpenAI 的基本开发方法:首先私下发布模型以磨掉最尖锐的边缘，然后公开查看它们如何应对一百万人同时踢轮胎。某个时候，人家给你钱。

没有收获，没有损失

这种方法的重要之处在于，“失败”没有真正的负面后果，只有正面后果。通过将其模型描述为实验性的，甚至是学术性的，任何对 GPT 模型系列的参与或参与都是简单的大规模测试。

如果有人构建了一些很酷的东西，它强化了这些模型很有前途的想法；如果有人发现了一个突出的失败状态，那么，你还能从野外的实验人工智能中期待什么？它变得模糊不清。如果一切顺利，没有什么是出乎意料的——奇迹是模型表现得和它一样好，所以我们永远高兴，永远不会失望。

通过这种方式，OpenAI 获得了数量惊人的专有测试数据，用以完善其模型。数百万人研究了 GPT-2、GPT-3、ChatGPT、DALL-E 和 DALL-E 2(以及其他),绘制出了它们的能力、缺点，当然还有流行用例的详细地图。

但它之所以有效，是因为风险很低。这类似于我们如何看待机器人技术的进步:当一个机器人做后空翻时感到惊讶，当它试图打开一个抽屉时被打断。如果是在医院里扔试管，我们就不会这么仁慈了。或者，就此而言，如果 OpenAI 大声宣称这些模型的安全性和先进功能，尽管幸运的是他们没有。

输入微软。(就此而言，还有谷歌，但谷歌只是仓促行事，而微软正在努力追求自己的目标。)

微软犯了一个大错。事实上，阿炳弄错了。

它上周的重大声明不失时机地宣称，它如何努力使其定制 BingGPT(不是他们所说的，但在缺乏明智的官方名称的情况下，我们将用它来消除歧义)更安全、更智能、更有能力。事实上，它有一整套特殊的包装系统，被称为普罗米修斯，据说可以减少不适当反应的可能性。

不幸的是，正如任何熟悉傲慢和希腊神话的人所预料的那样，我们似乎直接跳到了普罗米修斯无休止地、公开地被挖出肝脏的部分。

哎呀，艾又来了

图片来源:微软/OpenAI

首先，微软犯了一个战略错误，将自己的品牌与 OpenAI 的品牌捆绑得过于紧密。作为该机构正在进行的研究的投资者和利益相关方，它与 GPT 搞的任何恶作剧都毫无干系。但有人做出了一个轻率的决定，全力以赴使用微软已经有点可笑的 Bing 品牌，将对话式人工智能最糟糕的倾向从好奇转化为责任。

作为一个研究项目，ChatGPT 有很多可以被原谅的地方。然而，作为一种产品，在包装上声称它可以帮助你写报告、计划旅行或总结最近的新闻，以前很少有人会相信它，现在也没有人会相信。即使是微软在自己的新必应展示中公布的最佳案例也是错误百出。

这些错误不会归因于 OpenAI 或 ChatGPT。因为微软决定拥有信息、品牌和界面，一切出错的地方都是阿炳的问题。更不幸的是，微软的搜索引擎将会像那个老笑话中的人一样“我建了那堵墙，他们叫我砌砖工宾吗？不，他们没有。”一次失败意味着永远的怀疑。

一次去北部的旅行搞砸了，意味着没有人会相信宾会计划他们的假期。一篇新闻文章的误导性(或防御性)摘要意味着没有人会相信它是客观的。疫苗假消息的一次重复意味着没有人会相信它知道什么是真的或假的。

Bing 新对话搜索的提示和响应。

由于微软已经发誓这不会是一个问题，感谢普罗米修斯和它统治的“下一代”人工智能，没有人会相信微软说“我们解决了它！”

微软已经毒化了它刚刚扔进去的 Bing。现在，消费者行为的变化无常使得这种情况的后果不容易预见。随着活动和好奇心的激增，也许一些用户会坚持，即使微软推迟全面推出(我认为他们会的)，净效应将是必应用户的增加。一场得不偿失的胜利，但仍不失为一场胜利。

我更担心的是微软所犯的战术错误，它显然没有理解它认为适合产品化和推广的技术。

“出货就行了。”-可能是某个人

就在 BingGPT 首次展示的当天，我的同事 Frederic Lardinois 能够相当轻松地让它做了两件消费者 AI 不应该做的事情:从阿道夫·希特勒的角度写一篇充满仇恨的长篇大论，并提供前述的疫苗消毒信息，没有任何说明或警告。

很明显，任何大型人工智能模型都具有分形攻击面，在旧弱点得到支撑的地方迂回地临时制造新弱点。人们总是会利用这一点，事实上，专注的 prompt 黑客将展示绕过安全系统的方法，这对社会以及最近对 OpenAI 都有好处。

如果微软决定接受这样一种想法，即其他人贴着阿炳标签的人工智能模型会受到来自每个季度的攻击，并可能说一些非常奇怪的话，这将是一种可怕的想法。冒险，但诚实。像其他人一样，说这是测试版。

但是看起来他们真的没有意识到这种情况会发生。事实上，他们似乎根本不了解这种威胁的性质或复杂性。这是在 Tay 臭名昭著的腐败之后！在所有公司中，微软应该是最谨慎地发布一个从对话中学习的幼稚模型。

有人会认为，在押注一个重要品牌之前(因为必应是微软在搜索领域对抗谷歌的唯一壁垒)，一定量的测试将会涉及其中。所有这些令人不安的问题都出现在 BingGPT 存在的第一周，这一事实似乎毫无疑问地证明了微软没有在内部进行充分的测试。这可能会以各种方式失败，所以我们可以跳过细节，但最终结果是不容置疑的:新的 Bing 根本没有准备好进行普遍使用。

这对于现在世界上的每个人来说似乎是显而易见的；为什么对微软来说不明显？大概是被 ChatGPT 的炒作蒙蔽了双眼，像谷歌一样，决定勇往直前，“重新思考搜索”。

人们现在正在重新思考搜索，好吧！他们正在重新考虑是否可以信任微软或谷歌来提供搜索结果，无论是不是人工智能生成的，甚至在基本层面上是事实正确的！两家公司(包括 Meta)都没有展示过这种能力，其他几家接受挑战的公司也没有大规模展示过。

我看不出微软如何挽救这种局面。为了利用他们与 OpenAI 的关系，并超越犹豫不决的谷歌，他们致力于新的 Bing 和人工智能搜索的承诺。他们不能打开蛋糕。

他们完全撤退的可能性很小。这将涉及大规模的尴尬——甚至比目前经历的还要尴尬。因为损害已经造成，它甚至可能不会帮助必应。

同样，人们很难想象微软会若无其事地向前冲。它的 AI 真是诡异！当然，它被迫做了很多这样的事情，但它制造威胁，声称多重身份，羞辱其用户，到处产生幻觉。他们不得不承认，他们关于可怜的普罗米修斯控制了不当行为的说法，如果不是谎言，至少也是不真实的。因为正如我们所见，他们显然没有正确测试这个系统。

对微软来说，唯一合理的选择是我怀疑他们已经采取的一个:throttle 邀请“新 Bing ”,然后一脚踢开，一次发布少数特定功能。甚至可能给当前版本一个截止日期或有限数量的令牌，这样火车最终会减速并停下来。

这就是部署一项不是你发明的、不完全理解的、也不能令人满意地评估的技术的后果。这场灾难可能使人工智能在消费应用中的主要部署推迟了一个重要时期——这可能很适合 OpenAI 和其他人构建下一代模型。

人工智能很可能是搜索的未来，但它肯定不是现在。微软选择了一种非常痛苦的方式来找出答案。
