欧盟最高外交官就推特虚假信息向埃隆·马斯克鸣枪示警

原文：https://web.archive.org/web/https://techcrunch.com/2023/02/07/elon-musk-twitter-disinformation-josep-borrell/

Twitter 所有者埃隆·马斯克(Elon Musk)在 2 月 9 日(T1)之前终止免费获取其 API 的争议性举措招致了欧盟最高外交官的批评，他警告说，这可能会在关键时刻威胁到研究人员研究虚假信息的能力——俄罗斯正在积极地将虚假信息武器化，以试图为其在乌克兰的战争提供掩护。

在今天的一次讲话中，欧盟高级专员约瑟夫·博雷尔(Josep Borrell)详细阐述了欧盟如何回应俄罗斯自去年 2 月入侵乌克兰以来加强在线虚假信息活动，他敦促对社交媒体平台如何被用来传播克里姆林宫宣传进行更多研究。

“我们必须对社交媒体平台做更多的研究。他在对欧盟外交部门(EEAS)的讲话中说:“研究(虚假信息)是如何流动的，它来自哪里，结果是什么。”。

博雷尔点名批评了 Twitter——以及它的所有者马斯克——称欧盟担心新闻称 Twitter 正计划限制免费访问其 API，他警告称这将是“从早期承诺严重倒退”。

“早期承诺”很可能是指 Twitter 从一开始就成为欧盟关于网上虚假信息的行为准则的签署者，追溯到 2018 年的，当时欧盟委员会公布了鼓励社交媒体平台解决“假新闻”危机的自愿倡议(当时仍有时提及)。

欧盟还禁止了一些俄罗斯国家媒体——因此，自 3 月起，Twitter 被法律要求确保其平台不托管克里姆林宫支持的媒体机构今日俄罗斯(RT)和斯普特尼克(以及任何子公司)。任何阻止这些频道在欧盟传播内容的失败都将使禁令落空。

自欧盟 2018 年的虚假信息代码制定以来，虚假信息研究出现了很多——人们对用于操纵信息、传播和放大错误观点以操纵公众舆论、侵蚀对民主机构的信任以及干涉自由公平选举的技术和策略类型有了充分的了解。

但是——就像网络安全一样——对信息完整性的威胁并没有停止。博雷尔的演讲强调了西方民主国家需要采取更多措施，通过增加对信息威胁研究的投资来保护自己免受外国恶意信息的攻击，这位专员还挥舞着欧盟关于外国信息操纵和威胁的首份报告的副本，他说这份报告显示了“对我们信息空间的威胁的明显趋势”。

该报告的一个主要发现证实了“新一波的假情报技术”——他说，俄罗斯人正在利用这些技术编造虚假图像和视频，以传播和扩大反乌克兰宣传——并警告称，俄罗斯和中国等威胁行为者之间的合作“令人担忧”。

博雷尔表示:“外交账户和国家控制的渠道操纵了对欧盟的看法——将乌克兰战争的所有后果归咎于西方，它们扩大了关于西方军队在乌克兰以邻国为目标的生物武器的谎言。”他补充道:“这是需要回应的事情。

“我们需要以具体的行动和措施来预测和阻止这种活动。我们需要继续支持乌克兰……最后，我们必须更加雄心勃勃地建设抵御试图制造虚假信息和操纵信息的独裁政权的能力。”

尽管宣传作为一种策略在人类历史上并不新鲜，但博雷尔表示，互联网和加速信息传播的数字工具显然加剧了对信息操纵的威胁——他认为，自由的西方民主国家因此需要对快速扩大的虚假信息风险等问题做出相应的严肃回应。

尽管欧盟高层对这一问题给予了一些新的关注——欧盟高级专员试图在现有努力的基础上提高人们对克里姆林宫围绕乌克兰战争的宣传的认识(比如欧盟对抗虚假信息运动——T1)——但可以公平地说，欧盟的立法者仍未能找到一个令人信服的“前脚”来有效对抗在线虚假信息。

这是一个棘手的问题，因为更强有力的行动可能会成为兜售恶意主张的一个容易的目标，即保护信息完整性的干预相当于对言论自由的审查。然而，大量人为制造的虚假信息显然是对民主言论自由的真正威胁——正如博雷尔在演讲中指出的那样，专制政权的武器化和系统性大规模制造虚假言论意味着人们的真实观点被淹没。

“我们已经看到有人在制造人工网络来传播(克里姆林宫的)虚假信息。他们一直充斥着信息空间，以避免任何其他的声音被听到。我们看到有人试图用不断变化的叙述和事件版本来迷惑和误导人们。目的是让没有人再相信任何事情，任何报告，”他争辩道。

“让人们相信，嗯，到最后，一切都是谎言——我如何区分谎言和真相？他们想侵蚀对所有媒体和我们机构的信任。今天，我想说的是，我们必须严肃对待这件事。这不仅仅是专家的事。这不仅仅是信息系统工作人员的事。这是公民必须意识到的事情——我们必须在尽可能高的政治层面上解决这个问题。”

欧盟关于虚假信息的行为准则仍然没有法律约束力——不出所料，近五年的倡议未能阻止各种宣传浪潮，无论是与冠状病毒虚假信息还是乌克兰有关。

欧盟至少部分承认了这一失败。因此在去年宣布了一个 强化规范——而且，最引人注目的是，它表示将把遵守规范与遵守(具有法律约束力的)数字服务法案(DSA)联系起来——DSA 于去年生效，并将在今年晚些时候开始申请一组更大的平台(大部分数字服务预计将在 2024 年符合规范)。

因此，如果 DSA 作为一根棍子，迫使平台更认真地对待反虚假信息，欧盟立法者将希望未来会更好。

然而，就目前而言，欧盟迄今为止清除虚假信息的努力和网上实际发生的事情之间的差距看起来很大。

而且，这种脱节越来越令人尴尬。

事实是，在纸面上，马斯克领导下的 Twitter 仍然是欧盟准则的签署者。然而，在实践中，Twitter 的新主人已经采取了一系列非常明显与倡议背道而驰的决定——例如，废除了新冠肺炎上现有的误导性信息政策，并在去年接管账户验证后很快制造了混乱，让任何人付钱给他以获得蓝色复选标记，引发了恶意假冒的即时泛滥。

尽管欧盟立法者声称在数字监管方面走在了前面，但他们迄今为止除了发出几句警告——告诉 Twitter 履行其“承诺”之外，还能做什么。或者警告它“巨大的工作”,如果 Twitter 要遵守 DSA——无论它何时适用于平台。

对于欧盟作为数字规则制定者的声誉来说，更尴尬的是，马斯克经常被指控个人放大克里姆林宫的宣传。

在去年的一个臭名昭著的例子中，这导致了乌克兰总统本人的干预，他在推特上发布了一个讽刺性的投票——要求他的粉丝投票选出他们“更喜欢”哪个@elonmusk:选项是“支持乌克兰的人”或“支持俄罗斯的人”。

不用说，支持乌克兰的选项赢得了投票。但西方民主机构继续像虚假信息问题上的大输家一样出现——因为他们似乎无力阻止马斯克这样的人故意(或者充其量是轻信)传播克里姆林宫的谎言，马斯克现在是 Twitter 的首席执行官。

博雷尔今天对“推特的所有者”的抨击可能是欧盟对马斯克的最后一击。更广泛地说，认识到如果要根除日益有害的独裁虚假信息的威胁，就必须采取更系统的方法。

与此同时，马斯克继续使用他去年借了数十亿美元购买的平台来放大克里姆林宫的虚假信息。

就在本周，它被留给了 Twitter 用户，这些用户还没有放弃这个平台，因为马斯克是如何将它废弃的召唤出另一个“首席推特”为俄罗斯政权工作的例子——此前他轻信了一些虚假的指标，这些指标旨在通过不加批判地评论“悲惨的生命损失”来列出乌克兰的高伤亡率。

马斯克因放大克里姆林宫的乌克兰战争宣传而被点名，他的回应不是删除他对虚假声明的橡皮图章式回应——这是由一个账户发布的，字面上使用了俄罗斯总统弗拉基米尔·普京戴着光环的照片——，而是他建议Twitter 的社区笔记功能可以用来“纠正数字”。

正如 Twitter 用户大卫·罗斯柴尔德(David Rothschild)迅速指出的那样，你不能用一个“小修正”来纠正一个巨大的谎言。这样做表明，你串通一气，允许一个巨大的谎言继续流传——这个谎言描绘了一个关于俄罗斯在乌克兰战争的虚假叙事，有助于俄罗斯削弱对乌克兰继续与俄罗斯作战的支持。

“我们需要更多的透明度和问责制，而不是更少，”博雷尔今天说，他在推特和马斯克上呼吁不要违背早先打击虚假信息的承诺，并补充说:“我呼吁推特——及其所有者——确保他们承担的所有义务都将得到履行。”

在演讲中，他继续敦促那些致力于解决信息操纵问题的人变得更有组织性——并拿出可互操作的系统来共享分析和最佳实践——宣布欧盟也将通过建立一个新的中央资源来做更多的事情，他说该资源将用于收集虚假信息威胁的信息，并促进情报共享。

“这是一场远程战斗。这不可能一蹴而就，”他警告说。“我们必须有工具。这个信息共享和分析中心将加强我们的应对措施，使我们能够更好地保护我们的民主。”

TechCrunch 向 Twitter 寻求对 Borrell 言论的回应，并询问它是否计划重新考虑终止研究人员对其 API 的免费访问。

在批评马斯克终止免费 API 访问的计划可能会扼杀大量有用的 Twitter 机器人之后，他最近宣布对提供“免费的好内容”的机器人实施任意缓刑——无论“好”在这种情况下意味着什么。但到目前为止，他似乎对研究人员 API 问题保持沉默。(关于“不良内容”对民主利益构成的威胁，他有传播自己的倾向。)

在撰写本文时，Twitter 尚未回应我们的问题，即考虑到这样做会对专家研究虚假信息的能力产生影响，它是否会重新考虑切断研究人员的联系。然而，马斯克接管 Twitter 的另一个决定是清算其外部通信部门——并忽略媒体的评论请求——所以我们不期待回应。
