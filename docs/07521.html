OctoML 让 AI/ML 模型投入生产变得更加容易| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/06/22/octoml-makes-it-easier-to-put-ai-ml-models-into-production/

帮助企业优化和部署模型的资金雄厚的机器学习初创公司 OctoML 今天发布了一项重大产品更新，这将使开发人员更容易将 ML 模型集成到他们的应用程序中。有了这个新版本，OctoML 现在可以将 ML 模型转换成可移植的软件功能，开发者可以通过一致的 API 与之交互。这样，将这些模型集成到现有的 DevOps 工作流中也将更加容易。

正如 OctoML 创始人兼首席执行官 Luis Ceze 告诉我的，他认为这是公司的一个重要时刻。Ceze 与该公司的首席技术专家陈天琦、CPO Jason Knight、CTO Jared Roesch 和技术合作伙伴副总裁 Thierry Moreau 一起创建了该公司，以将 TVM 产品化，这是一个开源的机器学习编译器框架，可以帮助 ML 工程师针对特定硬件优化他们的模型。

然而，模型作为功能的能力只是该公司今天宣布的一部分。该平台中还有一个新工具，可以帮助 OctoML 使用机器学习来优化机器学习模型。该服务可以自动检测和解决依赖性，并清理和优化模型代码。还有一个新的本地 OctoML 命令行界面和对 Nvidia 的 Triton 推理服务器的支持，现在可以用于新的模型功能服务。

OctoMl 首席技术官 Jared Roesch 表示:“Nvidia Triton 是一个强大的抽象，允许用户跨 CPU 和 Nvidia GPU 利用多个深度学习框架和加速技术。“此外，通过将 Nvidia Triton 与 OctoML 相结合，我们使用户能够更轻松地选择、集成和部署 Triton 驱动的功能。OctoML 工作流通过无缝集成 OctoML 加速技术进一步增加了基于 Triton 的部署的用户价值，使您能够充分利用服务层和模型层。”

展望未来，Ceze 指出，该公司自 2020 年以来已从 20 名员工增长到 140 多名员工，将专注于将其服务带到更多的边缘设备，包括智能手机，以及由于与高通的合作关系，其他骁龙驱动的设备。
