报道:微软计划在未来几周内用更快版本的 ChatGPT 更新必应

原文：https://web.archive.org/web/https://techcrunch.com/2023/02/01/report-microsoft-plans-to-update-bing-with-a-faster-version-of-chatgpt-in-the-coming-weeks/

报道:微软计划在未来几周内用更快版本的 ChatGPT 更新必应

据 Semafor 的一份新报告称，微软正致力于在未来几周内将 OpenAI 的更快版本 ChatGPT，即 GPT-4，整合到必应中，此举将使该搜索引擎与谷歌更具竞争力。整合后，必应将使用 GPT-4 来回答搜索查询。

知情人士告诉 Semafor，ChatGPT 和 GPT-4 的主要区别在于速度。虽然 ChatGPT 有时需要几分钟才能形成响应，但据说 GPT-4 在响应查询方面要快得多。据说最新软件的反应更详细，更像人类。

将 ChatGPT 并入微软产品的计划预计将在互联网搜索领域引发新的竞争，该领域在很大程度上一直由谷歌主导。通过使用 GPT-4，必应将能够为用户提供类似人类的答案，而不是仅仅显示一个链接列表。

微软和 OpenAI 拒绝置评。

Semafor 报道称，OpenAI 还计划在未来推出一款移动 ChatGPT 应用。由于 ChatGPT 目前只有网页版，移动版的引入将使 OpenAI 能够通过使该软件更容易使用来扩大 ChatGPT 的用户群。此外，该报道称，该公司正计划测试其流行的图像生成软件 Dall-E 的一项新功能，该功能将使用人工智能生成视频。

据《T2》上个月的报道,有传言称微软正在准备将 ChatGPT 与 Bing 搜索结果整合，并将 OpenAI 的语言人工智能技术引入 Word、PowerPoint 和 Outlook 等应用程序。

这份报告发布之际，微软上周通过一项“多年、数十亿美元”的投资扩展了与 OpenAI 的合作关系。微软首席执行官塞特亚·纳德拉最近也告诉华尔街日报，微软计划将 OpenAI 的基础系统作为商业平台，以便任何行业的任何实体都可以在其上建立。

在一篇博文中，微软表示，它计划投资部署专门的超级计算系统，以加速 OpenAI 的人工智能研究，并将 OpenAI 的人工智能系统与其产品集成，同时“引入新类别的数字体验”

值得注意的是，尽管必应加入 GPT-4 将挑战谷歌在互联网搜索领域的主导地位，但据报道，该公司表示不会推出 ChatGPT 的竞争对手，因为它比 OpenAI 等初创公司有更大的声誉风险。据报道，谷歌高管还告诉员工，他们不认为聊天机器人已经准备好取代搜索。
