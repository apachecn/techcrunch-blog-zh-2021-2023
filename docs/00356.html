Activeloop 获得 500 万美元种子，为人工智能应用构建流数据库| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/11/02/activeloop-snags-5m-seed-to-build-database-for-ai-applications/

Y Combinator2018 年夏季团队的成员 Activeloop ，正在建立一个专门为以媒体为中心的人工智能应用程序设计的数据库。今天，该公司宣布了由 468 Capital 和 CM Ventures 领投的 500 万美元种子投资，参与投资的有 Tribe Capital、Shasta Ventures 和各种科技行业天使投资人。

该公司创始人兼首席执行官 Davit Buniatyan 表示，该公司是在他在普林斯顿大学所做的研究基础上发展起来的，他在那里看到了专门为人工智能用例设计的图像和视频等非结构化数据的流数据库的需求。该公司今天还推出了一款商业产品的 alpha 版本。

“我们正在为人工智能引入一个数据库，特别是一个存储层，它有助于非常有效地存储数据，然后将这些数据传输到机器学习应用程序或训练模型，以进行计算机视觉，音频处理，NLP(自然语言处理)等，”Buniatyan 解释道。

实际上，这采用了开源 API 的形式，使用户能够转换数据(视频、音频等)。)转换成机器可以理解的数学表示。此外，API 允许您跟踪数据的不同版本，并最终允许您将数据存储在像亚马逊 S3 这样的存储库中。

通过流式传输数据，数据科学家或开发人员不再需要将所有这些数据下载到本地笔记本电脑上，而是将它们发送给模型使用，就像网飞将电影流式传输给你一样，因此你不必将所有数据下载到本地计算机上。

他说开源项目有 55 个贡献者，总共有 700 个社区成员。该程序迄今已被下载 30 万次。

他在普林斯顿大学神经科学实验室处理大量数据时，想到了流式传输数据的想法。他发现这些文件太大了，下载它们是不现实的，所以他想出了这种流式传输的方法，这将成为 Activeloop 的基础。

到目前为止，该公司有 15 名员工，并打算招聘大约 6 名高端工程师。他表示，在创建公司的过程中，他一直在寻找各种各样的人才。他指出，他的工程副总裁是一名女性。最近，该公司向一名埃塞俄比亚工程师提供了一份工作，但他最终拒绝了，但他表示，这是一个在任何地方寻找人才的例子。

该 API 是开源的，但该公司的商业产品(今天在 alpha 中发布)将在此基础上添加一些功能，例如能够在流数据上运行 SQL 查询，这是开源产品所不具备的。该公司目前处于预收入阶段，但希望明年在 Q1 发布商业版。
