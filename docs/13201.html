迪士尼+广告支持的计划目前在 Roku 设备上不可用

周四，Disney+在美国推出了有史以来第一个广告支持的计划“ Disney+ Basic ”，每月 7.99 美元，与迪士尼将价格提高到10.99 美元/月之前的无广告计划价格相同。然而，Roku 用户想要转换到新的计划是不走运的——至少现在是这样。

据迪士尼 Plus 的支持网站称，广告支持层“目前在 Roku 设备上不可用”该网站称，微软 Windows 桌面应用程序也没有提供这一功能。因此，目前，拥有 Disney+ Basic 或 Disney Bundle Duo Basic(Disney+Basic 和 Hulu 的广告计划)或 Trio Basic (Disney+ Basic、Hulu 的广告计划和 ESPN+)等 Disney Bundle 的美国订户无法在 Roku 或 Windows 上播放流媒体。

迪士尼告诉 TechCrunch，它仍在与 Roku 就达成一项对双方都合适的协议进行谈判。我们猜测，这场纠纷是因为广告分成协议，因为默认情况下，频道必须与 Roku 分享广告收入。然而，迪士尼拒绝提供细节。Roku 也拒绝对谈判发表评论。

Roku 已经巩固了自己作为美国顶级智能电视平台的地位。因此，Disney+在发布时没有在 Roku 设备上提供新的广告支持层是一个主要的劣势。

网飞在一个月前推出广告支持的计划时也遇到了类似的问题。

当时，网飞告诉 TechCrunch，在发布时，tvOS 设备上不支持其“带广告的基本”计划，但很快就会支持。根据网飞支持网站的消息，苹果电视和 PlayStation 3 游戏机仍然无法使用该软件。
