Platform.sh 获得 1.4 亿美元的网络应用开发工具

原文：https://web.archive.org/web/https://techcrunch.com/2022/06/20/platform-sh-secures-140m-for-web-app-development-tools/

web 基础设施日益增长的复杂性是当今许多组织面临的一个挑战。当构建和部署 web 应用程序时，IT 团队必须努力应对一系列选择，从选择使用哪个后端系统到决定在哪个框架中编码。根据软件开发商 Infragistics 最近的一项调查，超过三分之一(40%)的开发人员正面临着日益增长的客户需求，而 39%的开发人员资源有限，包括预算紧张。

没有灵丹妙药。但是弗雷德·普莱斯认为，他的服务 Platform.sh 可以通过抽象掉大量底层网络应用基础设施，让开发者的生活变得更加轻松。为了支持这一愿景，摩根士丹利扩张资本(Morgan Stanley Expansion Capital)、Revaia 和 Digital Partners 共同领导了 Platform.sh 的 1.4 亿美元 D 轮融资，该轮融资于今天结束，BGV、Eurazeo、Hiinov、Eurozeo 和 Partech 参与了融资。这使得 Platform.sh 的总收入达到 1.87 亿美元，Plais 表示，这笔资金将用于招聘工作和自动化方面的“双倍下注”。

在与 Damien Tournoud 和 Ori Pekelman 共同创建 Platform.sh 之前，Plais 共同推出了法语搜索引擎 Infoclic 和由开源项目 Drupal 支持的电子商务平台 Commerce Guys。Pekelman 曾与人合伙创立了两家咨询公司，Internet Patrol 和 Constellation Matrix，并且是 Commerce Guys 的董事会成员。至于图尔努，他帮助创办了商业公司，并在 Fontaine Consultants 担任 IT 战略顾问。

“我在 2015 年与我的合伙人图尔诺德和佩克尔曼一起创立了 Platform.sh，目标是简化 web 开发人员的生活，”普莱斯在接受电子邮件采访时告诉 TechCrunch。“管理 web 基础架构非常耗时，我们知道我们有解决方案可以让这一过程更加高效和经济。七年过去了，云并没有变得更简单。相反，由于容器和微服务的出现，it 变得更加复杂，更不用说对更快更好的性能和管理越来越多应用的能力的需求了。”

开发者可以使用 Platform.sh 的工具来部署和测试 web 应用。图片来源: Platform.sh

就 Plais 的观点而言，由低代码工具供应商 OutSystems 进行的一项民意调查发现，近一半的公司在 2019 年交付网络或移动应用的平均时间是五个月或更长。一个估计认为成本在 2 万到 6 万美元之间——当然，根据能力不同而不同。

使用 Platform.sh，不同的开发团队可以使用不同的语言和不同的框架来构建网站和 web 应用程序。从控制面板上，开发人员可以管理数据库和工作流，并创建生产副本，用于工作、测试和与利益相关方共享。在跨多个云提供商部署之前，可以对想法进行试点并将其融入生产。基础设施变更是有版本控制和可审计的，而构建依赖和测试由平台自动解决。

普莱斯认为 Heroku 是 Platform.sh 的主要竞争对手之一，其次是规模较小的公司，包括 Vercel 、 Netlify 和Acquia(Drupal 的商业部门)。Vercel 最近筹集了 1.5 亿美元，这凸显了对 web 应用开发工具的需求，与此同时，Netlify 获得了 1.05 亿美元。与此同时，Acquia 在 2019 年被私募股权公司 Vista Equity Partners 以 10 亿美元收购。

Plais 断言 Platform.sh 是“极具竞争力的”,尽管它在资金上处于劣势，其与众不同之处在于它支持混合的 web 开发解决方案(例如，框架)——这不是所有的竞争对手都能比拟的。

“Platform.sh 为公司提供了一个端到端的平台，可以帮助建立、托管和扩展一系列网站和网络应用，同时消除对 IT 和云操作的需求，”Plais 说。“首席执行官们，尤其是首席信息官们，非常清楚管理网站基础设施所需的时间和资源投入，尤其是当乘以几十甚至几百个网站时。Platform.sh 是预算敏感的 IT 领导者的理想解决方案，他们希望成为领航者，管理和允许团队进行实际的创新工作，而不是将所有的“创新令牌”投资于基础设施。”

当被问及 Platform.sh 的财务状况时，Plais 令人耳目一新地透明，透露当前的年度经常性收入为 4500 万美元——比 2021 年增长 50%。该公司拥有大约 5000 名客户，包括 Adobe、雀巢、英国《金融时报》和美国商会等机构，以及大学和政府机构，如法国军队和文化部。

“虽然市场显然会有一些逆风，我们需要适应，但我们实际上已经做好了受益的准备。作为一家公司，我们一直很节俭，资本效率很高，运营非常精益，并注重可持续发展，”普莱斯说。“在疫情早期，我们冻结招聘一个季度，以评估形势，并确保我们能够根据不断变化的市场调整我们的支出。这是一个艰难的决定，但在当时可能是正确的，否则我们的运营不会受到影响。”
