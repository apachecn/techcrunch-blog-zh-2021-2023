Rewind 希望通过 a16z | TechCrunch 的数百万用户来改变你的记忆方式

原文：https://web.archive.org/web/https://techcrunch.com/2022/11/01/rewind-wants-to-revamp-how-you-remember-with-millions-from-a16z/

虽然已经有不少破坏搜索引擎的尝试， Rewind 可能是我见过的第一次尝试改变我们在网络生活中搜索的方式。一次一个应用程序。

由丹·西洛克、Optimizely的联合创始人和前首席执行官和布雷特·贝切克创建的 Rewind 希望帮助人们恢复记忆。今天启动的这家初创公司，使用漂亮的技术来记录人们如何滚动和聊天度过他们的一天。它创建了一个可搜索的记录，记录了在 Zoom 会议期间发生了什么，谁说了什么，以及有人提出费用报告黑客攻击的每个实例。

“会议一结束，讨论、辩论和决定的内容往往就永远消失了，”西罗克说。“有了 Rewind，您再也不用担心会丢失这些内容……你只需搜索一个说过的词，一个出现在你屏幕上的词，就可以回到你正在寻找的会议的确切时刻。”

Siroker 将该应用程序比作助听器，他说在他 20 多岁开始失聪后，助听器改变了他的生活。“失去一种感觉，又重新获得感觉就像获得了一种超能力，”西罗克说。在 2020 年 10 月离开 Optimizely 后，他开始寻找不同的方法来增强人类的能力。根据 LinkedIn 的说法，他在 2018 年成立了一个基金会，“资助和开展科学研究，以加速我们走向人类思维仿真的道路。”

在产品形态上，这个目标看起来像倒带。Siroker 说，这家初创公司“使用 API 来确定在任何给定时间处于焦点的特定应用程序”，然后创建该行为的时间线。它还使用一个 API 来允许网站的深度链接，因此人们可以直接从搜索结果中打开 Chrome。

用户不需要集成 Gmail、Dropbox 或 Slack，而是可以下载并“倒带”开始捕捉应用程序。在实践中，如果你忘记了某人在一次开发者大会上提到的一个新的竞争对手应用程序的登录页面的 URL，你可以倒回去——哈哈——在你的一天中，找到会议中有人在屏幕上扔出链接的时刻，复制链接并粘贴。

至于“为什么是现在？”对于这个问题，赛洛克立即给出了答案:“苹果芯片(即 M1 和 M2 芯片)。没有它，我们就不能做我们现在做的事情。我们利用片上系统(SoC)的每一部分，在您的机器上本地完成所有工作。”

Rewind 声称，它可以在不损失质量的情况下，将原始视频记录数据压缩 3750 倍；“这意味着，即使你今天能从苹果买到的最小的硬盘，你也可以存储多年的连续录音，”该公司在今天的声明之前向 TechCrunch 提供的一份声明中说。(苹果不是投资人。)

Rewind 正面解决了任何应用面临的最大挑战之一——用户信任。这些录音存储在个人的 Mac 电脑上。理论上，这意味着该公司不接触数据。Rewind 说只有用户可以访问他们的数据。

Siroker 补充说，用户可以随时暂停和删除录音，不包括 Signal 或 1Password 这样的特定应用程序，或者进入隐身模式，称“默认情况下，我们不记录 Chrome 隐身或 Safari 私人浏览窗口)。

图片来源:倒带

在一台机器上存储一个敏感的、无所不包的、包含你所看到的、所说的或听到的一切的存储库仍然有一些风险。例如，如果你的电脑被入侵，恶意软件可能会窃取敏感数据。还有一种尴尬的舞蹈，用户未经允许在屏幕上录制某人，这在一些州是非法的。

Siroker 说，他建议用户向他们正在录音的人征求口头许可，强调只有用户才能访问录音。尽管如此，使用 Rewind 的用户可能不太倾向于抱怨他们的公司母公司或分享个人故事，因为他们知道在某个地方有记录(不幸的是，删除按钮不能追溯工作)。(更新:西罗克给我发邮件说删除按钮确实可以追溯使用)。

“虽然各州的法律有所不同，但我们认为隐私非常重要，因此我们建议我们产品的用户以比最低法律标准高得多的标准对自己负责，”西罗克在谈到主动录音同意时说。“这不仅是法律上最安全的方法，也是正确的做法。”

至于这是如何让人们更好地记忆，而不仅仅是更好地记住工作日中的随机事情，现在还没有定论。如果你回顾你的一天，这并不一定意味着你会记得更多的事情，这可能只是意味着你有更多的现成记忆。我想这有助于记忆，但绝对需要你坐在电脑前。

“长期的愿景是给人类完美的记忆，今天的产品是关于搜索的，”西罗克说。“这是我们可以产生最大影响的地方。如果你想一想，如果我们都已经有了完美的记忆力，我们就不需要搜索我们的电子邮件、短信、文件等等。去寻找我们以前见过的东西。我们只会记住他们。”

到目前为止，该公司已经在由安德森·霍洛维茨(Andreessen Horowitz)牵头的一轮融资中以 7500 万美元的估值筹集了 1000 万美元，首轮资本(First Round Capital)和其他公司也参与了融资。

Siroker 说，这款应用目前是免费的，但将来会有免费的每月订阅。而且，尽管他有帮助公司开发更好的市场广告活动的历史，西罗克说，倒带“永远不会出售你的数据或做广告。”
