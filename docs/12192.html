Genesis 用 Genesis X 敞篷车调侃其电动汽车的未来| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/11/15/genesis-teases-its-ev-future-with-the-genesis-x-convertible/

现代豪华部门 Genesis 周二晚上发布了 Genesis X 敞篷车，这是该品牌电动汽车未来的第三款也是最后一款电动汽车概念。

在 2022 年洛杉矶车展之前举行的一场引人注目的活动中，reveal 遵循 X 概念和 X Speedium Coupe 所有这些都属于 X 系列的灵感。这些都只是概念，很少在活动中分享任何细节。但是 Genesis X 敞篷车和它的概念兄弟不仅仅是设计练习。目的是为车辆，特别是 Genesis X 敞篷车的前面板，展示品牌的新电动汽车组合。

当被问及 X 敞篷车或类似的东西是否会很快出现在消费者的车库里时，现代汽车集团和创世纪汽车品牌的首席创意官卢克·唐克沃克表示反对。“这么说吧。我投入了巨大的能量，我的能量，让它成为可能，”他告诉 TechCrunch。

图片来源:阿比盖尔·巴塞特

X 敞篷车有一个带有透明天窗的硬顶车顶，当车顶关闭时，可以为四名乘客提供宽敞的感觉。车辆的前部得到了创世纪的新电动汽车的脸，这是一个创世纪盾徽格栅的抽象。期待这张新面孔出现在未来的电动汽车上。

根据新闻稿，概念车上的白色油漆被称为鹤白色，并加入了珍珠颗粒。内部是 Giwa 蓝色，带有 Dancheong 橙色缝线和面向驾驶员的驾驶舱。

图片来源:阿比盖尔·巴塞特

Genesis 称 X 敞篷车为“品牌的灯塔”。从反应来看，或许它有这个机会。当这款车在加利福尼亚州马里布的一座价值数百万美元的海滩别墅向媒体和嘉宾展示时，观众听到了喘息声——这种反应在此类事件中并不常见。

“我们必须利用这个机会为品牌注入更多的肾上腺素，”唐克尔沃尔克说。"如果有人认为电动汽车不性感，创世纪将展示完全相反的情况."

X 敞篷车与 2021 年宣布的 X Concept 和 2022 年宣布的 X Speedium Coupe concept 共享其架构和电动动力总成。

唐克尔沃克不愿证实这些概念的基础。他告诉 TechCrunch:“我们正在与正确的合作伙伴合作，以获得适当的性能，在我看来，这足以满足这一细分市场的需求。”

他继续说，X 敞篷车将“开辟一个新的，嗯，让我们说平台的基础，一个新的平台和模块化系统。”Donckerwolke 拒绝详细说明。

Genesis 继续推动设计向前发展，正如 Donckerwolke 提到的，“通常你去创造一些东西，你必须将你的设计限制到最常见的标准，”他说。“我们没有这个，我们和内部供应商谈过，我们说，我们需要那些座位。我们需要那些灯。这是，这基本上是让我们推动边界很多。这基本上就是让我们能够制造出这样的汽车的原因。”

Genesis 表示，其目标是“为客户提供享受的产品”，鉴于与会者的反应，Convertible X 概念车将品牌推向了这个方向。
