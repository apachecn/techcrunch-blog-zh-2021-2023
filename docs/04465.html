欧盟对今日俄罗斯和人造卫星的禁令现已生效

原文：https://web.archive.org/web/https://techcrunch.com/2022/03/02/eu-rt-sputnik-ban-live/amp/

图片鸣谢: 图片联盟 / Getty Images

在欧盟正式采取制裁措施后，欧盟对今日俄罗斯(RT)和斯普特尼克(Sputnik)的禁令刚刚生效，这意味着媒体监管机构现在预计将监督合规情况，欧盟各国的监管机构可能会对任何被发现继续传播克里姆林宫相关媒体公司内容的平台处以罚款。

这是对 RT 和 Sputnik 及其子公司*发行的广泛制裁——不仅包括传统广播频道(如卫星电视)，还包括在线平台和应用程序，正如我们之前报道的。

虽然两家媒体的个别记者目前没有受到制裁(RT 的总编辑已经受到制裁)，但法律文书包括一项反规避条款，该条款最终可能针对个人，即如果他们被认为试图规避对这些频道的限制。

欧盟官员表示，这也意味着互联网提供商预计将采取积极措施，确保 RT 和 Sputnik 的内容不会出现在他们的平台上。

因此，基本上，仅仅禁止官方渠道可能是不够的——如果其他用户/账户上传受制裁的内容，社交媒体和其他技术平台可能会采取进一步措施来防止禁令被规避。

除了社交媒体网络和视频流媒体服务，欧盟官员表示，原则上互联网服务提供商也包括在内。

鉴于可用的数字分销渠道范围广泛，委员会官员承认立即结束这两个渠道的所有区域分销存在挑战——暗示他们预计会有一定程度的“泄漏”，尽管他们强调——从法律上讲——遵守禁令现在是一种预期。

尽管欧盟表示，对 RT 和 Sputnik 的制裁是有时间限制的，但在实践中，取消制裁的附加条件让人们很难想象禁令会被取消——至少在俄罗斯现任总统弗拉基米尔·普京仍在克里姆林宫期间。

这是因为委员会规定，为了解除禁令，俄罗斯必须停止在乌克兰的侵略和停止反对欧盟及其成员国的宣传。许多媒体观察家认为，像今日俄罗斯这样一个频道的主要目的就是制造和扩大反西方的宣传。

为什么欧盟特别挑出 RT 和 Sputnik？该集团的评估是，它们是亲克里姆林宫工具箱中的关键虚假信息工具，被普京用来发动针对西方的破坏稳定的“信息战”,这场战争已经升级为俄罗斯通过入侵乌克兰对东欧的血腥入侵。

例如，委员会官员对 rt 和 Sputnik 获得的巨额国家预算(估计 2021 年€为 13 亿美元)和其他国家支持进行了标记，并指出对这些频道编辑独立性的怀疑。

他们还认为这些限制是经过精心平衡的，强调它们针对的是俄罗斯政府用来操纵和干涉外国信息的两个最突出的工具，而且这两个工具显然是可归因的。

为了支持这一说法，他们提到了美国国务院的一份报告，该报告分析了 rt 和 Sputnik 在俄罗斯“虚假信息和宣传生态系统”中的作用，这只是一个例子——他们认为这两个实体显然是普京宣传机器中“必不可少的工具性”部分。

他们还声称，欧盟的回应——针对两家媒体公司分销的制裁——并不审查意见。

该委员会进一步强调了过去一年俄罗斯国内对独立媒体的重大打压——指出使用严厉的国内立法，特别是其关于“外国代理人”的法律来压制被视为批评政府的独立媒体和记者个人——并将俄罗斯对专业记者的威胁描述为不可接受的。

来自欧盟委员会的明确信息是，俄罗斯入侵乌克兰已经不可逆转地改变了这一格局；欧盟对普京宣传机器的容忍已经结束。

欧盟主席乌尔苏拉·冯·德·莱恩在一份声明中证实了制裁的正式通过，他说:“在这个战争时期，语言很重要。我们正在目睹对一个自由和独立国家的这一无耻攻击的大规模宣传和虚假信息。我们不会让克里姆林宫的辩护者散布他们为普京的战争辩护的恶毒谎言，或者在我们的联盟中播下分裂的种子。”

在另一份支持声明中，欧盟外交事务高级代表约瑟普·博雷尔(Josep Borrell)补充道:“克里姆林宫系统性的信息操纵和虚假信息被用作袭击乌克兰的作战工具。这也是对欧盟公共秩序和安全的重大和直接威胁。今天，我们正在对普京的操纵行动采取重要步骤，并在欧盟关闭俄罗斯国家控制的媒体的水龙头。我们早些时候已经对 RT 的领导层实施了制裁，包括主编[Margarita] Simonyan，并且也将这些组织在我们的联盟内开展的活动作为目标是符合逻辑的。”

当被问及是否会有更多限制措施出台时——鉴于冯德莱恩周日的评论，她说欧盟正在“开发工具，在欧洲禁止(俄罗斯的)有毒和有害的虚假信息”——欧盟委员会官员拒绝对任何具体政策发表评论。

但他们总体上指出了欧盟多年来一直在制定的一系列措施，如其民主行动计划和通过行为准则采取的应对网上虚假信息传播的措施，并补充说，欧盟正在建立一套连贯的政策方法和措施，包括在态势感知和弹性建设等领域——它打算特别针对外国信息操纵。

欧盟官员补充说，对 RT 和 Sputnik 的制裁符合这一更广泛的战略。

对 RT 和 Sputnik 制裁的法律依据是欧洲理事会根据《共同外交和安全政策规则》(第 29 条)以及《欧洲联盟运作条约》第 215 条做出的一致决定，该条规定了欧盟委员会的限制性措施。

自由获取信息是《欧盟宪章》中规定的一项基本权利，但欧盟委员会表示，它相信这些有针对性的措施将抵御任何法律挑战。

欧盟官员证实，现有的欧盟媒体规则继续照常与制裁并行。

六家被制裁实体完整名单为:RT——今日俄罗斯英语；RT–今日俄罗斯英国；RT–今日俄罗斯法国；RT–今日俄罗斯德国；RT–今日俄罗斯西班牙语；还有人造卫星。
