测试自动化平台 Tricentis 收购 Tx3 服务

Tricentis ，一家资金雄厚的以企业为中心的测试自动化平台，最近正在进行一场收购狂潮。去年，收购了性能测试服务公司 Neotys。本月早些时候，该公司宣布它已经收购了 UI 测试初创公司 Testim，该公司现在宣布它也已经收购了 Tx3 服务。

与其他一些测试服务不同，Tx3 专注于一个单一的垂直领域:医疗保健和生命科学公司。它为他们提供了一个专为他们的合规性和审计需求量身定制的解决方案。例如，该公司提供了一个 DevOps 平台，允许生命科学公司遵守 FDA 的电子记录和签名法规。

不出所料，该公司计划利用 Tx3 的专业能力来加强其在生命科学行业的地位。它已经向这些公司推销其服务，部分是通过与 Tx3 的现有合作关系。

Tricentis 董事长兼首席执行官凯文·汤普森(Kevin Thompson)表示:“Tricentis 已经在帮助成千上万的组织更快、更高效地生产高质量的软件。“通过将我们强大的软件测试产品组合与 Tx3 的专业数字验证平台相结合，Tricentis 将为希望推进其数字化转型之旅的生命科学和医疗保健组织提供一个极其全面和强大的解决方案。”

就在几天前，Tricentis 还推出了一套新的解决方案，帮助其客户完成数字化转型项目，并采用基于云的测试。该公司最近还任命阿曼达·鲍里切夫斯基为首席法律官。她之前在网络安全管理软件产品担任副法律总顾问(在臭名昭著的黑客曝光之前)，最近在 Thrive Pet Healthcare 担任法律总顾问。该公司最近还聘请了达伦·贝克(也曾任职于网络安全管理软件产品)担任新的首席营销官。
