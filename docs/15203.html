亲爱的基恩:

感谢您向我提出您是否有资格获得 E-1 条约贸易签证的问题，并分享您关于扩大在美国业务的激动人心的消息。很高兴听到您已经在美国建立了您的母公司，以促进您在美国的融资。

2019 年，当美国与新西兰建立了商业和航行条约时，新西兰人有资格获得 E-1 条约贸易商签证和 E-2 条约投资者签证。设立 E-1 条约交易者和 E-2 条约投资者签证是为了促进和加强美国和几个其他国家之间的经济联系(值得注意的是，目前美国和印度、中国或巴西没有这种条约！).我在我的播客节目中讨论了这两种电子签证，以及更多针对特定国家人才的特殊签证。

对你的问题的快速回答是肯定的，你可能有资格获得 E-1 商人签证，但这将取决于你公司所有权的国籍和其他几个因素，因此值得进一步研究。

根据您在美国的市场扩张或建立团队和运营的计划，您也可能有资格获得 E-2 签证(甚至是其他几种类型的签证，如 O-1 或 L-1 、，这些签证并不针对特定国家)。让我来看一下每一种的要求、优点和缺点，并更深入地研究一下 L-1A。

E-1 条约贸易商签证

E-1 签证适用于在美国和条约国之间进行或帮助其公司进行商品、运输或服务(包括技术、银行和保险)的“实质性贸易”的条约国公民。移民法没有明确说明什么是“大量贸易”，所以对交易的美元价值或数量没有最低要求。然而，大规模和频繁的国际交易更有可能获得批准。
