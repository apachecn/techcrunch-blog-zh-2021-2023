凭借 1700 万美元的资金，Immerok 推出了面向实时流媒体数据的云服务

2011 年，Apache 软件基金会发布了 Flink，这是一个高吞吐量、低延迟的引擎，用于传输各种数据类型。十年后，由 David Moravek、Holger Temme、Johannes Moser、Konstantin Knauf、Piotr Nowojski 和 Timo Walther 创建的名为 Immerok 的初创公司开发了一个名为 Immerok Cloud 的 Apache Flink 云服务，它是无服务器的，抽象出了处理流媒体数据所需的服务器管理任务。

在过去几个月与“几十家”公司合作并在早期访问计划启动前夕，Immerok 今天宣布，它在由 CUSP Capital、468 Capital、Cortical Ventures 和 Essence VC 牵头的种子资金阶段筹集了 1700 万美元，天使投资人包括 Apache Flink 的联合创始人 Stephan Ewen。Immerok 首席执行官 Temme 表示，收益将用于扩大公司团队和进一步开发其云产品。

“Immerok 正在通过开发无服务器的 Apache Flink 云服务将实时带入主流，这将使所有公司都有可能创新新的实时客户体验和业务运营，”Temme 在一次电子邮件采访中告诉 TechCrunch。“当然，我们将继续与开源社区合作，推动 Apache Flink 的创新和传播。”

Temme 承认，Apache Flink 还有其他云服务，如亚马逊网络服务(AWS)的 Kinesis 数据分析和 Aiven 的全面管理的 Apache Flink 服务。(巧合的是，Temme 自己的云服务是由 AWS 托管的。)但他认为，Immerok 让开发人员可以以接近零的操作开销获得 Apache Flink APIs 的自由功能，让他们运行“全系列”的流处理用例，包括人工智能和机器学习。

“目前最具战略意义的 IT 计划包括人工智能、实时和预测分析。流处理是这些系统的一个基本组成部分，直到现在，组织都需要专家来构建、操作和扩展该基础设施，而专家是昂贵和稀缺的，”Temme 说。“Immerok 云用户创建和部署任意应用程序，这些应用程序持续处理数据……我们合作过的许多公司都依赖于[服务] 来处理数据，这些数据驱动着电子商务、安全、金融服务、娱乐和其他行业的实时人工智能系统。”

Immerok 目前在美国和欧盟拥有 20 名员工，其中 90%从事工程设计。在寻求提升服务的同时，Temme 预计该公司将在 2022 年底前在其工程、社区关系和客户成功团队中再雇佣 30 名员工。

在经济低迷时期，这可能是一次艰难的攀登。但 Temme 认为，公司将越来越多地依赖实时、数据驱动的平台来创造新的用户体验并提高业务运营效率，从而巩固对 Immerok 等解决方案的需求。

“对流处理应用的需求正在迅速增长。但是开发和操作流处理基础设施所需的技能是稀缺的——大多数已经被云和互联网巨头雇佣。Immerok 解决了这一挑战，它让公司能够大规模构建和部署流处理系统，同时消除了对熟练技术人员的依赖，”Temme 说。“我们正在让实时计算成为标准，而不是利基市场；公司将能够实时做出任何数据驱动的行动。”
