苹果讨论 iPadOS 16.4 的新铅笔悬停功能

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/27/apple-discusses-ipados-16-4s-new-pencil-hover-features/

去年 10 月，苹果宣布 hover 将被加入 Apple Pencil 的锦囊妙计中。对于非艺术家来说，这听起来像是一个小小的更新，但却是工作流的一个重要补充。虽然数字绘图在纠正错误方面比现实世界中的同行更加宽容，但停止和开始仍然是一个非常令人沮丧的方面，最终可能会阻碍创作过程。

悬停添加了在将线条提交到作品之前预览线条的功能。通过将铅笔 2 悬停在新 iPad Pro 上方 12 毫米处(苹果指出，正是 M2 使该功能成为可能，因此硬件选项有限)，该系统提供了线条和线条宽度以及颜色预览，以及在标记中选择绘图工具的能力。

苹果输入体验总监 Leslie Ikemoto 告诉 TechCrunch:“铅笔发出信号，iPad 随后会解释并计算出铅笔尖端在 3D 中的位置以及握笔的角度。”。

制作图形编辑和数字绘画应用程序 iPadOS 的 Savage Interactive 在该产品发布时似乎特别乐观。“这确实对我们的设计阶段产生了深远的影响，”首席执行官詹姆斯·库达当时指出。我想，当你的应用程序是 iPad 独有的时候，这些事情会产生特别大的影响。

“首先，IPad 是一款触摸优先的设备。你拿着这块玻璃，”平台产品营销总监 Stephen Tonna 说。“当你给它添加不同的配件时，比如键盘或触控板，它就会释放出不同的功能。铅笔增加了难以置信的体验。这是思考铅笔和 iPad 的另一种方式。”

今天早上，iPadOS 16.4 在倾斜和方位角方面对该功能进行了进一步的改进。倾斜实际上就是它听起来的样子。方位角对我来说是一个新的(事实证明，我们的工作人员设计师布莱斯)。鉴于我高中几何学得如此糟糕，我将在 TechCrunch 上做我做过的最多的高中学期论文，并引用牛津语言词典:

观察者观察到的天体的方向，用从地平线的北极或南极到穿过该天体的垂直圆与地平线相交的点的角距离来表示。

这些工具现在对希望在现有 Pencil 功能基础上构建的开发人员开放。“如果你看生殖，他们有铅笔刷，当你垂直于显示器时，它又小又细，然后随着你倾斜以获得阴影而变宽，”Ikemoto 说。“通过倾斜和方位角，Procreate 可以在你放下铅笔画笔时呈现出你要做的标记的精确轮廓，这对他们的用户来说是一个巨大的加速器。”

该公司大约在 7.5 年前进入平板电脑手写笔游戏，宣布了第一支与最初的 iPad Pro 一起的铅笔。与此同时，Pencil 2 在三年后与第三代 Pro 一起上市，将无线磁充电带到了桌面上。当然，苹果不会对路线图或即将推出的新功能的开发发表评论。但除了上述研究，该公司表示，更新在一定程度上是由用户反馈决定的。毕竟，艺术家通常是一个特殊的群体，致力于随着时间的推移而发展起来的特定工作流程。

“我们一直在倾听客户的反馈，”Tonna 说。“打造优秀产品的一个重要部分是听取反馈并倾听正在发生的事情。当然，还要制造我们的客户想都没想过的东西。他们看到它，他们喜欢它，他们走得更远。我们倾听。”

这些特性通过各种 API 提供给开发人员。“我们希望 hover 尽可能容易适应，所以我们使用了用于 Magic Keyboard 触控板的现有 API，”Ikemoto 补充道。“那是 UIPointerInteraction 和 UIHoverGestureRecognizer。对于已经在其应用程序中采用 UI 指针交互的开发者来说，他们无需做任何工作就可以免费获得 Apple Pencil hover。它会工作的。想要更多自定义悬停体验的开发人员可以使用 UI 悬停手势识别器，我们已经扩展了该识别器，以在 3D 中报告笔尖的位置以及握笔的角度。”

自然，生育是新技术的早期接受者。“自从 Apple Pencil 推出以来，我们一直非常期待 Apple Pencil hover 的可能性，”Procreate 首席产品官克莱尔·德埃斯特(Claire d'Este)在对 TechCrunch 的评论中说。

然而，该公司告诉 TechCrunch，对于铅笔来说，在 iPad 上画画并不比手写更受欢迎。悬停功能也适用于各种应用程序，包括 Notes 和 Safari。它提供了类似的工具和预览功能，就像你在《生育》中看到的那样。

“你有这个装置，你可以把它放在你的桌子上，在讲座中做笔记，然后当你完成时，你可以把它放在一个神奇的键盘上，”Tonna 说。“你可以用键盘写你的学期论文。[……]你可以把这些手写的笔记直接粘贴到你的论文中，它们就变成了打印文本。”

在生产线上工作总是会涉及到大量的狗食。对于 Pencil 来说，这意味着确保团队中有一些业余艺术家，在向开发人员和用户更广泛地推广之前，实际上使用新的补充产品。

Ikemoto 说:“该团队每年 10 月都会参加 Inktober，在那里我们每天都会画出不同的概念。“Inktober 是艺术界的一件大事。一天发表一个词，像‘植物’，每个人都会画一株植物。这是在团队中分享作品的好方法。”

刚刚发布的 iPadOS 16.4 还带来了 Safari web 应用主屏幕推送通知、21 个新表情符号和各种错误修复。
