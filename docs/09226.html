Pliops 获得 1 亿美元用于加速分析的芯片

原文：https://web.archive.org/web/https://techcrunch.com/2022/08/16/pliops-lands-100m-for-chips-that-accelerate-analytics-in-data-centers/

分析企业内部生成的数据，例如销售和采购数据，可以获得改善运营的洞察力。但是一些组织正在努力有效地处理、存储和使用他们的大量数据。根据 Seagate 委托进行的 IDC 调查 显示，组织仅收集了其业务线中 56%的可用数据，而在这 56%中，他们仅使用了 57%。

部分问题在于数据密集型工作负载需要大量资源，而添加必要的计算和存储基础架构通常成本高昂。IDG 报道称，对于那些专门转向云计算的公司，他们计划今年在基础设施上投入 7800 万美元。36%的人认为控制成本是他们面临的最大挑战。

这就是为什么 Uri Beitler 推出了 Pliops ，这是一家为企业和云数据中心开发他所谓的“数据处理器”的初创公司。他声称，Pliop 的处理器旨在提高数据库和其他运行在闪存上的应用程序的性能，从长远来看可以节省资金。

“很明显，今天的数据需求与过去的数据中心体系结构不兼容。Beitler 在一次电子邮件采访中告诉 TechCrunch 说:“大规模数据增长与传统的计算和存储缺点相冲突，导致计算速度变慢，存储瓶颈和网络效率下降。“虽然 CPU 性能在不断提高，但速度跟不上，尤其是在加速性能至关重要的领域。事实证明，添加更多基础架构往往成本高昂且难以管理。因此，组织正在寻找将 CPU 从计算密集型存储任务中解放出来的解决方案。”

Pliops 并不是第一个销售用于数据分析的处理器的公司。Nvidia 出售 BlueField-3 数据处理单元(DPU)。Marvell 拥有 Octeon 技术。甲骨文的 SPARC M7 芯片有一个数据分析加速器协处理器，带有一套专门的数据转换指令。在初创公司领域，Blueshift Memory 和 Speedata 正在创建硬件，他们说这些硬件执行分析任务的速度比标准处理器快得多。

但 Pliops 声称比大多数公司走得更远，与客户(尽管未具名)进行了部署和试点，包括金融科技公司、“中型”通信服务提供商、数据中心运营商和政府实验室。这家初创公司的早期吸引力似乎赢得了投资者的支持，他们向今天结束的 D 轮融资注入了 1 亿美元。

Koch Disruptive Technologies 领投，SK Hynix 和 Walden International 的 Lip-Bu Tan 参与，使 Pliops 迄今筹集的总资本超过 2 亿美元。Beitler 表示，这笔资金将用于构建公司的硬件和软件路线图，加强 Pliops 与合作伙伴的合作，并扩大其国际员工人数。

“我们的许多客户在新冠肺炎疫情期间取得了巨大的增长，这在一定程度上归功于他们对新的工作环境和不确定性条件的快速反应能力。普利奥普斯当然有。虽然一些客户受到供应链问题的影响，但我们没有，”贝特勒说。“我们看不到数据增长有任何放缓，也不需要利用它。Pliops 在最近一轮融资之前就很强大，现在甚至更强大。”

加速数据处理

Beitler 是三星以色列研究中心高级内存解决方案的前总监，他在 2017 年与 Moshe Twitto 和 Aryeh Mergi 共同创立了 Pliops。Twitto 是三星开发闪存信号处理技术的研究科学家，而 Mergi 在加入 Pliops 之前，共同创办了许多初创公司，包括两家被 EMC 和 SanDisk 收购的公司。

Pliop 的处理器为固态硬盘(SSD)和在线压缩提供了驱动器故障保护，在线压缩是一种通过找到相同的数据序列，然后只保存第一个序列来缩小数据大小的技术。Beitler 声称，该公司的技术可以在扩展容量的同时减少驱动器空间，在存储中映射“可变大小”的压缩对象以减少浪费的空间。

Pliops 处理器的核心组件是其硬件加速的键值存储引擎。在键-值数据库中——数据以“键-值”格式存储并为读写而优化的数据库——键-值引擎直接管理所有持久数据。Beitler 认为，运行这些引擎时，CPU 通常会被过度利用，导致应用程序无法充分利用 SSD 的功能。

“组织正在寻找将 CPU 从计算密集型存储任务中解放出来的解决方案。我们的硬件通过利用新一代硬件加速数据处理和存储管理技术来帮助创建现代数据中心架构，这种技术在性能、可靠性和可扩展性方面提供了数量级的改进，”Beitler 说。“简而言之，Pliops 能够从现有基础设施投资中获得更多收益。”

Pliops 的处理器于去年 7 月上市。Beitler 说，开发团队当前的重点是加速机器学习用例的数据摄取，这些用例在 Pliops 的当前和潜在客户中已经增长。

前方的路

当然，Pliops 有自己的工作要做。Nvidia 在数据处理加速器领域是一个强大的竞争对手，它花了数年时间开发其 BlueField 产品线。AMD 以 19 亿美元收购了 DPU 供应商 Pensando ，显示了其更大的野心。

加入开放可编程基础设施项目(OPI)是一个可能为 Pliops 带来红利的举措，OPI 是 Linux 基金会下一个相对较新的项目，旨在围绕数据加速器硬件创建标准。虽然 Pliops 还不是会员(目前的会员包括英特尔、英伟达、Marvell、F5、红帽、戴尔和是德科技)，但有理由相信成为会员可以将其技术展示给更大的客户群。

当被问及 OPI 时，Beitler 表示反对，但他指出，数据加速市场仍处于萌芽状态，并在不断增长。

“我们继续看到基础设施和应用程序团队因性能不佳的存储和无法满足公司数据需求的应用程序而不堪重负，”Beitler 说。“总体反馈是，我们的处理器是一种改变游戏规则的产品，如果没有它，公司需要在软件和硬件工程方面进行多年投资，以解决同样的问题。”
