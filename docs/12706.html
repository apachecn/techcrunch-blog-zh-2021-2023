AWS 推出 DataZone，一种新的基于 ML 的数据管理服务

在其 re:Invent 会议上，AWS 今天宣布了亚马逊数据区，这是一种新的数据管理服务，可以帮助企业编目、发现、共享以及最重要的是管理他们的数据。这里有趣的部分是，AWS 正在使用机器学习来帮助企业建立这些数据目录，并生成元数据以使其可搜索。

AWS 首席执行官 Adam Selipsky 在今天的主题演讲中表示:“为了释放数据的全部力量和价值，我们需要让合适的人和应用程序在需要时轻松找到、访问和共享合适的数据，并确保数据的安全。”。

图片来源: AWS

该工具将为用户提供细粒度的控制来管理和治理这些数据。长期以来，这一直是企业面临的一个主要问题，但随着数据量的增加，确保正确的用户能够访问正确的数据，而不泄露个人身份信息等问题变得更加困难。

Selipsky 解释说:“数据区使管理员和数据管理员能够轻松管理和控制对数据的访问，从而使您能够在整个组织中安全地释放数据。“它让数据工程师、数据科学家、产品经理、分析师和其他业务用户能够轻松地发现、使用这些数据并围绕这些数据开展协作，从而为您的业务提供见解。”

数据区用户将可以访问一个门户，在那里他们可以设置自己的数据目录并定义分类。一旦数据区连接到数据源，它将使用机器学习用其元数据填充其目录，用户可以根据需要添加附加标签和描述。

在一个有点相关的声明中，AWS 也昨天发布了其数字主权承诺，承诺给用户提供他们需要的工具来控制他们的数据存储和访问的位置。数据区确实提供了其中一些控制，尽管它的重点显然不是数字主权。
