亚马逊通过企业级桥梁| TechCrunch 扩展其 Sidewalk 物联网网络

原文：https://web.archive.org/web/https://techcrunch.com/2022/01/06/amazon-expands-its-sidewalk-iot-network-with-an-enterprise-grade-bridge/

早在 2019 年，亚马逊首先宣布其人行道网络，这是一种新的低带宽、长距离无线协议和网络，用于连接智能设备——例如，当你自己的 Wi-Fi 网络出现故障时，通过搭载邻居的网络保持它们在线。自去年以来，亚马逊一直在将其 Echo 设备转变为人行道桥，选择环和瓷砖设备现在可以访问网络。现在，亚马逊正在推出其首款专业级人行道设备，旨在覆盖大学校园或公园等大片区域。

这款新设备的全名很拗口:亚马逊人行道桥 Pro by Ring。它可以安装在室内，但主要是为了安装在室外——最好是在高处——并且可以覆盖五英里以外的数百台设备(当然，这取决于当地的情况)。

为了测试这些设备，亚马逊与亚利桑那州立大学合作，将在坦佩校园的灯柱上安装这些新的人行道桥。大学技术办公室计划将其用作概念验证，计划连接阳光和温度传感器、二氧化碳探测器和粒子计数器。

图片来源:亚马逊

亚马逊还与专门从事环境监测的物联网公司 Thingy 合作，安装其空气质量监测工具，以提醒潜在野火的第一反应人员。

“亚马逊 Sidewalk Bridge Pro 为我们带来了在大量所需位置的[远程]能力，与 AWS 中现有应用程序的轻松集成，以及设备和应用程序的可信安全性。Thingy 首席执行官兼联合创始人斯科特·沃勒(Scott Waller)表示:“我们很高兴与亚马逊 Sidewalk 合作，用我们的传感器测量空气质量和野火，并帮助解决这些关键应用的连接挑战。

但除了设备本身，亚马逊继续投资人行道生态系统才是最重要的。

“我们正在建立一个网络，我们正在让参与者帮助物联网行业，”亚马逊 T2 人行道的主管斯特凡诺·蓝迪告诉我。"归根结底，如果我们希望推动智能和互联设备在各地的普及，您需要拥有合适的网络。如果你今天与物联网开发商交谈，是的，有许多选择，但要么从连接角度与蜂窝相比非常昂贵，要么范围有限，要么正在耗尽电池，要么只是整体开发周期太复杂。因此，我们认为我们应该投资，这就是我们所做的，我们将继续投资支持这些网络，以便物联网社区可以构建任何类型的应用:消费者、企业、公共部门。”

蓝迪指出，在推出该网络仅几个月后，该公司现在已经在美国 100 多个主要城市地区拥有非常强大的住宅覆盖。当然，在某种程度上，这是因为美国家庭中有很多 Echo 设备，除非用户选择退出，否则大多数现代 Echo 智能扬声器现在都默认启用了 Sidewalk。并非所有人都对此感到满意，尽管亚马逊会辩称，它将自己的网络设计为隐私优先，不会占用太多带宽(毕竟，它主要是用于传递警报，而不是你的环形摄像头的视频馈送)。但是可以合理猜测，大多数用户甚至不知道 Sidewalk 这个名字。

不过，覆盖一个住宅区是一回事。有了 Sidewalk Bridge Pro，企业现在还可以覆盖整片土地来连接他们的传感器。这似乎有一些需求，因为蓝迪指出，“超过几千家公司”已经联系亚马逊，询问商业用例——大多与该公司基于云的受管物联网服务 AWS IoT 有关。蓝迪说，这种兴趣很大程度上来自那些希望构建公共部门解决方案的公司，主要围绕智能城市服务。

“人行道桥专业版是一种专业级的桥梁，完全是为部署在那些(住宅区)以外而定制的，”蓝迪解释道。“所以现在你几乎可以覆盖任何地方。想想商业中心、公园、城市公园、州立公园、市政公园、荒野地区、商业区等等。现在，你真的带来了无处不在的连接，所以当你在那里构建解决方案时，你知道覆盖将几乎覆盖你需要的任何地方。”

蓝迪指出，虽然他预计大多数用户将在室外安装该桥，但它也可以在室内用于覆盖仓库或大型商店。尽管它被明确称为“专业版”，但我们不应该期望该公司很快就会推出消费者风格的“非专业版”。毕竟，这就是 Echo 和 Ring 设备的用途。
