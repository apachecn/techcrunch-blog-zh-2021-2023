Web3 并不是你所认为的安全补丁

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/12/web3-security-web2-problems/

谎言之网:Web3 并不是你所认为的安全补丁

web3 的拥护者会告诉你，与 web 2.0 相比，去中心化的 Web 带来了更大的弹性和安全性，这要归功于其基于 T2 区块链 T3 的技术。

Web 2.0 于 21 世纪初首次亮相，专注于用户生成的内容、丰富的用户界面和合作服务，同时也带来了新一轮的安全威胁，包括恶意软件、网络钓鱼、社交工程、欺骗、跨站脚本、SQL 注入和数据泄露，仅举几例。

Web3，一个包含多种技术的术语，如加密货币，NFTs 和 DAOs ，无疑给人一种印象，它将使这种威胁成为过去:web3 不仅让人们对他们的数据有更多的控制权，而且它依靠分布式技术，如区块链，来消除其前身的许多缺陷。

然而，事实上，web3 并不比 Web 2.0 更安全，它已经为机会主义网络罪犯创造了一个新的游乐场。这是因为尽管它代表了互联网的功能和用途的转变，但它并没有从根本上改变互联网的工作方式。

新的和未改进的

虽然它有望完全去中心化，但 web3 面向用户的组件主要基于 Web 2.0 技术，如 API 和端点，尽管它们是基于区块链技术构建的。这意味着 web3 服务和分散式应用程序(dApps)的用户继续依赖传统技术进行交易，并最终意味着 web3 容易受到困扰其前身的所有经典安全问题的影响，从 DNS 劫持到跨站脚本。Web3 公司还必须与他们的用户沟通，主要是通过 Web 2.0 技术，如电子邮件或在线消息，这些也容易产生遗留的安全问题。
