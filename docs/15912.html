Firefox 将其反跟踪保护扩展到 Android

Firefox 今天宣布，其保护用户免受追踪器攻击的 Total Cookie Protection (TCP)功能现在可以在 Android 上使用。该功能在默认情况下是打开的，它将阻止跨站点跟踪。这样，追踪者就无法收集你的浏览行为数据来做有针对性的广告。

TCP 于 2021 年推出，但仅限于 Firefox 的增强跟踪保护(ETP)模式。因此，用户必须手动选择该安全级别来防止基于 cookie 的跟踪。去年，该公司在 Windows、Mac 和 Linux 的 Firefox 上提供了 TCP，并在所有模式下默认打开。浏览器并没有完全摆脱 cookies。相反，它为每个站点维护一个“单独的 cookie jar ”,以在这个竖井中维护您的数据。

Android 上的 TCP 将从今天开始向用户推出，并将在下个月向所有用户提供。

与 Firefox 相反，谷歌已经推迟了在 Chrome 上取消第三方 cookies 的行动——现在计划在 2024 年的某个时候推出。

Mozilla 还在测试一项新功能，即在网站注册过程中动态生成 Firefox 中继——一种电子邮件代理服务。该公司表示，在一些网站上——Firefox 没有指明是哪一个——该工具会提示用户使用现有的代理电子邮件地址或创建一个新的。Mozilla 表示，它计划在今年晚些时候将这一功能扩展到所有用户和更多网站。

Firefox Relay 的屏蔽服务提供了五个免费的电子邮件地址。但是有了它的高级服务——每月 0.99 美元起的计划——你可以获得无限的电子邮件地址。

上个月， Firefox for Android 也获得了新的扩展,可以在分享 URL 和文章列表之前删除跟踪元素。
