Facet 筹集了 1300 万美元来增加 Photoshop 梦寐以求的照片编辑功能

原文：https://web.archive.org/web/https://techcrunch.com/2021/11/16/facet-raises-13m/

简而言之，该公司已经创建了一个人工智能驱动的照片编辑工具，可以使用 API 访问。这意味着你可以进行非常强大的批量照片编辑，这有点像 Snapchat 的照片过滤器、Adobe Lightroom 的批量编辑功能、Photoshop 的灵活性和 Figma 的协作能力等的混合。然而，该工具能够做迄今为止照片编辑领域尚未见到的事情。刻面刚从两个适马风投 处融资 1300 万美元有来自 Accel ， 基础设置风投 ， 慢风投南园公地 。

说我对摄影的兴趣超过一般水平有点轻描淡写——我曾经是一名专业摄影师，写过 20 多本关于摄影的书。因此，当 Facet 联系我，告诉我他们在照片编辑领域有了一些全新的东西时，我非常兴奋，随后是“极度困惑”即使在与公司创始人和投资者交谈了一个小时后，我仍然无法理解这个工具到底是什么，也不知道它是为谁服务的。

当我有机会亲自使用这个工具时，一切都变得清晰多了。我上传了一组舞蹈照片，是我最近在奥克兰湖边的呼唤舞蹈活动上拍摄的，让 Facet 去做吧。基于网络的编辑器有一个非常陡峭的学习曲线——如果你愿意，可以说是一个学习墙——但我很快就能完成一些在 Photoshop 中很难甚至不可能完成的编辑工作。

我创建的一个过滤器是“检测背景，然后模糊和去饱和背景。”图像上的前景/背景检测并不完美，但对于它工作的照片来说，这是一种非常快速的方法来使照片流行起来——而不必在照片编辑套件中打开和编辑每个单独的图像。

左边的图像的背景被小平面自动去饱和和模糊。原件在右边。结果并不完美，但 Facet 能够在几分钟内对 200 张照片做到这一点——这是人工智能图像编辑的一个令人难以置信的壮举。照片由 Haje Kamps 拍摄，由 Facet 的 AI 编辑。

Facet 特别针对商业级的图像编辑，在这种情况下，你需要准备和展示大量的照片，但是——和 Photoshop 一样——它可以有几十种不同的用途，只受图像制作者的创造力限制。

“当您编辑图像并对这些变化进行分层时，我们会分析每一次编辑，并找出如何在更大的内容库中进行传输，从而自动构建您的预设。Facet 的首席执行官兼联合创始人乔·雷辛格解释说:“这对于在整个广告活动中保持品牌一致性非常重要，可以确保你的所有产品和照片都是一致的。”。“Spotify 就是一个例子。你可能已经看到了它们在专辑封面上非常出名的双色调效果。我们可以创建并为您提供一个带有 API 端点的可重复使用的图像编辑管道，因此您可以非常快速地处理数千张照片。”

该公司的选择和过滤工具非常强大，并且具有无限的可扩展性。该公司确实有一个消费者级平台，爱好摄影师可以使用，但该公司真正闪光的地方是创意软件开发人员使用该工具来使用 API。

雷辛格说:“我们没有试图让陈旧的、以印刷为中心的软件适应互联网时代，而是通过一个内容感知的图像编辑平台，从头开始构建创意人员需要的工具。”。

刻面接口的示例。在这个截图中，我已经让工具去饱和并模糊背景。当它工作时，这是不可思议的，这是我多年来一直希望 Lightroom 能够做到的事情。当它不起作用时(见中间两张照片，只有女人的腿是彩色的，或者工具未能检测到舞者的脸作为前景)，它有点令人失望。话虽如此，还是有可能将图像下载为分层的 Photoshop 文件，因此整理这些文件是很简单的事情，而且在批量编辑过程中节省的时间将是巨大的。图:来自刻面工具的屏幕截图

“我非常喜欢 Facet 的一个方面是它支持异步协作。您可以定义照片的样式，设计师可以在许多不同的照片上使用相同的样式，而不必手动编辑每张照片。你可以通过编程对照片的外观和感觉进行编码，并将其从一幅图像复制到另一幅图像，”两个适马风险投资公司的合伙人丹·阿贝隆说，他是 Facet 这轮投资的主要投资者。“你开始进入更加社区化的一面——如果你喜欢某人的风格，你可以将其应用到你自己的图像中，这开启了一个实时协作的世界。”

“这不仅仅是为了赚钱。Facet 将对创意社区和更广泛的网络产生重大影响。我可以说，他们想在整个网络上留下印记，这也是我们真正感兴趣的地方，”Abelon 说。

该公司将利用其最近扩大的战争资金，在首轮融资后采取显而易见的下几个步骤:扩大团队，寻找更多的牵引力，并建立一个走向市场的战略。你可以在 Facet 上注册一个免费试用版来踢踢轮胎，和计划的起价是专业用户每月 24 美元，有 API 需求的高端团队用户每月 50 美元。
