Instagram 正在测试一个用于帖子和卷轴的应用内调度工具

社交媒体经理和创作者长期以来一直依赖第三方工具在 Instagram 上安排帖子。但现在该公司正在测试一个应用内工具来安排帖子和卷轴。

该公司证实了这一进展，并表示确实在进行这方面的实验。Meta 的一位发言人在一份声明中说:“我们正在测试与我们全球社区的一定比例安排内容的能力。”。该公司没有提供该功能全面上市的任何细节。

Twitter 用户 @WFBrother 贴出了一张展示其工作原理的截图。拥有此功能的用户可以在创建新帖子或卷轴时进入高级设置，并打开“安排此帖子”开关来选择时间和日期。他们还指出，你可以通过汉堡菜单下的“预定内容”部分看到预定的帖子。如果需要，该部分还允许您重新安排内容。

应用研究员 Alessandro Paluzzi 注意到 Instagram 早在七月就在开发日程安排功能。今年 8 月，该公司告诉 TechCrunch，“这是一种体验的早期原型，我们没有公开测试。”

2018 年，该公司开始向企业提供通过 API 安排帖子的能力，以便 Hootsuite 和 SocialFlow 等第三方软件可以利用这一点。

自 2020 年以来，Meta 一直通过脸书 Creator Studio 提供 Instagram 日程安排功能。然而，新的应用内日程安排工具可以让那些没有社交媒体管理专用软件的创作者受益。Instagram 引入原生日程安排较晚，因为 Twitter 已经有这个功能好几年了。

本月早些时候，该公司宣布增加平台上的广告负载，以对抗 Meta 的收入下滑。
