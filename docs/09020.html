亚马逊将手掌扫描支付技术扩展到全食超市的 65 个位置

原文：https://web.archive.org/web/https://techcrunch.com/2022/08/10/amazon-expands-palm-scanning-payment-tech-to-65-more-whole-foods-locations/

亚马逊的“One”手掌扫描仪支付技术将在加州超过 65 家全食超市推出。这是迄今为止最大的一次部署，马里布、蒙大拿大道、圣莫尼卡、洛杉矶、奥兰治县、萨克拉门托、旧金山湾区和圣克鲁斯的商店都获得了旨在实现零售购物现代化的技术。

当 2020 年第一次宣布收银设备时，Amazon One 在 Amazon Go 商店有售，最终扩展到奥斯汀、西雅图、洛杉矶和纽约市的全食商店。顾客也可以尝试在加利福尼亚州格伦代尔的亚马逊时尚商店以及 select Fresh and Go 商店用手结账。

亚马逊 One 是该公司使命的一部分，使用“非接触式”技术使支付更快。这项技术的工作原理是这样的:用户访问参与地点的信息亭或销售点，将他们的掌上电脑和支付卡与这项服务联系起来。然后，在结账过程中，他们所要做的就是将手悬停在扫描仪上以完成交易。

Amazon One 使用机器学习创建手掌签名来识别客户。虽然 kiosk 拍摄了用户手掌的照片，但该公司表示，它不会将图像存储在那里，而是对其进行加密，并将其发送到服务器进行匹配。

随着亚马逊客户继续放弃他们的数据以获得更方便的购物体验，对隐私的担忧也在增加。

如果你使用 Face ID 或指纹扫描仪，你已经使用了生物特征数据。然而，一些用户可能不同意亚马逊 One 允许公司跟踪你的行动。

甚至一群美国参议员也表达了他们对手掌扫描系统的担忧。参议员艾米·克洛布查尔、比尔·卡西迪和乔恩·奥索夫在给亚马逊首席执行官安迪·贾西的一封公开信中写道，“与苹果的 Face ID 和触控 ID 或三星 Pass 等生物识别系统不同，这些系统将生物识别信息存储在用户的设备上，据报道，亚马逊 One 将生物识别信息上传到云端，带来了独特的安全风险。”

去年，亚马逊与票务公司 AXS 合作，计划在科罗拉多州丹佛市的红石圆形剧场实施亚马逊一号。声明发布后不久，数百名音乐爱好者、艺术家和人权组织要求红石放弃这项技术，并禁止所有生物监测工具，如手掌扫描和面部识别。他们甚至签署了一封信，提到了对亚马逊与政府机构共享掌纹数据以及潜在黑客从云中窃取数据的担忧。

人们被它困扰是没有错的，因为亚马逊已经因存储 Alexa 语音数据而闻名。此外，亚马逊向美国执法部门出售生物面部识别服务，其环形相机公司也与警方合作。

该公司可能还发现了更多的数据收集方法，因为上周收购了 iRobot ，这是一款装载先进传感器的智能房间测绘机器人吸尘器。
