AWS 表示，随着客户继续削减成本，新年伊始，增长率降至 15%左右

昨天，亚马逊公布了收益，AWS 本季度同比增长 20%,达到 210 亿美元。但也许更令人不安的是，该公司在与分析师的收益电话会议中报告称，新年第一个月的增长率进一步下降至 15%左右，因为随着客户寻找削减账单的方法，云继续普遍放缓。

“展望未来，我们预计至少在未来几个季度，这些优化工作将继续阻碍 AWS 的增长。到目前为止，今年第一个月，AWS 的同比收入增长在 15%左右，”CFO Brian Olsavsky 在电话会议的开场白中说道。

对于一个多年来一直享有高增长率的部门来说，15%左右的增长率代表着一个非同寻常的下降，这在财报电话会议上没有被忽视。分析师当然很好奇这是否是一个长期趋势，但 Olsavsky 真的没有准备好预测这个季度以后的情况。

“因此，就 AWS 增长率而言，我不确定我能为你预测本季度之后会发生什么。这是一个经济上的未知领域。正如我们提到的，客户群正在发生一些独特的事情，我认为这个行业的许多人都看到了同样的事情，”他说。

值得注意的是，Olsavsky 还报告了 850 亿美元的年营收增长率，这表明尽管面临经济逆风，AWS 仍然是一项非常健康的业务。“也就是说，退一步说，我们的新客户渠道保持健康和强大，而且有许多客户继续实施迁移到云的计划，并长期致力于 AWS。”

亚马逊首席执行官安迪·贾西(Andy Jassy)职业生涯的大部分时间都在运营 AWS 的公司度过，他表示，随着客户寻求削减成本，短期增长将会放缓，但仍有大量云市场可以征服低迷后的市场。

“因此，我认为记住全球 90%到 95%的 it 支出仍在内部也是有用的。如果你相信这一点，这个等式将会改变和翻转。他说:“我认为内部部署永远不会消失，但我真的相信在未来 10 到 15 年内，大部分都将出现在云中……这意味着我们在 AWS 业务方面还有很大的增长空间。”。

或许如此，但对于一个多年来一直是该公司增长引擎的业务来说，AWS 目前的放缓仍然令人担忧。
