富士康凭借电动皮卡和跨界车进一步进军汽车领域 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/10/18/foxconn-pushes-deeper-into-automotive-with-an-electric-pickup-and-crossover/

制造苹果(Apple)iphone 手机的台湾制造业巨头富士康(Foxconn)周二在其第三届鸿海科技日(Hon Hai Tech Day)上发布了两款电动汽车概念，该公司试图实现业务多元化，并进军汽车市场。

这两款车，一款 V 型电动皮卡和一款 B 型电动跨界掀背车，是给汽车制造商(特别是特斯拉)的一个信息:富士康开始营业了。

在活动中，富士康董事长刘永好表示，该公司希望将其在生产消费电子产品方面的成功复制到为汽车制造商生产电动汽车上。杨威甚至说，他希望富士康有一天能为特斯拉生产汽车。

富士康的 Model V 电动皮卡。图片来源:富士康

在美国，该公司已经在与洛兹敦汽车公司(Lordstown Motors)和菲斯克汽车公司(Fisker)合作生产电动汽车，并且还在与台湾汽车制造商裕隆集团(Yulon Group)合作生产其电动 SUV Model C，这是一款于去年这个时候在富士康最后一次科技日首次亮相的电动 SUV。

虽然主要活动是 Model B 跨界车和 Model V 皮卡，但富士康也展示了 Model C，该公司表示，这款车可以在 3.8 秒内从 0 到 60 英里/小时，续航里程为 435 英里。富士康预计 Model C 将于明年下半年交付台湾裕隆。

富士康 B 型电动跨界车。图片来源:富士康

据公司称，B 型跨界车将基于 C 型平台，但将采用新的车身设计。它将采用飞机上常见的 S 形管道设计，这是一种流线型的屋顶和空气幕，可以减少空气湍流，并允许 0.26 的阻力系数。富士康称，B 型车的续航里程约为 280 英里。

Model V 的细节很少，但富士康表示，这将是台湾第一款自主开发的电动皮卡，有效载荷可达 1 吨，牵引能力为 3 吨(当然没有说明牵引将如何影响范围)。

这款全地形电动皮卡将有双驾驶室、五个座位的配置，并将在车辆周围配备一系列传感器，富士康表示，“不仅提高了安全性，还为用户提供了智能技术。”我们将其解释为高级驾驶辅助系统。

富士康的战略严重倾向于垂直整合。该计划旨在实现软件和硬件的标准化，这些软件和硬件将在汽车上共享，并帮助其汽车制造商客户，如洛兹敦汽车公司和菲斯克公司。

垂直整合当然不是什么新鲜事。特斯拉因此而闻名。今天，越来越多的汽车制造商，包括像通用汽车这样的传统公司，正在进一步推进这一战略。

富士康称其 HHEV。操作系统软件平台将有助于缩短车辆的开发时间，以便更快地投放市场。

“我们以前做 PC 和手机。从现在到未来，我们将创造电动汽车，”Young-way 在一份声明中说。“在电动汽车行业，我们对 CDMS 很坚定:这意味着合同设计和制造服务。这一承诺不会改变。未来 10 年，EV 行业的鸿海将重新定义汽车领域的 CDMS，继续推进垂直整合的技术服务。”
