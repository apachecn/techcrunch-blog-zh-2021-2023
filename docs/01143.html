脸书下令出售吉菲

原文：https://web.archive.org/web/https://techcrunch.com/2021/11/30/faecbook-giphy-cma-order/

在对大型科技公司通过纯粹的购买力保持市场主导地位的能力的重大打击中，英国竞争监管机构已经命令脸书(现为 Meta)撤销其对动画 GIF 平台 Giphy 的收购——证实了金融时报早些时候的报道。

竞争和市场管理局(CMA)表示，其第二阶段调查巩固了其早先对 Meta 拥有和运营 Giphy 的影响的竞争担忧，现在它命令 Meta 出售 Giphy。

在一份声明中，负责 CMA 调查的独立调查小组主席斯图尔特·麦金托什(Stuart McIntosh)表示:“脸书和 Giphy 之间的合作已经消除了展示广告市场的一个潜在挑战者。如果不采取行动，它还将允许脸书通过控制竞争对手对 Giphy GIFs 的访问，进一步增加其在社交媒体上的重要市场力量。"

“通过要求脸书出售 Giphy，我们正在保护数百万社交媒体用户，并促进数字广告的竞争和创新，”他补充说。

监管机构的干预是在对脸书于 2020 年 5 月宣布(并完成)的收购进行了扩展调查之后，CMA 将于 2020 年夏季进行初步调查——并在接下来的几个月加大审查力度。

2020 年 6 月，它下令停止脸书进一步整合 Giphy，同时继续监督。

上个月的另一个先例是，监管机构对脸书处以近 7000 万美元的罚款，原因是其故意隐瞒与收购的持续监管相关的信息——称这一侵权行为是“重大”违约。

8 月 13 日，CMA 关于脸书-吉菲收购案的初步报告得出结论，脸书对吉菲的收购引发了许多竞争问题——包括它会损害社交媒体平台之间的竞争，因为动画 gif 的供应缺乏选择。

监管机构不仅担心脸书可能会拒绝竞争对手访问 Giphy 内容，让他们的用户重新分享，还担心这个数据挖掘巨头可能会改变访问条款，例如，可能会要求抖音、Twitter 和 Snapchat 等竞争对手提供更多用户数据，以便访问 Giphy GIFs。

CMA 似乎一直担心通过从其他服务以及其他更明显的风险(如脸书切断竞争对手对该平台的访问)中提取数据来损害竞争的风险，因此拒绝了这家科技巨头提出的出售 Giphy 的所有替代“补救措施”，认为这些措施不够充分。

“在咨询了感兴趣的企业和组织，并评估了脸书提出的替代解决方案(被称为‘补救措施’)后，CMA 得出结论，其竞争问题只能通过脸书将 Giphy 整体出售给经批准的买家来解决，”CMA 在一份新闻稿中写道。

今年夏天，该监管机构还表示，它担心对数字“展示”广告的影响——因为 Giphy 在合并前一直在美国提供付费广告服务(并考虑扩展到包括英国在内的其他国家)，它说这有可能与脸书的广告服务竞争。然而，这场竞争因脸书的收购而终止。

“CMA 发现 Giphy 的广告服务有可能与脸书自己的展示广告服务竞争，”该监管机构写道。“他们还会鼓励市场中的其他人进行更大的创新，包括社交媒体网站和广告商。脸书在合并时终止了 Giphy 的广告服务，消除了潜在竞争的一个重要来源。鉴于脸书控制着英国 70 亿英镑展示广告市场的近一半，CMA 认为这一点尤其令人担忧。”

CMA 最终报告的摘要可以在这里找到。

监管机构对合并的评估取决于——根据“可能性平衡”标准——如果收购继续进行，是否会“大大减少竞争”(SLC)。在脸书-吉菲的案例中，这就是它的结论——在(其他)社交媒体平台上，几乎找不到吉菲的替代者,脸书 在英国的展示广告中拥有巨大的市场力量，以及其他相关因素。

CMA 对 Giphy 开发其付费广告模式的潜力特别感兴趣，包括可能在英国推出的广告，指出广告商对基于 GIF 的广告格式的反馈是积极的，并且“由于其 GIF 格式，广告的付费调整模式是微妙的，是信息的内在组成部分，而不是打断它”——其报告指出，这一点“反映在 Giphy 的内部文件中”，“脸书的内部文件也讨论了信息货币化的重要性。”

它还得出结论，脸书将有动机取消竞争对手对 Giphy 的访问权——这导致了另一个结论，即合并“将导致社交媒体的 SLC，这是垂直效应的结果，表现为输入取消权。”

竞争委员会评估了一些其他潜在的“补救办法”来解决竞争问题，而不是要求出售 Giphy 考虑并拒绝了脸书提出的三个选择。

脸书的建议是:a)开放获取——保持现有的和新的 API 伙伴对 Giphy 的获取；b)“混合”补救措施，取消 Giphy 的 ToS 中禁止将其搜索结果与另一家 GIF 提供商的结果混合的限制，脸书认为这将“使潜在的付费对齐提供商能够通过允许其在自己的广告中散布 Giphy 的 GIF 来增加其产品的吸引力”；以及 c)白标许可补救措施，包括创建和销售 Giphy 内容库的白标副本以及五年内使用其搜索算法的许可证。

CMA 将脸书的“补救措施”描述为“行为性的”而不是“结构性的”——并且在拒绝它们时认为它们不合适，值得注意的是，它强调了持续合规监督的挑战，以及提案本身不足以解决所有问题。“结构性补救通常比行为性补救更可取，因为它们从源头上解决了合并的负面影响，”报告写道。"虽然行为补救在某些情况下可能是合适的，但这次合并不具备这些特征."

(这是值得注意的，因为 CMA 已经在一个单独的监督程序中——与谷歌反对支持跟踪 cookies 的提议有关——表示，它愿意接受谷歌做出的许多行为承诺，例如限制它可以用于广告的数据，围绕它向竞争对手提供的透明度水平，以及任命一名监督受托人来审计合规性。当然，这不是合并审查。这是对竞争的投诉。)

“特别是，我们发现的 SLC 本质上是动态的，没有时间限制，减少了行为补救措施提供有效和全面解决方案的可能性，”CMA 在关于脸书-吉菲的最终报告摘要中写道。“我们还发现了脸书提出的补救方案的一些具体风险，包括它们无法全面解决 SLC，在具体说明脸书的义务方面的挑战，脸书能够规避这些义务的风险，以及在监测和强制执行脸书遵守这些义务方面的困难。因此，我们认为，脸书的补救建议将不会有效地解决我们所发现的特殊情况。”

虽然资本市场委员会的结论是，只有完全剥离 Giphy 才会有效，但这里存在一些复杂因素，因为合并已经完成，脸书已采取措施整合 Giphy，包括终止其收入职能和团队，将后台职能转移到脸书，以及 Giphy 员工转到脸书就业合同。

为了应对这些“特殊的挑战”，资本市场委员会表示，脸书必须恢复一些解散的业务职能和资产，“以确保 Giphy 拥有必要的管理、技术和创新人才，使其能够在整个剥离过程中以及剥离后有效竞争。”

“我们预计，脸书将需要提供适当的财政和其他激励措施，以鼓励前 Giphy 员工调回 Giphy，并为选择不这样做的 Giphy 关键员工招聘适当的替代人员。我们还预计，Giphy 将需要足够的财务资源进行剥离，以使其能够像未被脸书收购时一样运营和竞争，”它补充道。

今年夏天，梅塔/脸书对英国医学协会的临时调查结果做出了激烈的回应— 谴责该分析，并质疑英国监管机构对其业务的管辖权。

在一份简短的声明中，一位 Meta 发言人回应了 CMA 的最后一句话:

“我们不同意这个决定。我们正在审查这一决定，并考虑所有选择，包括上诉。在我们的基础设施、人才和资源的支持下，消费者和 Giphy 都变得更好。Meta 和 Giphy 将一起为英国和世界各地每天使用 Giphy 的数百万人、企业、开发者和 API 合作伙伴增强 Giphy 的产品，为每个人提供更多选择。"

扼杀收购杀手？

多年来，对所谓“杀手级收购”的担忧一直是行业观察人士的一个主要话题。所谓“杀手级收购”是指科技巨头通过收购初露头角的竞争对手来化解创业公司和新服务带来的风险(有时实际上是在收购后关闭它们)，以此来展示自己的财务实力，从而保护市场力量。

批评的焦点是竞争监管机构未能发展出与数字市场动态同步的伤害理论。例如，未能考虑如何将数据本身用作对抗竞争的工具。主导平台也可以通过自我偏好等策略，轻松利用其在一个渠道中的市场力量，迅速扩展到一个新的细分市场。虽然服务在使用时是“免费的”,但仍可能给消费者带来重大伤害——比如侵犯他们的隐私。

近年来，立法者和监管者已经开始回应这种担忧——例如，德国在今年年初通过了一项制度更新，以涵盖数字平台。(该国现在有许多针对科技巨头的公开程序，包括脸书，以确认其实施先发制人措施的能力。)

在美国，拜登政府今年早些时候提升 Lina Khan 担任联邦贸易委员会主席，标志着美国变革的关键时刻——标志着立法者对监管技术的改革方法的支持。

在此之前，可汗发表了具有里程碑意义的论文(关于亚马逊)，该论文研究了政府识别垄断的过时方法如何无法跟上现代商业现实。最初被一些人斥为“潮人反垄断”的东西，现在正在设定机构监管议程。尽管可汗在本土仍然面临着通过美国商会等渠道开展工作的技术游说团体的巨大反对。

在欧盟，欧盟委员会也一直在努力解决技术和反垄断之间的差距。

自去年 12 月以来，它已经有了一份适用于中介平台巨头(根据数字市场法案被归类为“看门人”)的一套事前规则的提案草案。本月早些时候，欧洲议会议员投票支持委员会有权在看门人有系统性违规行为时实施结构性或行为性补救措施，从而支持对杀手级收购进行干预。

DMA 是否能真正帮助重新启动竞争还有待观察。

现在已经退出欧盟的英国也有自己的国内竞争法更新，也旨在解决平台权力——新制度的“支持竞争”为被视为具有“战略市场地位”的平台定制规则

然而，所有这些都为时已晚，无法撤销大量固有的技术整合。但是现在撤销脸书-吉菲法案还为时不晚——因为监管者显然已经加强了他们对数字市场(和危害)的理解很多年了。

然而，过去几十年来，监管数字市场的过时方法使得数以千计的技术收购被搁置——包括脸书收购照片共享网站 Instagram、消息平台 WhatsApp 和虚拟现实耳机制造商 Oculus，这是三起跨越核心社交网络领域的战略收购，脸书/Meta 拥有并希望在未来几十年继续拥有这些领域(以更具沉浸感/侵入性的形式，因此有了“元宇宙”)。

仅在去年年底的，欧盟委员会未能阻止谷歌收购健康可穿戴设备 Fitbit，尽管民间社会发出了强烈抗议，警告称，让这家广告科技巨头通过吞噬如此敏感的数据来进一步巩固其主导地位会带来权利风险。欧盟竞争委员会没有阻止收购，而是接受了谷歌提出的一系列行为“补救措施”——其中包括承诺不会将第三方开发者排除在 Fitbit 的 API 之外，并承诺在整整 10 年内不会使用 Fitbit 的健康数据进行广告定位。

最近，CMA 还批准了脸书对 CRM 制造商 Kustomer 的收购——再次使用了一种相当狭隘的潜在竞争风险评估——并完全忽视了隐私倡导者对这家广告技术巨头将如何处理 Kustomer 用户数据的担忧。

CMA 现在决定命令脸书撤销对 Giphy 的收购，这是一个重大进展，尽管这仍然只是一个没有对 Big Tech 有利的决定。

在回答 TechCrunch 的问题时，托马索·瓦列蒂教授讨论了这一举措，他是委员会内的前首席竞争经济学家，曾在现任执行副总裁玛格丽特·维斯塔奇手下工作，他将 CMA 的举措描述为“高度象征性的决定”。但他警告说，不要对一个“不”字做过多解读。

“我一直在重复 1，000 和 0 的数字:GAFAM 完成的并购和过去 20 年里被阻止的并购。所以最终有了一个并不会改变整个画面，但它是一个信号，”他告诉我们。

今年早些时候，欧盟委员会允许成员国在合并案可能处于国家反垄断政策的夹缝中时，提交合并审查，因为创新技术或业务有被更成熟的竞争对手(以低价)收购的风险，从而扼杀萌芽中的竞争。

Valletti 还指出，Vestager 最终发出了与美国立法者讨论大型技术收购的意向-他称之为“另一个好迹象”，称欧盟“在这方面一直(并且仍然)落后。”

对反垄断在美国的应用方式进行重大改造，显然对于遏制(主要是)美国科技巨头至关重要——无论其他地方的个体(甚至是泛欧盟)监管机构的行动多么创新。

“至于‘新’的伤害理论，我认为这只是因为 CMA 有优秀的经济学家，他们知道经济学在过去 10 年里一直在说什么和发现什么:数据是商业模式的一部分，所以它们也必须是竞争力评估的一部分，”Valletti 在进一步评论 CMA 撤销脸书-吉菲的决定时补充道。“这不‘仅仅’是别人处理的隐私问题。

“良好的经济状况、开放的思想以及领导层更高的风险偏好意味着，资本市场管理局正试图在一个典型的极度保守、监管者害羞的领域里提高标准。让我们满怀希望吧！”

如上所述，英国正在进行竞争法改革，专门针对平台巨头——具有所谓的“战略市场地位”——未来将根据定制规则的事前要求进行监管。尽管授权专门负责该领域的数字市场部门的必要立法仍悬而未决。

不过，与此同时，CMA 并没有坐以待毙，它对 Big Tech 业务的各个方面进行了一系列公开调查，并对收购进行了持续审查。

鉴于英国不再是欧盟成员国，英国的监管制度可以在科技政策和大型科技并购方面自行其是，尽管英国监管机构表示，他们将继续就共同关心的问题与国际同行进行磋商。

尽管欧盟正在寻求在即将到来的立法更新和扩展(如 DMA 和数字服务法案)下协调数字法规，但有人担心，欧盟立法者减少“碎片化”的努力最终可能有利于科技巨头——即，如果它消除了单个成员国通过更雄心勃勃的立法的能力。

因此，英国监管机构最终可能会解决欧盟一刀切计划的不足之处——为平台巨头列出行为“该做和不该做”的清单——方法是应用一种更严格定制的制度，旨在解决每个科技巨头具体的有问题的做法。因此，在 CMA 中拥有有创造力的思考者是至关重要的。
