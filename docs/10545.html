OG 应用承诺提供无广告的 Instagram feed，但随后被 App Store TechCrunch 撤下

原文：https://web.archive.org/web/https://techcrunch.com/2022/09/27/og-app-promises-you-an-ad-free-instagram-feed/?guccounter=1

Instagram 目前一片混乱，用户——包括金·卡戴珊——对此不满。该公司试图在应用程序中塞进太多的功能，如卷轴、混音、算法馈送和 NFTs。开发人员 Ansh Nanda 和 Hardik Patil 感到非常沮丧，他们决定开发一款名为 OG App 的应用程序，为用户提供无广告、无建议的 Instagram feed。 [ 更新:App 已经从 App Store 下架。苹果表示，它以未经授权的方式访问了 Instagram 的服务。另外，Meta 称该应用违反了规定。】

 

在 iOS 和 Android 上可用的 OG 应用程序并没有试图创建一个新的社交网络。相反，它只是根据用户的选择重新调整 Instagram feed。用户可以登录他们的 Instagram 帐户，它会立即显示主页。但该应用程序会从他们的主页源中删除广告和建议帖子。

此外，它还允许用户创建替代源，并与朋友分享。把它们想象成 Twitter 列表，但是对于 Instagram 来说。例如，你可以为食物、鸡尾酒、网球、电影明星和摄影师创建不同的 feeds，而无需关注所有这些账户。用户可以在启动时选择默认提要。通过这种方式，用户可以让另一个列表优先于他们自己的主页提要。当用户的兴趣相关提要比默认提要有更多吸引人的内容时，这很有用。

图片来源:OG 应用

该应用程序还有一个选项，可以在 24 小时内停止获取新内容。这个选项就像一个数字福利功能一样，可以阻止用户不断查看 Instagram。他们仍然可以查看所有提要中已经获取的帖子以及保存的帖子。

图片来源: OG 应用

去年 6 月，南达和帕蒂尔合并了发布 OG 应用程序的 Un1feed 公司，并开始开发社交应用程序，以提供更清洁的体验。他们首先选择了 Instagram，因为他们认为它的广告最为复杂。这两人是在 KiranaKart 工作时认识的，KiranaKart 是一款杂货递送应用程序，现在被称为 Zepto，今年 5 月以 9 亿美元的估值筹集了 2 亿美元。

Nanda 和 Patil 表示，他们希望通过这家公司将更多的 feeds 控制权交还给用户。

“我们看到我们的朋友和家人受到社交媒体的影响，甚至删除应用程序，因为他们没有足够的选项来修改他们看到的内容。我们想把用户，而不是广告商，放在这个应用程序的首位。我们从 Instagram 开始，因为我们认为该应用程序与其用户的关系是最有害的，”创始人在与 TechCrunch 的电话会议上说。

在 OG 应用程序上创建自定义订阅源图片来源: OG 应用程序

Nanda 表示，该公司与数百名用户进行了交谈，发现 Instagram 用户对默认的社交网络体验不满意。该公司因优先考虑算法建议的内容而不是朋友和家人的帖子而受到了很多批评。今年 7 月，这家社交网络不得不撤回其通过算法反馈所做的更改。

Un1feed 已获得由 Neo 的 Ali Partovi 牵头的 100 万美元种子前资金，参与方包括 Tribe Capital、Great Oaks VC、Global Asset Capital、Pareto20 和 Cory Levy。联合创始人告诉 TechCrunch，该公司目前不打算向用户收费或寻找其他盈利机会。这家初创公司有八个人在开发这款应用，包括南达和帕蒂尔。

在未来几周，开发者希望推出一些功能，如私人账户定制订阅源，与朋友分享定制订阅源，以及下载故事供离线使用。此外，他们将很快允许用户使用“最近”、“热门”和“周/月/所有”等过滤器对任何提要进行排序应用程序制造商也在计划长期功能，如分享收藏作为供稿和制作协作供稿。

虽然社交媒体公司试图为用户提供对其订阅源的更多控制，但他们往往不得不求助于第三方应用程序来获得最佳的社交网络体验。例如，Reddit 的客户有 Apollo 、 Joey 和 Boost ，Twitter 的客户有 Tweetbot 和 Talon 。OG 应用希望为 Instagram 用户提供类似的可修改体验。

所有这些客户端都必须依赖核心社交网络提供的 API。众所周知，Twitter 在过去十年里通过限制其 API 削弱了第三方客户。但 OG 应用的创始人表示，他们目前并不担心 Instagram API 的任何变化，而是专注于为用户打造一款可靠的产品。

早期用户对隐私的担忧

该应用发布后，许多早期用户质疑该应用的隐私做法，包括 App Store 上列出的数据收集和一种登录方法，即将一些用户锁定在他们的账户之外，或者向显示从不同位置登录的提示。

应用程序开发者表示，他们必须使用与 Instagram 类似的权限来使用 API 并从社交网络中获取数据。关于登录方法，该公司表示，它对 Instagram 的 Android API 进行了逆向工程，以使该应用程序工作，这就是为什么用户会看到来自另一个位置的登录提示。然而，这家初创公司明确表示，它不会在其服务器上存储任何数据。

Un1Feed 还指出，在未来几天，它将为用户推出一种新的登录方法，这将解决许多问题，并提供更多关于如何使用 Instagram API 的信息。

应用研究人员 Alessandro Paluzzi 指出，OG 应用可能正在使用 Instagram 的私有 API ，这可能会面临 Instagram 的一些审查。我们已经联系了 Meta 进行评论，如果有回音，我们会更新这个故事。

发布几天后，苹果就从应用商店撤下了这款应用。在 Twitter 的一条帖子中，OG 应用将删除归咎于脸书(Meta)。但这表明该应用在某种程度上违反了 Meta 指南或政策，因为该线程解释了该应用如何成功通过苹果自己的应用审查。
