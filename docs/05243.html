电子邮件营销巨头 Mailchimp 证实了一次数据泄露

原文：https://web.archive.org/web/https://techcrunch.com/2022/04/04/mailchimp-internal-tool-breach/amp/

图片来源: 拉斐尔·恩里克/ SOPA 图片社/ LightRocket /盖蒂图片社

电子邮件营销巨头 Mailchimp 证实了一起数据泄露事件，此前恶意黑客侵入了一个公司内部工具以获取客户账户。

在给 TechCrunch 的一份声明中，Mailchimp CISO·西沃恩·史密斯表示，该公司在 3 月 26 日发现一名恶意行为者访问该公司客户支持和账户管理团队使用的工具后，意识到了这次入侵。在成功的社会工程攻击之后获得了访问权，这是一种利用人为错误并使用操纵技术来获取私人信息、访问权或贵重物品的攻击。

“我们迅速采取行动，通过终止对受损员工账户的访问来解决这一问题，并采取措施防止其他员工受到影响，”Smyth 说。

该公司表示，但速度不够快，黑客查看了大约 300 个 Mailchimp 账户，并成功导出了其中 102 个账户的受众数据。Mailchimp 拒绝透露具体访问了哪些数据，但告诉 TechCrunch，黑客的目标是加密货币和金融领域的客户。除了查看帐户和导出数据之外，威胁参与者还获得了数量不详的客户的 API 密钥的访问权限，这使得攻击者有可能发送欺骗性电子邮件，但这些电子邮件现在已被禁用，无法再使用。但是 Smyth 说 Mailchimp 收到了一些黑客利用他们从用户账户获得的信息向他们的联系人发送网络钓鱼广告的报告。

“当我们意识到任何未经授权的账户访问时，我们会通知账户所有者，并立即采取措施暂停任何进一步的访问，”Smyth 告诉 TechCrunch。“我们还为我们的用户推荐双因素身份验证和其他帐户安全措施，作为保护帐户和密码安全的附加措施。”

Smyth 拒绝回答我们的问题，即 Mailchimp 正在采取哪些额外的安全措施来防止未来的攻击。

这一事件最初是由哔哔声计算机报道的，在加密货币钱包制造商 Trezor 在 Twitter 上证实其用户因 Mailchimp 的漏洞而成为钓鱼电子邮件的目标后，该事件在周末被曝光，Trezor 使用 Mailchimp 向客户发送简讯。这些恶意邮件促使 Trezor 用户通过下载恶意软件来重置他们的硬件钱包 pin，如果安装了恶意软件，黑客就可以窃取客户的密码。

Mailchimp 不愿透露有多少其他加密货币服务或金融机构受到了此次事件的影响。
