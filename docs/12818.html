Google | TechCrunch 称，间谍软件供应商 Variston 利用了 Chrome、Firefox 和 Windows zero-days

原文：https://web.archive.org/web/https://techcrunch.com/2022/11/30/variston-spyware-chrome-firefox-windows/

谷歌称，间谍软件供应商 Variston 利用了 Chrome、Firefox 和 Windows zero-days

谷歌安全研究人员表示，一家总部位于巴塞罗那、自称是定制安全解决方案提供商的公司利用 Windows、Chrome 和 Firefox 浏览器中的几个零日漏洞植入间谍软件。

在周三发表之前与 TechCrunch 分享的研究中，谷歌的威胁分析小组(TAG)表示，它已经将声称提供量身定制的网络安全解决方案的 Variston IT 与一个利用框架联系起来，该框架使间谍软件能够安装在目标设备上。

“我们的团队由一些业内最有经验的专家组成，”Variston IT 的网站上写道。“我们是一家年轻但发展迅速的公司。”

谷歌研究人员在收到对其 Chrome 漏洞报告计划的匿名提交后，开始意识到所谓的“Heliconia”利用框架。在分析了框架之后，谷歌的研究人员在源代码中发现了暗示 Variston 可能是开发者的线索。

Heliconia 包含三个独立的利用框架:一个包含对 Chrome 渲染器漏洞的利用，该漏洞允许它逃离应用程序的沙箱，在操作系统上运行恶意软件；另一种是部署恶意 PDF 文档，其中包含对 Windows Defender(现代 Windows 版本中的默认防病毒引擎)的利用；另一个框架包含一组针对 Windows 和 Linux 机器的 Firefox 漏洞。

谷歌指出，Heliconia 漏洞利用对火狐 64 至 68 版本有效，这表明该漏洞利用早在 2018 年 12 月火狐 64 首次发布时就已使用。

谷歌表示，虽然它没有看到这些漏洞在野外被积极利用，但这些漏洞很可能被用作零日漏洞——这样命名是因为公司没有时间或零日来推出修复程序——后来被用作 n 日漏洞——当漏洞被利用时，但补丁已经发布。谷歌、微软和 Mozilla 在 2021 年初和 2022 年修复了这些漏洞。

当通过电子邮件联系到 Variston 的 IT 主管拉尔夫·韦格纳(Ralf Wegner)时，他告诉 TechCrunch，该公司不知道谷歌的研究，也无法验证其发现，但“如果在野外发现这样的(原文如此)物品，会感到惊讶。”

谷歌在其博客文章中称商业间谍软件，如 Heliconia framework，包含了曾经只有政府才拥有的功能。这些功能包括偷偷录制音频，拨打或重定向电话，从目标设备上窃取数据，如短信、通话记录、联系人和精确的 GPS 位置数据。

谷歌表示:“间谍软件行业的增长将用户置于风险之中，并使互联网变得更不安全，尽管根据国家或国际法律，监控技术可能是合法的，但它们经常被以有害的方式用于对一系列群体进行数字间谍活动。”。“这些滥用行为对在线安全构成了严重的风险，这就是为什么谷歌和 TAG 将继续采取行动打击商业间谍软件行业，并发布相关研究。”

几个月前，将一个以前未被提及的 Android 手机间谍软件，绰号“隐士”,与意大利软件公司 RCS Lab 联系起来。
