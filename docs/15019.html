研究人员称，一种新的 Linux 版本的 Clop 勒索软件存在重大缺陷

原文：https://web.archive.org/web/https://techcrunch.com/2023/02/07/clop-ransomware-linux-flaw/

研究人员称，一种新的 Linux 版本的 Clop 勒索软件存在重大缺陷

安全研究人员首次观察到针对 Linux 系统的多产 Clop 勒索软件操作。好消息是，新变种使用的有缺陷的加密技术意味着受害者有可能免费恢复他们被盗的文件。

SentinelLabs 的研究人员 Antonis Terefos 发现并详细描述了 Clop 勒索软件的新 Linux 变种。在的一篇博客文章中，Terefos 说他在 12 月 26 日首次观察到针对 Linux 系统的文件加密恶意软件，此前勒索软件团伙使用这种新的恶意软件针对哥伦比亚的一所大学，这种新的恶意软件在 1 月份被添加到 Clop 的黑暗网络泄漏网站。Clop 的泄漏网站仍然活跃，目前将哥伦比亚的拉萨尔大学列为其最近的受害者，与英国水供应商南斯塔福德郡水务公司并列。

Terefos 指出，新的 Linux 变种与 Windows 版本相似，使用相同的加密方法和相似的处理逻辑，但包含几个缺陷，包括一个有缺陷的勒索软件加密逻辑，它可以在不支付赎金的情况下解读原始文件。因此，SentinelLabs 为 Clop 受害者建立了一个免费文件解密工具，该公司告诉 TechCrunch，它也与执法部门共享了该工具。

Terefos 表示，一些缺陷的存在是因为 Clop 黑客决定建立一个定制的 Linux 版本的勒索软件，而不是移植到 Windows 版本，但警告说，更多针对 Linux 的勒索软件将会出现。

Terefos 说:“虽然 Linux 风格的 Clop 变种目前仍处于起步阶段，但它的发展以及 Linux 在服务器和云工作负载中几乎无处不在的使用表明，防御者应该会看到更多针对 Linux 的勒索病毒活动。

TechCrunch 已经向 SentinelLabs 询问了它观察到多少 Clop 感染，如果我们得到回复，将会更新。

说俄语的 Clop ransomware 团伙自 2019 年以来一直很活跃，但在 2021 年似乎遭遇了重大挫折，在代号为“旋风行动”的国际执法行动后，6 名与该团伙有关联的人被逮捕。乌克兰警方当时还表示，已成功关闭该团伙使用的服务器基础设施。但是 Clop 继续声称有新的受害者，包括一家农业设备零售商和一家建筑师事务所，就在警察突袭的几周后。

该团伙在 2022 年卷土重来，仅在一个月内，Clop 就在其黑暗的网络泄密网站上增加了 21 名受害者。

NCC 集团战略威胁情报全球负责人马特·赫尔(Matt Hull)表示:“Clop 活动的增加似乎表明，它们已经回到了威胁格局中。”。“Clop 最有针对性的行业中的组织——尤其是工业和技术——应该考虑这个软件集团带来的威胁，并做好准备。”
