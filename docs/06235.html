拥抱脸获得 1 亿美元来建立机器学习的 GitHub

原文：https://web.archive.org/web/https://techcrunch.com/2022/05/09/hugging-face-reaches-2-billion-valuation-to-build-the-github-of-machine-learning/amp/

图片来源:拥抱脸

抱紧脸已经关闭新一轮融资。是 1 亿美金的 C 轮，估值很大。在今天的融资后，拥抱脸现在价值 20 亿美元。

Lux Capital 领投，红杉和 Coatue 首次投资该公司。这家初创公司的一些现有投资者再次参与进来。这些投资者包括 Addition、Betaworks、AIX Ventures、Cygni Capital、凯文·杜兰特和 Olivier Pomel。

尽管生命短暂，拥抱脸已经有了有趣的演变。当我在 2017 年第一次报道这家公司时，这家初创公司专注于一款消费者应用。它看起来像又一个消息应用程序。然而，没有人在交谈的另一边，因为这是一个无聊的青少年聊天机器人应用程序。

消费者的赌注没有得到回报，但该公司一直在迭代其自然语言处理技术。拥抱脸在 GitHub 上发布了变形金刚库，并立即吸引了大量的关注——它目前在平台上有 62，000 颗星星和 14，000 个叉子。

使用 Transformers，您可以利用流行的 NLP 模型，如 BERT、GPT-2、T5 或 DistilBERT，并使用这些模型以某种方式操纵文本。例如，您可以对文本进行分类、提取信息、自动回答问题、汇总文本、生成文本等。

由于这个库的成功，拥抱脸很快成为所有与机器学习模型相关的东西的主要仓库——不仅仅是自然语言处理。在该公司的网站上，你可以浏览数千个预先训练好的机器学习模型，用你自己的模型参与开发者社区，下载数据集等等。

本质上，拥抱脸是在构建机器学习的 GitHub。这是一个社区驱动的平台，有大量的存储库。开发人员可以创建、发现和协作 ML 模型、数据集和 ML 应用程序。

拥抱脸还提供托管服务，如推理 API ，让你通过编程接口使用数千个模型，以及“自动跟踪”你的模型的能力。

随着今天的融资，该公司计划做更多同样的事情；10，000 家公司正在以这样或那样的方式使用拥抱脸，所以现在不是再次转向的时候。
