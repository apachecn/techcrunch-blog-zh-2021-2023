微软为 Windows 11 带来了新的必应，并为 iOS 推出了 Phone Link

微软正在抓紧时间将其新的人工智能驱动的必应带到其所有产品中。在将其引入 Edge 浏览器，然后将其引入手机上的 Edge、Bing 和 Skype 应用之后，今天轮到 Windows 11 的在其任务栏上欢迎新的 Bing 。

这是 Windows 11 重大升级的一部分，还包括 iOS 的第一个 Phone Link 预览版，可以让你在 Windows PC 上接听电话和回复来自 iPhone 的消息，在默认的记事本应用程序中添加标签，并有新的能源建议，重新设计的快速协助应用程序，可以更容易地远程协助父母解决他们的计算机问题，等等。

如果你是 Windows 11 用户，你肯定熟悉默认情况下集成到任务栏中的搜索框。这就是你现在会发现新的必应，等待你的问题，不愿意谈论自己。

正如微软指出的，每月有超过 5 亿的用户，搜索框是 Windows 中使用最广泛的功能之一。这是新必应的一个非常突出的位置。事实上，市场份额较低的必应搜索引擎的月用户数可能比这还要少。要访问新的 Bing，你仍然需要在等待名单上(如果你还没有这样做)，所以不是每个 Windows 11 用户都可以立即访问。

图片来源:微软

众所周知，自本月初 Bing 首次推出以来，微软不得不忍受相当多的批评。从很多方面来说，这看起来像是一个仓促的发布，但值得称赞的是，Bing 早期用户发现的许多问题已经得到了解决。使用 Bing 聊天机器人获得某种对抗性反应是一回事，而使用它来帮助日常搜索则是完全不同的体验。但我们显然需要探索边缘案例，微软在推出它之前显然应该自己做更多的工作。在这一点上，微软已经控制了机器人太多。它不想谈论自己很好——毕竟它不是政客——但每次对话只有六次机会，我不断地在对话中探索一个有趣的话题，它就让我闭嘴(或者它会问我是否想要某样东西的配方——它现在真的很喜欢问问题——当我告诉它我想的时候，它就让我见鬼了……)。

图片来源:微软

至于 iPhone 的 Phone Link 预览版，值得注意的是，这仍然是一个非常有限的预览版，仅适用于选择加入 Windows 11 Insider 发布渠道的用户——即使如此，它目前也只会向有限数量的测试人员推出。我使用 Android 版本已经有一段时间了，它非常稳定，信息、电话和提醒都在我的桌面上弹出。这是该应用程序承诺要做的，也是它所提供的。当然，我们将不得不看看 iOS 版本是否同样有效。
