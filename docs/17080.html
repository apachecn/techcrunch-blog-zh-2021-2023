创业公司股票的合理溢价是多少？

对于一家快速成长的初创公司来说，什么是合理的溢价？答案是不断变化的，但最近的历史告诉我们，它可能不会超过公开市场 comps 的 200%。甚至可能更接近 100%。

这不是一个学术问题。今年初创公司价格溢价基准的确定，可能有助于确定一系列 B 轮和 C 轮融资的估值是糟糕、合意还是令人兴奋。

该交易所探索初创企业、市场和资金。

根据红点风险投资公司的一份报告，今天早上我们正在研究 2021 年中期创业公司的估值如何达到令人难以置信的水平，以及这些价格点对可比公司的预期收入倍数急剧收缩的反应。这将涉及一些数字、一些图表和一点点数学，但我希望当我们完成时，我们将更好地理解初创公司的估值如何在上一次科技繁荣期间离开大气层，以及它们在 2023 年回归地球的程度。

希望在今年晚些时候筹集资金的初创公司创始人，这是为你准备的。

走向疯狂

风险资本市场对股票市场反应灵敏。随着科技股的价值上升，创业公司的价值也会上升，反之亦然。这是合理的，因为大多数创业公司都在朝着最终 IPO 的方向发展，所以他们今天的价值应该反映他们未来退出时的潜在价值，这部分是由最近的股市表现决定的。

Redpoint 的新数据清楚地表明了这一点，并详细说明了初创企业融资市场的不同领域对价格的依赖程度可能超出你的预期。和所有事情一样，让我们从头开始。

关于这些数据的几点说明。首先，风险投资公司董事总经理洛根·巴特利特(Logan Bartlett )表示，B 系列和 C 系列中的公司是红点“希望关注”的公司组合。这包括美国、加拿大和欧洲的公司。这是一个受地理限制的数据集，而不是一个全球性的数据集，但鉴于北美和欧洲松散地代表了风险市场的大部分，这是一个不错的样本。
