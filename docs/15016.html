Twitter 将停止在 iOS 和 Android 上强制其算法时间表

原文：https://web.archive.org/web/https://techcrunch.com/2023/02/06/twitter-will-stop-forcing-its-algorithmic-timeline-on-ios-and-android/

Twitter 将停止在 iOS 和 Android 上强制推行其算法时间表

Twitter 决策圈已经完成。这家社交网络将最终停止向 iOS 和 Android 用户强制推行其“为你”的算法时间表。Twitter 表示，当你关闭应用程序时，它们会记住你选择的时间线。

本质上，如果你在退出应用程序时正在查看按时间顺序排列的“跟随”时间线，当你再次打开 Twitter 时，你不会被迫回到算法时间线。该公司上个月在网上推出了这一更新，现在正在将其推广到 Twitter 移动应用的最新版本。

埃隆·马斯克领导的公司上个月开始推动算法上的“为你”时间表，将其作为 iOS 的默认馈送。后来，它还将双时间轴视图带到了 Android 和 web 上。

许多用户对无法将时间顺序作为默认时间表感到不满，因为 Twitter 会在刷新或应用程序重新打开时将他们推回到“为你”的时间表。最新的更新解决了这个问题，但是双时间线视图仍然存在。

在此之前，Twitter 有一个星号按钮，可以让你轻松地在时间顺序或算法时间线之间切换。

目前，你必须使用 Twitter 的官方应用程序来访问社交网络，因为该公司通过修改其 API 使用条款杀死了第三方应用程序。
