Grafana 开源了其随叫随到管理工具

去年 11 月，Grafana Labs 推出了 OnCall ，这是一款针对开发运营和现场可靠性工程(SRE)团队的随叫随到管理工具，作为其完全管理的 Grafana Cloud 服务的一部分。今天，该公司宣布它正在开源这一工具，这将使受监管行业中无法使用 Grafana 云的公司也能更容易地使用它。值得注意的是，OnCall 也是免费的 Grafana Cloud 产品的一部分。

“一旦我们觉得 OnCall 已经准备好让用户自己轻松可靠地运行它，我们就想让它开源，”项目联合创始人兼 Grafana Labs 高级工程经理 Matvey Kukuy 说。“我们觉得现在已经准备好了。”

图片来源: Grafana

为了建立 OnCall，Grafana Labs 去年收购了 Kukuy 的初创公司 Amixr ，这是他在 2018 年与 Ildar Iskhakov 一起创立的。这里的想法是给团队一个单一的工具，可以管理来自 Grafana，Prometheus 和 Alertmanager 的警报。该服务允许团队创建和管理随叫随到的时间表，在出现问题时建立自动升级，当然，它也与 New Relic 和 Datadog 等服务相集成。

除了 open sourcing OnCall 之外，该公司还于本周举办了 GrafanaCONline 社区活动，并于今天发布了 Grafana 9.0。它包括 Prometheus 和 Grafana Loki 的可视化查询生成器等功能，使更多开发人员可以更容易地创建查询，而不必学习底层查询语言，以及仪表板预览和现代化的热图面板。
