微软宣布新的工具来实现供应链和制造业的现代化

原文：https://web.archive.org/web/https://techcrunch.com/2021/11/02/microsoft-announces-new-tools-to-modernize-supply-chain-and-manufacturing/

由于多种原因，疫情对世界供应链造成了严重破坏——从原材料缺乏到劳动力短缺——但无论是什么原因，问题都持续存在，微软已经决定将其大量资源用于供应链和制造的现代化。

为了帮助解决这些问题，该公司宣布了一种新的制造解决方案，称为 Microsoft Cloud for Manufacturing，以及 Dynamics 365 Supply Chain Insights，这是一种旨在让客户更好地了解供应链路线上发生的事情，并在出现问题时智能处理问题的工具。这两款产品都将在 Microsoft Ignite 上发布，并从今天开始提供预览版。

微软制造和供应链副总裁 Caglayan Arkan 表示，该公司一直在思考如何帮助公司解决这些问题，同时发展成为更加专注于数字的公司——这对制造商来说是一个挑战。

Arkan 说，微软检查了事情的状态，看到了脆弱的系统，当面对疫情的压力时，不能很好地坚持。“制造业和供应链一直非常稳定；非常非常瘦，也许太瘦了。他们在非常长的周期和手动孤岛式数据状态中变得非常自在，”他说。

他补充说，制造商多年来一直以这种方式工作，他们没有真正的理由改变，直到疫情出现。“一切都停止了。音乐停止了，你不能再派员工去车间了，那里没有供应品。你甚至不能看到你的供应商知道你的货物在哪里，这是一个巨大的破坏，”Arkan 解释说。

制造云旨在帮助数字化工厂车间，并通过从传统的记录系统转移到 Arkan 所谓的“现实系统”来提供一条完全数字化的道路。

这意味着制造商需要能够将市场上发生的需求情况、车间里发生的生产情况以及世界上发生的供应情况联系起来，以了解全局，这样他们就不会像疫情早期卫生纸和个人防护设备需求激增时那样措手不及。

制造云旨在帮助收集这些信号，并在制造商可能需要额外供应时向他们发出警告。供应链洞察工具专门设计用于绘制供应链路线，并在瓶颈发生之前根除可能影响关键原材料交付的问题。它们旨在为制造企业带来更大的敏捷性和灵活性。

微软动态 365 供应链洞察工具。图片来源:微软

与过去不同的是，微软认识到，这类传统上现代化进程非常缓慢的公司不希望淘汰和替换现有系统，并承担一个庞大的项目。正如 Arkan 所说，你需要从一个成功的项目开始，然后转移到下一个项目，他说这个解决方案就是以这种方式设计的。

更重要的是，他说每个项目可以建立在另一个基础上，并通过节约和创新为下一个项目提供资金。“你数字化转型的每一步，与我们的每一次合作都创造了经济空间，可以为下一步增加资金，因为我们要么在 8-12 周内为你提供更多的顶线收入、容量，要么提供节约或质量改进，”他说。

这是一个大胆的承诺，但如果微软能够实现这一点——这是一个很大的假设——它将为传统上拒绝这种重大变革的公司创造一条舒适的现代化之路。

如果微软和其他大型企业公司，如 SAP 和 Salesforce 能够开始解决这些基本问题，它可能会开始缓解我们今天看到的供应链问题。虽然软件本身不能神奇地生产稀缺的原材料或雇佣足够的人来制造和运送供应品和商品，但它可以成为解决方案的一部分，帮助减轻未来的这类危机。
