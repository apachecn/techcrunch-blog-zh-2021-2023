俄罗斯数十名 IT 工作者签署公开反战请愿书 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/03/01/russian-it-workers-anti-war-petition/amp/

图片来源:肖恩·盖洛普/盖蒂图片社

自周四入侵乌克兰以来，一封公开信在该国的 IT 行业流传，抗议军事侵略行为，呼吁和平，这显示了俄罗斯专业人士反对乌克兰战争的强烈程度，这封公开信聚集了约 3 万名署名 IT 工人的签名。

请愿书的标题是:“俄罗斯 IT 行业代表反对乌克兰领土上军事行动的公开信。”

名单上的名字和职位——有些还注明了这个人为哪家公司工作——包括 IT 专业人士，他们自称是企业家、产品经理、客户体验主管、分析、后端开发人员、产品设计师、营销专员、devops 工程师、iOS 工程师、游戏开发人员、系统分析师、IT 招聘人员等等。

用来存放请愿书的谷歌文档长达 652 页。

这封信是用俄语写的，全文如下(用机器翻译):

我们，俄罗斯信息技术行业的雇员，坚决反对俄罗斯联邦武装部队在乌克兰领土上发起的军事行动。

我们认为任何导致战争爆发的武力表现都是没有道理的，并呼吁取消可能不可避免地导致双方人员伤亡的决定。我们两国一向关系密切。今天，我们担心我们的乌克兰同事、朋友、亲戚。我们对乌克兰城市现在发生的事情感到担忧，也感到道义上的压力。

在我们的工作中我们制造最好的产品，最好的服务，我们真诚地做每一件事，让俄罗斯的 IT 解决方案能够引以为豪。我们希望我们的国家不与战争联系在一起，而是与和平和进步联系在一起。

在战争和威胁人民生命和健康的情况下，造福人类的技术进步和发展是不可能的，只有在合作、观点多样化、信息交流和公开对话的条件下才有可能。

我们请求我国领导人重视我们的呼吁，找到和平解决这一局势的方法，防止人员伤亡。

不可能核实所有签名都是真实的，但我们已经确认请愿书是由一位名叫 Natalia Lukyanchikova 的女士发起的，她告诉我们她是一名“It 专家”。

她也是这封信的第一个签名者，在信中她将自己列为招聘网站 hh.ru 的产品分析师。

TechCrunch 通过电子邮件联系到了 Lukyanchikova，她解释说，她上周在脸书的公共页面上分享了和平请愿书，呼吁其他 IT 工作者签名，并动员起来，试图让媒体报道这一倡议。

我们还与另一名俄罗斯 IT 工人进行了交谈，他告诉我们他们已经签署了请愿书，并表示他们科技公司的许多人，包括其首席执行官，也签署了请愿书，但这位消息人士要求匿名，以避免引起对他们行动的关注，因为存在报复的风险。

Lukyanchikova 在她发起请愿的最初的脸书邮报中写道:“以下是来自 IT 社区的一封公开信。我不知道这是否行得通，但我知道集体行动有时会有帮助。这也有助于人们理解他们并不孤独。到目前为止，这是我看到的唯一没有被禁止的合法行为。您可以通过第一条评论中的链接注册。”

在她脸书主页的后续帖子中，随着签名数量的增加，也可以看到她在几天内跟踪进展。

随着俄罗斯对乌克兰的入侵加剧，这份请愿书似乎在俄罗斯 IT 行业迅速升温，截至周六，签名人数超过 1 万人，周日超过 2 万人，截至本文撰写之时，签名人数已接近 3 万人。

当被问及她是否相信请愿会产生影响时，卢克扬奇科娃告诉我们:“我想相信我们的声音会被听到，和平会得到恢复。”

俄罗斯国内的一些媒体报道了这封反战信。

例如，科技行业出版物 vc.ru 周六报道了这件事，当时报道称，签名者——当时人数约为 1300 人——包括俄罗斯科技巨头 Yandex 和 VKontakte 的员工，以及其他许多类型的企业。

它的报告还指出，国内其他行业的代表，如教师、科学家和医生，也发表了类似的反战信件。虽然 IT 行业的信似乎已经收集了最多的签名。

本报告更新了一些小的更正:我们最初声明申请已经结束；事实上，它仍然开放给更多的签名。此外，Lukyanchikova 的职位是“产品分析师”，而不是“食品分析师”，因为请愿列表中显示的西里尔文字是机器翻译的，我们曾用它从俄语翻译过来，(错误地)翻译了它。
