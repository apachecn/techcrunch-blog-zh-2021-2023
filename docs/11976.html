Equals 获得了 1600 万美元的投资来增强电子表格| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/11/09/equals-secures-15m-investment-to-supercharge-spreadsheets/

总部位于旧金山的初创公司 Equals 雄心勃勃地打算用超级电子表格挑战 Excel 的主导地位，今天宣布在由安德森·霍洛维茨(Andreessen Horowitz)领投的 A 轮融资中筹集了 1600 万美元，Craft Ventures、Box Group、Worklife 和 Combine 也参与了投资。Equals 由两名 Intercom 前员工本·麦克雷德蒙(Ben McRedmond)和鲍比·皮内罗(Bobby Pinero)共同创建，该公司声称，其电子表格是少数几个内置数据库连接、版本控制和协作功能的电子表格之一。

Equals 并不是第一家试图消灭传统电子表格的初创公司。当然，还有 Airtable，以及像 Spreadsheet.com、 Actiondesk 和 Pigment 这样的新贵——最后一家在去年 11 月为其数据分析和可视化服务筹集了 7300 万美元。但是 Equal 的首席执行官 Pinero 声称 Equals 是独一无二的，因为它没有取代电子表格，而是加入了额外的工具，如实时数据集成。

Pinero 在一次电子邮件采访中告诉 TechCrunch:“Equals 来自一个非常简单而明显的见解:电子表格是进行分析的最佳方式。”“Excel 是在将近 40 年前构建的。16 年前的谷歌床单。如今，公司的工作方式有了显著的不同。我们的数据更容易获取。我们应该将大部分将数据输入电子表格的痛苦的手工工作自动化。在过去的十年里，我们学到了很多关于团队如何更好地合作的知识。电子表格应该包含这些知识。这是平等的。”

无论是否使用 sequel 查询语言，客户都可以使用 Equals 直接从数据库或数据仓库中获取实时数据进行分析。它支持标准公式，并为常见用例提供模板，如跟踪经常性收入和测量用户参与度。

很快，Equals 将能够导入脚本，允许用户使用 JavaScript 或 Python 将电子表格连接到不同的 API 和内部工具。此外，Salesforce、QuickBooks、Stripe 和 Google Analytics 的数据透视表和商业智能应用的连接器也在路上。

“Equals 提供了一个巨大的机会，让业务利益相关者——通常是那些无法获得自己数据的人——访问数据。皮内罗说:“为了能够在一个他们已经熟悉并知道如何使用的工具中处理数据，他们需要一个电子表格。”。“不再需要花费数小时在整个团队中手动更新电子表格。不再需要将 BI 工具中的数据转储到电子表格中进行分析。”

这是一个很大的承诺，但皮内罗很清楚前面的障碍。他预计 10 人公司暂时不会盈利——他说，首轮融资所得将主要流向 R&D；Equals 迄今为止已经筹集了 2300 万美元——在下一个主要产品发布之前，该平台将一直处于等待名单中。皮内罗声称到目前为止已经有“数千”人注册了。

皮内罗说:“在这个市场上，我们能够筹集到一笔重要的 a 轮融资，这表明了人们的兴奋和牵引力。以我们目前的烧钱水平，我们已经有八年多的时间了。”“无论这次衰退会持续多长时间，我们都有能力挺过去。俗话说，世代公司是在经济低迷时期建立的，我们计划让 Equals 成为其中之一。”
