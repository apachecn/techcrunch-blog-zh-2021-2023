Sense75 是 Drop 自 2020 年以来第一款新的内部定制机械键盘

原文：https://web.archive.org/web/https://techcrunch.com/2022/08/30/drop-launches-the-sense75-its-first-new-in-house-keyboard-since-2020/

在疫情早期，随着机械键盘爱好的激增，许多公司竞相推出新产品。然而，在通过向更多受众提供定制机械键盘和许多配件来普及定制机械键盘方面，它可能比任何人都做得更多，在此期间，它大多将第三方键盘添加到其产品阵容中。然而现在，它正在推出 Sense75 ，这是两年来第一款全新的内部键盘。

顾名思义，这是一个 75%的键盘，这意味着你可以获得全套的功能键和箭头键，以及右侧的三个按钮(默认情况下，这些是删除、上一页和下一页)，以及一个旋钮，这已经成为这些天的标准。当然，它们是 RGB LEDs，包括底部的扩散器，这将产生 Drop 所谓的“视觉上吸引人的光环”，当然，还有热插拔插座，这样你就可以轻松地更换开关。键盘将支持通过 QMK 和 VIA 定制，以适应您的打字需求。

图片来源:掉落

Drop 还首次将越来越受欢迎的垫圈安装系统用于其内部键盘，这应该会增强声音，并在打字体验中增加一点弹性。

“当我们追求开发 75%的机械键盘时，我们致力于设计一款非常注重细节和优质材料的产品，它不仅超越了市场上的其他产品，还提供了我们社区会喜欢的打字和定制体验，”Drop 首席执行官 Jef Holove 说。“经过密集的设计和开发过程，我们很自豪地推出 Drop SENSE75，我们迫不及待地想让我们的社区试用它，因为没有他们我们就无法创造它。”

你可以选择一个预装版本，带有该公司的 Holy Panda X 开关，其新的 DCX 键帽和工厂调整的幻影稳定器，或者一个只包括键盘框架和 PCB 板的准系统版本。我很想看到更多的颜色选择，但目前你的选择是黑色(或 Drop 称之为夜幕)和电子白色(极地)，其中白色版本的溢价为 50 美元。

预装版本的黑色版本将花费 349 美元，白色版本将花费 399 美元，而准系统版本的黑色版本将花费 249 美元，白色版本将花费 299 美元。不过，目前你只能预订预装版本，发货时间是 11 月(在机械键盘领域，这是一个合理的提前期)。准系统版本要到“晚些时候”才能上市，这可能会让许多已经有一套最喜欢的开关和键帽的 Drop 粉丝有点失望。

仅凭外观很难判断一个键盘(尽管根据 Drop 迄今为止分享的图像，我确实喜欢这些键盘的极简风格)。我们将不得不看看几个月后他们到来时的声音和感觉。Drop 的新键帽做得很好，所以我对这款新主板也寄予厚望。但这个价位的市场竞争非常激烈，像 Keychron、NovelKeys、KBDfans 和其他许多公司都在这个价格范围内提供非常有竞争力的定制键盘。键盘迷如果不吹毛求疵的话什么都不是，所以几乎没有出错的余地。
