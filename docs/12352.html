web3 真的是互联网的新阶段吗？

还是 FTX 的崩溃揭示了它一直是一个噱头？

我们正处于网络新阶段的边缘，或者说故事是这样的。它的支持者已经将其命名为 web3 。虽然上周具有系统重要性的FTX 密码交易所的内爆表明，科技行业在执行方面还远远没有实现这一愿景，但 web3 的概念在过去几年中一直是初创公司和风险投资的基本驱动力。

如果我们真的处于第三次浪潮之中，那么理解网络的历史、它是如何演变的以及这个新阶段(如果它确实是一个阶段的话)在这个年表中的位置是很重要的。所谓的 web3 是互联网的下一个合乎逻辑的步骤，是对其发展产生持久影响的一个步骤，还是其他一些事情？

我们需要将我们所谓的 web3 放在适当的历史背景下，判断它的崛起是否真正标志着一个创新周期，或者只是现有技术的简单重新包装，以使其更适合渴望下一个重大事件的投资者生态系统。

很明显，确定网络新阶段的主要动机之一是伴随前两个阶段的难以置信的财富创造。web3 中的风险资本家、企业家和经营者不断重复着这样一句格言:web3 的现状类似于互联网的早期。因此，合乎逻辑的是，web3 领域的早期采用者和建立者相信，成为第一批进入者之一可能会有巨大的经济回报，就像 web 本身的早期一样。

当我们涉猎互联网的历史时(是的，真的！)，考虑一下网络是如何发展壮大的，你现在是否在 crypto 中看到了类似的动态。在这个 web3 愿景的动荡时刻，值得研究的是，FTX 的崩溃是否表明这一新浪潮一直是一个纸牌屋，或者今年 web3 公司的一连串破产是否只是在接受互联网不可避免的未来方面的一个挫折。
