Animoca、Galaxy Interactive 和 Polygon back 游戏公司 nCore 在 web3 中推出 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/01/17/animoca-galaxy-interactive-and-polygon-back-gaming-firm-ncore-in-web3-push/amp/

印度顶级游戏公司 nCore Games 的高管周一表示，该公司在新一轮融资中筹集了 1000 万美元，准备推出 web3 产品。

Animoca Brands 和 Galaxy Interactive 是加密领域的两家知名投资者，它们牵头提供了资金。流行的第二层以太坊扩展解决方案 Polygon 和 Hyperedge Capital 也参与了该轮投资。

包括 Amitabh Singhal、Sandeep Nailwal、Kanwaljit Bombra、Sohil Chand、Ashish Chand、Ram Madhvani、Rakesh Kaul、Mannan Adenwala、Sanjay Narang、Taj Haslani、Kristian Segerstrale、Sanjay Gondal、Vedant Baali、Kartik Prabhakara、Peter Leung、Yashraj 明石和阿克谢·查图维迪在内的数十位天使投资人也对该轮投资。

nCore Games 拥有许多游戏工作室，包括 Studio nCore、Dot9 Games 和 IceSpice。它的投资组合包括多人动作游戏 Fau-G 和职业板球手机，已经获得了数千万的下载量。

nCore Games 的联合创始人 Vishal Gondal 在接受 TechCrunch 采访时表示，这家初创公司计划今年推出并投资更多游戏。

但他说，融资的更大重点是向元宇宙扩张。他说，nCore Games 正在评估推出自己的 NFTs 和代币，可能会在未来几个月内推出。

按下载量计算，印度已成为全球最大的手机游戏市场之一。PUBG Mobile 在被新德里禁止之前，在印度已经积累了超过 5000 万的月活跃用户。但由于应用内购买的低渗透率和小额广告费，游戏公司一直难以有效地从这一用户群中赚钱。

银河互动公司的普通合伙人萨姆·恩格尔巴特在一份声明中说:“游戏在印度的增长已经是不可否认的，这个市场已经准备好再向前迈出一大步。”

Gondal 是一位游戏行业的资深人士，他将之前的公司卖给了迪士尼。他说，游戏经济正在发生转变，特别是在发展中市场，公司在很大程度上依赖广告来创收。他提到了 Axie Infinity 在菲律宾的受欢迎程度，这个市场类似于印度，应用内购买收入不高。

Gondal 将印度、印度尼西亚和菲律宾进行了比较，他乐观地表示，已经在几个类别和许多市场取得进展的 web3 产品将在印度的游戏中受到欢迎。Gondal 说，web3 的定位是为所有利益相关者提供正确的激励结构。

“目前，当你玩游戏时，对你来说最大的好处是成为某个社区的一部分。现在，当你在玩游戏时，你也有机会拥有游戏的价值，”他说。

IceSpice 的首席执行官 Tejraj Parab 表示，nCore Games 正在考虑制作一个通用令牌，其所有游戏以及合作伙伴公司的游戏都将提供给社区。他说，这家初创公司还希望将其技术提供给其他游戏开发商使用。

工作室 nCore 的首席执行官 Dayanidhi MG 说:“我们很高兴与全球领导者和非常成功的基金、公司和退伍军人一起拥抱新兴趋势和技术，以指数级增长将 nCore 带到下一个水平。”

Web3 在印度开始受到关注。由 Tiger Global 和 Sequoia Capital India 支持的印度初创公司 Faze 去年与全球板球管理机构 ICC 合作，为板球运动员推出了 NFTs。据两位知情人士透露，这家初创公司目前正在进行谈判，以从 Insight Partners 筹集新一轮融资。一些宝莱坞名人也与公司合作推出他们自己的 NFTs。

“nCORE Games 是由经验丰富、背景深厚的游戏行业专业人士创立的。这是我们在印度最有前途的投资之一，印度有一个巨大的游戏市场，具有巨大的增长潜力，”Animoca Brands 的执行主席和联合创始人 Yat Siu 在一份声明中说。“nCORE Games 拥有合适的团队来利用游戏中的区块链和 NFTs 向玩家提供数字产权，随着我们向全球开放的元宇宙迈进，我们期待着许多成功的产品。”
