SnapLogic 以 1.65 亿美元的 1B 估值筹集了 1.65 亿美元，帮助企业整合并自动化其分散的应用和数据 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/12/13/snaplogic-raises-165m-at-a-1b-valuation-to-help-enterprises-integrate-and-automate-their-disparate-apps-and-data/

随着越来越多的企业加入数字化转型的趋势，并将更多的传统工作带入现代工作时代，一家名为 SnapLogic 的公司建立了一个平台来整合这些应用和数据，并自动化使用它们的一些活动，该公司已经筹集了一大笔增长资金。该公司已经筹集了 1.65 亿美元，这笔钱将用于继续扩展其产品——特别是支撑其平台工作方式的人工智能——以及业务发展。

该公司已经证实，这笔融资的估值为 10 亿美元。第六街增长公司领投了这一轮，它没有披露其他投资者。之前的支持者包括 Arrowroot Capital(领投了前一轮 7200 万美元的融资)、Golub Capital、Andreessen Horowitz、Vitruvian(也领投了前一轮)、Capital One、Ignition Partners、微软和其他一些公司。迄今为止，该公司已经筹集了 3.73 亿美元。

新的估值对总部位于圣马特奥的 SnapLogic 来说是一次大幅度的上涨，snap logic 在 2019 年最后一次融资时的估值刚刚超过 3 亿美元(基于 PitchBook 的估计)。

然而，当你考虑到它所涉足的市场领域、客户基础和规模时，这并不奇怪。它为已经严重依赖数据服务的大型企业、It 和技术巨头以及其他主要企业提供服务，包括 Adobe、Aramark、Drax、Emirates、Qualtrics、Magellan Health、Schneider Electric、Siemens、work day——它表示，总共有“数千个”组织，每月通过约 310 万条“管道”处理约 2.7 万亿份客户文档。

SnapLogic 在与 Workato(今年早些时候筹集了资金)、Tray.io、MuleSoft 和其他公司的竞争中，最初是作为一家帮助企业在不同的应用程序之间整合和使用数据的公司而出名的——事实证明，这一领域对于多年来在多个云、容器、数据仓库和内部数据中心采用了许多应用程序和架构的企业来说具有难以置信的吸引力；现在我们正坐在所有这些数据的宝库中，他们需要找出如何以更好的方式来平衡和使用。

这最近已经让位于对自动化的大力推动，其逻辑是，一旦将这些应用程序和数据集中在一个平台上，人工智能和其他工具实际上可以使用这些应用程序和数据来完成一些更平凡和重复的工作。事实上，SnapLogic 如今不仅提供了从一个地方集成和管理应用和数据的技术，还提供了一系列可应用于这些资产的自动化功能。

该公司表示，事实上，从单一平台提供集成和自动化，从而使采购更容易，成本可能更具竞争力，一直是 SnapLogic 的增长动力之一。

SnapLogic 首席执行官 Gaurav Dhillon 在一份声明中表示:“企业自动化市场正在蓬勃发展，我们最新的融资进一步证实了我们在该领域的增长势头和产品领先地位。”“与点对点玩家不同，我们对企业的关注将释放应用程序、数据和 API 的力量。在后大流行时代，我们的客户将使用人工智能和自动化来彻底改变他们的混合劳动力。借助 SnapLogic，混合云和多云企业可以确保他们在公共云和私有云、SaaS、云数据仓库和 API 方面的大量投资获得回报。”

然而，这一领域并不能保证完全成功。拥有 MuleSoft 的 Salesforce 的首席收入官最近承认，该公司正在经历一段“艰难时期”，部分原因是员工流动，部分原因是销售增长放缓。这可能部分是竞争的结果，但部分也是因为企业涌入大型 IT 数字化转型项目的热潮可能正在进入一个稍微不那么活跃、更实际的磨合期。

当投资者评估更大的机会以及他们认为将使 SnapLogic 这样的公司保持业务增长的潜在趋势时，他们不会被这一点吓倒。

“现代企业正在使数据和应用程序的访问民主化，并使业务团队能够使用低代码、自助服务技术来构建他们需要的解决方案，以便更好、更快地工作，”第六街增长公司的合伙人兼联席主管迈克尔·麦吉恩在一份声明中说。“SnapLogic 经验丰富的管理团队、稳健的经济模式和可持续发展计划使其在利用蓬勃发展的企业自动化行业和不断扩大的混合工作场所趋势方面处于有利地位。我们很高兴领导这轮融资，并与 SnapLogic 合作，将其市场愿景、无与伦比的平台和强大的合作伙伴关系带给全球更多的企业。”

第六街是 SnapLogic 这一轮领先的有趣支持者。该公司已经支持了许多规模扩大的初创公司，这些公司已经上市或进行了大规模收购，如 Airbnb、MDLIVE、Spotify 和 Sprinklr。这就提出了一个问题，SnapLogic 可能会考虑什么样的长期未来。我们希望稍后能和迪隆谈谈，当我们了解到更多信息时，我们会更新这篇文章。
