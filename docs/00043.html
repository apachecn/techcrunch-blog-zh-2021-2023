Ghostery

，一家数字隐私公司。

当你想到今天的互联网格局时，谷歌压倒性的市场份额立即浮现在脑海中。

最新报告显示，谷歌占据了全球搜索市场近 87% 的份额，而 Chrome 占据了全球浏览器市场超过 67% 的份额，在全球范围内拥有26.5 亿用户。因此，任何创建在线工具的初创公司都必须以某种身份将自己视为科技巨头的合作伙伴，但这种市场控制也意味着谷歌可以利用其浏览器主导地位，随心所欲地改变监管规定。

事实上，在过去十年里，谷歌已经在 Chrome 的架构中加入了对在线创业公司的某些限制。几年前，谷歌开始执行“单一目的扩展”政策，要求开发者将其扩展限制在一个狭窄的焦点或浏览器功能上。

最近，谷歌为 Manifest V3 推出了计划——这是谷歌构建 Chrome 扩展的新规范，旨在提高用户的安全性和隐私性。从本质上讲，Chrome 网络商店从 2022 年 1 月开始将不再接受使用以前的 V2 清单指南构建的新扩展，到第二年，任何和所有不符合清单 V3 规定的现有程序都将关闭。

在这种情况下，一些初创公司面临着生死存亡的局面——要么花费时间和资源来调整他们的产品以适应 Manifest V3，要么完全停止在 Chrome 中存在。我的雇主 Ghostery 正面临着同样严峻的挑战。

虽然这些障碍可能会让一些人感到沮丧，但谷歌正在倾听，渴望听到这可能会对该领域的个人玩家产生怎样的影响。它甚至延长了部署时间表，以收集更多开发人员的见解，初创公司应该充分利用这一延迟，让人们知道他们具体的迁移挑战。

Chrome 的这次更新只是一个例子，说明了正在进行在线创新的初创公司可能会在任何时候被推翻，因此需要始终分配资源，以跟上这些政策的步伐，并让合适的工程师在甲板上导航这些细微的变化。

尽管如此，让你的核心业务托管在互联网上有着不可否认的好处，如广泛的可见性，易于安装和访问一个热切的用户群。如果像谷歌这样的科技巨头愿意与创业公司合作，寻找对各方都有利的最佳解决方案(看起来确实如此)，那么小公司就有责任利用这些开放的沟通渠道，为自己的产品宣传。

向你的关系网寻求支持

幸运的是，每个创业公司不必独自面对不断变化的互联网格局。由于一系列参与者在同一个生态系统中工作，公司有一个庞大的相关业务网络来获得支持和建议。在线领域的初创公司不应该感到孤立于他们自己的挑战，而应该感到被影响生态系统中所有公司的不断变化所团结。

例如，在导航清单 V3 的过程中，我们加入了用于 web 扩展的 W3C 小组。在这个社区中，我们可以分享我们的使用案例，并与其他受影响的公司合作调整我们的技术，以符合最新的准则。通过集中资源和参与这种交叉协作，我们可以在适应过程的早期排除故障。

无论是寻找相似的群体，还是直接咨询你的创始人和商业领袖网络，在产品和总体商业方向的过渡时期，依靠创业社区是至关重要的。

变得适应多种方式

除了主动分配资源以适应你的英雄产品，在线创新的初创公司需要不断重新评估他们的产品路线图，并寻找独特的机会来多样化他们的产品。

互联网历史上没有两年是相同的，但有一点是不变的，那就是个人在这个生态系统中拥有定制选项的愿望——敏捷的初创公司准备好提供最新、最棒的解决方案，从电子商务工具到密码保护系统、隐私套件等等。

对于一些公司来说，这可能意味着在 Chrome 以外的产品开发上投入更多时间，并研究 Firefox、Safari 或其他替代浏览器可能提供的功能。由于每个科技巨头都有自己与浏览器定制相关的政策，初创公司可以向全新的用户群开放，并建立与不同系统兼容的独特功能。

创业公司也可以扩展到移动领域，尝试他们的在线创新如何在 iOS 或 Android 环境中发挥作用。最终，拥有一个延伸到我们在线生活各个角落的产品范围，从浏览器到移动应用，将使你的公司能够适应我们在线世界的不断变化，如果多样化从一开始就是你商业计划的一部分，那么建立这种弹性就容易得多。

拥抱不断的变化

尽管创建在线工具和程序的初创公司不断受到政策变化、监管变化和市场需求的挑战，但那些不怕与大型技术合作并追求适应性产品战略的公司总是会找到一种方法，继续向消费者提供尽可能最好的产品和体验。

坚定不移地坚持你的使命，并愿意在前进的道路上修改你的方法，将会让你在未来的岁月里成功地创新。
