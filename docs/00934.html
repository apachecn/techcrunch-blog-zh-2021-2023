DevOps 市场繁忙，资金充足。举个例子，前几天我赶上了 Opslyft 的。该公司横跨印度和美国，正在构建一个统一的 DevOps 服务，该服务将用于创建软件的部署后方面的工具整合在一起。这是一家不错的公司，当它宣布融资时，我可能会花更多的时间来写它。GitLab，一个预先部署的 DevOps 服务，最近在上市，这是记忆中的另一个例子。

这就是说，大大小小的科技公司都在开发 DevOps 工具。我们看到机器学习操作(MLOps)市场开始快速模仿它的兄弟姐妹。TechCrunch 指出，MLOps 初创公司 Comet 本周筹集了，这提醒交易所我们最近看了一下最近的权重&偏向轮，这是 MLOps 初创公司的另一个资本事件。

我提出这些是因为前几天我们采访了蓝宝石风险投资公司的 Jai Das，为我们的文章《T2》收集更多的背景信息，这篇文章探讨了人工智能的筹资趋势。在聊天中，我提出了 AIOps 的想法，以及它是否会成为我们需要关注的第三个“Ops”类别。然而，根据 Das，“MLOps 基本上是 AIOps，”他说，所以我们可以将我们的思维限制在两个主要类别。

也就是说，AI 和 ML 并不是完全相同的东西——我们不要在这里争论，我只是随便说说——所以看看这两种不同类型的工作是否可以放在同一个软件篮子里将会证明是有趣的。

虽然地理上的差异引起了我们的注意，但随着人工智能的定义和应用范围的扩大，我们预计美元将得到更均匀的分配。例如，第三季度新诞生的两只拉丁美洲人工智能独角兽是食品科技公司 NotCo 和数字身份证提供商 Unico，而墨西哥贷款公司 Kueski 也获得了一大笔资金，我们本来会称之为金融科技，但也是人工智能支持的。如果这是人工智能的新现实，我们不会惊讶地看到更多的资金流入初创公司，利用它来解决世界任何地方的现实问题，包括拉丁美洲，但也包括非洲。

在回答我们关于人工智能聚焦的初创企业经济学的问题时，这位投资者不得不说:

在我看来，最近人们对人工智能的兴趣，主要是由正在进行大规模融资的公司的收入增长推动的。但收入增长背后的原因很简单:对产品的高需求和低劳动参与率。我们在 Point72 Ventures deep tech 投资组合中看到了这一点。人工智能有能力增强人类的能力，让他们更具生产力，在某些情况下，还能在非常适合自动化的任务中取代他们——让他们专注于更具附加值的战略活动。从历史上看，引入这种自动化的摩擦一直很大，但当你无法雇人处理客户服务请求或管理办公桌时，自动化突然变得有趣多了。

我们最近学到了很多关于宏观环境如何影响创业的知识。从不断上升的通货膨胀降低保险技术利润，到推动人工智能软件需求的巨大辞职。一些需要记住的事情。
