南非 Root 获得 300 万美元，计划将低代码保险平台扩展到欧洲——TechCrunch

在采用数字技术方面，保险业落后于其他金融服务。 诚然，现代数字保险领域已经冒出了一些独角兽，比如 Tractable 和 Lemonade 然而 ，这个部门还没有迅速发展到像那样的阶段。

全球保险市场价值超过 5 万亿美元，尽管有大型企业的存在，但仍有不同的机会被挖掘。

在非洲，一家初创公司正在为自己开拓利基市场。通过使用 API， Root 正在帮助企业和开发者以比传统方式更快的速度推出保险产品。这家总部位于南非的公司已经筹集了 300 万美元的种子资金，用于在国外扩大规模，并为全球数字保险经济建立基础设施。

风险投资公司 Invenfin ，Base Capital，Savannah Fund，P1 风险投资公司，Luno 和 FireID 参与了该轮投资，其中包括一些高影响力的天使投资人 。

Root 的出发点是，包括合规和资本在内的许多障碍阻止了软件开发人员和企业围绕保险进行构建和创新 。

首席执行官劳·霍普利告诉 TechCrunch，他在 2015 年注意到，当其他金融服务开始使用 API 来无缝地帮助开发者构建针对他们的产品的产品时，没有任何东西是围绕保险 构建的。

“我们有一种强烈的感觉，保险业停滞在 1900 年代。尽管消费者行为已经转向数字实时，但仍有大量纸张被使用。几乎每个行业都有一种个性化的环境，但保险业正在努力赶上，”他说。

“这就是我们试图打破这些障碍并将其打包在易于使用的 API 之后，以便人们可以开始解决这个新的现代数字个性化时代的问题 。”

Hopley 在 2016 年推出 Root 就是为了迎合这种需求。 Jonathan Stewart 于 2018 年作为联合创始人加入，领导销售和合作伙伴关系。

Root 的第一个产品是可编程的银行账户和银行卡，但它已经转向提供保险 API，这是它现在的核心产品。

Root plugs 在保险公司之上，保险公司 承保开发人员和业务分析师使用 Root 构建的产品，并集成到他们现有的客户体验中，这些体验可以存在于网站、聊天机器人或移动应用 上。

图片来源:根

成立五年的公司 主要是 瞄准非保险公司，拥有稳定的客户群；将保险作为次要产品的企业 。例如，电信公司、零售商和银行。然而，它也赋予了亲和保险公司权力；实际上是南非最大的一些。

这些客户包括 Mr Price Money、FinChoice、Telkom、Metropolitan、Sanlam 和 Guardrisk。 借助 Root 的保险基础设施，这些公司每月处理数百万份保单和数千份索赔 。

这家南非公司通过基于其客户需求的规模和复杂程度的订阅许可费来赚钱 。

除了 帮助开发商和企业推出保险产品外，Root 还为这些企业提供接触更多客户的渠道，并处理合规问题 。

有了这笔新资金，Root 打算推出其灵活的低代码数字保险平台，以扩展到其他市场，尤其是欧洲。

霍普利没有提到国家根会扩展成； 然而 从该地区的保险水平来看，英国、德国和法国可能会名列前茅 。

虽然 Root 计划在欧洲雇佣更多的技术和商业人才，但 Hopley 说公司团队的很大一部分将留在南非。

Hopley 说，尽管如此，该公司的重点是为全球保险业构建操作系统，尤其是在它扩展到新领域的时候。

我们认为 需要一个标准的平台，人们可以在这个平台上创新、制造产品并进行整合。 本质上是 ，这意味着我们希望成为一个平台，所有产品创新者、精算师和开发人员都将登录该平台，以构建保险产品并将其推向市场 。 我们认为 如果你这样做，你可以更快地推动更多的创新，因为你可以让次级经济体加速发展，让其他人开始将具体的解决方案融入保险业 。它 实质上 整合了整个产业

在一份声明中，Base Capital 的保罗·卢瑟福表示，Root 的 API 为复杂的保险世界带来了简化，并改善了消费者和保险公司的体验。另一方面，P1 风险投资公司的 Mikael Hajjar 指出，Root 可能是非洲 insurtech 初创公司获得欧洲和美国市场份额的最佳选择 。
