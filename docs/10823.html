快速的数据增长和不断扩大的数据法规(如 GDPR 和 CCPA 法规)给公司带来了巨大的压力，他们需要弄清楚自己拥有什么，并提供不断发展的法律体系所需的必要控制。

进入 Securiti ，一家 2019 年推出的创业公司，最初的想法是专注于数据控制的一个方面，数据隐私。但该公司的愿景是建立一个广泛的服务平台，不仅处理隐私，还处理数据安全、治理和合规性。

今天，它推出了名为 DataControls Cloud 的综合数据安全云，并宣布了 7500 万美元的 c 轮融资。该公司在短短三年内筹集了超过 1.55 亿美元。

数据安全云背后的想法是在数据所在的任何地方提供一层数据保护，无论是在三大云基础设施参与者 (AWS、微软和谷歌)，数据基础设施参与者如雪花或 Databricks，还是 SaaS 应用程序如 Box 或 Salesforce。

公司创始人兼首席执行官 Rehan Jalil 表示，该产品旨在控制所有形式的数据。“我们学到的是，控制是关键词，围绕数据的风险控制，无论是通过安全、隐私、治理还是合规的视角，”他告诉 TechCrunch。

他表示，监管环境和取悦消费者的愿望意味着公司必须在所有这些不同的层面上保护数据，如果他们获得了安全、隐私和治理权，合规性就会自然而然地随之而来。

它似乎在起作用。贾利勒表示，该公司已经获得了七位数和八位数的合同，这对一家成立三年的初创公司来说是不寻常的。它带来了三位数的季度环比增长，这反过来又引起了投资者的关注，尽管与前几年相比，获得风险投资的难度有所增加。

该公司从不羞于雇佣员工。在 2020 年获得 5000 万美元的 B 轮融资时，它已经拥有 185 名员工。今天它有大约 370 个，并计划在下一年翻一番。事实上，它现在有 50 个空缺，他说。

贾利勒说，随着他建立这家公司并继续增加员工，创造一种多元化和包容性的文化很重要。“如果你没有包容性的文化，没有多样性，没有人们能够以开放的方式进行合作的环境；他们有选择，尤其是在过去几年，”他说。他认为，如果员工不喜欢工作环境，他们会用脚投票。

今天的投资由猫头鹰岩石资本(Owl Rock Capital)领投，它是蓝猫头鹰资本的一个部门，现有投资者 Mayfield 和 General Catalyst 也参与了投资。根据协议条款，蓝枭资本的 Pravin Vazirani 将加入 Securiti 的董事会。
