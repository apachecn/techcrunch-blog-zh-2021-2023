FTC 起诉 Kochava 出售人们的敏感位置数据

原文：https://web.archive.org/web/https://techcrunch.com/2022/08/29/ftc-sues-data-broker-kochava-for-sale-of-peoples-sensitive-location-data-including-visits-to-reproductive-health-clinics/

美国美国联邦贸易委员会(FTC)周一宣布，已对数据经纪商 Kochava Inc .提起诉讼，原因是该公司出售“数以亿计的移动设备”的地理位置数据，这些数据可用于跟踪个人的移动，包括进出敏感地点的个人。具体来说，联邦贸易委员会表示，这些数据可以揭示人们对生殖健康诊所、家庭暴力或无家可归者收容所、成瘾康复中心和宗教场所的访问。

这些个人和私人信息可能会使人们面临“耻辱、跟踪、歧视、失业甚至身体暴力的威胁，”联邦贸易委员会在新闻稿中解释道。

该诉讼旨在停止 Kochava 涉及敏感地理位置数据的数据收集做法，并将要求该公司删除已经收集的数据。

它的到来进一步表明，联邦贸易委员会正在打击移动数据经纪人，这些经纪人的业务依赖于收集和转售消费者智能手机的数据——这是一种长期的行业做法，具有许多隐私影响，但往往不为受影响的最终用户所知。此前，苹果公司对追踪进行了重大反思，更新了其移动操作系统，允许消费者在每个应用程序的基础上选择退出一些数据收集做法。

据 TechCrunch 报道，最近，美国众议院监督委员会开始调查在后 Roe v. Wade 时代，周期跟踪应用程序和数据经纪人的商业行为如何可能将消费者的私人健康数据武器化。

总部位于爱达荷州的 Kochava 并不是一个家喻户晓的名字，但在数据收集行业有着相当大的影响。该公司是一家位置数据经纪人，提供消费者智能手机的精确地理位置数据，并从其他经纪人那里购买数据转售给客户。这些数据馈送通常由想要分析诸如本地商店或其他位置的客流量之类的事情的客户使用。这些数据本身非常精确，包括显示移动设备确切位置的带时间戳的纬度和经度坐标，此外还与唯一标识符相关联，如设备 ID 以及其他信息，如 IP 地址、设备类型等。

这个设备 ID 或移动广告 ID 是分配给消费者的移动设备的唯一标识符，以帮助想要向最终用户做广告的营销人员。虽然消费者可以在任何时候重置这个 ID，但是他们必须知道这样做，并且了解在他们设备的设置中这个选项在哪里可用。

根据联邦贸易委员会的投诉中引用的 Kochava 自己对其产品的描述，该公司为客户提供“每月约 940 亿+地理交易量的原始纬度/经度数据，1.25 亿每月活跃用户和 3500 万每日活跃用户，平均每台设备观察超过 90 次每日交易。”直到 2022 年 6 月，它在公众可访问的网站上，包括在 AWS 市场上，以订阅为基础出售其数据馈送。要访问该 feed，购买者需要一个免费的 AWS 帐户和 25，000 美元的 Kochava 位置数据订阅费。还提供了一个包含超过 3.27 亿行和 11 列数据的数据样本，这些数据与 6180 多万个不同的移动设备相关。

FTC 表示，这些数据是不匿名的，可以用来识别移动设备的用户或所有者。这是可能的，因为其他数据经纪人专门出售服务，将这些移动广告 id 与消费者的姓名和物理地址等离线信息进行匹配。

除了能够跟踪访问敏感地点的消费者，FTC 指出，这些数据可以用于推断消费者的 LGBTQ+身份或访问提供生殖保健以外的其他医疗机构。它也可以用来将该活动与某人的家庭住址联系起来。

此外，鉴于 Roe 诉 Wade 案的逆转，联邦贸易委员会指出，这一数据不仅可用于识别前往生殖健康诊所就诊的人，还可用于识别实施或协助实施堕胎服务的医疗专业人员。这是 VICE 的主板最近报道的主题，但它关注的是另一个名为 SafeGraph 的数据代理。在参议员沃伦和其他 13 名参议员写信给这些公司，要求他们就数据收集做法做出答复，并要求他们停止出售与堕胎诊所就诊相关的数据后，该经纪人和 Placer.ai 在 7 月份同意停止出售堕胎诊所就诊者的位置数据。

同月，谷歌表示将自动删除用户账户中“特别私人”地点的位置历史记录，包括堕胎诊所、收容所、戒毒中心等。它还建议 Fitbit 用户如何手动删除他们的日志。

FTC 旨在根据大量违反 FTC 法案的行为起诉 Kochava，包括涉及敏感数据不公平销售和消费者伤害的行为。它正在寻求一项永久禁令，以防止未来的侵权行为和法院决定的任何额外救济。

美国联邦贸易委员会消费者保护局局长塞缪尔·莱文在一份声明中说:“消费者寻求医疗保健、接受咨询或庆祝信仰的地方是私人信息，不应该卖给出价最高的人。”。“联邦贸易委员会将 Kochava 告上法庭，以保护人们的隐私，并停止出售他们的敏感地理位置信息。”

委员会以 4 比 1 的投票结果授权对科查瓦提起诉讼，委员诺亚·约书亚·菲利普斯是唯一投反对票的人

这一最新行动的消息并不令人惊讶。该机构曾在 7 月份警告企业，计划对非法使用和分享敏感消费者数据的行为进行执法，并在本月表示,正在探索新的规则，进一步打击“收集、分析和利用个人信息”的企业

然而，这也不是联邦贸易委员会第一次直接针对参与敏感数据收集的企业采取行动。去年，联邦贸易委员会对生育追踪应用程序 Flo 与第三方共享敏感数据采取了行动。该应用没有受到经济处罚，但值得注意的是，这是监管机构首次下令通知此类隐私行动。

Kochava 表示将于美国东部时间今天下午 2:30 发布声明。我们将更新它的回应。[参见下面的声明]

“收集我们的位置行为已经成为应用程序、移动电话运营商和其他‘位置情报’公司将我们的信息货币化的主要方式，”数字权利和消费者保护倡导者数字民主中心的执行董事杰夫·切斯特在 FTC 声明后的一份声明中指出。“联邦贸易委员会表示，我们访问的地方的信息是敏感数据，不能以监督营销业务所期望的方式使用。随着两党投票支持该诉讼，今天的委员会行动表明隐私对两党来说都是一个关键问题。它让数据和平台行业意识到，它面临着一场严峻的战斗，”他说。

Kochava 发布了以下声明:

这场诉讼表明了一个不幸的事实，即联邦贸易委员会对 Kochava 的数据市场业务和其他数据业务有着根本性的误解。Kochava 始终如一、积极主动地遵守所有规则和法律，包括专门针对隐私的规则和法律。在法律诉讼之前，Kochava 主动宣布了一项新功能，通过隐私块阻止敏感位置的地理数据，有效地将这些数据从数据市场中删除，目前正在添加该功能的实施过程中。由于 FTC 没有明确说明，我们一直在监控并主动调整我们的技术，以阻止来自其他敏感位置的地理数据。Kochava 在我们的数据市场中 100%的地理数据来自第三方数据经纪人，他们都表示数据来自同意的消费者。在过去的几周里，Kochava 一直致力于向 FTC 介绍数据的作用、收集数据的过程以及数据在数字广告中的使用方式。我们希望与联邦贸易委员会进行富有成效的对话，就这些复杂而重要的问题找到有效的解决方案，并在未来对他们持开放态度。不幸的是，联邦贸易委员会想要的唯一结果是一个没有明确条款或解决方案的解决方案，并将问题重新定义为一个移动的目标。通过浮夸的新闻稿和琐碎的诉讼，无法在改善消费者数据隐私方面取得真正的进展。令人失望的是，该机构继续规避立法程序，并延续围绕数据隐私的错误信息。
