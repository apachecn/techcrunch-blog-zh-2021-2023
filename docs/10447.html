肯尼亚初创公司 Ponea 在推动医疗服务普及方面势头强劲 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/09/23/kenyan-startup-ponea-gaining-momentum-in-driving-access-to-medical-services/

连续肯尼亚技术企业家迈克·马查里亚在过去的二十年里，他帮助像东非最大的电信公司 Safaricom 这样的企业创建基础设施，以促进它们的发展。他的公司七海科技之前也与政府合作过——参与了分散服务交付中心 Huduma 中心的初步实施，该中心通过一个单一的门户网站向公民提供几乎所有的政府服务。

几年前，马查里亚还与肯尼亚政府签订合同，建造现已废弃的国家医院信息系统，他说这将改变该国的医疗保健服务。然而，2019 年那个项目的结束并没有扼杀他的创新性；相反，这激励他创建了 Ponea Health 作为医疗服务市场。

“项目的结束给了我自我反省的机会，我意识到我一生都在为企业构建硬件和软件。但我从未想过为我们(大众)建造技术。我决定自下而上地建设，这就是 Ponea 的诞生，”这家初创公司的首席愿景官(CVO)马查里亚告诉 TechCrunch。

一次个人突发事件证实了这个想法的及时性。

“我开车从我家出来，流了鼻血，这是第二次了。我联系了我的医生，他建议我做一些测试。但我认为他怀疑我不会带他们；不管怎样，这是忙碌的一天。所以，他派了一名实验室技术员到我的办公室取样。在结果和医生的审查之后，我联系了一家药店，这家药店使用了一个附加条款来运送药物，”他说。

“在那一瞬间，我意识到一切都存在。医生、实验室、药房、支付和交付提供商都存在，但为什么它们没有互联互通？我发现没有人试图将整个生态系统结合成一个整体；这是我们需要做的事情。为什么不呢？”

这就是他如何着手将 Ponea 建设成一个“真正以病人为中心的平台”

Ponea Health 是一个多层次的市场，聚集了患者、医疗保健和其他服务提供商，包括支付领域的提供商。

它使用户可以根据各种因素(包括需求、位置和费用)轻松确定医生、设施和/或医疗保健套餐，因为费用也包含在列表中。

一旦用户与医生取得联系，根据其病情的严重程度，可以通过虚拟或物理方式进行咨询。此外，在医生建议进行测试的情况下，Ponea 会将患者与实验室提供商联系起来进行样本采集。

“我们也有自己的抽血师，他们有时会收集血液样本，因为我们意识到我们必须开始控制患者的最后一英里体验，”马查里亚说，他与阿克谢·沙阿共同创立了 Ponea。

他补充说，整个过程是由客户驱动的，并由呼叫中心(医疗运营中心)支持，以确保患者从登记到药物交付的平稳过程。

要想在 Ponea 的平台上上市，首先要使用其自己的评级系统对供应商进行审查和评级，该系统使用来自政府数据库和医生证书的数据。此外，患者还能够根据设定的指标对提供商进行评级，帮助根据客户体验和满意度对医生进行排名。

Ponea 成立于 2019 年 6 月，自去年 1 月以来，除了 15，000 多名客户外，还签约了 400 多名健康和福祉专业人士，随着该行业 COVID 诱导的增长，人们越来越多地寻求远程医疗，转化率为 54%。据说远程医疗弥补了非洲的医疗差距，非洲是世界上疾病负担最重、病人与医生比率最低的大陆。

随着远程医疗的不断普及，Ponea 的目标是在未来三年内将其客户群增加 500，000 人，并在其他四个市场建立业务，包括南非、尼日利亚、埃及和摩洛哥。

“我们已经建立了一个可扩展的产品，可以轻松地与其他产品集成，因为从一开始，我们就打算建立一个可以在世界任何地方工作的平台。这意味着我们不建造可用的东西。我们着眼于战略合作，允许在全球范围内进行 API 集成。

“而且，我们已经找到了全球和本地的优秀公司，我们以惊人的方式与之合作，这包括我们目前正在评估的全球精神健康症状检查器，并将很快集成到我们的平台中，”他说，并补充说该平台也能够从可穿戴设备中收集数据。

Ponea 的产品包括针对患者及其护理人员(如护士)的慢性病管理元素，有助于同步信息和数据，以便更好地监控疾病。

这家初创公司还迎合中小企业或无法负担综合保险的企业，或者那些只寻求门诊服务的人。员工可以从一系列预先选择的提供商那里获得服务，而且与保险不同，Ponea 钱包余额不会过期。

迄今为止，Ponea 已经吸引了来自 Afya Partners、以色列 Shield Capital、七海技术公司和一些天使投资人的 430 万美元资金，其中包括 Bhavesh Shah、Herman Langen、Franciscus Olsthoorn 和 Kalpesh Mehta。
