Cloudflare 希望用 Turnstile | TechCrunch 取代 CAPTCHAs

原文：https://web.archive.org/web/https://techcrunch.com/2022/09/28/cloudflare-wants-to-replace-captchas-with-turnstile/

在 10 月份的 Connect 大会之前，Cloudflare 本周宣布了一个名为 Turnstile 的雄心勃勃的新项目，该项目旨在消除整个网络中用于验证人们是否真实身份的验证码。无论是否是 Cloudflare 客户，网站所有者都可以免费使用 Turnstile，它从一系列“浏览器挑战”中进行选择，以检查网页访问者实际上是否是机器人。

验证码，我们大多数人在填写表格时都会遇到的挑战-反应测试，已经存在了几十年，它们在阻止机器人流量方面相对成功。但是，廉价劳动力的兴起、各种验证码风格的漏洞和自动解算器已经开始在系统中戳洞。一些网站提供人类和人工智能支持的验证码求解服务，每千个求解的验证码低至 0.50 美元，一些研究人员声称基于人工智能的攻击可以成功求解世界上最受欢迎的网站使用的验证码。

Cloudflare 本身也曾经是 CAPTCHA 用户。但是根据首席技术官 John Graham-Cumming 的说法，该公司从未对其感到满意——如果 Cloudflare 的公众集会口号没有明确表达这一点的话。在与 TechCrunch 的一次对话中，格雷厄姆-卡明列出了他认为验证码技术的许多缺点，包括可访问性差(视觉障碍可能导致无法解决验证码)、文化偏见(验证码假设熟悉美国出租车等物体)以及验证码给移动数据计划带来的压力。

图片来源:云闪

“验证码最大的问题是用户体验很糟糕。随着计算机在解决这些问题方面变得越来越好，用户体验只会越来越差，”格雷厄姆-卡明在一次电子邮件采访中说。

Cloudflare 一度转向了一项名为 hCaptcha 的服务，对此褒贬不一。一个常见的挑战要求用户输入他们的名字，说他们喜欢茄子还是胡萝卜，并点击 27 张显示火车的图片中的每一张。根据 Graham-Cumming 的说法，这种反作用以及一些验证码服务收取的费用是促使 Cloudflare 开发自己的替代品的一部分。

“我们几年来一直在研究一种解决方案，几个月前，我们在博客上写道，我们如何将自己的验证码使用量减少了 91%。因为我们已经证明它对我们有用，所以我们想给每个人一个摆脱验证码的选择，”他补充道。

Cloudflare 表示，Turnstile 会根据“会话期间展示的遥测和客户端行为”自动选择浏览器挑战，而不是登录 cookies 等因素。在运行非交互式 JavaScript 挑战来收集关于访问者和浏览器环境的信号，并使用人工智能模型来检测功能和以前通过挑战的访问者之后，Turnstile 根据具体的请求微调挑战的难度——避免让用户解决难题。

要部署 Turnstile，网站管理员需要创建一个 Cloudflare 帐户，并获取必要的嵌入代码，然后将这些代码粘贴到他们网站的代码中。在向 Cloudflare 的 Turnstile API 添加服务器端调用后，该服务就上线了。任何站点都可以调用 API。

“如果你今天使用现有的验证码服务，它只是在代码串上查找和替换，”Graham-Cumming 说。“它与任何其他网络提供商兼容…您不必使用任何其他 Cloudflare 服务，如我们的内容交付网络，就可以使用 Turnstile。”

展示 Cloudflare 十字转门系统工作原理的图表。图片来源:云闪

Cloudflare 声称 Turnstile 与 CAPTCHA 一样安全，利用了私有访问令牌等功能来最大限度地减少收集的数据量。在 iOS 16 和 macOS Ventura 中新实现的私人访问令牌通过让设备向兼容网站发送匿名认证信息(令牌)来工作，而不会暴露任何关于自身的敏感信息。

Cloudflare 和竞争对手 service Fastly 是首批宣布支持苹果硬件私人访问令牌的公司之一。

问题是网站是否会被说服在现任验证码上部署十字转门。根据一项指标，访问量排名前 100 万的网站中有 97.7%使用谷歌的 reCAPTCHA，这是目前市场上最受欢迎的验证码服务。Cloudflare 表示，它正在为 WordPress 等主要平台开发插件，以使 Turnstile 更容易部署，但可能需要时间来说服管理员，这是值得努力的——假设他们曾经被说服。

Graham-Cumming 似乎对此漠不关心，指出 Cloudflare 没有明显的商业动机来推动采用。

“我们建立了一个替代方案，证明它对我们很有效，并尽快向其他网站开放，”他说。“既然我们已经证明它对我们有用，我们想给每个人一个摆脱验证码的选择。帮助互联网变得更好是我们的使命。我们认为向任何网站提供这种服务都是一种方式。”

就下一步而言，Graham-Cumming 表示，私有访问令牌是 Cloudflare 未来发展方向的最佳指标。他承认，该公司过去测试过一个基于 USB 的安全系统，但需要硬件会增加很大的摩擦。

“客户和网络都越来越关心隐私和数据分割。Graham-Cumming 补充道:“我们可以将部分验证工作抽象给其他方，而无需自己收集数据。“例如，[人们]提到生物认证。我认为我们更有可能与硬件制造商合作，使用私人访问令牌为我们进行生物识别验证，并将证明验证的加密令牌传递给我们，而不是我们自己进行生物识别验证。”
