向来自 web3 加速器联盟 DAO 演示日的最新加密初创公司问好| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/11/09/say-hello-to-the-newest-crypto-startups-from-web3-accelerator-alliance-daos-demo-day/

在周三的 Alliance DAO 演示日期间，新的加密初创公司在 FTX 内爆中稳步前进。

“这实际上比 2018 年的熊市更糟糕，”Alliance DAO 的核心撰稿人王峤表示，他指的是币安收购竞争对手 FTX 的初步计划。“今天，每个人都措手不及，包括我自己……在过去的三个月里，我与我们的创始人密切合作，我再次充满希望。正是像他们这样的人将推动我们的行业向前发展。”

web3 加速器和构建者社区 Alliance DAO 的最新团队 All9 在周三的演示日展示了他们的想法，TechCrunch 对此进行了独家报道。我们讲述联盟刀之前的试玩日仿佛就在昨天，但是四个月过去了。

每年两次，Alliance DAO 邀请 web3 创始人参加为期三个月的项目。王说，到目前为止，联盟道校友已经创造了超过 100 亿美元的市值。大约有 953 人申请这个项目，但是只有 17 个团队被选中并从这个项目中毕业。

王说，许多团队正在研究改进加密主题，如物理工作证明、日常用户的钱包体验、产品驱动协议、加密 B2B 产品、链上数据和垂直化。"这个群体是这个行业正在发生的事情的一个相当有代表性的样本。"

该团队专注于整个 web3 的一系列子行业和产品，如身份认证、流动赌注、加密钱包和分散式机器学习等。大约 38%的初创公司专注于以太坊生态系统，而 Polygon 占 21%，而 Solana 占 17%。

导师包括 Brevan Howard Digital 的风险投资联席主管科琳·沙利文(Colleen Sullivan)；该街区 LinksDAO 的创始人、第六人风险投资公司的投资者迈克·杜达斯；索拉纳联合创始人阿纳托利·亚科文科；Messari 创始人兼首席执行官 Ryan SelkisMounir Benchemled，ParaSwap 创始人；0x Labs 联合首席执行官阿米尔·班迪利(Amir bande Ali)；多边形工作室的首席执行官瑞安·怀亚特。

以下是 17 家初创公司的分类:

公司名称:OraT3】

  • 它做什么:web3 的搜索引擎
  • 创始人:丹尼斯【安特拉】马丁内斯T6】，T8】马斯聪金，T12】雅各布·希奥拉
  • 阶段:种子
  • 推介:Ora 的联合创始人 Jacob Shiohira 在推介中表示，随着生态系统的不断扩张，Ora 正在为 web3 打造一个搜索引擎。“区块链数据令人困惑，我们可以在今天的探索者身上看到这一点，”他说。他补充说，为了让 crypto 成为主流，需要有每个人都能理解的产品。Ora 旨在通过一个界面将 sequel 引擎、仪表盘和浏览器结合起来。它从索引第一层区块链索拉纳的数据开始，但计划扩展到整个 web3。

公司名称:门沙里T17】

  • 它做什么:QuickBooks for web3
  • 创始人:维杜尔·贾因，马努杰·帕利瓦尔
  • 阶段:种子
  • 推介:Mensari 旨在通过自己的 QuickBooks 平台帮助参与 web3 的企业。联合创始人 Vidur Jain 在他的演讲中说，随着组织集成加密，他们的簿记复杂性正在“飙升”。Mensari 建立了一个复式记账系统，为具有 web3 特质的组织提供运营会计和投资组合会计，如资产互换、支付流、NFTs 和流动性供应。该平台在区块链以太坊和 Polygon 上运行，目前已经有 15 个测试版客户。

公司名称:张量T25】

  • 业务:聚焦索拉纳的 NFT 聚合器
  • 创始人:理查德·吴、伊尔贾·莫伊塞耶夫
  • 阶段:种子
  • 推介:Tensor 的目标是建立一个聚焦索拉纳的 NFT 聚合器。Ilja Moisejevs 在他的推介中说，目前的 NFT 市场迎合散户投资者，而不是交易者，随着越来越多的专业人士进入 NFT 市场，他们需要更复杂的工具。Tensor 的目标是为 Solana 的 NFT 交易员提供更多服务，让他们“只需点击一下鼠标，就能在 8 个最大的市场进行数百笔交易”

公司名称:RaleonT7】

  • 它的作用:Web3 营销分析
  • 创始人:内森·斯内尔，亚当·拉森
  • 阶段:种子
  • 推介:Raelon 正在为 web3 建立一个营销分析公司。联合创始人内森·斯内尔说，目前的营销技术不能满足 web3 工具或链上和链下数据的需求，因此需要进行改进，以考虑新技术。Snell 说，Raleon 将区块链数据与 Web 2.0 数据结合起来，创建了一个“丰富的身份”，因此项目可以针对 web3 用户。该平台正在寻找战略投资者，以获取客户。

公司名称:SpexigonT15】

  • 它的作用:无人机图像的物理验证工作
  • 创始人:亚当·基拉姆，彼得·辛姆扎克，比尔·莱克兰，亚历克·威尔逊
  • 阶段:种子
  • 推介:Spexigon 是一个验证无人机图像的物理工作平台，无人机所有者可以通过在飞赚奖励系统中捕捉图像来获得加密。一旦算法验证了图像并将其推向市场供企业和开发者消费，飞行员就会获得令牌。在获得政府机构、工程公司和铁路公司等的兴趣后，该平台在过去的八个月里已经获得了超过 150 万美元的收入。在 Alliance DAO、Dapper Labs 和其他公司的参与下，它已经在种子期筹集了 550 万美元。

公司名称:SlashAuthT3】

  • 它的作用:Web3 身份验证
  • 创始人:奈德·洛克森，尼古拉斯·萨尔瓦娜
  • 阶段:种子前期
  • 推介:SlashAuth 是一个 web3 身份验证平台，旨在帮助开发人员跨 Web 2.0 和 web3 生态系统创建分布式安全身份。联合创始人 Nicolas Salhuana 在他的演示中说，它以加密方式合并了两层的身份，因此用户可以通过一次登录访问他们所有的帐户。它目前处于封闭测试阶段，已经从 Alliance DAO 和 Y Combinator 等投资者那里筹集了 300 万美元。

公司名称:跨步T11】

  • 它做什么:宇宙聚焦液体打桩
  • 创始人:赖利·埃德蒙兹，艾丹·萨尔兹曼，维沙尔·塔拉萨尼
  • 阶段:种子
  • 推介:Stride 正在为第一层区块链宇宙建造一个液体打桩平台。用户可以用他们的代币换取 Stride 的 ST 代币，然后 Stride 用这些代币换取用户在 Cosmos DeFi 生态系统中使用的代币。该平台于八周前推出，但其总价值为 700 万美元，经常性收入为 15 万美元，用户超过 8000 人，联合创始人 Vishal Talasani 表示。Stride 已经从潘迪拉、Distributed Global、北岛风险投资公司、1 confirmation and Staking Facilities 等投资者那里筹集了 670 万美元。它正在寻找 LP 承诺，以换取 Stride 令牌。

公司名称:风气T21】

  • 作用:Sui 专用加密钱包
  • 创始人:娜迪亚·埃尔迪布，贾里德·科苏利奇
  • 阶段:种子
  • 推介:Ethos 是 Sui 区块链的“钱包超级应用程序”。该钱包提供类似 Web 2.0 的功能，如用电子邮件登录，这使开发者能够提供更好的用户体验，联合创始人 Nadia Eldeib 在她的演示中说。钱包已经建立了一个连锁游戏，Ethos 用户可以通过它的钱包预先批准交易。自上个月推出以来，其 Chrome extension wallet 的每周用户数已超过 3.2 万。Ethos 已经从 FTX、部落资本和 Mysten 实验室筹集了 240 万美元。

公司名称:连锁 MLT7】

  • 它的作用:分散式机器学习
  • 创始人:大卫·穆勒，伊森·杰克逊，罗恩·博德金
  • 阶段:种子
  • 推介:Chain ML 是一个去中心化的机器学习平台。它允许智能合同和 dApps 利用机器学习，旨在不损害审查阻力、防篡改或可用性。它利用分散的数据库、oracles 和运行机器学习服务的计算网络来提供审查阻力。Chain ML 最近在 iOS G 牵头的种子轮融资 400 万美元，其 MVP 将于 2023 年第一季度推出。

公司名称:维纳里传说T17】

  • 它的功能:web3 的口袋妖怪 Go
  • 创始人:杰森·杜，德鲁·埃尔金斯
  • 阶段:种子
  • 推介:Venari 传奇采用了 Pokémon GO 的概念，并通过 metaverses 将其引入数字世界。联合创始人 Jason Du 在演示日表示，其软件开发套件将重点从游戏资产的所有权转移到整个游戏生态系统的所有权。自 10 个月前推出以来，这款游戏平均每周有 3000 名活跃用户，并与几个 NFT 社区如 Fractal 合作。它已经在由 Lattice Capital 牵头的种子轮融资中筹集了 540 万美元。

公司名称:排序T3】

  • 它做什么:Web3 firebase
  • 创始人:杰森·祖切托
  • 阶段:种子
  • 推介:Sort 旨在成为开发者构建 dApps 的一种更简单的方式。创始人 Jason Zucchetto 在演示日说:“编写一个分散的应用程序仍然非常痛苦。Sort 希望通过为开发人员提供工具包和剪切粘贴代码来简化编写开源 dApps 的过程。Zucchetto 说，每天有超过 200 人查看他们的教程。Sort 目前正在筹集种子资金。

公司名称:人物角色

  • 它的作用:像 Dao 一样建立群聊
  • 创始人:雷兹·洛加特，威尔·多恩伦，亚力山大·格林
  • 阶段:种子
  • 推介:Persona 正在将群聊打造为“轻量级 DAOs”，联合创始人 Raeez Lorgat 在演示日说道。Dao 的受欢迎程度在过去的一年里有了显著的增长，但是形成和管理 Dao 的工具仍然是跨平台的。Persona 希望简化流程，以便用户可以在几分钟内为任何特定的共享目的汇集和管理资金。迄今为止，Persona 已经从 ECCO 资本、Alliance DAO、CYGNI Capital 和 character 等投资者那里筹集了 450 万美元。

公司名称:口口

  • 它的作用:连锁商品
  • 创始人:卡莱多拉·基尔南-林，马可·安东尼奥·里贝罗
  • 阶段:种子
  • 推介:Ostium 是一个旨在将大宗商品市场带上链的平台。联合创始人 Kaledora Kiernan-Linn 在演示日表示，非机构投资者无法进入许多商品市场。Kiernan-Linn 补充说，商品市场的问题可以通过分散融资来解决。通过像 python 这样的 oracles 镜像链上的链外资产，像 Ostium 这样的协议旨在以市场价格提供资产的流动部分暴露。Ostium 正在为种子轮融资，并寻求与具有流动性供应专业知识的投资者合作。

公司名称:SliseT3】

  • 业务:web 3-原生广告平台
  • 创始人:奥莱克西·西多罗夫，格里芬·高
  • 阶段:种子前期
  • 推介:Slise 是一个专注于 web3 数字广告的平台，旨在分析链上活动，以提供跨 web3 生态系统的定制广告。它可以在区块链像以太坊，索拉纳和多边形。Slise 在去年夏天赢得了 Polygon Build It hackathon，并正在筹集种子资金。

公司名称:时空T11】

  • 它的作用:分散的数据仓库
  • 创始人:内特·哈乐黛，斯科特·戴克斯特拉
  • 阶段:A 系列
  • 推介:Space and Time 是一个去中心化的数据平台，旨在通过智能合同技术帮助企业发展。它使用“SQL 证明”加密技术，允许传统集中式系统中的业务逻辑实现自动化，并直接连接到智能合同。它还计划将该平台与微软 Azure 集成。9 月，时空在由微软 M12 牵头的一轮战略融资中筹集了 2000 万美元。

公司名称:0x passT21】

  • 工作内容:为分散式应用提供类似 Web 2.0 的用户体验(dApps)
  • 创始人: Keon Kim ， Krish Chelikavada
  • 阶段:种子前期
  • 推介:0xPass 旨在为 dApps 带来类似 Web 2.0 的用户体验，以便主流用户能够以一种与钱包无关的方式享受现代 UX 的优势。其 web3 用户体验的一些变化包括不带钱包的用户加入，预先批准交易，以及将 0xPass 与几行代码集成。上周发布了针对 onboarding SDKs 的封闭测试版，正在 10 个项目中进行测试。到目前为止，它已经筹集了 60 万美元，并正在筹集种子资金。

公司名称:纽拉尔T3】

  • 它的作用:符合加密工资
  • 创始人:纳宾·班斯科塔，娜美·巴拉
  • 阶段:种子
  • 推介:Niural 是一个兼容的加密薪资平台。它在 Polygon 上建立了一个支付协议，能够以较低的汽油费为菲亚特和 crypto 提供实时工资，并建立了一个以 web3 为中心的人力资源平台，可以处理税收和错误分类风险。它与拥有 20，000 多名员工的 Renfroe 和团队成员遍布 72 个国家的 Scrum Launch 等公司合作。Niural 已在种子轮融资 500 万美元，参与方包括 Inspired Capital、New Form Capital、Alliance DAO 等。它目前正在寻找战略投资者。
