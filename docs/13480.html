PayPal 和 MetaMask 联手让购买 crypto | TechCrunch 更容易

原文：https://web.archive.org/web/https://techcrunch.com/2022/12/15/paypal-and-metamask-team-up-to-make-it-easier-to-buy-crypto/

PayPal 主要被认为是一种在线支付方式。但该公司希望成为一种简单的加密货币入门方式。在这方面，MetaMask 背后的公司 ConsenSys宣布将在其加密钱包中添加一个集成，以便用户可以使用他们的 PayPal 帐户购买加密货币。

MetaMask 是最受欢迎的非保管加密钱包之一。它可以让您存储加密资产并与 web3 产品交互，因为您可以使用您的钱包作为您的身份验证方法。

但是，如果你有一个空的 MetaMask 钱包，你就不能做太多事情。这就是为什么用户依赖比特币基地、北海巨妖和 FTX 等中央加密货币交易所来购买加密货币并将其转移到他们的 MetaMask 钱包中。MetaMask 在其移动应用程序中也有自己的入口功能，因此您不必切换到另一个服务并通过许多中间步骤。On-ramp 合作伙伴包括 MoonPay 、 Wyre 和 Transak 。

如果您与这些合作伙伴之一购买 crypto，您将必须通过 KYC 程序(“了解您的客户”)。这意味着你将不得不输入一堆个人信息，并用某种形式的 ID 来验证你的身份。

MetaMask 和 PayPal 的合作将使两家公司受益。在 MetaMask 方面，现有入口解决方案的转换率可能并不高。KYC 进程可能会令人生畏。

根据该公司最近的收益报告，全球已经有 4.3 亿个贝宝账户。如果 MetaMask 用户看到一个大按钮，上面写着你可以用 PayPal 账户购买加密货币，这听起来会很容易，很熟悉。至于贝宝，更多的活动意味着更多的收入。

起初，MetaMask 用户将只能使用 PayPal 作为支付方式购买以太坊(ETH)。在向美国所有人推广之前，它将对美国的一些用户开放。

如果您的 PayPal 帐户中已经有 ETH，您可以使用这些 ETH 为您的 MetaMask 钱包充值。如果不是这样，PayPal 将帮助您用 PayPal 余额或其他支付方式购买 ETH。

这将为贝宝带来一些收入，因为该公司对购买加密货币收取费用。这是 PayPal 首次作为 web3 钱包的入口提供商进行整合。但如果我们在加密钱包中看到更多的贝宝按钮，我不会感到惊讶。

今年早些时候，PayPal 也在 T2 增加了对加密转账的支持。美国的 PayPal 用户可以获得钱包地址，用加密资产为他们的 PayPal 账户注入资金。同样，PayPal 用户可以向第三方加密钱包汇款。

由于许多人认为加密货币是互联网货币，他们认为加密可以完全取代贝宝，成为从电脑和手机收发资金的方式。但传统银行账户和加密钱包之间总会有桥梁。PayPal 计划利用这一点。
