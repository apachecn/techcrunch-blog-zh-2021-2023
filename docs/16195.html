生殖人工智能已经看起来像是 2023 年的主要科技趋势。通过算法生成新鲜内容的能力已经被像 ChatGPT 这样的推向了公众意识，ChatGPT是一种聊天机器人式的技术，它经过大型语言模型 (LLMs)的训练，能够生成散文、诗歌、歌词、新闻文章甚至计算机程序。然后是 DALL-E ，来自同样的微软支持的 OpenAI 产生了 ChatGPT，它服务于类似的目的但是用于视觉创作。

虽然有些人认为 ChatGPT 标志着人工智能进入主流，但事实是，我们正处于人工智能应用的新时代的开端，这将改变工业的方方面面，从消费者搜索、股票摄影到房地产、内容营销。

正是在这种背景下，一家名为 Tavus 的初创公司正寻求通过让公司创建为特定个人量身定制的“独特”视频来留下印记，但完全基于单一的初始记录。

这个想法是，例如，销售和营销团队可以向潜在客户发布源源不断的视频推销，可能基于潜在客户通过在线表格提交的文本数据。或者，一家猎头公司可能会利用该平台，利用从潜在候选人 LinkedIn 个人资料中收集的数据，向他们发送多个个性化视频。

Y Combinator (YC) 校友 Tavus 于 2020 年由首席执行官 Hassaan Raza 和 Quinn Favret 在旧金山创立，今天宣布已在硅谷投资者红杉(Sequoia)领导的种子轮融资 610 万美元，包括 Accel Partners、Index Ventures、Lightspeed Ventures 和 YC 连续性在内的一系列高调支持者参与了融资。

它是如何工作的

任何希望制作多个个性化视频的公司都知道，这是一个极其耗时、重复的过程:用实质上相同的内容录制相同的信息，但针对不同的客户或候选人进行调整。这就是 Tavus 正在寻求解决的问题:允许用户在几分钟内创建自己的人工智能视频模板，然后从原始来源生成无限数量的视频版本。

初始入职流程要求用户——例如，招聘人员或销售主管——根据 Tavus 提供的脚本录制 15 分钟的视频，用于训练 AI。然后，用户为他们想要创建的每个活动记录一个模板。

Tavus:读取脚本以创建基础模板。图片来源 : Tavus

使用基于网络的编辑器，用户可以选择他们想要个性化的视频元素，指定每个变量(例如，公司、高管姓名或位置)，添加行动号召等。

Tavus:用变量个性化脚本。图片来源 : Tavus

Tavus 还通过 ChatGPT 支持的片段支持更长形式的变量，以实现更个性化的介绍，Favret 表示这是其用户高度要求的。例如，创建视频的基本脚本可以配置为包含从特定个人的 LinkedIn 个人资料生成的一句话的介绍。

Tavus:生成个性化内容。图片来源 : Tavus

从本质上说，Tavus 正在努力复制大众营销软件多年来在电子邮件领域所做的事情，将它带入更具视觉吸引力的视频世界。事实上，这可能会给一些人带来一些危险信号:当人们发现发送者没有真正努力为他们制作视频时，他们会接受个性化的推销吗？

但更重要的是，人工智能生成的视频使用从数据库中收集的个人信息，这可能有点令人毛骨悚然——当用户发现它是如何制作的时，一些旨在成为个人的东西最终可能会被视为难以置信的非个人。

随着人工智能在我们的日常生活中变得更加根深蒂固，这种场景提出的问题将继续出现。Favret 很快强调，虽然销售和营销是其技术的明显用例，但它并不完全是针对这些垂直行业的——它看到了“一个不拘一格的用户群体”的吸收，包括招聘人员、大学院长和首席执行官。

“有一种普遍的误解，认为 Tavus 只与销售和营销团队合作，”Favret 说。“虽然这是我们的一个重点，但我们有用户在整个客户旅程中以创新和强大的方式应用 Tavus。我们的许多高级用户在他们的组织中广泛应用 Tavus，包括客户成功、产品、招聘和其他上市相关职能。”

到底是谁在申请克隆呢？

“通常，用户克隆他们自己，但对于企业来说，有一个核心人物，如高管或发言人，录制视频以保持公司的一致形象也很常见，”法弗雷特说。" Tavus 是为所有类型的用户设计的，他们可以在几分钟内轻松地克隆自己."

在引擎盖下，Tavus 表示，它使用机器学习来训练一个面部手势和嘴唇运动的模型，创建一个系统来逼真地模仿这些运动，并与合成音频同步。

至于部署，公司可以通过两种主要方式访问 Tavus:一半的用户通过其 web dashboard 使用它，而其他人则通过 API 或本机将其集成到自己的系统中。

“我们经常看到销售团队直接通过该平台使用和部署 Tavus，因为他们能够高效地为活动生成大批量视频，”Favret 说。“其他团队将以更加程序化的方式使用 Tavus，将它直接集成到他们的系统中。这允许用户创建‘事件驱动’的工作流，其中 Tavus 视频可以在触发后生成并发送。”

举一个“事件驱动”的例子，如果一个潜在客户要在一个公司的网站上提交一个表单，该公司可以使用该客户提交的数据自动生成一个定制的 Tavus 视频并发送给那个潜在客户。

“这使得公司能够利用合适的时机获得最大的转化率，而不必等待团队成员录制视频，”Favret 说。

滥用

在生殖人工智能领域的高端，我们看到像微软和谷歌这样的公司一决雌雄，看谁能以最快的速度将他们各自的智慧送到企业和消费者的手中，这场战斗微软目前似乎正在赢得。与此同时，我们看到一大批生成性人工智能初创公司脱颖而出，如 GlossAi ，这是使用人工智能来帮助企业轻松创建可共享的营销短剧，而 Typeface 正在为企业中的营销文案和图像生成做类似的事情。

具体到 Tavus，几年前就已经有类似的公司了，比如 Windsor ，它也做类似的事情，尽管重点放在电子商务上。我们还有总部位于伦敦的 Synthesia，由一批知名投资者支持，主要是从文本中创建数字化身，用于培训和操作视频。

因此，很明显，即使在过去几个月所有的生殖人工智能炒作之前，初创公司界也在朝着这个方向发展，Tavus 现在正在寻求利用这一点。事实上，在其短暂的生命中，Tavus 已经获得了一些相当大的客户，如房地产数据公司 CoStar 和法国科技公司 scaleup AB Tasty 。

然而，值得考虑这种技术的潜在误用。例如，有没有什么可以阻止任何人上传其他人说话的视频，然后根据该模板创建新的视频？当然，视频和语音领域不乏深度造假的例子。一些公司，比如南方公园创造者的创业公司 Deep Voodoo ，已经在为他们的 deepfake 项目筹集风投资金。随着这种技术在社会中变得越来越根深蒂固和正常化，将会有更多关于这一切的道德含义的问题，即使潜在的意图是善意的。

根据 Favret 的说法，Tavus 有一个内置功能，使其更难欺骗系统，因为它要求用户在平台上进行语音验证和现场录音。

“这意味着用户不能上传其他人谈话的视频，”法弗雷特说。“此外，用户对他们的数据和肖像拥有所有权利，允许他们随时删除或移除他们的视频。我们还非常关注 Tavus 的使用方式:在用户开始使用 Tavus 之前，我们会筛选每个用例，确保用例符合我们的社区指南，并且符合道德规范。安全和道德对我们来说非常重要，尤其是考虑到这项技术在社会中的年轻性。”

给我看看钱

在商业模式方面，Tavus 向提供了一个针对小型企业的基本介绍计划，每月费用为 275 美元，并有限制，如他们可以创建的视频数量。定制“商业”计划消除了这些限制，尽管没有广告价格——它基本上是根据公司打算如何使用它来定制价格。

“Tavus 计划是根据公司的具体使用情况和需求定制的，但在高层次上，我们采用基于使用的模式，根据用户拥有的座位数量和他们生成的视频数量向用户收费，”Favret 说。

在这一轮种子期之前，Tavus 已经筹集了少量资金，作为其在 2021 年参与 YC 计划的一部分。其种子轮投资者的完整名单包括:红杉、Accel Partners、Index Ventures、Lightspeed Ventures、YC 连续性、SV Angel、Hack VC、Remus Capital、Mantis Capital、Liquid2 Ventures、Zillionize、Soma Capital、GTMfund、Terra Nova 和几家未披露的天使投资者。
