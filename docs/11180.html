印度电子商务巨头 Flipkart 推出元宇宙购物体验

Flipkart 推出了一款元宇宙产品，供消费者以更互动的方式发现和购买新产品，这是这家印度电子商务巨头的最新赌注，它正在试验 web3 产品，以增强其客户体验。

沃尔玛支持的总部设在孟加拉鲁鲁的公司与 Polygon 孵化的公司 eDAO 合作，在元宇宙发行股票，称之为 Flipverse。该产品正处于试点阶段，旨在本月的节日期间吸引人们的兴趣。

在 Flipverse 上，该公司为消费者提供“游戏化、互动和身临其境”的体验，消费者可以在购买时收集公司的忠诚度积分——super coins——以及合作伙伴品牌的数字收藏品。Flipverse 也有一些休闲游戏。

在周一的一次简报会上，Flipkart 表示，包括 Puma、Noise、Nivea、Lavie、Tokyo Talkies、Campus、VIP、Ajmal 香水和 Himalaya 在内的“各种品牌”正在合作在 Flipverse 上建立体验影院。

“这个想法是让数百万用户体验 Flipverse，并打开未来购物的大门，”该公司表示。该公司的高管承认，其 web3 产品仍处于试验阶段，但他们表示，他们相信它有能力最终成为 Flipkart 未来的重要组成部分。

“15 年前，我们是第一家推出基于 Web 2.0 的电子商务的印度公司。我觉得今天我们是第一家推出 web 3.0 商务的公司，”Flipkart 的一位高管在简报会上说。

Flipkart 演示文稿中的一张幻灯片。图片来源: Flipkart

Flipkart 及其在印度的主要竞争对手亚马逊(Amazon)正日益扩大产品范围，以接触南亚市场的新客户，并留住忠诚的客户。亚马逊上个月底在印度推出了 QVC 式的直播购物，带来了超过 150 名创作者的大军来主持直播并在视频中插入产品。

“虽然我们刚刚开始触及元宇宙的表面，但我们认为电子商务是最重要的使用案例之一。在虚拟环境中，顶级品牌与 Flipkart 的电子商务专业知识相结合，必将彻底改变我们所知的在线零售。Polygon 的联合创始人 Sandeep Nailwal 在一份声明中说:“Flipverse 将是元宇宙的一种充满活力的可见表达，我很自豪这种激活发生在 Polygon 上。”

与 Flipkart 更广泛的合作是 Polygon 的最新胜利，因为它积极吸引大品牌。最近几个月，以太坊扩展平台已经与包括 Stripe、Meta 和星巴克在内的许多公司达成合作。

Flipverse 并不是 Flipkart 首次涉足 web3。今年早些时候，该公司与 Carl Pei's Nothing 合作，向从该平台购买智能手机的人提供独家 NFTs。

Flipkart 实验室副总裁兼产品战略和部署负责人那仁·拉乌拉在一份声明中说:“电子商务的未来增长将受到今天沉浸式技术的影响，元宇宙是这个领域中具有巨大潜力的重大变革之一。”

“Flipverse 的推出将继续对电子商务等创新行业产生影响，并在提供游戏化和沉浸式购物体验的同时增强客户体验，特别是在印度多个品牌采用元宇宙和 web3 平台的情况下。通过为客户提供访问他们喜欢的品牌、优惠、超级商品和数字收藏品的机会，我们旨在改善他们在虚拟和沉浸式环境中的购物体验。”
