以太坊开发者工具平台 Hardhat 在 a16z 和其他公司的捐赠下成为非营利组织——TechCrunch

流行的以太坊开发者工具平台 Hardhat 的创建者 Nomic Labs 今天宣布，它将成为一个非营利组织，目标是改善该协议的开发者生态系统。

首席执行官兼联合创始人 Franco Zeoli 在接受 TechCrunch 采访时表示，2018 年首次推出的 Nomic 正在更名为 Nomic 基金会，并已经获得了 3000 万美元筹款目标中的 1500 万美元捐款。根据 Nomic 的说法，最初的承诺来自以太坊基金会、以太坊联合创始人维塔利克·布特林和许多加密交易所和风险投资公司，包括比特币基地、孔森斯、安德森·霍洛维茨、The Graph、Polygon、Chainlink、a_capital 和卡泽克风险投资公司。

为了实现 3000 万美元的目标，该集团计划向几个分散自治组织(Dao)提交资助提案。

开发者工具可以让软件工程师在以太坊这样的区块链上创建应用程序变得更加简单，从而加速以太坊这样的特定协议的发展。事实上，根据 Zeoli 的说法，23，000 个 GitHub 存储库或开发者项目使用 Hardhat，代表了数万名活跃用户。他补充说，包括 Uniswap、ENS 和 AAVE 在内的知名加密项目都是 Hardhat 用户。

Zeoli 说，Nomic 基金会的主要目标之一是通过建设基础设施为他们提供高质量的体验来吸引更多的开发者加入以太坊协议。

“我觉得以太坊需要真正钉钉的核心方面有两个。一个是扩展…但如果你有一个高度可扩展的系统，但没有人使用，那么这是毫无意义的，”Zeoli 说。他继续说，开发者的采用是以太坊协议未来的核心。

虽然 Zeoli 和他的联合创始人在 2015 年开始探索比特币的潜力，并且成立了 Nomic Labs，致力于各种协议的加密项目，但该公司在获得以太坊基金会的拨款后，于 2019 年转向专注于 Hardhat 产品。Zeoli 说，这笔赠款引发了 Nomic 和以太坊基金会之间的密切合作关系，导致后者成为 Nomic 转向非营利地位之前的唯一资金来源。

泽奥利说，泽奥利和他的联合创始人帕特里西奥·帕拉迪诺都生活在他们的祖国阿根廷，阿根廷经历了货币的大幅波动和贬值。根据 Zeoli 的说法，阿根廷公民面临的与这些波动相关的挑战清楚地说明了加密可以作为有价值的替代方案的领域——这是 Nomic 决定成为非营利组织的一个激励因素。

Nomic 基金会将不仅仅支持 Hardhat，还将致力于支持其他开发者工具，以改善整个以太坊生态系统。

Zeoli 说:“(对经济基金会来说)未来的一个成功场景将是对安全帽的竞争显著增加，因为我们知道，随着整个行业的不断增长，单靠安全帽不足以满足整个行业的需求。”。“需要有多样化的解决方案、不同的方法、不同的策略，甚至不同的偏好。”
