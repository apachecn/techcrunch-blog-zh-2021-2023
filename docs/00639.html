Workato 在为其企业自动化平台| TechCrunch 融资 2 亿美元后，市值飙升至 57 亿美元

原文：https://web.archive.org/web/https://techcrunch.com/2021/11/10/workato-storms-to-a-5-7b-valuation-after-raising-200m-for-its-enterprise-automation-platform/?guccounter=1

商业世界自动化的竞赛正在进行，今天，市场上一个较大的参与者已经获得了另一轮重大融资，以满足加快工作速度、降低重复程序的一些成本以及或许在这个过程中获得一些新的见解的需求。

Workato 建立了一个平台，让组织建立和运行自动化解决方案，以加快工作流程中的重复流程，该公司已经完成了 2 亿美元的 E 轮融资，估值为 57 亿美元。Workato 表示，在考虑未来的首次公开募股之前，它计划利用这笔资金继续投资其平台，并进行收购。

Workato 首席执行官兼联合创始人 Vijay Tella 告诉 TechCrunch，“这是我们最终上市的计划，但还没有正式的时间表。”“现在，我们专注于为您想到集成和自动化时会想到的公司提供服务。我们相信客户可以从墙到墙的集成中受益匪浅，因此我们的目标是让我们的客户使用量随着时间的推移继续增长，并保持我们已经走上的收入道路。”

与此一致，Workato 在发布融资消息的同时，还披露了它用部分资金进行的一项收购: RailsData ，一家建立了基于 API 的技术来整合应用程序、数据和设备的初创公司。在被收购之前，Workato 一直是总部位于印度钦奈的 RailsData 的客户，因此它已经在使用这家较小的初创公司的技术来支持自己的一些自动化解决方案，特别是通过 API 为 Workato 的大客户大规模部署服务。

该公司表示，Battery Ventures 领导了最新一轮融资，Insight Partners、Altimeter Capital 和 Tiger Global 也以同等金额参与了融资。Workato 目前已经筹集了 4.21 亿美元，今年更普遍的是融资和增长:仅在今年 1 月日，这家总部位于山景城的初创公司以 17 亿美元的估值筹集了 1.1 亿美元。

估值大幅上升的部分原因是该公司的增长。现在有许多公司，包括 Salesforce 的 MuleSoft、UiPath、Boomi、ServiceNow、Tray.io、SnapLogic、Blue Prism、IBM 和许多其他公司，提供不同的方法来为组织提供这种或那种自动化，随着这个领域的成熟，我们甚至开始看到越来越多的方法针对特定的用例和垂直领域。

Workato 的宣传是，它已经成功地在自动化和非技术用户之间架起了一座桥梁，这些非技术用户最有可能想要使用(并使用这些工具构建机器人流程自动化和其他工作流自动化体验)。

它有大约 50 万个“配方”可供公司使用，涵盖人力资源、营销、销售、财务、It、收入运营和产品等运营领域(其自动化功能可以嵌入到其他服务中)，以及 Salesforce、Slack、ServiceNow、雪花和 Workday 等流行的企业应用程序。尽管许多 RPA 和其他自动化供应商吹捧面向非技术人员的“低代码”甚至“无代码”解决方案，但事实往往并非如此。

Tella 说:“ Workato 从不同的角度看待集成和自动化问题，它们是一枚硬币的两面。“如今，iPaaS、ETL/ELT 和 API 管理等集成工具都专注于数据层面，而不是端到端的流程自动化或工作流。RPA 和 BPM 之类的工具从业务流程的角度解决问题，但缺乏强大的连接或集成基础，导致脆弱、难以实施和更改工作流。

“我们认为业务流程与底层数据密不可分，”Tella 继续说道。" Workato 的自动化建立在行业领先的 iPaaS 或集成基础之上。集成或数据是橡胶遇到道路的地方，自动化是如何完成工作的。我们在一个平台上独一无二地做到了这两点。一个work torecipe 是一个处理数据和流程级集成的单一概念。这不是一把隐藏在集成和自动化技术背后的瑞士军刀。“这个想法是，这种方法对非技术业务团队和 IT 团队都有效。

该公司表示，它有大约 11，000 家企业作为客户，名单上有大量科技公司和非科技公司。它包括 Atlassian、美国电话电报公司、Adobe、Autodesk、Box、Stitch Fix、GitLab、NYU、诺基亚、Lucid Motors、Broadcom 和 Toast。

Workato 在 1 月份提到了 7000 名客户，该公司的年度经常性收入在去年翻了一番多。它似乎也与客户进行了更深入的互动，用户从 1 月份的 7 万人增加到 10 万人。Tella 说，它目前最大的垂直行业是技术、金融服务、零售、物流和运输以及医疗保健。

当然，在过去两年中，企业 IT 的一个大趋势是“数字化转型”，特别是系统的现代化，以应对这样一个事实:由于新冠肺炎，我们现在的工作方式不同了，消费者对他们所接触的组织提出了不同的要求。这是投资者对该公司感兴趣的一个重要原因。

Altimeter Capital 合伙人 Pauline Yang 在一份声明中表示:“过去 10 个月的事件使卓越运营变得比以往任何时候都更加重要，企业明白，企业自动化是实现更高敏捷性、创新和性能的基石，他们需要在任何环境中取得成功。“自动化产品的市场只会继续扩大，我们相信 Workato 拥有最好的技术、合作伙伴和领导团队，可以抓住这个令人兴奋的机会。”

Tella 指出，Workato 的方法是最独特的可扩展方法之一，这要归功于它的构建方式。

他说:“如今，销售、营销、财务、人力资源、客户和其他业务的规模和范围意味着，1 万家公司中的 1 万名销售人员不是在重复发明同一个轮子。”"这是站不住脚的，无益的，结果也不会很好."他指出 Workato 的食谱生成器是人工智能驱动的，“因此它会推荐下一步措施，并帮助你默认构建正确的食谱。”

他将 500，000 个社区创建的食谱描述为“自动化和运营的 GitHub ”,用户可以找到一个自动化，并使其适应自己的公司和用例。“这是建立在 Workato- 专利克隆技术的基础上的，该技术使得重复使用不同公司创建的食谱成为可能，这些公司拥有像 Salesforce 这样的应用程序，这些应用程序的定制与你自己的非常不同。模板模型不工作为重用。”

该公司未来几年的重大机遇将是利用全球对更多自动化工具的需求。Workato 计划在亚太地区增加一个新的数据中心，以补充现有的业务；随着美国市场的进一步增长，它还将在 EMEA 地区带来更多的火力

Battery 是该公司的忠实支持者，为该公司提供视角和内部人士对 Workato 如何发展的看法。

“自 2018 年领导 Workato 的 B 系列以来，我们很高兴地看到 Workato 的客户——其中许多是各自行业的领导者——的使用量稳步增长，以及 Workato 在自动化市场的领导者地位不断巩固，这是技术领域发展最快的领域之一，”Battery Ventures 的普通合伙人尼拉杰·阿格拉瓦尔(Neeraj Agrawal)在一份声明中说。“我们一直坐在前排观看 work 真正转变其客户的许多业务，这是我们和该公司其他现有投资者对领导新一轮融资感到兴奋的一个原因。"

更新了客户总数的新数字，从 7，000 增加到 11，000。
