Appsmith，一个构建商业应用的低代码平台，以 4100 万美元成交

原文：https://web.archive.org/web/https://techcrunch.com/2022/06/22/appsmith-takes-open-source-route-to-accelerate-internal-business-app-development/

构建商业应用的低代码平台 Appsmith 以 4100 万美元成交

在过去的几年里， Appsmith 一直在悄悄地为内部开发团队构建一个开源平台，以构建他们需要的定制应用。这些是公司倾向于自己构建的业务应用程序，以便以更简化和可视化的方式访问数据。

虽然该公司将其设计为低代码，但它旨在加速有经验的开发人员的开发过程，而不是像一些低代码应用程序环境的目标那样，帮助业务线用户构建应用程序。

今天，该公司宣布了 4100 万美元的 B 轮融资，在风投资金越来越难获得的时候，这是一个非常重要的轮次，尤其是对于一家预营收公司来说，但投资者正在关注这家初创公司的增长潜力，将开源项目作为跳板。

公司联合创始人兼首席执行官 Abhishek Nayak 表示，虽然它们属于低代码类别，但它实际上是一款面向希望在构建内部应用程序时速度更快的开发人员的产品。

“这些应用程序旨在供内部团队成员使用，而非外部客户。所以这就是 Appsmith 的重点。Nayak 解释说:“对于开发人员来说，它的工作速度快得多，这就是为什么你可以将它视为低代码产品，因为它确实意味着开发人员编写的代码量更少……但许多开发人员将 Appsmith 视为构建应用程序的框架。

它通过提供构建应用程序所需的大量组件来实现这一点，无论是前端组件、数据连接器还是访问控制，开发人员通常都必须从头开始构建这些组件。这涉及到提供拖放用户界面组件。其次，开发人员可以连接到数据源，或者使用 API 连接到现成的数据源。最后，它处理访问控制，因此您可以控制谁可以在内部使用该工具。每个组件都是可定制的，因此开发人员有很大的灵活性来适应 Appsmith 提供的组件。

他说，该公司目前有 95 名员工，分布在 16 个国家，目前大多数在印度，其次是美国和尼日利亚。他计划在明年初之前将员工人数增加一倍，并相信，在打造一支多元化的员工队伍时，地域多元化会有所帮助。

他指出，偏远地区尤其有助于雇佣更多女性。“我们已经看到，远程和灵活的工作方式确实有助于我们吸引更多女性来和我们一起工作。虽然我们肯定还有很多可以做的，但我们对能够雇用女性的速度感到非常高兴，尤其是在世界各地，而不仅仅是在印度，”他说。

虽然该公司仍处于预收入阶段，但其用户群在去年增长了 15 倍，每月有超过 7，000 家活跃公司使用该平台。新的投资为他赢得了时间，让他在构建不可避免的商业产品之前专注于开源社区。他认为他至少有四年的时间，所以有足够的时间来开发产品并获得收入，但他说，他将在年内开始考虑这一点。

今天的 4100 万美元 B 轮投资由 Insight Partners 领投，Accel、Canaan Partners、OSS Capital 和个人天使投资人参与。
