谷歌地图以全新的城市“沉浸式视图”挑战苹果的 3D 模式 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/05/11/google-maps-challenges-apples-3d-mode-with-a-new-immersive-view-for-cities/amp/

图片来源:谷歌

面对来自苹果地图及其 3D 城市视图的日益激烈的竞争，谷歌今天推出了自己对下一代谷歌地图的愿景，预览了更加“身临其境”的新观看体验。谷歌 I/O 大会主题演讲期间展示的这一增强功能，利用计算机视觉和人工智能技术的结合，将街景和航空图像融合在一起，以提供一个世界的数字模型，并提供一种探索城市、关键地标、餐馆、场馆和其他名胜的新方法。

谷歌表示，它已经融合了“数十亿”图像来创建这种身临其境的视图，允许用户通过在某个区域上空飞行来探索，看看它可能看起来像什么。例如，如果您计划去伦敦旅行，您可以使用该功能查看大本钟或威斯敏斯特等标志性建筑，以更好地了解这个地方、体验和建筑。你还可以使用“时间滑块”来调整该地区在一天中不同时间的样子——这一功能有点类似于苹果地图的夜间模式，即使在浏览 3D 城市时，月光也会在黄昏时激活。

图片来源:谷歌

然而，谷歌的沉浸式模式将额外允许用户查看当地天气和交通状况，以帮助规划。

这种新模式不会仅仅停留在以更加身临其境的视角展示主要城市，它还将使更容易探索内部场所成为可能，包括附近的餐馆和其他受欢迎的场所。

图片来源:谷歌

用户将能够滑行到街道级别，然后点击查看他们可能想去的地方的内部情况。这可以帮助人们弄清楚一家餐馆可能会有什么样的氛围。从这一层还可以看到该地区的生活繁忙程度和附近的交通。

图片来源:谷歌

不过，这项功能不会马上在所有地方都可用。

最初，它将在年底前开始推广到包括洛杉矶、伦敦、纽约、旧金山和东京在内的主要城市，未来几个月还会有更多城市跟进。谷歌表示，它将跨平台和设备运行，今年晚些时候将在 Android 和 iOS 上推出。

该公司还在活动期间宣布了其他一些谷歌地图更新。该公司表示，其环保路线功能于去年秋天在美国和加拿大推出，旨在帮助司机找到最省油的路线，并将于今年晚些时候扩展到欧洲。

图片来源:谷歌

到目前为止，这一增加估计已经减少了超过 50 万公吨的碳排放，相当于减少了 10 万辆汽车上路。谷歌表示，欧洲的扩张将使这一数字翻倍。

与此同时，对于开发者来说，谷歌宣布推出新的 ARCore 地理空间 API，这将为开发者带来地图的实时视图功能，供第三方应用程序使用。

图片来源:谷歌

实时视图是将 AR 箭头和方向覆盖在真实世界之上的功能，就像通过智能手机的摄像头看到的一样。这个想法是，当你在一个新的地方，你可能会迷失方向时，你可以拿出你的手机，看看到底该走哪条路。这就好像你把自己放在了谷歌街景里。

现在，谷歌还将允许其开发合作伙伴开发利用实时取景技术的产品。

图片来源:谷歌/Docomo

一个合作伙伴是微移动公司 Lime，该公司正在使用 API 帮助伦敦、巴黎、特拉维夫、马德里、圣地亚哥和波尔多的通勤者找到停放电动滑板车和电动自行车的地点。Telstra 和 Accenture 正在使用它来帮助体育迷和音乐会观众在墨尔本的漫威体育场找到他们的座位、特许摊位和餐厅。在日本，Docomo 和好奇号(Curiosity)正在开发一款新游戏，让玩家在东京地标(如东京铁塔)前与机器人同伴一起抵挡虚拟龙，这款游戏也是通过 API 提供动力的。
