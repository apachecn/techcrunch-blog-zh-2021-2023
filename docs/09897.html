蒂姆·库克宁愿让你买一部 iPhone，也不愿让你使用 RCS

蒂姆·库克并不喜欢丰富的通信服务(RCS)。苹果首席执行官在 Code 2022 大会上明确表示了这一点，当时一名观众要求苹果采用 RCS，以便他的母亲可以更好地看到他发送的视频，库克告诉他，“给你妈妈买一部 iPhone。”

对于外行人来说， RCS 就像超负荷短信，让你像 WhatsApp 和 Telegram 等其他聊天应用一样发送高质量的多媒体文件，而还提供现在标准的消息功能，如打字指示、发送和阅读回执、等。

关于 iPhone 上 iMessage 的蓝色气泡和其他短信的绿色气泡的争论由来已久。谷歌一直在向苹果施压，要求其采用丰富的通信服务(RCS ),以便 Android 用户可以向 iPhone 用户发送多媒体消息，泡沫战争将永远结束。最近，它发起了一场新的运动，试图将消费者的情绪转移到这场斗争的一边，声称现在是苹果“解决短信问题”的时候了

该运动的网站指出:“这与泡泡的颜色无关。视频模糊、群聊中断、阅读收据和打字指示器丢失、不能通过 Wi-Fi 发短信等等。这些问题之所以存在，是因为苹果拒绝在 iPhones 和 Android 手机用户相互发短信时采用现代短信标准。”

目前，全世界有超过 5 亿 RCS 用户使用谷歌的消息应用程序。

但是库克声称苹果的客户并不需要这种支持。据《The Verge》报道，在回答一个关于 RCS 支持的问题时，库克说“我没有听到我们的用户要求我们在这一点上投入大量精力”。

如果你使用 WhatsApp、Signal、Messenger 或 Telegram 等其他消息应用，就不存在不同气泡的问题——这是客户转向第三方解决方案而不是内置客户端的一个原因。但第一方解决方案仍然有吸引力，许多 iPhone 用户继续使用 iMessage 作为他们首选的聊天应用程序。苹果明白，保持一个封闭的 iMessage 系统也鼓励人们购买 iPhones，因为他们不想成为他们群聊中的绿色泡沫。
