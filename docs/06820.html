一个人工智能筹集了 800 万美元来策划特定业务的自然语言处理模型

原文：https://web.archive.org/web/https://techcrunch.com/2022/05/30/one-ai-raises-8m-to-curate-business-specific-nlp-models/

无论是翻译还是文档摘要，企业都在增加对自然语言处理(NLP)技术的投资。根据 John Snow Labs 和 Gradient Flow 的 2021 年调查, 60%的技术领导者表示，与 2020 年相比，他们的 NLP 预算增长了至少 10%,而三分之一的人表示支出增长了 30%以上。

这是一个竞争激烈的市场。除了资源丰富的初创公司，如 OpenAI 、 Cohere 、AI21 Labs 和humping Face以及包括谷歌、微软和亚马逊在内的科技巨头，还有一批新的供应商在开源人工智能模型的基础上构建 NLP 服务。但是 Yochai Levi 并不气馁。他是 NLP 平台 One AI 的联合创始人之一，该平台今天以 800 万美元的价格悄然出现，由Ariel mais los、 Tech Aviv 、Sentinel One 首席执行官 Tomer Wiengarten 和其他未透露姓名的风险公司和天使投资人领导。

“虽然市场正在快速增长，但先进的 NLP 仍然主要由专家研究人员、大型科技公司和政府使用，”Levi 通过电子邮件告诉 TechCrunch。“我们相信这项技术已经接近成熟，在过去几次从零开始构建 NLP 之后，我们决定是时候将其产品化并提供给每个开发人员了。”

图片来源:一个 AI

Levi 列出了他认为困扰 NLP 开发的主要挑战。他认为，通常很难管理开源模型，因为它们必须与正确的领域和任务相匹配。例如，一个被训练用来对医疗记录进行分类的文本生成模型不太适合一个设计用来制作广告的应用程序。此外，模型需要不断地用新数据重新训练——以免它们变得“陈旧”举个例子，OpenAI 的 GPT-3 回答了“谁是美国总统？”这个问题答案是“唐纳德·特朗普”，因为它是根据 2020 年大选前的数据训练的。

Levi 认为解决方案是为特定商业用例训练的一套 NLP 模型——换句话说，就是一个人工智能的产品。2021 年，他与首席执行官阿密特·本(Amit Ben)、首席采购官阿维夫·德罗尔(CPO Aviv Dror)和首席安全官阿西·谢弗(CSO Asi Sheffer)合作，追求这一想法。在 LogMeIn 收购了他的第二家初创公司 Nanorep(一家人工智能和聊天机器人供应商)后，Ben 之前是 LogMeIn 的人工智能负责人。Dror 帮助共同创建了 Nanorep，并在 Wix 担任平台产品经理。Sheffer 是 Nanorep 的前数据科学家，也是 LogMeIn 的首席数据科学家。至于李维，他是 LivePerson 的在线营销副总裁，也是 WeWork 的营销主管。

One AI 提供了一组模型，可以在管道中混合和匹配，通过单个 API 调用来处理文本。Levi 说，每个模型都是根据其对企业的适用性进行选择和训练的，并由平台自动匹配客户的任务和领域(例如，对话摘要、销售见解、话题检测和校对)。One AI 的模型也可以与开源和专有模型相结合，以扩展 One AI 的功能。

该平台的 API 接受各种格式的文本、语音和视频输入。有了 One AI 的 language studio，用户可以试验 API 并生成代码调用。

“随着语言人工智能技术的成熟，机器开始适应我们的时候终于到了，”本通过电子邮件告诉 TechCrunch。"更广泛的开发人员社区采用语言理解工具是实现这一目标的途径."

图片来源:一个 AI

One AI 将其 NLP 服务分为几个等级，包括一个免费等级，每月处理多达 100 万个单词。可堆叠的“增长层”以 1 美元的价格增加了 10 万个单词。

One AI 必须克服的障碍是让客户相信，它的服务比现有的服务更有吸引力。今年 4 月，OpenAI 表示，成千上万的开发者正在通过其 API 使用 GPT 3 为 300 多个应用程序生成单词。但是，随着《财富商业观察》将 2020 年的 NLP 市场锁定在 165.3 亿美元，可以说新来者有足够大的份额。

TehAviv 创始人兼管理合伙人 Yaron Samid: 补充道:“语言是最有价值的未开发资源，是大多数产品和公司无法企及的。这是大多数行业和领域的真实情况，会导致负面结果，包括从销售损失到用户参与度和忠诚度降低，再到声誉受损。 释放语言人工智能使我们能够驾驭这种非结构化数据的力量，并将其转化为有用的信息和见解。”

One AI 表示，种子轮收益的一部分将用于扩大其 22 人团队，其中包括 10 名 NLP 数据科学家。
