NFT 版税怎么了？TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/11/02/whats-going-on-with-nft-royalties/

最近几个月，随着一些平台放弃版税转而寻求其他选择，围绕 NFT 创作者版税的讨论发生了转变。

不是每个人都对此感到高兴。

“大约一年前，每个平台都有版税，”NFT 市场 Rarible 的首席战略官和联合创始人 Alex Salnikov 对 TechCrunch 说。然后半年过去了，一些市场停止了实施，他补充道。

创作者版税最初在整个 NFT 社区推出，作为一种支付艺术家一手和二手作品的方式。一般来说，内容创作者版税是一个项目购买价格的 2.5%到 10%。萨尔尼科夫说，大多数版税平均约为 5%。

专注于艺术品的索拉纳 NFT 市场交易所首席执行官亚历克斯·弗莱舍留说，许多创作者的最初收入来自第一手销售，但随着时间的推移，第二手销售可以通过版税增加收入。阿特告诉 TechCrunch。“它支撑着他们的成功，对他们来说，谋生是非常重要的。”

“让我们从这些方法中选一个，让所有的 NFT 市场都支持它。我们通过争夺市场份额来诅咒市场。” Rarible 联合创始人亚历克斯·萨尔尼科夫

web3 游戏和开发工作室 Find Satoshi Lab 的首席运营官对 TechCrunch 说，版税和奖励创作者是建立长期价值的基础。“创作者和艺术家将与重视他们作品的平台合作，停止对他们的剥削，从而让他们有能力创作出最好的作品。”

周二，Find Satoshi Lab 推出了一个强制收取版税的多链 NFT 市场。Manghani 说:“Web3 的诞生在许多方面都是为了解决创作者面临的挑战，因为中央集权制度不允许公平的奖励。”“(我们)希望保持这种精神。”

分开，交换。ART 于周三推出了“版税保护标准”，该标准对其平台上的二次销售征收创作者版税。这意味着其市场上的新 NFT 收藏可以利用该标准来确保艺术家的作品不会在未经他们同意的情况下在市场上交易。

“我们最近已经看到版税面临很大压力，”Fleseriu 说。“我们已经看到市场和协议基本上允许买家和卖家规避这些版税，这加剧了 NFT 生态系统整体的掠夺性，特别是在(个人资料图片)市场。”
