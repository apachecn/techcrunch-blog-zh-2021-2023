借助镜头云，Snap 为其开发者平台| TechCrunch 提供后端服务

Snap 不仅仅是热门社交应用 Snapchat 背后的公司。它还建立了一个强大的增强现实开发平台，名为Snap AR——它的传播超越了 Snapchat，这要归功于 Camera Kit，这是一个 SDK 解决方案，可以让你将 Snap 的相机功能集成到其他应用程序中。

用 Snap 的行话来说，镜头本质上是你可以在 Snapchat 中访问的增强现实应用。在过去的几年里，该公司已经建立了一个巨大的镜片“应用商店”。有些很有趣，有些很有用，有些帮助你以一种全新的方式交流。它正在成为一个丰富的增强现实生态系统。

“已经有超过 250，000 名创作者制作了超过 250 万个镜头，这些镜头被观看了超过 5 万亿次，”Snap 首席执行官埃文·斯皮格尔在 Snap 合作伙伴峰会主题演讲中说道。

今天，该公司推出了其开发平台的另一个组件，镜头云。顾名思义，它是一个服务器端组件，将帮助开发人员构建动态的多人游戏体验。

镜头云有三个组成部分。首先，开发人员可以利用多用户服务。它允许您为一组朋友创建一个实例，以便他们可以在同一时间在同一个镜头内进行交互。

其次，基于位置的服务让开发者可以从伦敦市中心开始定位。例如，当你把相机对准一个特定的地标时，博物馆可以利用这一点来启用某些镜头。

图片来源:抓拍

最后，还有存储服务。开发者可以将资产存储在 Snap 的服务器上，并按需加载。它也可以作为一种存储卡。用户可以留下一个镜头，稍后从他们停止的地方继续。

“存储服务使开发人员能够扩展到 8 兆字节以上。他们通过在我们的云中存储他们将实时加载到镜头中的资产来实现这一点，”Snap Head 或 AR 平台合作伙伴 Sophia Dominguez 告诉 TechCrunch 的 Sarah Perez。

存储服务目前还不可用，但该公司预计将在未来几个月推出。这一系列后端服务将免费提供给 Snap AR 开发者。

想要开始为 Snapchat 制作镜头的创作者可以从下载 Lens Studio 开始。他们可以导入 2D 和 3D 资产，使用 3D 面部网格，添加自定义着色器，编写脚本，并通过 SnapML 利用 Snap 的机器学习模型。今天，Snap 发布了一个新版本的 Lens Studio，其中也包含了一些新功能。

Lens Studio 已经允许您通过使用 API 动态调整镜头。例如，如果下雨，您可以更改镜头的外观。该公司正在增加新的 API 合作伙伴，以开拓更多的可能性。多亏了占星雅皮和 Sportradar，您可以创建根据占星或体育数据进行调整的内容。

该公司还致力于光线跟踪支持，这应该会大大改善反射和表面渲染的一般情况。事件洞察力也改进了分析。它尤其应该有助于调试。

为迷你和游戏添加社交元素

除了增强现实应用程序之外，Snap 还允许您创建轻量级应用程序，可以在与朋友的对话中启动。这些名为 Minis 的应用程序是用 HTML5 构建的，可以和你的朋友一起使用。

如今，该公司正在推出 Minis Private Components System，这样开发者就可以在不损害隐私和安全的情况下添加评论和评级等社交元素。

当开发人员想要添加私有组件时，Snap 会在编译代码时自动创建一个安全的沙箱环境。这样，社交元素会自动从 Snap 的数据库中填充。

如果您是从事 Minis 工作的第三方开发人员，看到这方面的一些进展也很好。当平台仍在发展时，这总是一个好迹象。
