GetHenry 将名称改为 Cycle

原文：https://web.archive.org/web/https://techcrunch.com/2022/07/20/gethenry-becomes-cycle/

我们不需要臭商标:GetHenry 改名为 Cycle

刚刚在 5 月份筹集了 1740 万美元的资金后，GetHenry 宣布将更名为 Cycle ，这既反映了该公司生产自行车和三轮车的事实，也突出了该公司绿色可持续发展的雄心。

“新的名称和强大的品牌形象符合我们的使命，即消除商业运输领域的碳排放。公司创始人之一 Luis Orsini-Rosenberg 说:“我们专注于电动自行车，而不是内燃机汽车。“作为一个品牌，Cycle 是都市、大胆和酷的，这就是为什么我们认为快递员会认同 Cycle。”

对我来说，名字的改变有点让人头疼。GetHenry 可能不会马上喊出“电动自行车”，但至少它有一个好处，那就是足够低调，可以被打上商标。对于自行车和摩托车来说，Cycle 是一个相当通用的术语，虽然有点不合时宜，但却没有这个好处。事实上，该公司告诉我，它并没有寻求“循环”一词的商标。

当被问及这个问题时，该公司的创始人告诉我，他们正在注册商标的具体形状，而不是文字标志。这使他们面临着另一家同名电动自行车公司的风险，或者更有可能的是，一家山寨制造商加入进来，开始生产与德国品牌毫无关系的自行车产品。作为一个对模仿硬件产品的模仿者有很大问题的人(包括我们的商标！)并最终通过商标法终止了这一行为，我希望该公司已经仔细考虑了其名称和商标策略。但愿它不会回来咬他们的车座。

无论如何，该公司值得关注，因为它继续兜售其最后一英里递送解决方案。
