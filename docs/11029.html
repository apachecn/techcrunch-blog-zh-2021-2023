Meta 与微软合作，将团队、Windows 应用和游戏引入 Quest 设备

Meta 今天宣布与微软合作，为 Meta 的元宇宙硬件项目带来新的内容，包括 Windows 应用程序和团队合作。在今天上午的 Meta Connect 会议上，微软首席执行官塞特亚·纳德拉表示，微软团队将与 Quest devices 集成，微软将提供一种将 Windows 应用程序传输到 Meta 耳机的方式。纳德拉还透露，微软的流媒体游戏服务 Xbox Cloud Gaming 将在未来几个月内的某个时候登陆 Quest 设备。

图片来源:元

纳德拉说:“我们正在将微软团队的沉浸式会议体验引入 Meta Quest，以便为人们提供新的相互联系方式。”他指出，定制头像最终将带来这种体验。他补充说，Meta 的虚拟现实协作空间 Horizon Workrooms 将与团队连接，允许人们直接从工作室参加团队会议。

纳德拉补充说:“现在，你可以联系、分享和协作，就像你亲自在一起一样。”

在 Windows 端，纳德拉表示，微软 365 将通过一种让用户与 Word、Excel、PowerPoint 和 Outlook 等生产力应用程序的内容进行互动的方式来实现 Quest。The Verge 的汤姆·沃伦指出这些并不是为虚拟现实设计的成熟版本，重要的是；确切地说，它们是的进步网络应用。

纳德拉说:“你(将)有一种新的方式来安全地传输整个 Windows 体验，包括所有个性化的应用程序内容设置到你的 VR 设备上，充分发挥(Windows 的)力量。”。“我们一直在思考如何将 Microsoft 365 和 Windows 365 的强大功能引入 3D 空间，真正帮助提高工作效率，让您能够以全新的方式进行创作、交流和协作。”

图片来源: Meta

至于 Xbox 云游戏，在 Quest 上，它将把游戏传输到 2D 的虚拟现实屏幕上，支持现有的 Xbox 控制器。但纳德拉暗示，未来可能会推出更多功能。

微软与 Meta 的合作正值前者回调其内部 VR 和 AR 硬件项目，包括 HoloLens。Windows 混合现实平台是微软在 Windows 10 中内置的支持 VR 头戴设备的软件基础，从未像公司希望的那样取得成功。今年年初，商业内幕报道微软放弃了第三代 HoloLens 的计划，转而与三星合作开发一款新的“混合现实”设备。

微软对 HoloLens 的断言进行了反击。但后来，领导 HoloLens 开发的高管之一，亚历克斯·基普曼在被指控不当行为(包括不适当的性行为)后辞职，使该部门处于动荡之中。

也就是说，微软对投资为元宇宙开发软件表现出了浓厚的兴趣——不管它会采取什么形式。在去年的 Ignite 大会上，该公司宣布了 Mesh for Teams，它将该公司的 Mesh 平台与团队及其内置的生产力工具相结合，为虚拟现实、增强现实和其他领域的共享体验提供动力。
