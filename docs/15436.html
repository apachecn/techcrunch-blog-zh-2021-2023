参议员马基呼吁埃隆·马斯克恢复推特的无障碍团队

经过几轮裁员后，Twitter 的员工从大约 7500 人减少到不到 T2 的 2000 人——去年，全公司众多裁员中的一次淘汰了该平台的整个 T4 无障碍团队。

在给埃隆·马斯克的一封公开信中，参议员埃德·马基呼吁新的推特所有者重新组建无障碍团队。

Markey 写道“毫不奇怪，自从你关闭 Twitter 的无障碍团队后，残疾用户报告使用 Twitter 的难度和挫败感增加了。

像任何社交平台一样，Twitter 在可访问性方面也有自己的缺点——2020 年，Twitter 甚至没有可访问性团队，只有在公众抗议该公司推出不带字幕的语音推文后建立了一个。但在 Twitter 拥有无障碍团队的几年里，该公司推出了图像上的alt 文本功能，视频上的自动字幕以及空间直播音频室和语音推文的字幕。许多残疾用户在 Twitter 上找到了社区，因为其内置的辅助功能使其比其他社交平台更容易使用。

由于 Twitter 不再有可访问性团队，这个平台的功能并没有简单地保持休眠状态。Twitter 空间上的标题已经完全消失，使得任何失聪或听力不好的用户都无法使用该功能。对于残疾用户来说，这传达了一个信息，即无障碍不再是 Twitter 总部讨论的一部分(顺便说一下，该公司已经停止支付租金)。

当平台切断对其 API 的访问时，Twitter 的可访问性受到了又一次打击。现在，第三方开发者将不得不支付尚未确定的月费来建立一个以前免费欢迎他们的平台。这意味着像 Ironfactory 的twit terrious这样给用户提供扩展的辅助功能的长期受欢迎的应用程序将不再可用。

“我收到了不少盲人用户的电子邮件，他们感到不安和愤怒，因为他们很可能不得不停止使用 Twitter，而没有像 Twitterrific 这样的第三方客户端，”Ironfactory 联合创始人 Gedeon Maheux 告诉福布斯。

马基的信向马斯克提出了一些问题，马基要求马斯克在 3 月 17 日之前做出回应。Markey 询问了 Musk 为什么取消了可访问性团队，他是否会恢复该团队，Twitter 是否符合 ADA 和 FCC 的可访问性法规，Twitter 为什么从 Spaces 中删除了字幕，以及该平台是否会致力于为所有类型的内容创建用户友好的体验。

马基在给马斯克的信中写道:“在你的领导下，所有这些变化都表明了对残疾人需求的忽视。”。
