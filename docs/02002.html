火狐制造商 Mozilla 在反对声中暂停了加密捐赠

原文：https://web.archive.org/web/https://techcrunch.com/2022/01/06/mozilla-foundation-crypto/

在遭到包括 Mozilla 项目创始人在内的数十人的强烈反对后，Mozilla 基金会暂停了接受加密货币捐款的能力。

该基金会负责监管火狐浏览器的开发，周四承认围绕加密货币可能造成的环境影响进行了讨论，并表示正在评估其当前的加密捐款政策“是否符合我们的气候目标”

“去中心化的网络技术仍然是我们探索的一个重要领域，但自从我们开始接受加密捐赠以来，已经发生了很多变化，”该基金会在一条推文中表示，并承诺将随时更新其审查。

该基金会在邀请人们通过使用包括比特币在内的各种加密令牌向美国非营利公司捐款后，开始面临反弹。

杰米·扎温斯基(Mozilla 的创始人)在推特上回应道，他对基金会的举动表示失望。“参与这个项目的每个人都应该为这个与毁灭地球的庞氏骗局合作的决定感到无比羞耻，”他说，还加上了一句脏话。

浏览器引擎 Gecko 的创始人彼得·林斯加入了对话，告诉 Mozilla 它“应该做得更好”

这并不是第一次有大公司因为环境问题而遭到反对，或者远离比特币。去年 5 月，特斯拉暂停接受比特币支付其车辆，仅在推出选项后个月。

马斯克表示，该公司“担心比特币采矿和交易对化石燃料的使用迅速增加，特别是煤炭。”几周后，他表示，一旦加密货币的开采率达到 50%的可再生能源，该公司将“很有可能”再次接受比特币。

围绕比特币环境足迹的问题与各种代币的创造方式有关。比特币和以太坊使用一种称为工作证明的机制来为其网络提供动力，并铸造每种货币的新块。这种计算，从设计上来说，随着网络的发展，已经变得越来越复杂，并催生了一个公司行业，这些公司昼夜不停地运行数千个 GPU 来参与解决这个难题。

剑桥替代金融中心估计，比特币采矿每年消耗约 148 太瓦时的能源。然而，许多加密货币的支持者要么对这一发现提出质疑，要么提出理由说明为什么加密货币是值得的。

最近几个月，该行业开始出现更大的分歧。批评家们对 web3 的一些基本价值主张提出了质疑。支持比特币的 Twitter 创始人杰克·多西,驳斥了风投没有从 web3 项目中获取大部分利润的说法。

在持续的讨论中，许多公司已经缩减了他们的 web3 野心。在许多人反对这个想法后，不和谐在 11 月份推迟了探索 crypto 和 NFTs 的计划。游戏公司 GSC Game World 取消了将 NFTs 纳入其即将推出的游戏《潜行者 2:切尔诺贝利之心》的计划，此前一些游戏玩家提出了强烈的反馈。
