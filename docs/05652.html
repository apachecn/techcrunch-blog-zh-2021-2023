如果您共享您的帐户，网飞将收取更多费用

原文：https://web.archive.org/web/https://techcrunch.com/2022/04/19/netflix-to-charge-global-members-sharing-their-accounts-an-additional-fee-following-tests/amp/

图片来源:切斯纳特/盖蒂图片

鉴于网飞关于用户损失的，该公司今天在第一季度收益电话会议上宣布，它将扩大测试，如果会员与家庭以外的人分享账户，将收取更高的费用。该公司今年 3 月首先开始在智利、哥斯达黎加和秘鲁测试该功能，但现在该公司表示，计划用一年左右的时间在包括美国在内的全球市场实施该解决方案。

该流媒体工具澄清说，它需要在大约一年左右的时间里继续迭代这一功能，以确保在向与自己家庭以外的其他用户分享网飞账户的用户收取多少额外费用方面取得平衡。

网飞首席产品官 Greg Peters 在财报电话会议上解释说:“坦率地说，我们已经在这方面工作了近两年时间……一年多以前，我们开始进行一些小规模的测试，这些测试……为我们的想法提供了信息，并帮助我们建立了我们现在正在部署的机制。“我们刚刚进行了第一次大国测试，但需要一段时间来解决这个问题，并获得正确的平衡。”

目前，网飞在几个测试市场中的标准用户和高级用户可以选择为不住在一起的人添加“子账户”。每个子账户都有自己的个人资料和个性化推荐，但他们也有自己的网飞登录名和密码。这为他们将来成为拥有自己账户的固定会员做好了准备。如果他们选择这样做，他们的观看历史、观看列表(“我的列表”)和个性化推荐将转移到他们自己的帐户，并带有他们自己的账单信息。(由于共享他们账户的会员现在必须支付更多费用，当新的费用开始生效时，他们可能会选择将免费者从他们的账户中删除。)

网飞早些时候曾表示，这种解决方案不依赖于基于位置的数据，如 GPS。取而代之的是，它正在利用它今天用来向其最终用户提供服务的相同信息，包括 IP 地址、设备 id 和家庭中登录到网飞帐户的设备的其他信息。通过这种方法，网飞可以识别出家庭以外的持续分享。

它指出，当这些子账户仍在与另一个家庭共享账户时，它们不能算作用户。

今天，网飞估计全球大约有 1 亿个家庭共享他们的用户账户，其中仅在美国和加拿大就有超过 3000 万个。

网飞表示，通过要求分享账户的会员支付更多费用，它希望在允许分享发生的同时，帮助每个观看并从其服务中获得价值的人带来收入，从而达到适当的平衡。当然,“价值”转化成的实际金额还有待确定——可能因市场而异。

在其测试市场中，智利非家庭成员的额外费用为 2，380 CLP，哥斯达黎加为 2.99 美元，秘鲁为 7.9 PEN。这比完整的网飞帐户计划便宜，但也比以前免费分享某人的网飞帐户要贵。

这并不是网飞计划利用其用户群赚钱的唯一方式。该公司还表示将推出一项广告支持计划。
