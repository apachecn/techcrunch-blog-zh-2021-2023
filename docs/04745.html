Stripe 再次与 crypto 友好相处

支付巨头 Stripe 已经为网络金融基础设施的相当大一部分提供了支持，现在它正在推出加密支付支持，为客户提供更简单的途径来加入 web3 用户并与加密货币进行交互。

该公司今天宣布了一套产品，旨在让客户获得工具和 API，使客户更容易购买和存储加密令牌、套现、交易 NFT 和处理合规工作流，如了解你的客户(KYC)。Stripe 的支持页面指出，该公司的产品将支持用户在 180 个国家用法定货币购买超过 135 种加密货币。

Stripe 联合创始人约翰·科利森周四在 Twitter 上宣布了新功能。

多年来，Stripe 与 crypto 有着复杂的关系，在 2018 年结束了对比特币的支持，称加密货币在 2014 年最初支持加密货币后，作为一种支付手段“不太有用”。10 月份，该公司开始发布招聘信息，以组建一个加密团队，几周后，又将加密风险投资公司的马特黄招入董事会。该公司重返这一领域是对现有加密支付处理器的一大威胁，这些处理器缺乏该公司的广泛影响力。

与此同时，FTX 加密交易所宣布，它正在与 Stripe 合作，以改善其身份合规性功能和用户的 fiat onboarding 工作流。
