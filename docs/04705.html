为什么 Studio Display 不是苹果电视？TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/03/09/why-isnt-the-studio-display-also-an-apple-tv/

最终，苹果推出了一款不少苹果发烧友梦寐以求的产品:价格不低于 5000 美元的第一方显示屏。新的 27 英寸 Studio Display 的价格为 1600 美元，虽然远高于很多人想要的价格，但对很多人来说仍然是个不错的选择。苹果甚至在内部放了一个超级大的 A13 芯片，这比目前为 4K 苹果电视供电的芯片更有 T2 功能。

这就引出了一个问题:为什么 T4 的 Studio Display 不是苹果电视？

它已经有了芯片，虽然可能没有那么大的板载存储空间，但相对于运行 tvOS 和一些媒体应用程序所需要的内存数量，增加存储空间并不是什么难事。其他公司也有这样做的例子:三星 M 系列智能显示器(Samsung M-series Smart Monitor)有一个智能电视模式，内置糟糕的土布软件。

我是在 TechCrunch Slack 上提出这个想法的，后来我对它的不合理性做了一些温和的解释——比如你把 Studio Display 和一个 Mac 挂钩，这个 Mac 表面上可以复制苹果电视的所有功能，而且除此之外还能做很多事情。

但实际上，苹果 macOS 上的原生电视应用程序已经坏了。并获得网飞，迪士尼+，亚马逊 Prime 视频等。需要使用浏览器分别访问各自的网站，这比仅仅在简化的主屏幕上安装应用程序要笨重得多，也不那么优雅。再者，三星推出混合型智能电视显示器是有原因的，而且随着时间的推移，它甚至会在产品线中增加更多型号。

此外，我敢打赌，Studio Display 本身就不仅仅是一台电视，它的购买吸引力一定很大。这种工业设计恰如其分，它打败了大多数现代电视设计，即使是在客厅里看起来更漂亮而价格较高的电视类别(例如三星的 Frame 和 Serif 系列)中也是如此。

可以认为这是一个不切实际的目标，但 Studio Display 内置网络摄像头、扬声器和麦克风，堪称苹果公司的杀手锏。它也可以是一款令人称奇的无头 Zoom(或类似机型)，让视频会议体验更贴近人体。

不过，我认为这还远远不够，还需要对 tvOS 以及它支持的应用和服务进行一些重新架构。苹果本来可以在这款显示器上增加 Apple TV 的功能，而基本上不需要额外的成本。不过，A13 芯片的加入实在是太神奇了，消费者既可以利用它，也可以把它当成普通的显示器使用——这一切都有好处。

在理想情况下，未来的固件更新将包括 tvOS，并在购买后实现这一点。就目前的情况来看，我认为演播室的展示没有任何(足够吗？)来合理地运行 Apple TV 的操作系统和应用程序，这在我看来只是错过了一点机会。
