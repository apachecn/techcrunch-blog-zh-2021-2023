Sound.xyz 从 a16z 融资 500 万美元，探索 web3 如何颠覆音乐行业的经济

今年，我们已经看到大量证据表明，NFTs 可以动摇销售视觉艺术的经济学，但当涉及到 web3 颠覆艺术世界中不同垂直领域的经济学时，就没有那么活跃了，包括音乐。

Sound.xyz 是一家新的初创公司，希望帮助唱片艺术家通过 NFTs 将他们的社区货币化，因为这家初创公司希望建立一套工具，旨在将音乐人纳入所谓的“web3”阵营。这家初创公司在本月早些时候发布了第一款产品 Listening Parties，它使艺术家能够销售与新歌发布相关的 NFT。

这家初创公司告诉 TechCrunch，它已经在 Andreessen Horowitz 领导的种子融资中筹集了 500 万美元。本轮其他投资者包括 Variant Fund、Weekend Fund、Scalar Capital、Flamingo、Packy McCormick 和 21 Savage 等。

这家初创公司正寻求推动新兴艺术家通过利用 NFT 更有效地将他们的工作货币化，并帮助他们在 Spotify 等平台上超越每流赚取几分之一便士的水平。

“我们正在建立一套经济工具来帮助艺术家以新的方式将他们的歌曲货币化，”Sound 首席执行官大卫·格林斯坦告诉 TechCrunch。“我们想要解决的问题是:我们能让音乐被听到吗？我们能让艺术家得到报酬吗？”

这家初创公司已经与许多“加密友好”的录音艺术家合作，这些艺术家一直在与社区合作。早期的下降仅限于几十个限量版的 NFT，立即销售一空。虽然在一个有利于百万美元销售额的行业中，销售价值 1 万美元的 NFTs 听起来可能不太值得注意，但格林斯坦表示，这些收入对那些可能需要数十万个流才能在 Spotify 上赚到相同金额的艺术家来说意义重大。随着这家初创公司评估与某些艺术家的歌曲下载相关的 NFT 下载的需求，他们有雄心向社区发布更多的 NFT。值得注意的是，这家初创公司不从销售额和非功能性交易中抽取佣金。

该团队一直在尝试 drops 的其他功能，比如让 NFT 持有者能够在流媒体上留下评论，供艺术家阅读。未来，这家初创公司希望创造更广泛的工具，让音乐人从粉丝那里众筹资金，以创作新作品和处理新项目，同时为策展人创造新的激励模式，以发现和展示新音乐。

声音不是 NFT 音乐空间的唯一播放器。上个月，a16z Crypto 支持了 NFT 音乐平台 Royal，该平台希望将版税令牌化并出售给粉丝。区块链的音乐流媒体平台 Audius 已经从包括通用催化剂和比特币基地风险投资公司在内的公司筹集了超过 1350 万美元的资金。
