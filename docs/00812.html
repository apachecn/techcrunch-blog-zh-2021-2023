Jiko 宣布合作推出针对跨性别群体的银行应用程序| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/11/17/jiko-announces-partnership-to-a-launch-banking-app-for-the-transgender-community/

使用客户存款购买短期国库券的金融科技初创公司 Jiko 今天宣布了其首个 银行即服务(BaaS) 客户端。它将与 Euphoria 合作，推出一款名为 Bliss 的银行应用，这是一款针对跨性别群体的技术套件 ，旨在帮助缓解与性别转换相关的困难。

该公司在新闻稿中写道，美国有 200 万至 600 万人认为自己是跨性别者，34%的人缺乏与其性别相匹配的身份证明，这使得他们更难获得银行产品。大多数银行的合规流程不适应这种情况下的个人，但 Jiko 与 Euphoria 合作，为 Bliss 定制了“了解您的客户”流程，允许用户使用替代形式的验证开立账户。

“官方身份对变性人来说是一个巨大的挑战，”Euphoria 创始人兼首席执行官罗比·凯瑟琳·安东尼在一封电子邮件中告诉 TechCrunch。

更新你的法律文件需要时间、资源，在某些情况下，还需要正确的法律。安东尼写道:“如果你正在更新你的文件，这就为银行审查不匹配创造了机会，并可能导致某人无法开立账户。”

安东尼指出，Euphoria 是通过其投资者之一与 Jiko 联系在一起的，他是 Jiko 联合创始人洛奇·莫特瓦尼的朋友。从最初的谈话中，一种伙伴关系出现了。

Euphoria 估计，与传统银行合作为其社区构建一个平台将需要 12 到 18 个月的实施时间，而根据新闻稿，Jiko 能够在短短几个月内帮助他们推出 Bliss。

这一举动与 Jiko 的 意图的支点 一致，即从消费者服务转向 B2B 产品。它向合作伙伴收取每个账户的固定月费。每个账户持有政府支持的国库券，允许其账户持有人通过借记交易实时花费这些资金。

Jiko 的构想背后的理念是提供一个最安全的可能风险水平，以最佳的利率提供充分的支出能力。历史上提供这种风险回报水平的工具是美国国债，“ Jiko 首席执行官兼联合创始人夏羽·林特纳在一封电子邮件中告诉 TechCrunch。

虽然 Euphoria 是 Jiko 的第一个 BaaS 客户，但 Lintner 说 Jiko 拥有强大的渠道。根据林特纳的说法，该公司正在与拉丁美洲社区的汇款平台进行谈判，并看到了希望储存现金的企业和加密公司的需求激增。

" 我们核心产品的伟大之处在于它对每个人都是一样的。 新银行、社区、生活类 app 主要需要发卡和零售账户。但一个拥有财富 500 强公司的非常大的企业资金平台也会使用相同的 API 为其企业客户开户，并帮助他们存储日常流动性，”林特纳写道。
