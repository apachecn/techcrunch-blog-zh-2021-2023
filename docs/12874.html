由 Polychain Capital TechCrunch 领投的 Web3 开发平台 Fleek 融资 2500 万美元

Web3 开发者平台 Fleek 在由 Polychain Capital 牵头的首轮融资中筹集了 2500 万美元，该公司与 TechCrunch 独家分享。

本轮投资者还包括比特币基地风险投资公司、数字货币集团、Protocol Labs、Arweave、北岛风险投资公司、Distributed Global、LAO 和 Argonautic Ventures。

据其网站称，这家初创公司的目标是建立一个接口和协议层，“让任何人都可以访问存储、托管和计费等网络服务的基础层”。

“我们最初的主要关注点是内容交付市场，”Fleek 联合创始人哈里森·海因斯(Harrison Hines)对 TechCrunch 表示。“这就是 Fleek 目前所提供的服务，也是我们在 web3 基础架构堆栈中发现的一个巨大缺失。这是所有 web3 协议的问题。”

海因斯说，内容交付网络(CDN)由几个大玩家主导，仅举几个例子，如亚马逊网络服务( AWS 和 Cloudflare )。Hines 补充说，虽然 Fleek 最初与 AWS 和 Cloudflare 等 Web 2.0 基础设施提供商合作，但它计划在 2023 年推出自己的 Fleek 网络，并提供去中心化等 web3 技术，同时仍然实现类似 Web 2.0 的性能。

“我们对 Fleek 网络的愿景核心是一个分散的边缘网络，任何人都可以运行节点并为网络提供资源，”Hines 说。“互联网现在正走向边缘。大多数最大的平台都与边缘相关。”

Hines 将边缘定义为将内容从一个中央服务器位置转移到离最终用户更近的不同基础设施和云服务的松散耦合。

Hines 指出，Fleek 今天在其平台上托管了大约 50，000 个应用程序，主要是在以太坊生态系统内，但也有其他协议。迄今为止，所有 Fleek 产品都建立在加密协议上，如以太坊、Filecoin、互联网计算机、星际文件系统(IPFS)和纺织品。

Hines 说，新的资金将用于建立 Fleek 网络和平台，同时引进额外的人才并发展其社区。

海因斯说，这家初创公司将首先专注于建立 web3 生态系统，但随后将扩展到 Web 2.0 公司，如视频游戏平台、流媒体服务或任何有大量流量的平台，这通常是他们最大的成本之一。“在这个市场，大公司都在寻求削减成本，我们确实认为 Fleek Network 可能是一个有吸引力的解决方案，也更容易实现。”

Hines 说，Fleek 的定价是“浮动的，因此指标可以随着增长而调整”，但与每千兆带宽收费约 5-15 美分的 Cloudflare 相比，Fleek 的目标是保持在每千兆带宽不到 1 美分，使其价格便宜 5-15 倍。

海因斯说:“我们多年来一直在努力做到这一点，在(加密生态系统的)可扩展性以及如何实际构建这些网络方面有了突破，这给了我们信心，我们可以做到这一点，并与现有的 web2 系统的规模、吞吐量和延迟相当。”。"这是一个完美的时机。"
