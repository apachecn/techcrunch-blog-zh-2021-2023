Polygon 创始人为 web3 风险基金筹集了 5000 万美元

原文：https://web.archive.org/web/https://techcrunch.com/2022/08/25/polygon-web3-crypto-nailwal-symbolic-venture-capital-emerging-markets/

以太坊二层扩展平台 Polygon 在今年早些时候的第一轮重大融资中筹集了 4.5 亿美元。现在，该协议的联合创始人 Sandeep Nailwal 正在启动另一个项目，他独家告诉 TechCrunch 这次是以象征资本的形式，这是一个由 web3 创始人建立并为其服务的风险投资基金。

Nailwal 和 Cere 的联合创始人 Kenzi Wang 已经从包括其他风险公司、密码交易所、家族办公室和机构在内的投资者那里筹集了 5000 万美元，尽管他们没有透露具体的名称。Nailwal 表示，象征性的计划主要是支持公司构建面向消费者的去中心化应用程序(dapp)，这一举措似乎与 Polygon 加快 web3 应用程序开发的目标一致。

“我的核心任务是为 web3 带来大规模采用，而这种大规模采用只能通过应用程序来实现。这并不是说我要建造一个区块链，人们就会来使用区块链——没有人会直接使用区块链。他们总是通过一些应用程序来使用它，”Nailwal 告诉 TechCrunch。

根据 Nailwal 的说法，该基金已经进行了 15 笔投资，包括对 web3 游戏工作室 BlinkMoon、基于 Polygon 的元宇宙星球 Mojo 和电子竞技平台 Community Gaming 的投资。

作为一名印度创始人，Nailwal 经常寻求支持他的祖国——去年，当 Vitalik Buterin 向他安排的印度新冠肺炎救济基金捐赠了价值 10 亿美元的 crypto 时，他成为了头条新闻。Nailwal 说，通过这个新的风险基金，他的目标是将 80%到 90%的资金分配给像他一样来自新兴市场的创始人。

“它不会对任何在硅谷从事建筑的人关闭，当然不会。但是作为一个印度人，从技术上来说，我会是一个天使投资人，或者是来自印度次大陆的 95%的好项目的顾问——他们会以某种方式来找我，寻求帮助或指导，”Nailwal 说。

Nailwal 补充说，王同时对中国和东南亚的 web3 生态系统非常熟悉，这使他们对世界各国有了广泛的了解。他认为，新兴市场的创业者通常更务实，因为与资源更丰富的硅谷创业者相比，他们面临着种种限制。

Nailwal 说:“因为他们都被逼到了生存的绝境，他们需要建立一些能够产生收入的东西，他们可以靠这些东西生存。”他补充说，他在 growing Polygon 中采取了类似的方法，他说他将一次测试一项新技术，等到它获得一定数量的用户后，再尝试下一个想法。

这种背景是 Nailwal 认为 Symbolic 特别适合帮助新兴市场创始人的部分原因。

“他们中的许多人有能力建立最终获得用户的业务，但你需要帮助他们不断重塑自我，提高他们的标准，这样一旦他们在小规模上有了一个成熟的模式，他们就可以在更大的规模上保持增长，并重塑他们的想法和商业模式，”内尔瓦尔说。

他说，Nailwal 推出 Symbolic 背后的一个主要动机是有机会更正式地支持已经在他的轨道上的公司，并补充说，他认为他不必花太多额外的时间来管理基金，因为这将帮助他实现他已经建立的关系的协同效应，部分是通过 Polygon 自己的风险基金。他和王已经是创业公司的多产天使投资人-自 2019 年在币安实验室相遇以来，他们已经共同投资了 40 多家公司，他说。

Nailwal 表示，在 Symbolic 对应用的关注中，他对属于“创造者经济”子行业的初创公司特别感兴趣，如幻想体育公司。

“在 web3 fantasy 中，你可以拥有 NFT，你可以在不玩的时候把那些 NFT 出租，这样你就可以获得一些被动收入，再加上你可以赚取平台上的代币，从很早的阶段就成为平台的一部分，”Nailwal 说。

他补充说，这类激励措施通常对新兴市场的用户更有意义。他举出了中国“移动挣钱”应用 StepN 最近用户增长和“玩挣挣”视频游戏 Axie Infinity 作为发展中国家低收入工人补充收入来源的例子来说明他的观点。(Nailwal 和王恰好都是 Axie Infinity 母公司 Sky Mavis 的天使投资人)。

“在生活成本低得多的情况下，许多加密模型能够实现更好的可持续经济，”Nailwal 解释说。他补充说，印度的 Jio 4G 网络可以说是世界上最快的，甚至可以深入农村地区——如果提供正确的经济激励，这种基础设施可以帮助 web3 初创公司在全国范围内吸引用户。

一些 web3 play-to-earn 初创公司因在出现任何问题时将已经脆弱的人置于风险之中而受到批评，例如 3 月份 Axie Infinity 的 Ronin bridge 发生的价值 6 . 25 亿美元的黑客攻击。Nailwal 承认这种风险是“一个问题”，他说，他建议 web3 的企业家在他们的平台上发生交易时，向用户提供尽可能多的教育和披露。

Nailwal 表示，Symbolic 的一个关键区别在于其内部数据平台，该平台是该公司为支持其投资组合公司以及支持其自身的尽职调查过程而建立的。据该公司称，该平台将汇总和分析数据，如 GitHub 贡献、不和谐参与和象征性表现，以及就业和招聘趋势。

最终，Nailwal 希望围绕这个平台创建一个“专有的社交生态系统”，他将这个平台比作 Y Combinator 的网络。

Nailwal 说:“我们希望为投资组合公司和创始人做这件事，我们相信他们最终将成为许多新项目的天使投资者，这些项目将通过象征性项目实现。”。
