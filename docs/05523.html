中国银行业协会提议限制非金融交易

原文：https://web.archive.org/web/https://techcrunch.com/2022/04/13/chinas-banking-association-proposes-restrictions-on-nfts/amp/

图片来源: 截图:哔哩哔哩的数码收藏品项目

中国互联网社区等待数月的判决终于出来了:中国银行业、证券业和互联网金融协会周三在 T4 发布公告称，用于证明物品所有权和真实性的代币不得用于证券化或加密货币交易。

中国已经禁止首次发行硬币、加密货币交易和加密采矿。排除 NFT 的财政可能性可能会使该国进一步远离世界其他地方正在发生的 web3 浪潮，这一浪潮正在建立一个基于加密令牌的分散式互联网。

尽管厌恶加密的自由本质，中国还是将区块链视为建设互联网经济的关键基础设施。中国证券监督管理委员会的一名官员最近称赞 web3 是互联网的未来，称它可以解决 Web 2.0 时代的问题，如缺乏隐私保护。就像区块链推动的运动的其他方面一样，中国已经定义了自己版本的附带条件的非关税壁垒。

根据在中国金融监管机构监督下运作的三个协会的说法，非金融交易的问题是它们所带来的金融风险。这些团体警告说，非功能性交易涉及投机、洗钱和其他非法金融活动，但他们也认识到非功能性交易在推动中国数字和创意经济方面可以发挥的作用，例如，让艺术家能够控制自己的艺术品。

为了阻止 NFTs 的财务风险并利用基础技术，协会为行业发布了一套指导方针:

  • 非金融交易的基础资产不应包括债券、保险、证券、贵金属或其他金融资产。
  • NFT 的不可兑现性不应被削弱，以间接促进首次发行硬币。
  • 平台不应为 NFT 提供集中交易。
  • NFT 不应以加密货币进行交易。
  • 平台应该对客户进行真实身份检查并存储交易记录，以根除洗钱行为。
  • 实体不应直接或间接向 NFT 投资提供融资支持。

科技巨头已经在 NFT 的努力中与中国监管机构合作。哔哩哔哩、腾讯和阿里巴巴旗下的蚂蚁集团已经创建了许可区块链，创作者可以在这里创作和销售他们的作品，这些作品仅限于指定的参与者，并与开放的以太坊网络分开。交易只能用人民币进行，市场不允许像 OpenSea 和其他全球交易所那样转售人民币。中国的科技公司也将 NFTs 打造成“数字收藏品”，以此与加密世界拉开距离。
