如何将 CarPlay 黑进特斯拉或 F-150

原文：https://web.archive.org/web/https://techcrunch.com/2022/05/09/heres-how-to-hack-carplay-into-a-tesla-or-f-150/amp/

图片来源:罗伯特·罗森菲尔德/马修·伯恩斯

想让你的旧车感觉更新吗？添加新的立体声系统。20 年前，这将涉及到撕掉股票收音机和滑动在一个新的先锋头单元，也许一对超重低音扬声器。这一次，这意味着在现有的操作系统上增加一个新的用户界面。

上周，我在我的 2016 款福特 F-150 上添加了 CarPlay。这个黑客需要一个硬件和大约三分钟。我所要做的就是把 2016 年的 USB 端口换成 2017 年的 USB 端口。由于 2017 年的更新，CarPlay 立即可以在 Sync 3 系统中使用。有特斯拉吗？这需要更多的工作。

从本质上来说，这种黑客行为包括欺骗特斯拉，让它认为自己运行的是安卓平板电脑，让它可以运行苹果 CarPlay 或安卓 Auto。这样做需要两个树莓和一些软件。详细说明可在特斯拉 Android 项目网站上获得。

CarPlay 和 Android Auto 说的是安静的部分:车载信息娱乐系统仍然很糟糕。有些比其他的好，但大多数仍然缺乏用户交互。CarPlay 和 Android Auto 更好，但仍不完美。然而，在大多数情况下，他们在让司机安全访问他们设备上的相关信息方面做得很好。

支持 CarPlay 的售后设备屈指可数。最佳途径通常是 OEM 升级，在标准车载系统中增加功能。有时候，就像我的 F-150 一样，这是一个微不足道的升级，但其他时候，这很昂贵，需要经销商更换昂贵的控制系统。不同的解决方案来自第三方，如索尼或先锋，需要用户用新的收音机替换旧的。如果这不是一个选项，少数仪表板安装的屏幕提供 CarPlay 作为补充服务——这些很可爱，但在我看来提供了不合标准的体验。

特斯拉一直是反对 CarPlay 的主要力量。在某种程度上，这是合乎逻辑的，因为特斯拉汽车使用一个屏幕来控制车内的一切，并且可能会犹豫是否允许在唯一的屏幕上使用另一个用户界面。但是其他制造商已经成功了。沃尔沃和特斯拉一样，用大屏幕代替了收音机和气候控制的专用按钮。司机仍然可以使用 CarPlay 并访问车辆控制系统。在 RAM 卡车上也是一样——有一个巨大的安装在中央的屏幕可以控制卡车上的一切，但 CarPlay 仍然可以工作。直到(或者如果)特斯拉官方支持 CarPlay，这里总是有黑客链接。
