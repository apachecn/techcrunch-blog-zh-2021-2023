Cloudflare 将收购 Area 1 Security，这是一家阻止网络钓鱼电子邮件的初创公司| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/02/24/cloudflare-to-acquire-area-1-security-a-startup-that-blocks-phishing-emails/amp/

图片来源: Daria Nepriakhina / Unsplash

Cloudflare 已经宣布计划收购 Area 1 Security ，这是一家安全初创公司，开发了一种产品，可以在网络钓鱼攻击进入员工的收件箱之前阻止它们。Cloudflare 计划斥资约 1.62 亿美元进行收购。该公司将同时使用现金和股票为收购提供资金。

Cloudflare 开发了自己的安全产品套件，采用零信任安全模式。这些安全解决方案可防止数据丢失、恶意软件或网络钓鱼攻击，即使员工不在办公室或没有使用公司 VPN。

通过今天的收购，该公司在其零信任产品中添加了一个成熟的电子邮件安全产品。Area 1 Security 表示，仅在 2021 年，它就为其客户阻止了超过 4000 万次网络钓鱼活动。

“电子邮件是互联网上最大的网络攻击载体，这使得集成的电子邮件安全对于任何真正的零信任网络都至关重要。这就是为什么今天我们欢迎 Area 1 Security，以帮助 Cloudflare 的平台成为零信任的明确领导者，”Cloudflare 联合创始人兼首席执行官 Matthew Prince 在一份声明中说。“对我们来说，零信任的未来包括一种集成的一键式方法来保护组织的所有应用程序，包括其最普遍的云应用程序电子邮件。我们希望共同努力，提供市场上最快、最有效、最可靠的电子邮件安全性。”

这不是 Cloudflare 的第一款电子邮件产品。该公司去年推出了第一款电子邮件产品。例如，您可以将域名与您的 Cloudflare 帐户相关联，并从 Cloudflare 的界面创建自定义电子邮件地址。收到的电子邮件可以被重定向到任何其他电子邮件收件箱。

Cloudflare 还指导您改进 DNS 记录，使您的电子邮件更加安全。例如，该公司可以帮助您防止欺骗攻击。

区域 1 安全性填补了 Cloudflare 电子邮件产品中的一个重要空白。1 区安全部门不断尝试发现和识别新的网络钓鱼攻击。当员工收到新的电子邮件时，1 区安全部门会扫描入站电子邮件中的网络钓鱼企图。

可疑邮件在进入收件箱之前会被自动拦截。它们不仅会出现在“垃圾邮件”文件夹中，还会被完全阻止。

在后台，客户可以通过在 DNS 级别更改 MX 记录来开始使用区域 1 安全性。之后，它是一个云优先的产品，意味着客户不必安装和修补软件。如果你的公司正在使用谷歌工作空间或 Office 365，1 区安全与这些电子邮件提供商合作。

鉴于 Cloudflare 在 DNS 服务器方面的深厚专业知识，此次收购是有意义的。Cloudflare 已经对其员工的收件箱使用了 Area 1 安全性。换句话说，他们已经知道他们将获得什么。
