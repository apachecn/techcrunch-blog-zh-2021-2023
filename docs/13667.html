Quora 推出 Poe，这是一种与 ChatGPT | TechCrunch 等人工智能聊天机器人对话的方式

原文：https://web.archive.org/web/https://techcrunch.com/2022/12/21/quora-launches-poe-a-way-to-talk-to-ai-chatbots-like-chatgpt/amp/

图片来源: Quora

Quora 本周推出了一个名为 Poe 的平台，让人们可以提出问题，获得即时答案，并与人工智能聊天机器人进行互动对话，这表明了它对文本生成人工智能系统的兴趣。

Quora 发言人通过短信告诉 TechCrunch，Poe 是“开放探索平台”的简称，只接受邀请，目前只在 iOS 上可用，“旨在成为人们可以轻松与许多不同的人工智能代理互动的地方”。

“在过去 12 年创建和运营 Quora 的过程中，我们学到了很多关于构建消费者互联网产品的知识。我们在为寻求知识的人提供服务方面经验丰富，”这位发言人说。“我们相信，我们所学到的大部分知识可以应用到这个新领域，在这个领域，人们正在与大型语言模型进行交互。”

因此，Poe 并不是试图从零开始构建一个类似 ChatGPT 的人工智能模型。ChatGPT——它有能力回答从诗歌到编码等话题的问题— 一直是争议的对象，因为它有时能给出听起来令人信服但事实上不真实的答案。本月早些时候， Q &一个编码网站 Stack Overflow 暂时禁止用户分享 ChatGPT 生成的内容，称人工智能让用户太容易产生回应，并让可疑的答案充斥网站。

例如，如果 Quora 在其平台的大量众包问答中训练一个聊天机器人，它可能会陷入困境。用户可能会对他们的内容以这种方式使用有异议——特别是考虑到一些人工智能系统已经被证明会回流他们接受训练的部分数据(例如代码)。一些团体抗议像 Stable Diffusion 和 DALL-E 2 这样的生成艺术系统，以及像 GitHub 的 Copilot 这样的代码生成系统，他们认为这些系统窃取了他们的作品并从中获利。

也就是说，微软、GitHub 和 OpenAI 在一场集体诉讼中被 T2 起诉，指控他们允许 Copilot 在不提供信用的情况下重复使用部分授权代码，违反了版权法。在艺术社区门户网站 ArtStation 上，今年早些时候开始允许人工智能生成的艺术在其平台上，成员开始广泛抗议，在他们的作品集里放置“无人工智能艺术”的图像。

图片来源: Quora

在发布时，Poe 提供了对几个文本生成人工智能模型的访问，包括 ChatGPT。(OpenAI 目前没有为 ChatGPT 提供公共 APIQuora 发言人拒绝透露 Quora 是否与 OpenAI 就 Poe 或其他形式的早期访问建立了合作关系。Poe 就像一个短信应用程序，但对于人工智能模型来说——用户可以单独与模型聊天。在聊天界面中，Poe 为对话主题和使用案例提供了一系列不同的建议，如“写作帮助”、“烹饪”、“解决问题”和“自然”

Poe 在发布时只提供少数模型，但 Quora 计划为模型提供商(如公司)提供一种方式，在不久的将来提交他们的模型。

“我们认为这将是人们互动和探索不同语言模型的一种有趣方式。这位发言人说:“Poe 旨在通过自然对话，成为人们获得任何问题即时答案的最佳方式。”。“为了提高这些模型的功能，已经进行了大量的研究和开发，但为了将所有这些价值带给世界各地的人们，需要易于使用的良好界面。我们希望我们能够提供这样的接口，这样，随着未来几年所有这些发展的发生，世界各地的每个人都可以尽可能多地分享好处。”

众所周知，人工智能聊天机器人，包括 ChatGPT，可以生成偏见、种族主义和其他有毒内容——更不用说恶意代码。Quora 自己并没有采取措施来解决这个问题，而是依靠 Poe 中模型的提供者来调节和过滤内容。

这位发言人说:“模型提供商已经付出了很大努力来防止机器人产生不安全的反应。”

这位发言人很清楚 Poe 现在不是 Quora 的一部分——将来也不一定是。Quora 将其视为一个独立的项目，就像谷歌的人工智能测试厨房一样，它计划随着时间的推移不断迭代和完善。

当被问及 Poe 背后的商业动机时，这位发言人表示异议，称这还为时尚早。但是不难想象，Quora 的大部分收入来自付费墙和广告，如果它成长的话，它可能会在 Poe 中加入额外的功能。

不过，Quora 表示，目前它专注于解决可扩展性问题，从测试人员那里获得反馈，并解决出现的问题。

这位发言人说:“整个领域现在发展非常迅速，我们更感兴趣的是弄清楚我们能为 Poe 患者解决什么问题。”
