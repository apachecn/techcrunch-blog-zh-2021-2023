欧洲科技界发起 OneUkraine 援助乌克兰

原文：https://web.archive.org/web/https://techcrunch.com/2022/05/24/key-european-tech-founders-and-investors-launch-oneukraine-charity-to-assist-ukraine/amp/

图片来源: OneUkraine

许多主要的欧洲科技创始人和投资者今天支持启动 OneUkraine ，这是一个新的慈善机构，为乌克兰人民提供可持续的人道主义救济。

OneUkraine 将支持国内外的乌克兰人，提供人道主义援助，并致力于通过当地的中小企业和初创公司重建乌克兰的技术和更广泛的基础设施。由于该组织的许多创始成员来自乌克兰或与该国有家族关系，该组织希望利用对当地网络和关于该国需求的真实世界数据的直接访问。

该组织表示，他们已经疏散了 5500 多人，其中大部分是妇女和儿童，并且已经提供了价值超过 400 万€的援助。它还在立陶宛建立了一所乌克兰学校，为难民提供教育。

OneUkraine 首席执行官兼联合创始人马丁·赖特尔(Martin Reiter)在一份声明中表示:“我们的许多创始人都在乌克兰出生和长大，我们觉得有必要帮助我们的朋友、家人和同事。我们所有的创始人都有领导或创立科技独角兽的记录，我们现在正在集体做我们最擅长的事情:快速有效地大规模工作，为乌克兰人民提供即时和急需的人道主义救济。我们相信，我们可以通过为人道主义援助界带来一种创业、数据驱动和可持续的方法来提供最好的帮助，从而补充其他重大救援工作。”

OneUkraine 的创始团队:

  • 马丁·赖特尔:马丁早在 2011 年就在乌克兰运营 Groupon，他在乌克兰仍有许多朋友。马丁曾在 Airbnb 和 Wayfair Europe 担任领导职务，并与人共同创立了 mademoisellemartina.org，帮助疏散乌克兰妇女和儿童。
  • Martina Kojic: Martina 在战后的克罗地亚长大，战后局势的影响和创伤塑造了她的童年。玛蒂娜是帮助疏散乌克兰妇女和儿童的 mademoisellemartina.org 的创始人之一。
  • 马库斯·富尔曼(Markus Fuhrmann):马库斯的妻子和家人来自乌克兰马里乌波尔，是 GROPYUS 的首席执行官和联合创始人，之前是 Lieferheld.de 和 Delivery Hero 的联合创始人。
  • 沃尔夫冈·海格尔:现居立陶宛的沃尔夫冈是 NFQ 和 HomeToGo 的创始人。
  • Viktoriya Tigipko: Viktoriya 出生并成长于乌克兰基辅，是 TA Ventures 的创始合伙人，也是网络天使投资人 ICLUB Global 的创始人。她是拥有 5000 多名科技行业女性的网络 WTECH 的创始人，乌克兰创业基金董事会主席，Odesa 国际电影节的创始人兼主席。
  • Johannes Reck: Johannes 是 GetYourGuide 的联合创始人兼首席执行官，他的祖父在今天乌克兰西部的一个小村庄长大。
  • Dmitry Gorilovskiy: Dmitry 的母亲来自乌克兰的罗夫诺，自 2014 年俄罗斯政府禁止外国人收养俄罗斯残疾孤儿以来，他一直在组织收养俄罗斯残疾孤儿。他是 Woodenshark 和 Moeco 等产品设计、物联网和机器学习业务的系列创始人。
  • Klaus Hommels: Klaus 是 Lakestar 的创始人和董事长，也是欧洲领先的风险投资家之一。
  • 延斯·希尔格斯(Jens Hilgers):延斯的妻子是乌克兰人，她是一名连续创业者，在中欧和东欧以及亚洲创建并管理国际游戏和科技公司。其中，延斯是 BITKRAFT Ventures 的创始 GP，也是 G2 Esports 的联合创始人和董事长。
  • Alexa Sinyachova: Alexa 是乌克兰人。她是 Moeco 和 WTech Berlin 策展人的联合创始人兼首席执行官。

OneUkraine 是一项长期计划，需要大量捐助资金。运营团队的费用将完全由创始人的个人资金提供，但 OneUkraine 正在邀请捐助者和其他合作伙伴帮助资助这些项目，并有效地支持乌克兰。该平台在德国注册为 gGmbH，是一家慈善有限公司，由专门从事慈善事业的领先审计机构 Ecovis 进行审计。
