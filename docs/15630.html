D-ID 的新网络应用为 OpenAI 的 ChatGPT | TechCrunch 提供了一个面孔和声音

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/07/d-ids-new-web-app-gives-a-face-and-voice-to-openais-chatgpt/

D-ID,深度怀旧背后的以色列初创公司今天宣布，它将推出新的网络应用的测试版，允许用户与逼真的人工智能面对面交谈。名为 chat 的网络应用。D-ID 结合了 D-ID 的文本到视频流技术和 OpenAI 的 ChatGPT，使与 AI 的对话更加容易。

这家初创公司的首席执行官兼联合创始人吉尔·佩里(Gil Perry)告诉 TechCrunch，D-ID 认为给 ChatGPT 一个声音和面孔将允许更多人使用这项技术，因为不会读写的人现在可以与人工智能交谈。聊天的另一个目的。D-ID 让老年人更容易使用 AI。该公司认为，其新的网络应用程序可以更广泛地访问 ChatGPT。

“该应用程序是一种更简单的方式来使用人工智能的力量，并与 ChatGPT 进行交谈，”佩里说。“我们天生就喜欢和面孔交流，当我们这样做的时候，我们能更好地理解情况。当我们处于感觉像真实的场景中时，我们会感觉更舒服，可以更好地观察复杂的信息。视频比文本更有效，因此该应用程序通过添加人脸来增加大型语言模型的能力。”

当你在桌面或移动设备上打开网络应用程序时，你会看到一个名为“爱丽丝”的虚拟形象然后，您可以选择键入问题或单击麦克风图标大声说出您的问题。D-ID 注意到爱丽丝几乎可以回答任何问题。你可以让爱丽丝模拟一次工作面试，甚至主持你的家庭琐事之夜。鉴于这款网络应用仍处于测试阶段，它并不完美。

图片来源: D-ID

该网络应用程序将很快提供各种各样的虚拟形象供用户选择。用户还将能够上传他们选择的任何图像。然而，佩里指出，你不能上传名人或公众人物的照片。几周后，这款网络应用将让用户生成一个角色，比如《哈利·波特》中的邓布利多，并与他们对话。佩里认为，当与人工智能交谈时，拥有正确的面孔很重要。例如，如果一个小学生想学习量子物理，他们可以将应用程序的脸改为阿尔伯特·爱因斯坦的脸。

人们可以使用聊天。D-ID 是免费的，但 Perry 指出，如果网络应用开始发展，并变得太多，以至于初创公司无法独自处理，公司可能会开始向用户收费。佩里指出，D-ID 已经准备好让这个应用程序像病毒一样传播，目标是保持它的免费。

有些人可能会觉得这个新的网络应用有点令人毛骨悚然，D-ID 也意识到了这一点。然而，佩里认为，人类有一天将在日常生活中使用人工智能，D-ID 认为，给人工智能一张脸将使人们更容易与技术交流。

当微软上个月在必应中加入类似 ChatGPT 的功能时，这个科技巨头在用户收到来自搜索引擎的令人不安和不安的回复后，面临了一些反弹。D-ID 认为它不会面临类似的情况，因为 OpenAI 已经改进了它的 ChatGPT API 并添加了护栏。尽管 D-ID 表示，它对爱丽丝的回答不负任何责任，但它已经在其网络应用程序中纳入了保护措施，因为用户将无法上传色情或暴力图像。

图片来源: D-ID

在 D-ID 推出了一个针对企业的聊天 API 用于品牌推广和客户体验的一周之后，新的消费者应用程序发布了。文本聊天机器人是消费者与品牌互动的一种流行方式，D-ID 指出，这代表了其新 API 的一种可能用途，因为它将允许品牌通过更个性化和互动的体验与客户互动。

该公司的最新进展是在 D-ID 和包括 Adobe 和 OpenAI 在内的许多其他公司宣布参与合成媒体的道德和负责任的开发、创作和共享框架之际。

自几个月前推出以来，Open AI 的 ChatGPT 一直主导着互联网，并越来越受欢迎。因此，在过去的几个月里，人工智能已经成为一个越来越热门的话题，所以 D-ID 希望给这项技术增加一层真实感也就不足为奇了。

D-ID 今天还宣布，它将与非营利组织 Spring ACT 一起发起一项活动，利用生殖人工智能来加强对家庭暴力受害者的全球在线支持。微软正在支持这项活动，它利用了微软 Azure AI 的文本到语音功能和 Azure AI 云基础设施。
