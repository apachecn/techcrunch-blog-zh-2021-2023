Twitter 告诉用户，他们已经超过了每日推文限额

原文：https://web.archive.org/web/https://techcrunch.com/2023/02/08/twitter-tweet-limit-bug-error/

Twitter 告诉用户，他们已经超过了每天的推文限额

Twitter 用户最近已经看到了不稳定的政策变化和奇怪的错误，但今天他们甚至不能通过它发微博。

许多用户报告说，从太平洋时间下午 2 点左右开始，他们突然无法发推特了。现在，这个错误似乎正在影响所有非预定的推文。从我们的测试来看，之前预定的或预定继续的推文似乎仍然可以通过。该错误在该应用的网络和移动版本中是一致的。

在手机上，用户会看到一个错误，说明 Twitter“无法发送你的推文”在网络上，用户会看到一个错误，通知他们已经超过了推文限制——这条消息表明这可能是与 Twitter 的 API 速率限制配置有关的错误或漏洞。

实际上，Twitter】确实有每日推文限额，但你不会超过这个限额，除非你在一天内超过 2400 条推文的门槛。根据今天收到错误之前零次发推特的经验，目前平台上似乎有问题。

另一个错误显然是阻止用户关注新账户，许多 Twitter 用户报告看到一个弹出窗口通知他们“目前无法关注更多人”

许多用户也报告了在 Twitter 上发送直接消息的问题。当我们试图为这个故事发送一个测试 DM 时，页面无法加载并显示一条错误消息。Twitter 客户端 TweetDeck 也于周三下午停止工作，但尚不清楚该问题是否相关。总的来说，该平台目前几乎无法使用，尽管在太平洋时间下午 5 点左右，一些人的出站非预定推文似乎又恢复了功能。

Twitter 周三承认了这些普遍存在的问题，但尚未提供具体问题或何时解决。
