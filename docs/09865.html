与 YC 创始人关于他们创业历程的 7 次对话

原文：https://web.archive.org/web/https://techcrunch.com/2022/09/07/yc-demo-day-found-podcast/

本周是 YC 展示日，这让我们怀念起主持人与经历过著名加速器项目的创始人的对话。在《发现》的每一集中，主持人达雷尔·埃瑟林顿和乔丹·克鲁克都会挖掘创始人的背景，创办和经营公司的巅峰和低谷，以及一路上给予帮助的关系和导师。不足为奇的是，当一位 YC 的创始人出现时，谈话内容往往会涉及他们在加速器中的时光是如何对他们的公司和他们作为领导者的形成性经历。所以，如果你曾经想听 YC 的创始人们打开他们三个月来最好的教训和一般的学习，系好安全带。

以下是我们最喜欢与这些创始人的一些对话，排名不分先后。尽情享受吧！

1.魔铃乐队的哈娜·莫汉

在这一集中，达雷尔和乔丹与 MagicBell 的联合创始人兼首席执行官进行了交谈，magic bell 是一家初创公司，正在处理解决其他软件产品通知的艰巨任务。他们进入了 Hana 的创业之路，被贴上了“高维护员工”的标签，加入了 Y Combinator，以及她在创业世界的经历。

我们最近采访了 Hana，看看 MagicBell 是如何发展的。点击查看那一集。

2.来自阿德恩的伊丽莎白·鲁佐


我们的工作是与那些致力于解决难以置信的难题的令人印象深刻的人交谈——但 adyn 的创始人兼首席执行官 Elizabeth Ruzzo 可能是我们交谈过的最令人印象深刻的创始人之一。她不仅为女性开发了唯一的测试，以确保她们被处方的节育措施最不可能产生有害的副作用，她还创建了公司，并作为公司的唯一员工筹集资金。她与达雷尔和乔丹谈论了她作为一名单独的创始人/员工为节育解决方案筹集资金时所面临的挑战，她为什么决定离开学术界，以及她必须穿越复杂的监管迷宫才能让 adyn 起步。

3.汪薇薇从登陆

在本期节目中，Landed 创始人兼首席执行官汪薇薇谈到了她为蓝领工人提供高质量工作机会的使命。Landed 处理从招聘到审查到安排面试的招聘流程，并为总经理提供反馈，使他们的工作场所更令人满意。他们还通过在员工找到工作后帮助他们升级来改善他们的财务状况。Darrell、Jordan 和 Vivian 谈到了 COVID 如何向我们展示了蓝领工人是多么重要，如何让这些工人明显得不到充分的服务，以及她如何计划通过帮助他们更快地获得工资、建立信用和减少流动来改善这些工作的体验。

4.来自 Epsilon3 的 Laura Crabtree


Space enterprise 创始人 Laura Crabtree 与 Darrell 和 Jordan 讨论了启动 Epsilon3 以及他们如何帮助整个航天行业(及其他行业)的公司规划、跟踪和记录他们的大型项目和任务。他们讨论了他们日益增长的市场适应性、与投资者建立关系以及拥有三位创始人的好处。

5.《退出天才》的优素福·舍瓦尼


Yusuf Sherwani 是 Quit Genius 的首席执行官和三位医生/联合创始人之一，Quit Genius 是一款应用程序，使用 CBT 疗法和其他成熟的方法来帮助患者解决成瘾问题。他与 Darrell 和 Jordan 讨论了在 COVID 之前创办一家远程医疗公司的情况，以及他们必须克服的监管障碍，以及他们如何说服投资者，在打造以患者为中心的产品时，以研究为导向是至关重要的。

6.Nextmv 的卡罗琳·穆尼


卡罗琳·穆尼是 Nextmv 的联合创始人兼首席执行官，该公司帮助公司在大规模范围内做出高效决策——想想亚马逊分发包装或优步为优步拼车绘制路线。在这一集中，她与达雷尔和乔丹谈论了 Nextmv 的软件，该软件不仅优化了决策和路线规划，还使工程师能够在许多不同类型的团队中工作。此外，他们还讨论了高中排球教练如何让她成为一名更好的领导者，并迫使她优先考虑工作和生活的平衡。

7.高罂粟的利·霍尼韦尔

利·霍尼韦尔在她的职业生涯中一直试图阻止不好的事情发生在互联网上的人们身上。她曾在 Slack、Heroku 和微软工作过，精通在线骚扰的技术和人性两方面，亲眼目睹了这种骚扰如何升级为黑客攻击甚至更糟。这就是她如何想到 Tall Poppy 的，这是一个帮助拥有面向公众的员工的组织，如媒体组织，积极防止这种类型的升级的平台。高大的罂粟模式将一个不可扩展的业务变成了一个快速增长的创业公司。
