自 2015 年创办风险公司 Backstage Capital 以来，阿尔兰·汉密尔顿已经向超过 195 家由代表性不足的创始人领导的公司投资了数百万美元，从阿朵的汽车保险到反思我们如何虚拟学习的团队。尽管业务范围很广，但汉密尔顿表示，她的投资组合公司一直会问她两个问题:

“你能帮我们筹集资金吗？“你能帮我们招聘吗？"

虽然汉密尔顿的基金是对前者的回应，但她最近的赌注——由汉密尔顿自己建立——是一家探索后者的初创公司。 Runner 是一个劳动力市场，为寻找兼职工作的创业公司和运营人员牵线搭桥。它试图解决初创企业建设初期的一些最大的矛盾，比如决定什么时候雇佣你的第一个人才主管，或者决定外包什么，或者在员工配备方面内部建设什么。它的发布明确关注运营角色。

“如果你想学习如何编程，或者想找一份技术性更强的工作，有很多地方可以去，”汉密尔顿说。“但是如果你想成为某人的得力助手，你现在该去哪里，首席运营官，等等。…对大多数(公司)来说，这是一种事后想法。”

从概念上讲，跑步者不是逆向投资者。Upwork 和 Fiverr 在自由职业者经济的基础上建立了稳固的业务。然而，这家初创公司的不同之处在于它的目标是谁——技术领域的运营人员——以及它如何雇佣他们。每一个寻找新工作的“跑步者”或兼职专业人士，都被该公司以 W-2 分类雇用。今天大约有 200 名选手在台上。

汉密尔顿告诉 TechCrunch，在 Runner 将在 app store 上发布之前的几周，这种方法已经吸引了 150 万美元的种子前支持。对于整个故事，包括该公司的一群投资者如何提出一系列有趣的问题，请阅读我在 TechCrunch 上的故事:阿尔兰·汉密尔顿想要改变创业公司如何招聘。

本周交易

由于最近的融资事件，Flutterwave 现在是非洲估值最高的初创公司。跨境支付平台凭借其精明的 API 方法击败了 OPay 和 Chipper Cash。

这很重要，原因如下:非洲的科技行业可能会经历大量的整合。正如塔吉·肯-奥卡弗报道，“未来，Flutterwave 将寻求收购，进一步巩固其在金融科技领域的权威。随着这家支付巨头继续深化其在中小企业和消费者金融科技领域的影响力，我们可以推测，较小的初创公司——包括它支持的公司，如 cinet pay——可能会成为收购目标。”
