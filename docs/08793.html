微软将收购 RiskIQ 付诸实施

原文：https://web.archive.org/web/https://techcrunch.com/2022/08/02/microsoft-puts-its-riskiq-acquisition-to-work/

微软今天为其 Microsoft Defender 安全平台增加了两项新功能: Microsoft Defender 威胁情报和 Microsoft Defender 外部攻击面管理。这些功能是基于该公司对 RiskIQ 的收购，随着这次发布，微软现在将 RiskIQ 的一些核心功能引入自己的安全平台(同时 RiskIQ 也继续运营自己的服务)。

“我们的使命是为所有人建立一个更安全的世界——威胁情报是它的核心，”微软的瓦苏·杰克卡尔告诉我。“如果你不知道你周围的世界正在发生什么，就很难理解该做些什么以及如何采取行动。今天，微软拥有最广泛和最深入的威胁信号——正如我们刚刚在财报中宣布的那样，我们正在跟踪(每天)43 万亿个来自身份、设备、平台、电子邮件和协作工具的信号。”

借助 Defender Threat Intelligence，微软正在使用 RiskIQ 的技术扫描互联网，并向现有的 Defender 实时服务提供额外的数据，以帮助安全团队主动保护其基础设施。当然，微软已经有了一个大型信号图来支持其 Defender 平台，但 Jakkal 指出，RiskIQ 的数据不仅有助于丰富现有的数据集，还可以在 Defender 上增加一个层，让安全团队可以看到整个攻击链。

“他们可以看到整个攻击链，他们可以采取行动，然后结合他们自己的人类智能，他们可以看到攻击的方向以及如何主动预防它，”Jakkal 解释道。

图片来源:微软

该服务还为用户提供了来自微软安全专家的原始威胁情报和分析库，这反过来应该有助于安全团队发现、删除和阻止可能隐藏在其组织内的对手工具。

同时，新的外部攻击表面管理服务可帮助这些安全团队了解潜在攻击者如何看待他们的网络。与类似的服务一样，它为安全团队提供了一种方法来发现他们的所有资源，并找到那些未知的和/或非托管的资源。大多数开始使用此类服务的企业最终会惊讶地发现有这么多面向互联网的非托管资产。

形象积分:微软

“所有公司都在问这样一个问题:我的安全程度如何？这个问题很简单，但很难回答。因为第一点是，首先，我们需要了解威胁世界正在发生什么。我们需要了解这是什么样子。我们需要了解的第二件事是我们的资源在哪里，”杰克卡指出。有了这些新工具，Microsoft 可以为安全团队提供更多可用于保护其网络和其他资产的数据。
