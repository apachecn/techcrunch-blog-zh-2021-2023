Tier 以 2 亿美元的全股权“首次完成”D 轮投资| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/10/25/tier-scoots-off-with-first-close-of-its-200m-series-d/

总部位于柏林的电动滑板车租赁公司 Tier Mobility 宣布融资 2 亿美元，称这是其 D 轮融资的“第一笔交易”，因为投资者继续将资金投入城市微型汽车。

这家初创公司向 TechCrunch 证实，在 D 轮融资结束前，它将筹集更多资金(整轮融资也将是债务和股权的混合；尽管这第一部分只是股权)。

不过，该公司拒绝透露 D 轮融资的目标金额。

一位 Tier 发言人表示:“作为 D 轮融资的一部分，我们刚刚完成了第一笔交易，寻求增加股权融资，并筹集新的债务融资。”。

Tier 的公关还将今天的首次交易描述为“更广泛的股权和债务融资的一部分”。

今天的公告紧随今年夏天 6000 万美元的债务融资和去年 11 月2.5 亿美元的 C 轮融资(由软银牵头)。

微型汽车初创公司通常有很高的资本支出成本，尤其是在扩张期间扩大车队，因此债务增加相当常见。

在一份声明中，Tier Mobility 首席财务官亚历克斯·盖耶(Alex Gayer)表示:“这笔股权融资为扩大我们在全球多式联运市场的存在提供了进一步的火力，并寻求战略投资和收购。我们的车辆资本支出需求将通过释放债务能力来满足。我们的目标是将 Tier 打造成为欧洲微型移动强国，巩固我们目前作为共享电动滑板车市场头号玩家的地位。”

现有投资者软银愿景基金 2 与阿联酋的 Mubadala Capital 共同领导 D 轮融资，Mubadala Capital 去年也支持了 Tier 的 C 轮融资，而其他现有投资者 RTP Global、Novator、White Star Capital、Northzone 和 Speedinvest 也参与了融资。

Mubadala 对 Tier 越来越多的支持是在它去年进入阿联酋市场之后——在经过长时间的小型摩托车租赁试验后，它被当地道路和运输管理局选中。

新的投资者也加入了融资行列，包括绿色影响基金 M&G Investments 和多元化全球投资控股公司 Mountain Partners。

Tier 表示，其微移动业务现在的价值为 20 亿美元(比去年 C 轮融资时报道的价值高出约一倍)，称迄今为止已经筹集了总计 6.6 亿美元的股权和债务资金。

2018 年推出的电动滑板车，电动自行车和电动助力车初创公司在一个竞争激烈的领域与无数参与者竞争，包括 Bird，Dott，Lime，Voi 和 Wind(仅举几例)——尽管世界各地的城市当局试图通过限制每个城市允许的运营商数量，为快速发展的微移动市场带来一点结构。这意味着作为城市供应商赢得一个位置可以帮助在当地超越竞争对手。

例如，在新闻稿中，Tier 吹嘘其最近赢得了在伦敦(与 Dott 和 Lime 一起)提供电动滑板车出租的招标。它吹捧的其他新城市都在中东:麦纳麦(巴林)和多哈(卡塔尔)。

它也在巴黎运营——巴黎市政当局一直在大力推动城市交通组合从汽车转移到自行车和公共交通。因此它可以宣称取得了一些重大胜利。

迄今为止，Tier 表示，它已经在 16 个国家的 150 个城市部署了 135，000 辆电动滑板车、电动自行车和电动助力车，声称已经“通过无与伦比的资本效率和卓越的运营”成为欧洲市场的领导者。

这家总部位于柏林的初创公司的新融资计划是为了“收购和战略投资”，以及进一步的国际扩张 Tier 表示，它将瞄准欧洲和中东的战略增长市场。

因此，随着新的资金涌入，听起来像是更多的整合正在走向快节奏的电动滑板车市场。(规避城市对运营商数量限制的一种方法是收购那些在你没有中标的城市中标的竞争对手，以进一步扩大规模……)

Tier 还表示，它将引导部分投资继续推出由当地企业托管的电池充电站网络，也就是 Tier 能源网络。

“随着电动自行车在几个欧洲国家的推出，Tier 正在扩大其不断增长的多模式选择范围，使其成为首家在一个应用程序中为用户提供三种不同类型车辆的欧洲微型移动提供商，”它补充道。

在一份声明中，首席执行官兼联合创始人劳伦斯·洛伊施纳(Lawrence Leuschner)表示:“这笔资金为 Tier 提供了额外的资源，以实现我们永远改变移动性的使命。在如此短的时间内，超过 8000 万次出行，取代了 1300 多万次汽车出行，这充分证明了世界各地的城市都在寻找使其交通网络更加安全的方法，并朝着零排放的未来迈进。”

欧洲 Mubadala Capital Ventures 董事 Amer Alaily 在另一份支持声明中补充道:“从 Tier 迪拜总部到柏林总部，整个组织都可以感受到 Lawrence、Matthias 和 Alex 对变革的热情。“他们不仅迅速成为欧洲微移动领域的领导者，而且致力于可持续发展，使他们从竞争对手中脱颖而出。我们很自豪能够成为他们旅程的一部分，并期待在未来的岁月里继续成为劳伦斯及其团队的合作伙伴。”

在 Tier 向我们确认这 2 亿美元的融资是其打算进行的更大规模 D 轮融资的第一笔全股权融资后，该报告得到了更新，D 轮融资一旦完成，将是债务和股权的混合。
