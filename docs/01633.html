抖音正在测试一款名为抖音直播工作室 TechCrunch 的桌面流媒体软件

原文：https://web.archive.org/web/https://techcrunch.com/2021/12/15/tiktok-is-testing-a-desktop-streaming-software-called-tiktok-live-studio/

抖音正在测试一款名为抖音直播工作室的桌面流媒体软件

如果你不在 Twitch 上看你最喜欢的抖音明星视频，而是在抖音上看他们玩视频游戏，会怎么样？据该平台称，在过去的几天里，抖音一直在测试一个名为抖音直播工作室的 Windows 程序。

一旦下载到你的桌面上，该程序允许用户用他们的抖音帐户登录，直接向抖音直播。在该程序中，你可以通过聊天功能与观众交流，还可以从你的电脑、手机或游戏机上播放内容。抖音告诉 TechCrunch，这个程序目前只在少数西方市场为几千名用户提供。

抖音测试这项功能的一个明显好处是，创作者可能会被激励留在他们的应用程序中，而不是要求他们的观众在 Twitch 或 YouTube 游戏上观看他们的流媒体。从抖音的宣传图片来看，该公司似乎是在迎合那些希望观看视频游戏的人。但在其雏形中，抖音直播工作室缺乏经验丰富的流媒体工具在 OBS 或 Streamlabs 等广播软件上可以找到的功能——这是有意义的，因为该产品仍在测试中。例如，即使你可以将视频游戏控制台连接到抖音直播工作室，你也不能使用浏览器窗口作为来源，也不能设置提示或新关注的流内提醒。

但是这个功能测试的一个目标是看创作者如何使用这个软件。该公司告诉 TechCrunch，抖音直播工作室并不保证会推出——每当抖音测试一个功能，并不意味着它会永远存在。但如果我们真的看到抖音直播工作室的推出，这个平台将研究创作者使用该软件的目的，然后对其进行调整，以更好地适应这些用例。

创作者已经在使用抖音的直播工具来为 T2 赚钱并扩大他们的追随者。创作者可以接受提示，安排活动，使用直播 Q & A 工具，与其他用户一起直播，过滤某些关键词，分配主持人。目前，抖音直播工作室允许用户选择是否要启用礼物和评论，他们可以设置关键字过滤器。但其他现有的直播功能，如分配主持人和安排流媒体，可能会被纳入未来更强大的产品中。

这款流媒体软件也标志着抖音有机会接触到更多的桌面用户。对于创作者来说，桌面流媒体软件是必不可少的，因为智能手机根本不是为了连接到游戏主机并同时播放流媒体而构建的。虽然人们确实在手机上观看视频游戏流，但用户在桌面上的体验通常更好，这可以帮助用户养成在电脑上使用抖音的习惯。
