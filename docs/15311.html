当你回复或转发的推文收到社区通知时，Twitter 会发送通知

在社交媒体上盲目放大观点或帖子是错误信息迅速传播的关键原因之一。多年来，知名人士在 Twitter 上发布或转发虚假信息。该社交网络现在通过一个新的社区笔记(其众包事实核查程序)功能，为这种情况提供了一个撤回转发的机会。

Twitter 现在会通知用户，如果他们喜欢、转发或回复的推文收到了来自社区注释贡献者的上下文信息。“这有助于给人们提供额外的背景，否则他们可能会错过，”该公司在一条推文中说。

喜欢、转发或回复一条推文让它在推荐算法中具有某种相关性 Twitter 已经推动其算法提要有一段时间了。如果一个社区笔记贡献者的背景可以揭穿原来的观点，人们可能会删除他们的喜欢或转发。

该项目于 2021 年以的名字“观鸟”首次引入美国用户。在埃隆·马斯克收购 Twitter 后，他将该程序重新命名为“社区笔记”——尽管杰克·多西认为这是“有史以来最无聊的脸书名字”

12 月，Twitter 开始向全球所有用户显示社区笔记。今年 1 月，该公司开始接受来自美国以外的英国、爱尔兰、澳大利亚和新西兰的贡献者的笔记。因此，社区笔记计划仍然缺少全球背景。

在过去的几个月里，马斯克公司已经对算法进行了修改，比如调整低质量笔记的可见性，扩展贡献者笔记的类型，稳定贡献的影响分数。

鉴于新管理层已经解雇了几个人，包括在安全和信任部门工作的承包商，Twitter 对算法和众包进行内容审核的依赖迅速增加。这一负担可能会进一步增加，因为该公司已经关闭了对研究人员的免费 API 访问，他们中的许多人在平台上指出了仇恨言论和错误信息。
