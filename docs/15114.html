微软与人工智能: 本周在一次新闻发布会上，微软首席执行官塞特亚·纳德拉表示“搜索的新一天开始了。”他指的是该公司将 OpenAI 的 ChatGPT-4 整合到其 13 岁的搜索引擎 Bing 中。雷德蒙的人说，希望是整合将有助于必应更好地与谷歌竞争。在人工智能新闻发布后，这款应用的下载量猛增了 10 倍，因为微软承诺新必应的安装者可以优先使用。想要更深入的了解吗？查看 Frederic 对搜索引擎的实际操作。

谷歌和人工智能: 为了不被超越，谷歌本周宣布了 Bard，它是 OpenAI 的 ChatGPT 的对等物。它现在正处于测试阶段，它将使用谷歌的对话应用语言模型来驱动一个对话式人工智能，该人工智能将利用网络上的信息。但是 Devin 也说公司正在失去控制。

猜猜多重搜索的动力是什么？人工智能技术称为多任务统一模型。妈妈！

GitHub 裁员: 本周，微软的 GitHub 宣布将在 3000 名员工中裁员 10%。作为“保护公司短期健康”努力的一部分，GitHub 还将关闭其所有办公室，并完全远程化。

M2 上的苹果高管: 在一次与苹果副总裁的广泛采访中，我的老板兼 TechCrunch 总编辑马修·潘扎里诺(Matthew Panzarino)探讨了该公司最新的 M 系列芯片，并深入探讨了 M2 MacBook Pro 和 Mac mini 型号。剧透:他们更快。

印度封杀博彩和贷款应用: 印度电子和信息技术部呼吁封杀 232 款博彩和贷款应用，以保证用户数据安全，原因之一。

足球！: 现在是超级碗 LVII 时间，你可以通过多种方式进行直播。在此获取所有详细信息。

声音的

本周，Natasha 采访了 Cleo 的首席商务官和前首席执行官 SJ 萨凯蒂，谈论了自我、设定界限、辞职和成为“统计数据”以及为什么一家公司需要在没有你的情况下取得成功。在 Found 节目中，达雷尔和贝卡采访了直接面向消费者的香水公司 Ourside 的创始人兼首席执行官凯塔·伯克-威廉姆斯(Keta Burke-Williams)，了解是什么让她对颠覆这个庞大且过时的香水行业感兴趣。

科技博客+

TC+订阅者可以获得深入的评论、分析和调查——如果你已经是订阅者的话，你会知道的。如果你不是，考虑报名吧。以下是本周的一些亮点:

AI 的偏见: Dom 报告称，偏见出现在 AI 的大部分方面，从投资和雇佣到数据收集和制作。所以问题仍然是:人工智能的下一个前沿到底是为谁服务的？

非洲的创业生态系统: 去年，非洲的创业公司获得了创纪录的投资。Tage 与八位投资者进行了交谈，发现其中的关键是种子前期和种子阶段的投资者。但是还有很长的路要走。

菠菜. io 的推介平台拆除: Haje 将注意力转向菠菜. io 的种子平台，这是一家为工程师打造会议工具的公司。

网络安全专家: 撰稿人大卫·j·比安科(David J. Bianco)写道，防御者进退两难:“攻击者拥有所有优势，防御者必须被动等待，等待回应，这几乎是网络安全的一个公理。也是骗人的。”
