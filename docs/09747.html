LG 电子推出 NFT 平台，让用户买卖数字艺术品

LG 电子今天表示，它已经发布了不可替代的令牌(NFT)平台 LG 艺术实验室，用户可以在 LG 的智能电视上发现、购买、销售和交易 NFT。

大约八个月前，LG 电子在新闻发布会上表示，计划将 NFT 特色融入其智能电视。该平台目前在美国仅面向运行 webOS 5.0 或更高版本的 LG 电视用户，可以直接从主屏幕访问。

LG 电子与 Hedera network 合作提供了一款应用程序，用户可以通过 Wallypto 购买 NFTs，Wallypto 是 LG 的智能手机加密钱包。

LG 电子家庭娱乐公司平台业务负责人、高级副总裁克里斯·乔(Chris Jo)告诉 TechCrunch 说:“Wallypto 建立在 Hedera 之上，自 2021 年 9 月以来一直在开发。"它的钱包测试版于 8 月份在本周推出."

Hedera 和 LG 电子之间的合作始于 2020 年，当时 LG 加入了 Hedera 管理委员会，该委员会包括谷歌、IBM、德国电信和印度理工学院等。

“虽然许多人听说过 NFTs，并希望参与到不断增长的生态系统中，但它可能过于复杂，难以开始，”Jo 说。“LG 艺术实验室旨在让美国数百万用户轻松访问和显示 NFT，而无需与代码交互或直接与区块链互动。”

NFT 平台提供了一个 Drops 功能，可以描述艺术家并展示他们的新作品预览。它的实时实时投放功能会发出通知，帮助用户获得投放的 NFT，以确保他们不会错过任何机会。

图片来源: LG 电子 Barry X Ball Studio，Inc .提供。

Jo 还表示，Hedera 允许用户以每笔不到 0.0001 美元的价格进行交易(没有高额的汽油费或隐性成本)，这意味着没有与 NFT 平台相关的隐性成本。

“Hedera 网络消耗的能源远远少于任何其他公共分类账，这使其成为可持续计划的理想选择，也意味着它可以满足现代企业和投资者的 ESG，”Jo 解释道。他说，用户可以在 Hedera 网络上购买、销售和展示 NFT，而不必担心与其他公共网络相关的高能耗。他补充说，Hedera 保证了用户的低成本、可预测的费用，这与其他许多分散式网络不同。

南韩的 IT 和娱乐公司正在跳入不可替代的 NFT 产业。一月份，LG 的竞争对手三星公布了其新的基于智能电视的 NFT 市场计划。三星表示，将在三星电视上添加 NFT 功能，如 MICRO LED 和 Neo QLED。
