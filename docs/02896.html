三观:NFT 艺术市场的现状“直线上升”告诉了我们什么？TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/02/02/3-views-what-does-line-go-up-tell-us-about-the-state-of-the-nft-art-market/

NFT 艺术品市场异常火爆。

2021 年，收藏家在 NFT 上花费了 220 亿美元，高于前一年的 1 亿美元。

上个月末，加拿大视频制作人丹·奥尔森发布了一个两小时的视频，讲述了他对 web3 和区块链技术的强烈看法，题为“线路上升 NFTs 的问题”

简而言之:奥尔森断言，web3 的技术局限性如此之大，以至于它永远不会兑现自己吹嘘的承诺。他走得更远，基本上将加密货币描述为一种制造的问题比解决的问题更多的方案，本质上是低效的，并允许早期采用者榨取后来的进入者，直到系统耗尽更大的傻瓜。

我请 John 和 Alex 分享他们对 Olson 视频的看法，作为讨论加密行业总体状况的出发点。这里是我们结束的地方:

  • 沃尔特·汤普森:非功能性食物几乎不是最低限度可行的产品
  • 约翰·比格斯:科技的腾飞必须经历一次洗牌
  • 亚历克斯·威廉:我只是不想要你的 NFT

沃尔特·汤普森:非功能性食物几乎不是最低限度可行的产品

将加密货币项目和不可替代代币描述为投机性质是事实。同样无可争议的是，早期的加密采用者比后来的进入者享有巨大的优势，他们倾向于掩盖广泛讨论的道德问题，即生产这些数字产品需要多少能源以及如何营销它们。这很重要，但这里还有另一个问题:

目前，NFT 项目只是区块链投资者用来验证其假设的最低可行产品。

不可替代的令牌具有 MVP 的所有特征:除了最热情的早期采用者之外，阻止所有人的高准入门槛，最小的功能集，不固定的用例，以及对未来好处的建议，这些好处将远远抵消今天的成长烦恼。

皮尤研究中心的一项调查发现，到目前为止，大约 16%的美国人已经购买了加密货币，这表明它的流行速度有多快。像耐克和维萨这样的上市公司正花费数百万美元来早期介入这些不受监管、高度投机的投资。

同样值得注意的是:如果有紧急开支，大多数美国居民都拿不出 1000 美元。

正如 Steve Blank，一位帮助推广 MVP 框架的企业家曾经写道，“你是在向有远见的人，而不是每个人，出售远见和交付最少的功能集。”

NFT 拍卖市场 OpenSea，Bored Ape 游艇俱乐部的所在地，上个月完成了对该公司价值 133 亿美元的 3 亿美元 C 轮融资。据首席执行官 Devin Finzer 称，该平台在 2021 年的交易量增长了“超过 600 倍”，赚取了数亿美元的交易费。
