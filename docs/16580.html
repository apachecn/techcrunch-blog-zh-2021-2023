Neptyne 正在为数据科学家构建一个基于 Python 的电子表格

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/27/y-combinator-backed-neptyne-is-building-a-python-powered-spreadsheet-for-data-scientists/

Douwe Osinga 和 Jack Amadeo 在 Alphabet 建立高科技城市的风险投资公司 Sidewalk Labs 合作，当时他们得出的结论是，大多数电子表格软件无法应对今天的数据挑战。像 Pandas 和 Jupyter Notebooks 这样的数据科学工具确实有，但它们对于外行人来说太难接近了——至少在 Osinga 和 Amadeo 看来是这样。

“与任何分析师或财务建模师交谈，他们都会告诉你 Excel 不再适用，”Osinga 说。“每个人都意识到需要转向更强大的解决方案，Python 是显而易见的候选者。然而，与今天的工具合作并不令人印象深刻。”

Osinga 和 Amadeo 的解决方案是 Neptyne ，这是一款使用人工智能助手帮助用户编写电子表格而无需学习如何编码的应用程序。作为 Y Combinator winter 2023 班的一员，Neptyne 本月完成了 Y Combinator 和一群高调天使投资人的 200 万美元种子前轮投资，其中包括谷歌人工智能负责人杰夫·迪恩和谷歌地图联合创始人拉斯·拉斯穆森。

Neptyne 加入了一大批创业公司，致力于改变传统的电子表格。当然，还有 Airtable，以及像 Spreadsheet.com、 Actiondesk 和 Pigment 这样的新贵——最后一家在去年 11 月为其数据分析和可视化服务筹集了 7300 万美元。最近，等于，一家总部位于旧金山的风险投资公司，为其电子表格平台筹集了 1600 万美元，该平台整合了实时数据集成等工具。

图片来源:海王星

Neptyne 的不同之处在于它打包了一个基于 Python 的电子表格引擎，Python 是一种广泛用于数据科学的编程语言。Osinga 这样描述它:它是一个电子表格，一切都按照你期望的方式在电子表格中工作，但它也提供了对 Python 生态系统的访问——包括库、框架和工具。

“大型语言模型(如 OpenAI 的 GPT-4)最近产生了令人震惊的结果，恰好非常擅长编写 Python 代码，”Osinga 说。“由于 Neptyne 天生会说 Python，这意味着人工智能不只是帮助你编写公式或可视化数据——你可以与人工智能就你面前的电子表格应用程序进行对话，并让它为你修改它。”

例如，假设您有一个使用 Pandas 构建的数据透视表，Pandas 是一个用于数据分析的开源 Python 库，它按产品和地区汇总数据。使用 Neptyne，您可以要求 AI 助手修改数据透视表计算，以包括或排除产品，或者更改分组标准，例如按产品类别而不是单个产品。Neptyne 实时更新计算，让您探索不同的选项。

“今天，许多重要的建模和计算隐藏在复杂的电子表格或数据处理系统中，修改需要专家深入研究，”Osinga 说。“Neptyne 强大的人工智能集成消除了普通电子表格的限制，同时大大降低了使用高级数据工具的复杂性。”

他们的战斗文字。但对 Neptyne 来说还为时尚早——该公司还处于预营收阶段，没有披露其客户群的规模。Osinga 乐观地预计今年会有增长，以至于他预计会雇佣 6 名员工来充实 Neptyne 目前的 4 人员工队伍。
