在过去的几年里，风险投资资金充足且相对便宜。这创造了一种环境，在这种环境中，每个人的座右铭都变成了“不惜一切代价实现增长”从表面上看，一家成功的风险投资公司的秘诀变得非常千篇一律:每 18 个月融资一次；大力投资走向市场；以“标准”速度增长收入，第一年增长两倍，第二年再增长两倍，之后再翻倍。

这些源自富足时代的“风险主义”已经渗透到了各地的董事会和投资者会议中。事实上，问题是，“你预计筹集的资金能维持你多久？”实质上变成了智力测试。唯一正确的答案是 18 到 24 个月，没有考虑公司的具体情况。

人们可能还没有大声说出来，但这些 VCisms 开始感觉过时了。当资本不容易获得，或者从稀释的角度来看资本非常昂贵时，不惜一切代价的增长是行不通的。此外，当一轮融资不再需要一个月，而是需要三到六个月或更长时间时，每 18 个月进行一次融资就感觉非常繁重。

是时候问问我们自己这些 VCisms 是否仍然相关，或者是时候改变了。首先，让我们回顾一下。

我们是如何形成“不惜一切代价实现增长”的心态的？

在这一点上，众所周知，资本成本近年来有所下降。这主要是通过参考公司在不同阶段获得的估值增加来讨论的，如下图 b 所示。

资料来源:2012-2022 年的 PitchBook 数据。图片来源: Neotribe Ventures

我们越早开始针对具体公司的对话，越早承认千篇一律的成功秘诀是不够的，对所有相关方都越好。

与 2012 年至 2018 年相比，在所有阶段，公司都看到了更高的融资后估值，从最早阶段的约 40%到 2018 年至 2022 年期间的增长阶段的 200%以上。

换句话说，在过去三年里，一家公司可以筹集同样数量的资本，而稀释程度更低。

但是经常被忽略的事实是，数据告诉我们，公司实际上并没有筹集到同样多的资金。他们在每个阶段都筹集了更多的资金——明显更多。如图表 C 所示，与 2012 年至 2018 年相比，中值系列 C 在过去几年中的规模增加了一倍多。

资料来源:2012-2022 年的 PitchBook 数据。图片来源: Neotribe Ventures

如下图 D 所示，平均而言，各公司的稀释程度略低。

资料来源:2012-2022 年的 PitchBook 数据。图片来源: Neotribe Ventures

例如，在 2012 年至 2018 年的后续 B 轮融资中，现有股权持有人(种子和 A 轮投资者和创始人)的股权稀释率中值为 22%，而在 2018 年至 2022 年期间，他们的股权仅稀释了约 20%。那只有 10%的差别。

值得注意的是，在过去三年中，公司因这种稀释获得了两倍多的资本(即 2500 万美元对 1100 万美元，如图 C 所示)。这些资本可以通过市场营销和雇佣销售人员来推动增长。

如果投资者看到了类似的稀释水平，这是否意味着他们看到了类似的回报？

肯定不是。

我们来对比一下两个时间段。2012 年至 2018 年间，一家 C 轮公司的估值中值为 1.2 亿美元。假设一位投资者在一轮种子融资中以大约 100 万美元的支票获得了 15%的所有权，然后看到他们的股份在此后的每一轮融资中被稀释，数据表明他们在 C 轮融资后的所有权约为 7.2%。
