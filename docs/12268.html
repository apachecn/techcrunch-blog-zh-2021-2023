伊隆·马斯克，灾难艺术家| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/11/16/elon-musk-disaster-artist/

我们正处于 Elon Musk/Twitter 崩溃的关键时刻，这是一场狗屎秀的事实是我们的新常态，任何类似正常运作的科技公司或领导层的事情都比相反的事情更有新闻价值。但即使我们理所当然地认为马斯克的统治将继续走向混乱，也有必要退后一步，看看这位亿万富翁高管将煽动灾难作为首选做生意方法的历史。

危机导致迫切需要解决方案

马斯克一直将他的企业定位为旨在为整个人类的长期利益服务，值得称赞的是，他似乎一直真诚地相信这是真的，这是他与超人的共同特征，也是莱克斯·卢瑟的共同特征。通过这样做，马斯克利用了在灾难和外星人入侵电影中经常被用作巨大努力背后的统一动力的东西:即，如果我们面临一个存在的威胁，我们更有可能作为一个具有超人能力的统一力量来面对它。

从特斯拉开始，马斯克的业务都被定位为最终威胁人类长期生存的重大问题的解决方案。X.com，也就是后来的贝宝，可能是这一规则的例外，但事实上，它在更多方面是一个例外，这可能比其他任何事情都更具规范性。

特斯拉旨在通过成为第一家以大众市场规模有效制造电动汽车的公司，帮助人类避免气候衰退的生存威胁——特别是在碳排放的影响下。

SpaceX 是解决同一问题的不同方法——一种“让人类成为星际物种”的手段，它想象了一种未来状态，在这种状态下，特斯拉和相关的气候变化减缓努力充其量为我们赢得了额外的时间，让我们离开这个溃烂的污垢球，前往另一个(尽管更不友好？)像火星这样的天体。

马斯克还建立了不止一个，而是两个组织，目的是打击许多人认为更加遥不可及的威胁，但如果最坏的情况发生，存在的挑战也不会减少:即人工智能。为了采取一种更能影响其发展方向的人工智能方法，马斯克在 2015 年与前 YC 总统萨姆·奥特曼等人共同创立了 OpenAI 。

虽然他似乎不再直接参与该组织的日常运营(他在 2018 年离开了董事会)，但马斯克后来还创立了 Neuralink ，这是一家专注于更“实用”(只要这个词在这种背景下有任何意义)应用的初创公司，旨在创造一种可植入的大脑计算机，帮助人们增强他们的智能，以跟上人工智能最终被认为具有的超级天才能力。

作为宏观和微观激励因素的混乱

正如马斯克使用迫在眉睫的全球威胁作为其所有业务背景下发出威胁的一贯基调一样，他也利用具体的即时危机来“激励”他的员工进行快速(往往是不计后果的)变革。公平地说，马斯克似乎经常因为同样的鲁莽硬收费而引发这些危机，你可以说这让他得以闯入汽车和航空业等行业，在这些行业，既得利益和高成本通常意味着新来者不会持续很久。

马斯克拥有促成他成功的特殊智慧和天赋，但根据我对他职业生涯的长期观察，准备和规划并不在其中。一些领导人在职业生涯中寻求通过培训和自我完善来弥补自己的不足:我认为，马斯克看到了混乱产生的力量，并选择了不同的方向，经常制造灾难，促使他自己的团队发生突然转变和紧急消防演习——这也促进了他在公共政策方面的商业利益。

作家兼科技行业评论家 Paris Marx 著名地指出，马斯克对他提出的 hyperloop 技术的大部分宣传实际上是为了支付对加州高速铁路项目的支持，将他在交通领域的大部分工作框定为试图“扼杀个人汽车所有权的替代方案”，并推而广之，特斯拉。

也许马斯克将灾难作为动力的最阴险(但也可以说是最有效)的方式是让他的员工行动起来。特斯拉 Model 3 的生产流程就是一个最好的例子:马斯克自己在早期曾将其描述为“生产地狱”，他经常被发现睡在工厂的地板上，同时试图将他的员工团结在他们所面临的挑战周围。但大部分挑战都归结于马斯克的一个决定，即放弃传统的汽车装配线方法，转而采用超密集、最终不可行的自动化机器人装配单元。

从表面上看，尽管马斯克尽了最大努力，但这是一个没有得到回报的大赌注。然而，一个更具批判性的观察者可能会认为，马斯克选择了一条对他的员工不利的风险更大的道路，因为他知道，一旦他们进入紧缩模式，无论自动化游戏的结果如何，他都能够收回更多的汗水。

推特:埃隆的灾难杰作

然而，到目前为止，当涉及到引发大规模问题，然后给他手下的人增加责任时，埃隆的忍耐只能是Twitter。从一开始，当他通过裁员一半来清理门户(就连锁基础设施影响而言，这种连锁反应是可以预见的，就像灭霸消失了半个漫威电影世界一样)时，他正在播下混乱的种子。

从那以后的几周里，他似乎几乎每天都在引入新的灾难，包括 sprint 产品的推出(和回滚)，公司在家工作政策的突然逆转，以及就在今天，一个向那些仍然过度工作的人承诺的最后通牒(T2)。

马斯克显然在一个混乱的环境中茁壮成长，Twitter 是他按照自己喜欢的环境设计景观的最好例子。在这个过程中，他还透露了更多关于他的人文主义“英雄主义”的特殊品牌——这最终更像《坚不可摧》中的格拉斯或《守望者》中的奥兹曼迪斯，而不是任何一个直白的主角。
