联邦调查局是如何抓住黑客的

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/24/how-the-fbi-caught-the-breachforums-admin/amp/

图片来源:斯科特·奥尔森/盖蒂图片/盖蒂图片

周五，美国司法部宣布现已被捕的臭名昭著的黑客论坛 BreachForums 的所谓管理员帮助买卖属于“数百万美国公民和数百家美国和外国公司、组织和政府机构”的私人信息

在一份声明中，检察官确认逮捕了来自纽约 Peekskill 的 20 岁的 Conor Fitzpatrick，又名 Pompompurin。菲茨帕特里克被控一项共谋实施接入设备欺诈的罪名，如果罪名成立，最高可判处五年监禁。

为了证明 BreachForums 促进了被盗或被黑数据的买卖，FBI 卧底特工购买了五组数据:一组是从一家未具名的美国互联网托管和安全服务公司窃取的数据，其中包含大约 8000 名客户的姓名、地址、电话号码、用户名、密码哈希和电子邮件地址，以及 1900 名客户的支付卡信息；另一个数据集是从一家不知名的美国投资公司窃取的，包含至少 500 万个电子邮件地址；一个包含“大量美国人”的私人信息，包括全名、电子邮件地址、电话号码、家庭住址、出生日期、社会安全号码、驾照号码、银行名称、路由号码和账号；另一个来自同一个卖家，其中包含大约 1500 万美国人的私人信息和银行账户信息；另一组数据来自一家美国医疗保健公司。

联邦调查局收集了几件证据来逮捕庞波林。首先，他们获得了庞波普林用来访问破坏论坛的前身【RaidForums 的 IP 地址，该论坛于 2022 年 4 月被联邦调查局查封。根据菲茨帕特里克的互联网服务提供商威瑞森的说法，其中 9 个 IP 地址与菲茨帕特里克有关，这是联邦调查局特工约翰·朗米尔在 3 月 15 日的证词中写的，在菲茨帕特里克被捕前两天。

在黑客方面的一个惊人的混乱中，Longmire 写道，第二条证据来自 Pompompurin 本人。在与 RaidForums 管理员的聊天中，Pompompurin 说他注意到网站上发布的数据泄露不包括“我的一封旧电子邮件”，他在合法的数据泄露通知网站上查找了这些邮件。

尽管 pompumpurin 随后表示“(出于显而易见的原因，我不想分享我的实际电子邮件，但这封电子邮件似乎与我的情况相同):conorfitzpatrick02@gmail.com，”该代理在宣誓书中写道，该电子邮件地址确实是 pompumpurin，因为 FBI 从谷歌获得的记录显示，fitzpatrick 在那次聊天的几个月前注册了该地址。根据证词，被指控的黑客还拥有与该电子邮件地址和一个新的电子邮件地址“conorfitzpatrick2002@gmail.com”关联的谷歌支付账户，这两个账户都与菲茨帕特里克拥有的一个号码关联。

此外，该代理人写道，他从谷歌获得了更多记录，显示 conorfitzpatrick2002@gmail.com 有一个恢复电子邮件地址 funmc59tm@gmail.com 链接到一个 IP 地址，注册人的姓氏是菲茨帕特里克，电话号码不同，该代理人说，他相信这是属于菲茨帕特里克的父亲。

然后，根据证词，Pompompurin 使用几个 VPN 连接到他的 Gmail 帐户，其中一些与他在互联网上其他地方的活动重叠。

该代理人还表示，联邦调查局从加密货币交易所 Purse.io 获得了记录。该公司的记录显示，用于连接交易所的四个 IP 地址也用于连接 conorfitzpatrick2002@gmail.com Gmail 帐户和庞波普林的 RaidForums 帐户。证词称，此外，那个 Purse.io 账户是以康纳·菲茨帕特里克(Conor Fitzpatrick)的名字和“conorfitzpatrick2002@gmail.com”的电子邮件地址注册的。

据该代理称，这四个 IP 地址为 VPN 提供商所有，Pompompurin 也用这些地址连接到“conorfitzpatrick2002@gmail.com”账户。

根据证词，另一个 VPN IP 地址也被用来登录一个名为“pompompurin”的 Zoom 帐户，该帐户与一个 Riseup 电子邮件地址相关联，该电子邮件地址也被用来注册他的 RaidForums 帐户。

来自 purchase . io 的记录还显示，Fitzpatrick 的账户购买了“几样东西”,并用联邦调查局已经确定是他的电话号码将它们寄到了他的地址。此外，九个 IP 地址中有七个用于连接到 Purse.io，也用于连接到 Pompompurin 在 RaidForums 上的帐户。最后，宣誓书称，Purse.io 账户“完全由 Pompompurin 在 RaidForums 上讨论过的一个比特币地址提供资金”。

证据不止于此。根据证词，在 RaidForums 论坛活动的数据库中，联邦调查局发现 Pompompurin 从注册在 Fitzpatrick 父亲名下的 IP 地址访问了他的账户，该地址与当局之前确定的家庭地址相同。

Longmire 在证词中写道，同一个 IP 地址被用来访问与 Fitzpatrick 相关的 iCloud 账户。

此外，Longmire 指出，RaidForums 和 break forums 上使用 pompumpurin 的账户很可能是同一个人所有的，正如 pompumpurin 在 break forums 上的一篇帖子中写道:“如果你使用 RaidForums，你很可能记得我，我是那里更活跃的用户之一，”break forums 上的新 pompumpurin 账户“暗示了 pompumpurin 账户在 RaidForums 上的过去活动。”

最后，Longmire 写道，联邦调查局获得了从威瑞森获得 Fitzpatrick 实时手机 GPS 位置的许可，允许代理人观察到 Pompompurin 登录了 breakthrough forums，而他的手机位置显示他在家中。

联邦调查局还监视了菲茨帕特里克的家，同时特工注意到庞波普林的账户在论坛上很活跃。

这些证据让执法部门获得了搜查 Fitzpatrick 家的搜查令，在那里他同意与特工谈话，并“承认他是 pompumpurin 账户的用户”，而且“他拥有并管理 BreachForums，以前在 RaidForums 上运营 pompumpurin 账户。”

FBI 没有立即回应置评请求。菲茨帕特里克的律师也没有回应置评请求。

具有讽刺意味的是，当 Fitzpatrick 推出 BreachForums 时，他可能已经想到这一天会到来。在数据骑士网站的一次采访中，采访者问他，“你不认为联邦调查局关闭 RaidForums 是有原因的吗？既然知道你可能会面临同样的命运(不管是什么)，你为什么要重提此事？”

庞波普林回应道:“这并不真正困扰我。如果有一天我被捕了，我也不会感到惊讶，但正如我所说，我有一个值得信任的人，他可以在没有我的情况下完全接触到重新启动它所需的一切。”

司法部在周五的声明中表示，它还“进行了一次破坏行动，导致 breakthrough forums 离线。”当联系到 DOJ 发言人 Joshua Stueve 进行评论时，他拒绝详细说明。在本文发表时，BreachForums 无法访问，显示一条错误信息“错误的网关”，但该域名似乎仍在该网站当前管理员的控制之下。

在司法部宣布菲茨帕特里克被捕后，接替他的人，被称为 Baphomet，宣布他们将关闭论坛。

周五，在宣誓书在网上流传后，Baphomet 在一个电报频道上写了一条消息，说“我们社区目前最重要的事情是意识到联邦调查局现在被确认有权访问被攻破的数据库，”并且“在这一点上，整个文件将清楚地显示我在被攻破的整个时间里所说的话，并且你不应该相信任何人来处理你自己的 OPSEC。作为一名管理员，我从来没有做过这样的假设，其他人也不应该做这样的假设。”

Baphomet 补充说，这就是为什么“简单地将每个人都堆回同一个社区，而不考虑我们如何正确安全地前进基本上是一个死亡陷阱。”
