Afero 为其专注于安全的物联网平台 TechCrunch 筹集了 5000 万美元的 C 轮融资

原文：https://web.archive.org/web/https://techcrunch.com/2021/12/08/afero-raises-50m-series-c-for-its-security-focused-iot-platform/

随着 Samsara IPO 走向定价，现在是花多一点时间挖掘物联网市场的好时机。事实证明，这不仅仅是其中一个参与者的流动性问题。

例如，Afero 今天完成了 5000 万美元的 C 轮融资，由交叉点资本合伙人领投。新的资本筹集大约相当于 Afero 在最近一轮融资前筹集的所有资本，根据 Crunchbase 数据，这意味着这家初创公司现在手头的现金比以前任何时候都多。

在对我们没有注意到的情况下，Samsara 自己以物联网为重点的业务增长如此之大感到有点惊讶之后，TechCrunch 联系了 Afero 首席执行官乔·布里特(Joe Britt)和交叉点(Crosspoint)的休·汤普森(Andrew Britt ),就其业务以及为什么选择此时进行融资进行了更多的讨论。

阿费罗是做什么的？

根据 Britt 的说法，当他的团队将 Afero 整合在一起时，他们注意到预测表明世界上联网设备的数量将会以一个数量级的速度增长。

鉴于联网设备、传感器等的预期繁荣，Afero 的物联网平台在构建时就考虑到了安全性。这听起来可能很直观，但 Britt 认为，网络最初并没有将安全作为首要任务，这导致了一轮又一轮的围绕违规和黑客的问题。

如果我们要将我们的生活和业务与周围的互联网联系起来，安全性将是一个非常重要的问题。否则你的烤面包机或产品会被黑掉。

拥有安全工作背景的汤普森强调了这一立场对他投资该公司的兴趣有多么重要，称这家初创公司“从一开始就(在安全方面)把事情做好”是“罕见的”(感谢 TechCrunch 的扎克·惠特克(Zack Whittaker )( T15 ),我比以往任何时候都更清楚大公司被入侵的频率，这让我对以安全第一的心态进行建设的重要性的论点产生了共鸣。)

Afero 的物联网平台还包括一个软件开发工具包(SDK)，供客户公司在其上构建移动应用，以及根据客户用例进行定制的能力。

在实践中，这家初创公司与家得宝(Home Depot)等公司合作，将联网设备引入智能家居等市场。根据 Britt 和 Thompson 的说法，该公司的平台还可以支持线下项目，如标记的实物商品，以帮助跟踪供应链。

为什么现在加注？

在我接触过的所有 C 轮融资后的创业公司中，Afero 是全职员工人数最少的，只有 24 人。显然，该公司将利用其新资本进行一些招聘。它还计划投资于其走向市场的努力——在营销、销售等方面投入更多资金。

当 TechCrunch 问及该公司目前的市场足迹有多大时，Afero 的首席执行官拒绝透露具体数字，在试图找到一种不太精确地解释一般规模的方法后，他只是说世界上有“很多”Afero 连接设备。通常我们会抱怨缺少硬数据，但考虑到初创公司的客户可能不希望它披露他们有多少设备安装在 Afero 的平台上，这是有一定道理的。

随着时间的推移，当 Afero 拥有更广泛的客户群时，我们会期待更多的特殊性。

几年前，我们可能会怀疑 Afero 正在应对的市场规模，但 Samsara 的增长改变了我们的总体看法。

这家即将上市的公司的平台从现实世界的商业运营中收集物联网数据，允许在来自传感器的数据基础上创建应用层。Samsara 上个季度 1.138 亿美元的收入证明了市场对物联网平台更普遍的需求。

Afero 最著名的合作伙伴关系更多的是面向消费者——家得宝的 HubSpace 系列面向公众——而不是 Samsara 已经建立的产品，所以两者现在不是直接的竞争对手。我们并不是想说阿费罗是在轮回之后到来的，或者相反。至少现在还没有。

展望未来，我们很想知道 Afero 在未来几个季度能吸引多少新客户公司，以及它多快能扩展到可以分享更难的指标。
