GitHub 关闭包含 Twitter 源代码的知识库

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/27/github-takes-down-repository-containing-twitters-source-code/amp/

图片来源:布莱斯·德宾/TechCrunch

在社交网络 DCMA 提出关闭请求后，GitHub关闭了一个名为“FreeSpeechEnthusiast”的用户的知识库，其中包含了 Twitter 的专有源代码。这个用户名显然是对推特所有者埃隆·马斯克的讽刺，他多次声称是一个“言论自由绝对主义者”。

周五，Twitter 在北加州地区法院提交了一份请愿书，要求 GitHub 删除代码，并帮助它找到肇事者。传票要求 GitHub 披露与“FreeSpeechEnthusiast”相关的“姓名、地址、电话号码、电子邮件地址、社交媒体个人资料和 IP 地址”。

这一进展发生在 3 月 31 日之前，届时马斯克据称将开源 Twitter 用于推荐推文的算法，这是他已经承诺了一段时间的事情。

尚不清楚 Twitter 的哪部分源代码在 GitHub 上泄露，也不清楚泄露了多长时间。GitHub 的 DCMA 撤下博客只说它撤下了包含“Twitter 平台和内部工具的专有源代码”的知识库

代码托管网站没有说明在该公司关闭它之前，是否有任何用户能够访问这个存储库。我们向 GitHub 征求了意见，如果有回音，我们会更新这个故事。

Twitter 可能也担心没有上传到 GitHub 的代码副本。根据《纽约时报》的一篇报道，Twitter 的一项内部调查显示，对泄密事件负有责任的人已于去年离开公司。这篇报道还暗示，这家社交网络的高管最近才知道泄密事件。

自去年马斯克接管以来，该公司一直处于动荡时期。最近的报道 暗示这位特斯拉 CEO 现在对 Twitter 的估值为 200 亿美元，不到他为这家社交网络支付的 440 亿美元的一半。据纽约时报报道，马斯克给员工写了一封电子邮件，宣布一项新的股票补偿计划，称 Twitter 有一天可能价值 2500 亿美元。

马斯克已经采取了激进的措施来削减 Twitter 的成本，包括大规模裁员，并重新启动了一个新的订阅计划，该计划将验证作为福利之一。根据分析公司 Sensor Tower 的数据，Twitter 已经从这项新服务中获得了 1100 万美元。

在最近的一次会议上，马斯克表示，用户在 Twitter 上的时间没有得到很好的货币化。

“2.5 亿(月活跃用户)每天花在 Twitter 上的平均时间大约是半小时左右。所以我们所拥有的是——我认为最有趣的是——推特上每天大约有 1.2 亿到 1.3 亿小时的人类关注，”他说。

“平均每一天，我认为这是一个非常有趣的点，这是令人吃惊的货币化程度，因为你必须说这种注意力有多有价值每天阅读的人的 1 亿到 1.3 亿小时的人类注意力，所以这些通常是世界上最聪明的人，世界上最有影响力的人。”

不出所料，当我们联系 Twitter 时，我们得到了一个便便表情符号。
