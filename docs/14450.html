拥有 170 万用户的游戏化网络安全培训平台 Hack The Box 筹资 5500 万美元 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2023/01/11/hack-the-box-a-gamified-cybersecurity-training-platform-with-1-7m-users-raises-55m/amp/

在计算机黑客世界中，长期以来一直存在着这样一种分歧，一种是采取恶意方法来破解系统的人，另一种是使用相同技术来了解系统的漏洞，帮助修复漏洞，同时与恶意行为者进行斗争的人。今天， Hack The Box 是一家初创公司，它建立了一个平台，以游戏化的方式帮助培养更多的后一种群体，在积累了 170 万用户后，它宣布提供 5500 万美元的资金来扩大业务。

这笔资金由凯雷牵头，帕拉丁资本集团、奥塞奇大学合伙人、马拉松风险投资、Brighteye Ventures 和 Endeavor Catalyst Fund 也参与了进来。

这家英国初创公司目前没有披露估值。但在某些背景下，根据 PitchBook 的说法，这家总部位于英国、但在纽约设有办公室、创始地在希腊的初创公司——它在希腊也有一个办公室(几天前，似乎有关于这轮融资的传言在希腊泄露)——自 2017 年成立以来，已经筹集了超过 2400 万美元(其中约 1500 万美元为股权:该公司表示，现在已经筹集了约 7000 万美元)。它最近一次估值是在 2021 年，在它筹集了 1060 万美元之后，当时的估值是非常适中的 5200 万美元。

“适度”是因为该公司取得的成就令人印象深刻。使用该平台的 170 万社区成员既包括自己加入 HTB 学习技能并获得认证的个人，也包括约 1，500 家企业、大学、政府和其他组织，他们派遣团队到 HTB 接受考验。

该公司表示，它目前在 300 多台机器上运行着大约 450 个“黑客实验室”。与 Kahoot 这样的公司(确切地说，它在一个非常不同的环境中工作，即 K-12 教育和企业培训)类似，HTB 的想法是，它的学习环境是围绕游戏化、虚拟角色模拟和叙事场景建立的，这些场景旨在将用户投入到模仿不同且日益复杂的经典网络黑客的环境中。它还有一个“专业实验室”层，采用典型的网络配置，如活动目录或完全修补的环境，以围绕常见的企业工具和场景针对不同的攻击和方法进行测试和培训。渗透测试、错误配置和逃避端点保护都是用户面临的情况。

最重要的是，除了为个人和团队提供培训平台之外，它还提供了一个职业平台，那些希望雇用道德黑客或寻找工作的道德黑客可以在这里联系。

HTB 不是第一家也不是唯一一家围绕游戏化环境开展网络培训的公司。美国网络游戏，与美国政府组织联合开发，是一个大众玩家环境，用于识别和训练可能的白帽黑客。(它也有职业服务。)HTB 实际上是美国网络游戏的发起者和支持者之一。其他如 SafeTitan、Phished 和 Immersive Labs 为技术团队和员工提供了一系列方法来帮助提高意识。后者目前还不属于 HTB 的范畴，尽管这是一个很明显的领域，它可能会发展壮大。

首席执行官兼联合创始人哈里斯·皮拉里诺斯(Haris Pylarinos)在一份声明中表示:“我们的使命是通过高度参与的黑客体验，培养创新思维，创造并连接网络就绪的人类和组织。”。“网络游戏已经发生了变化，面对不断增加和不断演变的复杂攻击浪潮，防御、反应和恢复态势不再适用。需要一种新的主动进攻和防御方法来打击网络犯罪分子，而不是等待被打击。从个人安全专家到公司，这意味着采用“黑客思维”，学会像攻击者一样思考和行动。这就是我们通过黑盒子培养的那种心态。”

目前，我们在 TechCrunch 上经常提到的一个事实是，在某些技术领域，融资变得更加困难了。HTB 是继续受到关注的类别之一，尤其是因为安全漏洞肯定没有随着经济的其他部分而放缓。这就是为什么投资者会支持该领域那些正在扩大规模、迄今为止外部资本相对较少的公司的原因之一。

“对安全和 IT 专业人员的需求从未像现在这样大。凯雷董事康斯坦丁·博耶(Constantin Boye)在一份声明中表示:“全行业人才短缺，网络威胁呈指数级增长，因此专业人士和组织保持一流的安全实践非常重要。”。“Hack The Box 是在完全游戏化和直观的环境中不断提供新鲜和有针对性的培训和技能提升内容的先驱，使个人和组织能够解决现实世界的黑客问题。我们对 Hack The Box 的下一阶段发展感到兴奋，并为成为这一旅程的一部分感到自豪。”
