ChatGPT 是网络安全威胁吗？TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2023/01/11/chatgpt-cybersecurity-threat/amp/

图片来源:Korakrich suntornites/Getty Images

自 11 月首次亮相以来，ChatGPT 已经成为互联网上最受欢迎的玩物。人工智能驱动的自然语言处理工具迅速积累了超过 100 万用户，他们使用基于网络的聊天机器人做任何事情，从生成婚礼致辞和嘻哈歌词到撰写学术论文和编写计算机代码。

ChatGPT 的类似人类的能力不仅在互联网上掀起了风暴，而且也让许多行业处于紧张状态:纽约的一所学校禁止了 ChatGPT，因为担心它可能被用来作弊，文案已经被取代，有报道称谷歌对 ChatGPT 的能力感到非常震惊，以至于发布了“红色代码”，以确保公司搜索业务的生存。

长期以来，网络安全行业一直对现代人工智能的潜在影响持怀疑态度，该行业似乎也在关注 ChatGPT 可能被资源有限、技术知识为零的黑客滥用的问题。

就在 ChatGPT 首次亮相几周后，以色列网络安全公司 Check Point 展示了基于网络的聊天机器人如何在与 OpenAI 的代码编写系统 Codex 配合使用时，可以创建一封能够携带恶意有效载荷的钓鱼电子邮件。check Point threat intelligence group 经理 Sergey Shykevich 告诉 TechCrunch，他认为像这样的用例表明，ChatGPT 具有“显著改变网络威胁格局的潜力”，并补充说，它代表着“在日益复杂和有效的网络能力的危险演变中又向前迈出了一步。”

TechCrunch 也能够使用聊天机器人生成看起来合法的钓鱼邮件。当我们第一次要求 ChatGPT 制作一封钓鱼邮件时，chatbot 拒绝了我们的请求。“我没有被编程来创建或推广恶意或有害的内容，”一个提示回应道。但是稍微改写一下请求，就可以让我们轻松绕过软件的内置护栏。

TechCrunch 采访的许多安全专家认为，ChatGPT 编写听起来合法的钓鱼电子邮件的能力——勒索软件的最大攻击载体——将使 chatbot 受到网络犯罪分子的广泛欢迎，特别是那些母语不是英语的人。

Sophos 的首席研究科学家 Chester Wisniewski 表示，很容易看到 ChatGPT 被滥用于“各种社会工程攻击”，其中犯罪者希望看起来用更令人信服的美国英语写作。

Wisniewski 告诉 TechCrunch:“在基本层面上，我已经能够用它编写一些很棒的网络钓鱼诱饵，我希望它可以用来进行更现实的互动对话，用于商业电子邮件妥协，甚至通过 Facebook Messenger、WhatsApp 或其他聊天应用进行攻击。”

“实际上，获得恶意软件并使用它只是成为底层网络罪犯的一小部分。”格鲁格，安全研究员

聊天机器人能够写出令人信服的文字和真实的互动，这种想法并不牵强。Darktrace 的威胁研究负责人 Hanah Darley 告诉 TechCrunch 说:“例如，你可以指示 ChatGPT 假装一个 GP 手术，它会在几秒钟内生成逼真的文本。”"不难想象威胁分子会如何利用这一点来增强力量."

Check Point 最近也对聊天机器人帮助网络罪犯编写恶意代码的明显能力发出了警告。研究人员表示，他们目睹了至少三起没有技术技能的黑客吹嘘他们如何利用 ChatGPT 的人工智能达到恶意目的的例子。一名黑客在一个黑暗的网络论坛上展示了 ChatGPT 编写的代码，据称这些代码窃取了感兴趣的文件，对它们进行压缩，然后通过网络发送出去。另一个用户发布了一个 Python 脚本，他们声称这是他们创建的第一个脚本。Check Point 指出，虽然代码看起来是良性的，但它可以“在没有任何用户交互的情况下，很容易地被修改以完全加密某人的机器。”Check Point 表示，同一名论坛用户此前曾出售被黑客入侵的公司服务器和窃取的数据。

会有多难？

安全研究员、Picus Security 的联合创始人 Suleyman Ozarslan 博士最近向 TechCrunch 演示了 ChatGPT 是如何被用来编写世界杯主题的钓鱼诱饵和编写针对 macOS 的勒索软件代码的。Ozarslan 要求聊天机器人为 Swift 编写代码，Swift 是一种用于开发苹果设备应用程序的编程语言，它可以在 MacBook 上找到微软 Office 文档，并通过加密连接将其发送到网络服务器，然后对 MacBook 上的 Office 文档进行加密。

“我毫不怀疑 ChatGPT 和其他类似工具将使网络犯罪民主化，”Ozarslan 说。“人们已经可以在暗网上买到‘现成的’勒索软件代码，这已经够糟糕了；现在几乎任何人都可以自己创造。”

毫不奇怪，关于 ChatGPT 编写恶意代码的能力的消息让整个行业都皱起了眉头。也看到一些专家开始消除人们的担忧，即人工智能聊天机器人可能会把想成为黑客的人变成成熟的网络罪犯。在乳齿象上的一篇帖子中，独立安全研究员 Grugq 嘲笑了 Check Point 的说法，即 ChatGPT 将“对不擅长编码的网络罪犯进行超级指控”

“他们必须注册域名并维护基础设施。他们需要用新内容更新网站，并测试几乎不能工作的软件在稍微不同的平台上能否继续工作。他们需要监控其基础设施的健康状况，并检查新闻中正在发生的事情，以确保他们的活动不是在一篇关于“最令人尴尬的 5 大网络钓鱼骗局”的文章中。“实际上，获得恶意软件并使用它只是成为底层网络罪犯的一小部分。”

一些人认为，ChatGPT 编写恶意代码的能力带来了一个结果。

“防御者可以使用 ChatGPT 生成代码来模拟对手，甚至自动执行任务以使工作变得更容易。它已经被用于各种令人印象深刻的任务，包括个性化教育、起草报纸文章和编写计算机代码，”F-Secure 的威胁情报主管劳拉·坎卡拉说。“但是，应该注意的是，完全信任 ChatGPT 生成的文本和代码的输出可能是危险的——它生成的代码可能有安全问题或漏洞。生成的文本也可能有明显的事实错误，”Kankaala 补充道，对 ChatGPT 生成的代码的可靠性提出了质疑。

ESET 的 Jake Moore 表示，随着技术的发展，“如果 ChatGPT 从其输入中了解到足够多的信息，它可能很快就能分析潜在的攻击，并提出积极的建议来增强安全性。”

不仅仅是安全专业人士对 ChatGPT 将在未来的网络安全中扮演什么角色感到矛盾。当我们向聊天机器人提出这个问题时，我们也很好奇 ChatGPT 会怎么说。

“很难准确预测 ChatGPT 或任何其他技术在未来将如何使用，因为这取决于它如何实现以及使用它的人的意图，”聊天机器人回答道。“最终，ChatGPT 对网络安全的影响将取决于如何使用它。意识到潜在的风险并采取适当的措施来降低风险是非常重要的。”
