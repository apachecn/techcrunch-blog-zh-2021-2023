筹资阶段与美元价值无关，而是与风险有关

对于快速的估值攀升，思考一下，“现在最大的风险是什么，我如何消除它？”

你可能听说过pre-seed、seed、A 轮、B 轮等等。这些标签通常不是非常有用，因为它们没有明确的定义——我们已经看到非常小的首轮融资和巨大的种子前融资。每一轮的决定性特征与其说是换手了多少钱，不如说是公司的风险有多大。

在你的创业之旅中，有两种动力同时在起作用。通过深入了解它们——以及它们之间的联系——你将能够更好地理解你的筹资之旅，以及如何在你发展和进步的过程中思考创业道路的每一部分。

总的来说，从广义上讲，融资通常如下:

  • 4f:创始人、朋友、家人、傻瓜:这是进入公司的第一笔钱，通常只够开始证明一些核心技术或商业动态。在这里，该公司正试图建立一个 MVP 。在这些回合中，你经常会发现不同复杂程度的天使投资人。
  • 前期种子期:令人困惑的是，这通常与上述相同，只是由机构投资者(即家族办公室或专注于公司早期阶段的风投公司)完成。这通常不是一轮“定价轮”——公司没有正式的估值，但筹集的资金是可兑换的或安全的票据。在这个阶段，公司通常还没有产生收入。
  • 这通常是机构投资者向一家已经开始证明其活力的公司投入大量资金。创业公司将有一些业务开始运行，可能有一些测试客户，一个测试产品，一个门房 MVP，等等。它不会有增长引擎(换句话说，它还没有一个可重复的方式来吸引和留住客户)。该公司正致力于积极的产品开发，并寻找适合市场的产品。有时这一轮是定价的(即投资者协商对公司的估值)，也可能是不定价的。
  • 首轮融资:这是一家公司筹集的第一轮“增长轮”。它通常会在市场上推出一种产品，为客户提供价值，并正在以一种可靠、可预测的方式向客户获取投入资金。该公司可能即将进入新的市场，扩大其产品供应或追求新的客户群。A 轮融资几乎总是“定价”，给公司一个正式的估值。
  • B 轮及以后:在 B 轮中，一家公司通常会认真投入比赛。它有客户、收入和一两款稳定的产品。从 B 系列开始，你有 C 系列、D 系列、E 系列等等。轮和公司变得更大。最后几轮通常是为公司扭亏为盈(实现盈利)、通过首次公开募股(IPO)或两者兼而有之做准备。

在每一轮融资中，一家公司都会变得越来越有价值，部分原因是随着它找到了自己的增长机制和商业模式，它会获得越来越成熟的产品和更多的收入。沿着这条路，公司也以另一种方式发展:风险降低。

最后一点对你如何看待你的筹款之旅至关重要。你的风险不会随着你的公司变得更有价值而降低。随着风险的降低，公司变得更有价值。你可以利用这一点，通过设计你的融资轮，明确消除你公司“最可怕”的东西的风险。

让我们仔细看看风险出现在创业公司的什么地方，以及作为创始人，在公司存在的每个阶段，你可以做些什么来尽可能地消除风险。

你公司的风险在哪里？

风险以多种形式出现。当你的公司处于创意阶段时，你可能会和一些创始人与市场非常契合的联合创始人在一起。你已经发现市场存在问题。您早期的潜在客户访谈都同意这是一个值得解决的问题，并且有人——理论上——愿意花钱来解决这个问题。第一个问题是:这个问题有可能解决吗？
