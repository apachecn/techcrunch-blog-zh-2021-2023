You.com 筹集 2500 万美元用于人工智能搜索引擎

原文：https://web.archive.org/web/https://techcrunch.com/2022/07/14/you-com-raises-25m-to-fuel-its-ai-powered-search-engine/

人们对替代搜索引擎的需求越来越大。至少，这是 Salesforce 前首席科学家理查德·索彻(Richard Socher)喜欢提出的论点的核心。2020 年，Socher 共同创立了 You ，这是一个搜索引擎，它使用 AI 来理解搜索查询，对结果进行排序，并将查询解析为不同的语言(包括编程语言)。你汇总来自网络的信息，并提供内置应用程序，如 Twitter 的搜索工具，让用户无需离开结果页面就能完成任务。

他的话似乎有些道理。Socher 声称 You 拥有成千上万的用户，上个月注册人数增长了 70%,独立搜索量逐月增长了 30%。虽然这与拥有数亿用户的世界上最受欢迎的搜索引擎(即谷歌、必应)相比相形见绌，但 Socher 将注意力吸引到了你的保留率上。50%将你设为默认搜索引擎的人会在事后继续使用它。

这些数字正合投资者的胃口。今天，您完成了由 Radical Ventures 牵头的一轮 2500 万美元的融资，Salesforce 首席执行官马克·贝尼奥夫(Marc Benioff)的 Time Ventures、Breyer Capital、Norwest Venture Partners 和 Day One Ventures 参与了融资。这使得这家初创公司的总融资额达到了 4500 万美元，Socher 表示，这笔资金将用于开发“优质功能”，并与外部开发者合作，在你的搜索结果页面上“显示更多有用的可操作应用”。

“我们设想 You.com 成为一个开放的搜索平台，允许其他人在我们创造的所有搜索技术的基础上进行开发。数据透明、用户定制、摘要、隐私和最先进的搜索是我们平台的基础，从长远来看，我们通过让世界共同构建下一个搜索体验来击败谷歌，”Socher 在一次电子邮件采访中告诉 TechCrunch。“谷歌是一个单一的、垄断性的搜索引擎，它是封闭的，最终为了服务于其真正的目的——广告——而将人工智能武器化来对付用户。我们正在将 You.com 打造为一个开放的搜索平台，强调通过 You.com 应用直接服务于用户需求，而不是用广告轰炸用户。”

Socher 之前创立了 MetaMind ，这是一家人工智能初创公司，于 2016 年被 Salesforce 收购。在 Salesforce 时，索赫尔帮助建立了该公司的 庞大的爱因斯坦人工智能平台，并与你的第二位联合创始人布莱恩·麦肯(Bryan McCann)合作进行自然语言处理研究。

毫不奇怪，鉴于 Socher 的数据科学背景，你在整个搜索体验中自由地散布人工智能功能。作为今天新推出的一项功能，YouCode 可以根据搜索查询生成类似 GitHub 的 Copilot 的代码。最近首次亮相的 YouWrite 由 OpenAI 的 GPT-3 驱动，可以被提示写文章、博客帖子甚至样板信。

Socher 认为应用程序，而不是广告，是 You 增长的核心——传统搜索引擎公式的一个转变。(广告占谷歌收入的大部分——2021 年为 2567.3 亿美元。)虽然它没有排除引入广告的可能性，但 You 提供了一种“隐私模式”，不存储查询、偏好、IP 地址或位置数据，并将发送给你的第三方合作伙伴的查询数据匿名化。

“我们的隐私模式提供了所有搜索引擎中最隐私的搜索体验。用户也可以要求删除他们的数据，并在如何[和]使用什么数据方面拥有完全的透明度，”Socher 声称。“在最大程度上，我们相信未来在应用而不是广告，经济将越来越成为展示多于讲述的经济。”

当然，你已经把它的钱用在了它的嘴上，在像前面提到的 YouCode 这样的体验上投入了大量的开发。除了生成代码之外，YouCode 还可以从 Stack Overflow 和 GitHub 等源代码中找到代码片段，并过滤与特定开发环境和软件堆栈相关的结果。一个特殊的预览面板显示来自在线发布的文档的顶级信息，而内置工具可以验证 JSON 文件，并以十六进制、RGB 和 HSV 为颜色生成代码。

索彻说，虽然你的大部分应用程序是内部开发的，但下一步是将平台暴露给第三方。在未来，你将开放对其数据集和定制搜索技术的访问，以便开发人员可以在他们的内部数据的基础上进行构建。到那时，你可能会开始收费，比如对特性的 API 访问，“高级”和“个性化”的文本和代码生成，以及通过内部开发人员知识库的现成搜索支持。

你的商业模式能否胜出，还有待观察。其他低调的搜索引擎，如 Neeva ，已经走上了不同的盈利道路，对搜索结果的访问收费。还有一些人一直在努力寻找可靠的收入来源——由前谷歌工程师创建的 Cuil ，在运营了两年多之后，于 2010 年关闭之前筹集了 3300 万美元的资金。

在任何情况下，按照索彻早先的观点，广告不太可能是搜索引擎初创公司赚钱的首选方式。在企业层面，隐私转发功能，如苹果的应用程序跟踪透明度可能会破坏许多广告的幕后机制，而美国和海外的监管机构正在关注对广告商为定向目的收集的数据量的限制——使某些广告产品不那么有吸引力。

“我们希望成为一个开放的互联网入口，而不是一个通过广告网络向所有公司征税的封闭门户，广告网络最终将成为信息共享进展的瓶颈，”索切尔说，并对 You.com 的主要竞争对手进行了挖苦。“一个更好的搜索引擎是可能的——一个更好的隐私，对程序员等专业人士更好的结果，更少的广告，更多的社交结果，更少的 SEO 网站，更多的选择和排名中的人的代理。”
