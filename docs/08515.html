人力资源自动化平台 Omni 希望成为“东南亚的涟漪”

原文：https://web.archive.org/web/https://techcrunch.com/2022/07/24/hr-automation-platform-omni-wants-to-be-the-rippling-of-southeast-asia/amp/

图片来源:Jimmy fam/Getty Images

Omni 希望成为一个人力资源平台来管理他们所有人——或者至少是所有与人力资源相关的任务。该软件使人力资源团队能够数字化员工记录，自动化员工入职和休假管理等管理任务，并整合来自不同系统的员工数据。总部设在新加坡，目前活跃在新加坡和印度尼西亚，并计划推出后，在其他东南亚市场本地化的就业法规。

这家初创公司今天宣布，它正在走出隐形模式，在由阿尔法 JWC 风险投资公司和 Picus Capital 共同牵头的种子前一轮超额认购 240 万美元，参与方包括 FEBE 风险投资公司、Basis Set 风险投资公司、Ratio 风险投资公司和 Horizons 风险投资公司的弗朗西斯康。它还包括天使投资者的投资，包括美国人力资源软件公司和终极软件公司的前高管。

Omni HR 于 2022 年 3 月试运行，已经被几家公司使用，包括印度尼西亚投资应用程序 Ajaib 。这笔资金将用于为 Omni 增加更多功能，包括第三季度的招聘模块和年底的绩效提升模块。

该公司由高盛(Goldman Sachs)前高管布莱恩·叶(Brian Ip)和数据工程师 YC·陈(Michael Chan)于 2021 年创立。Ip 告诉 TechCrunch，他之前曾在高盛增长基金从事软件投资，并研究了许多人力资源技术交易，这是他和 Chan 最初了解该行业的方式。

“通过研究和与最终用户的交谈，我们意识到人力资源软件是一个需要大量本地化的类别，目前还没有针对东南亚的成功产品，”Ip 说，并补充说大多数本地解决方案仅解决有限的功能，如工资单。

但是，Chan 和 Ip 采访的大多数人力资源团队都想要一个一体化的解决方案。许多人仍在使用电子表格或基本的薪资软件。他们手动完成的工作可以通过 Omni 实现自动化，包括新员工入职、招聘员工、绩效评估、收集员工 id 等文档以及为内部管理准备人力资源报告。

“从战略的角度来看，我们认为使这个创业机会更加有趣的是，我们不认为人力资源软件是一个仅供人力资源部门使用的筒仓工具，”陈说。"相反，我们把它看作是员工信息的‘记录系统’。"

公司内的几乎每个应用程序或业务功能，包括软件、设备、办公室管理和财务，都可以连接到 Omni，将其转变为软件基础架构层。

就竞争而言，Chan 说他看到了两个类别:本地薪资软件和从海外进口的软件。他补充说，工资软件的缺点是它们只提供工资计算的基本管理功能，并且不可扩展。它们也没有绩效评估、招聘、入职和员工文档管理的功能。

另一方面，进口的人力资源软件没有本地化，这意味着它们缺乏东南亚国家的工资模块、本地客户支持等功能，“有时甚至连请假跟踪或考勤管理等模块都不够灵活，无法适应一个市场的政策，”Ip 说。

他补充说,ripping和其他像 Gusto 和即这样的美国顶级人力资源平台目前在美国之外无法使用。“我们相信，即使他们在某个时候确实进行了国际扩张，本地化要求和地理重点将允许我们建立一个强大的护城河。”

针对每个市场进行本地化可能相当复杂。不同国家的人力资源经理需要收集不同的员工信息。例如，在新加坡，员工提供他们孩子的出生证明，这样当他们休育儿假时，公司可以用这些证明申请政府报销。另一方面，印尼的公司收集多种形式的身份信息，包括 KTD(居民卡)、KK(家庭卡)和 NPWP(税号)。

每个国家也有不同的工作流程。Ip 说，在新加坡，永久员工的试用期可以“延长”，但在印度尼西亚最多只允许三个月，而且不能延长或续签。

薪资计算也因国家而异，包括税收、养老金和其他法定扣缴等因素。休假规则也各不相同。对于招聘，Omni 可以通过连接当地的工作委员会而不是以美国为中心的工作委员会进行本地化。

Ip 说，新加坡和印度尼西亚被选为 Omni 的第一个市场，因为这家初创公司的初始客户群是科技和科技相关垂直行业的公司，特别是其他风投支持的公司。他补充道，“就软件/云的采用和支出意愿而言，新加坡可能是东南亚最成熟的市场。印尼是东南亚最大且快速增长的市场机遇之一。”
