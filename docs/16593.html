亚马逊向开发者开放其低带宽、长距离的人行道网络

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/28/amazon-opens-its-low-bandwidth-long-range-sidewalk-network-to-developers/

早在 2019 年，亚马逊就宣布了“T2 人行道”这一使用 900 兆赫频谱连接物联网设备的低带宽长距离无线网络。它通过将这些设备连接到亚马逊自己的 Echo 和 Ring 设备上来做到这一点，这两个设备共享所有者的一小部分带宽。理想情况下，这意味着人行道将能够连接蓝牙或无线信号无法覆盖的设备。然而直到现在，只有为数不多的 T4 开发商能够为网络构建应用程序。但今天这种情况正在改变。该公司现在正在运送软件和硬件开发工具包，以及一个易于使用的测试工具包，以测试您周围的人行道连接情况。

使用新的测试套件，开发人员将能够在地图上检查他们的本地信号强度，以更好地了解他们的设备在开始开发产品之前是否能够连接到网络。

亚马逊给我发了一个戒指品牌的设备来试用。开始使用它只需要打开电源(它确实有一个移动使用的电池)并登录亚马逊基于网络的人行道覆盖服务。从那里，您可以快速查看启用了全球定位系统的测试套件中的所有原始数据，并访问亚马逊的全美覆盖地图。

为了确保用户隐私，覆盖地图仅显示了 900 米乘 900 米见方区域内的覆盖范围。您将能够看到自己设备的确切位置，但无法看到报告到覆盖范围图中的其他设备。

亚马逊设备与服务高级副总裁戴夫·莱姆(Dave Limp)表示:“我们已经快速构建了一个覆盖 90%以上美国人口的长距离、低带宽网络，这是对开发者进行测试的公开邀请。”。“许多类型的连接设备都受到 Wi-Fi 范围和蜂窝技术成本的限制，这阻碍了连接环境传感器、泄漏检测器和智能锁等设备的能力。人行道旨在提供一种安全、低成本的方式来发明和连接一系列全新的设备，我们迫不及待地想看看开发人员会构建什么。”

人行道覆盖图。影像学分:亚马逊

对于软件开发套件，亚马逊与北欧半导体、硅实验室、德州仪器和模块供应商 Quectel 合作。这些公司还将提供硬件开发套件，Quectel 推出了 Sidewalk 连接模块，允许开发人员快速将支持 Sidewalk 的设备投入生产。亚马逊也在为 iOS 和 Android 推出移动 SDK，以帮助开发者将 Sidewalk 构建到他们的移动应用中。该公司还推出了一款应用程序，帮助开发者在现场调试和排除设备故障。

值得注意的是，Sidewalk 还与 AWS 的物联网核心服务进行了原生集成，允许企业连接和管理他们的物联网设备车队。“AWS 物联网核心和亚马逊 Sidewalk 的整合标志着开发者、制造商和客户的一个重要里程碑，简化了基于亚马逊 Sidewalk 的物联网解决方案的设计、连接和部署，”亚马逊网络服务的物联网副总裁 Yasser Alsaied 说。“现在，借助亚马逊 Sidewalk 的 AWS 物联网核心，开发人员可以访问 200 多种 AWS 服务，在高度可靠、安全和免费连接的无线网络基础上构建可扩展的解决方案。”
