Heartex 的 Label Studio 让为 ML 标记音频变得更加容易

Heartex,资金雄厚的机器学习数据标签初创公司，今天对其平台进行了一次重大更新，将通过使其标签工作室的商业和开源版本中的注释音频文件对其用户来说更加容易，为其工具开辟新的用例。这些更新包括一个新的用户界面，针对音频文件进行了优化，能够注释较长的文件(长达两个小时)，以及内置注释工具中的毫秒级控制。现在还有一个新的渲染引擎用于显示波形。在之前的更新中，Heartex 还增加了其工具功能，如逐帧视频对象跟踪和全面修改的注释 UI。

该公司表示，目前约有 15 万用户依赖 Label Studio，迄今已创建了超过 9500 万条注释。

图片来源: Heartex

“当我们最初考虑建立数据标签解决方案时，我们做了大量的数据科学家采访，”Heartex 联合创始人兼首席执行官迈克尔·马柳克告诉我。“我们发现，他们中的许多人说他们没有足够的灵活性来使用现有的工具。有一个图像工具，但是你只能在屏幕上放一个图像。我们的音频有一个工具，但它在用例方面是非常硬编码的。我们认为，对于 Label Studio 来说，当数据科学家在他们的头脑中有一个问号时，“它支持我的用例吗？”答案总是肯定的——它必须支持您的使用案例。"

当然，要标记数据，你只需要成为主题专家，而不是软件工程师。Label Studio 背后的理念是让几乎任何人都可以标记你的数据。“我们认为，每一家人工智能公司都将转型为数据标签公司或数据集开发公司，”Malyuk 说。他说，本质上，Heartex 希望让 Label Studio 成为数据集开发的事实上的 IDE。

展望未来，他指出，该公司计划对其用户社区进行大量投资，并计划明年举办首届 Label Studio 用户大会。
