Thirdweb 为其 web3 开发套件筹集了 2400 万美元

原文：https://web.archive.org/web/https://techcrunch.com/2022/08/24/thirdweb-raises-24m-at-a-160m-valuation-from-haun-ventures-shopify-and-coinbase-for-its-web3-development-kit/amp/

图片来源: Getty Images

有很多关于“web3”的宣传，但现实是它仍然是一个很大程度上新生和分散的概念，这导致 web3 初创公司建立平台，更容易获得大量关注。在最新的发展中，一家名为 thirdweb 的初创公司已经筹集了 2400 万美元，A 轮融资对该初创公司的估值为 1.6 亿美元。thirdweb 创建了一个开发工具包，使构建和推出区块链游戏、NFTs、DAOs、marketplaces 等 web3 产品变得更容易。

Thirdweb 的业务在伦敦，但总部设在旧金山，它计划利用这笔资金继续增强其开发者工具包，目前涵盖智能合同、分散登录、发布工具等领域的约 10 个功能，以扩大对更广泛的区块链的支持，通过招聘和潜在的收购来吸引更多用户和壮大团队，所有这些都有助于让 web3 变得更加主流。

Thirdweb 上线仅 9 个月，但它一直在蓬勃发展:联合创始人 Steven Bartlett 告诉 TechCrunch，迄今为止约有 55，000 名开发者——从独立创作者到 Afterpay 和纽约时装周等组织——使用其框架开发了一系列 NFT 商品、Dao、游戏和其他应用程序。截至本周，大约 150，000 份智能合同已经在六个区块链部署。(如果你想知道，他说 Polygon 是目前最受欢迎的:“更便宜的费用是最直接的答案，”他说。)

这些项目每周总共产生 150 万美元的收入，尽管 thirdweb 没有从中分一杯羹。Bartlett 说，目前 thirdweb 上没有收费或佣金，一个例外是 thirdweb 的一些企业客户支付费用，作为他们采购政策的一部分。

今天宣布的资金也显示了一些势头。这轮融资由 Haun Ventures(由人脉广泛的前 a16z 合伙人凯蒂·霍恩创建的公司)以及战略支持者比特币基地风险投资公司和 Shopify 共同牵头。

比特币基地已经与 thirdweb 合作——例如后者为创作者提供开发平台为比特币基地 NFTs 建立 NFTs 这将使两者更加紧密地联系在一起。

与此同时，Shopify“将我们视为投资，同时也是合作伙伴，”Bartlett 说。他补充说，在工作中有一个集成，thirdweb 的工具包将提供给希望引入 web3 产品的 Shopify 客户，以补充他们的 Shopify 支持的店面。

另一个战略靠山 Polygon 也在参与此轮。

这三人加入了 third web 500 万美元种子期投资者的长长名单(包括马克·库班、加里·维纳查克等)。

Bartlett 说，thirdweb 正在解决的市场缺口是他和他的联合创始人 Furqan Rydhan 在他们自己开发区块链应用程序时发现的。

“我们看到从头开始构建应用程序是多么困难的工作，”Bartlett 说。他将可用于加快区块链工作的工具的缺乏与我们今天所知的早期网络进行了比较，当时人们还必须从头开始编写网站及其所有功能，而在现代，人们可以通过几行代码和一个 API 来集成支付流程，该 API 连接到由其他人构建和操作的技术。或者，实际上，使用像 WordPress 或 Wix 这样的引擎构建一个完整的站点。

机会很明显:为 web3 开发同样的工具。

“我们意识到，我们可以把自己开发的应用程序作为产品发布，”Bartlett 说。“我们希望为区块链开发者提供与 Web2 相同的易用基础设施和工具。”

联合创始人过去的经历可能也在其中发挥了作用。巴特利特喜欢在机会刚刚出现时抓住它们:他是社交媒体的早期采用者和支持者，尤其是在品牌认识到社交媒体战略是必须的之前很久，品牌就利用它来发展。

与此同时，Rydhan 在开发基于 API 的工具方面有着丰富的经验。在他过去的许多角色中，他是应用程序货币化的主要参与者 AppLovin 的创始 CTO。他现在是 thirdweb 的首席执行官。

web3 仍然是一个新生的、理想化的概念，这部分是因为 web3 的现实世界在发展过程中仍然充满了坎坷。

现在，你感觉没有一天不听说另一个区块链项目如何被黑，或者在曾经大肆宣传的 NFTs 世界中价格如何下跌；这还是在你开始考虑另一面的时候，感觉这只是一大堆空洞和非常自私的猜测。

Thirdweb 是这个生态系统的一部分，即使它不是任何区块链交易所或智能合约的直接运营商，但 Bartlett 声称，他们意外地发现，它的关联业务没有下降。

“令人惊讶的是，建筑商一直在继续施工，”他说。“我们每周的增长活动没有放缓，而且还在继续增加。我们的假设是，尽管存在波动，但从长远来看，不管是秘密还是宏观前景，从事项目建设的人都会继续下去。”

他补充说，这也是吸引投资者的因素之一。"我们的数据显示，我们逆势而上。"

Haun Ventures 的创始人兼首席执行官 Katie Haun 在一份声明中表示:“我们的使命是加速下一代互联网的发展，我们相信 thirdweb 将在实现这一目标方面发挥关键作用。”“随着 web3 开发的复杂性不断增加，由 Furqan 和 Steven 领导的经验丰富的 thirdweb 团队已经构建了一个优雅的解决方案，允许开发人员快速构建，同时避免代价高昂的错误。我很高兴看到这种水平的成熟创始人将他们的下一章献给 crypto，并期待支持他们的努力。”

从长远来看，我想知道 thirdweb 是否会继续投入资源开发自己的工具，还是会像市场一样开放，引入第三方开发者，包括那些将成为特定领域专家的开发者。

目前，Bartlett 说，该计划是继续依靠自己的力量追求全栈方法——可能通过收购团队和技术来帮助——尤其是因为它仍然是先行者，并且有机会这样做。

这也是它选择不收取费用的原因:尽你所能尽早获得足够数量的资金。

“我们相信这种特殊的游戏是赢家通吃，”他说。"我们的目标是尽可能多地获得基地。"
