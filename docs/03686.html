福特野马 Mach-E 取代特斯拉 Model 3 成为《消费者报告》的顶级电动汽车

乔恩·芬加斯是

Engadget

.

More posts by this contributor

在过去的两年里，特斯拉的 Model 3 一直是《消费者报告》( Consumer Reports)的首选电动汽车，但该出版物准备宣布一个新的冠军。 CR 透露福特野马 Mach-E 已经取代 Model 3 成为其电动车首选。编辑们表示，野马跨界车不仅“更实用”，而且第一年的可靠性更好，信息娱乐系统也“简单得多”，基本任务不需要多个步骤。更好的乘坐体验和更低的噪音也有帮助。

福特的 BlueCruise 驾驶员辅助技术也给了 Mach-E 一个优势，这要归功于一个更有效的驾驶监控系统，现在计入车辆得分。当司机看向别处时，特斯拉的自动驾驶仪因功能正常而被停靠。

《消费者报告》仍然推荐 Model 3，因为它具有跑车般的性能、续航里程、充电网络和技术。然而，该经销店无法推荐 Mach-E 更直接的竞争对手 Model Y 作为电动汽车的首选。特斯拉的 SUV 般的乘坐可靠性比阵容中的平均车辆“差得多”，明显比平均评级的 Model 3 更差。

这不会让特斯拉高兴，多年来，特斯拉与《T2 消费者报告》的关系并不友好。两者对测试结果有争议， CR 已经暂时撤回对部分车型的推荐。然而，这也反映出对特斯拉可靠性的担忧挥之不去。最近几个月，这家电动汽车生产商发布了一系列召回，车主也频繁报告制造质量问题。这本身可能不会让特斯拉失去领先地位，但它肯定无助于该公司的机会。
