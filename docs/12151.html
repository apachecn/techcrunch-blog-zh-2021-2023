FTX 和 Avalanche 共同为 Joepegs NFT 市场牵头 500 万美元融资| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/11/14/ftx-and-avalanche-co-led-5m-round-for-joepegs-nft-marketplace/

虽然上周 FTX 崩溃了，但是他们的风投团队仍然被宣布加薪。

位于雪崩区块链的 NFT 市场 Joepegs ，在由现已解散的 FTX 风险投资公司和雪崩基金会领导的种子轮融资中筹集了 500 万美元，其联合创始人(笔名 Cryptofish 和 0xMurloc)独家告诉 TechCrunch。

该团队在一份声明中表示:“来自 FTX 风险投资公司的资金已于 6 月完成，并在最近的破产事件之前被转移出 FTX。”。

该市场于 5 月推出，并迅速发展成为 Avalanche 上最大的 NFT 市场，二级 NFT 销售额超过 340 万美元，拥有 12，000 名用户。该公司还表示，它还有一个内部制作单位 Joe Studios，以及一个 NFT 发射台，该发射台已经为雪崩生态系统提供了 50 多个项目。

这位联合创始人还创立并继续参与了 Trader Joe、的运营，这是一家位于 Avalanche(不要与美国连锁超市混淆)的分散式交易所，于 2021 年 7 月初推出，交易总额超过 880 亿美元。

“当我们开始构建这个平台时，我们很快意识到，为了提供一个真正帮助用户发现伟大的 NFT 的平台，我们必须投资更多的平台功能，所以这就是筹资的方向，”0xMurloc 说。“除此之外，我们还创造了大量内容。我们一开始这样做是为了满足需求。市场的好坏取决于生态系统中的内容。”

除了 Avalanche，Joepegs 还投资运营方面，与“整个生态系统”的不同交易商、项目和艺术家合作，0xMurloc 说。“这是我们坚决要做的事情。”

今年早些时候，Avalanche 在与 NFT 最大的市场 OpenSea 合作后，进一步进入 NFT 市场，open sea 现在与 Joepegs 和 Kalao 等其他平台一起在区块链运营。cryptosslam数据显示，Avalanche 总销售额约为 4.082 亿美元，是 NFT 销量排名第七的区块链。

“人们关注的是更大的市场正在发生什么，”0xMurloc 说。“是的，现在玩 crypto 和整个 NFT 市场的人越来越少，但是，我们确实看到创作者、品牌和项目对深入 web3 和 NFTs 的兴趣——这种兴趣没有减弱。”

有很多公司、创作者和艺术家“渴望探索这种形式的商业和社区建设，”0xMurloc 补充道。

“我们非常看好 NFTs 的未来以及它可能带来的东西，”Cryptofish 说。“你可以拥有由 NFTs 支持的服装的想法非常乐观。你可以看到志那都红豆的滑板和耐克运动鞋，我们希望走在 NFT 数码产品和服装创新的前沿。”

Cryptofish 补充说，随着更多替代 NFT 产品的出现，NFT 市场将不得不适应适应。“随着时间的推移，我们对 NFT 市场的愿景必须像亚马逊一样。最初这是一家书店，现在他们已经扩大经营范围，什么都卖。这就是我对事情发展的看法。”

在短期内，该团队计划继续推动内部内容，并在不久的将来推出新的系列，0xMurloc 说。

“从长远来看，我们希望进入不同口味的非功能性交易，探索 Fish 提到的东西，无论是时尚、实体商品还是游戏。我们对未来充满期待。”
