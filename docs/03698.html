inter giro de-pages 是“嵌入式银行”中一家新的金融科技初创公司 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/02/17/intergiro-de-cloaks-as-a-new-fintech-startup-in-embedded-banking/

“嵌入式商务”始于 PayPal 之类的东西，多年后随着 Stripe 之类的产品发展。但“嵌入式金融”正作为金融科技领域的一股新浪潮出现，比如 Contis、Solaris、Swan 和 Stripe Treasury(尚未在欧盟上线)。事实上，你可能还记得安吉拉·斯特兰奇(a16z 的)在 2019 年宣布“每家公司都将成为金融科技公司”，一些预测认为，到 2030 年，嵌入式金融机会的价值约为 7.2 万亿美元。

过去五年来，Intergiro 一直是一家秘密运作、由私人资金支持的创业公司，但现在它正成为嵌入式银行运动的一部分，自称为“金融云”,几乎任何类型的公司都可以在其上提供银行服务。

联合创始人兼首席执行官尼克·鲁特毕业于计算机科学专业，曾任首席技术官、首席财务官和首席执行官，在创办 Intergiro 之前，他在伦敦银行业工作了 12 年。

在一次独家采访中，他告诉我:“使用我们工具的工程师不需要跳来跳去，把许多不同的提供商拼凑在一起来构建金融工具。他们可以进入整个金融价值链……从收银台到多币种钱包，到 SWIFT 或当地银行电汇或 Visa 或 Mastercard 网络。一体式方法不仅对产品构建者来说更方便，还能带来全新的产品体验。”

故事开始于几年前，当时 Root 想要建立新的银行服务。然后，他组建了 Intergiro 团队，使用传统方法来完成这项工作，但最终他也将目光投向了加密市场。Intergiro 自 2014 年以来一直存在(名称不同)，2017 年重新塑造，2019 年推出封闭测试版。但公开上市是在 2021 年。这是它第一次与 TechCrunch 这样的主流媒体进行详细的对话。

它是这样运作的:一家企业通过数字形式与 Intergiro 合作。一旦 KYC 获得批准，他们就可以访问 B2B 和 B2B2C API 套件，从而开始构建。他们可以自动处理自己的流程(工资单、对帐、发票支付等)。)或者他们可以将金融产品放入他们的应用程序(收银台、钱包、卡等)。).他们可以以各种方式结合这两个平台，例如创建“闭环”卡支付系统或加密交易应用程序。因为它是收单方、账户提供方和发卡方，所以它有自己的卡方案，所以市场和加密交易所(例如)可以从即时结算中受益。收入来自卡支付、外汇交易和卡支付(互换)。

Intergiro 自称是市场上为数不多的供应商之一，或许是唯一的供应商，它是卡收单机构、发卡机构、外汇和银行账户供应商，在 B2B 和 B2B2C 平台上进行营销:“我们不仅仅是坐在顶端的技术层，我们拥有完整的监管和技术体系，”Root 说。

“如果你看看科技是如何出现的，英特尔是如何抽象出芯片的，谷歌是如何抽象出互联网链接的，脸书是如何抽象出社交网络的。类似的事情正在金融领域发生，它将渗透到每一次消费者互动中。所以，你会在你意想不到的产品中看到钱，”他告诉我。

到目前为止，Intergiro 拥有大约 2000 名客户，并表示每月增加 200 名左右。自推出以来，这相当于约 25 亿€的交易额。这家总部位于斯德哥尔摩的初创公司现在拥有 140 名远程员工。
