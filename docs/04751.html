YC 支持的 Curacel 推出新的 API 平台，使技术领先的企业能够提供保险

原文：https://web.archive.org/web/https://techcrunch.com/2022/03/10/yc-backed-curacel-unveils-new-api-platform-that-enables-tech-led-businesses-to-offer-insurance/

YC 支持的初创公司 Curacel 正在为非洲市场开发保险基础设施，该公司推出了一个新的界面，允许零售、金融科技、电子商务和物流等数字企业将保险添加到其核心产品中。

这款基于 API 的工具名为 Grow，迄今已与非洲各地的 22 家科技实体整合，包括 Topship、Barter by Flutterwave、Float、Fingo 和 Payhippo，使他们能够将主要产品与保险套餐捆绑在一起，或作为独立服务提供保险。该工具还为旅游、医疗和汽车行业的保险提供商提供了新的增长途径，帮助他们以数字化方式分销产品。

Curacel Grow 通过让流行数字技术的用户随时可以获得保险产品，预计将促进保险在非洲的普及和推广，在非洲，的渗透率不到 3%——如果没有南非的超大市场，渗透率为 1.2%。

“机会是巨大的。Curacel 首席执行官兼联合创始人亨利·马斯克在尼日利亚经营这家初创公司，他说:“我们不仅仅关注人们为自己购买保险的情况，我们也关注可保险的交易——比如你可以捆绑贷款和人寿保险。”。

Mascot 的创业之旅始于 2017 年，当时他帮助医院数字化记录。然而，直到 2019 年，他才与联合创始人约翰·达达(首席技术官)合作建立保险技术，从一家健康保险公司的欺诈检测系统开始，这是一个试点项目，揭示了保险市场对更多技术的需求。

“我们意识到存在一系列问题。保险公司因为欺诈损失了很多钱。所以我们为他们建立了这个系统，”Mascot 说。

Curacel 正在为新兴市场建设保险基础设施。图片来源: Curacel

Curacel (YC W22)允许任何拥有管理系统的医院将其系统连接到初创公司的 API，以便与保险公司无缝集成，并实现支付流程的自动化。Mascot 表示，约有 4000 家医院和 15 家合作保险公司，包括 Old Mutual 和 AXA Mansard，已经使用了他们的平台，并通过他们的系统处理了价值 2500 万美元的索赔。

“我们围绕保险建立了基础设施，使保险行业能够更好地工作，从销售保险到确保真正拥有保险的人快速有效地获得支付，”Mascot 说。

Curacel 建立的基础技术现在使他们能够建立一种产品，这种产品有可能接触到他们运营的九个市场(包括尼日利亚和科特迪瓦)的数百万用户，因为他们寻求的增长不仅仅是为企业提供软件。Curacel 也将进入埃及市场，去年筹集了 45 万美元的种子前资金，为其扩张提供资金。

“实现嵌入式保险需要一整套技术，如对账、支付、报告和数据，以确保您向正确的人推荐正确的产品。我们从构建 API 基础设施开始，但我们是一家强大的技术公司，这使我们有可能提供这种产品。”

根据麦肯锡的一份报告，非洲的保险仍然是一种边缘产品，整个大陆的渗透率是世界平均水平的一半，人均保费比世界平均水平低 11 倍。尽管这些数字很低，但非洲市场仍是仅次于拉丁美洲的第二大增长市场。非洲大陆的保险业仍然高度分散，面临着分销的挑战，按毛保费计算，目前的价值为 680 亿美元(T2)。由于人工处理索赔造成的延误或拒绝而导致的缺乏信任也是造成接受率低的原因之一。

Mascot 说:“人们并不真正信任保险公司，因为他们没有以应有的速度支付赔款，这可能与缓慢的人工流程有关。”

“因此，我们为现有的保险公司和生态系统提供基础设施，使它们更有效率，更快地扩大规模，以抓住非洲的机会，”他说。

Curacel 现在加入了一个已经拥有几项保险技术的领域，包括米拉技术公司，该公司正在扩大其在非洲大陆的 API 保险平台。这是因为智能手机和平价互联网在整个非洲大陆的渗透程度加深，为数字产品的采用和保险技术公司介入并提供创新产品提供了机会。
