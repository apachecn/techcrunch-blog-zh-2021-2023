Noname Security 在 1.35 亿美元的 C 轮融资后达到 1B 的估值

最近，API 安全风靡一时，在一系列高调的安全事件导致大量用户数据暴露或泄露后，它被推到了聚光灯下。Peloton 泄露了用户的私人账户信息；益百利揭露了数百万美国人的金融历史；还有脸书、 LinkedIn 和 Clubhouse 都因其糟糕的安全 API 而遭到用户数据集体抓取。

对于像 Noname Security 这样旨在解决 API 安全问题的公司来说，业务正在蓬勃发展。这家总部位于帕洛阿尔托的公司今天宣布，它已经筹集了 1.35 亿美元的 C 轮融资，使这家初创公司的估值超过了 10 亿美元，成为最新一家加入网络安全独角兽俱乐部的初创公司。

本轮由 Georgian 和 Lightspeed 牵头，Insight Partners、Forgepoint 和其他公司参与。这笔资金是在 Noname 完成 B 轮 6000 万美元融资后仅六个月获得的，不到一年前，Noname 在去年 12 月隐身后在 A 轮融资中筹集了 2500 万美元。对于那些保持计数的人来说，该公司迄今已筹集了 2.2 亿美元。

Noname 通过分析配置设置、网络流量和代码来防止误用，从而帮助公司主动发现和补救 API 安全问题。简单来说，API 允许两个或两个以上的东西通过互联网相互对话，包括硬件和软件。API 几乎无处不在，即使你看不到它们。但是，如果没有适当的控制，API 可能会被滥用，从公司的服务器上窃取大量敏感或私人的内部数据。

这家初创公司表示，计划利用新投资来扩大研发规模，并改善其上市策略。Noname 现在声称支持五分之一的财富 500 强公司，并在过去的六个月里将其员工人数增加了两倍，达到 200 多人。
