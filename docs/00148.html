随着后大流行时代电子商务的繁荣，Rally 为无头结账平台筹集了 600 万美元

你可能听说过“无头”内容管理系统，通过它你可以运行一个内容管理系统而不会被锁定在一个前端平台上。嗯，类似的运动正在电子商务领域展开。

这场运动的最新体现是“无头结账”初创公司 Rally Commerce Inc. 获得了 600 万美元的种子资金。这里更广泛的想法是通过允许商家与 Shopify 等公司脱钩来“分散”电子商务生态系统。Rally 将 headless commerce 与 Web3 令牌方法结合起来，他们声称这将允许商家“拥有”网络的一部分。

这轮融资由 rain Ventures 领投，Felix Capital、Long Journey Ventures、Afore Ventures 和 Commerce Ventures 跟投。

Rally 背后的团队之前开发了 CartHook，这是一个结账解决方案。

Rally 表示，它的收银台是平台无关的，允许商家用 Rally 的收银台取代现有的传统收银台，或者建立一种无头体验，将 Rally 作为收银台和编排层。

Rally 的联合创始人兼首席执行官乔丹·加尔(Jordan Gal)表示:“商家经常被迫接受底层平台向他们提供的平庸检查，而应用程序开发者则屈从于他们为之构建的平台。我们为商家提供了更高的平均订单价值(AOV)和更好的转换率，以及根据他们的需求建立业务的自由。”

在大流行后，电子商务正在爆发，因为所有人都被迫使用在线平台，所以电子商务显然正在受益。

Felix Capital 的投资者 Joseph Pizzolato 表示:“随着商家和投资者认识到在所有接触点创造无缝和独特客户体验的潜力，无头商业空间正在蓬勃发展。CommerceTools、Nacelle、Fabric 和 Bolt 等公司正在彻底改变旧的电子商务体系，推动创纪录的资金流入和极高的估值。我们看到了同类的反弹。”

作为背景，Commercetools 是一家电子商务 API 提供商，大型零售商可以使用它来建立定制支付、结账、社交商务、市场和其他服务——最近完成了1.4 亿美元的融资。

但是商业工具和拉力赛并不孤单。其他包括 Spryker，Swell，Fabric，Chord 和幕府。
