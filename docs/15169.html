谷歌开始在 Android | TechCrunch 上测试其基于兴趣的广告定位品牌

原文：https://web.archive.org/web/https://techcrunch.com/2023/02/14/google-starts-beta-testing-its-rebrand-of-interest-based-ad-targeting-on-android/

谷歌已经开始让 Android 开发者踢它声称重新启动广告定位的轮胎——宣布推出其“隐私沙箱”的第一个测试版，这是一个广告技术栈提案，旨在迭代广告跟踪，定位和报告是如何完成的，以便它对个人用户来说不那么令人毛骨悚然，同时保持网络用户眼球上基于兴趣的行为定位能力。

随着这家广告科技巨头开始逐步(但它说是全球)推出测试版，符合条件的“一小部分”Android 13 设备将从今天起参加测试版的试用——这将“随着时间的推移而扩展”。(这里发布了开发者参与测试的指南。)

该试验的广告合作伙伴包括 TechCrunch 的母公司雅虎、移动游戏制造商 Rovio、移动公司 Wolt、跨平台游戏引擎 Unity 和移动营销平台 AppsFlyer、InMobi Exchange 和 Adjust。

“如果你的设备被选中进行测试，你会收到一个 Android 通知让你知道，”谷歌在一篇博客文章中补充道——这意味着 Android 用户将选择加入实验性的、基于兴趣的广告定位(如果他们不想让自己的眼球成为试验品，他们将不得不主动选择退出)。

当被要求确认这一选择加入点时，谷歌表示，欧洲经济区、瑞士和英国的 Android 用户将被要求自愿参与该计划——通过选择加入的方式。

这表明，在世界其他地方，用户正在被选择加入——因此，如果他们不想参与这些基于兴趣的广告测试，他们将不得不手动选择退出。

早在 2019 年，当谷歌宣布所谓的“隐私沙箱”提案，以使广告跟踪对个人网络用户的隐私不那么糟糕(或者，嗯，使其业务不那么成为隐私反弹的目标)时，它专注于其基于桌面的 Chrome 浏览器，以及反对第三方跟踪网络 cookies。但是一年前它宣布它打算将沙盒方法扩展到 Android——今天，它让第一批 Android 开发者体验它所构建的东西(并提供反馈)。

“隐私沙盒测试版提供了以隐私为核心的新 API，不使用可以跨应用程序和网站跟踪您活动的标识符。选择参与测试的应用程序可以使用这些 API 向你展示相关广告，并衡量它们的效果，”谷歌的安东尼·查韦斯在博客中解释道。

目前还不清楚到底有多少 Android 用户会接触到这种实验性的广告定位。或者这些用户会在世界的什么地方。或者有多少比例的人会被要求同意参加试验——而不是被选择参加，如果他们不想参加，就不得不通过设置选择退出。

查韦斯的博客文章仅指出，Android 用户可以通过进入设置的隐私沙盒部分来“控制”他们的“测试版参与”——他说，在这里，屏幕将让用户“查看和管理”广告兴趣，参与应用程序可以使用这些兴趣向他们显示“相关广告”，他称之为基于跟踪的营销。

“例如，你可以看到安卓系统已经估计到你对电影或户外等话题感兴趣，如果这些话题不符合你的兴趣，你可以屏蔽它们，”他写道，并补充道:“如果你改变主意不参加测试，你可以在设置中关闭或打开它。”

图片来源:谷歌

深入研究这些设置并发现自己被分配了并不持有的“话题”的 Android 用户可能有理由质疑谷歌基于兴趣的广告定位方法。

希望不会有比这更糟糕的冲击——因为谷歌已经表示主题分类将是“人类策划的”——正是为了避免将“敏感”主题作为“估计兴趣”包含在内。

然而，敏感话题(比如，代表政治观点的某类电视节目)可以轻易找到代理人，这表明这可能相当幼稚。用户可能希望考虑他们生成的兴趣是否会被其他可以访问他们设备的人看到，这可能会根据他们使用设备的目的而产生隐私风险。

谷歌的博客文章没有详细讨论这些敏感的隐私问题。

相反，该公司选择抨击移动平台竞争对手苹果的应用程序跟踪透明度(ATT)功能——该功能要求 iOS 上的第三方开发者征得用户同意进行应用内跟踪——查韦斯称 ATT 是一种“直言不讳”的方法，并链接到第三方跟踪拦截器进行的一项研究,该研究声称 ATT 实际上并没有停止跟踪，因此只是给用户一种“错误的隐私感”。

查韦斯的博客文章进一步声称，苹果的方式将导致通过设备指纹等方式增加对用户的跟踪，同时也损害了开发者产生广告收入的能力——因此谷歌显然旨在将其对广告跟踪的技术改造定位为一种更微妙的工具。一种避免二元选择的方法——比如事先询问人们是否希望被跟踪。

相反，对于大多数 Android 用户来说，谷歌将在多层设置菜单中埋下禁用跟踪的选项——这进一步模糊了选择，因为它建议用户可以调整他们对跟踪的适应程度，从而让行为定位继续进行(并在他们不看的时候产生新的估计兴趣)。

谷歌在这些年的发展中一直在琢磨的沙盒推销已经凝结和凝结在一个声明上，即其进化的广告技术栈将赋予用户“更多隐私”(如谷歌所定义的那样——因此肯定不是完全隐私)，同时让广告跟踪行业的肥缺产生收入，但将其锁定在谷歌的“进化”基础设施上；随着跟踪 cookies 被粉碎，它需要更多的内部跟踪组件(因此，嗯，比以往任何时候都更加依赖谷歌)。

客气地说，这是一个对双方都妥协的提议——最终，可能不会让任何一方满意。如果你愿意的话，一个双输。但谷歌显然认为这场赌博是值得的，因为如果它能摆脱追踪广告的更名，让沙盒符合隐私和竞争监管机构以及向他们投诉的广告商的要求，它就能获得真正的大胜利，继续成为广告技术的赢家，让行为广告的大部分资金流入它的金库……
