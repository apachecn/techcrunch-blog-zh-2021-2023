Aspecta 获得了 350 万美元来建立人工智能审查的编码器配置文件| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/21/aspecta-nabs-3-5m-to-build-ai-vetted-coder-profiles/

虽然 LinkedIn 有助于展示人们的教育和职业成就，但存在着一个自学成才的技术人才的世界，他们的技能不那么容易在这个社交网站上反映出来。相反，他们的专业知识隐藏在他们编写的代码行中。

Aspecta 正试图通过为希望为自己创建类似 LinkedIn 的身份页面的开发人员提供一个人工智能驱动的个人资料生成器来填补这一空白。这是通过使用大型语言模型来审查他们所参与的项目中的代码质量来实现的。该平台还考虑了社会认可，并应用网络分析来查看程序员的工作是否得到了其他公认专家的“喜欢”。

尽管经历了加密的寒冬，web3 开发者的数量一直在激增，自然地，Aspecta 的数据触角延伸到了区块链世界。它的算法从用户的钱包地址如何与智能合约互动中获取线索，并相应地标记他们的活动类型。据 Aspecta 的联合创始人 Jack He 称，这是一个比审查 GitHub 数据“更容易”的过程，因为“链上数据更加结构化”

Aspecta 目前与 GitHub 集成在一起，所以当用户在代码托管网站上用他们的帐户登录时，他们会收到一个自动生成的身份页面，上面有他们所有的技术成就，看起来有点像游戏玩家的奖杯页面。该平台还将谷歌、Stackoverflow 和 MetaMask 纳入其数据源，并正在加入 Twitter。

Aspecta 的 alpha 版本已经吸引了大约 13 万人加入它的等候名单。投资者已经注意到了它的吸引力，今天，这家初创公司宣布完成了一轮 350 万美元的种子融资，这将使它能够推出旗舰数字身份产品 Aspecta ID 的测试版。

这轮融资的主要机构投资者包括真格基金(ZhenFund)，该基金以支持中国企业家在全球扩张而闻名，此外还有专注于加密的 HashKey Capital 和 Foresight Ventures。这家初创公司还吸引了几家可能会利用其身份服务的战略投资者，包括 Galxe、 CyberConnect 、 Mask Network 和 P12。

道治

像如今在数字身份领域工作的许多初创公司一样，Aspecta 发现自己被 web3 的去中心化治理机制所吸引。

“我们本质上是在创建一个身份生态系统，我们不认为它应该由第三方公司来管理。相反，它应该由 DAO 来管理，”他告诉 TechCrunch。

DAO，即分散式自治组织，由编码为计算机程序的规则来运行，该程序由组织的成员而不是中心方来控制。DAO 成员拥有允许他们对关键决策进行投票的令牌。这个概念在加密牛市期间风靡一时，吸引了艺术家、企业家和投资者发起他们的 Dao，但是自从市场下跌以来，许多 T4 人失去了动力。

Aspecta 的 DAO 还没有实现。何表示，该平台人工智能挖掘的人才洞察力最终将产生收入，这些收入最好根据社区规则进行分配。换句话说，它希望让它的 用户，或数据提供商，决定他们的数据如何被数据消费者使用，这些人利用他们的数字足迹。

除了为程序员服务，该平台还面向那些能够通过其“身份即服务”系统利用易消化的开发人员数据的组织。例如，黑客马拉松可能会发现 Aspecta 很有用，因为它的 ID 系统不仅可以帮助注册参赛者，还可以提供黑客的熟练程度。或者，如果一家公司正在招聘开发人员，它可以使用 Aspecta 的数据来快速了解候选人，而不是逐行检查他们的代码。

Aspecta 的下一步是瞄准内容创作者，他们面临着与软件开发者相同的困境——他们的成就在现有的专业社交网站上不容易展示。因此，这家初创公司计划应用其算法来检查创作者推文或 YouTube 视频的受欢迎程度，并使用图形学习算法来识别虚假评论。

“你可以伪造你的 Twitter 关注，但你不能玩阿斯佩特卡的算法，”他说。
