苹果发布 iOS 和 macOS 补丁以修补新的零日攻击| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/09/13/apple-ios-macos-zero-day-active-attack/

苹果发布 iOS 和 macOS 修复程序，以修补新的零日攻击

苹果发布了另一轮安全更新，以解决 iOS 和 macOS 中的漏洞，包括一个正在被攻击者积极利用的新的零日漏洞。

苹果公司周一在安全顾问中表示，被追踪为 CVE-2022-32917 的零日漏洞允许恶意应用程序以内核权限在受影响的设备上运行任意代码，这意味着对设备及其数据的完全访问。苹果警告说，它知道这个漏洞“可能已经被积极利用”，据信是苹果自今年年初以来修复的第八个零日漏洞。

苹果表示，它已经修复了 iOS 15.7 和 iPadOS 15.7、macOS Monterey 12.6 和 macOS Big Sur 11.7 更新中的漏洞。

苹果没有透露任何关于 CVE-2022-32917 的进一步信息，也没有透露它是如何被网络犯罪分子利用的。苹果没有回应置评请求。

苹果公司本周为运行 macOS Big Sur 11.7 的 MAC 电脑重新移植了另一个被利用的零日漏洞(CVE-2022-32894)的补丁。此前几周，该公司在旧款 iPhones 和 iPads 上修补了同样的漏洞——苹果称之为可远程利用的 WebKit zero-day，该漏洞可能允许攻击者在未打补丁的设备上执行任意代码。

除了这些修复，苹果周一还发布了其他一些安全更新，包括一个可能导致地址栏欺骗的 Safari 漏洞，一个可能让攻击者读取敏感位置信息的地图问题，以及一个可能让应用程序绕过隐私偏好设置的联系人漏洞。

安全补丁与 iOS 16 一起发布，iOS 16 带来了许多安全和隐私增强，包括支持苹果密钥和锁定模式。
