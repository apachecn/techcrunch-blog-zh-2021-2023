Tabnine 为自动完成代码的人工智能筹集了 1550 万美元

原文：https://web.archive.org/web/https://techcrunch.com/2022/06/15/tabnine-raises-15-5m-for-ai-that-autocompletes-code/

为软件开发人员创造“人工智能助手”的初创公司 Tabnine 今天完成了一轮 1550 万美元的融资，由高通风险投资公司、OurCrowd 和三星 NEXT Ventures 共同牵头，现有的支持者 Khosla Ventures 和头条风险投资公司也参与了融资。首席执行官 Dror Weiss 表示，收益将用于改善开发者体验，添加新功能和“加强”Tabnine 的企业产品。

人工智能正越来越多地推动编程工具。也许最好的例子是 OpenAI 的 Codex ，它为 GitHub 的 Copilot 功能提供了支持，该功能可以为微软 Visual Studio 等开发环境中的代码行提供建议。这些工具承诺削减管理成本，同时允许编码人员专注于更具创造性、重复性更少的任务——或者说是销售宣传。

Tabnine 成立于 2012 年，名为 Codota，它利用人工智能来理解代码，自动完成功能或带有目的和内容想法的代码“块”。该平台利用训练有素的算法来理解代码的语义模型，试图学习个体的最佳实践，并警告偏离这些实践的行为。

“Tabnine …是由 Eran Yahav 和我自己在 2017 年创立的，”韦斯在一次电子邮件采访中告诉 TechCrunch。他说，“Tabnine”这个名字来自一家总部位于滑铁卢的初创公司，与 Codata 在 2019 年收购的公司同名。“基于我们之前在代码分析和模拟方面的工作，我们意识到，随着代码中大量的通用性和标准模式，人工智能将不可避免地成为开发过程中的一个关键部分。我们开创了人工智能代码助手类别。”

Tabnine 在包括 Android Studio、VSCode、IntelliJ、Webstorm 和 Eclipse 在内的集成开发环境中，为每一次按键提供建议，并提供整行或功能建议——由 Weiss 描述为小型、“代码原生”人工智能模型驱动，这些模型在特定编程语言或领域上从头开始训练。他说，Tabnine 目前有十几个这样的流行语言模型，以及由生态系统合作伙伴培训的“社区”模型。

图片来源: Tabnine

韦斯声称，Tabnine 的方法使平台能够比其他代码生成解决方案更好地学习代码中的“规律”和模式，并且非常有效。“我们的模型让客户能够灵活地在我们的云或他们的网络上运行 Tabnine ，并能够训练定制的人工智能模型，以捕捉他们仓库中的特定模式，”韦斯说。“[像 Copilot 这样的工具]仅限于在新行上提供建议，因为推理成本和延迟要高得多。此外，它们依赖于一个巨大的单一人工智能模型，这个模型只能由(大型科技公司)托管。”

韦斯说，在过去的一年里，Tabnine 为 Python、JavaScript、TypeScript、Java 和其他语言推出了十几个新的人工智能模型，以及与 GitHub、GitLab 和 BitBucket 的集成。最新一代的 Tabnine 平台让开发人员决定在哪里运行 Tabnine 的人工智能助手，无论是在单台机器上，在 Tabnine 的云服务上，还是在自托管安装上。

韦斯声称，现在有超过 100 万名开发人员正在使用 Tabnine 的人工智能技术，每天完成超过 400 万行代码。付费用户数以千计，包括埃森哲和 LG 这样的品牌。

“人工智能平台现已被证明对软件开发的生产率和质量具有极大的重要性。随着组织寻求更快地创新，提高开发人员的生产力和更快地加入新团队成员对每个公司都具有不可思议的价值，人工智能是实现这一目标的唯一可扩展和具有成本效益的方式，”韦斯说。“组织人工智能平台是组织开发堆栈的下一个进化步骤，它将组织软件资产转化为一个活跃的知识库，使每个开发人员变得更好，就像与公司内部的领域专家结对编程一样。”'

当然，代码生成系统是有价值的。剑桥大学贾奇商学院发表的一项研究显示，程序员 50.1%的工作时间不是在编程；另一半是调试。Standish Group 发现那些“有挑战的”项目——即那些未能满足范围、时间或预算预期的项目——约占软件项目的 52%。

但是 Tabnine 面临着激烈的竞争。GitHub 最近宣布，已经拥有数万用户的 Copilot 将在经过几个月的技术预览后，于今年夏天全面上市。除了 Copilot，还有像 Ponicode 这样的平台，它利用人工智能来检查代码的准确性。 DeepCode 也提供了一个机器学习驱动的系统，用于整个应用程序的代码审查——和亚马逊一样。

Tabnine 面临的挑战将是利用迄今为止筹集的 3200 万美元资本继续保持差异化。韦斯指出，该公司计划在年底前雇佣新员工，将美国和以色列的员工人数从 30 人扩大到 40 多人。

科斯拉风险投资公司的合伙人布莱恩·拜恩在一份声明中补充道:

人工智能辅助的软件开发将在未来几年成为所有开发者事实上的标准。Tabnine 以最广泛使用的、开发人员喜爱的、开发人员优先的编码助手开创了市场。Tabnine 以其独特的组合人工智能模型为基础，准备为企业提供最准确、合规和可采用的人工智能辅助和代码智能平台。
