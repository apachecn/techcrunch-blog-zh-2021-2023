OP3N 筹集 2800 万美元为 web3 | TechCrunch 打造“WhatsApp 遇上亚马逊”

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/21/op3n-raises-28m-to-build-whatsapp-meets-amazon-for-web3/amp/

图片来源:阿丽娜·库里安诺娃/盖蒂图片社

web 3 人工智能平台 OP3N 筹集了 2800 万美元的 A 轮融资，对该公司的估值为 1 亿美元，这家初创公司与 TechCrunch 独家分享。

“我想在一个地方为创作者、聊天和商业建立联系，”OP3N 的联合首席执行官马正远对 TechCrunch 说。“这是 WhatsApp 遇到亚马逊的 web3 版本，或者微信遇到阿里巴巴。”

OP3N 的基于聊天的产品 Superapp 旨在通过将区块链和去中心化技术实施到 Web 2.0 友好的界面中来弥合 Web 2.0 和 web3 之间的差距。该应用程序将有能力添加音频，视频和混合媒体内容的链上互动，使用户可以有一个“一站式”的体验。

马云说，这个平台的诞生是出于对类似于 Web 2.0 平台的用户体验和界面的需求，但由 web3 技术提供支持。

马分享说，这个平台允许艺术家和创作者为粉丝提供新的参与方式，通过 NFT 门体验，粉丝可以以不同的价格购买。他补充说，客户可以通过电子邮件注册，该平台将为用户创建一个加密钱包。

这轮融资由 Animoca Brands 牵头，包括来自蜻蜓资本、SuperScrypt、创意艺术家机构和新企业协会的 Connect Ventures、Republic Crypto、雪崩的暴雪基金、银河数字和华纳音乐互动的投资。其他投资者包括 GSR Markets、斯巴达集团、BRV 资本管理公司和 NBA 球员拉塞尔·维斯特布鲁克等。

“这些都是长期投资者，他们会留下来，为我们带来战略协同效应，”马云谈到投资者时说。“我们来这里是为了打造一款产品，我相信它有望在两年内成为 web3 的去中心化脸书、YouTube、Spotify 或 WhatsApp。”

2021 年，OP3N 完成了来自 Goodwater Capital、Soma Capital、500 Global、WhaleShark、Twitch 联合创始人贾斯汀·坎(Justin Kan)和沙盒联合创始人首席运营官·塞巴斯蒂安·博尔杰(Michael sébastien Borget)等投资者的 1000 万美元种子轮投资。

马云表示，新的资金将用于进一步开发其应用程序以及培养其领导团队。他补充说，该产品目前处于测试模式，已经在不同的社区进行了测试。

“通过我们在第一版中的学习，我们意识到当我们做[NFT]下降时，粉丝们不知道 AVAX，Solana，Polygon 或任何加密货币之间的区别，”马说。仅仅那一次经历就引发了如此多的摩擦和负面影响。"

因此，展望未来，马意识到 OP3N 必须以一种更容易理解的方式将传统市场连接到 web3。“10 亿人知道如何下载和使用钱包不会在一夜之间发生，所以我们正在解决这个问题。”
