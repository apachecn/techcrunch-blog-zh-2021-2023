软件股甚至放弃了更多的地盘

快速博客在这里更新你一些相当重要的运动在市场上。今天，在股票普遍低迷的一天，软件和云公司的股票遭受重创。

根据美国消费者新闻与商业频道的数据，纳斯达克综合指数下跌了 2.51%。对于一个庞大而关键的公开交易财富类别来说，这是非常糟糕的一天。随后，Bessemer Cloud Index 在常规交易期间下跌了 5.45%。bes semer Cloud Index 是我们最喜欢的跟踪一篮子更有针对性的现代软件公司的方法。

一天之内删除了很多价值。但由于下跌发生在创业关键指数最近已经经历了大幅下跌之后，这是对伤害的侮辱。这是图表:

您需要从这个图表数据集合中收集两件事情:

  1. 软件股从 2020 年末以来的涨幅超过了所有涨幅，从近期高点回落了 30%以上。这太糟糕了。
  2. 软件股票仍然价值不菲，价值远高于大约两年前的 2020 年初。那很好。

因此，对于现代公共软件公司来说，事情并不伟大，但也不可怕。

TechCrunch 继续追踪的问题是，上面显示的下跌多快开始影响到初创公司的估值。我们正看到公私市场的分裂，首次公开募股和直接上市试图将公司从一个海岸带到另一个海岸。但就纯粹的创业融资势头而言，你不会知道收入倍数正在对公开市场产生巨大影响。对于大多数创业公司来说，这仍然是令人兴奋的日子。
