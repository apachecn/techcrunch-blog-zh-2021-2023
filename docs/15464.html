谷歌称 Chrome 不会耗尽你的 MacBook 电池

原文：https://web.archive.org/web/https://techcrunch.com/2023/02/28/google-says-chrome-wont-drain-your-macbook-battery-as-much/

谷歌表示，Chrome 不会消耗你的 MacBook 电池太多

谷歌表示，它正在推出对最新版本 Chrome 的改进，使其在 MacBooks 上不那么耗电。该公司表示，将于本周向所有用户推出的 110 版本将为采用苹果处理器的新款 MAC 电脑甚至旧款电脑提供性能提升。

谷歌进行的测试显示，用户可以在运行新版 macOS Ventura 的 MacBook Pro 上观看 18 小时的 YouTube 视频或浏览网页 17 小时。谷歌没有提供任何数据来表明新版本会对采用英特尔芯片组的旧款 MacBook 产生多大影响。

值得注意的是，谷歌去年开始测试内存和节能模式。但是这些增强功能从上周开始接触到用户。这家搜索巨头表示，节能模式可以进一步延长电池寿命，因为 Chrome 可以比上述测试多运行 30 分钟。

谷歌指出，Chrome 团队对 iframes(一种在页面中加载 HTML 页面的元素)进行了修改。它还阻止了浏览器不必要地重绘 UI 元素，并调整了计时器以获得更有效的 CPU 访问。

去年，微软还调整了其 Edge 浏览器，以占用更少的系统资源，实现更好的能源管理。

你可以通过点击右上角的“更多”按钮，打开“帮助”菜单，然后点击“关于谷歌浏览器”，来检查你正在使用的谷歌浏览器的版本，并(可能)更新到最新版本。否则，Google 会时不时地在后台自动下载并应用新的更新。
