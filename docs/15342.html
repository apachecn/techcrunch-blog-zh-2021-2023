比特币基地启动区块链基地，帮助开发者在链上构建 dApps

原文：https://web.archive.org/web/https://techcrunch.com/2023/02/23/coinbase-launches-blockchain-base-to-help-developers-build-dapps-on-chain/

交易量第二大的加密交易所比特币基地(T1)推出了专注于以太坊的第二层(L2)区块链 base ，比特币基地 Base 负责人兼协议主管杰西·波拉克告诉 TechCrunch。

波拉克说，过去，比特币基地一直专注于其业务的交易和交换方面，但从实用的角度来看，开发人员仍然很难构建有用的分散式应用程序(dApps)，用户也很难在链上实际使用这些东西。

Pollak 说，为了进一步扩展到开发者领域，比特币基地正在建立一个基础，使开发者构建 dapp 和用户通过比特币基地产品访问这些 dapp 变得“非常容易”。“我们的目标是实现比特币基地总体规划的第四阶段:将 10 亿用户带入密码经济。”

该公司表示，L2 是一个“安全、低成本、开发者友好”的连锁店，旨在帮助建筑商创建 dapp on-chain。Base 建立在麻省理工学院许可的 OP 栈上，与第二层区块链乐观组织合作，后者也专注于以太坊链。

一位比特币基地发言人告诉 TechCrunch，许多加密业务、平台、市场和基础设施公司都致力于在此基础上进行建设。

那些计划参与其中的包括 Blockdaemon、Chainlink、Etherscan、Quicknode、Aave、Animoca Brands、Dune、Nansen、Magic Eden、python、Rainbow Wallet、Ribbon Finance、The Graph、Wormhole 和意式冰淇淋等等。

波拉克说:“我们努力向每个人开放，让他们进入这个开放的生态系统。”。“我们要取得成功，就必须继续坚持去中心化、开源的价值观，让它为所有人服务，并成为一座桥梁，将我们的用户连接到加密经济中，将他们带到他们可以去的地方，这些地方不受 Coibase 控制，但可以为他们提供宝贵的体验。”

Pollak 说，开始时，Base 的费用将与 Arbitrum 或乐观公司的费用相当，在 10-50 美分的范围内，计划在未来一年内下降近 1 美分。波拉克说，该链目前处于测试网阶段，还没有 mainnet 的具体推出日期，但预计将在今年的 Q2 期间推出。

波拉克说，虽然基地是在比特币基地内部孵化的，但它的计划是“在未来几年内”将其分散化。团队“在过去的六个月里做了大量的工作”来解决其他人最关心的一些问题，包括一个大的集权公司如何以分权的方式参与。

“这将是一个旅程，但这就是为什么我们乐观地工作，为什么我们建立在开源技术栈上，并且从第一天起就如此谨慎地对待我们带入生态系统的利益相关者，”波拉克说。

波拉克说，比特币基地对 base 的目标也是为开发者和用户提供基础，使人们能够进来。“这将要求我们尽可能便宜、尽可能安全、尽可能去中心化，因为去中心化是实现生态系统开放性的一个关键方面。”

Pollak 分享道，该链将整合比特币基地的交易所、钱包、NFT 市场和开发者产品，并计划与以太坊以外的生态系统中的其他区块链进行互操作。波拉克补充说，基地已经与建设 Solana、Avalanche、Polygon 和其他人的团队进行了对话，以实现这一愿景。

波拉克说，Base 还与比特币基地风险投资公司合作推出了一个生态系统基金，主要对整个加密生态系统的建设者进行种子前投资。“我们不想只关注金融工具、非金融交易或游戏；我们希望它受到所有人的欢迎。”
