好像成为一名古希腊学者从根本上来说还不够困难，他们所依赖的原始文献经常被损坏得无法修复，因为它们已经有几千年的历史了。历史学家可能会在 Ithaca 中拥有一个强大的新工具，这是一个由 DeepMind 建立的机器学习模型，可以对缺失的单词以及文本的位置和日期做出惊人的准确猜测。这是人工智能的一个不同寻常的应用，但它展示了它在科技世界之外是多么有用。

不完整的古代文本的问题跨越了许多学科，专家们在这些学科中研究退化的材料。原始文件可能由石头、粘土或纸莎草纸制成，用阿卡德语、古希腊语或线性字母 A 书写，描述了从杂货商的账单到英雄的旅程的任何事情。然而它们的共同点是历经数千年累积的损伤。

文本磨损或撕掉的空隙通常被称为空白，可以短到一个字母丢失，也可以长到一章，甚至整个故事。填写它们可能是琐碎的或者不可能的，但是你必须从某个地方开始——这就是伊萨卡想要帮助的地方。

伊萨卡岛(以奥德修斯的家乡命名)在一个巨大的古希腊文本图书馆中接受训练，它不仅可以说出一个丢失的单词或短语可能是什么，还可以猜测它有多古老以及写于何处。它不会自己填充整个史诗循环——它是为那些处理这些文本的人准备的一个工具，而不是一个解决方案。

发表在《自然》杂志上的一篇论文展示了它的功效，并以雅典的一些法令为例。被认为是写于公元前 445 年左右，伊萨卡根据其文本分析认为它们实际上是来自公元前 420 年左右——与最近的证据一致。这听起来可能不是很多，但是想象一下如果权利法案实际上是 20 年后写的！

图片来源: DeepMind

至于文本本身，研究中的专家在第一遍中有 25%的正确率；不完全是明星，虽然文本恢复当然不是一个下午的游戏，而是一个长期的项目。然而，与伊萨卡配对，他们很快就达到了 72%的准确率。在其他情况下，人们最终会更准确，但可以通过快速消除死角或建议起点来加快过程，这种情况经常被发现。在医疗数据中，很容易监督人工智能可能迅速标记的异常——但最终是人类的专业知识感知细节并找到正确的答案。

你可以在这里测试一下精简版的伊萨卡，如果你手边有一些充满空白的古希腊文本，或者使用他们提供的例子来看看它如何填补要求的空白。对于较长的部分或超过 10 个字母的缺失，在这款 Colab 笔记本中尝试一下。代码可在 GitHub 页面获得。

虽然古希腊语是伊萨卡开始的一个明显且富有成效的领域，但团队也已经在努力学习其他语言。阿卡德语、通俗语、希伯来语和玛雅语都在名单上，希望随着时间的推移会有更多的人加入。

“伊萨卡展示了自然语言处理和机器学习在人文学科中的潜在贡献，”参与该项目的雅典大学教授 Ion Androutsopoulos 说。“我们需要更多像伊萨卡这样的项目来进一步展示这种潜力，但也需要合适的课程和教材来教育未来的研究人员，他们将对人文学科和人工智能方法有更好的共同理解。”
