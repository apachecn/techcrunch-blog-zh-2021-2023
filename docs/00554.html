NFTs 和加密钱包可能是不和谐的未来

原文：https://web.archive.org/web/https://techcrunch.com/2021/11/09/discord-nfts-metamask-crypto/

Discord 已经是 NFT 社区事实上的家，但该公司可能正在计划以一种大的方式加深这些联系。

周一，Discord 创始人兼首席执行官杰森·香橼(Jason Citron)在推特上发布了一张截图，描绘了 Discord 与流行的加密钱包服务 MetaMask 和 WalletConnect 的集成，wallet connect 是许多移动加密钱包所基于的开放协议。香橼采用了 NFT 推特上的短语“可能没什么”，这是对这将是一笔大交易的简写。

钱包支持目前看起来纯粹是探索性的。Citron 没有提供任何关于假设的加密集成将如何工作的细节，或者关于探索加密集成的分歧有多严重，该公司也拒绝提供细节。

“我们一直在探索和黑客攻击我们认为会改善我们服务的所有社区的不和谐，”一位不和谐发言人告诉 TechCrunch，并补充说截图来自最近的黑客马拉松。

Discord 对在社交聊天应用中添加加密功能的兴趣可能是假设的，但如果该公司打算支持以太坊，这些计划可能远远超出支付范围。Discord 最近还调查了用户对 NFTs 的看法。

今年,“不和谐”成了数千个 NFT 项目的天然家园。其中许多项目通过专用的 Discord 服务器与关注者保持联系，发送更新信息，跟踪销售和市场动向。它也是协调分发活动的首选应用程序，在最终进入 OpenSea 等交易平台之前，NFT 被“铸造”并出售或分发给支持者，通常价格很高(是的，通常是 JPEGs)。

虽然 Discord 可能只考虑以太坊和其他数字货币的交易，但 Citron 关于元掩码支持的暗示也有可能是一个以 NFTs 为中心的更雄心勃勃的计划。像以太坊这样的硬币可以用于基本的支付和交易，但加密货币也是大多数 NFT 的技术支柱，通过以太坊区块链上的智能合约进行跟踪和交易。

Discord 是一家精明的公司，它可能对人们如何使用 NFT 来表达他们的数字身份有很好的想法。Discord 是一个文本和语音聊天应用程序，用户身份的主要表达是通过你选择的化身——这是 NFT 目前的全部内容。在 Twitter 和 Discord 上，熟悉 NFT 的用户已经选择了他们最稀有的——通常也是最昂贵的——NFT 作为他们的个人资料图片。有了 MetaMask 的支持，Discord 可以成为一个地方，人们可以在与用户资料链接的画廊中展示他们的 NFT，或者选择“经过验证的”头像图像，所有权通过区块链得到支持。

该公司尤其有能力利用其在蓬勃发展的 Web3 领域的立足点——许多人预测，互联网的下一阶段将由一波去中心化、数字商品和基于所有权的虚拟身份来定义。但是不和谐也有好的一面。

在 Twitter 上，Citron 的推文结合该公司发出的关于“Discord & Web3”的调查证明是两极分化的。许多账户召集 Discord 用户取消他们的高级 Nitro 订阅，这些订阅按月收费提供功能津贴，理由是担心 Discord 可能会在 NFTs 上全押。“取消我的硝基潜艇，去他妈的@discord，去他妈的 NFTs，”一条有近 10，000 个赞的推文说。

其他人涌向 Citron 的推特，哀叹 Discord 的新生加密计划，因为诈骗和垃圾邮件堵塞了他们的社区服务器。Citron 向一些 Twitter 用户保证，Discord 创建了一个新团队，专门解决困扰许多服务器的猖獗的粗略行为，包括那些围绕交换和销售 NFT 的服务器。“垃圾邮件和安全是我们在这个领域的首要任务，”Citron 写道。“我们最近成立了专门研究这个问题团队。更多即将推出。”
