AWS 推出物联网 TwinMaker，这是一种轻松创建数字双胞胎的新服务

在今天早上 AWS 的 re:Invent 大会上，该公司宣布了 AWS IoT TwinMaker ，这是一项新服务，可以轻松创建和使用现实世界系统的数字双胞胎。

就上下文而言，数字双胞胎是建筑物、工厂、生产线和设备等事物的虚拟表示，它们定期用真实世界的数据进行更新，以模拟它们所代表的系统的行为。

该公司概述说，通过这项新服务，用户可以通过连接来自视频源和应用程序等来源的数据来创建数字双胞胎，而不必将数据转移到一个单一的存储库中。

“您可以将内置数据连接器用于以下 AWS 服务:AWS IoT SiteWise 用于设备和时序传感器数据；亚马逊 Kinesis 视频流的视频数据；以及亚马逊简单存储服务(S3 ),用于存储视觉资源(例如 CAD 文件)和来自业务应用的数据。AWS IoT TwinMaker 还提供了一个框架，让你可以创建自己的数据连接器，以便与其他数据源(如雪花和西门子 MindSphere)一起使用，”AWS 在一篇关于这项新服务的博客文章中解释道。

该公司指出，一旦创建了数字孪生图，用户可能会希望在物理环境的背景下可视化数据。为了解决这个问题，AWS IoT TwinMaker 创建了一个数字孪生图，它结合了用户物理系统的虚拟表示和连接的数据源之间的关系。这允许用户精确地模拟他们的真实世界环境。用户还可以导入现有的 3D 模型来布置物理空间(如工厂)的 3D 场景。从那里，您还可以添加交互式视频和传感器数据叠加，以及来自互联机器学习服务的见解。

AWS 指出，这项服务附带了一个亚马逊托管 Grafana 的插件，这是 Grafana 实验室为开放仪表板和可视化平台提供的托管服务。

AWS IoT TwinMaker 今天在北弗吉尼亚、俄勒冈州、爱尔兰和新加坡提供预览，并在未来的其他 AWS 地区提供。
