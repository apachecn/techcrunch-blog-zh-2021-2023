  • 然后，我们以与汽车无关的自动驾驶汽车部分结束。是的，我们正在谈论可爱的农业机器人，包括来自 Vecna 的轮、电动羊和 MongoDB 创始人做这个，这提醒我们机器人的 AWS 并不遥远。
