必应称要删除其 GPT-4 聊天服务的等待名单

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/16/bing-said-to-remove-waitlist-for-its-gpt-4-powered-chat/

必应表示要删除 GPT 4 聊天服务的等待名单

微软的必应在上个月发布了 GPT 驱动的界面后，十年来第一次受到关注。但到目前为止，这家科技巨头对向用户提供新的必应产品的速度保持谨慎——该产品由 OpenAI 的 GPT-4 技术提供支持。但是 Bing 似乎正在推倒这些墙。

OpenAI 的主要投资者微软似乎已经取消了新 Bing 的等待名单，表面上允许任何人立即获得新的体验。最先发现这一变化的 Windows Central 表示，用户不必再等着试用新的必应。TechCrunch 用几个电子邮件 id(新旧都有)对此进行了测试，并立即获得了访问权限。然而，并不是所有我们测试的电子邮件 id 都能立即访问。

图片来源:TechCrunch 截图

虽然新必应的登陆页面仍然显示典型的“加入等候名单”按钮，但你可以登录并立即进入。该公司没有说明他们是否完全摆脱了等待名单，但在一份声明中，微软表示，它正在进行各种实验，以接纳更多的用户。

“在这个预览阶段，我们正在进行各种测试，这可能会加快一些用户使用新必应的速度。我们仍处于预览阶段，你可以在 Bing.com 注册，”微软发言人说。

微软将于美国东部时间周四上午 11 点举办一场名为“用人工智能重塑生产力”的活动。虽然今天的议程仅限于为微软 356 (Office)和该公司的 Salesforce 竞争对手 Dynamic 365 推出人工智能工具，但如果有一项宣布也与 Bing 有关，也不会令人惊讶。

这家总部位于西雅图的公司正在努力将人工智能聊天机器人集成到它的许多服务中。上个月，微软将 GPT-4 驱动的机器人引入 Windows 11 的任务栏。本周早些时候，Edge 浏览器的稳定版本获得了 Bing AI 聊天机器人功能。

OpenAI 的技术被证明是 Bing 的一大亮点，据最近报道，Bing 的日活跃用户达到了 1 亿。鉴于人工智能聊天机器人的大肆宣传以及它如何吸引了数千万希望尝试它的用户，这是意料之中的。在人们能够“越狱”聊天机器人说出有问题的事情后，微软开始测试对对话的各种限制。本周早些时候，它将限制提高到每天 15 次对话和 150 条消息。

微软的声明更新了这个故事。
