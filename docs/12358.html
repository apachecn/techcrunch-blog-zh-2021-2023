Mozilla 期待下一章| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/11/17/mozilla-looks-to-its-next-chapter/

Mozilla 今天发布了年度“Mozilla状态”报告，大部分消息都是积极的。Mozilla Corporation 是整个 Mozilla 组织的盈利方，2021 年从其搜索合作伙伴关系、订阅和广告收入中产生了 5.85 亿美元——比前一年增长了 25%。虽然 Mozilla 仍然主要依赖其搜索合作伙伴，但来自 Mozilla VPN、Mozilla Developer Network(MDN)Plus、Pocket 等新产品的收入现在占其收入的 5700 万美元，比上一年增长了 125%。在很大程度上，这是由 Firefox 和 Pocket 的新标签上的广告推动的，但安全产品现在也有 400 万美元的年收入。

随着今年报告的发布，Mozilla 领导团队也花了一些时间来展望未来，因为在许多方面，这是 Mozilla 的一个拐点。

当 Mozilla 成立时，互联网本质上是网络，浏览器是访问它的方式。从那时起，我们体验互联网的方式发生了巨大的变化，虽然浏览器仍然是最重要的工具之一，但它不是唯一的工具。因此，Mozilla 也必须做出改变。它的 Firefox 浏览器已经从主导这个领域变成了一个利基产品，但是该组织的使命(“确保互联网是一个全球公共资源，对所有人开放和可访问”)今天和 25 年前 Mozilla 成立时一样重要，甚至可能更重要。

为了谈论业务的现状并展望 Mozilla 的下一步，我采访了 Mozilla 的执行主席兼首席执行官米切尔·贝克、Mozilla 首席产品官史蒂夫·特谢拉(他在离开 Twitter 后最近加入了 Mozilla)和 Mozilla 基金会执行董事马克·苏尔曼。

Mozilla 的执行主席米切尔·贝克。PATRICIA DE MELO more IRA/法新社通过 Getty Images 拍摄

在谈到业务状况时，贝克指出，鉴于疫情，2021 年显然不是正常的一年。她指出，在 2022 年上半年，随着乌克兰战争和整体经济逆风，Mozilla 的财务状况看起来有些类似。但更重要的是，Mozilla 多元化的尝试开始获得回报。“2021 年，我认为我们正在做一些事情，我们打算继续做下去——多产品、不同的商业模式、吸引消费者——[……]这些确实代表了未来，也是我们开创 Mozilla 新篇章的最初步骤，”她解释道。

有了 Mozilla 的三种商业模式(隐私保护广告、订阅和搜索合作)，该组织现在更依赖于整体广告市场。但正如特谢拉强调的那样，市场可能会更加疲软，但这主要表现在增长放缓，而不是市场下跌。“对我们来说，作为一家非上市公司，这没什么，”他说。“我们可以做得很好。因为我们不需要管理公众的市场认知，所以我们需要的是一个好的业务，并让我们开心。”他还认为，虽然其安全产品目前只占 Mozilla 总收入的一小部分(虚拟专用网几乎驱动了全部收入)，但仍有很大的上升空间。Mozilla 正在考虑如何向其整体安全套件添加更多功能和产品，既可以作为 Firefox 的一部分，也可以作为独立产品。

今年早些时候，Mozilla 为其 Mozilla 开发者网络(MDN)引入了付费订阅。Teixeira 表示，它满足了该组织的“适度期望”，但也代表了 Mozilla 尝试这些商业模式的意愿。像 MDN 这样有着悠久历史的产品，Mozilla 必须小心如何管理这样的实验，因为它很容易适得其反，损害其品牌，贝克承认这一点。“我们真的下定决心要尊重 MDN，并确保我们增加的是真正的优质服务，”她说。

然而，尽管 Mozilla 继续扩大其产品供应，但对很多人来说，它的品牌仍然是 Firefox 的同义词。贝克和特谢拉都强调，Mozilla 将继续投资 Firefox，特谢拉特别强调，他认为移动领域有很多增长机会。但贝克也指出，她希望 Mozilla 能代表比这更普遍的东西。“我发现 Mozilla 品牌
激发了希望，代表了渴望，”她说。“过去几年我一直在测试这一点，因为我们知道 Firefox 是一个全球品牌——这很难获得，尤其是一个持续了 20 年的品牌——但我一直在全球测试 Mozilla 品牌，并发现了一些事情:它是一个全球品牌，不如 Firefox 知名，但比你预期的更知名。”

但她认为，人们希望从 Mozilla 获得更多。毕竟，修复 Firefox(Mozilla 可以说已经做到了)并与 Edge 和 Chrome 竞争是一回事，与微软和谷歌竞争并在很难保护你的在线隐私和安全的情况下将代理感还给用户是另一回事。“[互联网]是由少数针对我的组织控制的，旨在让我做一些事情，购买一些东西，相信一些东西，采取某种行动，”她谈到互联网的现状。“我们目前所没有的，也是我在 Mozilla 花费毕生精力的原因——我们没有那种力量为我工作，接触世界，寻找我正在寻找的东西或影响世界，或找到我想要的东西或代表我。它是针对我的，没有一个人有资源真正理解或回应或关注正在发生的事情。”然而，如果不解决网上隐私和安全问题，很难预见接下来会发生什么。“隐私和安全是必要的。我们现在就需要它们。人们意识到了它们。但我认为，如果我们这样做，它可以展示这种技术的不同模型看起来是什么样子的承诺，然后人们可以看到它并选择它，”她补充说。

今天的 Mozilla 已经远远超越了浏览器。这不一定是新的，但我认为它从来没有更清楚。这意味着 Mozilla 公司和基金会都在将注意力转移到新的话题上。Mozilla 基金会的苏尔曼说:“在这个时刻感觉很好，特别好的是，在一个从一开始就说我们需要以不同的方式做事情的组织，然后问我们自己，以不同的方式做事情对现在和未来十年意味着什么。”。“所以我们在[我们的报告]中使用了‘下一章’或‘未来 25 年’这样的措辞，这是你在这样的报告中所说的一种诗意。但也确实有这种感觉，随着基金会的重点转向人工智能，以及更广泛的问题，即以不同方式对待今天的时代和明天的技术时代意味着什么。”

最近，这意味着 Mozilla Ventures 的推出，这是一个 3500 万美元的风险基金，该组织计划用它来投资那些想要建立一个更好的、尊重隐私的互联网的产品和创始人。“作为一个公益产品组织，我们拥有巨大的杠杆，”他说。“这正在扩展到新的表面区域。但另一方面，Mozilla 最深刻的一点是这种社区方式，这种运动方式或生态系统方式。”然而，尽管 Mozilla 能够让 Firefox 在一个小团队中运行，并建立其 Mozilla 公司/基金会的双重性，但这对创始人来说很难复制。苏尔曼说:“我们遇到了创始人、从学校出来的人，各种各样的人，他们说:‘是的，我们想做一些有助于改变这种情况的事情，但我们如何才能真正朝着那个方向前进，而不是回到科技行业的错误中去。毕竟，传统的风险公司希望看到他们投资的高额回报——而且越快越好。苏尔曼强调，Mozilla Ventures 也是为了产生回报，但它可以有耐心，并投资于与它有共同价值观的创始人。明年，Mozilla 基金会将针对更广泛的技术生态系统推出几项新举措。

虽然 Mozilla Ventures 是一种方法，但 Baker 预计，我们将在未来几年看到该组织在这一领域做得更多，部分是通过与更多分享其愿景的组织合作。

“我遇到了很多人，他们都渴望丰富的互联网——但更好或更少令人毛骨悚然——并希望与一些正在这样做的组织建立联系，”她说。“我有一个假设，有一种新形式的开源社区，它应该是 Mozilla 和我们更大的运营模式的一部分。Mozilla 是什么？我们有不同的组织和员工，但越来越多的是，我们也应该隶属于、联系、代表和认同那些希望有所不同的人。”
