Finix 融资 3000 万美元，fintech 的聚光灯选择了自己的立场 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/08/10/finix-raises-30-million-as-fintechs-spotlight-picks-its-sides/

Finix 今天宣布，它已经筹集了 3000 万美元的新风险资本，使其已知筹集的资金总额达到 1.33 亿美元。这一轮融资是在 DIY 支付公司关闭对其3500 万美元 B 轮的 3000 万美元延期近两年后，以及它筹集由拉丁裔和黑人投资者领导的300 万美元 SPV 约 18 个月后。

现金部分明显小于其延长的 B 轮融资总额，但 Finix 确实包括了新投资者参与的这轮融资，这意味着这一轮不是延长轮，“由于它在过去六个月中表现出的增长”。

新老支持者普通合伙企业(TheGP)、富兰克林邓普顿(Franklin Templeton)、Acrew Capital、美国运通创投、贝恩资本创投、Cap Table Coalition、Homebrew、Insight Partners、Inspired Capital、光速创投合伙人、Precursor Ventures、PSP Growth 和 Vamos Ventures 参与了该公司的最新一轮融资。

Finix 没有披露其估值，在新闻稿中指出——尽管承认“当前的融资环境”——资本是在今年夏天筹集的,“估值增加了”TechCrunch 向 Finix 寻求进一步的评论，但在撰写本文时尚未收到回复。

这家 SaaS 初创公司的核心业务是帮助软件公司通过灵活的软件处理他们自己的支付，尽管它已经扩展成为一个直接支付服务商。该公司坐在 Stripe 等公司旁边，毫不掩饰要与之竞争。

在这家初创公司于 2020 年筹集了由红杉(Sequoia)牵头的 2020 年 B 轮融资一个月后，这家风险投资公司退出了交易，据报道，该公司向 Finix 返还了一张 2100 万美元的支票，代表其投资的全部价值以及董事会席位、信息权和股份。TC 的 Connie Loizos 当时报道说，红杉决定退出，因为它认为 Finix 与其投资组合的宠儿 Stripe 直接竞争。

今年 5 月，Finix 在 Stripe 竞争中加倍下注，宣布将通过其内部平台直接促进支付，这在历史上是无法做到的，因为它只是一个 API 提供商。它向直接便利化的过渡使其能够捕捉到低于以前最佳服务点的较小客户，交易量约为 5000 万美元。它还进入了面对面支付领域，允许不同类型的企业接受信用卡支付。正如金融科技领域经常发生的那样:范围越广越好。

这两项举措将 Finix 置于 Stripe 的地盘，尽管其首席执行官兼联合创始人 Richie Serna 告诉 TC 的 Mary Ann aze vedoFinix 与 Stripe 不同，它专注于创建一个开放的生态系统。Serna 将他的公司比作 Android，将 Stripe 比作苹果，后者以封闭其 iOS 平台而闻名。

他在 5 月份的一次采访中告诉 TechCrunch，“我们正在开发一种技术，这种技术需要几十名工程师花三年时间在内部开发，需要数千万美元的技术 R&D 和投资，并通过让开发者友好的 API 开始将他们的支付货币化，将这一过程缩短到几个月。”“这是我们最大的核心产品。我们现在所做的是自己成为支付服务商，这样我们不仅可以提供支付，还可以提供所有后台要求和合规性认证，这样我们的客户就可以在几天内启动并运行，而不是几个月。”

在宣布新现金的新闻稿中，该公司表示，就新交易而言，Q2 2022 是其有史以来最好的季度。对于金融科技公司来说，这是一个受欢迎的消息。因此，很明显，菲尼克斯的战略转变已经产生了足够的影响，让一批投资者将资金投入这家风险投资公司。
