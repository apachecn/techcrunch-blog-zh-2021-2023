如果你的笔记本电脑被抢或被盗，这种 USB“杀死线”可以立即清除你的笔记本电脑

记者、活动人士和人权维护者面临着一场持续的战斗，以保护文件安全，免受越来越多的数字威胁和监控。但是，无论是机会主义的小偷还是暴虐的政府破门而入，人身攻击都很难防范。

本周，一个名为 BusKill 的项目推出了一种定制的 USB 磁性分离电缆，它可以充当“死人开关”，如果有人物理地抢走电脑并切断磁性连接器，就会锁定电脑。

总线电缆切断时锁定计算机。(图片来源:巴斯基尔)

布斯基尔作为一个自己动手的项目已经筹备了两年多。任何有硬件的人都可以编译源代码，但它只能在 Linux 上运行，而且组件很快就销售一空。

经过的众包努力，这种电缆现在可以以 59 美元的价格购买，并且有一个配套的应用程序，可以在 macOS、Windows 和 Linux 上工作，允许使用电缆的人通过触摸一个按钮来轻松启动和解除电缆。

“大多数人不会处理来自告密者的绝密文件，并担心秘密警察会破门而入，但这就是我设计 BusKill 的风险水平，”该项目的创始人迈克尔·奥尔特菲尔德(Michael Altfield)告诉 TechCrunch。“我想让那些不一定使用 Linux、不知道如何使用 CLI(命令行界面)的记者也能使用它。”

BusKill 旨在当你的计算机与你物理分离时锁定你的计算机，但 Linux 用户可以进一步配置该应用程序来触发自毁命令，该命令会扰乱设备的加密密钥，使计算机的数据在短短几秒钟内无法访问。

该项目还计划在磁性电缆被切断时释放关闭电脑的触发器。

虽然 Altfield 说 BusKill 是为记者和活动家设计的，但该电缆也可以保护度假旅行者和其他高风险用户的计算机，如密码交易员。
