推动安全从秘密行动走向解决网络安全的人员问题 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/10/13/nudge-security-stealth/

社会工程攻击呈上升趋势。这些低技术但高影响力的攻击——黑客操纵员工授予他们访问公司服务和数据的权限——去年增加了几乎三倍,今年到目前为止已经造成了几个高调的受害者，从 Twilio 和 Mailchimp 到 Revolut ，以及最近的大多数优步。正如这些知名企业所展示的那样，即使是资源最充足的组织也很难防范这类攻击。

现在，网络安全初创公司 Nudge Security 正在悄然兴起，帮助组织解决他们认为最大的网络安全弱点:人。

这家完全偏远的公司——在德克萨斯州的奥斯汀和怀俄明州的杰克逊设有前哨——由前软件工程师拉塞尔·斯皮勒和杰米·布拉斯科于 2021 年创立，他们认为解决“人的问题”的唯一方法是让员工成为解决方案的一部分。顾名思义，其产品通过“轻推”员工采取最佳安全行为来实现这一点，例如开启多因素身份验证 (MFA)或在涉及违规时更改他们的密码。

该公司的安全产品持续揭示整个组织的历史和新的软件即服务资产，包括 SaaS 供应链和 OAuth 授权，而不依赖于网络基础设施、端点代理、浏览器扩展或 API 集成。当出现新的“安全关键”事件时，如创建新帐户或安装新应用程序，Nudge 会与该员工联系，以确保他们做出良好的安全选择。例如，如果一名员工下载 Dropbox，但该组织使用 Google Drive，Nudge 将启动一个对话，以了解为什么会做出这一决定。

Nudge 的 Spitler 告诉 TechCrunch 说:“我们作为一个助手，允许员工与安全团队合作，并允许集中的团队仍然可以看到正在发生的事情，制定政策，并让员工以不中断他们工作的方式参与这个过程。”“我们相信每个员工都有潜力以支持和加强组织网络安全态势的方式行事，只是这样做并不总是简单或直接。”

为了确保员工参与这些提示，Nudge 与杜克大学心理学教授 Aaron Kay 合作，他向这家初创公司展示了如何利用心理学方面的基础研究来建立我们的产品和最终用户之间的关系。“我们正在努力让员工参与进来，并确保我们不会以一种拍拍你的手或挥舞红色警告横幅的方式出现，”斯皮勒补充道。

Nudge 并没有声称它可以阻止优步的黑客或 Revolut 的入侵——斯皮勒告诉 TechCrunch，“我们在这个行业已经呆了太久，无法做出这样大胆的案例”——但该公司认为，它可以帮助组织告知他们的风险状况，不仅是在谁有权访问方面，而且在谁有权访问什么以及为什么有权访问方面。

斯皮勒说:“就像优步的情况一样，过去几个月的崩溃趋势之一就是这些组织的复杂性。”。“社会工程加上复杂性意味着，即使一个用户受到威胁，整个组织也会突然分崩离析。”

“我们还提供供应链信息，”Nudge 的联合创始人兼首席技术官布拉斯科补充道。“假设您的组织正在使用 Slack，而他们正在使用 Twilio，我们可以告诉您 Twilio 受到了威胁。”

Nudge 在获得 Ballistic Ventures 700 万美元种子投资六个月后推出了产品，Ballistic Ventures 是一家新的风险投资机构，专门为早期网络安全初创公司提供咨询和资金。自从这笔投资以来，Nudge 已经接纳了 10 家客户，还有十几家处于大型企业试点阶段。

“我们本周将推出的产品确实是我们目前的重点，然后我们将加大营销和销售力度，”斯皮勒说。“当我们开始在这方面扩张时，我们可能会寻求另一轮融资。”
