谷歌选择南非作为其在非洲的第一个云区域

原文：https://web.archive.org/web/https://techcrunch.com/2022/10/05/google-picks-south-africa-for-its-first-cloud-region-in-africa/?scid=6pCB_biezs

科技巨头谷歌今天宣布在南非推出云区域，这是其在非洲大陆的第一个云区域，赶上了其他顶级提供商，如亚马逊网络服务(AWS)和微软 Azure，后者几年前就进入了非洲大陆。

谷歌表示，它还在内罗毕(肯尼亚)、拉各斯(尼日利亚)和南非(开普敦和约翰内斯堡)建立专用的云互联网站，将用户的内部网络与谷歌的网格连接起来，以寻求为非洲的客户和合作伙伴提供全面的云能力。

谷歌计划利用其连接非洲和欧洲的私有海底电缆 Equiano T1 为这些站点供电。Equiano 自 2019 年以来一直在开发中，迄今已在多哥、纳米比亚、尼日利亚和南非进行了四次着陆。

南非现在加入了谷歌在全球 35 个云区域和 106 个区域的全球网络，该声明是在最近在马拉西亚、泰国和新西兰区域的预览发布之后发布的。谷歌云区域允许用户从特定的地理位置部署云资源，并访问包括云存储、计算引擎和密钥管理系统在内的多种服务。

“我们很高兴宣布在非洲建立第一个谷歌云区域。新区域将允许应用程序和服务的本地化。这将使我们的客户和合作伙伴更容易为他们的业务快速部署解决方案，从而他们能够利用我们的计算机人工智能或机器学习能力以及数据分析，在前进的过程中做出更明智的商业决策，”谷歌云非洲总监 Niral Patel 说。

他补充说，新的区域和互联网站将使其云计算服务更接近其客户，允许其客户选择在哪里消费产品。

“我们在这里所做的是让客户和合作伙伴选择他们希望在哪里存储数据，以及他们希望在哪里使用云服务，特别是在数据主权的背景下。这使得客户可以在他们选择的国家存储数据…我想对我来说最重要的因素是它给了客户选择的机会，”Patel 说。

随着肯尼亚等国家实施隐私和数据法，要求公司将数据存储在境内，并通过本地托管的服务器进行处理，用户选择存储数据的能力变得越来越重要。

在南非设立区域的决定是基于对云服务的需求和市场潜力。尽管如此，随着对其产品需求的飙升，该公司正寻求在非洲大陆的更多市场推出产品。它的早期采用者包括大型企业和电子商务公司，如南非的 TakeAlot 和肯尼亚的 Twiga。

“我们将继续评估市场需求，与客户一起见证他们在这些市场的转型和发展。我们不断进行这些评估，并在此基础上继续投资，”帕特尔说。

根据谷歌云委托 AlphaBeta Economics 进行的研究，到 2030 年，南非云地区将为南非的国内生产总值贡献超过 21 亿美元，并支持创造超过 40，000 个就业岗位。

根据 Gartner 的数据，谷歌云、微软 Azure 和 AWS 是世界上三大公共云存储公司，但目前还不清楚为什么谷歌一直没有在非洲出现。

微软于 2019 年在南非推出了两个云区域:开普敦和约翰内斯堡(只有后者的云区域仍然活跃)，同年谷歌宣布“没有在非洲建立云区域或数据中心的计划”，根据这份报告；然而，这并不排除它在可预见的未来会发生。

亚马逊在 2020 年紧随其后，通过开普敦将其 AWS 数据中心扩展到南非。另一家主要厂商甲骨文公司今年也在约翰内斯堡建立了数据中心。在回答谷歌是否正在追赶其他云存储公司时，Patel 和谷歌非洲的董事总经理 Nitin Gajria 描绘了一幅画面，其中每个主要公司都关心通过其数据中心扩大非洲的互联网生态系统，而不是争夺更大的市场份额。

Gajria 说:“就我们在互联网方面所处的位置而言，要做的工作是考虑我们如何让更多的人和企业上网，我们如何帮助更多的企业家获得资本等等。”。“用商业术语来说，这不仅仅是一场市场份额零和游戏，而是我们如何在私营部门、公共部门和公民社会中共同努力，建立一个庞大、充满活力的互联网生态系统，帮助扩大经济和商业，并创造就业机会。”

随着谷歌的推出，南非现在拥有了非洲大陆四大云存储提供商。
