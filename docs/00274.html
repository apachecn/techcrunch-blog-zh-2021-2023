【2021 年两个月后，南非金融科技公司 Stitch 走出了秘密，并筹集了 400 万美元的种子资金，使其能够在几个月前开始进入尼日利亚。

今天，Stitch 正式宣布向尼日利亚扩张，并为此提供 200 万美元的种子扩展资金；种子投资总额现已达到 600 万美元。

该轮投资者包括现有支持者 Raba、Firstminute Capital、CRE、Village Global、500 Fintech、Future Africa 和 Norrsken。 新投资者、 主要是全球金融科技公司的 高管也参与其中。他们包括 Monzo 和 GoCardless 的联合创始人汤姆·布洛姆菲尔德(Tom Blom field)；GoCardless 联合创始人马特·罗宾逊；比特币基地总统兼首席运营官 Emilie Choi 以及 ComplyAdvantage 的创始人查理·德林波尔。

今年 2 月，Stitch 推出了全球金融科技基础设施参与者使用的典型格子剧本:一种数据产品，帮助开发人员和金融科技人员将其应用程序与用户的金融账户联系起来。然后，在用户同意的情况下，这些企业可以访问他们的身份、账户、交易历史和余额。

因为 Stitch 刚刚秘密启动了和，所以该公司透露它已经连接了多少个金融账户还为时过早。九个月过去了，该公司仍不热衷于分享这些数据，尽管它一直在测试一款支付产品，并取得了重大进展。

非洲的金融科技大多数时候依赖于信用卡支付，尽管有 Yoco、Flutterwave 和 Paystack 等公司提供的基础设施，但仍存在退款费用和欺诈成本等问题。

在南非，Stitch 决定解决这些问题，并通过试运行其银行支付功能来帮助企业更好地处理支付。

“一旦你让银行支付的用户体验更加顺畅，就像信用卡一样可识别，选择使用这种方法(特别是对金融科技公司来说)就相当简单了，”Stitch 的联合创始人兼首席执行官基安·皮莱(Kiaan Pillay)告诉 TechCrunch。

Pillay 表示，在南非试运行的六个月内，该公司所有解决方案的客户和支付量环比增长超过 50%。

在尼日利亚，Stitch 也一直在测试其支付产品，而不是其主要数据产品。 据首席执行官称，在与 40 多位 fin tech交谈后，该公司在尼日利亚有了一个更令人信服的支付产品使用案例。

原因如下:尼日利亚拥有高效的即时银行结算系统，在各种金融科技应用中，通过银行支付的选项相对标准 。问题是，当客户试图通过他们的移动或互联网银行应用程序支付时，他们必须最终使用他们的移动银行应用程序完成交易。

该公司表示，有了 Stitch、 和 ，用户就不需要经历那些麻烦，而且可以 发起 一次性、重复且用户不在场的银行转账。

“时机已经成熟，”皮莱说，他与 娜塔莉·库斯伯特 和 普里安·皮莱共同创办了史迪奇。我认为 越来越多的 API fintechs 认识到，在将客户转化到他们的平台上时，通过信用卡交换支付会带来高额费用。 就在 仅这个月，我们就看到南非的呼入请求增加了五到十倍，尼日利亚的客户也在整合我们，尽管我们之前仍处于试运行阶段。”

现在完全住在南非和尼日尼亚，史迪奇说已经走上正轨促成 1000 万美元的月付款 。它的一些客户包括 Chipper Cash、Paystack、Franc、Sanlam、Yoco 和 Flexclub。

去年，当开放式银行初创公司进入非洲金融科技领域时，似乎有一场成为非洲格子 的竞赛。 但现在，每个玩家都在各自的市场定居——肯尼亚和尼日尼亚的 Pngme 尼日利亚和加纳的单核细胞增多症；尼日利亚的秋葵，南非和肯尼亚的贝塔；在南非和尼日利亚缝纫。

很明显，尼日利亚作为这些初创公司的市场有多重要。那么，在其他竞争对手提供相同支付产品的市场上，Stitch 如何计划 获得 客户呢？

“首先，我们向打算使用 Stitch 的尼日利亚企业提供免费产品，直到 2021 年底，”Stitch 尼日利亚的国家经理 Benjamin Dada 告诉 TechCrunch。

达达将领导 Stitch 在尼日利亚的业务。在加入公司之前，他是尼日利亚科技出版物Benjamindada.com的创始人和自由编辑。这位国家经理的金融科技经验来自与尼日利亚数字银行 Eyowo 的合作，以及在 Tiger Global 支持的 FairMoney 的短暂工作。

虽然达达认为市场足够大，可以容纳多个玩家，但他表示，该团队将努力通过为客户开发“包容性和可持续”的价格来超越其他人。

“他们不仅可以通过接受账户支付来实现成本节约，而且他们现在可以通过 Stitch 接受的每笔支付多赚一点，”他补充道。

该公司打算如何做到这一点，目前还不清楚。不过，可以确定的是，该公司计划在 11 月底 推出其数据解决方案。seed 扩展将是其中不可或缺的一部分，并会看到更多的员工加入 Dada 和他在尼日利亚拉各斯的团队 。

Blomfield 在一份声明中表示，他支持 Stitch，因为他认为该公司在建设基础设施以实现非洲数字金融公司的指数增长方面发挥着至关重要的作用。

“我在非洲市场看到了很大的潜力，那里的数字金融创新浪潮正在 真正 开始获得动力，Stitch 团队正在 准确地说是 正确的时间进入。这个团队是我在全球见过的最好的团队之一，我很高兴看到他们在尼日利亚和其他地方继续发展。”
