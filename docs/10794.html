Easop 帮助您向国际员工授予股权 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/10/04/easop-helps-you-grant-equity-to-international-employees/

认识一下 Easop ，这是一家新的软件即服务初创公司，帮助公司管理多个国家的员工股权激励。Easop 特别适合那些希望雇佣远程员工并为他们提供股权的初创公司。

如果你在一家科技初创公司工作，并且负责招聘，你知道找到合适的候选人会非常困难——如果你只寻找本地候选人，那就更困难了。这就是为什么现在越来越多的公司雇佣远程员工。新冠肺炎疫情促成了这一趋势。

向不住在公司总部附近的国际员工提供带福利的本地合同也比以往任何时候都容易。像 Deel 、 Oyster 和 Remote 这样的初创公司可以让你将团队扩展到多个国家，而不必在每个新国家都设立子公司。

然而，这一过程产生了一系列全新的障碍。例如，如果没有合适的子公司，向所有人提供相同的员工持股计划(ESOP)几乎是不可能的。

由于监管框架因国而异，因此存在一些法律上的复杂性。至于员工，他们不知道应该如何报税。他们甚至可能不知道他们赠款的价值。

Easop 希望为所有员工和承包商创造相同的体验。它直接连接到流行的 cap 表管理服务，如 Carta 和 Pulley 。

但 Easop 不想重新发明轮子，取代那些现有的解决方案。当 Easop 中发生股票事件时，它从这些服务中提取股票数据，并与 Carta 或 Pulley 同步。

与此同时，Easop 还与 Deel、Oyster 和 Remote 等人力资源信息系统同步，以获取国际员工的更多信息，如他们的居住国和开始工作的日期。

当 Easop 的客户希望向新员工提供资助时，他们可以在 Easop 中起草资助并将其发送给董事会进行验证。Easop 确保拨款符合当地法规和税收规定。

对于未来的员工，他们可以连接到 Easop 的网站，查看他们的资助状态。当应纳税事件发生时，Easop 可以生成年度税务报告表格，或者直接与某些国家的当地薪资系统集成。

Easop 已经筹集了 250 万美元的种子资金，由 Partech 和 Boldstart Ventures 牵头。基马风险投资公司、eFounders partners Thibaud Elziè re 和 Quentin Nickmans 以及几位商业天使也参加了这轮融资。

这个想法是 Easop 应该为公司的每个人工作——在美国为总公司工作的人，在另一个国家的子公司工作的人，以及依赖于像 Deel 这样的记录雇主(EoR)的国际团队成员。

我认为为每个人建立相同的平台是正确的方法。如果你试图建立一个真正分布式的团队，你应该能够使用相同的工具，不管你在哪里。
