海岸线获得 3500 万美元的 B 轮融资，用于构建自动化事件响应平台| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/03/28/shoreline-scores-35m-series-b-to-build-automated-incident-response-platform/

Shoreline 创始人兼首席执行官 Anurag Gupta 在创办自己的公司之前，在 AWS 负责基础设施长达八年。他负责确保有适当的系统来应对使 AWS 系统变慢或停止的事件。

这是一项艰巨的工作，虽然他帮助建立了内部系统来自动响应事件，但他发现市场上缺少帮助其他公司实现这一目标的工具。虽然有工具可以测试和部署软件，在生产中监控软件，并在事故发生时管理事故，但在他看来还缺少一块。

他指出，一旦生成了事故单，并且必要的人已经到齐，找出问题所在并解决问题通常就变成了一个非常手动的过程。系统停机的每一分钟都代价高昂。随着软件和系统变得越来越复杂，负责处理这些问题的现场可靠性工程师(SREs)要找出根本原因并修复它变得越来越困难。

“几乎在所有情况下，它都变成了一个手动过程，这就是人们精疲力竭的地方。这就是他们犯错的地方。这就是需要大量劳动的地方。这也是停机的原因，因为与机器相比，人需要花很长时间来修理东西，”古普塔解释道。

该公司已经开发了 Jupyter 风格的笔记本电脑，用于记录和自动响应给定系统的常见问题，提供解决问题的分步指导，同时尽可能自动响应。这样做的目的是帮助缓解当时的压力。

古普塔说，SRE 函数正呈指数级增长，以满足不断增长的解决系统问题的需求，但他说，简单地把尸体扔向问题不是一种可持续的方法。

Insight Partners 的管理合伙人乔治·马修(George Mathew)的公司正在投资海岸线，他说，这是关于让机器和人类一起更快地解决问题。

“但是，让更高级别的功能由人类完成，让更低级别的功能由机器学习算法自动化，这是我们在这个领域看到的一个令人信服的机会，”马修谈到他为什么投资该公司。

该公司于 2019 年推出，但花了两年半的时间来建立这样的自动化解决方案，由 2200 万美元的 a 轮融资。该产品已经上市约六个月，海岸线已经拥有近 50 名员工。

古普塔说，在建立公司的过程中，拥有一支能够反映公司运营环境的员工队伍是他的一个关键目标。

“我坚信，公司里的人应该看起来像整个社会，而不是技术社会，因为这已经存在系统性偏见，”他说。这意味着努力实现与实际人口比例相匹配的员工比例。

“我相信，如果你在招聘过程中引入多元化，你的公司最终也会多元化，”他说。

今天，海岸线宣布了一笔 3500 万美元的 B 轮投资，由 Insight Partners 领投，黎明资本参与其中。据该公司称，这笔投资使融资总额达到 5700 万美元。
