南非的 Revio 允许企业连接多种支付方式并减少故障 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/11/25/revio-allows-african-businesses-to-connect-to-multiple-payment-methods-and-reduce-payment-failures/

据报道，非洲十分之三的支付失败了。这背后的因素从分散的支付格局和无效卡到休眠账户和更高的争议率；它们每年都会出现，导致整个大陆的数字业务经常性收入损失 140 亿美元。

根据一些报告，随着非洲数字支付的持续增长，这些问题注定会增加，年增长率为 20%。虽然网关和聚合器使企业更容易接受多种支付方式，但很少有解决方案出于必要将它们聚合起来，并处理每个平台出现的支付失败。这就是南非 API 支付和收款公司 Revio T1 的用武之地。fintech 使非洲各地的企业更容易连接到多种支付方式并管理支付故障，该公司宣布已筹集到 110 万美元的种子资金。

金融科技投资者 Speedinvest 领投了这轮融资，Ralicap Ventures、该基金和两家文化资本参与了融资。根据这家初创公司分享的一份声明，一些天使投资者也参与了进来，包括来自红杉、Quona Capital 和 Circle Payments 的支付和收入回收专家。

Revio 于 2020 年由鲁安·博塔创立。作为一名在南非银行和保险业工作了十多年的专业人士，Botha 在看到企业在吸引客户支付未付款和失败付款方面花费了大量时间和人力后，决定推出 Revio。很明显，很少有公司对收入恢复进行有意义的投资。当询问超过 25 名客户，如果他们必须修复他们的支付系统，他们会在哪里投资 1 美元时，大多数人说他们会将至少 90%的钱用于管理支付故障和客户流失。

“我们将借记卡作为南非最大的经常性支付方式。但是，当企业想要开始增加其他不同的支付方式来满足客户需求时，他们很难做到这一点，”博塔在接受 TechCrunch 采访时表示。“正是因为银行、新型金融科技和支付聚合商之间的脱节，也使得企业难以持续获得经常性收入。因此，通过 Revio，我们希望让企业能够非常简单地连接他们需要的任何支付方式，不仅在南非，而且在非洲其他地区和全球范围内。”

除了博塔之外，还有三名管理公司事务的高管:首席商务官彼得·格罗贝拉(Pieter Grobbelaar)是 Flutterwave 的前国家领导人；首席技术官凯尔·泰特斯，他拥有与金融科技公司和风险投资工作室合作的经验；首席产品官Stefan Griesel，拥有八年以上 fintech 产品经验；和首席运营官 妮可·邓恩 ，一位与多家非洲初创公司合作过的风险投资家和经营者。

邓恩在与博塔一起与 TechCrunch 通电话时表示，Revio 在非洲聚合并协调了多种不同的支付方式，包括卡、银行转账、借记卡、移动货币、代金券和二维码。该平台通过 Flutterwave 、 Paystack 、 Ozow 和 Stitch 等支付提供商在 40 多个市场收集和结算支付。除了多种支付方式之外，它的一些功能还包括智能支付路由、自动化计费流程、自动退休以及实时分析和报告。

首席执行官鲁安·博塔。图片来源: Revio

在一年多的运营中，Revio 已经接纳了 50 多个客户，每月处理数千笔交易。从大型企业到中端市场企业，以及涉及经常性收入业务和高交易量的快速增长型企业，通常需要在多个市场中使用多种支付方式。这些企业通常是保险公司、电信公司、零售商、订阅软件或媒体、资产租赁或融资企业以及另类贷款机构。

Dunn 说:“我们还建立了协调能力，通过智能交易路由、智能重试等方式来减少支付失败，以确保客户不会拖欠，特别是在经常性支付方面。“我们与众不同的地方在于，我们为有经常性收入的企业服务，而不是典型的电子商务平台。”她补充说，Revio 有超过 100 个客户在等待加入。

在当今世界，支付协调正变得越来越重要，因为企业在多个国家开展业务，需要一系列支付方式来维持运营。虽然在美国和欧洲已经有一些这样的平台通过统一的支付 API 来处理这种繁重的工作，如 Primer、spreadly和 Zooz ，但发展中市场的企业开始看到相同的平台，如 Revio 和总部位于埃及的 MoneyHash 在各个地区占据中心位置。

在竞争和如何脱颖而出的问题上，Revio 声称它是第一个专注于支付失败和收入恢复的非洲支付平台。“与市场上的其他平台相比，我们在撒哈拉以南非洲地区也有更多的功能和覆盖范围，”邓恩补充道。无论如何，per 报告称，全球支付协调市场正在快速增长(根据一项研究,市场规模预计到 2030 年将达到 65.2 亿美元，从 2022 年到 2030 年将以 24.5%的 CAGR 增长),对于新平台来说，有足够的空间来抢占市场份额——以及 Revio 等现有公司来深化其影响力。

这是这家成立两年的金融科技公司筹集这笔资金的原因之一:进军非洲内外的新市场，在此过程中扩大团队，并为其日益增长的客户推出新产品。

博塔说:“我认为使用投资是双重的。“一个是获得更多关于机器学习和数据的战略技能，以帮助我们发展和推动与客户的更好互动，了解他们失败的原因以及如何获得更好的回应率。有了这些数据，我们可以开始在非洲的一些核心市场进行实验。未来 18 个月，我们希望在大约 13 个非洲国家开展业务，但重点是三四个大型市场。然后，获得足够的牵引力，使我们能够进入拉丁美洲等其他新兴市场。”
