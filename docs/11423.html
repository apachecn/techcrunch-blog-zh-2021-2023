微软称 GitHub 现在拥有 9000 万活跃用户，1B 澳元

原文：https://web.archive.org/web/https://techcrunch.com/2022/10/25/microsoft-says-github-now-has-a-1b-arr-90m-active-users/

微软表示，GitHub 目前拥有 9000 万活跃用户，价值 2000 万美元

作为其收益电话会议的一部分，微软今天宣布了 GitHub 的一些新数据点，这是一项非常受欢迎的代码库服务，它在 2018 年被以 75 亿美元收购。据微软称，GitHub 现在的年经常性收入为 10 亿美元，从收购时报道的 2 亿美元增加到 3 亿美元。

该公司还宣布，该服务现在在该平台上有超过 9000 万活跃用户，高于收购完成时的 2800 万人和去年 11 月托马斯·多姆克取代纳特·弗里德曼担任该服务首席执行官时的 7300 万人。

这标志着微软第一次分享关于收购完成的服务的财务数据。

“自从我们收购以来，GitHub 现在每年的经常性收入为 10 亿美元，GitHub 的开发者至上的精神从未如此强大，”微软首席执行官塞特亚·纳德拉在今天的财报电话会议上说。“现在有超过 9000 万人使用这项服务在任何平台上为任何云构建软件——增加了三倍。”

在很大程度上，微软让 GitHub 成为 GitHub，因为收购已经完成。早期，许多开发者——尤其是开源倡导者——担心微软会改变服务的运作方式，减少免费服务，以便从中榨取更多的钱。但是，GitHub 扩展了它的免费服务，并继续拥抱开源和开源开发者。与此同时，如果没有微软的帮助，像 GitHub Copilot 这样的项目可能是不可能的。虽然一些用户转向了 GitLab 和其他服务，但新用户的数量说明了一切。
