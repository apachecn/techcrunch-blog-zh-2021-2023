AWS 推出 Application Composer，这是一款用于构建无服务器应用的低代码工具

在今天的发布会上， AWS 宣布推出 AWS Application Composer，这是一款新的低代码工具，用于可视化设计和构建无服务器应用。该服务为开发人员提供了一个可视化画布和一个简单的拖放界面，以创建应用程序架构、连接他们的资源和设计他们的功能。

正如 AWS 首席技术官沃纳·威格尔在他今天的主题演讲中提到的，许多开发者都想开始构建无服务器应用，但对许多人来说，门槛仍然太高。在某种程度上，这是因为你必须以不同的方式考虑无服务器应用程序，因为它们通常是异步系统(这是沃格尔斯今天主题演讲的主题之一)。因此，该团队开始使用 Application Composer 来改进这一点。

图片来源: AWS

“有时开发人员选择同步系统是因为它很方便。他们看起来简单多了，”他说。“在同步系统中，你拥有所有这些组件。在事件驱动的[架构]中，它们如何一起工作有时看起来有点令人生畏。所以我们一直在思考如何简化这个问题。例如，对于以前从未使用过无服务器的开发人员来说，如何使它变得更容易呢？你怎么知道从哪里开始？他们需要哪些服务？他们是如何合作的？我们真的想让这变得更容易。”

例如，使用 Application Composer，开发人员可以轻松构建执行标准转换任务的功能，然后只需点击几下鼠标即可部署这些功能。

有一个出口，因此开发人员也可以获得这些代码，并在他们选择的 IDE 中继续工作。但正如沃格尔斯指出的，因为这是一个视觉系统，现在与同事分享代码和合作变得容易多了。
