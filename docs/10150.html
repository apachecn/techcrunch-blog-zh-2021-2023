Polywork 获得 2800 万美元的额外收入，为你的职位添加连字符 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/09/15/polywork-lands-28-million-series-b-professional-network/

不要试图让聚合工作发生。已经发生了。当你读这篇文章的时候，你甚至可能正在进行多重工作。

Polywork 是一家由风险投资支持的初创公司，它正在建立一个专业网络，帮助人们表达他们的多重职业生活。换句话说，如果你是一名记者，也是播客，想写一本书，并希望有一天能在大学教书，Polywork 想给你，嗯，在这种情况下是我，一个表达这种想法的地方。据首席执行官兼创始人彼得·约翰斯顿说，这个名字听起来像一个动词“那种工作”

Polywork 今天宣布，它已经筹集了 2800 万澳元的 B 轮资金，由前 GitHub 首席执行官 Nat Friedman 和 Caffeinated Capital 共同牵头。其他投资者也参与了该轮投资，包括 Instacart 首席执行官 Fidji Simo 以及 Instacart、Stripe、Lyft、Clubhouse、Lattice、Minted 和 Divvy Homes 的创始人。主导 poly work 1300 万美元 A 轮融资的 Andreessen Horowitz 也参与了这轮融资，但没有领投。

这一轮标志着 Polywork 正式推出私人测试版。虽然 Polywork 创始人兼首席执行官彼得·约翰斯顿拒绝透露具体的用户数量，但除了他们正在增长的事实之外，这表明了对该平台为用户提供表达自己的场所的能力的信心。

图片来源: Polywork

换句话说，这家初创公司认为自己已经找到了适合市场的产品。自首次推出以来，Polywork 跟踪了谁注册了网站，以更好地了解他们在进入网站后想要做什么——除了自我表达。约翰斯顿解释说，行动号召很快就变成了帮助人们寻找相互合作的机会。

“如果 LinkedIn 是一个全职机会的网络，我们就是一个合作机会的网络，”他补充道。

并非所有功能都将对公众开放。例如，Polywork 正在推出私人测试版俱乐部，这是一个旨在帮助人们合作的专用小组空间。想象一下一群人通过基于徽章的进入和经验证明而聚集在一起。每个小组都有一个主持人来促进小组内的合作。

该公司的早期用户大多是全职从事技术工作的人，他们希望拓宽自己的兼职工作，如天使投资人、播客作者或时事通讯作者。这位创始人解释说，一些人认为多重身份工作的兴起只是美化喧嚣文化的又一个例子。对此他说，“这一代人绝对从做更多的事情中获得能量；这不是让他们疲惫不堪，也不是让他们筋疲力尽。”与此同时，一些报告显示，千禧一代和 Z 世代的员工工作倦怠率最高，分别为 59%和 58%。其他报道称越来越多的美国人从事第二职业来对抗通货膨胀，与其说是他们的热情。与此同时，超过 5000 万人，据 Fast Company 称，认为自己是创造者。

这种相互冲突的趋势既体现了对流动性更强的职业网络的需求，也体现了让多重身份工作变得更加突出的现实。

约翰斯顿说:“他们的专业知识赋予了他们同时尝试多种事情的能力。”“很多事情确实回到了金钱和人们想要额外收入的问题上，但实际上更多的是从人们需要和想要从中获得能量开始的——打破了他们在疫情期间工作的事实……这使他们能够以一种轻量级的方式尝试一些新的东西。”

Polywork 没有收入计划，因为它更关注增长和产品。未来的盈利可能看起来像用户 Polywork 模板的定制或类似 LinkedIn premium 的高级搜索。

Polywork 正面临着任何市场都可能面临的挑战。如果它的价值主张是更多的合作机会，那么它如何不仅能吸引想要图书代理商的人，还能吸引足够多的图书代理商，让阅读成为一种选择呢？每个人都可能在寻找播客的共同主持人，但如果每个人都有自己的想法，不一定要与他人合作，那该怎么办？

合作的未来是丰富而复杂的。但是，聪明的是，Polywork 将其焦点定位为一个着眼于未来的职业网络，挖掘人们超越过去或仅仅是现在的希望和梦想。
