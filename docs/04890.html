由于中国 Covid 案件激增，特斯拉关闭上海 Gigafactory 两天

原文：https://web.archive.org/web/https://techcrunch.com/2022/03/16/tesla-closes-shanghai-gigafactory-for-two-days-as-covid-cases-spike-in-china/amp/

图片来源: Getty Images

特斯拉在上海的超级工厂已经停工两天，原因是中国的奥米克隆案件增多，促使政府加强了对那里的限制。

据查看了内部备忘录的路透社报道，汽车制造商周三向员工和供应商发出通知，告知他们工厂关闭。

这家电动汽车制造商没有证实周三和周四暂停生产的原因。然而，暂时停产之际，全球最大的两家汽车制造商丰田和大众本周也停止了在中国的业务，原因是新冠肺炎病例在当地增加，以及政府为管理激增而实施的额外限制。

也有可能是供应链的限制导致了停产。

昼夜不停的工厂是特斯拉全球运营的关键，也是其底线。上海超级工厂是特斯拉产量最大的工厂，向欧洲出口了大量 Model 3 和 Model Y 汽车。该工厂每天生产约 2000 辆汽车，因此即使停工两天也可能大幅降低特斯拉的产量，并进一步推迟交付。

该病毒在中国再次飙升，今年前三个月的病例数超过了 2021 年的总病例数。自 2020 年 3 月疫情抵达以来，每日新增病例数量已开始达到前所未有的水平。

在整个疫情，中国政府实施了大规模检测和隔离，以遏制传播。
