尼日利亚银行即服务平台 Maplerad 融资 600 万美元，由彼得·泰尔梵拉风险投资公司领投

原文：https://web.archive.org/web/https://techcrunch.com/2022/10/17/nigerian-banking-as-a-service-platform-maplerad-raises-6m-led-by-peter-theils-valar-ventures/

过去 18 个月，银行即服务(BaaS)平台在金融科技领域迅速发展。通过与银行合作，这些平台允许从初创公司和金融科技公司到大公司和银行的实体为客户提供量身定制的银行服务和体验。

在美国和欧洲提供 BaaS 服务的金融科技公司，如 Unit 、 Rapyd 和 Treasury Prime ，由于其在市场上享有发达的银行系统，已经取得了相当大的规模。然而，他们的同行正试图在非洲等不太先进的银行系统中复制这种增长，在那里，此类产品的需求和可扩展性未经证实。

在最新的发展中， Maplerad ，一家被其创始人 Miracle Anyanwu 和o binna chukwujoke描述为瞄准非洲的全球 BaaS 玩家的金融科技公司，已经筹集了 600 万美元的种子资金。据消息人士透露，总部位于美国的 Maplerad 正在秘密进行，该轮融资估值为 3000 万美元。创始人拒绝置评。

Maplerad 作为一家公司的历程可以追溯到 2020 年，当时创始人推出了第一款产品 Wirepay 。该应用程序通过提供菲亚特和加密货币的跨境支付解决方案，帮助用户进行国际支付。然而，它后来转变为一种自我描述的一体化金融产品，使用户能够接收、持有和支付多种货币，创建虚拟和实体卡，并支付账单。

去年，Wirepay 筹集了一笔未披露的前期种子资金，包括 OnDeck 的一张 12.5 万美元的支票。金棕榈投资公司(golden Palm Investments Corporation)、温室资本(Greenhouse Capital)、一些 Stash 高管和贝里伍德资本(Berrywood Capital)也是这轮投资者。

随着 Wirepay 的用户增加到 5 万多人，主要是在尼日利亚，安延武在与 TechCrunch 的通话中表示，企业开始询问支持其消费者应用程序功能的内部基础设施。首席执行官兼首席技术官表示:“人们希望使用支持 Wirepay、我们的许可证覆盖范围和银行关系的基础设施。

Anyanwu 还提到 Maplerad(母公司)一直想把这个 infra 剥离出来用于其他业务。然而，当外部要求蜂拥而至时，他们最终安排在今年 8 月测试推出 Maplerad，这是一款基础设施产品，允许公司将账户、支付、外汇和卡等强大的金融功能嵌入其产品。

“从第一天开始，当我们为我们的消费者构建 Wirepay 时，我们就知道最终的举措将是基础设施，尽管我们没有首先启动业务基础设施。这位首席执行官表示:“任何人要开发任何与金融相关的东西，他们都必须从一大堆银行业务开始，甚至在整合功能之前，他们都必须克服许多障碍。”。“其中之一是银行关系和合规。另一个是许可。所以 Maplerad 正在为非洲的这些企业解决金融基础设施问题。我们处理整个堆栈，并提供同类最佳的 API 来帮助您在五分钟内推出金融产品。因此，你可以集成我们的 API 并投入使用，而不是一家公司花费 8 个月和数百万美元来开始构建一个金融科技产品。”

银行即服务平台已经受到试图将金融服务嵌入其产品的公司的欢迎，因为大型现有银行在使其服务跟上科技和银行业变化的步伐方面相对缓慢。因此，银行即服务平台看到了以更低成本提供更多个性化服务和灵活性的机会。这个领域正在升温 Maplerad 在尼日利亚的一些竞争对手包括 YC 支持的主播、Bloc、 OnePipe 以及更大的 fintechs，如 Flutterwave 。

在今年 8 月 TechCrunch 的一次采访中，Anchor 首席执行官 Segun Adeyemi 在被问及这家初创公司相对于其他公司的优势时，提到了他的创始团队的技术经验、对安全性和可扩展性的关注以及企业在其平台上上线的速度。

提出了类似的问题，Maplerad 的创始人提到了该平台的“更广泛的银行关系”，“首屈一指的技术”，以及“最好的机构投资者/合作伙伴”。与此同时，梵拉风险投资公司的合伙人詹姆斯·菲茨杰拉德(James Fitzgerald)在谈到这笔投资时说，尽管非洲大陆的经济日趋成熟，但很大一部分非洲人口仍然被金融排斥在外，“Maplerad 是一流的银行即服务解决方案，它有巨大的机会为企业提供金融基础设施，以快速、无缝地在非洲和全球范围内扩展。”

这家彼得·泰尔创立的风险投资公司领导了 Maplerad 的种子轮，这是继 Kuda 和黄牌之后的第三次非洲投资。本轮其他投资者包括金棕榈投资公司(Golden Palm Investments Corporation)、迈克尔沃恩(Michael Vaughn)(前首席运营官，Venmo)、金融科技基金(Fintech Fund)、巴布斯奥贡代伊(Babs Ogundeyi，Kuda) Armyn Capital、邓巴资本(Dunbar Capital)、Strawhat Investment、Polymath Capital、冷门创投、肖恩马赫索尔(Sean Mahsoul)和 MyAsiaVC。创始人说他们也把钱投资到公司。

在秘密行动期间，Maplerad 每月为其平台上收购的 100 多家企业处理数百万美元的业务，包括 Pastel、sp lete、Bridgecard、Onboardly、Vella、Crowdforce、Dojah、GetEquity 等初创公司和几家银行。它计划利用这笔投资获得更多客户，获得额外的许可证，建立自己的团队，巩固其在非洲的存在。
