大锅筹集 660 万美元建造“网络 3 的皮克斯”

原文：https://web.archive.org/web/https://techcrunch.com/2022/07/06/crypto-gaming-firm-cauldron-raises-6-6m-to-build-pixar-of-web3/

许多加密游戏公司在过去的一年里增加了产量，希望能彻底改变这个不断增长的行业。总部位于伦敦的游戏公司 Cauldron 也加入了这个名单，该公司计划通过一个关键要素成为“web3 的皮克斯”:讲故事。

该公司创始人兼首席执行官马克·沃里克(Mark Warrick)独家告诉 TechCrunch，该公司刚刚完成了一轮 660 万美元的种子期，并计划在秋季推出其第一个 web3 游戏世界 Project Nightshade。

沃里克说:“web3 游戏缺少的一大部分是讲故事，人们不会真的回来，因为没有回来的理由。”。“如果你闭上眼睛听皮克斯电影，你看不到艺术或技术，但故事仍然吸引着你。重要的是讲故事。”

这家游戏工作室是由沃里克和另外两位富有创造力的资深人士福克斯·罗杰斯和马特·海德于 2021 年 8 月创建的一个副产品。此后，该公司聘请了以科幻和奇幻为主的作家来帮助建立其讲故事的重点。

“当你在考虑 web3 中的娱乐时，你必须看看 web3 是什么:它是关于资产所有权的，”Warrick 说。“如果你不是围绕拥有宇宙的一部分来设计的，那么现有的娱乐就不能转化为宇宙的一部分。”

Warrick 指出，大锅计划向公众发布三个不同的游戏世界:两个世界有一个幻想角度，而一个世界专注于科幻小说。

“我们关注皮克斯不是因为美学，而是因为他们用《玩具总动员》创造了一个食谱，23 年后，他们从未错过，”沃里克说。“对我们来说，我们想用龙葵项目创造一个配方，并想重复它，创造许多世界。五年后，我希望有三个世界受到数百万人的喜爱，十年后，再增加四倍。”

Warrick 说，这轮融资由 Cherry Ventures 牵头，参与方包括 Cassius、Seedcamp 和 Playfair，在 140 万美元的种子前融资后仅六个月就结束了。他拒绝透露估值。

“正常娱乐中的角色是由观众开发和观察的，但想象一下一个空无一物的世界，我们把这些角色放在那里，你可以定义那些起源故事、反派、英雄，创造你自己版本的‘星球大战’、‘权力的游戏’，并看着它发展。…这是一种新的娱乐形式，”沃里克说。

沃里克说，这笔资金将用于从游戏和电影行业招聘人才，同时积累经验。

沃里克说:“我们的重点纯粹是让我们的(虚拟)世界对外面的人来说更容易接近。“我们想不出比实际通过 web3 空间更好的方式，因为人们真的想在互联网上拥有一些东西，拥有一个世界，就像拥有一点《指环王》或一点《星球大战》我们希望我们有这样的机会，现在人们有了这样的机会，但是有了新的知识产权。"
