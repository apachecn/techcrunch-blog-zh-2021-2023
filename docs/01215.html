报告称亚马逊收取超过三分之一的卖家收入，2021 年带来 1210 亿美元 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/12/01/report-claims-amazon-collects-over-a-third-of-seller-revenue-bringing-in-121b-in-2021/

一项新的研究声称，亚马逊从其市场平台上赚取的费用远远超过了被称为 AWS 的摇钱树。该报告称，亚马逊有效参与其商店的费用已经增长到了卖家现在将收入的 34%交给该公司的程度——这最近已经成为亚马逊的主要收入来源。该公司对报告的发现提出质疑。

当地自力更生研究所的报告“亚马逊的收费之路”提出了两个主要观点。首先，ILSR 的研究人员表示，2021 年亚马逊将从卖家那里以费用和广告支付的形式获得约 1210 亿美元的收入，约占这些卖家总收入的 34%。根据该报告，这是 2019 年估计 600 亿美元的两倍，当时占卖家收入的 31%。

创始人杰夫·贝索斯本人试图反驳这种说法，他告诉国会，越来越多的钱从卖家流向亚马逊是一种错觉，因为更多的人选择支付附加服务，如更好的关键词搜索排名，并使用亚马逊自己的运输和仓库基础设施。

在发给 TechCrunch 的一份声明中，亚马逊称 ILSR 的报告“不准确”，称其“将亚马逊的销售费用与我们可选的附加服务混为一谈”，其销售费用与其他在线零售商相比具有竞争力——当然，该报告确实结合了这些数字。

但正如其作者 Stacy Mitchell 指出的那样，随着亚马逊为使用这些附加服务的卖家提供了一个又一个优势，这些附加服务已经从可选变成了必备。过去几年的报告显示，普通产品搜索上的广告和赞助列表数量大幅增加。亚马逊向使用“由亚马逊履行”服务的卖家提供分数奖励，这对于一个产品是否能在列表中获得某些令人垂涎的位置有很大的帮助。而且这还没有考虑复制成功产品的黑幕业务。

亚马逊没有提到卖家今天在广告和植入上的支出是 2016 年的 4-5 倍，这有助于收入的大幅增长。该公司只是说，有一系列的广告类型和过程，这是“卖家帮助提高其产品知名度的一个很好的方式。”它否认在搜索结果中偏向 FBA 用户，尽管正如上面的链接所示，它似乎是通过间接的方式这样做的。

该报告提出的另一个说法是，亚马逊正在使用创造性会计来掩盖卖方费用产生的巨额收入，将市场部门的巨额利润与建设运输基础设施产生的巨额损失归为一类。当然，它们是相关的——但是很难呈现两个完全不同的数字的总和，并声称它准确地代表了业务。这不是一个新的指控，但米切尔提出了 2020 年的具体数字，使它不仅仅是一个笼统的想法。

“我们的结论是，卖家费用可能比 AWS 产生更多利润。这与对该公司的传统看法相矛盾；新闻报道通常将 AWS 描述为亚马逊大部分收入的来源，”米切尔在摘要中写道。“根据分析师对亚马逊可能从卖家广告和其他卖家费用中获得的利润的估计，我们发现 Marketplace 可能在 2020 年产生了 240 亿美元的运营利润，远远超过亚马逊为 AWS 报告的 135 亿美元的利润。AWS 一直被视为亚马逊的摇钱树。但这份报告发现，这家科技巨头还有第二棵摇钱树，它一直悄悄地隐藏着。"

亚马逊告诉我，它“无法推测”今年 2021 年的营收数据，但没有回答随后的一个问题，即 ILSR 报告中前几年的数据是否准确。

其中一些做法正受到各种政府权力的审查，包括由现在可能是世界上最著名的亚马逊商业行为质疑者 Lina Khan 领导的 FTC。ILSR 的报告仅仅是信息性的，亚马逊可以置之不理，但如果 FTC 的一个工作组正在调查类似的问题并得出类似的结论，该公司可能有理由开始担心了。
