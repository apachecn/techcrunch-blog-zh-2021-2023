美国航天局奖励太阳帆项目$2M 登月补助金

原文：https://web.archive.org/web/https://techcrunch.com/2022/05/25/nasa-awards-2-million-to-moonshot-solar-sail-project/

太阳帆的概念并不特别新——只要看看行星协会的 LightSail 2 ，法国初创公司 Gama 或 NASA 的先进复合太阳帆系统就知道了。但是一组研究人员正在将这项技术推向新的高度。

衍射太阳能帆船项目在 NASA 创新先进概念(NIAC)项目中获得了第三阶段的地位，该项目拥有 200 万美元的预算来将团队的概念变为现实。

太阳帆是一种航天器推进方式，其工作原理类似于船上的帆，但不是由风驱动，而是由光驱动。衍射太阳帆，如 NIAC 正在开发的，是传统太阳帆的更有效的演变，有一天可以帮助太空任务。

“衍射帆是基于衍射的光学机制，而我们看到的传统太阳帆是基于反射定律，”约翰·霍普金斯大学应用物理实验室的项目负责人 Amber Dubill 告诉 TechCrunch。

像以前一样，光子撞击船帆将它们的动量传递给航天器，但杜比尔的团队正在研究这项成熟技术的新变化:

使用绕射代替反射的优点是能够调整入射光的转向角度，使得合成力对于轨道机动更有效，而不必固定大而薄的结构。

衍射太阳能帆船项目已经完成了 NIAC 计划的第一阶段和第二阶段，这是美国宇航局的高风险、高回报孵化器探索太空中不寻常技术的可能性。第一阶段致力于开发概念和证明它们的潜力。然而，第三阶段将概念转化为现实。(题外话:自 2012 年 NIAC 项目启动以来，只有五个项目获得了第三阶段的资格，因为大多数项目在第一阶段和第二阶段都没有足够的潜力让 NASA 继续推进。)

杜比尔和她的团队将使用 200 万美元的资金来设计和制造 sail 材料，他们计划在未来两年内在全国各地的不同设施进行测试。

“同时，我们计划通过建立帆的最佳轨迹和姿态控制来发展太阳极地轨道飞行器帆任务的愿景，以实现我们的太阳物理学家指定的有效载荷套件的太阳观测，”杜比尔说。“通过扩展衍射光帆设计和发展整个光帆概念，我们的目标是利用衍射光帆技术为未来的演示任务奠定基础。”

太阳极地轨道器任务是美国宇航局特别感兴趣的，因为我们的恒星的极点尚未被探索。“获得日冕和表面磁场的完整图像对空间天气预报和意识至关重要，一系列绕太阳航行的仪表化帆不仅会提高我们对离我们最近的恒星的了解，还会增加对可能损坏卫星和地面系统的太阳事件的预警时间，”NIAC 的代理项目执行官迈克·拉普安特告诉 TechCrunch。

拉普安特建议，同样的技术可以用于整个太阳系的类似任务。有了这样的潜力，难怪美国宇航局会资助衍射太阳能帆船项目——现在它只需要离开地面。
