索尼宣布新的 PlayStation Studios 移动部门

原文：https://web.archive.org/web/https://techcrunch.com/2022/08/29/sony-acquires-savage-game-studios/

索尼收购萨维奇游戏工作室，宣布新的 PlayStation 工作室移动部门

索尼正在收购手机游戏开发商 Savage Game Studios，该公司周一宣布。这笔交易的财务条款没有披露。Savage Game 正在加入一个新创建的 PlayStation Studios 移动部门，该部门将专注于基于新的和现有的 PlayStation IP 创建游戏。索尼表示，新工作室将独立于主机游戏开发运营，旨在吸引新的观众，并为游戏玩家提供更多参与其内容的方式。

野蛮游戏工作室已经在开发一款新的未公布的 AAA 移动直播服务动作游戏。索尼表示，现在透露这款即将推出的游戏的更多信息还为时过早。

PlayStation Studios 的负责人赫曼·赫尔斯特(Hermen Schmidt)在一份声明中表示，“此次收购再次确认了索尼互动娱乐公司通过扩展到更多平台，向全球新玩家提供创新体验的承诺。”。“PlayStation Studios 必须继续扩展和丰富我们的产品，使之超越主机，为更多人带来前所未有的新游戏。收购 Savage Game Studios 的天才团队是实现这一目标的另一个战略步骤。”

萨维奇游戏工作室于 2020 年由开发者米恰伊尔·卡特科夫、纳德吉姆·阿吉尔和迈克尔·麦克马纳斯创立。三人组拥有多年在众多开发商工作的经验，包括 Insomniac Games、Zynga 和 Rovio。该工作室似乎还没有制作任何游戏。

收购完成后，Savage Game Studios 的日常运营将继续由公司目前的管理层负责。

“萨维奇游戏工作室是由渴望挑战现状并为移动游戏带来有意义的创新的业内资深人士创立的，”萨维奇游戏工作室的首席执行官兼联合创始人米恰伊尔·卡特科夫在一份声明中说。“今天的公告反映了我们的坚定信念，即 SIE 与我们一样热衷于实验和冒险。我们非常钦佩 PlayStation 第一方目录的一致性和质量，知道在这个行业中像他们一样一次又一次地提供产品是多么困难。我们很荣幸成为这一遗产的一部分，并加入 PlayStation Studios 由才华横溢的设计师、艺术家和工程师组成的世界级集体。我们还处于这个未公布项目的早期开发阶段，但我们迫不及待地想向你展示更多我们正在做的事情。"

索尼表示，它在游戏机之外的努力并没有削弱它对 PlayStation 社区的承诺，也没有削弱它继续打造单人游戏、叙事驱动型体验的热情。该公司表示，其手机游戏努力将为更多人参与其游戏提供途径。

今天的声明是在索尼确认其以 36 亿美元收购游戏公司 Bungie 的交易顺利完成后几周发布的，Bungie 是《光环》和《命运》的开发者。在索尼，Bungie 仍将是一个独立的游戏工作室，但它的专业知识将融入该公司对 PlayStation Studios 的战略，PlayStation Studios 是索尼互动娱乐公司(Sony Interactive Entertainment)的一个部门，致力于制作展示该公司技术实力的 tentpole 游戏。
