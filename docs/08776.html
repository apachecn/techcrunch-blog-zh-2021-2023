网络安全是未被充分代表的人进入科技领域的一种方式

原文：https://web.archive.org/web/https://techcrunch.com/2022/08/02/cybersecurity-could-offer-a-way-for-underrepresented-groups-to-break-into-tech/

上周在波士顿举行的 AWS re:Inforce 公司安全活动上，有一条关于多元化的欢迎信息。鉴于未来几年网络安全领域所需人员的绝对数量，这似乎代表着历史上未被充分代表的群体找到进入科技领域的途径。

AWS 的 CJ Moses 在公司主题演讲中谈到了多样化思维方式在确保公司安全方面的重要性。“我们文化的另一个关键部分是让不同观点的人呆在一起。这可能是内向或外向，来自不同的背景或文化，只要能让你的文化以不同的方式看待事物并相互挑战，”他说。

他补充说，新的思维方式可能会给网络安全团队带来变革。“我还认为，新员工可以为团队提供高度的清晰度，因为他们没有多年的偏见，也没有固定的团队思维。因此，当你招聘时，我们的最佳做法是鼓励对面试小组的构成保持敏感，拥有多种观点和背景，因为多样性带来多样性。”

初创公司 JupiterOne 的领域安全主管贾斯敏·亨利(Jasmine Henry)最近帮助编写了一本名为重塑网络安全的书，讲述了女性和跨性别者如何帮助改变网络安全领域。但是要完全实现这种转变，公司需要雇佣更加多样化的候选人。亨利认为让员工更加多样化是行业的责任，尤其是像 AWS 这样的大型组织。

“我认为有很多人真的想闯进去。我认为这更像是一种技能错配，而不是技能差距，因为有些人愿意、有能力并希望(在这个领域)工作。因此，我认为雇主，尤其是大雇主，有很大的责任来培训这些学徒，提高他们自己的劳动力技能，与社区团体合作……来培训那些想担当这些角色的个人，”亨利说。

她说，当像她这样的人进入这个领域时，他们可以帮助其他人获得在这个领域工作所需的技能。“我是第一代大学毕业生，我并非来自财富。安全是我成为中产阶级的原因，我为此感到自豪。我非常热衷于指导他人，尤其是第一代大学毕业生，”她说。

总的来说，科技行业在多样性方面做得并不好。根据招聘网站 Zippia 的调查，只有 25%的科技员工是女性，尽管她们占美国总人口的一半；7%是黑人，尽管占美国总人口的 14%；8%是拉丁裔，尽管占美国总人口的 18%。

根据来自阿斯彭研究所的研究，当你具体看网络安全工作时，女性占这些工作的 24%，黑人占 9%，拉丁裔仅占 4%。

AWS 的安全主管珍妮·布林克利(Jenny Brinkley)表示，亚马逊确实非常认真地承担起了更加多元化招聘的责任。事实上，她说公司将安全视为一种给公司带来更多多样性的方式。“我们真的专注于如何(作为一家公司)做出贡献，无论是通过开源贡献来提升人才，创造和确定这些网络安全工作的技能缺口，”她说。

布林克利赞同摩西在主题演讲中的观点，认为安全尤其需要多元化的思维方式。“我们可以开始更多地谈论神经多样性，当我们从整体上思考包容、公平和多样性时。安全真的代表了一个时刻，我们可以开始讨论如何创造和找到个人来完成这些工作？”她补充说，这些工作有可能为个人创造多代人的财富，她认为那些历史上被该行业和这些高薪工作遗忘的人有很大的机会。

亨利说，当她今年早些时候整理这本书时，她看到了一种放大各种声音的方法，并看到了该领域已经存在的多样性。“在这一过程中，我真正了解了自己，因为我意识到，在召集作者时，我也必须有意识地关注多样性，并意识到许多人希望谈论身份。他们想通过交叉的视角来讨论安全问题，”她说。

阿斯彭研究所有一些具体的建议来增加网络安全的多样性，包括减轻认证费用的负担，这是大公司可以做的；与能够带来更多样化候选人的组织建立伙伴关系；创建关注不同人群的导师项目，等等。
