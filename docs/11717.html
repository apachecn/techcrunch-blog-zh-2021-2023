Web3 基础设施初创公司温和地接受 Infura，炼金术与新的节点产品| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/11/02/tenderly-web3-infrastructure-alchemy-infura-startup-blockchain-developer/

Web3 开发工具初创公司温柔地正在进入节点游戏,它今天宣布了一个名为 Web3 Gateway 的新产品。据该公司称，该产品将帮助 web3 开发者读取、传输和分析区块链数据。

该产品建立在该公司的可观察性堆栈上，据说该堆栈索引了 20 多个区块链网络中的 90 多亿笔交易。

虽然许多区块链和加密公司在不利的市场条件下难以增长，但在开发商对构建 web3 产品的稳定兴趣趋势的推动下，温柔等基础设施提供商对逆风保持了相对的弹性。

这一新产品标志着 web3 基础设施提供商之间的竞争正在升温，因为它使这家初创公司与 ConsenSys 和 Alchemy 直接竞争，ConsenSys 拥有流行的节点即服务提供商 Infura，Alchemy 是业内另一家广泛使用的节点提供商。在此之前，tentry 只专注于智能合同领域，其仪表板和 API 可帮助工程师开发、测试和监控分散式应用程序的健康状况。

“与其他节点不同，温柔 Web3 Gateway 与温柔开发平台的其余部分紧密集成，消除了 dapp 构建过程中的开发、测试和基础设施孤岛，并帮助智能合同开发者节省时间和成本，”温柔首席执行官 安德烈·本契奇在一封电子邮件中告诉 TechCrunch。

节点提供商通常被比作 web3 公司的亚马逊网络服务(AWS ),因为它们提供了区块链基础设施的关键层。thornely 的产品与其他节点提供商不同的一个方面是其交易模拟平台，Ben ici 表示，该平台每月已经为 Instadapp、Yearn、Safe 和 Yield 等 web3 应用程序提供 5000 万次模拟。T11】

这家位于塞尔维亚贝尔格莱德的初创公司最近一次融资 4000 万美元是在今年 3 月，就在加密价格开始大幅下降之前。这笔融资是在这家初创公司的 A 轮融资几个月后进行的，同月宣布的还有 Alchemy 的 2 亿美元 C 轮融资延期，后者的估值为 102 亿美元。

据 T2 科技博客 TechCrunch 月报道，该公司表示，其平台被来自 Uniswap、Yearn Finance 和 OpenSea 等应用的数万名开发者使用，并与前 100 名以太坊项目中的大多数项目合作。

Spark Capital 的成长型投资者 Yasmin Razavi 帮助领导了该公司对 tently 的投资，他告诉 TechCrunch，这家初创公司的新产品是因为其开发者发现他们不能依赖现有的节点提供商来实现他们的目的，并决定改为自己建立这种能力。

Razavi 说:“你听到的关于 Alchemy 和 Infura 的问题大多是围绕着它们无法扩展。

根据 Razavi 的说法，客户报告称，根据该公司进行的 beta 测试，softly 的产品性能是 Alchemy 的三倍。虽然其性能尚未在公共领域得到验证，但很明显，这一产品使其更接近于 web3 基础设施服务的全套提供商，因此是该子行业中更强大的力量。

本契奇说:“通过这个最新版本，我们完成了对开发人员的端到端承诺，从编写和调试智能合约的最初步骤，一直到 dapps 上线后为其提供全面的基础设施支持，我们都给予支持。”。
