谷歌 Chrome 增加了虚拟信用卡号来保护真实信用卡号的安全

谷歌今天宣布，其 Chrome 浏览器现在将为用户提供在网上在线支付表格中使用虚拟信用卡号码的能力。当你在网上购物时，这些虚拟卡号可以让你保持“真实”信用卡号的安全，因为如果商家的系统被黑客攻击，它们可以很容易地被撤销。许多信用卡发行商已经提供了这些虚拟信用卡号码，但它们可能远没有达到应有的主流水平。

图片来源:谷歌

谷歌表示，这些虚拟卡将于今年夏天晚些时候在美国推出。由于谷歌正在与两家发卡机构合作，如 Capital One，这是该功能的推出合作伙伴，但主要网络如 Visa，Mastercard 和 American Express 也将在推出时得到支持。在这里，获得网络的支持绝对是一件大事，因为试图让每一个发卡机构都参与进来将是一项艰巨的任务。

更新 : 谷歌最初告诉我们万事达卡支持将于“今年晚些时候”到来我们现在从万事达卡获悉，它将于今年夏天与其他网络一起推出。

这项新功能将首先在 Chrome 桌面和 Android 上提供，随后将推出 iOS 支持。

“这是将虚拟卡的安全性带给尽可能多的消费者的里程碑式的一步，”谷歌副总裁兼支付总经理阿诺德·戈德堡说。“在桌面和 Android 上使用 Chrome 的购物者可以在网上购物时享受快速结账体验，同时放心地知道他们的支付信息受到保护。”

从用户的角度来看，这一新的自动填充选项将只需为您输入虚拟卡的详细信息，包括您永远不会记得的实体卡的 CVV，然后您可以管理虚拟卡，并在pay.google.com查看您的交易。虽然这些虚拟卡通常用于一次性购买，但您也可以使用这些卡进行订购。

由于这是谷歌，一些用户显然会担心该公司会使用这些关于你购买习惯的额外数据，但谷歌表示，它不会将这些信息用于广告定位目的。
