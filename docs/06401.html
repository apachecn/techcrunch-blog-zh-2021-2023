Token.io 获得 4000 万美元用于扩展支付技术

原文：https://web.archive.org/web/https://techcrunch.com/2022/05/12/token-io-raises-40m-to-expand-open-banking-based-account-to-account-payments-in-europe/amp/

图片来源: 艾丽西娅·洛普 /盖蒂图片社

数字支付的兴起改变了人们彼此做生意的方式；开放银行——银行业的一场运动，在位者最终采用更新的技术，如 API，将其系统开放给现代集成——正在引领一波新的支付方式，所有这些都希望成为像现金或用卡支付一样的标准。在这一主题的最新发展中，一家名为 Token.io 的英国初创公司获得了 4000 万美元的资金，以扩大其在支付技术方面的特殊推动——账户到账户支付和通过单一 API 访问交易账户——在英国和整个欧洲更深入。

这笔资金由 Cota Capital 和 TempoCap 共同牵头，参与方包括 Element Ventures、MissionOG 和 PostFinance，以及过去的投资者章鱼风险投资公司、Opera Tech Ventures 和 SBI 投资公司。该公司没有披露估值，但首席执行官托德·克莱德证实，与之前的融资相比，这一数字有所上升。

但作为 Token 表现及其技术吸引力的衡量标准，其客户包括法国巴黎银行(BNP Paribas)、汇丰银行(HSBC)、万事达卡(Mastercard)、Nuvei、Paysafe、Ecommpay、Rewire、Coingate、Raypd、Sonae Universo、Volt、Vyne 和其他名字未公开的公司(一些公司将 Token 用作白标服务，这意味着没有会泄露这种关系的品牌)。克莱德还表示，Token 的支付量去年增长了 20 倍，目前占英国和欧洲所有账户对账户支付的 21%，随着越来越多的客户开始使用 Token 集成，这一比例将会上升。

Token 的一个可比公司是 TrueLayer ，这是一家主要的开放银行，总部也在英国。去年，该公司以超过 10 亿美元的估值筹集了 1.3 亿美元的资金。

如果“令牌”这个名字让你想知道这家初创公司如何以及是否以某种方式参与了加密货币，那你就错了，也对了。该公司几年前成立，其使命是建立替代支付技术，以改善跨境支付的方式，加密货币确实是这一前提的一部分。然而，在 2019 年，剥离了其加密业务，该业务被称为 M10 ，目前专注于帮助银行(国有银行、商业银行)建立和运行令牌，用于创建自己的数字货币和其他用途。

Token 保留了“token”在金融服务中传统使用方式的名称和一些遗产，以继续其对安全和现代开放式银行技术的关注，并成为一批公司中的一员，这些公司对构建新服务感兴趣，以与现有的支付轨道竞争，承诺在此过程中为消费者和企业提供更多选择并降低成本。

从最普遍的意义上来说，账户到账户支付并不是一个新概念，但它长期以来一直是一个复杂而昂贵的过程，不是日常支付中使用的东西，而是需要与银行建立联系，有时需要填写文件并支付费用。在过去几年里发生的变化——也受到 GoCardless 等公司的创新的刺激——是这样一种想法，即它成为一种快速而简单的事情，就像从你的钱包中掏出一张卡或现金一样简单。

“出现了一个转折点，”克莱德说。“开放银行解决了账户对账户支付的障碍，它将使这些成为主流。所以吸收是相当巨大的。我们看到英国的销量逐月增长 10%至 20%。”

然而，有趣的是，与交易总额相比，这似乎仍然相对较小，这凸显了在账户对账户(以及支持这一概念的公司)的概念有望获得任何主流接受之前，还需要发生多少事情。目前，Clyde 估计 A2A 仅占英国和欧洲所有支付的 1%左右，尽管所有迹象都表明它将在三年内占所有支付的 10%。

正如 Kevin 一样，Token 的方法是构建自己的 API，在此基础上，它完成了与不同银行整合的所有艰苦工作，以使支付成为可能。上周，我们刚刚报道了另一家帐户到帐户支付领域的公司，该公司也宣布为其在 A2A 的特殊业务提供资金，将其与零售商的 POS 服务整合。

在英国，开放银行业务已经在该国的主要银行中推行了多年，这相对简单，因为这些银行使用相同的标准；Clyde 说，在欧洲，这是一项复杂得多的工作，Token 采用了 14 种不同的银行标准，并“协调”它们，将其构建到其 API 中，有效地使 16 个国家数千家银行的账户持有人可以通过其单一 API 进行支付。

“聚合对我们来说是一个关键的价值主张，”他说。

凯文·雅克(Kevin Jacques)是 Cota Capital 的合伙人，负责 Token 的投资，他曾在 Visa 工作多年，并管理其风险部门(因此对如何建立和控制支付轨道有着特殊和深入的了解，也知道当今一些最大的支付轨道的缺点)，他指出推动这个市场的一个重要力量将是强大的客户认证。

这些是欧洲(和英国)的新法规，要求多因素身份认证以提高数字支付的安全性。直接来自客户账户的 A2A，将意味着那些管理支付的人将有更多的账户持有人数据存档，并因此能够加快一些处理速度:由于处理延迟和故障，许多交易经常被放弃，简化这可能会刺激零售商和消费者的更多采用。

未来，虽然 Token 将主要专注于扩大其足迹、合作伙伴以及与其平台进行更多的实时集成，但它也将考虑如何引入更多服务来补充基本的 A2A 支付，以服务于更广泛的用例。我向克莱德提到了信用:今天不是每个消费者都愿意或能够用现金购物，那么为什么在 A2A 他们能够这样做呢？一些人仍将需要融资和信贷选择。他的回答是:虽然 Token 不太可能在短期内自行构建这些服务，但它可能会与该领域的其他破坏者合作，提供替代方案:

他说:“信用卡是卡，但它将被现在购买，以后支付和其他形式的信用所取代。”

我们目前正处于一个特别艰难的融资市场，因此值得注意的是，Token 的这一轮交易在过去几周内结束，这表明仍在为正确的团队和企业进行交易。

“这一轮证明了 Token 作为一家企业的质量。TempoCap 的投资合伙人亚当·谢泼德(Adam Shepherd)领导了该公司的投资，他说:“我们看到了支付市场发生一些动态变化的真正机会，即使有开放的银行创新，这种机会也不会每两年出现一次。“Token 在团队中有着深厚的支付专业知识，我认为我们在这个时候宣布这一轮的事实表明，对于拥有正确计划和正确单位经济的正确企业来说，仍然有资金可用。但没错，这是一个更加艰难的市场。”
