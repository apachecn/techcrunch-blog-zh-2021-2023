菲律宾 MSME 平台 GrowSari 融资 7750 万美元 C 轮

总部位于马尼拉的菲律宾小企业数字化平台 GrowSari 今天宣布，它已经筹集了 7750 万美元的 C 轮融资。加上之前的融资，包括在 1 月份宣布的同一轮融资的 4500 万美元，这使该公司的总融资额达到约 1.1 亿美元。投资者包括国际金融公司、KKR、Wavemaker Partners 和淡马锡集团的 Pavilion Capital。

新的资金将用于扩张到新的商店模式，建立物流和履行网络，并为 GrowSari 的运营，技术和数据科学团队招聘人员。

联合创始人兼首席执行官 Reymund Rollan 告诉 TechCrunch，GrowSari 再次筹集资金是因为它希望为商店老板扩大其金融科技产品，并建立其供应商市场，包括大宗商品。它还计划为更多类型的微小中型企业提供服务，如 carinderias(小餐馆)、小型非处方药店和其他路边和市场商店。

GrowSari 成立于 2016 年，现在为小企业提供的工具包括库存管理、定价工具、物流网络和流动资金贷款。它还使零售商能够提供电信充值和账单支付。它的客户现在包括吕宋岛 220 多个城市的 100，000 家商店，并计划很快扩展到棉兰老岛。

在一份准备好的声明中，国际金融公司运营高级副总裁斯蒂芬妮·冯·弗里德伯格说:“疫情从根本上改变了商业运作方式……忽视数字技术的企业将自己置于直接劣势。我们的投资将使 Growsari 能够扩大中小微企业的数字应用和金融服务，这对于保持它们的竞争力以及实现弹性和包容性复苏至关重要。”

更正:本文之前的版本称 GrowSari 的 C 轮投资总额为 1.1 亿美元。这实际上是该公司自成立以来筹集的资金总额。
