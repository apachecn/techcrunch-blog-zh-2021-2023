在退款欺诈事件后，Union54 首席执行官表示，非洲的金融科技“不是儿戏”

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/29/after-chargeback-fraud-debacle-union54-ceo-says-fintech-in-africa-isnt-childs-play/

去年 7 月，TechCrunch 报道几个金融科技应用的客户无法访问这些平台提供的虚拟美元卡服务。此外，客户无法创建新卡、为现有卡提供资金，也无法使用新卡进行网上和店内支付和购物。

这个问题的核心是赞比亚初创公司 Union54 正在打击的退款欺诈。这家 YC 支持的初创公司的 API 允许公司向其客户和员工发行物理和虚拟的本地和美元借记卡，而不需要银行或第三方处理器，它是该地区 100 家金融科技公司的默认卡合作伙伴。停止其服务影响了许多初创公司数周，直到他们找到替代方案。

该事件比以往任何时候都更加突出了发卡领域(以及整个金融科技领域)更好的 KYC/反洗钱合规检查的必要性，因为对持卡人退款索赔的不一致尽职调查可能会引发更多欺诈活动。

Union54 从包括 Tiger Global 在内的全球投资者那里筹集了超过 1500 万美元，是万事达卡的官方发行商。当时，它进行了合规性审计，预计最多在两个月内完成平台的安装和运行。

然而，现在还没有。相反，它的联合创始人兼首席执行官珀尔修斯·姆兰博最近告诉 TechCrunch，它正在等待重新进入非洲的金融科技市场，打算应用从这次活动和目前 9 个月的中断中所学到的东西。

Mlambo 坦率地谈到了 Union54 必须应对的问题，该公司如何面临全面关闭的风险，以及为什么 fintechs 需要在欺诈暴露方面更加透明。为了篇幅和清晰起见，下面的对话节选做了轻微的编辑。

Union54 遭遇退款欺诈已经不是什么新闻了。然而，没有提供详细的解释。能分享一下发生的事情吗？

Mlambo: 在文章上线的前几天，我们注意到我们的平台上有许多欺诈企图，我们发现并阻止了这些欺诈。人们试图做的是有效地使用他们没有的资金。为了让您了解我们阻止的欺诈数量，他们试图使用这些卡进行超过 12 亿美元的欺诈。我们算了一下数字。这个数量是我们日常数量的 600 倍。

因此，我们向金融科技公司提供 API 访问权限，他们向客户提供卡访问权限。与我们合作的一家公司在非洲大陆的一个新地区开展业务，由于他们的配置方式，他们很容易受到这种欺诈企图的攻击。我们看到的是，人们试图在一个特定的加密货币网站上购买一个特定的密码。所以这可能有点容易很快识别出这是非常不寻常的。

union 54 是如何解决该公司的这一问题的？

我们有几个客户，我们要做的事情之一就是与他们合作，让他们知道这些事情也在影响着他们。所以我们对那家公司做的就是让他们注意到这一点，并说，“嘿，我们看到你们那边的销量很大。我们强烈建议您看一看。”起初，他们坚持认为这是正常的，是增长的一部分；他们甚至感到震惊，我们对进来的数量感到惊讶。但是几个小时后，他们说，“你知道吗？有点不对劲。”

最后，最终发生的情况是，当卡被使用时，我们拒绝了它们。我们会看到一个交易通过，并说这个交易没有相应的余额，所以我们停止了它。因为我们不断停止所有这些交易，这导致了我们整体产品的退化，因为网飞、谷歌或 Spotify 等商家也会写信给我们，问我们为什么停止所有这些交易，因为这影响了业务。

情况变得如此糟糕，以至于我们在此期间使用我们的自动化规则不断拒绝每一笔交易，用户不断重试这些交易，这延长了阻止这种欺诈的过程。

因此，Union54 停止了所有交易，包括那些想从这些国际网站上买东西的人的交易，因为这些交易发生在欺诈事件期间？

是的，这导致了另一个复杂的问题，我想我们可能是少数几个经历过的公司之一。因此，每次交易被我们的规则或系统停止或拒绝时，我们仍然必须为该交易付款，因为在交易的生命线中涉及到如此多的不同方，我们需要向供应商付款。给你一个概念，在这个活动的高峰时期，我们有来自取消交易的高达 50 万美元的发票。

我们认为可能会发生三件事情中的一件。要么我们将花费所有的运营资金来支付这些费用，然后耗尽资金，这并不好，因为我们的基本原则是活下去。或者我们会回到这些受影响的金融科技公司，说，“嘿，你欠我们 50 万美元，”明知他们没有 50 万美元。或者我们会说，“你知道吗，我们将能够从我们的供应商那里收回这笔费用，以便在以后上诉时获得退款。”

最终，我们选择了第三个【选项】，不得不用我们的资金支付这些被拒绝的交易。后来，我们告诉投资者，在解决所有这些问题之前，我们认为继续运营该产品并不明智。

回想起来，Union54 难道不能解雇造成这一欺诈的公司吗？

所以我们尝试了一下。但实际情况是，我们了解到的另一件事是，一旦我们启动了这家公司，试图进行欺诈的攻击者就会发现我们正在让另一家金融科技公司发行信用卡，并试图通过它们。

这是一次相当复杂的尝试，赶走一家金融科技公司最终不会有所帮助，因为这种情况经常发生，即使是全球金融科技公司也是如此。我想是时候我们在非洲开始这样的对话了。 非洲的金融科技很难，大多数利益相关者仍假装这是儿戏。他们没有谈论真正的问题。我认为，站出来说，“你知道吗，我们发现了欺诈，我们阻止了它，”这也给了其他人信心，因为这些事情一直在发生。

站出来分享你所学到的东西是一个大胆的举动，因为大多数非洲创始人通常不喜欢谈论失败，或者在这种情况下，谈论挫折。

老实说，当你经历这一切时，感觉世界正在分崩离析，因为你建立了一些东西，然后突然，你失去了控制，人们开始散布谣言。

当你生活和经营的市场陷入困境时，也有点令人扫兴。我觉得这就是为什么许多人不谈论这些问题，因为他们觉得这会对整个生态系统造成危害。这也是我们一开始什么都不说的原因之一。当指控出现时，我们也没有反驳任何东西，因为这意味着对我们的一些客户来说，这是一个非常生死攸关的情况，如果我们说什么，他们就必须反驳我们说的任何话。所以我们只是遵守我们的法律义务，为我们的股东做正确的事情，然后继续制造我们的新产品，这可以给我们带来可持续性。

关于新产品，消息人士去年年底告诉我，Union54 正在秘密地与一些客户合作。这个过程的结果是什么？

由于我们与卡计划合作，在此期间，我们为大约三家金融科技公司切换了对 API 的访问，以衡量并证明我们可以从我们的一家供应商那里获得特定的退款。为了能够获得这些数据，我们需要有效地证明，比如基线，重新打开对产品的访问，然后说，“在正常情况下，这是退款的样子，这是交易的样子，这是我们看到的欺诈的影响。”

有了这些比较点，我们就能回到我们的供应商那里说，“看，我们希望最终发展一家公司，可以帮助非洲任何地方的任何人发行卡片，这是我们学到的一些东西。”例如，我们能够确定与大公司合作是不灵活的，因为他们需要更长的时间来解决争端。此外，我们认为需要解决我们的 KYC 流程。

union 54 的运营规模令人着迷，因为我们会在每个国家找到金融科技人员，并让他们访问 API，因为他们在该国受到监管，他们自己会做 KYC。正因为如此，我们能够与 100 多家公司合作，并创造出在埃塞俄比亚和赞比亚都有效的解决方案，埃塞俄比亚当时正在对外开放，而赞比亚是一个独特的市场，那里的身份证件是层压的，你无法使用手机摄像头轻松扫描它们。

尽管如此，我们需要能够全方位地为客户服务，这从长远来看影响了我们。一些出色的公司已经在 KYC 出现，但它们主要经营或服务于一个利基市场。我们最后一次检查时，还没有一家公司能提供从海角到开罗的 KYC 数据库覆盖范围。希望到明年年底，我们会有人能做到这一点。这将会改变游戏规则。

我们也有两三家公司发行美元虚拟卡。但是仍然有对他们服务的抱怨，表明仍然需要一个有更好的 KYC 流程的工会 54，因为问题还没有消失。Union54 的用途越来越广泛，人们仍在给我们发电子邮件，询问我们何时将再次开始发行。

这个问题有答案吗？

在过去的几个月里，我们一直很悲伤，因为失去一些东西从来都不容易。但在那之后，我们鼓起勇气，知道我们仍然有充足的资金，我们有时间开发一些东西，无论生态系统中目前发生什么样的冲击，这些东西都可以持续下去。为了能够再次发行，我们在那里做了一些调整，我们将在下一个产品中推出。

与此同时，我们了解到的另一件事是，尽管出现了裁员和货币贬值的消息，但非洲的金融科技领域仍然充满活力，有许多优秀的产品正在开发，也有许多优秀的公司。然而，在我们拥有一个成熟的生态系统之前，还需要一段时间。
