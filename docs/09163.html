在黑帽和 Def Con 2022 上你可能错过了什么

这些是年度黑客聚会上最精彩的演讲

黑客、研究人员、网络安全公司和政府官员上周齐聚拉斯维加斯参加黑帽和 Def Con，这是一项网络安全双法案，统称为“黑客夏令营”

今年的网络聚会特别令人兴奋:它不仅标志着黑帽的 25 周年纪念，而且是自疫情开始以来，与会者首次完全回到广受欢迎的安全会议的地毯走廊。这意味着在面具混乱和随后涌入的阳性测试中，黑客社区有很多东西要补上。

我们从这两个节目中收集了一些最好的公告。

Starlink 用 25 美元自制的 modchip 黑了

一名网络安全研究人员透露，使用一台 25 美元的设备就可以侵入 Starlink 终端。比利时安全研究员伦纳特·伍特斯周四在黑帽公司的舞台上展示了他是如何使用自制电路板或“modchip”黑进 StarLink 的用户终端的这个小工具允许绕过 Starlink 安全系统的故障注入攻击，并允许访问 Starlink 原本打算锁定的控制功能。伍特斯去年向 SpaceX 透露了这一漏洞，从而在该公司的 bug bounty 名人堂中赢得了一席之地。在他的讲话之后，SpaceX 公司以一份六页纸的文件作出回应，解释了它如何保护其系统以及固件更新，“使攻击更加困难，但并非不可能。”

Zoom 安装程序缺陷允许 macOS 上的 root 访问

由于过去几年来远程和混合工作的广泛转移，Zoom 已成为许多组织的基本通信工具，并安装在全球数百万台设备上。但安全研究员帕特里克·沃德尔(Patrick Wardle)在 Def Con 的一次演讲中透露，Zoom 的 macOS 安装程序中的一个缺陷可能会允许攻击者获得操作系统的最高级别访问权限，包括系统文件和敏感的用户文档。沃德尔发现Zoom MAC OS 安装程序具有自动更新功能，该功能以提升的权限在后台运行，使得攻击者能够通过更新功能运行任何程序并获得这些权限。尽管在沃德尔发表演讲时，这个缺陷还没有得到修补，但 Zoom 在周末发布的更新中修复了这个问题。

乌克兰网络主管意外亮相

乌克兰首席网络安全官员维克托·卓拉(Victor Zhora)对黑帽进行了一次未经宣布的访问，他在那里向与会者讲述了乌克兰与俄罗斯冲突中的网络战状况。担任乌克兰国家特别通信和信息保护局副局长的卓拉透露自二月份俄罗斯入侵以来，该国的网络事件增加了两倍，并补充说，乌克兰到目前为止在 2022 年已经发现了超过 1600 起“重大”网络事件，包括发现了工业 2 恶意软件，它可以操纵电力设施中的设备来控制电力流动。

美国揭露涉嫌 Conti 勒索软件的特工

美国国务院也出人意料地出现了，它利用这个机会宣布奖励 1000 万美元用于提供信息，导致识别和定位臭名昭著的俄罗斯支持的 Conti 勒索软件团伙的五名成员。该奖励是国务院正义奖励(RFJ)计划的一部分，该计划于周四分享了一张名为“Target”的已知 Conti 勒索软件运营商的图像，这标志着美国政府首次公开确认 Conti 特工。

Virtru 展示加密的周期跟踪应用原型

最近罗伊诉韦德案的翻案引发了人们的担忧，担心月经和排卵追踪应用程序(T2)可能被用来起诉寻求堕胎或流产医疗护理的人以及帮助他们的人。作为回应，以为企业和消费者提供电子邮件加密服务而闻名的 Virtru在 Def Con 上展示了一款原型周期跟踪应用，声称可以让用户完全控制他们的私人信息。Virtru 员工团队在最近的一次公司黑客马拉松中用三天时间构建的 SecureCycle 利用了 OpenTDF 提供的开源端到端加密，如果任何第三方试图访问他们的数据，它将通知数据所有者。

“基本”安全缺陷造成重大 5G 风险

5G 商业网络开始铺开，承诺令人兴奋的新用例，如自动汽车、更智能的医疗保健和智能传感器网络。但柏林科技大学的研究员 Altaf Shaik 表示，这些 5G 网络也可能带来新的安全挑战。Shaik 和他的同事 Shinjo Park 研究了 10 家移动运营商提供的使开发者可以访问物联网数据的 API，并在每一家运营商中发现了“基本”API 漏洞。Shaik 告诉 Wired 尽管这些缺陷很简单，但它们可能被滥用来泄露 SIM 卡标识符、SIM 卡密钥、账单信息以及购买 SIM 卡的人的身份。
