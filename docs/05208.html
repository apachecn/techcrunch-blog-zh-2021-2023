谷歌开始隐私沙盒广告定位的全球测试

原文：https://web.archive.org/web/https://techcrunch.com/2022/03/31/privacy-sandbox-global-trials/

谷歌刚刚宣布了其隐私沙盒提案的下一阶段试验——重点是广告相关性和测量。

沙盒指的是一个不断发展的——现在受到非常密切监督的——广告定位技术栈，谷歌已提议(最早)在 2023 年下半年用其他替代技术取代 Chrome 中基于 cookie 的定向广告，谷歌认为这些替代技术将更好地保护用户隐私，但仍能有效产生广告收入。

Chrome 隐私沙盒产品总监 Vinay Goel 今天在博客中写道:“从今天开始，开发者可以开始在全球范围内测试 Chrome 金丝雀版本中的主题、羽翼和属性报告 API。

“我们将尽快向数量有限的 Chrome Beta 用户推广。一旦测试版一切顺利，我们将在 Chrome 的稳定版本中提供 API 测试，以将测试扩展到更多的 Chrome 用户。”

沙盒提案由多个部分组成，如主题——谷歌通过基于浏览器的用户网络活动跟踪实现基于兴趣的广告定位的想法(取代了 FLoCs；备受批评的前身谷歌最近放弃了)——以及谷歌关于在没有个人用户追踪的情况下重新营销和定制受众的提议。

除了复杂和充满缩写词，谷歌的沙盒计划也吸引了不少争议。

最值得注意的是，欧洲的反垄断监管机构在出版商和广告商的投诉后介入，他们认为谷歌反对跟踪 cookies 的计划只会巩固其市场权力。

但在获得谷歌关于如何开发沙盒的一些承诺(包括任命一名监督受托人)后，英国 CMA 上个月同意让该项目继续进行——为继续开发和另一批沙盒试验铺平了道路。(尽管欧盟监管机构仍在继续审查该计划。)

谷歌表示，它现在也将开始测试更新的隐私沙盒设置和控制——它表示，这将允许用户“查看和管理与他们相关的兴趣，或者完全关闭测试”。

它的博客文章给出了一些它将进行测试的设置的示例图——它显示了一个多层菜单结构，在顶层有一个主开关来关闭(或打开)测试，并向下钻取一个基于浏览器的广告个性化菜单，用户可以删除基于主题的浏览活动监控分配给他们的兴趣，并编辑系统从中推断兴趣的网站列表，以及另外两个菜单(一个与广告测量有关，另一个用于减少垃圾邮件和欺诈)。

图片来源:谷歌

值得注意的是，欧盟(和其他几个地区市场)的 Chrome 用户不会选择参加最新的沙盒起源测试——相反，根据谷歌的说法，他们只有通过将开关切换到 on 位置来主动选择参加才能参加。这可能是由于该地区对人们隐私的法律保护，如欧盟的一般数据保护条例。

沙盒中现已废弃的絮凝物成分的试验也于去年在欧洲之外开始。

“在即将到来的 origin 试用期间，Chrome 计划根据不同地区测试多种通知用户试用的方法，”谷歌发言人告诉我们。“在欧洲经济区，瑞士和英国用户将被要求通过选择加入的方式自愿参与试验。”

这位女发言人补充说:“所有用户都将有强大的控制，并可以在任何时候选择退出试验。”

谷歌的沙盒方法是否真的能保护隐私，这是悬在该提议上的一个相当突出但尚未回答的问题。

还有一个更广泛的问题是，基于个人推断兴趣的定向广告(通过监视他们在本地的浏览)是否会简单地复制掠夺性和歧视性的定向广告，这在当前基于个人跟踪的广告技术中是完全可能的。因此，隐私改革/重启谷歌沙盒的力度到底有多大，还有待观察。

独立隐私研究员和顾问 Lukasz Olejnik 博士对谷歌的提议进行了一般性评论，他说，谷歌开发“隐私保护”广告定位的方法借鉴了早年的研究。但是他同意这项任务具有挑战性。

“这样的研究现在有点被遗忘了，但它在大约 10-7 年前是隐私研究界令人兴奋的一部分。在这个领域不可能再有学术出版物了，因为学者们似乎认为这个问题已经“解决”了很多年。嗯，算是吧，不管这听起来有多荒谬。

“但谷歌对发布会议论文不感兴趣，他们实际上是在建设基础设施，这是一个具有挑战性的部分，因为网络是一个复杂的生态系统。它也很容易泄露——所以重要的是对网络架构的关键部分提出适当的修改或补充，使其能够保护隐私，”他告诉 TechCrunch。

Olejnik 还表示，谷歌的试验声明强调了自从这家科技巨头几年前开始沙盒迁移工作以来至少有一个值得注意的教训。

“这一声明也证明，谷歌艰难地认识到了用户控制此类测试的重要性，因此现在很清楚，用户从一开始就将处于控制之中，”他建议道。

在谷歌的博客文章中，Goel 写道，沙盒提案“从早期测试人员的深思熟虑的反馈中受益匪浅”，并补充说:“我们渴望为我们的更多提案开放测试。我们将继续从生态系统中收集反馈，并与全球监管机构合作。”

开发者被告知谷歌关于沙盒 API 的指南，以及如何参与测试的信息。
