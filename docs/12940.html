安娜·海姆(Anna Heim):2023 年，以 API 为先的创业公司将继续崛起

我相信 API 优先将是 2023 年的一个主要趋势，这种方法比以前更加普遍，也比不太依赖 API 的选项更加成功。

API 正在崛起并不是什么新鲜事——但 API 优先的初创公司是这个世界上的一个子群体，而且是一个享受顺风的群体。

根据 API platform Postman 根据内部数据和一项调查得出的发现，报告将 76%的开发工作花费在 API 上的公司“比将自己评为 API 优先的受访者预测未来一年招聘增加的可能性高 35%。”
