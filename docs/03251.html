购买 Docker 企业资产两年后，米兰蒂斯的运营成本超过 1 亿美元

原文：https://web.archive.org/web/https://techcrunch.com/2022/02/09/mirantis-on-run-rate-over-100m-two-years-after-buying-docker-enterprise-assets/

当 Docker 宣布将在 2019 年出售 Docker Enterprise 时，这对行业观察人士来说是一个很大的惊喜。或许更令人惊讶的是，买家是米兰蒂斯，一家因 OpenStack 项目商业化而闻名的公司。

出售后，Docker 转向了一家专注于开发者的公司，并于上周报告称其去年获得了 5000 万美元的 ARR 。在本周与 TechCrunch 的一次对话中，Mirantis 首席执行官 Adrian Ionel 表示，该公司预计本季度总收入将达到 2800 万美元，运营成本将超过 1 亿美元。他说，其中大约 80%是反复出现的。

他补充说，从 Docker 购买并更名为 Mirantis 的资产(包括 Docker Enterprise Engine、Docker Trusted Registry、Docker Unified Control Plane 和 Docker CLI)以及该公司在购买之前拥有的基于 Kuberentes 的云平台工具，收入分成约为 50/50。值得注意的是，还在 2020 年购买了 Lens ，这是一个专门为 Kubernetes 设计的 IDE(集成开发环境)，这也有所帮助。

但基本上 Docker 和 Mirantis 都获得了 5000 万美元的收入，这是因为 Docker 在出售企业产品后能够自己创造什么，以及 Mirantis 将这些 Docker 企业资产转化成什么。这是一个罕见的交易，在发生两年后，双方匹配得如此之好，但 Ionel 说，他当时肯定不知道这一点。

“这是一次奇妙的旅程，一次难以置信的旅程。它始于 2019 年 11 月签署协议。但当时每个人都不清楚这将如何实现，”他说。首先，它给客户造成了一些困惑，不知道这次收购对公司的未来意味着什么，但他说，他们很快就看到了公司对合并后的产品系列的愿景。

“事实证明，人们非常、非常迅速地理解了我们共同的产品愿景是什么，以及我们如何创建一家新公司，我认为这是关键主题。这真的不仅仅是关于米兰蒂斯收购 Docker Enterprise。Ionel 解释说:“这更多的是关于我们建立一家利用两家公司资产的公司，并成为一家新的、更强大的公司。

董事会还讨论了购买这些资产的好处。“在收购之前，我们的董事会对此进行了激烈的辩论，你可以想象，因为很多收购都可能出错，但它非常成功。它增加了巨大的股东价值。对我们来说，这绝对是一个梦幻般的赌注，真正加快了我们与 Kubernetes 的旅程，并走向未来，”他说。

不过，在交易完成后，事情并非一帆风顺。它从裁员 40%开始，Ionel 承认这是一个痛苦的开始。但通过将工程和其他一些业务职能整合到一个保护伞下，该公司能够节省资金，这将导致其最终的成功。

Ionel 报告说，在过去的两年中，Mirantis 的现金流一直是正的，在此期间产生了超过 1900 万美元的现金。它在合并时扩大了与 300 家现有客户的关系，包括苹果、维萨和 Booking.com，同时还增加了 100 家新客户。

总的来说，Ionel 不可能期待比他从 Docker Enterprise 交易中获得更好的结果。“这是一个奇妙的旅程，有时充满挑战，因为我们确实不得不重组 Docker Enterprise 业务，但我们快速果断地完成了，我认为这是成功的，最终结果是我们现在拥有了一个繁荣的业务。”

编者按:这篇文章最初称该公司上个季度的收入为 2800 万美元。这实际上是下一季度的预计总收入。这一点已经得到纠正。
