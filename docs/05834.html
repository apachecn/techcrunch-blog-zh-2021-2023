为什么埃隆·马斯克收购 Twitter 对隐私有害

原文：https://web.archive.org/web/https://techcrunch.com/2022/04/26/elon-musk-twitter-privacy/

埃隆·马斯克宣布推特已经接受他的出价，以每股 54.20 美元收购该平台，对该社交媒体平台的估值约为 440 亿美元，从而结束了数周的猜测。

虽然马斯克对 Twitter 的长期追求已经结束，但至少对他来说，Twitter 及其数亿用户历史的下一个篇章才刚刚开始。

这笔交易立即引起了人们的担忧，自称“言论自由绝对主义者”的马斯克可能在内容节制上开倒车，这可能会破坏多年来遏制仇恨言论和错误信息自由传播的工作。但是专家们很快就警告了 440 亿美元收购 Twitter 私有化的潜在隐私影响，在这个时候，甚至员工都不清楚公司的未来。

根据马斯克短短 78 个字的声明，他为 Twitter 提出的许多计划中有一个引起了业界的关注，那就是开源该平台的算法代码，使其公开可用。马斯克声称，这一变化——Twitter 已经考虑了一段时间——将有助于提高对该平台的信任，该平台多年来一直面临着虚假新闻和安全事件泄露，其中看到黑客劫持了高调的 Twitter 个人资料——包括马斯克的个人资料——以推广加密货币骗局。

但网络安全专家担心，马斯克对 Twitter 的开源愿景可能会使该平台更容易受到攻击者的攻击。

安全公司 ExtraHop 的高级技术经理杰米·莫莱斯(Jamie Moles)告诉 TechCrunch 说:“开源这一代码的决定可能意味着它将被其他社交平台、广告商和其他寻求完善用户定位的人所采用。“当然，与任何广泛采用的开放源代码一样，存在重大的安全隐患。正如我们在 Log4Shell 和 Spring4Shell 中看到的，广泛使用的开源应用程序中的漏洞具有指数级的价值。将代码开源可能会增加 Twitter 用户的透明度，但也可能使 Twitter 成为攻击者更大的目标。”

莫莱斯说，如果操作得当，马斯克对所谓的垃圾邮件机器人发动战争的计划，可以产生“新的技术，改善对垃圾邮件、垃圾邮件帖子和其他恶意入侵企图的检测和识别，”他补充说。"这对各地的安全从业者来说很可能是一个福音."

英国德蒙特福特大学计算机科学和信息学院院长 Eerke Boiten 教授警告说，开源 Twitter 的算法可能会导致恶意行为者“游戏”该算法，这可能会导致人们根据个人特征受到不同的对待。

Boiten 说:“例如，想想 Twitter 的目标广告方面的外部操纵，这是一个甚至在游戏化之前就令人担忧的隐私领域。”"这也将加速新游戏方式和寻找对策的军备竞赛."

马斯克的简短声明给人们留下了很多想象空间。他没有说他对“认证所有人类”的计划是什么一些人认为这是一个扩展 Twitter 现有用户验证系统的计划，或者计划引入实名制政策，要求用户提供合法姓名的证明文件。数字权利组织电子前沿基金会(Electronic Frontier Foundation)表达了对实名制政策对假名言论的人权价值的担忧，马斯克可能没有考虑到缺乏匿名可能对某些人群产生的影响。

EFF 在一篇博客文章中说:“假名和匿名对于保护那些观点、身份或兴趣与当权者不一致的用户至关重要。”。“例如，在脸书要求实名的政策被用来驱逐美洲原住民；使用传统爱尔兰、印度尼西亚和苏格兰名字的人；天主教神职人员；变性人；变装皇后；和性工作者。如果当权者能够发现他们的真实身份，持不同政见者可能会处于严重的危险之中。”

EFF 还对持续缺乏对 Twitter 直接消息的端到端加密表示担忧:“担心平台的新所有者能够阅读这些消息不是没有根据的，”EFF 补充道。

Boiten 也认为，马斯克的假名打击将是马斯克收购最令人担忧的方面。“在许多情况下，匿名是隐私的先决条件。一旦人们知道 Twitter 已经认证了它的用户，暴虐的政府就可以要求他们提供认证信息，从而危及这些国家当前的许多颠覆性使用，”他说。“我想知道目前有多少匿名 Twitter 账户是由特斯拉员工运营的——埃隆·马斯克严格遵守他自己的规则——所以潜在的特斯拉举报者或工会会员在 Twitter 上获得认证是不安全的。”

在周二的一条推文中，参议院情报委员会主席、参议员马克·华纳(Mark Warner)表示，Twitter“在努力解决虚假、欺骗性和受操纵的内容方面，比许多竞争对手更具前瞻性”，尽管他表示该公司有“巨大的改进空间”，但华纳表示，他希望马斯克将“真诚地努力，将这些必要的改革保持在适当的位置，并防止有害于民主的倒退。”

目前，马斯克对 Twitter 的收购要约仍有待股东和监管机构的批准。
