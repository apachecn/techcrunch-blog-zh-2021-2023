Twitter 是全世界分享观点和愿望的地方；它是品牌、名人和政治家与人互动、生活和公开的地方。

从 2006 年到 2012 年，Twitter 的公共 API 对所有人都是免费的，开发者利用它为不断增长的社区建立了广泛的增值服务(如 TweetDeck)。但在 2012 年之后，Twitter 大幅限制了数据访问，侵蚀了开发者的信任。为了帮助扭转这一趋势，让开发者社区蓬勃发展，Twitter 在 2021 年 5 月收购了我的初创公司 Reshuffle。

收购后，杰克·多西和布鲁斯·福尔克(分别是前首席执行官和收入产品总经理)让我负责重新开放 Twitter API。这是在我作为一名外部开发者在一个公共论坛上向 Twitter 的领导层提出严厉的反馈意见之后发生的，这些意见是关于开发者平台有多糟糕以及需要投资来纠正它。

我告诉他们，通过适当的关注，我们可以创建一个令人愉快的、成功的平台，为开发者提供工具和 API，以繁荣和改善 Twitter 的用户体验。套用 Twitter 时任首席财务官内德·西格尔的话:“阿米尔告诉我们它有多糟糕，所以我们买下了他的公司来修复它。”

我和我的创业公司加入了一个了不起的小团队，致力于重振 Twitter 开发者平台。除了确保开发人员有一个不断增长的 API 端点套件，使他们能够构建成功的解决方案，该团队还将旧的 API 迁移到新的基于 GraphQL 的基础设施。2021 年 11 月 15 日，我们正式推出了新的 Twitter V2 API，受到了很多开发者的喜爱和兴奋。

但是我们的野心不止于此。

多尔西和董事会甚至进一步资助我们(批准我们再雇佣 50 人)建造更大的东西——他们长久以来一直想建造的东西。我们的愿景是让 Twitter 成为一个真正的开发者平台。我们希望开发者能够创建与用户互动的 Twitter 内置应用。

我们设想了一个世界，在这个世界里，你可以从 Spotify 上分享你最喜欢的歌曲，并在 Twitter 上与你所有的粉丝一起现场聆听。我们希望您能够分享您对您最喜爱的事业的捐赠，并通过一种整合的 GoFundMe 风格的体验让您的追随者也进行捐赠。我们希望你在 Twitter 上玩 Wordle，而不仅仅是分享结果。我们希望你能够在 Twitter 用户体验中与开发者驱动的应用程序进行交互。

这仅仅是个开始:我们还设想了 Twitter 时间线的真正去中心化。我们想让开发者创建并分享他们自己的时间表。

上个月，我们兴奋地期待着在 Chirp 上向开发人员宣布我们的愿景，现在这个愿景只是一个开放的主题文档，在我的砖砌 Twitter 计算机上丢失了。

对科技感兴趣？这是 TechCrunch 策划的时间表。对电子游戏感兴趣？这是 Twitch 的游戏和彩带时间表。自定义时间线是 Twitter 列表的超级英雄进化，给予开发者高级权力来策划和创建他们喜欢的主题，或者代表他人这样做。

我们计划的下一部分是创建一个发现机制，类似于一个商店来发现和安装这些应用程序和时间线。我们甚至开始探索让开发者将这些体验货币化的可能性。

在 Twitter 收购期间的过山车时期，我们实现了这一切。我们(相当错误地)认为，埃隆会花时间了解 Twitter 内部的项目范围，以及它们对公众对话的影响。我们相信开发者平台是他直言不讳的“一切应用”愿景的关键部分。我们兴奋地向他展示我们的愿景，希望他会对我们的愿景感到兴奋。

然后，在 11 月 4 日，我们被解雇了。我们的工作电脑在半夜被封了，电子邮件出现在我们的个人账户里，告诉我们我们被解雇了。

根据我们的一位工程师的公开推文，有两人仍在我们 100 多人的组织之外。我们对开发者的所有梦想和规划都被吹得灰飞烟灭。

当我加入 Twitter 时，我告诉多尔西，“我们不能搞砸了。我们只能修复一次开发者和 Twitter 之间的关系。如果我们错过了这个机会，我不会再雇佣一个信任我们平台的开发者。”

这是因为当开发人员开始在一个平台上构建时，他们就在打赌这个平台将继续以高度的稳定性存在。在一个平台上构建需要做很多工作，开发人员在过去被不成功的平台如 Windows Mobile 和不可靠的平台如 T2 脸书 API T3 所困扰。

我在一些最好的平台上工作过——从 Android 和 Sharepoint 到 Twitch 和 Slack——它们都有一个共同点:开放和信任。

上个月，我们打破了这种信任，很抱歉我没能阻止这种事情发生。我半夜醒来还在想这件事。上个月，我们兴奋地期待着在上个月的 Chirp 上向开发者宣布我们的愿景，现在这个愿景只是一个开放的主题文档，在我的 Twitter 电脑上丢失了。

一位开发者曾经问我，我们如何才能确保 Twitter 继续维护和投资这个开发者平台。我的回答是，“只要我们有这个了不起的团队，我们的领导负责构建平台，开发者就会看到我们是认真的，当情况发生变化时，我会让你知道。”

让这成为我对 Twitter 开发者的个人通知:团队没了；投资被撤销了。爱已经不在这里了。

构建 Twitter 开发者平台的团队令人惊叹。他们将为其他公司的开发人员和其他开发人员工具构建令人敬畏的平台。我很荣幸能和他们每一个人共事。

对我来说，随着我向风险资本领域前进和过渡，我打算投资于有影响力的开发者平台——那些致力于开放和值得信赖的平台。
