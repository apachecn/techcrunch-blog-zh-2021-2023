Twitter 的限制性 API 可能会将研究人员拒之门外

原文：https://web.archive.org/web/https://techcrunch.com/2023/02/14/twitters-restrictive-api-may-leave-researchers-out-in-the-cold/

本月早些时候，Twitter 宣布将限制免费访问其 API(允许第三方开发者与 Twitter 互动的编程接口)。虽然此举肯定会影响独立开发者和初创公司开发工具，使平台变得有趣和安全，但它也给学生和学者带来了一个问题，他们使用 Twitter 进行不同主题的研究。

上周，埃隆·马斯克领导的公司给开发者发了一封电子邮件，其中提到访问 Twitter 的 API 的基本层——每月为“低级使用”支付 100 美元——将取代传统的访问级别，如基本、高级和学术研究。目前，几乎没有任何关于每月 100 美元能让开发者做什么的信息。根据平台开发者 Casey Newton 的说法，一个低级的企业应用程序接口每月要花费高达 42，000 美元。

研究人员的支付能力

对于研究界的许多人来说，每月花费数百或数千美元可能是不可行的。

“本周早些时候，HateLab 的一名本科生不得不改变他的论文设计，不再在 Twitter 上收集数据，因为他没有资金支付这笔费用。全国上下成千上万的人以及全世界数百万人将会分享他的经历。这真的具有难以置信的破坏性，并将对依赖于这些数据的研究生态系统产生重大影响，因为 Twitter 研究人员从本科生到专业人员的渠道已经被这种变化破坏了，”哈特拉布教授马修·威廉姆斯说，哈特拉布是卡迪夫大学社会数据科学实验室和在线仇恨言论研究的一部分。

具有讽刺意味的是，HateLab 在 Twitter 的开发者门户网站上被列为善用研究的成功故事。它在平台上发表了多篇关于仇恨言论的研究论文。该机构一直在使用社交网络的学术研究 API。但在新版本中，这种情况可能会不复存在。

Twitter 也没有考虑到一个月 100 美元的基本访问费对于发展中国家的研究人员来说可能不是一个大数目。

“停止免费访问 Twitter API 的决定将极大地影响在线研究仇恨言论的研究人员，特别是独立研究人员和发展中国家的研究人员。印度将深刻感受到它的影响，仇恨言论正在 Twitter 上以惊人的速度扩散。印度研究机构 Hindutva Watch 的创始人拉吉布·哈米德·纳伊克说:“每月支付 100 美元，每年支付 1200 美元，这对他们来说是一个沉重的经济负担。

对于可能愿意花钱的机构来说，这个过程甚至还不清楚。数据、民主和政治研究所的主任丽贝卡·特朗布尔(Rebekah Tromble)说，当他们试图填写企业级访问的表格时，他们被重定向到学术研究项目。她还提到，他们以前在 Twitter 上联系的那个人已经不在公司工作了。

研究 API 关闭的影响

独立研究是让 Twitter 更有用、毒性更小的关键因素。该公司利用 Twitter 数据展示了在医疗保健、在线仇恨言论和气候变化等领域开展的多个项目。

今年早些时候，该公司发起了 Twitter 适度研究联盟(TMRC) ，邀请来自学术界、公民社会、非政府组织和新闻业的成员研究该平台的治理问题。但是自从马斯克接手后，这个项目就停滞了，参与其中的员工也离开了。

在导致公司被收购的公开争论期间，特斯拉首席执行官本人使用了来自机器人测量仪的数据。这个工具是由印第安纳大学社交媒体观察站开发的。但是这个工具的未来可能会受到新 API 公告的威胁。

很多研究项目都考虑了大量的推文。他们向平台发送数百个查询来研究不同的主题。Twitter 还没有公布任何关于每月 100 美元的细节。但是对于大多数项目来说，这很可能是不够的。作为参考，在学术研究方面，Twitter API 以前每月提供 1000 万条推文，每个应用程序每 15 分钟提供 50 次请求。

Botometer 的研究人员之一杨凯成(音译)表达了对 Twitter 关闭学术机构免费 API 的担忧。

NYU 社交媒体和政治中心的联合主任 Joshua Tucker 最近在 Twitter 上发表了一篇关于 2016 年美国总统竞选期间俄罗斯错误信息活动的论文。他说，该运动研究了成千上万条推文的数据，所以如果社交网络让学者为这些数据付费，那么将很难进行大规模的研究。

“这(Twitter 的举动)本质上是朝着错误的方向迈出了一步。我们正处于这样一个时刻，全球重要的立法努力都集中在让外部研究人员更容易获得数据，Twitter 的这一决定只会让外部研究人员更难获得数据。这反过来意味着，对于政策制定者、媒体、公民社会和商界而言，这些平台对社会的影响存在更多盲点，”他说。

在过去的几天里，许多 的研究人员指出，免费的 Twitter API 访问也有助于应对自然灾害的危机，比如最近发生在土耳其和叙利亚的毁灭性地震。本月早些时候，一群独立研究人员给 Twitter 写了一封公开信，要求该平台保持免费 API 访问的开放。

“Twitter 的新任首席执行官埃隆·马斯克(Elon Musk)承诺将使该平台更加透明，并减少垃圾邮件和操纵账户的盛行。我们赞扬并支持这些优先事项，”信中说。

“事实上，独立研究社区已经开发了许多用于管理机器人的最前沿技术。API access 为这项工作提供了重要的资源。Twitter 的数据访问新壁垒将降低平台和我们的社会迫切需要的透明度。”

如果没有独立的研究，该公司可能会对平台上的错误信息和仇恨言论问题一无所知。欧盟已经对这家公司发出了一张“黄牌”,理由是其虚假信息报告中缺少数据。多家公司签署了行为准则，承诺向研究人员提供数据等。

虽然不遵守研究准则不会产生任何法律后果，但从明年数字服务法案(DSA)生效起，它可能会产生更大的影响。另外，欧盟高级专员 Josep Borrell 批评 Twitter 通过让其 API 付费来限制对其平台的访问。这一变化的影响比预期的更广泛，快速的收入增长可能会变成一个更重要的长期问题。
