Web3 艺术家 Latashá向创作者介绍加密技术的兴衰 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/06/26/latasha-web3-crypto-musician-rapper-artist-creator-education-zora-zoratopia/

尽管围绕加密世界存在诸多争议，但区块链技术的一个明显亮点是其支持创作者和艺术家将其作品货币化的能力。说唱歌手兼视觉艺术家拉塔沙，我们最新一集 连锁反应 播客的嘉宾，是 web3 支持独立艺术家潜力的生动体现。

2021 年，她制作并成功销售了自己的第一部 NFT 音乐录影带——这是她首次涉足网络 3 世界。从那时起，她以数万美元的价格出售了自己作品的 NFTs，并成为艺术家利用 web3 技术的一个非常明显的倡导者。她在仿植物怪兽佐拉 NFT 市场担任社区负责人，通过这个社区，她在全国各地举办了她的标志性活动 Zoratopia，以教育对进入这个领域感兴趣的艺术家。

拉塔沙说，对于艺术家来说，铸造 NFT 的价值主张是“就像在美术馆里陈列东西一样”。

“我提醒(艺人)，尤其是音乐行业的艺人，入行成本如此之高。我的意思是，要把你的歌曲放到播放列表上，要把你的歌曲放到收音机上，你通常要支付一些很荒谬的费用，从 10，000 美元起。所以我想，你愿意花一万美元去电台吗？或者你更愿意花 50 到 100 美元看着这个东西成长，尤其是在现在这片新的沃土上，”她补充道。

自从拉塔莎首次进入 NFT 空间，艺术家们面临着大量新的挑战，希望复制她的成功之路。她说，首先，以太坊的油价已经大幅上涨，从她创业时的每 NFT 约 25 美元。其次，整个加密行业正处于市场低迷期，随着代币价格的下降，NFT 价格也在下降(以本国加密货币报价)。

Latashá说，尽管面临这些挑战，她仍然相信利用 web3 对许多艺术家来说是一个明智的策略。她说，就像任何其他行业一样，web3 会起起落落，市场参与者将不得不经历不同情绪的波动。

“我认为有很多人渴望同时进入，”她说，沉思着 crypto 最近的牛市。“我认为这可能是我们之前需要考虑的部分，因为我们需要让人们为潮起潮落做好准备，”她继续说道。

拉塔沙说，更多的教育本可以帮助创作者为这次衰退做好准备。这也是为什么教育人才是她在仿植物怪兽佐拉的核心使命的部分原因。

“我总是(对艺术家)说，决定你想用你的钱做什么，对吗？如果你想把你的钱保密，那就去做吧。但我也想说，要有多种收入来源，”拉塔沙说。“总是要决定你是否想把你的一些秘密投入到不同的艺术形式中，或者为你的音乐找一个营销团队，或者做其他可以以不同方式增加收入的事情……我不认为我们应该被任何收入流所束缚。我们应该永远自由，尽可能开放。”
