SeMI Technologies 的搜索引擎开辟了查询数据的新途径 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/02/22/semi-technologies-search-engine-data/

公司拥有大量的非结构化数据，通常没有能力从中获取更多信息。

现在想象一下，有一种存储数据的方法，并且实际上能够向它提问，例如，“ABC 公司什么时候与我们签订了第一份合同？”或者“给我看包含蓝天的视频。”

这就是 SeMI Technologies 正在用矢量搜索引擎 Weaviate 构建的东西。SeMI 的首席执行官兼联合创始人 Bob van Luijt 表示，这是一种独特的人工智能优先数据库，使用机器学习模型输出向量，也称为嵌入，因此得名向量搜索引擎。

他解释说，矢量搜索引擎并不新鲜——谷歌搜索是建立在矢量搜索引擎之上的解决方案的一个例子。然而，SeMI 的目标是将这项技术商品化，并拥有开源商业模式，以便任何人都可以使用它。

去年，Van Luijt 让我的同事 Alex Wilhelm，通过创建一个语义搜索引擎，对 2021 篇 Techcrunch 文章进行问答，了解了技术的内幕。

“每个人都可以使用这项技术，我们为那些需要的公司提供工具和服务，”van Luijt 补充说。“我们不创建或分发实际的模型——这是像 Huggingface 或 OpenAI 这样的公司做的事情，或者公司自己制作模型。但拥有模型是一回事，在生产中使用它们来驱动搜索和推荐系统是另一回事，这正是 Weaviate 要解决的问题。”

自 2019 年与首席技术官艾蒂安·迪洛克(Etienne Dilocker)和首席运营官·米查·弗哈根(Michael Micha ver Hagen)一起创立该公司以来，范·路易特已经看到 SeMI 的技术激发了 100 多个用例，包括像 Keenious 或 Zencastr 这样的初创公司，基于矢量搜索引擎给他们带来的新可能性创建新业务，并使用 Weaviate 提供的结果直接帮助人们，例如在医疗领域。

van Luijt 的一些个人最爱是他所说的更“深奥”的，包括人类基因组的矢量化和搜索，整个世界在向量中的映射，或所谓的图形嵌入，可以通过 Weaviate 轻松搜索，如 Meta Researches 的图形嵌入上创建的演示 SeMI。

SeMI 于 2020 年 8 月从 Zetta Venture Partners 和 ING Ventures 筹集了 120 万美元的种子资金，此后一直受到风险投资公司的关注。从那以后，它的软件被下载了差不多 75 万次，每月增长约 30%。Van Luijt 没有详细说明该公司的增长指标，但表示下载量可以与企业许可证和托管服务的销售相关联。此外，对 Weaviate 附加值的使用和理解的激增导致所有增长指标上升，公司耗尽了种子资金。

尽管种子资金已经没有了，但该公司并没有积极寻求新的资金。然而，当半联合创始人与 Cortical Ventures 进行对话时，van Luijt 表示，这些公司向他们展示了他们将如何支持这项业务。Cortical Ventures 是由前数据机器人创始人和新企业联合公司(NEA)成立的一只新基金。

“这真的是‘捏我的手臂，令人瞠目结舌’的感觉，”他补充道。“他们过去所做的一切，支持我们的团队，正是我们所寻找的，我可以说，尽管这是非常新鲜的经历，但所有令人惊叹的故事都是真实的。”

这些对话导致 NEA 和 Cortical 共同领导了新一轮 1600 万美元的首轮融资。

SeMI 打算将这笔新资金用于雇佣美国和欧洲的人才，并在 Weaviate 和 vector search 领域加倍投入开源社区。它还将增加对上市和开源核心产品的关注，并在机器学习和计算机科学重叠的研究领域迈出第一步。

与此同时，van Luijt 认为，我们正在关注数据库技术的下一波浪潮，首先是 SQL 浪潮，它带来了 Oracle 和微软等大赢家，其次是无 SQL 数据库浪潮，赢家是 MongoDB 和 Redis。

“我们现在正处于新一代数据库的边缘，那些人工智能优先的数据库，Weaviate 就是一个例子，”他补充道。“我们不仅需要教育市场 Weaviate，还需要教育市场向量搜索数据库，或者人工智能优先数据库。这是一件非常令人兴奋的事情，因为机器学习带来了一些独特的东西。例如，让你的数据库回答数百万甚至数十亿份文档中的自然语言问题，或者“理解”数百万张照片或视频中包含的内容。”
