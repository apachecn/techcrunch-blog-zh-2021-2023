Join 获得新的资金来建立一个施工“决策”平台

原文：https://web.archive.org/web/https://techcrunch.com/2022/06/21/join-secures-new-cash-to-build-a-decision-making-platform-for-construction/

加入建筑、工程和施工决策平台，今天完成了由 SignalFire 领投的 1600 万美元 A 轮融资，参与方包括 Ironspring Ventures、Metaprop、Building Ventures 和 Standard Investments。据首席执行官安德鲁·祖考斯基(Andrew Zukoski)称，新资本使公司的总融资额达到 3000 万美元，将用于市场扩张和扩大 Join 的产品。

Join 于 2017 年上线。Zukoski、Drew Wolpert、Jim Forester、王也和其他创始团队成员通过谷歌实验产品实验室 X(前身为 Google X)的分支 Flux Factory 相遇，在那里他们开发了跟踪和记录建筑施工决策的技术。Flux 是 2014 年第一家从 X 中诞生的公司，但当一名建筑师起诉 Flux 和谷歌涉嫌窃取他的建筑设计软件创意时，Flux 陷入了争议之中。

Zukoski 希望加入将有一个安静的形象。他通过电子邮件告诉 TechCrunch:“我们设想这样一个世界，团队以信任为动力，建筑以信心为基础。”“总承包商必须赢得工作，进行规划，然后执行。十分之一的项目对总承包商来说是一场财务灾难。他们有很好的跟踪和控制执行的系统。然而，这些系统无法让他们看到早期阶段，随着向协作交付方法的转变，早期阶段变得越来越重要。”

Zukoski positions 的加入取代了 Excel 和其他基于电子邮件的工作流，许多建筑利益相关者使用这些工作流来制定项目决策。大多数项目的风险管理发生在早期阶段，称为“施工前”但是它经常发生在不同的电子表格中，导致不连贯的交流。

Join 从 Excel 和其他评估工具中获取数据，并提供一种表面上更简单的方法来评估施工前决策的影响。例如，使用 Join，用户可以提出“如果...会怎样”的问题，并实时查看某个选项的成本影响。

“主要的现任者，如 Procore 和 Autodesk，已经围绕 20 年前交付工作的方式进行了构建，其中所有者、架构师和承包商之间的关系类似于瀑布软件开发。在得到任何关于成本的反馈之前，团队会设计建筑的每一个细节。这是许多项目延期和超出预算的主要原因，”Zukoski 说。“这已经改变了。现在，80%的工作是通过早期参与设计和施工的协作方法交付的，允许成本通知设计。这是一个巨大的建设机会，但也意味着在建设前阶段，当这些利益相关者聚集在一起时，有一个巨大的合作需求。”

Join 通过仪表板提供实时数据、决策历史和信息，利益相关者可以查看这些信息并做出响应，以解决潜在的警告信号。该平台包括可视化工具以及时间表和预算规划器，根据过去的项目绩效提供见解，以便未来的项目能够受益。

“在疫情和技术放缓的情况下，我们都做好了准备，”Zukoski 说，他指的是 2020 年 Autodesk 的一项调查，该调查显示，糟糕的决策导致建筑业在可避免的返工或重做第一次没有正确完成的工作方面花费了 890 亿美元。“总的来说，建筑行业采用新技术的速度较慢，因此客户有很大的增长空间。”

事实上，建筑领域的技术通常发展缓慢。例如，最近的毕马威加拿大调查发现，加拿大四分之三的建筑公司认为他们的数字化成熟度“相当低”

但是 Zukoski 认为, 疫情驱使建筑公司拥抱技术和新的工作方式，激发他们对技术的兴趣，使承包商的工作总体上更有效率。也就是说，Join 拥有大约 50 家客户，他们已经将该平台用于医疗机构、研究中心、经济适用房以及娱乐和体育中心。

根据 Construction Dive 的数据，2021 年，美国建筑行业的投资者资金达到创纪录的 21 亿美元，比一年前增长了 100%以上，凸显了对建筑技术的需求。就在最近，人工智能建筑管理平台 Buildots 在一轮融资中获得了 6000 万美元。与此同时，建筑设计平台 Alice Technologies 获得了 3000 万美元的新资本，以扩大其平台。

Zukoski 说，A 轮融资的目标之一是到 2023 年将 Join 的团队规模从 52 人扩大到 75 人。
