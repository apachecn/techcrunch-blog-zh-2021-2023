伊菲希望在任何地方提供产品保护保险 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/10/26/evy-wants-to-offer-product-protection-insurance-everywhere/

认识一下法国初创公司伊菲，它致力于延长保修期和产品保护保险。该公司从红杉资本、La Famiglia 风险投资、全球创始人资本和几个商业天使那里筹集了 650 万美元的种子资金(€650 万美元)。

伊菲希望将 AppleCare 体验带给其他品牌和零售商。从本质上讲，这家初创公司希望在结账时增加产品保护和一些开箱即用的良好覆盖，从而创造无缝体验。

这家初创公司充当保险经纪人，并与 Wakam 合作承保风险——但未来它也可能与其他保险公司合作。另一方面，它与零售商合作，让他们在自己的网站上植入伊菲的保险产品，或者试图在商店里出售延长保修服务。

例如，伊菲已经与马诺马诺诺签署了合作伙伴关系，这是一个家装和园艺电子商务平台，我已经报道了多年。当客户在 ManoMano 上购买产品时，他们可以添加针对破损、故障和/或盗窃的多年保险。

让伊菲从传统参与者中脱颖而出的是，它可以在很短的时间内创建定制的保险产品。例如，马诺马诺在四个国家有 25 种不同的产品类别。伊菲在短短几个月内创建了 25 个量身定制的保险项目。

同样，随着 insurtech 初创公司专注于基于 API 的集成，零售商可以与伊菲进行深度集成。

如果产品有问题，伊菲首先会设法找到解决方案来修复产品。它计划建立一个维修网络。如果不起作用，伊菲会向顾客付款。

由于伊菲与其零售合作伙伴分享部分收入，这种分销方式应该会非常有效。一些大型零售商，如 Darty 或 Best Buy，已经从保险产品中获得了重要收入。伊菲希望为电子商务和实体店的长尾提供一个解决方案。

伊菲不仅仅是一个保险公司。围绕产品生命周期有一个更大的愿景。“最终，我们希望提供与产品相关的所有客户服务。他们对商家和终端用户都很有用，”联合创始人兼首席执行官西蒙·凯穆恩告诉我。

“他们都在转向循环经济。我们知道产品何时在保修期内，何时没有发生事故。我们可以提供以旧换新优惠，这样你就可以买到最新的产品，”他补充道。
