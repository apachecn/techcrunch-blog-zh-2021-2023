机器人还是 fauxbot？TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2023/01/12/robot-or-fauxbot/

这是我第周的汇报。CES 有办法把你扔进新的一年，又踢又叫，当你出现在另一边时，很难找到你的方向。随着尘埃落定，有一件事仍然非常清楚:没有两个人对机器人的构成有相同的概念。

这本身不是问题。语言在进化，技术也在进化。本周，我在股票播客上与我们的硬件记者之一 Haje 谈论 CES 。他认为，在这一点上，大部分技术本质上是一个机器人。人们可以相当可信地认为，“机器人”作为一个术语比我们倾向于如何部署它要宽泛得多。当然，机器学习正以很少有人想象的方式变得无处不在。

尽可能广泛撒网的好处是，人们也可以证明无处不在的机器人技术不是未来的愿景。它已经在这里了，也许这是令人振奋的。一个更严格的定义——也是我倾向于选择的一个——包括一定程度的自主和感知。你知道，从外部来源获取信息并做出相应的决定，就像我们一样。

我并不拘泥于这个定义，以至于它阻止我报道这个空间中其他不完全一致的故事。也不是所有符合条件的事情都在我的权限之内。首先，我们有几个非常擅长自己工作的汽车记者，所以他们首先尝试所有自动驾驶汽车的东西。事实是，我们对什么符合或不符合机器人的定义也受到一些编辑决策的支配。它也可以是多孔的。

这很容易成为关于 80 亿激光雷达初创公司的时事通讯，但我真的不希望这样，我怀疑你们大多数人也不希望这样。至少对我来说，当我决定这些页面上什么有意义，什么没有意义时，严格遵守迂腐或正统没有多大价值。但是有一些护栏是有帮助的。

第一次在时事通讯中看到“智能”洗衣机可能是新奇的，但是如果我突然开始每周强迫你使用它们，你会(理所当然地)生我的气。另一方面，机器人吸尘器确实满足了包括我在内的许多人或大多数人的机器人需求。我绝对会在这里介绍它们，但我也做了足够长的时间，知道介绍市场上的每一个机器人吸尘器都是让读者出血的好方法。

图片来源:机器人

在 CES 这样的展会上，所有这些东西都在一起流血。例如，我花时间研究了 Robosen 的极其整洁的变形机器人。但我认为，如果我们试图将这些玩具与展会上展出的一些工业履行机器人进行比较，这对双方都不利。我有点厚脸皮地提出，机器人是“用于不酷事物的酷技术”这显然不是一个指导原则，但它确实指出了一些值得在这里讨论的东西。

怪异和古怪的机器人玩具将在 CES 这样的展会上占据头条。我们明白。它们对交通有好处，而且很有趣。我已经写了很多这样的故事，将来当真正有趣的故事出现时，我会继续写下去。人们被它们吸引的另一个原因是，它们更像一种柏拉图式的机器人理想。机器人玩具看起来是我们认为机器人应该看起来的样子。无论你最终如何看待科幻小说对公众认知的影响，重要的是要认识到它永远不会消失。

然而，随着机器人在我们日常生活中变得越来越重要，这种影响将越来越成为一种对话。我花了一大笔钱在电影院看了新的《阿凡达》，里面有各种各样的装饰。让我震惊的一件事——可能你也一样——是潘多拉星球上的美国军用机器人与当今世界上存在的机器人有多么相似。想惹恼你的约会对象？指出科幻电影中每个机器人的灵感来源。

可以预见的是，我在过去一周写的机器人方面的文章很大程度上围绕着 CES，因为坦率地说，我在过去一周做的大多数事情都围绕着 CES。朋友们，没有人说过硬件编辑的生活会很轻松。

但是两个半的时事通讯对于展会来说已经足够了，所以让我们收集一些非 CES 的顶级新闻，好吗？

图片来源:矿物

另一家公司本周从 Alphabet X“moon shot”实验室毕业。退出秘密行动两年后，矿产公司现在是自己的字母公司。机器人方面在很大程度上围绕着数据收集，监控作物，为农民提供深入而丰富的可操作信息，以实现更可持续和高效的种植。

“在 Alphabet 的 moonshot 工厂 X 孵化了我们的技术五年后，Mineral 现在是 Alphabet 公司了，”首席执行官埃利奥特·格兰特(Elliott Grant)说。“我们的使命是帮助扩大可持续农业的规模。我们正在通过开发一个平台和工具来帮助收集、组织和理解关于植物世界的前所未有的信息，并使其有用和可操作。”

图片来源: iRobot

把这个归档:不是最好的样子。家用机器人和隐私已经到了不可避免的地步。《麻省理工评论》收集了来自 Roomba 的图片，这些图片最终登上了社交媒体；iRobot 回应称，这些镜头都是在用户同意的情况下拍摄的。随着我们将机器人和摄像技术带进家中，这一讨论变得越来越重要。这可能会成为反对亚马逊收购该公司的有力论据。

一位用户称之为“他们明显违反了协议，也违背了信任。”一个很大的问题是，当邀请这种技术进入你的家时，对隐私的合理期望是什么？
