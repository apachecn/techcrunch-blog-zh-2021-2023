亲爱的开创性，

这是多么好的消息啊，你正打算进军美国市场。在最近的播客(这实际上是最近几周在你的国家的趋势！)，我讨论了我对如何最好地建立一个移民成功的创业公司的看法。无论创始人是想去美国，还是已经在美国了，想自己创业，我们都讨论了细节。甚至对新成立的资金有限的公司也有选择。

在美国成立公司时，了解你的选择是很重要的。这种知识使你的公司能够成功地赞助你、你的创始团队和其他潜在雇员获得签证和绿卡(如果需要的话)——潜在投资者也会对投资你的公司感到更放心。超级重要，对吧？！在你深入计划之前，我建议你咨询公司律师和移民律师来帮助你。

有限责任公司对特拉华州 C 公司

你需要做的第一个决定是如何在美国组建你的公司。商业或公司律师可以帮助你做这个决定。一般来说，公司和有限责任公司都能够赞助个人获得美国签证或绿卡。

我在创业生态系统中通常看到的是，大多数风投更喜欢投资在特拉华州创建的 C 公司(特拉华州 C-corp)。这主要是因为特拉华州的法律保护投资者，特拉华州首席企业官可以向员工、投资者和董事会成员分配两种或更多种股票和股票期权。

设立特拉华州 C-corp 公司不需要美国居留权或公民身份，一天之内就可以完成。关于这个话题的更多信息，听听我和我的好朋友和值得信赖的同事林赛·米格纳米的聊天，他是 Smith Shapourian Mignano 的创始合伙人，这是一家位于三藩市的专注于创业的公司律师事务所。

此外，和你的商业律师谈谈这些实体的所有权结构。例如，一种可能性是你的危地马拉公司可能拥有你的美国公司。然而，大多数投资者更愿意投资总部设在美国的母公司，所以你可以考虑成立你的美国公司，在危地马拉拥有你的公司。然而，如果你已经在危地马拉发行了你公司的股票，这可能是一个繁琐的过程，最好等到你准备好融资的时候。然后，你可以做所谓的“特拉华翻转”——你绝对应该让你的公司律师建议你这么做。
