Redditors 创造了数百万个加密钱包来购买 NFT 头像

今年 7 月，Reddit 跳上了 NFT 的列车，推出了一个位于 NFT 的市场，允许用户以固定价格购买位于区块链的个人资料图片。考虑到今天围绕非功能性测试的普遍情绪，你可能会像我一样认为这个实验结果不佳。但显然情况正好相反。

今天在 TechCrunch Disrupt 的一个小组讨论会上，Reddit 首席产品官 Pali Bhat 透露，迄今为止，超过 300 万 Reddit 用户已经使用 Reddit 的 Vault 区块链钱包创建了超过 300 万个加密钱包。他说，其中大部分——250 万——是为了购买 NFT 头像而创建的，这些头像可以在平台上用作个人资料照片。

考虑到并非所有的 NFT 市场都愿意分享这些指标，很难将这些数据放入背景中。但 Dune Analytics 在最近的一份报告中估计，领先的平台之一 OpenSea 截至 1 月份托管了超过 100 万个活跃钱包。与此同时，截至 3 月，加密钱包 MetaMask 为超过 3000 万用户提供服务。

几个月前，Reddit 与大约 30 名艺术家合作，发布了大约 4 万个 NFT 头像设计，可以通过 Reddit 移动应用程序购买，价格从 9.99 美元到 99.99 美元不等。(目前都卖完了。)购买了其中一张限量版照片的用户获得了许可权，可以在 Reddit 上使用它作为头像，并可以使用内置的头像构建工具来混合和匹配他们头像的外观。

Reddit 与以太坊兼容的区块链 Polygon 合作，铸造这些头像。(在区块链行话中，“铸造”指的是出版一个独特的 NFT，以便可以买卖。)Vault 用于通过 Reddit 应用程序存储和管理 NFT。

继 Twitter 之后，社交网络开始允许用户将任何 NFT 设置为他们的个人资料图片，这是 Reddit 进军 NFT 空间的最新举措。当时，NFT 的头像只对被邀请的会员开放。

除了 NFTs 之外，Reddit 最近几个月已经向多个不同方向发展，寻找新的收入来源。该公司将引入类似抖音的视频编辑工具的想法玩弄于股掌之间，并进行了几次有针对性的收购，收购了内容审核初创公司奥特鲁，自然语言处理公司 MeaningCloud ，机器学习平台 Spell 和语境化公司 Spiketrap 。Reddit 还改进了其开发者门户，为第三方应用和机器人提供了动力，并升级了其直播音频产品 Reddit Talk ，增加了新的发现功能。
