为即插即用的数字银行工具存入 500 万美元

原文：https://web.archive.org/web/https://techcrunch.com/2022/08/23/deposits-5m-digital-banking/

越来越多的公司在他们的产品中加入支付和其他金融功能；然而，这通常需要一些没有的专业技术知识。

进入 Deposits ，这是一家位于达拉斯的金融初创公司，提供基于云的即插即用功能，以简化信用社、社区银行、保险公司、零售商和品牌等公司的数字银行工具的实施。

联合创始人兼首席执行官 Joseph Akintolayo 在与银行合作多年并听取人们描述他们希望自己的钱如何运作后，于 2019 年与丹尼尔·帕拉莫(Daniel Paramo)一起创办了这家公司。

“他们经常描述一些听起来有点像信用合作社的东西，”他告诉 TechCrunch。“他们想要拥有感。他们想确保它支持他们的社区。他们想要掌控一切，想要个性化，想要与他们交流。”

当他观察他的代理客户能提供什么时，他发现人们要求的和银行能提供的之间有很大差距。例如，只有最大的银行才能真正提供现代体验。然而，Akintolayo 解释说，这对那些银行服务不足或无法负担月费的人来说代价很大。

“剩下的是有‘意愿’的人，以及没有‘途径’的社区银行和信用合作社，”他补充道。“存款的诞生是为了在社区机构和品牌之间架起一座桥梁，这些机构和品牌真正热衷于为消费者提供他们以现代方式服务消费者所需的工具。”

该公司使其客户能够快速轻松地整合一系列功能，包括移动应用程序、借贷账户、移动存款、虚拟卡、点对点支付和带身份验证的在线账户。他们还可以提供房屋和汽车贷款以及外汇兑换等服务。

存款与许多银行、信贷和支付合作伙伴合作，使用无代码或低代码工具和 API 促进其模块化“套件”系列，包括商业银行、资金管理、身份验证、用于购物和结账的嵌入式金融以及工作场所需求。

该公司加入了其他迎合信用合作社和小银行的金融科技公司，如 Narmi 、 Bankjoy 和 Xend Finance 为他们的方法获得风险投资资金。就存款而言，它今天宣布了第一笔投资，由 ATX 风险投资公司牵头的 500 万美元种子轮投资，Cabal Fund，light speed Venture Partners Scout Program 等公司参与了投资。

“第一波嵌入式金融科技平台展示了需求，但需要深厚的技术专业知识，并遭受了延迟推出，”ATX 风险投资公司(Bernstein Venture Partners)合伙人兼联合创始人克里斯·肖克(Chris Shonk)在一份书面声明中表示。“存款公司通过一个几乎任何信用社或产品品牌都可以使用的低代码平台兑现了承诺。再加上约瑟夫组建的团队，其令人信服的使命和令人印象深刻的早期牵引力，这是一个很容易的投资决定。”

Akintolayo 说，该公司新注入的资本将用于产品开发，并增加其 27 人的销售、营销和工程团队。他最近带来了一位业务发展副总裁，并将充实高管团队。

存款平台仍处于早期阶段，刚刚推出一年多一点；然而，Akintolayo 表示，它已经“显示出强烈的信号”，表明它已准备好迎接更广阔的市场。

“对我们来说，业务并没有放缓，”他补充道。“我们非常看好我们的细分市场，并支持经济中非常需要和重要的部分。我们提供的产品和服务无论是在充裕的时候还是在需要的时候都是需要的，所以我们感觉非常好。”
