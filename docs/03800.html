菲律宾支付网关 PayMongo 获得 3100 万美元 B 轮融资，将探索区域扩张

原文：https://web.archive.org/web/https://techcrunch.com/2022/02/20/philippines-payment-gateway-paymongo-gets-31m-series-b-will-explore-regional-expansion/amp/

PayMongo 创始人(从左至右):首席技术官 Jaime Hing III、首席执行官 Francis Plaza 和首席商务官 Luis Sia

总部位于菲律宾的 fintech PayMongo ，使商家能够接受数字支付，今天宣布它已经筹集了 3100 万美元的 B 轮资金，着眼于区域扩张。投资者包括贾斯汀·马丁(Justin Mateen)的 JAM Fund、ICCP-SBI 风险投资公司(Bernstein-Bernstein Venture Partners)和丽莎·戈孔韦(Lisa Gokongwei)的 Kaya 创始人，以及回归投资者 Global Founders Capital 和 SOMA Capital。这家初创公司表示，这轮融资还包括来自 Qonto、Viva Wallet、Billie 和 Scalable 等欧洲金融科技公司的创始人。

这使得 PayMongo 的总资金接近 4600 万美元。它的最后一次融资是 2020 年宣布的由 Stripe 牵头的1200 万美元 A 轮融资。

该公司与各种规模的企业合作，但特别针对微型、小型和中型企业，使他们能够接受不同形式的支付，包括信用卡、在线钱包和柜台支付。其产品包括 PayMongo API 和电子商务插件。新的资金将用于进一步发展 PayMongo 当前的支付基础设施，并增加更多的金融服务，包括支付、资本借贷、BNPL 以及订阅和经常性支付。

PayMongo 产品路线图的一部分包括获得新的许可证，使其能够运营更多的金融服务。同时，公司也在探索区域扩张。

“在菲律宾还有很多工作要做。我们还预测我们的团队规模将增加一倍以上，以支持这一不断增长的需求，并实现我们积极的产品路线图。与此同时，我们已经开始了一些初步的探索和腿部工作，以扩大在东南亚地区的业务，这项工作我们已经在去年开始了，”联合创始人兼首席执行官 Francis Plaza 在一封电子邮件中告诉 TechCrunch。

菲律宾的其他数字支付网关包括 DragonPay、PesoPay、PayMaya 和 Paynamics。Plaza 在一封电子邮件中告诉 TechCrunch，该公司自 2019 年成立以来，一直专注于中小企业和高增长创业公司。

“除此之外，随着我们与该平台上的数千家企业合作，我们将致力于打造更多的产品和服务，不仅让商家能够轻松接受支付，还能通过获得其他金融服务来实现增长，”他说。“从转移资金的能力，商店余额，获得信贷和客户的其他扩展支付选项。”Plaza 补充说，它已经在与商家一起测试几个新产品和服务。

在一份声明中，Tinder 和 JAM Fund 的创始人贾斯汀·马丁(Justin Mateen)表示，“作为 PayMongo 的首批投资者之一，我看到了他们从为少数企业简化支付到现在成为成千上万商家日常运营所依赖的公司的道路。我对他们的进步感到兴奋，并很高兴再次支持该团队，因为他们通过数字经济创造了更多的经济机会。”
