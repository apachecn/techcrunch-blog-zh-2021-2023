Alphabet X 毕业生机器人农业技术公司矿产

在公开亮相两年多一点后，Mineral 成为了自己的字母公司。这个团队的前身是“计算农业项目”(猜测他们为什么采用这个新名字没有奖励)，刚刚从 X“moon shot”实验室毕业。

“在 Alphabet 的 moonshot 工厂 X 孵化我们的技术五年后，Mineral 现在是一家 Alphabet 公司，”首席执行官 Elliott Grant 在的一篇博客文章中说。“我们的使命是帮助扩大可持续农业的规模。我们正在通过开发一个平台和工具来帮助收集、组织和理解关于植物世界的前所未有的信息，并使其有用和可操作。”

在试图主要通过收购建立一个机器人部门多年后，Alphabet 似乎正在内部建立一个更加有机的部门。Mineral 跟随日常机器人和内在从 X 成长为完全发布的 Alphabet 子公司。

Mineral 使用其内部机器人来创建数据集，并对不同的作物进行研究。它解释说，在它存在的五年时间里(大多是秘密的)，人们发现大多数公司在收集利用机器学习所需的数据范围方面做得足够好。

“没有一种数据收集模式适合每一项农业任务或每一种作物，”格兰特说。“我们从可以捕捉大量高质量图像的植物漫游车开始，随着时间的推移，我们扩展到构建通用感知技术，可以在机器人、第三方农场设备、无人机、哨兵设备和移动电话等平台上工作。”

该公司的最终目标是创建详细而丰富的数据集，供世界各地的农民使用，以挖掘以前未知的种植因素。通过这样做，它希望能帮助培育对气候变化更有抵抗力的作物，而不会加剧这个紧迫的问题。
