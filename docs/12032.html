Directus 希望在整个企业内实现数据民主化| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/11/10/directus-wants-to-democratize-data-across-the-enterprise/

一家希望在企业中实现数据民主化的初创公司？这听起来可能非常熟悉，但今天宣布由 True Ventures 牵头的 700 万美元 A 轮融资的 Directus 正在采取一种不同于大多数竞争对手的方法，将传统的开发工具与无代码方法相结合，为其企业用户提供高度灵活的开源数据平台。使用该服务的工具，开发人员可以轻松地将任何 SQL 数据库转换为 API 来支持他们的应用程序，或者使用该服务的无代码工具来构建应用程序。

尽管它在 2020 年才推出，但总部位于纽约的 remote-first 公司已经将 Bose、Adobe 和 Tripadvisor 等企业加入了其付费客户名单。虽然公司本身现在只有几年的历史，但 Directus 首席执行官兼联合创始人本·海因斯实际上早在 2004 年离开空军并开始网络咨询业务后就开始考虑创办 Directus 的想法。

图片来源: Directus

“我发现工程中有许多重复的工作要做——认证和授权、连接、数据库、数据访问、缓存，”Haynes 解释道。“这就是构建可交付成果的全部内容，但是一旦你交付了，你就需要一种方法来管理它。”

当时，这主要涉及到像 WordPress 或 Drupal 这样的 CMS 和像 phpMyAdmin 这样的 LAMP 栈数据库管理工具。但是没有任何伟大的工具来为新项目构建信息架构，所以 Haynes 最终编写了 Directus 的第一个版本。虽然他在 SoulCyle 和 AOL 工作期间一直将它作为一个兼职项目，但直到 2020 年，当他和他的联合创始人 Rijk van Zanten 开始对这个他们以前只在咨询业务中使用的工具进行更认真的询问时，它才成为一份全职工作和一个初创公司。今天的 Directus 显然已经不是 PHP app 了。它已经被完全重写，位于现代 Jamstack 平台之上。

Directus 联合创始人本·海恩斯(左)和里克·范·赞滕(右)。图片来源: Directus

也许描述 Directus 用户今天体验的最佳方式是混合使用以代码为中心的数据库管理工具和类似 Airtable 的 T2 Directus Studio 无代码工具。

正如 Haynes 所强调的，该公司的业务不是管理数据库本身。相反，它可以位于任何 SQL 数据库之上。“数据库不是我们平台的一部分，”他指出。“那是你的数据。你有权力。我们在数据库管理员管理工具上加了一层。我们试图为你的模式、你的优化、你的外键约束提供工具和数据库入口——无论你优化了什么，都保持完全不变。我们不混合任何系统数据。如果你删除我们的软件，六个月或六年后，它会完全恢复原样。”

虽然业务用户也可以使用该服务，但核心受众——即使对于无代码/低代码工具也是如此——是开发人员。“我们仍然专注于开发人员，将其作为我们理想的客户群。我们正在与开发者交流和合作，”海因斯说。他认为，像 Retool 或 Airtable 这样的服务首先是无代码平台，然后试图回填技术。“你最终只能得到一个权宜之计——当需要扩展时，开发者可能不会满意的权宜之计，”他说。"我们首先是数据库，然后是 API，然后是连接，然后是无代码."

这种对开发人员的关注也体现在这样一个事实上，即该服务在命令行界面和 JavaScript SDK 之上提供 REST 和 GraphQL APIs 来连接到其服务。

图片来源: Directus

对于开发人员来说，这意味着他们在如何使用工具和操作数据(无论是文本、图像还是地理数据)方面获得了很大的灵活性。该工具既有开源的版本也有完全免费的管理服务，付费层的价格从每月 25 美元起。

该公司现在有 25 名员工，共筹集了 850 万美元。除了 True Ventures 领投这一轮 A 轮，握手创投也参与了。

“让非技术用户拥有无代码工具是企业界正在进行的一场巨大变革，”True Ventures 联合创始人菲尔·布莱克说，他将加入 Directus 董事会。" Directus 是一个开源项目，在不到一年的时间里已经被下载了超过 2000 万次。在众多好处中，该软件帮助团队大大减少了开发人员创建数据驱动项目的时间。更重要的是，我们喜欢创始团队在开始 Directus 之前花时间深入这个问题。动手奋斗会带来创新。”
