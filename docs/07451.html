摸摸你的脉搏:API-first 初创企业 Vivanta 希望为健康而格子化

API 至上的公司正在崛起，不仅是在金融科技领域，在医疗保健等领域也是如此。这种多样化是由这样一个事实推动的，即在银行业应用编程接口上获得成功的员工现在正把他们的技能应用到其他问题上。

医疗保健是可能受益于 API 解决方案的行业之一。虽然了解您的心率或血糖水平很有价值，但 API 可以帮助公司为其最终用户提供更全面的健康视图，使他们能够采取正确的措施来保持健康。

总部位于墨西哥的创业公司 Vivanta 正处于医疗保健和 API 的交汇点。它的重点是健康数据，着眼于可穿戴设备正变得无处不在的事实。但其联合创始人亚历克斯·埃尔南德斯(Alex Hernandez)和豪尔赫·马德里加尔(Jorge Madrigal)之前曾在金融科技公司合作过:去年 12 月被万事达卡收购的 Arcus，以及 API 主导、有时被称为“LatAm 的格子图案”的 Belvo

这种以 API 为中心的跟踪记录是这家初创公司在推出产品之前筹集 30 万美元种子期前资金的关键——Vivanta 下个月将进行一次私人发布。

“我们投资 Vivanta 是因为 Jorge 和 Alex 在将新的 API 产品推向市场方面有着成功的记录，”99startups 管理合伙人 Alejandro Gálvez 说，他与瓜达拉哈拉的 Redwood Ventures 、蒙特雷的 Angel Hub MTY 和专注于墨西哥的辛迪加 Lotux 一起参与了这笔资金。

Vivanta 的创始人还邀请了十几个人参与这轮融资，重点是拥有相关专业知识的运营商。“能够向以前做过的人提问是非常宝贵的，”马德里加尔告诉 TechCrunch。“我们希望有一个强大的可以依靠的人际网络。”

Vivanta 的一些天使投资者本身就是创始人，比如 Arcus 的 Edrizio de la Cruz、Madrigal's 和 Hernandez 的前老板。但这两人也有意接触了在 cap 表格上可能不太常见的技术资料——包括目前正在开发 API 的女性工程师和首席技术官。
