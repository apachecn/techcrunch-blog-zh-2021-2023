Box 以一个超出所有人预期的季度业绩反击，包括它自己的预期

去年的这个时候，Box 正在全力以赴阻止以 Starboard Value 为首的激进投资者接管董事会并主宰公司方向。如果维权投资者赢了，首席执行官兼联合创始人亚伦·李维可能会被迫出局，公司可能会被出售。

但是 Box 赢了，支持高管团队的股东本周获得了超出所有人预期的四分之一，包括公司自己。

现在，代理权之争已经结束，很明显，Box 在过去几年中为进一步进入真正的企业市场而建立的一些举措正在取得回报。 Alan Pelz-Sharpe，深度分析首席分析师

据该公司称，上个季度，Box 的收入达到 2.33 亿美元，同比增长 17%，超过了分析师预计的 2.29 亿美元。收入结果也高于 Box 自己的估计，即增长将在 12%左右。截至周五上午，该公司股价本周上涨逾 6%。

去年，当 Starboard 在公司下面点火时，Box 报告了一组非常不同的季度业绩。8.3%的增长乏力，给该公司领导层带来了开始改善增长和盈利能力的压力。

Levie 对他的公司的结果很乐观，这是可以理解的，他说 Box 的增长不是来自任何一个特定的领域，而是来自所有领域。

“从快速增长的小型企业到正在经历重大数字化转型的大型企业，无所不包。我的意思是，我们取得了一些重大胜利，包括大型银行、政府机构和大型工业产品制造商。所以它的基础非常广泛，”他说。

业务增长的一个主要原因是捆绑销售，去年 Zendesk 的捆绑销售也取得了巨大成功。事实证明，构建一套产品对 SaaS 公司来说效果很好。

“我认为从商业模式的角度来看，最重要的是我们向捆绑计划的转变。因此，我们有了 Enterprise Plus，这是一个新的企业版 Box，它在一个套件中集成了 Box Shield、Box Governance、我们的设计功能[以及其他功能]。这无疑加速了我们的增长，并导致越来越多的客户采用我们的完整平台，”Levie 解释道。

Deep Analysis 是一家关注内容管理空间 Box 运营的公司，其创始人兼首席分析师艾伦·佩尔茨-夏普(Alan Pelz-Sharpe)表示，代理权之争拖累了公司，现在它已经过去了，它已经能够向前发展了。
