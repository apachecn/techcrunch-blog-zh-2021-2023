Mambu 以 55 亿美元的估值获得 2.66 亿美元，对嵌入式金融服务和银行应用编程接口加倍投资| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/12/09/mambu-nabs-266m-at-a-5-5b-valuation-to-double-down-on-embedded-financial-service-and-banking-apis/

嵌入式金融是过去十年金融科技的主要趋势之一。在嵌入式金融中，公司利用 API 来集成银行和其他金融工具，而不必从头开始构建一切。今天，构建这些 API 及其背后的后端技术的最大的初创公司之一宣布了一轮重大的融资，以加倍抓住这个机会。

Mambu 已经开发了一套 API 来支持贷款、存款和其他银行产品，以及大量的分析和其他工具来建立金融服务，已经为€筹集了 2.35 亿美元，这一系列 E 对该公司的估值为 49 亿€(按今天的汇率计算，约为 2.66 亿美元，估值为 55 亿美元)。

这笔资金由欧洲私人股本公司 EQT 成长(Bernstein Growth)牵头，总部位于阿姆斯特丹/伦敦的 Mambu 没有披露其他投资者。(它的最后一轮融资是在 1 月份筹集了 1.35 亿美元，由 TCV 牵头，参与方包括 Tiger Global、Arena Holdings、Bessemer Venture Partners、Runa Capital 和 Acton Capital Partners。自那以来，它的估值已经翻了一倍多。)

如今，该公司拥有 200 多家客户，与 65 个国家的 5300 万用户合作。它们包括 N26 这样的大型新银行，也包括 Raiffeisen Bank、BancoEstado 和 ABN Amro 这样的现有银行。

更广泛地说，Mambu 还与电信和电子商务公司等金融服务的新进入者合作，这些公司可能没有能力或没有兴趣从头开始构建金融服务，但可以看到一个业务或运营机会，为他们的客户更直接地提供其中一些工具，并投资构建更定制的金融服务，利用另一家公司(即 Mambu)提供的构建模块，这种模式为 Mambu 打开了通往更广泛客户的大门。

事实上，Mambu 的联合创始人兼首席执行官尤金·达尼尔基斯(Eugene Danilkis)对他认为 Mambu 可以在更广阔的金融服务世界中占据的位置有着远大的抱负。

“我们的愿景是在金融服务和银行领域创造 Salesforce 在 CRM 领域的成就，”他说。

就这一点而言，Salesforce 自己的野心也很大。然而，这些天来，它不是一个竞争对手，而是一个强大的合作伙伴，Danilkis 说:Mambu 的部分力量不仅在于它提供的金融服务 API，还在于它建立的网络，以便这些服务可以与公司可能使用的所有其他 it 基础设施和应用程序“很好地配合”。除了 Salesforce，Mambu 还集成了许多其他第三方，如 AWS、Google、Marqeta、compliant Advantage、Deloitte 等。现在，Mambu 每天处理的 API 调用数量约为 1.3 亿。

在此次融资之前，Mambu 在银行中仍有大量资金，因此最新一轮融资的计划将是利用这些资金加倍将其业务扩展到更多市场，并开始考虑该公司接下来想要解决的其他邻近领域。Danilkis 说，保险等领域肯定在它的考虑范围之内，尽管他拒绝透露它可能何时推出服务，以及这些服务可能是什么。例如，一个合理的领域可能是财产保险，因为它已经提供抵押贷款 API，并与那些已经活跃在财产金融领域的公司合作。

更直接的是，他认为，在简单地改善金融服务和创造这些服务的基础领域，仍有大量机会。当你提到“嵌入式金融”时，他实际上有点恼火。他说，该公司更喜欢用“SaaS 银行平台”来形容其所做的事情，这说明了 Mambu 在帮助客户为自己打造量身定制的金融服务方面发挥着更为重要的作用。

“如果我们回到行业中，大多数银行和金融科技公司关注的是大批量产品，这些产品需要非常数字化，但仍需要感觉为个人定制，”他说。"它超越了贷款和存款."他说，从长远来看，Mambu 希望帮助他们制造“更复杂的商业产品，在这些产品中，你可能产量更低，但价值也更高。”他指出，金融服务中的集成和中间件也存在挑战，这是 Mambu 将向市场提供更好技术的另一个领域。他说，重点是“如何在生态系统中更好地集成”，这将包括为第三方解决方案创建更多预构建的集成。“我们正在投资将许多生态系统整合在一起，”他补充道。(这就是 Salesforce 的精神。)

可能还会有收购，这是曼布还没有做的事情，到目前为止，它只是有机增长。不过，它所工作的领域整合的时机已经成熟。首先，Mambu 本身有许多直接的竞争对手，如 think Machine、Rapyd 和 Airwallex(以及许多其他公司)。这三家公司的资本都非常充足，第一家在 11 月底筹集了 2 亿美元；迄今为止，T2 已经筹集了超过 8 亿美元的资金；来自亚洲的 Airwallex 现在估值 40 亿美元。还有许多较小的地区性参与者和许多“点解决方案”在云中处理金融服务和金融基础设施的不同方面。

实际上，EQT 在上一轮之前就在和曼布谈判，但这只是第一次投资 e 轮。

“我们几年来一直在关注曼布的旅程和出色的执行力，”EQT 成长顾问团队的合伙人卡罗丽娜·布罗查多说。“鉴于责任和可持续发展价值观的紧密结合，我们感到与 Mambu 作为一个组织的愿景有着紧密的联系，并相信 Mambu 的解决方案正在使全球数百万人更容易获得和负担得起金融系统。我们很高兴与欧洲领先的金融科技公司和管理团队之一合作，该公司提供行业独特的领先现代云平台，以应对由传统银行软件供应商和内部解决方案服务的€1000 亿美元的巨大市场。”她将在这一轮加入董事会。
