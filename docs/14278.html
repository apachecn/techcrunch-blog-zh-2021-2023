微软收购数据处理设备制造商 Fungible 来支持 Azure

12 月，报道暗示微软已经以大约 1 . 9 亿美元收购了一家名为“可替代的”,一家制造一种被称为数据处理单元(DPU)的数据中心硬件的初创公司。今天，微软证实了的收购，但没有透露收购价格，称其计划利用可替代公司的技术和团队提供“多种 DPU 解决方案、网络创新和硬件系统进步”

微软 Azure Core 部门的 CVP Girish Bablani 在一篇博客文章中写道:“可替代的技术有助于实现高性能、可扩展、分散、安全的横向扩展数据中心基础设施。”“今天的发布进一步表明了微软对我们数据中心基础设施的长期差异化投资的承诺，这将增强我们广泛的技术和产品，包括卸载、改善延迟、增加数据中心服务器密度、优化能效和降低成本。”

DPU 是一种专用硬件，用于处理特定的数据处理任务，包括数据流量的安全和网络路由。该方法旨在帮助减少 CPU 和 GPU 上与给定工作负载相关的核心计算任务的负载。

2016 年，前苹果软件工程师伯特兰·塞莱特与克里希纳·亚拉加达和瞻博网络联合创始人帕拉德普·辛杜一起，将云存储初创公司 Upthere 出售给了西部数据。可替代出售的 dpu 依赖于两个操作系统，一个是开源的，另一个是专有的，以及一个叫做 MIPS 的微处理器架构来控制闪存存储容量。

在微软收购之前，Fungible 设法从包括软银的愿景基金和 Norwest Venture Partners 在内的投资者那里筹集了超过 3 亿美元的风险资本。但据报道，其 DPU 架构很难开发，这可能会影响其发展势头。八月，在谣传未能出售给 Meta 之后，该公司透露已经裁员并缩减其产品组合。

dpu 市场竞争的加剧也给可替代产品带来了压力。Nvidia 在 2019 年收购了 DPU 制造商 BlueField，而 AMD 在去年年底收购了 Pensando。其他竞争对手包括 GigaIO、Liqid、Lightbits、VMware 的 Project Monterey 和 Amazon Web Services 的 Nitro cards，它们提供类似 DPU 的功能。

换句话说，微软得到了 DPU 的技术，可以用来支持 Azure——也许可以作为块存储的订阅产品或分层服务出售。有趣的是，这是这家科技巨头最近几个月的第二次以数据中心为中心的收购，继去年 12 月收购高速光纤初创公司 Lumenisity 之后。

“可替代的 DPU 是在 2016 年发明的，旨在解决横向扩展数据中心最重要的问题:服务器节点内以数据为中心的计算执行效率低下，”可替代在其网站上的声明中写道。“我们很荣幸能够成为与可替代公司有着共同愿景的公司的一员，并将利用可替代的 DPU 和软件来增强其存储和网络产品。”

Bablani 说，这个可替换的团队将加入微软的数据中心基础设施工程团队。
