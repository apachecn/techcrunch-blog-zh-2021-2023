Twitter 正式推出标签来识别“好机器人”

原文：https://web.archive.org/web/https://techcrunch.com/2022/02/16/twitter-officially-launches-labels-to-identify-the-good-bots/amp/

图片来源:布莱斯·德宾

Twitter 去年秋天推出了新标签，允许其服务中所谓的“好机器人”识别自己。虽然“机器人”这个词往往有负面含义，但 Twitter 指出，也有有用的机器人，它们会自动在 Twitter 上发布有用的信息，如新冠肺炎更新、地震警报、国会正在提出的法案等。这些“好机器人”标签，曾经在一个小团体中测试过，现在对所有自动账户持有人公开。

在测试期间，将自己标记为机器人的 Twitter 帐户将在他们的 Twitter 个人资料上显示这些信息。在帐户名称和@username 下面，一个小机器人图标出现在“Automated by”字样旁边，后面是帐户操作员的姓名。与此同时，推特上的个人简介将详细说明这个机器人的用途。

当机器人发微博时，它们的自动状态也会出现在用户的时间表中。

Twitter 表示，这些信息将帮助用户决定关注、参与和信任哪些账户。

图片来源:推特

9 月份，约 500 个 Twitter 开发者账户首次获得了账户标签，这些账户测试了该功能并提供了反馈，其中包括机器人，如 @earthquakesSF 、 @vax_progress 、 @last100bills 、 @AltTxtReminder 、 @met_drawings 和 @EmojiMashupBot 。这些帐户代表了机器人的使用范围，从关键更新到有趣的信息到有趣的内容。

除了“好机器人”，许多人以其他方式自动化他们的 Twitter 账户——例如，使用 IFTTT 集成来发布链接。但 Twitter 对这种自动推文不太担心，它表示——只是指出，用户应该调查他们在账户上使用的任何第三方应用程序，并确保遵守 Twitter 的规则。

图片来源:推特

机器人标签虽然有用，但不一定能解决 Twitter 服务中机器人的更大问题。

坏机器人的范围很广，从令人讨厌的——像发垃圾微博的机器人(或推广加密货币方案的机器人！)—更令人担忧的是，像试图影响即将到来的选举的机器人。作为一个选择加入系统，试图用他们的创造做好事的机器人所有者将能够添加标签，并宣传自己是帐户背后的人。但是坏的机器人不会参与。

使用 Twitter API 的自动账户今天可以使用该功能。但 Twitter 表示，目前没有对该系统进行审计。

“推特上的所有账户都要遵守推特规则，”推特发言人说。“由于该标签仅用于选择加入，我们将不会审核此时选择采用该标签的帐户，如果帐户违反规则，我们将依靠我们的报告流程。”

Twitter 将于美国太平洋时间今天下午 3 点在由 Twitter 安全团队主持的 Twitter 空间上进一步讨论该消息，随后将于太平洋时间周四上午 11 点在 TwitterDev 的 Twitch 频道上进行开发者演示。
