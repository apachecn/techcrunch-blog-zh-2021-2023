EdgeDB 希望为尖端应用程序实现数据库现代化

原文：https://web.archive.org/web/https://techcrunch.com/2022/04/22/edgedb-wants-to-modernize-databases-for-cutting-edge-apps/?guccounter=1

数据库是在数字世界中开展业务的一个重要组成部分，但并不是每个组织都能利用它们取得成功。在 2019 年的 Vanson Bourne 调查中，企业表示他们受到了传统数据库技术的阻碍，特别是关系数据库系统。受访者表示，关系数据库的设计没有考虑到新的应用程序或云，这使得它们有时难以适应和使用。根据调查，72%的公司认为他们对关系数据库(包括围绕关系数据库构建的架构)的依赖限制了他们实施数字化转型项目的能力。

不出所料，公司越来越接受关系数据库的替代品，如 NoSQL 。由于传统解决方案缺乏可扩展性，他们正在寻找现代系统，包括基于云的系统，支持扩展，同时降低成本并加快开发。高德纳预测到 2022 年，75%的数据库将被迁移到云服务——凸显了这一转变。

“数据库行业正面临着向新商业模式的重大转变，”创建下一代数据库架构的初创公司 EdgeDB 的首席执行官尤里·塞利万诺夫(Yury Selivanov)通过电子邮件告诉 TechCrunch。“很明显，有很多中小型企业需要快速构建软件，然后将数据托管在云中，最好是以一种方便而经济的方式。”

Selivanov 将他在 2019 年与 Elvis Pranskevichus 共同创立的 EdgeDB 吹捧为遗留数据库问题的解决方案之一。EdgeDB 的开源架构是关系型的，但 Selivanov 说，它旨在解决一些基本的设计缺陷，这些缺陷使得使用数据库(关系型和 NoSQL 型)对企业来说是不必要的繁重。

“EdgeDB 有一个非常雄心勃勃的目标:以开发者体验为重点，重新想象关系数据库，”Selivanov 说。“我们在这方面非常独特。虽然大多数数据库公司都关心可伸缩性，但我们希望让实际的开发人员在使用 EdgeDB 进行开发时，比使用任何其他数据库(无论是 SQL 还是 NoSQL)时更有效率。”

在推出 EdgeDB 时，Selivanov 和 Pranskevichus 借鉴了他们在 MagicStack 的经验，magic stack 是一家总部位于加拿大多伦多的软件咨询公司，他们在 2008 年帮助共同创立了该公司。2016 年，在观察到 MagicStack 的许多客户面临的数据库障碍后，Selivanov 说，他和 Pranskevichus 意识到前进的道路是成为一家产品公司。

EdgeDB 的 1.0 版本在 2 月份悄悄推出，并带来了一个集成的访问控制系统和一种查询语言 EdgeQL，Selivanov 声称它比传统的 SQL 快 10 到 1000 倍(取决于操作)。目前正在开发的是 EdgeDB 2.0，它将引入数据库可视化 UI 和对 WebAssembly 的实验性支持，web assembly 是在 web 浏览器中运行二进制程序的开放标准。

正如大多数参与开源的初创公司一样，EdgeDB 的目标是通过基于 GitHub 托管代码库的托管服务来赚钱。Selivanov 说，即将推出的 EdgeDB Cloud 将提供“丰富的”图形用户界面和对终端命令的支持，以创建云数据库实例，以及与前端 web 应用开发堆栈 Vercel 的集成。

“我们的云数据库将跟踪缓慢的查询，并建议如何优化数据库布局或查询。它将提供内置的性能跟踪和与 DataDog 等服务的交钥匙集成，”Selivanov 补充道。“我们还没有任何与机器学习相关的功能，但我们正在考虑直接在我们的数据库中建立一些数据科学功能……我们目前还没有任何具体的计划，但这对我们来说是一个有趣的未来垂直市场。”

EdgeDB 仍然是预收入，但 Selivanov 预计该公司将在 2022 年第四季度开始产生现金，这是高级 EdgeDB 云的暂定发布窗口。迄今为止，总部位于加利福尼亚州旧金山的 10 名员工 EdgeDB 已经从 Accel 和天使投资人那里筹集了 400 万美元，其中包括 OpenAI 的联合创始人兼首席技术官 Greg Brockman。

“在一些领域，我们将与各种公司争夺开发商的份额，如 Prisma、Supabase 和 PlanetScale，”当被问及他认为谁是主要竞争对手时，Selivanov 承认。“[但是] EdgeDB 的价值主张是独一无二的，因为我们同时极大地改善了许多领域的数据库数据交换…我们已经看到了一场成功的草根运动，人们开始用 EdgeDB 构建真正的生产应用程序。我们预计，一旦我们在大约一个月后推出我们的云产品以及与 Vercel 的原生集成，这种牵引力将会加快。”
