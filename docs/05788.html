Arrow 希望让结账过程变得更简单

原文：https://web.archive.org/web/https://techcrunch.com/2022/04/24/arrow-saves-online-shopping-carts-in-southeast-asia/

即使在信用卡渗透率很高的市场，购物车废弃仍然是在线供应商关注的主要问题。现在想象一下东南亚的情况，那里的许多国家都有许多电子钱包，现在购买，以后支付服务和其他形式的支付。银行转账也是网上购物的热门选择，但涉及几个步骤，这增加了放弃购物车的风险。

Arrow 希望通过充当支付网关上的一层来简化结账流程。它支持 50 多种支付方式，包括新加坡、马来西亚和印度尼西亚的所有主要支付方式(包括 Atome、GrabPay、Boost 和 GoPay)。

该公司今天宣布，它已经筹集了 480 万美元，由红杉资本(Sequoia Capital India)牵头，阿尔法 JWC 和 Zinal Growth 参与其中。包括 AIG 和 Maxis 董事会成员黄惠庭(Ooi Huey Tyng)、Paysend 首席运营官史蒂夫·维克斯和比特币基地东南亚负责人哈桑·艾哈迈德(Hassan Ahmed)在内的天使投资人也参与了这轮投资。

Arrow 的首席技术官 Sudhan Raj 以及创始人 Sebastian Roervig 和 Liat Beng Neo。图片来源:箭头

Arrow 于 15 个月前推出，由 Liat Beng Neo 和 Sebastian Roervig 创建，目前已有 100 家商家使用。Liat Beng Neo 告诉 TechCrunch，放弃购物车的一个主要原因是“该地区当前的结账流程没有考虑到其令人难以置信的多样性。东南亚由 11 个不同的国家组成，每个国家都有自己独特的电子商务习惯和细微差别。”

例如，他补充说，一些地区的互联网连接很差，所以如果涉及到点击多个网页，客户可能会退出结账过程。甚至像银行转账这样流行的支付方式也需要几个步骤，每一步都意味着顾客改变购买想法的风险。

除了支付方式，Arrow 还集成了运输信息和附属的忠诚度计划，因此客户可以在一个结账页面上看到所有内容。Arrow 可以集成到 WooCommerce 或 Magneto 等购物平台中，或者通过 API 让商家用 Arrow 替换他们现有的店面。对于社交商务，零售商获得一个结账链接，他们可以向客户发送消息。

Liat Beng Neo 说，Arrow 可以被所有类型的商家使用，但它专注于快速消费品和其他非必需的商品和服务，因为这些商品的弃车率往往很高。他补充说，这也特别迎合了大量交易的商家，因为他们将从购物车放弃率的提高中受益最多。

Arrow 目前活跃在新加坡、马来西亚和印度尼西亚，并计划目前专注于这三个市场，同时计划向菲律宾、泰国和越南扩张。
