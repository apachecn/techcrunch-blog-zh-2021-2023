为什么你不应该忽视欧洲的深度科技繁荣

新数据显示，非洲大陆正在远离消费者风险投资

如果你关注全球人工智能竞赛，你会认为只有两个国家在争夺领导权。但中国和美国远非仅有的拥有发达的初创企业和现有企业、强劲的风险资本活动以及能够将早期创意转化为上市公司的资本市场的科技市场。

人工智能竞赛是更大的深度技术竞争的一部分，包括芯片制造等事情，这是另一个经常作为美中问题讨论的热点。现实情况是，许多国家都在押注于深度技术——该交易所认为投资于可能缺乏直接商业应用的复杂技术，或者专注于为其他更有利于市场的产品奠定基础。

我们很好奇欧洲是否能形成第三个独立的中心或中心集团，用于深度技术投资、知识产权和公司创建，独立于中国和美国。

我们分两部分来探讨这个问题:今天，我们分析 Angular Ventures 关于欧洲和以色列的企业和深度技术风险投资的报告。然后，下周早些时候，我们将收集欧洲主要投资者和创始人对这个问题的评论，以更好地理解实地观点。

总数上升

首先，请注意几点。Angular Ventures 的报告包括来自以色列的数据，以色列是一个活跃的深度科技市场。风险公司的投资包括一些企业投资(SaaS、金融科技、电子商务等等)。所以我们需要做一些解析，但是没有一个数据集是完美的，Angular 发表的东西非常有用。

我们需要考虑的第一个数据点是焦点。Angular 将风险资本活动分为两类，这有助于进行广泛的比较。如果我们从企业和深度技术与消费者的角度来观察欧洲的风险资本市场，数据表明，欧洲大陆已经从平衡的投资重点彻底转变为非消费者工作的优势。

例如，从 2014 年到 2017 年，这两大类大致相当，各自持有欧洲约 50%的风险投资活动。但直到 2018 年的今天，这一比例已经转变为大约三分之二/三分之一，欧洲深度技术和企业投资占据了大部分交易流量。

最近在风险投资对欧洲的关注上出现了明显的分歧，这有助于解释为什么我们的问题不仅仅是无所事事。欧洲已经从一个消费者和非消费者投资相当平均的市场演变成一个严重偏向于后端技术而非面向消费者的产品的市场。
