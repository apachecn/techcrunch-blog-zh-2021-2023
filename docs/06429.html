最热门的拍摄

对于加密投资者来说，这是一个疯狂的一周，没有其他的说法。但与之前的崩盘相比，这是一种不同的疯狂。

简而言之，随着以太坊和比特币等顶级硬币大幅下跌，而其他区块链网络基本上崩溃，本周全球加密市场市值蒸发了数千亿美元。随着代币不分青红皂白地全面崩溃，数十万名加密投资者在交易中被清算，与此同时，Terra 的 stablecoin 惨败——我的同事 Jacquie 在这里有大量细节——似乎在一两天内蒸发了数百亿的加密财富。

对于长期的加密交易者来说，市场上的巨大下行压力可能看起来很老套，但亏损的金额和亏损的人数比以往任何时候都要多，因为加密市场在这轮牛市中扩张得如此之快。如果加密市场继续陷入困境，随着 web3 的付费收购预算随着数量的减少而枯竭，消费者的加入将会受到很多持久的损害。

经过几年的 Robinhood 和 r/wallstreetbets 散户投资者在公开股票上的赌博，消费者准备好了加密，行业张开双臂欢迎他们。在过去的几年里，风险投资者一直在押注面向消费者的加密垂直市场，将投资与拥有代币和 NFT 集成的实际游戏游戏化。与此同时，web3 的追随者强调“社区”是基于加密的平台的杀手级特征之一，并解释说，让用户在平台中拥有经济利益将导致他们按照平台的最大利益行事，并相应地传播福音。

在这种加密牛市的“只涨不跌”时代，这一切都表现得足够好了，但现在有趣的部分来了。

当有经济激励存在的时候，给予用户经济激励让他们享受你的产品就足够了，但是当空间里的空气被抽走，用户只剩下一个赤裸裸的、无趣的平台时，事情看起来就有点不一样了。“玩赚”游戏公司已经为游戏筹集了数十亿美元，这些游戏只有在你变得富有时才有趣，否则就很糟糕。NFT 的项目同样诱使用户使用类似交易卡的机制，这种机制只有在资金流动时才有趣。与此同时，风险投资为 web3 媒体公司、出版物和社交网络公司提供资金，这些公司都过度依赖加密投机，同时通常运输劣质产品。

有些人可能会将此解读为对加密庞兹经济学的总体控诉，但另一种解读方式是，在 web3 的淘金热中，区块链的创始人忘记了热爱某样东西意味着什么，因为它是一种伟大的产品，并且过度关注了消费者贪婪或财务绝望的可持续性。现在，加密市场明天可能会反弹，但它不会减少你只能为忠诚支付这么长时间的事实。

第四舱:凯文·罗斯

你好，安妮塔又来了。在本周的连锁反应播客中，卢卡斯和我谈到了投资者即将面临的加密冬天。公共股票总体上正在遭受打击，标准普尔 500 指数连续五天下跌，而比特币基地和罗宾汉等与加密相关的公司正承受着市场担忧的冲击。

加密货币的价格也在下跌。按市值计算，比特币是全球最大的加密货币，目前较去年 11 月的峰值下跌了 50%以上。过去几天，金价几次跌破 3 万美元，分析师表示，这标志着比特币的一个关键门槛——如果继续下跌，损失可能会继续增加。由比特币部分支持的 Terra UST 稳定币的惨败显然无助于局势。

但看多加密的人喜欢谈论几十年，而不是几天，而且往往对更广泛的市场中不存在的波动性有兴趣。这远不是比特币价格第一次暴跌，因此值得及时回顾一下，看看比特币在 2017 年最后一个主要加密冬天的表现。那年早些时候，比特币的价格达到了 2 万美元的峰值，但随着黑客攻击、监管和投资者的反复无常，比特币在 12 月下旬跌至 1.2 万美元以下。直到 2020 年末/2021 年初，金价才开始大幅升值，最终突破 30，000 美元大关，此后(大部分时间)一直保持在 30，000 美元之上。

这一次，OG 加密货币的情况可能会有所不同。现在持有比特币的散户投资者要多得多，只有时间才能证明他们是否有足够的资金度过这场风暴。此外，以太坊和索拉纳等新兴区块链已经在蚕食比特币的竞争优势。你可以在我的最新专题这里阅读更多关于困扰比特币的问题及其支持者正在做什么来帮助推动比特币。

添加了分析

随着监管的迫近，Terra 的 UST 崩盘将使加密行业的日子更加难过
在过去的一周里，随着一系列因素撼动了这个行业，稳定的消息成为了加密领域的主要话题。随着加密市场以悲观情绪做出回应，一个主要问题出现了:这一切对稳定密码的未来意味着什么？许多市场参与者都在思考未来的路会是什么样子。

鲨鱼池的凯文·奥利里谈论加密和他为什么支持 stablecoins
谈到 stablecoins，鲨鱼池的凯文·奥利里与 TechCrunch 坐下来分享他对许多加密相关话题的想法，如加密监管和他为什么支持 stablecoin。我们还讨论了进入该领域的机构公司，以及如果他决定这样做，他将创建的专注于加密的公司类型，等等。

比特币基地 NFT 市场开局惨淡
其他消息，比特币基地 NFT 公司三周前从今天开始推出测试模式，但仍未获得任何采用——即使在上周向公众开放之后。一位消息人士称，目前对它应该在哪里的预期与预期不符，而且不清楚它是否会如此。鉴于比特币基地加密交易所的规模，有人会认为它的 NFT 市场也会成功，但其他人认为这不太可能，而且它进入该领域的方式。
