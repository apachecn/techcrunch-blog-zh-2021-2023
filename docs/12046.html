谷歌称监控供应商零日锁定三星手机

原文：https://web.archive.org/web/https://techcrunch.com/2022/11/10/google-surveillance-samsung-spyware/

谷歌表示，有证据表明，一家商业监控供应商正在利用三星新款智能手机中发现的三个零日安全漏洞。

这些漏洞是在三星定制的软件中发现的，被一起用作漏洞链的一部分，以运行 Android 的三星手机为目标。这些连锁漏洞使得攻击者能够以根用户身份获得内核读写权限，并最终暴露设备的数据。

谷歌项目零安全研究员 Maddie Stone 在一篇博客文章中说，漏洞链针对的是运行特定内核版本的 Exynos 芯片的三星手机。三星手机主要在欧洲、中东和非洲销售 Exynos 芯片，这些地方很可能是监控目标的所在地。

斯通说，当时运行受影响内核的三星手机包括 T2 S10 T3 A50 和 A51。

这些缺陷被修补后，被一个恶意的 Android 应用程序利用，用户可能被诱骗从应用程序商店之外安装。恶意应用程序允许攻击者逃离旨在遏制其活动的应用程序沙箱，并访问设备操作系统的其余部分。斯通说，只获得了漏洞利用应用程序的一个组件，所以不知道最终的有效载荷是什么，即使这三个漏洞为其最终交付铺平了道路。

“这个链中的第一个漏洞，任意文件读写，是这个链的基础，使用了四次不同的时间，并且在每个步骤中至少使用了一次，”Stone 写道。“Android 设备中的 Java 组件往往不是安全研究人员最受欢迎的目标，尽管它运行在这样一个特权级别，”Stone 说。

谷歌拒绝透露商业监控供应商的名称，但表示这种利用遵循的模式类似于最近的设备感染，恶意 Android 应用程序被滥用来提供强大的民族国家间谍软件。

今年早些时候，安全研究人员发现了 Hermit，这是一款由 RCS 实验室开发的 Android 和 iOS 间谍软件，用于政府的定向攻击，已知受害者在意大利和哈萨克斯坦。“隐士”依靠诱骗目标从应用商店外部下载并安装恶意应用程序，如伪装的手机运营商辅助应用程序，然后悄悄地窃取受害者的联系人、录音、照片、视频和精确的位置数据。谷歌开始通知安卓用户，他们的设备已经被隐士攻破。监控厂商 Connexxa 也使用恶意侧装应用来针对 Android 和 iPhone 用户。

谷歌在 2020 年底向三星报告了这三个漏洞，三星在 2021 年 3 月向受影响的手机推出了补丁，但当时没有透露这些漏洞正在被积极利用。斯通说，三星已经承诺开始披露漏洞被积极利用的时间，继苹果和谷歌之后，它们也在安全更新中披露漏洞受到攻击的时间。

“对这一利用链的分析为我们提供了关于攻击者如何瞄准 Android 设备的新的重要见解，”Stone 补充道，并暗示进一步的研究可能会发现三星等 Android 设备制造商开发的定制软件中的新漏洞。

“这凸显了对制造商特定组件进行更多研究的必要性。它显示了我们应该在哪里做进一步的变异分析，”斯通说。
