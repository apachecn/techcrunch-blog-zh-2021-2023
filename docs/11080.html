微软通过设计师和图像创建者| TechCrunch 将 DALL-E 2 推向大众

原文：https://web.archive.org/web/https://techcrunch.com/2022/10/12/microsoft-brings-dall-e-2-to-the-masses-with-designer-and-image-creator/

微软正在对 OpenAI 的人工智能系统 DALL-E 2 进行重大投资，该系统可以通过文本生成图像，并将其引入第三方应用和服务。在本周的 Ignite 会议上，微软宣布将 DALL-E 2 与新发布的微软 Designer 应用程序和图像创建工具整合到 Bing 和微软 Edge 中。

随着近年来 DALL-E 2 和像稳定扩散这样的开源替代方案的出现，人工智能图像生成器已经大受欢迎。9 月，OpenAI 表示超过 150 万用户每天使用 DALL-E 2 积极创作超过 200 万张图像，其中包括艺术家、创意总监和作者。像 Stitch Fix、Nestlé和 Heinz 这样的品牌已经将 DALL-E 2 用于广告宣传和其他商业用途，而某些建筑公司已经使用 DALL-E 2 和类似的工具来构思新的建筑。

“微软和 OpenAI 自 2019 年以来一直密切合作，加速人工智能的突破。我们已经与 OpenAI 合作开发、测试和负责任地扩展最新的人工智能技术，”微软现代生活、搜索和设备 CVP Liat Ben-Zur 通过电子邮件告诉 TechCrunch。“微软是 OpenAI 的云计算服务独家提供商，也是 OpenAI 将新 AI 技术商业化的首选合作伙伴。我们已经开始通过 Azure OpenAI 服务和 GitHub Copilot 等程序来做到这一点，我们将继续探索利用人工智能和高级自然语言生成的解决方案。”

为了将 OpenAI 的技术带给更广泛的受众，微软推出了 Designer，这是一款类似 Canva 的网络应用，可以为演示文稿、海报、数字明信片、邀请函、图形等生成设计，并在社交媒体和其他渠道上分享。设计师——其公告在今年春夏反复泄露——利用用户创建的内容和 DALL-E 2 来构思设计，用下拉框和文本框来进一步定制和个性化。

在 Designer 中，用户可以从各种模板中进行选择，开始为 Instagram、LinkedIn 脸书广告和 Instagram Stories 等平台进行特定的定义维度的设计。预构建的模板可以从网上获得，形状、照片、图标和标题也可以添加到项目中。

微软 Edge 和 Bing 中的图像创建器。

“微软设计师由人工智能技术提供支持，包括 DALL-E 2，这意味着即时生成各种设计的能力，”Ben-Zur 继续说道。“[它]帮助你把你的想法变成现实。

微软表示，Designer 将在有限的预览期内保持免费——用户可以从今天开始注册。一旦 Designer 应用程序正式推出，它将包含在微软 365 个人和家庭订阅中，并为非订阅者免费提供“一些”功能，尽管微软没有详细说明。

DALL-E 2 支持的另一个新的微软开发的应用程序是 Image Creator，将在未来几周内登陆 Bing 和 Edge。顾名思义，Image Creator——通过 Bing 图像标签或 bing.com/create,或 Edge 侧边栏中的 Image Creator 图标访问——通过向 DALL-E 2 发送请求来生成带有文本提示的艺术，就像 OpenAI 仍在测试中的 DALL-E 2 服务的前端客户端一样。

输入一些东西的描述，任何额外的背景，如位置或活动，以及艺术风格，将从 Image Creator 产生一个图像。“Image Creator 将很快创建尚不存在的图像，只受你的想象力的限制，”Ben-Zur 补充说。

与 Designer 不同，Bing 和 Edge 中的 Image Creator 将完全免费使用，但微软担心潜在的滥用和误用，表示将采取“审慎的方法”推出这款应用。Image Creator 最初将只在特定地区的预览版中提供，微软表示，这将允许它在进一步扩展应用程序之前收集反馈。

微软设计师。

一些图像生成系统已经被用来创建令人反感的内容，如图片暴力和色情，非合意的名人假货。美国众议院议员安娜·g·埃舒(D-CA)最近给国家安全顾问(NSA)和科学技术政策办公室(Office of Science and Technology Policy)写了一封关键的信，其中她敦促 NSA 和 OSTP 解决“不安全的人工智能模型”的发布问题，这些模型“不会限制他们平台上的内容”

生成图像的人工智能还可以从用于训练它们的网络上的数百万张图像中发现偏见和毒性。OpenAI 本身在一篇学术论文中提到DALL-E 的开源实现可以被训练成产生刻板印象的联想，比如为“CEO”等术语生成穿着西装的白人路过的图像。

在回答关于 Designer 和 Image Creator 中缓解措施的问题时，微软指出，OpenAI 从用于训练 DALL-E 2 的数据集中删除了露骨的性和暴力内容。该公司还表示，它采取了自己的措施，包括部署过滤器来限制违反内容政策的图像的生成，对敏感主题进行额外的查询阻止，以及提供“更多样化”图像的技术。

用户必须同意使用条款和上述内容政策，开始使用他们的微软帐户的 Designer 和 Image Creator。如果用户请求微软自动过滤器认为不合适的图片，他们会收到警告。如果他们一再违反内容政策，他们将被禁止，但有机会上诉。

“对于像 DALL-E 2 这样的早期技术来说，承认这是新的很重要，我们希望它继续发展和改进，”Ben-Zur 说。“我们认真对待我们对负责任的人工智能的承诺……我们不会允许用户产生暴力内容，我们可能会扭曲人们的面部，不会显示用作输入的文本字符串。”

针对最近围绕人工智能图像生成系统出现的一些法律 问题，微软表示，用户将拥有“完全”使用权，可以将他们通过 Designer 和 Image Creator 创建的图像商业化。(在其他主机中，Getty Images 已经禁止上传和销售使用 DALL-E 2、Stable Diffusion 和类似工具生成的插图，理由是对包含版权图像的训练数据集的合理使用感到担忧。换句话说，该公司——采用类似于 OpenAI 的商业使用政策——不会声称用户提供、发布、输入或提交给应用程序的提示、字幕、创作或任何其他内容的所有权。

当被问及微软是否认为用于训练 DALL-E 2 的图像是合理使用时，Ben-Zur 拒绝回答。
