纳什筹集现金，帮助商家管理当地交货

原文：https://web.archive.org/web/https://techcrunch.com/2022/07/26/nash-raises-cash-to-help-merchants-manage-local-deliveries/amp/

图片来源: svetikd / Getty Images

疫情使美国的外卖标准化，但即使在健康危机爆发之前，仅餐馆的外卖销售额就以每年 7%到 8%的速度增长，大约是整个餐饮业销售额的两倍，根据麦肯锡的说法。问题是，从餐馆和商家的角度来看，送货可能是一场噩梦。除其他挑战外，企业通常依赖单一合作伙伴进行交付，这使得它们极易受到供需和服务可靠性波动的影响。

为了让送货变得不那么令人头疼，马哈茂德·古尔曼创建了纳什平台，这是一个聚集送货供应商的平台，允许企业选择那些在价格和可用性方面最有意义的供应商。Nash 插入来自 DoorDash、Lyft、优步和其他合作伙伴的第三方交付 API，为客户提供管理和向客户提供本地交付的软件。

“在过去的几年里，我和我的联合创始人看到，鉴于零售、送货和运输巨头所取得的成就，消费者已经开始期望以可承受的价格获得令人难以置信的快速、可靠的送货，”Ghulman 在一次电子邮件采访中告诉 TechCrunch。“然而，由于运营和维护最后一英里配送业务的复杂性和高成本，中小型企业很难满足消费者的需求。因此，我们对构建一个软件和物流解决方案非常感兴趣，该解决方案将允许任何企业通过构建定制工作流和利用大型交付提供商网络，轻松激活可靠的本地交付。”

图片来源:纳什

古尔曼在高中的一次科学博览会上遇到了纳什的另一位联合创始人阿齐兹·阿尔古奈姆，两人一起就读于麻省理工学院。Alghunaim 之前是 Palantir 的工程师，并帮助创办了 Y Combinator 支持的初创公司 Tarjimly，旨在减少难民的语言障碍，同时提供支持服务。

Ghulman 和 Alghunaim 在 2021 年初共同创立了 Nash，公司参与了 Y Combinator 的 S21 批次。显然，纳什给投资者留下了深刻印象，他今天完成了由安德森·霍洛维茨领投的 2000 万美元首轮融资，Y Combinator、Rackhouse Venture Capital 和“技术和供应链领域的领导者”提供了支持。

“今天新宣布的 2000 万美元 A 轮资本使我们的总资金达到 2780 万美元，因为它紧随我们之前未宣布的去年年底的 780 万美元种子轮，安德里森·霍洛维茨也领导了这一轮，”古尔曼说。“这笔资金将用于加倍招聘工程、运营、销售和其他关键业务部门的员工。我们计划在年底前将 25 名员工的人数增加 2 到 3 倍以上，以适应我们的爆炸性增长。”

使用 Nash，企业可以手动选择交付提供商，也可以让平台自动选择。无论哪种方式，他们都可以获得价格和时间估计，以及关于送货人的信息，包括他们的联系信息和位置。

Ghulman 声称，成千上万的商家可以通过 Nash 的合作伙伴关系访问该平台，获得单独和批量(即多次提货)交付路线，覆盖近 1000 个城市中高达 94%的美国人口。

为了优化路线，纳什使用人工智能和机器学习来预测快递员的准时交付表现，给定的变量包括取件时间、卸货时间、包裹价值、距离和驾驶时间。 Ghulman 说，Nash 支持一系列不同的交付选项，包括计划内和同一时间交付，以及店对店交付和客户退货。

“我们为交付所做的就是 Stripe 为支付所做的:我们已经建立了一个简单的 API 和工作流生成器，”Ghulman 继续说道。“企业可以利用我们的 API 平台构建定制的交付工作流程，或者通过我们与行业领先的市场、聚合商和销售点提供商的技术合作关系直接访问 Nash。这种方法使得团队很容易减少维护高性能交付操作所需的技术开销。”

图片来源:纳什

一直处于资本密集型的按需交付市场目前并不是特别健康，正如 Gopuff 等交付初创公司的大规模裁员所证明的那样。古尔曼拒绝提供纳什营收和客户群的确切数字，但他向我保证，纳什没有受到更广泛的经济放缓的影响，并且“有能力”抵御潜在的逆风。"

“疫情带来了各行各业对本地配送需求的惊人增长。许多公司跟不上。有些甚至被迫关闭。由于劳动力减少，启动或运营快递业务比以往更具挑战性，”Ghulman 说。“通过消除与提供可靠交付体验相关的技术、物流和运营开销，Nash 帮助数百家企业获得了新客户和收入流。”

至于下一步，纳什计划进一步扩展到垂直行业，如零售，汽车零部件，宠物用品，餐饮和餐包，洗衣，鲜花，印刷和框架，制药和小包裹。它还将目光投向了新的市场，包括加拿大和英国，纳什计划在今年年底前推出这些市场。
