Helm.ai 斥资 3100 万美元扩大其“无人监管”自动驾驶软件 TechCrunch 的规模

原文：https://web.archive.org/web/https://techcrunch.com/2022/12/19/helm-ai-snags-31m-to-scale-its-unsupervised-autonomous-driving-software/

尽管宏观经济逆风几乎关闭了风险资本的龙头，并导致了进一步的整合，但自动驾驶汽车行业仍有一些亮点。

Helm.ai 是一家开发高级驾驶辅助系统、自动驾驶和机器人软件的初创公司，就是其中之一。

总部位于加利福尼亚州门洛帕克的初创公司最近在弗里曼集团(Freeman Group)牵头的 C 轮融资中筹集了 3100 万美元，仅在一年前，它获得了 2600 万美元的风险投资。最新一轮投资包括 ACVC 伙伴公司、Amplo 和战略投资者本田汽车公司、固特异风险投资公司和 Sungwoo Hitech，这一轮投资使 Helm.ai 的估值达到 4.31 亿美元。

弗里曼集团创始人布兰登·弗里曼(Brandon Freeman)将加入 Helm.ai 董事会，作为此次融资的一部分。迄今为止，该公司已经筹集了 7800 万美元。

像许多其他自动驾驶汽车初创公司一样，Helm.ai 推出了一种新的方法来推动这项技术向前发展。Helm.ai 的联合创始人 Tudor Achim 和 Vlad Voroninski 将目标对准了软件，而不是传感器或计算。

Helm.ai 开发了可以像人类一样理解传感器数据的软件——这一目标与该领域的其他人没有什么不同。它的方法是值得注意的部分。自动驾驶汽车开发者通常依赖模拟和道路测试的结合，以及大量由人类标注的数据集，来训练和改进自动驾驶汽车的所谓“大脑”。

Helm.ai 表示，它已经开发出可以跳过这些步骤的软件，这加快了时间表并降低了成本；较低的成本也使它特别适用于高级驾驶辅助系统。这家成立六年的初创公司使用无监督学习方法开发软件，可以在不需要大规模车队数据、模拟或注释的情况下训练神经网络。该软件还与车辆中使用的任何计算和传感器无关，使 Helm.ai 能够面向不同的客户群。

根据 Voroninski 的说法，Helm.ai 向汽车行业的各种原始设备制造商和一级供应商出售其软件，以帮助他们“通过高端 ADAS 和 L4 解决方案实现软件差异化”。

“从战略上来说，我们已经知道我们的走向市场战略将集中在高端 ADAS 上，或者在几年内；我们的战略丝毫没有因为最近的事件而改变，”他说，指的是 AV 行业的整合。“几年来，我基本上一直在预测，由于过时的技术方法和低劣的商业模式，绝大多数自动驾驶公司将无法进入市场。所以这对我来说一点也不意外。由于所有错误的宣传，自动驾驶市场在过去几年里通常都不是很有效率。”

Helm.ai 吸引了一些客户，尽管沃罗宁斯基说，由于保密协议，他不能说出其他客户的名字。Helm.ai 此前曾披露本田是客户。这位数学家和网络安全机器学习初创公司 Sift Security 的前首席科学家确实表示，他在过去两年里一直专注于将这项技术商业化并确保合作伙伴关系。

他说，最近的资金将用于增加更多的员工到 50 人的劳动力，R&D 和建立那些商业伙伴关系。
