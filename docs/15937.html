微软新的 Bing AI 聊天机器人在其 Edge 网络浏览器| TechCrunch 的稳定版本中发布

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/14/microsofts-new-bing-ai-chatbot-arrives-in-the-stable-version-of-its-edge-web-browser/

除了今天发布 OpenAI 的 GPT-4,现在确认是运行 Bing 的 GPT 型号，微软还宣布其 Edge 网络浏览器的稳定版本现在将在侧边栏中包括新的 Bing AI 聊天机器人。这项功能最初是在二月份微软的人工智能新闻发布会上介绍的，但之前只作为开发者预览版提供，没有公开发布。

随着今天 GPT-4 的正式发布，微软在其微软 Edge 浏览器的稳定版本中推出了这一功能，它称之为“Edge Copilot”。

此次更新重新构想了侧边栏的概念，之前侧边栏拥有 Edge 的“发现”功能，为用户提供他们正在访问的页面的上下文。现在，新的侧边栏将提供一个人工智能聊天机器人。

当用户想要与侧边栏和人工智能功能进行交互时，他们只需将鼠标悬停在工具栏中的 Bing 图标上即可打开侧边栏。不使用时，侧边栏可以自动隐藏。

图片致谢:Edge 浏览器截图

微软表示，当开放时，用户可以利用 Edge Copilot，这种人工智能功能能够提供“基于网页上下文和用户目标的智能建议和见解”。

正如该公司在活动中解释的那样，人工智能聊天机器人在推出时有两个主要功能——聊天和撰写。总之，这些可以帮助用户做一些事情，如总结冗长的网页内容，运行比较，甚至在某些情况下创建内容。

例如，微软的一个演示展示了人工智能总结一家公司的财务报表——但不幸的是，它得到了错误的数字，这是后来发现的。在另一个项目中，一名开发人员要求 AI 在研究堆栈溢出技巧的同时编写一段代码。人工智能能够将 Python 代码转换为 Rust。微软甚至演示了人工智能在给出一些提示后编写 LinkedIn 帖子。

今天，该公司表示，人工智能副驾驶也可以用来帮助用户写更好的电子邮件，更快地搜索网络和学习新技能。

人工智能的生产力角度并没有在竞争中迷失。今天，谷歌公布了如何将人工智能整合到自己的生产力解决方案 Workspace 中。其计划包括将生成性人工智能融入 Workspace 的每个部分，包括帮助用户在 Gmail 中写电子邮件，在 Docs 中编写和编辑文档，在工作表中生成公式，在 Meet 中捕捉笔记，以及在幻灯片中创建文本、图像、音频和视频。预计微软将在本周晚些时候详细说明对谷歌举措的回应。

虽然侧边栏是最受期待的新边缘功能，但它可能会被 it 管理员锁定。微软 notes 管理员将能够控制用户是否可以访问 Edge 的侧边栏——这是一个重要的考虑因素，因为一些公司，包括沃尔玛和亚马逊现在已经制定了政策，禁止与 ChatGPT 和其他人工智能机器人分享公司的机密信息。

然而，AI 侧边栏并不是这个版本中唯一的新特性。

该公司还表示，它将在 Edge 企业新标签页的微软 365 标签中推出一种新体验。这将包括一个更大版本的 Microsoft Feed，其中包括更多的生产力内容，并将重要电子邮件、最近的 SharePoint 网站、即将举行的活动和待办事项的卡片移到选项卡的右侧。

此外，该浏览器将提供各种安全模式改进，并将支持一个新的策略，该策略控制在退出浏览器应用程序时是否删除用户的浏览历史记录。

微软表示，其 Edge 网络浏览器的稳定版本将在一天或几天内逐步推出。该浏览器适用于 Windows 和 Mac 平台。
