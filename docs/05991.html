研究人员担心马斯克可能会切断推特数据

原文：https://web.archive.org/web/https://techcrunch.com/2022/05/01/researchers-fear-what-a-musk-acquisition-might-mean-for-twitter-research-data/

研究人员担心马斯克的收购可能对 Twitter 的研究数据意味着什么

关于埃隆·马斯克(Elon Musk)收购 Twitter(T3)的报道很多，尽管得到了摩根士丹利的大力支持和 Twitter 董事会的批准，但这项努力目前在仍不确定。

报道和权威人士一直关注拟议收购的安全影响，以及马斯克对内容节制的潜在方法，以及在一个相关主题上，他对“言论自由概念的理解但这笔交易的另一个重要方面受到的关注要少得多:在马斯克的统治下，Twitter 的研究数据访问政策可能会发生什么变化。

Twitter 与研究人员的关系并不总是那么融洽。然而，近年来，当竞争对手采取了与 相反的步骤时，社交网络在提供其档案访问方面取得了长足的进步。2021 年 1 月，Twitter 声称学术研究人员是使用其 API 的最大群体之一。

一些研究人员担心马斯克不会分享同样的开放数据访问的承诺，特别是考虑到他过去对以不讨好的方式描绘他的风险投资(包括特斯拉)的报道表现出的尖酸刻薄。

迄今为止，Twitter 在为研究人员提供数据方面是主流平台中独一无二的。大卫·g·兰德

2018 年，马斯克承诺——但最终没有——建立一个网站，对文章和记者的“核心真相”进行评级，以回应涉及特斯拉汽车、特斯拉劳工问题以及他与华尔街关系的撞车报道。

康奈尔理工大学信息科学教授莫尔·纳曼(Mor Naaman)设想，未来马斯克会对揭露 Twitter“挑战和不足”的研究人员变得充满敌意。

自 2009 年以来一直研究 Twitter 数据的纳曼通过电子邮件告诉 TechCrunch，“我对 Twitter 在马斯克的领导下将继续努力成为一家私营公司感到悲观。”“我不相信像我们对(前总统唐纳德·特朗普的)停止偷窃运动所做的这样的研究——以及我们从推特上收集并向其他研究人员提供的数据，自去年以来在 12 份不同的论文中使用——会在马斯克的领导下发生。第二，我无法想象审查公司系统的道德和偏见的内部团队会继续很好地运作，更不用说公开发布他们的发现了。

“如果他们继续出版，这些出版物将很难克服已经存在的对发表自己研究论文的平台对企业友好的偏见的怀疑。”

在其他承诺中，马斯克曾表示，他计划在 Twitter 上“击败垃圾邮件机器人”——似乎暗指鹦鹉学舌式传播错误信息和延续骗局的恶意账户。普林斯顿的博士后研究员 Orestis Papakyriakopoulos 通过电子邮件向 TechCrunch 指出，并非所有的机器人都是有害的。
