Pearpop 推出动态 NFT，随着社交媒体帖子的广泛传播，动态 NFT 获得了价值 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/02/10/pearpop-pearproof-nfts-crypto-social-media-creators/

社交媒体合作市场 Pearpop 正在迎头痛击加密技术。

有了链烟者、亚历克西斯·奥哈尼安、马克·库班和史努比·道格等大牌支持者的，Pearpop 推出了 Pearproof ，这是一款 web3 应用，允许创作者制作他们社交媒体帖子的 NFT。这个概念不一定是新的——Cent帮助杰克·多西以相当于 290 万美元的价格在联邦理工学院出售了他的第一条推文的 NFTs 但 Pearproof 的 NFTs 使用一种专有算法，随着帖子本身获得更多的社交参与，这种算法允许资产增值。

Pearpop 和 Pearproof 的首席产品官李双雷说:“我们希望创建一个平台或一种格式，不仅能让我们建立所有权，还能让他们的 web2 元数据真正联系起来，让 NFT 充满活力。”。

这些 NFT 从 Pearproof 开发的分层系统的“乙烯”层开始。随着它越来越受欢迎，NFT 可以“升级”到银，金，白金和其他水平。创造者可以决定与这些级别相关联的奖励——例如，也许 NFT 达到“银”意味着所有者将获得私人不和。

“这是关于编程的一种方式，解锁访问创作者或私人社区，”李告诉 TechCrunch。作为与创作者的成功相关的资产的所有者，粉丝几乎像创作者的投资者一样运作——他们希望创作者成功，因为他们在创作者的成功中有直接的经济利益。

这些功能并不适合所有人——首先，这些工具也是在这样的假设下运行的，即喜欢、评论和分享是社交媒体帖子的价值所在。并非所有创作者都是为了喜欢和分享——一些创作者努力与一小部分追随者建立更密切的关系，而不是寻求纯粹的病毒式名气。Pearproof 的成功也很大程度上依赖于创作者对密码的理解，以及他们的观众的购买意愿。随着加密诈骗和 T2 黑客的大量出现，普通消费者会相信社交 NFT 的长期价值吗？虽然这些功能可能有助于创作者开发新的赚钱方式，但粉丝可能会倾向于将其视为抢钱，即使创作者是出于好意。

该项目将利用索拉纳区块链，Pearproof 选择它是因为其交易成本低，对环境的影响较小。只有创作者自己才能将自己的社交帖子铸造成 NFT——Lee 告诉 TechCrunch，将会有“大量验证”来试图防止欺诈。NFT 卖出去，创作者拿 90%，Pearproof 拿 10%。在随后的每一笔交易中，创作者都会获得 10%的版税。

Pearpop 的商业模式依赖于外部社交平台(“我们以创作者为中心，平台不可知，”Lee 说)。除了蓬勃发展的 NFT 业务，Pearpop 自 2020 年推出以来，通过将粉丝与社会明星联系起来，从索默·雷和姚一奇·哈克这样的有影响力的人到麦当娜和莉莉·纳斯 x 这样的偶像，Pearpop 已经取得了增长。这些名人可以在抖音出售与他们“二重唱”的机会(对于像 Snoop Dogg 这样的人来说，二重唱可能需要几千美元)，或者他们可以通过 pear pop 运行“挑战”，邀请人们使用抖音上的特定声音，以便有机会赢得现金。麦当娜的团队最近与 Pearpop 合作，为推广她的歌曲《冰雪奇缘》的挑战赛分配了 2 万美元。

但是，像抖音、Instagram、Twitter、Spotify 和 YouTube 这样的平台会因为 Pearproof 把媒体变成非功能性传播而变得酷吗？虽然 Pearproof 没有直接询问这些平台，但 Lee 认为这不是问题。

“NFT 将代表网络 2 上的媒体，而不是媒体本身，”李开复说。因此，YouTube 视频的 NFT 可能只显示几秒钟，而不是整个视频。“你购买的是一个为你打开社区大门的代理，但你也在帮助推动特定视频的流量。这就是我们的设想。”

另一个潜在的问题是，如果一家竞争的初创公司采用了同样的想法，那么一条推文的 NFT 同时出现在 Cent 和 Pearproof 上意味着什么？但 Lee 认为 Pearproof 的专有算法将资产与社会参与动态地联系起来，使他们的 NFT 与众不同。尽管如此，对于 NFT 的所有者来说，不知道如果竞争对手抢了风头，Pearproof 会受到怎样的影响可能是有风险的(就像任何投资一样……练习小心！).这些 NFT 不仅是对创造者成功的赌注，也是对 Pearproof 成功的赌注。

在发布时，Pearproof 与 TikToker Josh Richards 和 deadmau5 合作，后者将于 2 月 15 日以 Pearproof NFT 的身份独家发布音乐。
