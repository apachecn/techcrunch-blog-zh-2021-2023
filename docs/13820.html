阿里巴巴首席执行官将在重大服务器故障后监管云部门 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/12/29/alibaba-reshuffle-2022/amp/

图片来源:张杰夫/阿里巴巴集团

当阿里巴巴前首席执行官马云将火炬传递给丹尼尔·张时，他建立了一个系统定期轮换高管以确保公司在快速变化的互联网领域始终保持敏捷。今年的洗牌时间到了——这家电子商务和云计算巨头在周四宣布了几项重大的人事变动。

最突出的决定发生在阿里云，根据 Gartner 的数据，阿里云是第三大基础设施即服务 (IaaS)公共云服务，但没有进入按照科纳仕公司或 Synergy 排名的前三名云提供商，可能是由于不同的计算方法。阿里云智能(Alibaba Cloud Intelligence)前总裁张志勇(Jeff Zhang)将离职，阿里巴巴首席执行官张勇(Daniel Zhang，无亲属关系)将接任代理总裁。

重组的时间引发了猜测。就在不到两周前，阿里云的香港服务器遭遇了严重的宕机，导致该地区许多服务关闭，包括主要的加密交易所 OKX。对于一些客户来说，系统故障持续了长达一天的时间，这使得该事件成为中国云提供商近年来最大的事件之一。

杰夫没有离开，而是将注意力集中在阿里巴巴的基础研究机构达摩院。作为阿里巴巴合伙企业、的 29 名成员之一，张自 20 年前加入以来一直扮演着举足轻重的角色。首先，他在为全球最大的在线市场之一淘宝设计系统基础设施方面发挥了重要作用。

在达摩，他将继续负责物联网计划和阿里巴巴的专有芯片开发团队 T-Head，随着中国对美国技术制裁的升级，该团队变得更加重要

“在过去的四年里，杰夫带领阿里云智能团队在技术创新和行业影响力方面取得了出色的成绩，”丹尼尔在给员工的内部电子邮件中表示。

除了执掌云业务，这位首席执行官还将管理阿里巴巴的企业通信应用程序 DingTalk，该应用程序与云部门密切合作。正如我们在 2020 年写的:

在本周的年度峰会上，阿里云重申了其最新战略，即将云融入类似 Slack 的工作协作应用。

该口号表明了阿里巴巴希望 DingTalk 扮演的战略角色:基于阿里云的操作系统，阿里云是仅次于亚马逊和微软的全球第三大基础设施服务。正如阿里云总裁张建锋此前在一次采访中所说，这种关系与微软 365 和 Azure 的关系类似。

很难想象丹尼尔会长期致力于云业务，这可能是他获得代理总裁头衔的原因。作为占 Q2 总收入 10%的阿里巴巴第二大业务，云计算部门肯定会竭尽全力寻找下一个合适的老板。

无论继任者是谁，都不会是一个小壮举。由于失去了重要的海外收入来源，云部门最近几个季度增长乏力。阿里云曾经是许多中国互联网公司在海外扩张的首选解决方案，但随着地缘政治紧张局势的加剧，他们开始转向 AWS 等外国提供商，以断绝与中国的关系。一个恰当的例子是，据报道，在抖音将其数据移出服务后，阿里云受到了重创。

正如该公司在 T4 11 月份收益电话会议上所说的:

“来自互联网行业客户的收入下降了约 18%，这主要是由于顶级互联网客户的收入下降，这些客户已逐渐停止将我们的海外云服务用于其国际业务，在线教育客户，以及中国互联网行业其他客户的需求疲软。”

在国内，阿里云面临着对手腾讯的竞争，后者在游戏领域占据重要地位。它还与华为和天翼竞争，天翼是国有电信巨头中国电信的云分支，这两家公司在支持政府和公共基础设施方面都处于领先地位。
