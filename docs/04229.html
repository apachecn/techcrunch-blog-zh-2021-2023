DeepScribe 的 Akilesh Bapu 说，在筹集首轮融资时，利用早期投资者

对于人工智能医疗转录平台 DeepScribe 的首席执行官兼联合创始人 Akilesh Bapu 来说，给潜在投资者一个严格的截止日期，同时依靠早期投资者的支持和指导，这一切都不同。

巴普在回顾 2021 年夏天时说:“作为一家公司，我们处于这种轨迹，我们的产品表面上符合市场。”“我们已经证明了我们的产品。我们在平台上有大约 200 个活跃客户……我们很高兴将 DeepScribe 带给更多的客户，并在那里寻找我们的最佳合作伙伴——不仅是短期的，也是长期的。我们有一个长远的愿景…我们想要一个认同这一愿景的合作伙伴。”

最终，该公司完成了由 Index Ventures 合伙人 Nina Achadjian 领导的 3000 万美元的首轮融资，两人在最新一集的 TechCrunch Live 中进行了讨论，这是一个以企业家、开发者和投资者为特色的每周节目。下面是整个剧集，以及 DeepScribe 的系列 A 推介资料的一部分。

如果不是因为 Bapu 和他的团队为这轮融资设定了最后期限，他说 DeepScribe 可能不会与 Index 合作——当 Achadjian 阅读他们的推介并试图将会议推迟到下周时，她正在度假，但 Bapu 说这个过程进展很快。他们第二天见面了。

后来，Achadjian 被卖掉了。“当我走出会议室时，我立即去找我的一个合伙人，就像是，‘终于，我找到了遵循正确方法的公司。’”她说。

她补充说，这对 DeepScribe 来说是一个关键的胜利，因为让潜在投资者充满热情并掌握一些要点是必不可少的，包括团队和市场。

准备尽职调查

由于 Index 对 DeepScribe 感兴趣，该公司开始进行尽职调查。

Achadjian 说，创始人可以通过预测问题来加快这个过程，特别是关于市场规模和竞争格局的问题。公司还可以向投资公司提供客户电话记录的摘要。

Achadjian 说:“然后我们列出一系列我们想深入研究的关键问题。“商业模式是什么？你是如何衡量的？参考文献。我打电话给阿基利什在伯克利的一位教授。我们做了大量的客户电话，并检查了企业家的参考资料。然后，老实说，我们喜欢花时间和团队在一起，在不同的环境中看到他们。”
