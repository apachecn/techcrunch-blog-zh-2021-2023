风投关注 Polygon 的投资——TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/12/05/vcs-eye-investment-in-polygon/amp/

三位知情人士告诉我，包括红杉资本(Sequoia Capital India)和 Steadview Capital 在内的许多投资者正在与 back Polygon 进行谈判，后者运营一个框架，通过购买令牌的方式，建设和连接兼容以太坊的区块链网络。

消息人士称，投资者希望购买价值 5000 万至 1.5 亿美元的代币，由于谈判是私人的，因此要求匿名。与这些代币交易一样，投资者可以以轻微的折扣购买硬币。(近一个月 MATIC 均价 8 折，从我听到的情况来看。)

商议仍在进行中，因此条款可能会有所变化。上周早些时候没有人发表评论。

Polygon，以前称为 Matic，已经成为最流行的第 2 层解决方案之一。该公司的市值已超过 140 亿美元，每天处理超过 750 万笔交易，并允许数以千计的分散式应用程序继续使用以太坊作为结算层，但避免了日益昂贵的天然气费用。

Aave、Sushi Swap 和 Curve Finance 是部署在 Polygon 上的一些最大的蓝筹项目，Polygon 已经积累了最大的开发者生态系统之一(即使与一些一级区块链相比)。

图像信用:多边形

一项投资将标志着投资者对总部位于印度的 Polygon 的看法发生转变，直到最近几年，该公司一直难以获得南亚市场最知名风险公司的支持。(值得注意的是，直到几个季度前，印度的大多数风投也没有积极跟踪 web3 领域。)此外，据两位知情人士透露，Polygon 不得不面对至少一次情况，即一些早期投资者在熊市周期中要求返还资金。

该公司向其中一些投资者返还了资金，并幸存了下来。“这是 Polygon 团队的主题之一。他们的毅力更上一层楼，”一名前员工表示。

Polygon 今年获得了企业家和投资者马克·库班的支持，它是几十个侧链和累积网络之一，希望以太坊将继续保持其主导地位，即使 Polkadot 和 Multicoin Capital 和 A16z 支持的 Solana 等少数其他第一层项目正在试图吸引新生但快速增长的开发者生态系统。

在今年早些时候的无银行播客上，Polygon 的联合创始人 Sandeep Nailwal(如上图)说，今天的 web3 开发者生态系统以以太坊为中心，他希望网络效应不会消失。在同一个播客中，Nailwal 和 Polygon 的另一位联合创始人 Mihailo Bjelic 表示，Polygon 正在不断扩大其产品，以建立区块链基础设施。
