Brex 刚刚以 123 亿美元的估值签署了一份 3 亿美元的投资意向书

原文：https://web.archive.org/web/http://techcrunch.com/2021/10/21/brex-raises-300m-at-a-12-3b-valuation/

据知情人士透露，快速增长的 fin techBrex已经筹集了 3 亿美元的资金，使其成为十玉米公司，仅在六个月前，该公司的估值为 74 亿美元。

由于交易尚未公开(尽管条款清单已经签署)，不愿透露姓名的消息人士表示，企业支出初创公司 Brex 目前的估值为 123 亿美元。据称，Greenoaks 是此次投资的主导者，据报道，投资者还包括在看到该公司的“实力”后希望获得更多股份的 return backers。据消息人士称，Brex 今年的收入有望翻番。

Brex 和 Greenoaks 都没有回应置评请求。

总部位于旧金山的 Brex 于 2017 年由 Pedro Franceschi 和 Henrique Dubugras(他们现在都是 25 岁左右)创立，在老虎全球管理公司(Tiger Global Management)领导的 a 轮 4.25 亿美元的 D 轮 融资后，今年 4 月的估值为 74 亿美元。据 Crunchbase 数据 显示，该公司已经通过债务和股权融资筹集了 12 亿美元。随着最近一次注资，这一数字攀升至 15 亿美元。

考虑到它相对年轻的年龄，Brex 现在是十粒玉米的事实有点不寻常。如上所述，它现在有望在 2021 年实现收入翻番，尽管我们手头没有任何硬数字。该公司在上一次融资时告诉 TechCrunch，它“每个月都在招募数千名新的科技和非科技客户。”Brex 当时还表示，它在 2021 年第一季度的“客户总数”增长了 80%，“每月客户总数增加了 5 倍。”

在过去的一年里，企业支出领域大幅升温。Spend management 初创公司 Ramp 在 8 月份宣布，它已经在 C 轮融资中筹集了 3 亿美元，对该公司的估值为 39 亿美元。Brex 专注于早期创业公司和更成熟的企业，尤其是中端市场类别的企业。Ramp 倾向于为更大、更成熟的公司服务。在融资时，Ramp 告诉 TechCrunch，自今年年初以来，其收入和交易量激增了 1000%。

上周，TripActions 透露，疫情转向帮助企业解决企业开支，帮助提高了其收入， 将其估值提升至 75 亿美元。因此，它现在也出人意料地成为企业支出竞赛中的一个竞争者。首席执行官 Ariel Cohen 告诉 TechCrunch，他认为 TripActions 与 Brex 和 Ramp 不同，因为这两家初创公司“与旅行截然不同”，因此更专注于中小企业，而 TripActions 则更专注于企业公司。有趣的是，Greenoaks 也在该公司最近的 2.75 亿美元融资中领投。

Brex 也在继续改进它的模型。今天早些时候，这家初创公司宣布了新的“ Brex API ”该公司表示，所有 Brex 客户都可以免费使用新的开放 API，旨在让他们“在可定制的界面中无缝管理财务信息”对于没有内部开发人员的客户，Brex 还宣布了与 Zapier 的合作伙伴关系，这将允许产品之间的工作流程自动化，“而无需编写任何代码”

该公司补充说，有了 Brex API，开发人员可以构建为他们公司的个人需求定制的工作流。

今年早些时候，该公司宣布推出一项新服务 ，名为 Brex Premium ，每月费用 49 美元。

“我们现在的付费用户数量绝对超出了我们的预期，”杜布格拉斯在 8 月份告诉我。

2 月，Brex 被最新的 fintech 申请银行执照 。

该公司销售专为初创公司定制的信用卡，目前由移民银行担任发行人，该公司已向联邦存款保险公司(FDIC)和犹他州金融机构部(UDFI)提交了设立 Brex 银行的申请。

但在 8 月，该公司表示将 自愿撤回其银行章程和联邦存款保险申请，以努力“修改和加强”其申请，然后在日后重新提交。

同样在 8 月，Brex 以 5000 万美元收购了成立一年的 Weav， 一家为商业平台开发通用 API 的开发商，这是其首次重大收购。该公司表示，此举旨在让使用其平台的企业能够“更快、更准确地”获得金融服务和新产品。
