看看神经果蝇，迄今为止最好的果蝇模拟器

原文：https://web.archive.org/web/https://techcrunch.com/2022/05/18/behold-neuromechfly-the-best-fruit-fly-simulator-yet/

果蝇，普通的果蝇，在某些方面是一种简单的生物。但在另一些情况下，它是如此复杂，就像任何形式的生命一样，我们对它的了解还只是皮毛。研究人员通过创建迄今为止最准确的数字双胞胎，对 D. melanogaster 迈出了重要一步——至少在它如何移动以及在某种程度上为什么移动方面。

EPFL 大学的研究人员称他们的新模型为“神经苍蝇”，这是一个“形态逼真的生物力学模型”，基于对真实苍蝇的仔细扫描和近距离观察。结果是一个 3D 模型和运动系统，当受到提示时，它会像真正的苍蝇一样四处走动或对某些基本刺激做出反应。

需要说明的是，这并不是一个完整的逐细胞模拟，在过去的几年里，我们已经看到在小得多的微生物上取得了一些进展。它不模拟饥饿、视觉或任何复杂的行为——甚至不模拟它如何飞行，只模拟它如何在表面行走和梳理自己。

你会问，这有什么难的？嗯，这是一件事近似这种类型的运动或行为，并作出一个小的三维飞行，移动或多或少像一个真实的。在物理模拟环境中精确地做到这一点是另一回事，包括生物精确的外骨骼、肌肉和控制它们的类似于苍蝇的神经网络。

为了制作这个非常精确的模型，他们从一只苍蝇的 CT 扫描开始，以便创建形态逼真的 3D 网格。然后，他们记录了一只苍蝇在非常小心控制的环境下行走，并追踪其精确的腿部运动。 EPFL 的研究人员然后需要准确地模拟这些运动如何与物理模拟的“关节连接的身体部分，如头、腿、翅膀、腹部、长鼻、触角、笼头”相对应，后者是一种在飞行中有所帮助的运动传感器官。

图片来源:帕万·拉姆迪亚(EPFL)

他们表明，这种方法的工作原理是将观察到的苍蝇的精确运动带入模拟环境，并用模拟的苍蝇重放它们——真实的运动正确地映射到模型上。然后，他们证明他们可以在此基础上创造新的步态和动作，让苍蝇跑得比他们观察到的更快或更稳定。

图片来源:帕万·拉姆迪亚(EPFL)

确切地说，并不是说他们在改进自然。他们只是展示了对苍蝇运动的模拟延伸到了其他更极端的例子。他们的模型甚至对虚拟投射物也很健壮…在一定程度上，正如你在上面的动画中看到的。

“这些案例研究增强了我们对模型的信心。但是我们最感兴趣的是当模拟无法复制动物行为时，指出改进模型的方法，”EPFL 的 Pavan Ramdya 说，他领导着建造模拟器(和其他与黑腹果蝇相关的模型)的小组。看到他们的模拟哪里出了问题，就知道哪里还有工作要做。

“神经蝇可以增加我们对行为如何从复杂的神经机制系统和它们的物理环境之间的相互作用中产生的理解，”上周发表在自然方法上的论文摘要写道。通过更好地理解苍蝇的运动方式和原因，我们也可以更好地理解其背后的系统，从而在其他领域产生见解(果蝇是最常用的实验动物之一)。当然，如果我们出于某种原因想要创造一只人造苍蝇，我们肯定会首先想知道它是如何工作的。

虽然 NeuroMechFly 在某种程度上是数字模拟生命领域的一个巨大进步，但它仍然(正如其创造者将是第一个承认的那样)令人难以置信地有限，只专注于特定的物理过程，而不是使果蝇成为果蝇的微小身体和心灵的许多其他方面。你可以在 GitHub 或 Code Ocean 查看代码，也可以投稿。
