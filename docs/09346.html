视频网站在 YouTube 和 Twitch 上赚了数百万

原文：https://web.archive.org/web/https://techcrunch.com/2022/08/20/vtubers-are-making-millions-on-youtube-and-twitch/

这是全球最大的网络名人聚会。当我在 VidCon 的一家酒店大厅等待会见 Twitch streamer Code Miko 时，我发现了一只 Instagram 上著名的哈士奇，它是网飞“The Circle”粉丝最喜欢的参赛者，也是一位有争议的美容博主。但是当一个时髦的韩裔美国女人走近我时，我意识到我有点期待在我面前看到一个 3D 的超现实动画，而不是一个真实的人。也许这是大规模在线视频会议第三天近乎幻觉般的疲惫，但与回声酒店门厅中的许多社交媒体明星不同，像 Code Miko 这样的 VTubers 有时无法被人认出来。

“VTuber”是一项起源于日本的运动，意思是“虚拟的 YouTuber”，但这种文化也在 Twitch 等其他流媒体网站上流行，Code Miko 在那里有近百万粉丝。为了建立他们的虚拟角色，streamers 使用动作捕捉(甚至只是 AR 面部追踪)技术来实现虚拟化身，并围绕角色编织背景故事和神话。

“我认为成为另一个角色真的很有趣，”流光告诉 TechCrunch。“我只是觉得我有这个愿景。我想控制一个虚拟角色，让观众能够在直播中与她互动。我是《预备玩家一号》的忠实粉丝，所以当我觉得我可以做其中的一小部分时，我真的很兴奋。”

例如，Code Miko 这个角色是一个 NPC(不可玩的角色)，她梦想出演一款大型视频游戏，但她太浮华了，所以她转而求助于流媒体。粉丝们称化身背后的真人为“技术人员”，但她的名字是尤娜。由于尤娜在被疫情解雇之前是一名虚拟现实动画师，并创造了 Code Miko——这是她现在的全职工作——所以她的化身远比大多数虚拟人物更现实。此外，大多数 VTubers 从来不敢亲自见记者，更不用说在网上露面了。但尤娜有时会露脸，让观众一窥她的 mocap 技术的幕后。

虚拟化身通常类似于动漫角色，因为这种类型首先出现在日本。粉丝们对谁是第一个 VTuber 有不同意见——有人说这种文化是由初音未来(T1)引发的，初音未来是一个人声音乐制作软件的化身，他为 Lady Gaga开了，T4 出现在大卫·莱特曼 David Letterman 的节目中，并为体育场大小的观众现场表演。其他人则归功于日本科技公司 Activ8 的项目绊爱，她在 2016 年创办了自己的频道，并创造了“VTuber”这个术语

绊爱的走红催生了日本新一代的网络明星。与日本偶像文化不同，日本偶像文化对现实世界的名人有着不可思议的高标准，虚拟粉丝更自由地做自己，即使他们是作为虚拟角色表演。

“他们存在于动漫角色和真人之间的空间，”动漫 YouTuber Gigguk 在一个视频中说道。“但他们可以探索原创的想法，或者摆脱生活在同一空间的其他人无法摆脱的东西。”

虚拟青春剧在日本盛行多年，但在疫情时期风靡全球。随着世界大部分地区进入封锁状态，大受欢迎的视频网站 HoloLive 推出了英语部门，吸引了新的西方观众。

这个计划不仅奏效。它永远改变了流媒体的格局。

在短短两年内，HoloLive 英语最受欢迎的视频网站 VTuber Gawr Gura 已经积累了超过 400 万 YouTube 用户。这位白发苍苍的动漫女孩穿着一件超大号的蓝鲨连帽衫，她的脸被连帽衫的鲨鱼牙框住了。当然，她明亮的蓝眼睛和她头发的颜色一样，当她笑的时候，她可爱的鲨鱼一样的牙齿露了出来。她是一名音乐艺术家，就像许多 VTubers 一样，她也在流媒体上播放游戏，如【《我的世界》】、马里奥赛车，甚至日本 Duolingo 。根据她的频道描述，她是“失落之城亚特兰蒂斯的后裔，她一边游向地球，一边说‘下面太无聊了，洛洛尔！’"

与此同时，HoloLive 还引入了像Mori Calliope(200 万订户)这样的人才，她自称是“死神的第一个徒弟”，成为一名 VTuber，从她的观众那里“收集灵魂”。卡莉欧碧是一个红眼的叛逆者，她用黑色的皇冠和面纱装饰着她淡粉色的头发。

我们无法确认她灵魂收割的进度，但说到钱，卡莉欧碧肯定是成功的。根据独立的 YouTube 分析网站 Playboard 的数据，2021 年，Calliope 仅从 super chats(YouTube 直播货币化功能)就赚了 854，595 美元，使她成为世界上第七大 super chats YouTube。

谁的六条彩带赢得了卡莉欧佩的超级帽子？当然还有 VTubers。

为什么要成为 VTuber 呢？

对于一个 VTuber 来说，很少像 Code Miko 那样暴露自己的身体——对于许多这样的横幅来说，匿名是最重要的。

你不必签约像 HoloLive 这样的大机构来成为 VTuber。尽管 Code Miko 的技术非常先进，让马克·扎克伯格的元宇宙相形见绌，但这并不常见。只需一部 iPhone 手机，一个新的流媒体工具就可以创建一个面部跟踪的 2D 虚拟角色。

现在，有一个越来越多的跨性别虚拟人社区，其中一些人说，采用虚拟形象帮助他们克服性别焦虑。与社交媒体的抖音方面不同，在那里展示你的脸几乎是没有商量余地的，VTubers 可以在屏幕上展示自己的另一面。例如，VTuber Ironmouse 是 Twitch 上订阅最多的女飘带。但在现实生活中，这位波多黎各游戏玩家患有慢性疾病，有时卧床不起，所以 VTubing 帮助她享受乐趣和社交，特别是在与冠状病毒隔离时。

对于一些飘带来说，这些头像也是防止骚扰的屏障。

尤娜告诉 TechCrunch:“我在网上受到的不良待遇没有我的女同事多。”。"很难找到卡通人物."

此外，在最近的一个视频流中，她展示了自己最先进的 mocap 套装，一名观众评论说她的技术是“色情业的未来”虽然有些虚拟人物确实变得有点活泼——毕竟这是互联网——但这些数字角色不仅仅是性感。

香港城市大学研究 VTubers 的助理教授卢智聪解释说:“我认为，对于观看 VTubers 的人来说，他们中的许多人甚至不关心谁是化身背后的人，谁是配音演员。”“更多的是关于角色和化身，他们对配音演员的真实生活知之甚少。”

不过，匿名也带来了一系列新的挑战。

“特别是对于公司经营的虚拟城市，配音演员可能会被取代，他们的劳动可能会被剥削，”陆说。许多最受欢迎的视频网站都是由 HoloLive、Nijisanji 和 VShojo 等机构创建或管理的。VTubers 有着鲜明的个性，这是由他们的配音演员决定的，但代理公司有可能在粉丝没有注意到的情况下带来一个新的配音演员。此外，人才从经纪公司获得的薪酬比例并不是众所周知的。

“棘手的是，人们实际上什么也看不见，”卢告诉 TechCrunch。“完全不透明。不透明，因为头像。”

当然，企业希望从中获利

八月中旬，作为与 Twitch 合作的一部分，老虎托尼的视频首次在流媒体上亮相。是的，那只老虎托尼，凯洛格冰淇淋的吉祥物，从 1952 年开始就出现在麦片盒子上。

营销和视频网站专家泰迪·坎博萨告诉 TechCrunch，像 T2、网飞、SEGA 和亚洲航空这样的品牌已经在他们的营销中使用了视频网站。但是，激活 VTubers 的庞大粉丝群并不像参与那么简单。

“品牌需要更好地理解，进入 VTuber 领域需要理解，人口统计不仅仅是短期的，”坎博萨说。“一旦他们了解这些粉丝的文化和行为，他们就可以利用粉丝的忠诚度，以获得他们作为潜在客户，并在更长时间内留住他们。”

老虎托尼的 VTuber 首秀很尴尬。吉祥物实际上并没有和加入他的四条 IRL 彩带一起扮演“替罪羊”，他离开了人群很长一段时间，促使成千上万的观众在 Twitch 聊天中要求托尼回归。不过，他稍微弥补了自己的缺席——老虎托尼告诉他的 13000 名观众，他们是他的“ pog champs

在 VTuber 领域之外，PacSun 和 Calvin Klein 等品牌已经与 Lil Miquela 合作，Lil Miquela 是一个完全虚构的 Instagram 影响者，由一家名为 Brud 的风险投资公司运营。但是这些广告活动经常遭到反对——为什么不找一个真实的、非 CGI 的女性来做这些衣服的模特呢？社交媒体已经因伤害少女而受到批评，部分原因是宣传不切实际的美丽标准。但是没有一种美的标准像女性身体的虚拟理想那样不切实际。

老虎托尼和莉尔·密凯拉拥有技术和资金支持，可以在技术上给人留下深刻印象，并进行良好的营销，但 VTubers 必须是真实的，才能与粉丝建立联系。即使对于只通过虚拟形象与观众联系的虚拟人来说，这种现象最终也是关于人与人之间的联系。毕竟，在那双动漫大眼睛后面还有一个真实的人——即使你永远看不到他们的脸。
