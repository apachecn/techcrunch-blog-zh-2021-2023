今天，我们很高兴向大家介绍 Antaris，这是一个基于云的卫星软件平台，致力于简化卫星设计和部署，同时降低成本和缩短入轨时间。

当 Antaris 的联合创始人兼首席执行官 Tom Barton 和 Epsilon3 的联合创始人兼首席执行官 Laura Crabtree 登台参加名为“ Hardware？那是什么？为什么软件是太空经济的未来。

发射得到了所有媒体的关注，但是卫星和它们背后的软件才是太空的主力。Barton 和 Crabtree 都是精力充沛的创始人/首席执行官，他们将谈论 SaaS、开源和基于云的平台如何革新卫星产业并加速太空经济。他们还将讨论第一次创业是什么感觉，以及将优秀代码投入太空需要什么，他们还将分享给其他太空创业者的技巧。

汤姆·巴顿是早期卫星软件创新公司 Antaris 的联合创始人兼首席执行官。巴顿和他的团队在 2021 年创建了这家公司，愿景是彻底改革太空软件——包括利用开放 API 和开源来大幅降低成本和卫星入轨时间。

Antaris 被广泛认为是卫星生态系统的新空间颠覆者，它得到了洛克希德·马丁风险投资公司、Acequia、Possible Ventures 和 E2MC 的支持。巴顿此前曾担任卫星领导者行星实验室的首席运营官三年，是负责将历史上最大的成像卫星星座送入轨道的领导团队成员。

巴顿也是 Diamanti，Rackable Systems 和 Cygnus Solutions 的首席执行官，他在 Red Hat 和 Lightspeed Venture Partners 担任过高级领导职务。他拥有斯坦福大学的理学学士和工商管理硕士学位。

Laura Crabtree 是 Epsilon3 的首席执行官和联合创始人，epsilon 3 是一个复杂工程、测试和操作程序的软件平台。作为 SpaceX 龙飞船运营团队的最初成员之一，Crabtree 帮助美国重返人类太空飞行业务。她想继续革新航天工业，从操作工具开始。
