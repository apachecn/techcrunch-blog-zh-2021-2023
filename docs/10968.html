企业能成为创业公司和风投的好搭档吗？

Cloudflare 上周 宣布一项 12.5 亿美元的资助计划，面向基于其软件 Cloudflare Workers 的初创公司。但这不是企业风险基金，这笔钱也不是公司的钱。

相反，这是一项计划，云基础设施公司策划了一组初创客户，并将其介绍给风险投资家，每个风险投资家承诺 5000 万美元支持基于 Cloudflare 员工的公司。26 家风险基金包括 NEA 和 Boldstart 这样的大公司和 Pear VC 这样的小公司。Cloudflare 首席执行官 Matthew Prince 告诉我，自该项目于 9 月份宣布以来，这一数字一直在持续增长。

有趣的原因是，虽然上市公司近年来大幅增加了初创企业融资，但主要是通过两种方式之一:公司要么在资产负债表上留出一部分资本来支持与自己相邻或互补的行业的初创企业，要么推出一个加速器项目。

Cloudflare 的这个策略感觉很新鲜。如果成功的话，这将是一个非常明智的选择。该计划本质上有助于将资金输送给客户，从而确保他们对该平台的需求，同时也吸引初创公司考虑在其他平台上构建 Cloudflare-而无需 cloud flare 花费任何资金。值得注意的是，加入这个项目的公司，不管他们是否被推荐给风投，都可以免费获得一年的多种软件功能。

但是像 Cloudflare 这样的公司会是一个好的媒人吗？普林斯似乎是这样认为的——他告诉我，这个项目的想法来自公司与风险投资家的对话。
