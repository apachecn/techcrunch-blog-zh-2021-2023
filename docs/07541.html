由 Raba Partnership | TechCrunch 领投的基于 API 的初创公司 Thepeer 筹集了 210 万美元

原文：https://web.archive.org/web/https://techcrunch.com/2022/06/22/thepeer-an-api-based-startup-that-connects-african-businesses-raises-2-1m-led-by-raba-partnership/

连接企业钱包的非洲科技基础设施初创公司 Thepeer 已经筹集了 210 万美元的种子资金，由 Raba Partnership 牵头。一年前，这家初创公司从一些天使投资者那里筹集了 22 万美元的种子期前投资，其中包括 Paystack 首席技术官 Ezra Olubi 和 Edenlife 首席技术官 Prosper Otemuyiwa。

peer 种子轮的参与投资者包括 RaliCap、Timon Capital、BYLD Ventures、Musha Ventures、Sunu 和 Uncovered Fund。非洲的金融科技公司也投入了大量的现金和资金。

peer 不是一个纯粹的金融科技游戏；它位于数据和金融的交叉点。然而，当他们在 2021 年 8 月首次推出该公司时，他们想要解决的问题kosinuchkwu Chike Ononye和 Michael“特洛伊”Okoh 对 fintechs 的影响超过了任何其他部门。像其他基于 API 的创业公司一样，peer 主要为中小型金融科技企业提供基础设施。

根据当地出版物 Disrupt Africa 的数据，去年，非洲金融科技公司的数量从 2019 年的 491 家增加了 17.3%，达到 573 家。这些金融科技大多面向企业和消费者，提供不同的服务，如支付、新银行、贷款、投资和交易。虽然他们提供数字钱包来帮助促进资金转移，但在他们的生态系统之外缺乏移动钱包的互操作性；本质上，将资金从一个金融科技钱包转移到另一个金融科技或非金融科技钱包(在嵌入式金融游戏的情况下)是很难的。

Thepeer 表示，其 API 提供了一个替代网络，金融科技公司和企业可以将不同系列的产品嵌入到他们的应用程序和网站中，以便客户轻松转移资金。

该公司推出的 Send(发送)服务允许整合了其 API 的企业客户，如 Eversend(跨境金融科技)和 Nguvu Health(远程治疗平台)，使用电子邮件或用户名等标识符在两个平台之间汇款。

“我从一个供应商那里买了东西，在使用了几个应用程序后无法支付，也无法将钱从一个有现金的钱包转移到另一个钱包，”首席执行官 Ononye 在接受 TechCrunch 采访时解释了他和首席技术官 Okoh 创办该公司的原因。“在我们推出第一款产品[Send]后，我们又回到了绘图板，因为有些事情我们需要做好。在这个过程中，我们发现了更多的消费者和企业问题。”

在 Send 之后，peer 与 Flutterwave 合作，flutter wave 是一家独角兽公司，其 API 网关是非洲最广泛的移动钱包和银行账户网关，并在其平台上建立以获得更多影响力。这种合作关系帮助它推出了另外两种 B2B2C 产品:直接收费和结账。

直接收费可以让大约 15 家企业的顾客互相掏钱。它是这样工作的。作为 Nguvu Health 的用户，有不同的方法来资助一个钱包，如卡和银行转账；peer 是另一种选择。因此，当 Nguvu Health 用户选择 peer 时，他们可以选择 15 个钱包中的任何一个，将它们与他们的 Nguvu Health 钱包集成，并启动直接收费交易。

Nguvu Health 首席执行官 Joshua Koya 在谈到 Thepeer 的整合时说:“关于 Thepeer 的一个令人惊讶的事情是，它使人们更容易在 Nguvu Health 应用程序上支付治疗费用，这使得非洲人可以通过智能手机轻松获得治疗。”“我们的用户现在可以选择将 peer 与其他金融科技钱包相结合，在那里他们可以支付治疗费用。”

然而，在诸如网上食品和服装店这样的商业网站上，结帐也是类似的。当顾客在网上购物时，peer 作为一种结账方式出现，让他们从 Bitsika、Eversend、Chipper Cash 和 Paga wallets 支付商品。

这家成立一年的金融科技初创公司自推出以来，平均每月交易增长 161%。随着 Send 的推出，其新产品专注于帮助企业通过其仪表板(不是面向消费者的仪表板)在彼此之间汇款，其月交易额增长了 65 倍以上，达到“八位数”百万美元。

“我们认为我们的产品是一个由 API 驱动的平台，通过提供无缝的资金流动来帮助连接金融科技公司、消费者和企业，”Ononye 说。“如今，整个欧洲大陆有近 600 家金融科技公司，其中大部分都在运营各自为政的钱包。我们的目标是让从任何钱包连接和支付成为可能。我们正在构建一个操作系统，以便企业能够为客户提供更多服务。”

对于创始人来说，围绕各种应用转移价值的挑战已经成为一个集成了多种业务的基础设施，并为他们的用户提供更多价值。这种进展使得 Raba Partnership 在这一轮中领先。其创始人 George Rzepecki 在一份声明中，将 Thepeer 统一非洲钱包生态系统的举动比作 Flutterwave 在 2015 年进入碎片化的卡和移动支付领域。

“随着非洲消费者和 B2B 金融科技的激增，对等体正在建立一个基于 API 的基础支付层，金融科技可以在各自的钱包和应用程序内实现资金流动，”他继续说道。“我们非常高兴能够支持这个雄心勃勃的团队构建下一代网络。”

Raba Partnership 在非洲的金融科技投资名单很长，peer 是其中的一员。尽管 Raba 投资了一些消费平台，如 Thndr T1 和 T2 Djamo T3，但它也是 Flutterwave、T4、Yoco T5、T6 Stitch T7、Axis Pay 和 one pipe T9 等基础设施的支持者。

这个人的下一步是什么？据 Ononye 称，这家初创公司正在寻求推出更多功能和其他货币的钱包(其目前的平台与 Naira 钱包交易)，以及雇佣更多人才和投资于产品开发。“我们的重点仍然是产品创新和服务于我们在不同地区经营的客户，”他说。
