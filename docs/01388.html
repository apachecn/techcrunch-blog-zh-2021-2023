开发者生产力初创公司 Gadget 融资 850 万美元，由 Sequoia 和 Bessemer 领投 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/12/08/developer-productivity-startup-gadget-raises-8-5m-led-by-sequoia-and-bessemer/?tpcc=tcplustwitter

开发者生产力初创公司 Gadget 融资 850 万美元，由红杉和贝塞麦领投

由两名前 Shopify 员工创建的开发人员生产力公司渥太华的 Gadget T1 已经筹集了 850 万美元的种子资金。这轮融资由红杉资本(Sequoia Capital)和 Bessemer Venture Partners 领投，OpenAI 的联合创始人兼首席技术官格雷格·布罗克曼(Greg Brockman)参与；Klarna 的 Koen KoppenShopify 的数据科学和工程主管 Solmaz Shahalizadeh 还有更多。

Gadget 由 Harry Brundage 和 Mohammad Hashemi 于 2020 年创立，旨在为电子商务应用程序开发人员提供一种减少他们花在编写代码上的时间的方法。该平台允许开发人员跳过与其任务相关的繁忙工作，从而更高效地构建和扩展应用。

首席执行官布伦戴奇在接受 TechCrunch 采访时表示:“我们创建 Gadget 是因为我们试图自己构建一系列不同的东西，并对花了很长时间才能让任何东西起飞感到沮丧。”“花上一周又一周的时间进行发布前的准备工作令人沮丧。我们发现很多其他人也有同样的问题，我们想做一些东西来解决它。”

Gadget 将开发人员所需的工具、库、API 和最佳实践捆绑到单一体验中。它允许开发人员定义他们的数据模型和编写代码，同时访问一组高级原语，如内置状态机、自动化访问控制、即时 API 生成和与其他 SaaS 平台的集成。

就最近的融资而言，Gadget 计划利用它公开推出其无服务器堆栈，并从 Shopify 开始建立与第三方 API 的连接。Brundage 和 Hashemi 计划首先关注这家电子商务巨头，以便利用他们在该公司的产品和工程专业知识。

“我们将把这笔钱用于雇佣一个更棒、更大的团队，然后尽可能多地推出新功能，”布伦戴奇说。

关于未来，Brundage 和 Hashemi 希望建立一个长期持久的公司，使重要的软件更容易开发。Brundage 指出，有许多可能有用的工具还没有被制造出来，该公司希望 Gadget 可以让这些工具成为现实。

“软件开发正到达一个临界点，”红杉的合伙人迈克·弗纳尔在一份声明中说。“随着我们对所用软件的期望越来越高，开发人员也需要更多的工具来满足需求。Gadget 的平台实现了帮助电子商务开发者以令人难以置信的速度构建可扩展软件的承诺。
