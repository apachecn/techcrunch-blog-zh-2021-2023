查尔斯斥资 2000 万美元将对话式商务引入 WhatsApp

原文：https://web.archive.org/web/https://techcrunch.com/2022/07/20/charles-raises-20m-to-bring-conversational-commerce-to-whatsapp-in-europe/

对话式商务并不是一个新现象，无数公司使用实时聊天、消息应用、聊天机器人、语音助手等等来鼓励消费者掏钱。作为这一更广泛运动的一部分，强大的 WhatsApp，信息世界的主导力量，一直在利用无数工具更深入地推动商业领域，将零售商和客户联系起来——从产品目录和系列，到购物车和 Instagram 商店整合。

对话式商务市场也相当可观，据报道，仅在 2020 年，中国的微信就促成了 2500 亿美元的交易。但是，虽然在全球许多市场，特别是在亚洲和拉丁美洲，从消息应用程序中购物是正常的，但在欧洲还没有达到同样的水平——这是德国初创公司 Charles 希望通过一个平台来改变的事情，该平台将关键的对话式商务组件与时事通讯的营销能力相结合。

在推出两年后，Charles today 宣布，它已经在由 Salesforce Ventures 牵头的首轮融资中筹集了 2000 万美元，Accel 和 HV Capital 也参与了融资。在此之前，它去年筹集了 650 万美元的种子资金。

它是如何工作的

Charles 于 2019 年在柏林成立，将自己定位为一个完整的端到端产品，涵盖后端和界面，将 WhatsApp 等消息服务的 API 与 Shopify 和 Salesforce 等流行的电子商务和 CRM(客户关系管理)系统连接起来。然后，企业可以销售产品，发送简讯并提供后续支持。

查尔斯的行动平台。图片来源:查尔斯

虽然销售和服务方面是任何会话式商业软件所期望的，但是时事通讯方面是一个有趣的补充。WhatsApp 简讯可能包括折扣、特价、产品发布或视频消息，但重要的是，它是为消费它的媒体(即消息应用程序)而设计的，而不是传统的电子邮件格式。

简讯本质上是一对多的“广播”，除非通过 API 访问，企业可以一次发送给无限多的接收者 WhatsApp 的标准广播仅限于 256 人。最重要的是，零售商可以使用 Charles 平台创建自动化的选择加入流程(例如，通过点击按钮或扫描网站上的二维码)，并访问性能分析，以显示 WhatsApp 简讯在参与度方面的增长情况。

WhatsApp 简讯:注册的动机。图片来源:查尔斯

对话式商务来到欧洲

快速浏览一下世界各地，就会发现对话式商务领域出现了一系列活动，像 Whym 、 Zeals 、 Yalo 和 Wizard 这样的公司最近都在为亚洲或美洲的目标市场筹集资金。在其他地方， Vonage 收购了新加坡会话商务公司 Jumper 。

这暗示了 Charles 是如何寻求差异化的——它希望在离家更近的市场复制其他公司的成功。这就是其新注入的 2000 万美元现金的用途，因为它希望将视野扩大到本土德国以外。

值得注意的一点是，Charles 在构建其平台时特别牢记欧洲的数据隐私法(即 GDPR )。例如，对于 WhatsApp 时事通讯，Charles 在客户表示希望收到 WhatsApp 消息后会向他们提供两次选择加入——所以客户可能会要求注册时事通讯(第一次选择加入)，然后他们会第二次被要求确认这是他们想要的。所有这些都是自动化的。

“以欧洲为目标，我们的主要优势之一是，我们从一开始就内置了 GDPR 合规性，”联合创始人兼首席执行官 Artjem Weissbeck 告诉 TechCrunch。

Charles 说这符合 GDPR 的规定。影像学分: Charles

不是每个人都会乐意看到有针对性的营销和广告每天都出现在 WhatsApp 的收件箱里，这也是 Charles 专注于选择加入/选择退出工作流程的原因。它与电子邮件不同，需要相应地对待。

“为了符合 GDPR 标准，品牌需要您的初始选择加入通知，而选择退出随时都有可能，”韦斯贝克继续说道。“这也是为什么我们通过自动选择加入和选择退出技术帮助品牌，我们的成功团队就通知的频率和相关性向他们提供建议。与电子邮件相反，它们应该是低频率和高相关性的，以尊重渠道的亲密度并保持消费者提供的信任。平均而言，这意味着每个月要进行一至两次活动。”

尽管查尔斯在德国国内市场的吸引力最大，但它已经从意大利、西班牙、法国、荷兰和英国获得了一些入境兴趣。从长远来看，它认为自己正在向其他市场扩张。

“这些是我们已经开始扩张的国家，超过 20%的客户来自这些国家——尤其是意大利和英国，”联合创始人兼联合首席执行官安德烈亚斯·图辛补充道。“我们的最终目标是全球化，我们有选择地与欧洲以外的公司进行试点。”

查尔斯表示，在其迄今相对较短的生命周期内，其 100 多个客户中的一些已经在今年达到 7 位数的 WhatsApp 收入，占其总销售额的 40%。但是 Charles 意识到了这样一个事实，虽然 WhatsApp 是欧洲一个卓越的信息平台，但它并不是这个城市里唯一的玩家——这就是为什么它也支持 Instagram Direct 和 Facebook Messenger，同时它还致力于支持电报、即时消息、RCS、谷歌我的商务聊天和手机短信。

“然而，欧洲大多数企业都专注于 WhatsApp，因为消费者渗透率最高，丰富的格式最先进，企业应用编程接口能力也最先进，”图辛指出。
