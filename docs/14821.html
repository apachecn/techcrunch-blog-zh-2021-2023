在 Twitter 终止其 Tweetbot 应用程序| TechCrunch 后，Tapbots 推出了新的乳齿象客户端 Ivory

原文：https://web.archive.org/web/https://techcrunch.com/2023/01/24/tapbots-launches-a-new-mastodon-client-ivory-after-twitter-kills-its-tweetbot-app/

广受欢迎的第三方 Twitter 应用 Tweetbot 的制造商 tap bots(T1)正在发布其下一款新产品，该应用最近被 Twitter 的 API 变化杀死，而则杀死了。为了填补 Tweetbot 留下的空白，该公司现在正在 App Store 上发布其预期的乳齿象客户端应用 Ivory 作为早期版本。

该公司告诉我们,“早期访问”标签是 Tapbots 在其发布时的副标题，表明在其首次发布时仍将缺少一些功能。然而，通过在 App Store 上公开发布，Tapbots 能够在填满其测试版本的有限数量的试飞位置后，将象牙放入更多人的手中。

Tapbots 创始人保罗·阿达德告诉 TechCrunch，Ivory 的应用商店发布昨天刚刚获得批准，但在公布发布之前，该公司还有一些关于订阅的技术问题需要解决。

对于长期使用 Tweetbot 的用户来说，象牙将提供一种熟悉的体验。但是，该公司不再是 Twitter 网络的客户，而是拥抱了前景广阔的开源平台乳齿象。尽管不如 Twitter 那么容易使用或理解，但在埃隆马斯克收购 Twitter 后的几个月里，乳齿象已经获得了关注。

随着 Twitter 的新主人实施了有争议的改变，网站的性能不时下降，一些人开始寻找一个新家来发布简短的更新，就像他们在 Twitter 上做的那样。这导致了对新应用程序和其他 推特 替代品的兴趣。截至去年 12 月，Mastodon 的月活跃用户已经增长到 250 万，分布在大约 8600 台不同的服务器上，用 Mastodon 的行话来说就是“实例”

然而，一些乳齿象新人不喜欢官方的原生移动应用程序——特别是在使用了更快、更精致的 Twitter 原生移动应用程序或来自第三方的应用程序，如 Tweetbot 之后。这有助于吸引人们对 Ivory 和其他由前 Twitter 第三方应用开发商(包括 Mammoth)开发的等新兴应用的兴趣。

Tapbots 表示，Ivory 的功能集利用了 Tapbots 在为 Twitter 平台构建其屡获殊荣的 Tweetbot 客户端方面十多年的经验。

在发布时，它展示了几十个功能，从支持基线功能到智能的铃铛和口哨，比如能够为应用程序设置主题或改变图标。

该应用程序支持多个帐户，并允许您查看您的本地和联合时间表，趋势帖子，帖子统计数据，通知等。它还实现了 Twitter 上没有的乳齿象特有的选项，如在帖子中添加内容警告的功能，以及更常见的功能，如发布 gif 和投票的功能。

图片来源: Tapbots

还有其他旨在吸引高级用户的贴心设计，如标签跟踪、支持正则表达式的静音过滤器和时间轴过滤器，让你显示或隐藏符合特定标准的帖子。这对乳齿象的老用户也有吸引力，他们可能希望屏蔽和避免乳齿象新人分享的一些帖子，这些新人将 Twitter 的文化带到了平台上，导致他们的时间线中没有内容警告的不想要的帖子。

象牙还允许用户以其他方式个性化应用。除了过滤器、主题和图标选项之外，用户还可以配置屏幕底部的导航按钮，通过按下按钮在书签、收藏、统计信息、个人资料、通知、列表和搜索等应用程序不同区域的选项卡之间切换，向用户显示他们想要查看的内容。

此外，该应用程序还提供了快速的性能、平滑的滚动以及有吸引力的用户界面和声音设计。

就首次发布而言，象牙海岸已经是一个相当强大的客户。不过该公司也表示，该应用还有很多需要改进的地方，比如可以编辑个人资料或帖子的工具(比如“编辑推特”(Edit Tweet)按钮！)，以及其他用于读取媒体的 alt/description 文本、改进的哈希表、对自定义实例表情的支持、具有更好过滤的改进的通知标签、改进的导航栏和抑制重复帖子的功能。

和 Twitter bot 一样，这款应用也将通过应用内订阅产生收入。App Store 列出的这些服务的月成本为 1.99 美元，年成本为 14.99 美元。“卓越理财”套餐每年还可享受 24.99 美元的折扣。后者包括所有的特点，在 Pro 订阅，稍后将包括一些额外的，该公司说。哈达德表示，增加 Premier 一级的主要原因是，人们要求用更多的钱来帮助公司摆脱困境，但公司计划在未来为这些人提供一些额外福利，比如额外增加一些图标或壁纸。

这些订阅目前可以在象牙海岸应用程序中获得，并支持一些高级用户功能，比如配置要接收的通知等。用户还可以免费“演示”该应用程序，而无需提交订阅。

由于 Twitter API 的改变，Twitter bot 的收入被削减到了零。因此，只有当消费者愿意采用 Mastodon 并订阅其他客户端时，像象牙海岸这样的应用才有前途。同时，因为和 Twitter 相比，Mastodon 的全球用户基础要小得多，所以和 Twitter bot 应用程序相比，象牙海岸的收入潜力上限要小得多。(如果你有怀旧之情，可以登陆 Twitter 的悼念页面。)

几周来，我们一直在通过 TestFlight 版本测试 evolve，结果发现它提供了一种性能卓越、内容丰富的用户体验，在很多方面都优于官方应用，这让它很容易成为 iOS 设备上使用 Mastodon 的首选应用之一，甚至在早期阶段也是如此。
