特斯拉电动车不再有雷达驶向欧洲、中东

特斯拉将开始向欧洲和中东的客户交付没有雷达的 Model Y 和 Model 3 汽车，这是该公司计划的延续，也是首席执行官埃隆·马斯克(Elon Musk)希望仅使用结合机器学习的摄像头来支持其高级驾驶辅助系统和其他主动安全功能。

2021 年 5 月，特斯拉首次开始制造没有雷达的 Model 3 和 Model Y 汽车。这些车辆仅面向北美市场。该公司在其网站上的更新中表示，它将于本月开始向欧洲和中东交付其仅有摄像头的 Y 型和 3 型汽车。

该公司表示，这些车辆交付时可能会有一些暂时受限或不活动的功能，包括让它保持在其车道内的功能。这项名为 Autosteer 的功能将被限制在最高时速 80 英里和更长的最小跟随距离。特斯拉将在未来几周通过一系列无线软件更新慢慢恢复功能。所有其他可用的自动驾驶和特斯拉全自动驾驶功能将在交付时激活，具体取决于订单配置。

特斯拉停止使用该传感器的决定背离了行业标准。汽车制造商通常使用雷达和摄像头的组合来提供提供高级驾驶辅助系统功能所需的传感，如自适应巡航控制(使汽车速度与周围交通相匹配)，以及车道保持和自动变道。一些汽车制造商，如奔驰和沃尔沃，甚至增加了第三个传感器，称为激光雷达，或光探测和测距传感器，以提供 ADAS 系统的冗余。

大多数汽车制造商都在朝着增加而不是减少传感器的方向发展。特斯拉及其首席执行官马斯克是逆向思维者。

马斯克长期以来一直在吹捧其品牌“特斯拉视觉”系统的潜力，该系统仅使用摄像头和所谓的神经网络处理来检测和理解车辆周围环境中正在发生的事情，然后做出适当的反应。

神经网络是机器学习的一种形式，其工作方式类似于人类的学习方式。它是一种复杂的人工智能算法，允许计算机通过使用一系列连接的网络来识别数据中的模式，从而进行学习。

许多为 robotaxis 或自动驾驶卡车开发自动驾驶汽车技术的公司使用深度神经网络来处理特定问题。但他们隔离了深层网络，并使用基于规则的算法来连接到更广泛的系统中。这些 AV 开发者还使用相机、雷达和激光雷达的组合来为系统提供冗余。
