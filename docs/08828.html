足迹改变了公司收集/存储/共享数据的方式

原文：https://web.archive.org/web/https://techcrunch.com/2022/08/03/footprint-wants-to-change-how-companies-collect-store-and-share-personal-data/

任何申请过公寓或抵押贷款的人都知道，这些公司往往会收集比他们需要的更多的信息，以确定你是否能负担得起月供。申请过程通常会涉及数页关于你和生活的数据，但如果公司能够以一种安全的方式获取他们需要的数据呢？

这是 Footprint 背后的前提，这是一家早期创业公司，希望改变公司收集信息的方式，同时帮助消费者控制自己的数据。这是一个雄心勃勃的想法，这家初创公司今天宣布了 600 万美元的种子投资。

“对于最终消费者来说，足迹是他们填写的最后一张身份表。对于企业来说，足迹只是五行代码。公司联合创始人兼首席执行官 Eli Wachs 告诉 TechCrunch 说:“它让我们的客户既能管理用户，又能创建账户，或者‘了解你的客户’(KYC)，还能分担存储数据所需的所有安全成本和风险。”。

该公司的目标是创建一个类似于苹果钱包的工具，但用于你的数据，因此它会使用移动设备以加密格式保存你的私人信息，以确保身份。“Footprint 在一个受保护的“安全飞地”中安全地保存消费者可验证的凭证(如社会安全号码、出生日期、电子邮件、电话号码)，由硬件级加密认证提供支持，”创始人解释道。

然后，该公司利用你手机上的 Face ID 或触控 ID 作为身份证明，与可信的合作伙伴访问和共享这些信息。

图片来源:足迹

公司联合创始人兼首席技术官 Alex Grinman 曾是 2019 年被 Akamai 收购的初创公司KryptCo 的联合创始人，他表示，在后端，该公司正在利用最初由 AWS 开发的安全飞地概念。他举了一个例子，证明某人超过 21 岁就可以进入酒吧。

“嗯，你不必解密他们的出生日期，然后做计算和应用程序代码。你可以在安全区域内做，所以我们只发布你需要的最少量的信息，”他说。这种方法还有额外的优势。

“因为它是解密任何数据的唯一安全途径，所以我们还有非常精细的审计日志记录。所以我们确切地知道谁在解密什么数据。如果他们被允许，我们可以给他们数据。如果他们不被允许，我们可以阻止他们，”他说。所有内容都被记录到只附加的审计日志中。

Wachs 解释说，今天，该公司可以收集和存储姓名、电子邮件、电话号码、出生日期、地址、社会保险号，并可以选择添加诸如验证收入来源(包括您的雇主和收入)和验证您拥有的企业等内容。随着时间的推移，我们的目标是能够处理申请公寓等东西所需的所有数据，并消除大多数公司仍在使用的当前基于纸张的系统中涉及的所有摩擦，同时允许用户与任何可信的合作伙伴共享他们的足迹数据。

投资者指数风险投资公司(investor Index Ventures)的合伙人沙尔杜尔沙阿(Shardul Shah)认为，一家拥有聪明创始人的公司正试图解决一个有很大增长空间的行业中的难题。

“我认为身份市场是巨大的，围绕该行业有许多商业活动，因为有许多有价值的交换，但它从根本上被打破了。大多数人都有一个观点，那就是要有所取舍。要么是低摩擦，要么是高精度。我认为勇敢和困难的是亚历克斯和伊莱的观点，那就是你可以两者兼得。顺便说一句，你可以是低摩擦和易于使用的，由一系列的利益相关者，以及令人难以置信的准确，”沙阿说。

该公司目前有 7 名员工，但 Wachs 表示，随着公司的发展，多元化对这家初创公司来说是一个重要的价值。“我们认为多样性非常重要。亚历克斯和我真的为一支看起来不像我们的球队感到骄傲。我们是队里唯一两个长得像我们的人。据我们所知，许多公司在后期阶段会大力推行多元化招聘，对我们来说，如果这不是一开始的重点，那就不是真正的重点，”Wachs 说。

该产品目前处于早期发布阶段，但该公司计划在秋季全面推出。

今天的 600 万美元投资由 Index Ventures 领投，参与投资的还有行业天使，以及 BoxGroup、Operator Partners、Lerer Hippeau、Palm Tree Crew、Not Boring Capital 和 K5。
