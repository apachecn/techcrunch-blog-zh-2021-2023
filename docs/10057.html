DJI 用 Osmo Action 3 颠覆了动作摄像机的世界

DJI 最出名的是它的四轴无人机，但在过去几年里，该公司一直在与邻近市场的老牌公司进行一些有趣的斗争。今天，DJI 发布了其 Osmo 系列动作相机的第三代产品 Osmo 3。新相机大大增加了电池容量，使相机比以往任何时候都更加方便创作者。

如今，很多内容都是垂直消费的——想想抖音和各种平台上的故事功能，相机的一个更新(回想起来很明显)是 Osmo 3 的设计既可用于横向也可用于纵向。设计师们用 1770 毫安时的电池建造了这台相机，它可以让相机记录两个半小时。他们还增加了快速充电功能，可以在 18 分钟内从没电状态充电到 80%。不到一个小时，它就会充满电。

“自从我们在 2019 年发布最初的 Osmo Action 以来，我们一直受到用它拍摄的惊人镜头的启发，”DJI 高级产品线经理 Paul Pan 说。“随着我们的用户不断向更远、更快、更深的方向前进，我们的使命是提供一款能够跟上他们步伐的设备。每一项新功能都使 Osmo Action 3 成为最可靠、最坚固、最易于使用的动作相机。我们迫不及待地想看看我们的用户通过 Osmo Action 3 推动他们的极限时创造了什么。”

该相机在 4K 以 120 fps 的速度拍摄，拥有超宽的视野，具有先进的图像稳定功能。相机也比以前更好地管理它的温度；该公司声称相机不需要一点休息来冷却。如果有电池，可以录音。

DJI Osmo Action 3 有一个“删除自拍杆”功能，当你在山上巡航时，它会让杆子看不见。图片来源: DJI

相机喜欢极端条件，无论是温度还是防水。冲浪、滑雪和一些浮潜都非常壮观:你可以将它带到 16 米深的地方，而无需额外的防水外壳，相机可以在零下 20 摄氏度(-6 华氏度)的温度下保持工作。它还具有大约 1.5 米(5 英尺)的抗摔性。

Osmo Action 3 今天上市，有两种包装:Osmo Action 3 标准组合零售价为 329 美元，包括 Osmo Action 3，一个 Osmo Action 3 Extreme 电池，Osmo Action 3 水平-垂直保护架，Osmo Action 3 快速释放适配器底座和 Osmo 平板粘合底座。Osmo Action 3 Adventure Combo 的零售价为 439 美元，包括相同的包装，但有三块电池，两个快速释放支架和一个电池盒。一系列附件可供选择，包括皮带支架、车把支架、头盔和吸盘支架、镜头盖等。你可以在 DJI 网站上了解更多关于这款相机的信息。
