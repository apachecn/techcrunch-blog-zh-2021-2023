D-ID 是 Deep 怀旧背后的公司，它让你从一张图像中创建人工智能生成的视频 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/09/19/d-id-the-company-behind-deep-nostalgia-lets-you-create-ai-generated-videos-from-a-single-image/

以色列人工智能公司 D-ID ，为深度怀旧等项目提供技术，正在推出一个新平台，用户可以上传单个图像和文本来生成视频。通过这个名为 Creative Reality Studio 的新网站，该公司瞄准了企业培训和教育、公司内部和外部沟通、产品营销和销售等领域。

该平台使用起来非常简单:用户可以上传一个演示者的图像，或者从预先创建的演示者中选择一个来开始视频创建过程。付费用户可以访问更具“表现力”的高级演示者，因为他们比默认演示者有更好的面部表情和手部动作。之后，用户既可以键入脚本中的文本，也可以简单地上传某人演讲的音频剪辑。用户可以选择一种语言(该平台支持 119 种语言)，声音和风格，如欢快，悲伤，兴奋和友好。

该公司基于人工智能的算法将根据这些参数生成一个视频。用户可以在任何地方分发视频。该公司声称，该算法只需要一半的视频持续时间来生成一个剪辑，但在我们的测试中，它需要几分钟来生成一个一分钟的视频。这可能会根据演示者的类型和您选择的语言而有所不同。

“新冠肺炎疫情加速了全球对数字内容的需求。组织面临的一个大问题是教育内容的创建。阅读文件和浏览演示文稿可能会枯燥乏味。此外，他们还必须花费数千美元来雇佣演员和制作教育视频。因此，我们正在使用我们的人工智能来创造主持人和导师，以重演人类，并使内容更加迷人和有效，”D-ID 的首席执行官吉尔·佩里在接受 TechCrunch 采访时表示。

Perry 指出了这项技术的许多使用案例——从首席执行官给员工的多语言消息到组织用户的个性化愿望。

D-ID 在 8 月中旬推出了该工作室进行测试，以便在公开发布之前消除错误。虽然它的主要重点是迎合各种规模的公司，但该公司看到平台上的创作者对很感兴趣。

制作攻击性的 deepfake 视频是有风险的。这就是为什么这家以色列公司安装了护栏，如过滤脏话和种族主义言论，以及图像识别，以避免使用名人的脸。它使用微软 Azure 文本审核 API 来清除视频脚本中的性言论和攻击性语言。D-ID 表示，该平台的使用条款禁止用户创建政治视频。如果违反这些规则，公司可以暂停违规者的帐户，并从图书馆删除他们的视频。

D-ID 在 3 月份由麦格理资本牵头的 B 轮融资中筹集了 2500 万美元，迄今总共筹集了 4700 万美元。到目前为止，该公司一直依赖其他人使用其 API 来创建内容——深度怀旧就是一个主要例子——客户包括 Modelez、华纳兄弟和印度短视频应用 Josh 。现在，该公司正在通过推出一个 PowerPoint 插件和这个自助服务平台来扩展其赚钱的产品。该插件为幻灯片添加了一个交互式演示者，因此用户不仅仅需要读出幻灯片。他们可以选择不同的头像、声音和语言，就像自助服务平台一样。但是目前还没有定制演示者的规定。

在发布时，用户将能够免费注册一个 14 天的试用账户，并创建长达 5 分钟的人工智能生成的 720p 视频。之后，他们可以每月支付 49 美元，获得 15 分钟的全高清人工智能视频、PowerPoint 插件和电子邮件支持。

D-ID 联合创始人埃利兰·库塔、吉尔·佩里和塞拉·布隆德海姆。图片来源: D-ID

用户还可以上传自己的音频片段进行语音克隆。此外，该公司正在开发一种工具，让用户上传自己的镜头，以训练人工智能更具表现力，从而更好地模仿视频中的人。所有这些特性都将局限于公司的企业级。

虽然该公司在人工智能生成视频领域面临来自 Rephrase.ai 和 Soul Machines 等公司的竞争，但它声称几乎没有任何公司声称可以从单一图像生成高质量的视频。

佩里说，D-ID 的目标不是将自己局限于企业培训、交流和营销视频。它还雄心勃勃地推动实时视频通话翻译和克隆主持人——在你口述音频时，让一个化身代替你出现在视频上。

该公司还在考虑成为 web3/元宇宙开发的主要参与者。“鉴于我们拥有从单一图像生成视频的专业知识。我们正在考虑为元宇宙创造数字化身的方法，”佩里说。
