随着苹果 WeatherKit 的发布，iOS 的黑暗天空将在年底结束运营

原文：https://web.archive.org/web/https://techcrunch.com/2022/09/13/as-apples-weatherkit-launches-dark-sky-for-ios-to-wind-down-operations-by-year-end/

对黑暗天空用户来说，坏消息是:苹果在 2020 年 3 月收购的热门天气应用现在有了官方关闭日期。根据该应用程序主屏幕上出现的通知，自 2023 年 1 月 1 日起，黑暗天空将不再可用，因为其预测技术现已集成到苹果自己的天气应用程序中。

这一消息并不令人惊讶，尽管令人失望。

收购完成后，苹果迅速关闭了黑暗天空的 Android 和 Wear 操作系统版本，并表示将不再接受黑暗天空 API 的注册，该 API 允许第三方应用程序开发者访问该公司的天气预报和历史天气数据数据库。后来将 API 服务的关闭时间推迟到 2022 年底，并表示 iOS 应用将在同一时间关闭。

写在墙上。毕竟，黑暗天空的应用并不是苹果交易的关键焦点——它是预测技术和天气服务本身。

苹果看到了将 Dark Sky 的 API 转化为自己创收业务的潜力。在此之前，该 API 已经免费提供了 1000 次 API 调用，之后每次调用收费 0.0001 美元。在苹果 2022 年 WWDC 大会上，在发布 iPad 默认天气应用的同时，苹果宣布将 API 的付费用户转移到其新的天气服务 WeatherKit，在测试期及以后，它将每月提供多达 50 万次 API 调用，作为其苹果开发者计划成员的一部分。

然后，它将继续按照以下费率向企业收取额外数据费用:

  • 100 万次通话/月:49.99 美元
  • 每月 200 万次通话:99.99 美元
  • 500 万次通话/月:249.99 美元
  • 每月 1000 万次通话:499.99 美元
  • 每月 2000 万次通话:999.99 美元

截至昨日 iOS 16 发布，苹果也正式推出了 WeatherKit 订阅，并指出开发者现在可以通过特定平台的 Swift API 在为 iOS 16、iPadOS 16、macOS 13、tvOS 16 和 watchOS 9 构建的应用程序中访问这些天气数据，并通过 REST API 在其他平台上访问这些数据。该公司指出，开发者可以通过苹果开发者应用中的账户标签随时升级或降级。

鉴于这项服务现在已经上线，并与苹果的天气应用集成在一起，苹果将关闭 Dark Sky 的 iOS 应用是可以理解的。但有趣的是，即使黑暗天空应用程序直接将用户指向苹果天气应用程序，作为其通知现有用户的一部分，苹果也不一定要说服这些用户进行转换才能受益。

因为苹果购买了 Dark Sky 的后端天气数据服务，现在每当用户转向另一个接入 WeatherKit 的第三方天气应用程序时，它就有机会产生收入。然而，目前还不清楚有多少以前的黑暗天空 API 客户会进行这种转变，因为 WeatherKit 服务刚刚上线。

然而，看到 Dark Sky 面向消费者的产品消失是一种耻辱。这个设计良好的应用程序长期以来一直是许多人的最爱，这要归功于它易于阅读的用户界面，带有可点击的按钮，可以在当天的温度、降水概率、湿度、紫外线指数等预测之间快速切换，以及它的雷达地图和可定制的通知。根据移动智能公司 Sensor Tower 的数据，iOS 应用程序迄今已有近 200 万次下载

值得庆幸的是，如果苹果自己的天气应用程序不能激发灵感，仍然有一个第三方天气应用程序的生态系统可以为预测带来更多的视觉效果。我们最喜欢的有:胡萝卜天气、(不无聊)天气、迈拉德和 Clime (如果你愿意订阅的话)。
