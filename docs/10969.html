埃隆·马斯克的 X 应用程序“一切”在美国可能不会成功

原文：https://web.archive.org/web/https://techcrunch.com/2022/10/08/elon-musk-x-everything-wechat-tesla-app/

随着埃隆·马斯克(Elon Musk)再次接近收购 Twitter 的交易，关于这位亿万富翁计划如何改变社交网络的猜测再次浮出水面。马斯克本周的推特(tweet)提供了一个线索:“收购 Twitter 是创造 X 这个万能应用的催化剂。”

虽然马斯克没有详细说明 X 会是什么样子，但许多人认为他希望复制微信的成功，微信在过去十年中几乎已经成为中国的一切应用。人们用它来看新闻、打招呼、预约医生、缴税和进行各种各样的日常活动。

鉴于马斯克对腾讯旗下的 messager 赞不绝口，这或许的确是他的主意。6 月份，在他与 Twitter 员工在 T2 的第一次市政厅会议上，这位特斯拉创始人大谈微信是美国社交网络的一个可能愿景。

你知道，如果我想到中国的微信，它实际上是一个非常非常好的应用程序，但在中国以外没有微信运动。我认为这是一个真正的创造机会。在中国，你基本上生活在微信上，因为它对你的日常生活非常有用，非常有帮助。我认为，如果我们能做到这一点，或者用 Twitter 接近这一点，那将是一个巨大的成功。

微信在西方一直被誉为中国互联网最伟大的发明之一。腾讯对特斯拉的投资可能让马斯克了解了中国互联网巨头。但微信模式真的是美国想要的产品吗？

给马斯克留下深刻印象的确切微信功能也是该应用的批评来源。批评者说，多功能信使实际上建立了一个有围墙的花园，在那里电子商务交易只通过其支付应用进行，用户消费的信息要么在微信的基础设施内发布，要么由腾讯支持的第三方服务发布。在中国最近的反垄断运动开始拆除厚厚的墙之前，腾讯的死敌，如阿里巴巴和斗印(抖音在中国的妹妹)的链接在微信上是无法访问的。

超级应用可能会给用户带来便利，因为他们几乎不需要离开平台——这反过来有助于增加公司的收入——但这种模式可能会扼杀竞争，并排除用户的选择。

抛开这些担忧，马斯克到底能不能在美国复制微信的成功？

不太可能，至少目前状态的微信不会。这款即时通讯应用在中国特有的条件下蓬勃发展，不管是好是坏。在深入研究微信做对了什么之前，我们不要忘记，它作为社交消息应用的核心使它与社交媒体平台 Twitter 有着根本的不同。它是一款聊天应用，这意味着它的粘性很高。微信拥有超过 13 亿用户，是中国无处不在的信使，而人们在 Twitter 上主要是为了消费信息，而不是与现实生活中认识的人交谈。

微信的不公平优势

微信不是从零开始的。凭借 QQ，腾讯已经是中国的社交网络之王，QQ 是一种类似于 ICQ 的信使，在 PC 时代起飞。2011 年微信推出后不久，腾讯将 QQ 庞大的社交图谱开放给微信，让人们可以选择在微信上添加 QQ 好友。如果马斯克从零开始打造 X，他可以尝试将用户从 Twitter 转移到新平台。但是，坚持微信的类比，它将在一个挤满 WhatsApp、Messenger、Telegram 和其他应用的市场中竞争。

微信是许多应用程序合二为一，消息传递是一个门户，人们可以通过它来发现它多年来增加的过多功能。其早期的杀手级功能之一是名为公众账号的内容发布功能。2012 年公众账号推出时，微博和博客平台是中国网民分享心声的两个平台。前者像 Twitter 一样有 140 个字符的限制，而博客主要在中国精英中流行。基于一个日常聊天应用，公众账号很快吸引了从经济学家到想要传播思想的小企业主的所有人。

到 2021 年初，3.6 亿用户通过公众账号阅读文章。任何需要生产内容的机构都在微信上，从国家媒体到时尚品牌。美国的在线媒体格局更加多样化。人们在新闻应用上阅读新闻，在 LinkedIn 上寻找思想领袖，通过博客了解品牌故事。中国的大多数企业可能没有网站，但他们可能有一个微信公众号。

随着时间的推移，公共账户已经演变成一种类似于 Shopify 的企业数字基础设施。随着 2013 年微信支付的推出，这成为可能。虽然美国在过去十年里一直在改善磁卡交易，但中国从未广泛采用信用卡，而是直接从现金支付转向使用二维码的移动支付。

微信支付(WeChat Pay)迅速吸引了大量用户，成为一些热门应用的默认支付选项，包括打车新贵滴滴和送餐平台美团——这两家公司都得到了腾讯的支持，腾讯是全球最多产的企业投资者之一。如果马斯克按照微信支付的剧本开始一个新的支付解决方案，他必须与其他互联网巨头结盟，以推动其采用。

随着时间的推移，微信作为电子商务支柱的角色只会变得更加无所不能。2017 年，它开始让开发者构建在微信内运行的精简应用。过去通过静态公共账户页面兜售产品的企业现在可以运营基于微信的商店，这些商店拥有电子商务应用程序的所有基本功能。社交商务初创公司拼多多(Pinduoduo)凭借与微信社交功能的无缝集成，作为一款迷你应用脱颖而出，在半个世纪内成长为阿里巴巴(Alibaba)的竞争对手。想象一下，你可以在 WhatsApp 上浏览亚马逊，与你的联系人分享产品页面，甚至不用离开 messenger 就可以购物。

在把自己变成中国移动互联网的万维网之后，微信在自己内部建立了一个应用生态系统。亚马逊、优步、脸书和 Square 等科技企业不太可能想加入一个假想的 X super 应用，因为它们已经在自己的领域占据了一席之地，并控制着用户数据。按照微信的模式，马斯克本质上将与拥有数亿用户和自己支付产品的老牌科技巨头对抗。

特斯拉的边缘

马斯克可以利用腾讯梦寐以求的优势，而不是效仿微信。包括腾讯在内的大多数中国顶级互联网公司都投资了或者与一家电动汽车制造商合作。虽然他们的一些投资只是金融投资，但其他人押注于智能汽车成为娱乐、购物甚至信息消费的主要门户的未来——他们希望参与其中。

事实上，腾讯已经与其投资公司 Nio 合作，创建了一个为驾车人士量身定制的新闻节目。Nio 可以轻松地将一系列其他第三方应用程序，如视频流媒体服务和语音助手，插入其车载操作系统。特斯拉也可以。

特斯拉已经具备了这样做的基础设施。正如我的同事柯尔斯顿写的:

特斯拉对第三方开发者开放。特斯拉 API 在技术上是私有的。但它是存在的，允许特斯拉自己的第一方应用程序与汽车通信，以做一些事情，如读取电池充电状态和锁门。通过逆向工程，第三方应用程序可以直接与 API 通信。

微信的成就令人鼓舞，但它可能不是美国公众想要的，其庞大的在线消费帝国在中国之外不容易复制。马斯克可以利用特斯拉的应用程序，并将其融入他假设的 X 项目中，从微信如何与第三方开发者形成一个充满活力的应用内生态系统中获得启示，但即使特斯拉在电动汽车领域占据主导地位，并且如果 Twitter 交易成功，这仍然不能构成一个“一切”的应用程序。
