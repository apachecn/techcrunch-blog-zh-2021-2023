愚人节快乐！

图片来源: Formlabs

我不知道你们怎么想，但我已经筋疲力尽了，在选举、流行病、入侵和我最喜欢的乐队的鼓手最近去世之间，我在过去几年里失去了至少 95%的幽默感。尽管如此，科技创业公司每年都试图捉弄我们这个永远热爱上帝的人。以下是今年五大最不令人生厌的愚人节笑话。

  • 3D 打印宠儿 Formlabs 宣布将推出一款 2D 打印机 。鉴于我从 Formlabs 得到的早期打印机的极端混合结果，我会犹豫是否订购一台，但让我们面对它；如果他们推出了 2D 打印机，它可能会比这些年来我桌子上的许多其他垃圾要好，所以谁知道呢。我 99%确定这是个玩笑，不幸的。
  • Twitter 控制了它的用户群，声称它正在开发一个编辑按钮 。自从我们开始在网上发微博以来，这一直是这个平台最受欢迎的功能，每个人都知道这可能不会发生。(此外，这是个糟糕的主意。)但是是啊。激怒群众的方法！
  • 今天早上，听众朋友们开了一个微妙的愚人节玩笑，100%打动了我。面向乐迷的 Wordle knock-off 超级好玩；我对今天不能玩的愤怒真的让我失望，让我哭，伤害了我。
  • tvTV 推出专为 Apple TV 制作的电视 。我特别欣赏该公司为一款在很多方面都毫无意义的产品所做的渲染，我担心它可能会创造一个虫洞，把我们都吸进另一个世界，那里有 2D 打印机，有 Twitter 编辑按钮，Apple TV 变成了烤面包机的墨盒。
  • 最后， 一个愚蠢的啤酒订阅网站拉了一个愚蠢的噱头 通过只能被描述为愚蠢的偷梁换柱的愚蠢来销售其愚蠢的产品。我希望他们愚蠢的营销团队和愚蠢的主管们明白，你不能欺骗别人，然后逍遥法外。他们说，所有的关注都是好的关注，但这是我的愚蠢的热门话题:那是愚蠢的。让我们不要再做那种愚蠢的事情，不要把你的钱给愚蠢的公司。</肥皂盒>
