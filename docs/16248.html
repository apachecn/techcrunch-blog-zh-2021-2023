微软将 OpenAI 的 DALL-E 图像创建器带到了新的 Bing | TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/21/microsoft-brings-openais-dall-e-image-creator-to-the-new-bing/

微软今天宣布其新的人工智能 Bing 将允许用户使用 Bing 聊天生成图片。这项新功能由 OpenAI 的生成图像生成器 DALL-E 提供支持。该公司没有说它在这里使用的是哪个版本的 DALL-E，只是说它使用的是“最新的 DALL-E 型号”。

这一新功能被称为“Bing 图像创建器”，现在(慢慢地)在 Bing 预览版中向用户推出，并且只能通过 Bing 的创意模式使用。将来会发展到 Bing 的平衡和精确模式。新的图像生成器也将出现在 Edge 侧边栏中。

图片来源:微软

正确的提示将生成现在熟悉的四幅高分辨率 DALL-E 图像的正方形。不过有一个主要的区别:在左下角会有一个小小的必应标志。

早期的 Bing AI 版本缺少一些护栏，但微软很快修复了这些护栏。该公司显然希望通过此次发布来避免这些问题。

“我们整合了 OpenAI 的安全措施，并为 Image Creator 增加了额外的保护，”微软解释道。“例如，我们实施了旨在限制有害或不安全图像生成的控制措施。当我们的系统检测到一个提示可能会生成一个潜在有害的图像时，它会阻止该提示并警告用户。”

图片来源:微软

微软表示，它将继续迭代这个新的图像生成器。具体来说，该公司强调，它正在努力优化这项新功能在多回合聊天中的工作方式。“我们仍然相信，将这些技术推向市场的最佳方式是在公开的环境中仔细测试它们，每个人都可以提供反馈，”该公司的博客文章指出。

图片来源:微软

除了新的图像生成器，Bing 还增加了两个新的搜索功能:视觉故事和知识卡片 2.0。Bing 之前版本的知识卡类似于谷歌的知识面板，但现在它们增加了人工智能生成的信息图和更多互动元素，如图表、图形和时间线。

与此同时，视觉故事更类似于人工智能生成的 Instagram 故事。据我所知，自从它们在我的搜索中频繁出现以来，微软最近几周已经测试了很多次。这些故事配有人工智能生成的音频旁白，以及取决于主题的视频。有时它们会是最靠前的搜索结果，但也可能出现在知识卡片中。例如，当搜索“俄勒冈州波特兰”时，Bing 会突出显示“波特兰人如何在城市中找到真菌”和“波特兰:美国最古怪的城市”等故事可悲的是，似乎没有直接链接到这些故事的方法。
