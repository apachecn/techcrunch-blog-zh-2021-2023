将社交媒体受众带入网络的 5 大 NFT 趋势 3 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/01/06/5-nft-trends-that-will-bring-social-media-audiences-into-web3/

Prerna Gupta 推出了移动娱乐应用程序，如 Hooked 和 Smule，覆盖了超过 10 亿人。她目前正在 NFT 空间从事一项秘密项目。

随着 Twitter 上的辩论在 T1 激烈进行，大多数科技创始人和风投现在已经选择了一方:Web 2.0 或 web3。

web3 的支持者认为它是互联网的未来，在未来几年内，区块链的产品将完全取代 Web 2.0。

Web 2.0 的忠实拥护者说，这只是寻求盈利的加密公司的一堆炒作，区块链技术的范围从根本上说是有限的。

作为一个已经开发 Web 2.0 应用超过十年的创始人，我相信最激动人心的机会就在这两个领域的交汇处。

Web 2.0 和 web3 的合并将释放区块链真正的大众市场潜力。

看到投入 web3 的精力、人才和资源，不禁让人想起互联网的早期。对于我们这些在硅谷生活了足够长时间的人来说，相似之处是不可否认的。

除了这一次，增长率似乎呈指数增长。Web3 专家指出这些事实来支持 web3 是互联网的未来，Web 2.0 将很快消亡的论点。

但是互联网的早期和今天有一个重要的区别:在 Web 1.0 之前，没有互联网。我们都过着类似的生活。当互联网出现时，它正在与一种相对无聊的存在竞争。免费色情，聊天室，游戏，音乐，电子邮件，视频，世界上的信息触手可及…一个昏昏欲睡的周六百视达之旅。这从来都不是一场公平的战斗。

我花了 2000 美元买了一双我知道我永远不会穿的运动鞋，因为它们只作为 NFT 而存在。这比我之前花在踩扁脚趾的高跟鞋上的钱少吗？

Web3 正在进入一个不同的领域。今天的人们对数码产品有着强烈的吸引力。你不会让普通消费者停止滚动抖音，因为他们想帮助开创一个去中心化的时代，支持创造者经济或对抗通货膨胀。他们不在乎。他们买不起 BAYC 或者 cryptopunk 。

普通互联网用户关心的是他们如何在社交媒体上出现和展示自己。那是桥。这座桥的关键是 NFTs。

以下是 NFT 的 5 个趋势，它们将把社交媒体受众一起带入 web3:

NFT 验证

怀疑论者很快指出 NFT 是愚蠢的，因为您可以简单地右键单击并保存底层文件。这是暂时的问题。在不久的将来，所有主要的社交平台都将实现 NFT 验证，允许您连接您的钱包，并在您的个人资料中显示您验证的 NFT。与此同时，指纹识别技术将允许平台轻松检测被盗文件并将其删除。

一旦抖音用户可以在他们的帖子中显示和使用他们独特的、经过验证的 NFT，游戏就完全改变了，因为购买和广播 NFT 的社会价值呈指数级增长。

侧链

随着像 Polygon 这样的侧链越来越普遍，NFT 的价格将会下降。接近零的汽油费也将使开发者能够在 NFT 中构建更多的交互性和可组合性，使它们本质上更具社会性。

想象一下区块链上的神奇宝贝 Go，每个神奇宝贝都是可以交易或出售的 NFT。你获得的每一个神奇宝贝都有独特的特质，你解锁的基于位置的成就会对它的发展产生独特的影响。随着神奇宝贝等级的提升，它的能力会在链上更新。随着您在游戏中的进展，您将获得加密令牌，这些令牌在游戏之外也有价值。好玩！

音乐

2021 年的 NFT 热主要集中在视觉艺术上，但下一个前沿领域，也是我认为最终会更大的领域，是音乐。无论是收藏实物艺术还是 NFTs，驱动一个收藏家行为的冲动都是一样的:向同行表达自己艺术品味的欲望；表达个人或群体身份的愿望；在某些情况下，还有盈利的愿望。

虽然最初的 NFT 热背后有大量的投机狂热，但我认为围绕自我表达和身份的冲动是艺术收藏更根本的驱动力。人们表达品味和身份的最普遍的方式之一就是通过音乐。

抖音最初是一个音乐视频应用，如今音乐曲目仍然是该平台上共享视频的核心组成部分。但是今天可用的音乐是普通的。每个人都可以访问相同的歌曲。

想象一下，你最喜欢的艺术家发布了一首独特的 60 秒音乐曲目，数量有限，以 NFTs 的形式提供。你买了一个，用它做了一个很酷的抖音。它会像病毒一样传播。数百万人突然希望这首歌成为他们自己的抖音。但是哪里都只有 100 本。你每天都会收到想从你这里购买这首歌的人的报价，你可以决定是想把它翻过来还是留给自己。好玩！

衣服

在 Web 2.0 时代的开端，当我 20 多岁的时候，我会把几乎所有的月可支配收入都花在时尚上。花哨的牛仔裤、高跟鞋、太阳镜和手袋——我对漂亮衣服的欲望是无法满足的。

但是在过去的十年里，随着我大部分的社交活动转移到了网上，我在服装上的花费减少了。这些天，除了运动装备，我真正想要的是一把五颜六色的变焦上衣。买花哨的衣服已经失去了乐趣。

但是前几天，我做了一件令人惊讶的事。我花了 2000 美元买了一双我知道永远不会穿的运动鞋。我不能穿它们，因为它们只作为 NFT 而存在。对我来说，很难向一个没有掉进兔子洞的人解释为什么我会买下它，我意识到这可能看起来很疯狂。但这比我之前花在踩脚趾的高跟鞋上的钱更疯狂吗？

可穿戴设备将迎来一个数字商品大规模消费的时代，社交媒体观众将利用这些商品来打扮和打扮自己的数字自我。尽管 NFT 领域大多数现有的可穿戴设备活动都与游戏和虚拟角色有关，但将有助于大量受众转移到元宇宙的杀手级用例将是 AR 滤镜。

想象一下，金·卡戴珊(Kim Kardashian)推出了定制的 AR 滤镜——整容、丰唇、化妆、头发、连衣裙、珠宝——每一个都是独一无二的。如果你买了一个，你就可以在你的抖音视频里独家穿上这个样子，想穿多久就穿多久，然后当你觉得无聊的时候你可以卖掉它。把我加入你的白名单，金！

动态头像

好吧，我明白了。无论如何，你都无法想象为什么有人会把自己的头像改成一只丑陋的猿猴。我不得不承认，我有一只无聊的猩猩。但是我也不能把它作为我的个人资料照片。我选择了一个女人的世界。为什么我要把自己的专业头像换成一个算法生成的卡通形象，而这个卡通形象和我只有模糊的相似之处？很有趣。它让我表达了我在照片中难以捕捉到的个性的一面。

但是我的个人资料照片已经开始有点过时了。我希望它不是如此…静态的。

活力、个性化和运动正在成为 NFT 的化身。想象你自拍，人工智能生成一个独特的定制 3D 化身，看起来像你的理想版本。像 Bitmoji，但是更好。你可以更新角色的表情和姿势，服装和配饰；你可以让它跳舞；你可以让它用你自己的声音说话，只需输入。想想看，在你对自己的表现感觉良好的地方，创建吸引人的抖音帖子会容易得多。你可以花更少的时间来完善你的表演和拍摄技巧，花更多的时间在你想传达的信息上。

动态头像将是一个在社交媒体上民主化自我表达的机会，并将为更多的创作者观众提供发言权。

当脸书第一次开始的时候，有点傻，但是很有趣。当你每隔几天登录一次，你就会想起你真正朋友的生日，看到他们最新的劣质个人资料照片，然后戳他们。然后是 feed，手机和算法，Instagram 模型，拿铁和完美过滤的生活，在这个过程中的某个地方，乐趣被一种社会奴役所包容。我们不断滚动浏览我们几乎不认识的人的帖子，这些帖子让我们感觉很糟糕，而我们不断张贴令人讨厌的东西作为回报。这肯定不是一切的结局吧？

当扎克伯格公布他对 Meta 的愿景时，你可以感受到集体的眼球转动。你已经用社交媒体毁了我们的生活，我们想尖叫，现在你要杀死我们最后的身体联系的残余，把我们变成生活在矩阵中的蔬菜。

但问题是，无论利弊如何，我们已经在不可阻挡地向元宇宙前进。我们越来越多地过着虚拟的生活。web3 使虚拟世界中的体验更加丰富。一个不那么空洞，更有人情味的世界。

我期待着未来，NFTs 和 web3 的所有其他承诺一起，帮助我们创造一个虚拟现实，在这个虚拟现实中，人类的互动再次变得有趣。
