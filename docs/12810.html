稳定性人工智能在 AWS 上翻倍

原文：https://web.archive.org/web/https://techcrunch.com/2022/11/30/stability-ai-doubles-down-on-aws/

微软可能长期以来一直将 OpenAI 作为其可靠的合作伙伴(在其大规模投资之后)，但 AWS 今天宣布，稳定 AI 是生成人工智能领域最热门的新贵之一，稳定扩散背后的公司，正在加倍投资其云，使其成为“为图像、语言、音频、视频和 3D 内容生成构建和扩展其人工智能模型的首选云提供商”

此外，Stability AI 还将与 AWS 合作，将其开源工具和模型提供给更多的学生、研究人员、创业公司和企业(这听起来有点像微软和 OpenAI 宣布合作时所说的话)。

Stability AI 最近宣布一轮 1.01 亿美元的融资，估值超过 10 亿美元，已经在 AWS 云中使用数千个 Nvidia GPUs 来训练其模型。现在，两家公司正在正式建立这种关系，Stability AI 计划使用 AWS 的 SageMaker ML 平台，在其底层基础设施服务的基础上使用 GPU 和 AWS 自己的 Trainium 芯片。

通常情况下，这些交易还附带优先定价和其他优惠，尽管 AWS 和 Stability 都没有在今天的公告中提到这些。

“在 Stability AI，我们的使命是建立基础，通过人工智能激活人类的潜力，”Stability AI 的创始人兼首席执行官艾玛德·莫斯塔克说。“AWS 在跨模式扩展我们的开源基金会模型方面发挥了不可或缺的作用。我们很高兴在 Amazon SageMaker 上运行这些模型，让数万名开发人员和数百万用户能够通过一套强大的工具来利用人工智能的力量。我们期待看到开发人员利用集体智慧和增强技术构建、客户设计和实现的令人惊叹的东西。”

像最近推出 2.0 版本的 Stable Diffusion 这样的模型并不是没有争议的，因为它们生成成人内容的能力，Stable Diffusion 2.0 会自动过滤掉一些内容，或者因为这些模型经常根据艺术家的图像进行训练，这些艺术家没有明确选择他们的作品用于训练这些模型。然而，毫无疑问，这些天来，稳定性人工智能正在从 OpenAI 那里接管相当多的注意力，open AI 长期以来一直对如何公开其模型采取更谨慎的方法。Stability AI 的开源方法，至少目前看来，似乎正在吸引更多的开发者，并推动这一领域的创新。
