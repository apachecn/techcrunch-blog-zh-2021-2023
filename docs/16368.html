前 Stripe 工程师为 Beam 融资 400 万美元，Beam 是一家金融科技初创公司，旨在帮助承包商更快获得报酬

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/23/former-stripe-engineer-raises-4m-for-beam-a-fintech-startup-out-to-help-contractors-get-paid-faster/

成立五个月的初创公司 Beam 旨在更方便地帮助总承包商向分包商付款，并让他们自己获得报酬，该公司在由 Accel 牵头的种子资金轮中筹集了 400 万美元。

这家初创公司的创始人和主要投资者之前都曾在支付巨头 Stripe 工作多年。在 2022 年 10 月开始 光束 之前，Adam Eagle已经在 fintech 公司担任了五年半的软件工程师，为 Stripe 计费、开具发票、商务和支付构建了 核心 API 和基础设施。Amy Saper，本轮领投人，在 2019 年作为合伙人加入 Accel 之前，帮助 建立并壮大 Stripe 的产品营销团队。

当 Saper 开发 Stripe 计费和发票产品时，他与 Eagle 一起工作，作为他的产品营销对手，亲眼目睹了他的技术能力。因此，当该公司开始筹集资金时，Accel 挺身而出，牵头进行了融资，融资于今年早些时候结束。

为(字面上的)建筑者建造

建筑行业的一个普遍现象是，大多数承包商被迫在拿到钱之前完成项目，常常不得不自掏腰包支付材料和人工费用。对于较小的业务，没有足够的资金流入可能会带来压力，通过电子表格和有时使用纸质支票来了解谁欠谁的过程可能非常耗时和乏味。

总部位于旧金山的 Beam 致力于帮助中小型普通住宅承包商节省时间和金钱，为他们提供一种在一个地方“简化”付款、发票和收据的方法。它还将 ACH 支付直接整合到其应用程序中。Eagle 说，承包商只需“几分钟”就能安装到 Beam 上，一旦安装完毕，他们就可以立即开始付款。

从支付转向建筑技术似乎是一个巨大的转变，但对 Eagle 来说，这几乎是不可避免的。在开始写代码之前，Eagle 说他一直对建筑和住房“超级感兴趣”。

他在一次采访中告诉 TechCrunch:“多年来，我一直看到关于住房危机和我们老化的基础设施的头条新闻，我决定我真的想从事一些涉及建筑、基础设施和物理世界的工作。”。“这很大程度上是由改善生活质量和城市质量的愿望所驱动的。”

Eagle 在研究这个空间后得出结论，建筑企业的财务运作“非常繁重”。他还意识到，许多这样的企业是中小企业或小型家庭经营的组织，只有几个员工和最少的资源。因此，所有者要么不得不花费大量时间人工处理发票和付款，要么花钱雇佣簿记员或办公室经理。

Eagle 说:“当你开始考虑这些建筑企业的财务状况时，你会意识到他们的处境非常糟糕——很多时候他们都在等待从客户那里获得报酬，或者他们不得不向分包商支付大笔款项。”。“或者，如果你是一个分包商，你必须为材料支付大量资金，你必须预先支付工资，然后你可能要到完成劳动后 30、60 或 90 天才能拿到工资。”

为了帮助承包商降低成本，Beam 声称，与 PayPal 或 Zelle 相比，它收取的交易费用更低，PayPal 或 Zelle 也对交易金额设置了上限。

伊格尔说:“这是许多建筑企业仍然使用纸质支票的众多原因之一。“因为每一笔支付，他们都会因为支付网络收取的高额费用而损失 15 美元到数百美元不等。”

认识到其客户中很大一部分是西班牙语母语者， Beam 构建了其产品的第一个版本 以包括针对非西班牙语母语者的西班牙语本地化。

Briq 是另一个专注于建筑行业的金融科技公司的例子。Eagle 认为，Beam 的不同之处在于，Briq 更专注于为大型企业构建自动化计费流程的工具。

从长远来看，Beam 的目标是简化企业业务的支付。

Susa Ventures 和 Wischoff Ventures 也参与了种子资金，此外还有一批天使投资者，包括几家大中型金融科技和建筑企业的创始人和高管。

对于 Accel 的 Saper 来说，Beam 试图解决的问题导致了这个国家的住房短缺。

“我 太难建造了，”她在博客中写道。“作为金融科技相关公司(包括 Unit、Braintree 和 Venmo)的深度投资者，我们看到支付和发票领域的创新触及了许多其他行业。然而，与建筑相关的账单还有待复兴。”

Beam，Saper 补充说，通过其“易于使用的计费和合规平台”来解决这个问题，该平台将“Beam 网络中的各方聚集在一起，以实现无缝和及时的支付。”
