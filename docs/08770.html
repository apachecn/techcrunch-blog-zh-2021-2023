亚马逊推出 AWS 技能构建者培训订阅，起价为每月 29 美元

亚马逊的 AWS 正在为寻求加强云计算技能的个人和团队推出高级付费订阅。

亚马逊在去年 11 月宣布了一批免费的云技能培训产品，其中之一是 AWS 技能构建器，这是一个在线学习中心，提供数百门免费的云计算课程。通过 AWS Skill Builder 订阅，这家科技巨头现在正寻求通过每月订阅来赚钱，在基本免费培训计划的基础上引入一系列额外的功能和服务。

个人订阅的价格为每月 29 美元或每年 299 美元，他们为那些希望通过官方 AWS 认证计划的人提供三次模拟考试，该公司在过去十年中一直提供这种考试。除此之外，还为用户提供了“构建者实验室”等工具，这些工具基本上是一些实用的指导练习，涵盖了一些更常见的云环境——用户在练习时会获得一个沙盒 AWS 帐户。

与此同时，团队订阅费为每位学员每年 449 美元，尽管有大量折扣。该计划包括一系列以企业为中心的功能，如分配培训练习、进度报告的能力以及与公司的单点登录(SSO)提供商的集成。

提高技能

值得注意的是，所谓的“三大”云公司多年来都推出了各种各样的培训课程，试图将开发者和公司引入各自的生态系统。在很大程度上，它们基本上是免费的(尽管有许多通过第三方合作提供的高级课程)或者一次性收费，但我们开始看到云公司向他们的现有客户收取经常性费用的趋势，这就是亚马逊最新计划的全部内容。通过每月订阅，这可能会鼓励用户对课程赋予更大的价值，并确保他们跟上他们的学习——同时给 AWS 带来少量的收入增长。

去年，谷歌推出了谷歌云技能提升，目标是培训 4000 万人如何使用其云平台的各个方面。这是由它在 2016 年收购 Qwiklabs 的推动的，与亚马逊的新努力类似，根据用户是否希望承诺每月或每年订阅，它收取29 美元或 299 美元。还有一个团队订阅选项，显然可以根据要求提供。

如果没有别的，所有这些都有助于凸显云在大型科技公司的命运中变得多么重要。尽管全球经济环境不佳，亏损 20 亿美元，亚马逊上周公布云业务持续强劲增长，AWS 收入同比增长 33%，至 197.4 亿美元。

但是，虽然云支出明显上升，但 IT 技能差距并没有缩小，解决这一差距是亚马逊希望其客户愿意支付的。
