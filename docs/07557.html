亚马逊推出类似 GitHub Copilot 的人工智能配对编程工具 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/06/23/amazon-launches-codewhisperer-its-ai-pair-programming-tool/amp/

图片来源: TechCrunch

在其 re:Mars 会议上，亚马逊今天宣布推出 CodeWhisperer，这是一款类似于 GitHub 的 Copilot 的人工智能配对编程工具，可以根据一条评论或几个按键自动完成全部功能。该公司在数十亿行公开可用的开源代码和自己的代码库，以及公共论坛上公开可用的文档和代码上训练了该系统，该系统目前支持 Java、JavaScript 和 Python。

它现在可以作为 AWS IDE 工具包的一部分在 preview 中使用，这意味着开发人员可以立即在他们喜欢的 IDE 中使用它，包括 Visual Studio 代码、IntelliJ IDEA、PyCharm、WebStorm 和亚马逊自己的 AWS Cloud 9。对 AWS Lambda 控制台的支持也即将推出。

在今天的声明之前，亚马逊负责人工智能服务的副总裁 Vasi Philomin 强调，该公司不仅仅是为了提供 Copilot 的副本而创造了这个。他指出，凭借其人工智能代码审查员和性能分析器 CodeGuru 和用于发现操作问题的工具 DevOps Guru ，该公司在几年前就为今天的发布奠定了基础。

图片来源:亚马逊

在内部，亚马逊一直只在少数开发者中测试这项服务——主要是为了保密。

图片来源:亚马逊

该公司指出，系统会持续检查你的代码和注释，甚至会考虑你自己的编码风格和变量名。使用这些上下文信息——以及光标所在的位置——它将生成自己的定制代码片段。

值得注意的是，CodeWhisperer 做的一些事情与 Copilot 之类的不同。首先，虽然系统生成的大部分代码都是新颖的，但每当它生成与训练数据中的现有片段接近的代码时，它都会注意到这一点，并突出显示原始功能的许可证。然后由开发人员决定是否使用它。这应该会减轻一些(虽然可能不是全部)使用这样的工具可能带来的版权问题。

图片来源:亚马逊

Philomin 强调的另一个因素是安全性。基于亚马逊自己管理大型代码库和在出现问题后进行汇报的经验(使用其形式化的“纠错”过程)，以及其与 CodeGuru 的经验，CodeWhisperer 将扫描代码以寻找潜在的安全问题。

“安全性在 AWS 中总是很重要，所以我们想确保我们生成的代码是安全的，”Philomin 说。“很明显，我们已经生成了代码，开发人员可以对其进行修改——因此 CodeWhisperer 可以说:对当前源文件进行扫描。它将……扫描并……告诉您生成或更改的代码中的任何问题和安全漏洞。”

图片来源:亚马逊

Philomin 还强调，该团队努力确保 CodeWhisperer 生成的代码也不会生成任何有偏见的代码。不过，当它出现时，该团队已经实现了过滤器来自动删除这些代码。

实际的用户/开发人员体验非常简单。开发人员可以在不同的代码建议之间进行选择(在我看到的演示中，通常至少有两个选项)。它可以自动完成评论，并根据这些评论，它会建议功能。一个简单的例子是类似于#看看一个数是否是质数。

虽然该系统对于 AWS 生态系统之外的开发人员来说工作得很好，但 Philomin 指出，该团队非常注意确保它对于想要使用它来访问 AWS 服务的开发人员来说工作得很好(想想#创建一个 S3 桶)。

“当我们说它对 AWS API 有一流的支持时，这只是额外的东西。就我们所有的测量和测试而言，我们发现它的效果非常好，它将是最先进的，”他说。由于开发者现在可以测试这项服务，我们可能很快就会听到它的效果如何，以及它与 Copilot 等项目相比如何。不过，如果有什么不同的话，在这个领域有更多的选择是非常好的(肯定有人会编写一个项目，自动提供来自多对编程人工智能的建议)。
