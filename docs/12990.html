随着 ZeniMax QA 测试人员组织| TechCrunch，微软可能获得其第一个官方联盟

原文：https://web.archive.org/web/https://techcrunch.com/2022/12/05/zenimax-union-microsoft-gaming/

视频游戏公司 ZeniMax Media 的大约 300 名质量保证(QA)测试人员正在寻求在他们工作室的母公司微软成立有史以来第一个工会。ZeniMax 包括 Bethesda Softworks 和 id Software 等子公司，生产《上古卷轴》、《毁灭战士》和《辐射》等系列产品。

视频游戏行业的工会组织一直在增加，尤其是在质量保证人员中。动视暴雪的 QA 测试人员已经通过通信工人联盟(CWA)在渡鸦软件和暴雪奥尔巴尼成功组建了工会，这也将代表 ZeniMax 的工会。但是当动视暴雪试图处处阻挠工会组织的时候，微软在 6 月承诺不会阻碍员工组织工会。到目前为止，微软一直信守承诺(当然，如果微软以 690 亿美元收购动视暴雪的交易结束，这个承诺将变得复杂)。

周五，工会组织者开放了一个门户网站，ZeniMax QA 测试人员可以在这个月底投票支持或反对工会。如果超过一半的合格工人投赞成票，那么——如果该公司信守承诺——微软将承认该工会，而无需将其提交给国家劳工关系委员会(NLRB)的官方官僚投票。

id Software 的高级质量保证测试员 Zachary Armstrong 告诉 TechCrunch，该部门正在组织起来争取更高的工资。

阿姆斯特朗告诉 TechCrunch:“现在，我们的工资没有反映出我们给公司带来的尊重和价值。”"这是所有视频游戏 QA 都有的情况."

游戏行业的其他主要联盟力量也来自于 QA 测试人员，这并不是巧合。

“当涉及到游戏开发时，QA 测试人员总是被放在图腾柱的底部，以至于我们甚至不被认为是游戏开发者。”阿姆斯特朗说。QA 人员严格测试视频游戏的所有方面，以识别和解决影响用户体验的问题。“这反映在我们的薪酬中，也反映在我们的工作中，特别是在 crunch 方面。”

在一个主要游戏发布之前，QA 测试人员有时被期望工作不可持续的时间，这被称为“紧缩”在宣布他们加入工会的意图之前，Raven Software 的第一个美国主要游戏工会举行罢工，抗议影响 12 名承包商的裁员——在这些合同被终止之前，QA 测试人员已经连续加班五周了。

阿姆斯特朗预计，ZeniMax QA 测试人员有必要的票数在 12 月底赢得他们的工会。目前，阿姆斯特朗乐观地认为，微软将继续让工人们不受干扰地组织起来。

阿姆斯特朗谈到微软的工会政策时说:“这使得联系那些更关心报复和支持工会的后果的人变得容易得多。”。“我们知道在其他电影公司要困难得多，事实上我们没有受到那样程度的抵制，这对我们来说是一个巨大的安慰。”

如果工会投票通过，ZeniMax 的 QA 测试人员将成立微软的第一个工会，也是迄今为止美国最大的视频游戏工会。

“ZeniMax 员工最近的组织努力，以及微软和 ZeniMax 对此的中立态度，是我们劳工原则付诸行动的一个例子，”微软发言人在一份声明中说。“我们仍然致力于为员工提供自由、公平地选择工作场所代表的机会。”
