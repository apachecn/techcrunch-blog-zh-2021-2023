Solana 的联合创始人说 NFT 有“50 种不同的用例”,今年可以搭载数百万人

原文：https://web.archive.org/web/https://techcrunch.com/2022/08/05/solana-co-founder-says-nfts-have-50-different-use-cases-that-can-onboard-millions-this-year/

索拉纳的联合创始人说，NFT 有“50 种不同的用例”，今年可以搭载数百万人

感觉就像昨天一样，NFT 热潮吸引了加密社区的注意，甚至在 web3 世界之外也掀起了波澜。但一年左右的时间过去了，对 NFT 的炒作已经有所降温。但这并没有阻止加密领域的一些人对不可替代的令牌保持乐观。

虽然很多行业的焦点和努力都集中在像 Bored Ape Yacht Club 或 crypto 朋克这样的档案图片 NFT 上，但除了图像以外，提供实用功能的 NFT 越来越受欢迎。

Solana 的联合创始人 Raj Gokal 本周告诉 TechCrunch，“我认为在 NFTs 中，一切都只是刚刚触及表面。”“我认为 NFT 有 50 种不同的用例，它们似乎可以合并成一个。我认为我们希望大多数[加密]项目都使用 NFTs。”

新的 NFT 用例从游戏延伸到音乐，Gokal 看到了足够多的专注于这些领域的创业活动，从而对更广泛的数字资产感到乐观。

“有一波新的 NFT 产品[即将到来]，可能会在第三季度和第四季度搭载数百万用户，”Gokal 说。他补充说，与其只想着发展索拉纳生态系统，不如通过开发者和创造者来观察它的发展，这更令人兴奋。
