ChatGPT 来到微软以企业为中心的 Azure 托管服务| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/09/chatgpt-comes-to-microsofts-enterprise-focused-azure-powered-managed-service/

ChatGPT 是 OpenAI 的病毒式人工智能聊天机器人技术，现在可以在一个更加企业友好的包中使用。

微软今天宣布chat GPT 通常可以通过 Azure OpenAI 服务获得，这是该公司完全管理的、以企业为中心的服务，旨在让企业访问 OpenAI 的技术，并增加治理和合规功能。必须已经是“微软管理的客户和合作伙伴”的客户可以在这里申请特殊访问。

ChatGPT 加入了已经通过 Azure OpenAI 服务提供服务的其他 OpenAI 开发的系统的范围，包括文本生成 GPT-3.5 ，代码生成 Codex 和图像生成 DALL-E 2 。微软与 OpenAI 有着密切的工作关系，已经向这家初创公司投资了数十亿美元，并签署了独家协议，将 OpenAI 的人工智能研究商业化。

显然，这是有回报的。微软今天表示，超过 1000 个品牌注册了 Azure OpenAI 服务。

ChatGPT 的 Azure OpenAI 服务风格是定价为每 1000 个令牌 0.002 美元，约合 750 个单词，chat GPT 的使用计费将于 3 月 13 日开始。(ChatGPT 消耗表示为一系列记号的原始文本；例如，单词“fan”、“tas”和“tic”将被拆分成记号。)这与 3 月 1 日推出的以开发者为中心的 ChatGPT API 价格相同。

Eric Boyd，微软人工智能平台的 CVP，在一篇博客文章中提到,Azure open AI 服务的客户可以配置 ChatGPT 的响应以“与他们的组织保持一致”这是一个重要的标注，因为用户已经能够促使 ChatGPT 以种族主义和性别歧视的方式回答问题，捏造事实而不透露它正在这样做，并绕过某些高级别的安全措施。

Boyd 写道:“开发人员可以将定制的[ChatGPT-powered]体验直接集成到他们自己的应用程序中，包括增强现有的机器人来处理意外问题，概括呼叫中心的对话以实现更快的客户支持解决方案，创建带有个性化报价的新广告文案，自动化索赔处理等。”

尽管 ChatGPT 存在更多的问题，但它的发展速度很快。OpenAI 尚未提供 API 使用数据，但截至 12 月，面向消费者的 ChatGPT 应用程序估计每月活跃用户超过 1 亿。包括 Snap 和 Quizlet 在内的品牌已经将它集成到或计划很快集成到各自的平台中。例如，Instacart 表示，它将使用 ChatGPT 创建 Ask Instacart，这是一个允许 Instacart 客户根据该公司零售合作伙伴的产品数据获得“可购买”答案的工具。Office Depot 正在构建一个基于 ChatGPT 的聊天机器人来支持它的几个内部业务部门，特别是人力资源部门。

特别是企业中的人们似乎已经接受了它，这对 Azure OpenAI 服务的推出是一个好兆头。根据 Fishbowl 的一份报告，该报告调查了来自美国银行、亚马逊和麦肯锡等公司的专业人士，近 30%的人说他们使用 ChatGPT 来帮助完成与工作相关的任务，如营销和广告、编程、咨询、会计和教学。

不过，一些雇主对此表示反对。据报道，出于生产率和合规性的考虑，一些公司对 ChatGPT 的使用进行了限制，富国银行是其中之一。
