Moovit 与智能拐杖公司 WeWALK 合作，为视障人士提高移动能力

英特尔旗下的 Moovit 是一家移动即服务(MaaS)提供商，拥有一款全球流行的旅行规划应用程序，它与智能手杖公司 WeWALK 合作，帮助视障人士更安全、更高效地到达目的地。

WeWALK 的 R&D 负责人让·马克·费加利(Jean Marc Feghali)表示，WeWALK 的应用程序现在将与 Moovit 的 Transit API 集成，后者将官方的当地交通机构信息与众包信息相结合，以获得每次旅程的最佳路线，以便视障用户可以在公共交通上安全导航。

此次合作之前，Moovit 与微移动公司进行了几次整合，如 Lime 、 Bird 以及最近的 Superpedestrian ，在应用程序中展示他们共享的电动滑板车和自行车。Moovit 还推出了新的业务部门，例如为交通沙漠中的乘客提供的按需交通，以及与另一家英特尔公司 Mobileye 的合作，提供自主乘车服务。

目前在 3400 个城市运营的 Moovit 似乎正试图无处不在，迎合所有人的需求，这肯定包括视障人士。专门面向残疾人群体的交通技术并不多，但一些有益的创新开始涌现。例如，本田孵化的 Ashirase 最近推出了鞋内导航系统，有点类似于 WeWALK 的拐杖。

虽然拐杖本身通过轴提供模拟地面障碍物检测，但连接到它的智能设备通过超声波传感器提供上半身障碍物检测。通过手杖内置的振动电机产生的触觉反馈，用户可以警告在不同距离检测到的障碍物。

Feghali 告诉 TechCrunch:“事实上，WeWALK 可以做得更多，包括提供公交车站的方向。”“通过蓝牙，智能手杖连接到我们的 WeWALK 智能手机应用程序，我们认为这是目前最全面、最易于使用的视障导航应用程序之一。我们的应用程序集成了 Moovit 服务和我们定制的导航引擎和应用程序界面，以提供步行和公共交通导航以及城市探索功能。”

一旦用户在应用程序中输入他们的目的地并选择路线，手杖就会通过音频方向和低视力地图一步一步地指导用户的旅程，指引用户到达运输站，并在下一辆运输车辆到达时通知他们。该系统还会在乘客上车和到站时提醒用户，让他们知道自己到了正确的车站以及何时下车。

对用户来说，最好的部分是不要一只手放在手机上，另一只手放在拐杖上。这款手杖的手柄上有一个内置的触摸板，与应用程序相连，允许用户做出手势，以便他们可以与手机互动，找到自己的位置，查看公交时刻表和附近的交通站，并导航到目的地。

“例如，如果用户正在导航到伦敦帝国理工学院，智能手杖会宣布路线选项，并引导用户一步一步地通过每个阶段，”费加利说。“对于步行路线，WeWALK 会宣布，‘在白金汉宫路 12 点钟方向行驶 50 米，然后在 3 点钟方向右转，进入车站路。’在地铁站，WeWALK 会在用户到达时通知用户要搭乘的列车，并在用户需要下车之前警告他们。"

这一伙伴关系的时机与将于周五举行的国际残疾人日同步，希望在视力障碍者试图获得就业、教育和社会活动机会时，给予他们更多的自主权和行动自由。

Moovit 的首席增长和营销官 Yovav Meydad 在一份声明中说:“虽然盲人和弱视者比以往任何时候都更加独立，但通过公共交通出行仍然令人生畏，令人不知所措。”“通过我们的合作伙伴关系，我们旨在通过打破一些移动障碍，让人们获得更多的信心，让他们能够获得更多的机会。”
