Sylvera 获得 3200 万美元以建立对碳补偿评级的信任

原文：https://web.archive.org/web/https://techcrunch.com/2022/01/25/sylvera-series-a/?utm_campaign=Speedinvest%20News&utm_medium=email&utm_source=Revue%20newsletter

Sylvera 是一家总部位于英国的初创公司，它使用机器学习技术来分析各种视觉数据，如卫星图像和激光雷达，目的是提高碳抵消项目的问责制和可信度。去年 5 月，它在 T2 完成了一轮 580 万美元的种子融资，并完成了一笔 3200 万美元的 a 轮融资

这轮融资由 Index Ventures 和总部位于纽约的全球私募股权和风险投资公司 Insight Partners 共同牵头，Index Ventures 也是 Sylvera 的种子投资者。参与此轮投资的还有:Salesforce Ventures、LocalGlobe 和一些天使投资者。

这家成立于 2020 年的初创公司现在已经筹集了 3950 万美元的资金。

它表示，A 轮融资将用于推动进一步的业务扩张，包括进一步扩大团队和加强技术领导力。

扩展平台以包括所有补偿的全面覆盖也是可能的。

联合创始人兼首席执行官阿利斯特·弗里博士在一份声明中评论说:“市场是世界上应对气候变化最有力的工具之一。但我们需要可靠的数据来确定碳抵消的质量，以激励人们投资于实际上做得好的项目，并奖励做得好的项目开发者。

“这就是我们为自愿碳市场(VCM)建立最准确评级的原因。我们将利用这笔资金来扩大我们的覆盖范围，这样，通过我们的评级，企业可持续发展领导者、碳交易商和决策者在评估和投资碳项目时将有清晰的思路、信心和选择。这就是你如何将数十亿美元转移到碳减排、碳封存和碳去除上的。”

Sylvera 没有透露它的评级有多少客户——它通过一个网络应用程序和 API 提供——但表示，迄今为止，客户涵盖各行各业，包括达美航空、嘉吉、CBL(Xpansiv 市场)和贝恩公司等。

“我们的客户包括企业买家、交易商和交易所，”它补充道。“我们的客户通常是做出净零承诺的大型机构，他们是市场上碳信用的最大买家。”

虽然 Sylvera 的评级框架是专有的，但这家初创公司表示，它致力于公布如何进行项目评估的细节，以支持其成为领先的碳抵消数据可信来源的目标(重要的是，它本身不交易碳抵消，以避免直接冲突)。

它的口号是“独立、深入和最新的碳项目评估”——在一个充斥着虚假补偿和绿色清洗的空间里，这是一个听起来很有价值的目标——尽管这一主张需要对所涉及的方法、数据和算法进行独立、深入和最新的外部验证，才能真正获得信任。

“我们正在努力使 Sylvera 在未来两年内成为所有类型的碳抵消中最可信的真实来源，”当我们问及此事时，这家初创公司指出，并补充说:“要成为可信的数据来源，我们必须透明。这就是为什么我们将公布我们的评级框架、我们的模型准确性评估协议和潜在的模型准确性数字。今年晚些时候，我们还将与全球领先大学的合作研究人员一起，发表关于我们先进生物量推断工作的同行评审学术论文。”

Index Ventures 的合伙人、Sylvera 董事会成员卡洛斯·贡萨雷斯-卡德纳斯在一份声明中对 A 轮融资进行了评论，他补充道:“如果没有一个运转良好的碳抵消市场，我们就没有机会减少全球碳排放。每年在碳抵消上花费数十亿美元，然而却缺乏透明度和问责制，因此也缺乏信任。要达到应对气候紧急事件所需的规模，信任是绝对必要的。作为一家独立的数据提供商，Sylvera 见证了一些世界上最大的公司、政府和其他实体的需求呈指数级增长。这凸显了他们的工作有多么重要，我们很高兴能够扩大与 Sylvera 的合作伙伴关系。”
