无头商业公司 fabric 在新一轮 TechCrunch 之后完成了独角兽里程碑

原文：https://web.archive.org/web/https://techcrunch.com/2022/02/24/headless-commerce-company-fabric-sews-up-unicorn-milestone-following-new-round/

Fabric 是一家成立五年的公司，为数字商务提供可组合的 API 驱动技术，以 15 亿美元的估值获得了 1.4 亿美元的 C 轮融资。

这家总部位于西雅图的模块化无头商务公司提供 300 多个商务 API 和相关框架，将公司的销售渠道连接到一个地方，并能够使用整个商务平台或单个产品。

虽然类似的公司以小企业为目标，但 fabric 正在追逐那些难以创建良好在线商店的大品牌，部分原因是他们通过大型、单一的遗留软件来运行在线商务，首席执行官 Faisal Masud 说“将他们锚定在严酷的平台上。”

“这并不是说亚马逊在商品销售方面做得更好，但他们建立了一个 API 优先的平台，特别是第一个 headless 平台，在那里他们可以根据需要进行灵活调整，”他补充道。“其他品牌的包袱是他们的平台。你不必重新搭建平台，但你应该将商业领域分割开来，转向微服务方式。”

虽然 Masud 认为 Shopify 这样的市场非常适合小企业，但他也认为对于其他任何从事 B2B 业务、年商品总值超过 5000 万美元或寻求更少限制、更灵活和更具组合性的销售渠道的人来说，Shopify 是一个错误的平台。

Masud 说，Fabric 的最新一轮融资是该公司一年内的第三轮融资，继去年 2 月的4300 万美元的 A 轮融资和去年 7 月的1 亿美元的 B 轮融资之后，该公司的估值为 8.5 亿美元。

软银领投了 C 轮融资，其他投资者包括 Forerunner Ventures、Glynn Capital 和现有投资者 Redpoint Ventures、Norwest Ventures 和 Stripes，后者领投了 b 轮融资。新资本迄今为止为 fabric 筹集了 2.93 亿美元的资金。

马苏德将融资轮的快速连续归因于 fabric 的“大幅增长”，其中大部分是有机增长。自从该公司在 2020 年的种子期以来，fabric 的客户从两个增加到今天的 60 多个——包括新客户 Honest co——同期员工人数增长了 6 倍，从 45 人增加到 280 人。

他解释说，种子期是 fabric 最后一次推动投资过程。在那之后，投资者才是靠过来的人。最近这一轮是“被压抑的投资者兴趣和增长再次超越公司计划”的结果。

“我们会见了软银，他们正在大力投资商业，”马苏德补充说。“他们认为织物在模块化和无头商务中处于领先地位。我觉得时机对我们很重要。如果你看看现在的市场，有一个崩溃，但我们的融资是对产品和团队的认可。未来 12 到 18 个月，这一领域将会有快速发展，我们不想因为资金或产品限制而落后。”

2021 年，fabric 实现了 4.5 倍的年同比收入增长，并推出了可扩展的 B2B 商务平台，该平台支持制造、分销或批发业务的在线销售交易。

该公司还充实了其领导团队，最引人注目的是让曾在思科工作的凯伦·布鲁尔(Karen Brewer)担任新的首席营销官，让曾在亚马逊工作的斯泰西·萨尔(Stacy Saal)担任新的首席运营官。马苏德还开玩笑说，首席收入官的公告即将发布。

Masud 计划将这笔新资金用于自动化和智能领域，以加速产品开发和地理扩张。他正在寻求将 fabric 的产品国际化，并在他希望扩张的地区，如欧洲和中东，将本地功能融入 API 和副驾驶应用程序。资金还将投资于额外的技术招聘和围绕收入和销售方面建立一个机器人团队。

作为投资的一部分，软银投资顾问公司(SoftBank Investment Advisers)投资总监罗伯特卡普兰(Robert Kaplan)将加入 fabric 董事会。

卡普兰在一份书面声明中说:“零售商需要在任何地方满足现代消费者——无论是在线、线下、移动、社交还是任何未来的切入点。”。“这意味着各种规模的商家，尤其是中端市场和企业，需要合适的商务平台来跟上不断发展的趋势。我们相信，fabric 已经构建了业界最好的 API 优先的模块化商务平台和配套的商务应用程序，为商家提供了无与伦比的能力和灵活性，同时减少了从传统、过时的商务平台迁移的痛苦。”
