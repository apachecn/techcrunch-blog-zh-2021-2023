分公司计划在人们返回办公室时收集新的家具

原文：https://web.archive.org/web/https://techcrunch.com/2022/04/11/branch-series-a-furniture-office/

总部位于纽约的办公家具公司 Branch 在过去的两年里，专注于用其核心产品为每个人的居家办公场所配备家具。现在，它准备推出一些新产品，包括一些针对那些离开家回到办公室的人的产品。

Hayes 说，该公司由 Greg Hayes，Sib Mahapatra 和 Verity Sylvester 在 2019 年推出，这样人们就可以购买优质家具，而不必通过经销商，比从传统零售商那里购买平均节省 50%。

我们用企业办公室记录了该公司的开端，以及它在 2020 年与全球疫情的冲突，包括一些裁员和转向家庭办公家具。自那以后，海耶斯告诉 TechCrunch，此举为 Branch 提供了一种重振旗鼓的方式，并抓住了在家工作时间比预期长的人的一些宏观顺风。

“我们在大约一个月内卖完了库存，花了 2020 年试图跟上，并在 2021 年开始怀疑这里是否有真正的业务，”他补充道。“原来是有的。我们在 2020 年增长了 600%，然后在 2021 年增长了四倍。今年，我们计划再翻三番。”

分支机构创始人，左起，Verity Sylvester，Greg Hayes 和 Sib Mahapatra。形象学分:分部

该公司的家具系列包括符合人体工程学的椅子、站立式办公桌、团队工作站、文件柜、会议室家具和空间划分。椅子的价格从 250 美元到 400 美元不等，而桌子的价格从 500 美元到 1000 美元不等。

Hayes 说，Branch 筹集了 350 万美元，有些在 2019 年，有些在 2020 年，同时在首都建立了高达八位数的运行率。

现在，该公司又筹集了 1000 万美元，这一次是在 A 轮融资中，主要用于新产品发布。此次注资由 Springdale Ventures 牵头，包括 Maywic Select Investments、Group RMC 和 Agya Ventures。参与的还有一批来自 proptech 和直接面向消费者行业的个人投资者，包括 Saatva 床垫创始人 Ricky Joshi、Convene 创始人 Ryan Simonetti 和 Outer 创始人 Jiake Liu。现有投资者包括 Nine Four Ventures、Alate Partners、RRE 和本·齐塞斯的 SuperAngel。基金也参加了会议。

“我们在核心产品上已经走了很长一段路，特别是在居家办公市场，但开发新的家具产品很难，我们希望做得好，”Hayes 补充道。“加薪使我们能够招聘优秀员工，推出优秀产品，围绕家具开发技术，因此有了一种新的方式来与 it 互动，并对供应链进行数字化优化。”

他认为 Branch 的技术以两种方式互动:一种是在消费者方面，将首先推出，包括人体工程学和更时尚的应用体验，为人们提供如何设置椅子或桌子的指导。它还将与客户支持、订购产品和游戏化有更好的互动。

在企业方面，该公司发现办公室经理管理办公家具的方式存在差距；例如，报告损坏或地板空间配置。

随着越来越多的员工回到办公室，Branch 的企业收入在过去三个季度增长了 500%。它的最佳点是员工人数在 20 到 200 人之间，它的大部分业务来自客户向他们的办公室经理介绍家具。

此外，该公司还从那些希望预先装修自己空间的房东那里获得了业务。Hayes 预计这类客户占分行业务的 10%,在全球疫情之前没有这么高。

至于新的资金，布兰奇用它来填补一些关键角色，包括家具设计，营销，销售和运营主管。之前团队不到 20 人。该公司还专注于产品开发和 R&D，并在年底前瞄准了一个可能的实体店。

“我们的设计越来越出名，成为一个设计驱动的品牌，”海斯说。“今明两年，我们将提高这一比例。这是一个巨大的行业，比大多数行业都要大，但在家具领域没有哈利或全鸟，所以我们正在成为这里的领导者。”
