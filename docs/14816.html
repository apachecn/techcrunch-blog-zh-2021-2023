QuickNode 以 8 亿美元的估值融资 6000 万美元，成为“区块链的 AWS 或 Azure”

原文：https://web.archive.org/web/https://techcrunch.com/2023/01/24/quicknode-raises-60m-at-800m-valuation-to-become-the-aws-or-azure-of-blockchain/?utm_content=bufferce636&utm_medium=social&utm_source=twitter.com&utm_campaign=buffer

区块链部署平台 QuickNode 的首席执行官兼联合创始人 Alex Nabutovsky 告诉 TechCrunch，该公司已在 B 轮融资中筹集了 6000 万美元，估值为 8 亿美元。

这轮融资由 10T Holdings 牵头，Tiger Global、七七六风险投资公司和 QED 投资者等参与了融资。此次融资发生在该公司 3500 万美元首轮融资后约 15 个月。纳布托夫斯基说，该公司总共筹集了超过 1.05 亿美元。

“当我们融资 3500 万美元时，我们只有 20 个人。现在我们有 120 多人，”纳布托夫斯基说。“我们将继续扩大团队。进入的企业数量令人难以置信，因此我们希望继续构建这些抽象层，发展我们的技术，并扩展到新的领域。”

Nabutovsky 说，虽然资本将专注于扩张，但公司也希望将资金用于增加区块链的采用。“如果我们要像去年那样大规模采用区块链，你需要提供企业级基础设施，”他说。

Nabutovsky 说，QuickNode 成立于 2017 年，旨在为 web3 带来 Web 2.0 基础设施，但随着时间的推移，它越来越大。他补充说，2022 年第三和第四季度是“收入创纪录的季度”，由于 Web 2.0 巨头和 web3 本土公司涌入该领域，收入增长了 300%以上。

其客户群包括主要的 Web2.0 公司，如 Twitter 和 Adobe ，以及 web3 平台，如比特币基地，OpenSea，Chainlink，1inch Network，Dune Analytics 和 Phantom 。

根据 Nabutovsky 的说法，该平台每月处理超过 2000 亿个 API 请求，并且 99.99%的正常运行时间“始终可用”。“我们的许多客户来到这里，因为我们能够保证 99.99%的可靠性、延迟和速度，”他说。

根据其网站的说法，它还与 16 个区块链合作，包括比特币、以太坊、索拉纳、多边形、币安智能链、雪崩和 Abritrum。“我们计划在今年年底前至少再增加四到五家连锁店，”纳布托夫斯基说。

“我们平台上有超过 50，000 名开发人员不断要求新的东西。但对我们来说，路线图总是添加协议并与这些基础一起工作。这不仅仅是一个协议，而是一种与基础建立的关系，作为他们生态系统的延伸，这样他们和他们的开发者就可以有更好的体验，”他说。

Nabutovsky 说，QuickNode 的长期目标很“简单”。

纳布托夫斯基补充说:“我们希望达到这样一个地步，即任何希望与区块链[技术]合作或互动的公司都将使用我们。”。“我们希望成为区块链的 AWS 或 Azure。这个行业已经非常成熟，产量惊人。”
