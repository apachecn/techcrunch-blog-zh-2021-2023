分析/商业英特尔搜索引擎 AlphaSense 获得 2.25 亿美元

原文：https://web.archive.org/web/https://techcrunch.com/2022/06/15/alphasense-an-analysis-and-business-intel-search-engine-finds-225m-at-a-1-7b-valuation/

谷歌向世人展示了理解和重视一个好的搜索引擎的重要性的方法，它可以打开你能在互联网上找到的所有信息。下一个迭代是垂直搜索:为特定用途提供深入非常具体的信息库的工具。

在公司为此建立解决方案的浪潮中，总部位于纽约的一家名为 AlphaSense 的公司，专注于提供搜索工具来查询市场分析和商业智能，现在已经筹集了 2.25 亿美元。它在 D 轮融资中的估值为 17 亿美元，是 2021 年的两倍(当时它在 C 轮融资中筹集了 1.8 亿美元)。这不仅证明了 AlphaSense 的增长，也证明了它有机会填补垂直企业搜索领域的更多空白。

这笔资金由两家投资者共同牵头，分别是高盛资产管理公司(Goldman Sachs)的成长型股票业务和维京环球投资公司(Viking Global Investors)。高盛在这里既是金融投资者也是战略投资者，芬兰联合创始人兼首席执行官杰克·科科告诉我:公司的分析师利用这个平台进行研究并发布他们自己的报告。2.25 亿美元的数字还包括贝莱德的“大量”债务承诺，尽管 AlphaSense 没有说明具体金额。

当我们在 2019 年最后一次报道该公司时——一个【1,000 万美元的 B 轮——alpha sense 有 1000 名客户。这个数字现在已经增长到 3500 家，去年客户数量在 2021 年增长了 110%，典型用途包括标准普尔 100 指数中的公司(75%是客户)；道琼斯 50 指数(97%)；以及大型资产管理公司和银行(该公司声称，美国所有这些公司的 70%都是用户)；以及那些在能源、工业、消费品和技术领域工作的人。

ARR 目前超过 1 亿美元，尽管 Kokko 告诉我 AlphaSense 尚未盈利(也没有选择盈利)。

如果说有什么不同的话，那就是新冠肺炎实际上推动了这家初创公司的发展。该平台的典型用户——商业分析师、策略师、财务规划师、投资者和其他经常购买和使用市场报告和竞争情报的人——不仅仍然需要做他们的工作，而且他们不太可能面对面地与同事和客户讨论研究，以获取和提供信息。

最重要的是，由于缺乏对即将到来的疫情及其影响的了解，人们比以往任何时候都更需要研究、见解和更多的信息。在这两种情况下，访问和传播研究的有用搜索引擎就相当于亲临现场。

AlphaSense 发现的差距实际上在每个信息孤岛中都很常见，但如何成功地解决它仍是棘手的部分。在商业信息的情况下，Kokko 说问题是它来自大量的来源，并且以许多不同的格式，大多数情况下是以叙述的方式传递的。

“我们专注于搜索非结构化信息，并为其提供结构，”他说。网络搜索智能是一个通过机器学习算法不断反馈的问题。在谷歌上搜索的人越多，谷歌就变得越好。“但我们的系统必须理解语言，并在没有数十亿次网络搜索的帮助和洞察力的情况下找到正确的信息。这些都不是私人信息。”

换句话说，我们的世界——或者至少是与之相关的信息——是我们在互联网上的牡蛎，搜索引擎是剥开它的工具。但同样的情况不适用于非结构化信息，因为这些信息基本上是私人的，没有上传到互联网上。

除了基本的搜索，鉴于人们最初在 AlphaSense 上搜索的性质，该公司的第二个产品层是如何呈现信息的形式。

它的算法是基于人工智能的，专注于自然语言理解，既提取搜索查询的意义和目的，也解析研究本身。

因此，虽然 AlphaSense 确实与一些公司就提供他们的研究链接并使其可搜索进行了谈判，但 Kokko 解释说，它也将一些信息掌握在自己手中，以将众多来源消化为更简洁的搜索结果的方式呈现这些信息。

这一点尤其重要，例如，当您对基于云的安全服务的市场规模有几个预测时，您可以权衡不同的预测以及它们包含或不包含的内容，因为您经常会看到来自不同分析师的不同数字和不同 KPI。

“我们正在向非结构化数据添加更多的结构，但我们也在对其进行组织，例如，提供热图来显示谁对特定数据点更乐观或更不乐观，”他说。“如果用户询问可达到的市场规模，我们可以提供一个答案，查看并接受所有这些数据。”

Kokko 说，AlphaSense 希望做得更多的另一个领域是公司内部搜索，以帮助企业更好地组织和访问自己的数据。因此，虽然其目前的产品可能会与 Wolfram Alpha 或 LexisNexis 相比，但内部搜索产品可能会使其更接近 Elastic 或 Algolia 之类的产品。

总而言之，这些机会说明了这项投资背后的理由，高盛增长董事总经理霍尔格·斯陶德告诉 TechCrunch，尤其是在如此多的其他公司发现融资具有挑战性的时候。

“AlphaSense 是一个正在加速增长的规模化企业，我们将其归因于产品和团队的实力，以及市场机会的规模。不管更广泛的科技市场如何重置，我们都很高兴能够支持像 AlphaSense 这样基本面强劲的高质量公司。“AlphaSense 正在为广泛的用户建立一个搜索和情报平台，而不仅仅是金融服务公司。我们认为，在整个周期中，企业仍将关注对精选业务信息的访问。AlphaSense 很好地抓住了这个机会。”
