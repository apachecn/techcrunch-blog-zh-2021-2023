Box 首席执行官 Aaron Levie 谈 web3 在哪里没有意义

原文：https://web.archive.org/web/https://techcrunch.com/2022/06/19/box-ceo-aaron-levie-on-where-web3-doesnt-make-sense/

对于加密社区来说，这是艰难的一周，顶级令牌出现了大规模抛售，迫使该领域的一些人加倍下注，同时让其他人评估该行业是如何走到这一步的，以及随着加密互联网的成熟，哪些广为接受的真理需要重新评估。

没有多少科技高管反复批评“web3”加密互联网代表什么的想法，但 Box 首席执行官亚伦·列维(Aaron Levie)肯定比大多数人更直言不讳。本周早些时候，我们有机会在 TechCrunch 的加密播客 Chain Reaction 上采访到 Levie，促使他关注一些他最怀疑的关于 web3 的承诺。

下面可以听完整集:

“我认为 web3 背后的理念很有说服力。列维告诉我们:“我认为很难否认更加分散的创新不是一件好事。”。“我认为，我所看到的实施在实际实现这一理念方面面临许多挑战。”

列维不是一家加密初创公司的高管，他似乎也没有探索 Box 的 web3 支点，但他告诉我们，他在推特上谈论 web3 的次数和他一样多，因为“作为一名初创公司的创始人，你必须了解世界的发展方向——然后你必须做出选择，看看你是否相信世界确实朝着其他人所说的方向发展。”

一些人认为，在区块链这个去中心化的世界里，高度集中的参与者最近几周的高调失败证明，更多的组织应该集体运营。不过，李维似乎并不认为 DAOs 或集体所有制会很快取代创业世界的传统结构。

“我们依靠库比蒂诺的人们来做出生产 iPhone 的决定，然后我们再决定是买还是不买。“这是我们在 iPhone 中唯一能做的决定，我们不能对任何事情进行投票，如果我们对任何事情进行投票，都会大大降低系统的速度，你就不能很快地创新，”李维说。“对于集体运动来说，(Dao)是非常令人兴奋的，就像无可争议的，但要取代快速发展的初创公司或公司的组织结构——我认为这是行不通的。”

随着加密风险投资推动企业家考虑用代币和 NFT 取代传统的基于广告的商业模式，推动消费者拥有他们使用的服务，Levie 质疑这些机制实际上有多普遍。

“我们可能高估了消费者对‘所有权’的需求，我之所以能这么说，是因为当你决定要做一个可以拥有物品的产品，还是加入一个网络但并不真正拥有太多物品的产品时，你会在产品中得到真正的权衡，”李维指出。“我碰巧看好广告的力量，因为它确实让产品变得更便宜，也确实让企业能够去寻找消费者。有些人采取另一种方式——这非常好。我认为问题是愿意接受这种交换的市场规模有多大，市场规模是否大到足以让我们谈论互联网工作方式的革命？”
