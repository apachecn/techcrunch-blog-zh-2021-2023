数据泄露数周后，儿童技术营 iD Tech 仍然保持沉默

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/23/id-tech-kids-tech-camp-data-breach/

在黑客从儿童科技编码营 iD Tech 窃取了数千名用户的个人数据几周之后，家长们仍在寻找答案，一些人担心他们孩子的数据在数据泄露中受到损害。

iD Tech 为孩子们提供校园课程、在线技术和编程课程，它尚未承认违规行为，也没有通知家长。

数据泄露的消息在二月份爆出，此前一名黑客在网络犯罪论坛上声称在一个月前的 1 月 3 日入侵了 iD Tech。

该黑客声称窃取了近 100 万条用户记录，包括姓名、出生日期、明文存储的密码和大约 41.5 万个唯一的电子邮件地址，当 iD Tech 通过电子邮件联系到这些记录时，并没有提出异议。这相当于每个家长都有一个或多个孩子在科技夏令营上课。

一些家长直到 3 月 6 日才发现数据泄露通知服务，如我被告知获得数据，向受影响的家庭发出通知。当其他服务，如 Firefox 或他们的设备安全软件通知他们在被破坏的数据中发现了他们的信息时，其他父母发现了这一情况。

一位从违规通知服务中得知自己的数据被盗的家长告诉 TechCrunch，被盗信息只是 iD Tech 收集的账户持有人和使用其平台的儿童的数据的一部分，包括性别、账单信息和一些健康数据，如免疫接种。

这位家长说，被泄露的数据肯定与孩子的出生日期有关，因为他们从未提供自己的数据。

这位家长说，iD Tech 公司还没有通知他们有违规行为。当这位家长联系该公司询问时，iD Tech 声称已经通知了受影响的账户持有人。

事实上，iD Tech 尚未在其网站或任何社交媒体渠道上公开承认违规。也没有证据表明 iD Tech 已经通知了受影响的账户持有人。

当通过电子邮件联系时，iD Tech 首席执行官 Pete Ingram-Cauchi 拒绝解释该公司为什么没有公开承认违规。当被问及时，Ingram-Cauchi 拒绝提供 iD Tech 声称发给家长的通信副本。该公司拒绝透露是否已根据数据泄露通知法向州检察长办公室报告了此次泄露事件。

相反，iD Tech 提供了一份来自通用公司电子邮件地址的简短声明，拒绝置评，称正在进行调查。邮件的发件人拒绝透露他们的姓名。

Ingram-Cauchi 没有回复后续的电子邮件。
