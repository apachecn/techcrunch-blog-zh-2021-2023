Stoï将网络保险产品与主动安全监控相结合 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/01/13/stoik-combines-cyber-insurance-products-with-active-security-monitoring/amp/

图片来源:阿迪·戈尔茨坦/昂斯普拉什

来认识一下stok，这是一家新的法国创业公司，希望保护中小企业免受网络安全事件的影响。该公司提供一种保险产品以及一种监控你的攻击面的服务。

这家初创公司最近从 Alven Capital、Anthemis Group、基马风险投资公司以及 raphal Vullierme、Emmanuel Schalit 和 Henry Kravis 等几位商业天使那里筹集了 430 万美元(€380 万美元)的种子资金。

Stoï专门针对中小企业，因为它们在遭遇勒索软件和其他网络攻击时非常脆弱。然而，小公司通常在保护他们的软件基础设施方面做得不够。

“我们将为你投保并保护你，”联合创始人兼首席执行官朱尔斯·威拉特告诉我。“但我们要卖的是保险产品。如果你受到攻击，你有一个电话号码可以全天候拨打，所有的费用都有保险。”

与此同时，Stoï将提供监控工具，以便小公司可以修复其基础设施中的漏洞。在这种情况下，斯托克和斯托克的客户之间的激励仍然是一致的。

这个由 15 人组成的团队已经与保险公司签署了合作协议，共同设计保险产品。Stoï销售保险产品并直接向客户收费——它从每份合同中抽取佣金。它与一家名为调查的第三方公司合作处理危机管理。

Stoï的工作方式有点像美国的联合保险公司，除了它不与经纪人合作分销其保险产品。这家法国初创公司希望与客户建立直接关系。

至于技术产品，当你注册这项服务时，你输入你的域名并开始扫描。Stoï查看 DNS 记录，查找 IP 地址，并扫描在线数据库，查找与该域名相关的密码泄露。

你得到一个分数和一些提高分数的建议。例如，Stoï可以告诉您，有些服务是对外公开的，尽管它们不应该公开。如果您的分数高于某个阈值，并且您的年收入低于€5000 万美元，您可以订阅该保险产品。

该公司目前正处于预发射阶段，合同范围从€每月 50 到€每月 400。接下来，它计划为其监控服务增加更多功能。例如，Stoï想要扫描内部账户。您可以想象扫描您的 Amazon Web Services 配置来发现一些漏洞。在与潜在客户签订新合同时，这也会有所帮助。
