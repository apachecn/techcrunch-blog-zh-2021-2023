尼日利亚 OnePipe 融资 350 万美元，对其嵌入式金融产品加倍投资| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/11/23/nigerias-onepipe-raises-3-5m-to-double-down-on-its-embedded-finance-offering/

去年，fintech API 基础设施公司进入了非洲科技领域，为企业和开发者带来了类似格子图案的服务。

对这些公司的关注，尤其是来自风险投资家的关注，螺旋上升到今年，每个重要的参与者都筹集了大量种子资金进行首轮融资。

OnePipe ，一家与众不同的金融科技 API 公司，今天加入了这个名单，筹集了 350 万美元的种子资金，对其嵌入式金融产品进行了双倍的投资。

专注于非洲影响力的风投Atlantica Ventures，去年 OnePipe【95 万美元 pre-seed 轮的联合牵头投资者，也与Tribe Capital和 V & R Associates 共同领导了此轮种子轮。

新投资者 Canaan Partners、Saison Capital、Norrsken(Klarna 创始人 Niklas Adalberth 的基金)、the fund 和 Two Culture Cap 也参与其中。现有投资者克里斯·阿德尔斯巴赫(Chris Adelsbach)、Techstars、Ingressive Capital、Acquity、P1、拉巴(Raba)和 DFS 实验室(DFS Lab)以及一些天使投资者也跟进了新的支票。

通常有三种主要的 fintech API 基础设施。 一个是数据和财务账目聚合(格子、秋葵、 Mono 、 Stitch 和 Pngme 是空间里的一些玩家) 。

第二个项目专注于嵌入式金融和银行服务，其中 Treasury Prime 、 Marqeta 提供他们的交易。第三个是核心开放银行业务，由 TrueLayer 等人开创。

OnePipe 最初的计划是创建一个 API 网关，在统一标准下连接银行和金融科技公司，此举将使该公司能够执行核心的开放银行业务。

但创始人兼首席执行官 Ope Adeoye(自称是该公司的首席水管工)在与 TechCrunch 的通话中表示，随着与这些金融机构的持续整合，公司显然需要转向，因为它没有产生许多需求周期。

在与几家银行达成合作后，OnePipe 决定退一步，深入嵌入式金融领域。

与开放银行和数据聚合业务不同，在这些业务中，一家公司需要与所在国家的几乎每一家银行合作， 提供嵌入式金融的公司不一定需要 合作。这就是为什么 OnePipe 目前有六家合作银行。

这位首席执行官说:“警告是这样的，当你对银行即服务进行定位时，你真正需要的是一家让你深入发展的合作银行，因为嵌入式金融(产品)是深度而不是广度。”。

“如果你选择数据聚合或开放式银行，那么你选择的是广度，而不是深度。 所以在我们这边，我们说我们更愿意选择二级和三级品牌，一旦你向他们描述了这个概念，他们就明白了 。 它为他们的成长提供动力，对他们更有价值，不像其他大型金融机构 。”

通过代表这些合作银行运行 API 基础设施并帮助他们将其货币化，OnePipe 与非金融机构合作推出并交叉销售一系列金融服务，如其产品中的信贷、账户和支付。

Adeoye 声称:“我们去年筹集了一轮资金，专注于合作伙伴关系的一个用例，即整合一组固定银行的 API，并提供嵌入式银行或银行即服务。”。“也就是说，我们让非金融机构或一般企业有可能向其客户提供银行服务。”

因此，举例来说，一家快速消费品初创公司可以插入由 OnePipe 管理的银行 API，并开始向客户发放账户，允许他们从这些账户中付款，并在需要时获得信贷。

【OnePipe 表示，在改用这种模式的 10 个月里，它已经处理了超过 630 万笔交易，价值超过 4630 万美元。这些数字来自 100 多万个个人账户和 138 多家企业，从快速消费品和零售到贷款和农业。

OnePipe 从这些账户的交易中抽取一定比例的佣金，并与其合作银行分享。对于通过其 API 提供的贷款，OnePipe 从其贷款合作伙伴那里获取至少 1%的贷款利息，并与企业和合作银行分享。

凭借 OnePipe 迄今取得的成就，Atlantica Ventures 的创始合伙人 Aniko Szigetvari 认为，该公司不仅深化了尼日利亚的金融包容性，也深化了整个非洲大陆的金融包容性。

“在我们看来，嵌入式金融是传统和金融服务企业的下一个推动者，通过为客户提供广泛的第三方金融产品和收入流来提高客户忠诚度和收入，”她说。

尽管 OnePipe 目前仅在尼日利亚开展业务，但它正首次走出该国，以响应 Szigetvari 的声明。

OnePipe 正在走战略合作伙伴路线，正如 Adeoye 提到的那样 他的公司与非洲物流和货运公司熊智娟达成了一项协议，以扩展到其他非洲国家。根据该公司的声明，该计划是“拉一个条纹购物风格的标签团队。”

首席执行官说:“我们确保在进入其他非洲国家之前，我们已经有了当地的客户。“我们与熊智娟达成了一项协议，让他们参与了这一轮，然后我们将部署资本进行扩张。因此，当他们去埃及、南非时，我们将 与他们一起部署，共同成长。”
