这就是为什么 NLP 初创公司的淘金热即将到来

原文：https://web.archive.org/web/https://techcrunch.com/2022/07/28/a-gold-rush-of-nlp-startups-is-about-to-arrive-heres-why/amp/

图片来源:谷歌双面

还记得自然语言处理吗？NLP 出现在几年前，但直到 2018 年，人工智能研究人员才证明，有可能在大量数据的基础上训练一次神经网络，并将其反复用于不同的任务。2019 年，来自开放人工智能的 GPT-2 和谷歌的 T5 出现了，表明它们非常好(现在已经被纳入谷歌 Duplex，如图)。人们甚至担心它们可能被滥用。

但从那以后，事情发展得，嗯，相当迅猛。

去年，自然语言处理初创公司和大型语言模型出现了名副其实的“寒武纪大爆发”。

今年，谷歌发布了 LambDa，这是一种用于聊天机器人应用程序的大型语言模型。然后 Deepmind 发布了 Alpha 代码，然后是 Flamingo——一种能够视觉理解的语言模型。仅在今年 7 月，大科学项目发布了 Bloom，一个大规模的开源语言模型，Meta 宣布他们已经训练了一个能够在 200 种语言之间翻译的单一语言模型。

我们现在正在达到一个临界点，我们将看到更多的 NLP 商业应用程序进入市场，其中一些使用了这些开源的、公开可用的平台。你几乎可以说，随着大型语言模型提供商之间的军备竞赛，创业公司试图建立在这项技术上的淘金热已经开始。

其中一家初创公司是 Humanloop ，这是一所大学学院的人工智能衍生产品，它声称通过一套帮助人类“教授”人工智能算法的工具，使公司“显著”更容易采用这种新的自然语言处理技术。这意味着律师、医生或银行家可以将一项知识放入该平台，然后软件可以在大型数据集上大规模应用这些知识，从而将人工智能更广泛地应用于各个行业。

它现在获得了由 Index Ventures 牵头的 260 万美元的种子资金，Y Combinator、Local Globe 和 Albion 也参与了进来。

Humanloop 于 2020 年由来自 UCL 和剑桥的杰出计算机科学家团队以及谷歌和亚马逊的校友创建，它表示，human loop 的应用可能包括从互联网上的非结构化数据构建一幅全国房地产市场的图片；阅读电子健康记录，以识别可能成为尝试新疗法候选人的人；甚至缓和对脸书团体的评论。

“如果人们知道基于语言的人工智能现在的能力，他们会感到震惊，”首席执行官拉扎·哈比卜在一份声明中说。“但将数据转换成算法可以使用的形式是最大的挑战。通过 Humanloop，我们希望使人工智能的访问民主化，并实现下一代智能的自助式应用程序——通过允许任何公司利用其领域专业知识，并在机器学习模型中有效地提取这些专业知识。”

Humanloop 声称，它的成功是“概率深度学习”的发展，在这种学习中，算法可以通过过滤掉数据集中的噪音，找到好的东西，并在人类不懂的部分寻求帮助，来解决他们不知道的问题。

其他建立自己的大型语言模型并将其置于 API 之后的创业公司包括CohereAI(1.649 亿美元的资金)和 Open AI GPT-3。浮潜AI(1.353 亿美元融资)也是这个领域的一家新初创公司。

然而，Humanloop 表示，它不太关注开发模型，而是更关注使它们适应特定用例所需的工具。

“许多人不知道的是，阻碍人工智能在每个工作场所无处不在的不是缺乏合适的算法，而是缺乏正确标记的数据，”领导此次投资的 Index Ventures 合伙人艾琳·普莱斯-赖特(Erin Price-Wright)补充道。“事实上，机器学习本身正变得越来越商品化，越来越现成，但对于非技术人员来说，将他们的知识传输给机器并帮助算法完善其模型仍然非常困难。”因此，Humanloop 允许人们调整数据。

如果 NLP 淘金热真的开始了，预计很快会出现一大堆其他创业公司。
