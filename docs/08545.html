NFL+流媒体服务在美国推出，取代 NFL Game Pass | TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/07/25/nfl-streaming-service-launches-in-the-u-s-replacing-nfl-game-pass/amp/

图片来源:美国国家足球联盟

今天，美国国家橄榄球联盟推出了其流媒体服务 NFL+,官方定价为每月 4.99 美元或每年 39.99 美元。然而，随着这一推出，即将离任的 NFL 游戏通行证将不再在美国提供。

NFL+订阅将包括所有场外季前赛直播、本地和黄金时段常规赛和季后赛直播(仅限移动设备)、NFL 网络节目点播、NFL 电影档案、每场比赛的本地和全国直播音频等。

当地和黄金时段的常规赛和季后赛比赛过去在雅虎体育应用上是免费的。与此同时，场外季前赛直播此前只能通过订阅 NFL Game Pass 才能获得，每年的费用高达 99.99 美元。

该公司告诉 TechCrunch，如果美国的 NFL Game Pass 用户是自动续订用户，他们将自动注册 NFL+，以大约 20 美元的价格为他们提供更多内容。

要享受 NFL Game Pass 之前包含的所有功能，订户必须拥有 NFL+ Premium，每月 9.99 美元或每年 79.99 美元。这包括完整的比赛回放和压缩的比赛回放(没有广告)，以及对“All-22”的访问，这是一个鸟瞰比赛的视图，有助于教练看到所有球员在每场比赛中的移动。

这标志着 NFL 首次运营自己的流媒体服务。联盟进入流媒体世界给了它额外现金的机会，并有可能增加其订户群，球迷们远离传统电视并切断了联系。

图片来源:美国国家足球联盟

美国国家橄榄球联盟(National Football League)有一段时间一直是局外人，美国其他主要职业体育联盟已经推出了流媒体服务。这包括 NBA 联盟通行证，NHL。电视和 MLB.TV

NFL 专员罗杰·古德尔在一份声明中说:“我们期待继续发展 NFL+,深化我们与所有年龄和人口统计数据的球迷的关系，为他们提供大量 NFL 内容，包括媒体行业最有价值的内容:现场 NFL 比赛。”。

据美国消费者新闻与商业频道报道，NFL 媒体执行副总裁兼首席运营官汉斯·施罗德表示，NFL+不会包括独家常规赛，但最终会包括。

在本月早些时候的一次采访中，古德尔证实了 NFL 的直接面向消费者(DTC)流媒体服务的推出。他还宣布，今年秋天，NFL 将为其周日门票套餐选择一个流媒体合作伙伴。
