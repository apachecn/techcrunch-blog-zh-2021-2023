Opera 已经发布了他们的加密浏览器的 iOS 版本

原文：https://web.archive.org/web/https://techcrunch.com/2022/04/15/opera-brings-its-crypto-browser-to-iphones-and-ipads/

Opera 将其加密浏览器引入 iPhones 和 iPads

Opera 发布了他们的 iOS 版加密浏览器，专为访问 web3 而设计，内置非托管加密钱包。

这款浏览器是以前在 Windows、Mac 和 Android 设备上可用的。

Opera 表示，他们为那些已经熟悉加密货币的人以及新来者开发了这款浏览器。该应用程序内置的加密钱包支持以太坊、多边形和塞洛区块链技术，Opera 计划集成更多的生态系统。用户可以使用法定货币和交换购买加密，并发送和接收支持的令牌。用户还可以将他们现有的加密钱包集成到浏览器的内置钱包中，前提是它们与以太坊虚拟机兼容。

Crypto Browser 应用程序还包括一个名为 Crypto Corner 的功能，该功能汇集了有关 Crypto 的新闻和信息，以及事件、空投和播客，以帮助用户了解该技术的最新动态。

Opera 说，加密浏览器的目标之一是让浏览 web3 像浏览 Web 2 一样容易。通过加密浏览器，用户可以访问 web3 NFTs 和分散式应用，包括 7，000 个多边形支持的服务。

Opera 移动业务执行副总裁桂鑫·阿内森在一份声明中说:“人们对 Web3 的兴趣正在持续增长。”。“Opera Crypto Browser 项目旨在简化 Web3 用户体验，这种体验常常令主流用户感到困惑。Opera 认为 Web3 必须易于使用，才能发挥其全部潜力并被广泛采用。

该浏览器还带有安全功能，如广告跟踪器、弹出窗口拦截器和 cookie 对话拦截器。它还使用加密货币挖掘保护来阻止加密劫持脚本。

为了解决对区块链高能耗的环境担忧， Opera 表示，它正在研究集成以太坊第二层解决方案，该方案旨在减少该技术对环境的影响。今年 2 月，Opera 宣布它已经将第二层集成到加密浏览器 Android 应用程序中。iOS 应用程序还实现了第 2 层技术。
