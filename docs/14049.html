2023 年的 4 个数据科学预测

托尔斯滕抢投稿人

Torsten Grabs

是 Snowflake 的产品管理总监，负责监管其数据工程、数据湖、数据科学工作负载和 Snowpark 开发人员体验。

数据科学长期以来一直是铁杆数据专业人士的领域，他们了解复杂的框架和相关语言，但这些专业人士是出了名的短缺。

幸运的是，工具和框架的前景在不断发展，我预测在 2023 年，新的发展将缓解数据团队和企业面临的挑战。

一方面，由于纯粹的必要性以及相关工具和平台的简化，长期宣传的公民数据科学家最终将在分析中发挥更大的作用。另一方面，数据专业人员将开始受益于这些更简单的工具来加速他们的工作，推动更大的标准化将有助于整个行业。

数据科学可能是目前所有企业技术中最令人兴奋的领域，它正在以闪电般的速度发展。

以下是对新一年数据科学的四个预测，以及企业如何利用它们。

Python 的使用将从数据专业人员扩展到普通开发人员

业务人员不能等待数据科学家提供他们需要的分析，所以他们要自己解决问题。随着预配置的云运行时和可访问工具的出现，Python 对于非专业人士来说变得更加容易接近，例如用于数字数据的 NumPy 、用于预测的 Prophet 和用于地理空间数据的 H3 。因此，到 2023 年，Python 的使用将从数据专业人员扩展到业务分析师和其他技术含量较低的用户手中。

Python 新手用户不应该尝试构建自己的运行时环境，而应该选择任何提供内置安全性和治理的现代云平台。Anaconda 提供了一个流行的 Python 发行版，有助于确保正确管理更新和依赖关系，Snowflake 将这些包安装在我们基于云的 Python 运行时中。

有许多非专业人士入门 Python 的在线资源，包括来自 RealPython 的这份全面的初学者指南。
