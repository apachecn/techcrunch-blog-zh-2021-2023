OpenSea 收购 DeFi 钱包初创公司达摩实验室，任命新首席技术官 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/01/18/opensea-buys-defi-wallet-startup-dharma-labs-appoints-new-cto/amp/

图片来源:达摩

NFT 市场 OpenSea 正在很好地利用其新筹集的资金，收购加密钱包初创公司达摩实验室，该公司今天宣布。该声明是在本月早些时候 Axios 的一篇报道之后发布的，报道称 OpenSea 正在谈判以 1.1 亿至 1.3 亿美元的价格收购这家初创公司。

达摩的以太坊钱包允许用户连接他们的银行账户，购买和交换代币，同时参与 DeFi 生态系统的其他元素。作为交易的一部分，OpenSea 将关闭达摩的应用，此举肯定会引起加密钱包用户的争议，他们只有一个月的时间将钱包中的资产转移到另一个账户。

作为交易的一部分，达摩实验室的联合创始人 Nadav Hollander 将成为 OpenSea 的新首席技术官。这家初创公司的前首席技术官亚历克斯·阿塔拉(Alex Atallah)将进入一个新的角色，“他将监督(OpenSea 的)web3 和 NFT 生态系统的开发工作，”该公司表示。

“我们的团队有一个共同的愿景，即 NFTs 将成为未来几年 crypto 采用的文化焦点——只有当使用 NFTs 变得对普通人来说简单而愉快时，这一愿景才能实现，”OpenSea 首席执行官 Devin Finzer 在一篇博客文章中写道。

随着 OpenSea 平台从 NFT 的夫妻店发展成为、价值 133 亿美元的 crypto decadecorn，其用户群中的一些人越来越直言不讳地表达了这样的观点，即该平台对于“web3”世界来说过于集中，早期用户没有像风险投资家那样从初创公司不断膨胀的估值中受益。尽管像 Rarible、仿植物怪兽佐拉和最近的新贵 LooksRare 这样的竞争对手受到了关注，但 OpenSea 仍然是 NFT 世界中遥遥领先的发电站。
