ChatGPT:关于人工智能聊天机器人你需要知道的一切

原文：https://web.archive.org/web/https://techcrunch.com/2023/04/01/chatgpt-everything-you-need-to-know-about-the-ai-powered-chatbot/

OpenAI 的文本生成人工智能聊天机器人 ChatGPT 风靡全球。它能够写文章，代码和更多给定的简短文本提示，超充电生产力。但它也有更… 穷凶极恶 的一面。

无论如何，人工智能工具不会消失——事实上，自几个月前推出以来，它已经有了显著的发展。各大品牌都在尝试这种技术，例如，利用人工智能生成广告和营销文案。

OpenAI 正在大力投资。ChatGPT 最近被来自 OpenAI 实验室的最新语言写作模型 GPT-4 充了电。付费的 ChatGPT 用户可以使用 GPT-4，它比以前支持 ChatGPT 的模型写得更自然、更流畅。除了 GPT-4，OpenAI 最近将 ChatGPT 连接到互联网，并在 alpha 中为等待名单上的用户和开发者提供插件。

这里是 ChatGPT 产品更新和发布的时间表，从最新的开始，定期更新。我们还回答了最常见的常见问题(见下文)。

ChatGPT 常见问题

什么是 ChatGPT？它是如何工作的？

ChatGPT 是一个通用聊天机器人，它使用人工智能在用户输入提示后生成文本，由科技初创公司 OpenAI 开发。聊天机器人使用 GPT-4，这是一种大型语言模型，使用深度学习来产生类似人类的文本。

ChatGPT 什么时候发布的？

2022 年 11 月 30 日是 ChatGPT 公开发布的时间。

ChatGPT 的最新版本是什么？

免费版 ChatGPT 和付费版 ChatGPT Plus 都会定期更新新的 GPT 车型。最近的型号是 GPT-4 。

ChatGPT 免费吗？

有一个免费版本的 ChatGPT 只需要在付费版本 ChatGPT Plus 之外进行一次登录。

谁用 ChatGPT？

任何人都可以使用 ChatGPT！越来越多的科技公司和搜索引擎正在利用聊天机器人来自动处理文本或者快速回答用户的问题。

ChatGPT 和聊天机器人有什么区别？

聊天机器人可以是任何与你/一个人进行对话的软件/系统，但不一定必须是人工智能驱动的。例如，有些聊天机器人是基于规则的，它们会对问题给出固定的回答。

ChatGPT 是人工智能驱动的，利用 LLM 技术在提示后生成文本。

ChatGPT 能写作文吗？

是的。

ChatGPT 有 app 吗？

没有适用于 iPhone 或 Android 的应用程序，但用户可以选择通过浏览器或使用 ChatGPT 公共 API 的第三方应用程序在移动设备上启用聊天机器人。

什么是 ChatGPT 字符限制？

任何地方都没有记载 ChatGPT 有字符限制。然而，用户已经注意到在 500 个单词之后有一些字符限制。

ChatGPT 有 API 吗？

没错，就是 2023 年 3 月 1 日上映的。

ChatGPT 的一些日常使用示例是什么？

日常例子包括编程、脚本、电子邮件回复、列表、博客创意、摘要等。

ChatGPT 有哪些高级用途？

高级使用示例包括调试代码、编程语言、科学概念、复杂问题解决等。

ChatGPT 写代码有多厉害？

这取决于项目的性质。虽然 ChatGPT 可以编写可行的 Python 代码，但它不一定能编写出整个应用程序的代码。这是因为 ChatGPT 缺乏上下文意识——换句话说，生成的代码并不总是适合使用它的特定上下文。

可以保存 ChatGPT 聊天吗？

是的。OpenAI 允许用户在 ChatGPT 界面中保存聊天，存储在屏幕的侧边栏中。目前还没有内置的分享功能。

有 ChatGPT 的替代品吗？

是的。有多个人工智能聊天机器人的竞争对手，如 Together ，谷歌的 Bard 和 Anthropic 的 Claude ，开发人员正在创造开源T8】替代品。但是后者现在更难运行——如果不是不可能的话。

围绕 ChatGPT 有哪些争议？(提一下 Red Ventures 等。)

在《未来主义》杂志报道《未来主义》杂志以一个完全由艾创作的神秘署名发表文章后，发现自己处于争议之中。拥有 CNET 的私募股权公司 Red Ventures 被指控使用 ChatGPT 进行 SEO 农业，即使信息是不正确的。

几个主要的学校系统和学院，包括纽约市公立学校，已经禁止 ChatGPT 进入他们的网络和设备。他们声称人工智能通过促进抄袭和错误信息阻碍了学习过程，这一说法并不是每个教育工作者都同意的。

在哪里可以找到 ChatGPT 提示的例子？

一些市场托管并提供 ChatGPT 提示，要么免费，要么象征性收费。一个是 PromptBase 。另一个是 ChatX 。每天都有更多发布。

ChatGPT 能检测出来吗？

很差。几个工具声称可以检测 ChatGPT 生成的文本，但是在我们的测试中，它们充其量是不一致的。

ChatGPT 聊天公开吗？

没有。但是 OpenAI 最近披露了一个 bug，该 bug 已被修复，它将一些用户的对话标题暴露给了服务上的其他人。

谁拥有 ChatGPT 创建的内容或媒体的版权？

从 ChatGPT 请求输入的用户是版权所有者。

围绕 ChatGPT 有哪些诉讼？

没有专门针对 ChatGPT 的。但 OpenAI 至少卷入了一场诉讼，这场诉讼对根据公开数据训练的人工智能系统有影响，这将涉及 ChatGPT。

ChatGPT 有关于抄袭的问题吗？

是的。像 ChatGPT 这样的文本生成人工智能模型有一种从训练数据中回流内容的趋势。
