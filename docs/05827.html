苹果对工作室显示网络摄像头的修复将在测试版中推出 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/04/26/apples-fix-to-the-mac-studio-webcam-is-rolling-out-in-beta/

苹果对工作室显示网络摄像头的修复正在测试中推出

我们真的很喜欢 Mac 工作室。它既强大又漂亮，几乎能以一种有意义的方式抓挠所有 Mac 桌面用户的痒处。就像最新的 MacBook Pro 一样，它代表了苹果应该前进的方向。然而，这篇评论确实有一个重要的警告:工作室显示器的网络摄像头。在一个越来越以远程工作为中心的世界里，1599 美元的显示器不应该有摄像头。

好消息是，这个问题似乎更多地与软件/固件方面的事情有关，而不是与实际的硬件有关(它实际上与新 iPad 上的模块相同)。这意味着该公司应该能够发布一个修复程序，将事情调整到它们应该在的地方。

以下是马修在当时所说的，“我问苹果我得到的结果是否具有代表性，并发送了我所看到的样本图像和视频。经过审查，苹果发言人告诉我，该系统的表现不如预期，苹果将进行更新，以解决相机的性能问题。

从左上顺时针方向:iMac 2020，iMac 2021，欧泊 C1 外置网络摄像头，Mac Studio。图片来源:布莱恩·加热器

在承认我们的系统(似乎还有其他审查人员的系统)没有按预期工作后，该公司承诺进行修复，但没有具体的时间表。就在这些评论取消一个多月后，该公司今天为运行 macOS 测试版的用户推出了一个修复程序。

在我写这篇文章的时候，12.4 开发者测试版应该就要到了，今天晚些时候会推出更大的测试版。该公司的一位发言人告诉 TechCrunch，“随着今天 macOS Monterey 12.4 的测试版发布，Studio Display 固件的更新现已可用。这次测试版更新对 Studio Display 相机的调谐进行了改进，包括改进的降噪、对比度和取景。”

我会说，由于不稳定的白平衡和图像噪声，我一直依赖外部网络摄像头进行视频聊天。我期待着安装更新，看看我们的土地。我们会相应更新。

 

更新:

固件肯定比早期版本有所改进。安装需要将您的系统更新到最新的 12.4 测试版，然后专门为 studio 安装一个单独的固件更新。接下来，确保你的第三方电话会议应用程序正确镜像了视频。第一印象:图像比以前更清晰，色彩平衡更准确(见上图)。如果你经常进行网络会议或流式视频，我仍然强烈推荐罗技或 Opal 等公司的第三方网络摄像头(这里的图像仍然有噪声)，但重新校准肯定会有所不同。
