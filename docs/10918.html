SaaS 平台 klikit 将餐厅厨房从“平板电脑地狱”中拯救出来

原文：https://web.archive.org/web/https://techcrunch.com/2022/10/06/saas-platform-klikit-saves-restaurant-kitchens-from-tablet-hell/amp/

图片来源: Klikit

送货服务的激增给顾客提供了许多选择，但也意味着繁忙的餐馆陷入混乱，因为它们需要跨多个应用和渠道管理订单。许多厨房通过同时处理几个设备来处理这个问题，每个应用程序一个设备。 Klikit 希望通过将所有应用程序的订单信息整合到一个平台上，将东南亚食品企业从平板电脑地狱中拯救出来。这家总部位于新加坡的初创公司刚刚获得了 200 万美元的种子前资金，退出了隐形模式。

这轮融资由 Global Founders Capital 和 Wavemaker Partners 共同牵头，参与方包括 Gentree Fund、AfterWork Ventures、Reshape Ventures、Nordstar、Pentas Ventures、Moving Capital、Gojek 联合创始人凯文·阿鲁维(Kevin Aluwi)、NasDaily 的努塞尔·雅辛(Nuseir Yassin)、YouTuber 耶戈·比姆(Michael Beam)和萝卜小说创始人李承允(Seung-yoon Lee)。战略天使投资者包括来自 Gojek、YouTube 和闪咖啡的高管。

自 7 个月前推出以来，klikit 的 SaaS 平台 klikit Cloud 已被用于为菲律宾、马来西亚、印度尼西亚、新加坡、台湾和澳大利亚的 150 个品牌提供超过 280 万美元的订单服务。

目前的用户包括 Bistro Group(TGI 星期五餐厅、硬石咖啡馆和布法罗辣鸡翅、快闪咖啡和幽灵厨房初创公司 MadEats 和 Just Kitchen 的菲律宾特许经营商。

Klikit 由克里斯托弗·威瑟斯(Christopher Withers)于 2021 年创立，他在按需领域拥有丰富的经验——他之前是 GoJek 的市场副总裁，孟加拉国打车平台 Pathao 的首席战略官，并在亚太地区推出了 UberEats。

在疫情期间，在 GoJek，Withers 搬到了澳大利亚的家中远程工作。他还拥有并经营一家幽灵厨房。

威瑟斯告诉 TechCrunch，他一直对食品配送领域着迷。

“我创办了我的幽灵厨房，因为我一直想真正体验经营一家餐馆的困难，而不是在我建立许多这些超级应用市场时，坐在一旁或笔记本电脑后面假设，”他说。

在那段时间里，Withers 被他不得不应付的平台、设备、软件、广告和社交媒体的数量和成本所淹没。因此，他想找到更有效的方法来管理它们，并推出新品牌。

威瑟斯解释说，现有的 F&B 软件不适合许多外卖餐馆和云厨房，亚洲只有不到 2%的商家将他们的外卖订单与传统的销售点系统集成在一起。这使得厨房和员工需要在不同的应用程序和设备上管理订单，这不仅耗时，还会导致订单遗漏、错误、混乱和普遍混乱。

“许多运营商称之为‘平板电脑地狱’,我们的一些客户在一个厨房位置就有多达 20 多台设备，占据了整个储藏室的面积！”威瑟斯说。

克里奇特的团队。图片来源: Klikit

Klikit 与为单一品牌公司创建的传统 POS 系统不同，它使餐馆和幽灵厨房能够在一台设备上管理多个地点和渠道的多个食品品牌。功能包括跨交付应用更新菜单，klikit 能够快速完成这项工作，因为它与 GrabFood、foodpanda、GoFood 和 UberEats 等应用有官方 API 协议。它提供了对历史数据分析的按需访问(相比之下，许多 F&B 软件系统将数据限制在有时间限制的查看范围内)，包括每日销售、产品组合和渠道细分。

由于东南亚的许多餐馆经常通过 WhatsApp、SMS 或音频消息等社交媒体处理送货订单，klikit 还允许将这些订单添加到其订单仪表板中，以便将其纳入分析中。

如果 klikit 的一个客户有多余的产能和设备，他们可以注册进入其与创造者和消费者品牌的虚拟品牌合作伙伴关系。Klikit 现在正与在菲律宾和澳大利亚拥有 3800 万粉丝的创作者合作，在 2022 年底推出两个“创作者滴”。威瑟斯说，klikit 与顶级 YouTubers 联系在一起，因为他们有能力在营销方面与快餐巨头竞争。

Klikit 最接近的竞争对手包括 Deliverect 和 NextBite，但 Withers 表示，他相信像 klikit 这样的区域性初创公司会成功，因为它可以巩固 API 与主要交付应用程序的合作关系。

这家初创公司的新资金在秘密行动期间被用于在六个国家雇佣 30 人。它还将利用这笔资金进行地区扩张，并通过组建工程团队来增加更多功能。

在一份声明中，Wavemaker Partners 管理合伙人保罗·桑托斯说:“我们看到 klikit 为各地的餐馆老板解决了广泛未解决的问题，同时也为创作者和品牌创造了独特的解决方案，以全新的方式获得收入并与粉丝互动。他们的愿景战略性地汇集了食品交付和创造者经济的趋同和增长趋势。”
