Tatsumeeko 想让元宇宙不和 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/06/09/tatsumeeko-wants-to-bring-the-metaverse-to-discord/

Tatsumeeko ，一款以太坊和 Solana 上的“MMORPG 精简版”游戏，由跨游戏平台生态系统 Tatsu 背后的团队开发。GG ，今天宣布已经筹集了 750 万美元的种子资金。这轮融资由迪法恩斯资本(DeFiance Capital)、德尔福风险投资(Delphi Ventures)和比特克拉夫特风险投资(BITKRAFT Ventures)共同牵头，投资者包括币安实验室、Animoca Brands、Dialectic 和 GuilldFi。这笔资金将用于继续发展 Tatsumeeko。

这使得该公司的融资总额在两轮中达到 810 万美元。

达。GG 是为 Discord 社区游戏化和参与而创建的，目前声称拥有 6000 万用户和 140 万 Discord 社区。它于 2016 年推出，当时其创始团队看到了在 Discord 上创建 web3 游戏社区的机会，从而增加了用户保留率、获得率和互动。

Tatsumeeko 将是一款角色扮演游戏，可以通过 Discord 以及 iOS、Android 和 web 使用。它以一个名为 Ielia 的世界为中心，玩家可以在这里与怪物战斗，建立社区并结识其他玩家。在这个过程中，他们还被引入到 crypto 和 NFTs 中，这个团队称之为“无缝”的方式。

随着游戏的推出版本达到 60，000 个月活跃用户和 450 万个交易项目，Tatsumeeko 现在将推出一个更广泛的版本，包括主要和次要故事任务，合作突袭，次元入侵和玩家与环境的战斗，以及游戏内的令牌奖励。

NFT 的项目可以通过他们的不和谐直接融入达美子。7 月初，它将开始首次出售 Aethereal Parcels，即向收藏家和与之互动的玩家提供效用和特殊属性的土地。去年 11 月，该游戏推出了 Meekolony Pass，这是一系列关于 Solana 的 10，000 个 genesis NFTs，为持有者提供利益、奖励和对 Tatsumeeko 中物品的空投。

Tatsumeeko 的目标是扩展到其他社交平台，如 QQ、微信和 Telegram，以及以太坊和 Solana 社区。

联合创始人大卫·林(David Lim)说，他想创造一个 MMORPG 精简版，因为他从小玩传统的 MMORPG，如 Everquest、Guild Wards、Maplestory 和最近的最终幻想 14。

但他补充说，MMORPGs 需要玩家花费大量时间和精力来充分享受整个体验。另一方面，Tatsumeeko 旨在结合 MMORPG 的最佳部分，如强大的故事情节，社交功能，探索，角色发展，任务和战斗，而不是耗时的任务，如磨水平。取而代之的是，这些对于玩家来说是自动化的。

“MMORPGs 一直擅长一件事——帮助有机社区的形成，”Lim 告诉 TechCrunch。“我们与 Tatsumeeko 一起所做的是，我们将这一概念应用于许多社区已经存在的不和谐之处，并帮助这些社区在社区参与、保留和用户获取方面加速增长。”

Tatsumeeko 不仅被玩家使用，还被公司、游戏和 NFT 项目用作基础设施，将他们自己的项目集成到 Discord 的生态系统中，并建立一个社区，为他们的 NFT 提供实用程序，并通过游戏所在的社区提高品牌知名度。

“也许你可以认为，我们不是在建设一个元宇宙，并试图将整个不和谐的社区转移到元宇宙，而是在让元宇宙与立美子不和，”林说。

对于不熟悉加密和 NFTs 的人来说，Tatsumeeko 将针对 Web 2，0 和 web3 玩家进行介绍。Lim 说，人们可以开始玩这个游戏，而不必创建一个钱包，购买 NFTs 或密码。“你可以认为我们在幕后使用加密和 NFT 技术，以确保我们的玩家对他们的游戏内资产有更多的控制权，如果他们选择这样做的话，”他说。

“很长一段时间以来，游戏在教授玩家全新的概念方面做得非常好，”Lim 补充道。“这也包括在 Tatsumeeko 自己的市场内交易和购买游戏中的资产和货币。”
