谷歌希望把 Chrome 操作系统带到你的电脑或 Mac 上——TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/02/15/google-wants-to-bring-chrome-os-to-your-pc-or-mac/amp/

图片来源:谷歌

谷歌今天宣布了 Chrome OS Flex，这是一项新计划的早期预览，该计划旨在将 Chrome OS 引入几乎任何 PC(以及基于英特尔的旧 MAC)。Chrome OS Flex 建立在谷歌于 2020 年收购的 CloudReady 基础上，面向希望延长现有设备寿命的企业和教育用户，但任何人都可以使用 USB 驱动器来给旧 PC——甚至可能是低功率的新 PC——带来新的生机。这里有一个明显的可持续性角度。

这里的想法是将完整的 Chrome 操作系统体验带到几乎任何电脑上。现在，情况可能不是这样，因为有无数种可能的 PC 配置。谷歌已经公布了一份经过认证的机器清单(大多是笔记本电脑)，但是其他机器也有可能运行良好——尝试一下就像尝试一个现代的 Linux 发行版一样困难，Chrome OS 显然是基于这个发行版的。

谷歌 Chrome OS 产品管理总监 Thomas Riedl 指出，如果消费者不得不进入他们的 BIOS 并配置他们的机器从 USB 驱动器启动，那么安装体验对他们来说仍然有点粗糙。

图片来源:谷歌

Chrome OS 和 Chrome OS Flex 使用相同的代码库，并将遵循相同的发布节奏。对于用户来说，这两个系统应该看起来和感觉上完全一样。不过谷歌很清楚，这仍然是一个非常早期的版本，用户不应该期望一切都马上完美运行。

你会问，安卓应用呢？正如谷歌发言人告诉我的那样，该团队目前专注于尽可能稳定的核心用户体验，但没有任何计划将对谷歌 Play 商店和安卓应用的支持添加到 Chrome OS Flex 中。“但当然，我们正在不断评估如何为未来改进 Chrome OS Flex，”这位发言人说。

图片来源:谷歌

如果你用的是现代的 Mac 或 MacBook，你可能就没那么幸运了，因为苹果的专有硬件使得在这些机器上安装 Linux 发行版几乎是不可能的——尤其是较新的 M1 设备。不过，大多数 2016 年之前的设备应该都可以正常工作。如果你还有一台白色的 MacBook，也许现在是时候掸掉它的灰尘了。

因为它基于 CloudReady 的工作，所以这里还有一个企业组件。适用于 Chrome OS 的所有常见 IT 管理功能也适用于 Chrome OS Flex。企业还可以将他们的 Chrome OS 许可证应用到 Chrome OS Flex 设备上(而在以前，CloudReady 有不同的许可证结构)。

CloudReady 制造商 Neverware 的前产品总监、现谷歌 CloudReady 的产品经理 Forrest Smith 告诉我，大约在 2016 年，企业开始接受使用其服务来延长设备寿命的想法。

“有很多人想让设备的寿命延长 12 个月——他们真的愿意做任何交易，”他告诉我。“实际上，非常明显的是，在 2015 年、2016 年和 2017 年的那个地区，事情发生了转变，人们真正购买了 Chrome OS 生态系统，客户找到我们说，‘我只是不能得到我想要的那么多 Chrome book。’"

Riedl 说:“令我非常兴奋的是，我们已经成功地将一个非常有趣但独立的解决方案与 CloudReady 及其管理产品进行了无缝集成。“因此，这些 Chrome OS Flex 设备，即使它们过去运行 Windows 或 Mac，它们也只是出现在管理控制台中，这意味着您可以在单一控制台中管理所有这些设备，无论是 Chromebook、Chromebox 还是 Flex 设备。这让管理员轻松多了。”
