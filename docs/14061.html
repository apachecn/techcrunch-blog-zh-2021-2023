由于纽约公立学校屏蔽了 ChatGPT，OpenAI 表示正在研究“缓解措施”,以帮助识别 ChatGPT 生成的文本

原文：https://web.archive.org/web/https://techcrunch.com/2023/01/05/as-nyc-public-schools-block-chatgpt-openai-says-its-working-on-mitigations-to-help-spot-chatgpt-generated-text/amp/

图片来源:吉纳·穆恩/彭博/盖蒂图片社

纽约市公立学校限制访问 ChatGPT ，这是一个人工智能系统，可以在学校网络和设备上生成各种主题和各种风格的文本。正如今天上午 广泛报道并由纽约市教育局发言人向 TechCrunch 证实的那样，由于担心 ChatGPT 制作的内容“对学生学习的负面影响”和“安全性和准确性”,实施了限制。

这位发言人通过电子邮件告诉 TechCrunch 说:“虽然这个工具可能能够提供快速简单的问题答案，但它没有培养批判性思维和解决问题的技能，而这些技能对于学术和终身成功来说是必不可少的。”他补充说，限制访问是应学校的要求。

这本身并不是禁令。纽约市公立学校系统正在使用相同的过滤器来阻止学校财产上的其他应用程序和网站，例如 YouTube 和脸书。个别学校可以要求解除 ChatGPT 的封锁，该发言人表示，纽约市教育局将“欢迎”有机会与 ChatGPT 背后的初创公司 OpenAI 进行对话，讨论如何将该工具应用于教育。

当被联系到发表评论时，OpenAI 的一位发言人表示，该公司正在开发“缓解措施”，以帮助任何人发现 ChatGPT 生成的文本。这很重要。虽然 TechCrunch 最近报道说 OpenAI 正在试验人工智能生成文本的水印技术，但这是 OpenAI 第一次证实它正在开发专门用于识别来自 ChatGPT 的文本的工具。

“我们将 ChatGPT 作为研究预览提供，以从现实世界的使用中学习，我们认为这是开发和部署有能力、安全的人工智能系统的关键部分。OpenAI 发言人说:“我们不断地整合反馈和吸取的教训。“我们一直呼吁围绕人工智能生成的文本的使用保持透明。我们的政策要求用户在使用我们的 API 和创意工具时要坦诚面对他们的受众。。。我们期待与教育工作者合作，寻找有用的解决方案和其他方式，帮助教师和学生从人工智能中受益。”

ChatGPT 有回答从诗歌到编码等话题的能力，但它最大的缺陷之一是有时能给听起来令人信服但事实上不真实的答案。这导致 Q &一个编码网站栈溢出，暂时禁止用户分享人工智能生成的内容，称 ChatGPT 让用户太容易用可疑的答案淹没平台。最近，世界上最大的人工智能和机器学习会议之一机器学习国际会议宣布禁止包含 ChatGPT 和其他类似人工智能系统生成的文本的论文，因为担心“意想不到的后果”。

在教育界，争论主要围绕着作弊的可能性。在谷歌上搜索“ChatGPT 来写学校论文”，你会发现很多教育工作者、记者和学生通过使用 ChatGPT 来完成家庭作业和标准化论文测试的例子。《华尔街日报》专栏作家乔安娜·斯特恩使用 ChatGPT 写了一篇及格的美联社英语文章，而《福布斯》职员艾玛·惠特福德使用在 20 分钟内完成了两篇大学论文。在接受《卫报》采访时，亚利桑那州立大学教授丹·吉尔默回忆起他是如何给 ChatGPT 布置了一个他通常会给学生的作业，并发现人工智能的论文会获得“高分”

剽窃是另一个问题。像其他生成文本的人工智能系统一样，ChatGPT 在公共数据上接受训练，通常在未经同意的情况下收集-有时可以在不引用任何来源的情况下逐字复述这些信息。这包括前面提到的事实不准确，以及偏见——包括公然的种族主义和性别歧视。OpenAI 继续引入过滤器和技术来防止有问题的文本生成，但新的解决方法每天都会出现。

尽管存在这些限制和问题，一些教育工作者仍然看到了 ChatGPT 和其他形式的生成性人工智能技术的教学潜力。斯坦福大学教育学副教授维克多·李(Victor Lee)最近在斯坦福大学教育学院网站上撰文指出，ChatGPT 可能有助于学生“用他们目前没有的方式思考”——例如，通过帮助他们发现和澄清自己的想法。他认为，教师也可以从 ChatGPT 中受益，为学生创造许多基本内容保持不变，但风格、句法或语法不同的叙事实例。

“ChatGPT 可以[允许]学生多次阅读、思考和修改，而没有这种过程经常引起的痛苦或沮丧，[同时]教师可以使用该工具作为一种方式来生成一种形式或流派的许多例子和非例子，”Lee 在一份声明中说。“显然，老师们不太喜欢计算机为学生做大量的跑腿工作。学生们仍然需要学习写作。但是用什么方式，写什么样的文章呢？答……。这种新药的副作用是，它要求我们所有人都提出这些问题，并可能对我们教学的总体目标和方法做出一些实质性的改变。”

无论如何，纽约市的公立学校政策，这似乎是美国第一个此类政策，肯定会迫使其他地方的学区进行讨论。随着这项技术的使用越来越多——截至 12 月，ChatGPT 的用户超过了 100 万——的独立研究人员和的公司已经开始尝试使用工具来检测人工智能生成的文本在学生提交材料中的使用。一些教育工作者可能会选择接受它们，而其他人，比如 Lee，则鼓励使用 ChatGPT 作为辅助写作工具。
