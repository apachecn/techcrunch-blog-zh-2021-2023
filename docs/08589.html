Pogo 获得数百万美元，成为“现实世界的蜂蜜”

原文：https://web.archive.org/web/https://techcrunch.com/2022/07/26/pogo-lands-millions-to-become-the-honey-for-the-real-world/?utm_medium=TCnewsletter&tpcc=TCstartupsnewsletter&utm_campaign=TCstartupsweekly

如果 Pogo 得以实现，每当你漫步在旧金山的市场街上，你都会得到报酬。或者检查你的电子邮件。或者打开它的 app。唯一的问题是，作为回报，你会将你的个人数据提供给专注于消费者的金融科技公司。

“我们几乎可以为现实世界创造蜂蜜，”Pogo 联合创始人兼首席执行官 Dom Wong 说，他正与创始人奥斯卡·梅尔金(Oskar Melking)和什卡·莫汉(Shikhar Mohan)一起创建这家公司。这家初创公司试图利用一些科技趋势来获得优势。首先，随着数据隐私 变得更受关注，用户能够选择是否与公司分享他们的信息。Pogo 的观点是，如果他们真的分享，为什么不从中获利呢？

其次，Pogo 是对 Rakuten 和 Honey 等电子商务工具崛起的一种利用，这些浏览器扩展可以自动找到现金、优惠券和交易，以换取你的购物历史。第三，它使用一系列 API 来接入消费者存在的任何地方，从他们的收件箱到他们的交易历史，如果他们愿意的话，还有他们的位置数据。

为了扩大数据以换取现金，这家初创公司以 Josh Buckley 领导的 1230 万美元种子轮和之前未宣布的 250 万美元预种子轮的形式筹集了 1480 万美元的风险投资。这家初创公司的其他投资者包括 Slow Ventures、Village Global、哈里·斯蒂林的 20VC、MrBeast 的 Night Ventures、Hyper、耸肩以及包括 The Chainsmokers、索菲亚·阿莫鲁索、瑞安·泰德和兰尼·拉奇斯基在内的创造者。此外，这家初创公司还从 Front、Rent the Runway 和 Honey 的创始人那里获得了资金。

它是这样工作的:当用户加入应用程序时，他们被邀请将他们的数据发送给公司。一旦 Pogo 与不同的信息来源连接，它就开始显示购买奖励或建议“采取行动”以节省资金。

该应用的一部分是帮助那些选择加入的人从他们的数据中获得报酬:“Pogo 让你可以轻松地将你的数据聚集在一个地方，让你可以控制自己喜欢的用例获得报酬，无论是匿名市场研究还是可信品牌的个性化营销。”该应用程序的另一部分类似于 Honey，因为它是一种金融代理，可以看到你的支出并提出建议，无论是为了更好的交易还是你可能不知道的促销。

这并不是完全没有听说过。例如，Dosh 是一家由风险投资支持的初创公司，它与零售商、品牌和支付提供商达成交易，然后在用户购买与关联提供商相关的产品时向用户提供现金返还。根据 TechCrunch 的报道，这家初创公司在 2019 年的估值为 4 亿美元。还有尼日利亚的 ThankUCash，它从获得了 530 万美元来建立现金返还基础设施和 PayPal 的现有工作，PayPal 推出了一种新信用卡，为所有 PayPal 购买提供 3%的现金返还。

图片来源: Pogo

这里有活动是有道理的。随着广告收入从脸书和谷歌流失，零售商和品牌希望找到新的方式来吸引潜在客户。我是这样想的:你可以选择在浏览 Twitter 的时候出售你的新健身装备，或者在当地 Crunch 健身房的更衣室里出售。后者在消费者更注重健康和时尚的时候找到了消费者，而前者打赌某人可能只是在浏览科技新闻时碰巧有心情买一套新的。

Pogo 没有与当地企业建立独家合作关系，以获得一定的现金回报来换取潜在客户，这是一些新银行已经走上的路线。相反，这家初创公司只是将所有现有交易集中在一个地方 Wong 认为不同数据源之间的互操作性是最大的竞争优势。

Pogo 真正的与众不同之处在于它为仅仅存在于世界上的用户提供了一个可以将通知转化为金钱的隐形层。Pogo 的大多数用户链接位置数据，因为他们在现实世界中可以获得奖励。“你路过温迪快餐店，如果你在应用程序上购买，就会弹出一个免费早餐三明治的通知。你所要做的就是观看它，然后你会得到报酬，我们也会得到报酬。”

目前，该公司主要通过代销费和数据交换获得报酬:

  • 选择加入建议的保险计划。
  • 点击进入基于位置的报价。
  • 回答市场研究或活动测量问题。
  • 购买处方时扫描 Pogo 卡以获得折扣。

Wong 说，在下一次迭代中，Pogo 将建议用户转向免费的银行服务，然后 Pogo 可以通过代销费赚钱。

“我们还考虑如何帮助您利用今天唾手可得的优惠，而无需正式促成合作关系。他在一封电子邮件中写道:“这可能意味着在你的收件箱里，以一种私密、安全的方式，出现公开的内容，甚至是个性化的促销信息。”。“例如，想象一下，当你下次走进 CVS 时，Pogo 会提醒你，你的收件箱里有一张未开封的全店优惠券，上面的任何商品都有 40%的折扣。”

虽然 Pogo 肯定认为自己是与客户一致的激励措施，但为了赚钱，它需要向其他公司证明，个人数据是一种商品。Pogo 将为人们的生活提供一个亲密的窗口，从他们住在哪里到他们最喜欢的咖啡店，到他们拥有多少订阅。这类似于银行会看到的，但它是一家风险投资支持的初创公司，它希望你信任它。

电子前沿基金会(Electronic Frontier Foundation)是一家非营利组织，自 1990 年以来一直在捍卫数字世界的公民自由，它将用数据换钱的想法描述为“数据红利”在一篇文章中，该组织敦促消费者重新思考，为他们的数据付费是否真的能解决用户和企业之间现存的不平衡。EFF 会问一系列问题，比如谁来决定某些数据的成本，以及是什么让你的数据对公司有价值？此外，普通人从数据红利中获得了什么，他们为了换取额外的现金而失去了什么(T4)？

“EFF 强烈反对数据红利和为人们考虑其数据的货币价值而不是将其视为基本权利奠定基础的政策，”该组织在一篇帖子中写道。“你不会给你的言论自由贴上价格标签。我们也不应该把它放在我们的隐私里。”

在电子邮件中，Wong 说，普通数据费用有现有的市场价格，但这家初创公司正在制定基于“大量绿地的定价策略，这在今天根本不可能。”

“例如，我们能够提供基于位置的优惠或广告，同时考虑到您过去的购买情况，这使我们能够获得比您的典型广告投放高出许多数量级的费率，”他继续说道。“另一个例子是，品牌能够根据核实的购买情况获得实时消费者反馈。如今，这种能力慢得令人难以置信，而且根据现有年收入数十亿美元的巨头设定的费率，品牌为每次回复支付数百美元并不罕见。”

Wong 现在还不太担心失去用户的信任。到目前为止，该应用程序已经被“数十万个家庭”使用，这家初创公司声称它已经产生了数百万美元的储蓄和收入。“我们非常有意识地考虑我们在这里如何考虑合作关系”,重点是让用户“对他们熟悉的用例进行精细控制”
