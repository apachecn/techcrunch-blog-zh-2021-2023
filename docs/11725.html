Y42 希望成为您数据管道的任务控制者| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/11/02/y42-wants-to-become-mission-control-for-your-data-pipelines/

当总部位于柏林的 Y42 于 2020 年推出时，它的重点主要是为商业智能编排数据管道。在过去的几年中，这项任务已经扩展了很多，今天，Y42 宣布推出它所谓的“现代数据操作云”建立在数据仓库服务雪花和谷歌的 BigQuery 引擎之上， Y42 的新的完全托管服务旨在为企业提供更多的工具，使更多用户可以轻松访问他们的数据堆栈，同时还提供额外的协作工具和改进的数据治理服务。

“数据的使用案例已经超越了临时报告，成为公司的命脉。然而，临时构建的数据管道天生脆弱，随着时间的推移不可避免地会损坏，导致消防请求的泛滥，并最终导致对业务数据的不信任。Y42 创始人兼首席执行官 Hung Dang 表示:“对于依赖数据做出关键决策的组织来说，这可能是致命的。

图片来源: Y42

他认为，Y42 的新 DataOps 云将允许组织更容易地创建和运行生产就绪管道，并消费通过它们获得的数据。

像以前一样，Y42 完全管理数据堆栈，使用像 Airbyte 这样的开源工具来集成不同的服务，使用 dbt Core 进行转换。对于高级用户和数据团队，Y42 提供了基于 Git 的版本控制(尽管非技术用户也可以通过该服务的 web 应用来利用这一点)，通过这个新平台，该公司现在还提供了增强的治理工具，如数据目录、资产所有权分配、数据合同和多级访问控制。

“我们的愿景是让每个组织——无论是单个数据工程师、数据分析师还是整个数据团队——都能够高效地创建和运行生产就绪型数据管道，并在任何下游应用中使用数据，从而做出更好的业务决策。Y42 Modern DataOps Cloud 让这一愿景在今天成为现实。

除了新的托管服务，Y42 今天还宣布聘请朱尔斯坎特威尔担任总裁。在 Y42 之前，坎特威尔是夸尔特里茨 EMEA 的首席运营官。

“科技行业正处于一个转折点，公司正在收集的数据的广度和数量正在迅速超过他们有效管理数据的能力。坎特威尔说:“现代数据运营云以可扩展的方式管理数据管道的需求已经成为一项关键任务。“Y42 拥有产品愿景、客户证据和充满激情的团队，能够真正改变数据管理领域。”

该公司最近还聘请了数据集成平台木桶公司(谷歌于 2018 年收购)的前 CMO Max Herrmann，以及最近在 Swim.ai 担任 CMO 的高级营销副总裁。
