AWS 推出新的 SageMaker 功能，使扩展机器学习更容易| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/12/01/aws-launches-new-sagemaker-features-to-make-scaling-machine-learning-easier/

在其年度 re:Invent conference 上，AWS 今天为 SageMaker、该公司的托管服务推出了一系列新功能，用于构建、训练和部署机器学习(ML)模型。亚马逊机器学习副总裁 Swami Sivasubramanian 表示，新功能旨在让用户更容易在他们的组织中扩展机器学习。

首先，AWS 推出了一项新的 SageMaker Ground Truth Plus 服务，该服务使用专家劳动力来更快地提供高质量的训练数据集。SageMaker Ground Truth Plus 使用包括机器学习技术的标签工作流程进行主动学习、预标签和机器验证。该公司表示，新服务可以降低高达 40%的成本，并且不需要用户拥有深度机器学习专业知识。该服务使用户无需构建标注应用程序即可创建训练数据集。SageMaker Ground Truth Plus 目前在北弗吉尼亚可用。

该公司还推出了一款新的 SageMaker 推理推荐器工具，帮助用户选择最佳可用的计算实例来部署机器学习模型，以实现最佳性能和成本。AWS 表示，该工具会自动选择正确的计算实例类型、实例数量、容器参数和模型优化。亚马逊 SageMaker 推理推荐器一般在除 AWS 中国区以外的所有提供 SageMaker 的地区都有。

此外，AWS 发布了新的 SageMaker 无服务器接口选项的预览，该选项允许用户轻松部署机器学习模型进行推理，而无需配置或管理底层基础设施。新选项在北弗吉尼亚、俄亥俄、俄勒冈、爱尔兰、东京和悉尼可用。

图片来源: TechCrunch

凭借 SageMaker 训练编译器，AWS 今天推出了一项新功能，通过更高效地使用 GPU 实例，可以将深度学习模型的训练速度加快高达 50%。该功能涵盖了从高级语言表示到硬件优化指令的深度学习模型。这项新功能在北弗吉尼亚、俄亥俄、俄勒冈和爱尔兰普遍提供。

最后，AWS 宣布用户现在可以只需点击一下鼠标，就可以直接从 SageMaker Studio 笔记本上监控和调试他们在 Amazon Elastic MapReduce (EMR)上运行的 Apache Spark 作业。该公司指出，你现在还可以直接从 SageMaker Studio 发现、连接、创建、终止和管理 EMR 集群。

AWS 在一篇博客文章中解释说:“因此，与 EMR 的内置集成使你能够在单个通用 SageMaker Studio 笔记本电脑中进行 peta 字节规模的交互式数据准备和机器学习。”

新的 SageMaker Studio 功能可在北弗吉尼亚、俄亥俄州、北加利福尼亚、俄勒冈州、加拿大中部、法兰克福、爱尔兰、斯德哥尔摩、巴黎、伦敦、孟买、首尔、新加坡、悉尼、东京和圣保罗使用。

与此相关的是，AWS 今天还推出了 SageMaker Studio Lab ，这是一项免费服务，旨在帮助开发者学习机器学习技术并试验该技术。昨天，AWS 宣布了一项名为亚马逊 SageMaker Canvas 的新机器学习服务。这项新服务将允许用户使用点击式界面建立机器学习预测模型。
