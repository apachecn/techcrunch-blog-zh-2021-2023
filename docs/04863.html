谷歌文档现在可以让你和其他人一起起草电子邮件，并将它们导出到 Gmail

谷歌公司周三宣布，谷歌将在“谷歌文档”中引入一项新功能，旨在简化电子邮件草稿的协作。新的谷歌文档电子邮件草稿模板允许用户起草电子邮件，然后将其导出到 Gmail。

此次发布是谷歌智能画布的一部分，它利用了“@菜单”，让用户可以快速执行操作。在 Google 文档中输入“@email”即可访问电子邮件草稿模板。这样做将显示一个包含收件人、抄送、密件抄送和主题行的模板。当您准备发送电子邮件时，您可以选择 Gmail 图标将草稿导出到电子邮件服务。将弹出一个 Gmail 撰写窗口，所有电子邮件字段将自动填写您在 Google Doc 的电子邮件草稿中输入的信息。

图片来源:谷歌

“我们正在用新的电子邮件草稿模板使在 Docs 中协作电子邮件草稿变得容易，”谷歌在一篇关于新功能的博客文章中说。您可以使用@菜单在收件人栏中提及联系人，而不必记住他们的电子邮件地址，并使用评论和建议在邮件正文中进行协作

该功能于 2 月和 首次推出，现在开始向所有 Google Workspace 客户以及传统 G Suite 基本版和商业版客户推广。谷歌没有提到个人使用的可用性，但 TechCrunch 已经联系了该公司以了解更多信息。值得注意的是，Gmail 功能通常会先向 G 套件客户推出，然后再面向消费者推出。例如，商业客户比消费者更早接触到 Gmail 与聊天、会议和房间的深度整合。

新的电子邮件草稿模板将在团队需要一起协作处理电子邮件的情况下非常有用。虽然你总是可以从谷歌文档和 Gmail 中复制和粘贴文本，但这种新的集成使得在一个地方使用这两种产品更容易，更快地完成工作。
