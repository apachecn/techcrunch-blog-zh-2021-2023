俄罗斯游戏开发商禁令和 doxes 6,700 骗子| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/06/russian-game-developer-bans-and-doxes-6700-cheaters/

骗子是几乎所有在线视频游戏中令人讨厌的一部分。禁止它们已经成为游戏开发商和发行商取悦用户的重要手段。

俄罗斯公司 Battlestate Games 开发的游戏《逃离塔尔科夫 》的发行商为这一常规游戏增加了一个不同寻常的转折:点名羞辱作弊者。上周，Battlestate Games 表示，它禁止了 6700 名作弊玩家，并在上公开发布了他们所有的昵称电子表格。

“我们希望诚实的玩家看到骗子的昵称，知道正义得到了伸张，在突袭中杀死他们的骗子已经受到惩罚并被禁止，”Battlestate Games 的发言人 Dmitri Ogorodnikov 告诉 TechCrunch。

其他公司，如《使命召唤》发行商 Activision ，或 Valorant 的开发商 Riot Games ，通常只是宣布他们禁止的玩家数量——而不是他们的昵称和手柄。

业内人士认为，Battlestate Games 采取的方法可能更有效，也能起到更好的威慑作用。

“好的。我希望我们也这样做，”一家视频游戏公司的员工告诉 TechCrunch，他在反作弊部门工作，要求匿名，因为他们没有被授权对媒体说话。“许多骗子给他们的朋友和其他重要的人制造虚假的形象，当他们的欺骗行为被公开曝光时，这会给他们的名声带来耻辱，并阻止他们重复这种行为。

“然而，在某些情况下，尽管被禁止，这些人只是购买一个与旧帐户相似的新帐户，并继续欺骗而不承担后果，欺骗任何与他们建立了游戏关系的人。此外，试图成为职业选手并赢得比赛的骗子将被暴露在公众面前，这样其他选手就再也不会给他们机会了，”他们说。“你破坏了信任，你不应该有机会回来。”

作弊软件本质上是利用视频游戏中的漏洞，让玩家在玩在线游戏时获得不公平的优势。通常，作弊软件会提供一些功能，例如“wallhack”，让玩家可以透过墙壁看到他们的对手，或者“aimbot”，这是一个自动指向敌人的系统，可以更容易地杀死他们。

制作作弊器已经成为一项大生意。2021 年，中国警方与科技巨头腾讯合作，捣毁了一个视频游戏作弊组织，该组织通过向 PUBG Mobile 出售作弊软件赚了超过 7000 万美元。
