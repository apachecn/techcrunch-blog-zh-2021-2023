印度解除对 PayU 的 LazyPay 和其他一些借贷应用的禁令

原文：https://web.archive.org/web/https://techcrunch.com/2023/02/10/india-lifts-ban-on-payus-lazypay-and-some-other-lending-apps/

印度解禁 PayU 的 LazyPay 和其他一些借贷应用

据一位知情人士透露，印度 IT 部已经解除了对七款备受瞩目的贷款应用的禁令，包括 PayU 的 LazyPay、Kissht、KreditBee 和 Indiabulls 的 Home Loans，这为近几个季度受到巨大审查的金融科技行业提供了一些缓解。

根据 TechCrunch 看到的一份命令副本，上周五，IT 部还解除了对 mPokket、Buddy Loan 和 Faircent 的禁令。这位人士说，在这些公司证明它们的资本表中没有中国投资者后，禁令被取消了。

国有广播公司 Prasar Bharati 称，印度最近几周推动禁止了 90 多个贷款应用程序，以保护国家的诚信，遏制中国在南亚市场的影响力。

据 TechCrunch 本周早些时候报道，官员们表示，IT 部对中国投资者过去和现在出现在印度一些贷款应用的资产负债表上感到担忧。

另一个担忧是与中国有关的网络犯罪报告不断增加。据 TechCrunch 早些时候报道，官员们说，内政部收到了涉及中国公司的犯罪活动报告，这些公司利用 API 访问印度贷款应用程序，并获取和存储印度消费者在国外的数据。

在一份声明中，Kissht 证实其网站和应用程序的禁令已经解除。“我们感谢印度政府和 MeitY 撤销了屏蔽 kisht 的命令，”kisht 发言人在一份声明中说。

“政府在确保 Kissht 等可信且完全合规的应用程序继续努力提高该国的金融包容性方面表现出了不懈的支持。自 2017 年以来，我们已经为 800 多万印度人提供了服务，并期待在未来几年中为更多的人提供服务。Kissht 将继续提供无忧信贷，目标是在 2023 年将我们的客户群翻一番。”
