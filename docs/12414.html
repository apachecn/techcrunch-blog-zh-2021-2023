量化全球电子商务放缓

新冠肺炎疫情有很多东西。全球传染。健康灾难。预示着新的地缘政治紧张局势，并长期评论我们愿意在多大程度上保护——或不保护——我们的人类同胞。这也是一场震撼全球大多数行业的商业地震。

但随着 COVID 的出现，打破了供应线和商业模式，它也很快消失了。随着世界上大多数人开始学会忍受——或者仅仅是决定忍受——疫情对健康的影响，许多行业迅速恢复到以前的状态。航空公司从垃圾变成了头等舱；相比之下，科技公司从宠儿变成了贬义词。

该交易所探索初创企业、市场和资金。

一些科技公司在 COVID 期间顺风顺水，因为新改革的商业环境帮助它们增长了一段时间。你可以在这里撒一张大网:Robinhood 爆炸式增长的部分原因是消费者比平时拥有更多的现金，Instacart 看到了对其杂货配送服务的爆炸性需求。一些科技公司走了另一条路，就像在 COVID 的最初几个月，Airbnb 的业务陷入困境一样，因为去的地方一夜之间从理想变成了疯狂。

自回归常态以来，最初受 COVID 影响的业务已经制定了不同的路线。Robinhood 失去了一些光彩，因为它的用户群，按照通常聊天的说法，回到了外面。Instacart 增长缓慢，但成功保持了疫情时代的增长势头。

Airbnb 是 COVID 可能导致昔日健康公司裁员的一个早期例子，该公司恢复了元气，并自上市以来保留了大部分价值，这对其 IPO 同行来说是一个罕见的壮举。
