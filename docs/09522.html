T-Mobile 手机将从明年开始连接 Starlink

原文：https://web.archive.org/web/https://techcrunch.com/2022/08/25/t-mobile-phones-will-connect-to-starlink-for-free-starting-next-year/

T-Mobile 和 SpaceX 已经宣布，明年发射的 Starlink 卫星将能够通过现有的蜂窝频段直接连接到运营商的手机。这些公司希望在有卫星覆盖的地方实现全球漫游，这项服务可能会免费加入现有的 T-Mobile 计划。

SpaceX 的 Elon Musk 和 T-Mobile 的 Mike Sievert 在该航天公司位于德克萨斯州的 Starbase 宣布了“技术联盟”。

“这很像在天上放一个蜂窝塔，只是难度大得多，”西弗特说。“你的手机并不知道它正在连接太空。它会认为它连接到一个手机信号塔，因为该手机使用行业标准技术通信协议，并且它已经内置了频谱，就像当今流通的绝大多数手机一样。”

尽管两位高管都强调了声明的“深刻性”，但这并非完全没有先例。事实上，这正是 Lynk 去年用其仍在运行的测试卫星展示的内容——这家初创公司计划从一个小型星座向全球提供紧急信息和警报。(“让 Elon 验证这是一项服务，这很棒。这是一个巨大的需求，巨大的市场和巨大的验证，”Lynk 创始人查尔斯·米勒在宣布后告诉我。他还指出，他们拥有专利和 FCC 许可证，所以我们将看看情况如何。)

无可否认，Starlink 确实在轨道上有更大的存在，理论上允许它提供更频繁和高带宽的连接。

但理论上是这个词，因为这种连接性似乎还没有在轨道上演示过。(至少他们没有说是——我已经联系了 T-Mobile 以了解更多信息，如果有回音，我会及时更新。)

“我们需要做的不仅仅是重新编程卫星；我们正在制造一种特殊的天线，世界上最先进的相控阵天线。他们必须从你的手机上接收到一个非常安静的信号，”马斯克在活动上说。“有相当多复杂的硬件和软件，因为它移动得如此之快——它们正以每小时 17，000 英里的速度在头顶上行进。通常情况下，手机信号塔的速度不会达到每小时 17000 英里。”

起初，该服务将只支持短信，可能还有消息应用，尽管 Sievert 警告说，“我们实际上还没有开始与其他公司合作……这是一个技术问题，我们需要合作伙伴的帮助。”在发送或接收消息之前，可能还会有“半小时左右”的相当大的延迟。

然而，马斯克建议，天线和服务的规格可能会使多个语音通话和实时信息及时。

这项服务目前只能使用 T-Mobile 自己的授权频谱，这意味着这是一项独家服务。然而，Sievert 和 Musk 都表示，他们希望与使用兼容频谱的其他国家的提供商达成协议，提供“互惠漫游”协议。那也有待决定。

Sievert 反复指出，产品和服务本身仍在敲定中，但随着相关公司了解细节和技术本身得到展示，你可以期待在未来几个月听到更多消息。
