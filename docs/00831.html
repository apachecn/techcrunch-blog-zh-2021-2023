礼物指南:给有抱负的年轻建造者的 20+ STEM 玩具礼物想法

原文：https://web.archive.org/web/https://techcrunch.com/2021/11/17/stem-toys-for-kids-and-teens/

欢迎来到 TechCrunch 的 2021 年节日礼物指南！需要礼物创意方面的帮助吗？我们有很多。我们刚刚开始推出今年的礼物指南，所以请从现在起到 12 月底查看更多内容！

对于今年的 STEM 玩具礼物指南，我们按照年龄划分了我们的建议，以便于浏览。20 多个礼物创意(如下)涵盖了从由彩色积木控制的火车套装到会画画的机器人，一直到可爱的 DIY 掌上游戏机，这真是一个供青少年搭建的实验平台。

我们挑选的礼物达到了一个价格区间——从 15 美元起，最高不到 550 美元(适合你的孩子可能需要的所有教育乐高玩具！)，价格区间介于两者之间。

学习代码类别作为一个整体继续成熟，显示出对艺术和设计的加强(和受欢迎的)关注，而不仅仅是纯工程。与此同时，很明显，维持一个销售教育小发明/游戏的企业是具有挑战性的，自从我们上次入住以来，许多玩家眨眼就消失了(或退出了)。产品的新鲜感似乎也在减少，尽管制造商硬件本身正在蓬勃发展(多亏了像树莓派这样的产品)。但是，总的来说，这一类别的实验性“寒武纪大爆发”时刻似乎已经过去，可编程机器人(大部分)已经取而代之。

整合仍然是这个领域的一大主题。枢轴也是如此(例如，见卡诺的新果酱)。毕竟，孩子们是善变的，即使是最漂亮的玩具也可能很快被丢弃，换成更新、更闪亮的东西。此外，感觉像是一些早期的炒作(和大声宣称)——围绕着“教授编码”的小发明——已经褪色为一种更实际/现实和不那么浮华的潜在教育价值的投影。

STEM 玩具制造商面临的一个额外挑战是(现在)对儿童屏幕时间的高度关注。因此，许多产品的营销特点是大肆宣扬“无屏幕”替代教学编码(例如通过使用物理块/卡/按钮等)。与此同时，其他一些确实需要屏幕才能工作的人正试图区分他们提供的“好屏幕”时间与令人上瘾的非学习型游戏和/或社交媒体的“数字糖”……父母是否会购买这一点仍有待观察。

对于幸存的 STEM 玩家来说，他们越来越多的时间和精力正从消费者领域转向直接向学校提供学习装备和资源——追求更可靠的收入流，尽管向学校销售也不是轻而易举的事情。总的来说，成为更大的制造商市场或更广泛的教育企业集团的一部分似乎是许多幸存的 STEM 创业公司的方向。

Kibo 10 家庭版包(如上图)包含 Kibo 机器人，它有一个可抽拉的面板、轮子和马达，以及用于创建 Kibo 程序的可扫描卡。

Osmo Explorer 入门套件

印度教育技术巨头 Byju 拥有的 Osmo 已经为平板电脑制作教育游戏近十年了。它的大转折是结合物理(离线)发挥(笔，块，卡等。)与数字屏幕上的内容和互动。

它通过一个专用的平板电脑支架来实现这一点，该支架在 iPad 的前置摄像头上添加了一个反射器，这样它就可以直接看到屏幕下方的物理游戏区。这意味着该应用程序能够反映/反映孩子们在物理空间中的行为。(有限的视野意味着相机只能捕捉小手，而不是你孩子的脸，所以这是隐私的一大优势。)奥斯莫称这种物理和数字游戏的融合为“体验学习”

图片来源: Osmo

Osmo Explorer 初学者工具包被描述为“迄今为止最完整的 STEAM 学习工具包”，捆绑了一系列互动游戏和美术用品(注意:不包括平板电脑，所以你需要自己的 iPad 或等效设备)。该套件包括一套专注于编码的游戏，让孩子们通过将(物理)编码块组合成一组指令并点击屏幕来执行他们的(原型)程序，来操纵屏幕上的角色前进。

年龄: 5-10
价格:158 元起 Osmo

机器人意识

欧洲初创公司 Robo Wunderkind 已经建立了 STEM 学习业务，一次一个模块化机器人模块。孩子们可以通过将各种智能模块连接在一起来学习电路和工程概念——构建自己的机器人，并让它们移动或以其他方式制作动画。创作可以通过添加(实际的)乐高积木进一步扩展。

图片来源:机器人神童

除了纯粹的物理游戏，Robo Wunderkind 还有一个以配套应用程序形式出现的数字助手，可以将学习扩展到屏幕上的编码。在这里，它提供了分层的复杂性——从孩子们可能只是操纵彩色图标的编码基础到基于拖放块的编码(基于 Scratch)，有三个不同的级别——这引入了更复杂的概念，如变量、函数、运算符和输入/输出处理。更高级的程序员也支持 Python 和 Arduino APIs。

Robo Sense kit 提供了 Robo Wunderkind 方法的品尝者，有几个感测块可以玩，还有 30 多个项目和教程可以在线或在应用程序中访问。

年龄: 5-12 岁
价格:【99】来自 Robo 神童
制作人: Robo 神童

对于一个预算友好但无止境探索的挑战，为什么不给你的孩子一个树莓派呢？低成本的微处理器抢占了当前的 STEM 设备浪潮——提供了一种“无装饰”的方法来让孩子学习编码，这种方法结合了便宜但功能强大的硬件和最少的手持操作。Pi 已经成为一个制造商的发电站。

图片来源:树莓派基金会

Pi 基金会有一个社区构建的 Pi Zero 项目来激发你的孩子。

AdaBox

AdaBox 是 DIY 电子项目的季度(即每三个月一次)订阅箱，由 AdaFruit 策划。你不会提前知道你会得到什么样的电子设备和机器人——但是，对于更高级的有电脑的青少年来说，这个黑盒子的拼凑硬件可能正是他们需要的灵感。

图片来源:阿达果

AdaFruit 表示，订阅服务是“为所有水平的制造商设计的，特别关注刚刚起步的人。”教程和视频通过学习系统在其网站上提供。对硬件进行编程需要访问连接互联网的设备。

过去的项目包括建立自己的矩阵门户流可视化。

(注意:有一个专用的“赠送”选项，用于在结账时赠送 AdaBox。然而，随着假日季节的快速临近，AdaFruit 表示，第一箱新订单要到春天才会发货，所以作为假日礼物订购这种产品需要收件人有一点耐心。)

微微系统

为什么不从总部位于英国的 Pimoroni 公司购买这款由 Raspberry Pi Pico 驱动的小型手持游戏系统呢？这不仅是一款小型游戏主机，还是一个实验性的游戏平台。

图片来源:皮莫罗尼

可以使用各种语言为 PicoSystem 编写游戏代码，包括 C++/MicroPython(根据 Pimoroni 的说法:“我们官方的 PicoSystem API 旨在轻量级，易于使用，并且不会妨碍您开发游戏”)——请参见他们的教程开始吧。

开箱后，小小的手持飞船上闪现着超级方块兄弟。由蝎子游戏。

Piper Make Starter +机器人探险套件

Piper 是 STEAM 玩具领域的另一个长期参与者——早在 2014 年就开始提供 DIY《我的世界》电脑来教孩子们编码。它仍在销售其经典的、木制外壳的树莓 Pi 驱动的电脑套件，但已经扩展到销售一系列制作套件。

包括——2021 年的新品(定于本月发货)——机器人探险套件，，它捆绑了必要的 Pi Pico(和其他启动器电子必需品)。孩子们可以建造几个运动机器人(一个步行机和一个漫游车)，然后通过 Piper 的拖放编码平台 Make 帮助他们编写代码来移动它们。所以他们需要使用电脑或移动设备。

图片来源:派珀

Piper 的编码平台包括微控制器的虚拟表示和基于块的编程语言到基于文本的 CircuitPython 的自动翻译，以支持青少年学习硬件编码的基础知识。

各种其他项目也可以通过 Piper Make 门户网站获得——该网站称，它使用基于讲故事的课程来激励年轻的学习者。它的 Make 编程软件也可以通过一个手机应用获得。
