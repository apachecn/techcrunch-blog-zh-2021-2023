AWS 针对能源行业推出了 AWS 能源能力计划 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/11/30/aws-targets-the-energy-industry-with-launch-of-the-aws-energy-competency-program/

亚马逊今天宣布一项新计划，将正式确定为能源行业开发专业解决方案的 AWS 合作伙伴。新的 AWS 能源能力计划将确定这些专业合作伙伴，他们已经展示了技术专长和客户成功，可以帮助世界各地的能源生产商利用最新技术建立和实施 AWS 驱动的解决方案，同时引导向更可持续的能源未来过渡。

今天，许多 AWS 合作伙伴与能源行业合作，帮助他们利用 AWS 技术，包括管理从勘探阶段到采集、加工、运输、管理和维护这些操作的石油和天然气资产的操作，以及可再生和可持续领域的操作。亚马逊表示，这些行业现在正在加速采用云计算、物联网(IoT)、人工智能(AI)和机器学习(ML)等解决方案。

新的 AWS 能源能力计划将帮助行业更好地确定在与世界能源生产商合作时拥有特定专业知识的 AWS 合作伙伴，包括那些可以帮助他们开发包含可持续和可再生能源资产的投资组合的合作伙伴。

在启动时，该计划包括 32 个全球合作伙伴，他们也与 AWS 合作，帮助开发能源生产商的新计划。这个群体包括那些可以在专业解决方案领域、行业垂直领域或工作负载方面提供帮助的人。亚马逊指出，这是目前 AWS 合作伙伴能够实现的最艰难的指定之一，因为合作伙伴必须经历与能源行业特定最佳实践相关的严格技术验证。

图片来源:亚马逊

其中一个合作伙伴 Slalom 一直在帮助北美能源公司 TC Energy 管理 44，000 公里的天然气管道。TC Energy 与合作伙伴一起开发了一种商业智能应用程序，该应用程序使用机器学习来帮助客户优化天然气调度和吞吐量。另一个合作伙伴是人工智能视频分析提供商 Unleash Live，它帮助工业工程解决方案提供商 Worley，后者运营风力发电场、太阳能公园、水电、天然气项目等。Worsley 正在使用“释放现场”来利用无人机和计算机视觉进行检查。

亚马逊表示，其他合作伙伴可以帮助提供包括上游、中游、下游、新能源、核心商业应用、健康、安全和环境以及数据和分析在内的解决方案。现在有兴趣加入该计划的 AWS 合作伙伴可以查看针对 AWS 服务合作伙伴和 AWS 软件 合作伙伴的验证清单，开始行动。
