微软向 OpenAI 投资数十亿美元，扩展合作关系

原文：https://web.archive.org/web/https://techcrunch.com/2023/01/23/microsoft-invests-billions-more-dollars-in-openai-extends-partnership/amp/

图片来源: OpenAI

微软今天表示，它将扩大与 OpenAI 的合作伙伴关系，open AI 是艺术和文本生成人工智能系统背后的初创公司，如 ChatGPT 、 DALL-E 2 和 GPT-3 ，投资“多年，数十亿美元”。OpenAI 表示注入的新资金——具体数额没有透露——将用于继续其独立研究和开发“安全、有用和强大”的人工智能。

光学对微软来说不是最好的，上周宣布计划裁员 10，000 人，作为更广泛的成本削减措施的一部分。但是他们在本月早些时候已经收到了公司的电报——在与华尔街日报的采访中，微软首席执行官塞特亚·纳德拉说微软计划将 OpenAI 的基础系统作为商业平台，这样任何行业的任何实体都可以基于它们。

作为与微软新投资协议的一部分，OpenAI 仍将是一家利润受限的公司。在这种模式下，投资者的回报被限制在他们投资的 100 倍以内——或者将来可能更少。

在一篇博客文章中，微软表示，它将增加对专用超级计算系统部署的投资，以加速 OpenAI 的人工智能研究，并将 OpenAI 的人工智能系统与其产品集成，同时“引入新类别的数字体验”。这家科技巨头的 Azure 云平台将继续成为 OpenAI 的独家云提供商，为这家初创公司的研究、产品和 API 服务工作负载提供动力。

之前有传言称，微软正在准备将 ChatGPT 与 Bing 搜索结果整合，并将 OpenAI 的语言人工智能技术引入 Word、PowerPoint 和 Outlook 等应用程序。几年前，微软旗下的 GitHub 联合开发并推出了 Copilot ，一个代码生成人工智能系统。微软已经将包括 GPT-3 和 DALL-E 2 在内的 OpenAI 创新融入到应用和服务中，如 Power Apps、微软 Edge 和即将推出的设计器。

这些努力建立在微软和 OpenAI 多年的密切合作基础上。2019 年，微软宣布将向 OpenAI 投资 10 亿美元(大致一半以 Azure credits 的形式)，共同开发 Azure 平台的新技术，并“进一步扩展”OpenAI 的大规模 AI 能力。作为交换，OpenAI 同意将其部分知识产权许可给微软，该公司随后将这些知识产权商业化并出售给合作伙伴，并在 OpenAI 致力于开发下一代计算硬件时，在 Azure 上训练和运行人工智能模型。

一年后，微软透露它已经建造了一台 Azure 托管的、OpenAI 合作设计的超级计算机，在当时是世界上最强大的机器之一。然后在 2021 年，微软推出了 Azure OpenAI 服务，该服务旨在让企业访问 OpenAI 的人工智能系统，包括 GPT-3 以及安全、合规、治理和其他专注于商业的功能。

纽约时报报道称，微软在 2019 年至 2023 年初期间向 OpenAI 追加投资 20 亿美元。这个科技巨头也成为了 OpenAI 的创业基金的关键支持者，OpenAI 的专注于人工智能的风险投资和科技孵化器项目。

纳德拉在一份声明中说:“我们与 OpenAI 建立了合作伙伴关系，共同致力于负责任地推进前沿人工智能研究，并将人工智能作为一个新的技术平台民主化。”“在我们合作的下一阶段，各行业的开发人员和组织将可以使用 Azure 获得最好的人工智能基础设施、模型和工具链来构建和运行他们的应用程序。”

“过去三年我们的合作非常好，”OpenAI 首席执行官山姆·奥尔特曼在一份新闻稿中补充道。“微软分享我们的价值观，我们很高兴能够继续我们的独立研究，并致力于创造有益于每个人的先进人工智能。”

消息来源此前报道称，微软希望获得 open ai 49%的股份，对该公司的估值约为 290 亿美元。根据 Semafor 详述的一项提案的条款，微软将获得 OpenAI 四分之三的利润，直到它收回高达 100 亿美元的投资，包括 Khosla Ventures 在内的其他投资者持有 49%的股份，OpenAI 保留剩余的 2%股权。

OpenAI 面临着在 ChatGPT 等产品上实现盈利的压力。初创公司预计将在 2023 年从授权和高级产品(如 ChatGPT Professional )中赚取 2 亿美元，与迄今为止已经投入的数十亿美元相比，这是微不足道的。

这要归咎于培训、开发和运行大型人工智能系统的高昂成本。根据 Altman 的说法，仅 ChatGPT 的运营费用就“令人眼馋”，相当于每次聊天花费几分钱。(截至 12 月初，ChatGPT 拥有超过 100 万用户。)与此同时，据估计 GPT-3 的发射已经花费了数百万美元。

open AI——推而广之，还有微软——面临着严峻的法律挑战，这些挑战可能会阻碍他们人工智能探索的发展。美国专利商标局(USPTO)最近提出撤销对人工智能创作的漫画的版权保护，称版权作品需要人类作者。微软、GitHub 和 OpenAI 目前正被集体诉讼起诉,指控它们允许 Copilot(上述代码生成系统)在不提供信用的情况下重复使用部分授权代码，违反了版权法。

一些法律专家认为，如果 Copilot 不经意地将该工具中受版权保护的建议整合到他们的生产软件中，可能会给公司带来风险。正如 Elaine Atwell 在 Kolide 的企业博客上的一篇文章中指出的，因为 Copilot 剥离了其许可证的代码，所以很难区分哪些代码是允许部署的，哪些代码可能有不兼容的使用条款。

这场诉讼还对 DALL-E 2 这样的生成艺术人工智能产生了影响，类似地，DALL-E2 也被发现从它们被训练的数据集(即图像)中复制和粘贴。OpenAI 的几个竞争对手最近在一起法律案件中成为目标，指控他们侵犯了“数百万艺术家”的权利，他们在“没有原始艺术家的同意”的情况下，在从网络上刮来的数十亿张图像上训练人工智能艺术生成工具。

像 Getty Images 这样的平台已经禁止了人工智能生成的内容，因为担心潜在的法律反弹。

抛开法律问题不谈，OpenAI 的文本生成技术因其有时能够给出听起来令人信服但事实并非如此的答案而受到抨击。本月早些时候， Q &一个编码网站栈溢出暂时禁止用户分享 ChatGPT 生成的内容，称人工智能让用户太容易产生回应，并让可疑的答案充斥网站。ChatGPT 也被几个公立学校系统和至少一个学术会议禁止。

这就是说，OpenAI 和微软都有自己的工作要做。
