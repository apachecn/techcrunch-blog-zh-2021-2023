NetApp 收购 Instaclustr 以提供开源数据库即服务

生产软件的公司越来越依赖开源数据库来构建他们的程序，但是使用所有这些不同的产品是很复杂的。加州的一家初创公司 Instaclustr 希望通过提供流行的开源数据库服务来改变这一现状。

今天，NetApp 以未披露的金额收购了该公司。Instaclustr 为 NetApp 客户提供了一种安装流行开源数据库的方法，而无需担心处理原始开源数据库的问题。他们支持的项目包括 PostgresSQL、Apache Kafka、OpenSearch、ElasticSearch 和 Apache Cassandra。

这是该公司一系列小型收购中的最新一起，该公司传统上提供数据和存储管理服务。在宣布该交易的博客文章中，NetApp 执行副总裁兼总经理 Anthony Lye 表示，此次收购最终是为了通过简化软件安装来改善客户体验。

“收购 Instaclustr 是 NetApp 的下一个战略步骤；为客户做更多，简化体验，不断优化和保护平台，这样客户就不用自己动手了，”他写道。

结合该公司最近进行的其他收购，这里有一个模式，即该公司试图从传统的存储供应商转向与当今寻求管理云中工作负载的客户更相关的供应商。

“我们已经进行了一系列战略收购，包括 Spot、CloudCheckr 和 Fylamynt，以提供 FinOps。借助 Cloud Hawk、Cloud Secure 和 CloudCheckr，我们现在可以提供 SecOps。我们为客户最迫切的云需求提供解决方案——规模、性能、速度、效率、安全性和成本，”Lyn 写道。

Instaclustr 的首席技术官和联合创始人本·布罗姆黑德(Ben Bromhead)认为，这两家公司为客户创造了一个合乎逻辑的组合。“从技术和产品的角度来看，NetApp 强大的基础设施解决方案与 Instaclustr 的数据层即服务解决方案和服务完美结合，”他在一份声明中说。

根据 Crunchbase 的数据，Instaclustr 成立于 2013 年，融资约 2200 万美元。NetApp 成立于 1992 年，1995 年上市。股票今天下午略有上涨。该交易有待监管机构批准，预计将于 2023 财年第一季度完成。
