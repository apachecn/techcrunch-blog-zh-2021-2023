多样化是 TechCrunch 的核心使命。我们相信每个人，无论背景如何，都应该有机会接触科技世界的无限可能，同时，我们相信更加多样化的劳动力只会加强科技企业。TechCrunch 的 Include 计划旨在通过应用 TechCrunch 独有的资源(包括我们的编辑和活动平台)以及在 TechCrunch 自己的员工和文化中体现多元化使命来促进多元化。

核心原则

随着时间的推移，Include 计划将包含许多项目和实验，但无论 Include 计划采取何种形式，这些指导原则都将是核心:

  • 协同开放。包容性的努力将呼吁更广泛的社区的想法和支持。我们将与 TechCrunch 以外的个人和组织合作，他们可以提供建设性的见解和批评，并从 TechCrunch 的资源中受益。
  • 透明负责。TechCrunch 将公开宣布所有 Include 计划，并公布每个计划的结果。这些信息可以在 TechCrunch 的 Include 频道上追踪到。
  • 进化。随着 Include 的发展，我们预计会遇到成功和失败，但我们会将我们所学到的知识应用到新的项目和方法中。

从广义上来说，TechCrunch Include 旨在为那些在科技界历史上未被充分代表的人提供机会。我们最初的努力将集中在技术领域代表性不足的群体上，包括但不限于黑人和非洲裔美国人、拉丁裔和西班牙裔美国人、土著美国人、妇女、美国老年人、经济弱势群体和 LGBTQ 群体。我们认识到这不是全部情况。随着时间的推移，TechCrunch 将对谁有资格参加 Include 计划持灵活的观点。

包括 2016 年的举措

编辑的

  • 增加创始人、投资者、法官和创始人在 TechCrunch 舞台上的比例，扰乱来自代表性不足的背景的活动和 Crunchies。
  • 创建一个框架来评估对不同公司、创始人和主题的报道，
  • 聘请一名作家，主要关注科技领域的多样性问题，

报告

  • 成立一个顾问委员会，作为包括倡议和反馈意见论坛。

事件

至少跟踪和报告战场参赛者和裁判的性别和种族分类。只要有可能，将努力报告年龄、性别认同和社会经济地位。

  • 至少跟踪并报告 Disrupt 发言人的性别和种族分类。只要有可能，将努力报告年龄、性别认同和社会经济地位。
  • 向未被充分代表的团体或组织提供 X%的门票，免费或大幅折扣，以推广他们的技术。
  • 在每个 Disrupt 举办一场女性科技(Crunch) 活动。
  • 开发 an Include“gala”的概念，以筹集资金，并在 2016 年的一段时间内为致力于技术多元化的团体提供曝光机会。
  • 为松脆饼干设立一个多样性奖。
  • 主持人包括与合作伙伴风投和 TechCrunch 撰稿人的办公时间。目标是每年工作大约 12 个小时。

媒体

  • 在 TechCrunch 上推出 Include“频道”。
  • TechCrunch diversity beat 的相关帖子。
  • Include 计划的更新。
  • 来自合作组织的更新包括。
  • 致力于提高技术多样性的组织的众包目录。
  • 与 Include 任务相关的事件的众包列表。
  • 推荐链接和数据:链接到有用的帖子和数据，这些帖子和数据与科技领域的多元化努力有关。

员工和文化

  • TechCrunch 将每年报告自己员工的人口构成。
  • TechCrunch 将举办定期培训课程，主题包括与 Include 任务相关的管理主题。
  • TechCrunch 将发布一份年度报告，对 TechCrunch 的影响包括。
  • TechCrunch 将与其顾问委员会召开季度会议和晚宴。
