AWS 获得无代码 ML 服务 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2021/11/30/aws-gets-a-no-code-ml-service/amp/

图片来源:罗恩·米勒

AWS 今天宣布一项新的机器学习服务，亚马逊 SageMaker Canvas。与现有的机器学习服务不同，这里的目标受众不是高度技术化的数据科学家和工程师，而是公司内部的任何工程师或商业用户。SageMaker Canvas 的承诺是，它将允许任何人使用点击界面来建立机器学习预测模型。

如果这听起来很熟悉，那可能是因为 Azure 和其他公司提供了类似的工具，尽管 AWS 可能有很多公司已经在 AWS 中存储了所有数据的优势。

图片来源:亚马逊

AWS 的 Alex Casalboni 在今天的公告中写道:“SageMaker Canvas 利用与亚马逊 SageMaker 相同的技术，自动清理和组合你的数据，在引擎盖下创建数百个模型，选择表现最好的一个，并生成新的个人或批量预测。”“它支持多种问题类型，如二元分类、多类分类、数值回归和时间序列预测。这些问题类型让您无需编写一行代码即可解决关键业务用例，例如欺诈检测、减少流失和库存优化。”

不出所料，这项服务得到了 SageMaker 的支持，SageMaker 是 AWS 的完全托管的机器学习服务。

这里的一般思想是，用户可以使用任何数据集，下至基本的 CSV 文件，然后决定这个数据集画布中的哪些列应该预测。这种模式不用担心怎么训练。然而，虽然这是一个比使用传统的 ML 工具更容易的用户体验，我们仍然没有谈论拖放。毕竟这是 AWS。总体体验更类似于在 AWS 控制台中工作，而不是现代的无代码应用程序。
