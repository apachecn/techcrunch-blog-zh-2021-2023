凭借 4200 万美元的资金，Bonfire Union 希望投资 web3

原文：https://web.archive.org/web/https://techcrunch.com/2022/07/06/bonfire-union-mask-network-maiden-fund/amp/

图片来源: 杰弗里·柯立芝 /盖蒂图片社

币安支持的初创公司【Mask Network 有一个雄心勃勃的目标:像腾讯在 Web 2.0 中所做的那样，成为 web3 中的企业风险投资巨头。Mask Network 使用户能够在 Twitter 等成熟的互联网平台上发送加密货币和加密信息。

为此，这家成立五年的公司最近完成了其风险投资部门 Bonfire Union 的首次关闭。该公司有了 4200 万美元的新资金，计划投资于 web3 的“社交生态系统”,这是一个主要建立在区块链基础上的互联网版本，承诺更加去中心化。

尽管在令牌价格暴跌的情况下，该行业前景黯淡，但该公司仍设法吸引了密码界的一系列知名人士。该投资公司表示，Mask Network 的“很大一部分”支持者后来成为了 Bonfire Union 的投资者，但没有透露他们的身份。

马斯克过去的投资者包括加密货币集团数字货币集团；萨姆·班克曼-弗里德的阿拉米达研究；世界上最大的密码交易所，币安；最活跃的加密风投之一，Animoca BrandsDapper Labs，CryptoKitties 背后的团队；Protocol Labs，分布式文件存储项目 Filecoin 的开发者；还有 Arweave，一家比特币基地和 a16z 支持的公司，他们希望将数据存储永久化，例如，可以用来对抗审查的工具。

比特币基地、分散融资平台 Gitcoin 和多链加密投资组合追踪器 DeBank 的前任和现任高管也资助了 Mask。

由于植根于中国，Mask 还将 FreeS、Y2Z、HashKey Group、Hash Global、SevenX、IOSG 风险投资公司、SNZ 和基础实验室等中国机构作为其投资者。

还有更大的秘密基金。但 Bonfire Union 的管理团队认为，它可以更好地与初创公司联系起来，因为他们自己就是企业家。

在当前的“加密冬天”的背景下，面具网创始人严苏吉承认，烽火联盟可能会从熊市中的折价交易中受益。严苏吉也因在网上发起反对中国科技行业“过劳文化”的运动而闻名。但与此同时，“投资者需要有勇气扣动扳机。”

严认为，在牛市中，“投资者正在撒大网。如果产品不太差，初创公司可以轻松进入下一轮融资。但在熊市中，“即使投资者真的相信一个项目，许多人也会观望。”

向腾讯学习

腾讯的投资策略启发了面膜网。这个中国互联网巨头最出名的可能是它的超级应用微信；但与这一旗舰产品共生的是其风险投资帝国，迄今为止，该帝国已经为全球超过 1300 家公司提供了资金，并赢得了“中国软银”的绰号

腾讯素以投资于自身不足的领域而闻名，而不是试图自己打造一切。它对其投资组合中的公司采取相对不干涉的方式，并通过分享网络流量等资源来帮助它们增长——这反过来为自己带来了更大的投资回报。它最成功的合作之一是与阿里巴巴的挑战者拼多多(Pinduoduo)的合作，拼多多的腾飞得益于其与微信的深度整合，后者允许用户轻松地与朋友分享产品页面。

同样，Bonfire Union 希望与为 web3 开发社交服务、基础设施和游戏的初创公司建立相互依赖的关系。面具网已经涉足这种方式。去年，它与 Arweave 达成了一项代币交换协议，此前两家公司共同致力于让用户使用分散存储在 Twitter 和脸书上传加密文件。

Mask Network 是 Bonfire Union 在交易结束后的最大基石投资者，并将继续通过其国库和团队成员的共同投资注入资本。该投资工具由一个 10 人团队运营，遍布新加坡、中国、日本和美国。

Bonfire 的其他有限合伙人包括祁鸣风险投资公司、硅谷银行和复星集团的前任和现任高管。
