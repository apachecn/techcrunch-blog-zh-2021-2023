Motional，通过在拉斯维加斯推出免费自主乘车服务

原文：https://web.archive.org/web/https://techcrunch.com/2022/02/24/motional-and-via-launch-free-autonomous-ride-hail-service-in-las-vegas/amp/

图像来源:动态

Motional，Aptiv-现代合资公司旨在将自动驾驶技术商业化，已经在拉斯维加斯推出了新的 robotaxi 服务，通过提供点播和运输技术服务。

这两家公司于 2020 年 10 月首次宣布合作,将在拉斯维加斯市中心向公众免费提供自动驾驶汽车。无人驾驶汽车的方向盘后面将会有人类安全操作员。

这项服务本应在 2021 年上半年开始，但新冠肺炎疫情的不确定性导致这些公司推迟了发布时间。Motional 和 Via 最初计划为按需共享机器人轴开发一个“蓝图”，并了解如何将这些车辆集成到公共交通中。Motional 证实，这两家公司后来取消了共享服务计划，将只提供私人乘车服务。

这项新服务推出前几个月，Motional 宣布计划于 2023 年在拉斯维加斯与 Lyft 合作开展一项商业无人驾驶叫车项目，这是两家公司现有合作关系的延伸。Motional 和 Lyft 的目标是在今年下半年开始提供无人驾驶的免费乘车服务，公众可以通过 Lyft 应用程序预订 Motional 的电动现代 IONIQ 5 机器人轴。

Motional 和 Via 采取了与 Cruise 和 Waymo 类似的方式，最初提供免费乘车服务。然而，至少在法律上，Motional 并没有被禁止在内华达州对乘车收费。

内华达州的 AV 法规比加州少得多，Cruise 和 Waymo 目前正在加州寻求商业化。例如，内华达州对有或没有人类安全操作员的测试或操作没有区别，其立法也没有提到提供 AVs 作为送货或乘车服务。内华达机动车辆部的一位发言人表示，这意味着公司现在可以在技术上对商业影音服务收费，因为没有规定说他们不能这样做。然而，发言人指出，新的立法正在起草中。

Motional 表示，它有一个允许它收取车费的许可证，就像它在拉斯维加斯现有的无人驾驶 Lyft 服务一样，但没有详细说明这个许可证是与 AVs 有关还是给了该公司在该市经营出租车服务。

Motional 的发言人艾比·欧玛利说，在现阶段，Motional 和 Via 提供免费乘车服务，既是为了推广服务，也是为了自己的学习目的，这样他们可以收集乘客的反馈，研究他们的技术如何协同工作。这位发言人没有证实公司是否或何时打算运营商业服务，只是说，“Motional 期待从试点中学习，并在未来扩大与 Via 的合作伙伴关系。”

图片来源: Motional/Via

从周四开始，乘客可以在 Via 应用上预订 Motional 的自动驾驶宝马 5 系机器人轴，Motional 目前的 Lyft 服务使用的就是这款车。根据欧玛利的说法，这两种服务的车队将是分开的，他补充说，该公司还在使用其克莱斯勒 Pacificas 进行测试。

Via 服务时间为周一至周五上午 9 点至下午 5 点。Motional 表示，乘客可以在市中心的选择点上下车，这些点将在 Via 应用程序上突出显示，包括 RTC Bonneville Transit Center、拉斯维加斯市政厅、集装箱公园、拉斯维加斯艺术区和克拉克县政府中心。

据 Motional 称，robotaxi 服务依赖于 Via 的智能预订、路由和软件应用技术，为 Motional 的自动驾驶 robotaxis、车队管理和车内乘客体验提供支持。

欧玛利告诉 TechCrunch:“在[运营设计领域]内，基于乘客的上下车请求，车辆将在那时选择最有效的路线。”"这不像公共汽车那样是一条固定的路线。"

Motional 不愿透露其与 Via 在拉斯维加斯合作的车队规模，但欧玛利表示，这些公司将密切关注需求，并有基础在未来扩大服务规模。
