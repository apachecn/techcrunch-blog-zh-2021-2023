Wing Security 推出其端到端 SaaS 安全平台，融资 2600 万美元 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/03/30/wing-security-launches-its-end-to-end-saas-security-platform-raises-26m/

随着企业越来越依赖越来越多的 SaaS 产品，安全团队必须更好地了解公司员工实际使用的应用程序及其相关风险。毫不奇怪，我们也看到了 SaaS 安全解决方案的增长。 Wing Security 今天从 stealth 推出，并宣布其 2000 万美元的 A 轮融资，旨在提供一个全面的端到端 SaaS 安全平台，可以帮助企业发现、监控并在需要时自动修复其员工如何使用 10 万多种 SaaS 工具中的任何一种工具的潜在安全问题。

该公司的首轮融资由 GGV 资本牵头，而其 600 万美元的种子轮融资由 S-Capital 牵头，该公司此前没有披露这一消息。Harmony Partners，硅谷 CISO 投资集团和一些天使投资人，包括 Fireblocks 首席执行官兼联合创始人迈克尔·肖洛夫，Orca Security 联合创始人阿维·舒阿和吉尔·杰龙，前 Kenna Security 首席执行官卡里姆·图巴和 Claroty 联合创始人加利纳·安托瓦，也参加了会议。

让 Wing 立刻脱颖而出的一点是它的创始团队。在以色列国防军(IDF)前首席信息安全官、退休的准将将军 Noam Shaar 和 IDF 前网络行动负责人、退休的 Galit Lubitsky 上校的领导下，你很难找到一批更有经验的创始人，即使是在以色列安全初创企业生态系统中，也很大程度上借鉴了 IDF 的 unit 8200。也正是这种领导经验让创始人走上了创办 Wing 的道路。

“我们知道人们需要端到端的整体解决方案。他们不需要一个解决方案来处理未知的应用程序，只需要一个解决方案来确保 Slack 正在运行并被安全地使用，”Shaar 说。“作为以色列最大、最复杂的组织的前 CISO，我对我的安全产品的所有要求就是性能好、使用简单并能保护我。而不是说‘啊，你出了那件事？那是因为我们没有报道这个，”然后，你有很多不同的解决方案，你试图整合在一起，它从来没有工作。"

Wing 随后提供了一个平台，可以发现公司内部使用的所有 SaaS 服务，而无需在用户设备上安装代理。为此，该服务集成了给定公司中使用的一些主要 SaaS 应用程序(比如 Slack、Salesforce、Zoom 等)。)然后查看那里发生的活动以及与这些应用程序的连接。这并不容易，但相对简单，可以与这一领域的其他供应商相比。此外，Wing 还集成了端点安全解决方案，以定期查询这些端点，收集员工在这些机器上使用的 SaaS 应用程序的遥测信息。

“将这两种方法结合在一起，这是构建完整视图的唯一方法，”Shaar 解释道，他还暗示该团队正在寻找一种在没有这些现有端点管理服务的情况下也能工作的解决方案。

值得注意的是，Wing 团队强调该服务只查看元数据。它也不会查看员工收件箱的内容。

至于补救措施，Wing 不仅发现了未经批准的应用程序，还发现了经批准的 SaaS 服务的配置存在潜在问题，可能会产生安全问题。然而，该公司及其投资者认为，仅仅提供潜在安全问题的警报还不够好。

“当安全专家看到另一个工具向他们发出警报时，他们会问‘然后呢？我该如何处理这些警报？我从这里到无限期的警报。”我认为 Wing 采用的这种平台方法非常独特。他们说‘我们不会就此罢休，我们会帮你解决这个问题。’GGV 的柳文欢·云格说。

在实践中，这意味着 Wing 不仅提供了用于补救常见问题的行动手册，还提供了一些用于补救重复性问题的自动化工具。

"Wing 的整体平台可满足您在 SaaS 的所有安全需求。它可以识别所有 SaaS 应用，确定风险优先级，并自动修复，”比特币基地的菲利普·马丁说，他代表硅谷 CISO 投资集团这使得安全团队能够快速高效地应对 SaaS 安全挑战，使安全性成为希望使用 SaaS 工具的员工的助力，而不是生产力的阻碍。"
