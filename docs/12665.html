AirTree 和 Greycroft 回归，领导澳大利亚 regtech FrankieOne 的 A+ | TechCrunch 系列

原文：https://web.archive.org/web/https://techcrunch.com/2022/11/28/frankieone-series-a-plus/?utm_source=Telegram&utm_medium=iXNews&utm_campaign=PerspectiveIX.com

总部位于墨尔本的初创公司 FrankieOne 提供用于身份验证和欺诈检测的 API 平台，该公司表示，已在 A 轮+中增加了 2300 万澳元(约 1540 万美元)，使其 A 轮总额达到 4500 万澳元(约 3000 万美元)。FrankieOne 表示，这是迄今为止澳大利亚 regtech 筹集的最大一笔风险投资。

该基金包括一批回归投资者，包括主要支持者 AirTree Ventures 和 Greycroft，以及参与者 Reinventure(金融服务公司西太平洋银行的风险部门)、Tidal Ventures 和 Apex Capital Partners，它们也是现有投资者。新的战略投资者包括币安实验室和北海巨妖投资公司。

FrankieOne 于 2019 年由连续金融科技企业家 Simon Costello 和 Aaron Chipper 创立，将银行、金融科技、加密和游戏公司与数百个数据源相连，并与全球 170 家金融机构合作。这家初创公司表示，在过去 12 个月里，它的收入增长了 4 倍，现在它的客户包括 Westpac、Shopify、Afterpay、币安、Zipmex 和 PointsBet。在过去的一年中，它的团队增加了一倍，并在旧金山开设了一个新的办公室。

科斯特洛告诉 TechCrunch，FrankieOne 是从他与 Chipper 共同创立的 neobank 风险投资公司发展而来的，该公司名为 Frankie。Costello 和 Chipper 希望 Frankie 通过使用可用的顶级身份和欺诈预防工具，以最佳的入职体验从行业中脱颖而出，但“这一过程被证明是不必要的复杂，需要多次集成，很明显我们需要创建自己的基于风险的入职，”Costello 说。他们意识到其他金融科技公司也有同样的问题。

FrankieOne 创始人西蒙·科斯特洛和亚伦·齐珀。图片来源: FrankieOne

因此，创建了 FrankieOne 来简化有关法规和合规性的流程。它现在提供了一个单一的 API，将第三方供应商与 48 个国家的 350 多个数据源连接起来。

科斯特洛解释说，大多数金融科技公司、银行和加密公司都面临匹配率、覆盖范围或扩展能力不足的问题，因为他们为了满足身份需求而连接到单一供应商。这意味着他们的技术也需要不断维护。

FrankieOne 通过连接到来自多个供应商合作伙伴的数百个数据源解决了这一问题，这意味着他们的客户不再需要依赖单一供应商。其 API 的关联供应商包括 Equifax、Experian 和 Socure 等老牌金融服务公司以及 Sardine AI 等快速增长的初创公司。

“FrankieOne 的与众不同之处在于，它允许客户切换供应商，创建动态工作流，添加更多欺诈信号并开拓新市场，确保我们的客户能够快速响应不断变化的法规和更新的业务要求，而无需承担任何额外的工作负担，”科斯特洛说。

他补充说，FrankieOne 的客户通常是高度监管行业中的大中型组织，这些组织需要适应频繁变化的复杂政策。

作为客户如何使用 FrankieOne 的一个例子，科斯特洛提到了体育博彩应用 PointsBet，该应用以前使用单一提供商进行客户入职，但处理大量无法实时验证的潜在用户。这导致了客户的减少，或者进一步的人工工作来验证他们的细节。自从与 FrankieOne 合作以来，PointsBet 已经将客户入职率提高了 14%。

当 Westpac 开始对更多的金融服务进行数字化时，他们仍然使用独立的传统系统，因此需要提高通过率以减少掉线。他们在新的 BaaS(银行即服务)中集成了 FrankieOne 的 KYC(了解您的客户)功能平台。这大大提高了通过率，因此该公司在整个集团中增加了 FrankieOne。

最新一轮融资将使 FrankieOne 更快地扩大规模，并在北美、欧洲和亚太地区扩张。科斯特洛表示，这也将使 FrankieOne 能够对其供应商、数据源和欺诈能力的生态系统进行重大投资，使其能够扩大客户群。
