规模计算获得 5500 万美元帮助公司管理边缘基础设施

原文：https://web.archive.org/web/https://techcrunch.com/2022/07/12/scale-computing-secures-55m-to-help-companies-manage-edge-infrastructure/

随着企业在其网络边缘处理越来越多的数据，边缘计算正受到越来越多的关注。根据 2021 年的一项调查(尽管来自一家边缘计算服务供应商)，77%的公司表示，他们预计在 2022 年将看到更多的边缘项目支出。但是，尽管一些组织准备从边缘计算中受益，边缘计算指的是在最终用户附近存储和分析数据的实践，但并不是所有组织都知道它需要什么。对于缺乏必要基础架构的 IT 团队来说，管理跨位置的边缘设备群可能是一项负担。

Jeff Ready 声称，他的公司 Scale Computing 可以通过存储架构和灾难恢复技术帮助那些不确定从哪里开始边缘计算的企业。Ready 在印第安纳波利斯经营着一家啤酒酿造公司，他在 2007 年与贾森·科利尔和斯科特·拉夫米勒共同推出了规模计算。

Loughmiller 和 Collier 都在 Tumbleweed Communications 工作，为企业和政府客户开发消息和文件传输平台。在 Scale、Ready 之前，Loughmiller 和 Collier 共同创立了 Corvigo，为电子邮件提供垃圾邮件过滤工具。

早期，Scale 专注于销售装有面向中小型企业的定制存储软件的服务器。但该公司后来转向了“超融合”基础设施和边缘计算产品，通过将服务器、存储、称为虚拟机管理程序的虚拟机监视器以及备份和数据恢复结合到单个系统中，虚拟化客户的基础设施。

“Scale Computing 设计了一个 IT 基础架构平台，该平台消除了对虚拟化软件、灾难恢复软件、服务器和共享存储等传统 IT 孤岛的需求，代之以一个完全集成、高度可用的平台来运行应用程序，”Ready 在接受 TechCrunch 的电子邮件采访时表示。自我修复平台可实时自动识别、缓解和纠正基础架构中的问题，使应用程序即使在本地 IT 资源和员工匮乏的情况下也能实现最大的正常运行时间

这些都是崇高的承诺。但是——贷款规模的可信度——该公司今天筹集了 5500 万美元的新资金，由摩根士丹利扩张资本牵头，使规模的总融资额达到 2.02 亿美元。

“Scale Computing 的边缘计算平台的技术优势通过增强其 IT 基础设施的弹性、可管理性和效率来解决客户普遍存在的问题，”摩根士丹利扩张资本董事总经理兼负责人 Pete D. Chung 在一份声明中告诉 TechCrunch。“我们的调查清楚地表明，Scale Computing 的客户受益于材料成本的节约以及对其 It 基础设施信心的增强。”

Scale 的平台允许公司在靠近用户的地方运行应用程序，在单个仪表板中集中管理远程站点，如分支机构。该公司的设备管理工具使用户能够从控制台查看一系列设备，并选择性地检查它们的健康状况，自动标记问题并记录下来，以便 IT 团队可以做出诊断。

“各行各业的企业都渴望简化其 IT 基础架构、提高业务弹性并降低运营成本。疫情还展示了在大型集中式数据中心之外拥有边缘计算能力的重要性，”Ready 说。“Scale 的软件消除了对虚拟化软件、灾难恢复软件、服务器和共享存储等传统 IT 孤岛的需求，代之以完全集成、高度可用的应用运行平台。”

Ready 的语言可能有些夸张，但对边缘计算管理软件的需求确实在增长。根据 Grand View Research 的数据，全球边缘计算市场在 2021 年估计价值 74.3 亿美元，到 2030 年可能攀升至 1559 亿美元。

Ready 不愿透露收入，当被问及 Scale 是否有计划在今年年底前扩大其 160 人的员工队伍时，ready 表示反对，这或许是为了应对经济下滑。但他声称，该公司目前在北美、欧洲和中东以及亚太地区拥有 6000 多名客户。

据报道，在某个时间点上，这些客户包括杂货店特许经营杰里的企业和赌场公司云顶集团。

“尽管疫情带来了不确定性，或者可能正因为如此，大规模计算对其边缘计算、虚拟化和超融合解决方案的需求前所未有，”Ready 在一封后续电子邮件中补充道。“我们将利用新的资金来扩大我们在边缘计算领域的领导地位，包括在人力、R&D 和债务重组方面的投资。”
