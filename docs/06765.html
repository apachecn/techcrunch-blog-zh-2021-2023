客户支持管理平台组装土地 5100 万美元

原文：https://web.archive.org/web/https://techcrunch.com/2022/05/26/customer-support-management-platform-assembled-lands-51m/

自称为客户支持团队劳动力管理平台的 Assembled 今天宣布，它在由 New Enterprise Associates 牵头的 B 轮融资中筹集了 5100 万美元，Emergence Capital 和 Basis Set Ventures 也参与了融资。首席执行官王帅文表示，新资金将用于产品开发，包括日程优化算法和与人力资源软件的集成，并启动一项名为“CX 学者”的倡议，该倡议将把 Assembled 收入的一定比例用于支持员工的继续教育。

疫情给客户支持组织带来了巨大的负担，他们既要面对不断增加的客户请求，又要面对在家工作的挑战。根据技术供应商 Replicant 在 2021 年进行的一项调查，71%的人认为现在比疫情开始时更难联系到真人。质量保证和培训中心报道，这可能是真的——美国联络中心代理的年平均离职率现在保持在 30%到 45%。

王并不把 Assembled 作为万灵药，但他认为该平台可以帮助减轻过度劳累的支持人员的负担。2018 年，王与他的兄弟和 Brian Sze 共同创立了这家公司，他们都曾在 Stripe 工作过。

“在 Stripe 时，[我们]观察到支持职能迅速转变为复杂的、地理上分散的、由数千人组成的运营，”王帅文在电子邮件采访中告诉 TechCrunch。“大量辞职、人员短缺、不切实际的期望和令人震惊的离职率促使人们认识到，公司必须平衡业务需求和支持团队的需求。这为我们的业务提供了顺风，因为公司更加关注他们如何为代理人配备人员、组织和提供更大的授权和灵活性。疫情推动的数字化转型加速对我们也有好处。我们看到，一方面，以数字为先的新品牌正在崛起，另一方面，传统品牌正在将电子商务融入其业务。”

通过一系列预测和安排技术，尝试预测和准备大量的客户电话、短信和电子邮件。例如，该平台利用历史支持票甚至 Reddit 活动来预测未来的客户请求量(按队列、渠道和站点)。模拟模型预测了在不同的人员配置条件下，电子邮件积压等事情可能如何随着时间的推移而演变，而优化算法则创建了时间表——遵守轮班时间长度、轮班间隔时间以及轮班开始时的灵活性等规则。

使用组合排班的经理可以自己创建排班，如果他们选择并让座席代表请求更改这些排班的话。他们还可以查看响应时间和服务级别等指标，以及与历史性能的比较。

关于分析的话题，值得注意的是一些联络中心平台最近被指控强制监控，例如强迫远程员工在家中安装摄像头，以检查他们是否在工作。Assembled 声称，它不提供此类功能，并允许客户删除其存储的任何用户数据，包括基本识别信息(姓名和电子邮件)、日程安排和支持活动数据。

根据王帅文的说法，通过新的 CX 学者计划，Assembled 希望鼓励“上升的客户体验领导者”学习商业基础和其他技能，以促进他们的职业发展，同时使他们的团队受益。该计划的细节，包括选择和奖励过程，仍然有些模糊。一位发言人说:“我们正在从我们的用户社区收集英特尔，以更准确地了解什么样的教育和社交机会对那些从事支持工作的人最有用。”

王帅文认为，Assembled 与 Teleopti 和 Calabrio 等电子表格和“遗留”系统的解决方案，以及 WhenIWork 和 Tymeshift 和 Playvox 等初创公司的纯调度工具竞争。他断言，遗留工具是“为一个不同的时代建造的”，而大多数调度工具是针对更简单的环境，涉及比客户支持更小的人群。

“用于组织(呼叫中心)行业人员的技术已经过时，更重要的是，20 多年前的层级流程和做法已经根深蒂固。你如何处理需要及时回复的电话或电子邮件收件箱？您如何组织跨多个办公室、时区或完全分布式的运营？现代支持团队必须回答这些问题，同时为代表其品牌的人提供灵活性和信任、授权的环境，”王帅文说。“在任何给定的公司中，客户体验都是人员最密集的职能之一—在我们的客户群中，它通常占员工总数的 30%(在金融服务中高达 70%)。在一个人员短缺、工作分散的世界里，我们帮助公司解决与提供强大支持相关的棘手的人员物流问题。”

Assembled 在过去几年中取得了显著的里程碑式的成就，包括将其客户群增加了两倍，达到 180 多家企业，拥有 Etsy、Stripe、Zoom、TaskRabbit、Asana 和餐饮品牌国际(汉堡王和大力水手的母公司)，并与客户体验供应商 Five9 和 MaestroQA 建立了合作伙伴关系。为了保持领先地位，王帅文表示，Assembled 将开发针对信任和安全、履行和 IT 服务台场景定制的新产品，并开设一个纽约办事处，计划在亚太地区和欧洲扩张，Assembled 60%以上的支持代理都位于纽约。

“总共有 63 人，比去年增长了 4 倍。我们计划在年底前将团队规模扩大一倍，”王帅文透露。“我们目前不会披露公司的收入，但自上一轮(融资)以来，收入增长了 4 倍。”

迄今为止，总部位于加州旧金山的 Assembled 已经筹集了 7100 万美元。

计划加入 Assembled 董事会的 NEA 合伙人凡妮莎·拉科(Vanessa Larco)在接受采访时表示:“公司将所有的资源和注意力都投入到产品开发和销售中，保持对售后互动的这种关注程度对客户保留和品牌声誉至关重要，因为超过一半的客户愿意在一次负面互动后转向竞争对手。提供卓越的客户体验是一个与行业无关的目标，越来越多的公司开始关注这个目标……Assembled 诞生于 Stripe 扩展支持团队的第一手经验。王帅文和他的团队真正热衷于授权和优化支持团队。该团队的经验、技术能力和热情真正吸引了我们的注意力，我们相信他们是帮助解决企业在客户体验管理方面面临的日益严峻的挑战的合适公司。”
