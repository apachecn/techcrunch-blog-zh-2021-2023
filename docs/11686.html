乔治·霍兹，又名“地球热”，将离开 Comma.ai，投身于一个崇高的人工智能项目

原文：https://web.archive.org/web/https://techcrunch.com/2022/11/02/george-hotz-aka-geohot-is-leaving-comma-ai-for-a-lofty-ai-project/

四年前，Comma.ai 创始人乔治·霍兹求助于他的董事会——他是董事会的唯一成员——并解雇了自己的 CEO 职务。当时，著名的 iPhone 和 PlayStation 3 黑客 geohot 的目标是建立一个新的研究部门，专注于可以驾驶汽车的行为模型。

现在，Hotz 表示，他将从驾驶辅助系统初创公司“离开一段时间”，该公司承诺将特斯拉自动驾驶功能引入你的汽车。尽管如此，他仍将是唯一的董事会成员和总裁。

他告诉 TechCrunch，Hotz 已经有一段时间没有参与日常领导工作了。这就落到了首席运营官·阿莱克斯·马茨纳和首席技术官哈拉尔德·谢弗的肩上。自 2019 年里卡多·比亚西尼担任首席执行官以来，该公司一直没有首席执行官。(Biasini 于 2019 年离开了首席执行官一职，并留在 Comma 从事其开放试点软件的工作，直到 2020 年 2 月。)

霍兹是马茨纳所说的观察者和偶尔解决难题的人。

Hotz 告诉 TechCrunch，Comma.ai 开发并销售了一款售价 1999 美元的驾驶辅助系统 devkit，该系统可兼容 200 多辆汽车。现在的重点是将运行在 Comma 的开源软件 openpilot 上的 devkit 变成产品化的消费产品。

“我擅长战时的事情，”Hotz 在最近的一次采访中告诉 TechCrunch。“我不太擅长动手，好吧，让我们耐心地扩大规模。你想和一个一年能生产 10 万台设备的供应链打交道吗？好像，也不尽然。"

而这也是目标之一:Comma 3 年销量 10 万台。

这家初创公司去年悄悄地从个人手中筹集了 1000 万美元，并搬进了位于圣地亚哥的一个 20，000 平方英尺的设施。(Comma 的第一笔 810 万美元资金分两轮从硅谷 VC a16z 拿走。Matzner 在最近的一封电子邮件中告诉 TechCrunch，它现在正在“积极招聘”，并有望在本月晚些时候向 openpilot 推出一些主要的端到端机器学习更新。

Comma.ai 最初推出的计划是销售 999 美元的售后自动驾驶汽车套件，该套件将为某些车型提供类似于特斯拉自动驾驶功能的高速公路驾驶辅助功能。Hotz 在 2016 年 10 月收到国家公路和交通安全管理局的一封信后取消了这些计划。五周后，Comma.ai 向全球发布了其自动驾驶软件。所有的代码以及硬件计划都发布在 GitHub 上。

该公司继续开发硬件产品生态系统，旨在为汽车带来半自动驾驶功能。这些努力最终导致了 Comma 3 ，根据存储大小的不同，其价格在 1999 美元到 2499 美元之间。连接 devkit 和车辆的汽车线束也要 200 美元。

Comma 3 比它的早期版本更容易使用。Hotz 说，安装和设置它需要一些耐心，但不再需要任何技术专长。他补充说，现在，该公司把 Comma 3 变成“产品化”和可扩展的消费产品了。

下一步是什么？

Hotz 已经深入他的下一个项目，他称之为小公司。他的目标是为机器学习编写一个比 PyTorch 更快、更简单的新框架。Hotz 希望建立允许在边缘训练 ML 模型的工具，而不是在云中训练 ML 模型并将其运送到边缘。

“目前的 PyTorch 和 TensorFlow 无法满足边缘训练的需求，”他说。

包括自动驾驶在内的人工智能相关领域正在更多地转向深度神经网络——一种复杂的人工智能算法，允许计算机通过使用一系列连接的网络来识别数据模式，从而进行学习……有点像大脑的工作方式。但是正如 Hotz 指出的，“我们对神经网络这种东西都很陌生。”

深度学习和计算机视觉专家、特斯拉前人工智能总监安德烈·卡帕西(Andrej Karpathy)将这一阶段称为编程 2.0，或软件 2.0 ，其中编程是通过示例来完成的，人类实际上只是编写一般的脚手架。换句话说，软件自己写。

“你不应该建立一个(人工智能)芯片，直到你可以建立一个比英伟达上的 PyTorch 性能更好或至少相同的软件，”Hotz 说。"作为构建人工智能芯片的基础，首先让我们来构建软件."
