未来的犯罪

过去几年来，加密空间发展如此之快，以至于开发者们似乎普遍认为现有的规则不适用于它们。好吧，经过多年蜗牛的法律诉讼，美国检察官似乎开始觉得是时候挑战这种看法了。

本周，纽约南区的美国检察官办公室逮捕并起诉了一名前 OpenSea 高管，他利用自己的职位抢先运营了将在市场主页上上市的 NFT 项目。社区成员通过跟踪他在公共区块链上的活动发现了他的行为。

我很想在播客中对此大声疾呼，但是在我们录制的时候新闻突然出现了，所以我在这里给你留下一些想法。

这次逮捕对 NFT 圈子里的人来说是一个巨大的冲击，他们普遍认为内特·查斯坦的行为不道德，但这不可能是“内幕交易”，因为 NFT 不是证券。很多人都持有这种观点，包括查斯坦在 OpenSea 的老板，他解雇了查斯坦。

“我确实认为这被误解为内幕交易。我们不认为非金融资产是金融资产，所以这并不适用。这是一个非常具体的东西的一个非常具体的术语，”OpenSea Devin Finzer 在九月份告诉 Decrypt 。

有很多人非常仔细地阅读了 SDNY 的新闻稿，该新闻稿称，它特别指控查斯坦“与一项不可替代代币内幕交易计划有关的电信欺诈和洗钱”他们在后来的版本中将 NFT 描述为“数字资产”。此外，值得重申的是，这是 DOJ——而不是美国证券交易委员会——指控他，尽管这是办公室的证券和商品欺诈特别工作组处理此案。

现在，为什么加密人不希望 NFTs 被归类为证券？嗯，有很多现有的监管指导，大多数人认为，如果 NFT 单方面受制于证券法，这将基本上颠覆整个行业；这无疑会提高创建非功能性测试的门槛，并减少目前该领域的大量实验。

另一个重要的原因是，如果非金融资产被视为证券，那将是件坏事，因为这将意味着很多人在很长一段时间里一直在做非法的事情。

NFT 空间在没有任何有意义的监管出台的情况下，通过了最新的加密牛市。随着 NFT 交易量开始出现放缓迹象，人们担心更多监管可能即将出台。

随着 crypto 变得越来越主流，它还能保持去中心化吗？
无论是首次购买加密货币的人，还是了解 NFTs、比特币和一般加密生态系统的人，对加密的认识在全球范围内都有所上升。但随着它获得动力，世界各地的监管机构将继续更密切地监控这个领域，但标题本身就说明了问题:这对加密的未来意味着什么？该行业的一些创始人和高管发表了自己的看法。
