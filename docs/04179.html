苹果因荷兰约会应用支付遭遇第六次反垄断罚款——TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/02/28/apple-sixth-acm-fine/amp/

图片来源:切斯纳特/盖蒂图片

苹果仍未遵守荷兰的反垄断令，即允许当地约会应用选择使用第三方支付技术向其应用用户出售数字内容。

在今天的一份声明中，荷兰消费者与市场管理局(ACM)表示，已对这家科技巨头征收第六笔罚款(500 万€)，原因是其未遵守去年首次发布的一项命令。

这家 iPhone 制造商现在正面临着€3000 万英镑的罚款，因为自上周一以来，罚款又增加了 500 万€——如果它继续阻挠监管机构，未来几周€还有可能进一步增加 500 万英镑(€可能最多增加 5000 万英镑)。

“我们上周没有收到苹果公司的任何新提议，让他们遵守 ACM 的要求。因此，苹果也必须支付第六笔罚金，”ACM 发言人表示。

“在 2 月 14 日的那一周，我们再次向苹果解释了我们有哪些要求，以及为什么目前的提议不够充分。看起来苹果不打算对他们最初的提议做任何修改来满足要求。”

我们就最新的处罚联系了苹果公司，但该公司拒绝置评。

苹果几周来一直在这个问题上保持公开沉默——包括在欧盟数字战略主管 EVP Margrethe Vesta ger上周指出其行为后，指责该公司故意选择支付罚款而不是遵守竞争命令。

尽管新闻机构路透社今天早些时候报道了苹果发给 ACM 的一封信，该公司称已获得这封信，该公司在信中辩称，它已经遵守了监管机构的命令——辩称希望获得该权利的约会应用程序开发者只需进行“微小的技术更改”。

该公司此前表示不支持该命令，理由是它有降低用户体验的风险——同时坚持声称，它仍然通过向有问题的开发者提供两种权利来遵守。

在 TechCrunch 获得的苹果致 ACM 的一封日期为 2 月 28 日的完整信函中，苹果首席合规官凯尔·安德尔(Kyle Andeer)写道，“苹果认为其解决方案完全符合荷兰法律”。

在信中，这家科技巨头为其行为辩护的主要内容集中在要求约会应用提交一个新的二进制文件，以便能够利用非苹果支付技术——安德尔认为这并不是一个特别不寻常的步骤。

“这是一个简单的先决条件，可以确保苹果遵守其在荷兰的法律义务，同时有能力在世界其他地方维持其标准条款和条件，”他建议道，并补充道:“苹果的全球应用商店规则和政策要求在其应用中销售数字商品或服务的约会应用的开发者使用 IAP 功能进行这些交易，为用户提供安全、可靠和一致的体验。这一直都是事实。”

安德尔继续举了一些例子，他说在线约会巨头 Match Group(拥有大量约会应用程序，包括 Tinder)已经提供了不同版本/二进制的 Pairs、Match 和 Our Time 应用程序，“以适应不同司法管辖区的不同要求或偏好”。

因此，他认为这是“苹果和开发者在其他司法管辖区使用的相同方法，在这些司法管辖区，存在独特的法律问题，需要在特定的司法管辖区使用不同的方法”——进一步断言提交单独的二进制文件的要求对开发者来说“既不昂贵也不困难”。

“约会应用熟悉这个过程，事实上是自愿参与的，”安德补充道。“荷兰店面的新二进制程序只需要对现有应用程序进行微小的技术更改，包括有限的调整，允许约会应用程序的开发者使用第三方支付处理器或插入购买网站的链接。这种方法不会产生额外的成本。”

就其本身而言，荷兰监管机构此前表示，苹果对希望使用其应用内支付 API 替代品的开发者施加了“不合理”和“不利”的条件。

而且——从根本上来说——很明显，开发人员自愿做一些技术上的事情和技术动作是平台要求他们访问他们合法授权的条款之间是有区别的。

此外，ACM 此前表示，它对苹果试图限制开发者在支付技术上的非此即彼的选择感到不满——要么使用苹果的应用内 API 或第三方技术——而不是让他们能够利用同一应用中的所有选项。

(在给 ACM 的信中，苹果用这些术语描述了其对该命令的回应——称在荷兰应用商店提供约会应用的开发者可以选择使用“或苹果的应用内购买(“IAP”)功能、第三方支付处理器或他们应用的网站链接”【强调我们的】。)

我们联系了 Match Group，请其对苹果在给 ACM 的信中引用的有关二进制文件的论点做出回应，并将根据任何回应更新这份报告。

正如我们之前报道的那样，与此同时，欧盟的立法者正在就全面的事前竞争规则的细节达成一致，这些规则将专门适用于最强大的中介平台——所谓的“看门人”——上周维斯特格引用了苹果在荷兰逃避反垄断执法的案例，以突显数字市场法案面临的挑战。

这表明，苹果对荷兰当地反垄断令的回应可能会影响 DMA 的最终形式，如果地方立法者认为他们需要进一步加强一揽子计划，以降低不合规的风险。

也就是说，DMA 已经面临违规罚款的威胁——高达全球年营业额的 10%。(就苹果而言，这更接近于€的 300 亿，而不是€的 3000 万。)

因此，对于科技巨头来说，忽视未来根据 DMA 做出的委员会行为命令将是一个风险更大/成本更高的策略——至少对于那些属于即将到来的泛欧盟监管范围的公司来说。

这也是欧盟改革和加强数字竞争执法方式的重要原因。
