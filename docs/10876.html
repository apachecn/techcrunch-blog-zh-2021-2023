等待树莓派:Eben Upton 谈论供应约束和需求冲击 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/10/05/raspberry-pi-supply-chain-shocks-eben-upton-interview/

随着疫情引发的全球供应链中断继续挤压经销商库存，想要获得一些刺激的单板 Raspberry Pi 套件来支持低成本电子项目的硬件爱好者已经经历了一年多的沮丧时光。

去尝试购买 Pi，你不会错过任何地方的官方经销商网站——例如欧洲的 Pimoroni 和 SB Components 或美国的AdaFruit——都充斥着“缺货”通知。

继续寻找，你可能会在社交媒体上偶然发现发烧友通过非官方经销商途径(如亚马逊)在受欢迎的主板 RRP 上支付多倍价格的故事——这些硬件通常会有非常合理的价格标签。

遗憾的是，与 Pi 本身不同，这些 Pi 买家悲哀的故事在网上太容易找到了，因为黄牛已经利用了供应短缺——启动股票购买机器人，并试图在少数单位出现时突袭，以便他们可以通过利用人们的热情继续他们的项目，以大幅抬高的价格出售工具…

一位沮丧的潜在 Pi 购买密报者直接给我们打了电话——想知道:“树莓 Pi 在这种环境下如何生存？他们的许多产品多年来一直(并将继续)缺货。糟糕的是，许多产品都被推迟到 2023 年末。你也不会从他们网站上的购买按钮知道它…它链接到所有缺货的供应商，所以他们基本上在掩盖它有多糟糕。不管怎样，希望他们能挺过这一关，让他们痛不欲生。”

为 Raspberry Pi 基金会说句公道话，他们从去年开始就一直在写关于供应问题的博客(当时他们也不得不宣布一些与全球半导体供应的“奇怪时期”有关的临时价格上涨)——所以声称他们在“掩盖”坏消息听起来有点阴谋论的风险。

全球半导体短缺本身的根源与供应链中断有关，供应链中断是由疫情影响劳动力可用性和新冠肺炎停工同时刺激对各种消费电子产品(当然需要芯片来驱动)的需求所引发的。简而言之，众所周知的“完美风暴”。

如果可以这么说的话，圆周率的供给问题实际上是这两种现象的巧妙组合。(或者:“我们有分销商的订单积压，他们有客户的订单积压，”Raspberry Pi 的联合创始人向我们简明地总结了这个问题。)

当然，对于想买圆周率的个人来说，结果远非如此。但该公司已经挺身而出，在供应紧张时期支持其商业客户，特别是工业和商业原始设备制造商。这意味着它优先考虑商业客户的可用 Pi 库存，以便他们可以继续为他们的客户服务，因此业余爱好者感到(库存)紧张。

“这些公司(通常是小公司)对我们的平台抱有信心:生计危在旦夕，我们觉得对他们有义务，”联合创始人埃本·厄普顿(Eben Upton)告诉 TechCrunch，并补充说:“新的生产填补了这些积压，这就是为什么我们每周可以持续生产超过 10 万台，但渠道中仍然很少有空闲库存。”

因此，尽管圆周率创始者的核心“技术民主化”教育使命本质上与向好奇的个人提供负担得起的可编程计算硬件相关联——以便他们可以四处玩耍并在途中学习 STEM 但毫无疑问，它不得不做出艰难的选择，要求爱好者加入依赖圆周率的企业背后的队列。

“显然，这意味着降低了发烧友客户的可用性，”他证实。“我们的建议是从经过批准的经销商那里购买，他们中的许多人都有等待名单，并且/或者使用像 rpilocator 这样的工具来跟踪渠道中出现的股票。”

“去找经过批准的经销商，”他重申道。“我们不会向未经批准的经销商供货。去超市，排队。使用遥控器如果你想去一个有排队系统的商店，那就去排队。我们已经派出小队了。我们仍在设法将一些设备送到那个地方。”

因此，对业余爱好者来说，目前的信息仍然是一个非常英国化的信息:“很抱歉，但你必须排队等候。”

但厄普顿也暗示，明年 Pi 库存问题应该会开始缓解——他表示，他希望“不晚于 Q2”找到解决方案。“我们看到了供应链状况正在改善的积极迹象，但我们还需要几个月才能回到我们喜欢的强劲库存水平，”他告诉 TechCrunch。

应对需求冲击

在四月份的供应链更新中，厄普顿写道,“各种各样”的供应链挑战正在继续扰乱常规库存。

“目前的形势既是需求冲击，也是供应冲击，”他解释道。“从 2021 年初开始，对树莓派产品的需求急剧增加，供应限制使我们无法灵活满足这一需求，导致我们现在几乎所有产品都有大量订单积压。反过来，我们的许多经销商也有自己的积压订单，当他们从我们这里收到库存时，就会履行这些订单。”

他还重申了前面提到的对试图获得工具包的个人的官方 Pi 建议——这些建议仍然适用:即，从经批准的经销商处购买 Pi(即，避免被倒卖)；试着找一家接受预购的经销商，它可以(希望)让你知道你要等多长时间……嗯,(显而易见的)后续措施是:在你等待 Pi 的时候，做好耐心等待的准备。

Pimoroni 的联合创始人乔恩·威廉姆森(Jon Williamson)在一封电子邮件中对 Raspberry Pi(又名，慈善机构 Pi Foundation 的 Pi 制造部门)只有赞美之词，因为他在库存“SBC”(单板计算)套件方面做得“非常好”，“直到缺货很久，特别是考虑到它们的受欢迎程度”，他在一封电子邮件中这样对我们说。

他还把我们带回了 Pi 四月份的博客文章——称之为“相当彻底”的更新，提供了关于供应紧缩的“最佳信息”。

威廉姆森在谈到 Pimoroni 自己的 Pi 供应情况时说:“我们偶尔会有少量 SBC 库存，就像其他经销商一样。”他说，Pimoroni 用来应对核心产品库存挑战的策略是专注于供应更充足的其他 Pi 产品。“我们有大量的 RP2040/Pico 库存，并且正在专注于我们支持这种库存的产品，”他指出。

Upton 的 4 月更新还敦促买家考虑(2020 年推出)树莓 Pi 400；或者是(2021)Pico——它是使用 Pi 自己的硅和一个内部开发的芯片(RP2040)制成的——作为其他供应更受限制的电路板的替代品，因为他说这些生产线受供应问题的影响较小，所以你应该更少等待获得 Pi。

鉴于围绕短缺的持续聊天，我们联系了厄普顿，聊了聊它是如何管理如此持续的库存限制期的——应该说，厄普顿在度假期间主动打了一个电话(所以他当然不能被指责逃避这个问题)。

他也欣然同意，对于 Pi 的购买者来说，这是一个令人沮丧的局面——事实上，对于树莓 Pi 的每个人来说，它总是双重关注家酿(业余爱好者)和工业(商业)用户，所以个人购买者绝对是其客户群的核心组成部分(原谅双关语)。

“我们真的很抱歉——这不是我们选择的情况，”他在一次聊天中主动说道，聊天的背景是中世纪教堂的钟声和广场的嗡嗡声。“这不是任何人想要的局面。我们竭尽全力想摆脱困境。”

他也有一些好消息:告诉我们他希望持续的库存短缺将从明年年初开始缓解。所以——手指搜索——那些恼人的“缺货”通知应该会在 2023 年开始变得不那么明显。(通常的“黑天鹅事件”警告当然适用，特别是因为 2022 年不缺乏新的地缘政治冲击，例如俄罗斯在欧洲开始一场陆地战争。)

2014 年，艾本·厄普顿带着圆周率重返 TC 扰乱伦敦。图片来源: TechCrunch

“好消息是，不管宏观环境是否发生变化，我们都希望我们的具体供应挑战会有很大改善，比如说——不迟于——明年的 Q2，”他说。“我认为这是一个很好的猜测，因为有一些供应方面的因素会有所帮助。这里有更多的制造能力。”

他对这一点的信心与拥有更大的 28 纳米硅生产能力有关——这种硅用于 Pi 4 产品。

“未来 12 个月将会有更多的产能上线，我认为这可能会结束我们的特定短缺，”他预测道。“我可能对 Q2(作为 Pi 供应短缺的预计结束日期)相当满意，因此我们现在的挑战是如何将它引入 Q1——我们如何维持几个月……我们如何稍微挤压一下，我们可以使用什么杠杆来实现那种‘恢复增长’?

“因为这与收缩无关，而是缺乏增长。这就是我们如何比我们从系统中自然获得的早几个月恢复增长。”

当被问及迄今为止影响聚酰亚胺生产的最大供应链问题是什么，或者自疫情中断开始以来是否有许多不同的问题时，厄普顿说，这实际上是一个不断变化的供应限制，他们必须管理和应对每一个问题。

“实际上，它的基础相当广泛。在任何一天，都有一些挑战性的组成部分，但它的基础相当广泛。他解释说:“这是一种你可以感觉到很难超越的自然生产率，因为你知道，核心逻辑有一些限制，然后你通过它，无线是一个问题，或者一些电源组件是一个问题。”

“我们非常出色，因为主板上的大多数元件至少都是双源的。这给了我们一点弹性。但实际上它非常持久，基础非常广泛，”他补充道。“是一切。”

他讲述了一个电路板的供应问题，该问题是由上海下令实施的新封锁引发的——他形容这些“战术性的小短缺会让生产线停工一两天”,因为它们会波及(并破坏)供应链，因此仍有可能影响生产率。

“我们遇到了一个问题，我们正在从一家供应商——一家晶圆供应商——的硅前端问题中恢复过来，但他们的封装、测试和发货都在上海，没有人能去上海工作，”他说，并补充说，这种“地理”问题肯定是一个持续的头痛问题。“中国的滚动封锁可能是——我不会说是战略短缺的最大来源——但它们是日常(供应问题)的最大来源。”

与西方政府不同，中国政府继续奉行“零新冠肺炎”政策——这意味着大规模封锁仍不时发生(当新冠肺炎病例群被发现时)，例如 3 月份的上海全城封锁。

此类事件有可能引发本地化的劳动力限制，如果公司依赖中国的采购，这可能会抑制国际硬件生产(如果像 Pi 一样，你的业务是制造电子产品，这当然是难以避免的)。

不过，在 Pi 的案例中，Upton 告诉我们，它通常能够依靠在其他地方采购组件来避免局部中断。“幸运的是，这些组件中的大多数都是多源组件，你往往能够解决这些问题，”他在谈到上海事件时说。

他补充说，自从全球供应链中断开始以来，圆周率制造商已经做了更多的多方采购。“我们已经相当稳健了——因为我的意思是，即使在正常时期，你也必须如此——你不能把所有的鸡蛋放在一个篮子里。但是我们已经做了很多。”

当然，就其性质而言，在维持“正常”生产率方面，变通办法不太可能完全不受影响——尤其是在全球供应危机期间。当你的主要供应商都顺利运作时，总会有一些延迟/意外的转换成本。因此，这是关于修补供应链，以尽可能维持生产，而不是在一个位置，以提高利率。

此外，厄普顿认为，全球半导体短缺还有另一个更难量化的影响:工程时间成本。 因此——在某种程度上——产品创新。 简单来说，供应链中断不仅仅是减缓生产；它给你开发下一件大事的能力踩了刹车。这意味着对下一个Pi 的等待可能会更长。

“我想，对我们来说，一个真正的烦恼是，我们在合格的组件替代品上花费了工程努力，而我们宁愿在制造下一个大东西上花费工程努力，”他说，并补充说:“我认为这在整个行业都很常见。从创新的角度来看，这种方式浪费了工程时间。

“我们可能将 40%的工程能力花在了我传统上认为并不理想的事情上，因为它们并没有起到推动作用。他们把球放在原处；他们不让球向前移动……我认为这是一个挑战。”

让工业客户吃饱

全球半导体短缺是 Pi 供应问题的根源，因为它阻止了该公司扩大制造规模以满足不断增长的需求，当一家企业发现自己对其产品的需求增加时(否则)就会出现这种情况——正是因为电子产品供应问题如此普遍。

基本上:对于这些全球短缺，没有完美的解决办法——这意味着 Pi 现在根本无法满足对其产品的所有需求。

该公司几乎所有的产品都是在南威尔士的英国工厂组装的，但据 Upton 称，在需求高峰期，例如产品发布前后，该公司还依赖一个临时设施——在将电池板运回威尔士之前，使用索尼在日本的“取放”设施来完成部分前端组装工作，在威尔士，Pi 自己的工厂负责剩下的工作(也进行测试、包装和运输)。Pico 的产品也产自日本。与大多数其他套件相比，这些主板不太受供应限制。

但是，就目前而言，库存中断仍然是规则，因为对 Pi 产品的需求继续超过其可以提供的生产率。(在 4 月份的更新中，Upton 写道，尽管面临各种供应挑战，但在过去的 6 个月里，它每月“一直”能够生产“大约 50 万”块电路板和模块——因此，据报道，现在 Pi 的生产率每周超过 10 万块，似乎大约在这个水平左右，或者可能略低于这个水平。)

正如我们在之前报道的那样，在疫情期间，当许多人被困在室内无所事事时，对 Pi 产品的需求加速增长——一些人的额外可支配收入在他们的口袋里烧了一个洞。因此，锁定无聊似乎鼓励了一群人冒险尝试硬件黑客技术。

厄普顿还指出，在学校停课的时候，人们对儿童教育的担忧是另一个巨大的需求驱动力，因为 Pi 微处理器可以变成价格非常低廉的家庭学习电脑——只需抢救几台旧显示器和键盘，比如通过寻求当地企业的捐赠，就可以为儿童提供廉价电脑。(专业提示:他建议要求律师事务所植入未使用的外围设备，因为他认为他们倾向于保留旧的 IT 设备。)

不过，有趣的是，他表示，尽管大多数疫情的封锁已经放松，但对 Pi 的需求仍在持续上升。因此，它没有经历疫情加速视频会议平台遭遇的那种高峰崩溃，当时面对面会议又开始恢复了。

“目前需求非常旺盛，”他告诉我们。“一切都[起来]了。这个行业的需求非常非常强劲。好吧，也许会出现经济低迷……但这只鞋比我们预期的时间要长得多。”

目前还不完全清楚所有的增长动力到底是什么，但刺激 Pi 产品需求的一个因素是其业务的工业方面——正如 Upton 所说，其嵌入式计算产品线(又名计算模块)的最新更新自推出以来(就在疫情之前)一直需求特别高。

“可能最大的一件事是我们在 2019 年底推出了计算模块 4 。它的起飞非常艰难。这对我来说真的还是莫名其妙…它就像火箭一样起飞了。直到今天我也不知道为什么。

“可能是其他东西的可用性降低了。如果您想一想与计算模块竞争的是什么，那就是自己进行预测和设计。所以放下芯片，去买芯片，去买内存，去设计电路板，自己把它放到电路板上。计算模块可能会出现的情况是，较老的硅芯片不可用，这种全球性冲击已经影响到我们，因为人们说，我不能去买恩智浦芯片，所以我将使用该计算模块；我了解这些人，我们信任他们。”

“这只是一个非常奇怪的曲线，计算模块 4 一直在吹屋顶，”他补充说我们的业务很快就突破了每年 100 多万台。"

对计算模块的高需求对 Pi 供应 SBC 库存的能力产生了直接的连锁影响，因为他说两条生产线使用相同的组件-将该问题描述为“一种有限的[资源]”。“模块加单板计算机的总和在某种程度上是受限制的，”他补充道。"当模块上升时，单板计算机就会下降。"

供应紧张意味着 Pi 与工业和 OEM 客户的互动方式发生了巨大变化，这要求它建立比疫情时代之前更紧密的工作关系，当时它能够生产大量额外的库存。

自 4 月份的博客帖子以来，Upton 表示，Pi 已经与 2000 多家直接 OEM 供应商建立了关系。

“我们确实认为我们在原始设备制造商方面做得很好，”他告诉我们。“Raspberry Pi 的价值主张一直是，明天你就可以买到 10 万个——这在我们的行业中是不可思议的，对吧。没有人这样做…所以当你有这样的世界，你真的不需要与你的原始设备制造商有太多的联系；你知道，这种产品是现成的——你只要在需要的时候去买就行了。[但]我在这种[供应]受限的环境下，这促使我们与原始设备制造商建立更密切的关系。”

厄普顿敦促所有仍在努力获得所需产品的原始设备制造商通过专门的business@raspberrypi.com电子邮件联系——以便 it 部门能够尽力支持他们。

“如果人们有困难，他们可以发邮件给 business@raspberrypi.com，我们会找出他们是谁，”他说。“我们将尽最大努力发现他们不是黄牛——他们是有真实需求的原始设备制造商。我们不想要打算转卖它们的人。我们不希望那些正在建立库存的人，你知道，建立一种缓冲，让他们感到舒服。我们想知道人们的确切需求是什么——你每天需要多少人来保持生产线运转，这样我们就不会成为你生产中的一个问题。

“我们找出这些是什么，然后尽最大努力去实现它们。我们一个月生产成百上千个树莓馅饼——我认为如果管理得当，这就足够了，如果你摆脱了恐慌性购买的倾向，这就足够了；这将满足原始设备制造商的需求。因此，我们认为我们在保持原始设备制造商方面做得很好，但对我们来说，令人心碎的事情当然是我们是一个爱好者的企业——我们既是工业企业，也是爱好者的企业。”

“这对我们来说都非常重要，”他补充道。“因此，当我们谈论‘恢复增长’时，这就是我们所谈论的——恢复到那种‘库存充足’，‘10 万到 20 万库存在渠道中晃荡’的状态，这种状态持续了十年，也就是在此之前的八九年。”

厄普顿也重申了前面提到的给渴望 Pi 的个人的建议——让制造商回到供应较少受限的产品，比如 Pico。

“去年 1 月，我们推出了自己的第一款派对芯片。我们控制着 Pico 产品中的所有主要硅——因为我们自己制造了 RP2040 微控制器。而且，你知道，这是一个什么时候出现！所以 Pico 是一种[较少受供应限制的产品]。我们也有一些预算——如果需要，我们可以建造 1000 万到 2000 万个微微蜂窝。这没有问题，”他说，并补充说 Pico line 目前每年销售“几百万”台。

“显然，这不是树莓派平台——当人们想到树莓派时，他们往往指的是大型模型，我们称之为‘大树莓派’——Linux 树莓派。但实际上，你可以用新的无线设备 Pico W 做些什么，两者有很多重叠之处。”

他还观察到，一些爱好者已经能够切换到比他们计划使用的替代套件——指出 Pico 可以作为零的替身。

“有趣的是，许多人对 Raspberry Pi 的应用实际上包括运行 Python 脚本、报告结果以及通过 wi-fi 传输数据，仅此而已。因此，有趣的是看到人们想方设法做本来是 Raspberry Pi Zero 的项目，或者 Raspberry Pi Zero W 的项目，并想方设法使用 Pico 和 Pico W 平台做这些项目，因为它们的供应更好……因为 Pico W 当然有无线功能，它运行 Python。”

“微控制器上高级编程语言的数量改变了游戏，”他补充道，并对 Arduino 在使空间更易访问方面的工作表示认可。“微控制器总是有……有点令人毛骨悚然的名声。但你可以只写一个 Python 脚本——就像一段普通的编码一样。”

同样乐观的一面是，厄普顿对与英国当前经济动荡有关的任何担忧都不予理会——利率和通胀飙升，而英镑，呃，没有——他表示，他相信国内的经济问题不会在短期内导致 Pi 新鲜供应(和/或成本)问题。

这是因为这家英国公司是以美元业务为基础的，这为其抵御当地财政动荡提供了一些缓冲。

“我们是一元生意。我们用美元买东西，用美元卖东西，用美元报告我们所有的财务状况。因此，我们是一家总部设在英国剑桥的美元企业。唯一直接(以英镑计价)的是我们的工资支出，或者说我们的大部分工资支出，因为我们确实有一些美国员工，”他解释道。

“当然，从中期来看，我们的制造工资支出是以英镑计算的，因此你会预计到——这是一个缓慢的影响——基于工资成本的制造报价的变化，但随着时间的推移，你会预计到以美元计价的制造成本会发生变化……但是，总的来说，当然，目前汇率的走向对我们来说不是问题，因为这会使……目前英镑固定成本变得更便宜。”

“但这对我们来说并没有太大的影响。从某种程度上说，这是一种‘积极的’影响，”他补充道。“我的意思是，你不能让债务绿洲贬值，也没有人主张让英镑贬值，以改善英国制造业，但在某种程度上，这种影响是无法命名的。”
