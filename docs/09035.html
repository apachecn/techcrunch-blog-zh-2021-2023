DNSFilter 收购 iOS 防火墙 app Guardian

原文：https://web.archive.org/web/https://techcrunch.com/2022/08/10/dnsfilter-guardian-ios-firewall/amp/

图片来源: Getty Images

位于华盛顿特区的 DNSFilter 是一家提供基于 DNS 的网页内容过滤和威胁防护的公司，该公司宣布收购 Guardian，这是一家用于 iOS 的隐私保护防火墙。这笔交易的财务条款没有披露。

Guardian 于 2013 年由安全研究员、前 iPhone 越狱者 Will Strafach 创建，他在 2017 年发现 AccuWeather 在未经用户许可的情况下，秘密向第三方公司发送精确的位置数据。该公司的“智能防火墙”iPhone 应用程序通过加密的虚拟专用网络(VPN)传输数据，阻止应用程序与第三方共享用户的个人信息，如 IP 地址和位置数据。这家初创公司声称迄今为止已经阻止了超过 50 亿个数据追踪器和 10 亿个位置追踪器，最近与 Brave 联手，将其防火墙和 VPN 功能集成到其同名的非追踪浏览器中。

在宣布收购 Guardian 的一份声明中，DNSFilter 表示，这家初创公司的技术，加上其 DNS 层安全性，将有助于扰乱安全网络网关市场(或 SWG)，该市场依赖于静态签名和威胁馈送列表，很少识别变异或新的攻击载体。 Guardian 将为 DNSFilter 配备 完整的网址和 ip 地址、文件类型、端口过滤和 VPN 功能，实现“为 DNS 之外的客户过滤新的互联网流量层”

DNSFilter 联合创始人兼首席执行官 Ken Carnesi 表示，虽然安全的网络网关在现代网络安全战略中发挥着“关键作用”，但将 DNS 层安全与防火墙和 VPN 组合结合起来"会带来巨大的商机，并缓解客户在传统 SWG 技术中面临的难题

Guardian 将仍然是一个独立的应用程序，而 DNSFilter 的预测 DNS 保护将进一步帮助 Guardian 用户。

DNSFilter，成立于 2015 年，旨在保护用户免受密码劫持和勒索软件等威胁，目前拥有超过 15000 名客户，包括英伟达、联想和洛杉矶地铁。去年，该公司从 Insight Partners 获得了 3000 万美元的首轮融资。Guardian 是该公司最近收购的一家公司。2018 年，DNSFilter 收购了 Web Shrinker，这是一家使用人工智能和机器学习来实时分析和识别威胁和域的初创公司。T9】
