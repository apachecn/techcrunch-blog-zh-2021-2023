我们都一直在关注条纹与格子最近上演的戏剧。我不会在这里重复所有这些，我会给你看一些我们最近关于这个主题的文章，只是总结一下:这两家金融科技初创公司最近变得(更)有竞争力了。

如果事情还不够混乱，另一家初创公司已经非常公开地成为 Stripe 的强大竞争对手: Finix 。

现在，菲尼克斯不是凭空出现的。这家 SaaS 初创公司于 2020 年初通过向其他企业出售其支付技术起家，筹集了由红杉资本牵头的 3500 万美元 B 轮融资。一个不寻常的转折是，仅仅一个月后，红杉退出了交易，据报道，在交易中，红杉给自称的支付基础设施公司开了一张 2100 万美元的支票。正如 TC 的 Connie Loizos 当时报道的那样，Finix 告诉员工，在签发支票后不久，红杉认为 Finix 与支付公司 Stripe 的竞争过于直接，Stripe 是红杉最大的私人持股之一，也是红杉最大的外部投资者之一。

快进到上周。Finix 宣布，除了使其他公司能够促进支付之外，它正在成为支付服务商。这一举措使其与 Stripe 展开了直接竞争，首席执行官兼联合创始人里奇·塞尔纳并不羞于承认这一点。

在过去一周的一次采访中，Serna 详细说明了 Finix 确实是从开发软件开始的，这给了任何软件公司成为他们自己的支付服务商的途径。

“我们正在开发一种技术，需要几十名工程师花三年时间在内部开发，需要数千万美元的技术 R&D 和投资，然后通过让开发人员友好的 API 开始将他们的支付货币化，将这一过程缩短到几个月，”他说。“这是我们最大的核心产品。我们现在所做的是自己成为支付服务商，这样我们不仅可以提供支付，还可以提供所有后台要求和合规性认证，这样我们的客户就可以在几天内启动并运行，而不是几个月。”

他说，此举使 Finix 能够与处理量为 0 美元的公司和软件平台合作，一直到处理量为数十亿美元的公司。

Serna 补充说:“这使这些客户能够获得更好的产品体验和更快的上市速度，并使我们能够承担推广和货币化以及获得支付的非技术方面的工作。”

你看，从历史上看，公司需要达到一定的数量阈值，然后 Finix 才能与他们合作。但是现在，根据 Serna 的说法，他们可以在最早的状态下开始与它们一起工作。

“客户可以从第一天开始与我们合作，使用财务 API，当他们准备好承担更多的所有权和更多的风险、承保和合规运营责任时，他们可以毕业并成为自己的支付服务商，”他说，“因为我们仍然使用完全相同的 API。”

菲尼克斯还进入了这位高管所称的“卡片展示”或当面支付领域。这意味着它能够为许多类型的企业提供接受信用卡支付的软件。

Serna 说:“如果你考虑一家餐馆软件提供商，他们将需要一套不同于健身房或快餐车设备提供商的设备。“这就是我们独特的产品，并将其推向市场。”

所以，如果你还没弄明白，Stripe 确实有理由担心，因为 Finix 确实在直接与它竞争。那么它们有什么不同呢？

根据 Serna 的说法，答案在于 Finix 已经建立了“一个模块化和可配置的开放系统和开放架构”。另一方面，他说，Stripe“继续加倍锁定供应商，以便继续关闭他们的系统和架构。”

“我们认为它非常类似于 iOS，”Serna 告诉 TechCrunch。“我们认为自己更像 Android……我认为，随着我们继续打造我们的产品和公司，我们将继续看到这些特征被放大。”

Finix 只有 150 多名员工，目前使用其 API 为美国 12，000 多家商户提供支持。它已经从美国运通风险投资公司、贝恩资本风险投资公司、Homebrew、灵感资本、光速风险投资伙伴和 Visa 等投资者那里筹集了约 1 亿美元的资金。

与此同时，在《福布斯》最近的一篇文章中，Stripe 的联合创始人约翰·科利森对亚历克斯·康拉德说，据说耸了耸肩:“我们将与一群公司竞争，我们将与一群公司合作。每个人都需要成为一个成年人，并对此表现良好。”在同一篇文章中，消息人士告诉亚历克斯，Stripe 在 2021 年的总收入约为 120 亿美元，同比增长 60%。据报道，该公司的净收入约为 25 亿美元。

每周新闻

谈到 Stripe，Ingrid Lunden 在 5 月 24 日报道称，该公司首次推出了其应用程序市场，这是一项新产品，Stripe 将提供对第三方应用程序和由应用程序发行商、用户和 Stripe 自己创建的脚本的访问，这些脚本将这些应用程序与 Stripe 结合在一起。这可能代表着它在支付之外的最大飞跃。

据报道，瑞典支付巨头 Klarna 在过去的一周内裁员 10%,即 700 人。此前《华尔街日报》报道称，该公司将下调估值以筹集新资本。

一键式结账创业 Bolt 据信已经让多达 240 名员工在走向市场、销售和招聘岗位上下岗。早先的报道称将有 100 名工人受到影响，但随着细节的出现，似乎更多。2 月中旬，创始人瑞安·布雷斯洛(Ryan Breslow)在推特上宣布博尔特为每位员工提供从公司借钱行使股票期权的机会后，成为头条新闻。现在，还不清楚那些被解雇并向公司借钱的人会怎么样。该公司告诉彭博，贷款的受影响工人数量是“个位数”

但是并不是所有的金融科技公司都在裁员。 Fidel API 表示，在投资 6500 万美元宣布 B 轮融资后，它“正在快速增长”，并正在招聘工程、销售和客户体验团队的 60 多个职位。fintech 表示，在过去 6 个月中，其规模已经翻了一番，并打算在年底前再翻一番。

佩吉·曼格特(Peggy Mangot)已经辞去贝宝风险投资公司(PayPal Ventures)运营合伙人的职务，担任摩根大通商业银行金融科技合作伙伴的新主管。在 PayPal Ventures，Mangot 帮助领导全球金融科技、商业、基础设施和加密领域的投资。
