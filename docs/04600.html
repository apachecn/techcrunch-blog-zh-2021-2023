Cityblock 联合创始人讨论为什么一个负担得起的、以人为本的医疗保健模式行得通

原文：https://web.archive.org/web/https://techcrunch.com/2022/03/07/cityblock-co-founder-discusses-why-an-affordable-human-centered-healthcare-model-works/

Cityblock 的共同创始人讨论了为什么一个负担得起的，以人为本的医疗保健模式的工作

在 Found Live 上，Darrell 和 Jordan 与 Cityblock 的联合创始人兼总裁 Toyin Ajayi 进行了交谈。Cityblock 向历史上在医疗保健系统中被边缘化的人提供初级保健、行为健康和社会保健，主要是通过医疗补助计划接受医疗保险的人，其目标是以较低的成本提供高质量的整体保健。他们通过医生网络、社区外展团队以及与保险提供商合作来实现这一目标。当 Ajayi 医生在医院系统工作时，她对一次又一次将相同的病人送回医院的系统问题感到沮丧。因此，她开始尝试扩大护理范围的方法，比如给出自己的手机号码、上门服务以及解释药物治疗。这个想法发展到了 Cityblock，那里的个性化护理水平是成比例的。他们谈到了医疗保健系统，提供商如何改善医疗服务，Toyin 的辛辣 TechCrunch Disrupt 小组，并回答了一些观众的问题。
