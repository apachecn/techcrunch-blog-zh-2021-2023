Twilio 凭借基于 32 亿美元细分市场收购的新工具更深入地研究营销技术

原文：https://web.archive.org/web/https://techcrunch.com/2021/10/20/twilio-delves-more-deeply-into-marketing-with-new-tool-built-on-3-2b-segment-acquisition/

当 Twilio 去年以 32 亿美元的高价收购 Segment 时，你知道它将会投入使用。今天在该公司的信号客户会议上，它宣布了一个新的全渠道营销工具，称为 Twilio Engage，建立在细分市场之上。

Segment 首席执行官 Peter Reinhardt 表示，新工具旨在利用 Twilio 产品家族的几个部分，包括 Segment，来创建一个全新的云营销技术(martech)工具。

他说:“我们今天发布的是 Twilio Engage，这是一个全新的增长自动化平台，以客户数据平台(CDP)和 Twilio 的通信 API 为基础构建而成。新应用程序的目标是帮助营销人员更好地了解他们的客户，并提供一种交付更多定制信息的方式。

它首先利用细分市场，从 50 多个数据源收集客户信息。接下来，真正驱动这个工具的是一个新的旅程编排组件，它让营销人员为每一组客户定义一个工作流。然后，他们可以通过不同的通信渠道编写和发送定制的消息，包括电子邮件和短信，这就是 Twilio 的核心通信 API 发挥作用的地方，他们还可以连接到各种外部工具。

当旅程工作流程满足特定条件或遇到特定触发时，这些不同的通信组件可以以定制的文本消息、电子邮件或广告的形式交付。“你现在可以创建一个完整的宣传活动，一应俱全，所有的内容，所有的受众创造，所有的目标都在 Twilio Engage 内部，”他说。

举个例子，假设一家运动鞋公司有一款跑步应用。使用 Twilio Engage，他们可以建立一个活动，当用户在该公司的应用程序中记录 100 英里时开始。这个里程碑可以触发带有祝贺信息的 SMS 文本(使用 Twilio 的通信 API)。如果个人也是现有客户，系统可以自动发送电子邮件祝贺他们达到第一个目标，并包括 20%的优惠券。如果用户不是客户，该公司可以通过不同的广告渠道——脸书、Snapchat、谷歌——以类似的折扣和信息重新定位他们。

Twilio Engage 工作流程。图片来源: Twilio

Futurum Research 的创始人兼首席分析师丹尼尔·纽曼表示，尽管全面进军 martech 是一件新鲜事，但该公司已经凭借 CDP 及其消息平台在那里站稳了脚跟，他认为它可以利用其开发人员社区来成功实现这一转变。

“虽然 CDP 和 martech 环境竞争激烈，但我相信 Twilio 及其忠实的开发人员社区将会被该公司的扩展产品所吸引，因为了解客户的[完整]旅程仍然是营销人员和收入负责人的一个复杂但越来越可能实现的目标，”纽曼说。

很明显，Twilio 希望将其平台和收入机会扩展到公司赖以建立的核心通信 API 之外。Reinhardt 说，他和 Twilio 首席执行官 Jeff Lawson 清楚地认识到，结合两家公司的工具集的最佳用例涉及营销，这就是他们下注的原因。

“我们与客户进行了交谈，当我们看到人们如何在 Twilio 上使用消息传递 API，以及人们如何在细分市场上使用数据时，很明显，双方的[最大用例]都是营销。我们可以做很多事情来使这变得更容易和更好，”他说。导致他们建立了 Twilio Engage。

它将于今天在部分客户中进行试点，预计将于明年上半年全面上市。
