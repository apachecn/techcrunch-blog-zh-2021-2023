Built Robotics 再融资 6400 万美元，使建筑设备实现自动化

原文：https://web.archive.org/web/https://techcrunch.com/2022/04/07/built-robotics-raises-another-64m-to-make-construction-equipment-autonomous/amp/

图片来源:建造机器人

如果你还在思考无人驾驶汽车的想法，那就做好准备吧，这可能看起来更疯狂:无人驾驶建筑设备已经存在，而且已经在工地上挖掘了。

我们第一次写道在 2019 年建造了机器人，当时该公司正处于启动其自主挖掘机的早期阶段。几个月内，他们就开始将努力扩展到其他建筑设备，如推土机和装载机。

但是，随着疫情把所有人都送到了公司总部，公司知道必须改变现状。快速将任何公司转移到远程会带来挑战。但是一个致力于大型自主机器人的公司？他们不得不调整他们的方法。

“我们真的只是想加强我们的重点，”建立创始人诺亚准备-坎贝尔告诉我。因此，他们专注于似乎已经奏效的业务:挖掘机。具体来说，就是用挖掘机自主挖沟。

“几乎所有类型的建筑都需要壕沟，”Ready-Campbell 说。“这非常重复，但准确性非常重要。你需要确保你在正确的深度；如果你走得太深，就会有破坏沟壁的风险，而且对帮助安装管道或电缆的人来说也不安全。”

“制造”并不制造挖掘机本身，卡特彼勒和 John Deere 等公司几年前就锁定了这一部分。相反，Built 提供了一个被称为“exosystem”的售后附加服务，它表示可以为“任何挖掘机”增加自主性。他们对 exosystem 收取每台机器每月 3000 美元的费用，外加根据地点不同而不同的每小时使用费。

人们在大型自动驾驶挖掘机周围工作安全吗？当我问的时候，坎贝尔微笑。在超过 13，000 小时的运行后，他说他们“仍然有一个完美的安全记录。我真的为此感到骄傲。”他概述了一个“八层安全系统”，其中包括地理围栏，如果机器人离开特定区域，它会自动关闭机器人，工作现场周围的无线紧急停止按钮和挖掘机上的硬连线紧急停止按钮，以及不断扫描机器周围的人和障碍物的计算机视觉。

这一轮是 6400 万美元的 C 轮融资，由 Tiger Global 领投，并得到了 NEA、创始人基金、第五墙和 Building Ventures 的现有投资者的支持。这一轮融资使 Built 的总融资额达到 1.12 亿美元。
