大众全电动 ID 7 轿车最有趣的特点不是它的亮灯漆

原文：https://web.archive.org/web/https://techcrunch.com/2023/01/03/the-volkswagen-all-electric-id-7-sedans-most-interesting-feature-isnt-its-light-up-paint/

全电动大众 ID 7 轿车——德国汽车制造商阵容中的下一款电动汽车——于周二在拉斯维加斯举行的 CES 2023 上亮相。

首次亮相最令人兴奋的部分不是车辆的黄色、橙色、红色和蓝色伪装电致发光涂料的多层爆炸，这种涂料根据命令点亮(尽管它对于车辆的近生产版本来说是引人注目的)。相反，它是之前 ID 车型的一些功能和变化，表明大众正试图用一种更接近消费者在特斯拉车型、保时捷 Taycan 和 Rivian R1S 和 Rivian R1T 中可能找到的方法来提升其软件和人机界面设计游戏。

图片来源:柯尔斯顿·科罗塞克

大众乘用车首席执行官托马斯·谢弗(Thomas Schä fer)表示，ID 7 轿车是将于 2026 年投产的 10 款新电动车型之一，此前在中国以 ID Aero 的名字作为概念车展示。这款电动车将在中国、欧洲和北美上市。

与其他 ID 兄弟一样，ID 7 基于 MEB 平台，这是一个灵活的模块化系统，实际上是一个通用部件的矩阵，大众集团品牌，包括奥迪、西亚特、斯柯达和大众，使用它来提高生产电动汽车的效率和成本效益。仅在欧洲销售的电动掀背车大众 ID.3、大众 ID.4 SUV 和 ID.5 ，以及奥迪 Q4 e-tron 的各种变体都是在 MEB 平台上制造的。到 2025 年，汽车制造商预计超过 80%将是电动的。

ID 7 确实有一些值得注意的突出变化和功能，首先是汽车信息娱乐系统的新界面。中央显示屏更大，为 15 英寸。但更重要的是，用户访问位于屏幕上的功能并与之交互的方式已经改变。

气候功能现在突出并永久显示在中央信息娱乐屏幕的底部。沿顶部和左上角是驾驶员可以快速访问车辆信息和高级驾驶员辅助功能的地方。

图片来源:大众汽车

大众采用了一种新的空调系统，该系统使用了类似于特斯拉汽车配备的数控通风口，这在一些汽车圈可能会引起争议。这意味着没有更多的物理通风口，司机或乘客可以移动来改变气流的方向。相反，用户可以通过语音控制或中央触摸屏来改变气流方向。

空调系统的设计也更加直观。如果司机拿着钥匙靠近车辆，系统应该打开，并在热天开始冷却车内，或者在温度较低时加热车内。

大众汽车也在试图加强其自然语言处理游戏。该公司表示，用户将能够给出具体的语言提示，如“你好，大众汽车，我的手很冷！”该系统将启动方向盘加热功能，并将暖风吹到驾驶员的手上。

ID 7 的其他增加和变化包括增强现实平视显示器和触摸屏底部的照明触摸滑块。

图片来源:大众汽车

外观设计确实符合大众目前的电动汽车产品组合。轿车有一个空气动力学的前部和车顶，有一个像轿跑车一样的斜坡。据该公司称，这种设计在彩色迷彩的照明下很难识别，旨在帮助降低能耗和增加射程。进气口位于前端，引导空气通过进气口，沿车辆两侧流向后部，形成一个气帘，使车辆两侧的气流平稳。

最终结果是一个大约 700 公里的估计范围(在欧洲 ELTP 循环下)，或大约 434 英里的范围。
