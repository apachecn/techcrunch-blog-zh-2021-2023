有很多理由认为，传统风险行业在经历了一段非常好的时期后，可能正处于转折点。最明显的迹象是近年来私人投资者获得的般的丰厚回报，而公共股东却没有这么好的表现。看看《华尔街日报》昨天发布的数据，数据显示，过去 13 个月中，风险投资支持的上市公司，包括 DoorDash、Oscar、UiPath、Compass、Robinhood 和 Coupang，其股价分别比首日收盘价低 40%、-81%、-47%、-56%、-59%和-28%。

在光谱的另一端，事情看起来也不那么健康。正如 TechCrunch 上周指出的，种子期、A 轮和 B 轮公司最近几个季度的收入远低于过去几年。很可能，这是因为创业公司正在以更快的速度筹集资金(你只能在几个月的时间里取得这么大的进展！).但投资者也比以往更加放松。没有进展？显然，这没什么大不了的。重要的是对创始人的押注。

尽管如此，风险投资可能使用重置的最强指标与风险投资者为尚未创办公司的人提供资金的渴望有关。“这不是一个新趋势，”通用催化剂公司的总经理尼科·博纳索斯说。“但由于种子前投资和种子投资的大量增加，这种现象现在变得越来越明显，”他继续说道。“有大量的普通合伙人、更大的基金和更多的交易，我们都是靠投资获得报酬的。”

advanced Ventures 的马克·苏斯特正在这么做。“假设我们在 Riot Games 认识你，在 Snapchat 认识你，在脸书认识你，当你在 Stripe 或 PayPal 工作时认识你，”去年秋天 Suster 告诉我们。"我们会在第一天的编队中支持你."

基金会资本的董事总经理 Ashu Garg 上周分享了类似的策略。“我们的目标是，在他们准备好创办公司时，与他们握手言和。这就是我们所处的行业。没有公司。那就是我们的商业模式。”

问问许多老牌风险公司，你会听到差不多的答案。

克里斯·法默(Chris Farmer)于 2013 年创立了风险投资公司 SignalFire，他是最早也是最公开的“量化”风险投资倡导者之一。当时，SignalFire 的平台 Beacon 正在跟踪来自 200 万个数据源的超过 5 万亿个数据点，从专利到学术出版物，到开源贡献到财务文件，以努力确定工程人才的来去情况等。

Farmer 当时是唯一一个使用这些数据作为营销工具的公司，但许多公司后来都采用了类似的系统，尽管没有那么密集，包括使用公共和私人数据来跟踪尚未离职的个人，或者已经离职但尚未宣布任何计划的个人，或者只是注册了一家公司作为第一步的个人。

其中一些简单得荒谬。“有些数据，你可以追踪很多年，”Bonatsos 说。“如果有人在 Twitter 或 LinkedIn 上修改了自己的简历，这是他们向寻找这些东西的人发出的信号，这是一种表示‘我在做一些新东西’的方式。”"

对于大型风险投资公司来说，只需几名分析师和一些基本的脚本，就可以相对轻松地标记出加州等州可能感兴趣的每一家公司。“我们至少有六个主要同行在做同样的事情，”博纳索斯说，“因为当我们给创始人发电子邮件时，他们有时会说，‘有意思，今天又有几个人给我发了电子邮件。’"

事实上，现在风投们有很多常见的方式来跟踪有才华的个人。有些对风投来说是可见的，因为他们已经筹集了资金，建立了一家公司，然后卖掉了它。(风投通常认为，要么是公司的创始人，要么是他们的早期员工，会考虑很快创办另一家公司。)

有构建企业产品经验的人特别容易成为目标，这要归功于 GitHub 等代码共享平台，它让偷窥狂能够在任何商业活动之前看到哪些项目正在获得更广泛的开发人员社区的支持。

甚至童子军项目的兴起也与这一趋势有关。风险企业与运营高管和初创企业创始人建立关系，以改善他们的交易流程。但也存在这样一种预期，即如果这些星探创办了自己的公司，他们会知道先找谁。

当然，要问的问题是这些是否有意义，目前还没有足够的数据来知道被鼓励创办公司的人是否能胜过以更有机的方式创业的人，比如说通过创新、特别的洞察力——甚至是和。

有了这么多资本的支持，风投们无论如何都会继续这一战略。

就他而言，Bonatsos 已经学会了在寻找人才时考虑某些问题，包括如果某人已经卖掉了一家公司或以其他方式赚了一大笔钱，他是否还会感到饥饿。他试图衡量一个人的思想是“真正当代的还是过时的”。他们会像以前那样重新做事情吗？”他还指出，对于风投来说，“第二次创业可以招募新人，但支持他们最终推出的任何产品会更加昂贵”。

与此同时，法默还继续尽可能早地接触潜在的创始人，认为鉴于融资市场的泡沫，这比以往任何时候都更有必要。

正如他解释的那样，对于一个已经开始运营的初创公司来说，“进入成本”比过去两年高出了 200%。因此，你要么以更高的估值开出一张更大的支票来获得所有权，要么，如果你是反向投资者，冒超早期风险更有意义，因为随着时间的推移，有人愿意支付巨额溢价。”

这是简单的数学，他继续说道。“如果你以 3 倍于 3 年前的价格购买同样的资产，回报可能会低 3 倍。如果你要花 2000 万美元收购一位来自 YC 的校友，而他的过往记录很少，为什么不早点去找一位创始人呢，即使他们的想法还没有形成？”

这一切听起来有点不可思议，但法默观察到，这是“真正疯狂的”正在部署的“大量资金”。他认为，无论哪种方式，SignalFire 以这种方式投资都会活得更久。他说:“哪里有吸引力和成功的信号，哪里的竞争就最激烈。

“然后就是一场残酷的刀战。”
