Passbase 获得 1000 万美元，重新将身份验证的重点放在加密上

原文：https://web.archive.org/web/https://techcrunch.com/2021/12/08/passbase-series-a/

数字身份初创公司 Passbase ，提供用于运行远程身份检查的 SDK，已经筹集了 1000 万美元的首轮融资，因为它引起了人们对加密合规性的关注——宣传工具以帮助金融科技公司应对快速发展的监管要求。

该基金由 Costanoa Ventures 牵头，Lakestar、Eniac Ventures、Cowboy Ventures 和 Seedcamp 参与其中。

此前，Passbase 在 2019 年筹集了 360 万美元的种子资金，几乎翻了一番。因此，这家 2018 年创立的初创公司迄今为止的总融资额约为 1770 万美元。

数字身份领域是一个拥挤的领域，但在加密合规性方面，Passbase 热衷于回顾早期的经验，并声称这是一个差异化因素——该团队在转向身份认证之前，已经开始构建一个加密货币钱包——它认为这有助于了解这个特定(有些人可能会说是特殊)行业的需求。

这家初创公司表示:“我们的主要关注点是加密，”并将其 SDK 的典型客户描述为“快速增长的数字第一金融服务产品，尤其是对 crypto onramp services 的菲亚特”。

Passbase 的 subscription developer 产品提供了创建虚拟验证流程和后台合规性的工具，以满足 AML(反洗钱)、KYC(了解您的客户)和年龄限制等监管需求，并针对初创企业、成长型企业和企业用户提供不同的层级。

它的最高声明是，该产品可以在短短 30 分钟内集成，并覆盖 190 个国家和 15 种语言的 6，000 多个 id。

Passbase 表示，在这个阶段，它的开发产品有大约 180 个客户。

而在过去一年中，该公司表示收入增长了 8 倍。

虽然(高度泡沫化的)加密市场是 Passbase 发现其监管支持技术最大机会的地方，但它也表示正在关注在线年龄验证的发展——由于对儿童安全的担忧，这一领域在某些市场(如英国)受到监管机构越来越多的关注。

“年龄验证是次要的，但我们有一些用例，如在线游戏、数字大麻、在线销售电子烟和在线销售酒精……但我们确实会继续跟踪年龄验证方面的法规，如 COPPA、防止向儿童在线销售电子烟法案等，”它告诉我们。

Passbase 表示，A 轮融资将用于推出一个新的无代码政策系统，允许客户为多个地区和监管要求设计自己的入职流程——这家初创公司表示，它非常关注那些在扩张到新的市场和地理区域时寻求合规支持的公司。

Passbase 在谈到即将推出的政策工具时表示:“这将满足公司的需求，即了解他们需要做些什么来符合不同的市场(即通过政策模板)并管理如何做到这一点(即我们的验证产品中的工作流)。

另一个预定目标是扩展该系统的功能，以提供它所谓的“针对不同地区和用例的市场优化的入职解决方案”。

它的总体愿景是成为数字身份网络中的关键参与者，该网络由可共享的 KYC 令牌和可验证的证书模型组成——它认为这将实现“保护隐私的身份生态系统”，正如现在被炒作为“web3”的分散式身份模型所设想的那样。

然而，有许多(相当好的)资源丰富的初创公司也在出售身份验证工具——Passbase namesJumio、 Onfido 、 Persona 和 Veriff 是其最接近的竞争对手。

值得注意的是 Stripe 最近还增加了 ID 检查——于今年夏天发布，并通过自助工具为人工智能验证创造了一个新的竞争层。

Stripe 的到来显然让所有梦想成为“用于身份验证的 Stripe”的创业公司感到非常头疼(这正是 Passbase 在 2019 年谈论其业务的方式)。

但是，尽管在核心 ID 检查方面竞争激烈，Passbase 并没有认输——相反，它只是押注于专门的加密合规性需求(专门的加密合规性工具存在市场)。因此，它的营销重点现在谈到将“KYC 带入加密”——并对 Stripe 进入其地盘保持沉默——再次辩称，它在加密和 web3 方面的早期经验为它现在瞄准的加密利基提供了竞争优势。

“重要的是要记住，我们的团队来自 Coinance 的 web3 和加密领域。我们比 IDV(身份验证)领域的任何人都更了解 web3 的需求。当我们寻找解决方案时，这些竞争对手就已经存在，我们没有选择他们是有原因的，”它认为，这加剧了加密宣传。

“我们的种子期让我们在 IDV 成为一个有竞争力的解决方案，并有所改进，如为规模公司提供更灵活的定价，更方便用户的验证流程，为产品负责人提供更深入的定制，以及更好的文档，这样开发人员可以在几分钟内而不是几周内完成集成。A 系列旨在向可访问的全球合规性解决方案迈出下一步，而分散式身份模型 web3 旨在促进这一目标的实现。”
