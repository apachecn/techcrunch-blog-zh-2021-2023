Fixie 希望让公司更容易在语言模型上进行构建

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/30/fixie-wants-to-make-it-easier-for-companies-to-build-on-top-of-language-models/amp/

然而，另一家希望利用生殖人工智能热潮赚钱的初创公司获得了一笔令人瞠目的风险投资。

这家名为 Fixie 的公司由苹果和谷歌的前工程主管创建，旨在将类似于 OpenAI 的 ChatGPT 的文本生成模型连接到企业的数据、系统和工作流程。联合创始人兼首席执行官 Matt Welsh 将其描述为第一个以企业为中心的平台即服务，用于构建使用大型语言模型(LLM)的体验。

“本质上， Fixie 是一个无限可扩展的模型，企业可以将其集成到自己的产品和工具中，”联合创始人兼首席采购官扎克·科赫在一次电子邮件采访中告诉 TechCrunch。“Fixie 的核心是其 LLM 驱动的代理，任何人都可以构建并在任何地方运行。”

Fixie 是否是同类中的第一个是 略有 疑问，但不是的是创始团队的血统。

在加入 Fixie 之前，威尔士在谷歌 Chrome 团队担任了近十年的工程主管。Koch 曾是 Shopify 的产品总监，也是 Chrome 和 Android 团队的负责人。CTO Justin Uberti 是 AOL Instant Messenger 的创始人之一。至于 Fixie 的首席人工智能官 Hessam Bagherinezhad，他是苹果公司的机器学习执行官，负责 iPhone 和 Apple Watch 等产品。

这是 Fixie platform 的一万英尺视图:LLM 驱动的代理与外部系统接口。Fixie 代理可以与数据库、API(例如 GitHub 的)、生产力工具(例如 Google Calendar)和公共数据源(例如 web 搜索引擎和社交媒体)进行交互，以生成和处理任意事物，如图像和文本，然后以各种方式操纵它们。

例如，通过 Fixie，公司可以将语言模型功能整合到客户支持工作流中，方法是构建代理程序，这些代理程序接受客户的票据作为输入，自动查找客户的购买情况，在必要时退款，并生成对票据的回复草稿。

Fixie 代理可以用任何编程语言实现，并可以在任何基础设施上运行，每个代理都可以使用自己定制的 LLM。Fixie 支持 OpenAI 的 GPT-4 等流行模型，但客户可以提供自己的模型或选择其他商业和开放模型。

威尔士说:“最终，我们相信 LLM 会取代很多传统软件，因为这些模型可以充当自然语言驱动的‘问题解决引擎’。"使用 Fixie，不用编写一堆复杂的代码来将两个系统连接在一起，而是简单地将每个系统包装在一个自然语言代理接口中，并让这些系统用英语相互通信。LLM 本身充当了一个非常强大的符号操纵器，不需要编程就可以解析、操纵和合成数据。自然语言可以作为不同计算系统相互交流的通用语言。”

毫无疑问，这是一个引人注目的愿景——open ai 最近为 ChatGPT 推出了插件,从而实现了这一愿景。在本周的一篇文章中，媒体分析师 Ben Thompson 写道插件如何使 ChatGPT 更像一个聚合器或平台，而不仅仅是一个聊天界面——类似于 Welsh 对 Fixie 及其代理家族的描述。

事实上，ChatGPT 插件可能会对 Fixie 造成某种程度的威胁。但是 Welsh 认为 Fixie 平台提供了比 OpenAI 更多的可定制性和自由度，至少目前是这样。

借助 Fixie，公司可以通过预先训练或定制的语言模型将各种应用和服务连接在一起。图片来源:修复

“新的 ChatGPT 插件提供了一种将 OpenAI 的 LLM 与外部 API 连接起来的好方法。但我们对 Fixie 的关注是不同的，”他说。“由于 Fixie 与模型和提供商无关，企业可以利用任何类型的 LLM 并在自己的基础架构上托管代理… Fixie 处理底层 LLM 交互以及用户身份、身份验证、会话管理、存储和配置等细节。”

威尔士在 Zapier 的自然语言动作功能中看到了另一个竞争对手，该功能允许开发者使用自然语言在应用程序、产品和服务之间传递信息。但他并不认为 Fixie 直接具有竞争力，他指出 Fixie 并不是从零开始训练自己的 LLM，而是让客户能够使用专有数据或流经给定代理的历史数据，为他们的代理微调现有的 LLM。

事实上，威尔士提出，Fixie 通过解决采用生成式人工智能的一些主要障碍，即培训 LLM 的高成本以及与当今最佳模型相关的风险，超越了现有的几个步骤。Fixie 允许公司微调而不是自己训练模型，消除了成本支出，他断言，并约束模型的行为，以确保它们更可靠地执行任务和回答问题。

威尔士并没有夸张到他承诺 Fixie 可以完全修复(原谅文字游戏)LLMs 的倾向编造事实，这是一种被称为幻觉的问题现象。(Fixie 也不能解决他们的其他问题，比如偏见和短暂记忆。)他也承认 Fixie 平台的微调替代方案是存在的，比如开源的 LangChain 和 Llama Index。但威尔士强调，Fixie 是为拥有一系列专业知识的用户设计的——理论上降低了部署生成式人工智能的门槛。

也就是说，Fixie 在早期访问计划中拥有约 5000 名用户，并表示正在与“广泛”的公司合作，研究业务自动化、客户支持、生成式人工智能和图形等用例。它将在未来几天公开发布，供个人免费使用，并得到了来自 Redpoint Ventures、Madrona、Zetta Venture Partners、SignalFire、彭博贝塔和卡尼杰克森(Kearny Jackson)的 1700 万美元投资(1200 万美元种子资金，500 万美元前期种子资金)。

确切地说，在过去半年的时间里，Fixie 很容易就找到了资金，而且很快。根据本月发布的一份 PitchBook 报告，风投们稳步增加了他们在生殖人工智能领域的头寸，从 2018 年的 4.08 亿美元增加到 2021 年的 48 亿美元，再到 2022 年的 45 亿美元。天使和种子交易也有所增长，2022 年有 107 笔交易和 3.583 亿美元的投资，而 2018 年只有 41 笔和 1.028 亿美元。

Welsh 说，假设一切顺利，Fixie 计划在年底前将其 8 人团队扩大到 20 人。在此之前，它将主要关注客户获取。

“LLM 为所有类型的软件系统提供了全新的功能，但是企业还没有能够充分利用这些进步，”Welsh 说。“每个人都看到了 ChatGPT 这样的东西的巨大力量，而且人们普遍认识到这项技术将对整个信息技术行业产生巨大影响。问题是如何最好地利用这项技术，并以一种安全、可扩展、易于部署和管理的方式将其与现有的和新的系统集成。这就是 Fixie 的用武之地。”
