推特披露了它的一些源代码，包括它的推荐算法| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/31/twitter-reveals-some-of-its-source-code-including-its-recommendation-algorithm/

正如推特首席执行官埃隆·马斯克(Elon Musk)一再承诺的那样，推特(Twitter)已将部分源代码开放给公众查阅，包括在用户时限内推荐推特的算法。

在 GitHub 上，推特发布了两个 信息库，其中包含了让社交网络运转起来的许多部分的代码，包括推特用来控制用户在“为了你”时间线上看到的推特的机制。在一篇博客文章中，推特将此举描述为“提高透明度的第一步”，同时“防止”推特本身和平台上的人面临风险。

在今天的推特空间会议上，马斯克澄清道:

马斯克说:“我们最初发布的所谓算法将会非常尴尬，人们会发现很多错误，但我们会很快纠正它们。”。“即使你不同意某些东西，至少你会知道为什么它在那里，你没有被秘密操纵……这里，我们渴望的模拟是 Linux 作为一个开源操作系统的伟大例子……理论上，人们可以发现许多 Linux 的漏洞。事实上，社区会识别并修复这些漏洞。”

关于博客文章中关于防范风险的第二点，开源版本不包括支持推特广告推荐的代码，也不包括用于训练推特推荐算法的数据。此外，它们很少包含关于如何检查或实际使用代码的说明——强化了发布严格以开发人员为中心的思想。

“我们排除了任何会损害用户安全和隐私或保护我们平台免受不良行为者影响的代码，包括破坏我们打击儿童性剥削和操纵的努力，”Twitter 写道。这是一个有点混乱的消息，仅在 Twitter 解雇其道德人工智能和信任与安全员工后周，该员工负责其他用户安全相关任务中的内容审核。但该公司仍然坚称，在今天的代码发布中，它“采取了措施来确保用户的安全和隐私得到保护”。

展示 Twitter 推荐管道如何工作的图表。图片来源:推特

Twitter 表示，它正在开发工具来管理来自社区的代码建议，并将更改同步到内部存储库。据推测，这些将在未来的某一天推出——目前还没有任何迹象。

马斯克在 Spaces 会议上说:“我们将寻求建议，不仅仅是关于漏洞，还包括算法应该如何工作。”“这将是一个不断发展的过程。我不认为这是一个无休止的上升趋势…但我们对改善用户体验持开放态度。”

乍一看，该算法相当复杂，但从技术角度来看，这并不令人惊讶。它由多个模型组成，包括一个用于检测“不安全工作”或滥用内容的模型，确定 Twitter 用户与另一个用户互动的可能性，并计算 Twitter 用户的“声誉”。(不清楚“名声”指的是什么，确切地说；高层文档对此并不清楚。)几个神经网络负责对推文进行排名，并推荐要关注的账户，而一个过滤组件隐藏推文，以“支持法律合规性，提高产品质量，增加用户信任，通过使用硬过滤、可见的产品处理和粗粒度的降级来保护收入。”

Twitter 的为你时间线，由推荐算法驱动。图片来源:推特

在一篇工程学博客文章中，Twitter 披露了更多关于推荐管道的信息，它声称每天大约运行 50 亿次:

“我们试图从数亿条推文中提取出最好的 1500 条推文……今天，For You 时间轴平均由 50%(你没有关注的人的推文)和 50%(你关注的人的推文)组成，尽管这可能因用户而异，”Twitter 写道。“对[推文]进行排名是通过一个约 4800 万参数的神经网络来实现的，该网络在推文互动方面不断接受训练，以优化积极参与(例如，喜欢、转发和回复)。”

当然，Twitter 用户看不到全部 1500 条推文。它们是根据内容限制和模型考虑的其他标准和因素进行过滤的，比如推文是否有“负面反馈”，以及它们是否主要来自同一个 Twitter 用户，或者来自被屏蔽或静音的用户。

Gizmodo 指出有一件事似乎没有公开，那就是 Twitter 向用户推送的贵宾名单。本周，平台提供商报道【Twitter 有一份值得注意的用户的轮换名单，包括 YouTuber 野兽先生和每日电讯创始人本·夏皮罗，它通过增加这些“超级用户”的可见性来监控推荐算法的变化。

有更多的证据表明，该算法可能会根据来源对推文进行不同的处理。研究人员简·满春·王指出，Twitter 的算法专门标记了推文作者是否是埃隆·马斯克，还有其他标签表明作者是否是“超级用户”，以及他们是共和党人还是民主党人。

在今天下午的空间会议上，一位 Twitter 工程师说标签仅用于度量。但是马斯克-他说他在今天之前不知道这些标签-说它们不应该在那里。

“绝对不应该把人们分成共和党人和民主党人，这没有意义，”马斯克说。

在发布源代码之前，最近几个月发生了几起涉及 Twitter 推荐算法调整的争议。据平台开发商称，二月份，马斯克呼吁 Twitter 的工程师重新配置算法，这样他的推文将被更广泛地浏览。(Twitter 后来收回了这一改变——至少在某种程度上。)11 月，开始向用户展示更多他们不关注的人的推文——在马斯克收购之前，该平台曾试图采取这一举措，但后来在用户的强烈反对下撤销了。
