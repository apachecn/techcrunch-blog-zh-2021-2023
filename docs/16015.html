T-Mobile 将以 13.5 亿美元收购瑞安·雷诺兹的 Mint Mobile

T-Mobile 正在收购 Mint Mobile，这是一家预算宽松的无线服务提供商，部分归瑞安·雷诺兹所有，交易价值高达 13.5 亿美元，该公司周三宣布。此举表明，T-Mobile 正寻求扩大其预付费服务。

在一篇博客文章中，T-Mobile 表示，已达成协议，以 39%的现金和 61%的股份，以最高 13.5 亿美元收购 Ka'ena Corporation(预付无线品牌 Mint Mobile 和 Ultra Mobile 的母公司)以及无线批发商 Plum。最终购买价格将基于造币厂在收盘前后特定时期的表现。T-Mobile 预计将在今年晚些时候完成这笔交易。

T-Mobile 首席执行官迈克·希沃特(Mike Sievert)在一份声明中表示:“造币厂已经建立了一个非常成功的直接面向消费者的数字业务，它将继续在联合国领先的 5G 网络上为客户提供服务，现在我们很高兴能够利用我们的规模和所有者的经济实力来帮助推动它——以及超移动——走向未来。”从长远来看，我们也将受益于应用营销公式薄荷已成为著名的跨更多的部分 T-Mobile。我们认为，客户真的会用更有竞争力、更具扩张性的薄荷和 Ultra 赢得胜利。"

希沃特在周三发布的视频《T4》中表示，该公司将保留薄荷 15 美元的每月定价。

交易结束后，瑞安·雷诺兹将继续代表造币厂扮演他的创造性角色，并继续出现在商业广告中。造币厂创始人大卫·格利克曼和里兹万·卡西姆将继续留在 T-Mobile 管理这些品牌，这两个品牌通常作为一个独立的业务部门运作。

根据彭博的一份报告，T-Mobile 从一月份开始考虑购买薄荷手机。

Mint Mobile 没有任何实体零售店，因为它的业务完全是在线运营的。作为批发网络共享协议的一部分，电信公司的服务已经由 T-Mobile 提供。就上下文而言，Mint Mobile 是一家移动虚拟网络运营商(MVNO)，这意味着它购买无线网络的批发访问权，而不是拥有和运营无线网络。

T-Mobile 正在收购 Mint Mobile 的销售、营销、数字和服务业务，并计划利用其供应商关系和分销规模来帮助品牌发展，并为更多消费者提供有竞争力的价格和更大的设备库存。该公司还将利用薄荷的数字 D2C 营销，以达到新的客户群和地理位置。

该公司表示，Mint Mobile 和 Ultra Mobile 加入了其当前的预付费服务产品，包括 T-Mobile 的 Metro、T-Mobile 品牌预付费和 T-Mobile 的 Connect。
