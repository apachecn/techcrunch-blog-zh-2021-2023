抖音将向精选的研究人员提供更多关于其平台和审核系统的透明度

原文：https://web.archive.org/web/https://techcrunch.com/2022/07/27/tiktok-select-researchers-more-transparency-about-platform-moderation-system/amp/

图片来源:Nur Photo/Getty Images

抖音今天宣布它将为选定的研究人员提供更多关于其平台和审核系统的透明度。该公司正在开发一个研究 API，以改善对其应用程序内容和活动的公开和匿名数据的访问。抖音表示，研究人员目前没有一种简单的方法来评估内容或在其平台上进行测试。该公司计划在今年晚些时候向选定的研究人员提供 API 访问。

抖音还开发了一个审核系统 API，计划今年秋天在其虚拟透明和问责中心发布。审核系统 API 将为选定的研究人员提供一种方式来评估抖音的内容审核系统，并检查应用程序上现有的内容。此外，研究人员将能够上传他们自己的内容，以了解不同类型的内容是如何被允许、拒绝或传递给版主进行进一步评估的。

抖音还将允许独立专家访问其美国内容咨询委员会 API，以及访问机密信息，如关键字列表，这些信息用于帮助检测和标记潜在的违规内容。抖音说，这些变化将使专家有能力进行更深入的分析。最后，抖音将在其季度社区指导方针执行报告中公布其发现并从其全球平台中移除的秘密影响操作的见解。T15】

抖音首席运营官凡妮莎·帕帕斯(Vanessa Pappas)在博客中写道:“这些举措进展顺利，将在今年接下来的几个月推出。”。“当我们在透明度和问责制方面继续创新时，我们将更新我们的进展。”

今天的声明发布之际，抖音一直试图证明，由于其母公司与中国的联系，它不会对国家安全构成威胁。Gizmodo 今天早些时候发布了一份报告，指出抖音试图淡化与中国的关系。这份报告包括抖音公关人员在面对有关抖音与中国关系的问题时被要求使用的回答。

新的透明度承诺也是在 BuzzFeed News 报道披露抖音在中国的员工可以访问该公司的美国用户数据之后。与此同时，抖音表示，它正在将美国用户的数据转移到存储在美国的甲骨文服务器上。BuzzFeed News 的报道援引了它获得的 80 份抖音内部会议的录音，声称抖音的美国员工反复咨询他们在中国的同事，以了解美国用户数据的流动方式，因为他们没有“自己访问数据的许可或知识”

作为对该报告的回应，许多共和党参议员写信给抖音，表达了对该公司数据访问政策的担忧。抖音对这封信作出回应，承认一些在中国工作的员工可以访问数据,“这些数据受到我们在美国的安全团队的一系列强有力的网络安全控制和授权批准协议的制约”该公司试图安抚参议员，称其正在开展一项名为“德克萨斯州项目”的计划，以加强美国用户的数据安全。
