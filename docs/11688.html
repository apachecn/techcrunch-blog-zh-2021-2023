Gmail 将在假日购物季前增加新的包裹追踪功能

原文：https://web.archive.org/web/https://techcrunch.com/2022/11/02/gmail-to-add-a-new-package-tracking-feature-ahead-of-holiday-shopping-season/amp/

图片来源: 9gifts Kevalee / Getty Images

谷歌今天宣布对 Gmail 进行一个小而有用的更新，用户很快就可以直接从收件箱中跟踪即将到来的包裹递送。该功能的工作原理是查找包含跟踪号码的电子邮件，然后使用这些信息来确定订单的预计交付日期，并在您的收件箱中为您标记。

这意味着当你在 Gmail 中浏览邮件列表时，你不必点击订单确认邮件来查看你的包裹何时到达。相反，这些信息将显示在收件箱中发件人姓名和主题行下方的一个绿色小标签中。您会注意到一个小卡车图标，后面是指示订单状态的文本，后面是交付日期。谷歌表示，这个标签将随着订单的进展而更新，包括“标签已创建”、到货日期或交付日期等信息。

图片来源:谷歌

该功能将为在线购物者节省大量额外步骤，因为通常情况下，消费者必须打开他们的订单确认电子邮件，然后将跟踪信息复制并粘贴到适当的运营商系统、谷歌中，或者点击提供的链接开始跟踪订单。现在，他们需要做的就是查看他们的 Gmail 收件箱。

但是，如果您点击打开订单确认电子邮件，它现在会在顶部包含一个提供更多详细信息的摘要卡，包括一个带有复选标记的时间线，显示当前订单状态(订单已下、已发货或已交付)，以及一个带您进入订单详细信息页面的链接。

谷歌表示，这项新功能将在未来几周内在美国“大多数主要”运输公司推出。预计这一功能将在假日购物季到来之前推出，届时它将非常有用。

为了启用包裹追踪，Gmail 将首先询问用户是否希望在收件箱顶部的弹出窗口中接收追踪更新。用户将根据自己的喜好点击“允许”或“立即”。这也可以在 Gmail 的设置中启用。

图片来源:谷歌

当然，该系统包括扫描你的电子邮件以追踪信息，但这是自动的——人类不会阅读你的电子邮件。尽管如此，一些人可能会认为这是一个潜在的隐私问题，特别是如果 Gmail 选择使用这些数据来帮助通知其电子商务的各种发展和第一方购物功能。

新增加的功能可能会影响受欢迎的第三方包裹跟踪应用程序的采用，包括包裹、路线、AfterShip 和 Shopify 的商店应用程序——尽管后者提供了跟踪以外的更多功能，如浏览和从 Shopify 商家购买的功能。

后来，谷歌表示，它将扩展包裹跟踪功能，在包裹延误时主动更新标签，并将电子邮件放在用户收件箱的顶部，以确保他们知道。
