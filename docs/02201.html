Global Processing Services 增加了 1 亿美元的资金，用于发展其嵌入式金融和 API 支付平台| TechCrunch

原文：https://web.archive.org/web/http://techcrunch.com/2022/01/13/global-processing-services-adds-100m-to-its-coffers-to-grow-its-embedded-finance-and-api-payments-platform/

嵌入式金融仍然是推动金融科技增长的引擎，一批公司建立了核心的银行、支付和其他金融技术，还有一批更大的公司通过 API 利用这种技术来建立面向客户的业务。今天，核心技术方面的一个较大的参与者——全球处理服务——宣布获得 1 亿美元的资金，这不仅表明嵌入式金融作为一项业务仍然很受欢迎，也表明 GPS 在该领域的吸引力。

新加坡投资者淡马锡和美国公司 MissionOG 是最新一批资金的两个投资者，这笔资金是 GPS 早在 2021 年 10 月宣布的 3 亿美元投资的延续，以 4 亿美元结束整轮投资。Advent International 和 Viking Global Investors 共同领导了上一轮投资，这使他们获得了 GPS 的控股权。该公司的其他投资者包括 Visa。与这轮融资的早期阶段一样，总部位于英国伦敦的 GPS 今天没有披露其估值。

一位发言人在回答估值问题时表示:“这不是我们希望被利用的东西，但我们可以说的是，我们继续渴望成为世界上最大的支付科技公司之一，在支付的收购方，如 Adyen、Stripe 和 Checkout.com，以及发行方的 Marqeta，效仿提供商的成功。”“我们相信我们已经建立了一个特殊的平台。世界领先的支付和下一代技术专家的注资将使我们能够实现金融赋权，并使全球更多的金融科技客户踏上独角兽之旅。”

这笔资金将用于继续发展 GPS 的业务，包括一系列金融科技服务，如支付、直接借记和长期订单；虚拟卡；手机钱包；欺诈防范；费用管理；加密货币管理；BNPL 等(这些都是以 GPS Apex 品牌出售的)。

具体来说，该公司希望在欧洲和亚太地区以及中东和非洲的更多新兴市场进一步扩张；它想推出新产品。(值得注意的是，目前组合中没有贷款产品，所以这可能是它探索的一个领域；保险业可能是另一个例子，为特定垂直行业量身定制的解决方案也是如此。)

投资和投资者关注的原因是，全球定位系统及其活跃的空间都出现了大幅增长。一方面，消费者和企业的新银行服务越来越受欢迎(可信度也越来越高)；另一方面，我们已经看到越来越多的非金融科技企业(如电信和零售商)正在利用嵌入式金融的概念，为自己的平台增加新的功能和收入来源。

更普遍地说，随着新冠肺炎疫情统治世界，消费者和企业进行了一次重大转变，转而在网上进行所有的金融活动，即使这种转变减弱，看起来他们也不会完全回到他们的模拟方式。这对整个金融科技行业的风险投资产生了连锁效应。就在昨天，金融科技领域的另一个大玩家，支付初创公司 check out以 400 亿美元的估值筹集了 10 亿美元的巨额资金。

GPS 本身主要关注那些更直接在金融科技领域工作的人，其客户包括 Revolut、Starling、Curve、Zilch 和 Paidy。该公司表示，其服务目前在 48 个国家使用，去年处理了超过 13 亿笔交易，迄今已发行 1.9 亿张卡。

“GPS 是一家创新的技术公司，我们相信他们在全球支付生态系统核心的独特地位使他们能够推动下一代金融服务，”MissionOG 的普通合伙人吉恩·洛克哈特在一份声明中说。“凭借 MissionOG 提供的深厚网络和经验，我们期待成为 Joanne 和整个团队值得信赖和重视的合作伙伴。”值得注意的是，洛克哈特在这次投资中担任了 GPS 的主席。

GPS 首席执行官 Joanne Dewar 补充说:“最新一轮投资的扩大是公司向前迈出的重要一步，也是对我们战略的有力支持。”“我们是一家近年来发展迅速的公司，我们致力于创新和提供单一的可扩展技术平台。随着我们进入地理扩张和技术创新的下一阶段，我们的新合作伙伴为 GPS 带来的专业知识将是无价的。”

更新了公司对估价的回应，并指出 GPS 总部位于伦敦，而不是马恩岛。
