亲爱的饿了，

听说你和你爱的人分开了这么久，我很难过。我真的很钦佩你的力量和韧性。

我们很高兴分享一些期待已久的消息，可能会帮助你和其他人，并提供一些关于获得签证印章或签证印章更新的背景。

许多已经续签签证或更改签证类别的人因为担心无法及时返回美国而推迟了出国旅行。我们仍然建议需要签证的客户尽可能避免国际旅行，因为获得签证的风险太大。这种情况变化缓慢，目前的现状甚至可能在今年 3 月即将到来的 H-1B 签证抽签后继续存在。

尽管美国国务院(DoS)已经实施了一些政策来缩短在美国驻外使领馆获得签证盖章所需的时间，但拖延仍在继续。

然而，今年晚些时候，我们有可能不用出国旅行就能获得签证。

美国签证章更新试点计划

国务院计划在今年晚些时候实施一个试点项目，允许 H 和 L 签证持有者在美国时更新他们护照上的印章。目前，签证持有者必须出国旅行，才能从美国大使馆或领事馆获得护照上的签证箔(通常被称为“签证章”)。

根据彭博社的法律，国务院计划在华盛顿特区设立一个新的领事部门，在今年晚些时候开始处理 H 和 L 签证的续签。许多人多年来一直倡导这一点。在 2004 年国家安全法开始要求签证申请人收集生物特征数据之前，美国国内的签证部门一直处理许多类别的签证延期，包括 E、H、L 和 O 签证。

从那时起，除了某些外交和其他签证申请人，国务院要求所有在美国通过身份变更程序的签证申请人在美国驻外大使馆或领事馆获得签证印章。

1B 签证面谈的等待时间

DoS 有一个签证预约等待时间页面，可以让你查看美国大使馆和领事馆的等待时间。等待时间会根据你在印度的位置而有所不同，但它们一直在变化。
