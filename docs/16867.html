不，这不是愚人节玩笑:OpenAI 已经开始对其在意大利的生成性 AI 聊天机器人 ChatGPT 进行地理封锁。

在此之前，T4 当地数据保护机构周五发布命令，要求其必须停止为 ChatGPT 服务处理意大利人的数据。

在一份向试图访问 ChatGPT 的意大利 IP 地址用户在线发布的声明中，OpenAI 写道，它“遗憾地”通知用户，应数据保护局的“请求”，它已经禁止了意大利用户的访问，这被称为 Garante 。

它还表示，将向意大利所有上个月购买 ChatGPT Plus 订阅服务的用户退款，并指出，它将“暂时暂停”订阅续订，以便在服务暂停期间用户不会被收费。

OpenAI 似乎在这一点上应用了一个简单的 geoblock 这意味着使用 VPN 切换到非意大利 IP 地址为该块提供了一个简单的解决方法。虽然如果 ChatGPT 帐户最初是在意大利注册的，它可能无法再访问，想要绕过封锁的用户可能必须使用非意大利 IP 地址创建一个新帐户。

OpenAI 对试图从意大利 IP 地址访问 ChatGPT 的用户的声明。图片鸣谢:截图/Natasha Lomas/TechCrunch

周五the ga rante 宣布已经对 ChatGPT 涉嫌违反欧盟通用数据保护条例(GDPR)展开调查——称其担心 OpenAI 非法处理意大利人的数据。

OpenAI 似乎没有告知任何人它发现了谁的在线数据并用来训练这项技术，例如通过从互联网论坛收集信息。它也没有完全公开它正在处理的数据——当然不是它的最新型号 GPT-4。虽然它使用的训练数据可能是公开的(从发布在网上的意义上来说)，但 GDPR 仍然包含透明原则——这表明用户和数据收集者都应该被告知。

在昨天的声明中，Garante 还指出缺乏任何系统来防止未成年人接触技术，举了一个儿童安全的旗帜——例如，没有年龄验证功能来防止不适当的接触。

此外，监管机构对聊天机器人提供的信息的准确性表示担忧。

众所周知，ChatGPT 和其他生成性人工智能聊天机器人有时会产生关于指定个人的错误信息——人工智能制造商称这种缺陷为“幻觉”。这在欧盟看来是有问题的，因为 GDPR 为个人提供了一套对其信息的权利——包括纠正错误信息的权利。而且，目前还不清楚 OpenAI 是否有一个用户可以要求聊天机器人停止撒谎的系统。

这家总部位于旧金山的公司仍未回应我们对 Garante 调查的置评请求。但在其对意大利 geoblocked 用户的公开声明中，它声称:“我们致力于保护人们的隐私，我们相信我们提供的 ChatGPT 符合 GDPR 和其他隐私法律。”

“我们将与 Garante 合作，目标是尽快恢复您的访问，”它还写道，“你们中的许多人告诉我们，你发现 ChatGPT 对日常任务很有帮助，我们期待着它很快再次可用。”

尽管在声明的最后给出了乐观的基调，但尚不清楚 OpenAI 如何解决 ga rante——提出的合规问题，鉴于 GDPR 问题的广泛范围，它开始了更深入的调查。

泛欧盟法规要求通过设计和默认来保护数据——这意味着以隐私为中心的流程和原则应该从一开始就嵌入到处理人们数据的系统中(也就是说，与抓取数据并在后来请求原谅的方法相反)。

与此同时，对确认违反 GDPR 的罚款可高达数据处理器全球年营业额的 4%(或€2000 万，以较大者为准)。

此外，由于 OpenAI 在欧盟没有主要机构，欧盟的任何数据保护机构都有权监管 chat GPT——这意味着所有其他欧盟成员国的机构都可以选择介入并进行调查——并对他们发现的任何违规行为处以罚款(在相对较短的时间内，因为每个机构都只能在自己的领域内采取行动)。因此，它面临着最高级别的 GDPR 曝光，没有准备好玩其他科技巨头用来推迟欧洲隐私执法的论坛购物游戏。
