为什么越来越多的 SaaS 公司转向基于使用量的定价

原文：https://web.archive.org/web/https://techcrunch.com/2021/11/04/more-saas-companies-are-shifting-to-usage-based-pricing/

为什么越来越多的 SaaS 公司转向基于使用量的定价

采用产品导向增长和 UBP 的创业公司比同行表现更好

基于使用的定价 (UBP)正在成为主流吗？如果你考虑一下总部位于波士顿的风险投资公司 OpenView 的年度财务和运营基准调查的结果，或许是这样的。在参与调查的近 600 家 SaaS 公司中，45%表示他们正在使用这种灵活的定价模式，高于 2020 年的 34%。

该调查还研究了采用灵活定价模式的公司与同行相比表现如何，以及它如何对它们产生更广泛的影响。由于它没有回避提及挑战，我们觉得它对那些可能仍在观望的创业者来说是一个相关的读物。

为了更深入地了解调查结果，我们采访了 OpenView 的运营合作伙伴凯尔·波亚尔，他在过去支持基于使用的定价模式，并与合作伙伴 Sanjiv Kalevar 共同撰写了该公司的 2021 年基于使用的定价报告。

“基于使用的定价是一个令人难以置信的信息，即你支持你的产品，你真的相信你的客户会成功。”

主要观点如下，但在我们开始之前，这里有一个关于定义的说明:该报告对基于使用的定价采用的定义包括 Twilio 等公司，其定价几乎完全是按需付费，以及 Zapier 等提供基于使用的订阅层的公司。

换句话说，这份报告的共同作者并没有划定是否有订阅的界限；相反，他们会考虑定价是否与产品消费行为挂钩，而不是仅仅基于座位的定价，在这种情况下，公司还会根据客户规模、功能、服务或其他因素收费。

为什么座位是空的

据 OpenView 称，推动这种转变的一个因素是，对“座位”收费不再像过去那样有意义。Poyar 指出，客户获得的价值很少与有多少人登录直接相关，特别是现在越来越多的初创公司提供围绕自动化、人工智能或 API 构建的解决方案。

“事实上，”他说，“它甚至可能是负相关的:当人工智能能够自动化任务时，解决方案越成功，需要登录的人就越少。因此，座位只是一种过时的收费方式，不允许公司传达价值或投资于能增加更多价值的功能。”虽然 Poyar 承认并不是每家公司都这样做，但他指出，自动化、API 或 AI 往往在今天上市的公司的成功中发挥着重要作用。

这可能解释了思维的另一个转变:许多上市公司“正在呼唤基于使用的模式，并使之成为他们的 S-1 或投资者材料的焦点，”波亚尔说。他补充说，“过去人们担心投资者会因为基于使用的模式而惩罚他们，因为他们担心这不是经常性收入，也不可预测。[……]现在，基于使用的收入模式被视为一种竞争优势和长期增长的驱动力。”
