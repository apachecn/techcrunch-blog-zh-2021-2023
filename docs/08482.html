生成商业图像的人工智能引发棘手的法律问题

原文：https://web.archive.org/web/https://techcrunch.com/2022/07/22/commercial-image-generating-ai-raises-all-sorts-of-thorny-legal-issues/

本周，OpenAI 授予其图像生成人工智能系统 DALL-E 2 的用户将他们的世代用于商业项目的权利，如儿童书籍的插图和时事通讯的艺术。考虑到 OpenAI 自己的商业目标，这一举动是有意义的——政策变化恰逢该公司推出 DALL-E 2 的付费计划。但它提出了关于 DALL-E 2 这样的人工智能的法律含义的问题，这些人工智能是在网络上的公共图像上训练的，它们可能会侵犯现有的版权。

DALL-E 2 对从互联网上搜集的大约 6.5 亿个图像-文本对进行了“训练”，从该数据集学习图像和用于描述它们的词语之间的关系。但是，尽管 OpenAI 过滤掉了特定内容的图像(例如色情和副本，并在 API 层面实施了额外的过滤器，例如针对知名公众人物，但该公司承认，该系统有时可以创建包括商标标识或字符的作品。参见:

“OpenAI 将评估不同的方法来处理潜在的版权和商标问题，其中可能包括允许这种生成作为‘公平使用’或类似概念的一部分，过滤特定类型的内容，并直接与版权[和]商标所有者就这些问题进行合作，”该公司在周三 DALL-E 2 测试版发布前发表的一份分析中写道。

这不仅仅是 DALL-E 2 的问题。随着人工智能社区创建 DALL-E 2 及其前身 DALL-E 的开源实现，免费的和付费的服务都在不太仔细过滤的数据集上训练的模型上推出。其中一款名为 Pixelz.ai 的应用本周推出了一款由定制 DALL-E 模型驱动的图像生成应用，可以非常轻松地创建显示《银河护卫队》和《冰雪奇缘》等电影中各种神奇宝贝和迪士尼角色的照片。

当联系 Pixelz.ai 团队进行评论时，他们告诉 TechCrunch，他们已经过滤了模型的训练数据，以查找亵渎、仇恨言论和“非法活动”，并阻止用户在生成时请求这些类型的图像。该公司还表示，它计划增加一个报告功能，允许人们向人类版主团队提交违反服务条款的图像。但在涉及知识产权(IP)的地方，Pixelz.ai 让用户在使用或分发他们生成的图像时行使“责任”——不管是不是灰色地带。

“我们不鼓励在数据集和我们平台的服务条款中侵犯版权，”该团队告诉 TechCrunch。“也就是说，我们提供一个开放的文本输入，人们总是会找到创造性的方法来滥用平台。”

由 Pixelz.ai 的系统生成的迪士尼/漫威银河护卫队火箭浣熊的图像。图片来源: Pixelz.ai

MBHB 律师事务所的创始合伙人、知识产权法专家 Bradley J. Hulbert 认为，从版权角度来看，图像生成系统在几个方面存在问题。他指出，艺术作品“明显源自”一个“受保护的作品”——即一个受版权保护的角色——通常会被法院认定为侵权，即使添加了额外的元素。(想象一下迪士尼公主走过纽约一个布满砂砾的街区的画面。)为了免受版权索赔，作品必须是“变革性的”——换句话说，改变到知识产权无法识别的程度。

“如果在 DALL-E 2 生成的图像中可以认出迪士尼公主，我们可以有把握地假设华特·迪士尼公司可能会断言 DALL-E 2 图像是一种衍生作品，侵犯了其对迪士尼公主肖像的版权，”Hulbert 通过电子邮件告诉 TechCrunch。“在确定一个复制品是否构成‘合理使用’时，实质性的转变也是一个考虑因素。”但是，再一次，在某种程度上，迪士尼公主在后期作品中是可识别的，假设迪士尼将断言后期作品是侵犯版权的。"

当然，知识产权持有者和涉嫌侵权者之间的斗争并不新鲜，互联网只是起到了催化剂的作用。2020 年，华纳兄弟娱乐公司拥有哈利波特宇宙的电影描绘权，从包括 Instagram 和 Etsy 在内的社交媒体平台上删除了某些粉丝艺术。一年前，迪士尼和卢卡斯影业请求 Giphy 删除《尤达宝宝》的 gif 图片

但生成图像的人工智能降低了准入门槛，有可能极大地扩大这个问题的规模。大公司的困境不太可能获得同情(他们也不应该获得同情)，他们保护知识产权的努力在公众舆论的法庭上常常适得其反。另一方面，人工智能生成的艺术作品侵犯了独立艺术家的角色，可能会威胁到生计。

围绕 DALL-E 2 等系统的另一个棘手的法律问题与它们的训练数据集的内容有关。像 OpenAI 这样的公司使用受版权保护的图像和艺术作品来开发他们的系统是否违反了知识产权法？这是一个已经在 Copilot 环境中提出的问题，Copilot 是由 OpenAI 和 GitHub 联合开发的商业代码生成工具。但与 Copilot 不同的是，像 DALL-E 2 这样的系统从无数的公共网站上获取图片，Copilot 接受的是 GitHub 根据其服务条款有权使用的代码。

正如 Dave Gershgorn 在最近为 The Verge 撰写的一篇专题文章中指出的，在美国没有直接的法律先例支持公开的训练数据为合理使用。

一个潜在的相关案例涉及一家名为 5D 规划师的立陶宛公司。2020 年，公司起诉 Meta(当时的脸书),称其从规划师 5D 的软件中窃取了数千个文件，这些文件通过与普林斯顿大学的合作提供给 Meta 2019 年计算机视觉研究人员场景理解和建模挑战的参赛者。策划人 5D 声称，普林斯顿、Meta 和 Oculus(Meta 专注于虚拟现实的硬件和软件部门)本可以从从中获得的训练数据中受益。

该案件在 2023 年 3 月之前不会在进行审判。但去年 4 月，负责此案的美国地区法官驳回了当时的脸书大学和普林斯顿大学提出的驳回 Planner 5G 指控的动议。

不出所料，权利持有人不会被合理使用的论点所左右。Getty Images 的发言人在一篇文章中告诉 IEEE Spectrum,关于“像 DALL-E 2 这样的模型被训练的图像和图像中的人、地方和物体的权利”有“大问题”需要回答插画协会首席执行官雷切尔·希尔(Rachel Hill)也在文章中被引用，她提出了训练数据中图像的补偿问题。

Hulbert 认为，法官不太可能将训练数据集中版权作品的副本视为合理使用——至少在 DALL-E 2 这样的商业系统中是如此。他认为，知识产权持有者可能会在某个时候追踪 OpenAI 这样的公司，并要求它们许可用于训练其系统的图像。

“这些复制品……侵犯了原作者的版权。而侵权者须向版权拥有人作出赔偿。"。“[如果] DALL-E(或 DALL-E 2)及其合作伙伴制作了受保护作品的复制品，而该复制品既未得到版权所有者的许可，也未得到合理使用，则该复制品构成侵犯版权。”

有趣的是，英国正在探索立法，取消当前的要求，即通过文本和数据挖掘训练的系统，如 DALL-E 2，只能用于非商业目的。虽然在拟议的制度下，版权所有者仍然可以通过将他们的作品置于付费墙之后来要求支付，但这将使英国的政策成为世界上最自由的政策之一。

鉴于美国知识产权持有者的游说力量，美国似乎不太可能效仿。这个问题似乎可能会在未来的诉讼中出现。但是时间会证明一切。
