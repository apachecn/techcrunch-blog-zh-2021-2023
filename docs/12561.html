谷歌云与印度初创公司 SuperGaming 合作，为开发者提供游戏引擎

原文：https://web.archive.org/web/https://techcrunch.com/2022/11/23/google-cloud-supergaming-partnership/amp/

图像点数:超级游戏

谷歌云已经与 SuperGaming 合作，向全球开发者提供这家印度游戏初创公司的专有游戏引擎 SuperPlatform，这是这家安卓制造商最近一系列将重点扩展到游戏行业的举措中的最新举措。

这家搜索巨头的云部门周四表示，作为合作伙伴关系的一部分，它将为这家总部位于浦那的初创公司提供游戏引擎，以帮助全球开发者管理他们的直播运营、配对、玩家进度和数据、分析、服务器扩展和商品销售。这些工具旨在帮助公司维护、优化和扩展他们的游戏。

新贵 SuperGaming 在自己的游戏中使用其游戏引擎，并在移动设备上使用官方的 PAC-MAN 游戏，其移动游戏如 MaskGun，Silly Royale 和 Tower Conquest 已获得数百万次下载。

SuperGaming 最初开发 SuperPlatform 为自己的游戏提供动力，并于 2019 年开始向其他开发商授权该服务。

形象积分:超级博彩

这两家公司并不陌生。SuperGaming 最初依赖 AWS 来满足其云需求，但几年前在看到包括“大量节省”在内的优势后，转向了谷歌云，SuperGaming 联合创始人兼首席执行官罗比·约翰在接受 TechCrunch 采访时表示。

约翰说，这一举措推动了谷歌云作为独立软件供应商为开发者提供平台。“我很高兴将我们的平台引入谷歌云，谷歌云已经为我们 70%的顶级客户提供支持，”他补充道。

他说，开发者将继续选择在 AWS 和 Azure 上使用 SuperPlatform，尽管作为合作的结果，谷歌云将是 SuperGaming 的首选。

在谈到潜在的合作伙伴关系之前，约翰说，SuperGaming 一直在与谷歌云工程师密切合作，将云平台用于即将到来的皇家战役游戏 Indus。他说，双方的团队交换了见解，这有助于非常有机地建立伙伴关系。

“这种伙伴关系不仅仅是说，好吧，这里有计算机和基础设施以及所有其他东西。谷歌云印度公司董事总经理比克拉姆·辛格·贝迪在联合对话中说:“这是在说，我们如何才能齐心协力，实现成功的商业目标。”。

两人没有透露交易的财务条款。

谷歌云的竞争对手 AWS 和 Azure 确实为游戏开发者提供了原生 LiveOps 解决方案，让他们以服务的形式运行游戏，并获得实时遥测。然而，谷歌云似乎利用了 SuperGaming 的专业知识——以及它的平台——来带来一些区别。

“它总是与开发者有关，或者与玩家有关。这种伙伴关系使我们能够对双方都施加影响，”贝迪说。

SuperGaming 的投资者包括总部位于得克萨斯州的 Skycatcher、东京的晓娱乐技术基金、总部位于柯克兰的 1UpVentures 和蚂蚁集团支持的 BAce Capital，迄今为止已经筹集了 680 万美元，其中 550 万美元通过去年的 A 轮融资注入。

这家初创公司还推出了 TowerConquest:元宇宙版作为其免费的 web3 游戏，并表示该游戏也将在谷歌云上运行——与现有的游戏和即将推出的 Indus 一起。
