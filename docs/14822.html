在 Twitter 终止其 Tweetbot 应用程序| TechCrunch 后，Tapbots 推出了新的乳齿象客户端 Ivory

原文：https://web.archive.org/web/https://techcrunch.com/2023/01/24/tapbots-launches-a-new-mastodon-client-ivory-after-twitter-kills-its-tweetbot-app/amp/

图片来源: Tapbots

Tapbots ，广受欢迎的第三方 Twitter 应用 Tweetbot 的制造商，最近被通过 Twitter 的 API 变化杀死，发布了它的下一个新产品。希望填补 Tweetbot 留下的空白，该公司现在正在将其预期的乳齿象客户端应用象牙作为早期访问版本在 App Store 上提供。

该公司告诉我们,“早期访问”标签是 Tapbots 在其发布时的副标题，表明在其首次发布时仍将缺少一些功能。然而，通过在 App Store 上公开发布，Tapbots 能够在填满其测试版本的有限数量的试飞位置后，将象牙放入更多人的手中。

Tapbots 创始人保罗·阿达德告诉 TechCrunch，Ivory 的应用商店发布昨天刚刚获得批准，但在公布发布之前，该公司还有一些关于订阅的技术问题需要解决。

对于长期使用 Tweetbot 的用户来说，象牙将提供一种熟悉的体验。但是，该公司不再是 Twitter 网络的客户，而是拥抱了前景广阔的开源平台乳齿象。虽然不像 Twitter 那么容易使用或理解，但在埃隆马斯克收购 Twitter 后的几个月里，乳齿象已经获得了关注。

随着 Twitter 的新主人实施了有争议的改变，网站的性能不时下降，一些人开始寻找一个新家来发布简短的更新，就像他们在 Twitter 上做的那样。这导致了对新应用程序和其他 推特 替代品的兴趣。截至去年 12 月，Mastodon 的月活跃用户已经增长到 250 万，分布在大约 8600 台不同的服务器上，用 Mastodon 的行话来说就是“实例”

然而，一些乳齿象新人不喜欢官方的原生移动应用程序——特别是在使用了更快、更精致的 Twitter 原生移动应用程序或来自第三方的应用程序，如 Tweetbot 之后。这有助于吸引人们对 Ivory 和其他由前 Twitter 第三方应用开发商(包括 Mammoth)开发的等新兴应用的兴趣。

Tapbots 表示，Ivory 的功能集利用了 Tapbots 在为 Twitter 平台构建其屡获殊荣的 Tweetbot 客户端方面十多年的经验。

在发布时，它展示了几十个功能，从支持基线功能到智能的铃铛和口哨，比如能够为应用程序设置主题或改变图标。

该应用程序支持多个帐户，并允许您查看您的本地和联合时间表，趋势帖子，帖子统计数据，通知等。它还实现了 Twitter 上没有的乳齿象特有的选项，如在帖子中添加内容警告的功能，以及更常见的功能，如发布 gif 和投票的功能。

图片来源: Tapbots

还有其他旨在吸引高级用户的贴心设计，如标签跟踪、支持正则表达式的静音过滤器和时间轴过滤器，让你显示或隐藏符合特定标准的帖子。这对乳齿象的老用户也有吸引力，他们可能希望屏蔽和避免乳齿象新人分享的一些帖子，这些新人将 Twitter 的文化带到了平台上，导致他们的时间线中没有内容警告的不想要的帖子。

Ivory 还允许用户以其他方式个性化应用程序。除了过滤器、主题和图标选择之外，用户还可以配置屏幕底部的导航按钮，向他们显示他们想要看到的内容，方法是按下按钮，在应用程序不同区域的标签之间切换，如书签、收藏夹、统计数据、个人资料、通知、列表、搜索等。

该应用程序还承诺快速的性能，流畅的滚动和有吸引力的用户界面和声音设计。

对于首次发布的版本，Ivory 已经是一个相当健壮的客户端了。然而，该公司表示，该应用程序还有更多的东西要添加，包括编辑个人资料或编辑帖子的工具(拿着，编辑推特按钮！)，以及其他用于阅读媒体的 alt/description 文本的功能，改进的 hashtags，对自定义实例表情符号的支持，具有更好过滤功能的改进的通知选项卡，改进的导航栏和抑制重复帖子的功能。

该应用将像 Tweetbot 一样，通过应用内订阅的方式产生收入。App Store 列出了每月 1.99 美元或每年 14.99 美元的价格。每年 24.99 美元的“高级”订阅也是可用的。该公司表示，后者包括 Pro 订阅中的所有功能，稍后还将包括一些额外功能。Haddad 表示，增加 Premier tier 主要是因为人们要求支付更多的钱来帮助公司，但它计划在未来为这些人提供一些额外的福利，可能包括额外的图标或壁纸等。

这些订阅目前在 Ivory 应用程序中可用，并启用一些高级用户功能，如配置您想要接收的通知等。用户也可以免费“演示”这款应用，无需订阅。

由于 Twitter 的 API 变化导致 Tweetbot 的收入降至零，只有消费者愿意采用乳齿象并订阅替代客户端，像象牙这样的应用程序才有未来。由于乳齿象可利用的全球用户基础比 Twitter 小得多，象牙的收入潜力上限也比该公司的 Tweetbot 应用程序小得多。(如果你感到怀旧，访问它的纪念页面。)

我们已经通过它的 TestFlight 版本对 Ivory 进行了几周的测试，发现它提供了高性能和丰富的用户乳齿象体验，在许多方面都优于官方应用程序，这使它很容易成为在 iOS 设备上使用乳齿象的首选之一，即使在其早期也是如此。
