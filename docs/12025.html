OpenAI 在人工智能笔记应用 Mem 中领投 2350 万美元

原文：https://web.archive.org/web/https://techcrunch.com/2022/11/10/ai-powered-note-taking-app-mem-raises-23-5m-openai/

去年，OpenAI 宣布了 OpenAI 创业基金(OpenAI Startup Fund ),通过这一部分，它和包括微软在内的合作伙伴正在投资早期人工智能公司，解决重大问题。从那以后，接受该基金注资的公司都保持沉默。但今天，OpenAI 创业基金透露，它领导了对 Mem 的 2350 万美元投资，这是一款专注于工作的应用程序，利用人工智能自动整理笔记。

这笔投资对 Mem 的估值为 1.1 亿美元，使这家初创公司的总融资额达到 2900 万美元。

Mem 由凯文·穆迪(Kevin Moody)和丹尼斯·徐(Dennis Xu)共同创立，用穆迪和徐的话说，它通过强调“轻量级组织”来区别于传统的笔记应用。工作流程围绕着搜索和按时间顺序排列的时间线，允许用户添加主题标签，标记其他用户，并在笔记中添加重复提醒。

Mem 用户可以在任何地方使用短信、消息应用和平台的移动客户端捕捉快速笔记、发送链接和保存图像。协作功能允许团队共享、编辑和评论笔记，并直接将其附加到共享日历上，以便更快地参考。

Mem 的搜索体验使用人工智能来搜索笔记，旨在了解在特定时刻哪些笔记可能与特定的人最相关。穆迪和徐表示，该平台旨在增强知识工作者的典型职责，如通读信息页面，提取与特定问题相关的片段，并将信息转化为答案或报告。

Mem 点击 AI，实时整理笔记。图片来源: Mem

毫无疑问，寻求知识的任务是耗时的。根据 Gartner 的调查，专业人士花费 50%的工作时间搜索信息，平均花费 18 分钟查找文件(尽管这些指标的准确性多年来一直受到质疑)。一位消息人士估计，文档混乱每年会导致企业每个员工损失 3900 美元的生产力，如果这项技术像广告宣传的那样有效，那么 Mem 就是一个有吸引力的提议。

“我们从与我们交谈的组织那里听到的第一件事是，他们希望能够将他们大量的专有知识与生成性人工智能模型结合起来，以支持从开展研究到写作到销售等各种用例，”穆迪和徐在电子邮件采访中告诉 TechCrunch。“Mem 的神奇之处在于，我们将您自己的私有和专有数据与最先进的生成语言模型结合在一起，以解锁真正个性化的事实输出。我们将个人、团队和组织层面的知识来源结合起来，全面提升绩效。”

Mem 最近推出了 Mem It for Twitter，允许用户保存线程，获取人工智能生成的内容摘要，并查看类似推文的建议。它还继续完善 Mem 的内置工作助手 Mem X，增加智能写作和智能编辑等新功能，利用人工智能根据提示生成文本，汇总文件，为文档生成标题，并让用户使用自然语言命令编辑或格式化文本。

Mem 的人工智能写作工具，即将推出预览版。图片来源:记忆

穆迪和徐说，在可预见的未来，我们的计划是通过 OpenAI 启动基金，在 OpenAI 的支持下，越来越多地融入这种人工智能驱动的体验。OpenAI 创业基金的参与者除了获得资本外，还可以从微软获得新的 OpenAI 系统和 Azure 资源。

“OpenAI 显然正在引领我们正在乘坐的技术革命浪潮，”穆迪和徐说。“这使得 OpenAI 创业基金成为我们正在建设的理想合作伙伴——因为他们带来了技术专长和战略指导。”

OpenAI 首席运营官·布拉德·莱特卡普(Brad Lightcap)也管理着 OpenAI 创业基金，他在一份电子邮件声明中补充说:“Mem 使用强大的人工智能，通过消除组织和访问信息的单调乏味和繁重工作，让知识工作者更有效率，最终让人们专注于他们工作中重要的部分。他们的愿景与我们在 OpenAI Startup Fund 的目标完全一致，即加速公司使用人工智能来提高生产力，更广泛地说，提高人类潜力。”

Mem 与许多寻求解决相同的知识发现和笔记组织挑战的公司竞争。在企业搜索领域，有一家名为 Glean 的公司，它最近在一轮风险投资中筹集了 1 亿美元。在知识管理方面，Atlassian 的类似维基的协作工作空间融合和概念，在 2020 年被估价为100 亿美元，仍然占据主导地位。

但穆迪和徐认为，16 名员工的 Mem 具有优势，因为它是“自组织的”，表面上减少了人工管理和劳动。虽然他们拒绝透露 Mem 的收入或任何主要客户的名称，但他们断言 Mem 是成功的，因为它是人工智能驱动的技术。

“我们对自己组织和生成知识管理的独特方法充满信心。…我们的个性化机器学习模型不仅帮助知识工作者自动保持组织，而且不仅仅是帮助寻找东西——我们实际上帮助人们做他们的工作，”穆迪和徐说。“向远程工作的转变使得有效、异步的知识共享变得比以往任何时候都更加重要，而市场放缓也促使公司关注效率。我们的人工智能辅助知识工作节省了人们的时间，大型语言模型的快速改进给了我们进一步的顺风。”
