在 ChatGPT 狂潮中，中国涌现出一百名粉丝

原文：https://web.archive.org/web/https://techcrunch.com/2023/02/09/chatgpt-china-openai/

美国的技术突破总能激发中国的挑战者、追随者和机会主义者。轮到 ChatGPT 抓住世界上最大的互联网人口的想象力了。在微信上，ChatGPT 的“趋势指数”，一个衡量关键词在社交网络上受欢迎程度的指标，在过去 30 天内增长了 155 倍。观察 OpenAI 强大的语言模型如何在该国的科技巨头、创业公司和普通人之间引发巨大兴趣是很有趣的，尤其是因为它提供了一个了解两个超级大国之间人工智能竞赛状态的镜头。

与许多其他主要的西方互联网平台不同，ChatGPT 网站在中国还没有被屏蔽。但加载时间很慢，用户不能用中文电话号码注册，这给使用增加了额外的障碍。

正如我们之前写的，出于从语言到政治的各种原因，中国想要自己的智能自然语言系统。创建本土 ChatGPT 的重担不可避免地落在了中国的科技巨头身上。在推出了一个稳定的扩散式艺术生成器之后，百度最近宣布，它正在开发 Ernie bot，计划于 3 月份推出。电子商务巨头阿里巴巴随后 透露 其对 ChatGPT 的回答正在进行内部测试。就在我写这篇文章的时候，微信的母公司和视频游戏巨头腾讯表示，它也在进行“相关研究”。

百度和阿里巴巴的股票因其聊天机器人的声明而短暂上涨。但人们知道，当投资者开始聚集在远不能开发强大语言模型的较小人工智能玩家周围时，泡沫正在形成。中国音频到文本服务提供商 iFlytek 的股价今年上涨了 17%，尽管其业务线中没有 ChatGPT 风格的产品。

默默无闻的公司也搭上了人工智能狂潮的顺风车。上周，微信上开始出现大量山寨 ChatGPT，向发送提示的用户收费。社交网络似乎已经注意到了这一点。在撰写本文时，它已经隐藏了一些企业账户(比如脸书网页)，这些账户的名字具有误导性，在搜索结果中包含“ChatGPT ”,大概是为了保护用户免受这些可疑模仿者的攻击。

出于好奇，我测试了一个名为“ChatGPT”的账户。不幸的是，我不得不花 4.5 美元问了 20 多个问题，但我让它思考是谁。综上所述，本次私酒 ChatGPT " 与 OpenAI 无关；它的开发商是“一家名为 Cofoundry Limited 的英国公司；“它的创始人是一系列看似人工智能生成的白人名字；ChatGPT 由“GPT-2 技术有限公司开发”

不出所料，ChatGPT 山寨版不会回答任何在中国被视为政治敏感的问题。它确实以令人满意的流畅度交谈，这表明它可能使用了 OpenAI API 并增加了一个审查层。

一些狂热分子使用 OpenAI 的动机并不明显与金钱利益相关。一名开发者开发了一款工具，可以将聊天记录转化为易于理解的要点，当一个人醒来发现微信上有 100 条未读群发消息时就派上用场了。

我们之前写过审查制度可能是中国自然语言系统未来面临的最大挑战之一。如果百度和阿里巴巴只在微博、百度和微信数据上训练他们的模型，这些数据都有审查机制，他们的机器人就会知道什么是禁区。但是据报道，百度从中国精心设计的审查机制“防火长城”内外收集数据。

这完全有道理，因为有大量的英语源数据，如学术研究。然后，百度深度学习科学家的工作是确保其聊天机器人能够感知审查。Ernie Bot 和它的本地竞争对手是否足够机敏，能够发现用于绕过审查的不断变化的双关语和表情符号？

状态控制不仅决定了人工智能应用的规则，也决定了有多少资源用于特定类型的人工智能研究。这一点在人工智能投资者内森·贝纳奇和伊恩·霍加斯的《2022 年人工智能状态报告的 中显而易见。该报告显示，尽管中国在人工智能论文发表数量上迅速赶上美国，但其人工智能研究更倾向于与监控相关的任务，如“自主、物体检测、跟踪、场景理解、动作和说话人识别”。

展望未来，资源丰富的科技巨头，如百度、阿里巴巴和腾讯，无疑是最接近为中国创造一个有能力的 ChatGPT 的。从长远来看，随着美国切断供应，中国是否有能力获得用于训练大型人工智能模型的先进芯片组仍然是个问题。

我们也不要忘记小冰，这个曾经被大肆宣传的聊天机器人来自微软中国，于 2020 年从分离出来，专注于“本地化创新”当时，小冰表示将继续从微软获得技术许可。如果该协议保持不变，鉴于微软与 OpenAI 的密切关系，这是否意味着小冰可能会访问 GPT 的功能？这听起来可能有些牵强，但观察人工智能技术如何跨越国界肯定是值得的。
