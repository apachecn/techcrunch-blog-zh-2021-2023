微软的交易不是独家的，视频即将到来，更多来自 OpenAI 首席执行官 Sam Altman | TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2023/01/17/that-microsoft-deal-isnt-exclusive-video-is-coming-and-more-from-openai-ceo-sam-altman/

在上周的一次严格风险投资活动中， OpenAI 联合创始人兼首席执行官山姆·奥尔特曼(Sam Altman)接受了本刊编辑的广泛采访，回答了关于他在最雄心勃勃的个人投资以及 OpenAI 未来的问题。

有许多事情要讨论。自发布 ChatGPT 以来的两个月里，这家现已成立八年的机构主导了全国的讨论，chat GPT 是一个像人一样回答问题的聊天机器人。OpenAI 的产品不仅仅让用户感到惊讶；据报道，该公司正在进行谈判，以监督以 290 亿美元的估值向新投资者出售现有股票，尽管其相对的名义收入。

奥特曼拒绝谈论 OpenAI 目前的业务往来，当我们坐下来讨论一个相关问题时，他发出了警告。然而，他确实透露了一点公司未来的计划。首先，除了 ChatGPT 和该机构流行的数字艺术生成器 DALL-E 之外，Altman 证实了一个视频模型也将出现，尽管他说他“不想对何时出现做出有能力的预测”，并补充说“可能很快就会出现；这是一个合法的研究项目。这可能需要一段时间。”

奥特曼证实，OpenAI 与微软不断发展的合作伙伴关系——微软于 2019 年首次投资 OpenAI ，今天早些时候证实，它计划将 ChatGPT 等人工智能工具纳入其所有产品——并不是一项排他性协议。

此外，奥特曼证实，除了向其他公司授权其技术之外，OpenAI 还可以建立自己的软件产品和服务。对于那些想知道 OpenAI 是否有一天会通过自己的搜索引擎直接与谷歌竞争的行业观察人士来说，这是值得注意的。(当被问及这种情况时，奥特曼说:“每当有人谈论一项技术将成为其他一些巨头公司的末日时，这通常是错误的。人们忘记了他们可以在这里采取反制措施，他们非常聪明，非常有能力。”)

至于 OpenAI 计划何时发布第四版 GPT，ChatGPT 所基于的复杂语言模型，奥特曼只是说，这个备受期待的产品将“在我们有信心可以安全和负责任地(发布)它的某个时候推出。”他还试图缓和对 GPT-4 的期望，称“我们没有一个真正的 AGI”，这意味着人工通用智能，或一种具有自身涌现智能的技术，而不是 OpenAI 当前的深度学习模型，后者通过试错来解决问题和识别模式。

“我认为(AGI)是人们对我们的期望”，GPT-4“会让人们失望”，他说。

与此同时，当被问及时，奥特曼预计什么时候会看到人工通用智能，他假定这比人们想象的要近，但向“AGI”的转变不会像一些人预期的那样突然。“我们离 AGI 越近，我就越难回答，因为我认为这将比人们想象的更加模糊，更加是一个渐进的过渡，”他说。

很自然，在我们结束之前，我们花时间讨论了安全问题，包括社会是否有足够的护栏来保护 OpenAI 已经向世界发布的技术。许多批评家认为我们没有，包括忧心忡忡的教育者，他们越来越多地阻止访问 ChatGPT，因为担心学生们会利用它来作弊。(值得注意的是，据报道，谷歌一直不愿发布自己的人工智能聊天机器人 LaMDA，因为担心其“声誉风险”)

奥特曼在这里说，OpenAI 确实有“一个内部过程，我们试图打破东西并研究影响。我们使用外部审计师。我们有外部红队。我们与其他实验室合作，并让安全组织查看相关资料。”

他说，与此同时，技术正在到来——来自 OpenAI 和其他地方——人们需要开始弄清楚如何与之共存。“ChatGPT 将会或正在引发一些社会变革。现在正在发生的一个大问题是它对教育和学术诚信的影响，所有这些。”尽管如此，他认为，“现在就开始这些(产品发布)是有意义的，因为风险相对较低，而不是仅仅发布整个行业几年后将拥有的东西，而没有时间让社会更新。”

事实上，教育者——也许还有父母——应该明白不可能把精灵放回瓶子里。虽然奥特曼表示，OpenAI 和其他人工智能机构“将试验”水印技术和其他验证技术，以帮助评估学生是否试图将人工智能生成的副本作为自己的副本，但他也暗示，过于关注这一特定场景是徒劳的。

“我们可能有办法帮助教师更容易发现类似 GPT 系统的输出，但老实说，一个有决心的人会绕过他们，我不认为这是社会可以或应该长期依赖的东西。”

他补充说，这不会是人们第一次成功地适应重大转变。观察到计算器“改变了我们在数学课上测试的内容”，谷歌使记忆事实的需要变得不那么重要，奥尔特曼说，深度学习模型代表了这两种发展的“更极端的版本”。但他认为“好处也更极端。我们听到老师们担心这会对家庭作业产生影响，这是可以理解的。我们也听到很多老师说，‘哇，这对每个孩子来说都是一个令人难以置信的私人导师。’"
