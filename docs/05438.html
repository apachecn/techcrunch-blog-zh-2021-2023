苹果前员工将 Face ID 隐私投诉带到欧洲 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/04/11/gobbler-complaint-europe/amp/

图片来源:诺拉·谭/南华早报/盖蒂图片社

欧洲的隐私监管机构正在考虑一名前员工 Ashley Gjø vik 对苹果公司的投诉，她指控苹果公司在她提出一系列问题后解雇了她，包括对工作场所安全的担忧。

苹果公司前高级工程项目经理 Gjø vik 去年 9 月被公司解雇，原因是她对雇主对待员工隐私的方式表示担忧，其中一些问题在 2021 年 8 月The Verge的一份报告中有所报道。

当时，Gjø vik 因对工作场所的性别歧视以及敌意和不安全的工作环境表示担忧，被苹果公司处以行政休假，苹果公司表示正在对此进行调查。她随后向美国国家劳工关系委员会投诉苹果。

这些早期的投诉与她现在向国际监督机构提交的隐私投诉有关，因为 Gjø vik 表示，在苹果正式告诉美国政府解雇她的原因后，她希望审查苹果的隐私做法，并“坦然承认他们会解雇抗议侵犯隐私的员工”，正如她所说的那样，指责苹果利用她对员工隐私的担忧作为借口，终止她的，因为她报告了更广泛的安全问题，并与其他员工一起组织了劳工问题。

英国信息专员办公室(ICO)和法国都证实收到了 Gjø vik 对苹果的隐私投诉。

ICO 的一位发言人告诉 TechCrunch:“我们知道这件事，我们将评估所提供的信息。”

法国 CNIL 也确认正在关注 Gjø vik 的投诉。

“我们已经收到了这份投诉，目前正在调查中，”CNIL 的一位发言人告诉我们，并补充说:“目前我不能传达任何进一步的细节。”

此事最先被 《电讯报》——该报昨日报道称，这被认为是gj vik 首次在英国寻求对苹果提起隐私诉讼

爱尔兰数据保护委员会(DPC)是苹果在欧盟的主要数据保护监管机构，负责泛欧盟通用数据保护条例(GDPR)，根据该条例的一站式机制，该委员会可能会在任何与 GDPR 投诉相关的调查中发挥主导作用，该投诉也已提交给其他欧盟隐私监管机构(如法国的 CNIL)，该委员会拒绝置评。检察部也没有证实或否认收到了 Gjø vik 的投诉。

刑事检察委员会的一名发言人说:“刑事检察委员会不能对个别案件发表评论。在适当的情况下，DPC 收到的所有问题都将按照 DPC 的投诉处理职能进行评估和处理。”

爱尔兰正在对苹果公司的数据处理行为进行多项 GDPR 调查，包括对该公司隐私政策的调查，但爱尔兰检察委员会尚未就这些持续多年的调查做出任何决定。

考虑到爱尔兰监管机构在 GDPR 积压的大量案件，如果 DPC 决定这起投诉值得对苹果展开新的调查，可能需要数年时间才能得出公开结果。

在诉状的结论中，Gjø vik 敦促监管机构“调查我提出的问题，并在苹果全球公司办公室内就这些问题展开更大规模的调查”，进一步声称:“苹果声称人权不因地理位置而不同，但苹果也承认，法国和德国政府永远不会允许它在加州库比蒂诺和其他地方做的事情。”

Face ID Gobbler 应用

Gjø vik 表示，这份长达 54 页的“隐私侵犯投诉”在本月早些时候提交给了欧洲监管机构，它对该公司的员工隐私做法提出了质疑，引发了对许多做法的担忧，包括苹果公司使用一款名为“Gobbler”(后来的“Glimmer”)的应用程序从员工那里收集生物特征数据的内部程序，这显然是 Face ID 产品开发过程的一部分。

更广泛地说，投诉集中在苹果的保密范围和“反员工隐私”政策，以及 Gjø vik 所称的“非法限制”保密协议。

记者联系苹果公司就该投诉置评，但在撰写本文时，该公司尚未回应。

根据诉状，这家科技巨头邀请员工参与产品测试的方式，有时涉及到捕捉生物特征，这让 Gjø vik 觉得她的参与是强制性的，在她详述的一个例子中，她描述了对她认为是“强制性的社交活动”的回应，结果是在被关在一个安全的户外院子里，在阳光下，使用 Gobbler 应用程序手动测试 Face ID。

根据诉状，苹果公司内部向员工提供的关于 Gobbler 的信息敦促员工上传在家中捕捉的应用程序数据。

“苹果迫使员工将他们的‘面纹数据’上传到苹果内部服务器，捕捉员工的秘密照片和视频，并告诉员工，与面部相关的日志每天都会从他们的 iPhones 上自动上传，”Gjø vik 指控道。

她还声称:“非常不清楚哪些数据是自动上传的，是如何上传的，何时上传的。”。 “我的开放式问题包括我的个人数据是否被 备份在员工 iCloud 备份上，通过 iCloud 同步， 和/或被苹果公司的 MDM 配置文件访问/复制——或 员工手机的其他全球安全监控。此外， 让我不安的是，该应用程序在没有任何 通知(声音、信号等)的情况下拍摄照片/视频，这让我认为，如果苹果公司 愿意，它也可以在任何时候激活我的设备摄像头并在 我不知道的情况下监视我。我和其他员工谈过，包括经理，也有类似的担忧。”

Gjø vik 引用了苹果公司的一份公开声明，称其 Face ID 算法的开发中使用了超过 10 亿张图片——声称该公司从未回答参议员艾尔·弗兰肯在 Face ID 发布后提出的问题。她建议道:“(苹果副总裁克雷格)费德里基没有说的是，这些图片来自像我一样的员工，不管我是否愿意分享它们。”。

根据诉状，苹果告知员工，美国以外的国家限制员工向 Gobbler 上传数据——尽管诉状还引用了一封苹果经理的电子邮件，其中称一项此类研究正在“美国、巴西、特拉维夫”和欧盟进行，“但不包括法国或德国”。

“我还在笔记中看到，该应用在日本和中国被禁止使用，但后来在某个时候，苹果决定在那里收集一些日志，”Gjø vik 进一步建议道。

苹果在欧洲确实有办事处——包括英国、法国、爱尔兰和该地区的其他地方——所以这些地方的员工至少有可能使用天书应用上传他们的生物特征数据。如果发生这种情况，它可能会涉及数据保护方面的考虑，比如苹果在处理这些数据时可以依赖的法律依据。但是，收到她的投诉的欧洲监管机构是否认为这里有他们需要调查的事情，还有待观察。

根据 GDPR 法案，同意是处理个人数据的几种可能的法律依据之一。然而，要使同意成为有效的法律依据，它必须是知情的、具体的和自由给予的——而且，即使不考虑是否向工作人员提供了关于如何处理其生物特征数据的充分信息的问题，雇主-雇员权力动态可能会削弱他们自由同意的能力(即相对于感觉他们必须参与这种测试，因为这是他们的雇主要求的)。因此，有理由进行更仔细的审查。

Gjø vik 的投诉也已提交给欧洲数据保护监督员(EDPS)，尽管该机构的发言人证实，EDPS 不会调查此类事件，因为其监督职能集中在欧盟自身的机构、组织或部门。

该投诉还将加拿大隐私专员办公室以及数字权利组织 EFF 和老大哥观察列为另一个被提交的机构。

除了天书/微光应用程序之外，gjvik 还引发了人们对苹果软件开发票证/臭虫报告系统在员工没有适当意识到的情况下收集个人数据的可能性的担忧——声称该系统默认向公司所有软件工程部门(可能有数万人)共享报告。该公司还表示，这些罚单可能会要求员工填写诊断文件——gjvik 认为，这可能会导致员工个人设备中的额外个人数据，比如他们的即时消息，在员工没有完全意识到的情况下被传递给苹果公司。

《边缘》去年的一篇文章引用了 gjvik 和其他一些苹果员工的话，据报道，该公司的员工经常被告知将他们的苹果个人识别码与他们的工作账户联系起来。

“个人和工作账户的模糊导致了一些不寻常的情况，包括 Gjø vik 据称被迫将自己的照片交给苹果律师，当时她的团队卷入了一场不相关的法律纠纷，”The Verge 报道说，然后引用了它所描述的“严格的雇佣协议，该协议赋予苹果公司进行广泛的员工监控的权利，包括'物理、视频或电子监控'，以及在上搜索你的工作场所，如文件柜、办公桌和办公室(即使上锁)，查看电话记录，或搜索任何非苹果财产(如背包、钱包)

The Verge 的报道强调的另一项苹果政策是禁止员工在将设备归还给公司之前擦拭任何设备，包括他们是否/何时离开苹果——这表明，将个人 Apple ID 与工作账户关联的员工在归还公司设备时，可能会向公司暴露隐私数据。
