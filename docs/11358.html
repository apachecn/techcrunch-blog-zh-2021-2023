播客应用 Pocket Casts 开源

流行的播客平台 Pocket Casts 已经在开源许可下发布了它的移动客户端。

WordPress 母公司 Automattic 去年 7 月收购了 Pocket Casts，该公司已被一群公共广播公司收购，其中包括 2018 年的。Pocket Casts 是谷歌、苹果和 Spotify 等大型科技生态系统之外最受欢迎的“播客”应用之一，允许用户免费搜索和订阅播客，并提供收费的桌面应用等高级功能。

考虑到 Automattic 创始人兼首席执行官马特·莫楞威格是开源项目的大力支持者，并且 WordPress 是这个星球上最顶尖的开源项目之一，Automattic 选择将 Pocket Casts 应用程序代码发布到 GitHub 上可能不是最大的惊喜。

通过开源 Pocket Cast，这意味着任何人都可以访问代码，修复错误，创建新功能，甚至可以在 Pocket Cast 代码基础上构建自己的竞争服务。

Android和 iOS应用现在可以在 Mozilla Public License 2.0 下获得，这是一个 copyright left 许可证，规定所有衍生项目或修改都必须在同一许可证下发布。
