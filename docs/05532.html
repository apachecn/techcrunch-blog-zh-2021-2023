费用管理提供商 Emburse 与 Brex 和 Ramp | TechCrunch 正面交锋

原文：https://web.archive.org/web/https://techcrunch.com/2022/04/13/spend-management-provider-emburse-takes-on-brex-and-ramp-in-going-after-smbs/

凭借 1200 万用户和近 2 亿美元的年度经常性收入，费用软件提供商 Emburse 的大部分业务都建立在企业中。

从历史上看，它一直与 Concur 和 Expensify 等传统公司以及 ERP 公司竞争。

但现在，Emburse 正在大举进军中小企业领域，并与 Ramp 和 Brex 等快速增长的初创公司正面交锋——后者始于市场的中小企业端，巧合的是，今天它宣布正在大举进军软件和企业领域。

今天，emb course 发布了它的第一个产品——emb course Spend——它依赖于交换费而不是订阅费。同样，这与 Brex 采取的行动相反，它正在使其收入基础多样化，除了交换费之外，还包括更多的订阅。

在日益拥挤、竞争激烈的企业支出领域，企业要么主要关注订阅，要么以收费为基础。但随着时间的推移，越来越多的企业支出者正演变成两者兼而有之。

Emburse Spend 的目标是“将公司名片从高管身份的象征转变为团队的日常工具。”它针对的是那些“经常”购买小额商品的员工，比如软件订阅，而不是一年几次出差的费用。该公司宣称，有了 Emburse Spend，经理们可以“随时”发放信用卡，而无需等待财务部门的批准。

Emburse 的历史有点复杂，所以我们会尽量简化。这家最初名为 Emburse 的公司在 2015 年 Disrupt 阶段迅速向员工发放预付卡。

2019 年，一家名为 Certify 的私募股权所有者 K1 的公司收购了 Emburse 和另一家名为 Chrome River 的公司，后者确实在企业领域进行了支出管理和发票自动化。这三家公司合并成了今天的 Emburse。

该公司有三个核心产品:Emburse Chrome River，专注于企业；Emburse Certify 服务于中端市场公司，其最新产品 Emburse Spend 面向中小型企业。

“这对我们来说是一个很大的区别，”Emburse 首席执行官兼 Concur 校友 Eric Friedrichsen 说。“市场上确实需要一家具有一定规模的公司，能够真正满足特定类型的组织在规模和地理位置方面的定制需求。”

在该公司看来，埃克森美孚的需求“与 Pinterest 的需求大相径庭”。

如今，中小型企业约占 Emburse 客户群的 15%,其中 35%是中端市场公司，50%是大型企业。 Faraday.ai 和公寓生活就是其当前中小企业客户的例子。

“我们预计所有这些领域都将增长，”Friedrichsen 告诉 TechCrunch。“但我们预计中小企业市场将会以更快的速度增长……”我们在中小企业领域看到了巨大的商机。"

它是如何工作的

Emburse Spend 基于该公司现有客户已经使用的相同平台，旨在为公司提供一种“轻松”向员工分发和管理虚拟和塑料卡的方式，同时最大限度地降低政策外支出的风险。该公司表示，员工可以使用这些卡购物或出差，而无需担心收取收据或提交费用报告。

例如，员工可以根据购买需求提交卡的请求，购买需求的范围可以从软件订购到旅行预订。卡管理员可以“立即”发行虚拟卡，并在卡中自动编码消费水平和时间限制等限制。

当员工通过 Emburse 卡购物时，他们会在移动消费应用程序上收到实时提醒，以捕捉收据的图像。该公司的 OCR 引擎验证收据是否与卡交易数据相匹配。创建一个费用记录，然后确定是否符合政策。

Emburse 表示，它能够做到这一切，是因为它使用“复杂的”光学字符识别、审批工作流程和机器学习来为购买和对账提供“无缝解决方案”。

甚至在今天的新闻之前，纸牌就已经在 Emburse 的策略中发挥了作用。2021 年，宣布与万事达卡建立白色标签合作关系，类似于最近空军基地与美国运通签署的合作关系。该公司还推出了平台，允许客户从硅谷银行等合作银行发行公司卡。

“在发卡方面，我们有一个开放的平台。弗里德里希森说:“我认为这是我们这个领域的初创公司的一大区别。“在某些情况下，客户可能需要我们的特定解决方案，但通常我们的目标是与金融机构合作，而不是与之竞争。”

据 Friedrichsen 称，Emburse 是盈利的，在全球拥有 18，000 名付费用户，去年的 ARR 增长了近 30%。它每年处理大约 650 亿美元的支出。

“我们的总留存率非常非常高，”这位高管说。

值得注意的是，该公司最近聘请阿德里亚娜·卡彭特担任首席财务官。在她之前的角色中，她通过 IPO 帮助获得了 Ping 的身份。Friedrichsen 告诉 TechCrunch，首次公开募股是 Emburse 的未来，只是在正确的时间。

“未来几年，我们可能会上市，这是合乎逻辑的一步。当然，我们有足够的规模来做这件事。我们已经有了这样做的基础设施，如果这是我们最终的发展方向，我们将确保我们已经做好准备，”他说。

Emburse Spend 是一个免费的解决方案，该公司表示，它还通过卡回扣来节省费用。“附加功能”可通过订购获得。

目前，Emburse 主要集中在北美和欧洲，亚洲和澳大利亚也有一些客户。它希望向全球扩张。

那么，为什么近年来企业支出管理变得如此热门呢？弗里德里希森认为，新冠肺炎疫情确实加速了对更好的支出管理解决方案的需求。

“如今的员工比以往任何时候都更加分散。所以现在，员工需要能够以一种非常有效的方式代表他们的雇主花钱，”他告诉 TechCrunch。“它还需要非常高效，但要有一些控制措施。我认为这是这个领域真正腾飞的重要原因之一。”

此外，他说，从公司卡的角度来看，技术已经“有了很大的发展”，能够比以往任何时候都向公司卡中加入更多的控制。
