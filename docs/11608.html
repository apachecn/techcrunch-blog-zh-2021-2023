Shopify 收购了 Remix 来加强其店面设计工具

原文：https://web.archive.org/web/https://techcrunch.com/2022/10/31/shopify-acquires-remix-to-bolster-its-storefront-design-tools/?tpcc=TCdailynewsletter

两家公司今天在一份联合声明中宣布，雷米克斯(Remix)，一家开发类似于next . js的开源网络框架的初创公司，已被 Shopify 收购。财务条款未披露，但在博客帖子中，Remix 首席执行官迈克尔·杰克逊表示，Remix 将获得来自 Shopify 的“长期支持和支持”，这将使其“增长更快”并“更加注重性能和可扩展性”。

“你会看到更多的(Remix 框架)在野外，为一些最大的商业网站提供动力，”杰克逊说。“此外，shotify 本身将在许多项目中使用 Remix，随着时间的推移，您可以预料到 shotify 的开发平台会有更多对 Remix 的一流支持。”

2020 年，前推特工程师杰克逊和瑞安·弗洛伦斯共同创立了 Remix。在决定推出同名的 Remix 框架之前，两人合作多年，围绕 React(一个用于构建应用程序用户界面的 JavaScript 库)创建了开源工具。

杰克逊和佛罗伦萨最著名的项目之一是反应路由器，这是一个反应库，已经被下载了近十亿次。并非巧合的是，shotify 最初使用 React Router 来构建 Hydrogen，这是该公司用于构建定制 shotify 店面的前端网络开发框架。

至于 Remix，它是一个完整的网络框架，旨在利用分布式系统和本地浏览器功能，同时抽象出后端服务器任务。与公共云环境兼容，包括亚马逊网络服务、谷歌云、Netlify、Vercel 和 Cloudflare Workers，Remix 的关键功能之一是预取——该框架可以在用户点击链接之前并行预取网页的元素，包括按钮和表单，以最大限度地减少页面加载。

在收购 Shopify 之前，Remix 已经从开放源码软件资本和天使投资者 Naval Ravikant、Ram Shriram 和 Sahil Lavingia 那里筹集了 300 万美元的种子资金。

在 Shopify Engineering 博客上的一篇帖子中，shoppiy 工程副总裁 Dion Almaer 表示，购买 Remix 将通过给 Hydrogen 带来改进，使 shoppiy 开发者和商家都受益。

“Remix 将继续是一个独立的开源框架，”阿尔迈尔说。“Remix 将解决基于 Hydrogen 的开发人员在数据加载、路由和错误处理方面遇到的挑战……Shopify 将在许多有意义的项目中使用 Remix，随着时间的推移，你可以期待看到我们的开发人员平台更多地支持一流的 Remix。”

Remix 是 Shopify 自今年 5 月以 21 亿美元收购的履行技术提供商 Deliverr 以来的首次收购。今年早些时候，Shopify 收购了帮助品牌管理影响者营销活动的燕尾。在对 CMS developer Sanity 和 Klaviyo 营销自动化初创公司进行股权质押后，该公司最近还投资了 Shopify 上许多企业使用的音乐和视频应用软件 Single。

在经历了艰难的 Q2 之后，有迹象表明 Shopify 开始更好地度过经济衰退。该公司上周公布的第三季度亏损低于预期，导致股价飙升 17%。
