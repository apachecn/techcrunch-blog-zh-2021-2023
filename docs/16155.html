AWS 在亚马逊最新一轮裁员中受到打击

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/20/aws-takes-a-hit-in-latest-round-of-amazon-layoffs/

当亚马逊宣布今天再裁员 9000 名员工时，AWS 员工也未能幸免，亚马逊首席执行官(前 AWS 首席执行官)Andy Jassy 宣布云部门将包括在今天的一轮裁员中。

TechCrunch 听说今天总收入的 10%来自 AWS。该公司不愿证实这些数字，而是提到今天早上公布的 Jassy 给员工的备忘录作为其声明的要点。

根据这份备忘录，公司分阶段裁员的原因是一些经理仍在评估他们的部门，他们在第一轮时还没有准备好。“简短的回答是，并不是所有的团队都在晚秋完成了他们的分析；Jassy 写道:“我们选择分享我们做出的决定，而不是在没有适当努力的情况下匆忙完成这些评估，这样人们就可以尽快获得信息。”

Constellation Research 的创始人兼首席分析师 Ray Wang 表示，亚马逊必须仔细研究组织的每个方面，AWS 也不例外。“这是科技公司再次变得精简的更大趋势的一部分，亚马逊在过去几年变得臃肿。他们最终在几周前完成了分析，现在 AWS 也在裁员，”他说。

在上个月初该公司最新的收益报告中，云部门的增长率从上一年超过 39%的增长率下降到了 20%。更糟糕的是，首席财务官 Brian Olsavsky 打电报说增长放缓更多。“展望未来，我们预计至少在未来几个季度，这些优化工作将继续阻碍 AWS 的增长。他当时表示:“今年第一个月到目前为止，AWS 的同比收入增长在 15%左右。

在这种背景下，裁员并不令人意外。事实上，云基础设施市场整体增长缓慢。经过多年的数字失控后，云支出正在缩减，并开始对市场产生影响。在最近的收益报告周期结束时，云基础设施市场的整体增长率从去年的 36%下降到 21%。

更重要的是，亚马逊的长期竞争对手微软获得了市场份额。虽然微软上个季度的增长也有所放缓，但该公司的增长速度一直在加快，并开始在 AWS 上缓慢但稳定地增长。

尽管如此，观察云基础设施市场的研究公司 Synergy Researcher Group 的首席研究员兼研究主管约翰·丁斯代尔(John Dinsdale)表示，尽管有这些数字，但市场仍在大幅增长，并受到外部因素的困扰，如“历史上强劲的美元和严格受限的中国市场”。

丁斯代尔说，鉴于市场下滑，裁员是可以理解的，但他不认为这是恐慌的原因..“那么，市场增长率在下降吗？是的。对于收入和组织以惊人的速度增长的公司，是否有合理化和削减成本的空间？是的。天塌下来了吗？不，”他说。
