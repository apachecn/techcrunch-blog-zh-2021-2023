第三方 Twitter 应用制造商将注意力转向乳齿象| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/12/05/third-party-twitter-app-makers-turn-their-attention-to-mastodon/

在埃隆·马斯克收购 Twitter 之后，开源 Twitter 替代产品乳齿象获得了一点的关注。现在，它也引起了第三方 Twitter 应用开发者的兴趣。包括 Aviary 和 Tweetbot 在内的流行 Twitter 客户端的制造商最近将目光投向了为日益增长的乳齿象用户群构建类似的客户端。

虽然 Twitter 的流失只影响了社交网络整体用户群的一小部分，但新来者涌入规模小得多的乳齿象生态系统对其社区产生了重大影响。根据独立的估计，如今，乳齿象的活跃用户群已经从马斯克接管 Twitter 后的 655，000 人增长到 T4 的 330 万到 T6 的 360 万。

乳齿象的新用户正在学习如何选择服务器的基础知识，以及如何找到他们的朋友，同时也在学习其他的社区价值，比如在适当的时候为他们的帖子使用“内容警告”。他们也意识到乳齿象不是 Twitter 的克隆，尽管最初有一些相似之处。通用搜索不存在，需要大量使用转发和标签来增加帖子的可见性。直接消息的工作方式与 Twitter 不同。没有类似于引用推文的功能。诸如此类。

然而，现在正在开发的新的乳齿象客户端可能有助于让乳齿象对前 Twitter 用户更熟悉，这些用户正在寻求转向“Fe diversity”——这是为乳齿象和其他应用程序提供支持的服务器的互联集合。

公平地说，乳齿象社区并没有因为移动应用程序而受到伤害。除了官方应用，已经有不少第三方客户端可用，包括元文本、图特尔、嘟嘟！、桅杆、桅杆、獠牙、水银等。其中几个在过去一个月左右的时间里增加了开发活动，因为它们的开发者意识到他们的应用程序端项目突然获得了新的追随者。

但对于 Twitter 的长期用户来说，Twitter 生态系统中知名应用程序制造商的加入是一个令人兴奋的前景。

其中一个是 Tapbots，它因在 iOS 和 Mac 上流行的 Twitter 应用程序 Tweetbot 而闻名。被誉为与时俱进的第三方 Twitter 客户端之一的，该公司今年早些时候发布了 Tweetbot 7，增加了画中画、统计标签和小工具等功能。现在，推特机器人的开发者保罗·阿达德正在为乳齿象开发一款类似的应用。

哈达德表示，虽然自马斯克接管以来，该公司尚未经历 Twitter API 的中断(除了 bug 和其他问题略有增加)，但在裁员和随后的员工离职后，他们失去了与 Twitter 的联系。

为了解决当前离开 Twitter 的用户的“大量请求”，该公司目前正在为乳齿象用户开发一款基于订阅的应用程序，非常类似于 Tweetbot。

图片来源: Tapbots

这款名为 Ivory 的应用程序仍处于开发的初级阶段，但可能会吸引那些希望应用程序能够快速访问关键功能的 Tweetbot 用户——在这种情况下，导航按钮会带你进入你的家庭时间表、@提及、收藏夹、搜索和趋势，以及你自己的用户资料。如果您愿意，可以将一些按钮换成其他按钮。例如，长按搜索按钮可让您切换到统计功能或用于配置静音过滤器的另一个屏幕。除了帮助用户屏蔽他们不关心或感到不安的话题，后者还可以帮助年长的乳齿象用户不必经常在他们的订阅中阅读 Twitter 和 Musk 这是自 Twitter 离开以来一直存在的抱怨。

为了创建一个新的帖子，它添加了一个带小喇叭的浮动按钮——向乳齿象版本的推文致敬，有时被称为“嘟嘟”

哈达德说:“我们这款应用的初始目标是为乳齿象复制 Tweetbot 体验。”“我们希望在 Twitter 上熟悉 Tweetbot 的人感觉像在家里一样使用象牙。一旦我们有了一个稳定的 1.0 版本，我们将开始添加更多乳齿象特有的功能，以及一些我们想添加到 Tweetbot 但由于技术限制而无法添加的功能，”他指出。

象牙的一个区别在于，与 Twitter 不同，它不会显示提升次数(乳齿象的转发次数)或时间线视图中一篇帖子收到的收藏或评论数量等指标——这是一种旨在减少屏幕混乱(或许还有追逐影响力)的设计选择。

图片来源:tap bots/象牙

Tapbots 表示，象牙将与 Tweetbot 一起开发，因为它们共享大量代码，Tweetbot 和象牙的 Mac 版本也在开发中。

该应用程序已经证明很受欢迎，因为该应用程序的 TestFlight 版本在发布后迅速爆满。

另一个第三方 Twitter 应用程序开发者 Shihab Mehboob 开发了鸟舍，他也已经开始开发乳齿象客户端，名为 Mammoth 。新应用将是付费下载，价格待定，发布时将包括最新的 Mastodon API 功能，以及编辑帖子和编辑历史等 4.0 功能。iPad 和 Mac 版本也将发布。

图片来源:猛犸

“我的动机是希望有一个‘好的’乳齿象应用程序，因为所有现有的应用程序都以这样或那样的方式缺乏功能或设计范式。他们都有所欠缺，也没有感受到 iOS 和苹果平台的原生性。所以我开始制作自己的产品来实现这一切，”Mehboob 解释道。他说，这款应用与众不同，它以 iOS 为中心的设计“在你的设备上感觉就像在家里一样”，还有一套全面的功能集。

“这也是一种使用乐趣，并且有一些很酷的功能，如以图像形式分享帖子，在 AR 中查看它们，通过设置调整各个部分，等等，”Mehboob 补充道。

然而，一个著名的第三方 Twitter 应用程序开发商，iOS 和 Mac 流行 Twitter 应用程序的制造商 Iconfactory 表示，他们尚未决定进入乳齿象应用程序领域。联合创始人 Gedeon Maheux 承认，最近几周，他们也失去了在 Twitter 上的 API 联系人。但目前，该公司只是在观察这种替代生态系统的发展——并为自己使用乳齿象。

“成为一个正在成长并适应许多新用户的服务的一部分是令人兴奋的，就像 Twitter 当年做的那样，”Maheux 说。“至于 Iconfactory 的乳齿象应用，我们目前没有任何东西可以宣布，”他指出。

当然，所有这些第三方应用仍将不得不与乳齿象自己的应用竞争，自马斯克收购 Twitter 以来，已经看到了自己的改进，而获得了另一位开发者来帮助加快应用更新。但是与 Twitter 不同的是，乳齿象对新的发展持开放态度，在过去的几年里，Twitter 积极地 工作碾压第三方客户。
