Polygon 和不可变合作伙伴，帮助更多游戏玩家和开发者加入 web3 | TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/20/polygon-and-immutable-partner-to-help-onboard-more-gamers-and-developers-into-web3/

在周一的游戏开发者大会上，Web3 游戏公司 Immutable 和 layer-2 区块链多边形公司宣布，两家公司希望新的战略联盟将加速新生加密游戏领域的创新和采用。

“对我们来说，这是一部相当明显的戏，”Immutable 的联合创始人兼总裁罗比·弗格森(Robbie Ferguson)说。“我们很快意识到以太坊的规模限制，但我们从未想过与之竞争。”Polygon 可以帮助 Immutable 避免以太坊拥塞和成本，而不必构建自己的替代方案。

该联盟将致力于让更大的游戏工作室和独立开发者参与进来，使基于 web3 的游戏更快、更容易、风险更低。

去年，在这两个平台上开发的游戏获得了大约 20 亿美元的投资者资金，Ferguson 和多边形实验室总裁 Ryan Wyatt 与 TechCrunch 分享了这一消息。

“对我来说，这就像是主流收养的下一次演变，”怀亚特说。“你已经开始看到更高保真度的区块链游戏了。”

Immutable 的新零知识以太坊虚拟机(zkEVM)——压缩数据以提高区块链上的可伸缩性和安全性的虚拟机的花哨术语——将由 Polygon 技术提供动力，并在其平台上得到支持。Polygon 的 zkEVM 扩展技术旨在降低交易成本，同时保持与第一层区块链以太坊的兼容性和安全性，这对区块链及其生态系统的长期发展至关重要。

Wyatt 说，Polygon 是“为游戏开发者和游戏玩家提供的一个非常干净、完善的端到端解决方案和市场”。“有人怀疑它会走向何方，会是什么样子[……]现在你将看到它在未来几年内加速发展。”

弗格森说，将两家公司的技术结合起来，可以帮助交易规模比以前增加 100 到 1000 倍。

虽然这是一项新举措，但通过与游戏工作室、传统企业和加密公司的合作和整合，这两个平台已经形成了庞大的游戏生态系统。

Polygon Labs 是分散 Polygon 协议背后的团队，已经与 Square Enix、Neowiz、Midnight Society、Plai Labs 和 Tilting Point 等游戏公司合作。该公司的第二层扩展解决方案已经获得了显著的采用，包括部署了数万个分散的应用程序，服务了超过 2.2 亿个唯一地址，部署了超过 100 万个智能合同，自成立以来处理了数十亿笔交易。

Polygon 网络还被集成到 Aave、Uniswap 和 OpenSea 等加密项目以及 Robinhood、Stripe 和 Adobe 等知名企业中，这可能有助于进一步扩展其覆盖范围。

Polygon network 还托管了一些最大的 web3 游戏项目和发行商，如育碧、雅达利、 Animoca Brands 、分散和沙盒等。在过去的一年里，Polygon 与许多其他大品牌合作，如星巴克的奥德赛数字收藏品奖励计划和迪士尼的加速器计划，同时还有普拉达和阿迪达斯等主要服装品牌通过其区块链推出 NFT 项目。

另外，Immutable 在最近几个月已经将 web3 游戏和倡议纳入其平台，包括 GameStop、DC Comics、抖音和漫威等品牌以及迪士尼和星球大战的 IP。该平台还在 6 月推出了一个 5 亿美元的基金，以促进 web3 游戏的采用。

上述公告可能指向 web3 游戏行业的增长；更多的品牌进入 web3 游戏领域可能会扩大更大的分散式游戏市场，这有点类似于 Web 2.0 公司之间的整合在历史上帮助他们扩大了自己的影响力。

“游戏玩家只是想玩伟大的游戏；就这么简单，”怀亚特说。“你真的只需要达到这样一个程度，你将密码和行话抽象出来，然后就有了一个真正令人兴奋的游戏。这就是即将到来的事情。”
