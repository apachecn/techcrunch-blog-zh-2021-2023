报告显示，Safari 在全球拥有 10 亿用户，仍落后于谷歌 Chrome TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/06/01/report-shows-that-safari-reaches-one-billion-worldwide-users-still-behind-google-chrome/

根据最近的一份报告，苹果的 Safari 终于达到了 10 亿用户，成为第二个达到这一里程碑的浏览器，仅次于谷歌 Chrome，后者的人气仍然更高。根据 Atlas VPN 的数据，目前有 1，006，232，879 名互联网用户(占所有互联网用户的 19.16%)使用 Safari 浏览器。与此同时，谷歌 Chrome 的用户数量是谷歌的三倍，达到 3，378，967，819 人。微软 Edge 以大约 212，695，000 用户位居第三。

Atlas VPN 的发现是基于 GlobalStats 浏览器市场份额百分比，该百分比是使用互联网世界统计互联网用户指标检索精确数字而转换成数字的。它的报告似乎表明，Safari 的增长可能与浏览器采用隐私和安全功能有关，但它可能更多地反映了 Safari 和 Chrome 安装在苹果和安卓设备上的移动市场份额。

图片来源: Atlas VPN

苹果的网络浏览器会自动安装在每台苹果设备上，而 Chrome 则安装在大多数版本的 Android 上。这有助于两家科技巨头在竞争中占据优势。虽然微软在 Windows 上发布了 Edge 浏览器，但由于 Windows Phone 的失败，它在移动市场上失利了。

也就是说，苹果在过去一年中为浏览器引入了几个新的隐私功能，这可能有助于吸引用户。例如，浏览器的新隐私报告显示了智能跟踪预防(ITP)阻止了多少和哪些跨站点跟踪器访问您的信息。

然而，Chrome 通常被认为是更有吸引力的浏览器，其网络应用程序包括 YouTube 、Drive、Calendar、 Docs 、 Earth 和 Maps 等等。就在上周，谷歌推出了一种在桌面上使用谷歌镜头的新方式，允许桌面上的 Chrome 浏览器用户使用谷歌镜头搜索网页上的任何图像。4 月，谷歌手机应用程序也推出了一个新的“多重搜索”功能。

此外，谷歌 Chrome 版本 100 于 3 月下旬发布，Android 上的 Chrome 速度提高了 15%。

报告中的其他浏览器包括微软 Edge、火狐、三星互联网和 Opera。如上所述，微软 Edge 是第三大最受欢迎的浏览器，拥有超过 2.12 亿用户，超过了拥有 1.79 亿互联网用户的 Firefox。自 Windows 11 发布以来，微软 Edge 成为所有设备上的默认浏览器，反过来，其用户群也大幅增加。

Firefox 浏览器面向更关注隐私的用户，因为它提供了一些保护功能，如防止跟踪、阻止恶意软件弹出窗口和网络钓鱼。Mozilla 在 5 月初推出了浏览器的版本 100 。

排名第五的是三星互联网浏览器，用户超过 1.49 亿。最后是 Opera 浏览器，用户超过 1.08 亿。人们不仅可以利用 Opera 完成日常任务，它还拥有一个加密浏览器和应用，内置一个支持以太坊、多边形和塞洛区块链技术的加密钱包。
