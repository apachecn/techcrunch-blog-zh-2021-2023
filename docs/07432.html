亚马逊扩展了其量子努力，专注于网络

亚马逊今天宣布一项新的努力，将量子计算引入其云计算——至少从长期来看是这样的。该公司今天推出了 AWS 量子网络中心，这是一项新的研究工作，旨在推动量子计算机联网的科学和工程，既用于构建更强大的多处理器计算网络，也用于创建安全的量子通信网络。

近年来，亚马逊及其 AWS 云计算部门在量子计算领域进行了大量重大投资。通过亚马逊 Braket ，该公司为开发者提供了从 IonQ、Oxford Quantum Circuits、Rigetti 和 D-Wave 等公司获得量子计算机的途径，以及其他软件工具和模拟器。除此之外，该公司还正在开展另外两项以研究为中心的工作:位于加利福尼亚州帕萨迪纳的 AWS 量子计算中心，该中心专注于基础科学，如构建更好的量子位和纠错算法；以及亚马逊量子解决方案实验室，该实验室侧重于帮助企业为量子计算的未来做准备。

基本上，虽然 Braket 和量子解决方案实验室专注于近期的实际解决方案，但量子计算中心和现在的量子网络中心专注于长期的研究工作。

“为了释放量子设备的全部潜力，它们需要连接在一起成为量子网络，类似于今天设备通过互联网连接的方式，”该公司在今天的公告中解释道。“尽管没有得到与量子计算机同等程度的关注，量子网络还是有令人着迷的潜在应用。其中之一是使全球通信受到量子密钥分发的保护，其隐私和安全级别是使用传统加密技术无法实现的。量子网络还将通过连接在一起并放大单个量子处理器的能力，提供强大而安全的云量子服务器。”

截至目前，大多数围绕量子网络的研究工作都处于国家资助的研究实验室的水平。大多数商业量子计算的努力都是在处理器级别(以及围绕它的生态系统)，所以亚马逊现在专注于这一点肯定是有点新奇的，但现在可能是时候专注于量子计算的这一方面了，特别是考虑到量子处理单元现在开始达到一个成熟的水平，这在一二十年前还被认为是科幻小说。
