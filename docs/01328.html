亚马逊网络服务倒下了，带走了一堆互联网

杰斯·康迪特是

Engadget

用户报告称，亚马逊网络服务出现大范围中断，导致美联社、迪士尼+和 Vice 等机构的网站和后端服务瘫痪。关于停电的细节很少。

周二早上，AWS 状态页面报告其管理控制台的错误率增加。

“我们在 US-EAST-1 地区遇到了 API 和控制台问题，”该页面称。“我们已经确定了根本原因，正在积极努力恢复。此问题正在影响全球控制台登录页面，该页面也位于 US-EAST-1。

AWS 于 6 月关闭，关闭了 Twitch、Reddit、Twitter、Hulu、HBO Max、Shopify 和亚马逊等网站。停机问题在同一天得到了解决。AWS 在今年之前也遭遇过中断。
