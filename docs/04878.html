网飞测试一项新功能，将提高账户分享的价格

原文：https://web.archive.org/web/https://techcrunch.com/2022/03/16/netflix-tests-a-new-feature-that-will-raise-prices-for-account-sharing/amp/

图片来源:克里斯汀·博西/彭博/盖蒂图片社

网飞将开始测试一项新的选择性加入功能，如果用户与自己家庭以外的人共享该服务，将提示用户支付额外费用。该功能将允许家庭添加两个“子账户”,费用低于全价网飞服务的费用。新的选项将首先在智利、哥斯达黎加和秘鲁进行测试，并将在未来几周内推广到这些市场。该公司没有说这项功能是否或何时会向全球用户推出。

在推出时，网飞在测试市场的标准和高级用户将被提供选项，为他们的服务添加子帐户，供他们不与他们生活在一起的人使用。像往常一样，每个子账户都有自己的个人资料和个性化推荐。但是这个功能的不同之处在于，子账户也有自己的网飞登录和密码。此外，如果他们选择以自己的名字和账单信息建立自己的网飞服务，他们的观看历史、观看列表(“我的列表”)和个性化推荐都会转移过来。

为了让这项功能发挥作用，网飞并不依赖基于位置的数据，比如 GPS。相反，该公司正在利用它今天用来向其最终用户提供服务的相同信息，包括 IP 地址、设备 id 和家庭中登录网飞帐户的设备的其他信息。利用这些信息，网飞可以识别出家庭以外的持续分享。

为了启用新的子帐户，主帐户所有者将收到一封电子邮件，其中包含一个代码，他们需要使用该代码来验证附加设备是他们家庭的一部分。

测试市场中额外非家庭成员的成本在智利为 2，380 CLP，在哥斯达黎加为 2.99 美元，在秘鲁为 7.9 PEN。这比完整的网飞帐户计划便宜，但也比以前免费分享某人的网飞帐户要贵。网飞指出，额外的会员不会单独为他们的账户付费——网飞的账单仍然会到主要账户持有人那里。(如果那个人想向他们的额外会员收取新的费用，那就由他们来处理和收取。)

早些年，网飞基本上对用户与直系亲属以外的其他朋友和家人分享其网飞账户的普遍做法视而不见。事实上，网飞当时的首席执行官雷德·哈斯汀斯曾经将账户共享描述为一件“积极的事情”，因为它为新人们提供了一个发现这项服务的机会。但今天，网飞不需要证明太多——它的许多节目，如《奇异事物》或《乌贼游戏》，已经成为全球娱乐轰动。与此同时，它面临的真正挑战是在竞争加剧的情况下继续增加用户和收入。例如，流媒体工具的年终季度在尤为艰难，用户增长创下多年来的新低。

理论上，通过新的转移选项，为额外会员创造一条成为正式用户的途径，随着时间的推移，可能有助于扩大网飞的付费用户群。然而，在短期内，值得注意的是，额外的成员不会被算作订户，而仍然使用其他人的帐户。

尽管网飞过去很乐意接受账户共享，但它今天的立场不同了。网飞目前的服务条款规定，分享应该发生在家庭内部，而不是家庭之间。该公司还在去年三月测试了一个功能，即被描述为对密码共享的打击，这表明可能会有更大范围的打击。

然而，我们知道，网飞并不认为这项新的测试是一种镇压——它只是想尝试一种选择，看看它对成员国是否有意义。用户的强烈反对可能会导致公司搁置这个想法，转而尝试其他东西。或者，如果成员国接受这一功能，网飞可能会决定更广泛地推广这一功能。时间会证明一切。

“我们认识到人们有许多娱乐选择，所以我们希望确保任何新功能都是灵活的，对会员有用的，他们的订阅资助了我们所有伟大的电视和电影，”网飞产品创新总监成毅·龙在博客文章中写道。“在世界其他地方做出改变之前，我们将努力了解这两个功能对这三个国家的成员的效用，”龙补充说。
