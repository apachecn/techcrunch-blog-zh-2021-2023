Twitter 宣布推出只有免费、基础和企业级的新 API

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/29/twitter-announces-new-api-with-only-free-basic-and-enterprise-levels/

Twitter 宣布新的 API，只有免费、基础和企业级

经过数周的拖延，Twitter 终于在周三宣布了新的 API 价格结构。这三个等级包括主要用于内容发布机器人的基本免费等级、每月 100 美元的基本等级和昂贵的企业等级。该公司表示，订阅任何级别都可以免费获得 Ads API。

Twitter 提到，在未来 30 天内，该公司将停止旧的访问级别，包括标准(1.1 版)、基本和高级(2 版)以及高级。

Twitter 的 API 传奇始于该公司 2 月份宣布将在几天内结束免费 API 访问。在遭到严厉批评后，埃隆·马斯克表示，该公司将向提供“好内容”的机器人提供免费等级后来，它说基本层将从每月 100 美元开始，但没有给出任何关于访问级别的细节。2 月 13 日，该公司表示已经将发布时间推迟了“几天”超过 45 天后，该公司终于提供了新 API 的相关信息。

新的 API 产品看起来像是抢钱。免费层每月仅提供 1500 个帖子请求，以及使用 Twitter 登录的权限。基本层被认为是“为业余爱好者或学生准备的”，每个应用程序每月提供 50，000 次发布请求和 10，000 次阅读请求。想要访问更多数据的开发者将不得不申请企业访问，据报道，每月花费高达 42，000 美元。

早些时候，随着 2020 年 v2 的推出，Twitter 向开发者提供了多个访问级别，如 Essential 和 Elevated，每月可以访问 50 万到 200 万条推文。现在，属于这一使用类别的应用程序制造商将不得不订阅企业计划。

一些试图订阅新的基本层的开发人员发现他们已经达到了极限。

当 Twitter 上个月决定关闭其免费 API 访问时，TechCrunch 报道称，许多研究人员和学者担心此举将阻碍学生项目和通过数据实现的平台透明前景。

Twitter 的新公告提到，它正在“寻找新的方式”为学术界服务，但没有提供任何关于潜在解决方案的信息。该公司继续表示，尽管他们试图定义学术使用层，但研究人员可以订阅免费、基础和企业层。免费和基础层对学者来说可能没用，而企业层对预算有限的项目来说可能太贵了。

在过去的几个月里，Twitter 的举措排斥了开发者社区。去年，该公司关闭了几个与开发者相关的项目，包括用于应用发现的 Twitter 工具箱，其他几个项目也处于休眠状态。今年 1 月，该公司在没有任何明确沟通的情况下，将第三方客户拒之门外。后来，它悄悄地把开发者条款改成了屏蔽另类 Twitter 应用。
