特斯拉三个月内第二次下调 Model 3 和 Model Y 在中国的价格

原文：https://web.archive.org/web/https://techcrunch.com/2023/01/05/tesla-slashes-model-3-model-y-prices-in-china-for-the-second-time-in-three-months/

今天早些时候，特斯拉悄悄宣布了中国市场的新定价。Model 3 和 Model Y 的价格大幅下降，特斯拉最终公布了 Model S 和 Model Y 的高性能格子版的定价，这两款车此前都没有在市场上销售。

这是特斯拉在三个月内对低价车型的第二次降价。10 月下旬，汽车制造商宣布对 Model 3 和 Model Y 降价 9%。

Model 3 目前在 CNY 的售价为 22.99 万英镑(合 33415 美元)，低于 CNY 的 26.59 万英镑，比 CNY 低了 3.6 万英镑(合 5240 美元)。Model Y 目前在 CNY 的售价为 25.99 万英镑(合 37775 美元)，低于 CNY 的 28.89 万英镑，比 CNY 低了 2.9 万英镑(合 4220 美元)。Model S 格子将售价 789，900 英镑(114，809 美元)，Model X 格子将售价 879，900 英镑(127，890 美元)。

今天的声明是在该公司最近的财务报告几天后发布的，该报告显示全球汽车出货量低于预期。具体到中国，今天早些时候，中国乘用车协会(CPCA)报告称，11 月份特斯拉汽车出货量下降 44%，至 55，796 辆，原因是该汽车制造商在需求减少的情况下减少了工厂产量并降价。

CPCA 的报告显示，特斯拉的销量被两个竞争对手超过:比亚迪和通用汽车在中国的合资企业上汽通用五菱汽车有限公司。比亚迪的销量是特斯拉的四倍，达到 234，598 辆。

据该协会称，通用汽车在中国的合资企业上汽通用五菱汽车有限公司(SAIC-GM-Wuling Automobile Co .)生产的小型电动汽车销量也比特斯拉高出 53%。

尽管如此，特斯拉去年在中国还是实现了增长。CPCA 报告指出，与 2021 年相比，特斯拉上海工厂交付的汽车数量增加了 50%。

特斯拉上海工厂去年 12 月曾暂停生产，据报道，明年 1 月将减产。

特斯拉中国部门最近经历了一次重大的领导层变动。路透社报道两天前，特斯拉首席执行官埃隆·马斯克提拔汤姆·朱负责其美国组装厂以及北美和欧洲的销售业务。这使他成为特斯拉的第二高管，仅次于埃隆·马斯克。据路透社报道，目前还不清楚朱是否会保留目前的头衔和职务。
