Index Ventures 认为，新的创业公司将在经济低迷时期出现，并为此投入了 3 亿美元

原文：https://web.archive.org/web/https://techcrunch.com/2022/11/17/index-ventures-thinks-new-startups-will-emerge-in-the-downturn-and-is-putting-300m-behind-that-bet/

回到 2021 年 4 月 8 日，欧洲风险投资界为数不多的“原始黑帮”之一 Index Ventures 表示，这是一种“回到根源”它宣布推出一个新的 2 亿美元专用种子投资工具，名为“指数起源”

现在，如果你回想一下，这是在去年白热化的技术牛市期间，当时估值达到顶峰，初创公司很少需要融资。因此，Index 的新基金名称向该公司作为种子基金的起源致敬，因为过去它曾支持过 Robinhood、Figma、Deliveroo 和 Wise 等处于种子阶段的公司。

在过去的几年里——尽管有疫情，在某种程度上因为它——在创业的早期阶段，有很多竞争。但随着明年全球经济衰退的逼近，一个隐秘的“核冬天”以及乌克兰战争等外部因素，你可能会认为像 Index 这样的投资者会收手。

并非如此。或许回到了古老的观点，即最好的创业公司是反周期诞生的，Index 今天推出了第二只“原始基金”，这将是一只 3 亿美元的种子基金。是的，伙计们，这比去年的 Origin I 多了 1 亿美元。

凭借 Index Origin II，Index 目前投资于三只基金，总额达 32 亿美元。Index 的其他基金包括早期基金 Index Ventures Xi(9 亿美元)，以及成长型基金 Index Ventures Growth VI(20 亿美元)。这意味着 75%的指数初始投资是种子或首轮投资。

意料之中的是，新起源基金似乎也适应了更现代的环境，在这种环境下，初创公司的联合融资也可能来自完全不同的来源，如独立的普通合伙人、天使投资人和许多现任或离职的企业家。

Index 表示，它希望复制迪伦·菲尔德等企业家的成功，Index 为他开出了第一张支票。

例如，Index 寄希望于宏观经济低迷会产生下一个 Airbnb、Adyen、Slack、Skype、Google 或 Spotify——所有这些都是在更广泛的经济衰退期间诞生的。

因此，它计划投资于任何感兴趣的垂直领域和任何地理区域(主要是美国和欧洲，尽管它没有明确地局限于这些市场)。

我问，为什么要在指数风险投资的早期阶段双倍下注？

旧金山的指数合伙人尼娜·阿查德金(Nina Achadjian)通过电子邮件告诉我:“纵观我们作为早期投资者的经历，我们意识到需要一种不同类型的早期基金。企业家早就告诉我们，在种子阶段，他们要么选择拥有大量资源和庞大网络的成熟投资者，要么选择只关注种子阶段的种子基金。”

她说，这个想法是将这两种方法结合起来:“通过 Index Origin，我们希望让创始人能够获得两个世界的最佳利益——他们快速增长所需的资源，以及早期专业知识和实践方法。我们知道这需要一个村庄，这就是为什么我们在种子投资中采取合作的方式。我们主动引入种子基金、独立全科医生和天使投资人与我们共同投资，这样我们就可以一起为创业者提供尽可能好的支持网络和成功机会。”

但是，为什么要募集比前一个更大的基金呢？“Origin I 去年推出时，我们采取的策略引起了创始人的极大共鸣。自启动以来，我们已经投资了 32 家公司，我们决定募集一只新基金，并扩大规模，以建立这一势头，”Achadjian 说。

Index 是如何在这种“低迷”的环境下找到资金的？

驻伦敦的合伙人丹尼·里默(Danny Rimer)说(也是通过电子邮件):“指数完全是关于信念。由于把主要的事情作为主要的事情，我们在投资 crypto 和中国时采取了非常逆向的方法，因此，与我们的同行不同，我们没有在这些领域投资。此外，有限合伙人真正理解 Index Origin 的价值主张，它是一只为创业者提供两全其美的基金。

在欧洲和美国的种子期，Index 是否看到了更多的天使投资人和前企业家/经营者？

“我们看到越来越多有经验的天使投资人加入到各个地区的融资中，这是一件好事。建立一家公司需要不同的专业知识，拥有不同背景的天使是一个显著的优势。这就是为什么我们建立了 Origin II，作为一个高度合作的基金，它对种子基金、独立全科医生和天使投资人开放。

美国的早期环境如何？而在欧洲呢？你对明年的预测是什么？

里默补充说:“与投资速度大幅放缓的增长阶段不同，在早期阶段，我们在所有地区都看到了健康的活动。在关注领域方面，我们继续加倍关注我们的核心领域，包括游戏、市场、企业/云/SaaS 和垂直 SaaS、人工智能、安全、金融科技和开源。”

Index 是否计划用这只新基金进行任何加密投资？

里默:“我不排除这种可能性，但正如我最近写的那样，对我们来说，迄今为止我们在这个领域看到的大部分公司都不是我们会投资的。我们看到了区块链的本来面目:一项强大的新技术，但不是新的互联网。我们希望，鉴于过去一年这个领域发生的一切，我们将专注于那些希望为用户创造真正价值的公司，解决真正的痛点，而不是本质上的投机。”

在过去的几个月里，Index 在纽约开设了办事处，并在特拉维夫雇佣了一名新合伙人。
