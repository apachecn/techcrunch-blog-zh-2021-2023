Eden AI 统一了几家云厂商的 ML API | TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/04/27/eden-ai-unifies-ml-apis-from-several-cloud-vendors/

法国初创公司 Eden AI 已经筹集了 160 万美元的种子前融资(€150 万美元)，以建立一个编程接口，让你可以访问来自多家供应商的人工智能引擎。该公司统一了这些 API，并提供了一种简单的方法来测试、混合和匹配它们。

在过去的几年里，小型和大型云供应商一直在构建 API，这些 API 使用机器学习模型神奇地处理他们云基础设施中的数据。例如，您可以使用这些 API 从图像中检测和提取文本，识别对象，从文本块中提取关键字，将语音转换为文本，翻译文本等等。

然而，所有这些 API 并不完全等同。有些 API 比其他的好。有些 API 在一种语言中工作得很好，但在另一种语言中就不那么好了。一些 API 也更容易使用和实现。

Eden AI 正在构建 API 来管理它们。“我们最初是一家专注于数据科学和人工智能的咨询公司，”联合创始人兼首席执行官塔哈·泽姆莫里告诉我。“作为顾问，我们不能直接推荐最好的模型，因为我们必须先测试它们，”他后来补充道。

这家初创公司与人工智能引擎提供商建立了 20 个合作伙伴关系，如 Mindee、Dataleon、Deepgram、AssemblyAI、Rev.AI、Speechmatics 和 Lettria。它还兼容大型云提供商，如亚马逊网络服务、微软 Azure 和谷歌云。

“我们创建了一个覆盖层，作为单一入口。切换到另一个提供商就像改变 API 中的一个参数一样简单，”Zemmouri 说。

随着价格的演变，Eden AI 也提供了一些灵活性，因为你不再局限于单一的云供应商。在计费方面，您不必为每个提供商开设一个用户帐户。相反，你的 Eden AI 帐户允许你访问所有这些 API。

如果您已经在特定云上拥有帐户，您还可以添加自己的密钥来使用自己的帐户。Eden AI 的收费并不比你从云供应商那里得到的要高。创业公司从供应商那里拿提成。

接下来，该公司希望根据您的需求自动选择合适的 API。目前，客户必须在他们的 API 调用中定义提供者。但是你可以想象一个让 Eden AI 为你选择最佳路线的功能。

在总部位于巴黎的 accelerator 50 Partners 的支持下，该公司从几个商业天使那里筹集了种子前融资，如奥利维尔·波梅尔(Olivier Pomel)、尼古拉斯·德塞格内(Nicolas Dessaigne)、塞巴斯蒂安·帕尔(Sébastien Pahl)、朱利安·莱莫因(Julien Schmidt)、本杰明·法布尔(Benjamin Fabre)、劳伦特·勒图米(Laurent Letourmy)、让·巴普蒂斯特·阿维亚特(Sqreen)、乔治·戈麦斯(Georges Gomes)和托马斯·格兰奇(Thomas Grange)。
