中国增强现实公司 Rokid 完成 1.6 亿美元的 C 轮融资，开始全球扩张

中国初创公司 Rokid 成立八年来经历了几个阶段的转型，最近它筹集了一些新资金以维持运营。这家淡马锡支持的公司最初是一家智能扬声器制造商，当时垂直技术在 2015 年年中风靡中国，但近年来它更加关注增强现实。

本周，Rokid 表示，它已经获得了 1.6 亿美元的 C 轮融资，使其融资总额达到 3.78 亿美元。从 2018 年最后一次 1 亿美元的财务注入到最近一轮，我们没有听到 Rokid 的太多消息——至少从消费者方面来说。

虽然像高通支持的 Nreal 这样的新玩家已经出现，以时尚、轻便的 T2 眼镜吸引早期用户，但 Rokid 似乎一直忙于探索企业用例，如为汽车、石油和天然气以及其他传统行业的现场工人提供远程通信。例如，它的 X-Craft 耳机，可以防爆、防水、防尘，并带有 5G 和 GPS 功能。和许多在新冠肺炎疫情爆发时匆忙提供非接触式解决方案的硬件初创公司一样，Rokid 吹捧智能眼镜可以在两分钟内检测多达 200 人的温度 。

Rokid 拥有一支约 380 名员工的团队，该公司表示，将把新的收益用于研发和全球扩张，因此发达市场可能会期待更多 Rokid 的 B2B 产品。事实上，该公司刚刚聘请了一位能源行业的资深人士来领导其在 APAC 地区的销售。

该公司拒绝透露本轮投资者的姓名，只说他们是“相关生态系统的参与者”，这很可能是指客户和供应链合作伙伴。该公司拒绝透露收购后的估值。

文章已更新，澄清 Rokid 从两批 C 轮融资中筹集了总计 1.6 亿美元。
