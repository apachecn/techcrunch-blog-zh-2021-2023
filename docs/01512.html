贾斯汀·坎首次亮相 Fractal，这是一个基于索拉纳的游戏市场

原文：https://web.archive.org/web/https://techcrunch.com/2021/12/13/justin-kan-debuts-fractal-a-solana-based-marketplace-for-gaming-nfts/amp/

图片来源:分形

虽然 OpenSea 似乎已经赢得了 NFT 市场流量的最大份额，但竞争对手仍在努力找到自己的位置，并为未来做好准备，在未来，NFT 不仅仅意味着可交易的 jpg。

连续创业家贾斯汀·坎(Justin Kan)共同创立的新 NFT 市场分形(Fractal)T3 正寻求在一个新兴网络索拉纳·区块链(Solana Berkeley)上抓住这个机会。

该平台是围绕游戏和所谓的“玩即赚”游戏的机会构建的，允许用户购买虚拟形象和数字商品。该平台一直专注于在发布前扩大对现有加密游戏的支持，但 Kan 和 Fractal 团队希望将该平台变成区块链游戏的一种 Kickstarter，允许用户在预售中购买独家 NFT，帮助为游戏本身的创作提供资金。

Kan 和团队的其他成员希望在未来几周内公开推出分形平台。

像加密领域的许多其他网络一样，今年对索拉纳来说是一个好年景，尽管区块链的支持者比大多数人都有更好的一年，因为他们看到了与网络相关的货币价值从 2 美元以上飙升到今年的 250 美元以上(尽管货币在最近几个月出现了大幅回调)。该网络是对未来的赌注，以太坊网络的模块化和移动缓慢的中央基础无法跟上 web3 用户群的需求。支持者认为低费用高带宽结构是建设 NFT 项目的理想场所。

“我相信索拉纳是游戏的未来，”Kan 告诉 TechCrunch。

Kan 之前是 Twitch 的联合创始人，该公司以退出亚马逊而闻名。自关闭上一家初创公司 Atrium 以来，菅直人一直专注于他专注于创业的 YouTube 频道和播客，以及他与合伙人陈彦博(Robin Chan)共同创建的孵化器/投资基金山羊资本。Fractal 是这个孵化器的一个产品，Kan 希望通过加入联合创始人兼总裁，将他的创始人明星效应抛在脑后。Chan 还将与 Fractal 的其他联合创始人，CTO Mike Angell 和 CPO David Wurtz 一起担任业务发展主管。

Fractal 在一定程度上押注于监管在区块链未来的重要性，在那里，费用和上市摩擦将被更有利于消费者的区块链和缩放解决方案所消除。迄今为止，以太坊昂贵的上市成本是阻止垃圾邮件发送者用数以千计的低努力 NFT 充斥现有市场的少数因素之一。
