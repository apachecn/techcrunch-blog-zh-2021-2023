MoneyHash 获得 300 万美元，为非洲和中东地区的支付操作构建超级 API TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/02/14/moneyhash-gets-3m-to-build-a-super-api-for-payment-operations-in-africa-mideast/

大型商家，尤其是那些在多个地区经营的商家，必须支持多种支付方式。每个支付集成都需要内部技术工作和改造平台以适应他们的插件。

这是一个广泛的过程，一些大型科技公司建立了专门的团队，来管理它们所在市场中支付网关的不同连接、数据和周期。

在美国、英国和欧洲，支付协调平台如spreadely、 Zooz 和 Primer 通过统一的支付 API 代表商家处理大量繁重的工作。但在非洲和中东仍有一个未开发的机会，那里很少有公司给商户的支付带来透明度。

总部设在埃及和美国的 MoneyHash 就是这样一家公司，它刚刚从测试阶段脱颖而出，获得了 300 万美元的种子基金。该公司将自己描述为中东和非洲“支付协调和收入运营的第一个超级 API”

去年 6 月，投资者风险投资平台、Kepple Africa Ventures、LoftyInc Capital 和牵头的 COTU 风险投资公司宣布了六位数的融资，种子前轮融资是这笔融资的延伸。中东早期基金也领导了这一扩展，参与的有最初种子前一轮的先前支持者以及其他人，如 VentureSouq、VentureFriends、The Continent Venture Partners、女娲资本和 First Check Africa。天使投资者包括 NerdWallet 的陈永正、Jake Gibson 和 Belvo 的 Oriol Tintore。

当商家或公司的平台启动时，他们主要与一两家支付处理提供商合作。随着时间的推移，随着规模的扩大和在多个地区的扩张，这些商家会选择更多的支付提供商来满足他们不断增长的需求。

但是整合不同的支付平台绝非易事。除了运营效率低下和涉及的技术问题，内部技术团队可能需要 3 到 10 周才能完成集成。在非洲和中东，由于支付方式、货币和国家间孤立的差异，这些问题被放大了。

Nader Abdelrazik 、 Mustafa Eid 和Anisha Sekar——他们在微软、UpWork、NerdWallet 和 Sigfig 等公司工作了 30 多年，在 2020 年末创立了 MoneyHash。

MoneyHash 位于支付提供商之上，提供其基础设施作为公司产品后端的扩展。这种延伸成为他们与所在市场的整个支付生态系统的连接。

“超级 API 的理念是整合不同的支付账户，并在此基础上构建所有这些功能。MoneyHash 成为一站式产品，或者说是你放在产品中的支付栈，管理每个非洲和中东国家的所有这些不同的集成和结账体验，并将你的所有信息放在一个仪表板上，”首席执行官 Abdelrazik 在与 TechCrunch 的电话会议上说。

2021 年初，MoneyHash 在埃及起步，允许 17 家公司使用其沙盒环境与其 API 连接，并访问 Fawry、 Paymob 和 PayTabs 等支付网关。

MoneyHash 将在测试后接入中东和北非地区活跃的不同支付网关和处理器。一些包括 Checkout，Stripe，Ayden，Amazon Pay，Tap 和 ValU。

首席执行官表示，将随后与撒哈拉以南非洲(主要服务于尼日利亚、肯尼亚和南非)的支付提供商整合，如 Yoco、Paystack 和 Flutterwave，但没有透露何时在该地区推出该产品的具体细节。

MoneyHash 的客户来自不同行业:电子商务、旅行和旅游、汇款等。他们只需点击几下鼠标，就可以整合支付提供商，嵌入统一的结账系统，并在平台上访问交易路由、订阅管理和发票等微服务。

在 17 家免费测试其沙盒的公司中，有 5 家是活跃的付费用户。MoneyHash 每月向这些公司收取 150 美元至 1000 美元不等的费用，具体取决于它们连接的支付提供商的数量。该平台还收取 10 美分的交易费，随着支付量的增加而下降。

Abdelrazik 说 MoneyHash 计划成为 AWS。中东和非洲的支付。他说:“AWS 对云的贡献在于，它使公司可以轻松地在基础设施上构建尽可能多的服务，我们认为支付行业，特别是在新兴市场，非常分散，需要一个物有所值的 AWS，MoneyHash 正在做的就是与它连接，在不需要改变任何东西的情况下，根据需要构建尽可能多的服务。”

MoneyHash 在一份声明中表示，该公司将利用这笔资金推动其在中东和非洲的增长。该公司还计划扩大其团队——目前在美国、埃及、阿联酋、尼日利亚和欧洲部分地区有 15 个团队——并雇佣 中高级软件工程师。

“MoneyHash 由三位卓越的创始人领导，他们对支付有着深刻的了解，对产品导向的执行非常敏锐，”主要投资者 COTU 风险投资公司的创始人阿米尔·法尔哈在一份声明中说。“他们吸引了一个优秀的人才团队，在令人印象深刻的测试运行之后，他们的产品已经准备好进入黄金时段。我们很高兴能成为他们旅程的一部分。”
