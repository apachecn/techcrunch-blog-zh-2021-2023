目标是将星巴克的订单和退货添加到其路边取货服务| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/02/23/target-to-add-starbucks-orders-and-returns-to-its-curbside-pickup-service/

Target 正准备进行一系列改革，以使其名为 Drive Up 的路边接送服务对消费者更具吸引力。零售商周三宣布将开始测试一个选项，允许顾客将星巴克订单添加到他们的取件中，以及带回他们想要退货的商品的能力，以及其他变化。

Target 表示，新增加的服务尚未推出，而是计划在 2022 年秋季(第三季度)扩大免下车服务的一部分。

该公司告诉 TechCrunch，最初，这些改进将由商店员工进行试点，然后在选定的测试市场推广到目标客户。塔吉特百货的家乡明尼阿波利斯市场很可能是首批试水者之一。然而，Target 拒绝透露哪些城市或多少家商店将获得这些功能，也没有透露它预计这些变化何时会在更大范围内推广。

当可用时，客户将看到新的选项出现在 Target 移动应用程序中，该应用程序现在为 Target 的免下车服务提供支持。如今，顾客可以在这里下单购买路边外卖，然后在路上提醒商店。关于客户位置的实时通知可以与星巴克的订购选项结合起来，因为该公司将知道何时开始准备饮料订单。

塔吉特百货于 1999 年首次与星巴克合作，在其美国门店中至少有1300 多家星巴克咖啡馆，从长远来看这一特色的潜在规模。(总共，塔吉特在美国有超过 1900 家门店，一位发言人指出“绝大多数”现在包括一家星巴克咖啡馆。)

虽然在路边取货时喝杯咖啡是一项不错的福利，但不用下车就能管理退货的能力是一项极具竞争力的附加功能。目前，大多数零售商要求购物者进入商店，并访问客户服务柜台进行退货，或直接将商品寄回。亚马逊提供了各种目的地，购物者可以亲自带着退货前往，比如全食超市和美国柯尔百货公司。与此同时，沃尔玛在 2017 年推出了对店内在线退货的支持。然后在 2018 年秋季增加了对市场项目的支持。

除了这些变化之外，Target 表示，它将在更多类别中扩展其“备份项目”功能，包括美容和家居用品。这使得客户可以在第一选择不可用的情况下选择合适的替代品。

“我们的客人不断告诉我们，他们喜欢免下车服务的轻松和便利，并要求我们在服务中增加更多的目标体验。Target 的首席商店官 Mark Schindele 在一份声明中说:“增加星巴克点餐和轻松退货，同时扩大我们的备用商品选项，将让客人更快更方便地享受他们在 Target 购物的乐趣。”。“对我们当天服务的持续投资已经与我们的客人建立了信任和相关性，同时满足了他们的需求——无论他们选择如何购物，”他补充道。

“驱车到达”是塔吉特提供的几项当天服务之一，另外两项服务是传统的在线取货服务和专注于杂货的在线购物服务 Shipt，后者既可以在塔吉特的应用程序内、在 Target.com 上提供，也可以在自己的品牌网站和应用程序上提供，用户可以在非塔吉特商店购物。

在 Target 2021 年第三季度的收益中，该公司宣布，今年其当天服务合计增长了近 60%，高于去年 200%的增长。当然，后者是在冠状病毒疫情肆虐的时候，这给零售商的路边取货选择带来了巨大的推动。但是，尽管在疫情爆发期间，许多消费者第一次尝试路边取货，以避免在店内购物，但行业专家预测，即使顾客返回商店亲自购物，消费者的采用率仍将保持高位。

Target 自推出以来一直在为 Drive Up 添加新功能，包括支持像成人饮料提货这样的功能，支持“购物伙伴”(针对希望派别人去取订单的客户)；下单后“忘记了一些”附加产品；和其他人。该公司在 11 月指出，它还投资于资本项目，以增加 200 多家高容量商店的永久存储容量，以及提供临时存储区域以支持季节性高峰的灵活固定装置。
