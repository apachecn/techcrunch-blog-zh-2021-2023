Twitter 的网页游戏让隐私政策更容易理解

原文：https://web.archive.org/web/https://techcrunch.com/2022/05/11/twitter-launches-a-new-web-game-to-make-its-privacy-policy-easier-to-understand/

Twitter 推出了一款新的网页游戏，使其隐私政策更容易理解

Twitter 今天宣布推出了一款新的网络视频游戏，让用户更容易理解其隐私政策。这款名为 Data Dash 的游戏的目标是教育人们 Twitter 收集的信息，这些信息是如何使用的，以及用户对这些信息的控制。这家社交媒体巨头表示，该游戏旨在帮助用户学习如何“安全地浏览推特”

一旦你开始游戏，你将被要求选择你想玩的语言。之后，你可以选择一个角色。该游戏通过帮助一只名叫 Data 的狗安全地导航“隐私”，避开广告，避开垃圾短信和 Twitter 上的巨魔。每次你完成一个级别，你将了解更多关于 Twitter 的隐私政策，以及你可以做些什么来保证自己在平台上的安全。

简单的游戏设计是为了帮助玩家在每一关学到一些东西。这款游戏由 pixel 艺术家兼游戏开发商 Momo Pixel 打造。

该公司在一份声明中表示:“通过 Twitter Data Dash，我们希望鼓励世界各地更多的人在我们的服务上负责他们的个人信息，甚至可能在这个过程中获得一点乐趣。”。“透明度是我们方法的核心，我们希望帮助您了解我们收集的信息、信息的使用方式以及您可以使用的控制措施。”

Twitter 发布了这款游戏，作为其更广泛努力的一部分，以使其隐私政策更容易理解。这家社交媒体巨头已经修改了其隐私政策网站，减少了“法律术语”的内容，使其更容易理解。Twitter 还将该政策重新划分为三个主要类别:数据收集、数据使用和数据共享。对于 Twitter 如何个性化用户体验和他们看到的广告，也有一个更清晰的解释。新的隐私政策网站和游戏从今天开始提供九种语言版本:英语、西班牙语、法语、意大利语、德语、日语、韩语、葡萄牙语和俄语。

发布的前一天， Twitter 推出了一项新的“复制意大利面和重复内容”政策，以阐明该平台如何打击垃圾邮件和重复内容。这家社交媒体巨头于 2020 年 8 月在首次透露，它将限制 copypasta 推文的可见性，现在正在强调它认为是违规的行为，以及采取什么行动来限制此类违规行为的可见性。
