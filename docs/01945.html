谷歌将 Android 和(一些)Windows PCs 更紧密地结合在一起 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/01/05/google-brings-android-and-some-windows-pcs-closer-together/

谷歌拉近了安卓和(一些)Windows 电脑的距离

谷歌正在与英特尔、宏碁和惠普等公司合作，通过 Fast Pair 让你的手机与 Windows PC 更容易连接，通过 Nearby Share 在 Android 设备和 Windows PC 之间共享文件，设置蓝牙配件，并在两个计算生态系统之间同步短信。

这些新功能将在今年晚些时候出现在部分 Windows 个人电脑上，谷歌称之为“投资于更有帮助的方式，让你的设备更好地协同工作”的努力的一部分。

图片来源:谷歌

在过去的几年里，谷歌和微软都推出了许多举措，将 Android 设备和 Windows PCs 更紧密地结合在一起。这包括微软为其 Surface 手机开发自己的 Android 发射器，但更重要的是，微软的 My Phone on Windows 等应用程序，可以让你从个人电脑上打电话和发送短信，或者 Windows 的 Android 子系统，可以让你在 Windows 11 上运行 Android 应用程序，尽管这是与亚马逊而不是谷歌合作的。

就该公司今天宣布的新功能而言，谷歌也不喜欢微软成为其合作伙伴之一，所以我们在这里主要谈论的是一小部分制造商的新电脑上的一些预装软件，而不是 Windows 10 或 11 的新功能。
