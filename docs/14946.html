由山姆·奥特曼(Sam Altman)的联氨(肼)支持的 ValueBase 筹集了 160 万美元的种子资金

原文：https://web.archive.org/web/https://techcrunch.com/2023/02/01/valuebase-backed-by-sam-altmans-hydrazine-raises-1-6-million-seed-round/

OpenAI 首席执行官山姆·奥尔特曼认为人工智能可以帮助迎来“难以置信的丰富，”但他说他希望确保这样的丰富是共享的。为此，奥特曼接受了19 世纪政治经济学家亨利·乔治的理论，他在自己的一生中担心财富聚集在工业革命后的少数人手中。乔治认为，如果土地的经济价值平等地属于社会的所有成员，就可以实现更大的平等。

奥尔特曼同样认为，在一个工作创造的经济价值可能减少的世界里，土地税可以弥补所得税，并保证所有个人的资产随着土地——一种固定资产——的价值增加而增加。他也把钱花在了自己的嘴上，带领一家成立仅 6 个月的初创公司进行种子期投资，这代表着向同样的方向迈出了一步。

从表面上看，这套名为 ValueBase 的装备听起来相当平庸。它计划使用土地优先模型来创建大规模评估模型和财产估价。ValueBase 表示，通过将气象气球、航拍供应商和其他来源的数据输入其算法，它可以为市政财产税评估人员自动评估土地和建筑物，他们中的许多人仍然依赖纸笔，主要关注建筑物的特征，而不是这些建筑物所在的土地。

但 ValueBase 的联合创始人 Lars Doucet 说，这个想法是要改变模式。“如果你明白土地是价值的关键驱动因素之一，你就不一定需要爬进人们的窗户，搜查他们的房子，拍下所有这些侵入性的照片。如果你能首先对土地进行估价，你就能得到更准确的结果，因为你能更多地了解土地特征，包括到学校的距离、到工作地点的距离、街道噪音有多大、污染有多大，以及是否有好的风景。所有这些东西都非常清晰，无需进入物业，因此更容易计算。”

出于多种原因，这是一个聪明的方法。一方面，与市政财产税评估员合作可以帮助这些评估员更清楚地向房屋或建筑物的所有者解释为什么一处房产被赋予了特定的价值。随着许多税务员退休，ValueBase 对城市也变得更有价值。(大多数是美国中年白人男性)

ValueBase 的策略也是自私的。它可以访问关键数据，用于创建更多的商业应用，包括经纪公司和抵押贷款机构。

Doucet 坚持认为，不管怎样，ValueBase 都有更高的使命。它希望通过尽可能多地了解美国和其他国家的每个地块，为不同地方的公共工程铺平道路。“如果有人问，‘嘿，爱达荷州的所有土地值多少钱？’“你能告诉我包裹在哪里吗？”他说:“他们会第一个打电话给我们。”。

目前，很难想象有人会问这个问题，更不用说 ValueBase 了，它仍然只有四名员工，分布在北卡罗来纳州、弗吉尼亚州和德克萨斯州。

尽管如此，如果奥尔特曼关于我们共同面临的指数变化率是正确的，那可能会改变，他显然愿意打赌。

Altman 和商业伙伴 Ryan Cohen 经营的基金肼资本(肼 Capital)领投了 ValueBase 的 160 万美元融资。

GitHub 前首席执行官 Nat Friedman、连续创业者 Sahil Lavingia 和 Erik Torenberg 等人也加入了他们的行列。
