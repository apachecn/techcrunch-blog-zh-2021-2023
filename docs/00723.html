脸书新的学术研究 API 提前开放| TechCrunch

原文：https://web.archive.org/web/http://techcrunch.com/2021/11/15/facebooks-researcher-api-meta-academic-research/

本周，几个学术研究团队将获得一个来自脸书的新工具，该工具旨在聚合全球最大社交网络上近乎通用的实时数据。

谈到谁可以访问脸书的数据以及如何访问，现在被称为 Meta 的公司仍然感受到 2018 年剑桥分析公司丑闻的影响，在该丑闻中，一家政治咨询公司收集了数百万不知情的脸书用户的个人数据，以建立潜在选民的详细资料。该公司在随后的三年里关闭了数千个 API，现在才开始恢复学术研究的广泛访问。

TechCrunch 预览了脸书新的学术研究 API，并与脸书产品经理 Kiran Jagadeesh 进行了交谈，他与脸书开放研究&透明(FORT) 团队一起领导了这个项目。

“这只是一个开始，”Jagadeesh 告诉 TechCrunch，将研究人员 API 描述为它最终希望提供的工具包的测试版。今年在 F8 首次发布的 API，是基于 Python 的，运行在开源笔记本接口 JupyterLab 中

鉴于脸书过去的许多隐私问题，新的研究人员 API 提供了一些初步的警告。首先，API 将只通过一个只接受邀请的系统提供给一小部分知名的学术研究人员。该公司计划在 2022 年 2 月将访问范围扩大到最初的测试组之外，将试验的反馈纳入更广泛的面向所有学者的发布中。

另一个预防措施:研究人员 API 运行在一个非常受控的环境中，Jagadeesh 称之为“数字洁净室”。能够访问 API 的学术研究人员可以通过脸书的 VPN 进入环境，收集数据和处理数据，但原始数据不能导出——只能导出分析结果。

这个想法是为了保护用户隐私，防止任何分析过的数据被重新识别，但这一限制可能会让该公司的一些批评者产生误解，因为研究人员 API 收集的所有公共数据已经在那里浮动，但很难用脸书现有的工具进行汇总和分析。

在发布时，该 API 将提供对四桶实时脸书数据的访问:页面、群组、事件和帖子。在每种情况下，该工具将只从公共数据中提取，并且最初只从美国和欧盟内部的来源中提取。对于组、事件和页面，至少需要一名管理员位于受支持的国家/地区，以便通过 API 提供该数据。

通过该工具，研究人员可以使用情绪分析等方法分析大量原始文本，情绪分析可以跟踪人们在特定主题的演讲中表达的效价和情绪。除了包含大多数可用数据的基于文本的帖子，研究人员还可以访问相关信息，如群组和页面描述、创建日期以及帖子反应。

原始图像等多媒体数据不会被包括在内，评论或用户人口统计数据(年龄，性别等)也不会被包括在内。).API 也不会从 Instagram 收集任何数据，尽管 Jagadeesh 认识到该平台对研究人员非常有价值，该团队正在探索让 Instagram 数据可用的方法。

FORT 团队希望与学术研究人员密切合作，开发和构建当前的工具，Jagadeesh 将其描述为一项正在进行的工作。虽然 Meta 表示其最初的学术合作伙伴尚未确定，但该公司已经邀请了来自全球 23 个学术机构的研究人员进行考察。

11 月 15 日星期一，已经完成团队入职流程并同意其隐私政策的研究人员被授予访问权限。脸书要求任何访问研究的人同意隐私限制，包括不重新识别数据中的特定个人。

研究 API 目前只对少数学术机构开放，但 FORT 团队计划探索向包括记者在内的其他群体开放。目标是创建一个公共路线图，让研究人员和记者透明地了解团队的工作目标。

该公司在研究界有许多建立信任的工作要做。今年 8 月，脸书切断了与 NYU 网络安全促进民主项目相关的两名著名研究人员的广告数据访问，引发了许多学者和监管机构的指责。这些研究人员专注于通过一个名为“广告观察者”的选择加入浏览器工具来追踪错误信息和政治广告。9 月，脸书·T4 向一群被称为“社会科学一号”的精英研究人员道歉，因为他们提供了不完整的数据——这个错误破坏了他们几个月的工作和分析。
