这款卓越 2 是一款很棒的平板电脑，可以用来阅读长篇文档、给文档加注释、做笔记、画草图……但不适合用来进行全面的写作。至少，直到他们推出了这款伟大的键盘对开本，这款平板电脑才变成了一个非常引人注目的极简主义写作站——如果你不介意稍微溢价的话。

这款新保护套与 Surface、iPad 甚至 Boox Tab X 等其他电子纸设备的保护套没有什么不同。它有一个皮革或人造革外壳(有黑色或牛奶巧克力棕色)，然后是一种三层设计，盖子在上面，平板电脑在中间，键盘藏在下面，直到被召唤出来。

如果你熟悉其他键盘对开型的情况下，这不会是一个很大的惊喜，但我会说，这是非常好的完成和制作。等等，你说，你要对一个封面做一个完整的回顾？嗯，这是一种独特的，非凡的是一个很酷的设备，没有很多像它在那里！我知道不止我一个人在寻找这样的东西。毕竟，他们卖出了一百万台这种东西。

无论如何，如果你只是想做笔记或纵向阅读，对开本可以作为普通封面。但是，一旦你翻开盖子，你可以放下它，抓住两边的几个标签，把屏幕从下面的按键中取出来。它可以折叠起来，用几块磁铁吸住，你就可以开始打字了。

没有 Wi-Fi 或蓝牙来连接或配置；该设备和外壳具有一个小小的接触接口，可以激活键盘并为其供电。简单多了，但当然这意味着你不能把键盘放在桌子上，把平板电脑支在某个地方。

在敲击一个键和字母出现之间基本上没有延迟。这是我在非 LCD 设备中看到的最好的，我在这里为卓越的工作鼓掌。

键盘绝对是你倾向于找到的平板电脑的伴侣的紧凑种类；字母键具有全尺寸布局(与正常布局的大小和间距相同)，但是通过使数字键变得矮胖和修饰键变窄来节省空间。

一般来说这很好，虽然我更喜欢稍微修改一下，让长而奇怪的退格键变大，并且不需要大写锁定(谁需要它？).也没有 delete 键，尽管 shift-backspace 也有同样的功能。

chiclet keys 没有太多的旅行或游戏，但足够舒适；没有笔记本电脑好，但比我用过的许多键盘套要好，当然也比任何屏幕键盘好得多。(老实说，我没有测试过很多这些东西，因为一般来说我看不到它的效用——就我个人而言，我用平板电脑来消费，而不是创造。当我真的需要写东西的时候，我有一个很好的符合人体工程学的机械键盘。)

更新:卓越已经给键盘对开本的早期买家发了邮件，提醒他们“我们发现了一个质量问题，影响了第一批打字对开本的塑料书脊。我们暂停发货，同时进行调查，并找到原因。在我们看到的少数案例中，这个问题只是表面现象，不会影响使用中的对开字体。我没有注意到任何异常，未来的订单应该没有这个问题(无论它是什么)，但它在这里记录在案。

最少的编辑——也许太少了

通常这个小菜单根本不存在，只有几行。图片来源:德文·柯迪威/ TechCrunch

键盘将提供一个新的以文本为中心的界面，虽然你可以用屏幕键盘(可维修)来使用它，但物理键盘显然是一个更好的选择。

通过点击左侧下拉菜单中的文本 T，一个文本列会出现在中间——或者更确切地说，不会出现。它在那里，但几乎没有任何标记来表明它。为了保持卓越的最小美学，除了一个小的位置标记之外，什么也没有，指示你的文本将出现在哪里。

这个想法是让你专注于文本，提供一些简单的格式选项。事实上，太少了。

原则上来说，编辑器是可以工作的，我喜欢它很像一台填充空白页的打字机，而不是文字处理器。拿起手写笔在这里或那里草草记下一个笔记的能力——“更多关于这个”,或者只是一个连接两段的箭头——非常棒。手写和草图是附在它们旁边的文字上的，所以当你上下滚动的时候它们会跟着一起移动。另一方面，据我所知，你不能在现有的图纸旁边创建一个文本字段——它会在下面开始一个新的段落。

虽然这种简单性令人钦佩，我也能够很容易地起草一篇文章，但我立即想要几样没有的东西。

首先，文本有点太小，可读性受到影响，尤其是在没有背光的情况下。从两英尺远的地方看，它看起来有点轻，事实上当你靠近时，渲染是有斑点的——这是快速刷新时间的结果。

图片来源:德文·寇维兹/ TechCrunch

这并没有困扰我太多，但如果有一对夫妇的其他大小，我会撞了一个只是为了增加清晰度。粗体和斜体可能不错，但必要时我也可以不用它们。

如果能够在每个文档的层次上控制布局和格式，那就更好了。唯一的选择——也是一个不错的选择——是将文本放在屏幕的中间四分之二处，并在两侧留出草图空间。老实说，这些都很好，但我只是喜欢一个更宽和更窄的选项，也许是一个调整和对齐设置，以及将文本列放在其他地方的能力。

我知道重点是极简主义，在这里我要求所有这些设置。但这并不是说所有这些标记会塞满你的屏幕——这是一个设置好就忘了的事情，因为不是每个人都有相同的偏好和需求。手写和草图笔记本有很多模板和选项，我希望基于文本的笔记本也有同样的模板和选项。

通过触摸屏编辑是好的，但是卓越应该从这个领域的其他人(或用户)那里学习一些 UI 技巧。选择一个词很容易，但选择一个句子却不容易。在文本区域点击是有反应的，但是在草图区域不做任何事情(比如转到最近的行的末尾)。这里有几个这样的小失误和缺失，可以并且可能会在即将到来的软件更新中修复。

我问了一些事情，卓越说，“我们计划在未来几个月增加更多功能，完善和改进体验，”但没有分享路线图。我相信他们会做得正确，但是现在设计得还很简单。

综合的，有点

图片来源:非同凡响

卓越坚持一个或多或少包含和非干扰环境的另一个方面是有限的集成。我想我们应该感到庆幸:起初根本没有！但现在你可以用 Dropbox、Google Drive 或 OneDrive 同步文件，尽管这些都是其第一方连接功能的次要功能，每月收费 3 美元(至少第一年是免费的)。这包括在桌面和移动应用程序以及平板电脑本身之间同步你的文件——尽管你一次只能有一个活动的平板电脑，我希望他们能改变这一点。

在大多数情况下，同步工作正常，偶尔在处理多个文档时有点滞后(像任何云服务一样)，但一般来说，你可以指望它拥有最新版本的东西，你的添加和草图都是最新的。

显然，卓越想让你留在他们的生态系统中，但是，没有机会与 Simplenote、Evernote、Bear 等服务同步。我知道其中一些可能是包罗万象的平台，并不真正与卓越的使命相吻合，但老实说，如果我可以把 Simplenote 放在上面，我会使用卓越的 10 倍以上。

事实上，我仍然可以在我的工作流程中为它找到一个位置，特别是在编写不太适合我的其他写作平台的较长作品时:正在进行的长篇故事，正在进行的一本书的大纲，诸如此类的事情。

folio 的价格为 200 美元，是 300 美元的三分之二(尽管后者近年来确实下降了)。它的确制作精良，键盘也很漂亮——而且它无疑增加了设备的用途。让我们面对现实:当在电子纸设备上写作时，我们的选择是有限的。这是一个很好的选择，即使它可能对某些人来说太简单了，任何写长格式文章的人都可能会选择一个“真正的”键盘。

它离我的数码打字机的梦想越来越近了，总价格 500 美元，考虑到如今一台“好”笔记本电脑的价格，这个价格很高，但并不疯狂。如果像我一样，你需要一个环境转换来真正专注于阅读或写作，这 500 美元可能是值得的。
