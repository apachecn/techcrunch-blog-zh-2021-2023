Meta 因涉嫌违反反垄断法在南非面临起诉

原文：https://web.archive.org/web/https://techcrunch.com/2022/03/15/meta-faces-prosecution-in-south-africa-for-alleged-antitrust-breach/amp/

图片来源:布莱斯·德宾/ TechCrunch

Meta 在南非面临起诉，此前该国竞争监管机构竞争委员会发现其阻止政府初创公司 GovChat 和#LetsTalk 使用其 WhatsApp 商业 API 的意图不具竞争力。

该监管机构自 2021 年 3 月以来一直在调查对这家初创公司不公平行为的指控，并将 Meta(原脸书)提交给竞争法庭起诉，该法庭裁决限制性惯例和滥用支配地位的投诉。

在其提交的中，监管机构建议 Meta 支付“最高罚款”——美国公司当地营业额 10%的罚款。

该委员会声称，Meta 威胁要在 2020 年 7 月左右阻止 GovChat 和#LetsTalk 使用其 WhatsApp 商业 API。它补充说，Meta 对初创公司的数据使用施加了不公平的限制，限制了其创新和开发可能与 Meta 的产品竞争的新产品和服务的能力。

“……监管 WhatsApp Business API 的条款和条件旨在保护和隔离脸书免受潜在竞争，例如 GovChat 带来的潜在竞争以及它能够获得的大量数据，这些数据使它能够开发新的服务和产品，”监管机构在一份声明中说。

GovChat 由南非政府于 2018 年推出，是一个公民参与平台，使用 WhatsApp 商业 API 来促进实时沟通。根据政府数据，它现在有 870 万活跃用户，处理了超过 5.82 亿条消息。

除了作为提醒和投诉公民问题(如道路上的坑洞)的来源，GovChat 还被政府用于处理社会保障申请，包括在新冠肺炎疫情期间的遇险支持。通过 GovChat 平台提交的申请超过 1330 万份。

提起诉讼的几天前，包括南非在内的五个非洲国家的竞争监管机构签署了一份谅解备忘录，除其他议程外，该备忘录将促进合作行动，克服限制非洲数字平台出现和扩张的障碍。该协议的其他缔约方是埃及、肯尼亚、毛里求斯和尼日利亚。

Meta 在发给 TechCrunch 的一份声明中表示，“没有证据表明 WhatsApp 试图将任何公司排除在市场之外或从事反竞争行为”，同时表示 GovChat 是一家私营企业，它违反了设定的条款，“在没有经过我们的入职流程的情况下，就将组织注册到 WhatsApp API。所有希望使用我们服务的组织都需要这样做，这意味着我们知道谁在使用我们的服务，并且组织同意我们的隐私做法……WhatsApp 将捍卫其采取一切合理措施保护其专有 WhatsApp Business API 和全球用户利益的权利，”Meta 在声明中表示。

此外，WhatsApp 在南非的发言人表示，“WhatsApp 有助于为人们提供来自可信来源的重要信息，我们知道这项服务在连接南非公民和他们的政府方面发挥的作用。这就是为什么我们希望按照国际公认的监管标准与 GovChat 合作来提供这项服务。”

“然而，GovChat 一再拒绝遵守我们旨在保护公民及其信息的政策，宁愿将其自身的商业利益置于公众之上。我们将继续捍卫 WhatsApp 免受滥用，保护我们的用户，”这位发言人说。

在全球范围内，Meta 因可能的反竞争行为受到了各种审查。就在上周，欧盟委员会(European Commission)启动了一项正式的反垄断调查，以评估 Meta 与谷歌之间就在线展示广告服务达成的代号为“绝地蓝”(Jedi-Blue)的协议是否违反了欧盟竞争规则。

欧盟委员会指出，2018 年 9 月的 Jedi-Blue 协议寻求 Meta 的观众网络参与谷歌的公开招标计划，此举可能会排除其他广告技术服务提供商，“扭曲在线显示广告市场的竞争，损害出版商，最终损害消费者”。

与此同时，美国美国联邦贸易委员会正在起诉 Meta 非法垄断，指控这家社交媒体巨头通过反竞争行为非法维持其社交网络垄断地位，包括近十年前收购 Instagram 和 WhatsApp。
