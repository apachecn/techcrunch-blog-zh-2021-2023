约翰·卡马克 AGI 创业公司融资 2000 万美元

原文：https://web.archive.org/web/https://techcrunch.com/2022/08/19/john-carmack-agi-keen-raises-20-million-from-sequoia-nat-friedman-and-others/

游戏开发者约翰·卡马克是 id Software 的联合创始人，曾担任 Oculus 的首席技术官，他正致力于一项新的风险投资，并且已经吸引了一些大牌的投资。

卡马克周五表示，他的新人工通用智能初创公司 Keen Technologies(可能是指 id 的" Commander Keen ")已经在前 GitHub 首席执行官 Nat Friedman 和 Cue 创始人 Daniel Gross 的一轮融资中筹集了 2000 万美元。他说，Stripe 联合创始人兼首席执行官帕特里克·科利森、Shopify 联合创始人兼首席执行官鸢·吕特克以及著名的风险投资基金红杉也在这轮投资之列。

“对我来说，这显然是一种专注的努力。我自己可以开一张 2000 万美元的支票，但知道别人的钱岌岌可危，会让我产生更强的纪律性和决心。我已经谈论了一段时间，我很高兴 Nat 推动了我，”他在一条推特上说。

AGI 是“人工通用智能”的缩写，是一种人工智能系统，理论上可以执行人类可以执行的任何任务。这与今天的人工智能系统相反，后者是为狭窄、特定的应用程序设计的，如生成艺术、驾驶汽车和玩视频游戏。一些乐观主义者认为，由于新兴的算法技术和日益强大的计算机硬件，AGI 可能在下个世纪实现。其他人，包括卡马克现任雇主的首席科学家扬·勒昆，都对 AGI 将成为现实的想法表示怀疑。

就卡马克而言，他似乎坚定地站在乐观主义者 阵营。在最近的播客中，他对麻省理工学院的研究科学家 Lex Fridman 说，他相信人工智能有一天会达到类似于人类或生物的行为，能够成为“通用远程工作者”

“我不相信快速起飞……(但)我认为，我们将从开始看到(人工智能驱动的)东西可信地看起来像动物的那一点出发，”卡马克说。“我认为动物的智力比许多人想象的更接近人类的智力……这只是大脑如何发展的平滑光谱，大脑皮层和不同事物的缩放在那里进行。”

卡马克继续指出，他对 AGI 的兴趣部分是受山姆·奥特曼的刺激，他说，山姆·奥特曼试图招募他加入 OpenAI。(奥特曼是总部位于旧金山的高知名度人工智能研究实验室 OpenAI 的原始联合创始人之一。)奥特曼之前曾表示，他预计人工智能有一天会超越人类智能，他一直直言不讳地表达了他对 AGI 的担忧——特别是如果建造或使用不当，它可能会对社会造成破坏。

卡马克仍将继续就虚拟现实问题咨询 Meta，但他说，他与社交巨头的工作将只消耗他 20%的时间。卡马克于 2019 年 11 月转到 Oculus 的“咨询 CTO”职位，以投入更多时间研究人工智能。

卡马克并不是最近发现一家雄心勃勃的人工智能初创公司的唯一传奇技术专家。大约一年来，LinkedIn 联合创始人雷德·霍夫曼和 DeepMind 创始成员穆斯塔法·苏莱曼(Mustafa Suleyman)一直在打造拐点 AI ，这是一家寻求将人类互动翻译成计算机可以理解的语言的初创公司。OpenAI 前研究副总裁 Dario Amodei 是 Anthropic 的联合创始人之一，该公司在 2021 年的大部分时间里以及今年一直在研究像 OpenAI 的 GPT-3 这样的人工智能文本生成系统的行为。
