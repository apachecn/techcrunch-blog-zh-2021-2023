小生境密码技术可以改变 web3 中的隐私

原文：https://web.archive.org/web/https://techcrunch.com/2022/07/18/crypto-blockchain-web3-privacy-cryptography-fully-homomorphic-encryption-startup-sunscreen/

隐私是加密的核心问题。一旦你知道一个加密钱包地址对应于某一个人，你就可以跟踪这个人通过他们的钱包在任何公共区块链上进行的所有交易，包括比特币和以太坊。

一些密码学家认为,解决 crypto 隐私问题的方法在于零知识证明(ZKPs ),这种技术允许在不共享底层数据的情况下在区块链上验证交易。虽然零知识证明确实可以提高一些最流行的区块链的隐私性和可扩展性，但它们远不是唯一可以加速 web3 进步的加密方法。

Ravital Solomon 是防晒霜的联合创始人兼首席执行官，他认为全同态加密(FHE)在加强 web3 隐私方面更有潜力。所罗门解释说，这项技术允许个人在不需要解密的情况下对加密数据进行计算。我想到的一个明显的使用案例是金融机构，它们可以使用 FHE 来分割交易数据，以检测潜在的欺诈行为，同时为客户维护比目前标准更多的隐私。

“就零知识证明能为 web 提供什么而言，它确实令人兴奋。对于游戏和身份来说，有所有这些令人兴奋的应用，但是零知识证明不一定是加密或隐私的全部。我把它想象成你要去吃饭。现在，零知识证明被描述为，“这是你可能需要吃下整顿饭的唯一器具，”然而我不认为那是必然的。完全同态加密是一种不同的隐私技术，你可以把它看作是对零知识证明的补充，”所罗门说，她在牛津大学获得了数学和理论计算机科学硕士学位。

防晒霜创始人兼首席执行官 Ravital Solomon。图片来源:防晒霜

零知识证明的部分挑战是使用它们可能很昂贵。Solomon 解释说，许多用 ZKPs 构建的项目希望用户拥有远远超过普通消费者在家中拥有的计算能力，以便按预期执行。这就是为什么她决定从事防晒霜的工作，这是一家旨在推动 FHE 技术发展的初创公司。

“用户不需要这些疯狂的机器，甚至可能是服务器——忘掉笔记本电脑吧——对用户来说，全同态加密(比 ZKPs)要轻量得多。因此，你仍然可以在不同类型的交易和计算中获得隐私，但你可以以合理的成本获得它，即使是在笔记本电脑上，”所罗门说。

去年，所罗门与 threshold cryptography 的资深人士、Y Combinator 支持的 NuCypher 公司的麦克莱·威尔基森(MacLane Wilkison)共同创立了 Sunscreen，所罗门最初有想法将 FHE 技术推向市场时，就在这家公司工作。现在是防晒霜顾问的威尔基森鼓励所罗门围绕她的想法创办一家公司。

如今，Sunscreen 的三名全职员工(包括 Solomon)已经开发了一个编译器，让工程师更容易构建 FHE 程序。该公司还宣布启动一项赠款计划，将拨款高达 50 万美元(每个项目 5 万至 7.5 万美元)，资助来自利用 FHE 的开发者、学者和研究人员的提案，Solomon 说。

为了资助这些项目，Sunscreen 刚刚在由 Polychain Capital 牵头的种子轮融资中筹集了 465 万美元，参与方包括 Northzone、比特币基地风险投资公司、dao5 和 Naval Ravikant。所罗门的一些前 NuCypher 同事也作为天使投资者参与了这轮投资，包括熵创始人 Tux Pacific 。所罗门说，这次提价将补充此前提高的 57 万美元的播种前防晒霜。

我问 Solomon，如果 FHE 如此有前途，那么它为什么没有在 web3 中变得更受欢迎，而 web 3 似乎与它的属性非常契合。

Solomon 解释说，编写 FHE 程序非常困难，因为这个领域仍处于萌芽状态，这是为什么 Sunscreen 正在开发开发人员工具来简化相关流程的部分原因。

为了让这项技术在隐私领域发挥作用，Solomon 认为需要一种新的第一层区块链，她计划长期与防晒霜一起研究，她说这将成为通往以太坊的桥梁。Solomon 解释说，通过一个新的链，技术专家可以在该链和 FileCoin 等其他协议之间建立集成，以对用户隐藏底层数据，并可以以更低的成本同时利用 ZKPs 和 FHE。

FHE 是网格加密的一个子集，Solomon 说，即使在加密领域，这个领域仍然被认为是利基。

“我怀疑完全同态加密只是最近才变得足够有效，足以在过去三到四年的用例中发挥作用，”所罗门说。“也许你和我都认为零知识证明是小众的，而格密码[更是如此]。我想说，这只是一个非常小的群体，他们甚至有专门知识来坐下来，将这些不同的领域结合起来解决问题。在 web3 中，我认为空间正在增长，但它肯定落后于零知识证明五年。”
