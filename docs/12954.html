脸书将在美国的脸书约会上测试年龄验证技术

原文：https://web.archive.org/web/https://techcrunch.com/2022/12/05/facebook-to-now-test-age-verification-tech-on-facebook-dating-in-the-u-s/

Meta 正在将其年龄验证技术引入其另一款产品脸书约会。脸书今天早上宣布，尽管这项服务还没有成为 Tinder 的主要竞争对手，但它现在将成为进一步试验新时代验证技术的测试场，包括视频自拍和身份证上传。该公司将扩大与在线年龄验证提供商 Yoti 的合作关系，开始验证脸书约会用户的年龄，当它怀疑他们谎报了真实年龄时。

随着监管机构对在线公司施加越来越大的压力，要求它们制定轻微的保护措施，Meta 一直在投资各种依赖人工智能的年龄识别技术。虽然 Meta 尚未完全详细说明它会寻找什么样的信号来帮助它确定某人的年龄，但它此前曾表示，它可能会扫描用户的生日帖子等内容，因为朋友们经常在回复中评论发帖人的真实年龄。在去年提示用户开始提供他们的生日后，Meta 已经在 Instagram 上测试这项技术。今年夏天，它开始要求一些美国用户提供他们的身份证或视频自拍，如果他们说他们是 18 岁或以上。

现在，同样的测试将扩展到脸书约会，这是脸书产品的一个领域，有未成年人参与可能会更有问题。

“我们致力于确保人们通过我们的技术获得适合年龄的体验，作为这项工作的一部分，我们一直在测试年龄验证工具，并使用年龄检测技术来阻止 18 岁以下的人访问成人体验，”在一篇关于新测试的博客文章中解释道。

图片来源:脸书

与 Instagram 类似，当脸书的人工智能检测到有人可能未满 18 岁，试图使用脸书约会时，它会向他们发送提示，以验证他们的年龄。

用户可以选择拍摄视频自拍，记录他们的面部，然后将视频中的静态图像与 Yoti 分享，以验证年龄。Yoti 使用的技术是根据面部特征估计用户的年龄。之后，图像被删除。该公司表示，或者，如果用户愿意，他们可以选择通过将他们的 ID 上传到脸书来验证他们的年龄，在那里他们的 ID 会被加密然后存储起来。该公司指出，用户可以管理他们的 ID 保存多长时间，尽管它自己的帮助文件指出，它可以将 ID 保存长达 1 年。

图片来源:脸书

Meta 表示，其 Instagram 上的年龄验证工具已经产生了影响。自 6 月份测试以来，该公司发现，当试图将出生日期修改为 18 岁以上时，可能会有大约四倍多的人完成年龄验证要求。这导致“数十万”用户被归入相应的年龄组。它还声称已经阻止了 96%试图使用这些工具将自己的生日修改为 18 岁以上的青少年。据称，81%的人选择了 Yoti 的视频自拍来完成这一过程。

虽然年龄验证技术可能有助于解决未成年人试图使用为成年人设计的产品的问题，但它也引发了其他问题。隐私和消费者权益组织警告说，所使用的方法，尤其是那些涉及 ID 上传的方法，具有侵犯性和风险性。法国数据保护机构 CNIL 也警告说，当今的许多系统很容易被绕过。正如在脸书看到的，隐私风险已经很明显，因为该公司将在上传后无限期存储用户的 id。

该公司表示，其年龄验证工具的脸书约会测试将在美国上线，但未来将把该工具扩展到其他全球市场。
