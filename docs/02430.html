汤姆·布拉迪炙手可热的名人 NFT 创业公司从硅谷顶级加密投资者那里获得 1.7 亿美元

由运动员汤姆·布拉迪共同创立的 NFT 机构 Autograph 拥有深厚的明星影响力，该机构已经从加密投资者那里获得了新的资金，这些投资者希望该平台能够将新一代名人及其粉丝带入加密收藏品的行列。

这家初创公司告诉 TechCrunch，它已经完成了 1.7 亿美元的 B 轮融资，由 Andreessen Horowitz 和 Kleiner Perkins 牵头，Katie Haun 的新基金和光速合伙人 Nicole Quinn 参与。此前，01A 和 Velvet Sea Ventures 共同牵头的 A 轮融资已于今年 7 月结束。此次加薪为董事会增加了三名新成员，包括 Haun、a16z 的 Arianna Simpson 和 Kleiner Perkins 的 Ilya Fushman。

他们加入了一个庞大的董事会，其中有一些大牌明星，包括布雷迪、苹果公司的艾迪·库伊、FTX 公司的萨姆·班克曼-弗里德和艺术家阿贝尔·特斯法伊，也就是威肯。

签名会本质上是明星运动员和艺人的中介机构，他们希望在极其嘈杂的 NFT 世界中找到自己的一席之地。自从去年年初秘密收藏品受欢迎程度飙升以来，名人 NFT 滴就成了一个大杂烩。虽然有些人带着回报粉丝的深思熟虑的项目进入这个领域，但也有许多引起嘲笑的现金攫取。

对于有眼光的名人来说，在不损失一些文化资本的情况下，Crypto 通常是一个艰难的地方。马特·达蒙这个月因为出现在一个毫无新意的加密广告活动中而受到了相当多的嘲笑。本周早些时候，金·卡戴珊和弗洛伊德·梅威瑟被心怀不满的投资者提起诉讼，这些投资者支持两人早先签署的一个令牌。

虽然 NFT 独角兽 Dapper Labs 已经与包括 NFL 球员协会和 NBA 球员协会在内的少数几个美国体育联盟球员协会建立了合作伙伴关系，但 Autograph 似乎更专注于运动员个人以及他们所在球队或联盟之外围绕他们建立的个人崇拜。Autograph 有几个早期合作伙伴，主要是在体育界，但这些名字代表了他们各自运动项目中最著名的一些运动员，包括布雷迪、老虎伍兹、西蒙尼·比尔斯、德瑞克·基特、奥米·大阪、尤塞恩·博尔特、韦恩·格雷兹基和托尼·霍克。

Autograph 的目标似乎是建立一个对空间的冲击，在这个空间中，三线演员可以在一个更加高度策划的环境中涉足秘密影响。

7 月，Autograph 宣布与 DraftKings 合作，尽管这家初创公司告诉 TechCrunch，它一直在与许多市场合作。Autograph 主要在 Polygon 区块链上提供其 NFTs，它利用以太坊的基础设施，但每次交易使用的能源少得多，这一事实对名人来说可能很重要，他们可能会小心翼翼地向区块链技术的环境批评敞开心扉。

在体育世界之外，亲笔签名已经与威肯、Slam 杂志和官方授权的 NFTs 合作了《电锯惊魂》。
