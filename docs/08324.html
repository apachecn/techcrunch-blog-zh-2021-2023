Snap 推出网络版 Snapchat

原文：https://web.archive.org/web/https://techcrunch.com/2022/07/18/snap-launches-snapchat-for-web/

Snap 今天宣布，它将推出 Snapchat for Web，让用户可以从他们的桌面发送快照并通过视频电话与朋友聊天。首先，Snapchat for Web 将面向美国、英国、加拿大、澳大利亚和新西兰的 Snapchat+ 用户开放。在首次推出后，该公司计划向法国、德国、沙特阿拉伯和阿拉伯联合酋长国的 Snapchat+用户推出 Snapchat for Web，然后向全球所有用户开放。

使用 Snapchat for Web，您可以发送消息和快照，还可以通过视频和常规通话进行聊天。Snapchat for Web 还允许用户利用手机上可用的消息功能，包括聊天反应。Snap 表示，它计划在未来将镜头功能引入视频通话。

要访问 Snapchat for Web，用户需要前往 web.snapchat.com 的，用他们的 Snapchat 用户名和密码登录。默认情况下，系统会提示您在手机上完成两步验证。打开 Snapchat for Web 后，您可以从手机上离开的地方继续您的对话。当您通过桌面使用 Snapchat 时，您的 Bitmoji 将出现在与笔记本电脑的聊天中，以向其他人表明您正在通过 Snapchat for Web 访问聊天。此外，网络体验还包括一个隐私屏幕，如果你离开去执行另一个任务，它会隐藏 Snapchat 窗口。

图片来源:抓拍

与手机上的应用程序体验类似，通过 Snapchat for Web 发送的消息将在 24 小时后自动删除。该公司还表示，Snapchat for Web 将防止人们截图。然而，Snap 意识到人们仍然可以用手机在屏幕上拍照，并指出该产品并不完美。

在与记者的一次简报中，Snap 发言人表示，Snapchat 上的普通来电者每天花在通话上的时间超过 30 分钟。尽管 Snap 仍然将自己视为移动优先的平台，但它表示，在听取了用户的反馈后，它决定是时候将 Snapchat 的核心功能引入网络了。

该公司指出:“随着我们社区中如此多的人花更多的时间在线，无论是远程学习还是工作，流媒体还是单纯的浏览——我们看到了一个巨大的机会，让我们的社区更容易全天保持联系，我们已经迫不及待地将我们最喜欢的基本 Snapchat 功能带到网络上。”

图片来源:快照

目前，Snap 专注于将该应用的核心功能带到网络上，并将在此基础上打造用户体验。该公司计划在未来将该应用程序的其他功能，包括观看故事和回忆，引入其桌面体验。

两周前，Snap 正式推出了其付费订阅计划 Snapchat+，为用户提供独家功能，例如改变应用程序图标的能力，以及查看谁重新观看了你的故事。订阅费用为每月 3.99 美元，可以解锁一个特殊的徽章，可以选择将一个朋友列为你的头号朋友，还可以查看“朋友最近移动的大致旅行方向”。该公司表示，只有当你的朋友已经同意分享他们的位置时，最后一项功能才可用。

Snapchat for Web 和 Snapchat+的推出正值社交媒体平台寻求更好地与抖音和 Instagram 竞争，这两个平台都可以通过网络访问，并在一定程度上提供订阅服务。
