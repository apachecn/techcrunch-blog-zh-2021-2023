埃隆·马斯克建议 Twitter 可以在“下周”开源其算法

下面是原故事:

Twitter 所有者 Elon Musk 的一条新的推文表明该公司正准备最快在下周开放其算法的源代码——当然，除非这是一个玩笑。(这年头谁也不知道！)然而，马斯克长期以来一直支持 Twitter 的推荐算法应该开源的想法，甚至在他执掌这家社交网络之前就已经反复声明这一信念，并在 2022 年 4 月宣布他打算收购 Twitter 时再次声明。

在回应今天敦促他开源 Twitter 的一条推文时，Musk 评论“当我们的算法下周开源时，首先要做好失望的准备”，然后指出“它将迅速改进。”

如果是认真的，这将是第一个提及 Twitter 算法开源的任何形式的最后期限的承诺之一——观察家们可能会关注这个承诺是否真的得到了满足。

正如 TechCrunch 的 Paul Sawers 在 12 月报道的那样，Twitter 一直受到来自更广泛的开源社区的其他人越来越大的压力，包括 Twitter 替代乳齿象，在马斯克于 10 月接管微博网络后，其使用量激增。

与此同时，Tumblr 的所有者兼 Automattic 首席执行官马特·莫楞威格最近表示，他的公司的博客平台将采用目前支持乳齿象的 ActivityPub 协议。不久之后，Flickr 首席执行官唐·麦卡斯基尔开始考虑类似的计划。

开源 Twitter 的算法可能有助于阻止立法者和监管者，因为人们对社交平台内容推荐的工作方式越来越感兴趣。不仅美国最高法院现在正在听取关于 YouTube 算法在向用户推荐 ISIS 视频中的作用的辩论，而且还有关于抖音监督的必要性的对话，这些对话在报道揭露该公司操纵病毒趋势甚至监视记者之后变得更加激烈。通过开源，Twitter 可能希望避免类似的审查。

此外，马斯克可以将注意力从其他举动上转移开，比如解雇道德 API 团队，限制 API 访问，以及最近，据报道他如何要求 Twitter 工程师调查他自己的 Twitter 参与度下降的原因，然后迫使算法改变以增加他的推文——他现在将此归因于一个错误。

当然，马斯克并不是第一个暗示开源是最佳前进道路的 Twitter 高管。

值得注意的是，Twitter 的联合创始人和前首席执行官杰克·多西去年对 Twitter 最初被转变为一家公司表示遗憾，他说，相反，Twitter 应该被发展成为一个开放和可验证的协议。他还通过短信与马斯克分享了同样的想法——这是在马斯克试图退出交易时对 Twitter 提起诉讼的法律调查过程中发现的。在短信中，多尔西说 Twitter 应该基于一个“由基金会资助的开源协议”，对此，马斯克回复道，“超级有趣的想法。”

多尔西现在正着手与 Bluesky 一起建立这一愿景，Bluesky 是一个从 Twitter 分离出来的开源项目，正在开发一个名为 ADX 的分散式社交网络协议。虽然 Bluesky 尚未公开推出其应用程序，这将展示其系统的运行情况，但一些已经注册 Bluesky 等候名单的人上周收到了一份调查电子邮件，要求他们提供关于自己和他们首选平台的更多信息。这封邮件暗示，Bluesky 已经接近可用，指出等候名单上已经有超过 100 万人注册，人们将被邀请在“未来几周”测试 Bluesky 应用程序
