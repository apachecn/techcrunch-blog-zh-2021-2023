柏林初创企业，希望与 Zapier 一较高下 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/10/12/the-berlin-startup-that-wants-to-give-zapier-a-run-for-its-money/

今天，Zapier 和 IFTTT 是非常大的平台，可以为文本创建自动化规则，或者让两个应用程序通过 API 相互“对话”。然而，在处理企业内部所需的简单任务时，这些是“敲碎坚果的锤子”。此外，如果包含图像或视频，或者引用的文本是非结构化的，那么需要这种结构的工具就不太好用了。

这是这家总部位于柏林的初创公司背后的想法。它想出了一种方法，让企业创建人工智能驱动的“无代码”规则，以非技术人员可以使用的方式自动化任务。

它现在筹集了 830 万美元的种子资金，由 Balderton Capital(伦敦以外)和 Chalfen Ventures 以及一些天使投资人共同领导。

由 Gero Keil 和 Thilo Hüllmann 创建的 Levity 允许企业使用简单的模板来自动化工作流程，该公司表示，这是一个负责繁重工作的人工智能。它在单一水平平台中使用 NLP 和计算机视觉来解析非结构化数据类型，如图像、文本和文档。Levity 的客户范围从时尚和房地产到航运、营销、社交媒体、科学研究等。

典型的用例包括自动标记和路由收到的电子邮件或电子邮件附件；对客户支持票据进行分类；将收到的文档分类到各自的文件夹中；或者标记视觉库存数据，例如产品照片。

有点像 Zapier，该平台集成了 Gmail，Outlook，Google Drive，Dropbox，Airtable 等。这家初创公司表示，该系统也通过了 SOC2 Type I 认证，符合 GDPR 标准。

在一份声明中，Levity 的联合创始人兼首席执行官 Gero Keil 说:“企业及其客户应该像他们更大的竞争对手一样，有同样的机会获得人工智能和自动化的好处。”

该平台于今年 8 月推出；订阅价格每月 200 美元起。

Balderton Capital 的合伙人詹姆斯怀斯(James Wise)补充说:“有能力利用人工智能和自动化的公司与缺乏资源这样做的较小企业之间的差距越来越大。Levity 肩负着弥合这一鸿沟的使命。”
