Mammoth 是一款免费的 iOS 版乳齿象应用，让你更容易上手

原文：https://web.archive.org/web/https://techcrunch.com/2023/02/27/mammoth-is-a-free-mastodon-app-for-ios-that-makes-it-easier-to-get-started/amp/

图片来源:猛犸

一款名为 Mammoth 的新乳齿象应用在应用商店上市的头几天就已经获得了 1 万次下载。这款应用是由一个团队开发的，该团队包括 Twitter 的 Aviary 应用程序的开发人员，今年早些时候，在新所有者埃隆·马斯克(Elon Musk)决定更广泛的应用程序生态系统不再需要第三方 Twitter 客户端后，Twitter 毫不客气地杀死了和中的一员。

这一决定让长期活跃于 T4 的 Twitter 应用陷入困境，但也促使一些应用开发者将注意力转向开源、去中心化的社交网络乳齿象。例如，上个月，受欢迎的 Twitter 客户端 Tweetbot 的制造商推出了一款名为 Ivory 的新乳齿象应用程序，此前埃隆·马斯克(Elon Musk)的 API 变化毫无预警地摧毁了该公司专注于 Twitter 的前身。

现在，猛犸象正在进入市场，为用户提供另一种选择。

与 Ivory 不同，Ivory 试图建立一个业务来弥补 Tweetbot 关闭带来的损失，至少目前来说，Mammoth 是免费的，不需要任何应用内购买。然而，该公司表示，它最终计划提供 Mammoth 及其社区的订阅版本，即 moth.social ，但尚未确定细节。

与此同时，该公司在博客帖子中解释道，其近期目标是鼓励乳齿象的认养。

图片来源:猛犸

“……我们不认为我们的订阅将是一个付费墙，也不认为免费用户将获得该应用或 moth.social 的只读版本。简单地说:我们希望为拥有 1000 万活跃用户的乳齿象做出贡献，然后是 1 亿。“现在还为时尚早，我们不想做任何减缓采用速度的事情，”Mammoth 的联合创始人 Bart Decrem 写道。“因此，我们可能会订阅，但如果你不付费，这不会阻止你享受这个应用程序。我们还不知道所有的细节和时间。”

在发布时，Mammoth 应用程序提供了广泛的基线功能，如浏览和切换乳齿象时间轴的能力；帖子内容包括文字、图片、gif、投票；使用私人消息、列表和书签等功能。更改应用程序图标和主题；开启黑暗模式；访问多个帐户；甚至可以像 Twitter 一样，在自定义的持续时间内撤销帖子。有大量的手势和定制选项，使猛犸感到个性化。

图片来源:猛犸

该应用程序还有一系列不太常见的功能，如线程创建工具、用于将帖子固定到屏幕上的画中画、在 AR 中查看媒体的工具、情感分析工具、隐藏回复的能力、撰写时的翻译工具等等。作为一款 iOS 应用，用户可以利用 Siri 快捷方式、Face ID 和触控 ID、共享扩展和 VoiceOver 支持。该公司指出，这款应用很快也会出现在 macOS 上。

可能比 Mammoth 的所有花哨功能更有趣的是它的用户加入方式。

选择服务器经常被认为是在乳齿象上搭建的最困难的部分之一。当用户希望能够像他们在其他社交网络上习惯的那样创建用户名和密码时，这给设置过程增加了一层复杂性。但是加入一个服务器是参与乳齿象和被称为Fe diversity、的更广泛的互联服务器群的意义的一部分，所以它不能被绕过。

图片来源:猛犸

为了使这个过程更容易，Mammoth 提供了一个简化的用户界面，引导用户完成帐户设置过程，包括他们的个人资料创建。它还通过提供一个从不同类别中寻找推荐用户的界面，解决了在乳齿象上关注谁的问题 Twitter 当年也是这么做的。

图片来源:猛犸

Mammoth 几天前刚刚推出，但它的 1 万次安装表明消费者对乳齿象应用程序仍有需求，尽管竞争格局包括乳齿象的官方客户端和许多第三方应用程序，包括 Tapbots 新推出的 Ivory。

虽然自马斯克收购 Twitter 以来，一些消费者对乳齿象的兴趣可能已经减弱，但对更广泛的 Fediverse 的兴趣仍在增长。

在马斯克的 Twitter 交易完成之前，包括乳齿象和一系列其他社交应用在内的 Fe diversity 每月用户超过 57 万。如今，该网络拥有257 万左右。公司也开始注意到这一点。 Tumblr 的所有者已经谈到采用与乳齿象相同的协议，而 Flickr 也在考虑同样的协议。博客网站 Medium 上个月推出了自己的社区，从而在 fevariety 中确立了自己的立足点。

Mammoth 看好 Fediverse 的潜力，以及分散式互联网的总体潜力。

该公司表示，除了运营自己的实例之外，它还将开源其代码，最终的目标是让乳齿象对下一个 1000 万用户变得更容易，然后是下一个 1 亿用户。这是一个雄心勃勃的目标，因为 Twitter 在作为上市公司报告的上一季度只有 2.378 亿可变现的日活跃用户，而脸书在最近的收益中有 20 亿日活跃用户。一群分散的应用程序能否在科技巨头的阴影下开拓出足够大的利基市场，还有待观察。但是这可以说是一个比 web3 更有前途的发展，web 3 让骗子和传销泛滥，浪费了消费者的信任。

Mammoth 是 iPhone 和 iPad 的免费下载软件，并将很快登陆 Mac。
