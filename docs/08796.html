Previ 想用返现的方式从你的薪水中扣除个人开支

原文：https://web.archive.org/web/https://techcrunch.com/2022/08/02/previ-wants-to-subtract-personal-expenses-from-your-paycheck-with-a-cashback-twist/?mc_cid=5c572f4a69&mc_eid=fb20c942a3

当你可以直接找到所有消费支出的真正来源:个人的薪水时，为什么要扰乱银行账户呢？布兰登·罗德曼，他创立了客户沟通平台 Weave ，但在公司去年上市前离职，他创办了一家新公司，旨在让消费者直接从他们的薪水中扣除日常开支。

Previ 将希望获得现金返还和便捷支付的用户与希望以更便宜的方式获得客户的合作伙伴联系起来。为了进一步加强合作，Previ 承担了消费者的“风险”,并代表他们向合作伙伴付款(并计划在未来获得债务额度以资助这些付款)。作为销售线索生成的回报，合作伙伴向 Previ 支付固定费用，并向消费者返还固定百分比作为现金返还优势。

理解初创公司最简单的方法是考虑激励机制。用户有动力通过返现优势更直接地为长期交易付费。合作伙伴有动力以更便宜的方式获得客户，而 Previ 有动力押注于愿意定期为服务付费的人的粘性，以至于他们选择自动支付。

这家初创公司不收取任何利率、年费或通常与信用卡相关的滞纳金，罗德曼表示这一点不会改变。然而，如果用户失去了工作和薪水，Previ 的备份计划是从他们的银行账户中提款。用户被要求在注册时连接到银行账户，以防预定购买的金额超过他们的支票。因为它是从用户的银行账户而不是他们的薪水支票中提取的，所以用户没有收到现金返还的好处。然而，当用户得到下一份工作时，他们会被激励重新连接他们的薪水支票服务，从而恢复现金返还激励。

一些人认为拥有薪水是金融科技成功的关键，因为它创造了一种粘性，消费者将通过获胜的平台进行每一笔交易。有初创公司竞相接近消费者的收入，包括 Clair，它通过集成到人力资源中为用户提供免费的挣工资预付款。还有 Pinwheel，一个用于工资单的 API 平台，它最近以 5 亿美元的估值融资。

目前，Previ 并没有直接与雇主合作，作为员工工资单上的一个项目。相反，类似于员工可以将他们支票的一部分直接存入储蓄或 401(k)账户，Previ 是一种选择，员工可以在每次收到工资时分配他们工资的一部分。

这是一个微妙的过程。Previ 最大的挑战是人类。

“显然，信任是第一位的，”他说。“人们将会把他们薪水的一部分联系起来，不是全部，而是一部分。因此，他们必须相信，我们正在尽一切努力解决这个问题。”Previ 无法获得整个薪水，只能获得分配给它的部分，但对于不确定种子期前创业公司持久力的人来说，自动奉献仍然令人生畏。

它还有一个鲜为人知的竞争对手，叫做信用卡。首先，Previ 的目标用户是年龄在 25 岁至 40 岁之间的技术员工，这是一个众所周知的痴迷于信用卡积分的人群。现金返还是否足以击败比赛获得最多积分？

“我不认为我们会取代它，”罗德曼说。“我们希望成为一个合法的替代者——我们考虑人们的钱包，以及他们现在是如何破碎的；有时他们使用 Visa，因为他们总是去好市多购物……但他们有 Venmo 和现金账户，他们有 Amex……我们希望进行同样的对话。”

目前，投资者正在助长这种想法。去年年底，Previ 从阿什顿·库彻的 Sound Ventures 以及 Qualtrics 和 Divvy 的联合创始人那里筹集了 650 万美元。其他投资者包括 Weave 的联合创始人 Jared Rodman 泰勒·霍格，Divvy 公司产品部的 SVP；布莱斯·罗伯茨，OATV 公司董事总经理；还有德里克·安德森，他创办了创业公司 Grind 和 Bevy。罗德曼和他的妻子也向这家初创公司投入了 200 万美元的自有资金。

Previ 正在开放其等待名单，并通过类似礼宾服务的方式让人们慢慢加入其 alpha 测试。我们的目标是在年底前进入测试阶段。
