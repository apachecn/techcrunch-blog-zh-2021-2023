Brick 关闭了 850 万美元的种子基金，以在东南亚实现开放金融| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/02/09/brick-closes-8-5m-seed-to-enable-open-finance-across-southeast-asia/

金融应用正在东南亚激增，使记账或获得在线贷款等事情变得更容易。但这意味着金融科技公司需要访问大量数据，这些数据可以用来验证客户身份、信誉或从在线账户收集信息。Brick 希望通过一套将金融应用与“超本地”数据源(包括银行、移动钱包和电信)连接起来的 API 来简化这一过程。

该公司今天宣布，它已经完成了由繁荣风险投资公司和鹿角公司牵头的 850 万美元的种子轮投资。联合创始人兼首席执行官 Gavin Tan 表示，这笔金额包括 Brick 之前未披露的种子资金，该资金于 2021 年 5 月TechCrunch报道，但其中大部分是新资本。Brick 目前主要在印度尼西亚运营，但正计划扩展到新加坡和菲律宾，最终覆盖东南亚的所有市场。

其他参与者包括 Trihill Capital 和回报投资者 Better Tomorrow Ventures 和 Rally Cap Ventures，以及个人投资者，如 Creative Juice 联合创始人兼首席执行官和格子公司前业务发展和战略主管西玛·甘地；债券金融科技联合创始人吴言；Brian Ma，Divvy Homes 的创始人；迭代联合创始人兼管理合伙人 Ooi Hsu Ken 派恩实验室首席执行官阿姆里什·劳；和 Aspire 联合创始人兼首席执行官 Andrea Baronchelli。

Brick 于 2020 年由 Tan 和 CTO Deepak Malhotra 创建，现在拥有 50 多个付费客户端，每月支持超过 1300 万个 API 调用和近 100 万个消费者。谭是 neobank Slice 的早期员工，而马尔霍特拉是印度 neobank Slice 的联合创始人兼首席技术官。

Brick 的 25 个数据合作伙伴包括一些印尼最大的银行，但 Tan 表示，在过去的一年里，“由于印尼和东南亚的当地情况和大量没有银行账户/银行服务不足的人口，我们实际上有点偏离了对银行业的关注。在印度尼西亚，只有 25%的成年人经常使用银行账户，因此我们已经扩展到涵盖移动钱包、电子商务、电信和政府社会保障数据，这已被证明非常受欢迎。”

Brick 的 API 包括砖块数据、砖块验证和砖块支付，可以实现在线贷款的端到端流程，包括验证用户身份、承销和确保资金支付到正确的银行账户。

当 TechCrunch 去年首次与 Brick 对话时，它的许多客户都是在线贷款提供商，但此后它已经扩展到新的垂直领域。其第二大最受欢迎的垂直领域是个人财务管理应用，其 API 支持预算功能。其第三大垂直市场是企业使用的簿记和会计应用。其他客户包括投资公司、银行和一些印尼最大的企业集团，包括 Sinarmas Group 和 Astra Financial。

这笔资金将用于加大 Brick 在印尼的业务和地区扩张。谭说，菲律宾被选为金砖银行的下一个市场，是因为“开放银行业的发展”。“菲律宾的监管机构、许多开放的银行和开放的金融友好型银行已经起草了监管条例，”谭说。

选择新加坡是因为“监管机构在开放金融思维方面一直处于领先地位，所以这很有帮助，但另一个重要因素是那里有很多现代金融科技初创公司，这意味着他们已经知道开放金融解决方案。”

东南亚其他开放金融 API 创业公司还有 Finverse 、 Brankas 和 Finantier 。

Tan 说，Brick 的优势是“我们在印度尼西亚拥有 25 个以上不同数据连接的覆盖范围，是目前的市场领导者”。

在给 TechCrunch 的一封电子邮件中，fluorescent Ventures 全球投资顾问 Smita Aggarwal 表示，“Brick 作为连接金融机构、企业和客户的数字渠道发挥着重要作用。它允许平台、应用和金融科技通过简单的自动化流程整合身份和金融数据。这项投资与我们对支持金融生态系统的更大承诺完全一致，金融生态系统是包容性的，为所有人服务。”
