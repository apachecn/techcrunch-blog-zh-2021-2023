Reddit 推出新的 NFT 头像市场

原文：https://web.archive.org/web/https://techcrunch.com/2022/07/07/reddit-is-launching-a-new-nft-avatar-marketplace/

Reddit 推出了一个新的 NFT 头像市场

Reddit 今天推出了一个新的 NFT 头像市场，允许你以固定价格购买区块链的头像。该公司表示，你不需要有一个加密钱包来购买它们，所以你的信用卡或借记卡应该足够了，你可以使用 Reddit 自己的钱包产品来存储它们。

该社交网络表示，它正在发布 90 个设计，在这个早期访问阶段销售的 NFTs 总量为“数万”

该公司指出，如果你购买一个限量版的 NFT，你将拥有在 Reddit 上和离线使用它作为头像的许可权。这些权利不像拥有宇迦实验室的无聊猿游艇俱乐部收藏的 NFT 那样广泛，它允许你根据你拥有的无聊猿制作商品或电视节目。更重要的是，你可以在的虚拟形象生成器中混合搭配你的虚拟形象，虚拟形象会在你的社区评论旁边有一个“发光效果”。

图片来源: Reddit

这不是 Reddit 第一次与 NFTs 进行竞技。今年早些时候，该公司开始测试一项功能，允许你将任何基于以太坊的 NFT 设置为你的个人资料图片——几周前 Twitter 发布了一个类似的功能。去年，Reddit 发布了基于其吉祥物 Snoo 的限量版 NFTs，名为cryptosnous。

Reddit 已经与以太坊兼容的区块链 Polygon 合作，在链上铸造这些头像。你可以使用 Reddit 自己的名为 Vault 的区块链钱包来存储和管理这些 NFT，该钱包可在该公司的本地应用程序上获得。目前，Vault 用于赚取区块链的社区积分并将其用于特殊功能，如徽章和动画表情符号。

今天，r/collective avatars仅限受邀用户的 subreddit 成员将可以看到 NFT 的头像。该社区将有关于建立钱包的细节，艺术家的独家 ama 和关于一次性个人资料图片的幕后帖子。因为没有拍卖，所有的非金融债券都有固定的价格，可以通过美元等法定货币购买。这些 NFT 头像的标价是 9.99 美元、24.99 美元、49.99 美元、74.99 美元或 99.99 美元。如果你不是这个社区的一部分，该公司指出，这些可收藏的头像将在未来几周内在其头像生成器页面上为每个人购买。

图片来源: Reddit

该公司与艺术家合作——有些来自像 r/Comics 、 r/ProCreate 和 r/AdobeIllustrator 这样的社区——制作第一个系列。Reddit 指出，艺术家也从 OpenSea 和 Rarible 等市场上这些 NFT 的二次销售中获得版税——与 Reddit 对半分享。

Reddit 表示，虽然 NFT 个人资料图片公司是区块链的一个早期项目，但它希望利用这项技术探索更多功能。

“未来，我们认为区块链是给 Reddit 上的社区带来更多授权和独立性的一种方式。Reddit 一直是在线去中心化的典范；我们的社区是自建和运营的，作为我们更好地赋予社区权力的使命的一部分，我们正在探索帮助他们更加自我维持和自我管理的工具，”该公司在一份声明中写道。

Reddit 加入了一长串正在尝试 NFTs 的公司。在 Instagram 上推出 NFT 支持后，脸书最近将其中一些功能移植到了蓝色应用程序。继 5 月份推出自己的数字艺术收藏后，易贝于 6 月份收购了 NFT 市场诺瑞金。今年 5 月，Spotify 开始测试一项功能，该功能允许一些艺术家在其个人资料上推广数字收藏品。link-in-bio 初创公司 Linktree 为开发者推出了一些新功能，比如为特定 NFT 收藏品持有人提供不可解锁的链接。预计在不久的将来会有更多这样的测试。
