Unit 的“银行即服务”平台开始涉足信用卡游戏 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/11/15/units-banking-as-a-service-platform-is-getting-into-the-charge-card-game/

如果银行即服务金融科技部门做得好，它将在企业中无处不在，同时终端用户也不知道它的名字。该公司为企业提供了一种将金融服务嵌入其产品的方式——在已经推出借记卡之后，Unit 正式进入了借记卡领域。

单位客户现在可以使用初创公司的 API 为自己的最终用户构建定制设计的充值卡。客户可以向他们的客户提供签帐卡、信用卡、循环贷款或该单位的银行合作伙伴提供的任何其他信贷产品。在后端，该单元将处理卡的打印、合规性，以及一旦卡投入使用后的交易跟踪。

据联合创始人兼首席执行官 Itai Damti 称，作为一家风险投资公司，信用卡是 Unit 的第四个也是最后一个支柱，增加了其在借记卡、银行账户和支付领域的产品。

就在六个月前，Unit 宣布以 12 亿美元的估值筹集了 1 亿美元的 C 轮融资，使其自成立以来的总股本达到近 1.7 亿美元。

对于小企业来说，比信用卡更受欢迎的签帐卡为 Unit 提供了一种方式，使客户能够建立和提供贷款产品，即使初创公司本身不是贷款人。“一旦你可以为人们存储金钱，你可以为人们移动金钱，你可以给人们钱，这就是所有这些软件产品可以用来在其环境中启动的全部银行业，”Damti 说。

图片来源:单位

如果 Unit 的新卡系列听起来能与估值数十亿美元的 Brex 和 Ramp 等竞争——我也有同样的想法，只是稍微复杂一点。Unit 不是像资金雄厚的竞争对手那样向初创公司出售卡片，而是向客户出售一种为自己的最终用户制作个性化卡片的方式。它将采用经典的 B2BC 模式，而不是 B2B 模式。

“如果你是一家向建筑公司销售产品的公司，而不是你的客户在市场上寻找其他解决方案，你可以直接将[贷款]嵌入到你的软件中，”Damti 说。“我们本身并不与(Brex 和 Ramp)竞争，但我们确实允许企业提供基本相同的产品，并以嵌入的方式来做。”

对于金融科技公司来说，在经济特别困难的时期，该部门的扩张有所不同，如 Chime 和 Stripe ，它们在过去几周进行了裁员。在 Opendoor 工作了七年后，最近加入该公司的贷款部门副总裁大卫·辛斯基解释说，新产品可以通过交换费帮助其客户引入全新的收入来源。

辛斯基说:“可能花在谷歌和脸书广告上的风险投资资金会更少，但我们正与开发差异化软件的公司合作。”。“我认为单位[是]更好地服务这些用户并提高他们单位经济效益的机会。”Unit 声称，与借记卡相比，使用信用卡进行刷卡交易将多产生 0.5%的交易收入。

Damti 补充说，“垂直金融领域的红海更少了……这是一个巨大的机会，因为他们有数据，他们有分布，他们可以成为非常有效的承销商，他们在垂直领域是非常有效的贷款人。”
