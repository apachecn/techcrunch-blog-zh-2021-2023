微软发布主权云

原文：https://web.archive.org/web/https://techcrunch.com/2022/07/19/microsoft-launches-its-cloud-for-sovereignty/amp/

图片来源:克里斯汀·博西/彭博/盖蒂图片社

在其 Inspire 大会上，微软今天宣布推出 Microsoft Cloud for Sovereignty，这是一种面向公共部门客户(尤其是欧洲客户)的新解决方案，这些客户需要能够保证他们的用户数据在给定的地区存储和处理。

“今天，公共部门客户可以利用微软云的全部功能，包括广泛的平台功能、弹性、敏捷性和安全性，”该公司在今天的公告中解释道。“随着 Microsoft Cloud for Sovereignty 的加入，他们将能够更好地控制自己的数据，并提高云运营和治理流程的透明度。”

图片来源:微软

用户将能够在 Azure 的 60 多个数据中心区域中的任何一个运行他们的工作负载，并访问所有标准的微软云服务(想想微软 365、Dynamics 365 和 Azure 平台)，然后为所有这些服务启用驻留选项，以满足他们的监管要求。该公司还将提供一个“主权登陆区”，使用与其 Azure 登陆区相同的基础设施即代码和策略即代码功能来推荐和执行合规选项。该公司认为，这将使其公共部门的客户更容易启动和运行，同时也使他们能够灵活地根据自己的需求定制这些政策。

当然，其他主要的云提供商也提供自己的系统来确保遵守数据主权法规。例如，谷歌最近宣布了其 Workspace 产品的主权控制，而 AWS 提供了构建这些功能的所有原语，但也押注于其第三方供应商和顾问网络为其客户提供这些功能。然而，很少有人像 Azure 一样重视数据治理，Azure 可能会在寻求更有利可图的公共部门客户方面获得回报。
