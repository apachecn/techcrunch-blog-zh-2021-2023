Discord 用类似 ChatGPT 的功能更新了它的机器人，推出了人工智能生成的对话摘要等等

Discord 是最新一家加入人工智能狂潮的公司。公司今天宣布将向许多服务器推出一系列新的人工智能体验。

最值得注意的是，Discord 正在用 OpenAI ChatGPT 技术更新其 Clyde 机器人，这将允许用户与机器人进行更长时间的对话。你可以在服务器中键入@Clyde，在任何频道与 Clyde 聊天。你也可以让克莱德为你的一群朋友开一个帖子。Discord 指出 Clyde 可以推荐播放列表，并像任何 Discord 用户一样访问 gif 和表情符号。

“在整个 Discord 中， 你可能已经看到 Clyde 对斜线命令作出回应，或者如果你犯了一个错误就让你离开，”Discord 在一篇博客文章中说。下周，克莱德将使用开放人工智能技术，来到(人工智能)生活中，与生俱来的不和谐

你可以让克莱德给你发一个 gif 或者分享五个关于猫的有趣事实。该公司正在测试和迭代该产品，并相信它将成为 Discord 体验的一个基本部分。

图片来源:不和谐

Discord 也在更新其 AutoMod 审核工具，以利用大型语言模型的力量。 AutoMod 现在使用 OpenAI 技术来发现并提醒版主服务器规则可能已被破坏，同时了解对话的上下文。AutoMod AI 实验今天开始在有限数量的服务器上进行。

该公司表示，自去年推出 AutoMod 以来，该工具已经自动阻止了超过 4500 万条来自服务器的不需要的消息，甚至在它们有机会根据服务器规则被发布之前。

Discord 也在推出人工智能生成的对话摘要。新功能旨在将信息流捆绑到主题中，以便用户可以快速了解或加入他们感兴趣的主题。有了这些新的摘要，一些用户将开始在他们的屏幕右侧看到一个新的面板，它将概述已经讨论过的主题。可以在“服务器设置”中启用对话摘要，从下一步开始，对话摘要将推广到有限数量的服务器。

图片来源:不和谐

除了这三种新的人工智能体验，Discord 还分享了它可以将人工智能融入其平台的其他方式。Discord 宣布它是开源头像混搭，这是一款允许用户通过生成图像模型来混搭彼此头像的应用。开发者从今天开始可以在 GitHub 上找到头像混音代码。该公司还在探索一个与朋友和同事合作的共享视觉空间，其中包括一个人工智能驱动的文本到图像生成器，用户可以一起试验。

该公司还分享说，将推出一个人工智能孵化器， 将资源奉献给希望在不和谐的基础上构建人工智能的开发者。该孵化器是 Discord 500 万美元承诺的一部分，旨在资助开发者和初创公司将他们的专业知识带到平台上。参与者将获得现金补助、与 Discord 开发团队的办公时间，并提前获得 Discord 平台功能。

自几个月前推出以来，Open AI 的 ChatGPT 一直主导着互联网，并越来越受欢迎，这导致 AI 在过去几个月里成为越来越热门的话题。因此，Discord 在自己的消息平台上添加人工智能聊天机器人也就不足为奇了。

Discord 并不是唯一一个添加了新人工智能功能的平台，因为 Snapchat 和 Slack 最近也将类似 ChatGPT 的功能集成到了他们的产品中。
