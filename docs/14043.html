Withings 发布 U-Scan 用于家庭尿液分析

原文：https://web.archive.org/web/https://techcrunch.com/2023/01/04/withings-u-scan/

小便是一个神奇的数字，Withings 在你的厕所里放了一个尿液分析实验室

Withings 以其智能体重秤、手表和其他专注于健康的消费技术而闻名，今天在拉斯维加斯的 CES 上发布了一款新设备。U-Scan 在一个服务不足的市场上引起了轰动，它旨在帮助客户跟踪他们尿液中的情况，而不必担心用杯子接住他们的废水或摆弄试纸。该设备与该公司不断扩展的 Health Mate 应用程序同步，并承诺提供可操作的见解。

U-Scan 被设计为安装在马桶中，这使得用户可以不用手就可以进行尿液分析。虽然在医疗环境中很常见，但尿液是家庭健康监测的一个很少利用的机会。如果 Withings 能够如愿以偿，这种情况在未来几年可能会有相当大的改变。该公司指出，尿液中有 3000 多种代谢物，可以快速反映身体的平衡和健康状况。

Withings 首席执行官 Mathieu Letombe 在新闻发布会上表示:“U-Scan 在家中进行日常尿液分析的能力将使 Withings 能够承担起帮助消费者充分利用尿液数据的使命，达到一个全新的水平。”“这是我们宣布的最激动人心、最复杂的产品之一。我们从 U-Scan Cycle Sync 和 Nutri Balance 开始了这一旅程，并期待着不断推出更多墨盒以及该技术的医疗应用。”

可充电的 U-Scan 阅读器知道冲洗水和尿液之间的区别，并确保它只收集所需的样本。使用时，尿液有效地流到收集入口，当热传感器检测到尿液存在时，启动泵。样本被引导通过微流体回路，并被注入测试舱。在这里，反应由光学传感器读取，并报告给应用程序。马桶的每次后续冲洗都会清洁系统，为下一次样本采集重置系统。

在欧洲，U-Scan Nutri Balance 应用程序显示了对比重、pH 值、维生素 C 和酮水平的分析。这些测量的结合有助于人们监控他们的代谢摄入，以优化他们的日常水合作用和营养。“可操作”的部分是，该系统可以推荐锻炼，提供饮食建议和食谱——所有这些都有助于注重健康的用户实现他们的目标。该公司指出，Nutri Balance 在美国的功能可能会有所不同，这取决于 FDA 对此事的说法。

该产品将在欧洲首次亮相，针对消费者推出两种不同的健康墨盒。在不太遥远的将来，医用墨盒将紧随其后。价格标签是€499.95 英镑，包括一个 U 型扫描阅读器和一个提供三个月测试的墨盒。首批上市的两种试剂盒是 Cycle Sync 和 Nutri Balance 试剂盒，Cycle Sync 将帮助每月进行月经周期的人跟踪月经周期，Nutri Balance 试剂盒将为营养和水合作用提供详细的代谢指南。该公司不确定该设备何时在美国上市，因为它的上市将取决于 FDA 的批准。

令人难以置信的是，U-Scan 可以通过其令人惊叹的 Stream ID 功能区分不同用户(从而将结果分配给使用该设备的正确人员)。嵌入读取器内的低能雷达传感器测量多个变量，通过检测尿流的运动和距离来识别个人的“尿流特征”。流 ID 信息可以在 app 中确认。

除了以消费者为中心的产品，Withings Health Solutions 是该公司服务于医疗保健提供商市场的企业对企业部门，它正在为合作伙伴提供用于研究目的的技术。

该公司告诉 TechCrunch，它计划在订阅或个人的基础上提供墨盒。
