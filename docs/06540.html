QuestBook 旨在帮助 web3 开发者获得资金

原文：https://web.archive.org/web/https://techcrunch.com/2022/05/18/questbook-raises-8-3-million-to-help-web3-developers-secure-funds/?guccounter=2&guce_referrer=aHR0cHM6Ly90LmNvLw&guce_referrer_sig=AQAAAIskVMCrmyNuhN5ISBWnmNBz5-j5QM0MdibwK-ySY2x7qYikDwFbPO38UCC_9Kw_j9-eHro_6IcepO0pwGaem4yGIzmcjaoXWJKUsXNGNRDbSilxYO0pwaJaj8L5rZ29NRzWTYVw6Ow0OtWQDeOuBUsP2Yvvc_ZrKL1QbRJLdH1f

区块链和其他 web3 项目正在竞相吸引开发者，举办黑客马拉松，发放补助金和提供其他津贴来吸引那些可以建设的人。但是，他们目前没有足够的带宽来审查他们收到的大量申请，这将一些可以为这些项目带来巨大价值的相同的建设者拒之门外。

试图解决这一问题以及更多问题的初创公司 QuestBook 今天表示，它已经在首轮融资中筹集了 830 万美元，因为它希望扩大自己的努力。

这轮融资由 Lemniscap 牵头，有数十家投资者参与，包括比特币基地风险投资公司、阿拉米达研究公司、蜻蜓、哈希德、波利贡、巴拉吉·斯里尼瓦桑、索拉纳的拉杰·戈卡尔、部落资本的阿琼·塞西和 GitHub 的玛尼什·莎玛。

QuestBook 运营着一个以其名字命名的平台，该平台允许公司以一种更有效、更透明的方式向开发者提供资助和投资。它还使用许多因素筛选项目，如开发者的链上和 GitHub 历史，以承担各种区块链和 web3 公司的负担。

这家初创公司成立于 2021 年 5 月，最初是为了帮助建筑商在加密方面赚钱。但是它很快意识到，它的联合创始人之一马德哈万·马洛兰解释说，没有多少开发人员知道如何用 Solidity 和 Rust 编程，这两种编程语言分别支撑着以太坊和索拉纳·区块链。

今天在开发者生态系统中有一个很大的空白。正如当 crypto 进入熊市周期时，散户投资者的兴趣会下降一样，许多开发者也开始探索其他更熟悉的机会。

但更重要的是，首先没有太多的开发人员为 web3 编码，这个问题可以通过长期激励来解决。根据投资 web3 初创公司的风险投资公司 Electric Capital 最近在发布的一份报告，尽管 2021 年有超过 34，000 名新开发人员为 web3 项目提交代码，但这仍然只是全球软件工程师人数的一小部分。

作为开始，QuestBook 开始免费教授开发者如何用这些语言编写代码，还发布了超过 100 个教程供任何人入门。马洛兰说，这个项目很快就启动了，聚集了超过 18，000 名开发人员。由于吸引了足够大的基础，QuestBook 一直在探索帮助开发者获得资金来建设他们的项目的方法。

“这是我们一直都有的感觉，”他在一次采访中说。“例如，我一生中从未买过任何密码。我拥有的所有密码都是通过为开源项目做贡献而获得的。”

“我们在 12 月下旬的假期中看到的一件事是，人们没有把加密视为一份全职工作。他们认为这是一份兼职。我们想知道如何给这些人一个开始兼职赚钱的机会，这样他们就可以找到资金流动性，然后双倍下注，”他说。

几乎每一个区块链，他们的基金会，或者建立在这些项目之上的公司都会提供资助。他们通常通过自己的代币资助这些项目，从而如果更多的开发者在他们的平台上建立一些可行的东西，他们自己的数字资产的价值就会增加。

但尽管这笔交易看起来对双方都有利，但要确定各种公司希望支持的主题以及它们配置资金的速度，有时会令人困惑和不知所措。例如，开发人员经常需要几个月的时间才能收到协议的消息。

这些公司需要额外的帮助来审查申请，以排除潜在的不良玩家。Uniswap 是以太坊上的一个分散网络，因母船上的 DeFi 生态系统的出现而闻名，它以惨痛的代价吸取了这个教训。

该公司去年向开发商提供了 2000 万美元的无条件拨款，很快超过 50%的资金被换成了稳定的货币 T2。

QuestBook today 与 Polygon、Aave、Near 和 Harmony 合作，并帮助他们向开发者提供了超过 150 万美元的赠款。这家初创公司目前没有将其软件货币化，计划在未来两个季度将这笔拨款增加到 3000 万至 5000 万美元。

他说，除了帮助开发商获得资助，它还有更大的目标。

QuestBook 的工具已经在全球范围内帮助公司创建小型基金，以便当地人才可以找到并投资新项目。这些投资都是透明的，如果开发者认为这个人做得不好，他们可以投票更换这个人。

“这是以最少的拨款和最大的社区参与来实现的。我知道我们需要想出一个更好的名字，”马洛兰笑道。“这个想法是，让我们说你正在运行一个网络或一个协议，你能够看到一些创新发生在一个空间或一个地区，你没有专业知识或带宽来评估，你把资本委托给别人。”

QuestBook 还致力于拓宽开发者寻找工作机会的途径。

“现在，我们正在解决资本分配问题，”马洛兰说。“我们的目标是为 crypto 带来无需许可的作品。一个优步司机可以点击一个按钮，得到这份工作，并开始赚钱。开发人员无法使用相同的基础设施。当然，要解决这个问题，你需要强大的认证、强大的工作流程和资金。我们开始着手处理工作流程。”

马洛兰不愿谈论太多，他说努力还处于早期阶段，但他分享说 QuestBook 已经建立了自己的钱包，可以进行无汽油交易(这意味着用户不需要支付交易费来进行交易。)

“我们今天使用的钱包都是为 DeFi 设计的。因为在我们的案例中，我们交易的不是钱，而是信息，所以我们有理由建立我们自己的无油钱包，在那里人们保存信息不需要任何成本。在很大程度上，开发商甚至不知道，也不应该知道，有一个钱包在发挥作用，”他说。
