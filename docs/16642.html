Krisp 从噪声消除扩展到设备上的转录| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/28/krisp-expands-from-noise-canceling-to-on-device-transcription/

Krisp 最初是一项人工智能噪音消除服务，但现在这家初创公司正在超越这一点，在设备上即时记录你的所有电话和会议——当然，它还集成了 ChatGPT now，用于快速总结。公开测试版今天上线。

我的记录经验通常是下载与会议相关的视频或音频，假设有人想记录它(我知道那个人是谁)，然后将文件上传到像 Otter 这样的服务，等待 10-15 分钟。当然，与几年前相比，这是非常方便的，但是我们在网上召开的会议数量成倍增加。无论哪里，只要我能节省几分钟或点击一下我的工作流程，我通常都会感兴趣。

Krisp 多年来也表现出一定的专注，专注于许多设备和服务的有效降噪。使其成为可能的进步基本上是训练系统识别和隔离人类的声音，这些声音虽然多样，但也与狗叫、交通噪音等截然不同。对我来说，Krisp 通常是有效的。

因此，当他们告诉我他们正在进入转录阶段时，我很高兴听到:这是一个以语音为中心的平台的自然延伸，当然他们知道他们可以获得清晰的语音信号。

Krisp 已经或多或少地在幕后运作，充当音频源的通道。因此，“人工智能会议助手”没有必要与谷歌、团队、Zoom 或其他任何东西集成，或者成为一个幽灵与会者——它在你的音频信号发出之前或它们到达你的耳朵之前严格处理你的音频信号。转录代理不需要与任何服务或 API 交谈，它只是识别进出的语音，并在您的设备上转录它们。

设备上的转录是一个关键的区别，一方面因为它比处理音频文件更简单，另一方面因为你的会议音频永远不会离开你的电脑。我不确定哪些威胁模型是重要的，但我怀疑它们确实存在，并且在未来几年只会变得更加突出。

抄本本身被直接发送到 Krisp 的云服务，这在某种程度上否定了设备上抄本的安全优势。但是如果你愿意相信另一个转录者，为什么不相信这个呢？无论如何，很快也将有一个选项，让您的成绩单完全在设备上。Krisp 首席执行官兼联合创始人 Davit Baghdasaryan 拥有深厚的安全背景，并表示他期待实现这一点。(我怀疑开发一个 web 应用程序比给多个移动和桌面应用程序添加新的功能更容易。)

成绩单怎么样了？根据我的经验，和其他人工智能驱动的一样好，也就是说比我的笔记好，但还没准备好发表。立即抛弃音频否定了其他转录者提供的一个有用的功能，快速检查和编辑:点击一个可疑的单词，它就会跳到音频中的那个位置。所以至少目前来说，在需要准确、广泛引用的面试中，它可能不是最好的选择。

但 Krisp 将其定位为一个生产力工具，建立一个平台，从我们有时发现自己安排的无数会议中快速提取关键点。为此，该公司允许你快速将成绩单发送给 ChatGPT 进行分析，后者取出 TL；博士，分配的行动或任何其他需要退出会议。

坦白地说，我从来没有喜欢过这些功能，也从来没有使用过它们，所以我不能在这里谈论质量。它们可能和其他服务一样好，也就是说对发现这些有用的东西的人来说是有用的。我试图尽快忘记会议的内容，这无疑会让我的同事们懊恼不已——但我要为自己辩护的是，我只是记错了。对于这种类型的工作有流程的有组织的人可能会发现 Krisp 风格很适合他们，或者不适合。

我也不太愿意判断一个服务本质上是对另一个服务的 API 调用，而这个服务本身是不断变化的。随着 ChatGPT 的改进，依赖于它的应用程序也将改进——这与突然进化出自然语言理解能力的上百个其他平台一样。

如果你想尝试，Krisp 是免费的——该公司通过企业高级订阅来赚钱，当然，随着计算在你的设备上运行，它很容易扩展。

在你的 Mac 或 PC 上花一点时间来确保它被正确配置是值得的，也就是说它的直通正在按预期工作。我早期有一些奇怪的问题，但通过从头开始并只分配会议应用程序和麦克风通过 Krisp 来解决它们，所以它不会混淆我的音乐。这只是一些拨号盘和开关，所以花五分钟做好它，很快你的成绩单就会出现得如此之快，以至于有点烦人。
