谷歌的扩展 SDK 旨在为旧版本的 Android 带来最新的功能

分散是对 Android 生态系统的长期抱怨。用户经常会错过最新版本 Android 的功能，因为他们使用的是不再更新的旧设备。为了缩小差距，谷歌发布了第一个公开版本的扩展 SDK，旨在将最新 Android 版本的功能带到旧版本中。

作为此次宣布的一部分，谷歌将向 Android 11 和 Android 12 开放照片选择器 API 支持。Photo Picker 允许用户从他们的图库中选择照片到应用程序中——目前，只有 Android 13 用户可以使用最新的 photo Picker 界面。

值得注意的是，由于这是一个 SDK，应用程序开发人员必须在其应用程序中包含这一功能。但是用户不需要升级系统就可以使用照片拾取器。

谷歌照片选择器。图片来源:谷歌

扩展 SDK 还支持实现 AdServices APIs，为 Android 隐私沙箱的推出做准备。安卓隐私沙箱是谷歌的新广告栈，用来取代 Chrome 中的 cookies。目前，谷歌正在有限数量的运行 Android 13 的设备上测试隐私沙盒，预计今年将推出更大范围的测试版。通过扩展 SDK 等组件，谷歌将致力于向运行旧版本 Android 的设备推出隐私沙盒支持。

谷歌试图通过像项目主线这样的程序来压缩功能推出时间，该项目是与 Android 10 一起推出的，旨在使操作系统更加模块化，并通过 Google Play 推送功能。因此用户不必等待手机制造商发布软件更新。扩展 SDK 是朝着这个方向迈出的又一步。
