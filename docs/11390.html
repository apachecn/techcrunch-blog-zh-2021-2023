Merge 为其统一 API | TechCrunch 筹集了 5500 万美元的 B 轮融资

原文：https://web.archive.org/web/https://techcrunch.com/2022/10/24/merge-raises-55m-series-b-for-its-unified-api/

近几个月来，融资环境可能发生了变化，但基本面强劲的初创公司仍有能力融资。 Merge 一家提供 API 集成服务的初创公司，专注于人力资源、薪资和会计系统(等等)，今天宣布它已经筹集了 5500 万美元的 B 轮融资，由 Accel 领投，现有投资者 NEA 和 Addition 参与。有了这笔钱，Merge 现在已经筹集了 7500 万美元。

谈到基本面，Merge 今天还宣布其 ARR 在过去 12 个月中增长了 30 倍。超过 2500 家公司，包括 TripActions、Ramp、Drata、AngelList、Deed 和 Apollo 等，现在都使用这项服务来整合他们的 SaaS 应用。虽然 Merge 没有披露其目前的估值，但该公司表示，在新一轮融资中，其估值增加了 4 倍。

合并旧金山和纽约的团队。图片来源:合并

“我们的目标一直是:如果一家公司需要建立集成，他们甚至不会考虑在内部进行，就像大多数公司不在内部建立和维护服务器一样，”Merge 联合创始人沈思丁说。“我们在所有现有类别的额外类别扩展方面取得了很大进展，还推出了更大的徽标。”

在过去的一年中，这家于 2020 年成立的公司将其集成支持扩展到了人力资源、招聘和薪资系统之外，现在支持 CRM 工具，以及项目管理和票务系统。

Merge 的联合创始人吉尔·费格(Gil Feig)告诉我:“我们看到现有客户在扩大，增加了更多的集成和类别，但新客户也被新的垂直行业所吸引。”。“这些垂直市场中的一个通常是主要卖点，他们也会购买其他一些，然后一些人会认为我们从一开始就需要所有这些。”

图片来源:合并

虽然该公司开始吸引更大的企业客户，部分原因是其支持类别的扩展，也是其对上市组织的投资增加，但合并团队仍在中小企业上下大赌注。对他们来说，Merge 本周将推出一个扩展的免费层。根据这一新计划，公司现在可以免费向他们的第一批客户提供集成，然后他们必须为接下来的 15 个客户过渡到统一的月费。

图片来源:合并

由于在旧金山和纽约设有办事处，Merge 去年将其员工从 15 人增加到了大约 55 人。除了扩大其走向市场团队和向其统一 API 添加新类别之外，Merge 还计划扩大其企业产品，包括内部版本。

“多年来，我们一直听说构建第三方集成有多么困难—这样做困难重重，成本高昂，而且无法维护。将加入 Merge 董事会的 Accel 合伙人本·弗莱彻(Ben Fletcher)表示:“Merge 以轻松、实惠的方式解决了这一长期摩擦，并真正了解了初创公司和企业的需求。“从开发人员、产品经理和高管那里听到 Merge 如何简单地解决如此困难的问题，给人留下了深刻的印象。”
