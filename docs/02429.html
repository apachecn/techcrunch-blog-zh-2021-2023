为了在远程医疗领域重塑自我，Wheel 获得了 1.5 亿美元的 C 轮融资| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/01/19/to-reinvent-itself-in-telehealth-wheel-lands-150-million-series-c/

当米歇尔·戴维(Michelle Davey)第一次创建帮助临床工作人员的医疗保健平台 Wheel 时，她不得不在两个愿景上说服投资者:第一，消费者将很快要求高质量的虚拟护理，第二，行为将发生如此巨大的变化，以至于她的初创公司将建立重要的底层基础设施，以扩大混合医疗工作人员的规模。

“我们看到整个行业都在朝着同一个方向发展，那就是为更多的患者提供虚拟服务——从保险公司等传统医疗保健公司到沃尔玛和亚马逊等大型零售商，”她说。“但是，在这项创新的早期，我们看到了许多护理点解决方案……以及对患者体验的重新细分。”

戴维重塑虚拟医疗后端的愿景在疫情期间获得了成功，当时随着访问的在线化，远程医疗初创公司的使用率和影响力飙升。很快，她三岁的初创公司又有了新的发展:Wheel 需要从紧急护理模式发展成为一个更全面的平台，可以帮助任何程序为患者的旅程提供动力——从初级护理到儿科，再到慢性病管理。

希望为远程医疗服务提供更全面的基础设施，Wheel 今天宣布完成由 Lightspeed Venture Partners 和 Tiger Global 共同投资的 1.5 亿美元 C 轮融资。新投资者 Coatue 和 Salesforce 与现有投资者 CRV、Tusk Venture Partners 和 Silverton Partners 一起参与了这一轮投资。该公司上次筹集 5000 万美元的 B 轮融资是在 2021 年 5 月，使其迄今已知的融资总额达到 2.16 亿美元。

这笔钱将帮助 Wheel 向更多寻求向患者出售整体虚拟护理服务的公司提供其平台。这家初创公司希望现在能够帮助患者从远程医疗访问开始，然后被引导到实验室、诊断甚至亲自访问。Davey 说:“虽然看起来很明显，但医疗保健的[这种]转变不是微不足道的，它需要新的临床协议、新的法规、法规考虑因素、新的软件和基础设施以及全新的劳动力来提供跨连续体的护理。”

Wheel 并不是唯一一家致力于帮助现场护理解决方案扩大服务范围以覆盖更广泛的消费者患者群体的初创公司。提供商和医疗保健初创公司可以获得公开上市的 Teladoc 和 Truepill 等白标解决方案，这些解决方案已经存在了很长时间，并为 Hims and Hers、Nurx 和 GoodRx 等独角兽的运营提供了动力，因为它们希望以合规但高效的方式实现规模扩张；与此同时，SteadyMD 等公司建立了一个“医疗保健 AWS ”,将数字招聘与电子病历集成、医患沟通渠道和处方转诊相结合。

交钥匙解决方案带来了一系列问题，例如内部建设基础设施与外包 it 的重要性，或者依靠第三方为实践配备人员是否合理。许多初创公司将他们的临床医生基础视为他们的核心竞争优势，因此从其他地方寻找资源会降低其价值主张。

“这不是一个搅动和燃烧的第三方服务，”她说。“我们真正关注的是临床医生，以及我们如何让他们能够以最高的医疗价值执业。她说，Wheel 的临床医生网络今年增长了 60%以上，保留率为 90%。

她证明了 Wheel 将其远程医疗访问工作流程构建在一个平台内的事实，这样临床医生在处理患者时就不必拼凑不同的服务。此外，该平台的规模有助于临床医生访问“一个完全不同的维度”的工作，在面对面护理、虚拟护理或两者的灵活混合之间切换。

迄今为止，Wheel 已经帮助进行了 130 万次患者就诊。
