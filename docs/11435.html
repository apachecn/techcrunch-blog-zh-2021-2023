Shutterstock 将整合 OpenAI 的 DALL-E 2，并为投稿艺术家推出基金

原文：https://web.archive.org/web/https://techcrunch.com/2022/10/25/shutterstock-openai-dall-e-2/

股票图像巨头 Shutterstock】宣布今天将与 OpenAI 合作，大力推进人工智能生成的图像，扩大双方去年宣布的战略联盟。

Shutterstock 和 OpenAI 之间的合作将见证后者的 DALL-E 2 图像生成人工智能系统与 Shutterstock 内容集成，并向全球 Shutterstock 用户提供——这一集成将在“未来几个月”推出。

人工智能生成的图像是指经过视觉数据训练的机器学习技术，因此它可以用反映描述的图片来响应基于文本的提示。结果的质量可能会有很大差异，但这些人工智能系统最近一直在突飞猛进地发展——引起了同样多的敬畏和愤怒；随着许多技术用户庆祝艺术的“民主化”，而视觉艺术家的工作可能已经被这些人工智能作为训练饲料，可能会感到被敲竹杠。

毫不奇怪，考虑到这些敏感性，Shutterstock 对生殖人工智能的推动被框定为一项“道德”行动计划——正如其新闻稿所述，其中包括推出一项基金来“补偿艺术家的贡献”。

它还表示，R&D 将专注于“收集和发布与人工智能生成的内容相关的见解”，目标是将自己定位于“新兴技术的前沿”。

那么，呃，RIP stock 摄影师？或者，库存摄影师的工作最终会在很大程度上转向捕捉训练数据以磨练人工智能模型？(“人工智能不会杀死工作，而是改变它们”是应用于自动化崛起的常用咒语——尽管人工智能经常会用更少、更专业的工作取代大量工作，因此赢家和输家的比例不一定相等，财富通常也不会平均分配……)

Shutterstock 表示，贡献者将因其内容在该技术开发中发挥的作用而获得“补偿”——这提出了许多问题，例如如何识别贡献者以及他们将获得多少报酬；如何准确量化他们的贡献；他们如何知道他们的贡献是否得到了公平的报酬？谁来审核这些薪酬框架？那么，呃，艺术家们最初是怎么同意成为这些人工智能系统的贡献者的呢？

“Shutterstock 认为，人工智能生成的内容是其贡献艺术家的累积努力。为了创建新的行业标准，并为公司的艺术家社区开辟新的收入来源，Shutterstock 还创建了一个框架，为其作品为开发人工智能模型做出贡献的艺术家提供额外的补偿，”Shutterstock 写道，称其框架为“道德和公平”；并表示，当贡献者的知识产权被使用时，它还将“旨在”补偿贡献者(以版税的形式)。

Shutterstock 的首席执行官 Paul Hennessy 在一份声明中评论说:“表达创造力的媒介在不断发展和扩大。我们认识到，拥抱这一变革，确保推动创新的生产技术植根于道德实践，是我们的重大责任。我们在将人工智能融入我们业务的每个部分方面有着悠久的历史。这种专家级的能力使 Shutterstock 成为帮助我们的创意社区驾驭这项新技术的理想合作伙伴。我们致力于开发最佳实践和经验，以实现我们的目标，即让世界充满信心地创造。”

“我们从 Shutterstock 获得的数据对 DALL-E 的训练至关重要，”OpenAI 首席执行官 Sam Altman 在另一份支持声明中证实，然后补充说:“我们很高兴 Shutterstock 通过我们的 API 向其客户提供 DALL-E 图像，这是首批部署之一，我们期待着未来的合作，因为人工智能成为艺术家创作工作流程的一个组成部分。"

Shutterstock 正在运行一个等待列表，以获得即将到来的其内容与 OpenAi 的 DALL-E 2 图像生成器的集成——该列表可在其网站上获得。
