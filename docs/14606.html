用户称，第三方 Twitter 应用正面临问题

原文：https://web.archive.org/web/https://techcrunch.com/2023/01/12/third-party-twitter-apps-tweetbot-twitterrific-echofon-down/

用户称，第三方 Twitter 应用正面临问题

使用第三方 Twitter 客户端的人面临着许多问题，包括无法登录和访问 Twitter feeds。Tweetbot、Echofon 和 Twitterrific 这三个流行的第三方 Twitter 应用程序证实了这些问题，并指出他们不确定是什么引发了这个故障。

这些应用的制造商也在 Mastadon 上抱怨这些问题。Twitterrific 的开发者 Sean Heber 说，“Twitter 刚刚杀死了第三方客户吗？”而推特机器人的保罗·阿达德说，“我希望推特上发生的一切只是一些自动垃圾邮件防护机器人错误地暂停了正确的应用程序。”

在给 TechCrunch 的电子邮件回复中，哈达德说这个问题始于太平洋时间今天下午 7:30 左右。他还提到所有来自应用程序的 API 请求都失败了。

很可能是 Twitter 对其第三方客户端的 API 进行了一些更改，导致这些应用程序崩溃。尚不清楚这是否是阻止进入平台的一个步骤。

除了上述应用，用户抱怨无法从客户端访问 Twitter，如菲尼克斯、 Twitpane 、羽毛和 Talon 。因此，访问 Twitter 的唯一途径是通过官方客户端或网站。

Twitter 开发者论坛上的一个帖子说，在开发者门户上，这些应用显示为“暂停”

自从埃隆·马斯克接管以来，Twitter 已经杀死了许多开发者程序，包括用于应用发现的 Twitter 工具箱。第三方开发者对他们围绕 Twitter 的开发计划持谨慎态度，因为该公司尚未公布其生态系统计划。上个月，该公司前开发者平台主管阿米尔·谢瓦特(Amir Shevat)为 TechCrunch 撰文称，新管理层辜负了开发者的信任。

虽然 Twitter 还没有就这个问题进行任何沟通，TwitterDev 账户上个月表示该公司“将继续投资我们的开发者平台，尤其是我们的 Twitter API。”

本周早些时候，Twitter 决定将算法时间线命名为“For You”作为 iOS 的默认提要。
