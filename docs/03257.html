Reddit 推出了网络版的 Reddit Talk，这是它的现场音频产品| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/02/09/reddit-rolls-out-a-web-version-of-reddit-talk-its-live-audio-product/amp/

图片来源: Reddit

在去年四月推出其俱乐部会所克隆后，Reddit 正在为 Reddit Talk 添加旨在提高参与度的新功能。这些新增功能包括网络兼容性、过去会议的录音、会谈期间的实时评论以及主页顶端的实时栏(类似于 Twitter ),显示正在进行的实时聊天。

“在过去的三个月里，我们看到 Reddit Talk 的每日活跃听众增长了 250%以上，”该公司在其博客上写道。Reddit 补充说，在超过 1000 个 subreddits(或社区)中进行了现场音频聊天，包括与投资者凯文·奥利里(Kevin O'Leary)进行的加密货币对话，以及与“公驴”系列特技演员进行的加密电影对话。

图片来源: Reddit

Reddit 上直播音频室的想法起初看起来有点违反直觉，因为许多 Reddit 为了匿名而涌向这个平台。但是，在直播间评论和发送表情符号的能力将使害羞(或神秘)的用户更容易插话——以前，听众只能通过举起他们的(数字)手来说话。此外，web 兼容性、live bar 和异步收听等发现功能将使用户更容易参与 Reddit 内容的新领域。

Reddit 写道，仍然只能在 Reddit 的 iOS 和 Android 应用程序上创建会谈，但本月晚些时候将可以在桌面上开始会谈。

目前，只有 Reddit 版主(志愿者尽最大努力保持 subreddits 中文明有序的事物)可以开始现场对话。他们可以通过谷歌表格向申请主持一场讲座。鉴于像 Twitter Spaces T7 和 club house T9 这样的平台所面临的 T4 内容审核斗争，这可能是 Reddit 目前控制局面的一个好方法。

“没有任何计划将现场音频功能扩展到主持人以外的 Reddit 用户，”Reddit 发言人告诉 TechCrunch。
