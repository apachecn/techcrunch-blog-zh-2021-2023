SettleMint 获得€1600 万英镑以增强其 web3 低代码平台并进军日本 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/10/11/settlemint-low-code-web3-crypto-blockchain-development/

区块链已经迅速发展，超越了其金融起源和加密泵和转储的狂热。从农业公司到汽车制造商，成熟行业已经开始接受分散化技术。但是和其他新兴领域一样，web3 缺乏熟练的程序员，这阻碍了该行业的创造性发展。SettleMint 试图通过其面向区块链工程师的低代码平台来弥合这一差距。

“欧洲只有不到 1%的开发者有 web3 的经验，”SettleMint 的联合创始人兼首席执行官马修·范·尼克尔克说。他在一次采访中告诉 TechCrunch，从零开始学习 Solidity 的人可能需要 3 到 18 个月来磨练他们在以太坊编程语言方面的技能。

该公司的解决方案是一种工具，它消除了大量区块链编程的复杂性，并允许开发人员在“几周内”创建分散的应用程序开发平台是链不可知的，这意味着它兼容私有的、许可的网络和公共的、无许可的网络。

像 OpenSea 这样的市场降低了 NFT 创作者的准入门槛，但是区块链的效用远不止验证 JPEG 的所有权。SettleMint 的客户之一是印度尼西亚的一所大学，该大学为该国 2019 年选举创建了一个并行的基于区块链的投票系统，以确保公平的结果。另一个用户，印度的贾坎德邦，实施了区块链来跟踪种子供应链并增加其透明度。在金融领域，银行正在使用 SettleMint 对安全产品进行令牌化，以实现不可变的即时交易。

为组织量身定制的区块链即服务业务并不能真正实现华丽的 GameFi 或 NFT 初创公司所见的惊人增长。但是 SettleMint，总部设在比利时，在阿联酋、新加坡和印度都有团队，正是 T2 投资者在加密冬天寻找的“建筑商”类型。

“现在是有很多真正严肃的工作要做的时期。范尼克尔克说:“公司来这里不是为了抬高象征性的价格，而是询问我们如何使用这项技术。

“我们是这个群体中最无聊的一个。我们就像一个蓝筹股玩家，所以去年与不同投资者的讨论很困难。但随着市场下跌，突然之间(投资者说)这家公司有了真正的商业模式，有了真正的订阅收入。这不是炒作的问题，他们已经在市场上呆了这么久。”

这种转变的情绪推动了 SettleMint 最近的融资结束，超额认购了€1600 万美元的 A 轮融资，由总部位于伦敦的风险投资公司 Molten Ventures 共同牵头，该公司是 UiPath 和 Revolut 的投资者，以及专注于欧洲的 OTB 风险投资公司。据 SettleMint 首席执行官称，该公司“资本效率很高”，因此尽管成立了六年，但在首轮融资之前，它只为€筹集了 760 万英镑

此轮投资的其他投资者包括富士通风险投资公司(Fujitsu Ventures)、industry 和 Bloccelerate，以及现有投资者 Medici Ventures，L.P

日本的 web3 未来

SettleMint 最近在葡萄牙的异地。形象学分: SettleMint

日本电子巨头富士通(Fujitsu)的入驻为 SettleMint 提供了一个独特的机会，让它得以扩张到一个长期以来由国内企业主导的科技生态系统中。四月，日本政府发布了一份 NFT 白皮书，将 web3 命名为“数字经济的新前沿”尽管世界各地的监管机构都表达了对这一新兴领域的看法，但很少有国家像日本那样通过国家层面的行动计划明确表态。

“当政府对游戏规则和如何运营有了明确的监管和愿景时，私营部门就会真正介入，支持和投资，无论是制造业的 it 项目，还是丰田或日产等大型制造商，还是其他项目，”范·尼克尔克说。

SettleMint 仍在与富士通就其走向市场战略进行合作，但有迹象表明其低代码解决方案如何吸引日本。该开发平台已在三个场景中得到大量使用——金融服务、制造和供应链以及公共部门，因此该公司“在这三个领域有非常强大的参考用例”, Van Niekerk 建议，因为它进军一个拥有强大技术和汽车制造业的市场。

随着新资本的注入，SettleMint 还计划加强其在欧洲、中东和亚太地区的业务，并在美国试水。该团队目前约有 55 人，并计划增加 60 人。
