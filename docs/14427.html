Blaze 通过人工智能驱动的无代码应用构建器| TechCrunch 让编码变得更加容易

原文：https://web.archive.org/web/https://techcrunch.com/2023/01/11/blaze-makes-coding-more-accessible-with-ai-driven-no-code-app-builder/

有很多人掌握着业务的脉搏，他们知道如何构建有用的应用程序，但缺乏基本的编码技能和将想法付诸实践的能力。如果这些人可以描述他们想要构建的应用程序，然后将连接器添加到公共数据源，甚至构建一个复杂的工作流，而无需编写一行代码，会怎么样？

这就是作为一家早期创业公司，由两位已经创建并出售了一家公司的女性创始人组成的 Blaze 试图做的事情。今天，该公司宣布该产品首次全面上市，并宣布进行 350 万美元的种子前融资。

“我们解决的问题是让任何团队都能够构建非常定制、复杂、高度安全的应用程序和工具，而无需代码。对我们来说，真正令人兴奋的是它也是人工智能驱动的，”联合创始人兼联合首席执行官刘南希告诉 TechCrunch。

人工智能驱动的比特是通过 OpenAI API 提供的自然语言处理，它使用户能够通过键入描述来简单地告诉 Blaze 他们想要做什么，然后该服务将这些描述转化为代码。公司联合创始人兼联合首席执行官 Tina Denuit-Wojcik 解释说，Blaze 需要大量的预处理和后处理来将输入的命令转换为操作代码，但 API 有助于驱动它。

“用户可以只输入几个句子，但我们必须让它知道所有数据的上下文，以及页面上的其他内容，如何连接以及可能的操作，”Denuit-Wojcik 说。

当他们构建应用程序时，用户可以拖放连接器，如 Shopify 等电子商务工具或 Stripe 等支付工具。其他预构建的集成包括 Airtable、DocuSign、Freshdesk、Google Sheets、Salesforce 等。该公司甚至建立了自己的数据库，名为 Blaze Tables。

他们帮助人们构建的应用程序包括文档门户、客户成功仪表板、库存管理系统和合同工作流程。

两位创始人已经合作了十多年，在 2012 年创立了一家名为 Enplug 的数字标牌公司，在 2021 年初被 Spectrio 收购之前，该公司筹集了 370 万美元(根据Crunchbase)。刘说，她和 Denuit-Wojcik 受到启发，创办了另一家创业公司，并开始专注于无代码解决方案，这是基于他们在前一家公司的经验，他们认识到需要这样一种工具。

到目前为止，这家新公司有 12 名员工。作为两位创办公司的移民女性，她们敏锐地意识到需要建立一个多元化和包容性的团队。此外，刘表示，该产品本身将为更多人打开这个领域，这些人传统上因缺乏编程技能而被排除在技术工作之外。

“我认为这就是我们的运作方式。我们对此考虑得很周到。刘说:“我们也非常关心这个产品如何帮助更多可能没有技术技能的妇女，现在她们可以去制造技术产品了。”。

今天的 350 万美元投资于去年年底关闭。Flybridge Capital 和 MaC Venture Capital 领投了这一轮，许多行业天使投资人也参与了进来。
