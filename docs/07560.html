AWS 向太空发送了一个雪筒

原文：https://web.archive.org/web/https://techcrunch.com/2022/06/23/aws-sent-a-snowcone-to-space/

在其 re:Mars 会议上，亚马逊今天宣布，在前往国际空间站的 Axiom 任务中，它悄悄地将其一台 AWS Snowcone edge 计算和存储设备送入太空。

在很大程度上，这是一个现成的雪锥，AWS 已经建造了足够坚固的雪锥，可以由 UPS 运输，尽管该公司必须做几个月的测试才能获得这次飞行的认证。

“当你想到在偏远、不连通、崎岖的环境中向边缘提供云计算时——在航天业工作了 35 年——坦率地说，没有比太空环境更恶劣、更偏远或崎岖的环境或更无情的了，”AWS 航空航天和卫星主任克林特·克罗希尔(Clint Crosier)说，他是一名退休的美国空军少将，在退休前帮助监管美国航天部队的基础，去年加入 AWS。“如今，太空是一个价值 4250 亿美元的全球产业，据所有主要分析师预测，到 2040 年，这一产业的价值将达到 1 万亿美元，是 2018 年至 2022 年间发射的卫星数量的三倍。出于所有这些原因，客户告诉我们，他们需要与他们在太空中的工作负载接近的云计算能力，就像他们在地面上一样。”

在安装之前，在 Ax-1 任务期间，AWS Snowcone SSD 在国际空间站上。图片来源: AWS

为了认证 Snow cone(Snow 系列中最小的边缘计算和数据传输设备)，AWS 必须让它通过 NASA 为期五个月的热、真空、声学和振动测试(不需要辐射测试，因为该设备将在屏蔽的 ISS 环境中使用)。一旦它到达空间站，由 AWS 的 Daryl Shuck 领导的团队将它连接起来，上传一个用于物体检测的 ML 模型，并在 Axiom 任务的整个时间内运行它。

Axiom 任务中的宇航员总共进行了 25 项实验，包括雪球实验。正如 Crosier 指出的，他们必须拍摄照片并记录他们带到船上的所有设备，然后随船运送。Snowcone 上的对象检测模型帮助他们对所有这些项目进行分类(并标记出那些将被排除在公共分发之外的项目)。

Crosier 承认这是一个相对简单的演示，但通过认证过程教会了该公司很多东西，也为未来的任务奠定了基础。“T 帽子 被 演示那个 我们 做了我no但是整个 过程， 如同 我们 认为 那是 我们的 真的 兴奋 关于 因为 我们 认为 它 引者 中的 一个 整体 为时间带边计算 能力 到 轨道上。

这才是真正的原因。因为这里的目标不是将现有的雪锥或其更大的兄弟带到太空，而是将团队从这些任务中学到的东西(亚马逊已经在未来的任务中与 Axiom 合作)，然后可能将更复杂的边缘计算能力集成到卫星中。具体会是什么样子还有待观察。正如任何一位接受过亚马逊媒体培训的高管在每次采访中都会告诉你的那样，该公司倾听客户的意见，并以此为基础开展工作。

“我们与客户合作，以满足他们的需求，”Crosier 说。“这是 AWS 的标志之一，也是我在美国军队服役 33 年后加入他们所学到的东西之一。因此，如果客户看到了在卫星上部署[边缘]计算能力的价值和需求，你可以理所当然地期待我们会听取他们的意见，并想办法满足他们的需求。”

亚马逊和 AWS 已经与蓝色起源合作，为其商业轨道礁空间站提供计算能力。
