Shopify 在早期因一个平台而出名，该平台为渴望建立自己的在线商店的零售商提供了一种简单快捷的方式，在他们自己的网站和应用程序上建立和运行这些服务——这是一条通往“DTC”的快速途径，当时它开始感觉像亚马逊这样的市场在接触消费者方面受到了束缚。快进到今天，经过两年的新冠肺炎电子商务热潮，数字购物正在越来越多的平台上发生。因此，为了应对这种转变，如今 Shopify 正在为有兴趣提升社交商务、谷歌本地购物优惠、加密货币、B2B 销售等的客户推出大量工具——总共约有 100 项新功能。

如今，购物者无处不在是一场流动的盛宴，Shopify 的方法似乎是提供拉斯维加斯式的自助餐来解决这一问题:每个人都有一点点东西。或者如首席执行官兼创始人鸢·吕特克所描述的，商业的“无限游戏”。

更完整的清单是作为该公司今天推出的名为 Shopify Editions 的新半年度产品冲刺的一部分而公布的。需要澄清的是，在今天之前，Shopify 已经在社交商务中开发产品，与谷歌合作，与加密货币和其他开发项目建立联系。今天发布的产品中捆绑的一些功能甚至不是第一次公开。Editions 值得注意的是，Shopify 如何将如此多的开发整合成一个大的推动力，这是该平台本身如何扩张的一个迹象。聚合一条长长的新闻尾巴会让大局更有分量。

Sarah 特别报道了今天宣布的一个新的购物体验，在 Twitter 上，这里；我们在下面重点介绍一些其他更值得注意的发展:

B2B。 Shopify 的主要使用案例一直是直接面向消费者的销售领域，即面向大众市场受众销售的零售商。如今，随着面向企业销售的零售商框架的正式开放，这种情况正在发生改变。Shopify 上的 B2B，正如它对该产品的称呼，不仅仅是针对从事批发业务的公司，也承认了商业世界中的几个重要事实:首先，以消费者为主要受众的零售商也经常向企业销售产品；第二，那些不卖 B2B 的公司可以通过这样做来扩大销售。企业客户通常以不同于消费者的方式购买，通常需要不同类型的支付方式，并且还会有不同类型的税务需求(例如，在英国，企业买家支付的销售税不同于消费者支付的商品和服务销售税)。新的框架将使这些零售商更容易实现部分自动化，并有可能在此过程中增加 B2B 收入。

象征性贸易。最近几个月，NFT 的活动和估值急剧下降，目前的一个大问题是，这是否是人们醒来闻到郁金香的味道的结果，或者这是否是加密货币目前大幅下跌导致的暂时现象——如果是后者，一种货币是否可能有效地与另一种货币分开。不管了！Shopify 将为所有决定涉足非功能性网络营销的顾客提供服务。

不，这不是与 NFT 最大的市场之一 OpenSea 的合作，而是 Shopify 自己构建自己的 NFT 体验。它说，Shopify 的令牌化商务将使用 NFT，尽管尚不清楚 Shopify 是否将为它们的创建和管理提供动力。它指出，通过将加密钱包与 Shopify 在线商店相连接，这将是一种“奖励真正的粉丝和贵宾，让 NFT 持有者独家获得产品、津贴和体验”的方式。Shopify 表示，商家将能够在网上、手机上和实体零售中激活 Shopify tokengated 商务体验。它还将为来自一家商店或零售商或品牌的 NFT 在 Shopify 支持的商店中使用创造条件。

iPhone 上的 Shopify。Shopify 在实体销售方面越来越活跃——这并不奇怪，因为对于该公司来说，这种交叉将代表一个比仅仅服务于在线商家大得多的潜在市场；许多商家同时经营这两种业务。在增加交易的最新举措中，Shopify 为人们提供了在 iPhone 上通过 Shopify 进行交易的功能。

这实际上似乎是在五月推出的，虽然没有大张旗鼓。基本想法是，虽然许多其他销售点服务都依赖于额外的 POS 硬件，但 Shopify 正在利用苹果的 tap 支付功能，让商家只需一部 iPhone，无需加密狗或终端，就可以轻松实现当面支付。如果像 Square 这样的东西为第一次接受卡支付的新一波小卖家打开了大门，这是一个更低的准入门槛，这就是它如此有趣的原因。

本地 Shopify 库存现在出现在谷歌上。Shopify 和谷歌已经开始合作，让在 Shopify 支持的网站上销售的产品可以在谷歌搜索中找到；现在，这是一个特定位置的角度。当顾客在寻找一种特定的产品时，如果该产品附属于 Shopify 支持的零售商或品牌，购物者将能够看到它是否可以在实体店当地购买。尚不清楚该商店是否需要由该零售商专门经营，或者它是否将与其他商店的库存可用性一起工作。

功能。Shopify 一直在开发大量技术，以满足一些零售商对“无头”商务日益增长的兴趣。无头商务是一种系统，他们可以根据网站的运营和外观定制更多功能。功能是 Shopify 对这个问题的回答:规模稍大的零售商可以建立更动态的选择，比如为购物车中超过一定门槛的商品提供折扣。功能的想法是，它补充了氢和氧，该公司的无头商务框架，更多的技术功能来填充这些定制的网站。这样做的目的是创造更多选择，防止大客户转移到迎合那些觉得自己已经超越 Shopify 的公司的平台上。(有很多这样的例子:只要在谷歌上搜索“成长超过 Shopify”就能明白我的意思。)

更大的情况是，虽然 Shopify 是一个早期的推动者，并有先见之明地看到了为更多商家提供建立购物网站的工具的潜力，但它现在正试图采取更多措施来预测并反映商业的发展方向，无论这些步骤可能是什么，以通过让那些本质上是卖家而不是技术专家的人尽可能容易地使用这种体验来赢得胜利。

Core 产品副总裁 Glen Coates 告诉我们:“商业世界走到哪里，我们就跟到哪里。”“我们希望 Shopify 成为一个简单的大按钮。”
