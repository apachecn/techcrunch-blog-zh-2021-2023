Storj Next 可能会使分散存储对供需双方都更有吸引力

原文：https://web.archive.org/web/https://techcrunch.com/2023/02/23/storj-next-could-make-decentralized-storage-more-appealing-to-both-supply-and-demand-sides/amp/

图片来源: natrot / Getty Images

分散存储公司 Storj 推出了名为 Storj Next 的新版本，该版本引入了新功能和经济激励措施，使亚马逊 S3 的这一替代方案对双方都更具吸引力。

Storj 在全球范围内使用额外的存储容量，安全地存储文件。正如 TechCrunch 的罗恩·米勒在 2020 年解释的那样，“他们有效地利用了存储空间，就像 Airbnb 利用额外的卧室一样，让个人和组织能够出售多余的空间来赚取额外的钱。”

但与 Airbnb 不同，Airbnb 并不总是最便宜的选择，Storj 通常比传统的云对象存储更便宜，后者是在云中存储非结构化数据的选择，如照片和视频。据 Storj 称，其解决方案的成本只是大型云服务提供商的一小部分。但是，企业愿意冒隐私和安全风险将文件存储在任何地方吗？

Storj 不会把你所有的文件都放到别人的电脑上。相反，该公司首席执行官 Ben Golub 告诉 TechCrunch，加密文件被分解并安全地分散在 2 万名个人和出租备用存储空间的公司的驱动器上。

“存储在我们这里的任何数据——任何文件、视频、照片、病历等等——在进入我们的网络之前都会先进行加密。因此，它被只有顾客才有的密钥打乱[……]，然后被分解成 80 块，其中任何 29 块[都能]把它重新组合起来。然后，这 80 个片段中的每一个都进入世界上不同地方的不同驱动器，由不同的人使用不同的电源运行。”

图片来源: Storj

这使得 Storj 能够提供不仅安全、快速，而且比亚马逊、谷歌和微软的数据中心更实惠的云对象存储。

利用未充分利用的存储实现货币化

作为 Storj Next 的一部分，该公司希望帮助其他人以新的方式利用他们未使用的存储。

该公司将增加一个可选的存款结构，网络参与者可以通过存款 Storj 获得奖励，STORJ 是一种基于以太坊的可替代令牌，在 STORJ 网络中使用。

“我们还为社区中的人们增加了一项新功能，即运行卫星；不是真正的卫星，但基本上是一个机会，给那些想在世界其他地方复制我们正在做的事情的人。“我们正在努力实现这一目标，无论是盈利还是非盈利，我们称之为社区卫星。”

除了货币化，Storj Next 还增加了一个新的加密功能:永久存储，这将通过与 Storj 的以太坊智能合同支付来实现。“这是 web3 社区特别感兴趣的事情，”Golub 说。

Storj 自 2021 年以来一直在进行开放开发，接受用户的反馈和贡献 web3 社区无疑非常活跃。然而，如果认为 Storj 只吸引分权狂热分子，那就错了。

Web 2.0 和 web3 之间的桥梁

“我们的客户往往是产生大量数据并希望分享这些数据的人，”Golub 说，他曾是 Docker 的首席执行官。区块链的 Fast sync 是 Storj 的使用案例之一，但该公司也看到了与视频和媒体分发、IT 备份和灾难恢复以及共享大型数据集以进行机器学习相关的需求。

Golub 说，Storj 与 S3 兼容，入职也很简单。“如果你有一个现有的应用程序运行在一个大型云中，甚至是本地，你改变三行，突然你可以(通过 Storj)存储你的数据，”他说。“我们让它变得超级简单；唯一改变的是，突然之间，你的云账单从现在的样子变成了末尾少了一个零，这是人们喜欢的。”

虽然 Storj 的最新功能迎合了 web3 社区，但其绝大多数用户都是 Web 2.0 公司，Storj 在这两者之间起到了桥梁作用。

Storj CMO 罗西 Pongracz 说，她认为这种专注使其有别于分散存储领域的其他参与者，如 Internxt、Filecoin 和 Sia，后者可能更专注于 web3。“但存储市场是巨大的，有许多不同的使用案例，”Golub 说。
