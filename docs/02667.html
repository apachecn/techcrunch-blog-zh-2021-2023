不和谐被打倒了，不仅仅是你

群组聊天平台 Discord 今天遭遇了广泛的 API 问题，就在太平洋时间中午之前开始。这意味着，如果你是一个游戏玩家/加密兄弟/其他喜欢不和谐的人，你将无法在你的播客服务器的#pets 频道上传你的狗的照片(或者上传和查看任何东西)。由于一些初创公司使用 Slack 这样的不和谐手段(…大多数是在某个特定行业，那里的价格也在下降，如果你明白我的意思的话…)，短暂的中断可能会产生影响和/或给 a16z 资助的初创公司的每个员工一个糟糕的日子。

“我们已经发现了 API 中断的根本问题，但正在处理我们的一个数据库集群上的次要问题。平台在上发帖称:“我们整个随叫随到的响应团队都在线，并对问题做出响应。”。作为回应，在团队努力解决服务器问题的同时，Discord 制定了登录速率限制来管理流量。

网站关闭约 45 分钟后，Discord 表示，其数据库再次恢复健康，并慢慢允许用户重新连接，但你可能仍然无法立即登录，并可能会看到一条错误消息。

形象演职员表:不和

在最初的中断大约两个小时后，Discord 发布消息称，他们取消了所有的费率限制，平台几乎恢复正常。

“在接下来的一个小时里，一些 Discord 服务器可能会继续看到一些使用斜杠命令与机器人交互的问题。作为解决事件的一部分，我们需要减少我们数据库的负载，我们拒绝了我们 slash 命令系统的一些部分，”Discord 在其状态网站上写道。“我们将完成我们的内部事后分析过程，以真正深入了解这里到底发生了什么，但如果您今天无法登录或有其他问题，我们真的很抱歉给您带来不便。”

TechCrunch 联系了 Discord 寻求置评。
