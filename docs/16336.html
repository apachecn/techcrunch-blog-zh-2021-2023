Shadow 希望通过云计算让游戏演示变得更酷

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/22/shadow-wants-to-make-game-demos-cool-again-with-cloud-computing/

在游戏开发者大会上，Shadow 正在介绍其云计算服务的一些新用例。特别是，该公司希望与游戏发行商合作，以便他们可以创建虚拟展台来展示游戏演示。

虽然 Shadow 现在为 T2 提供几项服务，但该公司更出名的是为游戏玩家和其他需要强大电脑的人提供云计算服务。用户可以在附近数据中心的服务器上启动 Windows 实例，而不是在本地运行应用和游戏。

然后，视频馈送被流式传输到他们的本地设备。与传统的云游戏服务不同，Shadow 提供了完整的 Windows 实例，这意味着你可以安装任何你想要的东西。如果你住在一个影子数据中心附近，并且有可靠的互联网连接，这项服务在运行得很好。

点击链接开始游戏演示

在过去的几个月里，该公司试图找到新的方法来利用其流媒体和虚拟化技术。例如，该公司与 Bandai Namco 合作，帮助他们进行 Elden Ring 新闻评论。该公司没有将产品密钥发送给媒体，而是建立了带有 Elden Ring 的影子实例。

Shadow 希望将这项服务正式化，并称之为 Echo Sessions。这种设置有几个优点。首先，更容易控制体验，因为每个人都有相同的 CPU，GPU 和 RAM 设置。

第二，游戏发行商或公关机构可以实时查看会话。检查某人是否被卡住是有帮助的。除了 Elden Ring，Shadow 还促成了《小噩梦 2》和《海贼王奥德赛》的公关活动。

Echo Sessions 还可以用于 playtesting，这是一个在视频游戏行业广泛使用的过程，用于查看游戏是否令人愉快。影子可以记录游戏测试，并要求游戏后的反馈。游戏开发者也可以使用 Echo 会话进行内部 QA 测试——这种设置与远程工作兼容。

图像来源:阴影

Shadow 想通过虚拟展台更进一步。顾名思义，它的工作方式有点像游戏大会上的展台。你排队，玩一个演示，放下游戏手柄给下一个人。在这种情况下，你不需要亲自到场。

当谷歌推出 Stadia 时，该公司承诺你可以通过点击 YouTube 视频下方的链接开始玩游戏来启动游戏。Shadow 本质上想要实现这个承诺——但是是通过游戏演示。

当 Windows 实例在后台启动时，你点击一个链接并得到一些指示。然后，你立刻被传送到游戏中。经过一段时间后，流会自动停止。

它可以在您的网络浏览器中工作，并且可以配置一些演示前和演示后屏幕。例如，游戏发行商可能会在你第一次加载页面时询问你的联系方式。玩家可以在演示结束时对游戏进行评级，如果喜欢的话，还可以获得预订折扣。如果试玩不顺利，游戏发行商可以用其他游戏机制开始不同的试玩。该公司的目标是在今年秋天发布虚拟展台。

制造者的阴影

Shadow 正在为那些想要一台可以全天候访问的云电脑的客户澄清其订阅阵容。基本订阅配有 Nvidia GeForce GTX 1080、12GB 内存和 256GB 存储空间，每月 29.99 美元，在欧洲为€29.99 美元。

用户可以每月再花 14.99 美元(或€14.99 美元)购买英伟达 RTX A4500，这是一款 AMD EPYC 7543P CPU，具有 4 核 8 线程，16GB 内存和 256GB 存储空间。这个“能量升级”的额外费用计划一直运行良好，因为在推出几个月后，25%的 Shadow 用户就加入了这个额外费用计划。这两个选项不会改变。

企业客户可以为他们的整个团队订购几个影子实例。同样，有几种配置的价格从每月每个实例 59 美元/€59 美元到 139 美元/€139 美元不等。这也是该公司在 2022 年 11 月宣布的内容。

企业版的 Shadow 还有一些工作要做，因为该公司给了我一长串它计划添加的功能。例如，Shadow 希望提供一种创建映像的方法，以便管理员可以使用预配置的 Windows 映像创建新的 Shadow 实例。该公司还致力于 IPv6 支持、活动目录支持、认证、服务水平协议等。

Shadow 还推出了名为 Shadow for Makers 的新产品，面向第三类客户。这一层是为专业消费者、自由职业者、3D 艺术家、视频编辑和其他希望在影子 PC 上工作的人设计的。现在，配置看起来很像影子的基本配置和权力升级的影子。Shadow for Makers 的价格比普通产品略高(每月 5 美元/€5)，但存储容量更大(1TB)。

一个很大的区别是，Shadow for Makers 是 Windows 10 Home 自带的，而 Shadow for Enterprise 提供的是 Windows 10 Enterprise。这也是为什么创客的影子比企业的影子便宜的原因之一。该公司计划在未来几周内推出 Windows 11 支持。

通过 Shadow for Makers，该公司希望那些不需要管理工具、Windows Professional 许可证和一些认证的苛刻用户会考虑 Shadow。再次强调，这一切都是为了通过云计算服务解决尽可能多的用户。
