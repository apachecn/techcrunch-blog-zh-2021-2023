Gensyn 为人工智能开发人员的分布式计算应用令牌，筹集 650 万美元

对于自动驾驶汽车和其他使用人工智能开发的应用程序，你需要所谓的“深度学习”，其核心概念出现在 50 年代。这需要基于人脑中相似模式的训练模型。这反过来需要大量的计算能力，如长时间运行的 TPU(张量处理单元)或 GPU(图形处理单元)所能提供的。然而，这种计算能力的成本超出了大多数人工智能开发人员的承受能力，他们主要从 AWS 或 Azure 等云计算平台租用。要做些什么？

一种方法是英国初创公司 Gensyn 采用的。它采用了旧项目如 SETI@home 和新冠肺炎聚焦 Folding@home 的分布式计算能力的思想，并将其应用于人工智能开发者对深度学习的渴望。结果是一种从分布式计算机网络获得高性能计算能力的方法。

Gensyn 现在已经筹集了 650 万美元的种子资金，由 web3 风投公司 Eden Block 牵头。参与这一轮的还有 Galaxy Digital、Maven 11、Coinfund、Hypersphere、Zee Prime 和一些区块链协议的创始人。这增加了之前未宣布的 2021 年 110 万美元的种子前投资——由 7% Ventures 和 Counterview Capital 牵头，Entrepreneur First 和 id4 Ventures 参与。

在一份声明中，Gensyn 的联合创始人哈里·格里弗(Harry Grieve)表示:“不断膨胀的硬件需求和丰厚的利润率，是 AWS 和 Azure 等常见品牌争夺如此高的市场份额的原因。结果是一个昂贵而集中的市场…我们设计了一种更好的方式——价格优越，具有无限的可扩展性，并且没有看门人。”

为了实现这一目标，Gensyn 表示，它将推出用于训练人工智能模型的分散计算网络。该网络使用区块链来验证深度学习任务已经被正确执行，通过令牌触发支付。然后，这以可验证的方式将未使用的计算能力货币化。Gensyn 还声称这是一个更环保的解决方案，因为否则这种计算能力将无法使用。

Eden Block 的管理合伙人 Lior Messika 评论道:“Gensyn 通过分散技术真正实现计算民主化的目标可能是我们遇到的最雄心勃勃的努力……该团队旨在通过大幅降低与大规模训练神经网络相关的成本和摩擦，积极扰乱世界上最大和增长最快的市场之一。”

在与我的一次通话中，Grieve 补充道:“我们的估计是，每单位或标准 Nividia GPU 的平均价格便宜 80%，或每小时 45 美分，相比之下，其他云服务每小时约 2 美元。”
