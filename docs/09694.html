英国加大对微软 Activision 交易的审查

原文：https://web.archive.org/web/https://techcrunch.com/2022/09/01/microsoft-activision-uk-antitrust-concerns/

微软以 687 亿美元收购游戏巨头动视暴雪的全现金交易在英国面临更严格的反垄断审查。英国市场监管机构刚刚宣布将进行深入调查——除非双方在未来几天内提交合适的提案来解决其担忧。

竞争和市场管理局(CMA)于 7 月日在对该收购展开正式调查，就是否进入更深入的所谓第二阶段调查征求反馈意见。现在，人们认为这笔交易确实值得密切关注——认为它可能会大大减少游戏机、多游戏订阅服务和云游戏服务的竞争。

微软和 Activision 有五个工作日的时间向 CMA 提交补救措施，以避免这一更深入的调查(这将涉及任命一个独立的专家小组，深入调查第一阶段调查中发现的问题)。

在一份声明中，CMA 兼并高级主管 Sorcha O'Carroll 说:

在我们的第一阶段调查之后，我们担心微软可能会在合并后利用其对《使命召唤》和《魔兽世界》等热门游戏的控制来伤害竞争对手，包括多游戏订阅服务和云游戏领域最近和未来的竞争对手。

如果我们目前的担忧没有得到解决，我们计划在深入的第二阶段调查中探索这项交易，以达成符合英国游戏玩家和企业利益的决定。

CMA 表示，它担心如果 Xbox 品牌的所有者微软收购非常受欢迎的主机、PC 和移动游戏制造商动视暴雪，可能会损害主机竞争对手(“包括最近和未来的进入者”)——拒绝他们访问动视的游戏，或以更糟糕的条款提供访问。

动视开发的热门系列包括《使命召唤》和《魔兽世界》。它还拥有手机游戏《糖果粉碎》，以及其他热门游戏。

“CMA 认为，该合并可能会让微软将 ABK(Activision Blizzard)的内容，包括使命召唤，独家提供给 Xbox 或 Game Pass，或者以其他方式削弱其竞争对手对 ABK 内容的访问，例如通过推迟发布或强制提高许可价格，”该监管机构在其决定的摘要中写道，并指出这被称为“输入止赎”问题，(即一家公司利用其对重要输入的控制来伤害其竞争对手)。

CMA 继续评估，索尼将是微软在中短期内可能受到影响的主要竞争对手。

此外，监管机构表示，已经收到了有关动视暴雪与微软更广泛的生态系统合并的潜在影响的证据。“微软已经拥有领先的游戏主机(Xbox)、领先的云平台(Azure)和领先的 PC 操作系统(Windows OS)，所有这些对其在云游戏领域的成功都很重要，”它在一份新闻稿中写道。“CMA 担心，微软可能会利用动视暴雪的游戏以及微软在游戏机、云和 PC 操作系统方面的优势，破坏新兴的云游戏服务市场的竞争。”

关于云游戏，CMA 总结了它的担忧:

微软已经拥有了其他云游戏服务提供商难以匹敌的资产组合。通过拥有一个大型且分布良好的云基础设施，微软将能够以优惠条件在其服务器上托管游戏，并在无需向第三方云平台支付费用的情况下接触到全球的游戏玩家。通过使用 Windows 操作系统(绝大多数个人电脑游戏都是在这个系统上进行的)，微软可以将游戏流式传输到 Windows 个人电脑上，而不必支付昂贵的 Windows 许可费，并且能够比竞争对手更有效地设计和测试 Windows 游戏。通过现有的控制台生态系统，微软拥有了现有的游戏玩家用户群，可以向他们推广其云游戏服务，以及一系列热门游戏。

因此，合并将使该公司处于独特的强势地位，提供业界最强的游戏目录之一的云游戏服务。CMA 担心，通过利用 ABK 的内容和微软更广泛的生态系统，微软将比当前和潜在的云游戏服务提供商拥有无与伦比的优势。这可能会导致云游戏服务的集中度增加，或者市场向微软“倾斜”，并最终剥夺消费者在新的和新兴的供应商之间竞争的好处，这些供应商竞相在云游戏中取得成功。CMA 认识到，如果微软大幅提高其在云游戏服务领域的市场影响力，这可能会对与微软自己的游戏产品组合竞争的独立游戏开发商和发行商产生连锁反应，他们可能会在许多方面处于不利地位，例如不得不支付更高的费用或在微软的游戏生态系统中被降级。

“CMA 认为这些问题需要进行深入的 2 期调查。微软和动视暴雪现在有 5 个工作日提交提案来解决 CMA 的担忧。如果没有提交合适的提案，该交易将被提交第二阶段调查，”监管机构在新闻稿中补充说。

记者联系了微软，请其对此事发表评论。公司发言人发表了这份声明，署名为公司总裁兼副主席布拉德·史密斯:

我们已经准备好与 CMA 合作下一步的工作，并解决它的任何问题。作为行业领导者的索尼表示，它对《使命召唤》感到担忧，但我们已经表示，我们致力于在 Xbox 和 PlayStation 上同日推出同一款游戏。我们希望人们有更多的机会接触游戏，而不是更少。

这家科技巨头还提到了微软游戏公司的首席执行官菲尔·斯潘塞今天发表的一篇博文，他在博文中阐述了“为任何地方的任何人提供游戏”的愿景，这并没有浪费任何时间来拿出公关利器；吹捧一个“选择”战略，他说微软将通过其类似网飞的游戏订阅服务 Game Pass 提供备受喜爱的动视游戏，“以发展这些游戏社区”。

“我们听说这笔交易可能会从人们目前玩《使命召唤》的地方夺走这类游戏的特许经营权。这就是为什么，正如我们之前所说的，我们致力于在游戏在其他地方发布的同一天在 PlayStation 上提供相同版本的使命召唤，”斯潘塞继续说道。“我们将继续让人们能够跨平台、跨设备地玩游戏。我们知道玩家会从这种方法中受益，因为我们已经与《我的世界》一起做到了这一点，它将继续在多个平台上可用，自 Mojang 于 2014 年加入微软以来，已经扩展到更多。

“随着我们在新设备和平台上扩展我们的游戏店面，我们将确保我们这样做的方式能够保护开发者选择如何分发他们的游戏的能力。”

同样在今天，动视的首席执行官 Bobby Kotick 向员工发布了一封公开信，他在信中重申，公司的领导团队预计对收购的监督将是一个“漫长的过程”，但也暗示交易“可能”在 2023 年年中完成。

“鉴于所需的政府批准数量，我们仍然认为这笔交易最有可能在微软截至明年 6 月的财年完成。他写道:“我们很幸运，已经获得了几个国家的批准，所有监管机构的审批程序基本上都在按照我们的预期进行。”。“本周，我们收到了来自英国的消息，在英国，我们的员工人数超过了除北美以外的任何地方。我们在那里的审查已经进入第二阶段，我们将继续与那里的监管机构全面合作，所有地方都需要批准。”

“随着我们的行业继续看到众多公司积极投资游戏，包括许多世界上最大的技术和媒体公司，政府监管机构正在采取适当和审慎的措施，以更好地了解我们的行业和来自世界各地的日益增长的竞争，”科蒂克补充道。

考虑收购微软-动视的其他监管机构包括美国联邦贸易委员会。
