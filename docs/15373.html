Houseparty 的创始人推出了 Towns，这是一款开源的群组聊天网络应用和协议

原文：https://web.archive.org/web/https://techcrunch.com/2023/02/23/towns-protocol-app-ben-rubin/

在发布了两个广受欢迎的社交应用程序后，本·鲁宾开始全力以赴去中心化。

鲁宾之前创立了 Meerkat 和 Houseparty，这两款应用分别是移动直播和群组视频聊天的先驱，在最初阶段塑造了大规模的社交趋势。在工作场所的“反会议工具” Slashtalk 之后，鲁宾现在又回到了消费者应用上，尽管是以一种迂回的方式。

这些天来，鲁宾对社交媒体的愿景与他因普及而闻名的丰富多彩、用户友好的应用程序几乎没有相似之处。他显然已经通过史诗游戏吸取了猫鼬不幸死亡和豪斯派对在疫情之前过早被收购致死的教训。鲁宾现在是 Web3 的忠实信徒，乐于经受疯狂的投机投资周期和萨姆·班克曼-弗里德斯的考验，同时对底层技术仍能为 Web 的某些方面开启更美好的未来抱有希望。

鲁宾目前正在运行这里不是那里的实验室——“一个由 web3 创建者组成的团队，在我们进行的过程中放弃项目。”布莱恩·米克(Brian Meek)是微软的一名老员工，曾在 Skype 的移动应用上工作，现在是该公司的联合创始人兼首席技术官。

他们的新项目是 Towns ，这是一个协议和基于网络的聊天应用，旨在促进自我拥有、自我管理的在线社区。因为 web3 也有一个加密组件，所以 Towns 最初可能会比他过去的项目更吸引那个社区的爱好者。

一开始，Here Not There Labs 就从 a16z Crypto 获得了 2550 万美元的 A 轮融资，该公司加入了其现有的支持者 Benchmark 和 Framework Ventures。它还获得了 Towns.com 域名(价格非常非常低，只有几十万美元)，这是鲁宾的骄傲，也是他长期游戏计划的标志。

鲁宾告诉 TechCrunch:“这不仅仅是一个加密项目——这是一种将社交网络视为由用户拥有和运营的网络的新思路。”“我认为这让我非常兴奋，特别是从我与 Meerkat 和 Houseparty 的经历中，我们很多人都不想销售……但我们没有在[那个]点上管理我们自己的产品，这是一个拥有 1.5 亿用户的产品。”

Towns 协议将为用户提供一个基于以太坊的智能合约系统(想想通过区块链运行的微型程序)和端到端的加密聊天。这里不是那里实验室表示，Towns 的智能合同是“可扩展、可组合和可升级的”，最终将授权社区起草自己的规则，规定谁可以参与，允许什么，如何以及是否可以赚钱。Towns 后端最初将在一个相对集中的权威证明算法上运行，而团队将建立协议，然后及时转移到以太坊 2.0 的利害关系证明系统。

如果界面看起来很熟悉，那不仅仅是你。图片来源:城镇

对于对 wonkier 加密方面不太感兴趣但仍对去中心化承诺感兴趣的用户来说，Towns 也将表现为一组应用程序，首先是一个今天上线的仅限邀请的 alpha web 应用程序。正如该公司解释的那样，“Towns 应用程序采用了协议实现的所有技术内容，并使它们在开源、端到端加密的愉快聊天体验中可用”——这种体验对那些一提到 NFTs 和 DAOs 就目光呆滞的用户来说更有吸引力。

一个更广泛的测试版将于 9 月推出，任何人都可以使用该协议创建自己的社区，未来几个月将推出移动网络应用程序，之后将推出原生应用程序。鲁宾预计，将来会有许多用户创建的城镇应用程序使用该项目的协议来构建。

目前，最早的用户将不得不使用他们的加密钱包登录 town——一个普通的非加密人可能会感到困惑的登录页面——但 Towns 计划增加对苹果下一代登录技术 Passkeys 的支持。

Towns 希望社区“真正拥有他们的城镇广场”，尽管该项目计划从这些社区中交易的密码中抽取一部分来维持自身。鲁宾表示，随着治理结构的建立，这些细节将得到解决。这里不是那里实验室表示，它将只在开始时进行管理，并随着时间的推移进行分散，将控制权移交给特定于城镇的 DAO(分散自治组织)，该组织将监督其治理并做出决策。

“这有点可笑，有一个人控制着一个大家都在使用的平台，那就是脸书，而现在又有一个人控制着另一个大家都在使用的平台，那就是 Twitter，”鲁宾说，并指出他对其他行业的整合感到不安，比如雀巢在北美的大型瓶装水公司。

Towns web 应用程序的用户体验与 Discord 有着惊人的相似之处，似乎很明显，该项目计划吸引更传统的聊天应用程序的加密社区中不再抱有幻想的成员。Towns 可能会受到 Discord 的设计和实用性的启发，但如果你问 Rubin，Discord 不是一个应该启发其用户的社区模型。“他们租房——他们有一个叫‘不和谐’的房东，他们向房东租房，”他说。

不太清楚的是，对于那些已经对 Twitter 的埃隆·马斯克时代感到不舒服，并愿意搬到乳齿象这样分散但仍然非常友好的社交空间的人来说，城镇是否有更多的主流吸引力。乳齿象在 Twitter 之后走得更远——Towns 在很大程度上是一个聊天工具——但是风气是相似的，即使对加密的重视程度不同。

由于该项目仍处于早期阶段，并且确实有大量的加密连接，Towns 似乎旨在强烈吸引 Rubin 的 web3 忠实信徒。这种情况是否会随着时间的推移而改变还有待观察，但人们对在线社交体验的需求肯定会越来越大，这种体验不会受到当今反复无常的亿万富翁的影响。

鲁宾说:“我认为，确保人类协调和合作的能力真的很令人兴奋。”。“它是抗审查的，它是加密的，由用户拥有和操作。”
