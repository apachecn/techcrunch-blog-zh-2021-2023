Insurtech 分公司以 10.5 亿美元的估值完成 1.47 亿美元的交易

原文：https://web.archive.org/web/https://techcrunch.com/2022/06/08/insurtech-branch-becomes-a-unicorn2331736/

提供捆绑住房和汽车保险的初创公司 Branch 已经在 C 轮融资中筹集了 1.47 亿美元，估值为 10.5 亿美元。

家族所有的私人投资公司 Weatherford Capital 领投了这轮融资，现有和新的支持者也参与了融资，如 Acrew、American Family Ventures、Anthemis、Gaingels、Greycroft、HSCM 风险投资公司、Narya、SignalFire 和 Tower IV。凭借最新的融资，总部位于俄亥俄州哥伦布市的分行自 2017 年成立以来，已筹集了 2.295 亿美元的资金。

Branch 的联合创始人兼首席执行官 Steve Lekas 表示，branch 是他所知道的唯一一家可以通过 API 绑定保险的保险公司，也是唯一一家可以在单笔交易中捆绑汽车和房屋保险的保险公司。

他补充道，Branch 的另一个独特之处在于，它可以嵌入到购买体验中。换句话说，该公司与抵押贷款或安全系统提供商合作，在销售点将保险集成到他们的产品中。例如，如果一个人正在关闭一个家，他们可以选择同时购买分支保险。合作伙伴包括 Homepoint、OpenRoad Lending 和 SimpliSafe 等。

这种分销模式意味着，Branch 支付更少的费用来获得客户，因此，它声称，能够以比竞争对手更低的价格提供保费。但这家初创公司也直接向消费者提供保险，并通过代理机构提供保险。

该公司经历了令人印象深刻的增长。虽然莱卡斯拒绝透露硬收入数据，但他说，过去 12 个月，该分行的年化书面保费增长了 1，300%。自从几乎整整一年前宣布其5000 万美元的融资以来，这家初创公司也从大约 75 名员工增加到今天的 400 多名。

该公司计划利用新资本部分加速其在美国的推广。目前该公司在 28 个州开展业务，今年已扩展到 9 个新的州。

“Branch 所做的不同的事情是收取适当的金额来支付运行每项政策的成本，”Lekas 告诉 TechCrunch。“在现阶段，我们的价格和模式是其他任何人都无法复制的。因此，我们正试图加倍努力，在所有 51 个州迅速扩大规模……有趣的是，尽管我们比之前的 insurtechs 年轻 4 年，但我认为我们将是第一个现金流为正的公司。”

Lekas 在好事达开始了他的职业生涯，在那里他继续担任承保，技术和产品管理的角色。然后，他建立了 Esurance 的第一个在线家庭保险业务。

但在内心深处，勒卡斯渴望找到一种方法，让更多人更容易获得保险。所以他和乔·艾米森合作，布兰奇诞生了。自 2019 年上市以来，Branch 声称每年平均为其“会员”节省 548 美元。

图片来源:分公司

Branch 并不是威德福支持的第一家保险公司。它还支持了斑马公司，这家总部位于奥斯丁的公司经营着一个保险比较网站，去年获得了独角兽地位。

对于 Branch 来说，在充满挑战宏观经济环境和全球风险投资放缓的情况下筹集到 9 位数的资金绝非易事。此外，去年上市的许多保险科技公司——如 Root、Lemonade、Hippo 和 metro mile——正在与历史低点的股票作斗争，而 Policygenius 等其他公司正在裁员，Branch 的成就更令人印象深刻。

不过，莱卡斯承认，早期很难引起投资者的注意。

他告诉 TechCrunch:“我们来自一个我们不是第一个 insurtech 的地方，所以我们很早就开始战斗，因为 venture 不喜欢与 venture 竞争，所以一旦 venture 认为有一群资金充足、聪明、可信的人在解决问题，他们就会三思而行，在竞争对手身后投入更多资金。”“但我们现在的规模比我们之前的大多数公司销售的产品都多。我认为我们做的东西是每个人都认为他们一开始就在投资的东西。”

early and repeat Branch investor grey croft 的联合创始人兼管理合伙人伊恩·西格洛(Ian Sigalow)表示，他的公司“从一开始”就支持 insurtech 的每一轮融资。

Sigalow 告诉 TechCrunch 说:“最初吸引我们的东西没有改变——Branch 是首批提供嵌入式房屋和汽车保险的公司之一，为其成员节省了大量资金。”“Steve Lekas 也从许多其他保险公司创始人中脱颖而出，因为他之前的经验——他已经在 Esurance 扩大了房主的业务，并在 Verisk 管理产品、数据科学、战略和营销，Verisk 是保险行业最大的数据提供商之一。凭借他的经验，他能够很快抓住市场机会。”

他认为，Branch 最大的优势在于其技术平台允许它只需一个名称和地址就可以立即承保和绑定保单，最终实现更低成本的保单。

“作为一家企业，特别是在 insurtechs 在公开市场上苦苦挣扎的时候，我们相信 Branch 的技术脱颖而出，将使他们快速增长，同时也产生可持续的长期损失率，”Sigalow 补充道。

除了努力变得有利可图，布兰奇还有更多利他的目标。这家初创公司是一家公益公司，管理着一个互惠交易所，在这个组织中，投保人是保单保费的实际所有者。该结构旨在调整激励措施，并为会员提供尽可能多的储蓄。它还成立了一个非盈利组织“安全巢”来帮助那些没有保险或保险不足的人。
