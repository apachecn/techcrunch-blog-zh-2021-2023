微软推出基于 Arm 的 Azure 虚拟机，采用 Ampere 芯片

原文：https://web.archive.org/web/https://techcrunch.com/2022/08/29/microsoft-launches-arm-based-azure-vms-powered-by-ampere-chips/?tpcc=tcplustwitter

继 4 月份的预览之后，微软今天早上宣布基于 Arm 架构的处理器 Ampere Altra 的虚拟机(VM)在 Azure 上的普遍可用性。第一批由 Arm 芯片支持的 Azure 虚拟机，微软表示，它们现在可以在 10 个 Azure 地区访问，并可以包含在 9 月 1 日开始使用 Azure Kubernetes 服务管理的 Kubernetes 集群中。

基于 Azure Arm 的虚拟机拥有多达 64 个虚拟 CPU 核心，每个核心 8 GB 的内存和 40 Gbps 的网络带宽，以及 SSD 本地和可连接存储。微软将其描述为“旨在高效运行横向扩展的云原生工作负载”，包括开源数据库、Java 和。网络应用和游戏、网络、应用和媒体服务器。

Windows 11 Pro 和企业版以及 Linux 操作系统发行版的预览版(包括 Canonical Ubuntu、Red Hat Enterprise Linux、SUSE Enterprise Linux、CentOS 和 Debian)将在 VMs 第一天推出，未来还将支持 Alma Linux 和 Rocky Linux。微软特别指出，由于该公司对 OpenJDK 项目的贡献，Java 应用程序几乎不需要额外的代码修改就可以运行。

Azure VMs 的推出是 Ampere 的一个显著胜利，Ampere 在 2018 年走出了隐身状态，其雄心勃勃的目标是与英特尔竞争~ 100 亿美元数据中心芯片市场的一杯羹。该公司得到了 4.26 亿美元风险投资的支持，并由前英特尔总裁领导，近年来成功站稳了脚跟，与甲骨文、Equinix、谷歌云和中国云服务提供商腾讯云、京东云和 UCloud 达成协议，推出基于 Arm 的虚拟机。

Ampere 与亚马逊网络服务公司(Amazon Web Services)基于 Arm 的虚拟机竞争，亚马逊网络服务公司在 2015 年收购了初创公司 Annapurna Labs，以建立自己基于 Arm 的通用服务器硬件阵容，名为 Graviton 。据报道，微软也在追求自己的 Arm 芯片设计，中国科技巨头阿里巴巴和华为也是如此。

研究公司 Omdia 去年 8 月称预计到 2025 年 Arm 将占服务器市场的 14%。如果这一预测成为现实，这将是对英特尔 x86 芯片的一次重大打击，截至 2022 年 3 月，该芯片控制了预计 89%的市场份额。

对于微软来说，Ampere VMs 的推出是实现其五年前做出的承诺的一步，该承诺是用 Arm 兼容的服务器为超过一半的云数据中心容量供电。在与高通的 Centriq 服务器处理器的错误开始后，最终停产，该公司似乎更有能力达到这个阈值。

“微软 Azure 虚拟机在 Arm 上的普遍可用标志着重新定义云计算可能性的一个重要里程碑，”Arm SVP Chris Bergey 在一篇详述 Azure 虚拟机的博客文章中引用他的话说。“通过市场领先的可扩展效率和创新自由，Arm……使 Azure 客户能够以更低的总拥有成本和更清洁的云服务运营来应对日益多样化的工作负载。”
