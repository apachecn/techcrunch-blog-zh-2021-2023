Flipboard 将乳齿象支持扩展到其 Android 应用程序| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/30/flipboard-expands-mastodon-support-to-its-android-application/

杂志应用 Flipboard 正在将乳齿象整合到其 Android 应用中，因为该公司扩大了其对联合社交网络的承诺。Flipboard 首席执行官迈克·麦库埃也将在今天上午举行的 FediForum 会议上演示该应用的新乳齿象支持，这是一场在线活动，旨在讨论 Fediverse 面临的挑战，Fe diverse 是一组互连的服务器，为一系列开源、分散的应用程序提供支持，包括最近流行的 Twitter 替代乳齿象。

上个月， Flipboard 首次宣布它将加入 Fe diversity，计划与乳齿象整合，推出自己的乳齿象实例 flipboard.social ，并在稍后添加对 ActivityPub 的支持——该协议为乳齿象等 Fe diversity 应用提供支持，该应用受益于埃隆马斯克收购 Twitter 后围绕 Twitter 的混乱。在宣布之后，Flipboard 的乳齿象社区现在有大约 500 名活跃成员，他们已经建立了账户，以便参与到这个分散的社交平台中。

此外，Flipboard 正在自己的应用程序中推出对乳齿象的支持，允许用户浏览他们关注的人的简短更新，回复，喜欢和发布帖子，以及点击标签来关注讨论等。在某种程度上，这种整合是为了填补 Flipboard 服务器上的一个潜在漏洞，如果它的 Twitter API 访问停止工作的话。在马斯克的 Twitter 管理下，这是一个高度关注的问题，因为该公司一夜之间在没有任何警告的情况下剥夺了许多应用程序的 API 访问权限，并且最近改变了定价，使许多开发者无法获得它。

然而，Flipboard 的乳齿象集成最初只支持 iOS。如今，Android 支持也已推出。

和在 iPhone 上一样，Android 用户现在可以通过浏览账户部分找到新功能，然后使用他们所在实例的乳齿象凭据登录。如果用户还没有乳齿象帐户，他们可以从这里请求访问 Flipboard 自己的服务器。

在推出 Mastodon 集成后，Flipboard 增加了一些功能，让用户可以直接从 Flipboard 应用程序中跟踪其他人、标签或提要，并在 Flipboard 用户的“为了你”提要中添加了最新的 Mastodon 帖子。这也使得用户可以从其应用程序中访问他们的 Mastodon 书签和@提及，并将其添加到 Flipboard 上的关注项目列表中。

综合起来，这些特点使乳齿象成为 Flipboard 体验的一大部分。虽然不如其他一些客户端强大，比如 Tapbots 的象牙塔或的 Mozilla 支持的猛犸，但是通过 Flipboard 可以浏览和使用 Mastodon，这对于更普通的用户来说已经足够了。

或许同样重要的是，该公司正在对联合社交网络做出更大的承诺，其中包括积极参与围绕权力下放未来的对话。例如，麦库埃今天演讲的活动——FediForum——是一个讨论和分享想法的小型在线“无会议”。正如第一天总结中提到的，其他非活动项目也被邀请参加，比如杰克·多西支持的蓝天计划，但是他们拒绝了。

“我们非常相信来自多元化的创新和增长。麦库埃说:“随着越来越多的应用程序和平台接受像梅斯顿这样的联合社交网络，这个生态系统将越来越成为当今‘围墙花园’社交媒体网络的可行替代品。”。“我们目前与 Mastodon 的集成让我们看到了 ActivityPub 协议如何能够带来更多互联服务和 Fediverse 的扩展。”

Flipboard 拥抱 Fediverse 之前，博客平台 Medium 也发布了类似的公告，后者最近为其作者推出了自己的乳齿象社区。其他已经权衡了各种 Fediverse 集成的公司包括 Tumblr 和 Flickr ，这两家公司也讨论了增加对 ActivityPub 的支持。WordPress.com 的所有者 Automattic(也拥有 Tumblr) 本月早些时候收购了一个 ActivityPub 插件，这样博客就可以加入 Fe diversity。
