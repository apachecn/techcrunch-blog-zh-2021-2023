四个月大的 Doppel 正在帮助一些最大的 NFT 项目发现骗局 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/05/24/doppel-helps-nft-projects-spot-scams-fraud-counterfeit-crypto-yuga-dapper/

随着 NFT 越来越受欢迎，NFT 欺诈也越来越受欢迎。抄袭者可以轻而易举地将图片上传到 NFT 市场，伪造并出售给客户获利，而无需验证这些图片是他们自己的知识产权。随着像 Solana 这样的连锁店出现 NFT，这个问题变得更加普遍，这些连锁店提供相对于以太坊更低的汽油费，因此对潜在的欺诈者来说门槛更低。

虽然 NFT 市场被要求遵守版权法，但 NFT 欺诈可能很难被发现，尤其是因为加密钱包地址并不总是能追踪到其所有者的真实身份。Doppel 是由前工程师 Kevin Tian 和 Rahul Madduluri 在今年 2 月创立的初创公司，它刚刚获得了由 crypto exchange 的风险投资部门牵头的 500 万美元种子资金，以追踪假冒的数字资产。该公司表示，Polygon Studios，Solana Ventures，OpenSea Ventures，Dapper Labs，SVAngel，South Park Commons，Script Capital，Spartan Group，Paxos，6529 Capital，Quiet Capital 和 AlphaLab Capital 都与包括 Balaji Srinivasan 和 Gokul Rajaram 在内的天使投资者一起参加了这轮投资。

田在一次采访中告诉 TechCrunch，在成立后的短短几个月内，Doppel 已经确定了太空中最大的客户之一 NBA 顶级 NFTs 背后的公司 Dapper Labs。田表示，该公司还在与宇迦实验室(Bored Apes Yacht Club (BAYC)系列的创建者)一起试验其产品，并与其他一些客户一起处于早期设计阶段。

NFT 欺诈检测初创公司 Doppel 的创始团队。图片来源: Doppel

田说，Doppel 对 NFT 在各种区块链的数据进行了索引，包括以太坊、索拉纳、多边形、流量等，并使用包含关键字和元数据的模型来识别欺诈。田和马杜鲁里最初是在一起打篮球时认识的，他们指出，在他们已经索引的 1 亿份 NFT 中，有 3700 万份被他们的算法标记为潜在欺诈。

田说，这家初创公司已经与最大的市场 OpenSea 整合，并正在努力与其他公司整合。他补充说，其近期的首要任务是建立新的集成和对额外区块链的支持，并将链外数据添加到他们的模型中。田解释说，Doppel 如何使用链外数据的一个例子是浏览一个品牌的产品目录，并搜索与这些产品匹配的 NFT。

田指出，虽然 Doppel 帮助公司识别欺诈行为，但一旦发现欺诈行为，应由每家公司自己决定如何处理。

据田介绍，该公司的目标是支持基于 web3 的公司和基于 Web 2.0 的公司建立他们的 NFT 战略。他说，Doppel 正在积极争取奢侈时尚品牌的客户。

首席技术官 Madduluri 说，Doppel 的系统可以在一分钟内发现欺诈行为。他预测，随着 Doppel 的持续增长，其面临的最大挑战将是技术上的。

“要消耗和组织的数据量令人难以置信，有很多人试图解决这个问题。我们面临着这一挑战，还有在这些数据的基础上建立良好的人工智能模型以识别潜在的假冒产品的挑战，”Madduluri 说。

至于竞争，Madduluri 表示，包括 Hive 和 Skale 在内的项目正在数字资产领域更广泛地致力于信任和安全。他补充说，反向图像搜索 API 提供商如 NFTPort 也在与 Doppel 类似的领域运营。不过，Madduluri 解释说，这些公司对 Doppel 采取了不同的方法。他说，Doppel 的优势在于，它正在用更深入的端到端方法来解决这个问题，而不是简单地为开发者提供工具，因为它既在构建获取数据的索引器，也在基于这些数据为公司创造 SaaS 式的产品。

田说，这个三人团队由两位联合创始人和公司的创始工程师组成，将在未来几个月内增加两到四名新的工程人员。

尽管加密货币价格最近有所下降，但这两人对非加密货币的持久力充满信心。

“我们不关心市场。这个领域总是有周期，但我们真的相信 NFTs 将长期存在，并将成为几乎每个公司战略的一部分，”Madduluri 说。
