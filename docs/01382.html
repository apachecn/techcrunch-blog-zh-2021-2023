随着企业风险投资的加速，Twilio 也加入了竞争

Twilio 是一家直接与开发商合作的公司，因此拥有一个内部投资部门来帮助这些在 Twilio 基础上建立公司的开发商是非常有意义的。今天，通信 API 公司宣布启动 Twilio Ventures ，这是一个 5000 万美元的基金，旨在投资使用 Twilio 的公司或那些致力于创新邻近理念的公司。

负责新投资部门的企业发展高级副总裁布莱恩·瓦尼曼(Bryan Vaniman)于 3 月份从 Adobe 离职，他的使命是建立一只基金并开始投资。他说，许多公司都有这样的融资机制，这是公司使命的自然延伸，让开发人员拥有更快构建应用程序的工具。

“这实际上是为了支持开发下一代客户参与应用程序的开发者和初创公司。我们只是认为这是我们从一开始就做的开发者第一公司的自然延伸，”他说。

随着去年以 32 亿美元收购 Segment 公司，该公司已经转向以客户为导向的应用，客户体验已经成为 Twilio Ventures 在寻找投资公司时关注的一大部分。

“首先，我们认为这是一个机会，可以通过支持那些为开发人员和客户参与应用程序构建下一代功能的令人兴奋的公司，来扩大我们在该生态系统中的影响力，”他说。

他说，该公司通常会投资首轮融资，但如果公司是一个很好的匹配，会晚一点或早一点投资。通常支票金额在 100 万到 300 万美元之间。

正如你所料，他们已经投资了几家与 Twilio 有很多共同点的公司。“如果你看看我们所做的几笔投资， Algolia 就是一个很好的例子，它展示了一家与我们有着非常相似的开发者个人特质的公司。他们开发了一套 API，使开发者能够构建令人惊叹的搜索推荐体验，”他说。

虽然他们不能与这些公司分享确切的投资金额，但他说他们是二级战略投资者。该计划旨在成为有益的合作伙伴，为这些公司提供内部资源，并在销售和营销方面提供帮助，但在大多数情况下，它们不会拥有董事会席位。

虽然他们可能会考虑收购他们投资的一些公司，但这不是该基金的主要目标。“这更多的是一种附带利益。这不是该项目的主要理由，但投资和拉近与公司的关系并建立关系的好处是，有可能在未来加深关系，但这不是主要重点，”他说。

其他 SaaS 公司采取类似的方式建立投资部门，包括 Salesforce、Zoom、Hubspot、Workday 和 Okta。
