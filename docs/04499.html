尼日利亚的 Sudo Africa 为其发卡 API 平台筹集了 370 万美元的前期种子资金| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/03/02/nigerias-sudo-africa-gets-3-7m-pre-seed-for-its-card-issuing-api-platform/

为尼日利亚开发者和企业提供发卡 API 的金融科技公司 Sudo Africa 已经筹集了 370 万美元的种子前期资金。

总部位于三藩市的全球创始人资本 (GFC)领投此轮融资。参与的风投包括 Picus Capital、LoftyInc Capital、Rallycap Ventures、Kepple Africa、Berrywood Capital、ZedCrest 和 Suya Ventures。

几位非洲金融科技创始人，如 Olugbenga 'GB' Agboola、Ham Serunjogi 和 Odun Eweniyi 也是该公司的投资者。

像许多以 API 为主导的金融科技一样，发卡 API(由 Rapyd、Ayden 甚至 Stripe 等公司在全球率先推出)正越来越多地受到投资者的关注，他们认为这是一个在非洲吸引了最多风投资金的行业的下一件大事。

Sudo Africa 的创始人 Aminu Bakori 和 Kabir Shittu 告诉 TechCrunch，建立 Sudo 的机会是因为他们在之前的创业公司尝试发行卡时面临的一个问题:一个移动钱包系统允许用户将现有的金融机构聚合到一个平台并进行交易。

首席执行官 Bakori 说:“有一段时间，我们想与尼日利亚的一家当地银行合作发行信用卡。“他们打印了多达 1，000 张卡，但这花费了大量时间，而且没有一张卡正常工作，因为银行无法为我们提供任何 API 来管理这些卡，甚至无法控制这些卡的使用。那是我们第一次过来，想着怎么发卡。”

Bakori 描述的金融科技领域是这样一个领域，当初创公司创建筒仓接口，允许他们的客户在他们之间发送或接收资金时，当全球金融系统开始运作时，问题总是会出现。

例如，一个尼日利亚的移动钱包或本地卡试图在亚马逊上进行购物总是一种可怕的体验 。但是随着一些数字银行和金融科技平台引入虚拟美元卡，这已经不再是一个问题 。这项技术的背后是发卡 API fintechs，如卡杜纳的 Sudo Africa。

L-R:卡比尔·什图(首席运营官)和阿米努·巴科里(首席执行官)

它对顾客的宣传是，银行需要几周或几个月的时间来发放信用卡，而 Sudo Africa 只需要几天。通过与持牌发卡机构合作，该公司的基础设施允许其自身以及任何进入其平台的开发商或商家向其客户发行虚拟和实体卡。 和 o n 平台让商家根据自己的口味 对卡 进行控制和编程，打造自己的特色，安全地与其他服务集成。

这里有一个详细的解释。假设一家公司使用 Sudo Africa 发行员工费用管理卡；事情是这样的:员工 被给予 低余额卡，因此每当他们需要使用该卡时，每次都会调用一个 API来实时决定是批准还是拒绝该交易。

另一个例子是，一家快递公司向其派送员发送可编程卡，以进行交易并在特定商场消费一定金额。 Sudo 的 API 工作原理是，如果骑手出于任何原因决定在另一个商场使用该卡或花费超过预定预算，则该卡将不起作用 。

Bakori 说:“这不仅仅是创建一张预付卡，为这张卡提供资金，然后允许顾客在他想去的地方消费。

“那么我们在卡上设置的所有这些控制，比如支出控制，您可以设置 可以支出多少 ，在哪里可以支出，以及何时，将公司放在第一线，并说，‘嘿，这些都是交易细节和支出控制，但仍然，您要继续并批准这笔交易吗？’这为使用我们平台的金融科技公司和企业带来了大量机会。

创始人认为这些特征——支出和本地化控制——使 Sudo Africa 区别于 YC 支持的 Union54 和 unicorn flutter wave 56，后者是发卡 API 业务的新成员。 然而 ，考虑到每家公司都声称自己网站上有这些功能，这三家公司似乎几乎一模一样。

Sudo 的客户横跨多个行业，该公司的首席运营官什图告诉 TechCrunch。他说，他们包括金融科技公司、小额信贷银行、非科技企业、政府机构、物流公司、商业银行和电子商务公司。

公司在其发行的卡 用于 进行网上或 POS 交易时收取交换费，并在进行消费和基于位置的控制时收取授权费。Shittu 称，Sudo 收取的卡片制作和个性化费用比现有公司更便宜。

Sudo Africa 目前是该领域唯一在尼日利亚 单独提供虚拟和实体卡服务的公司。Union54 尽管总部设在赞比亚，但客户遍布非洲。Flutterwave 表示，它帮助其 35 个非洲市场的商家发行虚拟卡，而目前只有尼日利亚的商家有权获得实体卡。

因此，Sudo Africa——它为试图在拉各斯之外扬名的其他金融科技公司和初创公司树立了榜样——希望通过这笔投资向其他非洲市场扩张，这不是竞争的结果，而是必然的结果。“我们现在关心的是向其他非洲国家扩张。首席运营官说:“因此，我们期待在今年年底之前，我们将可能会在另外三四个非洲国家开展业务。”。
