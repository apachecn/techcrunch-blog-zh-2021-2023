比特币基地将允许你在浏览器钱包中查看你的 NFT 收藏

原文：https://web.archive.org/web/https://techcrunch.com/2021/12/14/coinbase-will-let-you-view-your-nft-collection-in-its-browser-wallet/

虽然比特币基地因其加密货币交易所而闻名，但该公司也一直在开发几款 T2 比特币基地钱包 T3 品牌下的自保管钱包。这种类型的钱包让您负责您的加密资产，因为私钥存储在您的设备上。

今天该公司将 NFTs 引入桌面浏览器。在桌面浏览器上使用比特币基地钱包的人很快就会看到一个名为“NFTs”的新标签，它可以让你访问与你的钱包地址相关联的不可替换令牌(NFTs)。

虽然许多手机钱包可以让你查看你的 NFT 收藏，如 Rainbow 、 Argent 以及手机上的比特币基地钱包，但浏览器扩展通常没有原生的 NFT 画廊功能。该功能将在几天后推出。

图片来源:比特币基地

除了新的 NFT 功能，比特币基地一直致力于更好地支持测试网和替代网络。虽然您已经可以从设置中在多个网络之间切换，但比特币基地钱包现在将显示您在所有支持的网络中的令牌余额，如 Arbitrum、Avalanche、币安智能链、Fantom、乐观、Polygon 和 xdai。这是你的加密资产的一种统一收件箱。

比特币基地钱包也将支持 EIP-3085 (EIP 代表以太坊改进提案)。如果比特币基地钱包与支持 EIP-3085 的分散式应用程序交互，将为最终用户带来更好的用户体验。

EIP-3085 允许 dApp 开发者为交易建议一个特定的网络。换句话说，它可以更容易地与多个网络进行交互。

使用自保管钱包的主要优势是，你不会局限于应用程序中可用的内容。您可以使用 WalletLink、WalletConnect 等连接到第三方 dApp。

但比特币基地也将一些分散的交易所直接整合到了比特币基地钱包的界面中。有趣的是，比特币基地在这些交易中使用了 Uniswap 和 0x。比特币基地通过其固有的 DEX 功能收取 1%的交易费用。

在许多方面，比特币基地钱包感觉像比特币基地的 web3 倡议。随着加密用户受教育程度越来越高，并开始与更多 dApps 互动，该公司希望保持相关性。

MetaMask 在想第一次购买 NFT 的新加密用户中很受欢迎。但是 MetaMask 的用户会告诉你，meta mask……并不伟大。比特币基地不想停滞不前。它拥有收购的加密钱包初创公司 BRD ，并计划推出自己的 NFT 市场。
