Pinata 筹集了 2150 万美元用于扩展 NFT 媒体基础设施 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/08/09/pinata-raised-21-5m-to-scale-nft-media-infrastructure/

Pinata 是一家专注于 NFT 的媒体分发平台，该公司独家告诉 TechCrunch，它已经筹集了总计 2150 万美元的资金。

这是这家成立四年的公司迄今为止筹集的唯一资金。Pinata 的联合创始人兼首席执行官凯尔·图特(Kyle Tut)对 TechCrunch 表示，这笔总额来自最近完成的 1800 万美元的 A 轮融资和 2021 年 350 万美元的种子融资。首轮融资由格雷洛克和潘迪拉共同牵头，种子轮由格雷洛克和线下风险投资公司共同牵头。其他投资者包括 Volt Capital、OpenSea 和 Alchemy。

这家位于内布拉斯加州奥马哈的公司为 NFT 市场、metaverses、web3 应用和其他加密项目提供媒体基础设施和支持。它允许用户在任何区块链上管理、分享和他们的媒体，比如以太坊、索拉纳、Polygon、Avalanche 和 Algorand 等等。

“我们为 web3 的很大一部分提供动力，”图坦说。“这个领域的一些大公司正在使用我们。这些用户从非技术创作者到软件工程师都有。”

图坦卡蒙说，在过去的一年里，Pinata 的收入增长了 42 倍。Tut 分享说，自 1 月份以来，它的用户群增长了约 131%，从 104，000 名用户增加到今天的 240，000 名用户。像 OpenSea、DraftKings、Protocol Labs 和宇迦实验室这样的公司也使用这个平台。

“当你快进到今天，我们已经看到，最终 NFT 的行为就像他们自己的应用程序，”图坦卡门说。“他们希望像 YouTube 或抖音那样，向数百万人提供内容，但许多这类应用程序还不具备这样的能力。”

Greylock 的合伙人 Mike Duboe 在与 TechCrunch 分享的一份声明中表示，该平台为新兴创作者和企业提供了存储、管理和构建更多功能的基础设施。

Duboe 说:“虽然今天的 NFT 活动可能专注于静态 jpeg 文件，但我们相信这只是这项技术的冰山一角，围绕 NFTs 的最有趣的应用涉及更丰富的媒体和更深入的功能。”。

图坦卡蒙说，最终，Pinata 的目标是为希望专注于构建媒体和内容的 NFT 项目提供大规模内容。新的资本将用于发展团队和改善其基础设施，使其更快、更稳定，并能够“推动下一代非功能性测试”

“对我们来说，[未来]看起来像成熟的应用程序，而不仅仅是简单的图像或与 NFTs 相关的复杂应用程序，”Tut 说。"非功能性思维将比今天人们所理解的要多得多."
