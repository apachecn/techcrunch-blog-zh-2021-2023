最热门的拍摄

即使有数以千万计的加密拥有者，事实是，相对于这些话题产生的愤怒和狂喜，web3 和 NFTs 的用户数量少得可笑。

然而，硅谷的风险投资家们押注这些平台将很快成为主流，并对旨在让用户接触 NFTs 等事物的私人初创公司给予了高额估值。只要看看 Dapper 和 OpenSea 这样相对较新的初创公司，它们的估值分别为 76 亿美元和 133 亿美元，你就可以看到风投有多饥渴。

但是说到密码发电站，我仍然认为在 web3 中没有比 Twitter 更大的存在了。社交网络是 crypto 和 NFT 的核心和灵魂(不管他们喜欢与否)，该领域的绝大多数交易和关系都是因为 Twitter 平台上的互动而产生的。

以上是我谈一谈埃隆马斯克(Elon Musk)对 Twitter 的收购以及可能出现的世界的理由。现在，马斯克声明的让 Twitter 成为更安全的言论自由中心的意图与 crypto 没有太大关系，但提出 430 亿美元的收购要约可能会让 Twitter 的董事会和股东有时间考虑业务的现状，以及它可以在哪些方面取得更协调的进展。

本周，比特币基地与一些顶级 NFT 收藏合作，推出了他们 NFT 市场的测试版。他们正在寻求与主导 NFT 销量的 OpenSea 竞争，看起来比特币基地提供的很大一部分将是社交功能，具有开创性的功能，如…评论。我不会过多地评价他们的努力，但很明显，他们主要是想通过自己的影响力来竞争。

但当我想到 NFT 市场以社交网站为核心，覆盖面广的时候，我不禁想知道 Twitter 怎么可能会忽视他们在 NFT 建立自己的店面和 OpenSea 竞争对手的机会。

这家社交媒体巨头在很大程度上将其热衷于加密的用户视为一个快乐的小意外。有趣的是，这些用户花费数十亿美元购买非功能性网络交易，他们主要是通过推特发现和出售，唯一的赚钱方式是通过他们每月 2.99 美元的推特蓝色订阅，这给了用户一个很好的小六边形，让他们的非功能性网络交易生活在里面。与此同时，Twitter 基本上是 NFT 每家店面的中央监管前端，让用户离开平台进行交易，然后再回来浏览时间轴。

该公司把钱留在桌面上，这不应该是一个巨大的惊喜。Twitter 在过去十年左右的时间里一直在吮吸产品，但在过去几年里，他们的吮吸少了很多，采取了更积极的货币化措施，收购初创公司，并在寻求与 Spotify 和 Substack 等竞争对手保持竞争力时，总体上证明了自己。

尽管如此，Twitter 比任何其他同规模的消费公司都有更多未实现的潜力，但它对发布机会的单方面关注正导致它错过更大的垂直机会，而其规模大得多的大型科技竞争对手将会扼杀这些机会。脸书可能无法用任何可能的钱来收买它的方式来废黜 OpenSea，但在这里 Twitter 有一条通向 web3 统治地位的大开的道路，而该公司只是忽略它。

多年来，Twitter 一直在尝试特定于社区的功能，包括他们羽翼未丰的社区产品，但像专用市场这样的东西真的会让事情发生变化。加密不是最容易开始的地方——NFT 垃圾邮件已经被证明是 Twitter 的一个强大对手——但垂直市场仍然成熟。

最终，Twitter 最大的绊脚石和游戏聊天应用 Discord 一直在努力克服的一样:主流用户的看法。Twitter 知道，如果他们建立自己的 NFT 平台，他们会激怒同样多的用户。几个月前，Discord 试图测试一些以 NFT 为中心的功能，一条展示应用内钱包插件的短推文导致了数千条关于 Discord 首席执行官的扣篮推文和一致的撤回。这无疑是 Twitter 的情况，它已经发现自己陷入了大量的争议，所以我可以理解他们可能会有点担心直接走进更多的争议。

但有了 430 亿美元的报价，值得质疑的是，一家处于价值数万亿美元的行业中心的公司，如何高兴地忽略了对其不可或缺的机会。
