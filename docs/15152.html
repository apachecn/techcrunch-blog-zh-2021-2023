Twitter 再次推迟发布新的 API 平台

本月早些时候，Twitter 宣布计划削减免费访问其 API 的权限，惹怒了很多人。该公司最初设定 2 月 9 日为执行该规定的最后期限，但后来将其推迟到 2 月 13 日。现在，社交网络再次推迟了免费 API 的关闭——这一次，埃隆马斯克拥有的公司越来越多的不确定性列表中没有日期。

Twitter 开发者账户周一晚间表示，该公司将把新 API 平台的发布时间推迟“几天”该公司坚称，人们对即将到来的平台变化有着“巨大的热情”。

在过去的几周里，开发者社区表达了对社交媒体公司 缺乏透明度和信息的担忧 。该公司对此作出了部分回应，上周在没有任何解释的情况下，简单地将开发者论坛的网站放在了一个登录信息的后面。该网站再次开放后，开发者提出了大量问题，要求澄清新 API 将如何定价。

这一延迟还危及了开发者和初创公司围绕 Twitter API 开发工具的计划，因为他们不清楚该平台未来的支出和预算分配。

Twitter 上周表示，具有“低级使用”的 API 的基本层将定价为 100 美元，但没有透露有多少使用属于“低级”类别。该公司还表示，它将提供一个免费的只写 API，每月可以发布 1500 条推文。这是在埃隆·马斯克承诺为“好的”内容提供机器人提供轻量级 API 之后。

成千上万的学者和 T2 的研究人员也担心他们不得不花钱访问 Twitter 数据，这些数据在 Twitter v2 API 的学术研究层是免费的。
