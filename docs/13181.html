星巴克向第一批测试者开放其 web3 忠诚度计划和 NFT 社区

原文：https://web.archive.org/web/https://techcrunch.com/2022/12/08/starbucks-opens-up-its-blockchain-based-loyalty-program-and-nft-community-to-first-beta-testers/

星巴克今天向其第一批美国测试者推出了其位于区块链的忠诚度计划和 NFT 社区 Starbucks Odyssey 。这项新举措包括可以转化为现实世界体验的咖啡主题 NFT，是星巴克现有忠诚度计划星巴克奖励的延伸，但利用了多边形区块链和 NFT 等 web3 技术。

今年早些时候向投资者宣布，这家咖啡连锁巨头表示，它设想星巴克奥德赛是其最忠诚的顾客获得更广泛、更多样的奖励的一种方式，这些奖励不仅仅是他们今天可以获得的额外津贴，如免费饮料。相反，Odyssey 推出了一个新平台，客户可以参与名为“旅程”的互动活动，完成后，会员可以获得可收集的旅程邮票——这是星巴克对 NFTs 的一个不太令人讨厌的名字。

图片来源:星巴克

这些旅程旨在推广星巴克品牌，并向顾客介绍咖啡和公司的历史。它们可以包括任何数量的活动，如观看视频或接受测验，玩拼图，甚至去商店尝试公司想要推广的新饮料。后一个例子是星巴克奥德赛比其他一些 NFT 企业计划更有趣的部分原因，因为它实际上与该公司现有的忠诚度计划、商业目标和移动支付技术相关，而不是作为一个快速添加的功能。

如果旅程与店内购物相关，奥德赛会员在购买所需的饮料或菜单项时，将扫描现有的星巴克积分卡，然后该活动将与奥德赛分享，从而获得会员积分。

大约 500 分，会员将获得邮票——即多边形区块链上的咖啡主题 NFT。这些邮票也开启了特殊的经历。将有三个级别的好处和经验可以解锁。在低端，这些可以是在线体验，比如教你如何制作浓缩马提尼酒的虚拟课程，或者提供独特的艺术家商品。随着您赢得更多积分和 NFT，您可能会开始获得真实世界的体验，如在星巴克 Reserve Roasteries 举办的特别活动，甚至是前往哥斯达黎加的星巴克 Hacienda Alsacia 咖啡农场。

积分会在一年中累积，而不是像明星那样兑换奖励。年末，积分会重置，你会重新开始。但你仍将保留你的 NFTs，它将随后集成到星巴克应用程序中，允许顾客定制他们的星巴克卡，或者打印在其他商品上。

随着奥德赛计划进入测试阶段，第一批旅程邮票将包括那些从星巴克历史中获得灵感的邮票，比如它在西雅图派克市场的第一个位置；其他的将以经典设计和今年的“礼品包装魔术”节日杯艺术为特色。

该公司拒绝证实有多少客户已经在等候名单上注册，但表示需求“远远超过”其预期。现在，它将开始让一部分等候名单上的用户进入测试奥德赛，并提供对初始体验的反馈，这将用于进一步调整程序。如果用户被邀请加入测试版，他们会收到电子邮件，然后有三天的时间加入。如果没有，他们将返回到等待名单中，等待下一次邀请。

图片来源:星巴克

要在收到邀请后加入 Starbucks Odyssey，会员需要有一个现有的 Starbucks Rewards 帐户，因为该计划与公司更大的忠诚度计划相关联。(并不是只有奥德赛才有单独的账号。)

作为奥德赛的成员，他们将获得 NFT，根据他们的稀有程度有一个点值。这意味着，除了与现实世界的体验相关联，NFTs 还可以在 2023 年推出的奥德赛市场上进行买卖。这部分由 Nifty Gateway 提供支持，NFT 的所有权由区块链担保。

星巴克还简化了购买 NFT 的过程，因为它不要求会员拥有加密钱包，拥有加密货币或对底层 web3 技术有任何了解。除了从奥德赛的行话中去掉“NFT”一词，会员们还可以用信用卡购买这些“邮票”，就像他们在网上购买其他商品一样。

“它恰好建立在区块链和 web3 技术的基础上，但老实说，客户可能甚至不知道他们正在做的是与区块链技术的互动。它只是推动者，”星巴克的 CMO·布雷迪·布鲁尔在九月的一次采访中告诉 TechCrunch 。

形象学分:星巴克

对于这家咖啡连锁店来说，该计划使公司能够吸引最忠实的客户并建立社区，同时也提供了潜在的收入来源。从明年开始，它将发行限量版邮票，会员可以购买这些邮票来支持各种事业。这些 NFT 上的艺术品由星巴克和外部艺术家共同创作，而“一部分”销售将支持星巴克员工和顾客选择的事业。(该公司尚未透露将保留多少剩余收入。)这些邮票每年将发行四至六次。

虽然许多企业进军 NFT 市场都是噱头和欠考虑的，但星巴克一直在更谨慎地权衡其对 web3 的态度。该公司表示，它已经对区块链科技公司进行了几年的调查，但它只参与了这个特定项目大约六个月。布鲁尔早些时候表示，该公司不想将其投资视为一个“噱头”项目，而是希望利用这项技术来扩大现有的忠诚度计划。

星巴克还邀请了移动订单支付系统和星巴克应用程序的设计师亚当·布罗特曼(Adam Brotman)担任奥德赛的特别顾问。该公司表示，现在 Brotman 是 web3 忠诚度初创公司 Forum3 的联合创始人，他的团队与西雅图咖啡连锁店自己的营销、忠诚度和技术团队一起工作。

图片来源:星巴克

在 Odyssey 的演示中，这个程序看起来很容易使用。你登录，选择一个头像，经历一次关于旅程和邮票的入门体验，然后开始。该公司表示，任何时候都会有至少两次旅行。

第一批邀请函将于 12 月 8 日发送给一小群候选人。2023 年 1 月，星巴克将开始向更广泛的等候名单成员发出月度邀请。
