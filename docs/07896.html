挑战封杀令的推特起诉印度政府

原文：https://web.archive.org/web/https://techcrunch.com/2022/07/05/twitter-sues-india-government/

Twitter 已起诉印度政府，挑战一些推文和账户的封杀令，进一步加剧了这个关键海外市场的紧张局势。

在周二提交给孟加拉国卡纳塔克邦高等法院的诉讼中，Twitter 指控新德里滥用权力，命令其任意和不成比例地从其平台上删除几条推文。

此外，Twitter 在诉讼中表示，一些封杀令“涉及政党官方发布的政治内容”。

“屏蔽此类信息侵犯了该平台公民用户的言论自由。此外，有争议的内容与第 69A 条下的理由没有任何明显的直接关系。

该公司还声称，新德里威胁称，如果该公司不遵守命令，将对其在印度的首席合规官提起刑事诉讼。

在这起诉讼之前，Twitter 在印度度过了艰难的一年半，它被要求删除数百个账户和推文，批评者认为，其中许多是令人反感的，只是因为它们谴责了印度政府的政策和总理纳伦德拉·莫迪。

Twitter 发言人拒绝置评。根据 Sensor Tower 的数据，Twitter 在其智能手机应用程序上已经在印度积累了约 4800 万月活跃用户。

在过去一年半的时间里，Twitter 已经部分满足了这些要求，但也试图反击许多挑战。根据去年生效的印度新 IT 规则，Twitter 几乎没有单独质疑撤下命令的空间，不遵守规定可能会导致对其在该国的合规官提起法律诉讼。

新的 IT 规则要求任何大型社交媒体公司在该国任命首席合规官、节点联系人和常驻申诉官，以解决当地的问题。

当被问及诉讼，印度的 it 部长阿什维尼 Vaishnaw 说:“这是每个人的责任，遵守该国议会通过的法律。”

两者之间的紧张关系在去年 5 月 24 日显而易见，当时由印度中央政府控制的德里警方访问了 Twitter 的两个办公室——位于国家首都德里和邻近的哈里亚纳邦的古尔冈——以寻求更多信息，了解 Twitter 通过裁定部分 BJP 发言人是“受操纵的媒体”来给其中一条推文贴上标签的理由

德里警方当时表示，他们收到了关于该发言人推文分类的投诉，并前往办公室向推特印度负责人送达了调查通知。警方在一份声明中表示，Twitter 印度公司的总经理对这个问题的回答“非常含糊”

Twitter 当时将这一事件描述为“恐吓”。

该公司表示，“对警方在执行我们的全球服务条款时使用恐吓战术以及新 IT 规则的核心要素感到担忧”。

Twitter 印度董事总经理去年从公司辞职。

Twitter 并不是第一个起诉印度政府的科技巨头。WhatsApp 去年起诉新德里，挑战允许当局让人们的私人信息“可追踪”，并进行大规模监控的新规定。(该案仍在审理中。)

目前还不清楚新的诉讼是否会对 Twitter 被埃隆·马斯克拟议中的收购产生任何影响。马斯克的特斯拉几年来一直试图进入印度市场，但希望政府允许它首先销售和维修进口汽车。
