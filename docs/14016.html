索尼的目标是通过莱昂纳多项目控制器让 PlayStation 更容易上手

几年来，索尼一直在游戏中采用无障碍选项，但它落后于长期竞争对手微软的一个地方是无障碍硬件。他们的目标是通过莱昂纳多项目来改变这种情况，这是一种新的游戏控制器，旨在根据任何人的需求进行定制。

该设备仅在舞台上进行了一般性描述，但它似乎是一个带有可交换部件和板的中枢，用户可以连接各种其他项目，如呼吸管、踏板和各种开关，以激活不同的按钮。

每个 UFO 形状的莱昂纳多项目设备可以处理一个模拟操纵杆和八个按钮，它们可以相互配对或与传统的控制器配对，以补充或提供任何功能的替代品。索尼与易访问群体 AbleGamers、SpecialEffect 和 Stack Up 合作，以确保它对广泛的人群有用。

索尼设计师 So Morimoto 在一篇博客文章中说:“我们的团队与辅助功能专家一起测试了十几个设计，寻找有助于解决有效使用控制器的关键挑战的方法。”“我们最终选定了一种‘分离式控制器’设计，这种设计允许近乎自由形式的左/右拇指操纵杆重新定位，可以在不需要握持的情况下使用，并且具有非常灵活的按钮和操纵杆帽交换功能。控制器还可以灵活地接受辅助功能配件的组合，创造出独特的美感。”

图片来源:索尼

这类似于微软的 Xbox 自适应控制器的工作方式——有些是内置的，有些是你自己提供的。每个人的可访问性需求都略有不同，因此支持人们已经拥有的解决方案非常重要。另外，那东西很贵！

莱昂纳多计划目前正在开发中，所以还没有发布日期或价格。我们希望很快听到更多消息，并将联系索尼，了解他们在无障碍硬件方面的工作。
