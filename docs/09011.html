Truework 筹集了 5000 万美元，帮助贷款人核实借款人的信息

原文：https://web.archive.org/web/https://techcrunch.com/2022/08/09/truework-which-helps-lenders-verify-borrowers-income-and-employment-raises-50m/amp/

图片来源:真实作品

Truework 是一家为抵押贷款和以消费者为中心的贷款机构开发技术以即时验证借款人收入和就业的公司，已经在 G Squared 牵头的 C 轮融资中筹集了 5000 万美元。

总部位于旧金山的 Truework 成立于 2017 年，与包括 Gusto、Zenefits 和 BambooHR 在内的薪资提供商集成，使抵押贷款提供商和贷方(如汽车、个人贷款和学生融资)能够使用 Truework 的 API 在自己的应用程序中快速验证和批准借款人。该公司还支持对雇主和招聘公司进行背景调查，以核实潜在的新员工。

Truework 估计，今年它将处理超过 1200 万份收入和就业验证。

虽然 Trueworks 平台的很大一部分是为了支持所谓的“即时验证”，但有时并不总是可能做到这一点，这取决于个人的个人就业历史和状态，这意味着贷方必须恢复收集纸质工资单和文件。因此，Truework 最近推出了一款名为 Truework Documents 的新工具，它可以将用户以数字方式上传的物理文档转换为结构化数据，从而更容易评估欺诈风险。

Truework 的加薪发生在美国消费者借贷继续激增之际，其中汽车贷款占据了很大一部分。另一方面，在 2020 年的高点之后，抵押贷款需求最近创下了 20 年来的新低，但 Truework 的联合创始人兼首席执行官 Ryan Sandler 认为，对自动化验证的需求只会上升。

桑德勒在一篇博客文章中写道:“2020 年住房市场对抵押贷款领域的需求不断增长，这意味着企业需要更灵活的解决方案，能够快速扩展并帮助提高验证效率。”虽然 2022 年的需求速度有所放缓，但对自动化收入和就业验证的需求却在不断增加。对于消费贷款机构来说，日益激烈的竞争加上市场的不确定性，推动了对更强大、更准确的数据集的需求，以建立更具弹性的承保模型。"

Truework 创始人:维克多·卡布德邦(产品兼首席技术官)、瑞安·桑德勒(首席执行官)和伊桑·温切尔(首席运营官)。图片来源:真实工作

市场不确定性

最近在收入核实领域出现了一系列活动， Pinwheel 在 1 月 宣布获得 5000 万美元的资金，而 Argyle 在 3 月 吸引了 5500 万美元。目前的经济环境似乎在这一趋势中扮演了不仅仅是一个小角色。“上升的利率和市场的不确定性正在促使贷款人优化他们的审批程序，”G Squared 的合伙人斯潘塞·麦克劳德在一份新闻稿中说。

Truework 之前已经筹集了大约 4500 万美元，包括两年前的 3000 万美元。该公司表示，由于银行还有 5000 万美元的存款，它计划加强产品开发和员工人数。

除了主要支持者 G Squared 之外，Truework 的 C 轮投资还包括来自红杉资本、Activant Capital、Khosla Ventures 以及人力资本和 Four Rivers Group 的投资。
