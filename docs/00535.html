从不平衡的创造者经济和糟糕的安全性，到集中控制和不满的社区——Web 2.0 的缺陷在过去几个月里得到了充分展示。

首先，前脸书产品经理 Frances Haugen 上个月向国会作证说，这家社交媒体巨头“选择利润而非安全”然后，好像是在暗示，脸书的集中式服务在全球范围内崩溃了。中断如此广泛，以至于脸书甚至无法访问服务器本身。

然后，一名心怀不满的匿名黑客发布了亚马逊旗下热门流媒体服务 Twitch 的大量内部数据。除了顶级创作者的源代码和支付信息，黑客还敦促进行改进，称该社区为“令人恶心的有毒污水池”，试图“在在线视频流媒体领域引发更多破坏和竞争。”

尽管这些平台有所增长、覆盖范围和盈利能力，但保守派犯了很多错误，这一点从未如此明显。Web 2.0 的集中版本，完全是关于网络效应、大规模和赢家通吃的经济学，不再为社会服务。

是时候做出改变了。随着 Web 3.0 企业家构建一个开放的基础设施，促进更具协作性、创造性和以用户为中心的互联网，我们有责任解决上一代技术的根本缺陷。

以下是 Web 3.0 如何解决当前数字经济中一些最突出的问题。

安全性和数据控制差

Twitch 继续受到恶作剧的困扰，比如这个用亚马逊亿万富翁创始人杰夫·贝索斯的照片替换背景。显然，这些安全问题是地方性的，因为前雇员报告说这家流媒体巨头的安全措施很差。

我们都已经意识到，我们与中央实体共享的任何数据都存在风险。多年来从银行、零售商和社交平台泄露的私人信息表明，你不应该指望互联网上的任何东西都是真正隐私的。

因为 Web 3.0 是建立在加密原语之上的，并且通常以开放源代码为特色，所以任何人都可以通过审查代码来为项目做出贡献。这提高了用户的安全性，并将透明度转化为竞争优势。收益不仅仅是基于隐私，而是实际上保护了用户的价值。安全研究员 @samczsun 已经识别出跨 0x、Livepeer、Kyber、Nexus Mutual、Aragon、Curve 等协议的潜在漏洞，挽回了数十亿美元的潜在损失。

互操作标准意味着基于 ERC-721 的 NFT 可以在一系列不同的前端应用程序上交易和查看，ERC-20 令牌可以访问整个竞争金融产品生态系统，争夺注意力和价值。该机构增加了平台的风险，这可能会在安全漏洞后引发用户流失。

毒性和平台责任

尽管 Twitch 黑客的行为是非法和不道德的，但他们在一个方面是正确的:流媒体平台越来越有害，大型科技公司一直在努力应对这个巨大的问题。然而，在 Web 2.0 世界中，streamers 没有可行的替代方案。他们可以转到 YouTube 或脸书直播，但他们将从一个有毒的注意力经济平台换成另一个。

这些现实存在于一个环境中，在那里创造者比以往任何时候都有更大的权力。粉丝会跟随他们喜欢的创作者到他们选择的任何平台，这给了他们巨大的影响力。为了摆脱毒性，创作者需要 Web 3.0 工具走出围墙花园，通过与社区的直接关系来控制自己的命运。

Web 3.0 还重新平衡了用户和平台之间的权力动态，让用户控制他们的数据。有了像云杉这样的数据管理平台提供的互操作性和可移植性，Web 3.0 平台可以让用户更容易“用脚投票”，从一个平台转移到另一个平台。

像 Conflux 和 Moralis 这样的公司使得跨越区块链和标准的扩展变得更加容易，竞争对手可以在机会出现时迅速采取行动。例如，当用户发现 NFT 交易平台 OpenSea 可能是内幕交易，因为他们知道哪些非金融交易将被报道时，替代平台如 Artion 出现，以纠正 NFT 市场的一些感知的不满。这种对市场动态的快速反应在传统的 Web 2.0 生态系统中是不存在的，传统的 Web 2.0 生态系统依靠规模和封闭的访问来扼杀新进入者。

然而，Web 3.0 远远超出了直接的用户关系。这些平台是用户拥有的，由社区驱动，因此社区的激励措施是一致的，可以自我调节。在视频流的情况下，肯定没有社区希望仇恨袭击迫使其宝贵的成员在其他地方。

在 Web 2.0 的世界里，用户需要等待平台采取行动。在 Web 3.0 世界中，他们可以通过内置的治理和调节机制采取行动。在镜像博客平台上，用户每周投票决定谁可以写作和发表。在 Web 3.0 索引中，列出的项目可以控制后续项目的添加和删除，从而确保生态系统的健康增长。

这位脸书告密者还透露，脸书有一个双重司法系统，对待名人和普通用户是不同的。虽然一个普通帐户可能会因违反服务条款而受到惩罚，但一个有许多追随者的帐户可能会因同样的行为而逃脱惩罚。

Web 3.0 也解决了这个问题，由于区块链的不变性，它有了更大的透明度和审查阻力。决策通过 Snapshot 等工具公开做出，并由更广泛的社区推动。治理发生在链上，每个人都可以看到。没有幕后交易或双层司法系统(当然，除非经过投票)。这都是社区驱动的，所以如果参与者不同意透明度的方向或水平，他们可以简单地继续前进。

不平衡的创造者经济学

Twitch 泄露事件揭示了顶级表演者和日常创造者之间的经济差异。这些动态将平台和一小部分创造者之间的激励联系在一起。当少数创作者赚了大部分的钱时，该平台会将注意力推向最重要的影响者。

Web 3.0 范式通过民主化访问和消除创作者和粉丝之间的隔阂来解决这些错位的激励。Web 3.0 为创作者提供的货币化机制，如 NFTs、数字支付、代币和众筹，以一种对创作者友好的方式创造了公平的竞争环境。使用像 glass.xyz 这样的平台的艺术家们发现，他们可以通过伴随着引人入胜的直播流的 NFTs 来货币化他们的内容，这比他们仅仅通过 Web 2.0 模式销售要好得多。

在 Web 3.0 中，用户也可以拥有自己的平台，通常通过令牌进行协调。由于他们直接受益于平台的增长，他们有动力提供关键的服务，比如适度。

用户还可以购买粉丝代币，进一步向他们喜欢的创作者承诺，并建立一个积极的反馈回路，培养围绕共同的激情建立的健康的粉丝社区。像 Rally、socos(建立在 Chiliz )和 Roll 这样的平台给创作者提供了直接将他们的声誉、权威和创造力货币化的工具，而无需中介。这进一步调整了激励机制，因为创造者就是平台。创建者可以定义参与的规则，并做需要做的事情来维护一个健康的社区，而不受无私的第三方的干扰。

互联网升级对社会有益

毫无疑问，社会的社会结构和经济结构如此依赖于由少数私人公司控制的基础设施，这是有害的。当这些公司逃避责任，承诺改变时，一旦注意力消退，这些承诺似乎就会半途而废，这种损害就会进一步加剧。

但这并不意味着完全废除 Web 2.0，而是给社会一个急需的互联网升级。Web 2.0 工具本身有巨大的积极影响，但也有权衡:Web 2.0 的结构性激励导致了不负责任和不灵活的大型技术垄断。Web 3.0 发展了互联网，通过整合所有用户的经济和激励机制，吸取其前身的优点并加以改进，从而避免了广告支持模式的负面影响。

当涉及到文化和控制时，分散的服务比集中的看门人有显著的优势。Web 3.0 提出了一种全新的方式来培育社区，为用户提供数据可移植性和互操作性，并重新调整支持自我调节社区的激励机制。

Haugen 说得对:“这不是要分裂脸书。前进的道路是关于透明度和治理的。”我们都应该得到更好的，优先考虑透明度和社区治理的服务、平台和产品将在下一个数字经济时代蓬勃发展。
