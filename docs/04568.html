AssemblyAI 斥资 2800 万美元购买一体化 API 来转录、总结和调节音频 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/03/04/assembly-ai-snags-28m-for-all-in-one-api-to-transcribe-summarize-and-moderate-audio/?tpcc=tcplustwitter

在过去的几年中，音频和视频内容和界面的爆炸是显而易见的，但是在幕后处理所有这些媒体的方法却没有跟上。AssemblyAI 由 2800 万美元的新资金支持，旨在成为分析语音的首选解决方案，提供极其简单的 API 访问，用于转录、总结和以其他方式一次找出数千个音频流中发生的事情。

多媒体在如此短的时间内成为了许多事情的标准:电话和会议变成了视频通话，社交媒体帖子变成了 10 秒钟的剪辑，聊天机器人学会了说话和理解语音。无数新的应用程序正在出现，就像任何新的和不断增长的行业一样，人们需要能够处理这些应用程序产生的数据，以便很好地运行它们或在它们的基础上构建新的东西。

问题是音频并不容易处理。如何“搜索”音频流？您可以查看波形或浏览波形，但更有可能的是，您想先转录波形，然后搜索生成的文本。这就是 AssemblyAI 介入的地方:虽然有许多转录服务，但将它们集成到您自己的应用程序或企业流程中通常并不容易。

AssemblyAI 首席执行官兼联合创始人迪伦·福克斯(Dylan Fox)表示:“如果你想对内容进行审核，或搜索，或总结音频数据，你必须将这些数据转化为更灵活的格式，并且你可以在此基础上建立功能和业务流程。”“所以我们想，让我们建立一个超精确的语音分析 API，任何人都可以调用，甚至在黑客马拉松上——就像 Twilio 或 Stripe 风格的集成。人们需要很多帮助来构建这些功能，但他们不想把一堆提供商粘在一起。”

AssemblyAI 提供了一些不同的 API，您可以非常简单地调用它们(一两行代码)来执行任务，如“检查此播客中的禁止内容”，或“识别本次对话中的发言者”，或“用不到 100 个词总结本次会议”

编码，调用，完成。图片来源: AssemblyAI

你很可能像我一样，怀疑一个小公司能生产出如此简单地完成如此多任务的工作工具，考虑到一旦你进入这些任务，它们会变得多么复杂。福克斯承认这是一个挑战，但表示这项技术在短时间内取得了长足的进步。

“这些模型的准确性有了快速提高，尤其是在过去几年里，”他说。“总结、情感识别……它们现在都非常好。我们实际上正在推动最先进的技术——我们的模型比外面的更好，因为我们是少数几家真正进行大规模深度学习研究的初创公司之一。仅在未来几个月，我们就将在 R&D 和培训的 GPU 和计算上花费 100 多万美元。”

它可能更难直观地掌握，因为它不那么容易演示，但语言模型已经出现，就像图像生成(这 __ 不存在)和计算机视觉(Face ID，安全摄像头)一样。当然，GPT-3 是一个熟悉的例子，但福克斯指出，理解和生成书面文字实际上是一个完全不同的研究领域，而不是分析谈话和随意的讲话。因此，尽管机器学习技术的相同进步(如变形金刚和新的更有效的训练框架)对这两者都有贡献，但它们在大多数方面都像苹果和橙子。

无论如何，结果是，只需调用 API，就可以对几秒钟或一小时长的音频剪辑执行有效的调节或总结过程。当你构建或集成一个功能时，比如短格式视频，这非常有用——如果你预计每小时会有 10 万个剪辑被上传，那么你第一次确保它们不是色情、诈骗或重复的过程是什么？在您构建该流程的过程中，发布会推迟多长时间？

相反，福克斯希望，处于这种地位的公司将寻找一种简单有效的前进方式，就像他们面临增加支付流程时可能会采取的方式一样。当然，您可以从头开始构建一个，或者您可以在大约 15 分钟内添加 Stripe。这不仅从根本上来说是可取的，而且它清楚地将它们与更复杂的多服务包区分开来，这些包定义了微软和亚马逊等大提供商的音频分析产品。

有问题的狐狸。图片来源:延斯·潘杜罗

该公司已经拥有数百名付费用户，去年收入增长了两倍，现在每天处理一百万个音频流。“我们 100%现场直播。有一个巨大的市场和巨大的需求，消费者的消费就在那里，”福克斯说。

这一轮 2800 万美元的融资“由 Accel 牵头，Y Combinator、约翰和帕特里克·科利森(John and Patrick Schmidt)、纳特·弗里德曼(Nat Friedman)和丹尼尔·格罗斯(Daniel Gross)参与其中。”该计划是将所有这些零分布在招聘、R&D 基础设施和建立产品管道上。正如 Fox 指出的那样，该公司将在未来几个月内在 GPU 和服务器上花费 100 万美元，一批 Nvidia A100s 将为难以置信的计算密集型研究和培训过程提供动力。否则你就要为云服务付费，所以最好早点撕掉那个创可贴。

至于招聘，我认为他们可能很难与谷歌和脸书等公司直接竞争，这些公司当然也在努力开发自己的音频分析管道。然而，福克斯很乐观，他觉得那里的文化可能是缓慢而令人窒息的。

“我认为真正优秀的人工智能研究人员和工程师肯定有一种渴望，希望在最前沿工作——以及生产中的最前沿，”他说。“你想出一些创新的东西，几周后就投入生产……创业公司是你能做这种事情的唯一地方。”
