敏捷的下一个数字机器人将会有脸和手

原文：https://web.archive.org/web/https://techcrunch.com/2022/07/21/agilitys-next-digit-robot-will-have-a-face-and-hands/

据联合创始人达米恩·谢尔顿和乔纳森·赫斯特介绍，由 Agility Robotics 开发的双足机器人 Digit 将继续进化和改进，包括增加一个头部和一些自己的手指。

只是不要指望数字会说话，或者有看起来像人手的数字。

Digit 于 2019 年推出，最初似乎注定要过最后一公里送货的生活。最近，这家从俄勒冈州立大学(Oregon State University)剥离出来的初创公司已经将重心转移到了物流领域。其目标是:将 Digit 转变为一个通用工作平台，如打开卡车包装和在仓库中移动箱子。

谢尔顿在周四的 TC Sessions: Robotics 上接受采访和演示 Digit 时表示:“我们对 Digit 的整体愿景是，作为一个平台，让你可以将物理工作转化为软件应用。”

这一愿景意味着敏捷性是在软件方面不断改进，并在有意义的地方添加硬件。

“我们现在正在做的应用是全垒打。在演示过程中，Digit 拿起一个手提袋，把它放到传送带上。“当我们以人类的速度可靠地做这些事情时，这才是公司真正的价值所在。我们已经有了很多解决方案。”

兼任 CTO 的 Hurst 指出，未来的改进可能包括 Digit 能够前往其充电坞充电或完成其他通用任务。而说到手和脸，赫斯特说不会像人类。

“对于我们想要实现的用例来说，它们将是非常具体的东西，”赫斯特解释道。“我们也要在机器人上有点头。除了我们所做的一切的主题，也就是物理第一和功能第一，这是关于向人们指示机器人将要做什么。这也是关于把天线和传感器放在那里。所以这些东西只是继续完善和进化机器人的功能。”

Digit 也将有一个长着一双数字眼睛的头，这将帮助人们理解它的方向和意图。机器人确实有一个扬声器，但它不会说话。

“我认为现在还不是和机器人说话的时候，”赫斯特说。“你知道，如果你和 Siri 交谈，或者你和 Alexa 交谈，他们并不总是做你想要的事情。我想确保我们管理好预期，让人们像对待工作动物一样对待它，也许它是合作伙伴，但不是同伴。”

这家初创公司最近在由 Playground Global 和 DCVC 牵头的一轮融资中筹集了 1.5 亿美元，其中包括亚马逊工业创新基金(Amazon Industrial Innovation Fund)，目前正忙着将这笔钱投入使用。Shelton 指出，自 1 月份以来，该公司的规模已经翻了一番，员工人数超过 130 人，并正在投资其他领域，如垂直整合自己的制造流程，实现云车队管理，与客户的系统合作，以及与合作伙伴位置的仓库管理系统集成。

谢尔顿表示，尽管 Agility 专注于仓库，但它不会放弃交付。

谢尔顿说:“我们实际上仍然非常强烈地感觉到，最后一英里的递送可能是机器人的永久家园，因为相对于其他解决方案，腿在这项任务中是非常独特的。”这是为数不多的可以在 100%的人类环境中导航的技术之一，因为我们有能力在这些环境中导航。因此，即使你假设一个未来 50 年的世界，所有的仓库都是全新的绿地建筑，完全自动化，仍然需要机器人在户外环境中移动。"

他补充说，从部署和安全角度来看，专注于室内任务更容易。

谢尔顿说:“现实情况是，这个市场太大了，它能让我们忙上好一阵子。”
