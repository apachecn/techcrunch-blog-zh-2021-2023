微软在 Windows 11 TechCrunch 中推出亚马逊应用商店预览版

原文：https://web.archive.org/web/https://techcrunch.com/2022/02/15/microsoft-rolls-out-access-to-the-amazon-appstore-preview-in-windows-11/

去年 10 月，微软开始与亚马逊及其应用商店合作，在 Windows 11 电脑上测试安卓应用。然而，该功能只对参与 Windows 11 Insider 计划的测试者开放，并且仅限于大约 50 个应用。如今，随着亚马逊 Appstore 预览版在微软商店的到来，该公司正在向更广泛的受众开放，现在将提供 1000 多种应用和游戏。

除了这次发布，微软还推出了精选的 Windows 任务栏改进和重新设计的媒体播放器和记事本应用程序。

最初，Android 应用测试人员可以试用亚马逊的 Kindle 应用、华盛顿邮报、王者冲突、硬币大师和乐高得宝世界等应用。这种体验是通过新的 Android Windows 子系统建立在 Android 平台上的，由英特尔的桥接技术提供支持。不过，AMD 和英特尔设备都受支持，前提是它们满足运行 Windows 11 的硬件要求。

今天早上早些时候，亚马逊 Appstore 预览版的扩展访问开始推出，因为需要进行工程准备工作来展示体验。但是完整的集成体验还没有实现。

现在，美国的 Windows 11 用户将能够访问亚马逊应用商店预览版，并与亚马逊合作下载 1000 多种应用和游戏，包括 Audible、Subway Surfers、Lords Mobile、Khan Academy Kids 等顶级应用。首先，用户需要打开，然后更新他们的 Microsoft Store(单击 Microsoft Store >库中的“获取更新”)。)然后你就可以搜索自己喜欢的应用和游戏，通过亚马逊 Appstore 下载。这个想法是，应用程序将以一种让它们感觉像是 Windows 的一部分的方式运行，因为它们将与 Windows 输入和窗口体验相集成，比如快照布局。

任务栏的改进也在今天推出。Windows 11 中重新设计的任务栏版本收到了很多来自用户的抱怨，他们不喜欢任务栏现在被锁定在屏幕底部，并且不允许用户添加自己的快捷方式。微软听到了反馈，并一直在计划解决这个问题，但今天的更新是关于解决一系列不同的问题。

任务栏现在将在辅助监视器的任务栏上显示时钟和日期；它现在会在任务栏的左下角显示实时天气内容；微软团队的客户(拥有工作或学校账户)将能够静音、取消静音并从任务栏共享任何窗口，以使视频通话更加流畅。

这些功能最初将在 2022 年 2 月的非安全预览版中提供。

与此同时，媒体播放器应用程序为键盘用户提供了改进的键盘快捷键和访问键，以及其他可访问性改进。记事本应用程序获得了一个新的用户界面，具有 Windows 11 的设计功能，黑暗模式支持，简化的菜单和其他所需的功能，如多级撤销，丰富多彩的表情符号和更好的查找和替换体验。

用户可以通过进入“设置”>“更新与安全”>“Windows Update ”,然后选择“检查更新”来请求这些更新。

微软表示，除了年度更新之外，它计划在 2022 年更频繁地更新 Windows 11。
