Twitter 称其 API 的基本层将每月花费 100 美元

原文：https://web.archive.org/web/https://techcrunch.com/2023/02/08/twitter-says-the-basic-tier-of-its-api-will-cost-100-per-month/

Twitter 表示，其 API 的基本层将每月花费 100 美元

在 Twitter 宣布关闭免费访问 API 一周后，该公司今天表示，将对 API 的基本层每月收取 100 美元。这将使开发者获得“低水平的 API 使用”——没有具体说明这到底是什么意思——和 Ads API。

该公司曾计划在 2 月 9 日关闭其 API 的免费访问，现在它已将这一期限延长至 2 月 13 日。但是没有关于 API 定价重组和访问级别的细节，这种扩展似乎是象征性的，因为开发人员将无法计划他们的变化。

上周的声明招致了来自开发者的批评——，尤其是那些制作有趣的机器人发布信息或图片的人。后来，埃隆·马斯克表示该公司将向发布“好”内容的机器人提供免费 API。在最新的一系列公告中，Twitter 表示，它将向此类机器人开发者提供一个轻型只写 API，每月最多发布 1500 条推文(或每小时两条推文)。

Twitter 还补充说，它将在 2 月 13 日对其 premium API 进行贬值，该 API 是 v1.1 的一部分。虽然该公司表示，开发人员可以申请企业访问，但尚不清楚订阅 API v2 提升访问权限的人是否会受到影响。

虽然新的声明为基本的 API 层和机器人奠定了基础，但学术研究存在许多不确定性。在以前的管理下，该公司为研究人员提供了 API v2 的特殊访问权限。然而，没有信息表明，随着社交网络停止提供免费 API 层，访问权限将被取消。研究人员依靠这些数据来表明平台上仇恨言论或错误信息的趋势。因此，这些研究人员获得访问数据的许可以观察 Twitter 上的安全问题至关重要。

除此之外，人们还指出，为地震等自然灾害构建解决方案的工程师也依赖于 Twitter API。因此，停止免费层将影响这些解决方案。

让 API 访问付费是马斯克为 Twitter 带来更多资金的最新举措。该公司现在已经将 Twitter Blue 扩展到 15 个国家。根据最近的报道，在美国只有 180，000 人注册了该服务。 The Information 指出，按照这个速度，Twitter 今年只能从订阅中获得 2，780 万美元。
