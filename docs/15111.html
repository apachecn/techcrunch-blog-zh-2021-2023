Reddit 称黑客在网络钓鱼攻击后获取了员工数据

原文：https://web.archive.org/web/https://techcrunch.com/2023/02/10/reddit-says-hackers-accessed-internal-data-following-employee-phishing-attack/

Reddit 称黑客在网络钓鱼攻击后获取了员工数据

Reddit 已经证实，在一次“高度针对性的”网络钓鱼攻击后，黑客访问了内部文档和源代码。

Reddit 首席技术官克里斯多佛·斯洛维(T3)在 T2 的一篇帖子解释说，2 月 5 日，公司意识到针对 Reddit 员工的“复杂”攻击。他说，一名身份不明的攻击者发送了“听起来似乎合理的提示”，将员工重定向到一个伪装成 Reddit 内部网门户的网站，试图窃取凭据和双因素认证令牌。

斯洛维表示，最近有报道称“类似的网络钓鱼企图”，但没有提到具体的例子，但他将这次入侵与最近的 Riot Games 黑客事件相提并论，攻击者使用社会工程策略来访问该公司遗留的反作弊系统的源代码。

Reddit 表示，黑客成功获得了一名员工的凭据，使他们能够访问内部文档和源代码，以及一些内部仪表盘和业务系统。

斯洛维表示，该公司是在被钓鱼的员工向 Reddit 的安全团队自我报告了这一事件后得知这一违规行为的。Reddit 迅速切断了渗透者的访问，并开始了内部调查。

拥有超过 5000 万日常用户的 Reddit 表示，其调查发现，数百名现任和前任员工的一些联系信息，以及一些广告客户信息也被访问。Reddit 表示，它“没有证据”表明个人用户数据和其他非公开数据被窃取、发布或在网上传播。

无论如何，Reddit 建议所有用户在他们的账户上设置双因素认证，并使用密码管理器。“除了提供非常复杂的密码，它们还能在你在钓鱼网站上使用密码之前发出警告，从而提供额外的安全保障，”斯洛维说。

“我们正在继续调查和密切监控情况，并与我们的员工合作，以加强我们的安全技能，”他补充说。“众所周知，人类往往是安全链中最薄弱的一环。”

Reddit 在 2018 年经历了一次更严重的数据泄露，攻击者访问了 Reddit 自 2007 年以来的完整数据副本，包括该网站运营的前两年。这包括用户名、散列密码、电子邮件、公开帖子和私人信息。
