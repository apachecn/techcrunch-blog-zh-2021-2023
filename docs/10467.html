创业公司正在利用现有销售平台的 DevOps 工具建立业务

销售工具不断增长的市场催生了一个奇怪的家庭手工业:专门针对销售和营销功能软件的 DevOps 创业公司。

(在这里，“DevOps”指的是软件开发和 IT 团队之间自动化过程的工具。)它本身已经成为一个蓬勃发展的领域，销售 DevOps 平台软件的供应商(包括 Salesforce)声称，销售技术可以更容易地集成到公司现有的工作流程中。

但是，等等，你可能会说:销售软件的承诺不是最大限度地减少对定制编码的需求以完成与销售相关的任务吗？嗯，是的——在某种程度上。

以 Salesforce 为例。它原本是一个客户关系管理平台，随着时间的推移，它演变成了一个开发平台——但没有许多组件来适应组织的特定需求。
