Mozilla 正在关闭其虚拟现实网络浏览器 Firefox Reality TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/02/03/mozilla-is-shutting-down-its-vr-web-browser-firefox-reality/

一款顶级 VR 网络浏览器即将关闭。今天，Mozilla 宣布关闭其 Firefox Reality 浏览器——这款已有四年历史的浏览器专为虚拟现实环境打造。该技术允许用户通过虚拟现实耳机访问网络，使用虚拟现实手柄而不是鼠标来访问 URL、执行搜索以及浏览 2D 和 3D 互联网。

Firefox Reality 于 2018 年秋季首次推出，已通过其各种应用商店在 Viveport、Oculus、Pico 和 HoloLens 平台上推出。虽然能够在 2D 网上冲浪，但人们的期望是用户将主要使用新技术来浏览网络的 3D 内容并与之互动，例如 360 度全景图像和视频、3D 模型和网络虚拟现实游戏。但是在今天发布的公告中，Mozilla 表示该浏览器将在“未来几周”从商店中移除

相反，Mozilla 正引导那些仍想在虚拟现实中使用网络浏览器的用户转向 Igalia 即将推出的开源浏览器 Wolvic，该浏览器基于 Firefox Reality 的源代码。这款浏览器将于下周开始提供下载，因此用户不必离开——他们只需进行切换。

Igalia 吹嘘其对 XR 空间的承诺——XR 是一个涵盖虚拟现实、增强现实和类似技术的总括术语。但它自己的声明暗示，该公司认为 Mozilla 在这方面的努力已经停滞，并指出它对进行实验和“作为一个完整的项目继续这项工作”感到兴奋

“我们 Igalia 相信网络在很多方面对 XR 领域都很重要。提供沉浸式操作系统的 XR 系统需要网络浏览器成为其中的一部分，”Igalia 网站上的公告称。“进入一个‘现实’而不能接触到网络上已经存在的一切将是非常可怕的。此外，WebXR 为导航、共享和体验来自浏览器本身的信息开辟了新的途径。为沉浸式操作系统重新设计浏览器是一个新领域，这种新意味着浏览器的选择目前是有限的，”它说。

最初，Wolvic 浏览器将在 Oculus、HTC Vive Focus、Pico Interactive、Daydream、华为 VR 眼镜和开源 Lynx 设备上运行。然而，下周推出的浏览器仍将处于“测试”阶段，因为它正在努力将 Mozilla 之前提供的一些功能转移到自己的浏览器上。

Igalia 已经获得了未来两年开发浏览器项目的部分资金，但表示需要找到更多的合作伙伴，才能成功构建一个健康的生态系统。它邀请那些有兴趣的人通过电子邮件联系。

在解释为什么决定关闭 Firefox Reality 时，Mozilla 表示，虽然它确实有助于开发新技术，如 WebVR 和 WebAR，但它并不总是继续长期托管和孵化这些技术。在某些情况下，它将努力寻找其他可以继续其项目的组织，例如 WebAssembly、Rust 和 Servo。当然，作为一个仍然因曾经大受欢迎的 Firefox 网络浏览器而为主流消费者所熟知的组织，它正在将其最新浏览器技术的未来交给另一家公司来运营。
