超现实数据库为其数据库即服务产品筹集了 600 万美元 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2023/01/04/surrealdb-raises-6m-startup-funding-database-as-a-service/amp/

图片来源: koto_feja / Getty Images

托管数据库服务市场上有许多供应商，而且许多供应商资金雄厚。看看的 SingleStore ，它在去年 10 月筹集了 3000 万美元，将其数据库技术带给新客户。还有 EdgeDB ，它在 11 月推出云数据库产品之前获得了 1500 万美元。

托比摩根希区柯克认为，至少还有一个球员的空间。他是超现实数据库的两位联合创始人之一，这是一个数据库即服务平台，在查询数据时提供了极大的灵活性。

希区柯克可能有偏见。但有第三方证据表明，公司越来越多地采用完全托管的、基于云的数据库服务。在最近的 MariaDB 调查中，61%的受访者表示，他们要么已经完全迁移到云，要么正在努力完成向云的完全数据库迁移，这是出于节省资金和“弥合云技能差距”的共同愿望

超现实数据库显然受益于某种繁荣。在自举三年后(尽管是预收入)，超现实数据库最近完成了一轮 600 万美元的种子融资。FirstMark 领导了它。

希区柯克在接受电子邮件采访时告诉 TechCrunch:“2015 年，在多年来构建基于云的软件即服务系统(具有实时 API、复杂的安全权限和多个独立的数据库后端)之后，我和我的兄弟杰米(Jaime)质疑是否有一个平台可以更快地构建和扩展应用程序，同时仍然允许以结构化而灵活的方式存储和查询数据——就像数据库而不是 API 一样。”“我们开始构思和规划超现实数据库需求，从我们在以前项目中使用的一系列数据库中获得灵感。”

在共同推出超现实数据库之前，托比和詹姆已经尝试了许多项目，包括一个让高尔夫球场跟踪高尔夫球员如何打每个洞的应用程序。他们还共同创立了 Hire Insight，这是一种公司可以用来在线评估、策划和选择求职者的服务，类似于 LinkedIn Recruiter 和 Workday。

Tobie 说，超现实数据库的目标是通过减少构建后端 API 和数据库层或使用云数据平台或单一数据模型的需求来改善应用程序开发流程。(数据模型是一种抽象模型，它组织数据元素并标准化元素之间的相互关系。Tobie 说，为此，超现实数据库支持实时查询、多用户访问的安全许可和“高性能”分析工作负载。

超现实数据库的在线配置仪表板。图片来源:超现实数据库

客户端应用程序可以直接连接到超现实数据库，而传统的服务器端开发设置可以利用平台的查询和分析能力。Tobie 说，在建立数据模型时，超现实数据库用户可以根据存储数据的性质，在简单文档、嵌入字段的文档或记录之间的相关图形连接之间进行选择。

“从数据和技术的角度来看，超现实数据库提供了……以多种方式查询个人数据的能力，”托比说。“此外，超现实数据库能够从数据库内部处理业务逻辑和用户验证。正因为如此，超现实数据库使开发人员和组织能够简化后端技术堆栈，加快开发速度，并降低复杂的多互联后端平台的成本。”

在过渡到云产品之前，超现实数据库首先作为开源数据库平台推出，尽管开源包仍然可用并正在积极开发中。目前的客户包括“许多”初创公司和“公开上市”的企业，尽管 Tobie 不会指名道姓；在 2023 年初推出之前，超现实数据库的完全托管数据库服务仍处于测试阶段。

托比说:“(来自种子期的)资金将用于壮大团队，并推出(超现实数据库云产品的)商业产品。”我们特意选择了超现实数据库的技术堆栈，以便超现实数据库可以快速投入使用，易于新工程师理解，并尽可能使用最少的不同技术……考虑到这一点，超现实数据库能够快速迭代新功能和新版本，同时能够与敏捷灵活的团队一起完成。"

值得一提的是，FirstMark 的马特·图尔克对此表示赞同。当然，他是一个投资者，但他也提到了数据库即服务产品的巨大且不断增长的市场，根据市场和市场的预测，到 2025 年，这个市场的价值可能高达 248 亿美元。图尔克雄心勃勃地认为，超现实数据库不仅要与数据库供应商竞争，如 MongoDB 、 Neo4j 、 Couchbase 和 DynamoDB ，还要与后端服务提供商竞争，如 Firebase 、 Supabase 和 Nhost。

“超现实数据库是一家雄心勃勃的公司，它构建了一个多模型数据库，将许多人认为不兼容的东西结合在一起，而没有重大的折衷。他们受到数据库抽象、云和无服务器等主要趋势的推动，”图尔克在一份电子邮件声明中说。“他们显然引起了开发人员社区的共鸣——我们几乎从未见过任何数据库开源项目增长如此之快。最后但同样重要的是，我们喜欢这家公司的创始故事，它是由两兄弟自己创建的，我们认为他们作为创始人有很大的潜力。”

超现实数据库目前有一个三人团队，目标是在明年内将员工人数增加到 18 人左右。
