脸书首次推出用于第三方应用共享的卷轴 API

Meta 正全力以赴扩大其短视频产品的覆盖范围。在这些产品广泛普及的最新步骤中，该公司推出了一个脸书卷轴 API，供开发者集成到他们的应用程序中。这将允许用户使用“共享到卷轴”功能，从第三方应用程序直接共享脸书页面上的卷轴。

“脸书卷轴 API 将使第三方平台能够直接启用‘分享到卷轴’功能，为之前使用第三方平台的用户消除摩擦，然后在稍后上传到脸书。该 API 将允许直接从他们首选的第三方平台一键分享，”该公司在一篇博客文章中说。

如果开发者集成了这一功能，用户可以使用第三方应用来发布卷轴，并发布到他们管理的页面上。然而，这个 API 不允许应用程序向群组或个人用户发布卷轴。这可能对视频编辑应用程序、社交媒体管理工具和企业社交媒体用户有用。

Meta 表示，其三个发布合作伙伴已经在他们的平台上集成了新的 Reels API:创造者经济平台杰利马克，社交媒体管理工具萌芽社交和网站创建平台 Wix 。

在新 API 发布之前， Meta 在 6 月也发布了类似的声明。该公司推出了 Instagram Reels API，供开发者将卷轴直接发布到 Instagram 商业账户。上个月，这家社交网络巨头推出了一项功能，允许用户将视频从 Instagram 交叉发布到脸书。它还为脸书推出了一个自动生成卷轴的工具，利用你以前分享的故事制作了一个记忆片段卷轴。

梅塔和马克·扎克伯格指望卷轴成为公司的摇钱树。在 Q2 2022 年的结果中，扎克伯格提到用户在 Instagram 上花费 26%的时间观看卷轴。此外，该公司在卷轴广告方面的收入超过 10 亿美元。
