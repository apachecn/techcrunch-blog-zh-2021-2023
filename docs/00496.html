当你被编程为从 A 点到 b 点时，礼貌实际上并不算什么。但如果机器人将在人类社会中扮演越来越重要的角色，那么它们与我们其他人相处的精确度就会出现问题。

“机器人很快就会生活在我们的世界里，它们真的需要学习如何以人类的方式与我们交流，”麻省理工学院 CSAIL 研究科学家 Boris Katz 在一份与一项新研究论文相关的声明中说。“他们需要明白什么时候他们需要帮助，什么时候他们需要看看自己能做些什么来防止事情发生。”

该团队称这篇论文为“第一次非常严肃的尝试，试图理解人类和机器的社交互动意味着什么。”这种主张的有效性可能会引起争议；随着机器人开始在我们的生活中发挥越来越大的作用，它试图在非常早期阶段解决的问题无疑是机器人专家越来越多地考虑的问题之一。

研究人员在模拟环境中进行了测试，以开发他们认为机器人之间“现实和可预测”的交互。在模拟中，一个机器人看着另一个机器人执行任务，试图确定目标，然后在任务中试图帮助或阻碍它。

“我们已经为如何模拟两个主体之间的社会互动打开了一个新的数学框架，”项目负责人 Ravi Tejwani 在一份声明中说。“如果你是一个机器人，你想去位置 X，而我是另一个机器人，我看到你正试图去位置 X，我可以帮助你更快地到达位置 X。这可能意味着将 X 移近你，找到另一个更好的 X，或者在 X 采取你必须采取的任何行动。我们的公式允许计划发现“如何”；我们根据社交互动在数学上的意义来定义‘什么’。”

该模型目前是一个相对简单的 2D 模拟。该团队正在努力转向 3D 版本，同时添加一个基于神经网络的机器人规划器，以提高机器人从这些动作中学习的速度。
