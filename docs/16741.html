太快了！微软在人工智能支持的必应聊天中植入广告

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/29/that-was-fast-microsoft-slips-ads-into-ai-powered-bing-chat/

微软正在“探索”在 Bing Chat 给出的回复中植入广告，Bing Chat 是其新的搜索代理，由 OpenAI 的 GPT-4 驱动。虽然这些赞助回复被清楚地标注为这样，但它确实提出了一个问题:我们离搜索引擎广告的旧模式到底有多远。

微软在今天发表的一篇博客文章中证实了这一点，尽管是以实验的形式。以下是“一点背景知识”之后的相关内容，没有人会对此感到惊讶:

我们还在为出版商探索更多功能，包括我们的 7，500 多个 Microsoft Start 合作伙伴品牌。我们最近会见了一些合作伙伴，开始探索想法并获得反馈，了解我们如何继续以对我们的合作伙伴的流量和收入有意义的方式分发内容。随着我们寻求继续共同发展这一模式，我们分享了一些我们正在探索的早期想法，包括:
扩展的悬停体验，将鼠标悬停在某个出版商的链接上，将显示该出版商的更多链接，为用户提供更多参与方式，并为该出版商的网站带来更多流量。
对于我们的 Microsoft Start 合作伙伴，在聊天回答旁边放置一个丰富的 Microsoft Start 许可内容的标题，有助于推动更多用户参与 Microsoft Start 上的内容，我们与合作伙伴分享广告收入。我们还在探索在聊天体验中放置广告，与内容有助于聊天响应的合作伙伴分享广告收入。

几周前就有传闻说有广告，但是今天,的一条推特上，来自《拾遗》杂志的 Debarghya Das 在野外捕捉到了这些广告:

其他人指出，广告自推出以来就一直存在，但并不是对所有用户开放。我尽了最大努力召唤一个，但无济于事——我的账户似乎还没有启用它们。但很明显，这是未来的一个迹象。

让我们承认，搜索引擎，现在这些搜索代理，是商业模式的一部分，因此需要某种形式的货币化。但我认为，随着所有关于改革搜索和重新开始的讨论，我们都更期待一些比赞助答案更好的东西。

在上面这个广告的例子中，用户不清楚这里赞助了什么。TrueCar 是在销售这款车，并为新本田的搜索排名付费吗？或者他们只是一个参考或价格检查网站，付费成为 Bing 的首选汽车价格提供商？为什么这些价格比本田列出的价格高？微软是否因为不包含 Autotrader 或 Cars.com 列表而获得报酬？用户可以要求非赞助的结果吗？

这并不是说有什么邪恶的事情发生。但是一般来说，用户应该明白广告的内容。我们已经学会了如何解析搜索结果:广告通常会有一个小方框围绕在它们周围，并且位于一堆广告的顶部。你不一定要喜欢它才能理解它——通过理解它，你可以更明智地与他们接触(或脱离)。

在这种情况下，我希望有一个更好的“元对话”，如果你愿意的话，在有引文和广告(如果有的话)的聊天中加入一点思想泡泡。答案本身不应该被篡改——但是在 metacon 上你有几个链接，也许还有一个“如果你想买一辆新的 HR-V，在 TrueCar 的最后一天列出了 3 个”。(广告)"

更可怕的副作用是，目前的工具无法屏蔽此类广告。这并不是说他们根本不能被阻止——不难想象一个 uBlock 起源可以取代被标记为广告的聊天机器人响应，并请求另一个响应。但是，广告作为贴有标签的位置上的本质上可预测的项目的范例显然要结束了。

一方面，嗯…很好！每个人都讨厌广告，即使它们是网络(不可否认，包括这个网站)自我支付的方式。另一方面，这是一种新的、潜在的更微妙、更具颠覆性的广告形式，可能不容易被描述和忽略。

让用户想知道聊天机器人是否被付钱来说它所说的话是一种减缓或侵蚀信任的好方法，即使有“广告”标签。人们不禁想知道这些公司对他们的交易有多诚实。为什么他们不应该有一个影响结果但不会被贴上标签的优质广告层呢？这完全符合广告业的把戏。

虽然没有人指望微软或谷歌、亚马逊、Meta 和所有其他人出于好心来操作这些昂贵且计算量大的语言模型(假设他们有心且有善心)，但如果能在如何更好地整合广告方面投入更多的思考，那将是一件好事。

当整个模型都在改变时，显而易见的解决方案——碰巧很像你在过去使用的方案——不太可能是最好的。顺其自然暴露了公司的优先事项，这可能会导致不信任和怀疑。
