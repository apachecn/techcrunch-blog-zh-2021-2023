金融科技公司 Railsr(原名 Railsbank)融资 4600 万美元

原文：https://web.archive.org/web/https://techcrunch.com/2022/10/04/railsr-formerly-railsbank-raises-46m-for-embedded-finance-says-it-now-has-300-customers/

在过去几年中，嵌入式银行业务一直在增长，因为公司转向 API 来嵌入信贷、支付和存款等金融服务，以增加收入。现在，随着科技融资格局陷入严重动荡，嵌入式金融市场的一家公司已经筹集了一轮增长资金，突显出金融科技这一特定领域正在感受到的压力。

今年早些时候从 Railsbank 更名而来的伦敦初创公司 Railsr 已经筹集了 4600 万美元，其中 2600 万美元为股权，2000 万美元为债务。股权部分是一个内部回合，意味着所有重复投资者，Anthos Capital 领投，Ventura、Outrun Ventures、CreditEase 和 Moneta 也参与其中。Mars Capital 提供了债务，首席执行官兼联合创始人奈杰尔·弗登(Nigel Verdon)表示，债务将用于与股权相同的目的，但成本较低。“目前股票很贵，”他说。

该公司没有透露其确切的估值:Verdon 称其为当前市场的“公允价值”，并间接证实了与一年前的 B 轮融资相比，这是一轮下跌，当时筹集了 7000 万美元(也是由 Anthos 牵头)，Verdon 表示其估值接近 10 亿美元，称其为“近乎独角兽”

“这肯定不是去年的估值，”他谈到今天的 c 系列时说，“价格已经大幅下降。”

Verdon 在今天的一份声明中指出，该公司尚未盈利，并将此轮融资描述为“我们盈利道路上的重要一步”

市场状况似乎与 Railsr 本身的状况相差不远。其国内市场货币英镑兑美元汇率一直在下跌，许多人认为世界正在走向更大范围的衰退。购买活动在宏观层面上放缓，即使在更加悲观的环境下仍有机会为客户服务。

这对 Railsr 来说，既有积极的一面，也有消极的一面。

Verdon 说，Railsr 经历了一轮裁员，今年早些时候裁员约 14%，即 70 人。该公司还缩减了在东南亚和澳大利亚的业务，现在只剩下“骨干员工”，转而专注于英国和欧洲的业务。他说:“战术手册关注的是我们的优势。”。

另一方面，该公司任命了万事达卡前董事长里克·海索恩思韦特(Rick Haythornthwaite)为新董事长，今年上半年收入增长了 50%，预计 H2 的增长虽然有所放缓，但仍低于之前的预测，但仍朝着正确的方向发展。

与此同时，它的客户数量目前为 300 家，而一年前为 220 家，这些客户使用 Railsr 的 API 来支持信用、信用卡、忠诚度和更多经典的银行服务。

纵向 it 目标包括零售、场馆、体育和活动领域的公司；以及其他金融科技。一家金融科技公司可能不会建立自己的金融科技服务，这似乎有些奇怪，但通常情况下，这是因为该公司可能更专注于保险或薪资等其他领域，并利用嵌入式金融服务快速扩张到邻近的非核心竞争力领域。对于 neobanks 等更直接的银行提供商，业务重点可能是个性化和客户服务。因此，银行业务成为一种基本的(近乎商品化的)产品，与 API 集成比从头开始构建更容易、更快。

Railsr 在其网站的案例研究中引用了 Wagestream、Aviva 和赛车品牌 McLaren (via QtmPay ),并指出 HelloCash、Sodexo 和 Payine 都是其客户。提供服务和集成的合作伙伴包括 AWS、Salesforce、Visa、MasterCard 和 Plaid。

嵌入式金融通常是金融服务市场中较为乐观的领域之一，以至于即使是目前考虑到市场状况的研究似乎也对其增长持积极态度。Railsr 引用贝恩&公司本月早些时候的数据称，2021 年，嵌入式金融推动了美国约 2.6 万亿美元的金融交易，预计这一数字将继续增长，到 2026 年将超过 7 万亿美元。

这是现有投资者愿意再次支持 Railsr 的原因之一。

“很高兴看到 Railsr 作为旧金融的挑战者和嵌入式金融经济的创造者不断壮大，”Moneta VC 的联合创始人兼管理合伙人 Meirav Harnoy 说。“自从我领导 A 轮投资以来，Railsr 的客户、技术和人员给我留下了深刻的印象。我很期待接下来会发生什么。”
