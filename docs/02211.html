肯尼亚 BNPL 创业公司 Lipa 在筹集了 1200 万美元后，着眼于更多的非洲市场

原文：https://web.archive.org/web/https://techcrunch.com/2022/01/13/kenyan-bnpl-startup-lipa-later-eyes-more-african-markets-after-raising-12-million/amp/

图片来源: 大树 83 / Getty Images

肯尼亚科技主导的消费信贷平台 Lipa Later 在筹集了 1200 万美元的首轮融资前资金后，正计划向非洲新市场扩张。

股权和债务融资轮由 Cauris Finance、Lateral Frontiers VC(Lipa Later 的首批投资者之一)和 GreenHouse Capital 牵头，参与方包括 SOSV IV LLC、Sayani Investments 和 Axian Financial Services。

这家初创公司成立于 2018 年，是一家现在购买，以后支付(BNPL)的公司，现在计划进入坦桑尼亚，加纳和尼日利亚，并在现有市场，即肯尼亚，乌干达和卢旺达进行扩张。

“我们很高兴能与我们的投资者合作，因为我们希望在非洲发展和扩大市场。Lipa 后来的联合创始人兼首席执行官 Eric Muli 表示:“在未来的 12 个月里，我们希望在现有市场上实现增长并扩大一倍，同时在非洲开辟三到五个新市场。

Lateral Frontiers VC 的合伙人 Samakab Hashi 在评论最新一轮融资时说:“最新一轮融资使他们能够扩大产品供应，并使 Lipa 更接近于成为非洲大陆上主要的先买后付参与者。”

Lipa 后来与这些市场的零售商建立了独家合作关系，允许购物者分期付款购买产品。例如，Lipa Later 与法国零售商家乐福(在该地区有业务)的合作允许顾客以每月分期付款的方式购买家具、电子产品甚至易腐物品。

顾客为他们获得的信用支付月息(家乐福的月息约为 2.3%)。这家初创公司正计划与商家建立更多的合作伙伴关系，以覆盖非洲更多的国家。

(从右至左)家乐福东非区国家经理 Franck Moreau 和 Lipa Later 首席执行官 Eric Muli，标志着 Lipa Later 支持的 LipaVismart 产品的推出。图像演职员表:后来的李帕。

“Lipa Later 不仅改变了整个非洲的消费信贷格局，迄今为止，大多数人都无法获得这种信贷，而且还催化了购物、电子商务和支付的未来。他们以真正的产品和客户为导向的方式做到了这一点，使商家和消费者都受益，并证明了在多个市场上的不可思议的可扩展性，”温室资本合伙人 Ruby Nimkar 说。

Lipa Later 专有的信用评分和机器学习系统允许消费者注册并几乎立即获得信用额度。这家初创公司还建立了一个集成到电子商务平台的 BNPL API，使商家能够直接向消费者销售产品，并使消费者能够每月支付所购物品的分期付款。

这笔新资金是对总部位于东京的 Uncovered Fund 的一项未披露的 2020 投资的补充，该基金投资于非洲的早期和种子期初创公司。Lipa 后来成为获得这笔资金的五家创业公司之一。

Lipa Later 是肯尼亚 BNPL 市场的主要参与者之一，与 Aspira、Miti、Flexpay Technologies 和 Julla 竞争。根据这份关于肯尼亚的 2021 年调查，由于电子商务的快速渗透以及新冠肺炎疫情经济放缓的影响，肯尼亚的 BNPL 支付行业经历了强劲的增长。2021 年至 2028 年，肯尼亚 BNPL 行业的 CAGR 有望达到 30.8%。BNPL 商品总值预计将从 2020 年的 5160 万美元增长到 2028 年的 5.895 亿美元。

随着电子商务和替代信贷来源的增长，非洲的机会很多。在进军非洲的过程中，Lipa 后来将不得不与南非的 Payflex ( 公司，最近被澳大利亚的 BNPL Zip 公司收购)、PayJustNow 以及尼日利亚的 PayQart 和 Carbon Zero 竞争。
