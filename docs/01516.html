耐克收购 NFT 收藏品工作室 RTFKT

耐克公司正在深入秘密收藏品的世界，宣布他们将收购 NFT 工作室 RTFKT(发音为“神器”)。

收购公告来得正是时候；RTFKT 目前是本月最受关注的 NFT 项目之一的幕后推手——与艺术家村上隆(Takashi Murakami)建立了名为 CloneX 的全面化身合作关系。根据 crypto tracker CryptoSlam 的数据，自不到三周前首次下跌以来，该项目已经实现了近 6500 万美元的交易额。

收购条款没有披露。这家初创公司在 5 月份筹集了 800 万美元的种子资金，由 Andreessen Horowitz 牵头，对该公司的估值为 3330 万美元。

“这是一个建立 RTFKT 品牌的独特机会，我们很高兴能够受益于耐克的基础实力和专业知识来建立我们热爱的社区，”RTFKT 联合创始人 Benoit Pagotto 在一份声明中说。

本月早些时候，阿迪达斯宣布与 NFT 项目无聊猿游艇俱乐部合作。

除了建立自己的 NFT drops，RTFKT 还与其他加密创作者合作，设计了一些项目，如利用 NFT 其他项目中的图像的实体鞋，包括 CryptoPunks 和 Bored Apes。
