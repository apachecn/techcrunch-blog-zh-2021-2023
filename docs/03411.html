Stitch 为其 API 基础设施和嵌入式金融平台筹集了 2100 万美元

原文：https://web.archive.org/web/https://techcrunch.com/2022/02/13/stitch-raises-21m-for-its-api-infrastructure-and-embedded-finance-platform/

随着企业利用 API 访问客户的金融账户，并提供各种集成和嵌入式金融服务，开放金融仍然是金融科技领域的一个重要主题。在最新的进展中，Stitch——在非洲构建和运营这些 API 的知名企业之一——向 TechCrunch 证实，它已经筹集了 2100 万美元的首轮融资。

这则新闻是在 2021 年 2 月秘密筹集了 400 万美元资金之后，这是四个月前获得 200 万美元扩展轮的的前传，使其种子轮达到 600 万美元。迄今为止，史迪奇总共筹集了 2700 万美元。

南非 API fintech 使企业能够构建、优化和扩展金融产品。根据该公司发布的一份声明，它计划利用新的资金在非洲各地创建一个 【财务图】 生态系统。

该公司将财务图描述为财务构建模块的基础设施，允许企业编写一次代码，在多个市场推出，并基于跨地区、提供商、银行和其他类型的财务账户的互操作性更快地扩展。

Stitch 首席执行官基安·皮莱在接受 TechCrunch 采访时表示:“我们在某种程度上将更广泛的金融生态系统视为一堆不同的节点——银行账户、金融科技公司(fintechs)等商家或最终用户——它们都有着内在的联系。”。“通常，我们认为地理位置和机构之间的这些联系尚不存在。我们努力做的很多事情就是在这些联系之间架起桥梁，让这些联系无处不在。”

Stitch 分三个阶段查看此图表。第一个是它秘密推出的东西:用一个 API 连接金融和银行账户的纯基础设施游戏。第二种是寻求收购商家和企业，在基础设施之上构建用例及应用。最后是让终端消费者通过这些业务链接他们的账户。

“从高层面来看，人们第一次从现金转向数字会导致更多的碎片化。我们的观点是，将所有这些都包含在一个网络或图形中有助于打开空间，打破孤岛，”皮莱说。“我们最终的想法是，人们可以轻松地在不同地区和机构的各种应用程序之间转移资金。”

Stitch's 为电子商务公司、市场和平台及其主要客户 fintechs 提供解决方案。与其他年份一样，非洲金融科技公司在筹集风险资本方面优于其他初创公司，特别是在创纪录的 2021 年，他们根据报告筹集了 50-60%的风险投资资金。它们的重要性怎么强调都不为过，像 Stitch、 Mono、 Okra、、 Plaid、和 OnePipe 这样的基础设施玩家——他们都在过去一年里筹集了资金，有些是种子资金，有些是 A 轮资金——对于推动整个大陆的金融包容性和支付便利性的集体增长至关重要。

这些企业客户将 Stitch 用于 KYC &入职、个人和企业财务管理、借贷、钱包充值和电子商务结账等用例。 平台的数据和身份产品允许企业访问客户交易历史和余额数据，验证账户信息，并执行欺诈检查。支付产品支持银行间转账，只需点击一次即可存入和支付款项。

一些客户包括基于钱包的公司，如 Chipper Cash 和Luno；像 ImaliPay 这样的嵌入式金融提供商；像 FlexClub 这样的订阅平台；以及像 Yoco 这样的支付聚合商。

2021 年 4 月，Stitch 在南非推出了其支付产品，并注意到在接下来的六个月中支付量每月增长 50%。10 月，Stitch 将支付产品扩展到了尼日利亚，当时有望在 12 月前实现每月 1000 万美元的支付。

在电话会议上，与娜塔莉·库斯伯特和普里安·皮莱共同创建了 Stitch 的皮莱没有给出任何关于这一指标的更新，但表示自去年 4 月推出该产品以来，Stitch 的支付价值每月增长 104%。2021 年第四季度，该平台的客户环比增长 44%，关联金融账户环比增长 72%。

Pillay 说:“随着我们继续深化支付产品并关注每月和定期支付，我们为我们在这里的合作伙伴和客户感到自豪，这对我们来说是有趣的功能集。

“我们最近有几个客户在尼日利亚上线了，这让我们非常兴奋。我们只是在那里提供支付，但渴望深化产品。我们今年将考虑添加数据和身份，并深化支付设置，类似于我们在南非的做法。”

总部位于纽约的长期投资公司云杉屋合伙公司领导了这轮融资。这一轮也欢迎新的和现有的投资者参与，如 PayPal Ventures、TrueLayer、firstminute capital、Raba Partnership、CRE Venture Capital、Village Global，以及金融科技创始人和公司，如 TrueLayer、Chipper Cash、Quovo 和 Unit 的创始人以及 Guillaume Pousaz 的 Zinal Growth。

“我们跟踪非洲的初创公司已经很多年了。Spruce House Partnership 的联合创始人本·斯坦说:“我们的勤奋非常明显，这是非洲大陆最有才华的团队之一，我们很高兴能够成为他们在 Stitch 建设的一部分。

皮莱表示，从风险投资公司和在多个地区推广金融科技产品的运营商那里筹集资金，让该公司在进入下一个增长阶段时有了“相当特别的东西”。Stitch 表示，这笔资金将使其能够在开普敦、约翰内斯堡和拉各斯的办事处扩大团队，推出新产品，并进入非洲大陆的新市场。
