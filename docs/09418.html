Qloo 筹集资金扩展其人工智能驱动的推荐 API

原文：https://web.archive.org/web/https://techcrunch.com/2022/08/23/qloo-raises-cash-to-expand-its-ai-driven-recommendation-api/

政策制定者一直在打击收集消费者数据的平台，实施要求公司对其收集和用于商业目的的细节保持透明的规定。例如，去年，弗吉尼亚州通过了《科罗拉多州数据保护法案》(Colorado Data Protection Act)，该法案要求企业在处理敏感信息之前获得同意，披露信息何时出售，并允许客户选择退出。加州、科罗拉多州和欧盟都有类似的框架，而其他州和 T2 国家也在考虑同样的框架。

一些营销人员认为，这些保护措施加大了建议或预测消费者可能需要哪些产品的难度。但是亚历克斯·埃利亚斯说事情不应该是这样的。他是 Qloo 的创始人，这是一个利用人工智能帮助品牌分析客户偏好数据以提供推荐的平台，包括娱乐和实物商品的推荐。

“围绕隐私的监管和平台环境极大地限制了身份驱动的理解消费者的方式。埃利亚斯在接受电子邮件采访时对 TechCrunch 表示:“这对从科技到包装消费品的各个领域都产生了巨大的影响，导致许多公司争相收集自己的第一方数据，这带来了巨大的风险。”。“与此同时，随着媒体和音乐消费的激增，消费者的口味变得越来越分散和细化，这使得接触消费者变得更加困难。”

伊莱亚斯告诉我，为了解决这些双重问题，他与 Qloo 首席运营官杰伊·阿尔杰(Jay Alger)共同创立了 Qloo。“Qloo 可以大规模阐明观众的偏好，这些数据可以提高销售效率，推动转化率的提升，从而增加底线，”他说大多数在更细的层面上了解消费者口味的公司都可以从 Qloo 中受益。"

为了显示投资者的信心，Qloo 今天获得了 1500 万美元的新融资，作为 Eldridge 和 AXA Venture Partners 牵头的 B 轮融资的一部分。这使得该公司的总筹资额达到 3000 万美元，其中包括演员莱昂纳多·迪卡普里奥、埃尔顿·约翰和喜达屋酒店创始人巴里·斯特恩里奇等知名人士的捐款。

Qloo 声称，其 API 关联了超过 5.75 亿个“主要实体”，包括电影、书籍、餐馆和歌曲，为“几十个”企业客户提供消费者口味预测，如百事可乐和埃尔顿·约翰的音乐公司 Rocket Entertainment。该 API 还为社交媒体应用 TasteDive 提供支持，该应用内置了 Qloo 在 2019 年收购的电影、电视节目、音乐、视频游戏和书籍的娱乐推荐引擎。

据 Elias 称，Qloo 不使用任何个人身份信息，保持所有请求“特设”并拒绝存储客户的客户身份。Elias 说，数据表面上是匿名和加密的，平台的数据处理“完全符合”法规，包括 GDPR 和加州消费者隐私法。

具体细节有点模糊，但在高层次上，Qloo 使用客户偏好的知识库来微调算法，从而产生产品建议和见解。例如，Qloo 可以为覆盖在地理区域上的实体(如音乐艺术家)创建“品味亲和力”，揭示特定城市甚至社区的品味和趋势(如哪些音乐家在布鲁克林市区受欢迎)。该平台还可以生成关于实体分组或实体比较的品味的描述，如耐克运动鞋客户和凡斯客户之间的音乐品味差异。

Elias 声称，有了 Qloo 及其与雪花、Tableau 和其他现有数据平台的集成，客户可以更好地解决诸如推动销售、减少广告支出和选择零售地点等问题。

“Qloo 的人工智能适应广泛的参数，因此最终客户可以根据他们对最终消费者口味相关性的‘新颖’或‘预期’程度，动态调整算法的权重，”Elias 继续说道。例如，亚洲的一个流媒体客户端能够将特定地区的结果优先于全球流行的推荐，并相应地调整算法

埃利亚斯承认，无数公司试图通过人工智能破解个性化和推荐，包括万事达卡拥有的动态收益和丰富相关性。但他认为 Qloo 平台是对竞争对手的补充，这些竞争对手运营着更通用的推荐引擎，如亚马逊个性化、Azure 推荐 API 和谷歌云的推荐人工智能，因为它可能会带来他们有时缺乏的“全套知识”。

“Qloo 之所以与众不同，是因为它在音乐、产品、旅游等领域拥有深厚的数据集和现有的知识库。这使得 Qloo 能够帮助客户用最少的上下文实现显著的个性化，”Elias 说。“Qloo 也是焦点小组或定制调查等传统、更昂贵的洞察工具的有利竞争对手，因为它可以快速、更有效地提供基于更大面板的特定洞察，例如，“上东区喜欢 Lululemon 的人也会喜欢什么电影？”"

Elias 说，B 轮收益将支持 Qloo 最新产品的发布——该平台的“精简”版本，为不太懂技术的用户提供可视化界面的订阅。除此之外，这笔资金将用于支持产品开发，在未来几个月将 Qloo 的 30 人团队扩大 30%以上，并建立公司的销售渠道。

当被问及收入时，埃利亚斯犹豫了。但他说，Qloo 迄今为止设法逆经济趋势而动，部分原因是旅游和娱乐等行业的需求复苏。

Elias 说:“进入疫情两年来，Qloo 的服务需求不断增长，收入和 API 利用率创下历史新高。“基本的顺风，包括对隐私的推动、对收入增长的关注和人工智能的广泛采用，大大超过了宏观环境和更广泛的技术估值的逆风。Qloo 实现了非常低的燃烧率，并接近盈利。”
