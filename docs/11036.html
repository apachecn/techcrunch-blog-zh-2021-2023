一家新公司正着手挑战 Airtable ，这家成立 10 年的公司最近估值高达 110 亿美元，它对无代码数据库平台的理解略有不同。

NocoDB 是众多计划取代强大的 Airtable 的创业公司之一，开源基金会是其核心卖点。虽然 NocoDB 在允许非技术用户创建新数据库方面以类似的方式工作，但它的独特之处在于，它还直接处理驻留在 Postrgres、MySQL 或 MariaDB 等数据库或数据仓库中的实时“生产”数据，并将它们转化为它所谓的“智能电子表格”。

这使得任何人都可以利用遗留数据库，而不需要 IT 人员的输入——不需要 SQL 查询或代码。这一切都是为了让业务、财务甚至营销团队能够连接到实时数据，并与开发人员合作构建无代码应用。

总部位于英国的创始人兼首席执行官 Naveen Rudrappa 声称，核心开源项目已经被 2000 多家公司使用，包括谷歌、沃尔玛、美国运通和迈克菲等巨头。

Rudrappa 告诉 TechCrunch 说:“我们看到的采用情况真的是前所未有的——我们在推出一年内就有 700 万次 Docker 下载，超过 30，000 个 GitHub stars，使我们成为世界上 350 个顶级开源项目之一。”

NocoDB:网格视图。图片来源: NocoDB

成立一年多后，该公司宣布从天使投资界的名人录获得一笔可观的种子资金。

自去年 6 月成立以来，这笔资金实际上已经分几次到位，但这轮融资总计约为 1050 万美元，机构投资者包括分贝公司、OSS Capital、不相关风险投资公司和 Together.fund。与此同时，天使投资方包括 YouTube 联合创始人查德·赫利；WordPress 创造者马特·莫楞威格；RedHat 联合创始人鲍勃·扬；早期谷歌投资人拉姆·施拉姆；以及来自 Cloudera、CockroachDB、PipeDream、Talend、AngelList、BrightRoll 和 Freshworks 的创始人。

故事到此为止

NocoDB 的起源可以追溯到 2017 年，当时 Rudrappa 正在以不同的名称开发一个相关的开源数据库“激情项目”，这是一个纯粹的后端，根本没有用户界面。他试图解决的问题涉及创建 API 来访问英国房地产数据的 MySQL 数据库——这并不容易实现。

“我意识到，使数据库 API 可访问的根本问题仍然没有解决，”Rudrappa 说。“所以，我建立了一个原型，在 GitHub 上发布，第二天早上醒来就看到了我的项目的一千个 GitHub 星星。这个问题比我想象的要普遍得多，我最初的原型引起了用户的共鸣。这个爱好项目获得了 25 万次下载，然后我决定与一个朋友合作，开始构建 NocoDB。”

去年 NocoDB 登陆 GitHub 时，Rudrappa 说它在头 10 周内就获得了超过 100 万的下载量。

“MySQL 或 Snowflake 等实时生产数据存储对企业用户来说是一种威胁，甚至对不习惯使用后端技术堆栈的开发人员来说也是如此，”他说。“但他们需要访问这些数据，以便快速构建有用的应用程序。NocoDB 使得将任何组织数据源连接到普遍理解的电子表格界面成为可能，允许零编码经验的用户构建与真实业务数据协同工作的工作流和自动化。”

拥有 1050 万美元的银行存款，以及技术领域一些知名人士的支持，NocoDB 完全有能力为主要的开源项目构建一个商业组件。这包括一个新的高级版本，目前处于私人测试阶段，允许公司连接到 Oracle 数据库和雪花。

“这个商业版本是客户方的要求，因为他们在使用软件时需要与我们签订工作合同，”Rudrappa 解释说。“企业客户需要不同的支持，我们希望在满足这一需求的同时，平衡我们开源社区的需求。”

最重要的是，NocoDB 还在开发一个托管和托管的云版本，该版本充满了企业级功能，包括连接器、单点登录(SSO)、访问控制、审计等。
