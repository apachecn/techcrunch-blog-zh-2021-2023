微软 M12 领投 web3 平台时空 2000 万美元| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/09/27/microsofts-m12-led-20m-investment-in-web3-platform-space-and-time/

空间和时间，一个去中心化的数据平台，已经筹集了 2000 万美元的战略资金，以帮助企业通过智能合同技术实现增长。

“我们的使命是让智能合约更加智能，并通过智能合约扩展业务逻辑的用例，”时空公司的联合创始人兼首席执行官内特·霍利迪(Nate Holiday)对 TechCrunch 表示。“我们看到智能合约将通过先进的自动化操作世界的商业逻辑。”

在软件中，业务逻辑有助于管理最终用户界面和数据库之间的信息交换。

Holiday 指出，Space and Time 将使用其“SQL 证明”加密技术，允许传统集中式系统中的业务逻辑自动化，并直接连接到智能合同。它还将把该平台与微软 Azure 整合，为用户提供访问和使用其区块链数据的入口。

此次融资由微软的风险基金 M12 牵头，投资者包括 Framework Ventures、HashKey、SevenX Ventures、Foresight Ventures、 Polygon 和 Avalanche 的生态系统基金 Blizzard 等。7 月下旬，时空公司在由 Framework Ventures 牵头的种子轮融资中筹集了 1000 万美元。

“如果你想一想，全球数字化转型是由微软领导的，”Holiday 说。“他们已经这样做了几十年，推动了业务逻辑和业务转型的自动化，所以我们认为这是一个令人兴奋的时机，可以在此基础上进行扩展，将区块链运营和智能合同纳入到今天已经存在于集中式数据库中的内容中。”

该平台是 Chainlink 计划的初创公司的一部分，将与 oracle network 密切合作，扩展智能合约的功能，以便区块链开发人员能够以低成本、分散的方式构建多链分散式应用程序(dApps)以及分析见解。

Chainlink 的联合创始人谢尔盖·纳扎罗夫(Sergey Nazarov)在一份新闻稿中表示:“鉴于信任最小化应用的潜在市场总额高达数万亿美元，为 Web3 开发者提供核心基础设施对于扩大规模和满足这一全球需求至关重要。”。

Holiday 说，该平台尚未向公众开放，但今年年底将有 DeFi 和游戏协议的有限版本。“然后，我们将进入测试网络阶段，我们将在 2023 年春季推出 SQL 验证，并在 2023 年秋季推出全面生产。”

Holiday 说，在短期内，Space and Time 希望帮助 web3 dApp 开发者简化他们的数据生态系统和架构。“从长远来看，我们将看到一个业务逻辑、自动化和验证都通过智能合同完成的世界。”
