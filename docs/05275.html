Dropbox 商店推出公测版，允许创作者出售数字内容

Dropbox 商店，一个允许创作者直接向他们的客户出售数字内容的平台，现在已经可以公开测试，Dropbox 周二宣布。该公司还推出了新的平台更新，并增加了小费的能力。您现在还可以定制您的店面和网址，并嵌入 HTML 代码。

Dropbox 商店去年推出了私人测试版，允许用户为数字内容创建产品列表。你可以直接从 Dropbox 添加内容，设置自定义图像、音频或视频预览，然后列出你的价格。创作者现在也可以选择启用小费。然后，Dropbox 将创建一个可共享或可嵌入的链接，你可以将其粘贴到社交媒体帖子、电子邮件、网页等内容中。

您可以创建自己的电子商务店面，编辑您的个人资料，并定制您的商店网址来展示您的产品。Dropbox 与 Stripe 和 PayPal 合作，让创作者除了主要的信用卡和 Google Pay 之外，还可以为客户提供多种支付选择。用户还可以查看和跟踪销售数据，并查看客户信息，如流量来源和历史交易。

可以通过商店销售的内容类型包括教育课程和材料、数字艺术和图形设计、音乐和音频、写作和出版、个人财务电子表格、游戏教程、健身和健康视频、电影、摄影、动画、数字工具等。Dropbox 商店支持图像、音频和视频上传的多种文件格式。您还可以在每个列表中上传多种文件类型。

图片来源: Dropbox

Dropbox 指出，它目前没有收取交易的百分比提成，因为它还处于开发该产品的早期阶段。一旦 Dropbox 商店超越测试阶段，这种情况很可能会改变。Dropbox 商店测试版今天面向美国 Dropbox Basic、Plus 和 Professional 计划的个人用户开放。

随着创作者经济的不断崛起，越来越多的人希望出售他们的数字产品。随着 Dropbox Shop 的推出，该公司正在寻求与其他允许创作者和用户销售数字产品的在线平台竞争，如 Shopify 和 SendOwl 。

除了关于 Dropbox 商店的消息，该公司还推出了Dropbox Backup 的最新版本。备份已升级，增加了新功能，包括恢复流程和管理计算机上备份的附加设置。Dropbox 指出，备份可以用来确保你采取了必要的措施来保护你最重要的文件。Dropbox Backup 适用于所有计划，也可作为独立产品提供给新用户和基本用户。
