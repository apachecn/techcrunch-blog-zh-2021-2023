尼日利亚的 Kippa 为其小企业财务管理应用程序| TechCrunch 获得了 320 万美元的前期种子资金

原文：https://web.archive.org/web/https://techcrunch.com/2021/11/18/nigerias-kippa-gets-3-2m-pre-seed-for-its-small-business-finance-management-app/

在过去十年的大部分时间里，全球数以百万计的小企业，尤其是新兴市场的小企业，都没有涉足互联网 。由于这个原因，他们中的大多数人仍然依赖于使用笔和纸或分类账来记账和存储重要信息。

在尼日尼亚，有些人到了将信息随手保存的程度。除了耗时之外，所有这些低效率还会导致错误，影响现金流和财务，这就是为什么该国几乎十分之九的小企业在头五年就失败了。

尼日利亚初创公司 Kippa ，试图通过其财务管理应用改善这些小企业的生命周期，已经筹集了 320 万美元的前期种子资金 。

这家初创公司的新一轮融资由总部位于柏林的风险投资 Target Global 牵头。entre Capital、Alter Global 和 Rally Cap Ventures 是其他参与的风险投资公司。

多位 天使投资人——Babs Ogundeyi，库达 CEOKhatabook 的投资者 Sriram KrishnanRaffael Johnen，Auxmoney 首席执行官；克里斯·布维尔；凯恩·卡西里；古德沃特资本公司的爱德华·苏；赛义德·拉赫曼投资了初创公司 。

Kippa 是一个简单的应用程序，小企业主可以在这里跟踪他们的日常收入和支出交易，创建发票和收据，管理库存，并通常 监控 他们的业务如何随着时间的推移而起伏 。

据该公司称，该应用程序最重要的功能之一是帮助商家跟踪债务人，并向他们发送自动提醒。该公司声称，以这种方式使用 Kippa 的商家“收回债务的速度快了 3 倍。”

这些功能中的大部分都是为了将尼日利亚的各种业务纳入该平台；然而，战略 是向他们介绍信贷和其他金融服务。

联合创始人兼首席执行官肯尼迪·埃克齐告诉 TechCrunch。小企业进行的大多数交易都是以现金为基础的，超过 30%的销售额是通过赊账完成的。因此，从本质上讲，企业面临的最大问题不是缺乏记账或工具，而是缺乏营运资本或信贷。

“对我们来说，我们所做的是我们有这样一个独特的机会向用户提供金融服务。对他们中的大多数人来说，Kippa 是他们使用的第一个 B2B SaaS 应用，”这位首席执行官说，他与杜克·埃克齐耶和耶弗他·乌切 一起创办了这家公司。

“我们确实有一个独特的机会来帮助他们接受在线数字支付，为他们提供营运资金、数字储蓄，并将他们融入金融生态系统 。”

在过去的一年里，许多解决小企业和初创公司各种需求的初创公司推出了大胆的承诺——采用不同方法的ll；有些人想管理簿记，有些人热衷于将小企业与供应商联系起来，有些人提供银行和软件服务。

但事实上，它们都汇聚在一点上，那就是提供获得信贷的机会。 首席执行官埃克齐说，虽然这是真的，但 Kippa“选择以数字方式成为本地人，而不是像以前的玩家那样追求模拟过程的数字化”据创始人说，这使公司与众不同。

由于 Kippa 目前仍然免费供企业使用，信贷和其他金融服务的引入将使该公司通过从贷款或营运资本中扣除佣金或利息来获得收入 。

记账和财务应用程序声称自 6 月份推出以来，平均月环比增长 126%。凭借 o ver 13 万活跃商家，从小报亭、街角小店到使用该应用的当地食品摊贩和高端商家， Kippa 声称在过去五个月里记录了超过 3 亿美元 。

对于主要投资者 Target Global，这些指标 表明 尼日利亚市场对该产品的强烈需求；因此，这就是它投资的原因。 根据该公司投资总监 Lina Chong 的说法，“我们对 Kippa 的投资将使其发展壮大，并成为非洲小型企业的首选财务管理解决方案 。”

Ekezie 和他的联合创始人于 2021 年 2 月创办了 Kippa。在 Kippa 之前，三人于 2019 年创立了软件人才匹配平台Africave，该平台于去年关闭。

根据 Ekezie 的说法，他们在认识到即将到来的供应限制将限制公司的增长后，从 Africave 离开。

这位首席执行官说，创始人试图解决另一个他们认为最适合他们优势的问题。 在进行了一次创始人市场契合之旅，并会见了尼日利亚各地的一些小企业主以了解他们的痛点后，基帕诞生了 。

Ekezie 说:“我们看到他们中的许多人使用分类账非常手工地操作，在一天结束时花费一个小时或更多的时间来平衡他们的账目，犯错误，取消账目，抱怨他们的记录不完整。

“我们还看到了一个更大的问题——这也是小企业面临的最大问题——缺乏信贷或融资来正常运营。所以我们认为这是一个值得解决的有趣问题。”

该团队还与其他新兴市场类似企业的创始人和一些创始团队成员进行了交谈，如印度的 Khatabook ，哥伦比亚的 Treinta ，印度尼西亚的 BukuWarung 和 Bukukas ，以了解他们如何扩大业务 。

虽然有一些相似之处，但 Ekezie 认为他和他的团队已经重新连接了 Kippa，以适应尼日利亚企业的需求。 例如，尽管 Khatabook 和其他公司在其他任何事情之前首先将簿记流程数字化，但 Kippa 正在解决更广泛的融资问题 。

Kippa 的种子前融资是尼日利亚和撒哈拉以南非洲最大的融资之一，将 投入 以扩大其商户基础，改进其产品，扩大团队规模，并涉足金融服务 。
