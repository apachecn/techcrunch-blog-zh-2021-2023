苹果高管为其他专业人士开发 Mac Studio 和 Studio Display

原文：https://web.archive.org/web/https://techcrunch.com/2022/03/17/apple-execs-on-developing-mac-studio-and-studio-display-for-the-other-pros/amp/

苹果工作室的种子是在 5 年前的下个月种下的，当时，苹果公司罕见地坦承失误，称是 按下了 Mac Pro 上的重置按钮。苹果公司已经把自己漆成“热角落”，它意识到它必须烧掉它创造的架构，重新开始。

一年后，它表示已经 创建了 Pro Workflows Team ，这是一个旨在与创意、科学和其他领域的专业人士互动并收集信息的团队，以更好地了解他们的工作流程。这些信息将继续用于开发 2019 年的 重新设计的 Mac Pro ，这是一个高度模块化的系统，带有一些高度专业化的可选组件，专为高端 3D 设计、音乐和电影工作量身定制。

Mac Studio 也是这一过程的产物，旨在为苹果的产品线引入一种全新的桌面，它结合了顶级的功能、可访问性和丰富的端口可扩展性，以及模块化的方法，包括提供 Studio 显示器——与 Studio 本身一起设计。

“我们非常关注 Mac studio，它是一个全新的 Mac 产品线。这很少见。苹果公司负责 Mac iPad 产品营销的副总裁汤姆·博格说:“我们不经常给 Mac 电脑添加产品线。“我们的理念根本不是拿一台 Mac Mini 并将其放大，而是‘我们知道我们正在开发这款 M1 Ultra 芯片，我们希望将其带给那些想要高性能、导电性和模块化系统的用户。让我们把它放在人们的桌子上，让它触手可及。这就是我们所交付的。”

我与博格、苹果公司 iPad 产品设计高级总监雪莉·戈德堡和苹果公司 Pro Apps 产品营销总监史云光·索伦谈论了 Mac Studio 的设计和开发。

在我们的会议中，Soren 指出，其创意和专业用户发出了一个明确的信号，即需要一个介于 iMac 和 Mac Pro 之间的模块化系统。模块化，在这个意义上，由提供两级 M1 芯片和一对演播室显示器的 Mac Studio 组成。

从苹果重置 Mac Pro，到组建 Pro Workflows 团队，再到开发 Mac Studio，有一条非常清晰的主线。这是一个端口丰富的产品，为用户提供了他们在社交媒体上公开要求的以及在与工作流团队的会议上要求的许多东西。

Mac Studio 有点像是对苹果创意客户的赞美，这些客户让苹果度过了艰难岁月，当时他们几乎是唯一看到苹果设计前沿软件和硬件套件的价值的用户。

自 2019 年以来，这群 Mac 忠实者并不总是受到我们所看到的连珠炮式发布的待遇。有一段时间，这个群体不禁怀疑，鉴于苹果 iOS 系列产品的巨大成功，苹果是否还会关心 Mac。

即使当 Mac Pro 上市时，考虑到如此强大的机器令人瞠目的价格和狭窄的市场，最终问题还是比答案多得多。

Mac Studio 提供了一种可以与苹果显示器或任何其他设备配对的机器，填补了一体机和 Mac Pro 之间的差距(以及 27 英寸 iMac 的永久删除留下的新差距)。它提供的性能是绝对不可否认的。

我买 Mac Studio 已经有一周左右的时间了，我一直在使用它，运行基准测试，并以各种方式推动它，同时也在尝试了解这种新的计算形式是什么样的。

我很高兴地说，这些印象大多是正面的——只有一个不幸的例外，我稍后会谈到。我可能会顺便在这里提到一些，但如果你想看看我们得到了什么样的基准和测试结果，以及对 Mac Studio 更深入的以硬件为中心的了解，请确保在这里阅读 TechCrunch 硬件编辑 Brian Heater 的文章。

举一个例子，在我的测试中，我正在试用的装有 M1 Max 的 Mac Studio 从 Final Cut Pro 导出 8K 时间轴的速度是 2020 年 M1 MacBook 的 3 倍，只比几乎完全饱和的 Mac Pro 慢 3 分钟。Mac Studio 的配置会让你花费大约 3200 美元。仅比我在 2020 年进行这项测试时使用的 Mac Pro 便宜约 20，000 美元。

这是一个巨大的性能增量，也是苹果在这款面向专业人士的产品中努力追求的。

索伦表示，对苹果来说，其 pro 应用程序的最高性能与民主化有关。

“说到专业技术，东西会变得很贵。你需要很多不同的装备。你需要一个特定的专业空间来工作。索伦说:“过去，你需要大团队。“所有这些都在不断发展。我认为进化在过去几年里真正加速的一个原因是因为苹果芯片。

我们正在讨论您可以拥有多少 8k 的屏幕，或者您可以在 96k 音频中进行 Atmos 混音，性能指标达到 30%。我们看到的另一个非常一致的情况是，你不需要超级昂贵的设施和长期租金。我们看到令人难以置信的作品在桌子上或桌子的角落里完成，这非常鼓舞人心。”

Boger 认为，人们需要一段时间来适应在苹果的产品线中拥有这种独立的桌面选项。但是苹果公司希望这些能成为许多工作室和机构的基础设备。为了让这台机器满足这些更灵活的工人的需求，它必须提供可选性。

“我认为我们看待它的方式是，我们很乐意为我们的用户提供多种工作方式，”Boger 说。“因此，你可以决定拥有一台内置 M1 Max 芯片的 MacBook Pro，也可以决定在你的工作室里设置一个可以来回携带 MacBook Pro 的装置。如果这是你选择的工作方式，很好。但是，我们也有用户喜欢将台式机放在办公桌上。”

Mac Studio 上可用的端口数量是在与 pro 客户交谈时得出的，询问他们正在使用多少设备。老实说，USB-A 选项对我来说有点惊讶，但 Boger 说他们的研究表明仍然有遗留需求。

“我们试图给我们的用户提供动态的选择范围。因此，在决定端口阵列和数量时，实际上只是与许多客户交谈，为我们的客户服务，并查看他们使用了多少设备。而 USB-A 端口是因为人们仍然有一些传统设备，他们只能在那里连接，还有一些软件仍然需要软件密钥，”Boger 指出。

他说，Mac Studio 是为了给他们提供一系列广泛的连接，同时还能满足绝大多数这类用户对端口数量的需求。

Mac 工作室团队的另一大设计试金石是这些端口的可访问性。M1 Max(理论上还有 M1 Ultra)是非常强大的芯片，可以在任何尺寸的计算机中提供最佳性能，更不用说 7×7 尺寸的计算机了。但紧凑的尺寸也是为了方便所有端口，甚至是后面的端口。

团队在设计时就考虑到了在任何给定的时刻能够到达 Mac Studio 的前面甚至后面。与任何中端机型相比，Mac Studio 真正被设计成了一个 桌面 桌面。

Boger 说:“我们把 IO 放在正前方，即使你需要到后面，你也只需把它转过来就行了。”。“比较轻；它非常小；它适合大多数 3.7 英寸高的显示器。我们真的给了用户前所未有的东西。他们总是不得不权衡取舍。如果我想要一台更小的电脑，我必须牺牲性能。我们想做的是给人们一些你不必做的事情。事实上，你可以做其他电脑做不到的事情，比如 18 个 8k 视频流，或者占用近 128 GB 视频内存的大型 3D 场景。”

谢利·戈德堡说，将这种能力融入 Mac 工作室是一项工程挑战。光是散热情况就已经是一项巨大的努力，这给 Mac Pro 带来了独特的挑战，因为那里有太多的空间来容纳大型风扇和大量的通风口。

由于 Mac Studio 被设计成一台真正的台式机，苹果公司设想它的操作贴近用户。

“从硬件的角度来看，这是一个非常有趣的挑战，因为我们试图提供如此巨大的性能，但却受到外形尺寸的限制，”Goldberg 说。“显然，如果你打算把它放在桌子上，你不会希望它在你做所有这些事情的时候打扰你，所以从硬件的角度来看，这是一个非常有趣的挑战。该团队对气流进行了数百次热模拟，试图找出气流通过系统的最佳模式，以优化性能和声学效果。最终，我们想出了我们的设计，底部有一个入口，通过 2000 多个加工孔进入，这些孔都是以[特定]角度加工的，当你绕着周边旋转时。”

这种特殊的通风设计减少了外壳内部的湍流，使空气能够从底部平稳地流过组件并通过鼓风机，而不会与自身发生碰撞。这些洞本身也不是正常钻成的；有一种专门为此目的定制设计的苹果机器，每秒钟在每个外壳上以特定的角度钻 3 个孔。

双鼓风机也完全是非标准的。戈德堡说，工程团队发现，通过将鼓风机的每个叶轮在高度方向上用隔板分开，他们能够使用圆盘进行调整，允许隔板上方和下方的叶片有不同的间距。

“这导致从盒子的每个部分吸入不同量的气流，因此我们可以根据冷却需求调整热性能，调整隔板的高度，调整叶片和每个部分的间距，然后同时优化热量和声音，”Goldberg 说。“每一代产品——尤其是 pro 产品——我们都觉得自己在不断挑战散热性能的极限，以及如何在特定的外形中实现更高的功率密度。对于我的团队来说，每一代人都能找到新的方法，这仍然非常令人兴奋。”

顺便说一句，一些人在 Mac Studio 的 M1 Ultra 型号规格上注意到的额外重量来自 Ultra 双层 SoC 顶部鳍状堆栈中的大量铜。这两种型号的鼓风机是一样的，当然是根据每个型号的需求来调整的。

我个人发现 Mac Studio 型号的热性能好得令人难以置信。我不仅没有注意到渲染或编译等扩展操作中的性能下降，而且我从未听到风扇发出任何真正可以听到的声音。我相信有些工作会让你听到，但这些工作必须是相当有挑战性的任务。

当然，让 Mac Studio 对任何人都有吸引力的另一个因素是工作室的显示屏。事实上，对苹果极受欢迎的 Thunderbolt 显示器替代品的等待是漫长的。虽然 Pro Display XDR 确实存在，而且是我见过的最好的显示器之一，但它 5000 美元的价格让许多不需要其功能集的参考显示器方面的专业人士望而却步。

演播室的展示非常出色。不，它不是微型 LED，但它是 P3，非常脆。在 iMac 27 上多年没有目标显示模式后，能够通过该显示器运行任何 Mac 电脑是一个非常受欢迎的选择。虽然 Studio Display 是为 Mac Studio 设计的，但苹果知道你也可以在其他机器上运行它。这也是 A13 在中央舞台和空间音响上承担重任的原因。它有效地给了旧的 MacBooks 一个很好的升级，以及清晰的显示。

“我们希望它对我们所有的 Mac 用户来说都是一个伟大的、非常易用的、非常主流的显示器，”博格说。“如果你想连接 MacBook Pro、MacBook Air、Mac mini、Mac Studio、Mac Pro 等，这是一个很好的显示器。我们知道，仍然有用户在使用基于英特尔的 MAC，因此将 A13 放在那里可以处理空间音频的音频，并使中央舞台的魔力发生。

虽然 Pro Display XDR 是一款令人惊叹的显示器，而且我们认为是任何地方的专业人士的最佳显示器，但也有数百万用户希望苹果为他们使用的任何 Mac 提供主流显示器。"

对于那些想知道的人，顺便说一句，你可以在非 Mac 机器上使用显示器；它将作为普通显示器、网络摄像头和立体声扬声器点亮。

这些扬声器是工作室展示的第二大特色，我很高兴地报告，我发现它们总体上听起来相当不错。

最大峰值音量大致在 27 英寸 iMac 的范围内，但音调更丰富、更宽广。对于小型室内扬声器阵列来说，低音响应确实令人印象深刻。即使它是“小低音”，你仍然可以在远处感觉到它，这增加了一点质感。低端仍然有那种受限的感觉，这种感觉在小音箱系统中很常见，但是中音非常宽广，一点也不狭窄。

就空间音频录音而言，Studio Display 呈现的前台远比我想象的要广阔。当在空间和非空间上比较同一首曲目时，你知道，你可以听到声音开放到一个比今天常见的非常响亮的前向混合更加细致入微的声场。

我很惊喜能够听到它，享受它，并且仅仅通过一次不经意的聆听就能发现它的不同。虽然声音无法与我的专用桌面显示器相比，但它将比大多数人现在在桌面上的声音更好。其他人可能会对此有更有见地的看法，但就我而言，我认为它相当不错。毫无疑问，这是监听音频的新高潮。

Goldberg 表示，Studio Display 中的空间音频系统还利用了他们为 16 英寸 MacBook Pro 构建的力消除驱动程序，这是桌面音频系统中的首例。这些专门的驱动器提供了一个内部的反作用力，可以抑制不必要的振动，起到减震器的作用，防止扬声器锥体的振动传入外壳。

“我认为这在声音质量上产生了很大的差异，因为通过这些相反的驱动因素，我们将我们创造的所有振动都发送到创造声音的有意振动中，而不是发送到外壳中，并且以有时不太可预测的方式振动外壳，这在声学上产生了不良的副作用，”Goldberg 指出。

不幸的是，正如你也将在 Brian 对 Mac Studio 的评论中读到的那样，当谈到 Studio Display 的第三大特色:新的 Facetime 摄像头时，我们遇到了一个大的性能问题。

在我们的测试中，Studio Display 的摄像头在本地和远程产生的图像颗粒大，对比度低，总体质量差。目前，我们看到的图像比 2021 年 24 英寸 iMac 的摄像头产生的图像更差。

第一次启动网络摄像头时，我就注意到了质量问题。我用其他设备检查了一下，注意到如果它是从运行 MacOS 12.2 的 MacBook Pro 上运行的话，实际上会稍微好一点，尽管仍然不是很好。鉴于存在差异，我的假设是这似乎是某种处理错误。我问苹果公司我得到的结果是否具有代表性，并发送了我所看到的样本图像和视频。经过审查，苹果发言人告诉我，该系统的表现不如预期，苹果将进行更新，以解决相机的性能问题。

我没有这些更新的时间表或任何细节，但苹果现在意识到工作室显示器的摄像头质量有问题，他们说他们正在修复。在你做出购买决定时，了解这一点是很有价值的，这也可能是等待这些更新是否能提高质量的一个原因。

截至目前，它并没有辜负显示器本身和扬声器的出色性能，这为显示器内音频树立了新的标准。

当 Mac Pro 下降时，我认为有一点点震惊，因为人们看到了 iMac 和 Mac Pro 的阵容，这实际上是针对整体用户的一小部分。以及介于两者之间的巨大鸿沟——无论是计算能力还是价格。

Mac Pro 成为了一种现象的受害者，我认为这种现象解释了为什么每次苹果发布新机时都会有很多关于定价的讨论。就像 80 年代和 90 年代经典的保时捷平面广告一样，苹果的大部分定位是关于其产品的可及性，同时将它们定位为努力争取的目标。

毫无疑问，它们是令人向往的产品，但你不可能买到比最好的 iPhone 更好的 iPhone，而且数十亿人拥有 iPhone，这一简单事实给人一种无产阶级的感觉——苹果自己的信息，甚至在上面的采访中，都是关于民主化的。

这绝对是一个值得讨论的话题，一个大约 3000 美元的工作工具，从各方面来看，它都是有史以来最强大的消费电脑之一。一旦价格达到 20，000 美元，就有点难了。事实是，虽然我们都喜欢想象自己坐在那些疯狂强大的 Mac Pros 的座位上，但与 iPhone、iMac 或 iPad 相比，这种机器的观众少得可怜。

对于观众来说，价格标签并不是一个大问题。对我们其余的人来说，它扼杀了梦想。

有是个好问题。每次苹果推出一款产品，都会让人产生一种‘我怎么能在那台 Mac 电脑上看到自己’的感觉。Mac Studio 在填补价格和性能差距方面走了很长的路，这一差距导致了一些存在性焦虑和一些专业挫折。这是一台很好的机器，如果网络摄像头的问题可以通过更新来解决，工作室显示器是一个值得的伴侣。

台式 Mac 又回来了。
