比特币基地和 Polygon 支持印度新的加密倡导组织 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/11/03/coinbase-and-polygon-back-new-crypto-advocacy-group-in-india/

包括比特币基地和 Polygon 在内的顶级加密公司在印度成立了一个行业机构，以促进主要利益相关者之间的对话，并提高对 web3 的认识，几个月前，印度最大的加密倡导组织被解散。

这个名为 Bharat Web3 Association (BWA)的新行业机构的成员包括顶级的本地加密交易所，包括 CoinDCX、CoinSwitch 俱吠罗和 WazirX。BWA 在一份声明中说，它还包括 Hike、Biconomy、ZebPay 和 Tax Nodes。

“就人才、投资和创新而言，印度的 Web 3.0 潜力是革命性的，必将使印度成为这一快速新兴领域的全球领导者，”比特币基地国际和商业发展副总裁纳纳·穆鲁格桑(Nana Murugesan)在一份声明中表示。

“我们支持 BWA 通过利益相关者协作、思想领导和教育来推进 Web 3.0 生态系统的使命。建设强大的基础设施和设计有利的环境，可以让像比特币基地这样的参与者建立一个更加自由、开放和安全的互联网。”

一位知情人士告诉 TechCrunch，Murugesan 和其他几位比特币基地高管目前正在印度，他们过去几天一直在印度与印度的主要部长进行对话。今年早些时候，在监管机构表示担忧后，比特币基地在印度推出了一项不成功的服务，并在推出周内取消了该服务。

该人士说，比特币基地和政府官员之间的对话尚未就恢复该国的服务取得进展，这位人士要求匿名谈论私人问题。

Bharat Web3 协会还将寻求为 Web3 行业制定标准化原则，并帮助培养印度的人才库。

知情人士表示，印度央行继续强迫银行与印度的加密平台合作，这一举动让 on-ramp 成为了相关公司的噩梦。

据直接了解此事的消息人士称，几个月来，该国的许多投资者和企业家一直在努力寻找更新、更有效的方式，包括与强大的智库 NITI Aayog 接触，以联络政策制定者。消息人士补充说，Niti Aayog 拒绝涉足加密行业。

消息人士称，就印度立法者而言，他们在过去一年里遇到了几个行业面孔，但迄今为止，他们认为加密交易的快速采用伤害了大多数消费者，应该采取更多的保护措施。

随着不确定性的出现，当地生态系统中的一些人才转移到了国外，越来越多的当地企业家开始为国外市场服务，而不是为印度这个全球第二大互联网市场的客户服务。

当地行业此前由区块链和加密资产委员会代表，该委员会是印度颇具影响力的技术游说团体互联网和移动协会的一部分。该倡导组织在 7 月份表示，它正在解散加密部门，因为“该行业监管环境的解决方案仍然非常不确定。”

TechCrunch 此前援引消息人士的话报道称，此举是印度密码行业多年挫折的高潮，该行业认为游说团体的影响力和影响力无法带来里程碑式的成果。

Polygon 的联合创始人 Sandeep Nailwal 在一份声明中说:“由于其蓬勃发展的开发者社区、企业家精神、快速增长的经济、健全的数字基础设施和深入的数字采用，印度将成为 Web3 领域的领导者。”

“印度企业家已经在生态系统中留下了印记，正在为世界创新，开发有价值的公共用例。BWA 将在帮助印度实现其作为全球网络 3 领导者的潜力方面发挥关键作用。”
