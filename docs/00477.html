WhyLabs 为其人工智能观察平台筹集了 1000 万美元的 A 轮融资

原文：https://web.archive.org/web/https://techcrunch.com/2021/11/04/whylabs-raises-10m-series-a-for-its-ai-observability-platform/

去年从艾伦研究所(Allen Institute)分拆出来的机器学习初创公司 WhyLabs ，帮助数据团队监控他们的人工智能模型和推动这些模型的数据管道的健康状况。去年，这家初创公司筹集了400 万美元的种子轮，今天，该团队宣布，它已经筹集了 1000 万美元的 A 轮融资，由 Defy Partners 和吴恩达的 AI Fund 共同牵头。现有投资者 Madrona Venture Group 和 Bezos Expeditions 也参与了此轮投资。

就在几年前，“MLOps”还不是一个真正的东西，但现在它已经成为一个既定的类别，我们现在看到越来越多的初创公司和老牌公司试图在这个市场上获得早期立足点。

“这是每个人的头等大事，”WhyLabs 首席执行官阿莱西娅·维森吉奇说。“去年，我们确实经历了真正的转变。今天，机器学习模型的部署后维护，我认为是比模型的实际构建和部署更大的挑战。这是推动整个 MLOps 转变的动力，组织非常关注围绕操作机器学习模型创建文化、流程和工具链。”

图片来源: WhyLabs

正如维森吉奇所指出的，每一家建立人工智能模型的公司都在努力提高它们在生产中运作的透明度和可观察性。训练模型是一回事，但要确保模型继续按预期运行，并确保为其提供动力的数据仍然可信，则是完全不同的挑战。大多数时候，模型将继续提供(错误的)预测，毕竟，即使输入数据被破坏，可能是因为 API 改变了。

Ng 说:“ML 工程师需要更好的工具来确保 ML 项目生命周期各个阶段的高质量数据。“人工智能基金很高兴支持 WhyLabs，其开源日志库和人工智能观察平台使开发人员可以轻松维护实时日志和监控 ML 部署。”

在过去的一年中，WhyLabs 让数据团队更容易使用其服务，这在很大程度上是因为它推出了带有免费自助层的 SaaS 平台。

“我们的许多 SAS 工作都专注于消除配置负担，确保您可以像监控特定计算实例一样进行监控，”维森吉奇说。“因为我们是 SaaS，我们非常注重隐私。关注隐私非常重要，因为我们同时监控数据健康和模型健康。”

具体来说，这意味着 WhyLabs 从不移动数据。正如维森吉奇解释的那样，这项服务只捕捉描述数据和模型的统计指纹。WhyLabs 推出后，该团队首先应对的挑战之一是如何监控潜在客户的高度专有数据。毕竟，这不仅仅是在公司的数据资产内移动数据，还要确保公司内部的权限得到良好管理，以确保敏感信息不会泄露。T2】

图片来源:why labs“SaaS 不仅仅是确保任何人都可以记录它，”她指出。“这是为了确保人们可以使用这个平台和以隐私为中心的方式，并且他们可以将其插入他们正在使用的任何类型的机器学习堆栈中。”

为此，该团队还花了一年时间来构建与所有主要云平台、机器学习框架和 Kafka、Spark 和 Ray 等数据服务的集成。

WhyLabs 首席运营官和联合创始人玛丽亚·卡拉伊凡诺娃指出，随着两周前 SaaS 平台的推出，该服务正在监测的模型数量已经增加了两倍。

WhyLabs 团队计划使用新的资金来建立其工程和走向市场团队。在工程方面，这也意味着增加对流数据和实时人工智能应用程序的支持。正如维森吉奇指出的，该公司的走向市场方法将包括重点教育客户有关 MLOps 的知识，因此该公司也在建立一支福音传道团队。

“WhyLabs 凭借其可观察性平台和数据记录库的快速采用，在改变任何企业治理人工智能和管理 MLOps 的方式方面处于独特的地位，”现已加入 WhyLabs 董事会的 Defy Partners 创始人兼合伙人尼尔·塞杰拉说。“他们建立了一个有效的控制中心来操作人工智能应用程序。反过来，他们的技术对智能应用构建者和每天被 AI 接触的数亿人产生了有意义的积极影响。”
