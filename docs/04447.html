区块链基础设施初创公司在收入同比增长 500%后温和融资 4000 万美元 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/03/02/blockchain-infra-startup-tenderly-raises-40m-after-seeing-500x-yoy-revenue-growth/

温柔地，一家旨在让“各种技能”的 web3 开发者更容易地 开发区块链产品的初创公司，已经在 B 轮融资中筹集了 4000 万美元——就在几个月前完成了其【a 轮】

此次融资是区块链基础设施领域——乃至整个金融科技领域——快速连续融资的又一个例子。本月早些时候，TechCrunch 还报道了 Alchemy 以 102 亿美元的估值为其 C 系列融资 2 亿美元。那家公司 的目标是成为考虑在区块链或主流区块链应用程序上构建产品的开发者的起点。

虽然两家公司都是类似的开发工具，用于构建分散式应用程序，但两家公司的主要区别在于，Alchemy 提供节点即服务，而 Tenderly 是一个平台，旨在通过其仪表板和 API 开发、测试和监控分散式应用程序的健康状况。据说它更侧重于智能合同。

“成千上万”来自 Uniswap、Yearn Finance、Circle、Chainlink、Gnosis、Nexus Mutual、Instadapp、DeFi Saver 和 NFT 市场(以及 decacorn) OpenSea 等应用的开发者小心翼翼地监控着应用和智能合约的健康状况。事实上，该公司表示，它与大多数100 强以太坊项目合作。

总部位于塞尔维亚贝尔格莱德的于 2018 年由朋友安德烈·本契奇(CEO)、波格丹一世·哈比克(CTO)、米尔詹·泰基奇(首席运营官)和内博伊沙·乌罗舍维奇(区块链架构师)成立，他们多年来一直在软件开发的前沿工作，并对他们所说的不存在的工具感到沮丧。

四方小组表示，他们很早就认识到区块链成为下一个互联网核心标准的巨大潜力，但也意识到为了实现这一点，这项技术需要让每个人都能使用。

Bencic 说:“我们开始时很温和，四名工程师专注于一项特殊的技术，那就是区块链和以太坊。

该团队开始为自己开发工具——最初是为了调试——但现在已经逐渐发展成为一个开发者平台，每天通过其虚拟机(VM)处理超过 2500 万个事务。它现在涵盖了整个区块链，如 以太坊，雪崩，Fantom，乐观主义和 Arbitrum。

图片来源:温柔地

Bencic 说:“我们与整个社区密切合作，创建了一个工具、服务和基础设施系统，以帮助开发者快速推出更好的产品。“现在，我们在开发的每个阶段都照顾到开发人员——从到开发智能合同，在生产前对他们进行测试，在支持基础设施和生产中帮助他们。

尽管该公司拒绝透露具体的收入数字，但该公司的收入同比增长了 500%，用户数量同比增长了 420%，尽管“销售部门没有适当的专业知识”，Bencic 表示，他将迄今为止的成功归功于与用户基础的“深厚联系”。

他说:“我们仍然主要是一个产品和工程团队，但我们希望在未来的一年里解决这个问题。”。

他坚持认为，加密领域目前的波动并没有影响其增长。

Bencic 补充道:“如果说有什么不同的话，那就是我们看到了开发者们探索 web3 领域的更大决心。”。

Spark Capital 领投了这家初创公司的最新一轮融资，该公司正在旧金山开设一个办事处。现有的支持者 Accel(领导其首轮融资)和 Point Nine Capital(领导其种子轮)也参加了，还有 Abstract Ventures，比特币基地风险投资公司，Uniswap，代达罗斯，Vercel 首席执行官 Guillermo Rauch，乐观主义联合创始人兼首席执行官王京兰，长期股票交易所创始人 Eric Ries 和 LTSE 软件首席执行官 Tihomir Bajic。

该公司计划利用新资本继续扩大产品供应，争取客户和合作伙伴，并雇佣技术和销售人才。

领投此次融资的 Spark Capital 普通合伙人 Yasmin Razavi 认为，这款温和的产品套件将“显著降低 web3 的开发门槛，并在整个开发过程中不断增强开发者的能力:从测试到部署再到监控。”
