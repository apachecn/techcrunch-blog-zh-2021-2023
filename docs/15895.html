Anthropic 推出聊天机器人 Claude，与 OpenAI 的 ChatGPT | TechCrunch 竞争

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/14/anthropic-launches-claude-a-chatbot-to-rival-openais-chatgpt/

Anthropic 是一家由前 OpenAI 员工共同创立的初创公司，今天推出了一款与病毒式轰动效应 ChatGPT 相抗衡的产品。

Anthropic 的 AI 是一个聊天机器人，名为 Claude，可以被指示执行一系列任务，包括搜索文档，总结，写作和编码，以及回答关于特定主题的问题。在这些方面，它与 OpenAI 的 ChatGPT 相似。但 Anthropic 认为，克劳德“不太可能产生有害的输出”，“更容易交谈”和“更容易控制”

“我们认为 Claude 是适合各种客户和用例的正确工具，”Anthropic 发言人通过电子邮件告诉 TechCrunch。“几个月来，我们一直在为服务模型投资基础设施，我们有信心满足客户需求。”

继去年年底的封闭测试之后，Anthropic 一直在悄悄地与包括 Robin AI、AssemblyAI、onception、Quora 和 DuckDuckGo 在内的合作伙伴一起测试 Claude。从今天早上开始，有两个版本可以通过 API 获得，一个是 Claude，另一个是速度更快、成本更低的衍生产品 Claude Instant。

结合 ChatGPT，Claude powers DuckDuckGo 最近推出的 DuckAssist 工具，直接回答用户直截了当的搜索查询。Quora 通过其实验性的人工智能聊天应用程序 Poe 提供与 Claude 的联系。在概念上，克劳德是概念人工智能技术后端的一部分，这是一个与概念工作空间集成的人工智能写作助手。

罗宾首席执行官理查德·罗宾逊在一份电子邮件声明中说:“我们使用克劳德来评估合同的特定部分，并建议对我们的客户更友好的新的替代语言。”“我们发现克劳德非常擅长理解语言——包括像法律语言这样的技术领域。它在起草、总结、翻译和用简单的术语解释复杂的概念方面也非常自信。”

但是 Claude 是否避免了 ChatGPT 和其他类似的 AI 聊天机器人系统的陷阱？众所周知，现代聊天机器人容易使用有毒的、有偏见的或其他攻击性的语言。(参见:必应聊天。)他们也倾向于产生幻觉，这意味着当被问及核心知识领域以外的话题时，他们会捏造事实。

Anthropic 表示，克劳德-像 ChatGPT 一样，无法访问互联网，并在公共网页上接受培训，直到 2021 年春天-被“训练以避免性别歧视，种族主义和有毒的输出”，以及“避免帮助人类从事非法或不道德的活动。”这在人工智能聊天机器人领域是意料之中的事。但是 Anthropic 声称，让 Claude 与众不同的是一种叫做“宪法人工智能”的技术。

“宪法人工智能”旨在提供一种“基于原则”的方法，使人工智能系统与人类意图保持一致，让类似于 ChatGPT 的人工智能使用一套简单的原则作为指导来回答问题。为了构建克劳德，Anthropic 从大约 10 个原则开始，这些原则放在一起，形成了一种“宪法”(因此得名“宪法人工智能”)。这些原则还没有公开。但是 Anthropic 说，它们是基于有益(最大化积极影响)、无害(避免给出有害建议)和自主(尊重选择自由)的概念。

Anthropic 然后有一个人工智能系统——而不是克劳德——使用自我完善的原则，对各种提示做出回应(例如，“以约翰·济慈的风格写一首诗”)，并根据宪法修改回应。人工智能探索了对数千个提示的可能反应，并筛选出最符合体质的反应，这些反应被 Anthropic 提取到一个单一的模型中。这个模型被用来训练克劳德。

然而，Anthropic 承认 Claude 有其局限性——其中一些在封闭测试中暴露出来。据报道，与 ChatGPT 相比，Claude 的数学更差，编程能力也更差。它产生幻觉，例如，为一种不存在的化学物质发明一个名字，并为生产武器级铀提供可疑的指示。

也可以通过巧妙的提示绕过 Claude 的内置安全功能，ChatGPT 就是这样。测试版中的一个用户能够让克劳德向描述如何在家制造冰毒。

Anthropic 的发言人说:“挑战在于制造既不会产生幻觉又仍然有用的模型——你可能会陷入一个艰难的局面，在这种情况下，模型认为永远不撒谎的一个好方法是永远不要说任何话，所以我们正在努力进行权衡。”。“我们在减少幻觉方面也取得了进展，但还有更多工作要做。”

Anthropic 的其他计划包括让开发人员根据自己的需要定制克劳德的宪法原则。不出所料，客户获取是另一个焦点——Anthropic 将其核心用户视为“进行大胆技术押注的初创公司”以及“更大、更成熟的企业”

Anthropic 的发言人继续说道:“我们目前并没有寻求一个广泛的直接面向消费者的方法。”。“我们认为，这种更窄的关注范围将有助于我们推出更优质、更有针对性的产品。”

毫无疑问，Anthropic 正感受到来自投资者的某种压力，要求收回投入其人工智能技术的数亿美元。该公司拥有强大的外部支持，包括一批投资者提供的 5.8 亿美元，其中包括名誉扫地的 FTX 创始人萨姆·班克曼-弗里德、卡罗琳·埃里森、吉姆·麦克拉夫、尼萨德·辛格、贾恩·塔林和新兴风险研究中心。

最近，谷歌承诺向 Anthropic 投资 3 亿美元，收购这家初创公司 10%的股份。《金融时报》最先报道了该交易的条款，根据该条款，Anthropic 同意让谷歌云成为其“首选云提供商”，两家公司“共同开发人工智能计算系统”
