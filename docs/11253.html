开放式 3D 引擎增加了改进的地形创建和协作工具 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/10/18/the-open-3d-engine-adds-improved-terrain-creation-and-collaboration-tools/

很长一段时间，3D 引擎的世界——尤其是对游戏开发者来说——都是关于 Unity 和 Epic 的虚幻引擎。然后，当亚马逊开始其命运多舛的进入游戏的尝试时，它推出了木材场引擎(它本身是基于 Crytek 的 CryEngine)。虽然你可能不记得那次大熔炉的失望，但那些游戏没有成功的事实产生了有趣的影响。并不总是被称为开源冠军的亚马逊开源了木材场，并推出了开放 3D 基金会(打着 Linux 基金会的旗号)。从那以后，Adobe、微软、英特尔、华为、Niantic、LightSpeed Studios 以及最近的 Epic 都成为了该基金会的会员。本周，该基金会刚刚成立一年，正在举办其 3DCon 会议，并发布最新版本的开放式 3D 引擎。

最新版本(22.10)关注于围绕性能、工作流和可用性的生活质量改进。例如，对于新用户来说，有一种新的入职体验，以及用于在远程项目上与其他团队成员协作的新工具，这在当今时代变得更加重要。例如，团队现在只需共享一个 URL 就可以共享和下载项目，新的项目模板使新团队成员更容易上手。开发人员还推出了新功能，使设置和调试多玩家应用程序更容易，对于艺术家来说，现在可以更容易地将他们的动画放入开放的 3D 引擎中。对于所有开发开放世界游戏和体验的开发者来说，现在有了一个改进的地形系统，可以扩展到 16x16km 的世界。

图片致谢: O3D 基金会

Linux 基金会数字媒体和游戏总经理兼 O3DF 执行董事 Royal O'Brien 表示:“通过这一最新版本，我们的社区将继续致力于让全球的开发人员、艺术家和内容创作者更轻松地构建令人惊叹的 3D 体验，重点是性能、核心稳定性和可用性增强。“很高兴看到他们辛勤工作的成果，因为开放式 3D 引擎正在加速成熟，成为想要以模块化方式构建沉浸式体验的创作者的首选。”

图片来源: O3D 基金会

当然，有趣的是，像 Epic 这样的公司正在加入像 Open 3D Engine 这样的努力，它起初看起来像是一个竞争对手——而且是免费的。当我与 O'Brien 谈论这个问题时，他指出这与其他将竞争厂商聚集在一起的开源项目并没有什么不同。不仅这些引擎变得越来越复杂，而且它们在这一点上提供的很多东西也是赌注。像开放 3D 基金会这样的努力让他们专注于真正让他们与众不同的功能。这有助于贮木场和现在的开放 3D 引擎，从一开始，就意味着模块化。但最重要的是，该基金会还提供了一个中立的地方，用于制定互操作性的共享标准，以帮助开发人员和艺术家使用他们想要的工具，然后将他们带入他们选择的引擎，无论他们是在构建游戏还是新的 AR/VR 体验。
