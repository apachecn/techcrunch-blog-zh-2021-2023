风投们加大对索拉纳不断扩张的 NFT 生态系统的押注

原文：https://web.archive.org/web/https://techcrunch.com/2022/04/04/vcs-scale-their-bets-on-solanas-expanding-nft-ecosystem/

作为一种资产类别，非金融资产的迅猛增长是一个似乎拒绝破裂的泡沫——至少目前如此——但 NFT 市场正继续吸引投资者的注意力，尤其是围绕市场领导者似乎尚未开发的机会。最近几周，越来越多的注意力集中在索拉纳生态系统的 NFT 机会上。

上周五，贾斯汀·坎(Justin Kan)的加密游戏市场分形(Fractal)在 Paradigm 和 Multicoin 的共同牵头下筹集了 3500 万美元，参与方包括安德森·霍洛维茨(Andreessen Horowitz)、Animoca、比特币基地和索拉纳实验室(Solana Labs)。这一轮融资是在 Solana 最受欢迎的 NFT 本土店面 Magic Eden 获得由 Paradigm 牵头的 2700 万美元融资后两周进行的，红杉等也提供了额外资金。

Solana 为消费者提供低成本交易费和更高的速度，是近年来区块链生态系统中更大胆的成功故事之一，因为该网络吸引了开发商和投资者。许多玩家将第 1 层(L1)替代方案视为 NFT 工作的更好枢纽，这些工作经常严重拖累以太坊网络，导致整个网络的高额费用和等待时间。

“我们从索拉纳开始，因为我们认为这对用户、游戏玩家和游戏公司来说是最好的用户体验，”Kan 在接受 TechCrunch 采访时表示。

所有这些关注吸引了对索拉纳·NFT 创业公司的新投资，也吸引了现有重量级公司的新努力。上周，OpenSea 在一条推特上分享了，它将在未来几周增加对索拉纳区块链的支持。

OpenSea 仍然是 NFT 总销售额中最大份额的中心。它的总交易额达到了极高的水平，使该公司的估值达到了 133 亿美元——但这家初创公司仍有大量工作要做，以培养主流消费者对该技术的广泛热情。虽然该平台在过去 30 天的交易量超过了 25 亿美元，但这些资金分散在大约 45 万个不同的钱包中——这是该平台的一个重大同比增长，但也表明 NFT 市场在这一点上仍然是多么孤立。与此同时，Magic Eden 的总交易量只是 OpenSea 的一小部分——过去一个月的交易量为 4100 万美元，而不是 25 亿美元，但 Magic Eden 在此期间的 95000 名活跃交易者表明，尽管平均交易量低得多，但用户对这个平台的兴趣正在上升。

一个最大的公开问题将是如何监管因素纳入 NFT 市场在处理垃圾邮件方面的持续挑战。

在过去的几年里，OpenSea 严重依赖于在以太坊上制作 NFT 的相关费用，以此作为管理其市场的一种手段。在过去的一年里，他们试图通过提供一种更简化的过程来降低费用，这种过程被称为“懒惰铸造”，但在 1 月份，他们不得不控制这一功能，因为他们注意到使用这种技术创建的 80%的 NFT 要么是垃圾邮件，要么是假货，要么是抄袭的作品。Solana 降低了费用，可能会加速大型市场的一些问题，并迫使他们建立更强大的内容过滤器，也许会为具有更好定义的利基或更积极的监管策略的新贵留下空间。
