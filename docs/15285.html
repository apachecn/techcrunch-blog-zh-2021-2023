Agtech Green Labs 是最新一家裁员的韩国创业公司

韩国农业科技初创公司绿色实验室，去年 1 月融资 1.4 亿美元(1700 亿韩元)的 C 轮，正在大幅裁员。

知情人士向 TechCrunch 证实了早些时候由当地媒体发布的关于裁员的传言。

格林实验室目前拥有 650 多名员工，在获得 C 轮融资后，员工人数增加了一倍，作为重组的一部分，该公司正在裁员。TechCrunch 报道称，2022 年 1 月，GreenLabs 拥有 320 名员工。

对于有多少绿色实验室计划裁员，我们没有确切的数字。格林实验室向 TechCrunch 证实，它正在进行一轮裁员，可能会影响至少 50%的员工。

除了裁员，这家初创公司在筹集新一轮资金时遇到了障碍。据知情人士透露，由于过度扩张，该公司资金告罄。

由 Sanghoon Shin，Scott Sungwoo Choi 和 Luke Donghyun Ahn 于 2017 年创立，Choi 和 Ahn 已经离开了这家创业公司，Shin 现在领导着这家公司。

绿色实验室的投前估值约为 7 亿美元，获得了 BRV 资本管理公司、Skylake Incuvest 和 SK Square 的 C 轮融资，以加速其核心业务的全球扩张:建立可持续的解决方案来解决全球食品供应链问题。自成立以来，绿色实验室已经筹集了 1.7 亿美元。

这家初创公司为农民构建了智能农业软件和硬件，以数字化农业产业从作物生产到分销的整个价值链。这家初创公司声称，超过 50 万农民使用绿色实验室的 B2C 应用 Farm Morning。绿色实验室表示，该应用程序聚合农业数据，然后人工智能提供对作物生命周期的见解。其 B2B 市场 Sinsun Market 为 10，000 多家企业买家提供采购渠道。
