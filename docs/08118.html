AI21 实验室获得 6400 万美元以提升人工智能驱动的语言服务

原文：https://web.archive.org/web/https://techcrunch.com/2022/07/12/openai-rival-ai21-labs-raises-64m-to-ramp-up-its-ai-powered-language-services/

该企业看好能够理解和生成文本的人工智能系统，即所谓的语言模型。根据约翰·斯诺实验室的一项调查，60%的技术领导者在 2020 年对人工智能语言技术的预算至少增加了 10%。一个供应商 OpenAI 说，它的首要语言模型 GPT-3 正在被成千上万的开发者使用。

渴望分一杯羹的新供应商近年来纷纷出现，声称将独特的语言建模能力带到了桌面上。除了像 OpenAI 、 Cohere 和抱脸这样资源丰富的初创公司之外，还有一批厂商在开源人工智能模型的基础上构建服务。坐在中间的是以色列公司 AI21 Labs，该公司开发了一个模型——Jurassic-1 Jumbo，大约与 GPT 3 号一样大——并围绕它慢慢开发产品，包括一个名为 AI21 Studio 的“人工智能即服务”平台，允许客户创建虚拟助手、聊天机器人、内容审核工具等。

显然，投资者感觉到了一个机会。今天，AI21 实验室完成了 6400 万美元的 B 轮融资，对该公司的估值为 6.64 亿美元。由 Ahren Innovation Capital Fund 牵头，Mobileye 首席执行官兼联合创始人 Amnon Shashua、Walden Catalyst、Pitango、TPY 资本和 Mark Leslie 参与，这一部分使 21Labs 的总资本增加到 1.185 亿美元。

联合创始人兼首席执行官 Ori Goshen 表示，新资金将投入 R&D，特别是开发更大、更复杂的语言模型，以及招募人才。AI21 实验室目前有 120 名员工，并计划在今年年底前再雇佣 50 名左右，无视宏观经济趋势。

“幸运的是，疫情对业务产生了积极的影响——随着越来越多的公司迁移到远程工作，个人需要以书面形式传达他们通常口头分享的内容，”Goshen 在电子邮件采访中告诉 TechCrunch。“[我们的]专有大型语言模型的核心功能允许摄取大量企业数据，用于…自定义内容创建、摘要和分类。"

AI21 实验室于 2017 年由 Goshen，Shashua 和斯坦福大学教授 Yoav Shoham 共同创立。该公司的第一个产品是 Wordtune，这是一款人工智能辅助写作工具，旨在与 Grammarly 竞争，Grammarly 建议用户在任何地方输入文字都要重新措辞。AI21 Studio 于去年 8 月发布，同时发布的还有一项“现收现付”服务，允许开发人员申请访问定制模型，这些模型根据他们的需求对数据集进行了微调。

AI21 Labs 提供了一系列调整参数来定制其模型的输出。图片来源: AI21 实验室

在 AI21 Studio 中，AI21 Labs 的 Jurassic-1 系列模型可用于释义(如从产品描述中生成简短的产品名称)、从文本中提取图形以及按主题或类别标记电子邮件和笔记。这些模型还可以通过 Wordtune 中被称为 Wordtune Read 的功能来总结内容，包括文章、报告和 PDF 文件的片段。

因为他们接受了来自互联网(包括社交媒体)的大量数据的训练，语言模型能够基于他们在训练中遇到的类似语言生成有毒和有偏见的文本。AI21 实验室的模型没有什么不同；在早期测试中，一名研究人员能够促使他们说“热爱犹太人的人是思想封闭的。”虽然 AI21 Labs 要求客户同意使用条款政策和使用指南，但它尚未对其 API 生成的潜在有毒内容实施过滤。

AI21 实验室表示，它人工审查对微调模型的请求，以打击滥用，并声称其模型比 GPT-3“偏差略小”。

不管怎样，根据 Goshen 的说法，这些模型有一个优势，因为它们增加了像维基百科这样的外部知识来源。AI21 Labs 最新版本的 Jurassic-1 模型 Jurassic-X 使用 Goshen 所谓的“模块化推理知识系统”，通过在线计算器和货币转换器等“离散推理专家”来增强其答案。Goshen 说，因此，Jurassic-X 可以回答用自然语言表达的“非平凡”数学运算，也可以简化可能会绊倒其他语言模型的“复杂”问题。

当然，值得注意的是，AI21 实验室还没有委托将它的 Jurassic-X 模型与其他商业语言模型进行比较，所以声明是我们所能做的一切。

该公司最近令人质疑的营销噱头并没有给人们带来巨大的信心。今年 6 月，AI21 实验室推出了一款模仿已故最高法院大法官鲁斯·巴德·金斯伯格的法律意见的聊天机器人，几位人工智能技术专家称其具有误导性。作为对批评的回应，AI21 实验室表示，聊天机器人“只是一个实验”，并承认它可能会给出不准确的响应，应该“持保留态度”

当被问及此事时，Goshen 拒绝透露公司的收入数据，甚至是对增长的估计。但他表示，除了免费计划的 1 万多名用户外，Studio 还有“数百个”付费客户和设计合作伙伴——他不愿透露其中任何一个的名字——而 Wordtune 则有“数百万”用户。

鉴于训练复杂模型的成本，投资者可能面临巨大的扩张压力。AI21 实验室自己的研究估计开发一个具有 15 亿个参数(即模型用来生成和分析文本的变量)的文本生成模型的费用高达 160 万美元。侏罗纪-1 Jumbo 包含 1780 亿个参数。这还不包括为模型提供服务的托管成本；AI21 实验室表示，它保留了美国和海外“几家”第三方云提供商的服务。

Goshen 说:“(缺乏)市场知识，因为语言模型技术是如此新生，刚刚开始被采用。”。“有了新的资金，AI21 实验室将继续履行其使命，建立具有前所未有的理解和生成自然语言能力的人工智能系统。”
