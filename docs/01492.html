Adobe 推出 Creative Cloud Express，这是对 Adobe Spark | TechCrunch 的更新

图片来源: Adobe

Adobe 今天发布了 Creative Cloud Express ，这是一款移动和网络应用程序，将该公司庞大的 Creative Cloud Suite 和 Acrobat PDF 工具的一些最佳功能整合到一个应用程序中，帮助用户快速创建从社交媒体帖子到宣传海报和视频的任何内容。Creative Cloud Express 取代 Adobe Spark ( 见下文更新)。

Creative Cloud Express 使用模板优先的方法，内置了对股票图像和其他资产的访问，这意味着比单个的 Creative Cloud 应用程序更容易访问。该应用程序将有免费版和每月 9.99 美元的付费版，具有更多功能和更复杂的模板库。对新应用程序的访问也将包括在 Adobe 的 Creative Cloud 所有应用程序和旗舰单应用程序计划中。

图片来源: Adobe

除了网络应用，苹果应用商店、Google Play 和微软商店也有免费应用。

Creative Cloud Express 背后的总体想法是为非专业人士提供实现其愿景所需的工具。正如 Adobe 的 Ashley 仍然指出的那样，该公司近年来已经看到非专业用户的大量增长。但是，虽然这些用户中的许多人最初可能认为他们想要完整的 Creative Cloud 应用程序的精确度和控制力，但现实是他们真正想要的是一种快速而简单的方式来完成相同的任务。

“我们在 Creative Cloud Express 上所做的是，我们从我们广泛的网络和移动应用程序以及我们的核心 Creative Cloud 技术(无论是 Photoshop 和图像还是视频)中汲取所有知识，并将它纳入一个名为 Creative Cloud Express 的统一产品中，”斯蒂尔说。“这真的是为那些关注结果而不是过程的人准备的。他们不想从一张白纸开始，他们想从 1.75 亿张 Adobe 股票库中的图像开始。他们不想创建一种字体，他们想从 Adobe 字体库中获得 20，000 种令人惊叹的字体中的一种。他们不想去多个应用程序创建一个传单，然后创建一个 PDF，以便他们可以打印它。他们希望能够在一个地方完成所有工作。”

图片来源: Adobe

在实践中，这意味着您可以访问大量的模板库来开始，还可以使用工具来快速删除图像的背景，或者对这些图像应用 Photoshop 风格的滤镜和效果。由于与 Creative Cloud Libraries 的集成，用户还可以从 Photoshop 和 Illustrator 中获取资源，这些资源可能是同事为他们创建的，然后在 Creative Cloud Express 应用程序中重用它们。

还有一些工具可以将视频转换成 gif，将文档转换成 pdf 等等。这里有一个有趣的方面是集成了 Adobe Stock，它没有免费计划，但被集成到 Creative Cloud Express 的免费版本中(有一些限制)。免费用户将获得约 100 万张图片和其他资产。Premium 计划用户将获得 1.75 亿张 Adobe 库存照片、20，000 种字体以及 Photoshop Express 和 Premiere Rush。

“在 Creative Cloud Express 中，少即是多，”斯蒂尔说。“你不需要能够在 Photoshop 中做任何事情。就像你不需要神经过滤器，但你需要能够做一些简单的事情，如删除背景或对图像进行简单的编辑。使用 Acrobat，您不需要密码保护 PDF，您需要能够创建 PDF，编辑 PDF。很多简单性来自于我们自己对人们需要实现的最重要的事情的编辑。他们的意图是什么？”

图片来源:土坯

在某些方面，Creative Cloud Express 有点像 Spark，它是一款开发社交图片、短视频和网站的工具。Adobe 不会这么说，但如果你认为这是类固醇上的火花，我不认为你离得很远，因为 Creative Cloud Express 分享了许多用户界面设计和整体理念。事实上，如果你在发布前去了 adobe.com/express，它会把你重定向到 Spark 主页。

更新 : Creative Cloud Express 取代了 Adobe Spark，该公司在新闻材料或发布前简报中没有提到这一点(对我们和其他所有人)。为什么？我不知道。我想更新 Spark 和发布一个新的应用程序没有相同的铃声。我们已经要求公司给我们一个说法。

如上所述，该公司并不认为这是对 Spark、Photoshop Express 或 Premiere Rush 等应用程序的替代，所有这些应用程序也旨在吸收 Adobe 的一些核心功能和人工智能工具，并使它们更易于使用。

Creative Cloud Express 的目标受众也更广泛一些。Adobe 希望它成为从学生到小企业主的任何人的首选内容创作工具。

Adobe Creative Cloud 首席产品官兼执行副总裁 Scott Belsky 表示:“每个人都有自己的故事要讲，我们的使命是让每个人都能表达自己的想法。“在这个独一无二的时代，数百万人正在建立个人和专业品牌，我们很高兴推出 Creative Cloud Express，这是一个简单的基于模板的工具，它统一了创作、协作和共享流程，因此任何人都可以轻松地进行创作。”
