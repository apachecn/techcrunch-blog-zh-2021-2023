开放金融初创公司 Ayoconnect 的 API 在东南亚实现金融普惠| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/10/13/open-finance-startup-ayoconnects-apis-enable-financial-inclusion-in-southeast-asia/

专注于东南亚，Ayoconnect 的 API 使企业能够更快地推出新的金融服务，而不需要建立自己的技术基础设施。它还获得了印尼央行的许可，从而能够提供更多的服务。这家开放金融初创公司今天宣布，它已经完成了由 SIG Venture Capital 牵头的 1300 万美元的 B 轮融资，CE Innovation Capital 和回归投资者 PayU(Prosus 的支付和金融科技业务)也参与了该轮融资。这使其总融资额达到 4300 万美元，其中包括由老虎环球牵头并于 2022 年 1 月结束的 B 轮融资的第一批超额认购。

Ayoconnect 成立于 2016 年，拥有约 250 人的团队，目前正致力于为印度尼西亚消费者和中小企业提供更多金融包容性。它与监管机构和现有银行合作，最近获得了印度尼西亚银行(BI)支付服务提供商 1 类许可证。Ayoconnect 表示，它是印度尼西亚唯一一家获得央行许可的开放金融公司。

Ayoconnect 的新资金将用于领导层招聘，以及其 Ayoconnect 的产品和技术，包括支付、数据和银行的新解决方案以及开户和发卡的新 API。

这家初创公司最近与印度尼西亚七家最大的银行(曼迪里银行、BRI 银行、BNI 银行、CIMB 尼亚加银行、达纳蒙银行、印度尼西亚伊斯兰银行和新商业银行)合作推出了自动循环直接借记。这允许 Ayoconnect 的客户使用其直接借记 API，并能够从客户在多家银行的储蓄账户中进行经常性借记。

在创办 Ayoconnect 之前，创始人兼首席执行官 Jakob Rost 是 Lazada 的董事总经理。离开 Lazada 后，他在印度尼西亚生活了几年，在那里他看到了该国如何从更多的数字金融包容性中受益。他说，例如，按人口计算，中国是世界第四大国家，但大约一半的人没有银行账户。它的地理位置也很复杂，导致金融基础设施薄弱，银行业分散且不太规范。此外，Rost 补充说，印度尼西亚面向消费者的企业没有数字金融基础设施来管理自己的财务，同时为客户服务。

Ayoconnect 在 B 轮融资完成后很快就再次融资，因为它增长迅速，并在获得商业智能许可后获得了重要的战略合作伙伴关系。Rost 表示，新资本将加强 Ayoconnect 的资产负债表，并为未来几年的增长做好准备。

该平台现在为 200 家 API 客户提供服务，包括大型银行、金融机构、科技独角兽和金融科技公司，并提供超过 4，000 种嵌入式金融产品。其 API 涵盖两大类:开放银行 API 和支付服务 API，目标是构建东南亚最完整的开放金融栈。

Ayoconnect 的客户已经推出的金融服务的一些例子包括前面提到的直接借记，嵌入式金融(它与 PT 合作。印度尼西亚国有铁路运营商 kereta Api Indonesia(KAI)将在 KAI Access 移动应用中推出新的票务和生产力功能，用户可以购买手机积分、互联网数据订阅和电力代币。它还与印尼最大的伊斯兰银行 Syariah 银行合作，为增加新的数字和移动功能，目标是提高其客户的金融包容性和经济增长。

开放金融领域的其他东南亚初创公司包括 Brick 、 Finverse 、 Brankas 和 Finantier。ayo connect 与众不同的一个原因是它是印度尼西亚唯一获得许可的开放金融平台，这使它能够提供市场上还没有的解决方案。

“尽管在欧洲和美国，开放式银行和开放式金融已经相当成熟，但在东南亚，这一行业仍非常年轻，但正在快速增长。在印度尼西亚，数亿人正在接受新的数字服务，而更多的人仍然无法获得基本的金融服务，如银行账户，”罗斯特说。

“因此，该地区开放金融的潜力巨大，该行业有大量进一步发展的机会。我们很高兴看到这个领域的活动，并在推动生态系统发展方面发挥作用。”

在一份声明中，SIG 风险投资公司的阿克谢·巴贾杰表示，Ayoconnect 团队“多年来一直在运行大量的 API，并且非常有能力帮助客户快速安全地推出令人信服和有利可图的用例。由于其不断扩展的能力，Ayoconnect 继续经历着来自银行和 API 客户的强劲和不断增长的需求。我们喜欢他们的愿景，并相信他们有潜力改变和增强东南亚支付的未来。”
