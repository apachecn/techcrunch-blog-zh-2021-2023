开放式银行初创公司 ably 获得 6.01 亿美元，以增强其基于人工智能的消费者借贷平台| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/06/open-banking-loans-platform-abound-nabs-601m-to-supercharge-its-consumer-lending-business/

在英国和欧洲缓慢起步之后，开放式银行正在受到金融科技公司的欢迎，这些金融科技公司使用 API 来访问银行数据和 rails，并将其作为传统信贷网络的替代方案来开发产品。

现在，一家总部位于英国的名为的消费贷款服务公司 abundy正加倍努力实现其在该领域的雄心，大举筹资，以推动其基于银行的开放式业务。这家初创公司已经筹集了高达 5 亿英镑(按今天的利率计算为 6.01 亿美元)——这笔钱将用于帮助贷款，为其平台带来更多客户，并投资于其技术，该技术结合了公开的银行数据和机器学习算法，以建立一个 afforbid 认为对申请人来说更好的“信用评分”。为了补充其在英国提供的直接面向消费者的服务，Abound 还计划在欧洲拓展 B2B 服务，该公司一直在构建自己的开放式银行框架 PSD2。

首席执行官兼联合创始人杰拉尔德·沙佩尔在接受采访时说，“我们认为自己不仅仅是信用评分，”他将银行交易数据描述为类似于“金融 x 光”，而银行交易数据是 ably 用来构建其基于人工智能的风险和贷款档案的在贷款方面，这些反过来又有助于“理解真正的支付能力”。

它的崛起与此同时，我们看到围绕开放银行的活动越来越多。去年， Visa 以超过 20 亿美元的价格收购了为数千家银行提供 API rails 的开放银行开发商 Tink 。另一家主要的 rails 提供商 TrueLayer ，最近一次融资超过 10 亿美元(假设那是在 2021 年……)。与此同时， Token.io 和 Vyne 是创业公司在开放银行标准(分别是账户到账户支付和商户服务)上构建更具体应用的例子。

Abound 的新融资包括债务和股权:美国花旗银行和瀑布资产管理公司的客户提供债务部分；K3 Ventures、GSR Ventures 和 Hambro Perks 提供股权。

作为贷款创业公司的典型，这 6.01 亿美元中的绝大部分是债务，将用于贷款；较小的股权部分将用于投资企业本身。比比皆是没有披露估值，但对于一些背景，沙佩尔证实，这家初创公司，以前被称为 芬特恩。AI (严格来说仍是其母公司的名称)之前筹集了不到 1100 万美元的股权和大约 6000 万美元的贷款。

更尖锐地说，这里筹集到大笔资金的原因是，自 2020 年推出以来，人们的兴趣一直在激增。

其服务——基于 1000 至 10000 英镑的贷款，还款期限长达 5 年(尽管平均还款期限为 2.5-3 年),公司担保的利率低于银行提供的利率(目前为 24.8%的年利率)——平均每月增长 30%；迄今为止，它已经向超过 15 万名客户发放了贷款，并表示有望在 2025 年前贷款 10 亿英镑(12 亿美元)。

所有这些信号不仅反映了当今的经济状况，也反映了金融科技的状况。是的，目前普通消费者肯定需要贷款来补充他们每月的固定收入。但也值得注意的是，新的金融科技服务如何被接受和采纳为获得流动性的一种手段。换句话说，用 neobanks 和 app 理财已经不是什么新鲜事了；这只是完成任务的另一种方式，也许对某些人来说是更好的方式。

沙佩尔说，他和他的联合创始人米歇尔·何(Michelle He)多年前就有了建立 Abound 的想法，当时两人都在管理咨询公司工作——沙佩尔在麦肯锡，何在 EY——他们与大型金融服务客户合作，帮助建立信贷产品，并围绕开放银行的基础开展工作。他说，两人看到 API 框架为那些了解如何以及在哪里使用它们的人提供了一个清晰的开端。

“消费信贷已经崩溃了，”他说。"其中大部分深深植根于 70 年代和 80 年代的技术。"

这种巩固包括 FICO 信用评分，以及对 Equifax 和 Experian 公司主导的数据的访问，以确定信用度。除此之外，消费者对贷款的体验普遍较差，而且我们已经看到了大量掠夺性贷款的剥削行为，你可以看到市场上存在更好的产品缺口，这些产品可以更好地满足客户的需求。

具有讽刺意味的是，对许多消费者来说，贷款产品的现状实际上可能是好的，特别是那些有信用记录的人，他补充说，可以明确地分为“优质”或“次级”案例。然而，对于那些新进入市场的人，即所谓的“准优质”消费者来说，它实际上是无法使用的。沙佩尔估计，仅在英国就有大约 1500 万个这样的人。

他说，大多数贷款机构会拒绝这些消费者的贷款或信贷申请:“他们太不确定了。”

因此，解决方案显而易见:建立一个系统，利用开放式银行获取个人如何管理普通银行账户收入和支出的基本实时细节；使用人工智能从数据中推断洞察力；创造一种新的信用评分。这是 Abound 在 2020 年推出之前花了大约三年时间建立的，也是现在形成其业务基础的东西。

似乎显而易见的是，一家银行本身可以、应该并且愿意推出类似的产品，提供自己的基于银行数据的贷款产品——至少是为自己的客户，如果不是为那些与他人一起存钱的人提供贷款的话。但是沙佩尔说这并不像看起来那么简单。

“这非常重要。银行需要五年或更长时间来改变他们的流程，”他说。他们的流程通常是那些在 70 年代和 80 年代根深蒂固的服务方法:FICO 分数，来自英国统计局的综合经济数据，以确定贷款资格，以及客户拖欠贷款或按约定还款的可能性。

与此同时，Abound 表示，它的证据已经在布丁中，可以这么说:沙佩尔表示，在过去两年中，这家初创公司的违约率比英国的行业平均水平低 70%

也就是说，它并不完美，但似乎比它想要取代的东西更好。“每 10 个竞争对手违约，我们就有 3 个，”他说。

但由于这是我们正在谈论的技术，当大银行或其他创业公司建造了 arbond 已经建造的东西时，这场比赛并没有结束:arbond 相信自己拥有先发优势，因此，在竞争对手建造类似东西的时候，沙佩尔相信 arbond 会开发出更好的人工智能算法，以更好地管理自己的费率。尽管如此，其他的如另一个借贷平台 Zopa ，已经推出了一些基于开放银行的信用评分服务，这是一个远离垄断市场的信号。

这种领先优势也是促使投资者支持该公司的原因。

K3 Ventures 首席执行官郭孟雄在一份声明中指出:“贷款行业由传统的信用评分等旧做法主导，这些做法忽视了过去十年的技术发展。”“Abound 正在提供一种独特的产品和一种与众不同的方法，这种方法已经证明了它能够为成千上万的客户服务。我们很高兴看到在未来的几年中，公司的报价会有所增长。”

瀑布资产管理公司的董事 Krishin Uttamchandani 在一份声明中补充说:“瀑布很高兴成为 Abound 业务扩张的一部分，因为它寻求以更明智的方式利用开放银行来帮助消费者。”“由一个强大的管理团队领导，我们很高兴与他们合作，我们相信他们有强大的技术支持、核保方法和风险观。在最初两年的放贷中，Abound 取得了令人印象深刻的成绩，这应该为一个强大的平台打下基础，以便更好地服务于那些应该能够通过开放银行业务获得更廉价信贷的客户。我们很高兴成为丰富旅程的合作伙伴。”
