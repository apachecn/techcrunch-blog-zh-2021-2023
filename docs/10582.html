CoRise 提升技能的方法包括更少的课程和更多的访问 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/09/28/corises-approach-to-up-skilling-involves-fewer-courses-and-more-access/

尽管过去几年教育技术投资和创新蓬勃发展，创始人朱莉娅·斯蒂格利茨(Julia Stiglitz)认为还有很大的发展空间，她作为 Coursera 的早期员工(T3)进入了教育技术领域。她的新公司 CoRise 向希望提升职业技能的人出售专家指导的编程。这是在一个拥挤的行业中的一个新的尝试，重量级人物包括 Udemy，Udacity，Guild Education，以及她的前雇主。

斯蒂格利茨在接受 TechCrunch 采访时表示:“我们还没有解决这些问题，事实上，它们还在增长。”。这位 edtech 资深人士是对的:下一代 edtech 仍在寻找平衡动机和行为改变的方法，以可扩展的形式以可接受的价格提供。在参与度和规模之间有一个固有的权衡——这是一只大象，即使是独角兽也无法完全避免。

进入 CoRise ，它想要做这一切。这家由斯蒂格利茨、 Sourabh Bajaj 和雅各布萨缪尔森、创办的初创公司将希望学习和提高高技术技能(如开发运维或数据科学)的学生与专家配对。CoRise 将专家定义为科技公司的领导者；例如，广告上的讲师包括 Drizly 的数据工程经理、Wikimedia 的前首席技术官和 ShareChat 的机器学习主管。有些课程，比如这个 SQL 速成班，甚至是由 CoRise 的员工教授的。

就早期用户而言，它不适合那些想进入科技领域的创业者。相反，CoRise 正在向需要为他们的人才量身定制解决方案的企业销售产品。在与学习和发展领导者的交谈中，创始人了解到组织要么向全体员工推出异步教育平台，要么引入顾问进行客户培训；“中间没有任何东西，”她说，所以她建造了它。

斯蒂格利茨不希望 CoRise 发展到托管由数千名教师教授的 20，000 门课程的地步。相反，这家初创公司希望提供一门应用机器学习课程，每次教授 1000 或 5000 名学生。

通过专注于更大的群体，CoRise 采取了一种不同于一些竞争对手的方法。例如，Udemy 创始人 Gagan Biyani 正在开发 Maven ，它提供专家指导的编程，将人们分成小组，以培养协作和思想交流。与此同时，斯蒂格利茨认为，较小的群体推高了项目的费用。在她看来，大班标准化课程是让编程“真正容易理解”的唯一途径。

她补充说，单一课程的平均费用为 400 美元，学生可以花大约 1000 美元购买所有课程的通票。相比之下，Maven 的一门课程——也许方正金融的这门课程——可能要花费 2000 美元。

她说:“我们试图弄清楚，在这种规模下，你如何为学习者获得成果或结果，并仍然让它真正变得容易获得，仍然让教师从中获得稳定的收入。”“我们需要弄清楚如何让很多人加入一个团队，同时还能有很好的体验。”

大班额和标准化课程的挑战，当然是缺乏个性化。CoRise 创建了一个“轻推基础设施”,用于查看单个学生如何与课程、相关讲座和到期作业进行交互。它还会查看学生是否去了办公室，或者他们是否及时提交了作业。

后端信息帮助 CoRise 向需要提醒以寻求额外支持的人发送自动“轻推”或推送通知。创始人解释说，课程经理还会跟进人类的回应，这样学生就不会觉得全是机器人和自动信息。

随着时间的推移，CoRise 可以变得更聪明，如何支持那些甚至在上班时间之前就陷入困境的学生，这是个性化学习运动的一个共同愿景。

“我们试图弄清楚的很多事情就像是，什么样的人才能保持这种激励因素？然后，我们可以在后端扩大规模，以扩大规模并降低成本，从而制定一个合理的价格，”她说。斯蒂格利茨说这门课程的平均完成率是 78%。这家初创公司的 nudge 框架当然很有吸引力，但这只是为学习者提供更加定制化和更具吸引力的体验的一步。虽然低成本当然很重要，但如果其他竞争对手也试图压低价格来赢得客户，就可能会出现一场竞争。

虽然这家初创公司没有披露通过其平台的学习者人数，但它确实表示，他们来自 500 多家公司，包括 Spotify、沃尔玛和 Lyft。它有 68 个 NPS 分数。

这家初创公司已经筹集了数百万美元，以更好地解决上述问题。迄今为止，CoRise 告诉 TechCrunch，自推出以来，它已经从格雷洛克、GSV 和牛仔风险投资公司筹集了 850 万美元，其中 550 万美元是第一笔支票，随后的 300 万美元是最近的牵引力。其他投资者包括 OpenAI 的联合创始人格雷格·布罗克曼(Greg Brockman)和 DeepMind 的联合创始人穆斯塔法·苏莱曼(Mustafa Suleyman)。

我问斯蒂格利茨的最后一个问题很烦人:她是如何专注于更少的课程和导师，与她的投资者坐在一起的？难道他们不想让她一直开设新的课程吗？

“压力会越来越大，越来越大，但在班级内部，压力会越来越大，”她说。“我们的目标是希望向 1，000 名员工提供 SQL 培训的大公司，但他们不会希望推出该课程的八个不同版本。这就是我们获得规模的方式。”
