苹果将通过即将推出的功能解决 AirTag 跟踪问题

原文：https://web.archive.org/web/https://techcrunch.com/2022/02/10/apple-to-address-airtag-stalking-problem-with-upcoming-features/amp/

图片来源:苹果

苹果公司今天宣布将对 AirTag 和 Find My network 进行一系列更新，旨在解决 AirTag 配件被用于在个人不知情或未经其同意的情况下跟踪个人或他人财产的问题。继 AirTags 2021 年春季推出、之后，来自当地警察局的众多 媒体 报道和更新警告了一些事件，其中 air tags 被用于不必要的跟踪，包括对人和有价值的财产的跟踪——如打算偷窃的汽车窃贼。这给苹果带来了越来越大的公关噩梦，苹果曾将自己定位为一家关注消费者隐私的公司。

如今，苹果正在回应消费者、安全专家和执法机构的反馈，计划改变空中标签的工作方式。

该公司宣布，它将很快对 AirTags 和 Find My network 进行一系列更新，首先是新的隐私警告、警报和文档。它还“研究”在未来版本中引入其他功能，包括新的精确查找工具和对 AirTag 警报和声音的调整。

首先，在即将到来的软件更新之后，苹果设备将在 AirTag 设置过程中显示新的隐私警告。这一警告旨在阻止人们将 AirTags 用于除跟踪自己的个人物品之外的任何用途。它将告知用户，未经同意跟踪他人是一种犯罪行为，执法部门可以要求提供关于 AirTag 所有者的身份信息。

图片来源:苹果

苹果还表示，在收到的所有与 AirTag 相关的请求上，它一直在积极与执法部门合作，并指出它能够提供账户细节，以回应传票或其他有效的执法请求。这是可能的，因为每个 AirTag 都有一个唯一的序列号，与 Apple ID 相关联。该公司表示，通过提供这些信息，执法机构在许多情况下都能够追踪到 AirTag 的所有者，然后他被逮捕和指控。然而，苹果拒绝透露它参与了多少起案件。

此外，苹果公司将改变其更令人困惑的警报之一，以减轻人们对被跟踪的恐惧，而实际情况并非如此。该公司表示，他们从用户那里听到了关于“检测到未知附件”的警报，这让他们认为一个未知的 AirTag 正在跟踪他们。但苹果公司证实，对于未知的附近 AirTags，只有 AirPods(第三代)、AirPods Pro、AirPods Max 或第三方 Find My network 配件才会显示这一警告。苹果现在将更新这一警告，以表明“AirPods”一直在与用户一起旅行，而不是“未知的配件”。

图片来源:苹果

苹果还将更新其关于不必要跟踪的支持文档，现在包括更多关于查找我的配件及其跟踪提醒的信息。将提供图像以提供警报的具体示例，文档将指出如果你是跟踪的受害者该做什么的进一步指示。

然而，对 AirTag 和 Find My 更大的改动仍在进行中。

其中包括一种新的精确定位能力，它将允许人们在收到警报后定位附近的未知航空标签。许多 AirTag 案件中的跟踪受害者抱怨当他们收到一个 AirTag 与他们一起旅行的警告时，他们在寻找后无法找到它。这让他们感到暴露和脆弱，因为他们不确定 AirTag 是否还在附近。

图片来源:苹果

IPhone 11、iPhone 12 和 iPhone 13 用户将能够利用精确定位功能来查看未知 AirTag 在范围内的方向和距离，就像他们使用自己的 AirTag 设备一样。该功能使用来自摄像头、ARKit、加速度计和陀螺仪的输入，以及声音、触觉和视觉反馈，将它们导向 AirTag。

从 AirTag 跟踪案件中出现的另一个投诉是，在未知 AirTag 被放置在个人身边以跟踪他们和受害者的设备实际警告他们之间有多少延迟。一些人说，在他们被软件提醒之前，AirTag 已经跟着他们走了几个小时，走遍了整个城镇。

苹果公司表示，它计划更新其警告逻辑，以“更早地通知用户”一个未知的 AirTag 或 Find My network 配件(将包括 AirTag 的替代品，如 Chipolo )可能与他们一起旅行。然而，该公司没有说用户多久会收到这些警告。

苹果还将在晚些时候调整其警报的工作方式。当你检测到一个未知的 AirTag 时，它会同时在你的 iPhone，iPad 或 iPod touch 上显示一个视觉警报并播放声音。该警报将指导用户使用 precision finding 工具进一步帮助定位 AirTag，或者允许他们触发 AirTag 发出声音——如果 AirTag 被放置在警报难以听到的地方，这可能会有所帮助。与此同时，如果 AirTag 扬声器被篡改，查找工具可以提供帮助，这是一些跟踪案件中的一个问题。事实上，最近几周，易贝和 Etsy 上甚至出现了禁用扬声器的航空标签。

苹果公司表示，以后还将调整音调顺序，使用更响亮的音调，使未知的 AirTags 更容易被发现。

在宣布这一消息时，苹果公司辩称，使用技术跟踪人们是一个社会问题，空中标签也被用来做好事。它推动其他设备制造商效仿它的做法，在自己的产品中加入更多类似这样的主动功能。例如，AirTag 的竞争对手 Tile,说它正在研究安全功能,但还没有推出。该公司最近被家庭追踪和交流应用 life 360收购。

包括国家终止家庭暴力网络和国家犯罪受害者中心在内的安全组织对苹果的努力表示赞赏。前者通过董事埃里卡·奥尔森(Erica Olsen)表示，很高兴“苹果正在参与关于受害者安全的对话”，并“继续改善安全措施”。后者通过执行董事雷尼·威廉姆斯发表了一份稍显温和的声明。

威廉姆斯说:“这些不必要的跟踪警报向我们展示了苹果的系统正在工作，同时也提高了人们对这个问题的认识。”她补充说:“如果任何人经历了技术导致的虐待、跟踪或骚扰，除了执法部门之外，我们还可以为他们提供资源。”

虽然苹果可能是正确的，它在解决不必要的跟踪问题上比其他公司做得更多，但它也坚持更高的标准。苹果不仅有资金和资源投资于更全面的产品开发，还将该公司定位为一家关心消费者隐私的公司，包括最近推出的像这样的应用反跟踪功能。

目前还不清楚为什么苹果没有先见之明，考虑到空中标签可能会被滥用。这也提出了一个问题，即科技行业雇佣更多女性的大规模斗争是否会成为这里的一个问题。女性代表了跟踪受害者的大多数，她们很容易发现空中标签功能的问题。

记者联系到苹果公司请其置评，但苹果公司拒绝透露更新将于何时推出，也拒绝透露包含这些变化的软件版本号——对于那些想知道自己是否以及何时受到新功能保护的消费者来说，这些信息可能会很有帮助。
