今天是谷歌 I/O 主题日！一年一度的谷歌开发者大会以一连串连珠炮似的公告拉开帷幕，最终揭开了他们最近在幕后所做的许多事情。

没有时间收听整个两小时的演示？我们明白这一点——这就是为什么我们把最大的新闻打包成一个易于消化、易于浏览的列表。让我们开始吧！

图片来源:谷歌

布莱恩·希特写道:“有件事需要你好好想想:谷歌从未制造过自己的智能手表——但“这一切将在今年秋天改变。”

细节仍然很少(现在外面的很多东西实际上已经提前泄露了)，但谷歌已经展示了其首款 Pixel watch 的首批官方图像，预计将于今年秋天晚些时候推出。在这里找到所有细节。

Pixel 7 和 Pixel 7 Pro

图片来源:谷歌

去年，谷歌在 I/O 上尝试了一些新的东西:他们展示了关于他们的新旗舰 Pixel 手机的一点信息——当时是 Pixel 6——但将大部分信息留到了几个月后的另一次发布会上。

他们今年发布了 Pixel 7 和 Pixel 7 Pro，做了同样的事情。价格和发布日期等细节仍然保密，但我们知道的是:它将运行 Android 13，并使用下一代张量芯片。它借鉴了 Pixel 6 的许多设计，包括背面凸起的“相机栏”。想要更多吗？点击此处查看完整帖子。

像素 6a

图片来源:谷歌

谷歌正在借鉴已经非常好的 Pixel 6 的大部分技术，削减一些规格，将价格从 599 美元降至 449 美元，并发布 Pixel 6a。它的屏幕更小(6.1 英寸对 6.4 英寸)，内存更少，摄像头也降级了——但它仍然拥有谷歌定制的张量芯片、泰坦 M2 安全芯片和 5G 支持。点击此处查看详细信息。

谷歌钱包

谷歌长期以来一直提供谷歌支付(Google Pay)，这是一款 Android 应用程序，你可以在其中存储数字信用卡，用于非接触式支付。他们现在通过一个名为谷歌钱包的新应用程序扩展了这一概念，该应用程序还将“允许用户存储信用卡、会员卡、数字身份证、公交通行证、音乐会门票、疫苗接种卡等物品。”推广过程会因您所在的国家而有所不同— 在这里找到所有这些细节。

谷歌助手的改进

图片来源:谷歌

谷歌助手变得更加智能了！弗雷德里克这里有的全面分解。

更自然的交流:谷歌助手现在能够更好地理解你何时犯了命令错误，或者何时你需要一秒钟来理解你想说的话。在 stage 上给出的例子中，演讲者说“你能播放来自 mmm 的那首新歌吗……”而谷歌助手的回应是“mmhmm？”等待他们完成他们的想法。

看和说:在内置摄像头的谷歌助理设备上(比如 Nest Hub Max)，你再也不用在问问题之前说“嘿，谷歌”——只需看着设备，它就会利用接近度/头部方向/凝视方向等信息来理解你在问它问题。

快速短语: Nest Hub Max 现在可以让你随意使用某些常用命令，而无需先说出热门词汇。所以你可以喊“几点了？”或者“关灯”进入房间，谷歌助手就会做出相应的动作。

谷歌地图的改进

图片来源:谷歌

谷歌地图正在使用一些新的技巧——点击这里查看莎拉的完整帖子。

“沉浸式”视图:谷歌地图正在获得一种新的 3D 探索模式，从选定的主要城市开始，允许你放大该城市的 3D 模型，以更好地了解一切。随着他们的数据集扩大，3D 模型将增长到包括受欢迎的餐馆和地点的内部。

环保路线扩展:去年年底，谷歌推出了一项功能，让你选择你的路线，以优化车辆效率，而不仅仅是最快的路线。今年晚些时候，它将把这一功能扩展到欧洲。

第三方实时视图:早在 2019 年，谷歌就开始推出一项功能，利用你手机的摄像头和你周围的建筑/地标来确定你在世界上的确切位置以获得更准确的导航——最常见的是，当你刚开始一条新路线时，确定你应该朝哪个方向走。谷歌表示，它正在向第三方开放这项技术，展示了一些例子，如帮助音乐会观众找到座位或帮助通勤者找到哪里停放他们租用的电动自行车。

谷歌翻译的新语言

谷歌翻译正在学习几十种新语言，重点是“拥有庞大但服务水平低下的人口的语言”。新增内容包括盖丘亚语、瓜拉尼语、艾马拉语、梵语和特松加语。谷歌表示，他们今天添加的许多语言在几年前在技术上是不可能支持的，今天只有通过机器学习的进步才能实现。在这里找到所有细节。

Chrome 的虚拟信用卡

谷歌 Chrome 现在将能够生成一个“虚拟”信用卡号，以确保你真实信用卡号的安全。如果虚拟号码被盗，你只需撤销它，并生成一个新的，而没有获得一个全新的卡的麻烦。详情如下。
