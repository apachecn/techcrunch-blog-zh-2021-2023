总部位于奥斯汀的 ICON 公司获得了美国宇航局价值 5720 万美元的月球建筑技术合同

原文：https://web.archive.org/web/https://techcrunch.com/2022/11/29/austin-based-icon-awarded-57-2-million-nasa-contract-for-lunar-construction-tech/

总部位于奥斯汀的 ICON 公司授予美国宇航局价值 5720 万美元的月球建筑技术合同

建筑技术公司 ICON 已经筹集了超过 4 亿美元的资金，并从美国宇航局获得了一份新合同，开发在月球和火星上建造的新系统。

这份价值 5720 万美元的合同是之前与美国空军的小型企业创新研究(SBIR)两用合同的延续，该合同部分由 NASA 资助。该奖项将支持 ICON 所谓的“奥林巴斯计划”的发展，这是一项利用原地资源在月球和火星上建造建筑物的雄心勃勃的计划。

ICON 首席执行官杰森·巴拉德(Jason Ballard)在一份声明中说:“为了将太空探索范式从‘往返’转变为‘停留’，我们将需要强大、有弹性、功能广泛的系统，这些系统可以利用月球和其他行星的本地资源。”很明显，美国宇航局同意这一点。事实上，该机构已经明确表示，其雄心勃勃的阿尔特弥斯月球计划的目标之一是在月球上建立长期的人类存在。但到目前为止，美国宇航局还没有制定出明确的计划，说明这些宇航员到达那里后将住在哪里。

ICON 以其 3D 打印房屋而闻名，它已经在奥林巴斯项目上工作了一段时间。该公司于 2020 年 10 月获得了美国空军的首批 SBIR 拨款，金额为 1455 万美元。这笔最新的资金将使该项目至少再存活几年:合同将持续到 2028 年。

根据合同条款，ICON 将与美国宇航局马歇尔太空飞行中心合作，开展一项名为“月球到火星行星自主建造技术”的项目。该公司正计划研究月球风化层的样本，并将其硬件和软件带入太空，以帮助其开发能够在月球寒冷、低重力的大气层中发挥最佳作用的建造方法。栖息地并不是该公司关注的唯一问题:它还关注着陆场和其他基础设施，以支持持续的月球探索。

ICON 自 2017 年底成立以来，经历了爆发式增长。该公司去年 8 月获得了一笔2.07 亿美元的 B 轮，半年后又完成了另一笔1.85 亿美元的。消息人士告诉 TechCrunch，最新的融资将 ICON 的估值推高至接近 20 亿美元。
