实时数据初创公司 Quix 获得由 MMC Ventures 领投的 1290 万美元 A 轮融资

流数据技术的复杂性——不仅仅是流视频，而是任何类型的流数据——使高速数据处理变得令人头疼。

因此，像 Spark 和 Flink 这样的公司已经涌现出来解决这个 ksqlDB。许多要么是基于 Java 的解决方案，要么是基于 SQL 的分析解决方案。然而，英国初创公司 Quix 表示，它是一个用 Python 开发事件驱动应用程序的平台，这些应用程序可以用于，比如说，基于物理的数据建模和机器学习中的异常检测。

它现在筹集了 1100 万美元/1290 万美元的首轮融资，由伦敦的 VC MMC Ventures 牵头，现有投资者 Project A Ventures(柏林)和 Passion Capital(伦敦)参与。

Quix 的创始人熟悉实时决策，曾在一级方程式赛车中工作过，那里的成功基于毫秒。事实上，它的客户之一是迈凯轮，以及移动创业公司 Voi 和国民医疗服务体系(英国)等。

Quix 的联合创始人迈克·罗萨姆(Mike Rosam)在一份声明中表示:“许多公司正在努力将 Kafka 等原始技术结合到实时数据能力中……这笔新资本将推动我们简化事件驱动的数据工程的使命，以便更多公司可以构建现代数据密集型应用。”

MMC Ventures 的合伙人奥利弗理查兹(Oliver Richards)补充道:“我们一直在数据基础设施领域进行越来越多的研究，很明显，消费者和 B2B 使用案例对实时流媒体数据的需求不断增长。”
