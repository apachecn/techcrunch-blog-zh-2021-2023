数据泄露数周后，儿童技术营 iD Tech 仍然保持沉默

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/23/id-tech-kids-tech-camp-data-breach/amp/

影像学分: Gorodenkoff / Getty 影像

几周前，黑客从孩子们的科技编码营 iD Tech 窃取了成千上万用户的个人数据，家长们仍在寻找答案，有些人担心他们孩子的数据会在数据泄露中受损。

为孩子提供校园课程、在线技术和编码课程的 iD Tech 尚未承认违规或通知家长。

数据泄露的消息在 2 月份传出，此前一名黑客在网络犯罪论坛上声称在一个月前的 1 月 3 日黑了 iD Tech。

该黑客声称已经窃取了近 100 万份用户记录，包括姓名、出生日期、以明文形式存储的密码以及大约 41.5 万个唯一的电子邮箱地址。当收到电子邮件时，iD Tech 并未对此提出异议。这相当于每个父母的账户上有一个或多个孩子在科技夏令营上课。

有些家长直到 3 月 6 日才知道，当时像【我被感染了吗】这样的数据泄露通知服务获得了数据，并且向受影响的家庭发送了通知。当其他服务，如火狐或他们的设备安全软件，通知他们他们的信息被发现在被破坏的数据中时，其他父母发现了。

一位从违规通知服务中得知其数据被盗的家长告诉 TechCrunch，被盗信息只是 iD Tech 从账户持有人和使用其平台的儿童身上收集的数据的一部分，包括性别、账单信息和一些健康数据，如免疫接种。

父母说，被泄露的数据必须与孩子的出生日期有关，因为他们从来没有提供自己的出生日期。

父母说，iD Tech 还没有通知他们违规的情况。当母公司联系该公司询问时，iD Tech 声称已经通知了受影响的账户持有人。

事实上，无论是在其网站上还是在其任何社交媒体渠道上，iD Tech 都没有公开承认这一违规行为。也没有证据表明 iD Tech 已经将违规行为通知了受影响的账户持有人。

当通过电子邮件联系时，iD Tech 首席执行官 Pete Ingram-Cauchi 拒绝解释该公司为什么没有公开承认违规。当被问及时，Ingram-Cauchi 拒绝提供 iD Tech 声称发给家长的通信副本。该公司拒绝透露是否已根据数据泄露通知法向州检察长办公室报告了此次泄露事件。

相反，iD Tech 提供了一份来自通用公司电子邮件地址的简短声明，拒绝置评，称正在进行调查。邮件的发件人拒绝透露他们的姓名。

Ingram-Cauchi 没有回复后续的电子邮件。
