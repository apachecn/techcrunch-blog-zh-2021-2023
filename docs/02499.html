Twitter 发布 NFT 个人资料图片——但仅针对 Twitter Blue 订户 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/01/20/twitter-blue-subscription-users-are-first-gain-access-to-a-new-nft-profile-picture-feature/amp/

图片来源:推特

Twitter 今天推出了一项新功能，允许用户展示他们的 NFTs，即不可替代的令牌，这是一种认证存储在区块链上的数字资产的方式。该公司正在通过 Twitter Blue 的早期访问实验室功能向 iOS 上的 Twitter Blue 订户推出 NFT 的个人资料图片。对 Twitter 的 NFT 产品的支持将在以后推广到 Android 用户和网络。然而，Twitter 表示，虽然只有 iOS 用户能够将他们的 NFT 设置为他们的个人资料图片，但 Twitter 上的每个人都可以看到新的六边形图片，不管他们目前使用的是什么平台。

Twitter Blue 订阅服务尚未在全球范围内提供，这将限制 NFT 个人资料图片在目前提供服务的早期市场的应用——美国、加拿大、澳大利亚和新西兰。

该公司早些时候暗示了其更全面地拥抱 NFTs 的计划，这已经成为平台上的一个大话题。许多加密爱好者也已经将他们的个人资料图片设置为他们的 NFT 照片，因为 Twitter 尚未提供任何官方手段来验证用户的加密钱包和验证所有权。

图片来源:推特

9 月，在一大堆其他更新中，Twitter 首次展示了它的 NFTs 计划。当时，Twitter 吹捧身份验证将如何让创作者在 Twitter 上更好地展示他们的作品，并表示正在探索让他们展示自己作品的想法。

“Twitter 是人们谈论他们关心的事情的地方，也是人们第一次体验加密和 NFTs 的地方，”公司发言人在一封电子邮件声明中告诉 TechCrunch。“我们现在看到人们将 NFTs 作为一种身份和自我表达的形式，以及加入蓬勃发展的社区和 Twitter 上越来越活跃的对话的一种方式。”

要作为 Twitter Blue 订户使用该功能，您只需像往常一样进入您的个人资料，更改您的个人资料照片。在这里，您将看到选择 NFT 的新选项。然后，您连接您的加密钱包。

在发布时，支持比特币基地钱包、彩虹、MetaMask、Ledger Live、Argent 和信托钱包。验证后，您将选择想要展示的 NFT。Twitter 表示，目前，以太坊上制作的 JPEG 和 PNG NFT(ERC-721 或 ERC-1155 令牌)可以用作 NFT 的个人资料图片。

在后来的更新中，Twitter 表示正在考虑增加对 SVG 的支持，但现在还不能呈现。此外，仅支持链上 NFT。举例来说，如果你在 Opensea 上偷懒，这些照片就不能作为你的 Twitter NFT 个人资料图片。

一旦上线，穿越 NFT 的 Twitter 用户将能够点击个人资料图片，了解更多关于艺术品、藏品、出处等信息。例如，您将能够看到一个项目或集合是否已经被 OpenSea 或另一个第三方市场验证。

Twitter 对将去中心化技术融入其平台越来越感兴趣，这包括支持比特币提示，以及最近聘请苔丝·里纳森担任其密码工程主管。该公司去年还尝试铸造自己的免费 NFTs 。

该公司尚未决定是否将在未来某个时候向非订户提供该功能，但表示将监测 Twitter Blue 用户的反馈，并随着时间的推移进行更多更新。

注意:一个不合时宜的 OpenSea 中断使 Twitter 的发布变得复杂，但后来得到了解决。
