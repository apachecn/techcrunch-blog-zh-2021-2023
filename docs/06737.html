MotiSure 利用微支付推动移动保险

原文：https://web.archive.org/web/https://techcrunch.com/2022/05/25/kenyas-motisure-rides-on-micro-payments-to-drive-personal-mobility-insurance-growth/

负担能力是肯尼亚保险渗透的最大障碍之一。通常，投保人需要为长期保险支付一次性费用，而报价通常超出了大多数人的承受能力。肯尼亚保险技术公司 MotiSure 正在努力填补这一空白。

这家初创公司的目标是摩托车出租车(boda boda)运营商、其乘客和其他公共交通工具的用户(以下简称通勤者)，正在围绕个人意外保险的日常小额支付建立一项业务，有些保费低至 0.1 美元。

用户可以选择加入服务 USSD 代码，并使用移动货币进行支付。这个由 API 驱动的平台正在利用技术来扰乱一个主要由传统保险公司主导的市场，其中许多公司不愿意放弃冗长的文书工作，以及保费和索赔的传统支付模式。

“我们通过价格合理的保险和便捷的渠道提供安全网，”首席执行官兼联合创始人乔尔·马查里亚告诉 TechCrunch。

摩托车出租车经营者的个人事故保险要求每天 0.1 美元(每月 3 美元)的保费，包括每年高达 6 000 美元的医疗费用，以及因事故、残疾或死亡后住院治疗而造成的收入损失付款。骑手们必须持续支付至少七天才能获得利益。

MotiSure 的联合创始人兼首席执行官乔尔·马查里亚(Joel Macharia)表示，该公司正依靠微支付推动个人移动保险的增长。图片来源: MotiSure

MotiSure 瞄准了一个根据政府数据(2018 年)拥有超过 140 万辆注册摩托车的市场。这个数字还在增长，2020 年将有 210，103 辆摩托车进口到中国；随着 2020 年进口数量增长 20%,去年增长 15%,摩托车仍然是肯尼亚和整个非洲受欢迎的交通工具。

通勤者也有资格为他们的每次旅行购买人身意外保险，他们支付的保费是基于旅程的持续时间。与乘客不同，通勤者立即有资格获得福利，MotiSure 正在关注大约 153 万每天使用公共交通工具的通勤者。

这家初创公司于去年 7 月成立，已经与肯尼亚的一些公共服务车辆公司合作，对他们的产品进行试点。

“进入的障碍之一是成本，这就是为什么我们已经满足于这些小额支付和按使用付费，以使成本变得更微不足道，这样人们就不会介意付费，”马查里亚说。

在许多新兴市场都有业务的小额保险公司(前身为 MicroEnsure)是这家初创公司的承保人。

马查里亚说，他们的方法受到研究的启发，研究显示，博达博达的乘客希望获得超出资产覆盖范围的保险产品。在肯尼亚，包括摩托车在内的所有车辆都被要求至少购买一份第三方保险(大多数投保人都购买了这种保险)，这通常是最便宜的选择，但它排除了发生事故时资产所有者的风险。尽管事实上摩托车事故占肯尼亚道路事故伤害的 20%。

“现有的综合保险或第三方保险不覆盖乘坐者或后座乘客。我们发现，由于他们的工作性质，他们(骑手)需要一份可以应对住院、死亡或残疾等意外事件的保险。这些经营者大多是养家糊口的人，他们希望家人得到很好的照顾，以防情况变得更糟。他们也热衷于小型自行车维修……我们为他们解决了这个问题，”Macharia 说，同时补充说，花时间做研究并与潜在客户交谈对于使产品适应市场非常重要。

“我们生产产品的方式非常独特。我们首先去客户那里了解他们的模式。我们和全国不同地区的人们坐在一起，问他们想要什么样的保险。然后，我们列出了他们想要的类似封面的清单，并记录下来，让他们提出最相关、最方便的付款方式，”马查里亚说。

马查里亚对该行业的兴趣始于 2018 年，当时他推出了一款针对机动车的按次付费小额保险产品——这是他在经营一家汽车修理店时观察到的数据和模式提供的信息。他想知道为什么他的大多数只在周末使用汽车的客户被迫签署年度保单。

然而，这个想法没有成功，因为他瞄准的当地合作伙伴没有提供帮助。他不是一个轻易接受拒绝的人，他开始建立必要的技术，并按照他的设想执行这个想法，从大众开始。

马查里亚说，他们有近 10，000 名投保人，目前的容量高达 15，000 人，但随着创业公司的发展，以及他们与第三方合作将保险嵌入其他服务，他们将向更多人开放。他还着眼于其他市场。

“东非也存在同样的问题，随着我们扩大业务规模、扩大客户群和增强产品，我们正在研究如何在非洲其他地区提供这种服务。”
