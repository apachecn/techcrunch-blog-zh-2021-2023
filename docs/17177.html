混合工作是工作场所的未来吗？TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/sponsor/epos/is-hybrid-working-the-future-of-the-workplace/

新冠肺炎已经大大加快了向混合工作模式的转变，但这意味着什么？它将对后疫情时代的企业产生什么影响？

当一级防范禁闭开始时，我们中的许多人立即过渡到在家工作，而且远程工作似乎会一直持续下去。根据国际设施管理协会(IFMA)的一项新研究，81%的中小企业希望超过四分之一的员工大部分时间远程工作，即使疫情已经结束——在科技行业更是如此。

这一发现对我们的工作方式以及“在工作中”的真正含义有着巨大的影响。远程工作不仅仅是在家工作；这也可能意味着在客户的工作场所、你邻居的共同工作空间或者路边的咖啡店里工作。在大多数情况下，这种混合模式仍然意味着每周至少在办公室工作一次，但办公室将不再是工作的主要场所。相反，它将成为一个受欢迎的、供应充足的枢纽，为流动和分散的劳动力服务。

在这种情况下，有时办公室会从一个你必须去的地方变成你想去的地方。它可能以热桌子、创意和娱乐空间、数字展示和咖啡吧为特色。它将成为一个协作、面对面会议和团队合作的地方，而更多的日常功能发生在其他地方。

这不是白日梦。许多公司已经在评估他们的办公场所，IFMA 调查发现，56%的中小企业预计公司预算将从物理工作场所转向改善员工的数字体验。技术将是这一切的关键。“随处工作”的心态要求员工配备所需的数字工具和应用程序，无论他们身在何处，都能保持高效工作。但是，更多的远程工作意味着更多的电话、更多的视频会议，以及当你的技术或音频设备不合格时我们都会经历的日常压力。

混合动力设计

如果企业做得好，混合工作场所模式有显著的好处。组织可以在满足可持续发展目标的同时削减办公空间的成本。将弹性工作制作为标准有助于招聘和留住人才。越来越多的大量证据表明，员工重视离开办公室工作的选择，只要他们有工具高效地工作。

Covid lockdown 实际上是一个在家工作的大规模实验，许多企业和员工认为至少取得了部分成功。一个长期的混合工作模式将必须是一个更加顺畅和更加深思熟虑的体验，连接加上访问数字工具将是至关重要的。员工希望能够在办公室里尽可能自由地进行远程协作。IMFA 的研究结果强调，雇主现在正集中投资为远程员工提供数字工具、设备和培训，以确保工作场所的生产力(无论在哪里)不受过时技术的影响。

同样重要的是，必须考虑对员工的情绪和心理影响。如果员工不能自由、自然地交流，远程工作可能会让人感到孤立——通话质量差和连接性差的挑战本身也会让人精疲力竭。这种向虚拟世界的转变导致了员工日常微挫折的增加。我们没有设计我们的家来工作，背景噪音加上技术问题经常让我们错过重要信息或要求个人重复他们自己。音频解决方案领导者 EPOS 的研发副总裁 Jesper Kock 解释说:“如果你经历过充满中断和低效电话的一天，你会感到筋疲力尽。”“这是因为大脑浪费了不必要的能量和认知能力来专注于相关的和想要的声音”。

需要和平

虽然这些挑战是真实的，但它们远非不可克服。最新的视频会议和 IP 语音(VoIP)电话鼓励顺畅和自发的沟通。云中的软件即服务(SaaS)应用程序让同事们无论身在何处都能在项目上无缝协作。管理者需要确保团队在任何时候都是虚拟地联系在一起的，并且只要有机会就要进行物理上的联系。

但是，混合工作的一个挑战可能在开始时不太明显，那就是噪音干扰。对许多员工来说，嘈杂拥挤的家不是理想的工作环境。“随着全球大部分员工继续远程工作，背景噪音不仅对我们的工作效率，也对我们的健康构成了重大威胁，”Kock 解释道。“由于工人们基本上没有意识到它的影响，只有解决这些问题，我们才能开始释放新水平的福祉和生产力。”

强大的音频和视频通信对于支持混合工作至关重要，但越来越多的研究发现，即使是错误类型的低水平噪声也会妨碍有效的通信并增加压力。

根据 EPOS 的研究，95%的工人承认工作中的注意力和效率受到噪音的负面影响。混合工作可能会使情况变得更加严峻。同事们的背景喋喋不休可能是一种干扰，但吵闹的孩子、狂吠的狗或在咖啡店工作的嘈杂声也是如此。

当如此多的交流和协作在虚拟环境中进行时，噪音就成了一个更加紧迫的问题。EPOS 的研究计算出，由于语音通话的音质不佳，最终用户每周损失大约 29 分钟的工作时间，或者一年损失三个完整工作日。音频质量差的后果包括丢失关键作品、客户不满意和推销失败。经常在不利的听力环境下努力听音频的员工可能会感到疲惫，即所谓的“大脑疲劳”，并表现不佳。

如果企业投资明智，技术再次提供了答案。当远程员工出于专业目的使用基本通信工具时，音频质量差的挑战会更加严峻。另一方面，将专业降噪音频耳机与企业级通信和协作服务相结合的公司更有可能获得新工作方式的全部好处。混合模式为企业及其员工提供了巨大的优势，但前提是围绕沟通和协作的挑战得到了解决。
