AWS 与 Avalanche 合作，为企业、政府扩展区块链解决方案| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2023/01/11/aws-partners-with-avalanche-to-scale-blockchain-solutions-for-enterprises-governments/

两家公司独家告诉 TechCrunch，亚马逊网络服务(AWS)已经与艾娃实验室合作，该公司正在构建第一层区块链雪崩，以帮助企业、机构和政府扩大区块链的采用。

“展望未来，web3 和区块链是不可避免的，”AWS 副总裁兼全球初创公司负责人霍华德·赖特(Howard Wright)对 TechCrunch 表示。“没有人可以预测它将会发生的时间、日期或季度，它将成为主流，但我们以前已经看到过增长周期。这一次的速度似乎在加速，我们很高兴能成为其中的一员。”

该合作旨在使个人更容易启动和管理 Avalanche 上的节点，同时也旨在为开发人员提供更强的网络能力和灵活性。

AWS 将通过其市场支持 Avalanche 的基础设施和分散应用(dApp)生态系统，以及一键节点部署。合作还将包括艾娃实验室加入 AWS Activate ，这是一个帮助初创公司和早期企业家在其平台上起步的计划。

“对我们来说，这意味着很多事情，”艾娃实验室主席吴炯对 TechCrunch 说。“我们有 500 多个应用程序，我们希望给他们更好的体验，现在我们有了一个真正的合作伙伴关系，我们可以将其引入激活计划。最重要的是，我们的用户一直在寻求更好的体验。一键节点是一种不可思议的方式。”

许多区块链已经使用 AWS 来驱动他们的网络——根据其网站的统计，世界上大约 25%的以太坊工作负载运行在 AWS 上。Wright 说，该技术本身是“天生不可知的”，支持所有区块链协议，尽管这是 AWS 与区块链的第一次基础合作。

艾娃实验室计划将其子网部署作为托管服务添加到 AWS 市场，因此个人和机构都可以轻松地推出他们自己的定制子网。子网是 Avalanche 扩展解决方案的一部分，可将流量从主区块链分流出去，并允许项目使用其本地令牌 AVAX，同时创建自己的第 1 层或第 2 层区块链。

“这是一个非常非常大的事情的开始，”吴说，并补充说，子网将允许开发者“非常容易地在亚马逊上建立自己的区块链，一个完整的区块链。”

吴说，上个季度，Avalanche 开始开发五到六个实时子网。但是在 testnet 阶段，至少有 100 多个子网将在未来 6 到 12 个月内部署“我们期待着与今年将推出的数百个子网分享这种合作关系[……]因此，我对这将是什么感到兴奋，而不仅仅是它是什么。”

Wright 回应了这一观点:“当你将激活次数乘以雪崩次数乘以子网次数时，你会发现这是一个开创性的时刻。我认为区块链(技术)将会变得很普通，并被开发者应用于我们的市场。”

Wright 说，艾娃实验室还成为了 AWS 合作伙伴网络(APN)的成员，使该公司能够与 150 多个国家的 10 万多个合作伙伴一起在 AWS 上部署产品。“[APN]与艾娃实验室和雪崩是区块链和加密的喷气燃料，将使世界各个角落的访问民主化。”

Wright 分享道，Avalanche 远非第一家通过亚马逊网络发展起来的创业公司。“超过 200，000 家初创公司走进了我们的大门，所以我们知道卓越是什么样的。这包括网飞、优步和 Airbnb——他们重新定义了垂直市场，我们大胆地认为其他人(像他们一样)也在那里，包括艾娃。”

艾娃实验室的创始人兼首席执行官 Emin Gün Sirer 对 TechCrunch 表示:“我们仍处于企业和政府构建物联网的早期阶段。但这些举措的步伐将加快，因为 Avalanche 和 AWS 正在为他们的需求提供更完整、更可靠的解决方案，Sirer 补充道。

也就是说，这个网络的规模可以极大地扩展这家基于加密技术的公司及其基于区块链的开发者的影响力。“底层技术和能力是我们试图与约翰和他的团队一起挖掘的东西，”赖特说。“它又回到了易用性和可访问性。”

吴认为，AWS 和 Avalanche 旨在通过合作提供的“方便和访问”只会加速采用。

“许多开发者和新的企业家都是从 web2 到 web3 的跨界者，不再是 web3 的中坚力量，”吴说。“我认为，他们已经有了与亚马逊合作的丰富经验，拥有 Activate 和 Avalanche 只会让跨界变得更容易，这将是一个加速器和放大器。”

两家公司还通过 Avalanche Summit、Avalanche Creates 和 hackathons 为企业家和开发人员合作举办活动，帮助建筑商在区块链上进行建筑。

“我们渴望成为长期战略合作伙伴；对我们来说，这是一个与众不同的激励因素，”赖特说。“因此，子网的补充和我们的激活我们认为是完美的时间，完美的机会，我们谦卑地认为，我们会回顾过去几年，并认为这是区块链扩张的重要时刻。”

至于 AWS 的未来？Wright 说，它计划“更贴近开发者和合作伙伴”。“不仅仅是西雅图总部[……]我们正试图将它带给众所周知的某个地方车库里的七个孩子，我们将与艾娃这样的高价值合作伙伴更加密切和灵活。”

“我们想挑战一切可能的极限，”赖特说。
