由谷歌前广告主管创建的 Neeva 将其无广告搜索引擎带到了欧洲

原文：https://web.archive.org/web/https://techcrunch.com/2022/10/05/neeva-brings-its-ad-free-search-engine-to-europe/

Neeva 是一个无广告搜索引擎，由一位前工程师和高管于去年推出，他因领导谷歌广告技术业务的而闻名，Neeva 即将登陆欧洲。

总部位于加州山景城的公司于去年 6 月在美国国内市场正式推出了一个只订阅的搜索引擎，后来继续在组合中添加一个免费的“基本”层，并有一定的限制。在此期间的几个月里，Neeva 表示，它已经积累了超过 60 万名用户，其中绝大多数是免费计划用户，现在该公司正在寻求更远的市场以寻求增长。

虽然它尚未建立一个正式的欧洲枢纽，而是选择从 5000 英里以外的美国西海岸驾驶这艘船，但明天 Neevia 将在英国、法国和德国打开众所周知的灯光。欧洲可能是 Neeva 首次扩张计划的明显第一步，因为强大的现任谷歌目前面临着一系列反垄断投诉，立法者瞄准了从电子商务(谷歌购物)安卓到其在线广告市场的主导地位的一切。

搜索是许多这些产品的核心，是全球数十亿人的互联网门户——这也是为什么欧盟迫使谷歌在设置期间为 Android 用户提供默认搜索引擎的多种选择。虽然这在很大程度上是无效的，原因有很多与谷歌如何实施这一过程有关，但它确实将欧洲定位为任何羽翼未丰的搜索初创公司漫游的沃土。

TechCrunch 本周在伦敦采访了 Neeva 的联合创始人兼首席执行官斯里达尔·拉马斯瓦米，了解他成立三年的公司的现状，以及随着市场的进一步扩张，事情会向何处发展。

搜索任务

回溯一下，拉马斯瓦米在谷歌的工程部门工作了 15 年，成为高级副总裁，也是谷歌超过 1000 亿美元的广告和商业业务的驱动力。他于 2018 年离开，加入受人尊敬的风险投资公司 Greylock Partners，他今天仍在担任这一职务，尽管职位有所降低。

“这是一份兼职工作——我给他们提供投资建议；它非常轻，”拉马斯瓦米说。

并非巧合的是，Greylock 与红杉资本和其他受尊敬的投资者一起，在两轮独立的融资中投资了 Neeva。自成立以来，Neeva 已经筹集了超过 8000 万美元的资金，随着 Neeva 对其全球雄心的加倍努力，这一数字可能会上升——它的下一步将是扩展到其他英语国家，包括印度、加拿大和澳大利亚，这可能会在今年年底前发生。

拉马斯瓦米补充说:“初创公司创始人的主要工作是筹集资金，这样你就不会耗尽资金。”

Neeva 可以作为 Android 和 iOS 上的原生移动应用程序，同时也可以通过浏览器扩展在桌面上(Chrome，Edge，Firefox，Safari)使用，该浏览器扩展可以阻止跟踪器并提供 Neeva 自有品牌的有机搜索。这意味着您访问的新闻或信息网站将无法将您的数据发送给任何第三方，以用于个人资料或个性化广告。

尼娃防止网络跟踪器。图片来源:尼娃

无广告环境是 Neeva 的一大卖点。Neeva 不再需要滚动浏览六七个或更多的赞助结果来找到相关链接，而是从最顶端提供它认为最相关的结果。

Neeva:只有有机搜索结果。图片来源:Neeva

但是，虽然很少有人会错过付费的“无机”结果，但任何标榜用户隐私的搜索产品总会有其他的权衡。

事实上，隐私搜索引擎的一个主要症结是，“隐私”往往是以牺牲真正的效用为代价的，而这正是谷歌所带来的。谷歌搜索包括一系列增值功能，比如告诉你一家酒吧在特定时间可能有多忙，最近的酒吧与你当前的精确位置相关，以及你如何通过逐路段导航找到最近的酒吧。这就是我们谈论一个城市的每一个公共设施的最新开放时间、地址、电话号码等的全部内容。

与其他隐私搜索引擎一样，Neeva 通过与 Yelp 和 Tripadvisor 等第三方数据提供商的集成，在一定程度上解决了这一问题，而苹果地图有助于查看地点。但是信息并不总是完整的或最新的，而且没有分享你的确切位置，导航也不容易获得。

Neeva 搜索引擎。Tripadvisor。图片来源:尼娃

默认情况下，Neeva 使用一个人的 IP 地址来“推断”用户在哪里，尽管根据我的测试，这是几百英里以外。但用户可以选择切换到他们当前的确切位置，这使得 Neeva 能够提供更准确的结果。

图片来源:尼娃

其他卖点包括过滤掉特定信息源的能力。例如，如果你不想从亚马逊购买，你可以告诉 Neeva 在未来的结果中取消它的优先级。类似地，对于来自特定出版商的新闻，你可以告诉 Neeva，你希望从该渠道看到更少的故事，或者更多，如果你喜欢的话。

尼娃:降低新闻来源的优先级。图片来源:尼娃

这突出了 Neeva 商业模式的主要卖点之一。允许用户在搜索结果中降低特定公司的排名并不真正符合谷歌的利益，但在 Neeva 的案例中却是这样:它瞄准的是消费者的钱，而不是公司的钱。

或许存在这样一种风险，即这一切都可能会发挥备受指责的回音室效应，这种效应近年来一直困扰着大型科技公司——算法本质上是为人们提供来自相同来源的相同内容，而不是让他们接触到“新想法”或他们可能不会接触到的内容。如果有人讨厌一堆有特定政治倾向的新闻出版物，他们可以选择在搜索结果中最小化它们的存在。但 Ramaswamy 很快强调，质量是其搜索索引算法的基本要素。

“这有点微妙，如果你优先考虑一个网站，我们不会向你显示来自那个网站的垃圾——它仍然需要通过一个相关性阈值，”他说。“我们指向的搜索结果必须对您的查询有意义。大致意思是，在所有条件相同的情况下，“优先考虑这个网站”或所有条件相同的情况下，“让我们摆脱这个网站。”但是如果这个网站是唯一一个对你的查询有用的结果，即使你取消它的优先级，我们也会显示它，因为我们认为对你来说显示它比没有结果要好。"

还有一个棘手的问题是 Neeva 如何产生结果。与其他“替代”搜索引擎类似，Neeva 的一些网络搜索确实在一定程度上依赖于 Bing，尽管这是它在推动完全依赖自己的技术堆栈时寻求摆脱的东西——可能在明年的某个时候。

“所有的个人索引都是我们完全从零开始构建的技术，”Ramaswamy 说。“网络搜索目前分成两部分，我想大概是我们自己的技术和必应各占一半。我们正在运行一个非常大的索引和搜索系统，提供越来越多的结果。”

从隐私角度来看，Ramaswamy 指出，Neeva 不会向 Bing 发送任何个人数据。

“我们不发 IP 地址；我们不发送精确的位置，”他说。"对 Bing 来说，整个 Neeva 就像一个用户."

最重要的是，Neeva 还允许所有登录成员连接个人谷歌、Office 365 和 Dropbox 账户，从本质上把网络搜索变成一个完全个性化的搜索工具，涵盖电子邮件、文档、日历等。与此同时，高级用户可以连接 Slack、GitHub、Confluence、吉拉、Box、opinion 和 Figma 等其他网站。

尼娃:搜索个人账户。图片来源:尼娃

微妙的平衡动作

虽然许多人可能会发现通过 Neeva 搜索所有个人文件非常有用，但如果有人加入 Neeva 的动机首先是出于对竞争产品隐私的担忧，那么他们可能最好不要实施任何提供的集成。例如，当用户将 GitHub 连接到他们的 Neeva 帐户时， Neeva 请求读写权限，这是 GitHub 的 API 规定的下载数据的要求，尽管 Neeva 说它“不会以任何方式使用写权限”

正是因为这些原因，谷歌才避免在搜索结果中过于个性化。曾几何时，谷歌会使用从 Gmail 收集的数据在搜索结果中显示个人信息，如即将到来的航班或酒店预订，更不用说根据用户收件箱的内容提供个性化广告了。但最终，这造成的隐私谈判太多了，谷歌最终解除了这些产品给人一种它真的不在乎你的数据的印象——也安抚了担心谷歌可能会阅读他们的电子邮件的不安的企业客户。

这在很大程度上归结于隐私和实用性之间的微妙平衡，同时让用户对自己的偏好有所控制——毕竟，不是每个人都想从搜索引擎得到同样的东西。有些人可能会很高兴谷歌扫描他们的收件箱，这样他们就可以很容易地查询他们的航班预计何时起飞，而其他人可能会拒绝这个想法——但拒绝者可能会非常乐意拥有有针对性的广告支持的搜索结果，如果这意味着他们可以在没有经济负担的情况下访问谷歌的产品库。

对于 Neeva 来说，这是一个同样的困境:如何提供一个真正有用的产品，让人们不仅想用，而且(有一天)愿意付钱。以及如何在不采用涉及广告和企业干预的传统商业方法的情况下做到这一切。默认是游戏的名字——让产品简单，尽可能不像谷歌那样开箱即用，然后允许人们手动调整东西以适应他们自己的隐私标准。

简而言之，人们可以使用 Neeva 进行简单的网络搜索，而无需做广告或注册或做任何事情。如果他们愿意匿名，他们可以创建一个免费帐户来解锁其他功能，包括在第三方个人帐户中搜索和从搜索结果中取消来源的优先级。与此同时，那些以每月 4.95 美元注册付费计划的人，可以获得额外的专注于隐私的工具，包括 LastPass 和 Bitdefender 的高级密码管理和 VPN 服务。Neeva 计划在未来将更多这样的工具添加到选项列表中。

除了本周在欧洲推出之外，Neeva 还在其运营的所有市场进行了一项显著的全面变革。到目前为止，Neeva 已经默认存储了 90 天的搜索历史，如果用户不想要的话，就必须手动关闭这个功能。Neeva 之前默认开启该功能的原因是，搜索历史为最终用户提供了真正的实用工具——在主页上提供推荐，或者根据之前的搜索自动填充搜索词。但对于一个本质上定位于隐私凭证的产品来说，这可能会被解释为一个不合适的功能，这就是为什么前进搜索历史将被默认关闭。

“我们正在改变这一点，因为搜索是如此个性化；Ramaswamy 说:“我只是觉得默认设置是没有任何搜索历史会更好。“我们认为它作为一个明确的选择加入功能会更好。”

搜索并摧毁

然而，总的来说，任何习惯了谷歌搜索的即时性和巧妙性的人可能会很难过渡到另一个搜索引擎——Neeva 或其他——而不会注意到其固有的缺点。这就是为什么谷歌在它所做的事情上取得了如此大的成功——它创造了令人难以置信的粘性产品，吸引人们，使人们难以逃脱。

地图也许是谷歌成功的最好例子之一，它从一个产品中赚钱，投资到另一个产品中，有效地消除了任何竞争的迹象。

“我使用谷歌地图是出于你提到的原因——如果我想要一条火车路线，这是很难击败的，”Ramaswamy 说。“但另一方面，这也是一家公司，它把所有本可以在这些领域进行创新的人都挤出了市场——仅仅是因为他们的规模有多大。请记住，10 年前，提供地图曾经是一项独立的业务。有很多公司都这么做。他们都停业了。这是因为谷歌拿走了你从搜索广告中获得的钱，将其注入地图，并淘汰了整个类别。”

承认尼瓦面临的问题的规模是一回事，解决它是完全不同的另一回事。因此，归根结底就是一次解决一个问题，首先从那些认为广告很重要的人的搜索等式中剔除广告，并在组合中加入一些增值服务。

Ramaswamy 说:“创业公司需要解决这个问题的方法是从一个核心的公用事业领域开始，并确保他们向一部分人提供服务。”“创业公司就是这样成功的——如果你试图同时做所有的事情，这是有问题的。”

当然，房间里的大象之一是拉马斯瓦米从领导谷歌广告和商业部门到建立类似于两极对立的东西的旅程。

拉马斯瓦米说:“我只是觉得这是一种剥削模式，我看到谷歌、脸书和亚马逊等广告平台越来越多地从规模中获益。”。“所以我想回归基本，创造一个以用户为中心的产品。我觉得更好的产品是可能的，我们都已经放弃了——我认为自己首先是一名技术专家，其次才是广告人。”

隐私推送

快速浏览一下竞争格局，就会发现有许多可供选择的“隐私”搜索引擎，包括资金雄厚的现有公司，如 DuckDuckGo、You.com、T4 和 T5，这些公司最近推出了自有品牌的搜索引擎 T7，扩展到了浏览器之外。所有这些公司都有很多共同点；例如，他们将“隐私”作为核心卖点。但是他们也都在想办法建立一个可行的搜索业务，不一定要涉及无数的广告——如果一定要涉及广告，那就只基于侵入性较小的关键词搜索。

例如，Brave 正在尝试与 Neeva 类似的模式，因为它将很快提供基于订阅的付费服务，允许用户从搜索结果中删除广告。但这也告诉我们，那些免费计划的用户将不得不与广告竞争。在其他地方，DuckDuckGo 基于关键词的广告占据了其商业模式的很大一部分，而 You.com 仍然在很大程度上弄清楚其货币化哲学，但目前基于附属链接并且将来可能包括广告。

Neeva 强调，其商业模式绝对不是基于广告或附属链接——目前，这一切都是为了让人们注册一个溢价计划。它还在考虑其他的盈利渠道，包括与其他志同道合的公司捆绑合作，如加密电子邮件提供商 Proton，甚至是授权交易，以支持企业内部的应用搜索。

虽然 Neeva 已经可以在任何国家使用，但到目前为止，它只针对美国市场。随着这一最新推出，英国、法国和德国的消费者将能够访问该产品的完全本地化版本，其中包括当地语言翻译和为当地用户量身定制的搜索。

拉马斯瓦米说:“本地化是为了确保搜索质量之类的事情没有问题——从外面很难看到，但在幕后有很多工作。”“但你也可以从搜索引擎中看到一些看得见的东西，从天气到货币兑换器和本地翻译，无所不包。”

在推出时，Neeva 只在欧洲提供基本的无广告服务，未来几周将以每月 5 美元的当地货币推出高级服务。
