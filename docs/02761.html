活动营销是一项“团队运动”，Vendelux 正在指导公司寻找最佳投资回报率 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/01/28/event-marketing-vendelux/

当你参加活动、会议和贸易展览，并且在推动大量线索和机会方面表现良好时，这是一次很好的经历。然而，当事情进展不顺利时，就浪费时间和金钱而言，这是有风险的。

Shutterstock 的两位前高管 Alex Reynolds 和 Stefan Deeran 一起致力于将 API 和计算机视觉货币化，并参加了一些糟糕的活动，他们认为可以有更好的方法来决定哪些活动应该在列表的顶部，以及哪些公司团队成员应该去最大化投资回报。

雷诺兹告诉 TechCrunch:“我们认为事件营销是一项大规模的团队运动。“它涉及高管、销售团队、所有不同的营销团队、运营和采购，但没有平台或记录系统来帮助实现协作、问责和可见性。”

因此，他说公司很难回答基本的问题；例如，在每年举行的 30，000 多个贸易展会中，哪一个应该有该公司的存在，以及该公司过去从该活动中产生了多少收入。在 Shutterstock 的活动和营销团队的帮助下，Reynolds 和 Deeran 找到了在内部创建流程的方法，并看到渠道的投资回报率大幅上升。

就在那时，他们决定检验这种方法是否适用于其他企业。他们在 2021 年底开始建立 Vendelux ，已经有十几个客户，包括 ActiveCampaign、Gainsight 和高尔吉亚，访问其超过 30，000 个现场和虚拟活动的专有数据库，包括演讲者、赞助商和与会者级别的数据。

Vendelux 从公开来源获取数据，并能够显示哪些赞助商和发言人将出席某个活动。用户还可以从他们的客户关系管理系统(甚至 LinkedIn 联系人)上传信息，并能够看到公司未来的每次活动，或者他们网络中的某人是否在活动中发言。

该公司于本月公开上市，正经历着两位数的月环比收入增长。今天，它宣布了 240 万美元的种子资金，由 Tenacity Venture Capital 牵头，参与方包括 Earl Grey、Pareto Holdings 以及包括 Shafqat Islam 和 Avi Muchnik 在内的战略天使投资者。

Vendelux 的浏览选项卡。图片来源: Vendelux

新的资金将主要用于产品开发和团队扩展。在这轮融资结束之前，创始人们正在启动公司。然而，雷诺兹说，他们已经到了平衡客户来电和建筑特色的地步。

"这感觉像是在火上浇点油，建立团队的恰当时机，"他补充道。“在产品方面，我们希望增加很多东西，当然是从集成的角度来看。”

同时，Reynolds 和 Deeran 将公司的发展分为三个阶段:探索、执行和评估。目前，该公司正专注于事件智能领域的探索，现在随着种子期的到来，它将进入执行阶段。

一旦一家公司知道去哪里和派谁去，创始人下一步要做的就是确保提前预定足够多的会议，能够捕捉正在发生的数据，并实时查看销售代表的表现，并衡量投资回报率。

“我们认为活动营销人员是这些营销团队的无名英雄，他们正在制定这些庞大的计划，并将所有这些人聚集在正确的地方，但没有平台来帮助他们推动问责制和能见度，”雷诺兹补充道。“我们希望成为记录系统，并从本质上帮助企业围绕它们应该向何处去以及从价值角度来看这意味着什么，做出更多由数据驱动的决策。”
