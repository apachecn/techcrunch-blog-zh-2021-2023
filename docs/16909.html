人工智能初创公司 Fourthline 锁定 5400 万美元，为金融行业带来更好的身份检查和合规工具| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2023/04/03/ai-startup-fourthline-locks-down-54m-to-bring-better-id-checks-and-compliance-tools-to-the-finance-sector/

随着数字金融服务变得越来越先进，恶意黑客和欺诈者侵入这些有价值系统的努力也在增加，监管机构也在努力建立更好的结构来避免这种滥用。

为了帮助金融部门满足这些需求，阿姆斯特丹的一家名为 Fourthline 的初创公司建立了一套基于人工智能的解决方案，以帮助身份验证，帮助企业遵守反洗钱规则等。今天，它说它已经为€筹集了 5000 万英镑(5400 万美元)的资金来扩展这项技术。

全股权融资由芬奇资本(Finch Capital)牵头，还有其他未披露身份的投资者(有人告诉我，其中一个是战略机构合作伙伴)。Fourthline 自 2013 年以来一直存在——在 2018 年商业推出之前，它的技术建设了五年——它基本上是靠自己的力量增长的，迄今为止只有€筹集了 7000 万英镑，包括最新的€5000 万英镑的注入。该公司的估值尚未披露。

自推出以来的五年中，增长令人印象深刻:Fourthline 的客户包括 N26、Qonto、Trade Republic、FlatexDEGIRO、Scalable Capital、NN 和 Western Union，以及 Wish 等市场。在过去的五年里，业务每年增长 80%。总之，它说它的技术每年帮助审查“数百万”消费者。

当我们看到许多关于人工智能可能被误用和滥用的争议时，Fourthline 正在解决的问题在某种程度上完全适合人工智能的力量。

它的前提是基于几个关键事实:坏人可以通过许多途径来利用数字金融服务，无论是通过窃取数据，冒充他人还是采取其他方法来窃取金钱或非法转移资金。人类显然是解决这一问题的重要组成部分。但是，恶意方法的扩散和日益复杂，这些方法来自人类，但也来自机器，有时来自人工智能，这使得应对这些恶意方法和欺诈企图的挑战变得越来越困难。

因此，这种想法是，基于人工智能的方法使用计算机视觉，机器学习和巨大的数据处理来检测什么时候事情不应该是这样的，这不仅是有帮助的，而且是必须的。

Fourthline 的技术目前包括大约 200 项检查，涵盖的领域包括检查身份证件、解析生物数据、检查居住地记录、对照制裁名单验证姓名等。其中一些可能是简单的数据库检查，但其他的是非常复杂的移动目标。在 Fourthline 已经投资的基础上，这部分将通过这轮融资获得更多 R&D 资源。

Fourthline 的首席执行官兼联合创始人克里克·冈宁(Krik Gunning)在接受采访时说:“我们在认证方面投入了大量资金。他说，这包括“以不同的方式看待身份证和护照。”

该公司声称，这种方法能够识别 60%以上的欺诈，准确率为 99.98%。

已经有许多初创公司帮助 fintechs 和其他公司在 KYC(了解你的客户)监管和身份验证等领域合规，但 Gunning 指出，规范是采取一种非常不同的方法:公司在很大程度上建立了通常依赖于使用第三方构建的 API 和核心技术的解决方案，然后由有问题的初创公司进行定制。

Fourthline 采取了另一种观点:最好使用自己的专有数据集从头开始构建自己的技术，因为这更容易控制和修改，当然，从长远来看，这也给 it 部门带来了更好的服务利润。

“我们用自己的 OCR(光学字符识别)模型来识别数字区域，用另一个模型来识别视觉区域，以检查它是否被篡改过。是的，许多人也有这种情况，但我们也能够解决另一方面的问题，这要困难得多:确认某样东西是否真实，但也指出它可能是或可能不是真实的原因。”这涉及到更深入地研究鬼影，甚至能够理解创建图像时使用的照明。“我们为此投入了大量资金。”这是该公司花了数年时间才推出一款产品的一个关键原因。

该公司对如何走向市场有一个非常简洁的观点。目前，除了 KYC、反洗钱和身份验证之外，还没有建立其他服务的计划。因此，虽然信用评分可能看起来像一个非常接近和明显的机会，但它不是 Fourthline 目前将追求的机会。同样，这也是为什么它只专注于金融部门，而不应用其技术或考虑与许多其他部门的业务发展，这些部门也可能购买 ID 服务。试图创建通用身份认证方案的政府——结果非常零散——并不是消费者。至少现在不是。

“这个领域有很多公司在人工智能方面做了很多工作，但老实说，在你能让人工智能模型达到你想要的水平之前，需要大量的时间、投资、知识和培训，”他说。“你能做到的唯一方法就是专注。我们不可能在世界上每个地区的每个部门都这样做，所以这就是为什么我们在我们可以发挥作用的地方这样做。这就是我们关注欧洲金融机构的原因。”

在一个充斥着高度资本化的初创公司的市场中，这些公司发现很难达到自己的估值和增长预期，Fourthline 的精简重点、经证实的回报以及更重要的是，由自己的工程师开发的技术，是目前引起投资者共鸣的例子，也可能被视为未来增长的更健康基础。

芬奇资本(Finch Capital)的管理合伙人拉德布·瓦拉(Radboud Vlaar)用 Fourthline 自己的方法描述了它的经济性:“我们非常相信，通过利用专有技术的平台方法的专注增长战略，可以解决这个行业的合规性挑战。”
