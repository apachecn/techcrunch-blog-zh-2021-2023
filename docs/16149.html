亚马逊证实了又一轮裁员，影响到 AWS、Twitch 和其他部门的 9000 人

亚马逊宣布了又一轮大规模裁员，这家互联网巨头今天透露，又有 9000 人将失去工作。

在首席执行官 Andy Jassy 今天早上发布的一份备忘录中，该公司表示，裁员将影响其 AWS 云部门、Twitch 游戏部门、广告和 PXT(体验和技术解决方案)部门。

就在几个月前，亚马逊透露裁员 18000 人，其中大部分将来自其 PXT 和亚马逊商店业务。这一消息本身是对之前数字的延伸，据报道在 10，000 人范围内，尽管亚马逊从未证实 11 月份首次报道的任何数字。无论如何，今天的披露揭示了亚马逊对其裁员努力的交错方法 Jassy 在今天的备忘录中提到了这一点。

事实上，Jassy 指出，他们没有急于完成评估，这样他们就可以一下子宣布所有的裁员，而是选择慢慢来，一旦每个团队基本完成了他们的分析，就让人们知道。值得注意的是，亚马逊是世界上最大的雇主之一，在全球拥有约 150 万名员工，大约是谷歌母公司 Alphabet 的七倍，谷歌母公司在不到两个月前解雇了约 12，000 名员工。

此外，Jassy 说，他们还没有就确定哪些特定角色将会消失的问题得出裁员的详细结论。

“有些人可能会问，为什么我们不在几个月前宣布裁员的同时宣布裁员，”Jassy 写道。“简短的回答是，并不是所有的团队都在晚秋完成了他们的分析；我们选择分享我们做出的这些决定，而不是在没有适当努力的情况下匆忙完成这些评估，以便人们尽快获得信息。本说明也是如此，因为受影响的团队尚未最终决定哪些角色会受到影响。一旦做出这些决定(我们的目标是在 4 月中旬至下旬完成)，我们将与受影响的员工沟通(或在欧洲与员工代表机构沟通，如适用)。”

摇钱树

从表面上看，最大的惊喜之一是亚马逊的长期摇钱树 AWS 受到了裁员的打击。但是，尽管 AWS 仍然相对健康，但其最近的增长轨迹已经不像过去几年那样陡峭，这反映了云基础设施支出在更广泛的放缓。简而言之，由于经济衰退，公司正在寻求削减成本，这意味着在云计算等方面的支出减少——尽管 AWS 仍然是亚马逊的一个利润丰厚的实体。

亚马逊发言人表示，AWS 裁员是因为“重新确定优先事项的决定”，这要求它相应地“重新分配资源”。

“在大多数情况下，这涉及到人们转移项目、优先事项或团队，但在某些情况下，我们没有合适的技能来匹配这些优先事项，”该发言人说。“我们继续对我们的 AWS 业务感到非常兴奋和乐观，即使在充满挑战的经济环境中，该业务仍将继续拥有非常强劲的渠道、迁移和基本面。”*

微软的 Azure cloud 部门已经在 AWS 上取得进展，也在一月份宣布了的大规模裁员，据报道包括大约 150 名 Azure 员工，尽管这还没有得到独立证实。

虽然亚马逊之前没有证实 AWS 受到其在 1 月份宣布的一轮裁员的影响，但一些裁员似乎来自该部门，正如一些商店当时报道的那样。

因此，今天的新闻看到亚马逊官方证实，它正在缩减近来最赚钱的业务之一。这也是所谓的“五大”科技公司中第二家宣布两轮独立裁员的公司，此前脸书的母公司 Meta 上周宣布再裁员 1 万人。

* 文章更新了亚马逊 的附加评论。
