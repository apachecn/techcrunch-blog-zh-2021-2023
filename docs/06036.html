Truist 收购金融科技初创公司 Long Game

原文：https://web.archive.org/web/https://techcrunch.com/2022/05/03/banking-giant-truist-acquires-fintech-startup-long-game-in-effort-to-reach-younger-demographic/amp/

图像积分:真实/长期游戏

——美国最大的金融机构之一——已经收购了 Long Game ，一家 12 人的金融科技初创公司已经筹集了超过 2000 万美元的风险投资，高管们独家告诉 TechCrunch。

截至今年 1 月，总部位于北卡罗来纳州夏洛特的 Truist 是美国第六大银行拥有 4880 亿美元的资产。它于 2019 年由 BB & T 和 SunTrust banks 合并而成。

龙游戏成立于 2015 年，建立了一个游戏化的财务移动应用程序，旨在帮助人们“储蓄、学习和参与”他们的财务。具体来说，这家总部位于旧金山的初创公司希望利用与奖励挂钩的储蓄和休闲游戏来激励明智的金融行为，从而推动银行客户的参与。

两家公司都没有透露 Truist 为 Long Game 支付了多少钱，也没有透露公司的收入，只是说“逐年增长”

在联合创始人兼首席执行官林赛霍尔登的带领下，Long Game 已经从投资者那里筹集了超过 2000 万美元的资金，这些投资者包括残留风险投资公司、富兰克林邓普顿、茁壮成长资本和协作基金。TechCrunch 报道了这家初创公司 2017 年的【660 万美元融资 。

霍尔登告诉 TechCrunch:“超过一半的美国人储蓄不足 500 美元，所以很多人，如果发生意外，可能会对他们的财务生活非常不利。”“我们想以不同的方式解决这个问题。我们思考了很多关于行为经济学和人们如何被激励的问题，我们遇到的一件事是一种被称为与奖励挂钩的储蓄的行为经济学。”

她说，与奖金挂钩的储蓄是一种概念，人们将钱投入储蓄工具，以获得赢钱的机会。

霍尔登补充说:“这实际上是一种彩票动态，人们试图赢得大奖，并将其用于某些事情上。“你不是买了票就扔掉现金，而是把它存入储蓄账户。”

霍尔登说:“不出所料，Long Game 更多地面向新生代和千禧一代。”他指出，Zynga 的前员工都是普通员工。

“这是一个真实的游戏，因为我们真的在利用游戏动力学来推动金融知识。Long Game 利用游戏行业的最佳实践来吸引用户，”她说。"有一个琐事方面，客户从学习中获得奖励."

迄今为止，其主要营销渠道一直是社交媒体营销和游戏广告网络的使用。

对 Truist 来说，接触令人垂涎但难以接触到的年轻群体的机会很有吸引力。

Truist Ventures 的企业发展主管 Vanessa Vreeland 告诉 TechCrunch，虽然收购 Long Game 不是该行的第一笔金融科技收购，但这可能是最“重要”的一笔。

她说:“这部电影给人的感觉与众不同。“它没有资产负债表。这是我们第一次在没有资产负债表的情况下收购金融科技。” Truist/SunTrust 之前收购了在线贷款商 LightStream 和销售点软件提供商 Service Finance 。

作为此次最新收购的一部分，Long Game 的工程师、设计师和高管将加入 Truist 的创新团队。

“我们将自己视为一家巨型初创公司……当我们寻求扩大我们零售银行的产品时，我们不仅想方设法获得客户，还想方设法加深与他们的关系，”弗里兰说。“…我们想‘除了提供一种真正令人兴奋的游戏化储蓄和个人理财方式，还有什么更好的方式来真正吸引我们的客户并吸引年轻一代的新客户加入 Truist 呢？’…Truist 的应用程序将“与漫长的游戏体验融为一体。”

她补充说，该银行计划“重新推出”该应用程序的增强版，“让超过 1500 万个家庭可以使用它”。

如今，Truist 在旧金山确实设有办事处，为其财富客户以及企业和投资银行客户提供服务。

“但是我们还没有把重点放在西海岸的技术或创新团队建设上，”弗里兰告诉 TechCrunch。“我们很高兴能在我们的旧金山办事处安置足够数量的人员，我们将继续为 Lindsay 和她的团队提供技术、产品和设计人才，同时继续增强 Truist 的能力，以满足我们客户的技术需求。”

根据 Vreeland 的说法，Truist 通过一些不同的渠道开始意识到长期游戏。举个例子，龙游戏参加了 FIS 的 Fintech 加速器计划 。她也通过个人关系网认识了这家公司。

霍尔登说，她被 Truist 对客户的“强烈关注”所吸引，她认为这与 Long Game 的类似。

她补充道:“我们只关注真实财务进展的客户体验——把一些感觉困难、有时令人情绪激动的事情，变成激励人心、有趣甚至神奇的事情。”“能够成为一个团队的一员，让我们拥抱并投资于创造最佳消费者金融体验的梦想，实在是太棒了。”

图片鸣谢:龙游戏联合创始人兼 CEO 林赛·霍尔登/龙游戏

与此同时，银行和金融科技初创公司之间的交集越来越常见，两者要么合作，要么竞争，或者像在这种情况下，一方被另一方收购。许多银行已经意识到，收购一家成熟的金融科技公司，比试图自己开发技术更有意义——在财务和物流方面都是如此。

我们是一个非常注重面子的组织。我们花了相当多的时间与客户会面，所以我们知道客户希望继续与我们面对面地会面，但也希望与我们进行数字化合作，”弗里兰说。“我们希望通过我们追求的所有数字渠道保持这种接触平衡，因此我们对 Long Game 能够帮助我们做到这一点感到非常兴奋。”

作为数字关注度上升的证据，Truist 今年早些时候关闭了数十家实体分支机构。

Vreeland 说，她还被 Long Game 由一位女性首席执行官领导的事实所吸引。

“我们真的很高兴见到一位女性首席执行官——这对我们来说真的很重要，因为我们考虑在 Truist 内部建立一个产品和服务组合，这些产品和服务真的像我们服务的客户和社区一样多样化，”她补充道。
