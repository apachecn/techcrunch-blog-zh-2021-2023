A16z 支持 Rutter 的“商业格子”目标

原文：https://web.archive.org/web/https://techcrunch.com/2022/03/15/a16z-backs-rutters-goal-of-being-plaid-for-commerce/

在公开推出其公司七个月后， Rutter 带着新的资本注入回来了，这次是由 Andreessen Horowitz 领导的 A 轮融资中的 2700 万美元。

我们第一次见到创始人郁鑫和彼得·周是在去年，当时公司筹集了 150 万美元。他们开始构建一个通用的电子商务 API 来读取和写入数据，以便技术客户可以与商业平台集成，并访问财务数据点来提供贷款，在新的市场推出，并简化品牌管理的库存同步。

如今，Rutter 拥有类似的会计系统和支付处理器工具，包括 Shopify、亚马逊、WooCommerce、BigCommerce、PrestaShop、Quickbooks、Xero、Stripe 和 PayPal。

为了实现“商业格子”的目标，周说，Rutter 的核心重点是结束公司插入所有这些渠道的所有碎片化现象。

“令人惊讶的是，这比人们想象的更加分散，”他告诉 TechCrunch。“我们提供各种平台，并充当粘合剂，将所有商业工具、平台和产品连接到所有商家和小企业。”

参与新一轮融资的还有新老投资者，包括 Comma Capital、恶作剧资本、Basis Set Ventures、Haystack、the Chainsmokers 以及一个包括 Plaid、Modern Treasury、RedHat 和 Airwallex 等公司创始人和高管的团体。作为投资的一部分，a16z 的克里斯蒂娜·沈(Kristina Shen)将加入 Rutter 的董事会。

鲁特的授权流程。图片来源:鲁特

该公司正在与客户合作，包括金融科技方面的 Ramp、Airwallex、Uncapped 和 Lenkie，以及电子商务支持公司，包括 CJ Dropshipping、意式冰淇淋、Shirtsy 和 Moonshot 品牌。Rutter 目前每月推出大约五个新的集成，并计划在年底前支持超过 100 个集成。

周计划利用新的资金在年底前将鲁特的团队从 19 人扩大到 80 人。他还专注于产品开发，包括为商业建立数据基础设施层，并在金融科技方面建立更多功能。

“我们看到如此多的需求和产品从我们手中流失，所以我们需要与新人一起扩大规模，”周补充道。

自 2021 年第二季度以来，该公司的收入增长了 10 倍，在去年 8 月的融资后增长了 5 倍。

周说，电子商务实现的这一领域“现在是一个开放的领域”，有大量的需求，其中大部分尚未得到满足。他补充说，其最接近的竞争对手专注于金融科技，而不是电子商务，但鲁特的长期目标不是成为一家全栈金融科技公司，而是一个商业数据层。

与此同时，俞表示，该公司正在增加更多的平台覆盖面，并将深入集成和用例。到目前为止，金融科技是主要的用例之一，但该公司在商业方面也看到了其他用例，包括航运、直运和市场。

为了支持该公司在那里看到的情况，正在开发的新产品和新功能将快速实时地获取数据，包括一个基于 webhook 的系统，以通知客户订单。
