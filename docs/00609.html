在用户的反对声中，不和谐导致探索加密和非功能性加密技术的暂停

原文：https://web.archive.org/web/https://techcrunch.com/2021/11/10/discord-nfts-crypto-jason-citron/amp/

图片来源:不和谐

Discord 创始人兼首席执行官杰森·香橼(Jason Citron)周三试图让用户放心，该公司没有即将推出的将业务转向非功能性桌面系统的计划。

在本周早些时候的一条推文中，Citron 分享了一张集成到 Discord 用户界面中的 crypto wallet MetaMask 的图像，并配有文本“可能什么都没有”——这是 NFT 空间中对即将发生的大事的速记语言。他在周三晚上联系了之前的推文，指出 Discord“目前没有计划”将加密钱包集成到其应用程序中。

Discord 还提供了以下声明，向 TechCrunch 澄清了其计划:

我们感谢我们听到的所有对内部概念的回应，你可能已经在本周早些时候的一条推文中看到了，我们想澄清一下，我们目前没有计划推出它。我们对 web3 技术的潜力和这些社区在不和谐中走到一起的积极方式感到兴奋，特别是那些围绕环保、以创作者为中心的项目组织的社区。然而，我们也认识到有些问题需要解决。目前，我们的重点是保护用户免受垃圾邮件、诈骗和欺诈。

Citron 周一在回复中随意分享了这张截图，不是作为官方的公司公告，但热情的 Discord 用户立即在这条推文中跳了出来。该公司很快解释说，截图是社区黑客周项目的一部分，不是社交聊天应用即将推出的东西，但这匹马已经离开了马厩。

对加密空间心存警惕的 Discord 用户迅速鼓励彼此放弃他们的 Nitro 订阅，Discord 的付费高级服务有助于该平台保持无广告。随着反弹的蔓延，愤怒的不和谐粉丝也提到了公司最近的一项民意调查，询问他们对 Web3 和 NFTs 的看法。

虽然截图只是加密钱包集成的预发布模型，但 Discord 确实在积极探索区块链技术如何补充其现有任务。现在，随着 Discord 重新评估如何最好地保持其价值观与在该平台上建立了繁忙家园的社区保持一致，一些工作将暂停。

一些 Discord 用户记录了他们对 Citron 关于加密货币采矿的环境影响的推文的强烈反应。其他人更广泛地反对“NFT 兄弟”，但即使在 Discord 的许多 NFT 社区，用户也表示担心集成加密钱包只会加剧平台上猖獗的加密诈骗。

“立即取消你的 nitro，选择其他——引用首席执行官关于它的推文作为你的推理理由，”一条有 2 万个赞的推文说。“在这一点上，唯一能改变这种情况的是收入立即大幅公开下降。”

Citron 的澄清表明，Discord 正在听取其社区的意见，尽管许多 NFT 项目确实将该平台的服务器称为家，但 Discord 用户群中直言不讳的一部分显然不希望该公司涉足加密业务。

虽然没有一家公司喜欢反弹，但对 Discord 新生的加密兴趣的负面反应确实表明，该公司的用户有多喜欢其当前版本的产品。所有的社交媒体用户都害怕改变——只要看看 Instagram 用户界面的任何微小调整——但 Discord 的用户似乎特别热衷于保护他们拥有的产品。
