关于比特币基地 NFT 推的 3 个问题 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/04/24/3-questions-about-coinbases-nft-push/

终于，比特币基地的 NFT 推本周上市。

这家美国加密货币交易巨头进入不可替代令牌市场已经过了很长时间，并且在上市公司的一个重要时刻着陆。在将于 5 月初公布收益之前，比特币基地的股价今天接近历史低点。

比特币基地已经证明，通过帮助个人和机构购买、出售和持有 crypto 股份，它可以建立一个大型盈利企业。但随着其收入——以及由此而来的净收入——随着加密市场本身的宏观健康状况而波动，该公司的估值状况更像是一家银行，而非一家科技公司。

事实上，如果我们观察该公司第四季度的净收入为 24.9 亿美元，并将该数字的年化版本与比特币基地目前 363 亿美元的市场估值进行比较，我们可以看到该公司的市盈率不到 4 倍。

因此，NFT 在比特币基地的努力是其进入一个已经存在的市场的一次尝试，这个市场是有价值且受欢迎的 NFT 开放海洋平台——并且是一个可以挖掘的新的增长脉络。这种增长可能有助于比特币基地将其估值及由此产生的倍数提高到更接近其私有化时的水平。

遗憾的是，如今有关比特币基地 NFT 推动的数据微不足道。这并不是一个巨大的惊喜，真的，因为来自前独角兽的新服务是全新的。因此，与其试图解析来自第三方分析服务的第一批零零星星的数字，不如让我们问几个问题。今天，我们概述了三个关于新比特币基地·NFT 项目的顶级难题。

从根本上说，我们想知道比特币基地的 NFT 市场在短期内能以多快的速度发展，其潜在的经济状况，以及最终的长期增长前景。我们会按顺序处理。让我们找点乐子！

比特币基地多快可以测量 NFT 体积？

在接下来的几周里，我们很想知道总交易量有多少，以及由比特币基地 NFT 市场推动的这些交易的价值。

该公司有一个庞大的候补名单，并有一个简单的大规模用户群穿梭到其新产品。候补名单如何转化为活跃用户，进而转化为活跃用户，这是一个关键问题。在候选名单被仔细研究后，比特币基地流向 NFT 产品的份额将帮助我们确定公司近期的销量增长。

到目前为止，数据是有限的，坦率地说，是不完整的。所以我们今天不知道太多，但比特币基地在 NFT 比赛中的开局实力或许会让我们感受到它在未来几个季度可能会以多快的速度扩大规模，如果它想夺回过去几个月失去的一些价值，这是一个关键时期。然而，自然地，交易量只是交易收入等式的一部分。另一个是费用。

比特币基地的 NFT 推动的经济学看起来有多强？

从长远来看？很好。近期内？奇怪！

以下是本周早些时候我们对比特币基地·NFT 发布会报道的摘录:

“在有限的时间内”，其市场上的非金融交易也不会收取交易费。随着时间的推移，费用会增加，但将是一个“低个位数的费用，”比特币基地产品副总裁 Sanchan Saxena 在会议上说。用户可以使用比特币基地钱包或他们拥有的任何自保管钱包在平台上交易非金融交易。

通过不收取启动费，比特币基地展示了其极其富有和有利可图的事实。它可以放弃费用来鼓励销量。一旦其市场达到健康的供需水平，它就可以开始获取收入。
