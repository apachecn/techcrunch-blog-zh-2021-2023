Vantage 筹集 2100 万美元 A 轮融资，帮助降低云成本

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/08/vantage-raises-21m-series-a-to-help-bring-down-cloud-costs/

Vantage 是一家帮助企业更好地了解他们的云基础设施支出(并自动化他们的储蓄)的初创公司，今天宣布它已经筹集了 2100 万美元的首轮融资，由 Scale Venture Partners 牵头，参与投资的有 Andreessen Horowitz，它领导了种子轮，以及 Harpoon Ventures，CloudFlare 首席执行官兼联合创始人 Matthew Prince 和 Glenn Solomon(他是 GGV 的管理合伙人，但也在这里进行个人投资)。该公司此前在 2021 年筹集了 400 万美元的种子资金。

正如 Vantage 联合创始人兼首席执行官(前 TechCrunch 网站开发者)Ben Schaechter 告诉我的那样，该公司现在帮助其 300 多家客户管理每年超过 10 亿美元的云成本。这些客户现在包括 Square、NASA、BuzzFeed、PBS、Ripple、PlanetScale 和 Barstool Sports 等，他们在这一点上将总共超过 10，000 个基础设施帐户连接到 Vantage。

图片来源: Vantage

当 Schaechter 和他的联合创始人 Brooke McKim 开始时，他们的想法是建立一个更好的 AWS 仪表板。作为其中的一部分，该团队建立了一个仪表板，帮助其用户可视化他们的 AWS 支出，这导致该团队几乎完全专注于这个想法。从那以后，Vantage 增加了对 Azure 和 GCP 的支持，但也许更重要的是，它还增加了对 Datadog、Fastly、Databricks、Snowflake、New Relic、MongoDB 和 Kubernetes 集群等服务的支持，并在路线图上支持 Twilio、PlanetScale 以及 IBM 和 Oracle 的云。

正如 Schaechter 告诉我的，Vantage 也从当前金融市场的低迷中获利。不仅 FinOps 已经成为一种东西，而且公司现在也在积极寻找降低基础设施成本的方法。“环境的变化是，我们经历了长达 10 年的牛市，每个人都专注于增长。这显然已经停止或放缓——结果，每个人都转向了底线，”他说。“现在的问题是:你能优化什么？你可以通过[裁员]来优化人力成本。或者，通常情况下，第一行项目是云基础设施成本，如 AWS Azure 或 GCP 和雪花。因此，成本效率有了显著提高，Vantage 已经做好准备，从这个角度帮助客户。”

图片来源: Vantage

Schaechter 指出，由于该平台现在明显更加成熟，并且覆盖了更广泛的服务，Vantage 正在看到大型企业用户的涌入。他还认为，由于范围更广，与一些竞争对手相比，Vantage 能够更好地为这些客户服务。

有了坚实的基础，该团队现在正在考虑如何帮助客户更好地优化这些不同服务的成本。“Vantage 将是唯一一家拥有所有这些提供商的技术关键的提供商，这是我们可以开始提供和促进跨提供商成本建议和过渡的地方，”他解释说。“比如说。PlanetScale 是我们希望 Vantage 上的供应商之一。AWS 总是会告诉你在 AWS 上优化的最佳方式，但 Vantage 可以说:‘嘿，这是 RDS 上的特定数据库。你为此多付了钱。这是 MongoDB 的成本和行星尺度的成本。"

因为该团队已经可以访问这些资源，所以它还可以帮助用户在这些服务之间转移他们的工作负载。它已经提供了自动驾驶功能来帮助用户管理他们的财务承诺(作为回报，它向客户收取其节省成本的 5%)。计划是用这些额外的能力来扩展。

Schaechter 指出，虽然该公司目前的现金流实际上是积极的，但该团队认为，随着新客户的涌入，它需要加快步伐，以便更好地服务现有客户，并利用面前的机会。该公司目前约有 15 名员工，Schaechter 预计这一数字到今年年底将增长到约 40 至 50 名员工。
