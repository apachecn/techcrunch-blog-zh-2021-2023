Waymo Via 将为优步货运公司驾驶数十亿英里的无人驾驶汽车

原文：https://web.archive.org/web/https://techcrunch.com/2022/06/07/waymo-via-and-uber-freight-partner-up-for-the-long-haul/

Alphabet 自动驾驶子公司的交付部门 Waymo Via 正与优步物流子公司优步货运(Bernstein Freight)建立长期战略合作伙伴关系。作为协议的一部分，Waymo 承诺为优步货运网络保留数十亿英里的“纯货物”无人驾驶能力。

在过去的一年里，Waymo Via 已经与主要物流公司建立了其他长期联盟，如 C.H. Robinson 和 T2 j . b . Hunt。由于 Waymo 的目标是商业化，这种合作关系为该公司提供了一条通往优步货运公司 13 万多家承运商的途径。

多年伙伴关系将分阶段展开。第一阶段，Waymo 表示应该在年内进行，包括将 Waymo 的测试套件放到优步货运的网络上，这些测试套件是由 Waymo 司机改装的 Peterbilt 卡车。在这个试点场景中，Waymo 将充当承运人，在优步市场上为托运人提供其自动驾驶卡车作为资产。

Aurora Innovation 是 Waymo Via 的竞争对手，优步在向 Aurora 出售其自动驾驶部门优步·ATG后持有其股份，目前正在得克萨斯州与优步货运进行类似的试点。

最终，Waymo 正在寻求一种司机即服务(DaaS)的方法，即运营商将购买由 Waymo 司机运营的戴姆勒卡车。在 Waymo 与优步合作的第二阶段，运营配备 Waymo 司机的卡车的运营商将能够无缝选择进入优步货运网络，两家公司表示，这将有助于运营商扩大业务规模。

“这是我们产品之间的深度整合，我们将制定一个联合产品路线图，以建立专门用于在优步货运网络上成功部署自动驾驶卡车的工具和基础设施，例如，这种整合允许承运人在优步货运网络上与 Waymo 司机支持的卡车无缝货运匹配和执行负载，”Waymo Via 发言人在与 TechCrunch 分享的一份声明中表示。

这位发言人还表示，优步货运和 Waymo Via 在混合货运模式上保持一致，这种模式仍然依赖于人驾驶的卡车，但利用一种自主解决方案来帮助减轻面临卡车司机短缺的行业的压力。

“优步货运广泛、高效和可靠的数字网络对于实现自动驾驶卡车至关重要，”优步货运负责人 Lior Ron 在一份声明中表示。“我们有独特的优势成为自动驾驶卡车的首选网络，拥有规模和市场专业知识，可以部署自动驾驶卡车，使整个行业受益。”

Waymo 没有透露在最初的试点期间将使用多少辆测试车辆，但根据 Waymo 发言人的说法，在近期和长期内，Waymo 司机驾驶的卡车将开始在德克萨斯州达拉斯和沃思堡之间的 95 号州际公路上行驶，然后扩大规模。Waymo 没有提供时间表，说明它打算何时开始在 DaaS 的做法，或在得克萨斯州的初始自动车道以外扩大规模。
