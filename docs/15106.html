谷歌正在失去控制

原文：https://web.archive.org/web/https://techcrunch.com/2023/02/10/google-is-losing-control/amp/

图片来源: TechCrunch

谷歌正在挣扎。在多年来对虚假神虚拟助手的一心一意崇拜之后，该公司正在匆忙实施其人工智能战略，而其竞争对手则联手举起他们的干草叉。具有讽刺意味的是，这一切都是因为谷歌认为它垄断了干草叉市场。

看，2017 年，谷歌研究人员发表了论文“注意力是你所需要的一切”，引入了变压器的概念，极大地提高了机器学习模型的潜在能力。你不需要知道它的技术方面(事实上我不是教你的人)，但它已经有了巨大的影响力和授权；只要说它是 GPT 的 T 就够了。

你很可能会问，为什么谷歌免费赠送这个美好的东西？虽然大型私人研究机构在过去因隐瞒其工作而受到批评，但过去几年的趋势是向出版方向发展。这是一场声望游戏，也是对研究人员本身的让步，他们不希望自己的雇主隐瞒自己的观点。这可能也有傲慢的成分:发明了这项技术，谷歌怎么可能没有最好地利用它呢？

我们今天在 ChatGPT 和其他大型语言模型中看到的功能并没有立即出现。理解和利用一种新工具需要时间，每个主要的技术公司都开始研究人工智能的新时代可能提供什么，以及这样做需要什么。

协助助理

毫无疑问，谷歌像其他人一样致力于人工智能工作。在接下来的几年里，它在设计人工智能计算硬件方面取得了重大进展，为开发人员测试和开发机器学习模型建立了有用的平台，并发表了大量论文，内容涉及从深奥的模型调整到语音合成等更容易识别的事情。

但是有一个问题。我从谷歌员工和业内其他人那里听说过这一点，但该公司的工作方式有一种封建色彩:让你的项目得到现有主要产品的支持，如 Maps 或 Assistant，是获得资金和员工的可靠方式。因此，看起来尽管聚集了世界上许多最优秀的人工智能研究人员，他们的才华却被引入了企业战略的常规。

我们能看看结果如何吗？这里有一个小小的时间表(当然是有选择性的):

2018 年他们展示了对谷歌助手流程、照片(比如给单色图像着色)、带有“视觉优先版助手”的智能显示屏(你见过吗？)，地图助手，人工智能辅助的谷歌新闻和(值得称赞的)MLKit。

2019 年，更名和更大的智能显示屏，AR 搜索结果，AR 地图，谷歌镜头更新，网络双工(还记得 Duplex 吗？)，一个压缩的谷歌助手，在本地做更多的事情，Waze 中的助手，驾驶模式中的助手，直播字幕和直播中继(语音识别)以及一个更好地理解有语言障碍的人的项目。

可以肯定的是，这些东西有些很棒！然而，大多数只是现有的东西，但有人工智能的推动。回想起来，很多人感到有点畏缩。你真的会看到像谷歌这样的大公司如何受制于趋势，也如何驱动趋势。

图片来源: TechCrunch

与此同时，在那年的二月，我们也有了标题:“open ai 构建了一个非常好的文本生成器，发布它被认为太危险了。”那是 GPT 2 号。不是 3，不是 3.5… 2。

2020 年，谷歌制造了一个由人工智能驱动的 Pinterest 克隆人，然后在 12 月份，因一篇指出该技术的局限性和危险性的论文，谷歌解雇了人工智能伦理学的领军人物之一 Timnit Gebru。

公平地说，2020 年对很多人来说都不是一个好年景——open ai 是一个明显的例外，其联合创始人 Sam Altman 不得不亲自为 GPT 3降温，因为它已经超出了合理的水平。

2021 年见证了谷歌自己的大型语言模型 LaMDA 的首次亮相，尽管演示并没有真正销售它。大概除了让助手少出错之外，他们还在寻找它存在的理由。

OpenAI 以展示 DALL-E 开始了新的一年，这是文本到图像模型的第一个版本，很快成为一个家喻户晓的名字。他们已经开始展示，通过像 CLIP 这样的系统，LLM 可以执行不仅仅是语言任务，而是作为一个通用的解释和生成引擎。(需要澄清的是，我不是指“人工通用智能”或 AGI，只是说这个过程不仅仅是一组预设的口头命令。)

2022 年，对助手的更多调整，更多智能显示器，更多地图中的 AR，以及对人工智能生成的个人资料图片的1 亿美元收购。OpenAI 在 4 月发布了 DALL-E 2 ，在 12 月发布了 ChatGPT 。

在某个时候，我怀疑是 2022 年初，谷歌高管睁开眼睛，他们看到的东西吓了他们一跳。我在想象《指环王》中的场景德内瑟最终看向魔多集结的军队。但是，这些疯狂的副总裁们没有失去理智，被一个巫师打倒，而是发出电子邮件，询问为什么一些 pert 初创公司要绕着人工智能领域的世界领导者转圈。尤其是在他们几乎发明了这样做的方法之后。

证据就是在 DALL-E 2 发布一个月后 Imagen 的发布，尽管就像谷歌公布的几乎所有其他有趣的人工智能研究一样，任何人都无法测试它，更不用说连接到 API 了。然后，在 Meta 于 9 月发布了制作视频后，谷歌在一周后发布了 Imagen 视频作为回应。 Riffusion 为生成音乐兴风作浪，一个月后又来了 MusicLM (不能用的)。

但肯定是 ChatGPT 导致谷歌领导层迅速从焦虑转变为全力以赴的失败。

所有参与者都应该清楚，这种对话式人工智能与谷歌十年来投资的助理产品截然不同，实际上在做其他人的伪人工智能(实际上只是一系列 API 的自然语言前端)假装在做的事情。这就是所谓的生存威胁。

运气还是远见？

图片致谢:微软/OpenAI

现在，已经够糟糕的了，其他人，一些不受收购影响的暴发户，引发了搜索引擎的下一阶段进化，而且他们是以高度公开的方式这样做的，抓住了从行业领导者到技术回避者的每个人的想象力。刀的真正转折竟然来自 微软 。

称必应为谷歌搜索的“竞争对手”也许太过慷慨——必应占全球搜索市场的 3%,相比之下谷歌占 92%,必应更像是一只富有的牛虻。微软似乎已经放弃了对必应提升其地位能力的任何幻想，转而寻求外界的帮助。不管他们对 OpenAI 的投资是超自然的远见还是幸运的意外发现，从某种程度上来说，他们显然是押对了一匹快马。

也许在某个烟雾弥漫的房间里，塞特亚·纳德拉和山姆·奥特曼密谋将谷歌排除在他们的新世界秩序之外，但在公开场合，谈话的形式是金钱，而且是大量的金钱。无论背景故事如何，微软已经获得了对这位创新新人的忠诚，并有机会将其技术应用到最有益的地方。

虽然我们已经看到了一些有趣的想法，关于生成性人工智能如何帮助生产力、编码甚至管理，但它们尚未得到证明，要么是出于版权考虑，要么是人工智能在回应中有点太“创造性”的倾向。但是如果有适当的护栏，它显然非常擅长综合信息来回答几乎任何问题，从简单的事实问题到复杂的哲学问题。

搜索结合了微软的创新需求以在大型语言模型的核心竞争力方面领先，这是一个很好的机会或良好的感觉，它刚刚与世界上最重要的创造者成为合作伙伴。将最新的 GPT 模式(有人称之为 GPT-4，但我怀疑 OpenAI 会为自己的第一方模式保留这个名字)与 Bing 和 Edge 整合的举动是一种被迫的孤注一掷，是它在搜索引擎世界的最后也是最好的选择。

谷歌显然很慌乱，在微软计划发布基于 OpenAI 的 Bing 的大型活动的前一天，试图通过一篇空洞的博客文章进行一场搅局活动。Bard，显然是谷歌位于 LaMDA 的 ChatGPT 竞争对手的名字，以现在典型的备用方式推出。承诺能力，没有硬性的日期或访问计划。

这一宣布的尝试似乎是如此匆忙，以至于它的内容在两天后谷歌的“搜索和人工智能”活动中几乎没有被提及，事实上，它也逃脱了如果你在宣传知识图的未来，你会想要做的那种事实检查。用来说明巴德的图像包含一个不小的错误，称詹姆斯·韦伯太空望远镜“拍摄了太阳系外一颗行星的第一张照片。”这是不真实的，事实上，这种自吹自擂的机器智能搞错了，谷歌没有人注意到或足够关心去检查，这似乎吓坏了投资者。

ChatGPT 当然有问题，事实上，在微软推出增强版 Bing 后，TechCrunch 立即能够让据称安全且合适的人工智能即兴创作一篇希特勒的文章，然后反刍上个月其早期版本写的疫苗消毒。但是这些都是已经建立的记录上的污点，这个记录包括了数十亿次的提示和对话，让它的用户非常满意。

谷歌仓促出手，如此明显地犯错，说明它甚至在有限的实验水平上也缺乏准备——更不用说像微软已经开始的全球推广了。

在的投资者电话会议上，首席执行官桑德尔·皮帅说“我认为这是一个重新思考、重新想象和推动搜索为我们的用户解决更多用例的机会。现在还为时尚早，但你会看到我们变得大胆，把东西拿出来，获得反馈，不断迭代，让事情变得更好。”听起来像是有计划的人吗？

可以理解的是，谷歌不想过早地将搜索与他们现有的半生不熟的通用 LLM 合并，从而杀鸡取卵。他们已经成为部署高度专业化人工智能的专家，任务模型可以做一两件事。但当涉及到大动作时，他们舒适的位置让他们背负了惯性。

是谷歌的没落吗？当然不会，在不久的将来，它仍将是一家违约、利润惊人、有点可笑的公司。但投资者的信心已经动摇，因为事实证明，谷歌在过去几年中未能进行有意义的创新，可能不是出于智慧和信心，而是出于沉默和骄傲。(联邦贸易委员会和司法部再次打击其广告业务也于事无补。)

然而，这种转变才刚刚开始，当这种技术还没有证明自己像每个人希望的那样有价值的时候，我们不能推测得太远。否则，整个科技行业都将面临后果，不仅仅是谷歌。
