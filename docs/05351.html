Twitter 正在从网上删除嵌入的已删除推文

原文：https://web.archive.org/web/https://techcrunch.com/2022/04/06/twitter-deleted-embedded-tweet/

编辑推文按钮并不是 Twitter 唯一可以改写历史的新功能。该公司显然改变了处理事后被删除的嵌入式推文的方式，在互联网上到处都是漏洞。

以前，网页中嵌入的已删除推文仍会显示推文的文本内容。现在文本不见了，只显示一个空白框。Twitter 正在通过用 JavaScript 隐藏文本来修改带有已删除嵌入推文的网页——这一选择让许多开发者和开放网络倡导者大为光火。

在一篇的博客文章中，IndieWeb 开发者和前谷歌开发者倡导者凯文·马克斯猛烈抨击 Twitter 的改变，将其比作“篡改公共记录”马克斯引用前总统特朗普后来删除的推文作为应该保留的公共利益内容的例子，并补充说推特对删除的嵌入式推文的新做法“令人不安”。

Twitter 高级产品经理埃莉诺·哈丁在的一条推特上回应了马克的担忧，她说 Twitter 正在寻求“更好地尊重人们选择删除他们的推文”的改变。哈丁说，被删除的推文嵌入内容将很快显示一条信息，而不仅仅是留下一个完全空白的盒子，这就是现在正在发生的事情。

马克斯并不是唯一一个担心旧推文的非历史处理方式会损害网络的人。

“Twitter 所做的不仅仅是保护选择删除推文的人的隐私:他们使用 JavaScript 在嵌入的内容中追溯性地隐藏引用的纯文本，即使它仍然存在于 HTML 中，”Andy Baio 说，他创建了虚拟活动平台 Skittish T1，并曾任 Kickstarter 的首席技术官。"对于保存历史记录来说，这是一个巨大的问题."

“…任何在网上写作的人都可以使用一条推文的截屏或只是引用文本，但有信心知道，如果嵌入的推文被删除，纯文本仍将保留。Twitter 通过改变这种行为打破了这一约定。

Twitter 周二宣布，它将很快通过其付费订阅服务 Twitter Blue测试可编辑的推文。在嵌入式推文的变化和那条有争议的消息之间，该公司似乎正在转向一种将用户意图放在第一位的理念，远远高于对历史内容存档的任何担忧。但该功能的批评者认为，让用户在事后自由改变他们的推文可能会恶化 Twitter 的一些最棘手的平台问题，包括骚扰和错误信息。

就像公众对曾经神话般的编辑推特按钮的接受一样，人们可能会对推特删除互联网集体记忆是一个好的举动还是朝着错误方向迈出的不祥一步产生分歧。“被遗忘的权利”运动的精神特质与研究人员、开发者和其他开放信息倡导者之间存在天然的紧张关系，前者寻求授权人们从互联网上删除某些与他们有关的内容，后者将网络视为一个活的文档——一个不断更新但永远不会彻底改变的文档。
