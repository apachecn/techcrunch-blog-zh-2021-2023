欧洲首届风投界女性峰会是迈向平等的漫长征程中的第一步

本周，一些风险投资界最杰出的女性齐聚巴黎，参加首届欧洲版风险投资界女性峰会。

该活动由投资者索菲·温伍德和露丝·福克斯·布雷德组织，汇集了顶尖的头脑来讨论创造一个更加包容和公平的风险投资环境。会议只对那些被认定为女性的人开放，小组讨论的主题包括有限合伙人在科技多样化中的作用、寻找下一个由女性创立的独角兽以及激励更多女性基金经理等。

Winwood 和 Blader 在厌倦了欧洲妇女获得的微不足道的资金和很少的机会后，受到启发创建了这个会议。2021 年，欧洲仅有 1.1%的风险资本分配给女性创办的公司。虽然这个数字比 2020 年的 2.4%有所下降，但与 2017 年、2018 年和 2019 年的数字持平，当时女性分别获得了所有风险资本的 1.5%、1.8%和 1.5%。

诸如此类的多元化会议在美国相当普遍，但在欧洲却远非如此。风险投资峰会中的女性:欧洲的视野之广令人惊叹:通过日落晚会、品酒聊天和社交时间，来自欧洲大陆的女性聚集在一起，透过水泥天花板。

这项活动将于今天以一个被称为“盟友饮酒”的欢乐时光结束，届时男性将最终被允许进入房间。这是对改变是一个团队努力的微妙认可，那些有权力的人对他们如何运用权力负责。
