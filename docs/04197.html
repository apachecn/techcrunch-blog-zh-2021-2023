欧盟禁止 RT，Sputnik 有望覆盖在线平台

原文：https://web.archive.org/web/https://techcrunch.com/2022/02/28/eu-rt-ban-extends-online/

TechCrunch 证实，欧盟对克里姆林宫支持的媒体机构今日俄罗斯(RT)和人造卫星(以及任何子公司)的禁令预计将涵盖在线平台和应用程序以及传统广播频道。

这一“史无前例”的制裁是在昨日宣布的，当时欧盟加大了对俄罗斯入侵乌克兰的回应力度。

欧盟内部市场专员蒂埃里·布雷顿(Thierry Breton)的发言人告诉我们，该禁令“预计将涵盖所有分发或传输手段，包括互联网视频共享平台和应用程序”。

他还证实，欧盟行政部门打算对 RT 禁令使用制裁性法律工具，而不是试图修改现有的视听媒体服务指令——这样它可以更快地行动。

一个独立的委员会消息来源暗示，禁令甚至可能在几天内实施。

在欧盟制裁两家克里姆林宫喉舌的同时，许多科技平台今天宣布了对俄罗斯官方支持的媒体的新限制。

Twitter 表示，将降低俄罗斯官方媒体在其平台上的曝光率，并标记包含这些媒体链接的推文，正如我们之前报道的那样——扩大了之前标记媒体本身的政策。

这家社交媒体平台表示，这些变化将立即部署，并表示将在未来几周内为其他“国有媒体账户”添加类似标签。

同样在今天，脸书的母公司 Meta 推出了新的措施。

政策主席尼克·克莱格说，这将“限制整个欧盟对 RT 和 Sputnik 的使用”。尽管目前还不清楚该公司是完全禁止俄罗斯国家媒体公司，还是只是对它们进行地理封锁。

我们已经向 Meta 询问了更多细节。

 

在过去的几个小时里，微软也宣布了应对俄罗斯虚假信息的新措施。

该公司在一篇博客文章中写道:“我们正在迅速采取新措施，减少俄罗斯国家宣传的曝光，并确保我们自己的平台不会无意中资助这些业务。

“根据欧盟最近的决定，微软 Start 平台(包括 MSN.com)将不会展示任何国家赞助的 rt 和 Sputnik 内容。我们正在从 Windows 应用商店中删除 RT 新闻应用，并进一步降低这些网站在 Bing 上的搜索结果排名，以便只有当用户明确打算导航到这些页面时，它才会返回 RT 和 Sputnik 链接。最后，我们将禁止 RT 和 Sputnik 在我们的广告网络上发布所有广告，并且不会在这些网站上发布来自我们广告网络的任何广告。"

NPR 也刚刚报道抖音将在整个欧盟范围内对 RT 和 Sputnik 进行地理封锁。

欧洲各国政府一整天都在敦促美国科技平台对隶属于克里姆林宫的媒体采取更强硬的行动，因为俄罗斯武装部队继续在乌克兰进行侵略——对乌克兰第二大城市哈尔科夫进行野蛮轰炸，摧毁了居民楼，导致数十名平民死亡。

今天早些时候，政治报道了三个波罗的海国家和波兰领导人的一封信，信中敦促平台采取更多措施来扼杀他们所谓的俄国“大规模造谣运动”。

法国数字部长今天早些时候也在推特上发布了关于与社交网络公司和搜索引擎的会议——“讨论打击俄罗斯网上宣传”并谈论欧盟对克里姆林宫支持的媒体的新制裁。

正如我们今天早些时候报道的那样，Breton 本人也亲自推动谷歌和 YouTube 的首席执行官在视频电话中对“俄罗斯战争宣传”采取更强硬的行动。

欧盟主席乌尔苏拉·冯·德·莱恩昨天宣布，欧盟将对克里姆林宫支持的媒体实施禁令，这是针对俄国入侵乌克兰的一系列新制裁的一部分。“国有的今日俄罗斯(Russia Today)和斯普特尼克(Sputnik)以及它们的子公司将不再能够散布谎言，为普京的战争辩护，并在我们的联盟中制造分裂，”她说，这是她宣称的“前所未有”的举动。

欧盟在这方面的行动速度可能看起来令人惊讶，但欧盟已经花了数年时间与主流技术平台建立沟通渠道，专门用于处理在线虚假信息——通过一项自愿的行为准则，自 2018 年以来，许多主要平台都签署了该准则。

虽然该准则不具有法律约束力，但该机制产生了一种行动预期，并为委员会建立了快速通道，以达到主流(美国)技术平台并被其听到。它以前曾利用这些渠道敦促采取更多与冠状病毒“信息流行病”有关的措施，因为它称之为针对 2020 年夏季新冠肺炎危机的虚假信息浪潮。

战争显然是另一个紧迫的原因。

一位欧盟发言人告诉 TechCrunch:“在技术层面上，我们也在与这些平台的代表进行持续的协调。”。“平台同意根据当前形势不断调整和更新平台政策。

“关于接下来的步骤，我们正在探索各种选项，以最好地与平台协调。紧张的工作和协调正在进行。”

全面禁止俄罗斯媒体喉舌的内容发布渠道(包括所有数字内容)是欧盟迈出的一大步。

欧盟的立法者通常对可能面临言论管制指控的措施极为谨慎。然而，俄罗斯对乌克兰的入侵改变了其宣传的背景——改变了对媒体的控制，使其不再仅仅是民主欧洲边缘的永恒话题，而是普京的“战争宣传”机器不可避免的一部分，因为他的武装部队攻击了一个主权国家。

冯德莱恩昨日的言论也值得注意——她表示，欧盟正在“开发工具，在欧洲禁止有毒和有害的虚假信息”——这暗示着未来还会有更多行动。

除了即将出台的对 RT 和 Sputnik 的禁令，委员会打算扩大其切断克里姆林宫“有毒媒体机器”的方法还有待观察，但进一步的在线内容限制不太可能逃脱争议。

“开发工具”的提法表明，欧盟可能希望依靠技术平台，主动追踪通过非官方渠道传播的俄罗斯宣传——也许是通过应用人工智能或其他过滤技术。

尽管欧盟法律中有法律禁令，禁止对数字平台实施一般监控。

预先过滤内容的紧急命令也将是一个非常生硬的工具——有可能删除真正批评性的言论——这可能会无意中助长克里姆林宫的宣传机器，因为它会让真相是战争的第一受害者这一概念变得可信，从而在西方社会内部制造分裂。
