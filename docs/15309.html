抖音向美国非营利学术机构开放其研究 API

原文：https://web.archive.org/web/https://techcrunch.com/2023/02/21/tiktok-expands-its-research-api-to-nonprofit-academic-institutions-in-the-u-s/?scid=MB-UoVtY1I

抖音将其研究 API 扩展到美国非营利学术机构

抖音公司周二宣布正在扩大其研究 API 对美国非营利学术机构的访问。此次扩张正值抖音于 11 月开始测试研究 API 的早期版本。感兴趣的大学和研究人员必须申请访问并得到抖音美国数据安全(USDS)部门的批准。

去年夏天，当抖音在 T4 首次宣布研发 API 时，该公司表示，研究人员目前没有一种简单的方法来评估内容或在其平台上进行测试，这就是为什么它认为有必要开发研发 API。

“抖音正在努力提高研究界的透明度，”这家公司指出。“作为我们对如何管理和推荐内容负责的努力的一部分，我们已经建立了一个 API，其中包括我们平台上内容和帐户的公共数据。该 API 将向全球研究人员开放，首先是美国的学术研究人员，然后扩展到其他地区，随着我们能力的增强，还将包括非政府组织的研究人员。”

通过 research API，研究人员可以访问公共帐户数据，如用户配置文件和评论，以及性能数据，如用户收到的评论、喜欢和收藏的数量。研究人员还可以访问公共内容数据，如评论、字幕和字幕，以及性能数据，如视频收到的评论、分享、喜欢和收藏的数量。此外，研究人员可以访问关键字搜索结果的公共数据。

抖音最初宣布研究 API 之际，该公司正试图证明，由于其母公司与中国的联系，它不会对国家安全构成威胁。从那以后，该公司继续面临审查，因为美国众议院命令其工作人员和立法者在去年 12 月因流行的视频分享应用程序的安全问题而从任何政府发放的移动设备中删除抖音。

至少有 20 个州禁止在政府所有的设备上使用抖音。此外，许多大学也禁止抖音进入他们的校园网络和设备，包括德克萨斯大学和德克萨斯 A & M 大学。

研究 API 的更新正值抖音首席执行官周寿子计划于 3 月 23 日在众议院能源和商业委员会面前露面。这场听证会将标志着抖音首席执行官首次出现在国会小组面前。
