Mozilla 敦促采取行动解除平台浏览器锁定

原文：https://web.archive.org/web/https://techcrunch.com/2022/09/22/mozilla-browser-research/

“移动浏览器作为基本工具的体验以及它们之间缺乏差异化的感觉意味着，预装在设备上的浏览器具有巨大的优势，”它在报告中写道。“这有利于操作系统，但不一定有利于消费者。许多人对转换到新的浏览器犹豫不决，因为他们很快就习惯了他们预装的浏览器，没有强烈的动机去寻找替代浏览器，或者可能被阻止去发现一个。消费者行为的这种长期制约意味着，放弃令人满意的预装浏览器是一种主动选择，需要付出一定的认知努力。如果人们很忙或者过程太混乱，人们会推迟做出改变或者决定不做所有的改变。对许多人来说，简单地维持现状或推迟做出决定更容易。”

该报告还提出了桌面和移动浏览器使用之间的有趣联系 Mozilla 称“几乎所有”使用 Firefox(替代)移动浏览器的用户也在他们的桌面电脑上使用 Firefox。

“我们的研究显示，在美国，不到 6%使用 Firefox 以外的桌面浏览器的人报告说他们在智能手机上使用 Firefox，”它指出。“这表明，越多的人在台式电脑上使用 Firefox 或其他替代浏览器，他们就越有可能在移动设备上使用该浏览器。”

这反过来暗示，微软向 Windows 用户大力推广自己的浏览软件——尤其是它在桌面操作系统中注入的反 Firefox 信息——导致 Firefox 在移动浏览器市场的份额下降(尽管微软目前没有移动平台)。

然而，很明显，对于独立浏览器制造商来说，有多种因素使得移动领域的竞争变得尤为艰难。该报告强调了移动领域的挑战，因为与桌面操作系统相比，移动领域的控制和/或集成(以及品牌捆绑)更加严格。

例如，谷歌利用与 OEM 合作伙伴的合同限制，最大限度地提高预装 Chrome 浏览器等自有品牌服务的安卓设备的比例，尽管安卓是开源的。(当然，这个科技巨头因为这些限制而陷入了反垄断的困境——比如在欧盟，它被迫提供一个选择屏幕来推广搜索引擎竞争对手)。

然而，消费者对大型科技产品的熟悉度(和舒适度)显然可以与锁定同步工作——尽管平台可能会通过暗示性信息(和/或通过为替代品制造摩擦)积极过度销售集成优势，从而很好地塑造这一结果。

Mozilla 指出:“我们的研究表明，许多消费者认为 Chrome 是在 Android 手机上运行最好的浏览器，同一家公司的产品在一起运行会更好(例如，Gmail 在 Chrome 上运行会更好)。”Mozilla 指出，谷歌将此类信息作为其“跨产品推广”的一部分就是一个例子。

“它还与网络兼容性问题以及操作系统提供商限制或允许第三方浏览器互操作性的程度密切相关，包括访问提供给他们自己的浏览器的相同功能和 API，”它继续说道，还批评了苹果禁止其应用商店的替代浏览器引擎，这限制了与 Safari 竞争的差异化，因为竞争对手也必须在 Webkit 上开发(这在历史上减缓了他们的竞争能力，并继续限制他们能够提供多少差异)。

“iOS 上替代浏览器的功能开发仍处于停滞状态，因为苹果控制着浏览器引擎和操作系统，没有向竞争对手提供一些必要的 API 和功能，从而限制了差异化。”

选择被破坏

Mozilla 的报告还强调了一些例子，即使消费者成功地选择了一个替代浏览器作为默认浏览器，平台仍可能回到自助式选择——绕过他们的选择，在某些情况下重新展示他们的浏览器，例如在 iOS 中选择文本后执行“查找”(它指出“历史上总是会在 Safari 中打开网络搜索结果，不管用户选择了哪个默认浏览器”)；或者在 Windows 搜索栏或图标中打开一个 web 链接，这将打开 Edge("同样不管默认浏览器设置如何；或者使用 Android 上的搜索工具——它“总是会在谷歌浏览器中打开搜索结果”。

“OCA 的演示突出了操作系统偏好自己的浏览器并削弱消费者选择的一些做法。一些国家的立法者和政策制定者已经开始采取行动打击欺诈行为以保护消费者。其他人已经开始解决数字市场缺乏有效竞争的问题，包括通过引入监管。然而，很少有人认识到这些问题与浏览器竞争的重要性之间的联系，或者研究 OCA 实践作为实现(或阻挠)消费者选择和福利的一种方式的作用，”Mozilla 认为。

“我们相信，如果人们有一个有意义的机会去尝试替代浏览器，他们会发现许多浏览器是他们操作系统中默认浏览器的引人注目的替代品。这些机会多年来一直被在线选择架构和商业实践所压制，这些架构和商业实践有利于平台，但不符合消费者、开发者或开放网络的最佳利益。很难低估多年来自我偏好和破坏消费者选择的影响，包括其对消费者行为的影响。也很难估计破坏性创新、替代产品和功能，以及由于这些做法而失去的独立竞争对手。”

Mozilla 的报告没有就监管干预提出具体建议，以迫使平台“为消费者和开发者做得更好”，正如它所说的那样——因为它表示计划在未来几个月公布关于补救措施的进一步工作——但它敦促立法者采取行动，以防止“持续不作为和竞争停滞对消费者的进一步伤害”。

“由于这些公司迄今未能做得更好，监管机构、政策制定者和立法者已经花费了相当多的时间和资源来调查数字市场。因此，他们应该能够很好地认识到浏览器竞争的重要性，并采取行动防止消费者因持续不作为和竞争停滞而受到进一步伤害。

“我们呼吁他们执行已经存在的法律和即将生效的法律和条例。在缺乏现有法律法规的地方，我们呼吁引入这些法律法规，并强调它们对互联网未来的重要性。许多司法管辖区的监管者、政策制定者和立法者可以抓住这个机会，在互联网的故事中开创一个新时代——一个消费者和开发者从真正的选择、竞争和创新中受益的时代。”

如上所述，欧盟已经就谷歌的 Android 合同限制采取了反垄断执法行动，这导致向欧盟用户提供了一个选择屏幕——至少是默认搜索引擎。然而，Mozilla 的报告通常对现有的以在线选择架构和软件设计为特色的补救措施不屑一顾，认为: “迄今为止部署的补救措施有很多局限性，并且基本上都失败了。”

其结论得到了谷歌在欧洲移动搜索市场份额缺乏有意义的变化的支持——谷歌在欧洲占有 96.6%的市场份额，自 2018 年以来仅下降了 0.3%，当时欧盟委员会对该公司罚款 50 亿美元，并命令其起诉侵权消费者，正如非营利谷歌替代方案 Ecosia，最近指出的。

谷歌的竞争对手 DuckDuckGo 也呼吁监管机构在监管选择屏幕补救措施方面走得更远——近年来认为，如果这些工具要真正对抗根深蒂固的平台权力，它们的设计和集成必须实现真正的“一键”和普遍可用的体验。

Mozilla 的美国政策主管詹尼弗·泰勒·霍奇斯在评论当前选择架构补救措施的局限性时告诉我们:“OCA 补救措施需要仔细考虑并透明地实施。至少应在与利益攸关方协商的基础上建立这些机制，对其进行广泛测试，并共享关于其绩效的数据。以前的选择屏幕补救措施都不能满足这些要求。

“该报告强调的是，已经有很多关于消费者行为的研究，更多的研究正在进行中。这些专家和学者，以及像 Mozilla 这样的市场参与者，应该对如何开发 OCA 补救措施有发言权。”
