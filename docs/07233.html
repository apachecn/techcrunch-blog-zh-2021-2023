DealHub 获得 6000 万美元用于销售工具的配置、定价和报价

原文：https://web.archive.org/web/https://techcrunch.com/2022/06/14/dealhub-secures-60m-for-sales-tools-to-configure-price-and-quote/

配置、定价和报价软件，也称为 CPQ 软件，旨在帮助卖家对复杂和可配置的产品进行报价，如具有不同类型底盘和发动机的重型卡车。需求很高— 根据 Gartner 的数据，2019 年 CPQ 软件市场增长了 15.5%，估计达到 14.2 亿美元。但是这可能很难操作，因为不是所有的供应商都关注配置的相同方面。例如，一些 CPQ 软件强调以牺牲产品定制为代价的价格优化。

这促使 Eyal Elbahary、Eyal Orgil 和 Alon Lubin 成立了 DealHub ，这是一家专注于创建“准确”和“可预测”销售渠道的 CPQ 供应商。DealHub 于 2014 年推出，拥有 160 名员工，多年来已经扩展到包括合同生命周期和订阅管理工具，与 Conga、Tacton 和 Vendavo 等公司竞争。

“传统的 CPQ 解决方案通常需要很长的实施周期和大量的代码定制，几乎没有灵活性，”首席执行官 Elbahary 在一次电子邮件采访中告诉 TechCrunch。“DealHub 平台的混合引擎提供了一个独特的组合:定制编码解决方案的深度和复杂性，以及面向业务的无代码配置环境的灵活性。”

在创办 DealHub 之前，Elbahary 是 Check Point 的投资组合营销主管，并领导了 Mercury Interactive 的产品发布，Mercury Interactive 是一家 IT 管理初创公司，于 2006 年被惠普收购。Orgil 在惠普和 Infosys 子公司 Panaya 等公司担任过各种营销职务。至于 Lubin，除了 Panaya 之外，他还在 Veritas Software 和 Taboola 工作过。

Elbahary 将 DealHub 描述为买家和卖家参与交易不同部分的“数字工作空间”，拥有销售团队活动、行动和见解的“整体视图”。为此，DealHub 提供了“销售行动手册”，旨在帮助销售团队了解配置报价、创建销售材料和谈判合同的流程。该平台还可以动态生成相关的合同和法律条款，为收集电子签名和标记等任务提供便利，即在双方或多方谈判时编辑合同。

“疫情迫使销售团队迅速从传统销售流程转变为远程销售。随着旅行速度放缓，面对面的会议变得不可能，销售团队被迫采用数字工具。这种加速的数字化转型增加了对销售技术的现有需求，”Elbahary 说。

CPQ 系统不能解决所有问题。如果其中的数据不是最新的或不准确的，他们产生的建议可能包括夸大的价格或不准确的交货时间。也总有可能出现重大人为错误，比如销售人员忘记计入运输或交付成本，或者遗漏了另一个团队需要的组件。

但 Elbahary 声称，DealHub 的工具比该领域其他供应商的工具更复杂，包括 Oracle、Salesforce 和 SAP 等现有供应商。对收入数字三缄其口，但投资者似乎有足够的信心投资该公司。DealHub 今天完成了由 Alpha Wave Global 牵头的一轮 6000 万美元的融资，使这家初创公司的总融资额达到 9000 万美元。

“虽然头条新闻报道了贬值和裁员，但 DealHub 正在增长和招聘……我们正在经历连续第四年同比增长 250%，”Elbahary 说。“CPQ 市场充满了未开发的潜力，就像我们独一无二的产品一样。我们的投资者知道这一点，并通过这笔资金展示了他们对我们的信心。”
