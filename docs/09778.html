LineupSupply 的应用将音乐节海报变成 Spotify 播放列表

原文：https://web.archive.org/web/https://techcrunch.com/2022/09/06/lineupsupplys-app-turns-music-festival-posters-into-spotify-playlists/amp/

图片来源: gilaxia/E+ / Getty Images

如果你经常发现自己制作 Spotify 播放列表，为即将到来的音乐节做宣传，或者重温过去最喜欢的节目，一款名为 LineupSupply 的新移动应用现在可以帮助简化这一过程。这个聪明的工具允许你上传一张音乐节海报的照片，让它在几分钟内自动转换成 Spotify 播放列表。或者，你可以使用该应用程序来查找其他人创建的播放列表，或者一次性购买 1.99 美元，根据你上传的图片中的艺术家来获得音乐推荐。

这款新应用是由长期在西雅图工作的 iOS 开发者布雷特·鲍曼开发的，布雷特·鲍曼目前是 Loom 的高级 iOS 工程师。

鲍曼说，他是在搜索即将到来的音乐节 Portola 的 Spotify 播放列表后想到这个主意的，他计划参加这个音乐节。

“这让我想到，人们正在手工制作这些，我找不到另一种自动化的服务，”他说。

开发者在夏季开始开发这款应用，并在 8 月下旬发布了当前版本。鲍曼指出，大部分时间都花在了等待 Spotify 提供允许应用程序运行的 API 密钥上。

首次启动时，您必须先使用您的 Spotify 凭据进行身份验证，然后才能开始。然后，您可以将音乐节海报图像上传到应用程序，开始创建您的第一个播放列表。该应用程序目前支持带有英语、法语、意大利语、德语、西班牙语、葡萄牙语或中文文本的图像，所有图像扫描都在您自己的设备上本地完成。

除了 Spotify 的 API，LineupSupply 还使用苹果的视觉框架从音乐海报的图像中提取文本，然后将其与艺术家的名字进行匹配，并转换为音乐播放列表。虽然这款应用要求用户拥有 Spotify 账户，但你不必成为 Spotify 高级用户就能利用它的功能。

考虑到相关活动的艺术家阵容，该应用程序创建的播放列表可能长达数小时。

不幸的是，我们发现该应用程序与非典型字体斗争，并不总是能够识别一些海报上的所有艺术家。例如，当我们用一张即将到来的跳房子音乐节的图片测试该应用程序时，该图片以各种大小的彩色字母艺术家的名字为特色，该应用程序只能识别海报上的一小部分艺术家。这可以为创建自定义播放列表提供一个开端，但我们仍然需要像以前一样手动填写播放列表的其余部分。(鲍曼说他正在研究这个问题。)

但是我们发现，其他使用更标准字体的活动，比如EDC 奥兰多 2022 的这张图片，效果要好得多。

LineupSupply 还允许您在创建播放列表之前通过删除不想包含在播放列表中的艺术家来自定义播放列表。如果你不想寻找和上传自己的图片，你可以在应用程序的“发现”部分找到其他用户建立的现有播放列表。

免费版可以创建的播放列表数量没有限制。

只需 1.99 美元的一次性升级，你就可以获得一些额外的功能，包括设置自定义应用程序图标的能力，或者通过控制每位艺术家的歌曲数量、歌曲排序选项和播放列表描述来进一步定制播放列表。

付费用户还可以根据您上传的图片中找到的艺术家获得个性化的音乐推荐。

鲍曼说，他有一些其他的想法来继续开发应用程序，包括支持其他平台的计划，如苹果音乐。他也在考虑自动创建播放列表的其他方法。

他告诉 TechCrunch:“除了图像，我真的很有兴趣探索其他媒体来创建播放列表，比如视频和文本。”

在该应用发布的头两天，用户创建了 100 个播放列表，现在，大约一周后，这个数字已经增长到 250 个左右。当然，这并不代表用户数量，因为一些人可能会使用该应用程序访问其他人创建的播放列表，而不是上传他们自己的图片。

该应用程序是从应用商店免费下载的。
