跟上最新的技术资金动向

尽管风险投资活动放缓，但如今仍有大量资金流入初创企业。TechCrunch+正在发布一系列帖子，关注最近值得关注的风险投资回合、退出活动以及其他与建立新技术公司的财务方面相关的新闻。

虽然银行正在应对由著名的、对初创企业友好的硅谷银行的失败引发的危机，但新贵科技公司仍在忙于筹集资金。他们也在寻找出口。鉴于冻结的 IPO 市场，前者多于后者。但是，当我们等待创业公司的一个关键退出点被重新唤醒时，我们仍然可以密切关注资金从哪里以及如何流入他们的世界。

一周的精彩回合

  • 在特殊目的收购公司交易失败后，消费者交易服务公司 eToro 没有预期的新资本和新的估值。然而，该公司此前曾保证，如果特殊目的收购公司交易失败，它将获得新的资金，现在它已经筹集到了这些资金。
  • 这轮融资对其规模(9 位数)、行业(金融科技在最近几个季度受到了估值冲击)和潜在财务业绩至关重要。尽管自 2020 年以来，该公司在 2022 年出现了一些增长，但与去年的 2021 年相比，该公司出现了萎缩。这意味着我们看到一家大型消费金融科技公司在困难的条件下设定了新的估值。金融科技的创始人应该注意到这一点。
  • 过去一年，随着加密公司的财富减少，许多人认为对 Dao(分散的自治组织)的兴趣已经消退。但事实证明，仍然有很多人非常热衷于社区的概念，他们自己决定如何花费数百万美元。
  • Seed Club Ventures 是一个由风投、个人投资者、家族理财室和各种仍然相信 web3 的实体组成的 63 人财团，最近秘密成立了一个 2500 万美元的基金来帮助 DAOs 做到这一点。
  • 这很重要，因为这 2500 万美元将用于真正的早期项目建设【DAOs 急需的工具。它已经支持了 Guild、Stability AI、Lens 和 Metalabel 等项目。这种工具实际上有助于 Dao 实现完全分散系统带来的部分(如果不是全部)潜力。

IntegrityNext 筹集 1 . 09 亿美元帮助企业确保其供应链符合 ESG 标准

  • 围绕环境、社会和治理(ESG)投资政策有很多政治因素，这是有充分理由的:遵守 ESG 规范要求公司检查其运营的广度和深度，以确保负责任地做事。这可能会变得昂贵、乏味，而且需要很长时间。
  • 总部位于慕尼黑的 IntegrityNext 正在做一件非常特别的事情来帮助公司解决这个问题:它帮助公司审计他们的供应链，以便他们可以快速找到他们可以在哪里以及如何优化供应链并符合 ESG 要求。
  • 这次融资对欧洲公司来说确实是个好消息，因为随着欧盟监管的收紧，他们将更容易采纳以前“不错”的 ESG 政策，而这些政策很快就会成为“必须”。
  • 在一个丰富的世界里，有些东西是稀有的，这就是奢侈品经销商平台存在的原因。Kream 从韩国电子商务巨头 Naver 剥离出来，成立仅两年，但该公司已经取得了令人难以置信的成功，因为懂时尚的客户涌入了它的商店，寻找高端、稀有的运动鞋、手表、包、配饰和服装。
  • Kream 的 1.68 亿美元融资很有趣，因为该公司将在同行中投入大量资金，以建立一个覆盖亚洲大片地区的经销商网络——这意味着日本有人可以购买仅在日本推出的限量版运动鞋。
  • 这对亚洲不断增长的转售市场也是一个好消息，因为它表明消费者对收藏品和其他奢侈品的兴趣，这可能会推动这一领域的进一步投资。

克雷迪沃筹集了2 . 7 亿美元的巨额 D 轮融资，让银行服务不足的亚洲人更容易获得信贷

  • 众所周知，亚洲发展中经济体中大量银行服务不足的人口是金融科技要破坏的一个大市场，而旨在增加印度尼西亚和越南信贷获取渠道的 Kredivo 无疑已经获得了与印度尼西亚信用卡持有人口差不多大的用户群。
  • 该公司超额认购了 2.7 亿美元的 D 轮融资，这证明了一个事实，即在让人们的生活更轻松、帮助他们轻松无缝地获得银行服务方面，还有增长空间。

其他创业和风险投资新闻

风险投资放缓正在放缓甚至是最快的创业类别

  • 这是一个可悲的现实，有时甚至连钻石都没有人问津，这似乎正在初创公司的土地上淘洗:甚至是以前热门的 API 初创公司也在风险投资放缓中遭受损失。
  • 根据 GGV 追踪 63 家 API 公司融资情况的数据，2022 年这一类别的初创公司融资约 21.5 亿美元，不到一年前融资额的一半。交易数量也有所下降。2022 年第四季度，这些初创公司筹集了微不足道的 1.34 亿美元，低于今年前三个季度的水平。那一定很艰难。
  • 我们关心这一点，因为即使 API 初创公司以基于使用的定价模式领先，这可以说是软件销售的未来，他们仍然受到更广泛的市场压力。他们的挣扎表明，无论你所在的行业有多热门，赚钱的难度都可能会越来越大。

比特币基地高管对证券交易委员会给他们泼冷水感到愤怒

  • 加密世界对立法者对待它的方式不满意。比特币基地的首席执行官最近表示，在美国证券交易委员会向其发出威尔斯通知后，政府应该在监管方面下定决心，这基本上意味着政府将因“违反联邦证券法”而追捕比特币基地和像它这样的公司
  • 我们有点同意比特币基地的观点:密码世界正在经历的事情真的没有太多先例，将 SEC 近百年的法律适用于密码经济感觉非常像一个方形钉三角洞的情况。
  • 很明显，美国证券交易委员会需要真正巩固其对密码应该如何交易的信念，以便更广泛的生态系统可以遵循这些规则。

T4 在第二轮裁员中裁员 27%

  • Proptech 初创公司正在经历一个时刻，他们的员工似乎正在为此买单。不断上升的抵押贷款利率和普遍的房地产放缓对依赖人们实现他们的美国梦的公司来说并不好。
  • 但是在这种经济下买房？许多人基本上说，“是的，没错，”这基本上导致了 Roofstock，它允许人们在几十个美国市场上买卖出租房屋，决定它需要在不到两个季度内第二次裁员 27%。
  • 该公司试图在不断下滑的房地产市场中生存下去，这是有道理的，但没有道理的是，就在一年前，它的估值为 19 亿美元。这对目前更广泛的 proptech 市场来说不是好消息。
  • 印度初创公司在 2022 年初前景非常好，因为全球风险投资放缓尚未波及该国。但事实确实如此，导致今年下半年资金下降了 70%。

  • 随着 Deel、Velocity Global、Gusto 和 Ripple 等独角兽公司继续发展到新的市场和类别，这家初创公司的 ARR 增长和定期 EBITDA 产出——以及估值——似乎几乎不受经济放缓的影响。
