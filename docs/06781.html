数据观察初创公司 Manta 融资 3500 万美元

原文：https://web.archive.org/web/https://techcrunch.com/2022/05/26/manta-a-data-observability-startup-raises-35m-to-grow-its-workforce/amp/

图片来源:solar seven/Getty Images

在过去十年中，随着存储、访问和处理数据变得越来越便宜，组织创建数据的速度加快了。但是，随着数据的规模和复杂性不断增长，它变得分散在应用程序和平台之间，这往往会导致数据质量方面的问题。根据 2020 年 O'Reilly 的一项调查，超过 60%的公司认为他们有太多的数据源和不一致的数据，而超过三分之一的公司表示他们可用于解决数据质量问题的资源太少。

托马斯·克拉特基认为解决方案在于软件。他是 Manta 的创始人，Manta 是一个数据沿袭平台，它自动扫描一个组织的数据源，以构建数据流图。

“数据驱动的决策只能取决于底层数据集和分析的质量。Kratky 在一次电子邮件采访中告诉 TechCrunch，从充满错误的电子表格或商业智能应用程序中收集的见解可能会导致糟糕的决策，这可能会代价高昂并损害业务。为了支持大型数据现代化和转型项目，数据沿袭是解决大型数据基础架构层的复杂性和跟踪组织内数据流的关键组件

Manta 今天宣布，它在由 Forestay Capital 牵头的 B 轮融资中筹集了 3500 万美元，参与方包括 Bessemer Venture Partners、SAP.io、Senovo VC、Credo Ventures、Dan Fougere 和欧洲复兴开发银行。该公司表示，这笔收入将用于在年底前将其员工人数从 152 人扩大到 220 人。

Manta 于 2016 年由 Kratky 创立，他此前在捷克数据科学咨询公司 Profinit 领导 R&D。他说，目标是通过数据依赖关系图来解决企业数据环境的复杂性，这些数据依赖关系图旨在帮助用户避免“重大数据事故”

Kratky 认为，他已经或多或少地通过 Manta 平台的当前迭代实现了这一点。利用人工智能，Manta 追踪数据的来源——包括数据库、报告和分析软件以及建模工具——直到将数据交付给应用和服务的管道末端。“时间切片”功能允许用户查看数据在过去的样子，并了解沿袭是如何变化的，而影响分析工具则显示计划的更改将如何影响数据环境的各个部分。

图片来源:蝠鲼

“我们结合了许多不同的技术方法，因此我们的客户可以轻松消除盲点，并完全了解和控制其数据环境中最复杂的方面，”Kratky 说。“重要的是，由于 Manta 只绘制客户的数据处理环境，我们不接触数据本身，只接触处理数据的系统。"

Manta 还监控数据流中的差异和中断，结合不同的技术方法以尽量减少盲点。该平台的分析可以在本机仪表板中查看，也可以通过连接器和 Manta 的 API 发送到第三方软件，包括治理软件。

Kratky 说:“数据是每个组织的生命线，也是当今企业最重要的资产，[但是]大多数组织都面临着…大量的重大事故、重大风险、缓慢的变更管理，以及大量数据工程资源持续浪费在日常手动任务上。“数据沿袭和可观察性是解决这些复杂问题的关键能力。如果实施得当，它们可以帮助恢复可见性，从而恢复对数据的信任，提高实施变更和业务需求的效率和速度，并防止事故发生。”

虽然 it 团队确实在努力跟上他们组织的数据，但他们也面临着旨在帮助管理数据的软件的障碍。根据最近适马的一项调查，四分之一的商业专家说他们组织的数据分析花费太长时间，导致团队放弃寻找异常现象的答案。但是 2021 Era Software poll 发现数据可观察性工具的失败通常会导致故障排除和事件解决的延迟。

Kratky 指出 Manta 的客户群是该平台稳健性的证据:100 家客户，包括“财富 100 强和财富 500 强的超大型企业”。他认为 Manta 与 Collibra 和 Informatica 在数据谱系和可观察性工具市场上展开竞争。其他潜在的竞争对手包括蒙特卡洛和 T2 数据银行，这两家公司都已经筹集了数千万美元的风险投资。

Statista 预测到 2024 年，网络安全和可观察性市场的总价值将达到 282.6 亿美元。

“数据谱系和可观察性正在成为任何现代数据架构的核心组成部分。几乎数据管理领域的每一个参与者都至少在评估这个市场，他们中的大多数都试图进入这个市场—无论是数据编目、数据质量等方面的供应商。，”Kratky 说。“当企业完全了解其数据堆栈时，投资回报将对运营和业务决策至关重要…换句话说，通过了解自动化数据谱系在[其]数据质量战略中的作用，企业可以获得提高数据质量的好处。”

Manta 在纽约、布拉格、里斯本和都柏林都有办事处，在坦帕湾还有一个新的总部。通过 B 轮融资，这家初创公司的总融资额达到了 5300 万美元。
