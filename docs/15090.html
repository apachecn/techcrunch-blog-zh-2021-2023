低代码数据库 APITable 是 Airtable 的另一个挑战者| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2023/02/09/low-code-database-apitable-is-another-airtable-challenger/

Airtable 在 2021 年末达到的高达 110 亿美元的估值已经吸引了一大批雄心勃勃的企业家进入低代码数据库领域。他们甚至觊觎十年前的美国巨头所享受的一小块馅饼。虽然 SaaS 长期以来一直是美国、新兴市场，甚至是中国蓬勃发展的科技行业的成熟商业模式，但许多用户仍然希望软件是免费的。

为了增加付费客户，由经验丰富、价格合理的开发商创建的亚洲 SaaS 初创公司正在进军西方。总部位于香港的低代码数据库 APITable 就是其中之一(它正在建立加拿大基地)。该公司毫不掩饰其在 Airtable 上的雄心。举个例子，它的博客上有一个关于如何从 Airtable 迁移到自己平台的分步指南。

APITable 正在与一些新兴的初创公司竞争，如阿姆斯特丹的 Baserow 和旧金山的 NocoDB ，为创建智能、时尚的数据库提供开源、可视化的解决方案。它的名字暗示了对系统互操作性的关注。该公司的联合创始人和首席运营官的 Gary Li 在接受采访时说，在未来，用户可以使用 APITable API 将低代码工具连接到包括 Zapier，Slack，Google Workspace 和 red-hot ChatGPT 在内的平台。

对于希望获得只需要最低技术技能和第三方托管的现成产品的用户来说，该软件附带一个付费的高级版本。截至今天，APITable 已经在 GitHub 上被“加星”约 6500 次 ，这是一个项目在开发人员中受欢迎程度的指标。虽然李拒绝透露该公司的收入规模，但他声称，到 2022 年，包括免费增值和付费用户在内的“组织用户”将从 6000 人增长到 20000 人，遍布 30 个国家。

APITable 的创始团队首先在中国试水。它的中国版 Vika 于 2021 年 2 月推出，当时风险资本家受到西方 Zoom、Stripe、Canva、Figma、Airtable 和其他基于云的解决方案的热潮的鼓舞，纷纷押注于当地的替代方案。

APITable 本身获得了知名风投的资金，如 IDG 资本、GL Ventures (Hillhouse 的早期子公司)、5Y Capital(被一些人视为中国企业科技投资的领头羊)和 Eminence Ventures 。截至 2021 年初，这家初创公司已经筹集了 1000 万美元的资金，估值约为 7500 万美元。

回到中国，维卡不乏竞争对手。从字节跳动和腾讯这样的巨头到 Hipacloud 和 Treelab 这样的弱势公司，都在叫嚣着要为 Airtable 找到答案。展望未来，母公司将更加关注通过 APITable 在西方的扩张。“开源本身就是一种营销方式。不过，为了创收，我们仍然依赖 Saas 销售，”李估计。

APITable 的首席执行官陈柏瑜似乎对为非技术用户创建工具略知一二。他之前是 HeyTea 的首席技术官，hey tea 是一家风险投资公司，彻底改变了中国的泡沫茶业务。该连锁店不仅开创了新的奶酪口味，更重要的是，它投入巨资建立了一个相当大的开发团队，这在食品和饮料行业是一个相当不寻常的举动。结果令人放心。在成龙的领导下，HeyTea 开发了一个应用程序来加快点餐速度，缩短等待时间，并通过会员计划提高经常性收入。

更新:增加了 Eminence Ventures 作为投资者，并修正了 Vika 的推出日期。
