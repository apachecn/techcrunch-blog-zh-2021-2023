随着数据协作平台的持续发展，Hex 又获得了 2800 万美元的收入| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/23/hex-lands-another-28m-as-data-collaboration-platform-continues-to-gain-traction/amp/

图片来源: ArtemisDiana / Getty Images

如今，在日益紧张的投资环境下，许多公司都在努力筹集资金。风险投资公司向有前途的初创公司寻求资金而不是反过来的日子似乎已经一去不复返了，但数据合作初创公司 Hex 挑战了这种可能性。

公司联合创始人兼首席执行官巴里·麦卡德尔说，当红杉资本来敲他的门时，他并没有在寻找资金。该公司最终从红杉资本获得了 2800 万美元，现有投资者 Andreessen Horowitz、Amplify 和 Snowflake 也参与了投资。

麦卡德尔说，他还没有动用去年上一轮融资中存在银行的 5200 万美元。“我们没有筹集资金，也不需要这笔钱——上一轮我们一毛钱都没碰过。但红杉资本(Sequoia)的马特·米勒(Matt Miller)先发制人地向我们提出了一个几乎没有稀释的报价，让我们再投资 2-3 年，”麦卡德尔告诉 TechCrunch。

“在我们希望深入投资于我们的产品和团队，尤其是围绕人工智能机会的时候，这感觉是显而易见的，”他说。

投资者感兴趣的原因之一是，自去年融资以来，该公司一直在快速增长。“我们一直在撕。我认为从顶线业务的角度来看，我们在一年内收入增长了 4 倍，业务规模增长了 4 倍，这非常好。我们的用户群增长了 10 倍，因为我们推出了一个层，让更多的人可以访问。”

“因此，去年使用该产品的原始人数一直很高。我认为这显然吸引了新投资者的目光，”他说。

该公司现在有 450 个付费客户，包括 Brex、idea、Toast 和 Chegg。他认为，数据的协作方式以及对人工智能正在改变工作流程的日益认可，是该产品越来越受欢迎的最大原因。

最近，该公司增加了一个名为 Hex Magic 的工具，这是一种生成式人工智能方法，通过询问有关数据的问题，或要求 Hex 承担自动完成连接等任务，用简单的语言与数据进行交互，软件会对请求做出响应。他认为这是一种将用户群扩展到数据科学家和懂 SQL 或 Python 的人之外的方法。

“我们的产品独特地支持不同数据角色之间的协作。我认为这也非常强大。我认为人工智能的东西是与此相关的。你不一定要知道如何用 Python 写完美的语法来做某事，因为 Hex 可以帮你做到这一点。所以我认为这是一个非常强大的东西，我们将看到它在向更多人开放(平台)方面变得越来越重要，”他说。

Hex 成立于 2019 年，根据 Crunchbase 的数据，在短短四年内，hex 已经筹集了超过 1 亿美元的资金。最近一轮的资金。麦卡德尔说，他应该有五年的跑道，让他的公司安然度过可能存在的任何经济不确定性，同时给他必要的自由来发展平台。
