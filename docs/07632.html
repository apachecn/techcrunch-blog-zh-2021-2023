退出机会

在这一点上，很多旁观者兴高采烈地看着 crypto 崩溃，笑着看着代币暴跌和 NFT 交易量缩水，这已经不是什么秘密。在本轮牛市中，加密行业已经成功地为消费者树立了许多敌人——批评者强调了激进的能源使用、加密投资令人上瘾的特征以及 NFT 如何成为“男人的传销”——作为他们厌恶的理由。

随着牛市接近尾声，这可能是一个反思的好时机，投资者对网络的 web3 愿景如何能让消费者感到兴奋而不是怀疑，但有些事情告诉我，密码行业将变得比以往任何时候都更加孤立。

本周，NFT 居民聚集在纽约时代广场。昂贵的图片出现在巨大的广告屏幕上，代币门派对盛行，一群突然变得不那么富有的收藏家找到了同情和双倍下注的方法。我的共同主持人安妮塔有机会亲自访问 NFT 市，并给出了一些想法，但在某些方面，积极的氛围展示了一个行业从增长模式到生存模式。

当然，NFT 世界版本的生存看起来有些不同。在本周的活动中，Bored Ape 游艇俱乐部举办了一个有未来、液晶音响系统和艾米·舒默表演的节日。驯服的黑斑羚登上了凯文·罗斯月鸟活动的头条，代币持有者可以在现场纹上猫头鹰纹身。NYPD 捣毁了象征性的 NFT 聚会。一个项目雇佣了数十名抗议者，他们举着写有“上帝憎恨非功能性传播疾病”的标语站在他们活动的外面。一家 NFT 初创公司雇佣了一名史努比狗狗模仿者“杜普·斯诺格”在他们的活动中走动，作为一种默认的伪认可。

最后，众所周知，NFT 市场充斥着大量的废话，任何熊市都可以也应该恢复一些理智，但在 NFT，这些界限看起来有点模糊。

从某些方面来说，这感觉就像是 NFT 空间的富有收藏家在他们建立的世界为崩溃做准备时抛弃了他们。拥有 10+ ETH 楼层、风险投资和巨大交易量的所谓蓝筹项目在经济低迷时期表现出了令人惊讶的弹性，尽管它们所基于的基础加密货币的价值正在下跌，但随着不太富裕的收藏家尽可能寻找退出流动性，NFT 项目楼层全面遭受重大打击。

最新的豆荚

我们揭开了本周这一集的序幕，揭开了由狗爸爸埃隆·马斯克引起的一些争议。马斯克和他的公司 SpaceX 和 Tesla， 正被一名 Dogecoin 持有者 起诉，原因是涉嫌夸大加密货币的价值，此后该货币崩溃。

本周是 NFT 纽约市的时候了，这是一个吸引了有影响力的人、投资者、名人等来到纽约的秘密会议(安妮塔在下面做了更多介绍，她一直在纽约市与 NFT 社区交谈)。我们深入研究了 NFT 市场本身，以及在加密和科技整体市场形势如此严峻的情况下，是什么推动了 NFT 市场的繁荣。我们以两起与道有关的灾难结束了本周的新闻，这两起灾难对这个最近流行但不可否认地混乱的治理结构的未来可能不是好兆头。

跟着钱走

启动资金在密码世界流向何处:

  1. 面向机构投资者的数字资产平台 FalconX 宣布以 80 亿美元的估值获得 1.5 亿美元的 D 轮融资，由 GIC 和 B Capital 领投。
  2. NFT 收藏品项目 涂鸦 从亚历克西斯·奥哈尼安的七七六公司吸引了一笔数目不详的资金。
  3. 总部位于 NFT 的 Solana market place 在由 Electric Capital 和 Greylock Partners 共同牵头的 B 轮融资中筹集了 1.3 亿美元，使其估值达到 16 亿美元。
  4. 加密和金融科技基础设施初创公司 Prime Trust 从包括 FIS、Fin Capital 和北海巨妖风险投资公司在内的投资者那里获得了 1 亿美元的 B 轮投资。
  5. 无许可融资融券交易协议open leverage从币安实验室获得一笔规模未披露的战略投资。
  6. 总部位于 NFT 的喜剧和模因工具公司 可怕的宠物 ，一个来自电视节目《硅谷》制片人的项目，筹集了由首轮资本、XYZ 资本和 Moment 领投的约 400 万美元资金。
  7. NFT 流动性提供商 Astaria 完成了一轮 800 万美元的种子轮投资，投资者包括 True Ventures 和 Arrington Capital。
  8. 专注于运动鞋的 NFT 平台 Endstate ，从包括原型和城堡岛在内的投资者那里获得了 550 万美元的种子资金。
  9. 算法汇率协议 增量 为 ParaFi 领投的种子轮融资 156 万美元。
  10. Afropolitan 从 Balaji Srinivasan 和其他投资者那里筹集了 210 万美元的种子基金，为非洲人和非洲移民建立一个数字民族国家。

本周在网络 3

嘿，我是安妮塔，本周在纽约 NFT 为您做现场报道。包括我自己在内，每个住在曼哈顿的人都被一股喜悦的潮流所包围，对本周的低迷不屑一顾。你可以收听本周的播客来听听我对所有这些的想法，但我想在这里解决一个不同的问题: 密码社区正在实践他们所宣扬的吗？

排队数小时才拿到 NFT 纽约通行证的人们在推特上抱怨连连。他们告诉我，即使是那些在小组讨论会上发言的人，也不得不和所有参加活动的人一起排队，这显然要绕过多达三个街区。

我已经在纽约住了一段时间，所以我不容易被长长的队伍动摇，但这让我想到了整件事的讽刺之处。NFTs 及其相关技术可以提供简单的认证和身份验证。NFT·斯坦斯喜欢引用事件的例子作为该技术的主要用例，他们说这可以使管理负担(如登记人们参加会议)更加有效。那么，在本周的会议上，这项技术在哪里呢？

我确信举办一场秘密活动需要从混乱中创造秩序，这远远超出了我的能力范围，所以我没有特别挑出 NFT 纽约市的组织者或其他任何人。但是纽约 NFT 的台词在我心中提出了一个更大的问题，关于加密社区 所说的 是未来与加密社区实际行为之间的矛盾。比如说，为什么面对面的会议在认识 web3 中的人方面如此重要？我们不都应该过了需要呼吸彼此的空气来感受人类联系的阶段吗？

根据我在过去一年里从 web3 社区的很多人那里听到的，我本以为现在我们都已经和我们的闺蜜们在元宇宙全天候地闲逛了。加密会议本身似乎为 web3 爱好者提供了一个巨大的机会，让他们真正利用他们所说的将改变我们生活方式的技术。迄今为止，这个机会似乎在很大程度上被忽视了。
