特斯拉在美国大幅下调 Model S 和 Model X 价格

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/05/tesla-slashes-prices-for-model-s-and-model-y-in-us/

特斯拉在美国大幅降低 S 型和 X 型汽车的价格

据特斯拉公司网站和最先由彭博报道的消息，特斯拉已经降低了其 Model S 轿车和 Model X SUV 在美国的价格。这是特斯拉今年第二次降价，这可能是为了在本季度结束前提高销量。

Model S 全轮驱动现在的售价为 89，990 美元，比 94，990 美元下降了 5.2%，约合 5，000 美元。Model S 格子现在是 109990 美元，比 114990 美元下降了 4.3%。

Model X 全轮驱动的售价为 99，990 美元，比 109，990 美元下降了 9.1%或 1 万美元。格子现在是 109990 美元，比 119990 美元下降了 8.3%。

最近的降价发生在特斯拉在一月份大幅降价之后，这是该汽车制造商在过去几个月中至少第五次降低其车辆成本。上周在特斯拉奥斯丁工厂举行的投资者日，首席执行官埃隆·马斯克和其他领导人讨论了高效制造和削减成本的重要性。

“人们拥有特斯拉的愿望非常强烈，”马斯克说。"限制因素是他们支付特斯拉的能力."

本月剩余时间，特斯拉 Models 3 和 Y 仍可在美国享受 7500 美元的联邦税收优惠。
