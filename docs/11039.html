首席财务官离职，Noom 解雇更多员工

原文：https://web.archive.org/web/https://techcrunch.com/2022/10/11/noom-tech-layoffs-diet-app/amp/

图片来源:Piarasómídheach/sports file for Web Summit/Getty Images

TechCrunch 从消息人士处获悉，去年价值 37 亿美元的健康教练平台 Noom 在几个月内第二次裁员。

Noom 已经裁掉了 10%的员工，约 500 人，这一裁员主要影响其教练团队。这是 Noom 的教练团队在几个月内第二次裁员，影响了数百名员工。有消息称，自今年年初以来，Noom 的教练数量减少了近一半，目前约有 1000 名教练。

“Noom 在过去几年经历了非凡的增长，我们的结构必须能够让我们长期持续增长，”Noom 的一位发言人在电子邮件中说。“我们最近做出了一个艰难的决定，减少 Noom 的员工数量。我们对他们为 Noom 做出的贡献深表感谢，并祝愿他们继续取得成功。”除了这份声明之外，Noom 拒绝回答有关裁员规模、离职方案和战略的问题。

创业公司自 2008 年成立以来，已经筹集了超过 6.5 亿美元的资金，部分是因为其有争议的减肥和节食方法而闻名。然而，现在它显示出来自人事方面的紧张。Noom 对其教练服务的缩减表明，该平台偏离了最初的定位，即把智能营养与运动教练结合起来。2020 年， Noom 利用这一战略实现了 4 亿美元的收入。一年后，Noom 推出了心理健康垂直服务，扩大了其教练服务。

其目前的网站显示了 Noom 如何思考，或者至少是如何思考，将教练作为其战略。

“欢迎来到 Noomily，”Noom 的网站上写道。“我们的教练以同理心和同情心引导用户走过减肥旅程的起起落落。他们通过基于个人目标、偏好和生活方式的个性化行动计划，帮助用户更好地了解自己。”

教练的登录页面继续解释 Noom 教练的日常生活，包括帮助客户发展“健康的生活方式”和“为惊人的事情的发展做出贡献”该网站称，教练可以在就业 90 天后加入健康计划。

然而，现在广告有了不同的基调。据报道，该公司正在更多地转向预定视频通话，而不是实时聊天。

根据 TechCrunch 获得的一份备忘录，Noom 联合创始人阿尔特姆·佩塔科夫和 Saeju Jeong 向员工发表了裁员讲话，并表示尽管公司财务状况“显著改善”，但“管理支出、提高效率和更有效地实现我们的使命”对公司来说至关重要。

联合创始人在内部备忘录中写道:“今天的决定让我们能够继续为明年下大赌注，并在未来几年创新和增长。”据华尔街日报报道，今天的裁员正值该公司首席财务官迈克·努南离职加入猫途鹰。Noom 发言人通过电子邮件表示，裁员和首席财务官离职是两个“独立的、不相关的公告”

在内部备忘录中，联合创始人接着说，这家初创公司的大赌注集中在创建一个更全面的身心平台，个性化程度更高。联合创始人对他们的企业产品点头称是“另一个有可能极大地扩大我们的客户群的赌注。”

“最后，在接下来的几天里，我们可能会看到不友善的头条新闻，但请你们继续关注现在最重要的事情:像家人一样照顾我们即将离职的同事，以人性和尊重对待他们，”联合创始人写道。

据报道，该公司最近一次公开募股是在 2022 年，，估值 100 亿美元。
