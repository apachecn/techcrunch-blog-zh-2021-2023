在埃隆·马斯克的最新货币化计划中，Twitter 将终止免费访问其 API

原文：https://web.archive.org/web/https://techcrunch.com/2023/02/01/twitter-to-end-free-access-to-its-api/

在埃隆·马斯克的最新货币化计划中，Twitter 将终止其 API 的免费访问

埃隆马斯克拥有的微博网站表示，Twitter 将从 2 月 9 日起停止提供免费的 Twitter API，并将推出付费版本，该网站正在寻找更多的平台盈利途径。

在一系列的推文中，Twitter 开发者账户表示，该公司将停止对旧版 1.1 版和新版 2 版 Twitter APIs 的支持。该公司没有立即透露计划收取多少 API 使用费。

在此之前，Twitter 最近几周突然改变了其 API 的条款；这个 API 被许多流行的 Twitter 客户端使用，比如 Tweetbot 和 twitterrial T5。大多数第三方 Twitter 应用已经关闭了他们的移动应用。

“Twitter 数据是世界上最强大的数据集之一。我们致力于实现快速和全面的访问，以便您可以继续与我们合作，”Twitter Dev 帐户周四表示。“多年来，数亿人发送了超过 1 万亿条推文，每周还会增加数十亿条。”

在最近的变化之后，Twitter 关闭了第三方客户端，许多其他应用程序开发者对他们在 Twitter API 上进行开发的方式变得谨慎起来。这一新举措可能会让一些开发商要么放弃他们的产品，要么将成本转嫁给客户。

成千上万的开发者使用 Twitter API 做很多事情，比如跟踪 Twitter 账户的变化和提供提醒。这些都是有趣的副业，适合那些不愿意为自己没有赚钱的东西付费的人。

然后 Twitter API 还有另外一个特定的用户群:研究人员。Twitter 的新声明可能会影响不同领域的研究，包括仇恨言论和在线虐待。大学经常使用推特来研究不同地区的人类行为。限制免费 API 的使用也可以阻止公司在 Twitter 上检测错误信息的传播。

Twitter 从一开始就和开发者有着奇怪的关系。但是这种关系，不管多么不方便，对双方来说都是良性的。第三方公司通常为 Twitter 提供新产品和功能,这家社交网络也尽了自己的一份力量，没有向他们收取 API 使用费。

近年来，Twitter 甚至试图通过推出新的程序来修复与开发者的关系，比如用于应用发现的 Twitter 工具箱。在新管理层的领导下，几个这样的项目已经被终止。

在马斯克的领导下，Twitter 正在努力控制全球用户访问该平台的方式，同时扩大其服务货币化的尝试。

10 月下旬，马斯克以 440 亿美元收购了 Twitter。他获得了约 125 亿美元的贷款来为这笔交易融资。为了让 Twitter 这一不盈利的服务继续生存，马斯克每年仅利息就要额外支付 15 亿美元。因此，自然地，马斯克正在推动 Twitter 变得更加自我可持续。

Twitter 对其订阅服务进行了改进，并提高了价格，还调整了推文在用户时间表上的显示方式，以使该平台更具吸引力、更有利可图，并成为一个吸引日益萎缩的广告客户群的目的地。
