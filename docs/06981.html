苹果的“万能钥匙”可能最终会永远取消密码

原文：https://web.archive.org/web/https://techcrunch.com/2022/06/06/apple-passkeys-look-to-kill-off-the-password-for-good/

苹果的“万能钥匙”终于可以永远取消密码了

苹果在 2022 年 WWDC 车展上展示了“密码”，这是一种新的生物识别登录标准，最终可能会永远取消密码。

众所周知，密码是不安全的，根据威瑞森的年度数据泄露报告，在所有数据泄露中，80%以上是容易被猜到的凭证。据苹果公司称，密钥完全消除了对密码的需求，而且在数据泄露或网络钓鱼企图的情况下，更不容易被盗。

密钥基于 web 身份验证 API (WebAuthn)，这是一种使用公钥加密而不是密码来对用户进行网站和应用程序身份验证的标准，存储在设备上而不是 Web 服务器上。数字密码替换使用触控 ID 或 Face ID 进行生物特征验证，这意味着你不必输入一长串字符，你登录的应用程序或网站会向你的手机推送认证请求。

在无密码技术的 WWDC 演示中，苹果展示了如何在 iCloud 钥匙链中备份密钥，并可以通过端到端加密在 Mac，iPhone，iPad 和 Apple TV 之间同步。用户还可以使用 iPhone 或 iPad 扫描二维码，通过触控 ID 或 Face ID 认证，登录非苹果设备上的网站和应用。

苹果认证体验团队的工程师加勒特·戴维森(Garrett Davidson)说:“因为只需轻轻一点就可以登录，所以它比当今几乎所有常见的认证形式都更容易、更快、更安全。”

苹果公司并不是唯一一家努力取消密码的公司。上个月，谷歌和微软与苹果联手，扩大对手机、桌面和浏览器无密码登录的支持。美国网络安全和基础设施安全局(CISA)局长 Jen Easterly 称赞了这一新的集体承诺，他当时称之为“最终将使美国人民在网上更加安全的前瞻性思维”

苹果、谷歌和微软表示，他们的目标是在明年内在他们的平台上支持新的无密码认证标准——该标准由 FIDO 联盟和万维网联盟建立。如果苹果的 WWDC 演示是可靠的话，macOS Ventura、iOS 16 和 iPadOS 16 将是首批支持新登录标准的操作系统。

苹果在 WWDC 的主题演讲中悄悄宣布了另一项名为快速安全响应的安全功能，据称该功能通过在后台提供安全更新而无需重启，使 macOS 和 iOS 更能抵御攻击。“让你的设备更快地获得重要的安全改进，”苹果在其网站上简短地说。"这些改进可以在标准软件更新之间自动应用."
