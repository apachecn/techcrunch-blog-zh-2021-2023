Chris Dixon，马克·安德森支持 3000 万美元基金专门投资 NFT art TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/03/10/chris-dixon-marc-andreessen-back-a-30m-fund-investing-exclusively-in-nft-art/

非专利技术的世界永远不会无聊——只要钱永不停止。尽管投资者继续向该行业投入数十亿美元(尽管速度放缓)，但更多的加密买家正在增加资金，以支持 NFT 平台、项目和不可替代的令牌本身。

尽管专门投资于非金融资产类别的专项基金开始涌现，但对于较大的机构公司来说，这仍是一个充满风险的领域。不过，这并没有阻止这些投资者成为 NFT 基金的有限合伙人。许多著名的硅谷加密风险投资公司都支持由 Andrew Jiang 和 Todd Goldberg 领导的新贵 NFT 基金。

名为策展的 3000 万美元基金致力于购买和持有 NFT 艺术品。该基金得到了加密投资者名人录的支持，有限合伙人包括 a16z 投资团队的相当大一部分人(马克·安德森、克里斯·迪克森、陈楚翔、阿里安娜·辛普森和乔恩·赖都是支持者)，以及亚历克西斯·奥哈尼安、贾斯汀·坎、电气资本的阿维沙尔·加格和柯蒂斯·斯潘塞，以及该领域的许多其他投资者和创始人。

该基金计划将大约一半的资金投资于所谓的“蓝筹 NFT”，包括 CryptoPunks、Art Blocks 和 Bored Apes 等热门项目，以及出售奇异 NFT 作品的流行艺术家的作品。另一半资金将进入姜和戈德堡所认为的“高潜力收藏”，这些收藏来自现有市场规模较小的艺术家。

戈德堡在接受 TechCrunch 采访时表示:“我们认为，NFT 资产在未来十年将是一笔非常大的交易，比现在大几个数量级，因为 NFT 为我们提供了一种数字化的本土方式来投资互联网规模的文化。”。“我们希望成为一个非常高信号的收藏家，收购顶级资产，并对创作者和建设者有所帮助。”

戈德堡说，该公司也可能探索做自己的 NFT 滴，与他们的投资组合中的一些艺术家合作。

NFT 股市仍在吸金数十亿美元，但最近几周投资步伐似乎在放缓。根据区块链分析工具 DappRadar 的数据，在过去的 30 天里，OpenSea 平台已经完成了 23 亿美元的交易量，比前 30 天下降了 40%。

“市场一直存在泡沫，然而，我们正在进入一个熊市，这实际上对基金非常有利，因为我们是非常耐心的资本，”戈德堡说。“我们将等待进入这些头寸的最佳时机，然后我们将长期持有这些股票。长期、耐心的资本听起来没什么大不了的，但我认为在这个每个人都非常关注短期的市场上，这实际上是一种战略优势。”

蒋是该基金的主要负责人，他之前创办了硬件初创公司 Soda Labs，以及 YC 支持的非营利组织 Bayes Impact。戈德堡是票务初创公司 Eventjoy 的联合创始人，他已经有了一个独立的基金对初创公司进行股权投资，他与超人创始人拉胡尔·沃赫拉(Rahul Vohra)共同运营该基金。该基金的许多投资组合创始人都是策展的支持者，包括 Dapper Labs 和 NFT 初创公司 Manifold 的创始人。
