儿童数据担心在 Fortra 勒索软件攻击中被盗| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/28/children-data-fortra-ransomware/

Fortra 的大规模勒索软件攻击的余波继续扩大，黑客声称有了新的受害者:一家儿童虚拟精神健康护理初创公司。

在一份提交给缅因州司法部长办公室的数据泄露文件中，美国医疗保健巨头加州蓝盾公司证实，其供应商之一 Brightline 的 GoAnywhere 文件传输工具中存储的数据被盗。

为儿童提供虚拟教练和治疗的 Brightline 上周被 TechCrunch 认定为大规模违规事件的可能受害者。

泄露通知证实，黑客——可能是与俄罗斯有关联的 Clop 勒索软件团伙，声称通过使用一个未披露的安全漏洞，入侵了一百多个组织——访问并可能泄露了超过 63，000 名患者的个人数据。

Clop 的黑暗网络泄露网站称，该团伙将“很快”泄露从 Brightline 窃取的数据，除非支付赎金，否则该团伙将利用该网站发布被盗文件。

Brightline 尚未在其网站或社交媒体渠道上公开承认违规行为。Brightline 发言人约翰·奥康纳(John O'Connor)拒绝回答 TechCrunch 的问题，但并不否认此次违规影响了 63，000 名个人。目前还不清楚有多少 Brightline 的儿童用户受到了影响。

在其违规通知中，蓝盾表示，受影响的违规包括患者姓名、地址、出生日期、性别、蓝盾用户 id 号、电话号码、电子邮件地址、计划名称和计划组编号。

据说 Brightline 是受到 Clop 集团打击的 130 个组织之一，但不是唯一的医疗保健公司。提供企业健康和福利项目的 US Wellness 也证实，黑客已经访问了其用户的个人数据，包括姓名、地址、出生日期和会员 ID 号。

Fortra 漏洞对医疗保健组织的影响如此广泛，以至于它促使美国政府的卫生部门网络安全协调中心(HC3)在 2 月份发布了一个警告，以帮助防范 Clop 的攻击。

除了医疗保健机构，该组织不断增加的已知受害者名单包括多伦多市、加拿大融资巨头魁北克投资公司和维珍红色。

Virgin Red 的发言人朱迪·伯顿(Jodie Burton)告诉 TechCrunch，在与 Clop 联系后，他们得知攻击者“通过对我们的供应商 GoAnywhere 进行网络攻击，非法获得了一些 Virgin Red 文件”。TechCrunch 从其他受害者那里听说，他们也是在收到赎金要求后才得知数据被盗——尽管 Fortra 向他们保证他们的数据是安全的。

Fortra 发言人 Mike Devine 和 Rachel Woodford 一再拒绝回答 TechCrunch 的问题。当 TechCrunch 联系到代表危机沟通公司 Prosek 代表 Brightline 的安妮·哈特时，她拒绝置评。
