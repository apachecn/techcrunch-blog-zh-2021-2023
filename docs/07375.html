加拿大关闭其 COVID Alert 应用程序

原文：https://web.archive.org/web/https://techcrunch.com/2022/06/17/canada-sunsets-its-covid-alert-app-based-on-the-ios-and-android-exposure-notification-api/

加拿大关闭了基于 iOS 和 Android 暴露通知 API 的 COVID Alert 应用程序

加拿大政府今天宣布将停止使用基于谷歌和苹果联合开发的新冠肺炎暴露警报 API 的应用程序，作为帮助对抗疾病传播的措施。根据政府的警告，该系统立即被禁用，并建议用户从他们的设备上删除 COVID Alert 应用程序。

至于为什么要终止该计划，该发布表示，这与该国 PCR 检测的大幅下降有关，这导致很少向患者发放一次性密钥供其在应用中使用，因此使用量显然已经放缓至涓涓细流。

加拿大于 2020 年 7 月实施了 COVID Alert 应用程序，加拿大卫生部表示，自那以来，该应用程序已被 690 万人下载，并代表 5.7 万名检测呈阳性的人提供了暴露通知，并在应用程序中输入了他们的一次性密钥。

COVID Alert 应用程序部分由 Shopify 的工程师开发，由黑莓进行安全审查，并与省级和联邦政府资源合作。

与此同时，在安大略省，卫生系统正面临创纪录的需求——主要是由于 COVID 的附带影响，包括员工中断，而不是 COVID 病例本身。来自安大略省的废水数据(感染信息的最佳来源，因为大多数大规模 PCR 测试项目已经暂停)也表明在过去几周该省的 COVID 病例有所增加。
