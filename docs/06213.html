科技公司将如何应对没有办公室的未来？

科技公司办公室的会变成什么样子，如果没有传统上为我们提供的在同一栋大楼里一起工作的结构，公司将如何运作？这是当今科技公司面临的一个重大问题，因为它们正在努力确定自己在后大流行时代的工作方法。

当然，像 git lab 这样完全远程化的公司在 T4 已经存在一段时间了，但是在疫情之前的传统观点是，你需要在同一个地方完成重要的工作。

对于大型科技公司来说，这当然是事实。Salesforce、微软、谷歌、Meta、亚马逊和苹果在世界各地建造庞大的校园或摩天大楼，并不是为了毫无理由地放弃它们。他们建造这些房子是为了给工人提供住房，并炫耀他们纯粹的经济实力。但是当 2020 年 3 月疫情来袭时，它改变了我们对工作的看法，可能是永远的改变。

突然间，我们有了一个巨大的实验室来进行实验，每个人都在家工作，虽然肯定会有一些问题，但这取决于你的业务、你的工作，坦率地说，取决于你的生活状况，它表明所有类别的工人都不需要坐在一栋大楼内的小农场里完成他们的工作——当然不是一周五天。

我与科技界的各种人交谈过，从顾问和投资者到初创公司的创始人，试图准确把握下一阶段的工作将会是什么样子，毫无疑问，当涉及到在办公室见面的时候，科技公司至少变得更加灵活了。

公司政策在不断演变

这里没有放之四海而皆准的答案，但有足够多的例子表明，即使是老牌公司，也应该从传统的办公室设置中进行一些转变。
