  • 关于谁“拥有”web3: 杰克认为普通用户不是 web3 项目的所有者。相反，这位首席执行官声称，风险资本家，反过来，他们的投资者。更重要的是，杰克不认为这种动态会改变；相反，他预计，由于风险激励，支持过多加密项目的集中资本池将保持相当大的控制权。

上述问题的关键在于，最初的加密货币比特币的知名支持者杰克并不热衷于 web3 项目。

值得注意的是，这两个角色突然出现，反对过去几年在去中心化经济中出现的贪婪浪潮，每个传统的金钱都想获得假定的 web3 回报。

谁拥有什么？

杰克抱怨的核心是，领先的网络 3 公司由风险资本家所有，因此将不得不接受风险激励。

“风险激励”当然是一种礼貌的说法以这样一种方式经营公司，风险资本家和他们的支持者赚了很多钱。

这种抱怨是相当真实的。仅在 2021 年第三季度，风险投资家就向的加密项目投资了超过 60 亿美元。这应该让你对目前传统的私人市场投资者购买的密码世界的规模有所了解。更简单地说，养老基金拥有大量的秘密经济。

杰克对风险投资家和 web3 的诋毁不止于此。杰克不仅说 web3 项目由于其前身与外部资本达成的经济交易而在核心上有弱点，而且还说它们是集中的。

除了称某人为“无产阶级”，或者“保持贫穷”或“ngmi”，说一个加密项目是集中的可能是最粗鲁的评论。

杰克错了吗？不，他不是。

的确，风险投资者正在用现金和援助涌入 web3 领域，因为他们认为这会让他们赚钱。这导致了相当多的集中化。例如，我觉得非常有趣的是，a16z 同时是比特币基地和 OpenSea 的投资者。比特币基地打算参与 NFT 的游戏。你知道，开放海洋目前的规则是。

更有甚者，像 Solana 这样的项目以一首歌的价格向风险投资者出售了一批代币，有效地提供了已经膨胀的账户中永远冻结的上升空间。集权？所有权的和收益，在这种情况下。

集中化与最初有点朋克的比特币风气是对立的。据我所知，Web3 的支持者完全满足于使用外部资金来推动他们新的基于令牌的项目，以此来加速他们所预见的未来。我想，像拐杖一样平。

但比特币不需要在一年内筹集三轮风险投资来实现增长。它只是确定了一个想法，这个想法仍然引起人们的共鸣。一些值得思考的事情。

现在我们到了一个非常大的耶，但是。

比特币是宗教；web3 是贪婪

虽然同意某人的加密观点很好，但我不同意杰克的所有观点。

比如我不认同比特币没有中心化。在某种程度上，我认为没有多少人真的吸毒。一项新的研究发现，本周“大约 0.01%的比特币持有者控制了目前流通的 1900 万比特币中的 27%”，根据 Baystreet 的。

这就是为什么当我考虑加密世界中两个敌对的宗教阵营时——认为最初的加密项目是去中心化的耶稣和 web 3“GM”团队的比特币最大主义者——很明显他们是相当不同的。

比特币 maxis】宗教在他们的信仰中，OG crypto 钉死了它，并且连续的项目都是废话。web3 是贪婪，一种将一切在线金融化的方式，允许传统投资者在分散的环境中拥有收费站和税务员。

因此，虽然我非常满意地赞同杰克搅动网络 3 锅，但我不同意他的总体哲学。尽管如此，杰克还是设法指出，一群裸体的想要成为皇帝的人实际上是天生的。这是开始一天的好方法。
