远程工作和事件初创公司 twine 收购 YC 支持的 Glimpse，在 Zoom TechCrunch 上发布

原文：https://web.archive.org/web/https://techcrunch.com/2022/02/10/remote-work-and-events-startup-twine-acquires-yc-backed-glimpse-to-launch-on-zoom/

为虚拟活动和远程团队提供网络工具的公司 Twine ，由于刚刚完成对 Y Combinator 支持的初创公司Glimpse的收购，将很快将其服务引入 Zoom，该公司开发了一个为虚拟活动设计的“速度匹配”平台。Glimpse 的想法是提供一种方式来促进通常在现实世界活动中发生的联系，并通过使用人工智能在视频聊天中匹配参与者来将他们带到网上。最近，Glimpse 一直在测试一种新的集成，这种集成将允许活动主办方为他们的 Zoom 会议、网络研讨会和活动添加速度网络。

这种集成由 Zoom 的新“分支房间”API 提供支持，Glimpse 和少数其他公司已经提前使用了该 API。虽然两家公司都在类似的领域工作，致力于远程连接人们，但 Glimpse 的 Zoom 集成使他们在产品开发方面领先于 twine。此外，twine 的联合创始人兼首席执行官劳伦斯·科伯恩承认，他的公司甚至失去了一些交易。

通过此次收购，Glimpse 的技术将可用于 twine 的客户群，包括其扩展到更广泛的 Zoom 用户群的计划。

在接下来的几周内，一小组使用 Zoom 新的分会场 API 构建的应用程序将被添加到其应用程序商店 Zoom App Marketplace 中，该商店目前拥有数十个应用程序，这些应用程序要么旨在在 Zoom 客户端内工作，要么以其他方式扩展其功能。即将推出的“twine for Zoom”产品将是其中之一，它让客户可以使用匹配工具、网络和虚拟水冷工具，这些工具不仅可以用于虚拟活动，还可以用于其他类型的会议，如公司社交活动、全体会议、新员工入职会议、社区会议等。

“长期以来，我们一直钦佩远道而来的 Glimpse 团队和产品，我们很高兴能与他们合作，”Coburn 说。“他们在 Zoom 生态系统中成功构建的东西非常了不起，对远程团队和虚拟活动产生了改变游戏规则的影响。”

虽然是一家相对年轻的公司，收入很少，但 Glimpse 已经发展到 150 家客户，还有 700 多家企业对使用其平台感兴趣。这些公司从教育技术公司到风险投资公司，甚至是企业客户。后者对 twine 最有吸引力，twine 已经有更大的公司使用它的工具，例如亚马逊、微软和易贝。

Zoom Apps & Integrations 产品负责人 Ross Mayfield 在一份声明中表示:“Glimpse 是一家高度创新的公司利用 Zoom 应用市场提升客户体验的绝佳范例。“我期待着看到麻线团队将 Zoom 的麻线推向市场，”他补充道。

Glimpse 参与了创业加速器 Y Combinator 的 2020 年冬季项目，并获得了 YC 和 Maven Ventures 的种子期投资。其联合创始人海伦娜·默克(Helena Merk)和布莱恩·李(Brian Li)将继续留任，以便在过渡期间与 twine 合作。然而，它的三名员工团队正在加入 twine，该团队现在有 16 名全职员工。收购条款尚未披露，因为鉴于两家公司的早期性质，这是一个小的退出。然而，我们知道这是一个七位数范围内的全股票交易。
