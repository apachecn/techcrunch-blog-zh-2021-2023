针对拥有 1200 万用户的电话系统客户，出现了新的供应链攻击| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2023/03/30/theres-a-new-supply-chain-attack-targeting-customers-of-a-phone-system-with-12-million-users/

有一种新的供应链攻击，目标是拥有 1200 万用户的电话系统的客户

多家安全公司就 发出了警报，这是一种活跃的供应链攻击，它使用 3CX 的 的木马版本，广泛使用语音和视频通话客户端 来瞄准下游客户。

3CX 是一款基于软件的电话系统的开发商，该系统被全球超过 60 万家机构所使用，包括宝马、麦当劳和英国国家医疗服务系统。该公司声称在全球拥有超过 1200 万的日常用户。

来自网络安全公司 CrowdStrike、Sophos 和 SentinelOne 的研究人员周三发布了博客帖子，详细描述了一次网络安全管理软件产品式攻击——sentine lone 将其称为“平滑操作员”——该攻击涉及交付被木马化的 3CXDesktopApp 安装程序，以在企业网络内安装 info stealter恶意软件。

这种恶意软件能够从 谷歌 Chrome 、微软 Edge、Brave 和 Firefox 用户档案中收集系统信息并窃取数据和存储的凭证。根据 CrowdStrike 的说法，其他观察到的恶意活动包括向演员控制的基础设施发送信标，部署第二级有效载荷，以及在少数情况下的“手动键盘活动”。

安全研究人员报告称，攻击者的目标是 Windows 和 macOS 版本的受损 VoIP 应用程序。目前，Linux、iOS 和 Android 版本似乎没有受到影响。

SentinelOne 的研究人员表示，他们在 3 月 22 日首次发现了恶意活动的迹象，并立即调查了这些异常，结果发现一些组织正试图安装 3CX 桌面应用程序的木马版本，该版本已使用有效的数字证书进行了签名。 苹果安全专家帕特里克·沃德尔(Patrick Wardle)也 发现 苹果对恶意软件进行了公证，这意味着该公司对其进行了恶意软件检查，一个也没有检测到。

3CX CISO Pierre Jourdan 周四对表示，该公司意识到一个“安全问题”影响了其 Windows 和 MacBook 应用程序。

Jourdan 指出，这似乎是一次“来自高级持续威胁的有针对性的攻击，甚至可能是国家支持的”黑客。CrowdStrike 暗示，臭名昭著的拉扎勒斯集团的子集团朝鲜威胁演员迷宫·千里马是供应链攻击的幕后黑手。

作为一种变通办法，3CX 公司敦促其客户卸载该应用程序并重新安装，或者使用其 PWA 客户端。乔丹说:“与此同时，我们对所发生的一切深表歉意，我们将尽我们所能弥补这个错误。”。

关于 3CX 供应链攻击，我们还有很多事情不知道，包括有多少组织可能已经受到威胁。根据测绘互联网连接设备的网站 Shodan.io 的数据，目前有超过 24 万个公开曝光的 3CX 电话管理系统。
