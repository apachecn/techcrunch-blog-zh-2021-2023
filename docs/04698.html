永远的触控 ID，永远的 Face ID

原文：https://web.archive.org/web/https://techcrunch.com/2022/03/09/touch-id-forever-face-id-never/

感谢上帝，苹果公司已经屈尊继续向其遍布全球的豪华娱乐场提供廉价票:iPhone SE 3。如果他们停止生产这个有用的产品，我将不得不换成 iPad 或者回到 Android，因为唯一的替代品既有一个丑陋的缺口又有它包含的 Face ID。幸运的是，新 SE 应该会让我们这些通情达理的人坚持到底，直到苹果重新考虑其在触控 ID 的立场，就像它在 USB-A 和 SD 端口上所做的那样。

为什么我和其他明眼人会选择 se？首先，这是你能买到的最便宜的 iPhone。对一些人来说，争论就这样开始和结束了。对其他人来说，触控 ID 是决定性因素。但是也有一个合理的问题，当你付出更多的时候，你得到的是否真的值得。

更大的屏幕？当然，但是对一些人来说那是一种拖累。SE 可能不像它成为苹果有史以来最好的手机时那样便于携带，但与市面上的许多手机相比，它相当纤薄。“全屏”的想法也从未吸引过我。所以屏幕有圆角——按钮应该在的地方——然后从顶部伸出一块？不用了，谢谢。

更多更好的相机？嗯，我不想在这里提出“640 千字节应该足够了”之类的论点(这是为 90 年代的孩子准备的)，但实际上……我们已经在相机质量方面达到了一个平台期，你可以从 SE 的相机中获得的东西(我指的是最后一个)已经足够好，足以完成我们通常对这些图像做的所有事情。

实话实说:大多数人唯一一次看到比手掌大的 iPhone 图片是在苹果的活动上。当然，iPhone 13 Pro 中的摄像头更好。但所有这些都将上传到 Instagram，在 5 英寸的屏幕上，从两英尺外观看 3/4 秒。如果你想从手机摄像头中获得好的效果，你可以在过去几年的任何手机上实现。

还有什么…计算能力？你在手机上做很多视频编辑和特效工作吗？还是主要是信息、社交媒体和一些可能在高端图形计算器上运行的休闲游戏？虽然有足够的资源在应用程序之间快速切换，并且不会被大型网络服务所拖累是很好的，但芯片很久以前就把这个里程碑留在了尘埃中。

想想苹果一直在说它的新处理器，A11，A12，A13 Bionic 等等。，每次他们出来。快两倍！快四倍！快六倍！你知道，这些都是累积的——到目前为止，我们的手机应该比几年前快一千倍。他们会感觉快一千倍吗？不，因为人们在电话上积极做的事情不需要 T4 快一千倍。自然，有一些不太明显的过程，如由专业芯片实现的计算摄影和语言引擎，但性能多年来在手机上并不是一个问题。

然后是 Face ID。听着，我知道有些人喜欢这样。但是有很多人不知道。一部分是令人毛骨悚然的因素——你的脸，真的吗？我想我们在这方面已经过时了。但它的很大一部分是实用的——有太多太多的情况下，我宁愿用我的拇指或我的其他几个手指中的一个来解锁我的手机，而不是拿起它盯着它。至少对我来说，相反的情况很少发生。

如果解锁机制是一个永远在线的面部扫描仪，那就不是你能控制的了。如果你不想让激活怎么办？将手指放在扫描仪上是一个深思熟虑的选择，表明用户想要解锁手机的意图。我可以把它放在口袋里，或者放在桌子上，用任何一只手。这很简单——很有效。拿着手机看着它是大多数人半天都要做的事情——用户的意图在哪里？

更不用说有一个按钮和一些安全挡板的设计选择是一个很好的选择。自 iPhone 早期以来，最大的吸引力就是 home 键带来的简单性。无论你在做什么，无论你是在打电话，玩游戏，还是在处理工作文件…按下那个按钮一次，它就消失了，你到家了。有个按钮是个好的东西。有一个不是由触摸屏制成的可以抓握的角落是一件好事情。

我从来没有特别喜欢过 iPhone 6-8 的设计；太像软糖了。不过比那些新的好。如果有触控 ID 的话，我会改用旧的 5S 设计，甚至是无槽 13 迷你外形。当它第一次被介绍时，我们有我们的怀疑，但是它很快成为具有安全和方便的最好平衡的方法。

我真的不关心苹果在过去几年中增加的几乎任何功能。我知道有很多人会不同意我的观点，幸运的是，新手机对他们来说很棒。但是只要这是一个选择，我会选择和触控 ID 的传统风格。

当然，我知道以上所有内容看起来都有点生硬。没关系。我喜欢面包皮——你也可以。加入我，让我们一起对着云大喊！
