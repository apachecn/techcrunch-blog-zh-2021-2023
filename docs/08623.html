人类安全与 PerimeterX 相结合，以阻止机器人和自动化欺诈 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/07/27/human-security-merges-with-perimeterx-to-thwart-bots-and-automated-fraud/?&web_view=true

面向企业的 bot 缓解和欺诈检测平台 Human Security 正与专注于保护网络应用免受账户盗用和自动欺诈的公司perimerx合并。交易条款没有披露。

合并后的实体将包括近 500 名员工，一旦两个各自的平台整合，它最终将归入人类安全的名下——尽管何时整合尚不清楚。

“虽然我不能提供一个确切的时间表，但我可以说我们有两个优秀的团队在一起，我们已经一起做了令人难以置信的大量工作，”Human 联合创始人兼首席执行官塔梅尔·哈桑告诉 TechCrunch。“我们正在尽一切努力使集成尽可能地快速和顺畅，以便我们的联合客户真正获得两种产品的最佳优势。”

作为交易的一部分，Human 表示，它已经从黑石信贷获得了 1 亿美元的债务融资，这是继 7 个月前的 1 亿美元增长轮融资后的第六轮融资。与此同时，perimerx去年筹集了 5700 万美元的融资。

机器人的攻击

两家公司一直在着手解决的问题是，大量的在线安全和欺诈事件是由机器人驱动的——这些机器人多年来变得越来越复杂，因为它们调整了自己的行为，变得更加难以检测。邪恶的活动包括抓取内容，如来自网站和应用程序的个人身份信息(PII)，或所谓的“点击欺诈”，实质上是机器人利用点击付费(PPC)广告来无组织地增加网站的收入。

Human Security ( 以前称为 White Ops )和 PerimeterX 通过使用机器学习来理解真实的“人类”行为，帮助检测机器人驱动的异常行为，从而解决了机器人攻击问题。然而，它们并不是完全相同的产品，每个产品都专注于不同的领域，例如，PerimeterX 在帐户保护方面非常重要，而它们瞄准的是不同的领域。

“PerimeterX 主要保护了电子商务公司，而 Human 则保护了广告技术、绩效营销和网络安全/应用安全团队，”哈桑说。“我们的共同目标是联合这些孤岛，解决整个企业的这个重大问题。”

网络安全初创公司在 2021 年度过了辉煌的一年，吸引了创纪录的 295 亿美元资金——比前一年的数字增加了一倍多。随着勒索软件攻击和数据泄露激增，对这种技术的需求比以往任何时候都更大，而远程工作的激增继续快速发展——但是网络安全初创公司并非不受经济衰退的影响在过去几个月中，整个行业宣布大量裁员。

合并

鉴于攻击和违规行为不会很快消失，网络安全公司通常处于长期繁荣的有利地位，随着科技估值暴跌和巨大的“重置”继续影响各种规模的公司，通过并购进行整合(M & A) 是应对这种的一种策略。

在游戏领域，我们最近看到 Unity 与 IronSource 合并，而在航天领域，Eutelsat 和 OneWeb 昨天宣布计划将合并为一家单一的欧洲卫星公司。特别是在网络安全领域，今年有无数的并购交易取得了成果，涉及大型科技公司，如 IBM 和谷歌以及寻求快速扩张的小型风险投资公司。

Human Security 于 2012 年在纽约成立，实际上于 2020 年被高盛的商业银行部门 ClearSky Security 和 NightDragon 收购，因此它可能不像其他网络安全初创公司那样受到经济衰退的影响。但是，通过与该领域的竞争对手合并，它可以以更有机的方式难以实现的速度增长。

“通过合并这些公司，并增加多个垂直行业的联合数据量，我们有了一个无与伦比的来源来改善我们对所有联合产品的整体检测，以及为帮助保护我们的客户而提供的新产品奠定了坚实的基础，”哈桑说。
