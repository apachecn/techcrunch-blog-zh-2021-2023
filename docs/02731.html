最新的苹果操作系统测试版为 Face ID 提供了一个面具和通用控制技术

最新的苹果操作系统测试版为 Face ID 提供了一个面具和通用控制

苹果刚刚推出了一批新的操作系统测试版，带来了一对期待已久的功能。IOS Beta 15.4 提供了戴着面具用 Face ID 解锁手机的功能(真的是上 Face ID，如果我们对此感到不安的话)，而 iPadOS 和 MacOS 12.3 的 Beta 15.4 版本提供了延迟的功能，通用控制。

由于显而易见的原因，自《疫情》开播以来，戴着面具的 Face ID 一直是广受欢迎的一部电影。2020 年 4 月，首次报道了一项功能，当手机发现面具时，用户将进入密码输入屏幕，去年 2 月，iOS 14.5 的开发者测试版带来了通过连接的 Apple Watch 解锁手机的功能——这是一种巧妙的变通方法。

新的测试版操作系统终于提供了人们一直要求的功能。一旦启用，手机将使用眼睛周围的特征来确定是否是正确的用户。然而，如果没有整张脸，阅读就不太准确，并且潜在地不太安全，这就是为什么它是选择加入的。

图片来源:苹果

在 iPad/MacOS 方面，该公司终于推出了通用控制测试版。该功能与 macOS 12 Monterey 一起宣布，预计在去年推出，但此后出现了明显的延迟。目前预计它将在春天发布，尽管开发测试人员现在就可以得到它。

该功能是边车的一种替代品。然而，该功能将 iPad 变成了一个额外的显示器，Universal Control 将自动在 Mac 和 iPad 之间共享鼠标光标和键盘。用户还可以在一台设备上获取内容，并在不同的机器之间拖放。

iOS 还新增了一些表情符号，包括心形手、咬嘴唇和怀孕的人/男人。测试版从今天开始提供。目前还没有关于最终消费者可用性的消息。
