在过去一年的产品更新浪潮中，Twitter 正在缓慢地继续增强其直接消息传递界面。该公司今天宣布，它现在将使用户能够将六个对话钉在他们的 DM 收件箱的顶部，以便于访问。该功能可在 iOS、Android 和 web 上使用。

去年，Twitter 对 DMs 做了一些期待已久的改变，包括在个人对话中一次向多人发布一条推文的能力。Twitter 不再用日期和时间给单个消息打上时间戳，而是开始按天对消息进行分组，以“减少时间戳混乱

如果 Twitter 在接下来的几个月里推出更多新的 DM 功能，这并不令人震惊。Twitter 在 2021 年的最后几个月收购了两家消息公司:总部位于伦敦的群聊应用程序 Sphere ，以及潜在的 Slack 竞争对手 Quill 。Sphere 和 Quill 的员工都加入了 Twitter，前 Quill 员工专门负责消息传递。Quill 创始人 Ludwig Pettersson 是 Stripe 的前创意总监，他加入 Twitter，担任 Conversations 团队的项目经理。
