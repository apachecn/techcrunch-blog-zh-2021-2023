微软在 preview | TechCrunch 中用 DALL-E 2 扩展 Azure OpenAI 服务

原文：https://web.archive.org/web/https://techcrunch.com/2022/10/12/microsoft-expands-azure-openai-service-with-dall-e-2-in-preview/

微软在预览版中用 DALL-E 2 扩展 Azure OpenAI 服务

当 Azure OpenAI 服务于 2021 年推出时，该服务——Azure 认知服务的一部分——通过 Azure 平台为语言翻译和文本自动完成等应用程序提供了对 open ai API 的企业定制访问。这是不会改变的。但是在五月份通过 微调功能扩展了服务之后，微软今天为精选 Azure OpenAI 服务客户引入了只有邀请才能访问的 DALL-E 2。

客户可以使用 DALL-E 2 生成使用文本或图像的自定义图像。根据消费者 DALL-E 2 服务，除了在现有图像上生成变化的功能之外，他们还可以利用内涂和外涂功能，分别在图像的一部分内生成新内容或推动图像超出其原始范围。

DALL-E 2 通过 Azure OpenAI 服务定制的播客内容。图片来源:微软

早期采用者包括像美泰这样的品牌，他们使用 DALL-E 2 提出了一种新的风火轮模型车的想法。另一个试点客户，德国媒体集团 RTL 德国，正在考虑将流媒体内容元数据与 DALL-E 2 结合起来，为播客剧集和有声读物中的场景生成视觉效果。

为了防止误用，就像 Designer 和 Image Creator 一样，微软表示，它已经实施了过滤器，以拒绝来自 Azure OpenAI 服务客户的违反内容政策的 DALL-E 2 提示。该公司还声称，它的集成技术可以防止 DALL-E 2 创建宗教物品和名人的图像，以及通常用于欺骗系统生成性或暴力内容的物品。微软表示，它增加了一些模型，可以删除人工智能生成的图像，这些图像似乎包含成人、血腥和其他类型的“不适当”内容。

使用 DALL-E 2 的美泰一代。图片来源:微软

微软的一位发言人通过电子邮件告诉 TechCrunch:“微软正在让只有受邀者才能访问，以便让我们有机会与客户合作，并在客户将他们的应用程序投入生产时，建立保护措施来防止有害的使用和不想要的结果。”“与这些早期客户的合作将有助于我们确保负责任的人工智能保护措施在实践中发挥作用。”

除了 DALL-E 2，微软还对 Azure OpenAI 服务自大约一年前推出以来的增长进行了总体更新。该公司表示，使用该服务的公司现在横跨金融服务、保险和医疗保健等行业，包括埃森哲、埃维诺、欧特克、宝马集团、CarMax、EY 和普华永道等品牌。一些最常见的用例包括书写辅助、自然语言到代码的生成和解析数据以产生见解。例如，普华永道正在利用 Azure OpenAI 服务将各种新闻文章分类为环境、社会和治理主题，以进行基准测试，而 CarMax 正在使用该服务根据客户评论生成新的营销内容。
