Namecoach 筹集资金教用户如何正确发音 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/11/17/namecoach-raises-cash-to-teach-users-how-to-correctly-pronounce-names/

我们都面临着一个难以发音的名字。但是并不是每个人都考虑到他们发音错误的后果。在为《快公司》撰写的一篇文章中，Carta 的包容、公平和影响力负责人 Madhumita Mallick 回忆了她的名字是如何成为焦虑的来源的，甚至在她还是小学生的时候。

研究确实发现，当同龄人和老师错误地发音或更改有色人种学生的名字时，这些学生在课堂上参与较少，在社交上变得孤僻，以避免与他们的名字联系在一起。正如一项研究的共同作者在 2012 年写的:“在 K-12 学校经历的行为和态度(学生)凸显了一种文化‘他者’的类型，这与我们对多元文化学校环境的目标相矛盾……即使只是偶然发现一个他们从未见过的名字，老师对学生名字设定的基调也是有意义的。”

Praveen Shanbhag，Namecoach 的首席执行官，在他姐姐的大学毕业典礼上，听到她母校的一个演讲者念错了她的名字。这一点，加上他作为第一代美国移民的经历，激发他共同创立了 Namecoach，该公司开发可以嵌入 Salesforce、Canvas 和 Gmail 等现有平台的姓名发音工具。

Shanbhag 通过电子邮件告诉 TechCrunch:“我们的使命是让每个人都不再念错名字。”

Shanbhag 在斯坦福攻读哲学博士时开始涉足编程。他决定编写一个应用程序，收集学生说出自己名字的录音，并在毕业典礼上发给姓名阅读者，这就是后来的 Namecoach。Shanbhag 说，到 2016 年，已有数百所学校在使用 Namecoach 的软件和服务。

name coach——最终将其客户群扩大到客户品牌——在市场上并不孤单。name ranges 提供了一套强大的名字发音工具，脸书、Slack 和 LinkedIn 也是如此。但 Shanbhag 告诉我，在早期，他试图通过大量投资人工智能和集成来区分 Namecoach。

图片来源:名称教练

例如，Namecoach 使用人工智能系统，根据国籍、种族、性别和地点等因素，预测某人的名字有多个正确发音时的正确发音。该平台的另一个系统在没有姓名录音的情况下合成语音。

在高层次上，Namecoach 显示用户生成的和从音频姓名发音数据库中提取的姓名发音。当被问及系统的准确性以及用户是否可以纠正可能进入数据库的错误时，Shanbhag 说，Namecoach 会咨询语言学专家，并提供一份更正的提交表格。

它还没有上线，但 Shanbhag 说，Namecoach 正在开发一种人工智能，以提供考虑说话者母语的发音，而不仅仅是姓名所有者的语言。“你的名字是你身份的核心，准确的发音为双方的积极互动定下了基调，”他说。

Namecoach 的平台可以与包括 Microsoft Teams、Outlook 和 Google Workspace 在内的服务一起开箱即用，并提供 API 和软件开发工具包，让第三方将 Namecoach 的功能构建到他们的产品中。但 Shanbhag 表示，未来几个月的重点将是这家初创公司的第一款消费者应用，该应用将于明年作为 Chrome 的扩展推出。

name coach——今天完成了一轮 800 万美元的首轮融资，由 Impact America Fund 领投，Authentic Ventures、Metaplanet、Engage 参与。风投、创始人基金(Founders Fund)、比斯克风险投资公司(Bernstein Ventures)等也计划加大销售和营销力度，并扩大该平台的连接器集合。Shanbhag 暗示了除发音指导之外的功能正在开发中，包括在销售场景中启用热情互动的提示，以及帮助用户更普遍地改善对话的反馈。

考虑到 Namecoach 的大客户——sales force、NetJets 和普华永道，学习销售应用程序是有意义的。除了这三个人，这家 30 人的初创公司声称在全球拥有 300 多个教育和企业客户。

Shanbhag 表示:“首轮融资使我们能够加快实现我们的目标，即将新颖的语音技术集成到每个通信工作流程中，以解决任何语音交互过程中各种使用情况下的姓名发音问题。”当被问及收入数字时，他不愿透露。“Namecoach 的客户涵盖非常广泛的垂直行业、使用案例和组织规模，这意味着我们不依赖任何细分市场来在低迷时期蓬勃发展。”

迄今为止，总部位于帕洛阿尔托的 Namecoach 已经筹集了 1500 万美元的风险投资。
