亚马逊 Sidewalk 增加了新的合作伙伴，计划很快向开发者开放

原文：https://web.archive.org/web/https://techcrunch.com/2023/01/05/amazon-sidewalk-adds-new-partners-plans-to-open-to-developers-soon/

在今天的 CES 上，亚马逊宣布来自四家制造商的大量新设备将很快加入其人行道网络。亚马逊的长距离、低带宽物联网网状网络 Sidewalk 通过共享该公司的 Echo 扬声器和环形摄像机等设备的一小部分用户带宽来提供动力，目前支持瓷砖跟踪器、亚马逊自己的环形摄像机和传感器，以及 Level 智能锁和 CareBand 的老年人护理系统。现在，你可以将 Browan 和 New Cosmos ，Meshify 和 Deviceroy 的 Aria 的传感器添加到这个列表中，尽管目前只有将太阳能逆变器连接到互联网的 Deviceroy 系统可用，其余的将在今年晚些时候推出。

在很大程度上，这四位新合伙人并不是家喻户晓的名字。然而，亚马逊的总经理兼 Sidewalk 首席技术官塔努杰·莫汉告诉我，今年将会有许多新的合作伙伴上线。更重要的是，他还表示，亚马逊计划在今年上半年向开发者开放 Sidewalk 网络。他指出，这将使几乎任何想要开始构建物联网产品的人都可以从亚马逊的合作伙伴那里订购参考工具包，并在几天内开始使用，而不必担心连接性。

“任何有想法的人都应该能够访问 AWS 网站，从 Silicon Labs、TI 或其他人那里找到一个硬件套件，订购该套件，并能够通过 AWS 将数据传输到应用程序中，”他解释道。“一拿到工具包，他们就可以开始逐字书写了。所以这就是我们对开发者开放的期望。我的愿景是，利用其中的一些套件和设备，他们实际上可以尝试构建一些真实的东西，甚至可能以少量的概念证明来证明他们的商业案例，这在以前是不可能的。”

正如 Mohan 所指出的，Sidewalk 团队面临的主要挑战之一是让人们改变他们对物联网连接的看法。当我问他他的团队面临的最大挑战是什么时，他说:“市场并没有完全理解 Sidewalk 的与众不同之处。”。“他们听说过，哦，物质解决一切，或者 Zigbee 解决一切，或者我们永远拥有这个。但也不尽然。你还没有一个现成的网络和一个送到你家的设备，你可以打开它。是的，也许带 SIM 卡的手机可以做到这一点。但没别的了。”

他指出，一些制造商可能已经开始使用 Zigbee 或 Wi-Fi 来为水龙头等日常设备添加智能，但后来发现人们没有 Zigbee 集线器或只是不会配置它。“这是一项不值得的投资，”他说。“如果你看看这些被强迫变聪明的聪明事物的百分比，就会发现它们中很低的百分比曾经被联系起来。”

理想情况下，这不是 Sidewalk 的问题，虽然 Matter 正在试图解决其中的一些问题，但 Mohan 认为 Sidewalk 实际上可能有助于 Matter 的发展，因为它可以为 Matter 网络提供初始网络功能，以允许设置新设备。
