看起来性科技创业公司劳拉·迪卡洛完蛋了

被列入消费电子展黑名单后，在 2019 年成为头条新闻的性科技初创公司 Lora DiCarlo 似乎已经关闭。该公司的网站已经离线，据报道订单已经几个月没有完成。

TechCrunch 已经向这位同名创始人求证，但这看起来肯定是一家短暂有前途的高科技性玩具企业的末日了。

Lora DiCarlo 成立于 2017 年，是由女性领导的新一波科技前沿性健康公司之一。正如我们的作者当时所说，它在 CES 2019 上获得了创新奖，“一种免提设备，它使用仿生和机器人技术，通过同时刺激 g 点和阴蒂来帮助女性实现混合高潮。”

但是，负责 CES 的消费者技术协会撤销了该奖项，并禁止该公司在展会上展出。他们当时的解释是，该公司及其设备都“不符合产品类别”。

可以预见的是，这立即招致了强烈的反对和性别歧视、假正经和普遍判断失误的指控。每个人都站在劳拉·迪卡洛一边，公众的关注是无价的，她后来在 Disrupt 上告诉 TechCrunch:“我认为他们实际上帮了我们一个大忙。”该公司在那段时间筹集了 200 万美元，在五年的运营中总共筹集了约 900 万美元。

但是，尽管 2020 年该节目将大规模回归(当然还有令人垂涎的 TC+功能，该公司似乎在疫情期间步履蹒跚——也许是老牌硬件制造商遇到的同样的芯片短缺和制造问题的受害者。

正如《妇女健康》所记载的，过去的几个月似乎是劳拉·迪卡洛的最后几个月，因为一个正常运作的商业企业的各个方面都开始失败:订单没有了，零售合作伙伴的库存没有了，员工也离开了。该网站本月早些时候关闭了，现在仍在关闭。尽管还没有任何官方声明，但该公司确实似乎已经破产了。

这太糟糕了，但如果没有疫情奖和性玩具的耻辱会增加阻力，作为一家硬件初创公司取得成功就已经够难了。如果我们收到迪卡洛的回复，我们会更新这篇文章。
