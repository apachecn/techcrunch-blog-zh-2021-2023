Deepki 获得 1.66 亿美元

原文：https://web.archive.org/web/https://techcrunch.com/2022/03/31/deepki-grabs-166-million-to-help-real-estate-investors-reduce-carbon-emissions/

法国初创公司 Deepki 已经筹集了 1.66 亿美元的 C 轮融资(€1.5 亿美元)。该公司正在为房地产行业构建一个软件即服务平台。这是一款监控和分析产品，可帮助您做出更好的决策。

使用 Deepki 的公司可以跟踪和分析 ESG(环境、社会和治理)标准，从其房地产投资组合产生的碳排放开始。

你不能减少你不能测量的东西。由于房地产投资者希望降低新项目和现有项目对环境的影响，Deepki 首先让您全面了解您的碳排放表现。

Deepki 让你收集和汇总数据，如能源，水和废物。该公司试图使用 API、网络抓取或与 SFTP 服务器的连接来尽可能自动化这一过程。

这是一个很好的起点，可以让您了解自己与不同地区的法律要求相比的情况。您还可以提取数据并将其用于认证。

但是使用像 Deepki 这样的产品对于观察一段时间内的趋势特别有用。从 Deepki 的数据管理和监控平台上，你可以看到你的电力或水的消耗是如何随时间变化的。

拥有多个商业物业的大企业客户可以在 Deepki 上添加几个位置，并在平台上进行比较。通过这种方式，您可以优先选择一栋建筑来实施节能计划。这会改善你的整体碳轨迹。

C 轮融资由 Highland Europe 和 One Peak Partners 牵头。Bpifrance 的大型风险基金和 Revaia 也参与了投资。一些现有投资者也在拿出更多资金，如 Hi Inov 和 Statkraft Ventures。

“全球房地产行业要想在 2030 年前将排放量减半，并在 2050 年前实现净零排放的目标，现在就需要行动起来。这对 Deepki 来说是一个巨大的市场机会。今天宣布的新资金意味着 Deepki 可以产生更大的影响，并支持更多的资产所有者应对气候变化挑战，我们很高兴有我们的新合作伙伴 Highland Europe 和 One Peak，以及 Revaia 和 Bpifrance 大型企业踏上这一征程，”联合创始人兼首席执行官文森特·布赖恩特在一份声明中说。

如你所见，Deepki 为有特殊需求的客户开发了一个高度专用的 SaaS 平台。这就是为什么该公司还提供咨询服务，帮助 Deepki 的客户实施 ESG 战略和行动计划。

该公司目前在欧洲五个首都城市拥有 150 名员工。Deepki 目前监控着 38 个不同国家的建筑，面积超过 5 亿平方米。客户包括 AEW、Tikehau、Generali RE、DeA Capital、Allianz Real Estate、Warburg HIH、Azora Capital 和 Neinver。
