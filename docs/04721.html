小键盘触觉显示器让视障用户可以触摸图像| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2022/03/10/dot-pad-tactile-display-makes-images-touchable-for-visually-impaired-users/

盲文被视力障碍者广泛使用，但尽管网络和智能设备的可访问性得到了广泛改善，盲文阅读器硬件的创新基本上已经停滞不前。Dot 通过智能盲文设备向前迈出了一大步，这种设备不仅可以轻松显示文本，还可以显示图像的触觉表现，有可能为教育和无障碍内容开辟一个全新的层面。

小键盘由 2400 个类似像素网格的针组成，可以快速设置为上下位置，形成盲文字母或易于识别的形状。这是 300 个盲文符号的空间，再加上下面一条更传统的间隔线中的 20 个。至关重要的是，这款设备还直接集成了苹果的 VoiceOver 屏幕阅读功能，只需轻轻一点，就可以阅读文本、图标标签甚至图表或简单的图像。

这家总部位于韩国的公司是在联合创始人 Ki Kwang Sung 和 Eric Ju Yoon Kim 发现自己厌倦了缺乏学习和阅读的选择时成立的，尽管在计算和界面方面取得了许多其他进步。

这绝不是第一个数字盲文显示器——像这样的设备已经存在了几十年，但它们在数量和功能上都明显有限。最常见的是，你会发现盲文显示器用于阅读和输入数字文本，但这些通常是相对笨重的单行机器，多年来没有发生根本变化。

这些设备通常也不是为孩子和学习而设计的。有视觉障碍的儿童面临许多系统性的不利条件，例如缺乏教科书或为他们设计的活动。正是由于这种缺乏，使得一个这样的孩子的父母创造了 BecDot ，一种帮助幼儿学习盲文的玩具。

(这篇报道发表后，美国盲人印刷厂指出，传统盲文阅读器的开发仍在继续，新设备已经发布，包括一些针对学习者的设备。但总体设计和界面是熟悉的。)

“在 21 世纪，视障人士不能以数字方式访问图形信息是没有意义的，”Sung 说。“每个行业都有很多创新，包括教育、就业和社交网络服务……对图形信息的要求越来越高，这意味着视障人士越来越少。即使在疫情，远程工作和教育也是强制性的…但他们没有任何解决方案来做到这一点。”

他们决定制造一种显示器，让盲人和低视力用户能够访问基于像素的图像和表示，并与这些图像和表示进行交互，而视力正常的人认为这是理所当然的。

盲文阅读器通常在机械上极其复杂，依赖于数百个微小的铰链和齿轮来按需升高和降低针脚——它们还必须足够坚固，以承受触摸带来的持续压力。这些年来，我们看到了来自著名研究机构的各种创新，但没有一个真正进入市场。Dot 正试图改变这一切，不仅要有更好、更强大的硬件，还要与智能手机和平板电脑进行更深入的整合。

如你所料，Dot Pad 的核心创新是“点”本身。如何让几十或几百个这样的小针(每个盲文字母 6 个)可靠而快速地(并且不太大声地)伸出和缩回，已经产生了各种各样的解决方案，但 Dot 的是全新的。

图片来源:圆点

“我们从扬声器的机制中得到这个想法，”宋解释说。这种微小的电磁致动器在智能手机扬声器中振动，但该团队对其进行了改造，改为上下移动一根针，使用一个磁性球转子，该转子可以轻松锁定在上下位置，并可以快速解锁和消失。整个装置的尺寸只是以前装置的一小部分——“比现有的压电盲文致动器小 10 倍。”(Dot 私下给我看了引脚的示意图和剖面图，展示了它是如何做到的，但拒绝公开分享这些内容。)

这意味着该公司能够创建一个由数千个针组成的网格，针之间的空间非常小，足够大，可以作为字母阅读，但也足够密集，可以形成代表图像的图案。小键盘的底部有一个传统间隔盲文的专用部分，但主网格更好地描述为“触觉显示器”。

我玩了一个生产前的原型设备，它工作得非常好，从上到下刷新整个屏幕大约一秒钟(这也在改进，到了可以制作动画的地步)，用户的手似乎很容易扫描。这两种显示器都有一个灵活的保护屏幕，防止引脚被粘住，并可以很容易地更换。细胞本身也很容易被替换。

iOS 15.2 中为开发者提供了一个新的触觉图形 API，允许他们在应用中包含和调整这一功能。(我已经向苹果公司征求对 API 的意见，如果他们回复，我会更新这个帖子。)

“由于业界领先的屏幕阅读器 VoiceOver，世界上许多盲人/低视力用户都依赖 iPhone 和 iPad，”Kim 说。“我们非常高兴 Dot 的触觉技术现在针对 VoiceOver 进行了优化，这将扩大数字可访问性。除了语音或文学盲文，这些用户现在可以感受和提高他们对图像的理解。”

显然保真度有些有限，但它可以很好地显示图标、线条画和图形之类的东西。想象一下一篇关于股票的文章中的图表——视力正常的人一眼就能看出来，但其他人必须找到其他方法，比如内置在 VoiceOver 中的一种方法，它将图表表示为一种升调和降调。聊胜于无，但绝对不理想。Dot Pad 由 VoiceOver 和它自己的图像分析算法提供支持，将尝试在显示器上显示任何屏幕区域或元素。

文本也可以表示为一整页盲文字母(像正常情况一样按间隔行排列)或字母本身的形状。这使得用户可以更好地欣赏标识中的字体——当然，盲文中没有衬线。其实用那种触觉的方式去体验大尺度的铅字，听起来挺有意思的。

更重要的是，这对孩子们来说是一个很好的资源。一个患有视觉障碍的孩子会错过很多东西，并且能够轻松地描绘出字母、形状和其他人认为理所当然的简单图像，如房子、猫等……这可能会改变盲人社区 K-12 教育的游戏规则。

其中一部分当然是与最常见的设备紧密集成，因此它不仅仅是某个人在非常特定的情况下拥有的资源。iPhone 和 iPad 不仅是无处不在的现代数字设备，而且还有一个强大的可访问性堆栈，Dot 已经利用了这一堆栈。

当然，语音界面的巨大进步极大地增强了不能使用图形界面的人的能力，但盲文仍然是一个重要的选择，特别是对于阅读和学习来说。所有这些模式以及更多的模式都必须改进，这样机会才不会受到技术的限制。

来自机构群体的反馈是积极的；Sung 说，人们主要考虑的是可能性，而不是局限性。但从早期阶段开始，他们确实增加了“像素”数量，以更好地渲染图像，他们正在开发一个点 Pad 友好的自定义图形库，例如，如果 Twitter 徽标被软件识别，它就可以使用自己的版本，而不是每次都扫描轮廓。

Dot 将为即将到来的动态触觉设备项目提供核心技术，该项目由美国盲人和人类用品印刷厂领导，计划于 2023 年推出；开发人员社区将有机会根据他们在 API 方面的经验发表意见。

图片来源:圆点

未来的功能计划包括照片的触觉表现——不一定是图像，但布局、位置和对人的描述以及其他方面都可以放在显示器上。他们还在研究一种将别针锁定在中间高度的方法，用于感觉渐变和其他用途。此外，触摸板还可以用作输入和显示——能够按下触针向屏幕的适当部分发送触摸信号将是另一个有用的功能。

当然，像以前的盲文显示器一样，点板既不便宜也不简单——尽管它可能比其他产品更便宜、更简单。制造和组装不是一件容易的事情，总成本也很难说，尤其是现在芯片和其他组件的价格飞涨。(它使用了数以千计的微型集成电路，这些集成电路以前主要用于车窗控制开关——现在价格已经高得离谱了。)

幸运的是，这正是那种不需要任何人付费的设备，而且有许多补贴和其他项目。毕竟，孩子们不必为他们在学校使用的桌子等必需品付费。帮助残疾人接受良好的教育符合每个人的利益。更好的可访问性本身当然是受欢迎的，但它具有重大的连锁效应，因为无法学习或参与某个行业的人终于有机会了。

Dot 的创始人指出，他们正在与韩国和美国政府以及盲人社区和倡导组织合作，将 Dot Pad 与课程整合，并使用现有的资金和方法支付费用。开发者可以在这里和苹果开发者网站了解更多关于触觉图形 API 的信息。
