马斯克透露计划将特斯拉扩大到“极限尺寸”

埃隆·马斯克(Elon Musk)在开设该汽车制造商的第一家欧洲工厂的前一天，在推特上调侃特斯拉“总体规划第三部分”的发布时，暗示了将特斯拉规模扩大到“极端”的计划。

周一，马斯克在推特上透露了将主导特斯拉长期剧本下一部的主题:人工智能和扩大汽车制造商的运营。

马斯克在推特上写道:“主要的特斯拉主题将会扩展到极限尺寸，这是让人类远离化石燃料和人工智能所需要的。”。"但我也会包括关于 SpaceX、特斯拉和无聊公司的部分."

该计划可能会详细说明特斯拉的“极限规模”是什么样的，并概述了该汽车制造商在全球疫情和供应链紧缩的情况下扩大其制造和供应链的战略。

特斯拉计划于周二开设其在欧洲的第一家工厂，届时特斯拉将移交其在欧洲生产的第一批量产汽车。位于柏林的耗资 50 亿美元的工厂计划每年生产 50 多万辆电动汽车，以与欧洲最大的汽车制造商大众汽车及其 1000 亿美元的电动汽车投资竞争。

上周，由于中国奥米克隆案件的增加和供应链的限制，特斯拉的上海巨型工厂24 小时停工两天。该工厂每天生产约 2000 辆汽车，并向欧洲出口大量 Model 3 和 Model Y 汽车。

“总体规划”的前两部分被证明是公司产品和技术发展的准确路线图；第一部分是 2006 年的一篇题为“秘密的特斯拉汽车总体规划(只有你我知道)”的博客文章，概述了特斯拉的概念证明。

十年后，马斯克的总体规划更新，“第二部分”，讨论了开发电池存储和推出新车型的计划，包括一款电池电动皮卡和 SUV。现在该公司已接近第二部分的尾声，马斯克在 Twitter 上开始调侃特斯拉的下一章。

这也是总计划首次将重点放在独立的马斯克公司上，这加剧了人们的猜测，即他可能打算创建一个伞式公司，将他的所有公司置于一个屋檐下。然而，推文提到的 SpaceX 和 Boring 公司也暗示了两家公司未来的合作。

这篇文章已经更新，注意到伊隆·马斯克的一条推文，该推文指出柏林工厂生产的第一辆量产车将于周二分发。
