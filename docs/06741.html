马里兰州成为第二个在苹果钱包中提供驾照和州身份证的州

原文：https://web.archive.org/web/https://techcrunch.com/2022/05/25/maryland-becomes-the-second-state-to-offer-drivers-license-and-state-id-in-apple-wallet/

苹果公司今天宣布，马里兰州居民现在可以在苹果钱包应用程序中存储他们的驾照或州 ID。继三月份在亚利桑那州推出之后，该州现在是美国第二个支持这一新功能的州。一旦加载到苹果钱包中，马里兰德人将能够使用他们的 iPhone 或 Apple Watch 出示他们的 ID，以选择当地机场的 TSA 安全检查站。

最初，BWI 瑟古德·马歇尔国际机场和华盛顿国家机场罗纳德里根机场的 TSA 安全检查站将支持这些新的数字身份证。

要开始将支持的 ID 加载到苹果钱包应用程序中，马里兰州居民首先要在 iPhone 应用程序中点击屏幕顶部的加号“+”按钮，然后选择“驾照或州 ID”，并按照屏幕上的说明开始设置和验证过程。该人通过自拍进行验证，然后扫描其现有驾照或州身份证的正反面。

还有一个额外的防欺诈步骤，要求用户在设置过程中完成一系列面部和头部动作。该应用程序将呈现一个摄像头视图，用户需要将他们的头转向一侧。这是数字服务验证一个人确实是真实的，并且用户没有拿着照片对着相机试图进行欺诈的常用方法。

正如苹果之前解释的，这些扫描和用户的照片被安全地提供给发行国进行验证。此外，苹果公司还会发送一个数字指示符，表示其确信出示该 ID 的人是该 ID 的所有者。它没有像核查期间要求的那样发送该人移动头部的视频。

整个批准过程通常只需要几分钟，当用户的 ID 在 Wallet 中可用时，用户会收到通知，类似于添加新信用卡后收到的通知。

然后可以通过新的身份读取器在 iPhone 或 Apple Watch 上支持的 TSA 检查站使用该 ID。在获得 Face ID 或触控 ID 的授权后，所请求的信息会以数字形式从设备中释放出来。用户不必将手机或手表交给 TSA 代理。

“随着苹果钱包中马里兰移动 ID 的实施，马里兰州很自豪再次成为安全创新的领导者，”马里兰州州长拉里·霍根在关于此次发布的声明中说。“展望未来，我们致力于提高便利性和可访问性，同时为我们的州和公民保持最高的安全和安保标准。”

新闻稿指出，马里兰州的移动身份证，因为该州正在呼吁新的数字身份证，是唯一形式的手机为基础的身份证明由 MDOT MVA。数字身份证也只适用于持有马里兰州颁发的有效驾驶执照或身份证的个人，它旨在作为物理、塑料驾驶执照或身份证的伴侣，而不是替代物。换句话说，该功能不允许马里兰人跳过前往当地 MDOT MVA 办事处获取许可证或身份证的旅程。

据 MDOT MVA 管理员 Chrissy Nizer 称，该州预计接受这种移动 ID 的地点数量将随着时间的推移而增长，但目前，只有特定的 TSA 检查站可以使用它。
