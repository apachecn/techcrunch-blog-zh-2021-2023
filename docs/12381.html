抖音开始测试其平台研究 API 的早期版本

今年早些时候，抖音宣布它正在开发一个研究 API，以改善对其应用程序内容和活动的公开和匿名数据的访问。现在，该公司表示，它准备推出其平台研究 API 的测试版，并要求其内容和安全顾问委员会的成员测试该 API 的早期版本。

“为了开始，我们已经要求我们的内容和安全咨询委员会成员在错误信息、暴力极端主义、仇恨行为和新兴技术方面具有专业知识，以测试我们平台研究 API 的早期版本，”抖音在一篇博客文章中说。“当我们收集他们对可用性和整体体验的反馈时，他们可以访问公共数据。我们致力于听取和整合测试人员的反馈，并创建一个既能满足科学界需求，又能尊重我们社区隐私的 API。”

在最初宣布的时候，抖音说，研究人员目前没有一种简单的方法来评估内容或在其平台上进行测试，这就是为什么它看到了研究 API 的需求。

除了平台研究 API，抖音正在开发一个内容审核 API。该公司计划在未来几个月分享更多细节。审核系统 API 将为选定的研究人员提供一种方法来评估抖音的内容审核系统，并检查应用程序上现有的内容。研究人员还将能够上传他们自己的内容，以了解不同类型的内容是如何被允许、拒绝或传递给版主进行进一步评估的。

抖音对其研究 API 工作的更新正值 FCC 专员 Brendan Carr 再次呼吁禁止该应用。这不是卡尔第一次提出这个想法。在 BuzzFeed News 报道了泄露的内部通信所暗示的数据不当行为后，卡尔在 6 月份写信给苹果和谷歌称该应用程序是“不可接受的国家安全风险”,并要求两家公司将其从应用商店中移除。
