印度央行表示，类似于“庞氏骗局”的加密货币并禁止它是“也许是最明智的选择”

原文：https://web.archive.org/web/https://techcrunch.com/2022/02/15/india-central-bank-cryptocurrency-ponzi-banning/amp/

图片来源:PUNIT PARANJPE/AFP/Getty Images

印度央行的一名高级官员将加密货币比作“庞氏骗局”，并建议彻底禁止加密货币，这是印度政府提议对虚拟数字资产征税并为承认其为全球第二大互联网市场的法定货币铺平道路后的几周内最尖锐的批评。

T.印度储备银行(RBI)副行长拉比·桑卡尔(Rabi Sankar)在一次银行业会议上告诉观众，加密货币是“专门为绕过受监管的金融系统而开发的”，没有任何基础现金流作为支持。

“我们还看到，加密货币不适合定义为货币、资产或商品；它们没有潜在的现金流，没有内在价值；他们类似于庞氏骗局，甚至可能更糟，”他说。

桑卡发表此番言论之际，印度政府已经发出信号，表示正朝着承认数字虚拟资产为法定货币的方向前进。该国财政部长 Nirmala Sitharaman 本月早些时候提议在联邦预算中对加密货币和 NFT 的转让收入征税。

尽管监管存在不确定性，但过去一年加密货币和 NFT 的销售在印度取得了快速进展。根据研究公司 chain analysis 的分析，世界第二大互联网市场的加密货币投资采用率排名第二。

“这些交易的规模和频率使得提供一个特定的税收制度成为当务之急，”她在预算演讲中表示。

印度央行迄今对加密货币非常谨慎。2018 年，它禁止金融公司处理加密货币。两年后，印度最高法院推翻了该禁令，但是大多数银行继续遵循印度储备银行的方向。

Sitharaman 周一表示，新德里和印度央行正在进行讨论，以制定规则，双方“在船上”

桑卡的讲话清楚地表明，印度央行没有改变其长期以来的立场。“作为一种价值储存手段，比特币等加密货币迄今为止已经带来了令人印象深刻的回报，但 17 世纪荷兰的郁金香也是如此。加密货币非常像一种投机或赌博合同，就像庞氏骗局一样。事实上，有人认为，从社会角度来看，查尔斯·庞兹在 1920 年设计的原始方案比加密货币更好，”他说。

他警告说，加密货币可能会“破坏”货币体系、货币当局、银行系统，以及总体上政府控制经济的能力。

“它们威胁到一个国家的金融主权，并使其容易受到创造这些货币的私营企业或控制这些货币的政府的战略操纵。所有这些因素导致的结论是，禁止加密货币可能是印度最明智的选择，”他说。“我们研究了那些主张加密货币应该受到监管的人提出的论点，发现它们都经不起基本审查。”
