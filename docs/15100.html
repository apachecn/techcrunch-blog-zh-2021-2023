Twitter 在宣布新的 API 定价(更新)后，将其开发者社区网站放在登录之后

2022 年 13 月 2 日更新: Twitter 开发者论坛终于在周日开放了。然而，该公司没有提供任何理由来解释为什么该网站被放在登录之后。

Twitter 今天早些时候宣布了其新的 API 定价，称其“低使用率”的基本层将每月花费 100 美元。当开发者试图寻求新规则的清晰性时，他们访问了开发者论坛网站，发现该网站已经被关闭。

图片来源: TechCrunch

Twittercommunity.com一直是开发者们询问关于 Twitter APIs 的问题并获得为该平台开发应用的技巧的地方。Twitter 员工也回复了发布 bug 和问题的开发者，告知他们公司层面的进展。它也是 Twitter 上某些开发者相关公告的门户。

几位开发者在 Twitter 上指出该网站无法访问，要求该公司提供更新。

尚不清楚 Twitter 是否将开发者论坛置于登录之后，或者这是一个错误。从昨晚开始，平台上发生了很多事情，包括人们在发布推文、访问回复和直接消息方面存在问题。这也可能是因为该公司推出了一个更新，让 Twitter 蓝色用户发布 4000 字符长的推文。

几个小时前，埃隆·马斯克在推特上说，这家社交网络正面临“多重内部和外部问题”，但这些问题应该在美国时间晚些时候得到解决。

然而，在撰写本文时，人们在访问服务的不同部分时仍然面临问题。希望 Twitter 能尽快解决所有这些问题。
