为什么我使用信贷工具来发展我的创业

想要加速增长并保留更多股权的企业家可能会理解 SPACs、P2P 贷款和众筹，但对于一些初创公司来说，获得信贷服务也是一个可行的选择。

有了信贷额度，初创公司可以增加招聘和产品开发，利用额外的资源在市场上验证想法。根据你的业务，向客户提供信贷也可以促进增长并锁定财务稳定。

Torpago 首席执行官 Brent Jackson 表示:“对于我们的业务模式来说，提高信贷额度来为我们客户的所有支出提供资金是最明智的做法。

他的公司获得了 7700 万美元的融资，“其中 7500 万美元是循环信贷，其余的是股权。”

这样做允许杰克逊向客户提供信用额度，“并以最小化长期资本成本的方式将债务纳入我们的资本栈。”

在一篇 TC+客座博文中，他向读者介绍了提高债务净值、让员工了解情况以及寻找合作贷款人的过程。“有很多东西需要学习，”他承认道。

在两次被拒绝的交易后，Zendesk 考虑它的下一步

图片来源: NurPhoto / Getty Images

Zendesk 做得非常好:2021 年，营收同比增长 30%。看一眼其收益公告的前景部分，就可以看出更多的增长正在酝酿之中。

但在发布这些业绩的同一天，该公司还拒绝了一个由私募股权公司组成的财团提出的 170 亿美元的收购，称这笔交易“低估了公司的价值”。

当时，Zendesk 试图以 41.3 亿美元收购 Momentive/SurveyMonkey，但上周五，我们得知 Zendesk 的股东并不像首席执行官/创始人 Mikkel Svane 那样渴望进入客户体验业务。

既然重要的交易已经失败，罗恩·米勒和亚历克斯·威廉进行了一次事后分析。

“Momentive 的潜在收入是否足以证明 Zendesk 准备支付的价格、它对客户体验市场的投入以及它会导致收购者偏离其核心客户服务方向的事实是合理的？

你在 BNPL 的创业公司到底值多少钱？

消费者被停滞不前的工资和通货膨胀压得喘不过气来，但许多人不介意背负一些债务，如果这意味着他们可以拥有新的魅力。

Alex Wilhelm 在分析澳大利亚 BNPL 公司 Zip 收购竞争对手 Sezzle 的提议时指出，这种行为正在增加 BNPL 公司的财富，但它们的估值将取决于找到市场、客户群和收入模式的正确组合。

Sezzle 和 Zip 的收入远低于规模更大的竞争对手 Affirm，这可能是因为后者在美国的业务以及更高的使用率。

“但 Zip 和 Sezzle 与 Affirm 的 BNPL 收入之间的巨大差距应该让 BNPL 的初创公司暂停，”他写道。

“你的创业公司更像 Affirm 还是更像它的小竞争对手？如果你的定价更像 Affirm，为什么？保费你配得上吗？”

实施差异隐私以促进数据共享和合作

保护客户隐私至关重要，但随着越来越多的数据混合在一起，许多组织都做不到这一点。

Sarus 的联合创始人兼首席执行官马克西姆·阿戈斯蒂尼(Maxime Schmidt)和数据孵化器的创始人李立伟(Tianhui)写道，差分隐私是这个问题的答案，这要归功于一种方法，这种方法涉及“共享结合随机噪声的数据处理结果，以便输出不会告诉潜在攻击者关于目标的任何具有统计意义的信息”。

和李解释了差分隐私是如何工作的，如何选择正确的架构来实现它，并促进数据共享，还包括了一个公司可以开始使用的开源库列表。
