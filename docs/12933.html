埃及的 SideUp 筹集 120 万美元发展其电子商务支持平台 TechCrunch

在主要作为物流市场运营了四年之后，埃及的 Voo 已经更名为 SideUp ，并转变了其战略，以提供完整的电子商务支持服务，包括支付网关、针对运输、仓储、履行和咨询的 API 集成。

在筹集了 120 万美元的种子资金后，这家初创公司还扩展到了沙特阿拉伯，此后将把总部设在那里。最近一轮的参与方包括 Launch Africa VC、500 Global、Riyadh Angels、Alex Angels、Al Tuwaijri Fund 和沙特天使投资人 Faisal Al-Abdulsalam。

SideUp 创始人兼首席执行官瓦利德·拉希德告诉 TechCrunch，在意识到小商户被大型服务提供商忽视后，他受到启发，将小商户与电子商务支持联系起来。

“有很多关于电子商务如何扩大规模的讨论，但我们仍然没有为那些在线销售的企业(微型、小型和中型企业)提供足够的支持。商家需要许多服务和完整的生态系统才能成功，”拉希德说。

“这就是我决定支持中小企业的原因；SideUp 面向村里的商家，或者那些通过 Instagram、脸书或 WhatsApp 销售产品的商家。他们可以获得从快递公司、仓储和履行到营销服务的所有服务，”他说

拉希德在银行业结束职业生涯后，于 2012 年首次涉足创业，当时他创办了一家跑腿公司 Ingez，这让他获得了经营电子商务业务的第一手经验。

“在这四年中，我了解了很多关于物流、运营和电子商务的事情。我看到了小企业因为缺乏业务量而没有成为大型物流公司的优先考虑对象，”Rashed 说，他在离开 Ingez 后创建了 SideUp，以解决小企业面临的挑战并帮助它们扩大规模。

成长

SideUp 的合作伙伴服务提供商使商家能够在 45 个国家销售，Rashed 表示，这开辟了以前小企业无法进入的新市场。

SideUp 目前为 2，000 家电子商务企业提供服务，这些企业还可以获得现金收款服务和信贷，以扩大业务。其产品组合月环比增长 30%，每月总商品量超过 50 万美元。

利雅得天使联合创始人哈立德·阿尔·塔维尔博士说:“在该地区的大部分地区，电子商务仍然是分散的，给企业主带来了许多挑战。SideUp 的平台是一个巨大的飞跃，为他们提供了一个接触合作伙伴和技术的单一场所，使他们的业务呈指数级增长。我们很高兴看到他们来到沙特阿拉伯，支持企业度过下一个增长阶段。”

这家初创公司现在计划通过在埃及和沙特阿拉伯扩大规模来扩大客户群，并在 2023 年底前至少扩展到另外两个国家，以利用不同地区蓬勃发展的电子商务行业。

根据 Mordor Intelligence 的一份报告，未来五年，中东和非洲的电子商务市场将增长 11.5%以上，这是由互联网和智能手机的渗透、城市化和移动购物以及更多企业接受电子商务支撑的。
