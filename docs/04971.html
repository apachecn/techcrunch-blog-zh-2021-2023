NFT 初创公司宇迦实验室以 40 亿美元的估值筹集种子资金

许多 NFT 怀疑论者讨厌的价值数百万美元的猴子 JPEGs 图片的制造商宇迦实验室周二宣布，该公司刚刚以 40 亿美元的估值从安德森·霍洛维茨获得了 4.5 亿美元的融资。

Bored Apes Yacht Club 背后的 NFT 公司此前没有筹集到资金，尽管这家初创公司长期以来一直吸引着渴望支持 NFT 热潮中一个主要参与者的风投的关注。这轮投资的其他投资者包括 Animoca Brands、LionTree、Sound Ventures、Thrive Capital、FTX 和 MoonPay。

宇迦实验室正越来越多地对其地位加倍下注。本月早些时候，这家初创公司宣布从幼虫实验室收购了 NFT 热门项目 CryptoPunks 和 Meebits 的资产。这家初创公司还刚刚推出 ApeCoin，该公司的创始人和高管在其中持有大量股份。这种代币在交易的第一天就聚集了数十亿美元的市值。这家初创公司很快希望将这种势头融入自己版本的元宇宙，名为“ Otherside ”，该版本还将整合 NFT 其他一些项目的头像。
