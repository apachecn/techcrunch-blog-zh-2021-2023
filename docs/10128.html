网络安全公司 Fortanix 获得了提供机密计算服务的资金

原文：https://web.archive.org/web/https://techcrunch.com/2022/09/15/cybersecurity-firm-fortanix-secures-capital-to-provide-confidential-computing-services/amp/

图片来源: Traitov / Getty Images

公司(和风投)在网络安全上花费了数十亿美元，但主要集中在保护基础设施或终端上。由于疫情，数据越来越多地分布在云、软件即服务应用和存储系统之间，在这样的世界里，这种方法并不总是正确的。根据【2021 年的一项调查，61%的企业安全主管认为他们的网络安全团队人手不足。

Ambuj Kumar 通过电子邮件告诉 TechCrunch:“企业和政府机构正在寻找一种新的方法来保护他们的数据安全，无论数据在哪里，尤其是在云中。”他是 Fortanix 的首席执行官和联合创始人，该公司旨在将安全与网络基础设施分离，以确保数据安全，即使基础设施遭到破坏。“他们需要在其整个生命周期中保护敏感和受管控的数据，包括静态、动态和使用中的数据。”

Kumar 有点像推销员，他认为 Fortanix 是应对日益增长的数据安全挑战的更全面的解决方案之一。一些投资者同意这一观点。Fortanix 今天完成了由高盛增长股权(Goldman Sachs Growth Equity)牵头的 9000 万美元 C 轮融资，参与方包括 Giantleap Capital、Foundation Capital、Intel Capital、Neotribe Ventures 和 In-Q-Tel(美国情报界的非营利战略投资者)，使其融资总额达到 1.22 亿美元，库马尔表示，这笔资金将主要用于扩大销售和营销业务，并在荷兰埃因霍温开设新办事处。

“在整体市场放缓的背景下，增长轮次已大幅放缓。与此同时，投资者有很多干粉，收入状况和盈利商业模式强劲的公司会吸引很多注意力，”库马尔说。“我们很幸运能成为这样一家公司。历史表明，伟大的公司都是在困难的经济环境中建立起来的，因为竞争对手得不到资金，人才变得更容易获得，客户朝着更强的产品整合。”

库马尔在 2016 年与阿南德·卡什亚普一起创立了福塔尼克斯。Kumar 之前是 Nvidia 的硬件设计主管和 Rambus 加密部门的首席架构师。Kashyap 在成为 VMware 的员工工程师之前是赛门铁克的首席安全研究员。

Fortanix 出售对软件的访问，这些软件可保护公共云、混合云、多云和私有云环境中的数据，并对数据库进行加密，以及在云和内部管理应用机密(例如用户名和密码)。除了加密服务，Fortanix 还提供机密计算，这是一种云计算技术，在处理期间将敏感数据隔离在加密的 CPU 飞地中，以便飞地的内容只能由授权的编程代码访问。

Fortanix 的保密计算技术建立在英特尔成熟的 SGX 平台之上。基于硬件的安全技术使用 CPU 内的可信硬件来创建上述飞地，例如，允许金融服务和医疗保健等受监管行业的数据团队使用私人数据，同时保持匿名。

在这些用例的推动下，至少有一家公司预计到 2026 年机密计算市场将价值 540 亿美元。近年来，机密计算技术的采用确实在加速，英特尔、谷歌、微软、Arm 和 Red Hat 等科技公司成立了一个组织——机密计算联盟(Confidential Computing Consortium)——来推进数据保护标准。拥有竞争对手机密计算解决方案的初创公司包括 Opaque Systems ，Edgeless Systems 和 Decentriq。

Kumar 继续说道:“为了更好地保护敏感数据，从生命周期的多个维度来考虑是有帮助的，例如，当数据处于闲置、传输或使用状态时。“通常，第三个层面，即数据在使用时，会因为不充分的保护机制或错误的安全概念而被忽视。最近，在使用状态下发生了几起严重的恶意软件攻击，包括 Triton 攻击和乌克兰电网攻击。[Fortanix 的]技术可保护应用程序和“使用中”的敏感数据，使其在极易受到攻击的情况下免受未经授权的访问和篡改，从而扩展了网络中静态和动态数据的现有安全性。”

Fortanix 声称在全球拥有超过 125 个客户，包括阿迪达斯、谷歌、贝宝、通用电气医疗保健、美国司法部和疾病控制和预防中心。合作伙伴包括 Elastx 和阿里云，它们在其平台上运行 Fortanix 的密钥管理服务，以及 Equinix，它们利用 Fortanix 来支持其定制的安全服务。IBM Cloud 是另一个合作者——它与 Fortanix 合作提供保护使用中数据的服务。

“在大多数情况下，我们的主要工作是说服客户，仅仅努力保护他们的网络安全是不够的，并教育他们数据第一安全的好处。在少数情况下，我们与 Thales 和 HashiCorp 等传统数据安全供应商竞争，”Kumar 补充道。

当被问及收入数字时，库马尔拒绝透露。但他向我保证，Fortanix 将继续增长，计划在明年将 225 人的员工人数增加 50%。

“我们当然对宏观经济状况很敏感。然而，数据安全和隐私是全球企业最关心的问题，我们继续看到对我们产品的强烈需求，”库马尔说。
