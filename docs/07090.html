ComfortDelGro 使游客能够通过移动应用程序支付车费

原文：https://web.archive.org/web/https://techcrunch.com/2022/06/08/singapores-taxi-operator-comfortdelgro-enables-tourists-from-malaysia-and-south-korea-to-use-mobile-wallets-apps-for-paying-fares/

新加坡的出租车运营商 ComfortDelGro 允许来自马来西亚和韩国的游客使用移动钱包应用支付车费

新加坡出租车运营商ComfortDelGro周四宣布与支付宝+合作，使马来西亚和韩国的游客可以使用他们的手机钱包应用Touch ' n Go eWallet和Kakao Pay支付新加坡的出租车费。

ComfortDelGro Taxi 的首席执行官 Jackson Chia 说:“随着新加坡重新开放边境，我们期待着游客来参观我们。“因此，我们与支付宝+的合作至关重要，因为它允许在 ComfortDelGro 出租车上使用无现金支付选项，如马来西亚的 Touch 'n Go eWallet 和即将推出的韩国 Kakao Pay，使马来西亚和韩国游客支付出租车费更加方便。”

Chia 补充说，这一合作将提供便捷的支付选择，无需兑换当地货币，因为客户可以在自己的移动应用程序上预先支付出租车费——Touch ' n Go e wallet 或 Kakao Pay。

马来西亚的 Touch 'n Go 现在可以在出租车上使用，而韩国的 Kakao Pay 也将很快推出。

根据《海峡时报》的报道，2022 年前 4 个月，新加坡的游客数量已经超过了去年全年。该公司表示，马来西亚是游客最多的国家之一，韩国也是新加坡的重要客源市场。

新加坡最大的出租车运营商，在该国拥有约 9000 辆舒适出租车和城市出租车，目前支持 20 多种不同的无现金支付选项，而支付宝+支持全球各地的电子钱包，包括菲律宾的 GCash、泰国的 TrueMoney、支付宝香港等。

“通过与 ComfortDelGro Taxi 的支持和合作，我们的目标是向支付宝+支持的移动支付的所有用户提供无缝和智能的数字旅行体验，他们可以在新加坡用他们熟悉的移动钱包支付，”蚂蚁集团南亚和东南亚全球商户合作伙伴团队总经理兰子君说。“这种伙伴关系也来得非常及时，因为新加坡在开放边境方面处于领先地位，并欢迎游客回到这个城市国家。”
