Netlify 获得 1 . 05 亿美元的 D 轮融资，推动以 Jamstack 为中心的网络开发

原文：https://web.archive.org/web/https://techcrunch.com/2021/11/17/netlify-snags-105m-series-d-to-fuel-jamstack-focused-web-development/

Netlify 很早就有了将网站的前端设计和后端业务流程分开的想法。当这些技术还没有被很好理解的时候，它就采用了 Jamstack 和微服务。这是一个相当常见的方式来建立一个网站，但并不总是这样，当他们开始。

或许正是这种知识和理解吸引了投资者来到这家公司。今天，Netlify 宣布了 1.05 亿美元的 D 轮融资，估值为 20 亿美元。这距离该公司的5300 万美元 C 轮融资已经过去了 18 个月。

Bessemer Venture Partners 领投了今天的这轮融资，现有投资者包括 Andreessen Horowitz、Bond、EQT Ventures、Kleiner Perkins、Mango Capital 和 Menlo Ventures。根据 Crunchbase 的数据，该公司现已融资近 2 亿美元。

Netlify 首席执行官兼联合创始人马特·比尔曼(Matt Biilmann)表示，在早期，他们必须教育和说服人们，Netlify 试图带来的建筑变革是真实的。今天，情况不同了。

“因此，这种架构转变绝对正在发生，而且还在加速。这就是为什么我们也提出了下一轮的背景，以真正加速这一进程，推动开发者革命，推进[这种]现代的网站开发方式，并解决下一系列挑战。Biilmann 告诉我说:“当建筑领域的这种转变变得真实而持久时，这就开始了。

他说，现在的目标是解决下一个层次的摩擦，包括管理多个 API 以及开发人员如何处理这些不同元素之间的所有依赖关系。这就是为什么该公司最近收购了早期创业公司 OneGraph ，这是Y Combinator 2018 年夏季团队的成员，该团队基于 GraphQL 构建了一个解决方案，将这些不同的 API 在后端连接在一起。

“[OneGraph 技术]将成为我们下一阶段产品开发的重要组成部分，”比尔曼解释道。“我们如何帮助团队处理如此多不同 API 和服务的复杂性，并使 Jamstack 的构建体验真正像一个统一的应用程序体验，同时充分利用这个新世界，在这个新世界中，每样东西都有一个 API，并且您的 UI 与所有这些不同的部分都是分离的。”

该公司声称有超过 200 万开发人员使用该平台，尽管许多人都在使用免费层。该公司花了五年时间达到 100 万用户，仅用了一年时间就翻了一番，所以事情进展很快。

Netlify 目前有 200 名员工，并计划在下一年将这一数字翻一番，因为它将投入新的资本。公司联合创始人克里斯蒂安·巴赫说，Netlify 拥有多样化的客户群，公司员工反映这一点很重要。

“这对于我们来说一直都非常重要，因为我们是一家全球性的企业公司，为任何为网络而构建的人提供服务。因此，这是一个非常多样化的目标受众，为了让我们拥有一个足够聪明的组织，我们也需要多样化，”他说。

Netlify 还宣布了一项 1000 万美元的 Jamstack 创新基金，以鼓励和支持 Jamstack 生态系统中的创新，支票金额在 10 万美元左右。此外，它还将另外留出 100 万美元投资于生态系统的相邻开源工具。
