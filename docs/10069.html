谷歌已经发布了新的 31 个表情符号版本

原文：https://web.archive.org/web/https://techcrunch.com/2022/09/14/google-has-already-published-its-version-of-the-new-31-emojis/

谷歌已经发布了新的 31 种表情符号

Unicode Consortium 是一个维护智能手机官方表情目录的组织，它于周二发布了 Unicode 15.0 标准，其中包括 31 个新的表情，包括一只推手、一张颤抖的脸(或我被摇晃了)、一只驼鹿、一只鹅、期待已久的粉红色心形和一个 Wi-Fi/无线标志。

我们第一次看到这 31 个表情符号的插图是在 7 月份，这是过去几年 T2 新增数量最少的表情符号之一。现在它们被批准了，手机制造商、操作系统制造商和应用程序制造商将实现他们自己的版本，把它们带到你的设备上。为此，谷歌已经发布了这些新的表情符号及其网络的 Noto 字体——这样开发者就可以很容易地将它们嵌入到他们的项目中。

Unicode 15.0 中表情符号的草图。图片来源:表情百科

这家搜索巨头表示，新的表情符号将于今年年底在 Android 上推出，明年在谷歌的其他产品上推出。除了更新的 Noto 字体的彩色版本，该公司还发布了带有新表情符号的单色版本。谷歌今年早些时候首次推出单色表情符号，作为对老式功能手机表情符号的认可。

更重要的是，谷歌正在发布其第一套动画表情符号供开放使用，其中一些已经可以在该公司的 Messages 应用程序中使用。

图片来源:谷歌

该公司还采用了 COLRv1 标准，让你可以改变字体的一部分——在这里是表情符号。例如，你可以用鸭子表情符号来尝试这个演示，混合不同部分的颜色来创建一个新版本。目前，只有谷歌 Chrome 和微软 Edge 支持 COLRv1，Firefox 支持即将推出。然而，你还不能通过消息应用程序发送这些混合表情符号。

谷歌变色表情符号演示的结果。图片来源:谷歌

不过，谷歌允许用户通过基于 G-board 的表情厨房来改变他们的表情颜色。你可以在现有表情符号中添加一个彩色心形来改变它的颜色。例如，将一朵红玫瑰和一颗黄色的心混合在一起，就会产生一朵黄玫瑰。

图片来源:谷歌
