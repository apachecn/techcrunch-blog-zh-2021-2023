为什么埃隆·马斯克和特斯拉需要蒂姆·库克

原文：https://web.archive.org/web/https://techcrunch.com/2022/03/01/elon-musk-and-tesla-need-a-tim-cook/

同样的故事已经上演了几十年。

一个创业公司在一个传奇人物的领导下茁壮成长。随后很快就进入平台期。该公司的重点从不断推出新产品转移到确保所生产的产品满足客户的需求。蜜月期结束了，一个新的时代开始了，这个时代关注的是持续大规模交付高质量产品所需的日常工作。

在某些时候，每个公司都需要一个蒂姆·库克。如今，这家公司就是特斯拉。

现在，在特斯拉的等级制度中没有明显的继承人。如果首席执行官(也是特斯拉的技术大王】埃隆·马斯克决定辞职，专注于他的其他公司或他的音乐事业，没有人会接手。特斯拉就是马斯克。他什么都插手。

他解散了公关部门，并通过 Twitter 发布公司消息。给 X 型车加装猎鹰翼门是他最后一刻的想法。他坚持认为公司可以建造一个几乎完全自动化的汽车工厂。他还实现了用加密货币支付的能力，然后取消了这一选项，然后将其重新引入特斯拉汽车和服务。

马斯克承认，他在公司驱动大多数新程序，“不是因为我想，而是因为我不得不在公司历史早期的深度参与是有帮助的。但是，如果公司的未来依赖于制造高质量的产品，那么一个人统治一切可能是灾难性的。

例如，制造机器的决定将 Model 3 的生产推迟了几个月，并创造了首席执行官所说的“生产地狱”。马斯克最终承认创建一个几乎完全自动化的工厂是一个错误。因为这个决定，Model 3 的生产被推迟了。

的确，今天特斯拉正春风得意，公布了季度利润和创纪录的汽车交付量。这个好消息也伴随着一系列的问题。一个人对一个成长中的组织进行微观管理不可能跟上生产增长时出现的问题。在某些时候，值得信任的人需要有机会做好自己的工作和控制问题— ,并且能够对首席执行官提出的任何随机想法进行反击，而不用担心遭到报复。

目前，特斯拉正在处理过去几个月中发布的多起召回事件，该汽车制造商仍然受到车辆制造问题报告的困扰。另外，新车(包括赛博卡车)已经被推迟到至少明年。这些问题是首席执行官需要在能够独立工作的团队的帮助下解决的。

然后就是推文了。这么多推文。

马斯克在推特上的表现……两极分化。这也可能对品牌不利。特斯拉内部没有新闻部。所有信息要么来自特斯拉官方账户，要么来自马斯克。如果这位首席执行官的个人账户没有跑题，只提到拥有一辆特斯拉的诸多好处，那就没问题。相反，它是迷因、糟糕的笑话、侮辱、对马屁精的回复、充满政治色彩的镜头和偶尔的特斯拉新闻的混合。

作为一名首席执行官，你不能在 Twitter 上将某人比作希特勒，而不认为这会对你的公司产生负面影响。

为了公司的利益，特斯拉需要着眼于未来，而不是其首席执行官的下一个突发奇想。马斯克不在的时候。最好的情况是苹果发生的事情。

多年来，苹果公司的业务与其首席执行官史蒂夫·乔布斯紧密相连。他将公司从破产的边缘拉了出来，并与他的团队一起建立了一个持久的公司，帮助启动了智能手机革命。可以肯定地说，没有乔布斯就没有苹果。但是一个公司不可能永远是一个人。

蒂姆·库克于 1998 年加入苹果，担任全球运营高级副总裁，负责整个公司的日常管理。2011 年，他在史蒂夫·乔布斯去世前被任命为首席执行官。在他的领导下，库克的收入和利润翻了一番多。库克没有乔布斯那种虚张声势或远见卓识。相反，库克雇佣聪明人来做他们的工作，确保苹果继续盈利并提供优质产品。

特斯拉需要一个蒂姆·库克。这家汽车制造商需要一位领导者(或培训中的领导者)，能够继续提供屁声和视频游戏等有趣的功能，并专注于确保车辆按时交付给客户，没有问题。特斯拉需要确保在未来几年内，其客户可以随时找到服务中心，并能及时更换车辆。不管喜欢与否，其他汽车制造商正在迎头赶上，并乐于用自己的电动汽车吸引 Model 3 车主。

这家汽车制造商可能利润很高，但如果没有适当的培育，特斯拉的电动汽车优势可能会被其竞争对手之一篡夺。马斯克真的不是那种善于培养的人。在他的领导下，车辆交付量创下历史新高，但仍不断有车辆制造质量问题的报告。那是一个几年前就应该解决的问题。

最近的报告显示，特斯拉一直在交付没有冗余转向控制器的车辆，并且未能告知客户这一缺失项目。如果车辆是 FSD 就绪的，则需要该控制器。这种类型的决定会侵蚀客户忠诚度，并拒绝其他人进入该品牌。

如果当前的客户对他们的特斯拉有问题，让它得到服务仍然充满误解，漫长的等待时间和挫折感。厌倦了试图进行维修的客户已经在 Twitter 上分享他们的经历，却遭到了马斯克谄媚的追随者的攻击，马斯克本人甚至没有试图控制这个群体。

然后是召回。在过去几年中，特斯拉应美国国家公路交通安全管理局的要求，对 FSD 的软件问题进行了召回，这些问题可能导致制动器在没有警告的情况下接合、安全带编钟、Model 3 和 Y 的悬架问题以及 Model S 和 X 的腐蚀动力转向螺栓等等。最近，特斯拉为车辆增加的功能现在必须被删除。这些召回是最麻烦的。

每个汽车制造商都必须处理召回，但不是每个汽车制造商都必须逆转功能，因为它们被视为不安全。在马斯克的领导下，特斯拉推动了汽车在道路上行驶时从外部扬声器播放司机想要的任何声音的能力，潜在地掩盖了行人警告声音。它还推出了一项更新，允许车辆通过停止标志而不是停止。此外，它必须被告知在有人驾驶车辆时，取消在主信息娱乐屏幕上玩视频游戏的功能。

作为对召回的回应，马斯克在推特上称 NHTSA 是“有趣的警察”实际上，该机构是安全警察，对特斯拉和公众来说，汽车制造商与他们合作，为每个人保持道路安全，而不仅仅是特斯拉司机想在街区里玩 Humpty 舞，这符合他们的最大利益。

为了帮助马斯克、特斯拉及其客户，最好的情况是有人与首席执行官一起工作几年，并能在公司内部建立信任，与董事会建立信任，更重要的是与客户建立信任。围绕马斯克的个人崇拜要求他全心全意地接受这个人作为领导者，所以当马斯克离开或从首席执行官的位置上退下来时，群众得到了安抚。

马斯克放弃对一家公司的完全控制权并非没有先例。他可能是 SpaceX 的首席执行官，但他与总统和首席运营官格温·肖特维尔(Gwynne Shotwell)分担领导责任。马斯克可能会在推特上发布关于 SpaceX 的消息，但主持大局的是肖特维尔。她不仅仅是该公司的公众形象，尽管她是最经常代表 SpaceX 出席会议和活动的高管，似乎最能阐明 Starlink 产品背后的愿景和战略。作为总统和首席运营官，负责日常运作的是肖特维尔。如果马斯克决定辞去首席执行官一职，肖特维尔已经准备好接任。

对于 SpaceX、其董事会、美国政府和合作伙伴公司来说，这肯定是一种解脱。把太空旅行押在一个人身上是一个可怕的赌注，因为最终，那个人会离开。

可惜特斯拉和马斯克是一体的。据公众所知，现在没有 B 计划，除了可能让动力和能源工程高级副总裁安德鲁·巴格里诺接管首席执行官的职位。有可能已经有了接班人计划。但如果没有透明度，实施这一计划可能会导致混乱，这对股价可能是毁灭性的。

马斯克目前在运营特斯拉，据我们所知，未来还会继续运营。但他不会永远这样——即使他是，也不能保证他累积的失误不会最终影响到公司。为了员工、股东和客户的利益，我们需要制定一个计划，并立即采取行动。
